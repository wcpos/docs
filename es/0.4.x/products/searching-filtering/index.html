Searching and Filtering

With WooCommerce POS you can search for product by title and also by attribute. For example, on the demo site a search for woo will filter by title, a search for cat:posters will filter products by the Posters category and a search for cat:posters woo will filter products by Posters category with 'woo' in the title. If you wanted to select the Movie Posters category you could use cat:"movie posters".

Below is a list of some currently supported search filters.
