---
title: Filtrer les commandes par PDV ou ventes en ligne
description: Séparez les commandes PDV et en ligne en utilisant les liens rapides dans l'administration WooCommerce.
---

Vous pouvez rapidement séparer vos commandes PDV et en ligne dans l'administration WooCommerce en utilisant les liens rapides en haut à gauche.
**Veuillez noter :** ce filtre ne s'appliquera aux commandes PDV qu'à partir de la version 0.3.1 ; les commandes antérieures à la version 0.3.1 seront classées par défaut comme commandes en ligne.

![Filtrer les commandes](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrer les commandes")

De plus, si vous souhaitez filtrer toutes les commandes terminées de votre espace d'administration afin de vous concentrer sur les commandes en attente de paiement ou en cours, vous pouvez ajouter le code suivant à votre fichier [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
