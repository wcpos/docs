---
title: Filtrer les commandes par ventes en point de vente ou en ligne
description: Séparez les commandes de point de vente et en ligne à l'aide des liens rapides dans l'administration WooCommerce.
---

Vous pouvez rapidement séparer vos commandes de point de vente et en ligne dans l'administration WooCommerce en utilisant les liens rapides en haut à gauche.
**Veuillez noter :** ce filtre ne fonctionnera que pour les commandes de point de vente après la version 0.3.1, les commandes antérieures à 0.3.1 seront par défaut en ligne.

![Filtrer les commandes](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrer les commandes")

De plus, si vous souhaitiez filtrer toutes les commandes terminées depuis votre zone d'administration pour vous concentrer sur les commandes en attente/en cours de traitement, vous pouvez ajouter le code suivant à votre fichier [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Modifier le filtre par défaut des commandes dans l'administration WordPress
 * ajouter ce code à votre fichier functions.php
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
