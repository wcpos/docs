---
title: Filtrage des commandes par ventes POS ou en ligne
description: Séparez les commandes POS et en ligne en utilisant les liens rapides dans l'administration WooCommerce.
---

Vous pouvez rapidement séparer vos commandes POS et en ligne dans l'administration WooCommerce en utilisant les liens rapides en haut à gauche.  
**Veuillez noter :** cela ne filtrera que les commandes POS après la version 0.3.1, les commandes antérieures à 0.3.1 seront par défaut des commandes en ligne.

![Filter Orders](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrer les commandes")

De plus, si vous souhaitez filtrer toutes les commandes complètes depuis votre espace d'administration afin de vous concentrer sur les commandes en attente/traitement, vous pouvez ajouter le code suivant à votre fichier [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
