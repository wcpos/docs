WCPOS utilise plusieurs composants de pavés numériques pour permettre une saisie rapide à la souris ou sur écran tactile.
Vous trouverez ci-dessous des exemples des différents types de pavés numériques :

![Pavé numérique de quantité avec saisie rapide pour +1 et -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Pavé numérique de quantité avec saisie rapide pour +1 et -1")

![Pavé numérique de prix d'article avec saisie rapide pour le pourcentage de remise.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Pavé numérique de prix d'article avec saisie rapide pour le pourcentage de remise")

![Pavé numérique de montant payé avec saisie rapide pour les montants de paiement en espèces probables.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Pavé numérique de montant payé avec saisie rapide pour les montants de paiement en espèces probables")

![Pavé numérique de rendu de monnaie avec saisie rapide pour les montants de rendu courants.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Pavé numérique de rendu de monnaie avec saisie rapide pour les montants de rendu courants")
