WCPOS est le plugin Point de Vente pour [WooCommerce](https://www.woothemes.com/woocommerce/).
Cette documentation couvre le plugin open source, ainsi que le plugin Pro qui ajoute des fonctionnalités au plugin gratuit.

- Téléchargez [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) depuis WordPress.org
- Passez à [WCPOS Pro](http://wcpos.com/pro)

:::danger

Assurez-vous que le numéro de version de la documentation correspond à la version de vos plugins installés.

:::

### Autres documentations

Une liste de **Questions Fréquemment Posées** est disponible sur [faq.wcpos.com](http://faq.wcpos.com).
La **documentation pour développeurs** est disponible sur [api.wcpos.com](http://api.wcpos.com).
