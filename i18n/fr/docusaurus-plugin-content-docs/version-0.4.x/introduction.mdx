WCPOS est le plugin point de vente pour [WooCommerce](https://www.woothemes.com/woocommerce/).
Cette documentation couvre le plugin open source, ainsi que le plugin Pro qui ajoute des fonctionnalités au plugin gratuit.

- Télécharger [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) depuis WordPress.org
- Passer à [WCPOS Pro](http://wcpos.com/pro)

:::danger

Veuillez vous assurer que le numéro de version de la documentation correspond à la version de vos plugins installés.

:::

### Autres ressources {#other-documentation}

Une liste de **questions fréquemment posées** est disponible sur [faq.wcpos.com](http://faq.wcpos.com).
La **documentation pour les développeurs** est disponible sur [api.wcpos.com](http://api.wcpos.com).

### Aide et assistance {#help-and-support}

Si vous avez une question qui n'est pas couverte par cette documentation ou la FAQ, vous pouvez nous contacter à [support@wcpos.com](mailto:support@wcpos.com).

### Contribuer à cette documentation {#contribute-to-this-documentation}

Vous pouvez améliorer cette documentation sur [GitHub](https://github.com/kilbot/WooCommerce-POS-Manual) en soumettant des tickets ou en proposant des modifications.
