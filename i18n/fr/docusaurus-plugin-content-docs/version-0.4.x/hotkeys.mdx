Les raccourcis clavier sont des combinaisons de touches qui peuvent être utilisées pour déclencher rapidement des actions dans WCPOS.

| Touche                      | Description      |
| --------------------------- | ---------------- |
| <div align="center">?</div> | Écran d'aide     |
| <div align="center">B</div> | Mode code-barres |
| <div align="center">V</div> | Mode recherche    |
| <div align="center">S</div> | Synchroniser avec le serveur |
