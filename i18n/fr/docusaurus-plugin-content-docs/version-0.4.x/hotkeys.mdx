Les raccourcis clavier permettent de déclencher rapidement des actions dans WCPOS.

| Touche                      | Description              |
| --------------------------- | ------------------------ |
| <div align="center">?</div> | Écran d'aide             |
| <div align="center">B</div> | Mode code-barres         |
| <div align="center">V</div> | Mode recherche           |
| <div align="center">S</div> | Synchronisation serveur  |
