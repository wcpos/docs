---
title: "Produits uniquement POS"
description: "Les produits peuvent être cachés dans votre boutique en ligne ou au point de vente à l'aide du paramètre de visibilité POS."
---

Il peut y avoir des situations où vous devez cacher un produit de votre boutique en ligne ou de votre POS. Avec WCPOS, cela peut être réalisé en utilisant le paramètre de visibilité POS sur la page d'édition du produit.

### Activation des produits uniquement POS {#enabling-pos-only-products}

Les produits uniquement POS ne sont pas activés par défaut. Pour activer les produits uniquement POS, allez dans <code>WP Admin > POS > Paramètres > Général</code>

![Activer les produits uniquement POS](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "Activer les produits uniquement POS")

Activer les produits uniquement POS nécessitera une recherche supplémentaire dans la base de données et entraînera donc une légère diminution de la performance. Si vous n'utilisez pas les produits uniquement POS, il est recommandé de laisser cette option désactivée.

### Visibilité POS et en ligne {#pos-online-visibility}

![Visibilité POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "Paramètres de visibilité POS sur la page d'édition du produit")

| Paramètre                         | Description                                                                       |
| --------------------------------- | --------------------------------------------------------------------------------- |
| **POS et en ligne**<br />(par défaut) | Les produits seront visibles à la fois en ligne dans votre boutique web et dans votre magasin physique |
| **Uniquement POS**                | La page produit ne sera pas visible en ligne                                     |
| **Uniquement en ligne**           | Le produit ne sera pas visible dans le POS                                        |

### Visibilité du catalogue {#catalog-visibility}

Si vous accédez directement à une page de produit uniquement POS, vous obtiendrez <code>404 Page non trouvée</code>. Cependant, les produits uniquement POS peuvent encore apparaître dans la recherche en ligne et les pages de catégorie. Pour cacher complètement les produits de votre site web, vous devez définir la visibilité du catalogue sur Caché.

![Visibilité du catalogue](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "Paramètres de visibilité du catalogue sur la page d'édition du produit")

### Filtrer les produits uniquement POS {#filter-pos-only-products}

Vous pouvez rapidement voir quels produits sont visibles/cachés en utilisant les liens rapides sur la page d'administration des produits :

![Filtre de visibilité POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "Filtre de visibilité POS")

### Édition en masse {#bulk-edit}

Pour changer la visibilité POS de plusieurs produits, allez dans votre liste de produits et sélectionnez <code>Modifier</code> dans les <code>Actions en masse</code>.

![Édition en masse de la visibilité POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "Édition en masse de la visibilité POS")
