---
title: Champ de Code-Barres Personnalisé
---

Par défaut, WCPOS utilisera le champ SKU comme code-barres.

![Par défaut WCPOS utilisera le champ SKU comme code-barres.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Par défaut WCPOS utilisera le champ SKU comme code-barres")

Les utilisateurs de [WCPOS Pro](http://wcpos.com/pro) peuvent utiliser n'importe quel champ de méta produit en sélectionnant un Champ de Code-Barres dans `WP Admin > POS > Paramètres > Général`

![Sélectionner un champ de code-barres personnalisé dans WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Sélectionner un champ de code-barres personnalisé dans WCPOS Pro")

Les champs de méta de code-barres personnalisés peuvent également être définis en utilisant le filtre `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Champ de méta produit personnalisé à utiliser comme code-barres
 *
 * Par défaut, WCPOS utilisera le champ SKU - '_sku'
 * Vous pouvez modifier la clé de méta en utilisant le filtre WordPress suivant
 *
 * Voici des champs de méta de plugins de code-barres populaires
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // modifier selon votre clé de méta personnalisée
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un code-barres doit être unique pour chaque produit.
:::
