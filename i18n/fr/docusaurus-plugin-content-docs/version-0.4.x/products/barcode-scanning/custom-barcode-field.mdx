---
title: Champ de code-barres personnalisé
---

Par défaut, WCPOS utilisera le champ SKU comme code-barres.

![Par défaut, WCPOS utilisera le champ SKU comme code-barres.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Par défaut, WCPOS utilisera le champ SKU comme code-barres.")

Les utilisateurs de [WCPOS Pro](http://wcpos.com/pro) peuvent utiliser n'importe quel champ méta produit en sélectionnant un Champ de Code-barres dans `WP Admin > POS > Paramètres > Général`.

![Sélectionnez le champ de code-barres personnalisé dans WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Sélectionnez le champ de code-barres personnalisé dans WCPOS Pro.")

Les champs méta de code-barres personnalisés peuvent également être définis en utilisant le filtre `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un code-barres doit être unique pour chaque produit.
:::
