---
title: Produits Divers
---

Il peut y avoir des cas où un magasin doit vendre des articles qui ne font pas partie de l'inventaire WooCommerce.  
Vous trouverez ci-dessous deux options pour vendre des 'produits divers' dans WCPOS.

### Produit Divers uniquement POS (recommandé) {#pos-only-miscellaneous-product-recommended}

![Produit Divers uniquement POS](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Créer un Produit Divers uniquement POS")

En utilisant la fonction [Produit uniquement POS](./pos-only-products), vous pouvez créer un 'produit divers' à utiliser dans le POS.  
Le titre, le prix de vente, le prix régulier, le taux de taxe et les informations méta peuvent tous être modifiés à la volée lors du paiement.

![Modifier le Produit Divers](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Modifier le Produit Divers dans le panier")

### Ligne d'élément de frais {#fee-line-item}

![Ligne d'élément de frais](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Modifier la Ligne d'élément de frais")

Utiliser une ligne d'élément de frais est similaire à la création d'un Produit Divers uniquement POS.  
Les caissiers peuvent rapidement mettre à jour le titre, le prix et le taux de taxe de la ligne d'élément de frais.  
Le principal inconvénient est que les frais peuvent apparaître séparément des produits dans vos rapports.
