---
title: Serveurs Obsolètes
description: Si votre hébergeur bloque les méthodes HTTP REST modernes, vous devrez activer le support des serveurs obsolètes.
---

HTTP (et HTTPS) sont les protocoles utilisés par chaque navigateur pour communiquer avec les applications web.
Une partie clé du protocole est la [méthode HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Traditionnellement, les applications web utilisaient un ensemble très limité de méthodes HTTP telles que GET (récupérer une page web) et POST (soumettre un formulaire web).

Les applications web modernes, comme WCPOS, utilisent une approche REST/HTTP avec les méthodes HTTP suivantes :

- GET
- POST
- PUT
- PATCH
- DELETE

Dans certains cas rares, un hébergeur web peut choisir de bloquer certaines de ces méthodes HTTP.
Si votre hébergeur bloque les méthodes HTTP REST modernes, vous devrez activer le support des serveurs obsolètes.

![Support des serveurs obsolètes](https://wcpos.com/wp-content/uploads/2016/06/enable-legacy-server-support.png "WP Admin > POS > Settings > Tools > Activer le support des serveurs obsolètes")
