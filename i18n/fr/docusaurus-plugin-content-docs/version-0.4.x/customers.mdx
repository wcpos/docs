### Capacités

Les caissiers auront besoin des capacités _list_users_, _create_users_ et _edit_users_ pour utiliser toutes les fonctionnalités Clients.

### WordPress Multisite

Lors de l'utilisation de [WordPress multisite](https://codex.wordpress.org/Create_A_Network), il existe des [vérifications supplémentaires](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) pour empêcher les non [Super Administrateurs](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) de créer et modifier des clients.

Tout d'abord, pour permettre aux non Super Administrateurs de **créer** de nouveaux clients, veuillez activer l'option suivante dans `Administration du Réseau > Paramètres`.

![Option multisite pour créer de nouveaux utilisateurs](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Option multisite pour créer de nouveaux utilisateurs")

De plus, pour permettre aux non Super Administrateurs de **modifier** les détails du client, vous devrez ajouter la capacité `manage_network_users` au caissier.

```php title="functions.php"
// changez CASHIER_ID, et
// ajoutez à votre fichier functions.php

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
