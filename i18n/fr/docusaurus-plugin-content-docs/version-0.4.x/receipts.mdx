Les modèles d'impression de reçus POS se trouvent dans le fichier `includes/views/print/tmpl-receipt.php` des plugins gratuit et Pro. Les modèles de reçus peuvent être personnalisés en créant un fichier `woocommerce-pos/print/tmpl-receipt.php` dans votre répertoire de thème. Le code pour les deux modèles est inclus ci-dessous.

### Modèle de reçu Pro

Le modèle de reçu Pro permet des informations supplémentaires à partir de l'administration des [Boutiques](./stores) telles que le logo, l'adresse de la boutique, les heures d'ouverture et les messages spéciaux.

### Personnaliser la date du reçu

WCPOS utilise [moment.js](http://momentjs.com/) pour localiser les chaînes de date. Le format de date par défaut est `"MMMM Do YYYY, h:mm:ss a"`, par exemple : 31 mai 2015, 19:20:44. Veuillez consulter la [documentation de moment.js](http://momentjs.com/docs/#/parsing/string-format/) pour plus d'informations sur la personnalisation du format de date.

### Propriétés de la commande

WCPOS utilise la sortie json de l'API REST WC pour remplir le modèle de reçu de commande. [La documentation de l'API REST WC montre un exemple](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) de la sortie json standard. Des propriétés supplémentaires ont été ajoutées par WCPOS en utilisant le filtre `woocommerce_api_order_response`.

| Propriété                          | Description                                                   |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | La partie fiscale de la remise du panier                      |
| cashier.id                        | ID utilisateur du caissier                                    |
| cashier.first_name                | Prénom du caissier                                            |
| cashier.last_name                 | Nom de famille du caissier                                    |
| payment_details.result            | Succès ou échec de la passerelle de paiement                  |
| payment_details.message           | Tous messages de la passerelle de paiement                    |
| payment_details.redirect          | URL de redirection pour les paiements hors site, ex : PayPal Standard |
| payment_details.method\_`gateway` | Messages spécifiques à la passerelle, ex : Montant payé et rendu |
| shipping_lines[i].total_tax       | Ajouter le montant de la taxe pour chaque ligne de livraison  |
| subtotal_tax                      | La partie fiscale du sous-total                               |
