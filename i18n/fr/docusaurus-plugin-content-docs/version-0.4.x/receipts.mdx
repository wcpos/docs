Les modèles d'impression des reçus POS se trouvent dans le fichier `includes/views/print/tmpl-receipt.php` des plugins gratuits et Pro. Les modèles de reçus peuvent être personnalisés en créant un fichier `woocommerce-pos/print/tmpl-receipt.php` dans le répertoire de votre thème. Le code pour les deux modèles est inclus ci-dessous.

### Modèle de Reçu Pro

Le Modèle de Reçu Pro permet d'ajouter des informations supplémentaires de l'administrateur des [Magasins](./stores) telles que le logo, l'adresse du magasin, les heures d'ouverture et des messages spéciaux.

### Personnaliser la date du reçu

WCPOS utilise [moment.js](http://momentjs.com/) pour localiser les chaînes de date. Le format de date par défaut est `"MMMM Do YYYY, h:mm:ss a"`, par exemple : 31 Mai 2015, 19:20:44. Veuillez consulter la [documentation de moment.js](http://momentjs.com/docs/#/parsing/string-format/) pour plus d'informations sur la personnalisation du format de date.

### Propriétés de Commande

WCPOS utilise la sortie json de l'API REST de WC pour peupler le modèle de reçu de commande. [La documentation de l'API REST de WC montre un exemple](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) de la sortie json standard. Certaines propriétés supplémentaires ont été ajoutées par WCPOS en utilisant le filtre `woocommerce_api_order_response`.

| Propriété                         | Description                                                  |
| --------------------------------- | ------------------------------------------------------------ |
| cart_discount_tax                 | La portion de taxe de la réduction du panier                |
| cashier.id                        | ID de l'utilisateur du Caissier                              |
| cashier.first_name                | Prénom du Caissier                                          |
| cashier.last_name                 | Nom de famille du Caissier                                   |
| payment_details.result            | Succès ou échec de la passerelle de paiement                |
| payment_details.message           | Tous les messages de la passerelle de paiement              |
| payment_details.redirect          | URL de redirection pour les paiements hors site, par ex : PayPal Standard |
| payment_details.method\_`gateway` | Tous les messages spécifiques à la passerelle, par ex : Montant remis et Changement |
| shipping_lines[i].total_tax       | Montant de la taxe à ajouter pour chaque ligne de livraison  |
| subtotal_tax                      | La portion de taxe du sous-total                             |
