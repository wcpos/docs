En utilisant les paramètres d'accès POS, les propriétaires de magasin peuvent accorder ou supprimer les privilèges des utilisateurs pour le système de Point de Vente.
Par défaut, les rôles **Administrateur** et **Gestionnaire de boutique** sont activés avec toutes les capacités pour utiliser le POS.
Il est recommandé de conserver les paramètres par défaut sauf si vous êtes conscient des conséquences.
Par exemple, accorder la capacité `publish_shop_orders` au rôle `Abonnés` serait une _très_ mauvaise idée.

![Exemple de paramètres d'accès POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Exemple de paramètres d'accès POS")

| Capacité                                                                | Accès                                                   |
| ----------------------------------------------------------------------- | ------------------------------------------------------- |
| access_woocommerce_pos                                                  | Requis pour accéder à l'interface POS                   |
| manage_woocommerce_pos                                                  | Requis pour accéder aux paramètres d'administration POS |
| read_private_products                                                   | Requis pour visualiser les produits                     |
| edit_product <br /> edit_published_products <br /> edit_others_products | Requis pour éditer les produits et variations           |
| read_private_shop_orders                                                | Requis pour visualiser les commandes et reçus           |
| publish_shop_orders                                                     | Requis pour créer de nouvelles commandes                |
| list_users                                                              | Requis pour voir les clients                            |
| create_users                                                            | Requis pour créer de nouveaux clients                   |
| edit_users                                                              | Requis pour mettre à jour les clients existants         |
| read_private_shop_coupons                                               | Requis pour visualiser les coupons                      |
