En utilisant les paramètres d'accès POS, les propriétaires de magasin peuvent accorder ou retirer des privilèges utilisateur au système de Point de Vente. Par défaut, les rôles **Administrateur** et **Shop Manager** sont activés avec toutes les capacités pour utiliser le POS. Il est recommandé de conserver les paramètres par défaut à moins d'être conscient des conséquences. Par exemple, accorder le `publish_shop_orders` au rôle de `Abonnés` serait une _très_ mauvaise idée.

![Exemple de paramètres d'accès POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Exemple de paramètres d'accès POS")

| Capacité                                                                | Accès                                    |
| ----------------------------------------------------------------------- | ---------------------------------------- |
| access_woocommerce_pos                                                  | Requis pour accéder au POS frontend      |
| manage_woocommerce_pos                                                  | Requis pour accéder aux paramètres admin du POS |
| read_private_products                                                   | Requis pour voir les produits            |
| edit_product <br /> edit_published_products <br /> edit_others_products | Requis pour modifier les produits et les variations |
| read_private_shop_orders                                                | Requis pour voir les commandes et les reçus |
| publish_shop_orders                                                     | Requis pour créer de nouvelles commandes  |
| list_users                                                              | Requis pour voir les clients             |
| create_users                                                            | Requis pour créer de nouveaux clients    |
| edit_users                                                              | Requis pour mettre à jour les clients existants |
| read_private_shop_coupons                                               | Requis pour voir les coupons             |
