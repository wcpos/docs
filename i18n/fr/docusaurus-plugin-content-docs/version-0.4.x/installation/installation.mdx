### Installation automatique (recommandée)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installer et activer WCPOS en moins d'une minute")

1. Allez à `WP Admin > Plugins > Ajouter Nouveau`
2. Recherchez `WCPOS` dans le WordPress Plugin Directory
3. Installez le plugin
4. Activez le plugin

### Installation manuelle

1. [Téléchargez WCPOS](https://wordpress.org/plugins/woocommerce-pos/) depuis WordPress.org
2. Allez à `WP Admin > Plugins > Ajouter Nouveau > Télécharger le plugin`
3. Activez le plugin

:::info
Les versions précédentes de WCPOS peuvent être trouvées sous l'onglet [Développeurs](https://wordpress.org/plugins/woocommerce-pos/developers/) sur WordPress.org.
:::
