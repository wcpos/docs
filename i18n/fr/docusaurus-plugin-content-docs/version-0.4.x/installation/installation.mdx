### Installation automatique (recommandée) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installer et activer WCPOS en moins d'une minute")

1. Accédez à `WP Admin > Extensions > Ajouter`
2. Recherchez `WCPOS` dans le répertoire des extensions WordPress
3. Installez l'extension
4. Activez l'extension

### Installation manuelle {#manual-installation}

1. [Téléchargez WCPOS](https://wordpress.org/plugins/woocommerce-pos/) depuis WordPress.org
2. Accédez à `WP Admin > Extensions > Ajouter > Téléverser une extension`
3. Activez l'extension

:::info
Les versions précédentes de WCPOS sont disponibles sous l'onglet [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) sur WordPress.org.
:::
