:::important

[WCPOS Pro](http://wcpos.com/pro) est un plugin payant qui ajoute des fonctionnalités au plugin gratuit [WCPOS](http://wordpress.org/plugins/woocommerce-pos/). 

:::

Si vous avez acheté une licence pour WCPOS Pro, veuillez suivre les étapes ci-dessous pour installer et activer le plugin :

1. Allez sur : [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Sous **Téléchargements**, cliquez sur le lien de téléchargement et enregistrez le plugin sur votre bureau.
3. Ensuite, allez sur votre site, connectez-vous et allez à `WP Admin > Plugins > Ajouter Nouveau > Télécharger le Plugin`
4. Téléchargez le fichier zip du plugin depuis votre bureau et activez-le.
5. Ensuite, allez à `WP Admin > POS > Paramètres > Licence` et entrez votre Clé de Licence et votre Email de Licence pour compléter l'activation.

![Mes Téléchargements](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Vous pouvez télécharger WCPOS Pro sur votre page de compte")

Lorsque de nouvelles versions sont prêtes, elles apparaîtront dans votre tableau de bord des Mises à jour. 
Pour des informations sur ce sur quoi je travaille actuellement, veuillez garder un œil sur la [Feuille de route](http://wcpos.com/roadmap/). 
Gardez également un œil sur le journal des modifications qui listera les nouvelles fonctionnalités de chaque version et la [page de documentation](http://docs.wcpos.com) sera mise à jour avec des informations sur l'utilisation des nouvelles fonctionnalités.
