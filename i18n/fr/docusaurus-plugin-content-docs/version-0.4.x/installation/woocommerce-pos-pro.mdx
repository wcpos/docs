:::important

[WCPOS Pro](http://wcpos.com/pro) est un plugin payant qui ajoute des fonctionnalités au [plugin WCPOS gratuit](http://wordpress.org/plugins/woocommerce-pos/). 

:::

Si vous avez acheté une licence pour WCPOS Pro, veuillez suivre les étapes ci-dessous pour installer et activer le plugin :

1.  Accédez à : [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Dans **Téléchargements**, cliquez sur le lien de téléchargement et enregistrez le plugin sur votre bureau.
3.  Ensuite, allez sur votre site, connectez-vous et rendez-vous sur `WP Admin > Plugins > Ajouter > Téléverser un plugin`
4.  Téléversez le fichier zip du plugin depuis votre bureau et activez-le.
5.  Ensuite, allez sur `WP Admin > POS > Paramètres > Licence` et entrez votre Clé de Licence et l'E-mail de Licence pour terminer l'activation.

![Mes Téléchargements](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Vous pouvez télécharger WCPOS Pro sur votre page de compte")

Lorsque de nouvelles versions sont prêtes, elles apparaîtront dans votre tableau de bord des Mises à jour. 
Pour savoir sur quoi je travaille actuellement, gardez un œil sur la [Feuille de route](http://wcpos.com/roadmap/). 
Surveillez également le journal des modifications qui répertoriera les nouvelles fonctionnalités de chaque version et la [page de documentation](http://docs.wcpos.com) sera mise à jour avec des informations sur l'utilisation des nouvelles fonctionnalités.
