Vous avez besoin d'aide avec WCPOS ? Voici comment obtenir du support et trouver des réponses à vos questions.

## Support dans l'application

Le POS comprend un écran de support intégré accessible depuis le tiroir de navigation à gauche. Cet écran intègre directement la communauté Discord de WCPOS dans l'application, vous permettant de :

- Parcourir les discussions existantes
- Chercher des réponses à des questions fréquentes
- Poser des questions dans le canal #general
- Rester informé des annonces

## Communauté Discord

Le principal canal de support pour WCPOS est notre [serveur Discord](https://wcpos.com/discord).

### Canaux

| Canal | Objectif |
|-------|----------|
| **#welcome-and-rules** | Directives pour la communauté |
| **#announcements** | Mises à jour et nouvelles |
| **#general** | Canal de support principal - posez vos questions ici |
| **#forum** | Discussions plus longues et demandes de fonctionnalités |
| **#github-activity** | Mises à jour sur le développement |

### Avant de demander

Pour obtenir le meilleur aide :

1. **Testez d'abord** - Téléchargez et testez le plugin pendant quelques jours pour comprendre s'il répond à vos besoins
2. **Vérifiez la documentation** - De nombreuses questions trouvent réponse dans cette documentation
3. **Cherchez sur Discord** - Votre question a peut-être déjà été posée
4. **Fournissez des détails** - Incluez votre version WooCommerce, votre version WCPOS, et les étapes pour reproduire les problèmes

## Support par e-mail

Pour des problèmes liés à la licence ou des demandes privées, vous pouvez envoyer un e-mail à : **support@wcpos.com**

:::note
Les questions générales de support doivent être posées sur Discord où la communauté peut aider et d'autres peuvent bénéficier des réponses.
:::

## Écran des journaux

Le POS comprend un [écran des journaux](/support/logs) qui enregistre l'activité système. C'est précieux pour :

- Déboguer les problèmes de synchronisation
- Comprendre les messages d'erreur
- Fournir des informations détaillées lors de la demande de support

Lorsque vous demandez de l'aide, vérifiez l'écran des journaux pour tout message d'erreur et incluez-le dans votre demande de support.

## Support Pro

Les clients de [WCPOS Pro](https://wcpos.com/pro) reçoivent le même support communautaire via Discord. La licence Pro finance le développement continu des fonctionnalités gratuites et Pro.
