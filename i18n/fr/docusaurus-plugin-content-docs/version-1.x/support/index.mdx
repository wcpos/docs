Besoin d'aide avec WCPOS ? Voici comment obtenir du support et trouver des réponses à vos questions.

## Support dans l'application

Le POS inclut un écran de Support intégré accessible depuis le tiroir de navigation gauche. Cet écran intègre la communauté Discord de WCPOS directement dans l'application, vous permettant de :

- Parcourir les discussions existantes
- Rechercher des réponses à des questions courantes
- Poser des questions dans le canal #général
- Rester informé des annonces

## Communauté Discord

Le principal canal de support pour WCPOS est notre [serveur Discord](https://wcpos.com/discord).

### Canaux

| Canal | Objectif |
|-------|----------|
| **#bienvenue-et-règles** | Directives de la communauté |
| **#annonces** | Mises à jour et nouvelles |
| **#général** | Canal de support principal - posez vos questions ici |
| **#forum** | Discussions plus longues et demandes de fonctionnalités |
| **#activité-github** | Mises à jour de développement |

### Avant de demander

Pour obtenir le meilleur soutien :

1. **Testez d'abord** - Téléchargez et testez le plugin pendant quelques jours pour comprendre s'il répond à vos besoins
2. **Consultez la documentation** - De nombreuses questions sont répondues dans cette documentation
3. **Recherchez sur Discord** - Votre question a peut-être déjà reçu une réponse
4. **Fournissez des détails** - Incluez votre version de WooCommerce, la version de WCPOS et les étapes pour reproduire les problèmes

## Support par Email

Pour des problèmes liés à la licence ou des demandes privées, vous pouvez envoyer un email à : **support@wcpos.com**

:::note
Les questions générales de support doivent aller sur Discord où la communauté peut aider et d'autres peuvent bénéficier des réponses.
:::

## Écran des Logs

Le POS inclut un [écran des Logs](/support/logs) qui enregistre l'activité du système. Cela est inestimable pour :

- Déboguer les problèmes de synchronisation
- Comprendre les messages d'erreur
- Fournir des informations détaillées lors de la demande de support

Lorsque vous demandez de l'aide, vérifiez l'écran des Logs pour tout message d'erreur et incluez-le dans votre demande de support.

## Support Pro

Les clients de [WCPOS Pro](https://wcpos.com/pro) reçoivent le même soutien communautaire via Discord. La licence Pro finance le développement continu des fonctionnalités gratuites et Pro.

## Contribution

WCPOS est open source ! Vous pouvez contribuer en :

- Signalant des bugs sur [GitHub](https://github.com/wcpos)
- Suggérant des fonctionnalités sur Discord
- Traduisant la documentation
- Partageant votre expérience avec d'autres
