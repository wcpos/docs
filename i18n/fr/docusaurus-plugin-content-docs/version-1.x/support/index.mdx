Besoin d'aide avec WCPOS ? Voici comment obtenir du support et trouver des réponses à vos questions.

## Support intégré à l'application

Le POS comprend un écran de Support intégré accessible depuis le tiroir de navigation de gauche. Cet écran intègre directement la communauté Discord de WCPOS, vous permettant de :

- Parcourir les discussions existantes
- Rechercher des réponses aux questions courantes
- Poser des questions dans le canal #general
- Rester informé des annonces

## Communauté Discord

Le canal de support principal pour WCPOS est notre [serveur Discord](https://wcpos.com/discord).

### Canaux

| Canal | Objectif |
|---------|---------|
| **#welcome-and-rules** | Lignes directrices de la communauté |
| **#announcements** | Mises à jour et actualités |
| **#general** | Canal de support principal - posez vos questions ici |
| **#forum** | Discussions plus approfondies et demandes de fonctionnalités |
| **#github-activity** | Mises à jour de développement |

### Avant de poser une question

Pour obtenir la meilleure aide :

1. **Testez d'abord** - Téléchargez et testez le plugin pendant quelques jours pour comprendre s'il répond à vos besoins
2. **Consultez la documentation** - De nombreuses questions trouvent leur réponse dans cette documentation
3. **Recherchez sur Discord** - Votre question a peut-être déjà été résolue
4. **Fournissez des détails** - Incluez votre version de WooCommerce, la version de WCPOS et les étapes pour reproduire les problèmes

## Support par e-mail

Pour les problèmes liés aux licences ou les demandes privées, vous pouvez envoyer un e-mail à : **support@wcpos.com**

:::note
Les questions de support général doivent être posées sur Discord où la communauté peut aider et où les autres peuvent bénéficier des réponses.
:::

## Écran des journaux

Le POS inclut un [écran des journaux](/support/logs) qui enregistre l'activité système. Cela est inestimable pour :

- Déboguer les problèmes de synchronisation
- Comprendre les messages d'erreur
- Fournir des informations détaillées lors d'une demande de support

Lorsque vous demandez de l'aide, consultez l'écran des journaux pour tout message d'erreur et incluez-les dans votre demande de support.

## Support Pro

Les clients [WCPOS Pro](https://wcpos.com/pro) reçoivent le même support communautaire via Discord. La licence Pro finance le développement continu des fonctionnalités gratuites et Pro.

- Signalant des bugs sur [GitHub](https://github.com/wcpos)
- Suggérant des fonctionnalités sur Discord
- Traduisant la documentation
- Partageant votre expérience avec les autres
