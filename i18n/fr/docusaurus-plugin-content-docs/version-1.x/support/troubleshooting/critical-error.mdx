Si vous voyez ce message d'erreur, cela signifie que votre POS a rencontré une erreur fatale du serveur et ne peut pas continuer à fonctionner. La bonne nouvelle est que ce message d'erreur est enregistré dans vos journaux WooCommerce, et notre équipe de support peut vous aider à diagnostiquer le problème.

## Étapes pour Trouver l'Erreur

1. Connectez-vous à votre zone d'administration WordPress.
2. Naviguez vers "WooCommerce" > "État" > "Journaux" dans le menu latéral de WordPress.
3. Localisez la section des journaux "fatal_error".
4. Cliquez sur l'entrée de journal correspondant au jour où l'erreur s'est produite (généralement la date d'aujourd'hui) pour voir les détails de l'erreur.
5. Identifiez la cause de l'erreur comme enregistrée dans cette section.
6. Copiez les messages d'erreur pertinents ou prenez une capture d'écran et partagez-les avec notre équipe de support en les publiant dans le [chat Discord](https://wcpos.com/discord) ou en les envoyant par e-mail à support@wcpos.com.

## Obtenir de l'Aide

Partagez le message d'erreur avec notre équipe de support :

- **Discord** : [wcpos.com/discord](https://wcpos.com/discord)
- **Email** : support@wcpos.com
