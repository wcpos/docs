WCPOS stocke une copie locale de vos produits, commandes et clients afin que le POS puisse fonctionner rapidement, même hors ligne. Parfois, ces données locales peuvent devenir désynchronisées avec votre boutique WooCommerce. Effacer toutes les données locales supprime la base de données locale et déclenche une nouvelle synchronisation depuis votre serveur.

:::warning
C'est l'option nucléaire. Effacer les données locales supprime tout ce qui est stocké sur l'appareil et force une nouvelle synchronisation complète. En fonction de la taille de votre boutique, cela peut prendre du temps. Utilisez-la uniquement lorsque d'autres options — comme une [synchronisation](/products/sync) normale — n'ont pas résolu le problème.
:::

## Quand l'utiliser {#when-to-use-it}

Vous pourriez avoir besoin d'effacer les données locales lorsque :

- **Problèmes de synchronisation** — les produits, commandes ou clients ne se mettent pas à jour correctement
- **Après une mise à jour de plugin** — pour garantir la compatibilité avec les nouvelles structures de données
- **Données obsolètes** — les prix, niveaux de stock ou autres informations semblent périmés

## Ce qui est effacé {#what-gets-cleared}

| Effacé | Non effacé |
|---------|-------------|
| Base de données locale des produits | Votre session de connexion |
| Base de données locale des commandes | Données côté serveur |
| Base de données locale des clients | Paramètres WooCommerce |

Vous **ne** serez pas déconnecté. Les données de votre boutique WooCommerce ne sont pas affectées — seule la copie locale sur votre appareil est supprimée. Une nouvelle synchronisation commence automatiquement après l'effacement.

## Comment effacer les données locales {#how-to-clear-local-data}

### Web

1. Cliquez sur votre **menu utilisateur** dans le coin supérieur droit pour ouvrir le menu
2. Cliquez sur **Effacer toutes les données locales**
3. La base de données locale sera supprimée et une nouvelle synchronisation commencera automatiquement

### Bureau {#desktop}

1. Ouvrez le menu de l'application **WCPOS** dans la barre de menu supérieure
2. Survolez **Avancé**
3. Cliquez sur **Effacer les données de l'application**
4. La base de données locale sera supprimée et une nouvelle synchronisation commencera automatiquement

:::note
L'application de bureau utilise une base de données SQLite tandis que l'application web utilise IndexedDB, mais le comportement est le même sur les deux plateformes.
:::
