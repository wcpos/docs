# API03007 : File d'attente de requêtes pleine

## Ce que cela signifie

Le POS a trop de requêtes en attente à envoyer. La file d'attente interne est pleine et ne peut pas accepter davantage de requêtes avant que certaines ne soient terminées.

## Causes courantes

- **Réseau lent** — Requêtes s'accumulant en raison d'une connexion lente
- **Serveur non réactif** — Le serveur met trop de temps à répondre
- **Actions rapides** — Effectuer de nombreuses actions plus vite qu'elles ne peuvent être traitées
- **Surcharge de synchronisation** — Une opération de synchronisation importante bloque la file d'attente

## Comment résoudre

### 1. Attendre que la file d'attente se vide

Donner du temps aux requêtes en attente de se terminer :
- Attendre 30-60 secondes
- Surveiller les indicateurs de synchronisation jusqu'à leur achèvement
- Éviter des actions supplémentaires jusqu'à ce que ce soit terminé

### 2. Vérifier la connexion réseau

Une connexion lente provoque un engorgement :
- Tester votre vitesse internet
- Essayer une connexion plus rapide
- Se rapprocher du routeur WiFi

### 3. Vérifier le temps de réponse du serveur

Si le serveur est lent :
- Tester l'accès à votre site dans un navigateur
- Contacter l'hébergeur si le site est lent
- Attendre que la charge du serveur diminue

### 4. Redémarrer le POS

Si la file d'attente est bloquée :
1. Fermer l'application POS
2. Attendre un moment
3. Rouvrir le POS
4. Les requêtes en attente peuvent nécessiter d'être réinitiées

### 5. Éviter les actions rapides

Pendant la synchronisation ou les périodes lentes :
- Attendre qu'une action soit terminée avant d'en commencer une autre
- Ne pas cliquer répétitivement sur les boutons
- Être patient avec les opérations réseau

## Comportement hors ligne

Lorsque la file d'attente se remplit en raison de conditions hors ligne :
- Le POS conservera les requêtes en attente jusqu'au retour en ligne
- Vous pouvez toujours parcourir les produits et clients mis en cache
- Vous pouvez commencer de nouvelles commandes et ajouter des articles au panier
- La finalisation des commandes nécessite le rétablissement de la connectivité
