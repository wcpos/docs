# API03007 : File d'attente de requêtes pleine

## Ce que cela signifie

Le POS a trop de requêtes en attente qui doivent être envoyées. La file d'attente interne est pleine et ne peut pas accepter plus de requêtes tant que certaines ne sont pas complétées.

## Causes courantes

- **Réseau lent** — Requêtes en attente en raison d'une connexion lente
- **Serveur non réactif** — Le serveur met trop de temps à répondre
- **Actions rapides** — Exécution de nombreuses actions plus rapidement qu'elles ne peuvent être traitées
- **Surcharge de synchronisation** — Une grosse opération de synchronisation bloque la file d'attente

## Comment résoudre

### 1. Attendre que la file d'attente se libère

Donnez aux requêtes en attente le temps de compléter :
- Attendez 30 à 60 secondes
- Surveillez les indicateurs de synchronisation pour voir s'ils se complètent
- Évitez les actions supplémentaires jusqu'à ce que ce soit libéré

### 2. Vérifiez la connexion réseau

Une connexion lente cause des retards :
- Testez votre vitesse Internet
- Essayez une connexion plus rapide
- Approchez-vous du routeur WiFi

### 3. Vérifiez le temps de réponse du serveur

Si le serveur est lent :
- Testez l'accès à votre site dans un navigateur
- Contactez l'hébergement si le site est lent
- Attendez que la charge du serveur diminue

### 4. Redémarrez le POS

Si la file d'attente est bloquée :
1. Fermez l'application POS
2. Attendez un moment
3. Rouvrez le POS
4. Les requêtes en attente peuvent devoir être réinitiées

### 5. Évitez les actions rapides

Pendant la synchronisation ou les périodes lentes :
- Attendez qu'une action soit terminée avant d'en commencer une autre
- Ne cliquez pas de manière répétée sur les boutons
- Soyez patient avec les opérations réseau

## Comportement hors ligne

Lorsque la file d'attente se remplit en raison de conditions hors ligne :
- Le POS conservera les requêtes en attente jusqu'à ce qu'il soit de retour en ligne
- Vous pouvez toujours parcourir les produits et clients mis en cache
- Vous pouvez commencer de nouvelles commandes et ajouter des articles au panier
- Compléter des commandes nécessite que la connectivité soit rétablie
