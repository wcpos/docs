# Erreurs Système

Les erreurs système sont liées aux ressources de l'appareil et à la configuration système. Ces erreurs sont préfixées par `SY` et indiquent des problèmes avec votre appareil ou l'application POS elle-même.

## Catégories

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| [Ressource](#erreurs-de-ressource) | SY01xxx | Problèmes de mémoire, de disque et d'autorisation |
| [Service](#erreurs-de-service) | SY02xxx | Configuration et disponibilité des services |

---

## Erreurs de Ressource

Problèmes liés aux ressources de l'appareil comme la mémoire, le stockage et les autorisations.

| Code | Nom | Description |
|------|------|-------------|
| [SY01001](./SY01001) | Mémoire Saturée | L'appareil manque de mémoire |
| [SY01002](./SY01002) | Disque Plein | Le stockage de l'appareil est complet |
| [SY01003](./SY01003) | Autorisation Refusée | L'autorisation système a été refusée |

## Erreurs de Service

Problèmes de configuration système et de disponibilité des services.

| Code | Nom | Description |
|------|------|-------------|
| [SY02001](./SY02001) | Configuration Invalide | La configuration système est invalide |
| [SY02002](./SY02002) | Service Indisponible | Un service requis n'est pas disponible |
