# Erreurs système {#system-errors}

Les erreurs système sont liées aux ressources de l'appareil et à la configuration du système. Ces erreurs sont préfixées par `SY` et indiquent des problèmes avec votre appareil ou l'application POS elle-même.

## Catégories {#categories}

| Catégorie | Plage de codes | Description |
|-----------|----------------|-------------|
| [Ressource](#resource-errors) | SY01xxx | Problèmes de mémoire, de disque et de permissions |
| [Service](#service-errors) | SY02xxx | Disponibilité et configuration du service |

---

## Erreurs de ressources {#resource-errors}

Problèmes avec les ressources de l'appareil tels que la mémoire, le stockage, et les permissions.

| Code | Nom | Description |
|------|-----|-------------|
| [SY01001](./SY01001) | Mémoire insuffisante | L'appareil est en manque de mémoire |
| [SY01002](./SY01002) | Disque plein | Le stockage de l'appareil est plein |
| [SY01003](./SY01003) | Permission refusée | La permission du système a été refusée |

## Erreurs de service {#service-errors}

Problèmes liés à la configuration du système et à la disponibilité du service.

| Code | Nom | Description |
|------|-----|-------------|
| [SY02001](./SY02001) | Configuration invalide | La configuration du système est invalide |
| [SY02002](./SY02002) | Service indisponible | Un service requis n'est pas disponible |
