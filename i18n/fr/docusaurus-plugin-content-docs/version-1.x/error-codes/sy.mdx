# Erreurs système {#system-errors}

Les erreurs système concernent les ressources de l'appareil et la configuration du système. Ces erreurs sont préfixées par `SY` et indiquent des problèmes liés à votre appareil ou à l'application POS elle-même.

## Catégories {#categories}

| Catégorie | Plage de codes | Description |
|----------|------------|-------------|
| [Ressources](#resource-errors) | SY01xxx | Problèmes de mémoire, de disque et d'autorisations |
| [Service](#service-errors) | SY02xxx | Configuration et disponibilité des services |

---

## Erreurs de ressources {#resource-errors}

Problèmes liés aux ressources de l'appareil tels que la mémoire, le stockage et les autorisations.

| Code | Nom | Description |
|------|------|-------------|
| [SY01001](./SY01001) | Mémoire insuffisante | La mémoire de l'appareil est presque épuisée |
| [SY01002](./SY01002) | Disque plein | Le stockage de l'appareil est plein |
| [SY01003](./SY01003) | Autorisation refusée | Une autorisation système a été refusée |

## Erreurs de service {#service-errors}

Problèmes liés à la configuration système et à la disponibilité des services.

| Code | Nom | Description |
|------|------|-------------|
| [SY02001](./SY02001) | Configuration invalide | La configuration système est invalide |
| [SY02002](./SY02002) | Service indisponible | Un service requis n'est pas disponible |
