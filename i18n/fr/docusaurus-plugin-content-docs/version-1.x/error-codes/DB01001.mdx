# DB01001 : Connexion échouée

## Ce que cela signifie

Le système de point de vente (POS) n'a pas pu se connecter à sa base de données locale. WCPOS utilise une base de données locale pour stocker les produits, les clients et d'autres données pour un accès rapide et une fonctionnalité hors ligne.

## Causes courantes

- **Stockage plein** — L'appareil n'a plus d'espace de stockage disponible
- **Base de données corrompue** — Les fichiers de la base de données locale sont endommagés
- **Problème de permission** — L'application n'a pas les autorisations de stockage
- **Stockage du navigateur désactivé** — IndexedDB est désactivé (version web)

## Comment réparer

### 1. Vérifiez l'espace de stockage

Assurez-vous que votre appareil dispose d'un espace de stockage disponible :
- Libérez de l'espace si le stockage est plein
- Supprimez les applications ou fichiers inutilisés
- Vérifiez l'espace disque disponible

### 2. Effacez les données de l'application

Réinitialisez la base de données locale :
- **Application de bureau :** Cherchez "Effacer les données" dans les paramètres
- **Navigateur web :** Effacez les données du site pour le domaine POS
- **Remarque :** Cela nécessitera une resynchronisation de toutes les données

### 3. Vérifiez les paramètres du navigateur (Web)

Si vous utilisez la version web :
1. Assurez-vous qu'IndexedDB est activé
2. N'utilisez pas le mode privé/incognito (stockage limité)
3. Autorisez le site à stocker des données

### 4. Vérifiez les autorisations

Sur ordinateur de bureau :
- Assurez-vous que l'application a la permission d'écrire dans son répertoire de données
- Vérifiez si l'antivirus bloque l'accès à la base de données

### 5. Réinstallez l'application

Si d'autres solutions échouent :
1. Désinstallez l'application POS
2. Redémarrez votre appareil
3. Réinstallez l'application
4. Connectez-vous et synchronisez les données

## Récupération des données

Si vous avez des données non sauvegardées :
- L'application peut avoir des modifications en attente qui n'ont pas pu être synchronisées
- Contactez le support si vous avez besoin d'aide pour récupérer les données
- Dans la plupart des cas, les données peuvent être resynchronisées à partir du serveur
