# DB01001 : Échec de Connexion

## Ce Que Cela Signifie

Le système POS n'a pas pu se connecter à sa base de données locale. WCPOS utilise une base de données locale pour stocker les produits, les clients et autres données pour un accès rapide et un fonctionnement hors ligne.

## Causes Courantes

- **Stockage plein** — L'appareil n'a plus d'espace de stockage disponible
- **Base de données corrompue** — Les fichiers de base de données locale sont endommagés
- **Problème de permission** — L'application n'a pas les autorisations de stockage
- **Stockage du navigateur désactivé** — IndexedDB est désactivé (version web)

## Comment Résoudre

### 1. Vérifier l'Espace de Stockage

Assurez-vous que votre appareil dispose d'espace de stockage :
- Libérez de l'espace si le stockage est plein
- Supprimez des applications ou des fichiers inutilisés
- Vérifiez l'espace disque disponible

### 2. Effacer les Données de l'Application

Réinitialisez la base de données locale :
- **Application de bureau :** Recherchez "Effacer les données" dans les paramètres
- **Navigateur web :** Effacez les données du site pour le domaine POS
- **Remarque :** Cela nécessitera une re-synchronisation de toutes les données

### 3. Vérifier les Paramètres du Navigateur (Web)

Si vous utilisez la version web :
1. Assurez-vous qu'IndexedDB est activé
2. N'utilisez pas le mode privé/incognito (stockage limité)
3. Autorisez le site à stocker des données

### 4. Vérifier les Permissions

Sur ordinateur de bureau :
- Assurez-vous que l'application a la permission d'écrire dans son répertoire de données
- Vérifiez si un antivirus bloque l'accès à la base de données

### 5. Réinstaller l'Application

Si les autres solutions échouent :
1. Désinstallez l'application POS
2. Redémarrez votre appareil
3. Réinstallez l'application
4. Connectez-vous et synchronisez les données

## Récupération des Données

Si vous avez des données non sauvegardées :
- L'application peut avoir des modifications en attente qui n'ont pas été synchronisées
- Contactez le support si vous avez besoin d'aide pour récupérer des données
- Dans la plupart des cas, les données peuvent être re-synchronisées depuis le serveur
