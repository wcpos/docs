# SY01003: Permission refusée

## Ce que cela signifie

L'application POS n'a pas la permission d'accéder à une ressource requise. Il peut s'agir d'un fichier, d'un dossier ou d'une fonctionnalité système.

## Causes courantes

- **Permissions de fichier** — Impossible de lire ou d'écrire dans les fichiers nécessaires
- **Permissions de l'application** — L'application manque des permissions requises
- **Dossier protégé** — Tentative d'accès à un emplacement restreint
- **Logiciel de sécurité** — Antivirus bloquant l'accès

## Comment corriger

### 1. Exécuter en tant qu'administrateur (Windows)

Cliquez avec le bouton droit sur l'application et sélectionnez "Exécuter en tant qu'administrateur" si nécessaire pour la configuration initiale.

### 2. Vérifier les permissions de l'application

**macOS :**
- Préférences Système → Sécurité et confidentialité → Confidentialité
- Vérifiez que l'application dispose des permissions nécessaires

**Windows :**
- Paramètres → Confidentialité
- Passez en revue les permissions de l'application

### 3. Vérifier les permissions du dossier

Assurez-vous que l'application peut accéder à son dossier de données :
- L'utilisateur doit posséder le répertoire de données de l'application
- Les permissions d'écriture doivent être activées
- Pas de drapeaux en lecture seule sur le dossier

### 4. Examiner les paramètres de l'antivirus

Le logiciel de sécurité peut bloquer l'accès :
- Vérifiez les journaux de l'antivirus pour les blocages
- Ajoutez l'application POS à la liste autorisée
- Mettez sur liste blanche le dossier de données de l'application

### 5. Réinstaller l'application

Si les permissions sont corrompues :
1. Désinstallez l'application POS
2. Redémarrez votre appareil
3. Réinstallez à un emplacement standard
4. Exécutez avec des permissions normales (non administratives)

## Permissions du navigateur (version Web)

Si vous utilisez la version web :
- Autorisez le stockage local pour le site
- Activez les cookies pour le site
- N'utilisez pas de paramètres de navigateur trop restrictifs
