# SY01003 : Autorisation Refusée

## Ce que Cela Signifie

L'application POS n'a pas l'autorisation d'accéder à une ressource requise. Cela pourrait être un fichier, un dossier ou une fonctionnalité système.

## Causes Courantes

- **Permissions de fichiers** — Impossible de lire ou d'écrire les fichiers nécessaires
- **Autorisations de l'application** — L'application manque des permissions requises
- **Dossier protégé** — Tentative d'accès à un emplacement restreint
- **Logiciel de sécurité** — Antivirus bloquant l'accès

## Comment Résoudre

### 1. Exécuter en tant qu'Administrateur (Windows)

Faites un clic droit sur l'application et sélectionnez "Exécuter en tant qu'administrateur" si nécessaire pour la configuration initiale.

### 2. Vérifier les Permissions de l'Application

**macOS :**
- Préférences Système → Sécurité et Confidentialité → Confidentialité
- Vérifier que l'application a les permissions nécessaires

**Windows :**
- Paramètres → Confidentialité
- Vérifier les autorisations de l'application

### 3. Vérifier les Permissions de Dossier

Assurez-vous que l'application peut accéder à son dossier de données :
- L'utilisateur doit posséder le répertoire de données de l'application
- Les permissions d'écriture doivent être activées
- Aucun indicateur en lecture seule sur le dossier

### 4. Vérifier les Paramètres Antivirus

Le logiciel de sécurité peut bloquer l'accès :
- Vérifier les journaux de l'antivirus pour les blocages
- Ajouter l'application POS à la liste des applications autorisées
- Mettre le dossier de données de l'application sur liste blanche

### 5. Réinstaller l'Application

Si les permissions sont corrompues :
1. Désinstaller l'application POS
2. Redémarrer votre appareil
3. Réinstaller à un emplacement standard
4. Exécuter avec des permissions normales (non administrateur)

## Permissions du Navigateur (Version Web)

Si vous utilisez la version web :
- Autoriser le stockage local pour le site
- Activer les cookies pour le site
- Ne pas utiliser de paramètres de navigateur trop restrictifs
