# API03005 : Limite de Taux Dépassée {#api03005-rate-limit-exceeded}

## Ce Que Cela Signifie {#what-this-means}

Vous avez effectué trop de demandes au serveur en peu de temps. La limitation des taux protège le serveur d'une surcharge.

## Causes Courantes {#common-causes}

- **Opérations de synchronisation rapides** — Plusieurs appareils se synchronisant simultanément
- **Rafraîchissement agressif** — Rafraîchissement des données trop fréquemment
- **Plugin de sécurité** — Un plugin impose des limites de demande
- **Limites d'hébergement** — Votre plan d'hébergement a des limites de taux API

## Mappage des Erreurs Serveur {#server-error-mapping}

Ce code d'erreur est déclenché lorsque le serveur renvoie :

| Code Serveur | Source |
|--------------|--------|
| HTTP 429 | Toute réponse du serveur (fallback) |

## Comment Corriger {#how-to-fix}

### 1. Attendre et Réessayer {#1-wait-and-retry}

Les limites de taux sont temporaires :
- Attendez 1 à 5 minutes
- Le POS réessaiera automatiquement
- Évitez de rafraîchir manuellement à plusieurs reprises

### 2. Réduire les Opérations Concurrentes {#2-reduce-concurrent-operations}

Si vous utilisez plusieurs terminaux POS :
- Échelonnez la synchronisation initiale entre les appareils
- Évitez de rafraîchir tous les appareils en même temps
- Laissez-en un se compléter avant de commencer un autre

### 3. Vérifiez les Plugins de Sécurité {#3-check-security-plugins}

Des plugins comme Wordfence peuvent bloquer des demandes rapides :
- Ajoutez l'application POS à la liste blanche
- Ajoutez vos adresses IP de vos appareils POS à la liste blanche
- Ajustez les seuils de limitation de taux

### 4. Examinez les Limites d'Hébergement {#4-review-hosting-limits}

Certains hébergeurs limitent les requêtes API :
- Vérifiez les détails de votre plan d'hébergement
- Envisagez de passer à un plan supérieur si les limites sont trop basses
- Contactez le support pour augmenter les limites

### 5. Optimisez la Fréquence de Sync {#5-optimise-sync-frequency}

Dans les paramètres POS :
- Ajustez les intervalles d'auto-rafraîchissement
- Utilisez la sync manuelle lorsque cela est approprié
- Évitez les rafraîchissements de données inutiles
