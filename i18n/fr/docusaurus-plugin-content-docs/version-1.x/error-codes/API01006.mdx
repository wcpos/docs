# API01006 : Réseau Injoignable

## Ce Que Cela Signifie

Le système de caisse ne peut pas établir de route vers le serveur. Il s'agit d'un problème de niveau réseau — les paquets de données ne trouvent pas de chemin vers votre serveur.

## Causes Courantes

- **Problèmes de configuration réseau** — Paramètres de passerelle ou de routage incorrects
- **Problèmes de FAI** — Votre fournisseur d'accès Internet rencontre des difficultés
- **Interférence VPN** — Un VPN peut acheminer le trafic de manière incorrecte
- **Problèmes de réseau serveur** — Le réseau du serveur peut être hors service

## Comment Résoudre

### 1. Vérifiez Votre Connexion Internet

- Essayez d'accéder à d'autres sites web
- Redémarrez votre routeur/modem
- Vérifiez que vous êtes connecté au réseau

### 2. Désactivez Temporairement le VPN

Si vous utilisez un VPN :
- Déconnectez-vous et réessayez
- Essayez un autre serveur VPN
- Vérifiez si le VPN permet l'accès à l'emplacement de votre serveur

### 3. Contactez Votre FAI

Contactez votre fournisseur d'accès Internet si :
- D'autres sites sont également inaccessibles
- Le problème persiste après avoir redémarré les équipements réseau

### 4. Vérifiez l'État du Serveur

- Vérifiez si votre hébergeur signale des problèmes réseau
- Essayez d'accéder à votre site depuis un autre réseau (par exemple, données mobiles)
- Utilisez des outils en ligne pour vérifier si le site est inaccessible pour tout le monde

### 5. Vérifiez les Paramètres du Pare-feu

Assurez-vous que le pare-feu de votre appareil ne bloque pas les connexions sortantes vers votre serveur.
