# API01006: Réseau Inaccessible

## Ce Que Cela Signifie

Le POS ne peut pas établir de connexion avec le serveur. Il s'agit d'un problème au niveau du réseau : les paquets de données ne peuvent pas trouver de chemin vers votre serveur.

## Causes Courantes

- **Problèmes de configuration réseau** — Paramètres de passerelle ou de routage incorrects
- **Problèmes de fournisseur d'accès Internet** — Votre fournisseur d'accès Internet rencontre des problèmes
- **Interférence VPN** — Un VPN peut acheminer le trafic incorrectement
- **Problèmes de réseau du serveur** — Le réseau du serveur peut être hors service

## Comment Réparer

### 1. Vérifiez Votre Connexion Internet

- Essayez d'accéder à d'autres sites web
- Redémarrez votre routeur/modem
- Vérifiez que vous êtes connecté au réseau

### 2. Désactivez le VPN Temporairement

Si vous utilisez un VPN :
- Déconnectez-vous et réessayez
- Essayez un autre serveur VPN
- Vérifiez si le VPN permet d'accéder à l'emplacement de votre serveur

### 3. Vérifiez Avec Votre FAI

Contactez votre fournisseur d'accès Internet si :
- D'autres sites sont également inaccessibles
- Le problème persiste après le redémarrage de l'équipement réseau

### 4. Vérifiez l'État du Serveur

- Vérifiez si votre fournisseur d'hébergement signale des problèmes de réseau
- Essayez d'accéder à votre site depuis un autre réseau (par exemple, données mobiles)
- Utilisez des outils en ligne pour vérifier si le site est hors ligne pour tout le monde

### 5. Vérifiez les Paramètres du Pare-feu

Assurez-vous que le pare-feu de votre appareil ne bloque pas les connexions sortantes vers votre serveur.
