# Codes d'erreur

Lorsque WCPOS rencontre un problème, il affiche un code d'erreur qui aide à identifier le problème. Chaque code d'erreur suit le format `[DOMAIN][CATEGORY][SPECIFIC_CODE]` pour vous aider à comprendre rapidement le type et la nature de l'erreur.

## Format du code d'erreur

| Composant | Description | Exemple |
|-----------|-------------|---------|
| **Domaine** | La zone système | `API`, `DB`, `PY`, `SY` |
| **Catégorie** | La catégorie d'erreur | `01`, `02`, `03`, etc. |
| **Code spécifique** | L'identifiant unique | `001`, `002`, `003`, etc. |

**Exemple :** `API04001` = API (domaine) + 04 (erreurs de réponse) + 001 (format de réponse invalide)

## Domaines d'erreur

### [Erreurs API](./api)

Les erreurs API se produisent lors de la communication avec votre serveur WooCommerce.

| Catégorie | Plage de codes | Description |
|----------|----------------|-------------|
| Connexion | API01xxx | Problèmes de réseau et de connectivité |
| Authentification | API02xxx | Problèmes de connexion, de jetons et de permissions |
| Demande | API03xxx | Problèmes avec les demandes sortantes |
| Réponse | API04xxx | Problèmes avec les réponses du serveur |
| Plugin/WordPress | API05xxx | Problèmes avec les plugins WordPress ou WooCommerce |
| Configuration | API06xxx | Problèmes d'installation et de configuration |

### [Erreurs de base de données](./db)

Les erreurs de base de données se produisent lors du stockage ou de la récupération de données localement.

| Catégorie | Plage de codes | Description |
|----------|----------------|-------------|
| Connexion | DB01xxx | Problèmes de connexion et de transaction de la base de données |
| Données | DB02xxx | Problèmes d'enregistrements et de contraintes |
| Requête | DB03xxx | Problèmes de syntaxe de requête et de type de données |

### [Erreurs de paiement](./py)

Les erreurs de paiement se produisent lors du processus de paiement et de traitement.

| Catégorie | Plage de codes | Description |
|----------|----------------|-------------|
| Carte | PY01xxx | Problèmes avec la carte de paiement |
| Passerelle | PY02xxx | Problèmes de communication avec la passerelle de paiement |

### [Erreurs système](./sy)

Les erreurs système sont liées aux ressources de l'appareil et à la configuration système.

| Catégorie | Plage de codes | Description |
|----------|----------------|-------------|
| Ressource | SY01xxx | Problèmes de mémoire, de disque et de permissions |
| Service | SY02xxx | Problèmes de configuration et de disponibilité des services |
