# Codes d'Erreur

Lorsque WCPOS rencontre un problème, il affiche un code d'erreur qui aide à identifier le problème. Chaque code d'erreur suit le format `[DOMAINE][CATEGORIE][CODE_SPECIFIQUE]` pour vous aider à comprendre rapidement le type et la nature de l'erreur.

## Format du Code d'Erreur

| Composant | Description | Exemple |
|-----------|-------------|---------|
| **Domaine** | La zone du système | `API`, `DB`, `PY`, `SY` |
| **Catégorie** | La catégorie d'erreur | `01`, `02`, `03`, etc. |
| **Code Spécifique** | L'identifiant unique | `001`, `002`, `003`, etc. |

**Exemple :** `API04001` = API (domaine) + 04 (erreurs de réponse) + 001 (format de réponse invalide)

## Domaines d'Erreur

### [Erreurs API](./api)

Les erreurs API surviennent lors de la communication avec votre serveur WooCommerce.

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| Connexion | API01xxx | Problèmes de réseau et de connectivité |
| Authentification | API02xxx | Problèmes de connexion, jetons et permissions |
| Requête | API03xxx | Problèmes avec les requêtes sortantes |
| Réponse | API04xxx | Problèmes avec les réponses du serveur |
| Plugin/WordPress | API05xxx | Problèmes avec les plugins WordPress ou WooCommerce |
| Configuration | API06xxx | Problèmes de configuration et de paramétrage |

### [Erreurs de Base de Données](./db)

Les erreurs de base de données surviennent lors du stockage ou de la récupération de données locales.

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| Connexion | DB01xxx | Problèmes de connexion et de transaction de base de données |
| Données | DB02xxx | Problèmes d'enregistrement et de contraintes |
| Requête | DB03xxx | Problèmes de syntaxe de requête et de type de données |

### [Erreurs de Paiement](./py)

Les erreurs de paiement surviennent pendant le paiement et le traitement du paiement.

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| Carte | PY01xxx | Problèmes liés à la carte de paiement |
| Passerelle | PY02xxx | Problèmes de communication avec la passerelle de paiement |

### [Erreurs Système](./sy)

Les erreurs système sont liées aux ressources de l'appareil et à la configuration du système.

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| Ressource | SY01xxx | Problèmes de mémoire, de disque et de permissions |
| Service | SY02xxx | Configuration et disponibilité des services |

## Référence Rapide

Vous recherchez un code d'erreur spécifique ? Utilisez la barre latérale pour naviguer vers le domaine, ou recherchez directement le code.

| Domaine | Nombre de Codes | Problèmes Courants |
|--------|-------------|---------------|
| [API](./api) | 39 codes | Communication serveur, authentification, problèmes de plugin |
| [DB](./db) | 9 codes | Stockage et récupération de données locales |
| [PY](./py) | 6 codes | Traitement des paiements |
| [SY](./sy) | 5 codes | Ressources de l'appareil et configuration |
