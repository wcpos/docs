# API01001 : Délai de connexion dépassé

## Ce que cela signifie

L'application POS a tenté de se connecter à votre serveur WooCommerce, mais le serveur a mis trop de temps à répondre. Cela indique généralement des problèmes de latence réseau ou un serveur surchargé.

## Causes courantes

- **Connexion internet lente** — Votre connexion réseau peut connaître une latence élevée
- **Surcharge du serveur** — Votre serveur WordPress peut traiter trop de requêtes
- **Demandes de données volumineuses** — Demander trop de données à la fois peut provoquer des dépassements de délai
- **Délais de pare-feu ou de proxy** — Les dispositifs de sécurité réseau peuvent ajouter de la latence

## Comment résoudre

### 1. Vérifiez votre connexion internet

Testez la vitesse et la stabilité de votre connexion internet. Essayez de charger votre panneau d'administration WooCommerce dans un navigateur pour voir s'il répond lentement.

### 2. Réduire la charge du serveur

Si votre serveur est surchargé :
- Envisagez de mettre à niveau votre plan d'hébergement
- Désactivez temporairement les plugins inutiles
- Vérifiez les tâches cron ou les processus en arrière-plan gourmands

### 3. Optimiser le temps de réponse du serveur

Contactez votre hébergeur à propos de :
- L'activation de l'opcache PHP
- L'augmentation des limites de mémoire PHP
- L'utilisation du cache côté serveur

### 4. Augmenter les paramètres de délai d'attente

Si votre serveur a légitimement besoin de plus de temps :
- Vérifiez si votre hébergeur permet d'augmenter le temps d'exécution PHP
- Envisagez d'utiliser un CDN pour réduire la latence
