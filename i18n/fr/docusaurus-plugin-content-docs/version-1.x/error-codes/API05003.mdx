# API05003 : extension WCPOS obsolète {#api05003-wcpos-plugin-outdated}

## Signification {#what-this-means}

L'extension WCPOS installée sur votre serveur est obsolète et n'est pas compatible avec votre version de l'application PDV. L'extension doit être mise à jour.

## Causes fréquentes {#common-causes}

- **Extension non mise à jour** — les mises à jour automatiques sont peut-être désactivées
- **Échec de la mise à jour** — une tentative de mise à jour précédente a échoué
- **Incompatibilité de version** — l'application PDV a été mise à jour, mais pas l'extension
- **Site de préproduction/développement** — test sur une version plus ancienne

## Comment résoudre le problème {#how-to-fix}

### 1. Mettre à jour l'extension {#1-update-the-plugin}

Dans l'administration WordPress :
1. Accédez à Extensions → Extensions installées
2. Recherchez « WCPOS »
3. Si une mise à jour est disponible, cliquez sur « Mettre à jour »

### 2. Activer les mises à jour automatiques {#2-enable-auto-updates}

Pour éviter de futures incompatibilités :
1. Accédez à Extensions → Extensions installées
2. Recherchez « WCPOS »
3. Cliquez sur « Activer les mises à jour automatiques »

### 3. Mise à jour manuelle {#3-manual-update}

Si la mise à jour automatique échoue :
1. Téléchargez la dernière version depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Désactivez l'extension actuelle
3. Supprimez les anciens fichiers de l'extension
4. Téléversez et activez la nouvelle version

### 4. Vérifier les erreurs de mise à jour {#4-check-for-update-errors}

Si les mises à jour échouent :
- Vérifiez les permissions des fichiers sur wp-content/plugins
- Vérifiez que vous disposez d'espace disque
- Recherchez des problèmes de mémoire PHP
- Consultez les journaux d'erreurs pour plus de détails

### 5. Revenir à une version antérieure de l'application PDV (temporaire) {#5-downgrade-pos-app-temporary}

Si vous ne pouvez pas mettre à jour l'extension immédiatement :
- Utilisez une version antérieure de l'application PDV
- Il s'agit uniquement d'une solution temporaire
- Prévoyez de mettre à jour l'extension prochainement

## Compatibilité des versions {#version-compatibility}

Maintenez toujours les deux composants à jour :
- L'application PDV
- L'extension WordPress WCPOS

Consultez les notes de version pour repérer d'éventuels changements incompatibles lors de la mise à jour.
