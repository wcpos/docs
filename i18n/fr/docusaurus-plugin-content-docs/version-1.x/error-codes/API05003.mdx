# API05003 : Plugin WCPOS Ancien {#api05003-wcpos-plugin-outdated}

## Ce que cela signifie {#what-this-means}

Le plugin WCPOS installé sur votre serveur est ancien et n'est pas compatible avec votre version de l'application POS. Vous devez mettre à jour le plugin.

## Causes courantes {#common-causes}

- **Plugin non mis à jour** — Les mises à jour automatiques peuvent être désactivées
- **Échec de la mise à jour** — Une tentative de mise à jour précédente a échoué
- **Inadéquation de version** — L'application POS a été mise à jour mais le plugin ne l'a pas été
- **Site de staging/dev** — Test sur une version plus ancienne

## Comment réparer {#how-to-fix}

### 1. Mettre à jour le plugin {#1-update-the-plugin}

Dans l'admin WordPress :
1. Allez dans Plugins → Plugins installés
2. Trouvez "WooCommerce POS"
3. S'il y a une mise à jour disponible, cliquez sur "Mettre à jour maintenant"

### 2. Activer les mises à jour automatiques {#2-enable-auto-updates}

Pour éviter de futures inadéquations :
1. Allez dans Plugins → Plugins installés
2. Trouvez "WooCommerce POS"
3. Cliquez sur "Activer les mises à jour automatiques"

### 3. Mise à jour manuelle {#3-manual-update}

Si la mise à jour automatique échoue :
1. Téléchargez la dernière version depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Désactivez le plugin actuel
3. Supprimez les anciens fichiers du plugin
4. Téléchargez et activez la nouvelle version

### 4. Vérifier les erreurs de mise à jour {#4-check-for-update-errors}

Si les mises à jour échouent :
- Vérifiez les permissions des fichiers sur wp-content/plugins
- Vérifiez que vous avez de l'espace disque
- Recherchez des problèmes de mémoire PHP
- Consultez les journaux d'erreurs pour plus de détails

### 5. Rétrograder l'application POS (Temporaire) {#5-downgrade-pos-app-temporary}

Si vous ne pouvez pas mettre à jour le plugin immédiatement :
- Utilisez une version antérieure de l'application POS
- C'est une solution temporaire uniquement
- Prévoyez de mettre à jour le plugin bientôt

## Compatibilité des versions {#version-compatibility}

Gardez toujours les deux composants à jour :
- L'application POS
- Le plugin WordPress WCPOS

Vérifiez les notes de version pour tout changement significatif lors de la mise à jour.
