# PY02002 : Délai de paiement dépassé

## Ce que cela signifie

Le traitement du paiement a pris trop de temps et a dépassé le délai imparti. La passerelle de paiement n'a pas répondu dans le temps attendu.

## Causes courantes

- **Surcharge de la passerelle** — Le processeur de paiement connaît un volume élevé
- **Latence réseau** — Connexion lente au service de paiement
- **Délai de traitement** — Contrôles de fraude ou vérification complexes
- **Problèmes de serveur** — La passerelle de paiement rencontre des problèmes de performance

## Comment résoudre

### 1. Vérifier si le paiement a réussi

**Important :** Avant de réessayer, vérifiez le statut du paiement :
- Consultez le tableau de bord de votre passerelle de paiement
- Recherchez la transaction dans WooCommerce
- Ne traitez pas à nouveau si le paiement a effectivement réussi

### 2. Réessayer avec précaution

Si le paiement n'a définitivement pas abouti :
- Attendre 30-60 secondes
- Réessayer
- Surveiller les charges en double

### 3. Vérifier votre connexion

Assurez une connectivité stable :
- Tester votre connexion internet
- Essayer une connexion filaire si en WiFi
- Vérifier les problèmes réseau

### 4. Essayer à un moment moins chargé

Si la passerelle est surchargée :
- Les heures de pointe au détail peuvent causer des ralentissements
- Réessayer peu après
- Utiliser un autre moyen de paiement

### 5. Contacter le prestataire de paiement

Si les dépassements de délai persistent :
- Vérifier leur page de statut
- Contacter leur support
- Signaler les problèmes récurrents

## Éviter les charges en double

Lorsqu'un dépassement de délai se produit :
1. **Ne pas réessayer immédiatement** — Vérifier le statut d'abord
2. **Examiner les transactions** — Vérifier la passerelle et WooCommerce
3. **Rembourser les doublons** — Traiter les remboursements pour les doublons accidentels
4. **Informer le client** — Expliquer la situation en cas de délai

## Actions alternatives

Si le paiement continue de dépasser le délai :
- Accepter un paiement en espèces
- Enregistrer comme « à payer plus tard » si autorisé
- Utiliser un autre moyen de paiement
