# PY01002: Fonds Insuffisants

## Ce Que Cela Signifie

Le paiement a été refusé car il n'y a pas suffisamment de fonds sur le compte pour couvrir la transaction.

## Causes Courantes

- **Solde de compte faible** — Fonds insuffisants disponibles
- **Transactions en attente** — Autres transactions en attente réduisant le solde disponible
- **Limite de crédit atteinte** — Pour les cartes de crédit, la limite est atteinte
- **Fonds en attente** — Les fonds sont en attente pour d'autres transactions

## Comment Résoudre le Problème

### 1. Utiliser un Autre Moyen de Paiement

La solution la plus rapide :
- Essayer une autre carte
- Utiliser des espèces
- Répartir le paiement sur plusieurs méthodes (si pris en charge)

### 2. Réduire le Montant de la Commande

Si un paiement partiel est possible :
- Supprimer certains articles
- Appliquer des remises ou des coupons
- Effectuer une transaction plus petite

### 3. Actions du Client

Le client peut avoir besoin de :
- Transférer des fonds vers le compte
- Attendre que les transactions en attente soient traitées
- Régler le solde de la carte de crédit
- Demander une augmentation de la limite de crédit

### 4. Diviser la Transaction

Si votre POS prend en charge les paiements fractionnés :
- Payer une partie avec une carte
- Payer le reste avec une autre méthode

## Remarque sur la Confidentialité

Lors de la communication de cela aux clients :
- Être discret sur la raison spécifique du refus
- Indiquer simplement "la carte a été refusée"
- Leur permettre d'essayer un autre moyen de paiement en privé
