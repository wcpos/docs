# PY01002 : Fonds Insuffisants

## Ce Que Cela Signifie

Le paiement a été refusé car il n'y a pas assez d'argent sur le compte pour couvrir la transaction.

## Causes Courantes

- **Solde de compte bas** — Fonds disponibles insuffisants
- **Transactions en attente** — Autres transactions en attente réduisant le solde disponible
- **Limite de crédit atteinte** — Pour les cartes de crédit, la limite est épuisée
- **Blocage des fonds** — Fonds bloqués pour d'autres transactions

## Comment Résoudre

### 1. Utiliser Un Autre Moyen de Paiement

La solution la plus rapide :
- Essayer une carte différente
- Utiliser des espèces
- Répartir le paiement sur plusieurs méthodes (si possible)

### 2. Réduire Le Montant de la Commande

Si un paiement partiel est possible :
- Retirer certains articles
- Appliquer des réductions ou des coupons
- Effectuer une transaction plus petite

### 3. Actions du Client

Le client peut devoir :
- Transférer des fonds sur le compte
- Attendre que les transactions en attente soient traitées
- Rembourser le solde de la carte de crédit
- Demander une augmentation de la limite de crédit

### 4. Fractionner la Transaction

Si votre POS prend en charge les paiements fractionnés :
- Payer une partie avec une carte
- Payer le reste avec une autre méthode

## Note de Confidentialité

Lors de la communication avec les clients :
- Être discret sur la raison spécifique du refus
- Simplement indiquer que la carte a été refusée
- Leur permettre d'essayer une autre méthode de paiement en privé
