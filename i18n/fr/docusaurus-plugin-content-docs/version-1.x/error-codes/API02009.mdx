# API02009 : Jeton de Rafraîchissement Expiré

## Ce Que Cela Signifie

Votre jeton de rafraîchissement a dépassé sa durée de vie et ne peut plus être utilisé pour obtenir de nouveaux jetons d'accès. Contrairement aux jetons d'accès (qui expirent rapidement), les jetons de rafraîchissement durent généralement des jours ou des semaines mais finissent par expirer.

## Causes Courantes

- **Inactivité prolongée** — Vous n'avez pas utilisé le POS depuis longtemps
- **Durée de vie courte du jeton** — Serveur configuré avec une expiration courte du jeton de rafraîchissement
- **Décalage horaire** — L'heure de l'appareil ou du serveur est significativement différente

## Comment Résoudre

### 1. Se Reconnecter

Lorsque le jeton de rafraîchissement expire, vous devez vous authentifier à nouveau :
1. Déconnectez-vous du POS
2. Entrez vos identifiants
3. Un nouveau jeton de rafraîchissement sera émis

### 2. Utiliser le POS Régulièrement

Pour éviter cela à l'avenir :
- Utilisez le POS régulièrement pour maintenir les jetons actifs
- Les jetons sont rafraîchis automatiquement lorsque vous êtes actif
- Des périodes prolongées d'inactivité nécessiteront une reconnexion

### 3. Vérifier les Paramètres de Temps

Assurez-vous d'avoir un temps précis sur tous les systèmes :
- Activez la synchronisation automatique de l'heure sur votre appareil
- Vérifiez que l'heure du serveur est correcte
- Les différences de temps peuvent causer une expiration prématurée

### 4. Ajuster la Durée de Vie du Jeton (Avancé)

Si les jetons expirent trop rapidement pour vos besoins, un administrateur de serveur peut ajuster la durée de vie du jeton de rafraîchissement dans les paramètres d'authentification.
