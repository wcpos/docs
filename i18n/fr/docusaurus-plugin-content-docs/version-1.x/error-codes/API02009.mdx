# API02009 : Jeton d'actualisation expiré

## Ce que cela signifie

Votre jeton d'actualisation a dépassé sa durée de vie et ne peut plus être utilisé pour obtenir de nouveaux jetons d'accès. Contrairement aux jetons d'accès (qui expirent rapidement), les jetons d'actualisation durent généralement des jours ou des semaines, mais finissent par expirer.

## Causes courantes

- **Inactivité prolongée** — Vous n'avez pas utilisé le POS depuis longtemps
- **Durée de vie du jeton courte** — Serveur configuré avec une expiration de jeton d'actualisation courte
- **Désynchronisation de l'horloge** — L'heure de l'appareil ou du serveur est significativement décalée

## Comment résoudre le problème

### 1. Se connecter à nouveau

Lorsque le jeton d'actualisation expire, vous devez vous authentifier à nouveau :
1. Déconnexion du POS
2. Saisissez vos identifiants
3. Un nouveau jeton d'actualisation sera émis

### 2. Utiliser le POS régulièrement

Pour éviter cela à l'avenir :
- Utilisez le POS régulièrement pour garder les jetons à jour
- Les jetons sont actualisés automatiquement lorsque vous êtes actif
- Des périodes prolongées d'inactivité entraîneront une nouvelle connexion

### 3. Vérifier les paramètres de temps

Assurez-vous que l'heure est précise sur tous les systèmes :
- Activez la synchronisation automatique de l'heure sur votre appareil
- Vérifiez que l'heure du serveur est correcte
- Les différences de temps peuvent provoquer une expiration prématurée

### 4. Ajuster la durée de vie du jeton (Avancé)

Si les jetons expirent trop rapidement pour vos besoins, un administrateur du serveur peut ajuster la durée de vie du jeton d'actualisation dans les paramètres d'authentification.
