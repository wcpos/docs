# SY02001 : Configuration Invalide

## Ce Que Cela Signifie

La configuration du système est invalide ou corrompue. L'application POS a trouvé des paramètres de configuration qui n'ont pas de sens ou sont incomplets.

## Causes Courantes

- **Paramètres corrompus** — Le fichier de configuration est devenu corrompu
- **Configuration incomplète** — Le processus de configuration n'a pas été terminé
- **Erreur de modification manuelle** — Les fichiers de configuration ont été modifiés incorrectement
- **Problème de mise à niveau de version** — L'ancienne configuration est incompatible avec la nouvelle version

## Comment Résoudre

### 1. Réinitialiser aux Paramètres par Défaut

Si disponible dans l'application :
- Recherchez "Réinitialiser les paramètres" ou "Restaurer les valeurs par défaut"
- Cela efface les paramètres personnalisés mais corrige la corruption

### 2. Terminer la Configuration

Si la configuration n'est pas terminée :
1. Recommencez le processus de configuration
2. Complétez toutes les étapes requises
3. Vérifiez la configuration avant de terminer

### 3. Effacer les Données de l'Application

Supprimer la configuration corrompue :
- **Bureau :** Supprimer le dossier de données de l'application, redémarrer l'application
- **Navigateur :** Effacer les données du site, recharger

### 4. Réinstaller l'Application

Pour des problèmes persistants :
1. Désinstaller complètement
2. Supprimer tous les dossiers de données restants
3. Réinstaller à neuf
4. Terminer la configuration à partir de zéro

### 5. Vérifier les Mises à Jour

Les formats de configuration peuvent changer :
- Mettre à jour l'application POS
- Mettre à jour le plugin WCPOS
- Reconfigurer après la mise à jour

## Que Sera Réinitialisé

Lors de la réinitialisation de la configuration :
- Vous devrez vous reconnecter
- Les paramètres personnalisés seront perdus
- Les données devront se resynchroniser
- Mais vos données WooCommerce sont sécurisées sur le serveur
