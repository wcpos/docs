# SY02001 : Configuration invalide

## Ce que cela signifie

La configuration du système est invalide ou corrompue. L'application de point de vente a trouvé des paramètres de configuration qui n'ont pas de sens ou qui sont incomplets.

## Causes courantes

- **Paramètres corrompus** — Le fichier de configuration est devenu corrompu
- **Configuration incomplète** — Le processus de configuration n'a pas été terminé
- **Erreur d'édition manuelle** — Les fichiers de configuration ont été modifiés incorrectement
- **Problème de mise à niveau de version** — L'ancienne configuration n'est pas compatible avec la nouvelle version

## Comment résoudre le problème

### 1. Réinitialiser les paramètres par défaut

Si disponible dans l'application :
- Recherchez "Réinitialiser les paramètres" ou "Restaurer les valeurs par défaut"
- Cela efface les paramètres personnalisés mais corrige la corruption

### 2. Compléter la configuration

Si la configuration n'était pas terminée :
1. Relancez le processus de configuration
2. Complétez toutes les étapes requises
3. Vérifiez la configuration avant de terminer

### 3. Effacer les données de l'application

Supprimez la configuration corrompue :
- **Bureau :** Supprimez le dossier de données de l'application, redémarrez l'application
- **Navigateur :** Effacez les données du site, rechargez

### 4. Réinstaller l'application

Pour des problèmes persistants :
1. Désinstallez complètement
2. Supprimez tous les dossiers de données restants
3. Réinstallez à neuf
4. Complétez la configuration depuis le début

### 5. Vérifier les mises à jour

Les formats de configuration peuvent changer :
- Mettez à jour l'application de point de vente
- Mettez à jour le plugin WCPOS
- Reconfigurez après la mise à jour

## Ce qui est réinitialisé

Lors de la réinitialisation de la configuration :
- Vous devrez vous reconnecter
- Les paramètres personnalisés seront perdus
- Les données devront être resynchronisées
- Mais vos données WooCommerce sont en sécurité sur le serveur
