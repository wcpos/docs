# Erreurs de Base de Données

Les erreurs de base de données se produisent lors du stockage ou de la récupération de données dans la base de données locale. WCPOS utilise une base de données locale pour stocker des produits, des clients et d'autres données pour un accès rapide et des fonctionnalités hors ligne. Ces erreurs sont préfixées par `DB`.

## Catégories

| Catégorie | Plage de Codes | Description |
|----------|------------|-------------|
| [Connexion](#erreurs-de-connexion) | DB01xxx | Problèmes de connexion et de transaction de base de données |
| [Données](#erreurs-de-données) | DB02xxx | Problèmes d'enregistrements et de contraintes |
| [Requête](#erreurs-de-requête) | DB03xxx | Problèmes de syntaxe de requête et de type de données |

---

## Erreurs de Connexion

Problèmes de connexion ou d'opération sur la base de données locale.

| Code | Nom | Description |
|------|------|-------------|
| [DB01001](./DB01001) | Connexion Échouée | Impossible de se connecter à la base de données locale |
| [DB01002](./DB01002) | Délai de Requête Dépassé | La requête de base de données a pris trop de temps |
| [DB01003](./DB01003) | Transaction Échouée | La transaction de base de données n'a pas pu se terminer |

## Erreurs de Données

Problèmes liés aux enregistrements, doublons et contraintes.

| Code | Nom | Description |
|------|------|-------------|
| [DB02001](./DB02001) | Enregistrement Dupliqué | Un enregistrement avec cet ID existe déjà |
| [DB02002](./DB02002) | Enregistrement Non Trouvé | L'enregistrement demandé n'existe pas |
| [DB02003](./DB02003) | Violation de Contrainte | Les données violent les contraintes de base de données |

## Erreurs de Requête

Problèmes de syntaxe de requête et de types de données.

| Code | Nom | Description |
|------|------|-------------|
| [DB03001](./DB03001) | Erreur de Syntaxe de Requête | La requête de base de données contient des erreurs de syntaxe |
| [DB03002](./DB03002) | Type de Données Invalide | Le type de données ne correspond pas au type attendu |
| [DB03003](./DB03003) | Champ Obligatoire Manquant | Un champ obligatoire est vide |
