# Erreurs de base de données {#database-errors}

Les erreurs de base de données se produisent lors du stockage ou de la récupération de données dans la base de données locale. WCPOS utilise une base de données locale pour stocker des produits, des clients et d'autres données pour un accès rapide et une fonctionnalité hors ligne. Ces erreurs sont précédées de `DB`.

## Catégories {#categories}

| Catégorie | Plage de codes | Description |
|-----------|----------------|-------------|
| [Connexion](#connection-errors) | DB01xxx | Problèmes de connexion et de transaction de base de données |
| [Données](#data-errors) | DB02xxx | Problèmes d'enregistrement et de contrainte |
| [Requête](#query-errors) | DB03xxx | Problèmes de syntaxe de requête et de type de données |

---

## Erreurs de connexion {#connection-errors}

Problèmes de connexion ou d'opérations sur la base de données locale.

| Code | Nom | Description |
|------|-----|-------------|
| [DB01001](./DB01001) | Échec de connexion | Impossible de se connecter à la base de données locale |
| [DB01002](./DB01002) | Délai d'attente de requête | La requête de base de données a pris trop de temps |
| [DB01003](./DB01003) | Échec de la transaction | La transaction de la base de données n'a pas pu se terminer |

## Erreurs de données {#data-errors}

Problèmes d'enregistrements, de doublons et de contraintes.

| Code | Nom | Description |
|------|-----|-------------|
| [DB02001](./DB02001) | Enregistrement en double | Un enregistrement avec cet ID existe déjà |
| [DB02002](./DB02002) | Enregistrement non trouvé | L'enregistrement demandé n'existe pas |
| [DB02003](./DB02003) | Violation de contrainte | Les données violent les contraintes de la base de données |

## Erreurs de requête {#query-errors}

Problèmes de syntaxe de requête et de types de données.

| Code | Nom | Description |
|------|-----|-------------|
| [DB03001](./DB03001) | Erreur de syntaxe de requête | La requête de base de données contient des erreurs de syntaxe |
| [DB03002](./DB03002) | Type de donnée invalide | Le type de données ne correspond pas au type attendu |
| [DB03003](./DB03003) | Champ requis manquant | Un champ requis est vide |
