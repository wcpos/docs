# SY01001 : Mémoire Saturée

## Ce que Cela Signifie

Votre appareil manque de mémoire (RAM) et ne peut pas terminer l'opération demandée. Le système de Point de Vente a besoin de suffisamment de mémoire pour fonctionner correctement.

## Causes Courantes

- **Trop d'applications ouvertes** — Autres applications utilisant de la mémoire
- **Jeu de données volumineux** — Nombreux produits/clients chargés
- **Fuite de mémoire** — Application ne libérant pas correctement la mémoire
- **RAM insuffisante** — L'appareil n'a pas assez de mémoire

## Comment Résoudre

### 1. Fermer les Autres Applications

Libérer de la mémoire :
- Fermer les onglets de navigateur inutilisés
- Fermer les autres applications
- Fermer les programmes en arrière-plan

### 2. Redémarrer le Point de Vente

Actualiser la mémoire de l'application :
1. Fermer complètement le Point de Vente
2. Attendre quelques secondes
3. Rouvrir l'application

### 3. Redémarrer Votre Appareil

Un redémarrage complet efface toute la mémoire :
1. Sauvegarder tout travail en cours
2. Redémarrer votre ordinateur/appareil
3. Ouvrir uniquement le Point de Vente au démarrage

### 4. Effacer les Données du Navigateur (Version Web)

Si utilisé dans un navigateur :
- Effacer le cache du navigateur
- Fermer et rouvrir le navigateur
- Utiliser moins d'onglets de navigateur

### 5. Réduire la Charge de Données

Si vous avez un catalogue très volumineux :
- Utiliser des filtres pour charger moins de données
- Envisager la pagination des résultats
- Limiter la portée de synchronisation si possible

## Configuration Minimale Requise

Vérifier que votre appareil répond aux spécifications minimales :
- **RAM :** 4 Go minimum, 8 Go recommandés
- **Navigateur :** Navigateur moderne (Chrome, Firefox, Edge)
- **Application de Bureau :** Consulter les notes de version pour les exigences

## Problèmes Récurrents

Si vous manquez fréquemment de mémoire :
- Envisager de mettre à niveau la RAM de votre appareil
- Utiliser un appareil avec plus de ressources
- Signaler le problème s'il semble excessif
