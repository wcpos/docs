# SY01001: Mémoire Épuisée {#sy01001-out-of-memory}

## Ce Que Cela Signifie {#what-this-means}

Votre appareil manque de mémoire (RAM) et ne peut pas terminer l'opération demandée. Le POS a besoin d'une mémoire suffisante pour fonctionner correctement.

## Causes Courantes {#common-causes}

- **Trop d'applications ouvertes** — D'autres applications utilisant de la mémoire
- **Grand ensemble de données** — De nombreux produits/clients chargés
- **Fuite de mémoire** — Application ne libérant pas correctement la mémoire
- **RAM insuffisante** — L'appareil n'a pas assez de mémoire

## Comment Corriger {#how-to-fix}

### 1. Fermer D'autres Applications {#1-close-other-applications}

Libérez de la mémoire :
- Fermez les onglets de navigateur inutilisés
- Fermez d'autres applications
- Fermez les programmes en arrière-plan

### 2. Redémarrer le POS {#2-restart-the-pos}

Rafraîchissez la mémoire de l'application :
1. Fermez complètement le POS
2. Attendez quelques secondes
3. Rouvrez l'application

### 3. Redémarrez Votre Appareil {#3-restart-your-device}

Un redémarrage complet efface toute la mémoire :
1. Enregistrez tout travail ouvert
2. Redémarrez votre ordinateur/appareil
3. Ouvrez uniquement le POS lors du redémarrage

### 4. Effacer les Données du Navigateur (Version Web) {#4-clear-browser-data-web-version}

Si vous utilisez dans un navigateur :
- Effacez le cache du navigateur
- Fermez et rouvrez le navigateur
- Utilisez moins d'onglets dans le navigateur

### 5. Réduire la Charge de Données {#5-reduce-data-load}

Si vous avez un catalogue très large :
- Utilisez des filtres pour charger moins de données à la fois
- Envisagez de paginer les résultats
- Limitez la portée de synchronisation si possible

## Exigences Minimales {#minimum-requirements}

Assurez-vous que votre appareil respecte les spécifications minimales :
- **RAM :** 4 Go minimum, 8 Go recommandé
- **Navigateur :** Navigateur moderne (Chrome, Firefox, Edge)
- **Application de Bureau :** Consultez les notes de version pour les exigences

## Problèmes Persistants {#ongoing-issues}

Si vous manquez fréquemment de mémoire :
- Envisagez de mettre à niveau la RAM de votre appareil
- Utilisez un appareil avec plus de ressources
- Signalez le problème s'il semble excessif
