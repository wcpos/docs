# API01007 : Appareil Hors Ligne {#api01007-device-offline}

## Ce que cela signifie {#what-this-means}

Votre appareil n'est pas connecté à Internet. Le POS a détecté qu'il n'y a pas de connexion réseau active.

## Causes Courantes {#common-causes}

- **WiFi déconnecté** — Votre appareil a perdu sa connexion WiFi
- **Ethernet débranché** — Le câble réseau est déconnecté
- **Mode avion activé** — Toutes les connexions sans fil sont désactivées
- **Adaptateur réseau désactivé** — L'interface réseau est éteinte

## Comment Réparer {#how-to-fix}

### 1. Vérifiez la Connexion WiFi {#1-check-wifi-connection}

- Vérifiez que le WiFi est activé sur votre appareil
- Assurez-vous que vous êtes connecté au bon réseau
- Vérifiez la force du signal WiFi

### 2. Vérifiez les Connexions Physiques {#2-check-physical-connections}

Si vous utilisez Ethernet :
- Vérifiez que le câble est bien branché
- Essayez un autre câble ou un autre port
- Vérifiez les voyants du commutateur/routeur réseau

### 3. Désactivez le Mode Avion {#3-disable-airplane-mode}

Vérifiez si le mode avion est activé par accident et désactivez-le.

### 4. Redémarrez l'Adaptateur Réseau {#4-restart-network-adapter}

**Windows :**
- Ouvrez Connexions Réseau
- Cliquez droit sur votre adaptateur → Désactiver, puis Activer

**macOS :**
- Désactivez et réactivez le WiFi depuis la barre de menu
- Ou : Préférences Système → Réseau → Désactiver/Activer

### 5. Redémarrez Votre Appareil {#5-restart-your-device}

Parfois, un simple redémarrage résout les problèmes de détection du réseau.

## Mode Hors Ligne {#offline-mode}

Lorsque vous êtes hors ligne, WCPOS peut toujours :
- Parcourir et rechercher des produits mis en cache localement
- Parcourir et rechercher des clients mis en cache localement
- Commencer de nouvelles commandes et ajouter des articles au panier
- Visualiser les données précédemment synchronisées

**Nécessite une connectivité :**
- Compléter/vérifier des commandes
- Créer de nouveaux clients

Lorsque la connectivité est rétablie, le POS reprendra automatiquement ses opérations normales.
