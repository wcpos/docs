# API01007 : Périphérique Hors Ligne

## Ce Que Cela Signifie

Votre périphérique n'est pas connecté à Internet. Le système de caisse a détecté qu'il n'y a pas de connexion réseau active.

## Causes Courantes

- **WiFi déconnecté** — Votre périphérique a perdu sa connexion WiFi
- **Câble Ethernet débranché** — Le câble réseau est déconnecté
- **Mode avion activé** — Toutes les connexions sans fil sont désactivées
- **Adaptateur réseau désactivé** — L'interface réseau est éteinte

## Comment Résoudre

### 1. Vérifier la Connexion WiFi

- Vérifier que le WiFi est activé sur votre périphérique
- S'assurer d'être connecté au bon réseau
- Vérifier la force du signal WiFi

### 2. Vérifier les Connexions Physiques

Si vous utilisez l'ethernet :
- Vérifier que le câble est bien branché
- Essayer un autre câble ou port
- Vérifier les voyants du commutateur/routeur réseau

### 3. Désactiver le Mode Avion

Vérifier si le mode avion est accidentellement activé et le désactiver.

### 4. Redémarrer l'Adaptateur Réseau

**Windows :**
- Ouvrir les Connexions Réseau
- Clic droit sur votre adaptateur → Désactiver, puis Activer

**macOS :**
- Éteindre puis rallumer le WiFi depuis la barre de menu
- Ou : Préférences Système → Réseau → Éteindre/Allumer

### 5. Redémarrer Votre Périphérique

Parfois, un simple redémarrage résout les problèmes de détection réseau.

## Mode Hors Ligne

Pendant le mode hors ligne, WCPOS peut toujours :
- Parcourir et rechercher des produits en cache local
- Parcourir et rechercher des clients en cache local
- Commencer de nouvelles commandes et ajouter des articles au panier
- Afficher les données précédemment synchronisées

**Nécessite une connectivité :**
- Finalisation/paiement des commandes
- Création de nouveaux clients

Lorsque la connectivité est rétablie, le système de caisse reprendra automatiquement ses opérations normales.
