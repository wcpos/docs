# PY01003 : Carte Expirée

## Ce que Cela Signifie

La carte de paiement a dépassé sa date d'expiration et ne peut plus être utilisée pour les transactions.

## Causes Communes

- **Carte expirée** — La date de validité de la carte est dépassée
- **Date d'expiration incorrecte saisie** — La date d'expiration a été saisie incorrectement
- **Nouvelle carte non activée** — Carte de remplacement reçue mais pas encore activée

## Comment Réparer

### 1. Vérifiez la Date d'Expiration

Regardez la carte :
- Trouvez la date d'expiration (format MM/AA)
- Comparez avec la date actuelle
- Vérifiez que la date saisie correspond à celle de la carte

### 2. Utilisez une Autre Carte

Si la carte est réellement expirée :
- Utilisez une autre carte valide
- Utilisez de l'argent ou un paiement alternatif
- Demandez si le client a une carte de remplacement

### 3. Obtenez la Carte Mise à Jour

Le titulaire de la carte doit :
- Vérifier le courrier pour une carte de remplacement
- Activer la nouvelle carte si reçue
- Contacter sa banque si aucune carte de remplacement n'est arrivée

### 4. Vérifiez la Saisie

Vérifiez à nouveau la saisie de l'expiration :
- Format MM/AA
- Mois et année corrects
- Pas de fautes de frappe

## Conseils pour le Service Client

Lorsqu'une carte est expirée :
- Informez poliment le client
- Suggérez-lui de vérifier pour une carte de remplacement
- Proposez des options de paiement alternatives
- Soyez compréhensif — cela arrive à tout le monde
