# PY01001 : Paiement Décliné

## Ce que cela signifie

Le paiement a été décliné par le processeur de paiement ou l'émetteur de la carte. La transaction n'a pas pu être complétée.

## Causes Courantes

- **Fonds insuffisants** — Pas assez d'argent sur le compte
- **Limite de carte dépassée** — La transaction dépasse les limites de la carte
- **Protection contre la fraude** — La banque a signalé comme suspecte
- **Détails de la carte incorrects** — Numéro de carte, CVV ou date d'expiration incorrects
- **Restrictions de carte** — Carte non activée pour ce type d'achat

## Comment Réparer

### 1. Vérifiez les Détails de la Carte

Vérifiez que les informations saisies sont correctes :
- Numéro de carte (tous les chiffres)
- Date d'expiration
- Code CVV/code de sécurité
- Adresse de facturation (si nécessaire)

### 2. Essayez à Nouveau

Parfois, des problèmes temporaires se produisent :
- Attendez un moment
- Saisissez à nouveau les détails de la carte
- Essayez la transaction à nouveau

### 3. Utilisez un Méthode de Paiement Différente

Si la carte continue d'être déclinée :
- Essayez une autre carte
- Utilisez un paiement en espèces
- Utilisez un moyen de paiement alternatif

### 4. Contactez la Banque

Le titulaire de la carte peut avoir besoin de :
- Vérifier la transaction avec sa banque
- Lever tout blocage de fraude
- Confirmer que la carte est active
- Vérifier les limites de dépenses

### 5. Vérifiez les Restrictions

Certaines cartes ont des restrictions :
- Transactions internationales désactivées
- Transactions en ligne/POS bloquées
- Certaines catégories de commerçants bloquées

## Pour les Propriétaires de Magasins

Si vous voyez des déclins fréquents :
- Vérifiez que votre compte marchand est en bon état
- Vérifiez la configuration de la passerelle de paiement
- Révisez les paramètres de prévention de la fraude
- Envisagez de contacter votre processeur de paiement
