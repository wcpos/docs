# PY01001 : Paiement refusé

## Ce que cela signifie

Le paiement a été refusé par le processeur de paiement ou l'émetteur de la carte. La transaction n'a pas pu être effectuée.

## Causes courantes

- **Fonds insuffisants** — Pas assez d'argent sur le compte
- **Limite de carte dépassée** — La transaction dépasse les limites de la carte
- **Protection contre la fraude** — Banque a signalé comme suspect
- **Détails de carte incorrects** — Numéro de carte, CVV ou date d'expiration erronés
- **Restrictions de carte** — Carte non activée pour ce type d'achat

## Comment résoudre

### 1. Vérifier les détails de la carte

Vérifiez que les informations saisies sont correctes :
- Numéro de carte (tous les chiffres)
- Date d'expiration
- CVV/code de sécurité
- Adresse de facturation (si nécessaire)

### 2. Réessayer

Des problèmes temporaires peuvent survenir :
- Attendre un moment
- Ressaisir les détails de la carte
- Réessayer la transaction

### 3. Utiliser un autre moyen de paiement

Si la carte continue d'être refusée :
- Essayer une autre carte
- Utiliser un paiement en espèces
- Utiliser un mode de paiement alternatif

### 4. Contacter la Banque

Le titulaire de la carte peut devoir :
- Vérifier la transaction avec sa banque
- Lever les blocages de fraude
- Confirmer que la carte est active
- Vérifier les limites de dépenses

### 5. Vérifier les restrictions

Certaines cartes ont des restrictions :
- Transactions internationales désactivées
- Transactions en ligne/TPV bloquées
- Certaines catégories de commerçants bloquées

## Pour les propriétaires de magasin

Si vous voyez des refus fréquents :
- Vérifier que votre compte de commerçant est en bon état
- Vérifier la configuration de la passerelle de paiement
- Examiner les paramètres de prévention des fraudes
- Envisager de contacter votre processeur de paiement
