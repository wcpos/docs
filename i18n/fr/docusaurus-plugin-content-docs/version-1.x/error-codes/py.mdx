# Erreurs de Paiement

Les erreurs de paiement se produisent pendant le processus de paiement et de traitement des paiements. Ces erreurs sont préfixées par `PY` et concernent des problèmes liés aux cartes de paiement, aux passerelles et aux transactions.

## Catégories

| Catégorie | Plage de Codes | Description |
|-----------|----------------|-------------|
| [Carte](#erreurs-de-carte) | PY01xxx | Problèmes avec la carte de paiement |
| [Passerelle](#erreurs-de-passerelle) | PY02xxx | Problèmes de communication avec la passerelle de paiement |

---

## Erreurs de Carte

Problèmes liés à la carte de paiement du client.

| Code | Nom | Description |
|------|-----|-------------|
| [PY01001](./PY01001) | Paiement Refusé | Le paiement a été refusé |
| [PY01002](./PY01002) | Fonds Insuffisants | Pas assez de fonds pour le paiement |
| [PY01003](./PY01003) | Carte Expirée | La carte de paiement a expiré |
| [PY01004](./PY01004) | Numéro de Carte Invalide | Le numéro de carte n'est pas valide |

## Erreurs de Passerelle

Problèmes de communication avec la passerelle de paiement.

| Code | Nom | Description |
|------|-----|-------------|
| [PY02001](./PY02001) | Erreur de Passerelle de Paiement | Erreur de communication avec la passerelle de paiement |
| [PY02002](./PY02002) | Délai d'Attente de Paiement | Le traitement du paiement a expiré |
