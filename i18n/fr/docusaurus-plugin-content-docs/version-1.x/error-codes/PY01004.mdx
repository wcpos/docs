# PY01004 : Numéro de carte invalide

## Ce que cela signifie

Le numéro de carte saisi n'est pas valide. Les numéros de carte suivent des modèles spécifiques et incluent des chiffres de contrôle pour la validation.

## Causes courantes

- **Fausse frappe** — Un ou plusieurs chiffres saisis incorrectement
- **Chiffres manquants** — Tous les chiffres n'ont pas été saisis
- **Chiffres en trop** — Trop de chiffres saisis
- **Mauvaise carte** — Numéro de carte différent de la carte physique
- **Erreur du lecteur de carte** — La puce/le passage a lu les mauvaises données

## Comment résoudre

### 1. Saisir à nouveau le numéro de carte

Saisissez soigneusement tous les chiffres :
- Vérifiez la carte physique
- Saisissez chaque chiffre lentement
- Vérifiez le numéro complet avant de soumettre

### 2. Vérifiez le type de carte

Assurez-vous que le type de carte est pris en charge :
- Visa (commence par 4)
- Mastercard (commence par 51-55 ou 22-27)
- Amex (commence par 34 ou 37)
- Discover (commence par 6011, 622, 644-649, 65)

### 3. Réessayez avec le lecteur de carte

Si vous utilisez un lecteur de puce/passage :
- Nettoyez la puce de la carte
- Essayez une méthode de lecture différente (puce contre passage)
- Saisissez manuellement si le lecteur échoue

### 4. Utilisez une carte différente

Si le numéro de carte continue à échouer :
- La carte peut être endommagée
- Essayez une autre carte
- Utilisez un mode de paiement alternatif

### 5. Vérifiez les dommages sur la carte

Problèmes physiques de la carte :
- Bande magnétique rayée
- Puce endommagée
- Chiffres usés ou effacés
- La carte peut nécessiter un remplacement

## Formats de numéro de carte

| Type de carte | Longueur | Commence par |
|---------------|----------|--------------|
| Visa          | 16       | 4            |
| Mastercard    | 16       | 51-55, 22-27 |
| Amex          | 15       | 34, 37       |
| Discover      | 16       | 6011, 622, 644-649, 65 |
