# PY01004 : Numéro de Carte Invalide

## Ce que Cela Signifie

Le numéro de carte saisi n'est pas valide. Les numéros de carte suivent des modèles spécifiques et incluent des chiffres de contrôle pour validation.

## Causes Courantes

- **Faute de frappe** — Un ou plusieurs chiffres saisis incorrectement
- **Chiffres manquants** — Tous les chiffres n'ont pas été saisis
- **Chiffres supplémentaires** — Trop de chiffres saisis
- **Mauvaise carte** — Numéro de carte différent de la carte physique
- **Erreur du lecteur de carte** — Le chip/glissement a lu des données incorrectes

## Comment Résoudre

### 1. Ressaisir le Numéro de Carte

Saisissez soigneusement tous les chiffres :
- Vérifiez la carte physique
- Saisissez chaque chiffre lentement
- Vérifiez le numéro complet avant de soumettre

### 2. Vérifier le Type de Carte

Assurez-vous que le type de carte est pris en charge :
- Visa (commence par 4)
- Mastercard (commence par 51-55 ou 22-27)
- Amex (commence par 34 ou 37)
- Discover (commence par 6011, 622, 644-649, 65)

### 3. Réessayer avec le Lecteur de Carte

Si vous utilisez un lecteur chip/glissement :
- Nettoyer le chip de la carte
- Essayer une autre méthode de lecture (chip vs glissement)
- Saisir manuellement si le lecteur échoue

### 4. Utiliser une Autre Carte

Si le numéro de carte continue d'échouer :
- La carte peut être endommagée
- Essayer une autre carte
- Utiliser un mode de paiement alternatif

### 5. Vérifier les Dommages sur la Carte

Problèmes physiques de la carte :
- Bande magnétique rayée
- Chip endommagé
- Numéros usés ou effacés
- La carte peut nécessiter un remplacement

## Formats de Numéro de Carte

| Type de Carte | Longueur | Commence Par |
|--------------|----------|--------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55, 22-27 |
| Amex | 15 | 34, 37 |
| Discover | 16 | 6011, 622, 644-649, 65 |
