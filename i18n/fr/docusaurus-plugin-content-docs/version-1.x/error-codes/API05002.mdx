# API05002 : Plugin WCPOS Non Trouvé

## Ce que Cela Signifie

Le plugin WCPOS (WooCommerce POS) n'est pas installé ou n'est pas actif sur votre site WordPress. L'application POS nécessite ce plugin pour communiquer avec WooCommerce.

## Causes Courantes

- **Plugin non installé** — Le plugin n'a jamais été installé
- **Plugin désactivé** — Le plugin a été désactivé
- **Plugin supprimé** — Les fichiers du plugin ont été supprimés
- **Mauvais site** — Connecté à un site sans WCPOS

## Comment Résoudre

### 1. Installer le Plugin

Si non installé :
1. Allez dans Administration WordPress → Plugins → Ajouter
2. Recherchez "WooCommerce POS"
3. Installez et activez le plugin

Ou téléchargez depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Activer le Plugin

Si installé mais non actif :
1. Allez dans Plugins → Plugins installés
2. Trouvez "WooCommerce POS"
3. Cliquez sur "Activer"

### 3. Vérifier les Erreurs

Si le plugin ne s'active pas :
- Vérifiez les erreurs PHP dans votre journal d'erreurs
- Assurez-vous que WooCommerce est installé et actif
- Vérifiez la version requise de PHP
- Recherchez des conflits de plugins

### 4. Vérifier l'URL

Assurez-vous que le POS est configuré pour se connecter au bon site WordPress :
- Vérifiez l'URL dans les paramètres du POS
- Confirmez qu'elle correspond à votre installation WordPress

### 5. Réinstaller si Nécessaire

Si le plugin est corrompu :
1. Désactivez et supprimez le plugin
2. Réinstallez depuis WordPress.org
3. Activez le plugin

## Prérequis

WCPOS nécessite :
- WordPress (version récente)
- WooCommerce (version récente)
- PHP 7.4 ou supérieur
