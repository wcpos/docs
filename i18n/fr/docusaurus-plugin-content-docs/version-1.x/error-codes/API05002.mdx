# API05002 : plugin WCPOS introuvable {#api05002-wcpos-plugin-not-found}

## Signification {#what-this-means}

Le plugin WCPOS n'est pas installé ou n'est pas actif sur votre site WordPress. L'application PDV a besoin de ce plugin pour communiquer avec WooCommerce.

## Causes fréquentes {#common-causes}

- **Plugin non installé** — Le plugin n'a jamais été installé
- **Plugin désactivé** — Le plugin a été désactivé
- **Plugin supprimé** — Les fichiers du plugin ont été supprimés
- **Mauvais site** — Connexion à un site sans WCPOS

## Comment résoudre le problème {#how-to-fix}

### 1. Installer le plugin {#1-install-the-plugin}

Si elle n'est pas installée :
1. Allez dans Administration WordPress → Extensions → Ajouter
2. Recherchez « WCPOS »
3. Installez et activez l'extension

Ou téléchargez-la depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Activer l'extension {#2-activate-the-plugin}

Si elle est installée mais inactive :
1. Allez dans Extensions → Extensions installées
2. Trouvez « WCPOS »
3. Cliquez sur « Activer »

### 3. Vérifier les erreurs {#3-check-for-errors}

Si l'extension ne s'active pas :
- Vérifiez les erreurs PHP dans votre journal d'erreurs
- Vérifiez que WooCommerce est installé et actif
- Vérifiez les exigences de version de PHP
- Recherchez les conflits d'extensions

### 4. Vérifiez l'URL {#4-verify-the-url}

Assurez-vous que le PDV est configuré pour se connecter au bon site WordPress :
- Vérifiez l'URL dans les réglages du PDV
- Vérifiez qu'elle correspond à votre installation WordPress

### 5. Réinstallez si nécessaire {#5-reinstall-if-necessary}

Si l'extension est corrompue :
1. Désactivez et supprimez l'extension
2. Réinstallez depuis WordPress.org
3. Activez l'extension
