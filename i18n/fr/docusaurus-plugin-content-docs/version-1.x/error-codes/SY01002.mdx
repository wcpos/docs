# SY01002: Disque Plein

## Ce Que Cela Signifie

Le stockage de votre appareil est plein. Le POS ne peut pas enregistrer de données ni continuer à fonctionner sans espace disque disponible.

## Causes Courantes

- **Stockage plein** — Aucun espace libre sur le lecteur
- **Grande base de données locale** — Données du POS utilisant un espace significatif
- **Fichiers temporaires** — Fichiers temporaires consommant de l'espace
- **Autres applications** — D'autres applications remplissant le disque

## Comment Résoudre

### 1. Libérer de l'Espace Disque

Supprimez les fichiers inutiles :
- Videz la corbeille
- Supprimez les anciens téléchargements
- Supprimez les applications inutilisées
- Effacez les fichiers temporaires

### 2. Vérifier l'Espace Disponible

**Windows :**
- Ouvrez l'Explorateur de fichiers
- Faites un clic droit sur le lecteur → Propriétés
- Consultez l'espace utilisé et libre

**macOS :**
- Menu Apple → À propos de ce Mac
- Cliquez sur Stockage

### 3. Effacer les Données du Navigateur (Version Web)

Les navigateurs stockent des données localement :
- Effacez le cache et les cookies
- Supprimez les anciennes données de site
- Vérifiez l'utilisation du stockage du navigateur

### 4. Déplacer ou Supprimer de Gros Fichiers

Trouvez et gérez les gros fichiers :
- Anciennes vidéos ou photos
- Gros téléchargements
- Logiciels inutilisés
- Fichiers de sauvegarde pouvant être déplacés vers un stockage externe

### 5. Vérifier les Données du POS

Si les données du POS sont très volumineuses :
- La base de données locale peut nécessiter un nettoyage
- Envisagez d'effacer et de re-synchroniser
- Vérifiez si les anciennes données peuvent être purgées

## Espace Libre Recommandé

Gardez au moins :
- **10 Go** libres pour un fonctionnement normal
- **20 Go+** pour de grands catalogues
- Plus d'espace est toujours mieux

## Prévenir de Futurs Problèmes

- Nettoyez régulièrement les fichiers inutiles
- Mettez en place un nettoyage automatique des fichiers temporaires
- Surveillez l'utilisation de l'espace disque
- Envisagez un stockage plus grand si souvent plein
