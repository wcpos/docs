# Erreurs de l'API

Les erreurs de l'API se produisent lors de la communication avec votre serveur WooCommerce. Ces erreurs sont préfixées par `API` et sont organisées en catégories suivantes :

## Catégories

| Catégorie | Plage de codes | Description |
|-----------|----------------|-------------|
| [Connexion](#connection-errors) | API01xxx | Problèmes de réseau et de connectivité |
| [Authentification](#authentication-errors) | API02xxx | Problèmes de connexion, de jetons et de permissions |
| [Demande](#request-errors) | API03xxx | Problèmes avec les demandes sortantes |
| [Réponse](#response-errors) | API04xxx | Problèmes avec les réponses du serveur |
| [Plugin/WordPress](#pluginwordpress-errors) | API05xxx | Problèmes avec les plugins WordPress ou WooCommerce |
| [Configuration](#configuration-errors) | API06xxx | Problèmes de configuration et de configuration |

---

## Erreurs de connexion

Problèmes de réseau et de connectivité entre le POS et votre serveur.

| Code | Nom | Description |
|------|-----|-------------|
| [API01001](./API01001) | Délai de connexion | Le serveur a mis trop de temps à répondre |
| [API01002](./API01002) | Connexion refusée | Le serveur a refusé la connexion |
| [API01003](./API01003) | Réinitialisation de connexion | La connexion a été fermée de manière inattendue |
| [API01004](./API01004) | Échec de la résolution DNS | Impossible de résoudre l'adresse du serveur |
| [API01005](./API01005) | Erreur de certificat SSL | Problème avec le certificat de sécurité du site |
| [API01006](./API01006) | Réseau inaccessible | Impossible d'atteindre le réseau |
| [API01007](./API01007) | Appareil hors ligne | Votre appareil n'est pas connecté à Internet |
| [API01008](./API01008) | Site Web indisponible | Le site Web ne répond pas |

## Erreurs d'authentification

Problèmes de connexion, de sessions et de permissions.

| Code | Nom | Description |
|------|-----|-------------|
| [API02001](./API02001) | Informations d'identification invalides | Nom d'utilisateur ou mot de passe incorrect |
| [API02002](./API02002) | Jeton expiré | Votre session a expiré |
| [API02003](./API02003) | Jeton invalide | Le jeton d'authentification n'est pas valide |
| [API02004](./API02004) | Utilisateur non autorisé | Vous n'avez pas la permission d'effectuer cette action |
| [API02005](./API02005) | Permissions insuffisantes | Votre rôle d'utilisateur n'a pas les permissions requises |
| [API02006](./API02006) | Clé API invalide | La clé API WooCommerce n'est pas valide |
| [API02007](./API02007) | Échec du rafraîchissement du jeton | Impossible de rafraîchir votre session |
| [API02008](./API02008) | Jeton de rafraîchissement invalide | Le jeton de rafraîchissement n'est pas valide |
| [API02009](./API02009) | Jeton de rafraîchissement expiré | Le jeton de rafraîchissement a expiré |
| [API02010](./API02010) | Authentification requise | L'authentification est requise pour cette action |
