# Erreurs API

Les erreurs API se produisent lors de la communication avec votre serveur WooCommerce. Ces erreurs sont précédées du préfixe `API` et sont organisées dans les catégories suivantes :

## Catégories

| Catégorie | Plage de codes | Description |
|----------|------------|-------------|
| [Connexion](#erreurs-de-connexion) | API01xxx | Problèmes de réseau et de connectivité |
| [Authentification](#erreurs-dauthentification) | API02xxx | Problèmes de connexion, jetons et permissions |
| [Requête](#erreurs-de-requête) | API03xxx | Problèmes avec les requêtes sortantes |
| [Réponse](#erreurs-de-réponse) | API04xxx | Problèmes avec les réponses du serveur |
| [Plugin/WordPress](#erreurs-de-pluginwordpress) | API05xxx | Problèmes avec WordPress ou les plugins WooCommerce |
| [Configuration](#erreurs-de-configuration) | API06xxx | Problèmes de configuration et de configuration |

---

## Erreurs de Connexion

Problèmes de réseau et de connectivité entre le POS et votre serveur.

| Code | Nom | Description |
|------|------|-------------|
| [API01001](./API01001) | Délai de Connexion Dépassé | Le serveur a mis trop de temps à répondre |
| [API01002](./API01002) | Connexion Refusée | Le serveur a refusé la connexion |
| [API01003](./API01003) | Réinitialisation de Connexion | La connexion a été fermée de manière inattendue |
| [API01004](./API01004) | Échec de Résolution DNS | Impossible de résoudre l'adresse du serveur |
| [API01005](./API01005) | Erreur de Certificat SSL | Problème avec le certificat de sécurité du site |
| [API01006](./API01006) | Réseau Injoignable | Impossible d'atteindre le réseau |
| [API01007](./API01007) | Appareil Hors Ligne | Votre appareil n'est pas connecté à Internet |
| [API01008](./API01008) | Site Web Non Disponible | Le site web ne répond pas |

## Erreurs d'Authentification

Problèmes de connexion, sessions et permissions.

| Code | Nom | Description |
|------|------|-------------|
| [API02001](./API02001) | Identifiants Invalides | Nom d'utilisateur ou mot de passe incorrect |
| [API02002](./API02002) | Jeton Expiré | Votre session a expiré |
| [API02003](./API02003) | Jeton Invalide | Le jeton d'authentification n'est pas valide |
| [API02004](./API02004) | Utilisateur Non Autorisé | Vous n'avez pas la permission d'effectuer cette action |
| [API02005](./API02005) | Permissions Insuffisantes | Votre rôle utilisateur manque des permissions requises |
| [API02006](./API02006) | Clé API Invalide | La clé API WooCommerce n'est pas valide |
| [API02007](./API02007) | Actualisation du Jeton Échouée | Impossible d'actualiser votre session |
| [API02008](./API02008) | Jeton d'Actualisation Invalide | Le jeton d'actualisation n'est pas valide |
| [API02009](./API02009) | Jeton d'Actualisation Expiré | Le jeton d'actualisation a expiré |
| [API02010](./API02010) | Authentification Requise | Une authentification est requise pour cette action |

| Code | Nom | Description |
|------|------|-------------|
