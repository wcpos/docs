# SY02002 : Service Indisponible {#sy02002-service-unavailable}

## Ce Que Cela Signifie {#what-this-means}

Un service ou composant requis n'est pas disponible. Le POS dépend de divers services pour fonctionner, et l'un d'eux est actuellement inaccessible.

## Causes Courantes {#common-causes}

- **Service en arrière-plan arrêté** — Un service requis n'est pas en cours d'exécution
- **Problème de ressources système** — Pas assez de ressources pour démarrer le service
- **Échec de dépendance** — Un service dont cela dépend a échoué
- **Problème de démarrage** — Le service a échoué à démarrer correctement
- **Erreur de serveur** — Le serveur WooCommerce a renvoyé une erreur 5xx

## Cartographie des Erreurs Serveur {#server-error-mapping}

Ce code d'erreur est déclenché lorsque le serveur renvoie :

| Code Serveur | Source |
|--------------|--------|
| HTTP 5xx | Toute réponse d'erreur serveur (500, 502, 503, 504, etc.) |

## Comment Réparer {#how-to-fix}

### 1. Redémarrer l'Application {#1-restart-the-application}

Fermez et rouvrez le POS :
1. Fermez complètement l'application
2. Attendez quelques secondes
3. Redémarrez-la

### 2. Redémarrer Votre Appareil {#2-restart-your-device}

Un redémarrage complet résout souvent les problèmes de service :
1. Enregistrez tout travail important
2. Redémarrez votre ordinateur
3. Essayez à nouveau le POS

### 3. Vérifiez les Ressources Système {#3-check-system-resources}

Assurez-vous que les ressources système sont disponibles :
- Fermez les applications inutiles
- Vérifiez l'utilisation de la mémoire
- Vérifiez l'espace disque

### 4. Mettre à Jour l'Application {#4-update-the-application}

Assurez-vous d'utiliser la dernière version :
- Vérifiez les mises à jour disponibles
- Installez toutes les mises à jour en attente
- Redémarrez après la mise à jour

### 5. Réinstaller si Nécessaire {#5-reinstall-if-necessary}

Si le service reste indisponible :
1. Désinstallez l'application
2. Redémarrez votre appareil
3. Téléchargez la dernière version
4. Installez-la à neuf

## Spécifique au Navigateur (Version Web) {#browser-specific-web-version}

Pour la version web :
- Assurez-vous que JavaScript est activé
- Vérifiez que les fonctionnalités requises du navigateur ne sont pas bloquées
- Essayez un navigateur différent
- Désactivez les extensions de navigateur qui pourraient interférer

## Quand Contacter le Support {#when-to-contact-support}

Si cette erreur persiste :
- Notez quand elle se produit
- Vérifiez si des actions spécifiques la déclenchent
- Signalez avec des détails pour enquête
