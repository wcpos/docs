# SY02002 : Service non disponible

## Ce que cela signifie

Un service ou un composant requis n'est pas disponible. Le POS dépend de divers services pour fonctionner, et l'un d'entre eux est actuellement inaccessible.

## Causes courantes

- **Service en arrière-plan arrêté** — Un service requis n'est pas en cours d'exécution
- **Problème de ressources système** — Pas assez de ressources pour démarrer le service
- **Échec de dépendance** — Un service dont il dépend a échoué
- **Problème de démarrage** — Le service n'a pas démarré correctement
- **Erreur de serveur** — Le serveur WooCommerce a renvoyé une erreur 5xx

## Mappage des erreurs de serveur

Ce code d'erreur est déclenché lorsque le serveur renvoie :

| Code de serveur | Source |
|----------------|--------|
| HTTP 5xx | Toute réponse d'erreur de serveur (500, 502, 503, 504, etc.) |

## Comment résoudre

### 1. Redémarrer l'application

Fermez et rouvrez le POS :
1. Fermez complètement l'application
2. Attendez quelques secondes
3. Redémarrez-la

### 2. Redémarrer votre appareil

Un redémarrage complet résout souvent les problèmes de service :
1. Sauvegardez tout travail important
2. Redémarrez votre ordinateur
3. Réessayez le POS

### 3. Vérifier les ressources système

Assurez-vous que les ressources système sont disponibles :
- Fermez les applications inutiles
- Vérifiez l'utilisation de la mémoire
- Vérifiez l'espace disque

### 4. Mettre à jour l'application

Assurez-vous d'être sur la dernière version :
- Vérifiez les mises à jour disponibles
- Installez toutes les mises à jour en attente
- Redémarrez après la mise à jour

### 5. Réinstaller si nécessaire

Si le service reste indisponible :
1. Désinstallez l'application
2. Redémarrez votre appareil
3. Téléchargez la dernière version
4. Installez à nouveau

## Spécifique au navigateur (Version web)

Pour la version web :
- Assurez-vous que JavaScript est activé
- Vérifiez que les fonctionnalités de navigateur requises ne sont pas bloquées
- Essayez un autre navigateur
- Désactivez les extensions de navigateur qui pourraient interférer

## Quand contacter le support

Si cette erreur persiste :
- Notez quand elle se produit
- Vérifiez si des actions spécifiques la déclenchent
- Signalez avec des détails pour investigation
