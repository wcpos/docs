# API02008 : Jeton de Rafraîchissement Invalide

## Ce Que Cela Signifie

Le jeton de rafraîchissement stocké par le POS n'est pas reconnu par le serveur. Les jetons de rafraîchissement sont utilisés pour obtenir de nouveaux jetons d'accès sans avoir à vous reconnecter.

## Causes Courantes

- **Jeton révoqué** — Un administrateur a invalidé votre session
- **Modifications des secrets du serveur** — Les clés de sécurité WordPress ont été mises à jour
- **Corruption de données** — Le jeton a été corrompu lors du stockage
- **Connexions multiples** — Une connexion ailleurs peut avoir invalidé ce jeton

## Comment Résoudre

### 1. Se Reconnecter

Comme le jeton de rafraîchissement est invalide, vous devrez vous authentifier à nouveau :
1. Déconnectez-vous du POS
2. Entrez votre nom d'utilisateur et votre mot de passe
3. Cela générera de nouveaux jetons valides

### 2. Vérifier les Sessions Révoquées

Si un administrateur a révoqué des sessions :
- C'est une pratique de sécurité normale
- Reconnectez-vous simplement
- Votre nouvelle session fonctionnera normalement

### 3. Vérifier les Modifications du Serveur

Si cela affecte tous les utilisateurs :
- Les clés/salts WordPress peuvent avoir changé
- Les paramètres du plugin JWT peuvent avoir été modifiés
- Contactez l'administrateur de votre serveur

### 4. Effacer les Données Stockées

Si vous continuez à rencontrer des problèmes :
- Effacez les données de l'application POS
- Supprimez les identifiants mis en cache
- Recommencez avec une connexion vierge
