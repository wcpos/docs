# API02008 : Jeton de rafraîchissement invalide

## Ce que cela signifie

Le jeton de rafraîchissement stocké par le POS n'est pas reconnu par le serveur. Les jetons de rafraîchissement sont utilisés pour obtenir de nouveaux jetons d'accès sans vous obliger à vous reconnecter.

## Causes courantes

- **Jeton révoqué** — Un administrateur a invalidé votre session
- **Secrets du serveur modifiés** — Les clés de sécurité WordPress ont été mises à jour
- **Corruption des données** — Le jeton a été corrompu dans le stockage
- **Connexions multiples** — Se connecter ailleurs peut avoir invalidé ce jeton

## Comment résoudre

### 1. Reconnectez-vous

Puisque le jeton de rafraîchissement est invalide, vous devrez vous authentifier à nouveau :
1. Déconnectez-vous du POS
2. Entrez votre nom d'utilisateur et votre mot de passe
3. Cela génère de nouveaux jetons valides

### 2. Vérifiez les sessions révoquées

Si un admin a révoqué des sessions :
- C'est une pratique de sécurité normale
- Il suffit de se reconnecter
- Votre nouvelle session fonctionnera normalement

### 3. Vérifiez les modifications du serveur

Si cela affecte tous les utilisateurs :
- Les sels/clés WordPress peuvent avoir changé
- Les paramètres du plugin JWT ont pu être modifiés
- Contactez votre administrateur de serveur

### 4. Effacez les données stockées

Si vous continuez à avoir des problèmes :
- Effacez les données de l'application POS
- Supprimez les identifiants mis en cache
- Commencez avec une connexion propre
