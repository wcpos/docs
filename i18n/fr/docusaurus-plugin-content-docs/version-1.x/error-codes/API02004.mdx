# API02004 : Utilisateur non autorisé

## Ce que cela signifie

Vous êtes connecté, mais votre compte utilisateur n'a pas la permission d'effectuer l'action demandée. Il s'agit d'un problème d'autorisation (ce que vous pouvez faire) plutôt que d'un problème d'authentification (qui vous êtes).

## Causes courantes

- **Limitations du rôle utilisateur** — Votre rôle WordPress n'inclut pas l'accès POS
- **Accès POS désactivé** — Votre compte ne s'est pas vu accorder les autorisations POS
- **Restrictions de fonctionnalités** — Certaines fonctionnalités sont limitées à des rôles spécifiques
- **Restrictions de magasin** — Vous pouvez ne pas avoir accès à ce magasin particulier

## Mappage des erreurs serveur

Ce code d'erreur est déclenché lorsque le serveur renvoie :

| Code serveur | Source |
|-------------|--------|
| `rest_cannot_view` | API REST WordPress |
| `woocommerce_rest_cannot_view` | API REST WooCommerce |

## Comment résoudre

### 1. Vérifier le rôle utilisateur

Dans l'administration WordPress → Utilisateurs :
1. Trouvez votre compte utilisateur
2. Vérifiez le rôle (par exemple, Gestionnaire de boutique, Administrateur)
3. Assurez-vous que le rôle inclut les capacités WooCommerce

### 2. Activer l'accès POS

Dans l'administration WordPress → WooCommerce → POS → Accès :
1. Trouvez l'utilisateur ou le rôle
2. Activez les autorisations d'accès POS
3. Enregistrez les modifications

### 3. Demander des autorisations supplémentaires

Contactez l'administrateur de votre magasin pour :
- Accorder l'accès POS à votre rôle
- Vous attribuer un rôle avec les autorisations appropriées
- Activer les fonctionnalités spécifiques dont vous avez besoin

### 4. Vérifier l'attribution du magasin

Si vous utilisez plusieurs magasins :
- Vérifiez que vous êtes assigné au bon magasin
- Vérifiez les autorisations spécifiques au magasin

## Autorisations requises

Différentes actions nécessitent différentes capacités :
- **Voir les produits** : Accès en lecture aux produits
- **Créer des commandes** : Capacités de création/modification de commandes
- **Gérer les clients** : Capacités de gestion des clients
- **Accéder aux rapports** : Capacités de visualisation des rapports
