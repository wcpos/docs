L'écran POS est l'interface principale pour vendre des produits. C'est là où vous passerez la plupart de votre temps lors de l'utilisation de WCPOS.

{/* Screenshot: pos-desktop-layout.png */}

## Disposition de l'écran

### En-tête

La barre d'en-tête affiche :

- **Nom du magasin** - Le nom de votre boutique WooCommerce connectée
- **Indicateur de connectivité** - Point vert lorsque connecté au serveur, jaune/rouge hors ligne
- **Menu utilisateur** - Cliquez pour accéder aux paramètres, changer d'utilisateur ou se déconnecter

### Tiroir de navigation

La barre latérale gauche permet un accès rapide à tous les écrans principaux :

| Icône | Écran | Description |
|------|--------|-------------|
| <Icon name="cash-register" /> | **POS** | Interface principale de vente (vous êtes ici) |
| <Icon name="gift" /> | **Produits** | Gestion des stocks (Pro) |
| <Icon name="receipt" /> | **Commandes** | Historique des commandes (Pro) |
| <Icon name="users" /> | **Clients** | Gestion des clients (Pro) |
| <Icon name="chart-line" /> | **Rapports** | Rapports de ventes (Pro) |
| <Icon name="heart-pulse" /> | **Journaux** | Journaux système pour le débogage |
| <Icon name="circle-question" /> | **Support** | Canal de support Discord |

Le numéro de version est affiché en bas du tiroir.

## Disposition responsive

WCPOS s'adapte à différentes tailles d'écran :

### Bureau / Tablette (Grands écrans)

Sur les grands écrans, le POS affiche une **disposition à deux colonnes** :

- **Colonne de gauche :** Panneau Produits - rechercher et parcourir les produits
- **Colonne de droite :** Panneau Panier - visualiser et gérer la commande en cours

Les colonnes sont **redimensionnables** - faites glisser le séparateur entre elles pour ajuster les proportions.

{/* Screenshot: pos-desktop-layout.png */}

### Mobile (Petits écrans)

Sur les petits écrans, le POS utilise une **disposition à onglets** :

- **Onglet Produits :** Parcourir et ajouter des produits au panier
- **Onglet Panier :** Visualiser le panier, gérer les articles et passer en caisse

Basculez entre les onglets à l'aide de la barre de navigation en bas de l'écran.

{/* Screenshot: pos-mobile-layout.png */}

## Composants principaux

### Panneau Produits

Le côté gauche du POS est le [Panneau Produits](/pos/product-panel), où vous pouvez :

- Rechercher des produits par nom, SKU ou code-barres
- Filtrer les produits par statut de stock, catégorie, étiquettes, etc.
- Ajouter des produits au panier

[En savoir plus sur le Panneau Produits →](/pos/product-panel)

### Panneau Panier

Le côté droit du POS est le [Panneau Panier](/pos/cart), où vous pouvez :

- Voir les articles de la commande en cours
- Modifier les quantités et les prix
- Sélectionner ou ajouter des clients
- Accéder aux actions de commande (notes, annulation, paiement)

[En savoir plus sur le Panneau Panier →](/pos/cart)

### Paiement

Lorsque vous êtes prêt à finaliser une vente, cliquez sur **Paiement** pour ouvrir la [Fenêtre de paiement](/pos/checkout) :

- Appliquer des codes de réduction
- Sélectionner le mode de paiement
- Traiter le paiement
- Imprimer ou envoyer le reçu par e-mail

[En savoir plus sur le Paiement →](/pos/checkout)

## Raccourcis clavier

WCPOS prend en charge les raccourcis clavier pour un fonctionnement plus rapide. Accédez à la liste complète via la fenêtre de Paramètres ou consultez [Raccourcis clavier](/settings/store/hotkeys).

Raccourcis courants :

| Raccourci | Action |
|----------|--------|
| `Ctrl + F` | Mettre le focus sur la barre de recherche |
| `Ctrl + Shift + S` | Ouvrir les Paramètres |
| `Échap` | Fermer la fenêtre/le dialogue |

## Connectivité

L'indicateur vert dans l'en-tête affiche votre statut de connexion :

- **Vert** - Connecté au serveur WooCommerce
- **Jaune** - Problèmes de connexion, nouvelle tentative
- **Rouge** - Mode hors ligne

En mode hors ligne, vous pouvez toujours parcourir les produits et clients mis en cache, et commencer de nouvelles commandes. La finalisation des commandes nécessite une connexion.
