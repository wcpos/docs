Les produits variables dans WooCommerce sont des produits qui proposent plusieurs options, comme différentes tailles, couleurs ou matières. WCPOS propose plusieurs façons de travailler efficacement avec les produits variables.

## Identifier les produits variables {#identifying-variable-products}

Dans le panneau des produits, les produits variables se distinguent par :

- **Bouton flèche** (<Icon name="circle-arrow-right" color="green" />) au lieu du bouton plus (<Icon name="circle-plus" color="green" />) utilisé pour les produits simples
- **Options d’attribut** affichées sous le nom du produit (par exemple, "Couleur : Bleu, Vert, Rouge")
- **Lien "Développer"** pour afficher toutes les variations en ligne

## Ajouter des produits variables au panier {#adding-variable-products-to-cart}

### Méthode 1 : fenêtre contextuelle rapide {#method-1-quick-popover}

Cliquez sur le <Icon name="circle-arrow-right" color="green" /> bouton fléché vert d’un produit variable pour ouvrir une fenêtre contextuelle avec des menus déroulants pour chaque attribut :

1. Cliquez sur le <Icon name="circle-arrow-right" color="green" /> bouton
2. Sélectionnez des options dans chaque menu déroulant (par exemple, Taille : Grand, Couleur : Bleu)
3. La variation correspondante est ajoutée au panier

C’est la méthode la plus rapide lorsque vous savez exactement quelle variation il vous faut.

{/* Screenshot: variation-popover.png */}

### Méthode 2 : développer en ligne {#method-2-expand-inline}

Cliquez sur le lien **"Développer"** d’un produit variable pour afficher toutes ses variations directement dans la liste des produits :

1. Cliquez sur **Développer** sous le nom du produit
2. Toutes les variations apparaissent sur des lignes distinctes sous le produit parent
3. Cliquez sur le <Icon name="circle-plus" color="green" /> bouton sur n’importe quelle variation pour l’ajouter au panier
4. Cliquez sur **Réduire** pour masquer à nouveau les variations

Cette méthode est utile lorsque vous souhaitez voir toutes les options disponibles, leurs niveaux de stock individuels et leurs prix.

{/* Screenshot: variations-expanded.png */}

### Méthode 3 : scan de code-barres {#method-3-barcode-scanning}

Chaque variation de produit peut avoir son propre code-barres unique. Lors du scan du code-barres d’une variation :

- Si une seule variation correspond exactement, elle est ajoutée directement au panier
- Si plusieurs variations partagent le même code-barres, la barre de recherche affiche les résultats correspondants

Consultez [Scan de code-barres](/pos/product-panel/barcode-scanning) pour plus de détails.

## Informations sur les variations {#variation-information}

Chaque variation peut afficher :

- **Attributs de variation** - Les options spécifiques à cette variation (par exemple, "Taille : Grand, Couleur : Bleu")
- **SKU** - Identifiant unique de la variation (peut différer de celui du produit parent)
- **Quantité en stock** - Niveau de stock individuel de la variation
- **Prix** - Les variations peuvent avoir des prix différents de ceux du produit parent ou des autres variations

## Variations dans le panier {#variations-in-the-cart}

Lorsqu’une variation est ajoutée au panier, elle affiche :

- **Nom du produit** provenant du produit parent
- **Attributs de variation** sous le nom
- **Prix** de cette variation spécifique

Vous pouvez modifier la ligne du panier comme pour n’importe quel autre produit du panier.

## Conseils pour sélectionner efficacement les variations {#tips-for-efficient-variation-selection}

### Utiliser la lecture de codes-barres {#use-barcode-scanning}

Si vos variations disposent de codes-barres individuels (recommandé), les scanner est le moyen le plus rapide de les ajouter au panier.

### Configurer les options d’affichage {#configure-display-options}

Dans les [paramètres d’affichage](/pos/product-panel#display-settings) du panneau des produits, activez l’option **Attributs** pour afficher les attributs des variations directement dans la liste des produits, sans la développer.

### Navigation au clavier {#keyboard-navigation}

Lorsque la fenêtre contextuelle des variations est ouverte, vous pouvez utiliser les touches fléchées du clavier pour parcourir les options et Entrée pour sélectionner.

## FAQ {#faq}

<Accordion>
  <AccordionItem question="Pourquoi une variation spécifique n’apparaît-elle pas ?">

Vérifiez que la variation :
- Est publiée et n’est pas à l’état de brouillon
- Est définie sur "En stock" ou possède une quantité en stock > 0 (si la gestion du stock est activée)
- Tous les attributs requis sont définis

Vérifiez également que vous n’avez pas exclu le produit avec le filtre "En stock" si la variation est en rupture de stock.

  </AccordionItem>

  <AccordionItem question="Comment ajouter des codes-barres à des variations individuelles ?">

Dans WooCommerce, modifiez le produit variable et développez chaque variation. Vous pouvez ajouter un code-barres unique dans le champ de code-barres de chaque variation (généralement `_sku` ou `_global_unique_id` selon votre configuration).

  </AccordionItem>
</Accordion>
