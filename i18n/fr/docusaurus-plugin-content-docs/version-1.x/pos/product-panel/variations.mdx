Les produits variables dans WooCommerce sont des produits qui ont plusieurs options, telles que différentes tailles, couleurs ou matériaux. WCPOS fournit plusieurs moyens de travailler efficacement avec les produits variables.

## Identification des Produits Variables

Dans le Panneau de Produit, les produits variables se distinguent par :

- **Bouton flèche** (<Icon name="circle-arrow-right" color="green" />) au lieu du bouton plus (<Icon name="circle-plus" color="green" />) utilisé pour les produits simples
- **Options d'attributs** affichées sous le nom du produit (ex. : "Colour : Bleu, Vert, Rouge")
- **Lien "Développer"** pour voir toutes les variations en ligne

## Ajouter des Produits Variables au Panier

### Méthode 1 : Popover Rapide

Cliquez sur le bouton flèche verte <Icon name="circle-arrow-right" color="green" /> sur un produit variable pour ouvrir un popover avec des menus déroulants pour chaque attribut :

1. Cliquez sur le bouton <Icon name="circle-arrow-right" color="green" />
2. Sélectionnez des options dans chaque menu déroulant (ex. : Size: Large, Colour: Bleu)
3. La variation correspondante est ajoutée au panier

C'est la méthode la plus rapide lorsque vous savez exactement quelle variation vous avez besoin.

{/* Screenshot: variation-popover.png */}

### Méthode 2 : Développer en Ligne

Cliquez sur le lien **"Développer"** sur un produit variable pour afficher toutes ses variations directement dans la liste des produits :

1. Cliquez sur **Développer** sous le nom du produit
2. Toutes les variations apparaissent comme des lignes séparées sous le produit parent
3. Cliquez sur le bouton <Icon name="circle-plus" color="green" /> sur n'importe quelle variation pour l'ajouter au panier
4. Cliquez sur **Réduire** pour cacher à nouveau les variations

Cette méthode est utile lorsque vous souhaitez voir toutes les options disponibles, leurs niveaux de stock individuels et les prix.

{/* Screenshot: variations-expanded.png */}

### Méthode 3 : Scanner de Codes-Barres

Chaque variation de produit peut avoir son propre code-barres unique. Lorsque vous scannez le code-barres d'une variation :

- Si exactement une variation correspond, elle est ajoutée directement au panier
- Si plusieurs variations partagent le code-barres, la barre de recherche affiche les résultats correspondants

Voir [Scanner de Codes-Barres](/pos/product-panel/barcode-scanning) pour plus de détails.

## Informations sur les Variations

Chaque variation peut afficher :

- **Attributs de variation** - Les options spécifiques pour cette variation (ex. : "Size: Large, Colour: Bleu")
- **SKU** - Identifiant unique pour la variation (peut différer du produit parent)
- **Stock quantity** - Niveau de stock individuel pour la variation
- **Price** - Les variations peuvent avoir des prix différents du produit parent ou les unes des autres

## Variations dans le Panier

Lorsqu'une variation est ajoutée au panier, elle affiche :

- **Nom du produit** du produit parent
- **Attributs de variation** sous le nom
- **Price** pour cette variation spécifique

Vous pouvez modifier l'article de ligne comme n'importe quel autre produit dans le panier.

## Conseils pour Sélectionner Efficacement des Variations

### Utiliser le Scanner de Codes-Barres

Si vos variations ont des codes-barres individuels (recommandé), le scan est le moyen le plus rapide de les ajouter au panier.

### Configurer les Paramètres d'Affichage

Dans le Panneau de Produit [Paramètres d'Affichage](/pos/product-panel#display-settings), activez l'option **Attributs** pour afficher les attributs de variation directement dans la liste des produits sans développer.

### Navigation au Clavier

Lorsque le popover de variation est ouvert, vous pouvez utiliser les touches fléchées du clavier pour naviguer entre les options et Entrée pour sélectionner.

## F.A.Q.

<details>
<summary>Pourquoi ne puis-je pas voir une variation spécifique ?</summary>

Vérifiez que la variation :
- Est publiée et non en statut de brouillon
- Est réglée sur "En Stock" ou a une quantité en stock > 0 (si la gestion des stocks est activée)
- A tous les attributs requis définis

Assurez-vous également de ne pas avoir filtré le produit avec le filtre "En Stock" si la variation est en rupture de stock.

</details>

<details>
<summary>Puis-je ajouter une variation sans sélectionner tous les attributs ?</summary>

Non, vous devez sélectionner une option pour chaque attribut afin d'identifier une variation spécifique. Si un produit a des attributs Size et Colour, vous devez choisir les deux avant d'ajouter au panier.

</details>

<details>
<summary>Comment ajouter des codes-barres aux variations individuelles ?</summary>

Dans WooCommerce, modifiez le produit variable et développez chaque variation. Vous pouvez ajouter un code-barres unique au champ de code-barres de chaque variation (typiquement `_sku` ou `_global_unique_id` selon votre configuration).

</details>
