Les produits variables dans WooCommerce sont des produits qui ont plusieurs options, comme différentes tailles, couleurs ou matériaux. WCPOS offre plusieurs façons de travailler efficacement avec les produits variables.

## Identifier les Produits Variables

Dans le Panneau Produit, les produits variables se distinguent par :

- **Bouton flèche** (<Icon name="circle-arrow-right" color="green" />) au lieu du bouton plus (<Icon name="circle-plus" color="green" />) utilisé pour les produits simples
- **Options d'attributs** affichées sous le nom du produit (par exemple, "Couleur : Bleu, Vert, Rouge")
- **Lien "Développer"** pour afficher toutes les variations en ligne

## Ajout de Produits Variables au Panier

### Méthode 1 : Popover Rapide

Cliquez sur le bouton flèche vert <Icon name="circle-arrow-right" color="green" /> d'un produit variable pour ouvrir un popover avec des menus déroulants pour chaque attribut :

1. Cliquez sur le bouton <Icon name="circle-arrow-right" color="green" />
2. Sélectionnez les options dans chaque menu déroulant (par exemple, Taille : Grand, Couleur : Bleu)
3. La variation correspondante est ajoutée au panier

C'est la méthode la plus rapide lorsque vous savez exactement quelle variation vous voulez.

{/* Screenshot: variation-popover.png */}

### Méthode 2 : Développer en Ligne

Cliquez sur le lien **"Développer"** sur un produit variable pour afficher toutes ses variations directement dans la liste des produits :

1. Cliquez sur **Développer** sous le nom du produit
2. Toutes les variations apparaissent comme des lignes séparées sous le produit parent
3. Cliquez sur le bouton <Icon name="circle-plus" color="green" /> de n'importe quelle variation pour l'ajouter au panier
4. Cliquez sur **Réduire** pour masquer à nouveau les variations

Cette méthode est utile lorsque vous voulez voir toutes les options disponibles, leurs niveaux de stock individuels et leurs prix.

{/* Screenshot: variations-expanded.png */}

### Méthode 3 : Scan de Code-Barres

Chaque variation de produit peut avoir son propre code-barres unique. Lorsque vous scannez le code-barres d'une variation :

- Si exactement une variation correspond, elle est ajoutée directement au panier
- Si plusieurs variations partagent le même code-barres, la barre de recherche affiche les résultats correspondants

Consultez [Scan de Code-Barres](/pos/product-panel/barcode-scanning) pour plus de détails.

## Informations sur les Variations

Chaque variation peut afficher :

- **Attributs de variation** - Les options spécifiques pour cette variation (par exemple, "Taille : Grand, Couleur : Bleu")
- **SKU** - Identifiant unique pour la variation (peut différer du produit parent)
- **Quantité en stock** - Niveau de stock individuel pour la variation
- **Prix** - Les variations peuvent avoir des prix différents du parent ou entre elles

## Variations dans le Panier

Lorsqu'une variation est ajoutée au panier, elle affiche :

- **Nom du produit** du produit parent
- **Attributs de variation** en dessous du nom
- **Prix** pour cette variation spécifique

Vous pouvez modifier la ligne d'article comme n'importe quel autre produit du panier.

## Conseils pour une Sélection Efficace des Variations

### Utiliser le Scan de Code-Barres

Si vos variations ont des codes-barres individuels (recommandé), le scan est la méthode la plus rapide pour les ajouter au panier.

### Configurer les Options d'Affichage

Dans les [Paramètres d'Affichage](/pos/product-panel#display-settings) du Panneau Produit, activez l'option **Attributs** pour afficher les attributs de variation directement dans la liste des produits sans développer.

### Navigation au Clavier

Lorsque le popover de variation est ouvert, vous pouvez utiliser les touches fléchées pour naviguer dans les options et Entrée pour sélectionner.

## F.A.Q.

<details>
<summary>Pourquoi ne puis-je pas voir une variation spécifique ?</summary>

Vérifiez que la variation :
- Est publiée et non en brouillon
- Est définie comme "En stock" ou a une quantité en stock > 0 (si la gestion de stock est activée)
- A tous les attributs requis définis

Assurez-vous également de ne pas avoir filtré le produit avec le filtre "En stock" si la variation est en rupture de stock.

</details>

<details>
<summary>Puis-je ajouter une variation sans sélectionner tous les attributs ?</summary>

Non, vous devez sélectionner une option pour chaque attribut pour identifier une variation spécifique. Si un produit a des attributs de Taille et de Couleur, vous devez choisir les deux avant d'ajouter au panier.

</details>

<details>
<summary>Comment ajouter des codes-barres à des variations individuelles ?</summary>

Dans WooCommerce, modifiez le produit variable et développez chaque variation. Vous pouvez ajouter un code-barres unique au champ code-barres de chaque variation (généralement `_sku` ou `_global_unique_id` selon votre configuration).

</details>
