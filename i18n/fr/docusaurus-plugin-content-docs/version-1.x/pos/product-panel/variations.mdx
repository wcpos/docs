Les produits variables dans WooCommerce sont des produits qui ont plusieurs options, telles que différentes tailles, couleurs ou matériaux. WCPOS propose plusieurs façons de travailler efficacement avec des produits variables.

## Identification des Produits Variables

Dans le Panneau de Produits, les produits variables se distinguent par :

- **Bouton flèche** (<Icon name="circle-arrow-right" color="green" />) au lieu du bouton plus (<Icon name="circle-plus" color="green" />) utilisé pour les produits simples
- **Options d'attributs** affichées sous le nom du produit (ex. : "Couleur : Bleu, Vert, Rouge")
- **Lien "Développer"** pour afficher toutes les variations en ligne

## Ajout de Produits Variables au Panier

### Méthode 1 : Popover Rapide

Cliquez sur le bouton flèche verte <Icon name="circle-arrow-right" color="green" /> sur un produit variable pour ouvrir un popover avec des menus déroulants pour chaque attribut :

1. Cliquez sur le bouton <Icon name="circle-arrow-right" color="green" />
2. Sélectionnez des options dans chaque menu déroulant (ex. : Taille : Grande, Couleur : Bleue)
3. La variation correspondante est ajoutée au panier

C'est la méthode la plus rapide lorsque vous savez exactement quelle variation vous avez besoin.

{/* Screenshot: variation-popover.png */}

### Méthode 2 : Développer en Ligne

Cliquez sur le lien **"Développer"** sur un produit variable pour afficher toutes ses variations directement dans la liste des produits :

1. Cliquez sur **Développer** en dessous du nom du produit
2. Toutes les variations apparaissent comme des lignes séparées sous le produit parent
3. Cliquez sur le bouton <Icon name="circle-plus" color="green" /> sur n'importe quelle variation pour l'ajouter au panier
4. Cliquez sur **Réduire** pour cacher à nouveau les variations

Cette méthode est utile lorsque vous souhaitez voir toutes les options disponibles, leurs niveaux de stock individuels et les prix.

{/* Screenshot: variations-expanded.png */}

### Méthode 3 : Numérisation de Code-Barres

Chaque variation de produit peut avoir son propre code-barres unique. Lorsque vous scannez le code-barres d'une variation :

- Si une seule variation correspond, elle est ajoutée directement au panier
- Si plusieurs variations partagent le code-barres, la barre de recherche affiche les résultats correspondants

Consultez [Numérisation de Code-Barres](/pos/product-panel/barcode-scanning) pour plus de détails.

## Informations sur les Variations

Chaque variation peut afficher :

- **Attributs de variation** - Les options spécifiques pour cette variation (ex. : "Taille : Grande, Couleur : Bleue")
- **SKU** - Identifiant unique pour la variation (peut différer du produit parent)
- **Quantité en Stock** - Niveau de stock individuel pour la variation
- **Prix** - Les variations peuvent avoir des prix différents de ceux du produit parent ou entre elles

## Variations dans le Panier

Lorsqu'une variation est ajoutée au panier, elle s'affiche :

- **Nom du produit** du produit parent
- **Attributs de variation** sous le nom
- **Prix** pour cette variation spécifique

Vous pouvez modifier l'article de ligne comme n'importe quel autre produit dans le panier.

## Conseils pour une Sélection Efficace des Variations

### Utiliser la Numérisation de Code-Barres

Si vos variations ont des codes-barres individuels (recommandé), le scan est le moyen le plus rapide de les ajouter au panier.

### Configurer les Options d'Affichage

Dans le Panneau de Produits [Options d'Affichage](/pos/product-panel#display-settings), activez l'option **Attributs** pour afficher les attributs de variation directement dans la liste des produits sans développer.

### Navigation au Clavier

Lorsque le popover de variation est ouvert, vous pouvez utiliser les touches fléchées du clavier pour naviguer entre les options et Entrée pour sélectionner.

## F.A.Q.

<Accordion>
  <AccordionItem question="Pourquoi ne puis-je pas voir une variation spécifique ?">

Vérifiez que la variation :
- Est publiée et n'est pas en statut brouillon
- Est définie sur "En Stock" ou a une quantité en stock > 0 (si la gestion des stocks est activée)
- A tous les attributs requis définis

Assurez-vous également que vous n'avez pas filtré le produit avec le filtre "En Stock" si la variation est en rupture de stock.

  </AccordionItem>

  <AccordionItem question="Comment ajouter des codes-barres aux variations individuelles ?">

Dans WooCommerce, éditez le produit variable et développez chaque variation. Vous pouvez ajouter un code-barres unique au champ de code-barres de chaque variation (généralement `_sku` ou `_global_unique_id` en fonction de votre configuration).

  </AccordionItem>
