Le Panneau de Produit est le côté gauche de l'écran du POS où vous recherchez, parcourez et sélectionnez des produits à ajouter au panier.

{/* Capture d'écran: pos-product-panel.png */}

## Vue d'ensemble de l'interface

### Barre de recherche

En haut du panneau, la barre de recherche vous permet de trouver rapidement des produits par :

- **Nom du produit** - Tapez n'importe quelle partie du nom du produit
- **SKU** - Recherchez par Unité de Gestion des Stocks
- **Code-barres** - Scannez ou tapez un numéro de code-barres

Voir [Recherche & Filtrage](/pos/product-panel/search-filtering) pour des informations détaillées.

### Paramètres d'affichage

Cliquez sur l'**icône des curseurs** (<Icon name="sliders" />) à côté de la barre de recherche pour ouvrir les paramètres d'affichage. Cela vous permet de personnaliser quelles colonnes et informations sont affichées dans la liste des produits.

### Boutons de filtre

Sous la barre de recherche, les boutons de filtre permettent de restreindre les produits :

- **En Stock** - Affiche uniquement les produits avec un inventaire disponible
- **En Vedette** - Affiche les produits marqués comme en vedette dans WooCommerce
- **En Vente** - Affiche les produits actuellement en vente
- **Catégorie** - Filtrer par catégorie de produit
- **Tag** - Filtrer par tag de produit
- **Marque** - Filtrer par marque (si vous utilisez un plugin de marque)

Les filtres actifs apparaissent en surbrillance. Cliquez à nouveau sur un filtre pour le supprimer.

### Liste des produits

La zone principale affiche vos produits dans une liste déroulante. Chaque ligne montre :

- **Image du produit** - Vignette du produit
- **Nom du produit** - Avec des détails optionnels (stock, SKU, catégories, etc.)
- **Prix** - Prix de vente actuel (avec les prix de vente affichés lorsque cela est applicable)
- **Bouton d'action** - Ajouter au panier (<Icon name="circle-plus" color="green" />) ou sélectionner une variante (<Icon name="circle-arrow-right" color="green" />)

### Types de produits

- **Produits simples** affichent un bouton vert <Icon name="circle-plus" color="green" /> pour ajouter directement au panier
- **Produits variables** affichent une flèche verte <Icon name="circle-arrow-right" color="green" /> pour ouvrir le sélecteur de variante

Voir [Produits Variables](/pos/product-panel/variations) pour plus de détails.

### Pied de page

En bas du Panneau de Produit :

- **Statut fiscal** - Affiche la base de calcul de la taxe actuelle (par exemple, "Taxe basée sur : adresse de la boutique"). Cliquez pour voir les taux de taxe actifs.
- **Nombre de produits** - Affiche combien de produits sont affichés (par exemple, "Affichage de 10 sur 17")
- **Bouton de synchronisation** (<Icon name="arrow-rotate-right" />) - Rafraîchir les produits depuis le serveur. **Appui long** pour des options supplémentaires :
  - **Synchroniser** - Rafraîchissement standard depuis le serveur
  - **Effacer et Rafraîchir** - Effacer les données locales et recharger tout

### Afficher les produits hors stock

Contrôle si les produits qui sont actuellement hors stock sont affichés.

- **Activé** : Les produits hors stock restent visibles mais ne peuvent pas être ajoutés au panier
- **Désactivé** : Les produits hors stock sont cachés de la liste

### Colonnes

Configurez quelles colonnes apparaissent dans la liste des produits :

| Colonne | Description | Options d'affichage |
|---------|-------------|---------------------|
| **Image** | Vignette du produit | - |
| **Produit** | Nom et détails du produit | Stock, SKU, Code-barres, Catégories, Tags, Marques, Attributs, Métadonnées |
| **SKU** | Unité de Gestion des Stocks (colonne séparée) | - |
| **Code-barres** | Code-barres du produit (colonne séparée) | - |
| **Type** | Type de produit (simple, variable, etc.) | - |
| **Stock** | Quantité de stock actuelle | - |
| **Coût des marchandises vendues** | Prix de coût du produit | - |
| **Prix** | Prix de vente | Info taxe, Indicateur de vente |
| **Actions** | Bouton ajouter au panier | - |

### Clés de métadonnées

Spécifiez les clés de métadonnées du produit qui doivent être transférées aux lignes de commandes lorsque les produits sont ajoutés au panier.

Entrez une liste de clés de métadonnées séparées par des virgules :

```
_bottle_deposit,_custom_field,_tracking_code
```

**Cas d'utilisation :**
- Plugins de dépôt de bouteilles
- Champs de produit personnalisés
- Données d'intégration tierces
- Informations de conformité/suivi

:::note
Les clés de métadonnées sont sensibles à la casse et doivent correspondre exactement à celles stockées dans les métadonnées du produit.
:::

### Restaurer les paramètres par défaut

Cliquez pour réinitialiser tous les paramètres d'affichage à leurs valeurs par défaut d'origine.

## Ajout de produits au panier

### Produits simples

Cliquez sur le bouton vert <Icon name="circle-plus" color="green" /> pour ajouter un produit simple au panier. Chaque clic ajoute une unité supplémentaire.

### Produits variables

Les produits variables (par exemple, un t-shirt avec des options de taille et de couleur) affichent une flèche verte <Icon name="circle-arrow-right" color="green" />. Cliquez pour :

1. **Popover rapide** - Sélectionner une variante dans un menu déroulant
2. **Développer en ligne** - Cliquez sur "Développer" pour voir toutes les variantes dans la liste

Voir [Produits Variables](/pos/product-panel/variations) pour plus de détails.

### Numérisation de code-barres

Connectez un scanner de code-barres USB ou Bluetooth pour ajouter rapidement des produits. Lorsque vous scannez un code-barres, WCPOS recherche et ajoute automatiquement le produit correspondant.

Voir [Numérisation de Code-barres](/pos/product-panel/barcode-scanning) pour les instructions de configuration.

## Pourquoi certains produits ne sont-ils pas visibles ?

Si des produits manquent dans votre Panneau de Produit, vérifiez :

- **Visibilité POS** - Les produits définis sur "En ligne seulement" n'apparaîtront pas dans le POS. Voir [Produits uniquement POS](/products/pos-only-products).
- **Paramètres de stock** - Si "Afficher les produits hors stock" est désactivé dans les paramètres d'affichage, les articles hors stock sont cachés.
- **Statut de synchronisation** - Les nouveaux produits ou les produits récemment mis à jour peuvent nécessiter une synchronisation. Appuyez longuement sur le bouton de synchronisation et sélectionnez **"Effacer et Rafraîchir"**.
- **Filtres** - Vérifiez si vous avez des filtres actifs (Catégorie, Tag, etc.) qui pourraient cacher des produits.
