Chaque élément du panier est un élément de ligne qui peut être édité directement. WCPOS offre une édition en ligne flexible pour les quantités, les prix et d'autres détails.

## Types d'Éléments de Ligne {#line-item-types}

WooCommerce utilise trois types d'éléments de ligne :

| Type | Description | Exemples |
|------|-------------|----------|
| **line_item** | Produits de votre catalogue | T-shirt, Mug à café |
| **fee_line** | Frais supplémentaires | Emballage cadeau, Frais de service |
| **shipping_line** | Coûts d'expédition | Expédition standard, Livraison express |

:::note
Les coûts d'expédition sont saisis manuellement. Le POS ne calcule pas automatiquement les taux d'expédition.
:::

## Édition des Éléments de Ligne {#editing-line-items}

### Quantité {#quantity}

Cliquez sur le champ de quantité pour l'éditer directement. Tapez un nouveau numéro ou utilisez les contrôles de sélection.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nom du Produit {#product-name}

Pour les produits divers et les frais, vous pouvez modifier le nom directement en cliquant dessus.

### Prix {#price}

Cliquez sur le champ de prix pour changer le prix unitaire. Cela est utile pour :

- Appliquer des remises manuelles
- Match de prix
- Corriger les erreurs de prix

Le total se met à jour automatiquement en fonction de la quantité × prix.

### Menu à Trois Points {#three-dot-menu}

Cliquez sur le **⋮** (trois points) sur n'importe quel élément de ligne pour accéder à des options supplémentaires :

{/* Screenshot: cart-line-item-menu.png */}

#### Formulaire d'Édition {#edit-form}

Ouvre un formulaire d'édition détaillé pour l'élément de ligne, vous permettant de modifier tous les champs.

#### Vue JSON Brute {#raw-json-view}

Affiche les données JSON brutes pour l'élément de ligne. Cela est extrêmement utile pour :

- **Débogage** - Voir exactement quelles données sont envoyées à WooCommerce
- **Dépannage** - Identifier les problèmes avec les champs personnalisés ou les métadonnées
- **Développement** - Comprendre la structure des données pour les intégrations

### Supprimer un Élément {#remove-item}

Cliquez sur le bouton rouge **×** pour supprimer un élément du panier.

## Attributs de Variation {#variation-attributes}

Pour les produits variables, les attributs de variation sélectionnés s'affichent sous le nom du produit :

```
Hoodie
Colour: Green
Size: Large
```

Ces attributs font partie des données de l'élément de ligne et sont inclus dans la commande.

## Métadonnées {#meta-data}

Si vous avez configuré [Clés de Méta Données](/pos/product-panel#meta-data-keys) dans les paramètres du Panneau de Produit, ces données sont automatiquement copiées dans l'élément de ligne lorsque le produit est ajouté au panier.

Vous pouvez voir ces métadonnées dans la vue JSON brute.

## Fractionnement des Éléments de Ligne {#splitting-line-items}

S'il est activé dans [Paramètres d'Affichage](/pos/cart#display-settings), vous pouvez diviser un élément de ligne en plusieurs lignes. Cela est utile lorsqu'un client souhaite appliquer différentes remises à des portions du même produit.

## Conseils {#tips}

### Ajustements de Prix Rapides {#quick-price-adjustments}

Pour donner rapidement une remise :

1. Cliquez sur le champ de prix
2. Entrez le nouveau prix
3. Appuyez sur Entrée

### Navigation au Clavier {#keyboard-navigation}

- **Tab** - Passer entre les champs éditables
- **Entrée** - Confirmer l'édition
- **Échap** - Annuler l'édition

### Édits en Lot {#batch-edits}

Pour des commandes complexes, envisagez d'utiliser le menu à trois points pour accéder au formulaire d'édition complet, qui montre tous les champs à la fois.
