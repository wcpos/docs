Chaque article du panier est un élément de ligne qui peut être modifié directement. WCPOS offre une édition en ligne flexible pour les quantités, les prix et autres détails.

## Types d'Éléments de Ligne

WooCommerce utilise trois types d'éléments de ligne :

| Type | Description | Exemples |
|------|-------------|----------|
| **line_item** | Produits de votre catalogue | T-shirt, Mug à café |
| **fee_line** | Frais supplémentaires | Emballage cadeau, Frais de service |
| **shipping_line** | Frais de livraison | Livraison standard, Livraison express |

:::note
Les frais de livraison sont saisis manuellement. Le POS ne calcule pas automatiquement les tarifs de livraison.
:::

## Édition des Éléments de Ligne

### Quantité

Cliquez sur le champ de quantité pour l'éditer directement. Saisissez un nouveau nombre ou utilisez les contrôles de rotation.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nom du Produit

Pour les produits et frais divers, vous pouvez modifier le nom directement en cliquant dessus.

### Prix

Cliquez sur le champ de prix pour changer le prix unitaire. Cela est utile pour :

- Appliquer des remises manuelles
- Ajuster les prix
- Corriger des erreurs de tarification

Le total se met à jour automatiquement en fonction de la quantité × le prix.

### Menu à Trois Points

Cliquez sur **⋮** (trois points) sur n'importe quel élément de ligne pour accéder à des options supplémentaires :

{/* Screenshot: cart-line-item-menu.png */}

#### Formulaire d'Édition

Ouvre un formulaire d'édition détaillé pour l'élément de ligne, vous permettant de modifier tous les champs.

#### Vue JSON Brute

Affiche les données JSON brutes de l'élément de ligne. Cela est extrêmement utile pour :

- **Débogage** - Voir exactement quelles données sont envoyées à WooCommerce
- **Dépannage** - Identifier les problèmes avec les champs personnalisés ou les métadonnées
- **Développement** - Comprendre la structure des données pour les intégrations

### Supprimer l'Article

Cliquez sur le bouton rouge **×** pour supprimer un article du panier.

## Attributs de Variation

Pour les produits variables, les attributs de variation sélectionnés s'affichent sous le nom du produit :

```
Sweat à capuche
Couleur : Vert
Taille : Large
```

Ces attributs font partie des données de l'élément de ligne et sont inclus dans la commande.

## Métadonnées

Si vous avez configuré des [Clés de Métadonnées](/pos/product-panel#meta-data-keys) dans les paramètres du Panneau Produit, ces données sont automatiquement copiées dans l'élément de ligne lors de l'ajout du produit au panier.

Vous pouvez afficher ces métadonnées dans la vue JSON brute.

## Séparation des Éléments de Ligne

Si activé dans les [Paramètres d'Affichage](/pos/cart#display-settings), vous pouvez diviser un élément de ligne en plusieurs lignes. Cela est utile lorsqu'un client souhaite appliquer différentes remises à des parties du même produit.

## Conseils

### Ajustements Rapides de Prix

Pour donner rapidement une remise :

1. Cliquez sur le champ de prix
2. Saisissez le nouveau prix
3. Appuyez sur Entrée

### Navigation au Clavier

- **Tab** - Se déplacer entre les champs modifiables
- **Entrée** - Confirmer l'édition
- **Échap** - Annuler l'édition

### Éditions par Lot

Pour les commandes complexes, envisagez d'utiliser le menu à trois points pour accéder au formulaire d'édition complet, qui affiche tous les champs en une seule fois.
