Chaque article dans le panier est un article de ligne qui peut être modifié directement. WCPOS offre un éditeur en ligne flexible pour les quantités, les prix et d'autres détails.

## Types d'Articles de Ligne

WooCommerce utilise trois types d'articles de ligne :

| Type | Description | Exemples |
|------|-------------|----------|
| **line_item** | Produits de votre catalogue | T-shirt, Tasse à café |
| **fee_line** | Frais supplémentaires | Emballage cadeau, Frais de service |
| **shipping_line** | Coûts d'expédition | Livraison standard, Livraison express |

:::note
Les coûts d'expédition sont saisis manuellement. Le POS ne calcule pas automatiquement les tarifs d'expédition.
:::

## Modification des Articles de Ligne

### Quantité

Cliquez sur le champ de quantité pour le modifier directement. Tapez un nouveau nombre ou utilisez les contrôles de rotation.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nom du Produit

Pour les produits et frais divers, vous pouvez modifier le nom directement en cliquant dessus.

### Prix

Cliquez sur le champ de prix pour changer le prix unitaire. Cela est utile pour :

- Appliquer des remises manuelles
- Ajuster les prix
- Corriger les erreurs de prix

Le total se met à jour automatiquement en fonction de la quantité × prix.

### Menu à Trois Points

Cliquez sur le **⋮** (trois points) sur n'importe quel article de ligne pour accéder à des options supplémentaires :

{/* Screenshot: cart-line-item-menu.png */}

#### Formulaire d'Édition

Ouvre un formulaire d'édition détaillé pour l'article de ligne, vous permettant de modifier tous les champs.

#### Vue JSON Brut

Affiche les données JSON brutes pour l'article de ligne. Cela est extrêmement utile pour :

- **Débogage** - Voir exactement quelles données sont envoyées à WooCommerce
- **Résolution de problèmes** - Identifier les problèmes avec les champs personnalisés ou les métadonnées
- **Développement** - Comprendre la structure des données pour les intégrations

### Supprimer un Article

Cliquez sur le bouton rouge **×** pour supprimer un article du panier.

## Attributs de Variation

Pour les produits variables, les attributs de variation sélectionnés s'affichent sous le nom du produit :

```
Hoodie
Colour: Green
Size: Large
```

Ces attributs font partie des données de l'article de ligne et sont inclus dans la commande.

## Métadonnées

Si vous avez configuré [Clés de métadonnées](/pos/product-panel#meta-data-keys) dans les paramètres du panneau produit, ces données sont automatiquement copiées dans l'article de ligne lorsque le produit est ajouté au panier.

Vous pouvez voir ces métadonnées dans la vue JSON brute.

## Division des Articles de Ligne

Si activé dans [Paramètres d'Affichage](/pos/cart#display-settings), vous pouvez diviser un article de ligne en plusieurs lignes. Cela est utile lorsqu'un client souhaite appliquer différentes remises à des portions du même produit.

## Astuces

### Ajustements Rapides des Prix

Pour donner rapidement une remise :

1. Cliquez sur le champ de prix
2. Entrez le nouveau prix
3. Appuyez sur Entrée

### Navigation au Clavier

- **Tab** - Passer entre les champs modifiables
- **Enter** - Confirmer la modification
- **Escape** - Annuler la modification

### Modifications en Masse

Pour des commandes complexes, envisagez d'utiliser le menu à trois points pour accéder au formulaire d'édition complet, qui affiche tous les champs en une seule fois.
