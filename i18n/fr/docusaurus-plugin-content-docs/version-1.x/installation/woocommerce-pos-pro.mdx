:::important
[WooCommerce POS Pro](http://wcpos.com/pro) est un plugin payant qui ajoute des fonctionnalités au plugin gratuit [WooCommerce POS](http://wordpress.org/plugins/woocommerce-pos/).
:::

## Installation

Si vous avez acheté une licence pour WooCommerce POS Pro, veuillez suivre les étapes ci-dessous pour installer et activer le plugin :

1.  Allez sur : [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Sous **Téléchargements**, cliquez sur le lien de téléchargement et enregistrez le plugin sur votre bureau.
3.  Ensuite, allez sur votre site, connectez-vous et allez à `WP Admin > Plugins > Ajouter > Téléverser un plugin`
4.  Téléversez le fichier zip du plugin depuis votre bureau et activez-le.
5.  Ensuite, allez à `WP Admin > POS > Réglages > Licence` et entrez votre clé de licence et votre e-mail de licence pour compléter l'activation.

![Mes téléchargements](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Vous pouvez télécharger WooCommerce POS Pro sur votre page de compte")

## Mises à jour

Lorsque de nouvelles versions seront prêtes, elles apparaîtront dans votre tableau de bord des mises à jour comme n'importe quel autre plugin. Vous pouvez également télécharger la dernière version depuis votre [page de compte](https://wcpos.com/my-account/).
