---
title: Exigences minimales
---

Pour assurer un fonctionnement optimal du plugin WooCommerce POS, veuillez vous assurer que votre environnement répond aux exigences minimales suivantes :

### Exigences du serveur :

- **WordPress :** Version 5.6 ou supérieure
- **WooCommerce :** Version 5.3 ou supérieure
- **PHP :** Version 7.2 ou supérieure

Le respect de ces exigences minimales garantit la compatibilité et la stabilité entre le plugin WooCommerce POS et votre environnement WordPress. Cela garantit également que vous pouvez bénéficier des dernières fonctionnalités et améliorations fournies par le plugin.

### Exigences du navigateur :

- **Navigateur web moderne :** Le plugin WooCommerce POS est conçu pour fonctionner avec les navigateurs web modernes, y compris Google Chrome, Mozilla Firefox, Safari et Microsoft Edge. Veuillez vous assurer que vous utilisez la dernière version stable de votre navigateur préféré.
- **JavaScript activé :** Le plugin WooCommerce POS repose sur JavaScript pour ses fonctionnalités interactives. Veuillez vous assurer que JavaScript est activé dans les paramètres de votre navigateur.
- **Pas de mode navigation privée :** Le navigateur ne doit pas être en mode navigation privée car cela peut restreindre l'accès à la base de données IndexedDB (IDB) du navigateur. Le mode navigation privée peut empêcher le bon fonctionnement de certaines fonctionnalités du plugin WooCommerce POS.

En respectant les exigences minimales et en utilisant un navigateur web compatible, vous pouvez avoir une expérience transparente en utilisant le plugin WooCommerce POS et tirer pleinement parti de ses fonctionnalités pour gérer efficacement vos opérations de point de vente.
