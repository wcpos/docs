## Réinstallation d'une version antérieure

Si vous devez revenir à une version antérieure du plugin WooCommerce POS, suivez ces étapes :

1. Visitez la page WordPress.org [Vue avancée](https://wordpress.org/plugins/woocommerce-pos/advanced/) pour WooCommerce POS.

2. Faites défiler jusqu'à la section **Options avancées** et localisez le menu déroulant **Versions antérieures**.

3. Ouvrez le menu déroulant et vous verrez une liste des versions de plugin disponibles. Trouvez la version que vous souhaitez installer et cliquez sur le bouton **Télécharger** à côté. Enregistrez le fichier téléchargé sur votre bureau ou dans un emplacement pratique.

4. Connectez-vous à votre tableau de bord d'administration WordPress.

5. Accédez à la page **Extensions** en cliquant sur **Extensions** dans le menu de gauche.

6. Désactivez et supprimez la version actuelle du plugin WooCommerce POS. Cela supprimera le plugin de votre installation WordPress. Ne vous inquiétez pas, vos données ne seront pas perdues.

7. Cliquez sur le bouton **Ajouter** en haut de la page.

8. Sur la page **Ajouter des extensions**, cliquez sur le bouton **Téléverser une extension**.

9. Choisissez le fichier de plugin précédemment téléchargé depuis votre bureau ou l'emplacement où vous l'avez enregistré.

<div class="image-container">
  <Image

  />
  <p class="image-caption">Téléverser une extension dans l'administration WordPress</p>
</div>

10. Cliquez sur le bouton **Installer maintenant** et attendez que l'installation soit terminée.

11. Après la fin de l'installation, cliquez sur le bouton **Activer** pour activer la version antérieure du plugin WooCommerce POS.

C'est tout ! Vous avez réussi à réinstaller une version antérieure du plugin WooCommerce POS.

Si vous rencontrez des problèmes lors du processus de réinstallation ou si vous avez besoin d'une assistance supplémentaire, veuillez contacter notre équipe de support à l'adresse support@wcpos.com.
