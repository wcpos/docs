# Traduit: WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

Ajoute la compatibilité WCPOS pour les crédits boutique créés avec [Smart Coupons par StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/). Cette extension permet aux commandes du point de vente d'utiliser les crédits boutique StoreApps, maintient le solde StoreApps synchronisé et laisse une trace d'audit dans les notes de commande pour le personnel.

Cette extension est spécifiquement conçue pour le plugin Smart Coupons de StoreApps / WooCommerce.com. D'autres plugins portant des noms similaires comme « Smart Coupons » sont des produits distincts et peuvent nécessiter des intégrations séparées.

:::info Fonctionnalité Pro
L'installation et la gestion des extensions depuis le répertoire nécessitent [WCPOS Pro](/getting-started/pro-license).
:::

## Fonctionnement {#what-it-does}

- Enregistre les métadonnées StoreApps `smart_coupons_contribution` pour les commandes créées au point de vente lorsque le contexte de panier du passage en caisse standard n'est pas disponible.
- Permet à StoreApps de déduire le montant partiel correct du crédit boutique du solde du coupon après le passage en caisse au point de vente.
- Restaure le solde de crédit boutique StoreApps lorsque la commande du PDV est annulée ou remboursée via le cycle de vie normal des commandes WooCommerce.
- Ajoute une note de commande privée après l'utilisation, indiquant le code du coupon, le montant utilisé et le solde actuel.
- Affiche le solde restant du crédit boutique dans la ligne de remise existante du reçu en l'ajoutant au libellé de description/remise du coupon.

L'extension utilise les champs de coupon et métadonnées standard de WooCommerce. Elle n'ajoute pas de champs de réponse de coupon personnalisés à WCPOS et ne nécessite pas de modèles de reçu personnalisés.

## Installation {#installation}

<Steps>
  <StepCard number="1" title="FR: Installer StoreApps Smart Coupons">

Installez et activez [Smart Coupons par StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/). Créez vos coupons de crédit boutique ou cartes cadeaux comme d'habitude dans WooCommerce.

  </StepCard>
  <StepCard number="2" title="FR: Installer WCPOS StoreApps Smart Coupons">

Installez depuis `WP Admin > POS > Réglages > Extensions`, ou téléchargez la dernière version depuis la [page des versions GitHub](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases) et téléversez via `Extensions > Ajouter > Téléverser une extension`.

  </StepCard>
  <StepCard number="3" title="FR: Synchroniser les coupons avec le PDV" isLast>

Ouvrez le PDV et synchronisez les coupons. Les caissiers peuvent ensuite appliquer les coupons de crédit boutique StoreApps depuis le flux normal **Ajouter un coupon**.

  </StepCard>
</Steps>

## Sortie du reçu {#receipt-output}

Les modèles de reçus WCPOS impriment déjà les remises de coupons via le libellé de la ligne de remise. Lors du rendu du reçu, cette extension ajoute le solde StoreApps actuel à ce libellé existant.

Par exemple, une description de coupon **Carte cadeau** peut s'imprimer ainsi :

```text
Gift card — Store credit balance: £61.50
```

Si le coupon n'a pas de description, le texte du solde est utilisé comme libellé de remise. Les modèles de reçus existants n'ont pas besoin d'être modifiés.

## Piste d'audit des commandes {#order-audit-trail}

Après le traitement par StoreApps d'une utilisation d'avoir en point de vente, l'extension ajoute une note de commande privée similaire à :

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

Utilisez cette note pour vérifier quel coupon d'avoir a été utilisé, le montant utilisé et le solde restant après la vente.

## Notes de compatibilité {#compatibility-notes}

- **StoreApps uniquement :** cette extension cible StoreApps / WooCommerce.com Smart Coupons. Elle ne cible pas WebToffee ni d'autres extensions de coupons intelligents.
- **Données de coupon :** WCPOS continue d'utiliser les réponses de coupon WooCommerce standard. L'état spécifique à StoreApps reste dans les métadonnées WooCommerce telles que `smart_coupons_contribution` et les champs de solde du coupon.
- **Reçus :** le texte du solde est ajouté via le chemin existant de description/libellé de remise du coupon, sans modifier les données ni les modèles de reçus.
- **Extension StoreApps inactive :** si StoreApps Smart Coupons n'est pas activée, l'extension ne fait rien.
