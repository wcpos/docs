:::info Fonctionnalité Pro
[WCPOS Pro](https://wcpos.com/pro) est un plugin payant qui ajoute des fonctionnalités au [plugin WCPOS gratuit](https://wordpress.org/plugins/woocommerce-pos/).
:::

## Ce qui est inclus dans Pro

WCPOS Pro déverrouille des fonctionnalités supplémentaires :

- **Écran Produits** - Modifier les niveaux de stock et les prix directement dans le POS
- **Écran Commandes** - Consulter et gérer l'historique des commandes
- **Écran Clients** - Ajouter et modifier les informations client
- **Écran Rapports** - Générer des rapports de ventes de fin de journée
- **Passerelles de paiement supplémentaires** - Stripe Terminal, SumUp et passerelles personnalisées

## Installation

Si vous avez acheté une licence WCPOS Pro, veuillez suivre les étapes ci-dessous pour installer et activer le plugin :

1. Aller à : [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Sous **Téléchargements**, cliquez sur le lien de téléchargement et enregistrez le plugin sur votre bureau.
3. Ensuite, allez sur votre site, connectez-vous et accédez à `WP Admin > Plugins > Ajouter > Télécharger un plugin`
4. Téléchargez le fichier zip du plugin depuis votre bureau et activez-le.
5. Ensuite, allez à `WP Admin > POS > Paramètres > Licence` et entrez votre Clé de Licence et votre Email de Licence pour terminer l'activation.

![Mes Téléchargements](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Vous pouvez télécharger WCPOS Pro sur votre page de compte")

## Mises à jour

Lorsque de nouvelles versions sont prêtes, elles apparaîtront dans votre tableau de bord Mises à jour comme tout autre plugin. Vous pouvez également télécharger la dernière version depuis votre [page de compte](https://wcpos.com/my-account/).
