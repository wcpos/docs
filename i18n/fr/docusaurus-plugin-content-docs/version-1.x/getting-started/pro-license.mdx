:::info Fonction Pro
[WCPOS Pro](https://wcpos.com/pro) est un plugin payant qui ajoute des fonctionnalités au plugin gratuit [WCPOS](https://wordpress.org/plugins/woocommerce-pos/).
:::

## Qu'est-ce qui est inclus dans Pro ?

WCPOS Pro débloque des fonctionnalités supplémentaires :

- **Écran des produits** - Modifiez les niveaux de stock et les prix directement dans le POS
- **Écran des commandes** - Consultez et gérez l'historique des commandes
- **Écran des clients** - Ajoutez et modifiez les informations des clients
- **Écran des rapports** - Générez des rapports de ventes de fin de journée
- **Passerelles de paiement supplémentaires** - Stripe Terminal, SumUp et passerelles personnalisées

## Installation

Si vous avez acheté une licence pour WCPOS Pro, veuillez suivre les étapes ci-dessous pour installer et activer le plugin :

1. Allez sur : [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Sous **Téléchargements**, cliquez sur le lien de téléchargement et enregistrez le plugin sur votre bureau.
3. Ensuite, allez sur votre site, connectez-vous et allez à `WP Admin > Plugins > Ajouter Nouveau > Téléverser le Plugin`
4. Téléversez le fichier zip du plugin depuis votre bureau et activez-le.
5. Ensuite, allez à `WP Admin > POS > Paramètres > Licence` et entrez votre Clé de Licence et votre Email de Licence pour compléter l'activation.

![Mes Téléchargements](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Vous pouvez télécharger WCPOS Pro sur votre page de compte")

## Mises à jour

Lorsque de nouvelles versions sont prêtes, elles apparaîtront dans votre tableau de bord des mises à jour comme tout autre plugin. Vous pouvez également télécharger la dernière version depuis votre [page de compte](https://wcpos.com/my-account/).
