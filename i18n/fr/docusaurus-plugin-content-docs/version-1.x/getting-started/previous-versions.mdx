## Réinstallation d'une Version Précédente

Si vous devez revenir à une version précédente du plugin WCPOS, suivez ces étapes :

1. Visitez la page WordPress.org [Vue Avancée](https://wordpress.org/plugins/woocommerce-pos/advanced/) pour WCPOS.

2. Faites défiler jusqu'à la section **Options Avancées** et localisez le menu de sélection **Versions Précédentes**.

3. Ouvrez le menu de sélection et vous verrez une liste des versions de plugin disponibles. Trouvez la version que vous voulez installer et cliquez sur le bouton **Télécharger** à côté. Enregistrez le fichier téléchargé sur votre bureau ou à un emplacement pratique.

4. Connectez-vous à votre tableau de bord d'administration WordPress.

5. Accédez à la page **Plugins** en cliquant sur **Plugins** dans le menu de gauche.

6. Désactivez et supprimez la version actuelle du plugin WCPOS. Cela supprimera le plugin de votre installation WordPress. Ne vous inquiétez pas, vos données ne seront pas perdues.

7. Cliquez sur le bouton **Ajouter Nouveau** en haut de la page.

8. Sur la page **Ajouter des Plugins**, cliquez sur le bouton **Téléverser un Plugin**.

9. Choisissez le fichier de plugin précédemment téléchargé depuis votre bureau ou l'emplacement où vous l'avez sauvegardé.

<div class="image-container">
  <Image
    alt="Téléverser le Plugin"

  />
  <p class="image-caption">Téléverser le plugin dans l'Administration WordPress</p>
</div>

10. Cliquez sur le bouton **Installer Maintenant** et attendez que l'installation se termine.

11. Une fois l'installation terminée, cliquez sur le bouton **Activer** pour activer la version précédente du plugin WCPOS.

C'est tout ! Vous avez réinstallé avec succès une version précédente du plugin WCPOS.

Si vous rencontrez des problèmes pendant le processus de réinstallation ou avez besoin d'aide supplémentaire, veuillez contacter notre équipe de support à support@wcpos.com.
