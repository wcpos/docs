## Réinstallation d'une Version Précédente {#re-installing-a-previous-version}

Si vous devez revenir à une version précédente du plugin WCPOS, suivez ces étapes :

1. Rendez-vous sur la page WordPress.org [Vue Avancée](https://wordpress.org/plugins/woocommerce-pos/advanced/) pour WCPOS.

2. Faites défiler vers le bas jusqu'à la section **Options Avancées** et localisez le menu déroulant **Versions Précédentes**.

3. Ouvrez le menu déroulant et vous verrez une liste des versions disponibles du plugin. Trouvez la version que vous souhaitez installer et cliquez sur le bouton **Télécharger** à côté. Enregistrez le fichier téléchargé sur votre bureau ou à un emplacement pratique.

4. Connectez-vous à votre tableau de bord WordPress Admin.

5. Accédez à la page **Plugins** en cliquant sur **Plugins** dans le menu de gauche.

6. Désactivez et supprimez la version actuelle du plugin WCPOS. Cela supprimera le plugin de votre installation WordPress. Ne vous inquiétez pas, vos données ne seront pas perdues.

7. Cliquez sur le bouton **Ajouter un Plugin** en haut de la page.

8. Sur la page **Ajouter des Plugins**, cliquez sur le bouton **Télécharger un Plugin**.

9. Choisissez le fichier du plugin précédemment téléchargé depuis votre bureau ou l'emplacement où vous l'avez enregistré.

<div class="image-container">
  <Image
    alt="Télécharger le Plugin"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">Télécharger le plugin dans l'Administration WordPress</p>
</div>

10. Cliquez sur le bouton **Installer Maintenant** et attendez que l'installation soit terminée.

11. Une fois l'installation terminée, cliquez sur le bouton **Activer** pour activer la version précédente du plugin WCPOS.

C'est tout ! Vous avez réussi à réinstaller une version précédente du plugin WCPOS.

Si vous rencontrez des problèmes pendant le processus de réinstallation ou si vous avez besoin d'aide supplémentaire, veuillez contacter notre équipe de support à support@wcpos.com.
