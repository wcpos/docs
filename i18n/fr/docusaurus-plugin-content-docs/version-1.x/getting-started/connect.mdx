:::note Bureau & Mobile Seulement
Cet écran n'est affiché que dans les applications Bureau et Mobile. Les utilisateurs web accèdent directement au POS à `yourdomain.com/pos` et se connectent avec leurs identifiants WordPress.
:::

## Aperçu de l'écran de connexion

Lorsque vous ouvrez l'application WCPOS Bureau ou Mobile, vous verrez l'écran de connexion. C'est ici que vous gérez vos connexions à la boutique et les connexions des utilisateurs.

{/* Capture d'écran : connect-screen.png */}

## Ajouter un nouveau magasin

1. Entrez l'URL de votre magasin WooCommerce dans le champ de texte (par exemple, `https://mystore.com`)
2. Cliquez sur **Connexion**
3. Vous serez redirigé pour vous connecter avec vos identifiants WordPress
4. Après une connexion réussie, vous retournerez à l'application

## Magasins multiples

Vous pouvez vous connecter à autant de magasins WooCommerce que nécessaire. Chaque magasin apparaît sous la forme d'une carte distincte sur l'écran de connexion, affichant :

- **Nom du magasin** et favicon
- **URL du magasin**
- **Utilisateurs connectés** pour ce magasin

Ceci est utile si vous gérez plusieurs emplacements ou entreprises.

## Utilisateurs multiples par magasin

Chaque magasin peut avoir plusieurs utilisateurs connectés simultanément. Cela est utile pour :

- **Changements de quarts** - Un nouveau caissier peut se connecter avant que le précédent ne se déconnecte
- **Configurations multi-enregistreurs** - Différents caissiers sur différents appareils
- **Changement rapide** - Passer facilement entre les comptes utilisateurs

### Ajouter un utilisateur

Cliquez sur le bouton <Icon name="plus" /> à côté de "Utilisateurs connectés" pour ajouter un autre utilisateur à ce magasin.

### Changer d'utilisateur

Cliquez sur un badge utilisateur (par exemple, "Brenda") pour ouvrir le POS en tant que cet utilisateur.

### Supprimer un utilisateur

Cliquez sur le **×** sur un badge utilisateur pour déconnecter cet utilisateur du magasin.

## Supprimer un magasin

Cliquez sur le bouton rouge **×** sur la carte du magasin pour le supprimer de votre liste. Cela déconnecte tous les utilisateurs et supprime la connexion du magasin de l'application.

## Magasin de démo

En bas de l'écran, vous pouvez voir un lien "Entrer dans le magasin de démo". Cela vous connecte à un magasin WooCommerce de démonstration pour essayer les fonctionnalités de WCPOS sans affecter les données de votre propre magasin.

## Dépannage

### Impossible de me connecter à mon magasin ?

- Assurez-vous que le plugin WCPOS est installé et activé sur votre site WordPress
- Vérifiez que vous utilisez la bonne URL (incluez `https://` si votre site utilise SSL)
- Vérifiez que l'API REST WooCommerce est accessible
- Assurez-vous que votre compte utilisateur a les autorisations d'accès au POS

### La connexion échoue constamment ?

- Essayez d'accéder à `yourdomain.com/pos` dans un navigateur web d'abord pour vérifier que le plugin fonctionne
- Vérifiez les journaux d'erreur de votre site pour tout problème
- Assurez-vous que votre serveur répond aux [exigences minimales](/getting-started/installation)
