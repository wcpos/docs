## Vue d'Ensemble de l'Écran de Connexion

Lorsque vous ouvrez l'application Desktop ou Mobile de WCPOS, vous verrez l'écran de Connexion. C'est ici que vous gérez vos connexions de magasin et les connexions des utilisateurs.

{/* Capture d'écran : connexion-ecran.png */}

## Ajout d'un Nouveau Magasin

1. Saisissez l'URL de votre boutique WooCommerce dans le champ de texte (par exemple, `https://maboutique.com`)
2. Cliquez sur **Connecter**
3. Vous serez redirigé pour vous connecter avec vos identifiants WordPress
4. Après une connexion réussie, vous serez renvoyé à l'application

## Plusieurs Magasins

Vous pouvez vous connecter à autant de boutiques WooCommerce que nécessaire. Chaque magasin apparaît comme une carte distincte sur l'écran de Connexion, affichant :

- **Nom du magasin** et favicon
- **URL du magasin**
- **Utilisateurs connectés** pour ce magasin

Cela est utile si vous gérez plusieurs emplacements ou entreprises.

## Plusieurs Utilisateurs par Magasin

Chaque magasin peut avoir plusieurs utilisateurs connectés simultanément. Cela est utile pour :

- **Changements de poste** - Un nouveau caissier peut se connecter avant que le précédent ne se déconnecte
- **Configurations multi-caisses** - Différents caissiers sur différents appareils
- **Changement rapide** - Changer facilement entre les comptes utilisateurs

### Ajout d'un Utilisateur

Cliquez sur le bouton <Icon name="plus" /> à côté de "Utilisateurs connectés" pour ajouter un autre utilisateur à ce magasin.

### Changement d'Utilisateur

Cliquez sur un badge utilisateur (par exemple, "Brenda") pour ouvrir le POS en tant que cet utilisateur.

### Suppression d'un Utilisateur

Cliquez sur le **×** sur le badge utilisateur pour déconnecter cet utilisateur du magasin.

## Suppression d'un Magasin

Cliquez sur le bouton **×** rouge sur la carte du magasin pour le supprimer de votre liste. Cela déconnecte tous les utilisateurs et supprime la connexion du magasin de l'application.

## Magasin de Démonstration

En bas de l'écran, vous pourrez voir un lien "Entrer dans le Magasin de Démonstration". Cela vous connecte à un magasin WooCommerce de démonstration pour essayer les fonctionnalités de WCPOS sans affecter les données de votre propre magasin.

## Dépannage

### Impossible de se connecter à mon magasin ?

- Assurez-vous que le plugin WCPOS est installé et activé sur votre site WordPress
- Vérifiez que vous utilisez la bonne URL (incluez `https://` si votre site utilise SSL)
- Vérifiez que l'API REST WooCommerce est accessible
- Vérifiez que votre compte utilisateur a les autorisations d'accès au POS

### La connexion échoue en permanence ?

- Essayez d'abord d'accéder à `votredomaine.com/pos` dans un navigateur web pour vérifier que le plugin fonctionne
- Consultez les journaux d'erreurs de votre site
- Assurez-vous que votre serveur répond aux [configurations minimales requises](/getting-started/installation)
