Cet écran n'est affiché que dans les applications Bureau et Mobile. Les utilisateurs Web accèdent au POS directement à `yourdomain.com/pos` et se connectent avec leurs identifiants WordPress.

## Vue d'ensemble de l'Écran de Connexion {#connect-screen-overview}

Lorsque vous ouvrez l'application WCPOS Bureau ou Mobile, vous verrez l'écran de Connexion. C'est ici que vous gérez vos connexions de magasin et les connexions des utilisateurs.

{/* Capture d'écran : connect-screen.png */}

## Ajouter un Nouveau Magasin {#adding-a-new-store}

1. Entrez l'URL de votre magasin WooCommerce dans le champ de texte (par exemple, `https://mystore.com`)
2. Cliquez sur **Connexion**
3. Vous serez redirigé pour vous connecter avec vos identifiants WordPress
4. Après une connexion réussie, vous serez renvoyé à l'application

## Magasins Multiples {#multiple-stores}

Vous pouvez vous connecter à autant de magasins WooCommerce que vous le souhaitez. Chaque magasin apparaît comme une carte distincte sur l'écran de Connexion, affichant :

- **Nom du magasin** et favicon
- **URL du magasin**
- **Utilisateurs connectés** pour ce magasin

Ceci est utile si vous gérez plusieurs emplacements ou entreprises.

## Utilisateurs Multiples par Magasin {#multiple-users-per-store}

Chaque magasin peut avoir plusieurs utilisateurs connectés simultanément. Cela est utile pour :

- **Changements de poste** - Un nouveau caissier peut se connecter avant que le précédent ne se déconnecte
- **Configurations multi-enregistreurs** - Différents caissiers sur différents appareils
- **Changement rapide** - Passer facilement d'un compte utilisateur à un autre

### Ajouter un Utilisateur {#adding-a-user}

Cliquez sur le bouton <Icon name="plus" /> à côté de "Utilisateurs connectés" pour ajouter un autre utilisateur à ce magasin.

### Changer d'Utilisateurs {#switching-users}

Cliquez sur un insigne d'utilisateur (par exemple, "Brenda") pour ouvrir le POS en tant que cet utilisateur.

### Retirer un Utilisateur {#removing-a-user}

Cliquez sur le **×** sur un insigne d'utilisateur pour déconnecter cet utilisateur du magasin.

## Supprimer un Magasin {#removing-a-store}

Cliquez sur le bouton rouge **×** sur la carte du magasin pour le retirer de votre liste. Cela déconnecte tous les utilisateurs et supprime la connexion au magasin de l'application.

## Magasin Démo {#demo-store}

En bas de l'écran, vous pouvez voir un lien "Entrer dans le Magasin Démo". Cela vous connecte à un magasin WooCommerce démo pour essayer les fonctionnalités de WCPOS sans affecter vos propres données de magasin.

## Dépannage {#troubleshooting}

<Accordion>
  <AccordionItem question="Impossible de me connecter à mon magasin ?">

  - Assurez-vous que le plugin WCPOS est installé et activé sur votre site WordPress
  - Vérifiez que vous utilisez la bonne URL (incluez `https://` si votre site utilise SSL)
  - Vérifiez que l'API REST WooCommerce est accessible
  - Assurez-vous que votre compte utilisateur a les autorisations d'accès au POS

  </AccordionItem>

  <AccordionItem question="La connexion échoue constamment ?">

  - Essayez d'accéder à `yourdomain.com/pos` dans un navigateur web d'abord pour vérifier que le plugin fonctionne
  - Consultez les journaux d'erreurs de votre site pour d'éventuels problèmes
  - Assurez-vous que votre serveur répond aux [exigences minimales](/getting-started/installation)

  </AccordionItem>
</Accordion>
