Pour garantir le bon fonctionnement et des performances optimales du plugin WCPOS, veuillez vous assurer que votre environnement répond aux exigences minimales suivantes :

## Exigences du serveur {#server-requirements}

- **WordPress :** Version 5.6 ou supérieure
- **WooCommerce :** Version 5.3 ou supérieure
- **PHP :** Version 7.2 ou supérieure

Le respect de ces exigences minimales garantit la compatibilité et la stabilité entre le plugin WCPOS et votre environnement WordPress. Cela garantit également que vous pouvez bénéficier des dernières fonctionnalités et améliorations fournies par le plugin.

## Exigences du navigateur {#browser-requirements}

- **Navigateur web moderne :** Le plugin WCPOS est conçu pour fonctionner avec des navigateurs web modernes, y compris Google Chrome, Mozilla Firefox, Safari et Microsoft Edge. Veuillez vous assurer que vous utilisez la dernière version stable de votre navigateur préféré.
- **JavaScript activé :** Le plugin WCPOS repose sur JavaScript pour ses fonctionnalités interactives. Veuillez vous assurer que JavaScript est activé dans les paramètres de votre navigateur.
- **Pas de mode navigation privée :** Le navigateur ne doit pas être en mode navigation privée, car cela peut restreindre l'accès à la base de données IndexedDB (IDB) du navigateur. Le mode navigation privée peut empêcher le bon fonctionnement de certaines fonctionnalités du plugin WCPOS.

## Installation du plugin {#installing-the-plugin}

1. Allez dans `WP Admin > Plugins > Ajouter nouveau`
2. Recherchez "WooCommerce POS"
3. Cliquez sur **Installer maintenant**, puis **Activer**

Alternativement, vous pouvez télécharger le plugin depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) et le télécharger manuellement.

## Accéder au POS {#accessing-the-pos}

Après l'activation, vous pouvez accéder au POS à :

- **Web :** `yourdomain.com/pos`
- **Bureau :** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile :** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

En respectant les exigences minimales et en utilisant un navigateur web compatible, vous pouvez avoir une expérience fluide avec le plugin WCPOS et exploiter tout son potentiel pour gérer efficacement vos opérations de point de vente.
