Pour garantir le bon fonctionnement et des performances optimales du plugin WCPOS, assurez-vous que votre environnement répond aux exigences minimales suivantes :

## Exigences serveur

- **WordPress :** Version 5.6 ou supérieure
- **WooCommerce :** Version 5.3 ou supérieure
- **PHP :** Version 7.2 ou supérieure

Le respect de ces exigences minimales garantit la compatibilité et la stabilité entre le plugin WCPOS et votre environnement WordPress. Cela vous permet également de bénéficier des dernières fonctionnalités et améliorations fournies par le plugin.

## Exigences du navigateur

- **Navigateur web moderne :** Le plugin WCPOS est conçu pour fonctionner avec des navigateurs web modernes, notamment Google Chrome, Mozilla Firefox, Safari et Microsoft Edge. Assurez-vous d'utiliser la dernière version stable de votre navigateur préféré.
- **JavaScript activé :** Le plugin WCPOS repose sur JavaScript pour ses fonctionnalités interactives. Veillez à ce que JavaScript soit activé dans les paramètres de votre navigateur.
- **Pas de mode de navigation privée :** Le navigateur ne doit pas être en mode de navigation privée, car cela peut restreindre l'accès à la base de données IndexedDB (IDB) du navigateur. Le mode de navigation privée peut empêcher le bon fonctionnement de certaines fonctionnalités du plugin WCPOS.

## Installation du plugin

1. Accédez à `WP Admin > Plugins > Ajouter`
2. Recherchez "WooCommerce POS"
3. Cliquez sur **Installer maintenant**, puis **Activer**

Vous pouvez également télécharger le plugin depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) et le téléverser manuellement.

## Accès au POS

Après l'activation, vous pouvez accéder au POS à :

- **Web :** `votre-domaine.com/pos`
- **Bureau/Mobile :** Téléchargez l'application depuis [wcpos.com](https://wcpos.com) et connectez-vous à votre boutique

En respectant les exigences minimales et en utilisant un navigateur web compatible, vous pourrez profiter d'une expérience fluide avec le plugin WCPOS et exploiter pleinement son potentiel pour gérer efficacement vos opérations de point de vente.
