Pour garantir le bon fonctionnement et les performances optimales de l'extension WCPOS, veuillez vous assurer que votre environnement répond aux exigences minimales suivantes :

## Configuration requise du serveur {#server-requirements}

L'extension WCPOS s'exécute sur votre site WordPress, qui doit respecter ces minimums :

| Composant   | Version minimale | Notes           |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x recommandée |

Le respect de ces minimums garantit la compatibilité et la stabilité avec votre environnement WordPress, ainsi que l'accès aux dernières fonctionnalités de WCPOS.

## Configuration requise du navigateur {#browser-requirements}

La version web du POS fonctionne dans tout navigateur moderne. Utilisez la dernière version stable :

| Navigateur      | Version prise en charge |
| --------------- | ----------------- |
| Google Chrome   | Dernière version stable     |
| Mozilla Firefox | Dernière version stable     |
| Safari          | Dernière version stable     |
| Microsoft Edge  | Dernière version stable     |

:::warning Évitez le mode privé / navigation privée
La navigation privée peut bloquer l'accès à la base de données IndexedDB du navigateur, que WCPOS utilise pour stocker les données hors ligne — certaines fonctionnalités ne fonctionneront pas correctement. JavaScript doit également être activé.
:::

## Configuration requise de l'application {#app-requirements}

WCPOS est également disponible en tant qu'application de bureau et mobile. Chacune est conçue pour les systèmes d'exploitation minimaux suivants :

| Plateforme      | Système d'exploitation minimal | Architectures              |
| --------------- | ------------------------- | -------------------------- |
| Bureau Windows  | Windows 10                | x64, ARM64                 |
| Bureau macOS    | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Android         | Android 10 (niveau d'API 29) | —                       |

:::note Autorisations d'impression
L'impression Bluetooth et réseau nécessite d'accorder à l'application les autorisations Bluetooth et de réseau local pertinentes lorsqu'elles sont demandées.
:::

## Installation du plugin {#installing-the-plugin}

1. Accédez à `WP Admin > Plugins > Add Plugin`
2. Recherchez « wcpos »
3. Cliquez sur **Install Now**, puis sur **Activate**

Vous pouvez également télécharger le plugin depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) et le téléverser manuellement.

## Accéder au point de vente {#accessing-the-pos}

Après l'activation, vous pouvez accéder au point de vente à l'adresse :

- **Web :** `yourdomain.com/pos`
- **Bureau :** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile :** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

En respectant la configuration minimale requise et en utilisant un navigateur web compatible, vous bénéficierez d'une expérience optimale avec le plugin WCPOS et pourrez exploiter pleinement son potentiel pour gérer efficacement vos opérations de point de vente.
