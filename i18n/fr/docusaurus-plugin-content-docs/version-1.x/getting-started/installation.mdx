Pour garantir le bon fonctionnement et les performances optimales de l'extension WCPOS, veuillez vous assurer que votre environnement répond aux exigences minimales suivantes :

## Configuration requise du serveur {#server-requirements}

- **WordPress :** Version 5.6 ou supérieure
- **WooCommerce :** Version 5.3 ou supérieure
- **PHP :** Version 7.4 ou supérieure (8.x recommandée)

Le respect de ces exigences minimales garantit la compatibilité et la stabilité entre l'extension WCPOS et votre environnement WordPress. Cela vous permet également de bénéficier des dernières fonctionnalités et améliorations apportées par l'extension.

## Configuration requise du navigateur {#browser-requirements}

- **Navigateur web moderne :** L'extension WCPOS est conçue pour fonctionner avec les navigateurs web modernes, notamment Google Chrome, Mozilla Firefox, Safari et Microsoft Edge. Veuillez vous assurer d'utiliser la dernière version stable de votre navigateur préféré.
- **JavaScript activé :** L'extension WCPOS s'appuie sur JavaScript pour ses fonctionnalités interactives. Veuillez vous assurer que JavaScript est activé dans les paramètres de votre navigateur.
- **Pas de navigation privée :** le navigateur ne doit pas être en mode de navigation privée, car cela peut restreindre l'accès à la base de données IndexedDB (IDB) du navigateur. Le mode de navigation privée peut empêcher le bon fonctionnement de certaines fonctionnalités du plugin WCPOS.

## Installation du plugin {#installing-the-plugin}

1. Accédez à `WP Admin > Extensions > Add Plugin`
2. Recherchez « wcpos »
3. Cliquez sur **Installer maintenant**, puis sur **Activer**

Vous pouvez également télécharger le plugin depuis [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) et le téléverser manuellement.

## Accéder au point de vente {#accessing-the-pos}

Après l'activation, vous pouvez accéder au point de vente à l'adresse :

- **Web :** `yourdomain.com/pos`
- **Bureau :** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile :** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

En respectant la configuration minimale requise et en utilisant un navigateur web compatible, vous bénéficierez d'une expérience optimale avec le plugin WCPOS et pourrez exploiter pleinement son potentiel pour gérer efficacement vos opérations de point de vente.
