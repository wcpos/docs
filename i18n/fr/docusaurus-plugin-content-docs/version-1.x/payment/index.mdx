WCPOS prend en charge plusieurs méthodes de paiement pour s'adapter aux différents besoins des entreprises.

## Méthodes de Paiement par Défaut

La version gratuite inclut deux passerelles de paiement intégrées :

### Espèces

- Saisir le montant tender
- Calcul automatique de la monnaie
- Aucune configuration supplémentaire requise

### Carte

- À utiliser avec des terminaux de carte externes
- Il suffit de marquer le paiement comme complet après traitement sur votre terminal
- Aucune intégration directe requise

## Passerelles de Paiement Supplémentaires (Pro)

:::info Fonction Pro
Les passerelles de paiement supplémentaires nécessitent [WCPOS Pro](/getting-started/pro-license).
:::

Avec Pro, vous pouvez activer des passerelles de paiement WooCommerce supplémentaires dans le processus de paiement du POS :

- **Stripe Terminal** - Intégration directe avec les lecteurs de cartes Stripe
- **SumUp Terminal** - Intégration avec les lecteurs de cartes SumUp
- **Toute Passerelle WooCommerce** - Activez toute passerelle installée sur votre boutique

## Passerelles Personnalisées

Créez vos propres intégrations de paiement en utilisant le système des [Passerelles Personnalisées](/payment/custom-gateways).
