# Passerelles de paiement {#payment-gateways}

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent les fonctionnalités de votre système de point de vente. Ces passerelles sont conçues spécifiquement pour les environnements de point de vente et offrent une intégration transparente avec divers moyens de paiement et services.

## Passerelles personnalisées disponibles {#available-custom-gateways}

### Passerelles pour terminaux physiques {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accepter les paiements à l'aide de lecteurs physiques Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Traiter les paiements via les lecteurs de cartes SumUp
- **[Square Terminal](./square-terminal)** - Encaisser les paiements en personne sur les appareils Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Accepter les paiements par carte en personne sur un terminal PayPal Reader (Zettle)

### Passerelles de paiement mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accepter les paiements par téléphone via QR code ou notification push

### Passerelles de paiement numériques {#digital-payment-gateways}

- **[Facture par e-mail](./email-invoice)** - Envoyer des factures de paiement aux clients par e-mail
- **[Paiement en ligne](./web-checkout)** - Rediriger les clients pour finaliser le paiement en ligne

## Créer votre propre passerelle {#creating-your-own-gateway}

Si vous avez besoin d'une passerelle de paiement personnalisée pour vos besoins spécifiques, vous pouvez utiliser notre **[Modèle de passerelle](/reference/gateway-template)** pour créer votre propre passerelle de paiement WCPOS.

## Aperçu de l'installation {#installation-overview}

Toutes les passerelles personnalisées suivent un processus d'installation similaire :

1. **Télécharger** la dernière version depuis le dépôt GitHub correspondant
2. **Installer** l'extension via `WP Admin > Extensions > Ajouter > Téléverser une extension`
3. **Configurer** les réglages de la passerelle dans `WP Admin > WooCommerce > Réglages > Paiements`
4. **Activer** la passerelle dans `WP Admin > POS > Réglages > Validation de la commande`
