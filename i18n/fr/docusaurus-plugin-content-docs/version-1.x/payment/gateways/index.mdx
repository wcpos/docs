# Passerelles de paiement {#payment-gateways}

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent les fonctionnalités de votre système de point de vente. Ces passerelles sont conçues spécifiquement pour les environnements de point de vente et assurent une intégration fluide avec différents moyens et services de paiement.

## Passerelles personnalisées disponibles {#available-custom-gateways}

### Passerelles de terminaux matériels {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acceptez les paiements à l’aide de lecteurs matériels Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Traitez les paiements via les lecteurs de cartes SumUp
- **[Square Terminal](./square-terminal)** - Encaissez les paiements en personne sur les appareils Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Acceptez les paiements par carte en personne sur un terminal PayPal Reader (Zettle)
- **[Mollie Terminal](./mollie-terminal)** - Acceptez les paiements en personne sur les appareils Mollie Terminal

### Passerelles de paiement mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acceptez les paiements par téléphone via un code QR ou une notification push

### Passerelles de paiement numérique {#digital-payment-gateways}

- **[Facture par e-mail](./email-invoice)** - Envoyez des factures de paiement aux clients par e-mail
- **[Paiement en ligne](./web-checkout)** - Redirigez les clients pour finaliser les paiements en ligne

## Créer votre propre passerelle {#creating-your-own-gateway}

Si une passerelle de paiement personnalisée est nécessaire pour répondre à des besoins spécifiques, utilisez notre **[modèle de passerelle](/reference/gateway-template)** pour créer votre propre passerelle de paiement WCPOS.

## Présentation de l’installation {#installation-overview}

Toutes les passerelles personnalisées suivent un processus d’installation similaire :

1. **Téléchargez** la dernière version depuis le dépôt GitHub correspondant
2. **Installez** l’extension via `WP Admin > Extensions > Ajouter > Téléverser une extension`
3. **Configurez** les réglages de la passerelle dans `WP Admin > WooCommerce > Réglages > Paiements`
4. **Activez** la passerelle dans `WP Admin > POS > Réglages > Validation de la commande`
