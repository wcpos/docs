# Passerelle de Paiement Web

La passerelle de paiement web permet aux clients de terminer leurs transactions de point de vente (POS) en utilisant le système de caisse de votre boutique en ligne. Cette passerelle crée un pont transparent entre votre point de vente physique et les méthodes de paiement en ligne, parfaite pour les situations où les clients préfèrent payer en utilisant leurs propres appareils ou lorsque vous avez besoin d'accéder à des méthodes de paiement exclusivement web.

## Fonctionnalités

- **Intégration Transparente** : Lien direct du POS à la caisse web
- **Toutes les Méthodes de Paiement** : Accès à toutes les passerelles de paiement activées de votre boutique en ligne
- **Contrôle Client** : Les clients peuvent payer en utilisant leurs propres appareils et méthodes préférées
- **Synchronisation des Commandes** : Mises à jour automatiques des commandes entre le POS et la boutique en ligne
- **Workflow Flexible** : Terminez les transactions en ligne tout en conservant le workflow du POS

## Installation

### Étape 1 : Téléchargement et Installation

1. Visitez la [page des versions de la Passerelle de Paiement Web](https://github.com/wcpos/web-checkout-gateway/releases)
2. Téléchargez le dernier fichier **woocommerce-pos-web-checkout-gateway.zip**
3. Dans votre administration WordPress, allez à `Plugins > Ajouter > Téléverser un plugin`
4. Téléversez le fichier zip et activez le plugin

### Étape 2 : Activation dans WooCommerce POS

1. Allez à `Admin WP > WooCommerce POS > Paramètres > Caisse`
2. Trouvez la **Passerelle de Paiement Web WooCommerce POS** dans la liste
3. Activez la passerelle pour une utilisation dans le POS
4. Enregistrez vos paramètres

:::note
Cette passerelle utilise vos méthodes de paiement WooCommerce existantes, assurez-vous d'avoir au moins une passerelle de paiement web correctement configurée (Stripe, PayPal, etc.).
:::
