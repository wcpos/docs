# Passerelle de Terminal SumUp {#sumup-terminal-gateway}

La passerelle Terminal SumUp vous permet d'accepter les paiements par carte en utilisant les lecteurs de cartes SumUp directement au sein de WooCommerce POS. Cette passerelle offre une intégration transparente avec le système de traitement des paiements SumUp pour les transactions en personne.

## Installation

<Steps>
  <StepCard number="1" title="Télécharger et Installer">

1. Visitez la [page des versions de SumUp Terminal pour WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Téléchargez le fichier **sumup-terminal-for-woocommerce.zip** le plus récent
3. Dans votre admin WordPress, allez à `Extensions > Ajouter > Télécharger une Extension`
4. Téléchargez le fichier zip et activez l'extension

  </StepCard>
  <StepCard number="2" title="Configurer les Paramètres de SumUp">

1. Naviguez vers `WP Admin > WooCommerce > Paramètres > Paiements`
2. Trouvez **Terminal SumUp** dans la liste des méthodes de paiement
3. Cliquez sur **Terminal SumUp** pour accéder aux paramètres
4. Entrez votre **Clé API SumUp** (disponible depuis votre tableau de bord marchand SumUp)
5. Enregistrez les paramètres

:::note
Vous n'avez pas besoin d'activer la passerelle Terminal SumUp dans les paramètres de WooCommerce. Elle sera activée spécifiquement pour le POS dans une étape ultérieure.
:::

  </StepCard>
  <StepCard number="3" title="Appairer Votre Terminal SumUp">

1. Sur la même page de paramètres, localisez la section **Appairer le Lecteur**
2. Sur votre appareil SumUp, accédez à l'écran d'apairage pour afficher le code d'apairage
3. Entrez le code d'apairage affiché sur votre appareil SumUp
4. Cliquez sur **"Appairer le Lecteur"** pour établir la connexion
5. Attendez la confirmation que le lecteur a été appairé avec succès

:::warning Important
Le lecteur doit être appairé avec succès avant que vous puissiez traiter des paiements. Assurez-vous que le processus d'apairage est terminé avant de continuer.
:::

  </StepCard>
  <StepCard number="4" title="Activer dans WooCommerce POS" isLast>

1. Allez à `WP Admin > WooCommerce POS > Paramètres > Paiement`
2. Trouvez la passerelle **Terminal SumUp** dans la liste
3. Activez la passerelle pour utilisation dans le POS
4. Enregistrez vos paramètres

  </StepCard>
</Steps>

## Utilisation {#usage}

### Traitement des Paiements {#processing-payments}

1. **Ajouter des Articles** : Ajoutez des produits à votre panier dans le POS
2. **Sélectionner la Passerelle** : Choisissez "Terminal SumUp" comme méthode de paiement
3. **Démarrer le Paiement** : Cliquez pour initier un nouveau paiement sur votre appareil SumUp
4. **Paiement du Client** : Le client finalise le paiement sur le terminal SumUp
5. **Complétion Automatique** : La commande est automatiquement complétée lorsqu'un paiement est détecté

### Contrôles de Paiement {#payment-controls}

Lors de l'utilisation de la passerelle Terminal SumUp, vous avez les options suivantes :

- **Démarrer un Nouveau Paiement** : Initier une demande de paiement sur le terminal connecté
- **Annuler le Paiement** : Annuler un paiement en cours
- **Statut du Paiement** : Surveiller le statut actuel du processus de paiement

### Gestion des Commandes {#order-management}

- **Mises à Jour Automatiques** : Les commandes sont automatiquement marquées comme complètes après un paiement réussi
- **Suivi des Paiements** : Tous les détails de paiement sont enregistrés dans les notes de commande
- **Génération de Reçus** : Les reçus standard du POS sont générés après des paiements réussis

## Compatibilité Matérielle {#hardware-compatibility}

:::important Exigences de Connectivité
Cette mise en œuvre utilise le SDK JavaScript de SumUp, ce qui signifie qu'elle fonctionne via des applications web mais nécessite **uniquement des terminaux connectés à Internet**. Les terminaux Bluetooth ne sont pas actuellement supportés.
:::

### Terminaux Supportés (Connectés à Internet) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - Terminal connecté par WiFi/Ethernet

### Terminaux Non Supportés (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - Bluetooth uniquement
- **SumUp 3G** ❌ - Bluetooth uniquement

:::note Support Futur
Le support des terminaux Bluetooth sera ajouté aux applications iOS et Android dans **Q1 2026**. Cela permettra le support de tous les terminaux certifiés SumUp, y compris les modèles Air et 3G.
:::

## Dépannage {#troubleshooting}

### Problèmes Courants {#common-issues}

<Accordion>
  <AccordionItem question="Le lecteur ne paire pas">

  - Assurez-vous d'utiliser un terminal connecté à Internet pris en charge (Solo uniquement)
  - Vérifiez que votre appareil SumUp Solo est connecté au WiFi et en ligne
  - Assurez-vous que l'appareil est en mode d'appairage et que le code d'apairage est saisi correctement
  - Confirmez que votre compte SumUp a l'accès API activé
  - Assurez-vous d'une connexion Internet stable lors de l'apairage

  </AccordionItem>

  <AccordionItem question="Le paiement ne se traite pas">

  - Vérifiez que le lecteur est correctement appairé
  - Vérifiez que votre compte SumUp est actif et en bon état
  - Assurez-vous que l'appareil dispose de suffisamment de batterie et de connectivité
  - Confirmez que la clé API est correcte et active

  </AccordionItem>

  <AccordionItem question="La commande ne se complète pas">

  - Vérifiez la stabilité de la connexion Internet
  - Vérifiez les paramètres de webhook dans votre compte SumUp
  - Assurez-vous que le paiement a été réussi sur le terminal SumUp
  - Vérifiez les journaux d'erreurs WordPress pour les problèmes de communication API

  </AccordionItem>
</Accordion>

### Limites de Taux API {#api-rate-limits}

La passerelle est optimisée pour réduire les appels API et éviter la limitation de taux :

- La logique de nouvelle tentative automatique gère les défaillances temporaires
