# Passerelle Stripe Terminal {#stripe-terminal-gateway}

La passerelle Stripe Terminal vous permet d'accepter des paiements en personne en utilisant des lecteurs de matériel Stripe Terminal directement dans WCPOS. Cette passerelle prend en charge à la fois les lecteurs de cartes physiques et le mode simulateur pour les tests.

## Installation

<Steps>
  <StepCard number="1" title="Télécharger et Installer">

1. Visitez la [page de versions de Stripe Terminal pour WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Téléchargez le fichier **stripe-terminal-for-woocommerce.zip** le plus récent
3. Dans votre administration WordPress, allez à `Extensions > Ajouter > Téléverser une Extension`
4. Téléversez le fichier zip et activez l'extension

  </StepCard>
  <StepCard number="2" title="Configurer les Paramètres Stripe">

1. Accédez à `WP Admin > WooCommerce > Paramètres > Paiements`
2. Trouvez **Stripe Terminal** dans la liste des méthodes de paiement
3. Cliquez sur **Stripe Terminal** pour accéder aux paramètres
4. Entrez votre **Clé Secrète Stripe** (vous pouvez l'obtenir depuis votre tableau de bord Stripe)
5. Enregistrez les paramètres

:::note
Vous n'avez pas besoin d'activer la passerelle Stripe Terminal dans les paramètres de WooCommerce. Elle sera activée spécifiquement pour le POS à l'étape suivante.
:::

  </StepCard>
  <StepCard number="3" title="Activer dans WCPOS" isLast>

1. Allez à `WP Admin > POS > Paramètres > Paiement`
2. Trouvez la passerelle **Stripe Terminal** dans la liste
3. Activez la passerelle pour une utilisation dans le POS
4. Enregistrez vos paramètres

  </StepCard>
</Steps>

## Utilisation {#usage}

### Connexion d'un Lecteur {#connecting-a-reader}

Lorsque vous sélectionnez la passerelle Stripe Terminal lors du paiement dans le POS :

1. **Choisissez la Méthode de Connexion** : Vous pouvez soit connecter un lecteur physique, soit utiliser le simulateur
2. **Lecteur Physique** : Suivez les instructions à l'écran pour connecter votre appareil Stripe Terminal
3. **Simulateur** : Sélectionnez le mode simulateur pour tester divers scénarios de paiement sans matériel

### Traitement des Paiements {#processing-payments}

1. **Ajouter des Articles** : Ajoutez des produits à votre panier dans le POS
2. **Sélectionner la Passerelle** : Choisissez "Stripe Terminal" comme méthode de paiement
3. **Connecter le Lecteur** : Connectez votre lecteur ou choisissez le mode simulateur
4. **Traiter le Paiement** : Suivez les instructions pour terminer la transaction
5. **Confirmation** : La commande sera automatiquement terminée après paiement réussi

### Tests avec le Simulateur {#testing-with-simulator}

Le simulateur vous permet de tester diverses méthodes de paiement et scénarios :

- **Paiements par Carte** : Testez différents types de cartes (Visa, Mastercard, etc.)
- **Paiements Sans Contact** : Simulez des transactions de paiement sans contact
- **Scénarios d'Erreur** : Testez les paiements refusés et d'autres conditions d'erreur
- **Différents Montants** : Testez divers montants de transaction

## Compatibilité Matérielle {#hardware-compatibility}

:::important Exigences de Connectivité
Cette mise en œuvre utilise le SDK JavaScript de Stripe, ce qui signifie qu'elle fonctionne via des applications web mais nécessite **uniquement des terminaux connectés à Internet**. Les terminaux Bluetooth ne sont pas actuellement pris en charge.
:::

### Terminaux Supportés (Connectés à Internet) {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Terminal connecté par Ethernet/WiFi
- **WisePOS E** ✅ - Terminal connecté par WiFi

### Terminaux Non Supportés (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Bluetooth uniquement
- **BBPOS WisePad 3** ❌ - Bluetooth uniquement  
- **Verifone P400** ❌ - Bluetooth uniquement

:::note Support Futur
La prise en charge des terminaux Bluetooth sera ajoutée aux applications iOS et Android au **T1 2026**. Cela permettra de prendre en charge tous les lecteurs certifiés Stripe Terminal, y compris le M2 et le WisePad 3.
:::

### Problèmes Communs {#common-issues}

<Accordion>
  <AccordionItem question="Le lecteur ne se connecte pas">

  - Assurez-vous que vous utilisez un terminal connecté à Internet pris en charge (S700 ou WisePOS E)
  - Vérifiez que le terminal est connecté à WiFi/Ethernet et en ligne
  - Vérifiez que votre compte Stripe a Terminal activé
  - Confirmez que le terminal est enregistré dans votre tableau de bord Stripe

  </AccordionItem>

  <AccordionItem question="Paiement refusé">

  - Vérifiez que votre compte Stripe est actif et en règle
  - Vérifiez que la carte utilisée est valide
  - Assurez-vous que des fonds suffisants sont disponibles

  </AccordionItem>

  <AccordionItem question="Erreurs de certificat SSL">

  - Stripe Terminal nécessite HTTPS - assurez-vous que votre site dispose d'un certificat SSL valide
  - Vérifiez que votre certificat SSL est correctement configuré

  </AccordionItem>
</Accordion>
