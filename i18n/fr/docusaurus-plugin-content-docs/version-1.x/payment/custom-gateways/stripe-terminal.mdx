# Passerelle Stripe Terminal

La passerelle Stripe Terminal vous permet d'accepter des paiements en personne en utilisant les lecteurs matériels Stripe Terminal directement dans WooCommerce POS. Cette passerelle prend en charge à la fois les lecteurs de cartes physiques et le mode simulateur pour les tests.

## Caractéristiques

- **Intégration Matériel** : Connectez des lecteurs physiques Stripe Terminal via une connexion Internet
- **Mode Simulateur** : Testez les paiements sans matériel en utilisant le simulateur de Stripe
- **Méthodes de Paiement Multiples** : Prise en charge de divers types de cartes et méthodes de paiement
- **Traitement en Temps Réel** : Traitement et confirmation immédiats des paiements
- **Transactions Sécurisées** : Traitement des paiements conforme à la norme PCI via Stripe
- **SDK Basé sur le Web** : Utilise le SDK JavaScript de Stripe pour la compatibilité avec les applications Web

## Installation

### Étape 1 : Télécharger et Installer

1. Visitez la [page des versions de Stripe Terminal pour WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Téléchargez le fichier **stripe-terminal-for-woocommerce.zip** le plus récent
3. Dans votre administration WordPress, accédez à `Extensions > Ajouter > Télécharger l'extension`
4. Téléchargez le fichier zip et activez l'extension

### Étape 2 : Configurer les Paramètres Stripe

1. Allez dans `WP Admin > WooCommerce > Paramètres > Paiements`
2. Trouvez **Stripe Terminal** dans la liste des méthodes de paiement
3. Cliquez sur **Stripe Terminal** pour accéder aux paramètres
4. Entrez votre **Clé Secrète Stripe** (vous pouvez l'obtenir depuis votre tableau de bord Stripe)
5. Enregistrez les paramètres

:::note
Vous n'avez pas besoin d'activer la passerelle Stripe Terminal dans les paramètres WooCommerce. Elle sera spécifiquement activée pour le POS à l'étape suivante.
:::

### Étape 3 : Activer dans WooCommerce POS

1. Allez dans `WP Admin > WooCommerce POS > Paramètres > Paiement`
2. Trouvez la passerelle **Stripe Terminal** dans la liste
3. Activez la passerelle pour l'utiliser dans le POS
4. Enregistrez vos paramètres

## Utilisation

### Connexion d'un Lecteur

Lorsque vous sélectionnez la passerelle Stripe Terminal lors du paiement dans le POS :

1. **Choisissez la Méthode de Connexion** : Vous pouvez soit connecter un lecteur physique, soit utiliser le simulateur
2. **Lecteur Physique** : Suivez les instructions à l'écran pour connecter votre appareil Stripe Terminal
3. **Simulateur** : Sélectionnez le mode simulateur pour tester divers scénarios de paiement sans matériel

### Traitement des Paiements

1. **Ajouter des Articles** : Ajoutez des produits à votre panier dans le POS
2. **Sélectionner la Passerelle** : Choisissez "Stripe Terminal" comme méthode de paiement
3. **Connecter le Lecteur** : Connectez votre lecteur ou choisissez le mode simulateur
4. **Traiter le Paiement** : Suivez les instructions pour compléter la transaction
5. **Confirmation** : La commande sera automatiquement complétée après un paiement réussi

### Tests avec le Simulateur

Le simulateur vous permet de tester différentes méthodes de paiement et scénarios :

- **Paiements par Carte** : Testez différents types de cartes (Visa, Mastercard, etc.)
- **Paiements sans Contact** : Simulez des transactions tap-to-pay
- **Scénarios d'Erreur** : Testez les paiements refusés et d'autres conditions d'erreur
- **Montants Différents** : Testez divers montants de transaction

## Exigences

- **Compte Stripe** : Compte Stripe actif avec Terminal activé
- **Clés API** : Clé secrète Stripe de votre tableau de bord
- **WooCommerce POS** : Version gratuite ou Pro installée et activée
- **HTTPS** : Votre site doit utiliser SSL/HTTPS pour des raisons de sécurité

## Compatibilité Matérielle

:::important Exigences de Connectivité
Cette implémentation utilise le SDK JavaScript de Stripe, ce qui signifie qu'elle fonctionne via des applications Web mais nécessite uniquement **des terminaux connectés à Internet**. Les terminaux Bluetooth ne sont pas actuellement pris en charge.
:::

### Terminaux Pris en Charge (Connectés à Internet)

- **Lecteur Stripe S700** ✅ - Terminal connecté par Ethernet/WiFi
- **WisePOS E** ✅ - Terminal connecté par WiFi

### Terminaux Non Pris en Charge (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Bluetooth uniquement
- **BBPOS WisePad 3** ❌ - Bluetooth uniquement  
- **Verifone P400** ❌ - Bluetooth uniquement

:::note Support Futur
Le support des terminaux Bluetooth sera ajouté aux applications iOS et Android au **T1 2026**. Cela permettra la prise en charge de tous les lecteurs certifiés Stripe Terminal, y compris le M2 et le WisePad 3.
:::

## Dépannage

### Problèmes Courants

**Le Lecteur Ne Se Connecte Pas**
- Assurez-vous que vous utilisez un terminal connecté à Internet pris en charge (S700 ou WisePOS E)
- Vérifiez que le terminal est connecté à WiFi/Ethernet et en ligne
- Vérifiez que votre compte Stripe a le Terminal activé
- Confirmez que le terminal est enregistré dans votre tableau de bord Stripe

**Paiement Refusé**
- Vérifiez que votre compte Stripe est actif et en règle
- Vérifiez que la carte utilisée est valide
- Assurez-vous que des fonds suffisants sont disponibles

**Erreurs de Certificat SSL**
- Stripe Terminal nécessite HTTPS - assurez-vous que votre site dispose d'un certificat SSL valide
- Vérifiez que votre certificat SSL est correctement configuré
