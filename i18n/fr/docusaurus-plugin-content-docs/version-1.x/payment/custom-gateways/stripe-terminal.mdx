# Passerelle Stripe Terminal

La passerelle Stripe Terminal vous permet d'accepter des paiements en personne à l'aide de lecteurs de terminaux Stripe directement dans WooCommerce POS. Cette passerelle prend en charge les lecteurs de cartes physiques et le mode simulateur pour les tests.

## Fonctionnalités

- **Intégration matérielle** : Connectez des lecteurs de terminaux Stripe physiques via une connexion internet
- **Mode Simulateur** : Testez les paiements sans matériel à l'aide du simulateur Stripe
- **Méthodes de Paiement Multiples** : Support de divers types de cartes et méthodes de paiement
- **Traitement en Temps Réel** : Traitement et confirmation de paiement instantanés
- **Transactions Sécurisées** : Traitement de paiement conforme à la norme PCI via Stripe
- **SDK Web** : Utilise le SDK JavaScript de Stripe pour la compatibilité des applications web

## Installation

### Étape 1 : Télécharger et Installer

1. Visitez la [page des versions de Stripe Terminal pour WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Téléchargez le dernier fichier **stripe-terminal-for-woocommerce.zip**
3. Dans votre administration WordPress, allez à `Plugins > Ajouter > Téléverser un plugin`
4. Téléversez le fichier zip et activez le plugin

### Étape 2 : Configurer les Paramètres Stripe

1. Accédez à `Admin WP > WooCommerce > Paramètres > Paiements`
2. Trouvez **Stripe Terminal** dans la liste des méthodes de paiement
3. Cliquez sur **Stripe Terminal** pour accéder aux paramètres
4. Entrez votre **Clé Secrète Stripe** (disponible sur votre Tableau de Bord Stripe)
5. Enregistrez les paramètres

:::note
Vous n'avez pas besoin d'activer la passerelle Stripe Terminal dans les paramètres WooCommerce. Elle sera activée spécifiquement pour le POS à l'étape suivante.
:::

### Étape 3 : Activer dans WooCommerce POS

1. Allez à `Admin WP > WooCommerce POS > Paramètres > Caisse`
2. Trouvez la passerelle **Stripe Terminal** dans la liste
3. Activez la passerelle pour une utilisation dans le POS
4. Enregistrez vos paramètres

## Utilisation

### Connexion d'un Lecteur

Lorsque vous sélectionnez la passerelle Stripe Terminal lors du paiement dans le POS :

1. **Choisir la Méthode de Connexion** : Vous pouvez connecter un lecteur physique ou utiliser le simulateur
2. **Lecteur Physique** : Suivez les instructions à l'écran pour connecter votre appareil Stripe Terminal
3. **Simulateur** : Sélectionnez le mode simulateur pour tester divers scénarios de paiement sans matériel

### Traitement des Paiements

1. **Ajouter des Articles** : Ajoutez des produits à votre panier dans le POS
2. **Sélectionner la Passerelle** : Choisissez "Stripe Terminal" comme méthode de paiement
3. **Connecter le Lecteur** : Connectez votre lecteur ou choisissez le mode simulateur
4. **Traiter le Paiement** : Suivez les invites pour terminer la transaction
5. **Confirmation** : La commande sera automatiquement terminée après un paiement réussi

### Test avec le Simulateur

Le simulateur vous permet de tester divers méthodes de paiement et scénarios :

- **Paiements par Carte** : Testez différents types de cartes (Visa, Mastercard, etc.)
- **Paiements sans Contact** : Simulez des transactions par effleurement
- **Scénarios d'Erreur** : Testez les paiements refusés et autres conditions d'erreur
- **Différents Montants** : Testez divers montants de transaction

## Exigences

- **Compte Stripe** : Compte Stripe actif avec Terminal activé
- **Clés API** : Clé secrète Stripe de votre tableau de bord
- **WooCommerce POS** : Version gratuite ou Pro installée et activée
- **HTTPS** : Votre site doit utiliser SSL/HTTPS pour la sécurité

## Compatibilité Matérielle

:::important Exigences de Connectivité
Cette implémentation utilise le SDK JavaScript de Stripe, ce qui signifie qu'elle fonctionne via des applications web mais nécessite des **terminaux connectés à Internet uniquement**. Les terminaux Bluetooth ne sont actuellement pas pris en charge.
:::

### Terminaux Pris en Charge (Connectés à Internet)

- **Stripe Reader S700** ✅ - Terminal connecté par Ethernet/WiFi
- **WisePOS E** ✅ - Terminal connecté par WiFi

### Terminaux Non Pris en Charge (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Bluetooth uniquement
- **BBPOS WisePad 3** ❌ - Bluetooth uniquement
- **Verifone P400** ❌ - Bluetooth uniquement

:::note Support Futur
Le support des terminaux Bluetooth sera ajouté aux applications iOS et Android au **T1 2026**. Cela permettra la prise en charge de tous les lecteurs certifiés Stripe Terminal, y compris le M2 et le WisePad 3.
:::

## Dépannage

### Problèmes Courants

**Le Lecteur ne se Connecte Pas**
- Assurez-vous d'utiliser un terminal connecté à Internet pris en charge (S700 ou WisePOS E)
- Vérifiez que le terminal est connecté au WiFi/Ethernet et en ligne
- Vérifiez que votre compte Stripe a Terminal activé
- Confirmez que le terminal est enregistré dans votre Tableau de Bord Stripe

**Paiement Refusé**
- Vérifiez que votre compte Stripe est actif et en bon état
- Assurez-vous que la carte utilisée est valide
- Vérifiez la disponibilité des fonds suffisants

**Erreurs de Certificat SSL**
- Stripe Terminal nécessite HTTPS - assurez-vous que votre site a un certificat SSL valide
- Vérifiez que votre certificat SSL est correctement configuré
