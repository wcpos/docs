# Passerelles de Paiement Personnalisées {#custom-payment-gateways}

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent la fonctionnalité de votre système de Point de Vente. Ces passerelles sont conçues spécifiquement pour les environnements POS et offrent une intégration transparente avec divers méthodes et services de paiement.

## Passerelles Personnalisées Disponibles {#available-custom-gateways}

### Passerelles de Terminal Matériel {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accepter des paiements à l'aide des lecteurs de matériel Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Traiter les paiements via des lecteurs de cartes SumUp

### Passerelles de Paiement Mobile {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accepter des paiements par téléphone via QR code ou notification push

### Passerelles de Paiement Numérique

- **[Facture par Email](./email-invoice)** - Envoyer des factures de paiement aux clients par email
- **[Paiement en Ligne](./web-checkout)** - Rediriger les clients pour compléter les paiements en ligne

## Créer Votre Propre Passerelle {#creating-your-own-gateway}

Si vous avez besoin d'une passerelle de paiement personnalisée pour vos besoins spécifiques, vous pouvez utiliser notre **[Modèle de Passerelle](./gateway-template)** pour créer votre propre passerelle de paiement WooCommerce POS.

## Vue D'ensemble de l'Installation {#installation-overview}

Toutes les passerelles personnalisées suivent un processus d'installation similaire :

1. **Téléchargez** la dernière version du dépôt GitHub respectif
2. **Installez** le plugin via `WP Admin > Plugins > Ajouter Nouveau > Télécharger le Plugin`
3. **Configurez** les paramètres de la passerelle dans `WP Admin > WooCommerce > Paramètres > Paiements`
4. **Activez** la passerelle dans `WP Admin > WooCommerce POS > Paramètres > Paiement`
