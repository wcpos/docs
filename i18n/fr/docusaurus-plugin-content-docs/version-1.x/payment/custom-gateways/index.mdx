# Passerelles de Paiement Personnalisées

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent les fonctionnalités de votre système de Point de Vente. Ces passerelles sont conçues spécifiquement pour les environnements POS et offrent une intégration transparente avec différentes méthodes et services de paiement.

## Passerelles Personnalisées Disponibles

### Passerelles de Terminal Matériel

- **[Terminal Stripe](./stripe-terminal)** - Accepter les paiements à l'aide des lecteurs de terminal Stripe
- **[Terminal SumUp](./sumup-terminal)** - Traiter les paiements via des lecteurs de carte SumUp

### Passerelles de Paiement Numériques

- **[Facture par E-mail](./email-invoice)** - Envoyer des factures de paiement aux clients par e-mail
- **[Paiement Web](./web-checkout)** - Rediriger les clients pour terminer les paiements en ligne

## Créer Votre Propre Passerelle

Si vous avez besoin d'une passerelle de paiement personnalisée pour vos besoins spécifiques, vous pouvez utiliser notre **[Modèle de Passerelle](./gateway-template)** pour créer votre propre passerelle de paiement WooCommerce POS.

## Vue d'Ensemble de l'Installation

Toutes les passerelles personnalisées suivent un processus d'installation similaire :

1. **Télécharger** la dernière version depuis le dépôt GitHub respectif
2. **Installer** le plugin via `WP Admin > Plugins > Ajouter > Télécharger un plugin`
3. **Configurer** les paramètres de la passerelle dans `WP Admin > WooCommerce > Réglages > Paiements`
4. **Activer** la passerelle dans `WP Admin > WooCommerce POS > Réglages > Caisse`

## Conditions Requises

- Plugin WooCommerce POS (version gratuite ou pro)
- WordPress avec WooCommerce installé
- Clés API ou identifiants appropriés pour le service de paiement spécifique

## Support

Pour les problèmes liés aux passerelles personnalisées, veuillez visiter le dépôt GitHub correspondant et créer un ticket avec des informations détaillées sur votre problème.
