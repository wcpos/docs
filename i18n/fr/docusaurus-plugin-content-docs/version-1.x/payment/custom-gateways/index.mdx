# Passerelles de paiement personnalisées {#custom-payment-gateways}

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent les fonctionnalités de votre point de vente. Ces passerelles sont conçues spécifiquement pour les environnements de point de vente et offrent une intégration fluide avec divers moyens et services de paiement.

## Passerelles personnalisées disponibles {#available-custom-gateways}

### Passerelles pour terminaux matériels {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acceptez les paiements à l'aide des lecteurs matériels Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Traitez les paiements via les lecteurs de cartes SumUp

### Passerelles de paiement mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acceptez les paiements par téléphone via QR code ou notification push

### Passerelles de paiement numérique {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Envoyez des factures de paiement aux clients par e-mail
- **[Paiement Web](./web-checkout)** - Redirigez les clients pour effectuer leurs paiements en ligne

## Créer votre propre passerelle {#creating-your-own-gateway}

Si vous avez besoin d'une passerelle de paiement personnalisée pour répondre à des exigences spécifiques, vous pouvez utiliser notre **[Modèle de passerelle](./gateway-template)** pour créer votre propre passerelle de paiement WCPOS.

## Aperçu de l'installation {#installation-overview}

Toutes les passerelles personnalisées suivent un processus d'installation similaire :

1. **Téléchargez** la dernière version depuis le dépôt GitHub correspondant
2. **Installez** l'extension via `WP Admin > Extensions > Ajouter > Téléverser une extension`
3. **Configurez** les paramètres de la passerelle dans `WP Admin > WooCommerce > Réglages > Paiements`
4. **Activez** la passerelle dans `WP Admin > POS > Réglages > Validation de la commande`
