# Passerelles de Paiement Personnalisées {#custom-payment-gateways}

WCPOS prend en charge plusieurs passerelles de paiement personnalisées qui étendent la fonctionnalité de votre système de Point de Vente. Ces passerelles sont conçues spécifiquement pour les environnements POS et offrent une intégration transparente avec divers moyens de paiement et services.

## Passerelles Personnalisées Disponibles {#available-custom-gateways}

### Passerelles de Terminaux Matériels {#hardware-terminal-gateways}

- **[Terminal Stripe](./stripe-terminal)** - Accepter les paiements en utilisant les lecteurs de matériel Stripe Terminal
- **[Terminal SumUp](./sumup-terminal)** - Traiter les paiements via les lecteurs de cartes SumUp

### Passerelles de Paiement Numérique {#digital-payment-gateways}

- **[Facture par Email](./email-invoice)** - Envoyer des factures de paiement aux clients via email
- **[Paiement Web](./web-checkout)** - Rediriger les clients pour compléter les paiements en ligne

## Créer Votre Propre Passerelle {#creating-your-own-gateway}

Si vous avez besoin d'une passerelle de paiement personnalisée pour vos exigences spécifiques, vous pouvez utiliser notre **[Modèle de Passerelle](./gateway-template)** pour créer votre propre passerelle de paiement WooCommerce POS.

## Aperçu de l'Installation {#installation-overview}

Toutes les passerelles personnalisées suivent un processus d'installation similaire :

1. **Télécharger** la dernière version depuis le dépôt GitHub respectif
2. **Installer** le plugin via `WP Admin > Plugins > Ajouter Nouveau > Télécharger le Plugin`
3. **Configurer** les paramètres de la passerelle dans `WP Admin > WooCommerce > Paramètres > Paiements`
4. **Activer** la passerelle dans `WP Admin > WooCommerce POS > Paramètres > Checkout`
