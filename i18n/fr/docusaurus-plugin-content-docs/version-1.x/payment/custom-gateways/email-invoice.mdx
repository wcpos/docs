# Passerelle de Facturation par Email

La passerelle de facturation par email vous permet d'envoyer des factures de paiement aux clients directement par email depuis WooCommerce POS. C'est parfait pour les situations où les clients préfèrent payer plus tard ou doivent compléter le paiement en ligne en utilisant les méthodes de paiement de votre boutique en ligne.

## Fonctionnalités

- **Intégration Email**: Envoyez des factures directement depuis l'interface POS
- **Gestion des Clients**: Remplissage automatique de l'email à partir des données client existantes ou ajout de nouvelles adresses email
- **Commandes Invités**: Support pour les clients invités avec mise à jour facultative de l'adresse de facturation
- **Paiement en Ligne**: Les clients reçoivent un lien "Payer cette commande" pour compléter le paiement en ligne
- **Flux de Travail Flexible**: Terminez les commandes en magasin ou autorisez la finalisation du paiement à distance

## Installation

### Étape 1 : Téléchargement et Installation

1. Visitez la [page de publications de la Passerelle de Facturation par Email](https://github.com/wcpos/email-invoice-gateway/releases)
2. Téléchargez le dernier fichier **woocommerce-pos-email-invoice-gateway.zip**
3. Dans votre administration WordPress, allez à `Plugins > Ajouter > Téléverser un plugin`
4. Téléversez le fichier zip et activez le plugin

### Étape 2 : Activation dans WooCommerce POS

1. Allez à `Admin WP > WooCommerce POS > Paramètres > Caisse`
2. Trouvez la **Passerelle de Paiement par Facture WooCommerce POS** dans la liste
3. Activez la passerelle pour une utilisation dans le POS
4. Enregistrez vos paramètres

:::note
Cette passerelle ne nécessite pas de clés API ou de configuration de service externe. Elle utilise votre système d'email WordPress existant et les méthodes de paiement WooCommerce.
:::

## Utilisation

### Envoi de Factures

1. **Ajouter des Articles**: Ajoutez des produits à votre panier dans le POS
2. **Sélectionner la Passerelle**: Choisissez "Facture par Email" comme méthode de paiement
3. **Adresse Email**: 
   - Pour les clients existants : L'email est automatiquement rempli à partir des données client
   - Pour les commandes invités : Saisissez manuellement l'adresse email du client
4. **Optionnel**: Cochez "Enregistrer dans l'adresse de facturation" pour mettre à jour les informations de facturation du client
5. **Envoyer la Facture**: Terminez le processus pour envoyer l'email de facture

### Expérience Client

Lorsque vous envoyez une facture, le client reçoit un email contenant :

- **Détails de la Commande**: Décompte complet des articles, quantités et prix
- **Total de la Commande**: Montant final dû, incluant taxes et frais
- **Lien de Paiement**: Lien direct pour compléter le paiement en ligne
- **Informations de Commande**: Numéro de commande et détails de référence

### Finalisation du Paiement

Les clients peuvent compléter le paiement de deux manières :

1. **Paiement en Ligne**: Cliquer sur le lien "Payer cette commande" dans l'email
2. **Retour en Magasin**: Revenir pour compléter le paiement avec d'autres méthodes de paiement POS

Le lien "Payer cette commande" dirige les clients vers une page de paiement dédiée sur votre site où ils peuvent utiliser n'importe laquelle de vos passerelles de paiement web activées (Stripe, PayPal, etc.).
