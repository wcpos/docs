WCPOS fonctionne avec les périphériques physiques que vous utilisez déjà en caisse. Cette page est le point de départ pour connecter votre matériel — imprimantes de reçus, lecteurs de codes-barres, terminaux de paiement et tiroirs-caisses. Chaque type de périphérique dispose de sa propre page de configuration ici, ou renvoie vers la page où sa configuration est décrite.

:::note Un seul emplacement par appareil
Le hub Matériel renvoie vers la configuration de chaque appareil, mais les étapes détaillées se trouvent à un seul endroit. Le compte et le processus d'appairage d'un lecteur de cartes se trouvent sur sa page Passerelle de paiement ; le réglage d'ouverture d'un tiroir-caisse se trouve sur la page Imprimantes de reçus.
:::
