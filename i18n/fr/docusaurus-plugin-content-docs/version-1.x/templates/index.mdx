Les modèles vous permettent de personnaliser l'apparence et le contenu de diverses sorties de votre système POS. Vous pouvez accéder à l'éditeur de modèles en naviguant vers : `WP Admin > POS > Modèles`

Le système de modèles offre une flexibilité pour la personnalisation :

- **Modèles de reçus** - Personnalisez les reçus imprimés et numériques
- **Modèles de rapports** - Personnalisez les mises en page des rapports _(bientôt disponible)_
- **Modèles d'e-mails** - Personnalisez les notifications par e-mail _(bientôt disponible)_

## Types de modèles

### Modèles par défaut

Détectés automatiquement dans vos fichiers de plugin et de thème. Ceux-ci ne peuvent pas être supprimés mais peuvent être copiés pour créer des versions personnalisées.

### Modèles personnalisés

Modèles créés par l'utilisateur que vous pouvez entièrement personnaliser, éditer et gérer via l'interface d'administration de WordPress.
