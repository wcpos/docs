Les modèles vous permettent de personnaliser l'apparence et le contenu de diverses sorties de votre système de point de vente. Vous pouvez accéder à l'éditeur de modèles en naviguant vers : `WP Admin > POS > Modèles`

Le système de modèles offre une flexibilité pour la personnalisation :

- **Modèles de Reçus** - Personnalisez les reçus imprimés et numériques
- **Modèles de Rapports** - Personnalisez les mises en page de rapports _(à venir bientôt)_
- **Modèles d'Emails** - Personnalisez les notifications par email _(à venir bientôt)_

## Types de Modèles

### Modèles Par Défaut

Détectés automatiquement à partir de vos fichiers de plugins et de thèmes. Ceux-ci ne peuvent pas être supprimés mais peuvent être copiés pour créer des versions personnalisées.

### Modèles Personnalisés

Modèles créés par l'utilisateur que vous pouvez entièrement personnaliser, modifier et gérer via l'interface d'administration de WordPress.
