Les modèles vous permettent de personnaliser l'apparence et le contenu de différentes sorties de votre système de caisse. Vous pouvez accéder à l'éditeur de modèles en navigant vers : `WP Admin > POS > Modèles`

Le système de modèles offre une flexibilité pour personnaliser :

- **Modèles de reçus** - Personnaliser les reçus imprimés et numériques
- **Modèles de rapports** - Personnaliser les mises en page des rapports _(à venir)_
- **Modèles d'e-mails** - Personnaliser les notifications par e-mail _(à venir)_

## Types de modèles

### Modèles par défaut

Détectés automatiquement à partir de vos fichiers de plugins et de thèmes. Ils ne peuvent pas être supprimés mais peuvent être copiés pour créer des versions personnalisées.

### Modèles personnalisés

Modèles créés par l'utilisateur que vous pouvez entièrement personnaliser, modifier et gérer via l'interface d'administration de WordPress.
