## Activer les Produits POS Uniquement

Ce paramètre ajoute des options de visibilité en ligne et pour le POS sur vos pages d'administration de produits, vous permettant de contrôler si les produits apparaissent dans votre boutique en ligne, votre système POS ou les deux.

:::warning Impact sur la Performance
Activez ce paramètre uniquement si vous devez cacher des produits de votre boutique en ligne ou du POS. Lorsqu'il est activé, il ajoute une recherche supplémentaire dans la base de données pour chaque demande de produit, ce qui peut affecter la performance.
:::

**Quand activer :**
- Vous avez des produits qui ne doivent être vendus qu'en magasin (POS uniquement)
- Vous avez des produits qui ne doivent être vendus qu'en ligne (En ligne uniquement)
- Vous avez besoin de catalogues de produits différents pour les ventes en ligne par rapport aux ventes en magasin

**Lorsqu'il est activé, vous pouvez définir la visibilité des produits sur :**
- **POS & En ligne** (par défaut) : Produit visible à la fois dans la boutique en ligne et dans le POS
- **POS uniquement** : Produit caché de la boutique en ligne, visible seulement dans le POS
- **En ligne seulement** : Produit caché du POS, visible uniquement dans la boutique en ligne

**Définir la Visibilité du Produit :**
1. Modifiez un produit dans le WP Admin
2. Cherchez le paramètre "Visibilité POS" dans le panneau de données du produit
3. Choisissez votre option de visibilité souhaitée
4. Mettez à jour le produit

Vous pouvez également utiliser des actions groupées pour changer la visibilité de plusieurs produits en même temps depuis la page de liste des Produits.

## Activer les Quantités Décimales

Ce paramètre permet aux clients et au personnel d'entrer des quantités décimales lors de l'ajout de produits au panier.

**Exemples de quantités décimales :**
- 0.25 (un quart d'unité)
- 1.5 (une unité et demie)
- 2.75 (deux unités et trois quarts)

**Utile pour :**
- Produits vendus au poids (par exemple, 0.5kg de grains de café)
- Services fractionnaires (par exemple, 1.5 heures de consultation)
- Produits en vrac pouvant être vendus en quantités partielles

## Générer Automatiquement le Nom d'Utilisateur à partir de l'E-mail du Client

Lorsqu'il est activé, WCPOS créera automatiquement des noms d'utilisateur pour les nouveaux clients en fonction de leurs adresses e-mail lors du passage à la caisse. Cela simplifie le processus de création de client et garantit un formatage cohérent des noms d'utilisateur.

## Client par Défaut POS

Sélectionnez le client par défaut qui sera attribué aux nouvelles commandes POS. Les options courantes comprennent :

- **Invité** : Les commandes ne sont pas associées à un compte utilisateur
- **Caissier** : Les commandes sont associées au compte du caissier connecté
- **Client Spontané** : Les commandes sont associées à un compte client spécifique (vous devrez d'abord créer ce compte)

## Utiliser le Compte Caissier

Lorsqu'il est activé, le compte utilisateur du caissier sera utilisé comme client par défaut pour les commandes POS au lieu du Client par Défaut POS sélectionné.

## Champ de Code-barres

Ce paramètre détermine quel champ de produit WCPOS recherchera lors de l'analyse des codes-barres ou de l'utilisation de la fonctionnalité de recherche par code-barres.

:::warning Important : Effacement et Rafraîchissement Requis
Après avoir modifié le paramètre du champ de code-barres, vous **devez** nettoyer et rafraîchir vos données de produit dans le POS pour que le changement prenne effet.

**Comment nettoyer et rafraîchir :**
1. Allez à l'écran Produits dans votre POS
2. **Appuyez longuement** sur l'icône de synchronisation (↻) en bas de l'écran
3. Sélectionnez **"Nettoyer et Rafraîchir"** dans le menu contextuel
4. Attendez que les produits se rechargent avec les nouvelles données du champ de code-barres

Cela est nécessaire car le POS met en cache les données produit localement, et changer le paramètre du champ de code-barres ne met pas automatiquement à jour les données mises en cache.
:::

### Option par Défaut

**`_global_unique_id`** - Il s'agit du champ d'identifiant unique global de WooCommerce, introduit dans WooCommerce 8.2 (juillet 2024). Il est conçu pour stocker des codes GTIN, UPC, EAN ou ISBN et est l'option recommandée pour les nouvelles installations.

### Autres Options Courantes

**`_sku`** - Le champ SKU (Stock Keeping Unit) du produit. C'est une bonne option de secours si vous n'utilisez pas le champ d'identifiant unique global.

### Clés Meta de Plugins Tiers

Si vous utilisez un plugin de gestion des codes-barres, vous devrez peut-être utiliser leur clé meta spécifique :

| Type de Plugin | Clés Meta Courantes | Description |
|----------------|----------------------|-------------|
| Plugins EAN | `_ean`, `_ean13`, `_ean8` | Codes de Numéro d'Article Européen |
| Plugins UPC | `_upc`, `_upc_code` | Code Universel de Produit |
| Plugins GTIN | `_gtin`, `_gtin13`, `_gtin14` | Numéro d'Article Commercial Global |
| Code-barre Générique | `_barcode`, `_product_barcode` | Champs de code-barre génériques |
| Codes-barres de Produits WooCommerce | `_barcode_text` | Plugin de code-barre populaire |
| Gestion de Stock ATUM | `_atum_barcode` | Gestion d'inventaire ATUM |

### Trouver la Clé Meta de Votre Plugin

Pour trouver la clé meta correcte pour votre plugin de code-barres :

1. Modifiez un produit dans le WP Admin
2. Ajoutez un code-barres en utilisant votre plugin
3. Consultez la documentation du plugin pour connaître le nom de la clé meta
4. Ou inspectez la table de base de données `wp_postmeta` pour voir quelle clé meta a été créée

### Clés Meta Personnalisées

Vous pouvez également utiliser n'importe quelle clé meta personnalisée qui stocke des données de code-barres. Il vous suffit d'entrer le nom de la clé meta (y compris le préfixe de soulignement si applicable) dans ce champ.

:::tip
Testez votre scan de code-barres après avoir modifié ce paramètre pour vous assurer qu'il fonctionne correctement avec le champ choisi. N'oubliez pas de nettoyer et de rafraîchir vos données de produit dans le POS pour que le changement prenne effet.
:::
