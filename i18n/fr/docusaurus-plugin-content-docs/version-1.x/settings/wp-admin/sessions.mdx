L’écran **Sessions** permet de voir qui est actuellement connecté au PDV, sur quels appareils, et de déconnecter des appareils individuels à distance. C’est l’endroit à utiliser lorsqu’un appareil est perdu ou volé, lorsqu’un caissier quitte son poste, ou simplement pour vérifier les connexions actives.

Vous le trouverez dans **`WP Admin > POS > Réglages > Sessions`**.

> _Gérez les sessions utilisateur actives. Vous pouvez voir tous les utilisateurs connectés, consulter leurs appareils et mettre fin aux sessions si nécessaire._

## Fonctionnement {#how-it-works}

L’écran comporte deux volets :

- **Utilisateurs** (à gauche) — chaque utilisateur ayant au moins une session active, avec un point vert pour toute personne active au cours des dernières minutes, une pastille **Vous** pour votre propre compte, et le nombre de sessions de chaque utilisateur. Un champ de filtre permet de rechercher par nom ou par identifiant.
- **Sessions** (à droite) — sélectionnez un utilisateur pour voir une carte par session active.

Chaque carte de session affiche :

| Champ | Signification |
| --- | --- |
| **Type d’application** | Endroit où l’utilisateur est connecté : **application iOS**, **application Android**, **application de bureau** ou **application Web**, ainsi que le navigateur/la version |
| **Appareil actuel** | Une pastille indiquant la session en cours d’utilisation |
| **Dernière activité** | Date de la dernière requête effectuée par la session (par ex. _À l’instant_, _il y a 5 minutes_) |
| **IP** | L’adresse IP depuis laquelle la session s’est connectée (ou _S/O_) |
| **Créée** | Date de première connexion de la session |
| **Expire** | Date d’expiration du jeton de session |
| **Agent utilisateur** | Développez pour voir la chaîne user-agent brute du navigateur/de l’appareil |

Une session est affichée comme **Active maintenant** lorsqu’elle a effectué une requête au cours des 5 dernières minutes.

## Déconnexion des appareils {#signing-devices-out}

Trois actions sont disponibles, chacune avec une boîte de dialogue de confirmation :

- **Mettre fin** — déconnecte une seule session. Utilisez le bouton sur la carte de cette session.
- **Déconnecter les autres appareils** — déconnecte l’utilisateur sélectionné de tous les appareils, **sauf** celui qu’il utilise actuellement. Affiché lorsque l’utilisateur dispose de plusieurs sessions.
- **Tout déconnecter** — déconnecte l’utilisateur sélectionné de **tous** les appareils, y compris l’appareil actuel.

:::tip Appareil perdu ou volé
Si un téléphone, une tablette ou un ordinateur portable sur lequel le PDV est installé disparaît, ouvrez Sessions, sélectionnez l’utilisateur concerné, puis utilisez **Terminer** sur la session de cet appareil (ou **Tout déconnecter** par sécurité). L’appareil perdra l’accès lors de sa prochaine requête.
:::

:::note
Mettre fin à votre propre session actuelle vous déconnecte de l’appareil que vous utilisez en ce moment — une confirmation vous sera demandée avant que cela ne se produise.
:::
