Ces paramètres configurent la fonctionnalité backend de l'application POS, y compris les configurations PHP et WooCommerce. 
Vous pouvez y accéder en naviguant à : `WP Admin > POS > Settings`

Les configurations clés incluent :

- Client de panier par défaut pour tous les caissiers.
- Champ de code-barres de produit.
- Paramètres d'email de commande.
- Autorisations de rôle d'utilisateur.
