Ces réglages configurent la fonctionnalité backend de l'application POS, y compris les configurations PHP et WooCommerce. 
Vous pouvez y accéder en naviguant vers : `WP Admin > POS > Settings`

Les configurations clés incluent :

- Client du panier par défaut pour tous les caissiers.
- Champ de code-barres du produit.
- Paramètres de courriel de commande.
- Permissions de rôle d'utilisateur.
