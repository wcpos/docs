Ces paramètres configurent les fonctionnalités backend de l'application POS, y compris les configurations PHP et WooCommerce. 
Vous pouvez y accéder en navigant vers : `Administration WordPress > POS > Paramètres`

Configurations principales :

- Client par défaut pour tous les caissiers.
- Champ de code-barres du produit.
- Paramètres d'e-mail de commande.
- Autorisations des rôles utilisateurs.
