WCPOS a trois types de paramètres organisés par leur configuration.

## Où trouver les paramètres {#where-to-find-settings}

### 1. Paramètres de l'administrateur WordPress {#1-wordpress-admin-settings}

Paramètres de backend configurés dans votre tableau de bord WordPress :

**Emplacement :** `WP Admin > POS > Paramètres`

Ceux-ci contrôlent :
- Client par défaut pour tous les caissiers
- Configuration du champ de code-barres des produits
- Paramètres de courrier électronique de commande
- Autorisations de rôle utilisateur (qui peut accéder au POS)
- Disponibilité des passerelles de paiement

Voir [Paramètres de l'administrateur WordPress](/settings/wp-admin) pour plus de détails.

### 2. Paramètres de la boutique POS {#2-pos-store-settings}

Paramètres à l'échelle de l'application configurés au sein de l'application POS :

**Emplacement :** Menu utilisateur (en haut à droite) → Paramètres, ou `Ctrl + Shift + S`

Ceux-ci contrôlent :
- Options de calcul des taxes
- Configuration du scanner de codes-barres
- Raccourcis clavier
- Thème/apperance

Voir [Paramètres de la boutique](/settings/store) pour plus de détails.

### 3. Paramètres d'affichage (par écran) {#3-display-settings-per-screen}

Chaque écran du POS a ses propres paramètres d'affichage pour personnaliser le tableau de données :

**Emplacement :** Cliquez sur l'icône de curseurs (<Icon name="sliders" />) sur n'importe quel écran

Ceux-ci contrôlent :
- Quelles colonnes sont visibles
- Options d'affichage des colonnes
- Fonctionnalités spécifiques à l'écran

Les paramètres d'affichage sont documentés au sein de la documentation de chaque écran :
- [Panneau des produits POS](/pos/product-panel#display-settings)
- [Panneau du panier](/pos/cart#display-settings)
- [Écran des produits](/products#display-settings)
- [Écran des commandes](/orders#display-settings)
- [Écran des clients](/customers#display-settings)
- [Écran des rapports](/reports#display-settings)
