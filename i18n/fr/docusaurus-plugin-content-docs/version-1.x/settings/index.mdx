WCPOS a trois types de paramètres organisés par leur configuration.

## Où trouver les paramètres

### 1. Paramètres de l'administration WordPress

Paramètres backend configurés dans votre tableau de bord WordPress :

**Emplacement :** `WP Admin > POS > Paramètres`

Ces paramètres contrôlent :
- Client par défaut du panier pour tous les caissiers
- Configuration du champ code-barres produit
- Paramètres des e-mails de commande
- Permissions des rôles utilisateur (qui peut accéder au POS)
- Disponibilité de la passerelle de paiement

Voir [Paramètres de l'administration WordPress](/settings/wp-admin) pour plus de détails.

### 2. Paramètres du magasin POS

Paramètres globaux de l'application configurés dans l'application POS :

**Emplacement :** Menu utilisateur (en haut à droite) → Paramètres, ou `Ctrl + Shift + S`

Ces paramètres contrôlent :
- Options de calcul des taxes
- Configuration du scanner de codes-barres
- Raccourcis clavier
- Thème/esthétique

Voir [Paramètres du magasin](/settings/store) pour plus de détails.

### 3. Paramètres d'affichage (par écran)

Chaque écran du POS a ses propres paramètres d'affichage pour personnaliser le tableau de données :

**Emplacement :** Cliquez sur l'icône des curseurs (<Icon name="sliders" />) sur n'importe quel écran

Ces paramètres contrôlent :
- Quelles colonnes sont visibles
- Options d'affichage des colonnes
- Fonctionnalités spécifiques à l'écran

Les paramètres d'affichage sont documentés dans la documentation de chaque écran :
- [Panneau des produits POS](/pos/product-panel#display-settings)
- [Panneau du panier](/pos/cart#display-settings)
- [Écran des produits](/products#display-settings)
- [Écran des commandes](/orders#display-settings)
- [Écran des clients](/customers#display-settings)
- [Écran des rapports](/reports#display-settings)
