WCPOS a trois types de réglages organisés selon l'endroit où ils sont configurés.

## Où trouver les réglages

### 1. Réglages de l'administration WordPress

Réglages backend configurés dans votre tableau de bord WordPress :

**Emplacement :** `WP Admin > POS > Settings`

Ces réglages contrôlent :
- Client par défaut du panier pour tous les caissiers
- Configuration du champ code-barres produit
- Paramètres des e-mails de commande
- Permissions des rôles utilisateur (qui peut accéder au POS)
- Disponibilité de la passerelle de paiement

Voir [Réglages de l'administration WordPress](/settings/wp-admin) pour plus de détails.

### 2. Réglages du magasin POS

Réglages globaux de l'application configurés dans l'application POS :

**Emplacement :** Menu utilisateur (en haut à droite) → Réglages, ou `Ctrl + Shift + S`

Ces réglages contrôlent :
- Options de calcul des taxes
- Configuration du scanner de codes-barres
- Raccourcis clavier
- Thème/aspect

Voir [Réglages du magasin](/settings/store) pour plus de détails.

### 3. Réglages d'affichage (par écran)

Chaque écran du POS a ses propres réglages d'affichage pour personnaliser le tableau de données :

**Emplacement :** Cliquez sur l'icône des curseurs (<Icon name="sliders" />) sur n'importe quel écran

Ces réglages contrôlent :
- Quelles colonnes sont visibles
- Options d'affichage des colonnes
- Fonctionnalités spécifiques à l'écran

Les réglages d'affichage sont documentés dans la documentation de chaque écran :
- [Panneau des produits POS](/pos/product-panel#display-settings)
- [Panneau du panier](/pos/cart#display-settings)
- [Écran des produits](/products#display-settings)
- [Écran des commandes](/orders#display-settings)
- [Écran des clients](/customers#display-settings)
- [Écran des rapports](/reports#display-settings)
