:::warning Important
Au chargement initial, le POS récupère les paramètres de taxe par défaut de WooCommerce. En général, les utilisateurs doivent conserver les mêmes paramètres que WooCommerce sauf s'ils ont un besoin spécifique. Avoir des paramètres de taxe différents du serveur provoquera presque certainement des calculs de taxe différents entre le POS et WooCommerce, ce qui peut entraîner des incohérences et des problèmes de synchronisation des commandes.
:::

## Voir Tous les Taux de Taxe

Cliquez sur le bouton **Voir tous les taux de taxe** pour afficher tous les taux de taxe téléchargés sur le POS. Cela montre :

- Noms des taux de taxe
- Pourcentages des taux de taxe
- Régions/emplacements applicables

Cela est utile pour vérifier que les taux de taxe corrects sont chargés et pour déboguer les problèmes de calcul de taxe.

## Paramètres de Calcul de Taxe

### Prix Saisis Avec Taxe

Détermine si les prix des produits incluent la taxe :

| Option | Description |
|--------|-------------|
| **Oui** | Les prix des produits incluent la taxe. Le POS calcule la taxe en remontant du prix total |
| **Non** | Les prix des produits n'incluent pas la taxe. Le POS ajoute la taxe en plus du prix saisi |

### Calculer la Taxe Basée Sur

Détermine l'adresse utilisée pour le calcul de la taxe :

| Option | Description |
|--------|-------------|
| **Adresse de livraison du client** | Taux de taxe basés sur l'endroit où la commande sera livrée |
| **Adresse de facturation du client** | Taux de taxe basés sur l'adresse de facturation du client |
| **Adresse de base de la boutique** | Taux de taxe basés sur l'emplacement physique de votre boutique |

### Classe de Taxe de Livraison

Détermine comment les frais de livraison sont taxés :

| Option | Description |
|--------|-------------|
| **Classe de taxe de livraison basée sur les articles du panier** | La taxe de livraison est calculée en fonction de la classe de taxe des articles du panier |
| **Standard** | La livraison est taxée au taux de taxe standard |
| **Taux réduit** | La livraison est taxée au taux de taxe réduit |
| **Taux zéro** | Aucune taxe n'est appliquée aux frais de livraison |

### Arrondissement

| Option | Description |
|--------|-------------|
| **Arrondir la taxe au niveau du sous-total** | La taxe est calculée pour l'ensemble du sous-total, puis arrondie, plutôt que d'arrondir la taxe pour chaque ligne individuelle |

## Paramètres d'Affichage

### Afficher les Prix dans la Boutique

| Option | Description |
|--------|-------------|
| **Taxes incluses** | Les prix affichés incluent les montants de taxe |
| **Taxes exclues** | Les prix affichés n'incluent pas les taxes |

### Afficher les Prix Pendant le Panier et le Paiement

| Option | Description |
|--------|-------------|
| **Taxes incluses** | Les prix du panier et du paiement incluent les taxes |
| **Taxes exclues** | Les prix du panier et du paiement excluent les taxes |

### Suffixe de Prix

Texte qui apparaît après les prix pour indiquer le statut de la taxe (par exemple, "TTC" ou "HT").

### Afficher les Totaux de Taxe

| Option | Description |
|--------|-------------|
| **Comme un total unique** | Toutes les taxes sont combinées en une seule ligne de total |
| **Détaillé** | Chaque taux de taxe est affiché comme une ligne distincte |

## Restaurer les Paramètres du Serveur

Cliquez pour récupérer et appliquer les paramètres de taxe WooCommerce actuels de votre serveur.

:::tip Bonne Pratique
Utilisez "Restaurer les Paramètres du Serveur" régulièrement pour maintenir votre configuration de taxe POS synchronisée avec votre boutique WooCommerce, en particulier après avoir apporté des modifications aux paramètres de taxe dans votre administration WordPress.
:::
