:::warning Important
Lors du chargement initial, le POS obtiendra les paramètres de taxe par défaut de WooCommerce. En général, les utilisateurs doivent conserver les mêmes paramètres que WooCommerce, sauf s'ils ont un besoin spécifique. Avoir des paramètres de taxe différents de ceux du serveur entraînera presque certainement des valeurs de taxe différentes entre le POS et WooCommerce, ce qui peut conduire à des écarts et des problèmes de synchronisation des commandes.
:::

## Voir Tous les Taux de Taxe

Cliquez sur le bouton **Voir tous les taux de taxe** pour voir tous les taux de taxe qui ont été téléchargés dans le POS. Cela montre :

- Noms des taux de taxe
- Pourcentages des taux de taxe
- Régions/emplacements applicables

Ceci est utile pour vérifier que les bons taux de taxe sont chargés et pour déboguer les problèmes de calcul de taxe.

## Paramètres de Calcul de Taxe

### Prix Saisis Avec Taxe

Détermine si les prix des produits incluent la taxe :

| Option | Description |
|--------|-------------|
| **Oui** | Les prix des produits incluent la taxe. Le POS calcule la taxe en travaillant à l'envers à partir du prix total |
| **Non** | Les prix des produits n'incluent pas la taxe. Le POS ajoute la taxe au prix saisi |

### Calculer la Taxe Basé Sur

Détermine l'adresse utilisée pour le calcul de la taxe :

| Option | Description |
|--------|-------------|
| **Adresse de livraison du client** | Taux de taxe basés sur l'endroit où la commande sera expédiée |
| **Adresse de facturation du client** | Taux de taxe basés sur l'adresse de facturation du client |
| **Adresse de base du magasin** | Taux de taxe basés sur l'emplacement physique de votre magasin |

### Classe de Taxe d'Expédition

Détermine comment les frais d'expédition sont taxés :

| Option | Description |
|--------|-------------|
| **Classe de taxe d'expédition basée sur les articles du panier** | La taxe d'expédition est calculée en fonction de la classe de taxe des articles dans le panier |
| **Standard** | L'expédition est taxée au taux de taxe standard |
| **Taux réduit** | L'expédition est taxée au taux de taxe réduit |
| **Taux zéro** | Aucune taxe n'est appliquée aux frais d'expédition |

### Arrondi

| Option | Description |
|--------|-------------|
| **Arrondir la taxe au niveau du sous-total** | La taxe est calculée pour l'ensemble du sous-total et ensuite arrondie, plutôt que de arrondir la taxe pour chaque ligne de produit individuellement |

## Paramètres d'Affichage

### Afficher les Prix dans la Boutique

| Option | Description |
|--------|-------------|
| **TTC** | Les prix affichés incluent les montants de taxe |
| **HT** | Les prix affichés n'incluent pas la taxe |

### Afficher les Prix Pendant le Panier et le Paiement

| Option | Description |
|--------|-------------|
| **TTC** | Les prix du panier et du paiement incluent la taxe |
| **HT** | Les prix du panier et du paiement n'incluent pas la taxe |

### Suffixe d'Affichage des Prix

Texte qui apparaît après les prix pour indiquer le statut de la taxe (par exemple, "TTC" ou "HT").

### Afficher les Totaux de Taxe

| Option | Description |
|--------|-------------|
| **En un total unique** | Toutes les taxes sont combinées en une seule ligne totale |
| **Détaillé** | Chaque taux de taxe est montré comme un élément de ligne séparé |

## Restaurer les Paramètres du Serveur

Cliquez pour récupérer et appliquer les paramètres de taxe WooCommerce actuels depuis votre serveur.

:::tip Meilleure Pratique
Utilisez régulièrement "Restaurer les paramètres du serveur" pour garder la configuration de la taxe de votre POS synchronisée avec votre boutique WooCommerce, en particulier après avoir apporté des modifications aux paramètres de taxe dans votre administration WordPress.
:::
