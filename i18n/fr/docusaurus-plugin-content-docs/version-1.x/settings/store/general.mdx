Les paramètres généraux de la boutique contrôlent la configuration fondamentale de votre boutique POS, comprenant les informations de la boutique, la localisation, le formatage de la devise et les valeurs par défaut des clients.

## Vue d'ensemble

Au chargement initial, le POS récupère automatiquement les paramètres par défaut de votre configuration WooCommerce. Dans la plupart des cas, vous souhaiterez conserver ces paramètres synchronisés avec votre boutique WooCommerce pour maintenir la cohérence sur tous les canaux de vente.

:::tip Bonne pratique
En général, les utilisateurs doivent conserver les mêmes paramètres que leur boutique WooCommerce, sauf s'il existe un besoin commercial spécifique de les personnaliser pour le POS, comme changer de devise pour un marché différent ou définir un client par défaut différent pour les transactions en magasin.
:::

## Informations de la boutique

### Nom de la boutique

Le nom de votre boutique tel qu'il apparaît dans l'interface POS et sur les reçus. Celui-ci est généralement synchronisé avec le nom de votre boutique WooCommerce.

### Adresse de la boutique

Les champs d'adresse incluent :

- **Ville de base de la boutique** : La ville où votre boutique est située
- **Code postal de base de la boutique** : Le code postal de votre emplacement
- **État/Province de base de la boutique** : L'état ou la province où votre boutique opère
- **Pays de base de la boutique** : Le pays où votre boutique est située

:::warning Paramètres d'adresse désactivés
Les champs d'adresse de la boutique sont actuellement désactivés dans les paramètres du POS car les modifications de l'adresse peuvent impacter les calculs de taxes et causer des problèmes de traitement des commandes. Ces paramètres restent synchronisés avec la configuration de votre boutique WooCommerce.
:::

## Paramètres de localisation

### Langue

Définit la langue d'affichage de l'interface POS. Cela contrôle la langue utilisée pour :

- Les éléments de menu et la navigation
- Les étiquettes de boutons et le texte de l'interface
- Les messages système et les notifications
- Le formatage de la date et de l'heure

### Devise

Définit la devise principale utilisée pour les transactions et l'affichage des prix. Ce paramètre affecte :

- L'affichage des prix des produits
- Les totaux et calculs des commandes
- Le formatage des reçus
- Les rapports financiers

### Position de la devise

Contrôle l'emplacement du symbole de devise par rapport au montant du prix :

- **Gauche** : 10,00 $ (symbole avant le montant)
- **Droite** : 10,00 $ (symbole après le montant)
- **Gauche avec espace** : $ 10,00
- **Droite avec espace** : 10,00 $

## Formatage des nombres

### Séparateur décimal

Le caractère utilisé pour séparer la partie entière de la partie décimale (par exemple, "," ou ".").

### Nombre de décimales

Spécifie le nombre de décimales à afficher pour les prix et les valeurs monétaires (généralement 2 pour la plupart des devises).

### Séparateur de milliers

Le caractère utilisé pour séparer les milliers dans les grands nombres (par exemple, " " ou ".").

### Style de groupement des milliers

Détermine comment les milliers sont regroupés et affichés :

- **milliers** : Groupement occidental standard (123 456 789) - groupe tous les 3 chiffres
- **lakh** : Système de numérotation indienne (12 34 56 789) - groupe les 3 premiers chiffres, puis tous les 2 chiffres
- **wan** : Système de numérotation d'Asie de l'Est (1 2345 6789) - groupe tous les 4 chiffres

## Paramètres clients

### Client par défaut

Définit le client automatiquement sélectionné pour les nouvelles commandes. Les options incluent généralement :

- **Invité** : Pour les clients sans compte
- **Client spécifique** : Un compte client prédéfini
- **Caissier** : Le caissier connecté comme client

### Le client par défaut est le caissier

Lorsqu'il est activé, le caissier actuellement connecté est automatiquement défini comme client pour les nouvelles commandes.

## Restaurer les paramètres du serveur

Le bouton "Restaurer les paramètres du serveur" vous permet de réinitialiser tous les paramètres généraux aux valeurs par défaut de votre boutique WooCommerce.

:::info Synchronisation
Cliquer sur "Restaurer les paramètres du serveur" récupérera les paramètres par défaut actuels de WooCommerce et remplacera tous les paramètres POS personnalisés.
:::
