Les Paramètres Généraux de la Boutique contrôlent la configuration fondamentale de votre boutique POS, y compris les informations sur la boutique, la localisation, le formatage des devises, et les valeurs par défaut des clients.

## Vue d'ensemble

Lors du chargement initial, le POS récupère automatiquement les paramètres par défaut de votre configuration de boutique WooCommerce. Dans la plupart des cas, vous voudrez maintenir ces paramètres synchronisés avec votre boutique WooCommerce pour conserver la cohérence sur tous les canaux de vente.

:::tip Meilleure Pratique
En général, les utilisateurs devraient maintenir les mêmes paramètres que leur boutique WooCommerce sauf s'il existe un besoin commercial spécifique de les personnaliser pour le POS, comme changer de devises pour un marché différent ou définir un client par défaut différent pour les transactions en magasin.
:::

## Informations sur la Boutique

### Nom de la Boutique

Le nom de votre boutique tel qu'il apparaît dans l'interface POS et sur les reçus. Cela est généralement synchronisé avec le nom de votre boutique WooCommerce.

### Adresse de la Boutique

Les champs d'adresse de la boutique comprennent :

- **Store Base City**: La ville où se trouve votre boutique
- **Store Base Postcode** : Le code postal de votre boutique
- **Store Base State**: L'état ou la province où votre boutique opère
- **Store Base Country**: Le pays où se trouve votre boutique

:::warning Paramètres d'Adresse Désactivés
Les champs d'adresse de la boutique sont actuellement désactivés dans les paramètres POS parce que les modifications de l'adresse du magasin peuvent avoir un impact sur les calculs fiscaux et peuvent causer des problèmes avec le traitement des commandes. Ces paramètres restent synchronisés avec la configuration de votre boutique WooCommerce.
:::

## Paramètres de Localisation

### Langue

Définit la langue d'affichage pour l'interface POS. Cela contrôle la langue utilisée pour :

- Les éléments de menu et la navigation
- Les étiquettes de boutons et le texte de l'interface
- Les messages et notifications système
- Le formatage des dates et heures

### Devise

Définit la devise principale utilisée pour les transactions et l'affichage des prix. Ce paramètre affecte :

- L'affichage des prix des produits
- Les totaux et calculs des commandes
- Le formatage des reçus
- Les rapports financiers

### Position de la Devise

Contrôle où le symbole de la devise apparaît par rapport au montant du prix :

- **Gauche** : $10,00 (symbole avant le montant)
- **Droite** : 10,00$ (symbole après le montant)
- **Gauche avec espace** : $ 10,00
- **Droite avec espace** : 10,00 $

## Formatage des Nombres

### Séparateur Décimal

Le caractère utilisé pour séparer le nombre entier de la partie décimale (par exemple, "." ou ",").

### Nombre de Décimales

Spécifie combien de décimales afficher pour les prix et les valeurs monétaires (typiquement 2 pour la plupart des devises).

### Séparateur de Milliers

Le caractère utilisé pour séparer les milliers dans de grands nombres (par exemple, "," pour 1,000 ou "." pour 1.000).

### Style de Groupe de Milliers

Détermine comment les milliers sont groupés et affichés :

- **thousand** : Groupement occidental standard (123,456,789) - groupe chaque 3 chiffres
- **lakh** : Système de numérotation indien (12,34,56,789) - groupe les 3 premiers chiffres, puis chaque 2 chiffres
- **wan** : Système de numérotation est-asiatique (1,2345,6789) - groupe chaque 4 chiffres

## Paramètres des Clients

### Client Par Défaut

Définit le client qui est automatiquement sélectionné pour les nouvelles commandes. Les options incluent généralement :

- **Invité** : Pour les clients sans compte
- **Client Spécifique** : Un compte client prédéfini
- **Caissier** : Le caissier connecté comme client

### Le Client Par Défaut est le Caissier

Lorsqu'il est activé, le caissier actuellement connecté est automatiquement défini comme le client pour les nouvelles commandes.

## Restaurer les Paramètres du Serveur

Le bouton "Restaurer les paramètres du serveur" vous permet de réinitialiser tous les paramètres généraux aux valeurs par défaut de votre boutique WooCommerce.

:::info Synchronisation
Cliquer sur "Restaurer les paramètres du serveur" va récupérer les paramètres par défaut actuels de la boutique WooCommerce et remplacer tous les paramètres personnalisés du POS.
:::
