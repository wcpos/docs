## Aperçu

Lors du chargement initial, le POS récupère automatiquement les paramètres par défaut à partir de la configuration de votre boutique WooCommerce. Dans la plupart des cas, vous voudrez garder ces paramètres synchronisés avec votre boutique WooCommerce pour maintenir la cohérence sur tous les canaux de vente.

:::tip Meilleure Pratique
En général, les utilisateurs devraient maintenir les mêmes paramètres que leur boutique WooCommerce, à moins qu'il n'y ait un besoin commercial spécifique de les personnaliser pour le POS, comme changer de devise pour un marché différent ou définir un client par défaut différent pour les transactions en magasin.
:::

## Informations sur la boutique

### Nom de la boutique

Le nom de votre boutique tel qu'il apparaît dans l'interface POS et sur les reçus. Cela est généralement synchronisé avec le nom de votre boutique WooCommerce.

### Adresse de la boutique

Les champs d'adresse de la boutique comprennent :

- **Ville de base de la boutique** : La ville où se trouve votre boutique
- **Code postal de base de la boutique** : Le code postal pour l'emplacement de votre boutique
- **État de base de la boutique** : L'état ou la province où votre boutique opère
- **Pays de base de la boutique** : Le pays où se trouve votre boutique

:::warning Paramètres d'adresse désactivés
Les champs d'adresse de la boutique sont actuellement désactivés dans les paramètres POS car des changements dans l'adresse de la boutique peuvent impacter les calculs d'impôts et peuvent causer des problèmes avec le traitement des commandes. Ces paramètres restent synchronisés avec la configuration de votre boutique WooCommerce.
:::

## Paramètres de localisation

### Langue

Définit la langue d'affichage pour l'interface POS. Cela contrôle la langue utilisée pour :

- Les éléments de menu et la navigation
- Les étiquettes des boutons et le texte de l'interface
- Les messages et notifications du système
- Le formatage de la date et de l'heure

### Devise

Définit la devise principale utilisée pour les transactions et l'affichage des prix. Ce paramètre affecte :

- L'affichage des prix des produits
- Les totaux et calculs de commande
- Le formatage des reçus
- Les rapports financiers

### Position de la devise

Contrôle où le symbole de la devise apparaît par rapport au montant du prix :

- **Gauche** : 10,00 $ (symbole avant le montant)
- **Droite** : 10,00 $ (symbole après le montant)
- **Gauche avec espace** : 10,00 $
- **Droite avec espace** : 10.00 $

## Formatage des nombres

### Séparateur décimal

Le caractère utilisé pour séparer le nombre entier de la partie décimale (par exemple, "." ou ",").

### Nombre de décimales

Spécifie combien de chiffres décimaux afficher pour les prix et les valeurs monétaires (généralement 2 pour la plupart des devises).

### Séparateur de milliers

Le caractère utilisé pour séparer les milliers dans les grands nombres (par exemple, "," pour 1 000 ou "." pour 1.000).

### Style de groupe de milliers

Détermine comment les milliers sont groupés et affichés :

- **mille** : Groupement occidental standard (123 456 789) - groupe tous les 3 chiffres
- **lakh** : Système de numération indien (12,34,56,789) - groupe les 3 premiers chiffres, puis tous les 2 chiffres
- **wan** : Système de numération est-asiatique (1,2345,6789) - groupe tous les 4 chiffres

## Paramètres clients

### Client par défaut

Définit le client qui est automatiquement sélectionné pour les nouvelles commandes. Les options incluent généralement :

- **Invité** : Pour les clients sans compte
- **Client spécifique** : Un compte client prédéfini
- **Caissier** : Le caissier connecté en tant que client

### Le client par défaut est le caissier

Lorsque cela est activé, le caissier actuellement connecté est automatiquement défini comme le client pour les nouvelles commandes.

## Restaurer les paramètres du serveur

Le bouton "Restaurer les paramètres du serveur" vous permet de réinitialiser tous les paramètres généraux aux valeurs par défaut de votre boutique WooCommerce.

:::info Synchronisation
Cliquer sur "Restaurer les paramètres du serveur" récupérera les valeurs par défaut de la boutique WooCommerce actuelle et remplacera tous les paramètres personnalisés du POS.
:::
