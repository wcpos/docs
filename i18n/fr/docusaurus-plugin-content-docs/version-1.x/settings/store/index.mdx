Les Paramètres du magasin sont des configurations à l'échelle de l'application qui affectent la manière dont le POS fonctionne. Ces paramètres s'appliquent à l'ensemble du magasin, et non à des écrans individuels.

## Accéder aux paramètres du magasin

Ouvrez les Paramètres du magasin en :

- Cliquant sur le **menu utilisateur** (coin supérieur droit) et en sélectionnant **Paramètres**
- Utilisant le raccourci clavier `Ctrl + Shift + S`

## Onglets des paramètres

La fenêtre modale des Paramètres du magasin contient les onglets suivants :

| Onglet | Description |
|--------|-------------|
| [Général](./general) | Informations sur le magasin, langue, monnaie, client par défaut |
| [Taxe](./tax) | Options de calcul et d'affichage des taxes |
| [Code-barres](./barcode) | Configuration du scanner de code-barres |
| [Raccourcis clavier](./hotkeys) | Raccourcis clavier disponibles |
| [Thème](./theme) | Apparence et schéma de couleurs |
