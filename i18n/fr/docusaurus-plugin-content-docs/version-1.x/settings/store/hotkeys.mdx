## Raccourcis Disponibles

| Combinaison de Touches | Action | Description |
|------------------------|--------|-------------|
| `Ctrl + Shift + S`     | Paramètres | Ouvre le panneau des paramètres POS |
| `Ctrl + Shift + A`     | POS    | Navigue vers l'écran principal du POS |
| `Ctrl + Shift + P`     | Produits | Ouvre l'écran de gestion des produits |
| `Ctrl + Shift + O`     | Commandes | Ouvre l'écran de gestion des commandes |
| `Ctrl + Shift + C`     | Clients | Ouvre l'écran de gestion des clients |
| `Ctrl + Shift + ?`     | Support | Ouvre la section d'assistance/aide |
| `Ctrl + Shift + L`     | Déconnexion | Se déconnecte de la session actuelle |

## Utilisation des Raccourcis Clavier

Pour utiliser un raccourci clavier :

1. **Maintenez enfoncées les touches de modification** (`Ctrl + Shift`)
2. **Appuyez sur la touche d'action** (la lettre correspondant à l'action désirée)
3. **Relâchez toutes les touches** pour exécuter la commande

## Conseils pour une Utilisation Efficace

- **Mémorisez les raccourcis fréquemment utilisés** : Concentrez-vous sur l'apprentissage des raccourcis pour les actions que vous effectuez le plus souvent
- **Pratiquez les combinaisons de touches** : Une utilisation régulière aidera à développer la mémoire musculaire
- **Utilisez les raccourcis pendant les périodes chargées** : Les raccourcis clavier peuvent accélérer considérablement les opérations pendant les heures de pointe

## Personnalisation

:::info Demande de Fonctionnalité
La personnalisation des raccourcis clavier n'est actuellement pas disponible. Les raccourcis énumérés ci-dessus sont fixes et ne peuvent pas être modifiés via l'interface des paramètres.

Si vous souhaitez personnaliser les raccourcis clavier, veuillez partager vos commentaires sur [Discord](https://wcpos.com/discord).
:::
