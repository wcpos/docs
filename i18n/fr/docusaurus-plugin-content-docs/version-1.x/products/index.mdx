:::info Fonction Pro
L'écran des Produits nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs gratuits peuvent voir et ajouter des produits au panier depuis le [Panneau de Produits POS](/pos/product-panel), mais ne peuvent pas modifier les inventaires ou les prix.
:::

L'écran des Produits est une interface dédiée à la gestion des stocks. Contrairement au [Panneau de Produits POS](/pos/product-panel) (qui est destiné à la vente), cet écran est conçu pour gérer votre catalogue de produits.

{/* Screenshot: products-screen.png */}

## Écran des Produits vs Panneau de Produits POS

| Caractéristique | Panneau de Produits POS | Écran des Produits |
|----------------|--------------------------|---------------------|
| **But** | Ajouter des produits au panier | Gérer l'inventaire |
| **Stock** | Affichage uniquement | Édition en ligne |
| **Prix** | Affichage uniquement | Édition en ligne |
| **Actions** | Ajouter au panier | Éditer, Synchroniser, Supprimer |
| **Disponible dans** | Gratuit + Pro | Pro uniquement |

## Vue d'Ensemble de l'Interface

### Recherche et Filtres

En haut de l'écran :

- **Barre de recherche** - Trouvez des produits par nom, SKU ou code-barres
- **Boutons de filtre** - Statut du Stock, Mis en avant, En Vente, Catégorie, Étiquette, Marque
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurer les colonnes visibles

### Liste des Produits

La zone principale affiche vos produits dans un tableau défilable :

- **Image** - Vignette du produit
- **Nom du produit** - Avec SKU, code-barres, attributs optionnels
- **Stock** - Quantité de stock éditable avec bascule "Gérer"
- **Statut du Stock** - En Stock (vert), Rupture de Stock (rouge), Commande en Attente
- **Catégories** - Badges de catégorie de produit
- **Prix** - Prix actuel, prix régulier, prix de vente (tous éditables)
- **Actions** - Menu à trois points avec Éditer, Synchroniser, Supprimer

### Produits Variables

Les produits variables montrent :
- Options d'attribut (Couleur, Taille, etc.)
- Lien "Développer" pour voir toutes les variations
- Lignes de variation individuelles avec stock/prix séparés

### Pied de Page

- **Statut fiscal** - Base actuelle de calcul de l'impôt
- **Nombre de produits** - "Affichage de X sur Y" avec bouton de synchronisation

## Caractéristiques Clés

### Édition Inline du Stock

Cliquez directement sur la quantité de stock pour l'éditer :

1. Cliquez sur le nombre de stock
2. Entrez la nouvelle quantité
3. Appuyez sur Entrée pour enregistrer

Les changements se synchronisent automatiquement avec WooCommerce.

### Bascule de Gestion de Stock

La bascule "Gérer" contrôle si WooCommerce suit l'inventaire pour ce produit :

- **Activé** - Les niveaux de stock sont suivis et diminués lors des ventes
- **Désactivé** - Le produit est toujours disponible (stock infini)

### Édition Inline des Prix

Cliquez sur n'importe quel champ de prix pour l'éditer :

- **Prix** - Prix de vente actuel
- **Prix Régulier** - Prix sans réduction
- **Prix de Vente** - Prix réduit (lorsqu'il est en promotion)

### Actions sur les Produits

Cliquez sur le menu à trois points (⋮) pour chaque produit :

- **Éditer** - Ouvrir le modal d'édition du produit
- **Synchroniser** - Actualiser ce produit depuis WooCommerce
- **Supprimer** - Retirer le produit de la base de données locale

:::note
Supprimer un produit du POS ne le retire que localement. Le produit reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

### Colonnes Disponibles

| Colonne | Description | Options d'Affichage |
|---------|-------------|---------------------|
| **Image** | Vignette du produit | - |
| **ID** | ID du produit WooCommerce | - |
| **Produit** | Nom du produit | SKU, Code-barres, Attributs |
| **Type** | Simple, variable, groupé, etc. | - |
| **SKU** | Unité de Gestion de Stock | - |
| **Code-barres** | Code-barres du produit | - |
| **Stock** | Quantité actuelle (éditable) | - |
| **Statut du Stock** | En Stock / Rupture de Stock / Commande en Attente | - |
| **Catégories** | Catégories de produits | - |
| **Étiquettes** | Étiquettes de produits | - |
| **Marques** | Informations sur la marque (si le plugin est actif) | - |
| **Coût des Biens Vendus** | Prix de coût du produit | - |
| **Prix** | Prix de vente actuel (éditable) | Informations fiscales |
| **Prix Régulier** | Prix non réduit (éditable) | Informations fiscales |
| **Prix de Vente** | Prix réduit (éditable) | Informations fiscales |
| **Date de Création** | Date de création du produit | - |
| **Date de Modification** | Date de la dernière modification | - |
| **Actions** | Boutons Éditer, Synchroniser, Supprimer | - |

### Restaurer les Paramètres par Défaut

Cliquez pour réinitialiser tous les paramètres d'affichage à leurs valeurs par défaut d'origine.

## Filtrer les Produits

Utilisez les boutons de filtre pour affiner votre vue :

- **Statut du Stock** - En Stock, Rupture de Stock, Commande en Attente
- **Mis en Avant** - Produits marqués comme mis en avant
- **En Vente** - Produits actuellement en promotion
- **Catégorie** - Filtrer par catégorie de produit
- **Étiquette** - Filtrer par étiquette de produit
- **Marque** - Filtrer par marque (si le plugin de marque est utilisé)

## Tri

Cliquez sur l'en-tête de n'importe quelle colonne pour trier par cette colonne. Cliquez à nouveau pour inverser l'ordre de tri.
