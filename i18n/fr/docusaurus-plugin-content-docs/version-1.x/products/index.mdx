:::info Fonctionnalité Pro
L'écran Produits nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs de la version gratuite peuvent afficher et ajouter des produits au panier depuis le [Panneau Produit POS](/pos/product-panel), mais ne peuvent pas modifier le stock ni les prix.
:::

L'écran Produits est une interface dédiée à la gestion des stocks. Contrairement au [Panneau Produit POS](/pos/product-panel) (qui est conçu pour la vente), cet écran est destiné à gérer votre catalogue de produits.

{/* Screenshot: products-screen.png */}

## Écran Produits vs Panneau Produit POS

| Fonctionnalité | Panneau Produit POS | Écran Produits |
|---------------|---------------------|----------------|
| **Objectif** | Ajouter des produits au panier | Gérer les stocks |
| **Stock** | Affichage uniquement | Modification en ligne |
| **Prix** | Affichage uniquement | Modification en ligne |
| **Actions** | Ajouter au panier | Modifier, Synchroniser, Supprimer |
| **Disponible dans** | Gratuit + Pro | Pro uniquement |

## Aperçu de l'Interface

### Recherche et Filtres

En haut de l'écran :

- **Barre de recherche** - Rechercher des produits par nom, SKU ou code-barres
- **Boutons de filtre** - Statut du stock, Mis en avant, En promotion, Catégorie, Étiquette, Marque
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurer les colonnes visibles

### Liste des Produits

La zone principale affiche vos produits dans un tableau défilant :

- **Image** - Vignette du produit
- **Nom du produit** - Avec SKU, code-barres, attributs optionnels
- **Stock** - Quantité de stock modifiable avec bouton "Gérer"
- **Statut du stock** - En stock (vert), Rupture de stock (rouge), Réapprovisionnement
- **Catégories** - Badges de catégorie de produit
- **Prix** - Prix actuel, prix standard, prix promotionnel (tous modifiables)
- **Actions** - Menu à trois points avec Modifier, Synchroniser, Supprimer

### Produits Variables

Les produits variables affichent :
- Options d'attributs (Couleur, Taille, etc.)
- Lien "Développer" pour voir toutes les variations
- Lignes de variation individuelles avec stock/prix séparés

### Pied de page

- **Statut fiscal** - Base de calcul fiscale actuelle
- **Nombre de produits** - "Affichage X sur Y" avec bouton de synchronisation

## Fonctionnalités Principales

### Modification Inline du Stock

Cliquez directement sur la quantité de stock pour la modifier :

1. Cliquez sur le nombre de stock
2. Entrez la nouvelle quantité
3. Appuyez sur Entrée pour enregistrer

Les modifications sont synchronisées automatiquement avec WooCommerce.

### Bouton de Gestion du Stock

Le bouton "Gérer" contrôle si WooCommerce suit le stock de ce produit :

- **Activé** - Les niveaux de stock sont suivis et décrémentés lors des ventes
- **Désactivé** - Le produit est toujours disponible (stock infini)

### Modification Inline des Prix

Cliquez sur n'importe quel champ de prix pour le modifier :

- **Prix** - Prix de vente actuel
- **Prix standard** - Prix sans remise
- **Prix promotionnel** - Prix réduit (lors d'une promotion)

### Actions sur les Produits

Cliquez sur le menu à trois points (⋮) pour chaque produit :

- **Modifier** - Ouvrir la fenêtre de modification du produit
- **Synchroniser** - Actualiser ce produit depuis WooCommerce
- **Supprimer** - Retirer le produit de la base de données locale

:::note
La suppression d'un produit du POS ne le supprime que localement. Le produit reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

### Colonnes Disponibles

| Colonne | Description | Options d'Affichage |
|---------|-------------|---------------------|
| **Image** | Vignette du produit | - |
| **ID** | ID produit WooCommerce | - |
| **Produit** | Nom du produit | SKU, Code-barres, Attributs |
| **Type** | Simple, variable, groupé, etc. | - |
| **SKU** | Unité de gestion de stock | - |
| **Code-barres** | Code-barres du produit | - |
| **Stock** | Quantité actuelle (modifiable) | - |
| **Statut du stock** | En stock / Rupture de stock / Réapprovisionnement | - |
| **Catégories** | Catégories de produit | - |
| **Étiquettes** | Étiquettes de produit | - |
| **Marques** | Informations de marque (si plugin actif) | - |
| **Coût des Marchandises Vendues** | Prix de revient du produit | - |
| **Prix** | Prix de vente actuel (modifiable) | Informations fiscales |
| **Prix standard** | Prix sans remise (modifiable) | Informations fiscales |
| **Prix promotionnel** | Prix réduit (modifiable) | Informations fiscales |
| **Date de Création** | Date de création du produit | - |
| **Date de Modification** | Date de dernière modification | - |
| **Actions** | Boutons Modifier, Synchroniser, Supprimer | - |

### Restaurer les Paramètres par Défaut

Cliquez pour réinitialiser tous les paramètres d'affichage à leurs valeurs d'origine.

## Filtrer les Produits

Utilisez les boutons de filtre pour affiner votre vue :

- **Statut du stock** - En stock, Rupture de stock, Réapprovisionnement
- **Mis en avant** - Produits marqués comme mis en avant
- **En promotion** - Produits actuellement en promotion
- **Catégorie** - Filtrer par catégorie de produit
- **Étiquette** - Filtrer par étiquette de produit
- **Marque** - Filtrer par marque (si plugin de marque utilisé)

## Tri

Cliquez sur l'en-tête d'une colonne pour trier par cette colonne. Cliquez à nouveau pour inverser l'ordre de tri.
