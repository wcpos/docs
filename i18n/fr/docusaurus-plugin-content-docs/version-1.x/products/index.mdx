:::info Fonctionnalité Pro
L'écran des Produits nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs gratuits peuvent visualiser et ajouter des produits au panier depuis le [Panneau de Produit POS](/pos/product-panel), mais ne peuvent pas modifier l'inventaire ou les prix.
:::

L'écran des Produits est une interface dédiée à la gestion des stocks. Contrairement au [Panneau de Produit POS](/pos/product-panel) (qui est destiné à la vente), cet écran est conçu pour gérer votre catalogue de produits.

{/* Screenshot: products-screen.png */}

## Écran des Produits vs Panneau de Produit POS

| Fonctionnalité | Panneau de Produit POS | Écran des Produits |
|----------------|------------------------|---------------------|
| **Objectif** | Ajouter des produits au panier | Gérer l'inventaire |
| **Stocks** | Affichage uniquement | Édition en ligne |
| **Prix** | Affichage uniquement | Édition en ligne |
| **Actions** | Ajouter au panier | Éditer, Synchroniser, Supprimer |
| **Disponible dans** | Gratuit + Pro | Pro uniquement |

## Aperçu de l'Interface

### Recherche & Filtres

En haut de l'écran :

- **Barre de recherche** - Trouver des produits par nom, SKU ou code-barres
- **Boutons de filtre** - État du Stock, En Vedette, En Promotion, Catégorie, Tag, Marque
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurer les colonnes visibles

### Liste des Produits

La zone principale affiche vos produits dans un tableau déroulant :

- **Image** - Vignette du produit
- **Nom du produit** - Avec SKU, code-barres, attributs en option
- **Stock** - Quantité de stock éditable avec bascule "Gérer"
- **État du Stock** - En Stock (vert), En Rupture de Stock (rouge), Commande en Attente
- **Catégories** - Badges de catégorie de produit
- **Prix** - Prix actuel, prix régulier, prix de vente (tous éditables)
- **Actions** - Menu à trois points avec Éditer, Synchroniser, Supprimer

### Produits Variables

Les produits variables montrent :
- Options d'attribut (Couleur, Taille, etc.)
- Lien "Développer" pour voir toutes les variations
- Lignes de variation individuelles avec des stocks/prix séparés

### Pied de Page

- **Statut fiscal** - Base de calcul de la taxe actuelle
- **Compteur de produits** - "Affichage de X sur Y" avec bouton de synchronisation (<Icon name="arrow-rotate-right" />). **Appuyez longuement** sur le bouton de synchronisation pour les options Effacer et Actualiser

## Fonctionnalités Clés

### Édition en Ligne du Stock

Cliquez directement sur la quantité de stock pour l'éditer :

1. Cliquez sur le nombre de stock
2. Entrez la nouvelle quantité
3. Appuyez sur Entrée pour enregistrer

Les changements se synchronisent automatiquement avec WooCommerce.

### Bascule de Gestion des Stocks

La bascule "Gérer" contrôle si WooCommerce suit l'inventaire pour ce produit :

- **Activé** - Les niveaux de stock sont suivis et décrémentés lors des ventes
- **Désactivé** - Le produit est toujours disponible (stock infini)

### Édition en Ligne des Prix

Cliquez sur n'importe quel champ de prix pour l'éditer :

- **Prix** - Prix de vente actuel
- **Prix Régulier** - Prix non remisé
- **Prix de Vente** - Prix remisé (lorsqu'il est en vente)

### Actions sur les Produits

Cliquez sur le menu à trois points (⋮) pour chaque produit :

- **Éditer** - Ouvrir la fenêtre modale d'édition du produit
- **Synchroniser** - Actualiser ce produit depuis WooCommerce
- **Supprimer** - Retirer le produit de la base de données locale

:::note
La suppression d'un produit depuis le POS ne l'enlève que localement. Le produit reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

### Colonnes Disponibles

| Colonne | Description | Options d'Affichage |
|---------|-------------|---------------------|
| **Image** | Vignette du produit | - |
| **ID** | ID du produit WooCommerce | - |
| **Produit** | Nom du produit | SKU, Code-barres, Attributs |
| **Type** | Simple, variable, groupé, etc. | - |
| **SKU** | Unité de Gestion de Stock | - |
| **Code-barres** | Code-barres du produit | - |
| **Stock** | Quantité actuelle (éditable) | - |
| **État du Stock** | En Stock / En Rupture de Stock / Commande en Attente | - |
| **Catégories** | Catégories de produit | - |
| **Tags** | Tags de produit | - |
| **Marques** | Informations sur la marque (si le plugin est actif) | - |
| **Coût des Biens Vendus** | Prix de coût du produit | - |
| **Prix** | Prix de vente actuel (éditable) | Informations fiscales |
| **Prix Régulier** | Prix non remisé (éditable) | Informations fiscales |
| **Prix de Vente** | Prix remisé (éditable) | Informations fiscales |
| **Date Created** | Date de création du produit | - |
| **Date Modified** | Date de dernière modification | - |
| **Actions** | Boutons Éditer, Synchroniser, Supprimer | - |

### Restaurer les Paramètres par Défaut

Cliquez pour réinitialiser tous les paramètres d'affichage à leurs valeurs par défaut d'origine.

## Filtrage des Produits

Utilisez les boutons de filtre pour affiner votre vue :

- **État du Stock** - En Stock, En Rupture de Stock, Commande en Attente
- **En Vedette** - Produits marqués comme en vedette
- **En Promotion** - Produits actuellement en promotion
- **Catégorie** - Filtrer par catégorie de produit
- **Tag** - Filtrer par tag de produit
- **Marque** - Filtrer par marque (si utilisant le plugin de marque)

## Tri

Cliquez sur n'importe quel en-tête de colonne pour trier par cette colonne. Cliquez à nouveau pour inverser l'ordre de tri.
