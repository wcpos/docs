:::info Fonction Pro
L'écran Clients nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs gratuits peuvent sélectionner des clients existants pour les commandes, mais ne peuvent pas voir la liste complète des clients ou modifier les détails des clients.
:::

L'écran Clients offre une gestion complète des clients directement dans le POS. Affichez, modifiez et créez des clients sans passer par l'administration de WooCommerce.

{/* Capture d'écran : customers-screen.png */}

## Aperçu de l'Interface

### Actions de l'En-tête

En haut de l'écran :

- **Barre de recherche** - Trouvez des clients par nom, email, etc.
- **Ajouter un client** (<Icon name="plus" />) - Créer un nouveau client
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurer les colonnes visibles

### Tableau des Clients

La zone principale affiche les clients avec :

- **Avatar** - Photo de profil du client (ou image par défaut)
- **Prénom** - Prénom du client
- **Nom de famille** - Nom de famille du client (triable)
- **Email** - Adresse email de contact
- **Adresse de facturation** - Adresse de facturation complète
- **Date de création** - Date à laquelle le client a été ajouté
- **Actions** - Menu à trois points

### Pied de Page

- Nombre de clients avec bouton de synchronisation

## Fonctionnalités Clés

### Recherche de Clients

Trouvez des clients rapidement par :

- Prénom ou nom de famille
- Adresse email
- Informations d'adresse

### Ajouter un Nouveau Client

Créez des clients directement depuis le POS :

1. Cliquez sur l'icône <Icon name="plus" /> dans l'en-tête
2. Remplissez les détails du client
3. Enregistrez le nouveau client

Le client est créé dans WooCommerce et immédiatement disponible pour les commandes.

### Modifier un Client

Mettez à jour les informations du client :

1. Cliquez sur le menu à trois points d'un client
2. Sélectionnez **Modifier**
3. Modifiez les détails (nom, email, adresses)
4. Enregistrez les modifications

Les modifications se synchronisent automatiquement avec WooCommerce.

### Colonnes Disponibles

| Colonne | Description |
|---------|-------------|
| **Image** | Avatar du client |
| **ID** | ID client WooCommerce |
| **Prénom** | Prénom du client |
| **Nom de famille** | Nom de famille du client |
| **Email** | Adresse email |
| **Rôle** | Rôle de l'utilisateur (Client, Abonné, etc.) |
| **Nom d'utilisateur** | Nom d'utilisateur WordPress |
| **Adresse de facturation** | Informations de facturation |
| **Adresse de livraison** | Informations de livraison |
| **Date de création** | Date à laquelle le client a été ajouté |
| **Date de modification** | Dernière mise à jour |
| **Actions** | Modifier, Synchroniser, Supprimer |

## Actions sur les Clients

Cliquez sur le menu à trois points (⋮) pour options :

- **Modifier** - Modifier les détails du client
- **Synchroniser** - Actualiser le client depuis le serveur
- **Supprimer** - Retirer de la base de données locale

:::note
La suppression d'un client du POS ne le retire que localement. Le client reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

## Utilisation des Clients dans les Commandes

Pour attribuer un client à une commande :

1. Dans le [Panneau du Panier](/pos/cart), cliquez sur le badge du client
2. Recherchez le client par nom ou email
3. Sélectionnez le client
4. Le client est désormais associé à la commande

Les informations du client (adresses de facturation/livraison) sont automatiquement utilisées pour la commande.

## Synchronisation

Les données des clients se synchronisent entre le POS et WooCommerce :

- **Mises à jour en temps réel** - Les changements se synchronisent sur tous les appareils
- **Intégration WooCommerce** - Les clients sont stockés dans WooCommerce
- **Capacité hors ligne** - Consultez les clients même hors ligne
