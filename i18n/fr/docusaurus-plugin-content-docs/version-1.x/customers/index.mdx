:::info Fonctionnalité Pro
L'écran Clients nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs de la version gratuite peuvent sélectionner des clients existants pour les commandes, mais ne peuvent pas voir la liste complète des clients ni modifier leurs détails.
:::

L'écran Clients offre une gestion complète des clients directement au sein du POS. Visualisez, modifiez et créez des clients sans passer à l'interface d'administration de WooCommerce.

{/* Screenshot: customers-screen.png */}

## Aperçu de l'interface {#interface-overview}

### Actions de l'en-tête {#header-actions}

En haut de l'écran :

- **Barre de recherche** - Trouvez des clients par nom, e-mail, etc.
- **Ajouter un client** (<Icon name="plus" />) - Créez un nouveau client
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurez les colonnes visibles

### Tableau des clients {#customer-table}

La zone principale affiche les clients avec :

- **Avatar** - Photo de profil du client (ou image par défaut)
- **Prénom** - Prénom du client
- **Nom de famille** - Nom de famille du client (triable)
- **E-mail** - Adresse e-mail de contact
- **Adresse de facturation** - Adresse de facturation complète
- **Date de création** - Date d'ajout du client
- **Actions** - Menu à trois points

### Pied de page {#footer}

- Nombre de clients avec bouton de synchronisation (<Icon name="arrow-rotate-right" />). **Appui long** pour l'option Effacer et actualiser

## Fonctionnalités clés {#key-features}

### Recherche de clients {#customer-search}

Trouvez rapidement des clients par :

- Prénom ou nom de famille
- Adresse e-mail
- Informations d'adresse

### Ajouter un nouveau client {#add-new-customer}

Créez des clients directement depuis le POS :

1. Cliquez sur l'icône <Icon name="plus" /> dans l'en-tête
2. Renseignez les détails du client
3. Enregistrez le nouveau client

Le client est créé dans WooCommerce et immédiatement disponible pour les commandes.

### Modifier un client {#edit-customer}

Mettez à jour les informations du client :

1. Cliquez sur le menu à trois points sur un client
2. Sélectionnez **Modifier**
3. Modifiez les détails (nom, e-mail, adresses)
4. Enregistrez les modifications

Les changements sont synchronisés automatiquement avec WooCommerce.

## Paramètres d'affichage {#display-settings}

Cliquez sur l'icône des curseurs (<Icon name="sliders" />) pour personnaliser les colonnes visibles.

## Actions sur les clients {#customer-actions}

Cliquez sur le menu à trois points (⋮) pour accéder aux options :

- **Modifier** - Modifier les détails du client
- **Synchroniser** - Actualiser le client depuis le serveur
- **Supprimer** - Retirer de la base de données locale

:::note
La suppression d'un client depuis le POS ne le retire que localement. Le client reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

## Client par défaut (Invité) {#default-customer-guest}

Lorsqu'aucun client n'est sélectionné, les commandes sont passées en tant que commandes **Invité**. Les commandes Invité :

- N'ont aucun nom de client, e-mail ou adresse associés
- Ne peuvent pas être recherchées par client dans l'historique des commandes
- Restent visibles dans l'écran Commandes et dans l'interface d'administration de WooCommerce

Pour éviter les commandes Invité, sélectionnez un client depuis le Panneau du panier avant le passage en caisse. Si votre activité exige un client sur chaque commande, envisagez de former les caissiers à toujours en attribuer un — il n'existe aucun paramètre intégré pour l'imposer.

## Utiliser les clients dans les commandes {#using-customers-in-orders}

Pour attribuer un client à une commande :

1. Dans le [Panneau du panier](/pos/cart), cliquez sur le badge du client
2. Recherchez le client par nom, e-mail ou numéro de téléphone
3. Sélectionnez le client
4. Le client est désormais associé à la commande

Les informations du client (adresses de facturation/livraison) sont automatiquement utilisées pour la commande.

## Synchronisation {#synchronization}

Les données clients se synchronisent entre le POS et WooCommerce :

- **Mises à jour en temps réel** - Les changements se synchronisent sur tous les appareils
- **Intégration WooCommerce** - Les clients sont stockés dans WooCommerce
- **Capacité hors ligne** - Visualisez les clients même hors connexion
