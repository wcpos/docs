:::info Fonctionnalité Pro
L'écran Clients nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs de la version gratuite peuvent sélectionner des clients existants pour les commandes mais ne peuvent pas afficher la liste complète des clients ni modifier leurs détails.
:::

L'écran Clients offre une gestion complète des clients directement dans le POS. Visualisez, modifiez et créez des clients sans passer par l'administration WooCommerce.

{/* Screenshot: customers-screen.png */}

## Aperçu de l'Interface

### Actions de l'En-tête

En haut de l'écran :

- **Barre de recherche** - Rechercher des clients par nom, e-mail, etc.
- **Ajouter un Client** (<Icon name="plus" />) - Créer un nouveau client
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurer les colonnes visibles

### Tableau des Clients

La zone principale affiche les clients avec :

- **Avatar** - Photo de profil du client (ou image par défaut)
- **Prénom** - Prénom du client
- **Nom** - Nom de famille du client (triable)
- **E-mail** - Adresse e-mail de contact
- **Adresse de Facturation** - Adresse de facturation complète
- **Date de Création** - Quand le client a été ajouté
- **Actions** - Menu à trois points

### Pied de Page

- Nombre de clients avec bouton de synchronisation

## Fonctionnalités Principales

### Recherche de Client

Trouvez rapidement des clients par :

- Prénom ou nom
- Adresse e-mail
- Informations d'adresse

### Ajouter un Nouveau Client

Créez des clients directement depuis le POS :

1. Cliquez sur l'icône <Icon name="plus" /> dans l'en-tête
2. Remplissez les détails du client
3. Enregistrez le nouveau client

Le client est créé dans WooCommerce et immédiatement disponible pour les commandes.

### Modifier un Client

Mettre à jour les informations du client :

1. Cliquez sur le menu à trois points d'un client
2. Sélectionnez **Modifier**
3. Changez les détails (nom, e-mail, adresses)
4. Enregistrez les modifications

Les changements se synchronisent automatiquement avec WooCommerce.

### Colonnes Disponibles

| Colonne | Description |
|--------|-------------|
| **Image** | Avatar du client |
| **ID** | ID client WooCommerce |
| **Prénom** | Prénom du client |
| **Nom** | Nom de famille du client |
| **E-mail** | Adresse e-mail |
| **Rôle** | Rôle utilisateur (Client, Abonné, etc.) |
| **Nom d'Utilisateur** | Nom d'utilisateur WordPress |
| **Adresse de Facturation** | Informations de facturation |
| **Adresse de Livraison** | Informations de livraison |
| **Date de Création** | Quand le client a été ajouté |
| **Date de Modification** | Dernière mise à jour |
| **Actions** | Modifier, Synchroniser, Supprimer |

## Actions sur les Clients

Cliquez sur le menu à trois points (⋮) pour les options :

- **Modifier** - Changer les détails du client
- **Synchroniser** - Actualiser le client depuis le serveur
- **Supprimer** - Retirer de la base de données locale

:::note
Supprimer un client du POS ne le supprime que localement. Le client reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

## Utilisation des Clients dans les Commandes

Pour attribuer un client à une commande :

1. Dans le [Panneau du Panier](/pos/cart), cliquez sur le badge client
2. Recherchez le client par nom ou e-mail
3. Sélectionnez le client
4. Le client est maintenant associé à la commande

Les informations du client (adresses de facturation/livraison) sont automatiquement utilisées pour la commande.

## Synchronisation

Les données des clients se synchronisent entre le POS et WooCommerce :

- **Mises à jour en temps réel** - Les modifications se synchronisent sur tous les appareils
- **Intégration WooCommerce** - Les clients sont stockés dans WooCommerce
- **Capacité hors ligne** - Visualisez les clients même hors ligne
