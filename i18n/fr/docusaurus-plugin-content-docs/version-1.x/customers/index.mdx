:::info Fonction Pro
L'écran Customers nécessite [WCPOS Pro](/getting-started/pro-license). Les utilisateurs gratuits peuvent sélectionner des clients existants pour les commandes mais ne peuvent pas voir la liste complète des clients ou modifier les détails des clients.
:::

L'écran Customers offre une gestion complète des clients directement au sein du POS. Visualisez, éditez et créez des clients sans passer à l'interface administrateur de WooCommerce.

{/* Screenshot: customers-screen.png */}

## Aperçu de l'interface {#interface-overview}

### Actions dans l'en-tête {#header-actions}

En haut de l'écran :

- **Barre de recherche** - Trouvez des clients par nom, email, etc.
- **Ajouter un client** (<Icon name="plus" />) - Créez un nouveau client
- **Paramètres d'affichage** (<Icon name="sliders" />) - Configurez les colonnes visibles

### Tableau des clients {#customer-table}

La zone principale affiche les clients avec :

- **Avatar** - Photo de profil du client (ou image par défaut)
- **Prénom** - Prénom du client
- **Nom de famille** - Nom de famille du client (triable)
- **Email** - Adresse email de contact
- **Adresse de facturation** - Adresse de facturation complète
- **Date Created** - Date d'ajout du client
- **Actions** - Menu à trois points

### Pied de page {#footer}

- Nombre de clients avec bouton de synchronisation (<Icon name="arrow-rotate-right" />). **Appuyez longuement** pour l'option Effacer et Actualiser

## Fonctionnalités clés {#key-features}

### Recherche de clients {#customer-search}

Trouvez rapidement des clients par :

- Prénom ou nom de famille
- Adresse email
- Informations d'adresse

### Ajouter un nouveau client {#add-new-customer}

Créez des clients directement depuis le POS :

1. Cliquez sur l'icône <Icon name="plus" /> dans l'en-tête
2. Remplissez les détails du client
3. Enregistrez le nouveau client

Le client est créé dans WooCommerce et immédiatement disponible pour les commandes.

### Éditer un client {#edit-customer}

Mettez à jour les informations du client :

1. Cliquez sur le menu à trois points sur un client
2. Sélectionnez **Éditer**
3. Modifiez les détails (nom, email, adresses)
4. Enregistrez les modifications

Les changements sont synchronisés automatiquement avec WooCommerce.

## Paramètres d'affichage {#display-settings}

Cliquez sur l'icône de curseurs (<Icon name="sliders" />) pour personnaliser les colonnes visibles.

### Colonnes disponibles {#available-columns}

| Colonne | Description |
|---------|-------------|
| **Image** | Avatar du client |
| **ID** | ID du client WooCommerce |
| **Prénom** | Prénom du client |
| **Nom de famille** | Nom de famille du client |
| **Email** | Adresse email |
| **Rôle** | Rôle de l'utilisateur (Client, Abonné, etc.) |
| **Nom d'utilisateur** | Nom d'utilisateur WordPress |
| **Adresse de facturation** | Informations de facturation |
| **Adresse de livraison** | Informations de livraison |
| **Date Created** | Date d'ajout du client |
| **Date Modified** | Dernière mise à jour |
| **Actions** | Éditer, Synchroniser, Supprimer |

## Actions sur les clients {#customer-actions}

Cliquez sur le menu à trois points (⋮) pour les options :

- **Éditer** - Modifier les détails du client
- **Synchroniser** - Actualiser le client depuis le serveur
- **Supprimer** - Retirer de la base de données locale

:::note
La suppression d'un client du POS ne le retire que localement. Le client reste dans WooCommerce et réapparaîtra lors de la prochaine synchronisation.
:::

## Utiliser les clients dans les commandes {#using-customers-in-orders}

Pour attribuer un client à une commande :

1. Dans le [Cart Panel](/pos/cart), cliquez sur le badge du client
2. Recherchez le client par nom ou email
3. Sélectionnez le client
4. Le client est maintenant associé à la commande

Les informations du client (adresses de facturation/livraison) sont automatiquement utilisées pour la commande.

## Synchronisation {#synchronization}

Les données clients se synchronisent entre le POS et WooCommerce :

- **Mises à jour en temps réel** - Les changements se synchronisent sur tous les appareils
- **Intégration WooCommerce** - Les clients sont stockés dans WooCommerce
- **Capacité hors ligne** - Visualisez les clients même hors ligne
