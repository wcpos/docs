---
title: Servidores Legados
description: Se o seu provedor de hospedagem bloquear métodos HTTP RESTful modernos, você precisará ativar o suporte a servidores legados.
---

HTTP (e HTTPS) são os protocolos usados por todos os navegadores para se comunicar com aplicações web.
Uma parte essencial do protocolo é o [método HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Tradicionalmente, as aplicações web usavam um conjunto muito limitado de métodos HTTP, como GET (recuperar uma página web) e POST (enviar um formulário web).

Aplicações web modernas, como o WCPOS, usam uma abordagem REST/HTTP com os seguintes métodos HTTP:

- GET
- POST
- PUT
- PATCH
- DELETE

Em alguns casos raros, um provedor de hospedagem pode optar por bloquear alguns desses métodos HTTP.
Se o seu provedor de hospedagem bloquear métodos HTTP RESTful modernos, você precisará ativar o suporte a servidores legados.

![Suporte a Servidores Legados](https://wcpos.com/wp-content/uploads/2016/06/enable-legacy-server-support.png "Administração do WP > POS > Configurações > Ferramentas > Ativar suporte a servidores legados")
