---
title: Servidores Legados
description: Se o seu provedor de hospedagem bloquear métodos HTTP RESTful modernos, você precisará habilitar o suporte a servidores legados.
---

HTTP (e HTTPS) são os protocolos usados por todos os navegadores para se comunicar com aplicativos web. 
Uma parte fundamental do protocolo é o [método HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1). 
Tradicionalmente, aplicativos web usavam um conjunto muito limitado de métodos HTTP, como GET (recuperar uma página da web) e POST (enviar um formulário da web).

Aplicativos web modernos, como o WCPOS, utilizam uma abordagem REST/HTTP com os seguintes métodos HTTP:

- GET
- POST
- PUT
- PATCH
- DELETE

Em alguns casos raros, um provedor de hospedagem pode optar por bloquear alguns desses métodos HTTP. 
Se o seu provedor de hospedagem bloquear métodos HTTP RESTful modernos, você precisará habilitar o suporte a servidores legados.

![Suporte a Servidores Legados](https://wcpos.com/wp-content/uploads/2016/06/enable-legacy-server-support.png "WP Admin > POS > Settings > Tools > Enable legacy server support")
