WCPOS é o plugin de Ponto de Venda para [WooCommerce](https://www.woothemes.com/woocommerce/).
Esta documentação cobre o plugin de código aberto, assim como o plugin Pro que adiciona funcionalidade ao plugin gratuito.

- Baixe [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) do WordPress.org
- Faça o upgrade para [WCPOS Pro](http://wcpos.com/pro)

:::danger

Por favor, certifique-se de que o número da versão da documentação corresponde à versão dos plugins instalados.

:::
