WCPOS é o plugin de Ponto de Venda para [WooCommerce](https://www.woothemes.com/woocommerce/).
Esta documentação abrange o plugin de código aberto, assim como o plugin Pro, que adiciona funcionalidades ao plugin gratuito.

- Baixe o [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) no WordPress.org
- Atualize para o [WCPOS Pro](http://wcpos.com/pro)

:::danger

Certifique-se de que o número da versão da documentação corresponde à versão dos seus plugins instalados.

:::
