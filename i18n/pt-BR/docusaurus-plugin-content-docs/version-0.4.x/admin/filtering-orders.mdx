---
title: Filtrando pedidos por vendas no POS ou Online
description: Separe pedidos do POS e Online usando os links rápidos no painel administrativo do WooCommerce.
---

É possível separar rapidamente os pedidos do POS e Online no painel administrativo do WooCommerce usando os links rápidos no canto superior esquerdo.
**Observação:** isso filtrará apenas pedidos do POS a partir da versão 0.3.1; pedidos anteriores à versão 0.3.1 serão exibidos como Online por padrão.

![Filtrar Pedidos](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

Além disso, para filtrar todos os pedidos concluídos do painel administrativo e focar nos pedidos aguardando pagamento/processando, é possível adicionar o seguinte código ao arquivo [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
