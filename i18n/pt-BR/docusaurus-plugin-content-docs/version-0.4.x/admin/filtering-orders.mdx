---
title: Filtrando pedidos por vendas POS ou Online
description: Separe pedidos POS e Online usando os links rápidos no admin do WooCommerce.
---

Você pode rapidamente separar seus pedidos POS e Online no admin do WooCommerce usando os links rápidos no canto superior esquerdo.
**Por favor, note:** isso filtrará apenas pedidos POS após a versão 0.3.1, pedidos anteriores à 0.3.1 terão como padrão Online.

![Filtrar Pedidos](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrar Pedidos")

Além disso, se você quiser filtrar todos os pedidos concluídos da sua área de admin para que você possa se concentrar nos pedidos pendentes/processando, você pode adicionar o seguinte código ao seu arquivo [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
