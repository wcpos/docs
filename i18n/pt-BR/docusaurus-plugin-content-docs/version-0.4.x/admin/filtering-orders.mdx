---
title: Filtrando pedidos por vendas em PDV ou Online
description: Separe pedidos de PDV e Online usando os links rápidos na administração do WooCommerce.
---

Você pode rapidamente separar seus pedidos de PDV e Online na administração do WooCommerce usando os links rápidos no canto superior esquerdo.
**Observe:** isso só filtrará pedidos de PDV a partir da versão 0.3.1, pedidos anteriores à 0.3.1 serão considerados Online por padrão.

![Filtrar Pedidos](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrar Pedidos")

Adicionalmente, se você quisesse filtrar todos os pedidos concluídos da sua área administrativa para se concentrar nos pedidos pendentes/em processamento, pode adicionar o seguinte código ao seu arquivo [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Alterar filtro padrão de pedidos no admin do WordPress
 * adicione isso ao seu arquivo functions.php
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
