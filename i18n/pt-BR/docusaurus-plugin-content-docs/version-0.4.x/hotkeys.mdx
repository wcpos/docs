Teclas de atalho são atalhos de teclado que podem ser usados para acionar rapidamente ações no WCPOS.

| Tecla                       | Descrição        |
| --------------------------- | ---------------- |
| <div align="center">?</div> | Tela de Ajuda    |
| <div align="center">B</div> | Modo de Código de Barras |
| <div align="center">V</div> | Modo de Busca    |
| <div align="center">S</div> | Sincronizar com o servidor |
