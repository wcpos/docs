Hotkeys são atalhos de teclado que podem ser usados para acionar rapidamente ações no WCPOS.

| Tecla                       | Descrição         |
| --------------------------- | ----------------- |
| <div align="center">?</div> | Tela de ajuda     |
| <div align="center">B</div> | Modo de código de barras |
| <div align="center">V</div> | Modo de busca     |
| <div align="center">S</div> | Sync com o servidor |
