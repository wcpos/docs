### Instalação automática (recomendado)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Instale e ative o WCPOS em menos de 1 minuto")

1. Vá para `WP Admin > Plugins > Adicionar Novo`
2. Procure por `WCPOS` no Diretório de Plugins do WordPress
3. Instale o plugin
4. Ative o plugin

### Instalação manual

1. [Baixe o WCPOS](https://wordpress.org/plugins/woocommerce-pos/) no WordPress.org
2. Vá para `WP Admin > Plugins > Adicionar Novo > Carregar Plugin`
3. Ative o plugin

:::info
Versões anteriores do WCPOS podem ser encontradas na aba [Desenvolvedores](https://wordpress.org/plugins/woocommerce-pos/developers/) no Wordpress.org.
:::
