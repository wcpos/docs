### Instalação automática (recomendada) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Install and activate WCPOS in less than 1 minute")

1. Acesse `WP Admin > Plugins > Adicionar Novo`
2. Pesquise por `WCPOS` no Diretório de Plugins do WordPress
3. Instale o plugin
4. Ative o plugin

### Instalação manual {#manual-installation}

1. [Baixe o WCPOS](https://wordpress.org/plugins/woocommerce-pos/) do WordPress.org
2. Acesse `WP Admin > Plugins > Adicionar Novo > Enviar plugin`
3. Ative o plugin

:::info
Versões anteriores do WCPOS podem ser encontradas na aba [Desenvolvedores](https://wordpress.org/plugins/woocommerce-pos/developers/) no WordPress.org.
:::
