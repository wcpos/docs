:::important

[WCPOS Pro](http://wcpos.com/pro) é um plugin pago que adiciona funcionalidades ao plugin [WCPOS gratuito](http://wordpress.org/plugins/woocommerce-pos/). 

:::

Se você adquiriu uma licença do WCPOS Pro, siga as etapas abaixo para instalar e ativar o plugin:

1.  Acesse: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Em **Downloads**, clique no link de download e salve o plugin na sua área de trabalho.
3.  Em seguida, acesse seu site, faça login e vá para `WP Admin > Plugins > Adicionar Novo > Enviar Plugin`
4.  Faça o upload do arquivo zip do plugin da sua área de trabalho e ative-o.
5.  Depois, vá para `WP Admin > POS > Configurações > Licença` e insira sua Chave de Licença e E-mail de Licença para concluir a ativação.

![Meus Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Você pode baixar o WCPOS Pro na página da sua conta")

Quando novas versões estiverem prontas, elas aparecerão no seu painel de Atualizações. 
Para obter informações sobre o que estou trabalhando no momento, fique de olho no [Roteiro](http://wcpos.com/roadmap/). 
Também acompanhe o registro de alterações, que listará os novos recursos de cada versão, e a [página de documentação](http://docs.wcpos.com) será atualizada com informações sobre como usar os novos recursos.
