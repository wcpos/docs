:::important

[WCPOS Pro](http://wcpos.com/pro) é um plugin pago que adiciona funcionalidades ao plugin gratuito [WCPOS](http://wordpress.org/plugins/woocommerce-pos/). 

:::

Se você comprou uma licença para o WCPOS Pro, siga os passos abaixo para instalar e ativar o plugin:

1.  Acesse: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Em **Downloads**, clique no link de download e salve o plugin na sua área de trabalho.
3.  Em seguida, vá para o seu site, faça login e vá para `WP Admin > Plugins > Adicionar Novo > Enviar Plugin`
4.  Faça o upload do arquivo zip do plugin da sua área de trabalho e ative.
5.  Por fim, vá para `WP Admin > POS > Configurações > Licença` e insira sua Chave de Licença e Email da Licença para completar a ativação.

![Meus Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Você pode baixar o WCPOS Pro na sua página de conta")

Quando novas versões estiverem prontas, elas aparecerão no seu painel de Atualizações. 
Para informações sobre o que estou trabalhando atualmente, fique de olho no [Roadmap](http://wcpos.com/roadmap/). 
Também fique atento ao registro de alterações, que listará os novos recursos para cada versão, e a [página de documentação](http://docs.wcpos.com) será atualizada com informações sobre como usar os novos recursos.
