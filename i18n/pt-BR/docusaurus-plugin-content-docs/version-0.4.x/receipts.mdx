Os modelos de recibos podem ser personalizados criando um arquivo `woocommerce-pos/print/tmpl-receipt.php` no diretório do seu tema. O código para ambos os modelos está incluído abaixo.

### Personalizando a data do recibo

WCPOS usa [moment.js](http://momentjs.com/) para localizar strings de data. O formato de data padrão é `"MMMM Do YYYY, h:mm:ss a"`, por exemplo: 31 de maio de 2015, 19:20:44. Consulte a [documentação do moment.js](http://momentjs.com/docs/#/parsing/string-format/) para mais informações sobre como personalizar o formato da data.

### Propriedades do Pedido

WCPOS usa a saída json da API REST do WC para preencher o modelo de recibo do pedido. [A documentação da API REST do WC mostra um exemplo](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) da saída json padrão. Algumas propriedades adicionais foram adicionadas pelo WCPOS usando o filtro `woocommerce_api_order_response`.

| Propriedade                       | Descrição                                                     |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | A parte do imposto do desconto do carrinho                    |
| cashier.id                        | ID do Usuário do Caixa                                        |
| cashier.first_name                | Primeiro nome do Caixa                                        |
| cashier.last_name                 | Último nome do Caixa                                          |
| payment_details.result            | Sucesso ou falha do gateway de pagamento                      |
| payment_details.message           | Quaisquer mensagens do gateway de pagamento                   |
| payment_details.redirect          | URL de redirecionamento para pagamentos fora do site, ex: PayPal Standard |
| payment_details.method\_`gateway` | Mensagens específicas do gateway, ex: Valor Recebido e Troco  |
| shipping_lines[i].total_tax       | Adicionar valor do imposto para cada linha de envio           |
| subtotal_tax                      | A parte do imposto do subtotal                                |
