### Visibilidade de Produtos

Produtos exclusivos de PDV devem ser habilitados apenas se você precisar ocultar alguns produtos da loja online ou física.
Consulte o documento [Produtos Exclusivos de PDV](../products/pos-only-products) para mais informações.

### Cliente Padrão de PDV

Um novo pedido será inicializado com o cliente padrão de PDV. Opções comuns são:

- **Visitante:** Pedidos de PDV não são associados a nenhum ID de usuário
- **Caixa:** Pedidos de PDV são associados à conta do usuário conectado
- **Cliente Avulso:** Pedidos de PDV são associados a uma conta de usuário personalizada (_note: essa conta precisará ser criada_)

Usuários do [WCPOS Pro](https://wcpos.com/pro) podem criar contas de usuário _instantaneamente_ durante o checkout.
Um endereço de e-mail é necessário para criar uma nova conta, este é um requisito do WordPress.
**Criar uma conta para novos clientes é recomendado** pois permite rastrear o histórico de pedidos e potencialmente entrar em contato com seus clientes com ofertas promocionais no futuro.
