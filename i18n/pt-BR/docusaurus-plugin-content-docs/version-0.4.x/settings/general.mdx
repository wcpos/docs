### Visibilidade do Produto

Os produtos somente para POS devem ser ativados apenas se você precisar que alguns produtos fiquem ocultos na loja online ou física.  
Consulte a documentação sobre [Produtos Somente para POS](../products/pos-only-products) para mais informações.

### Cliente Padrão do POS

Um novo pedido será inicializado com o cliente padrão do POS. As opções comuns são:

- **Convidado:** Os pedidos do POS não estão associados a nenhum ID de usuário
- **Caixa:** Os pedidos do POS estão associados à conta do usuário logado
- **Cliente de Passagem:** Os pedidos do POS estão associados a uma conta de usuário personalizada (_observação: essa conta precisará ser criada_)

Usuários do [WCPOS Pro](https://wcpos.com/pro) podem criar contas de usuário _em tempo real_ durante o checkout.  
Um endereço de e-mail é necessário para criar uma nova conta, isso é uma exigência do WordPress.  
**Criar uma conta para novos clientes é recomendado**, pois isso significa que você pode rastrear o histórico de pedidos e potencialmente contatar seus clientes com ofertas promocionais no futuro.
