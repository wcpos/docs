Usando as configurações de Acesso ao PDV, proprietários de lojas podem conceder ou remover privilégios de usuário no sistema de Ponto de Venda.
Por padrão, os papéis **Administrador** e **Gerente de Loja** estão habilitados com todas as capacidades para usar o PDV.
Recomenda-se manter as configurações padrão, a menos que você esteja ciente das consequências.
Por exemplo, conceder a permissão `publish_shop_orders` para o papel de `Assinantes` seria uma ideia _muito_ ruim.

![Exemplo de configurações de Acesso ao PDV](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Exemplo de configurações de Acesso ao PDV")

| Capacidade                                                              | Acesso                                                |
| ----------------------------------------------------------------------- | ----------------------------------------------------- |
| access_woocommerce_pos                                                  | Necessário para acessar o frontend do PDV             |
| manage_woocommerce_pos                                                  | Necessário para acessar as configurações administrativas do PDV |
| read_private_products                                                   | Necessário para visualizar produtos                   |
| edit_product <br /> edit_published_products <br /> edit_others_products | Necessário para editar produtos e variações           |
| read_private_shop_orders                                                | Necessário para visualizar pedidos e recibos          |
| publish_shop_orders                                                     | Necessário para criar novos pedidos                   |
| list_users                                                              | Necessário para visualizar clientes                   |
| create_users                                                            | Necessário para criar novos clientes                  |
| edit_users                                                              | Necessário para atualizar clientes existentes         |
| read_private_shop_coupons                                               | Necessário para visualizar cupons                     |
