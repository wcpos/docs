---
title: "Produtos Somente para POS"
description: "Os produtos podem ser ocultados em sua loja online ou no ponto de venda usando a configuração de visibilidade do POS."
---

Pode haver situações em que você precisa ocultar um produto da sua loja online ou do seu POS. Com o WCPOS, isso pode ser realizado usando a configuração de Visibilidade do POS na página de edição do produto.

### Habilitando Produtos Somente para POS {#enabling-pos-only-products}

Produtos Somente para POS não estão habilitados por padrão. Para habilitar Produtos Somente para POS, vá para <code>WP Admin > POS > Configurações > Geral</code>.

![Habilitar Produtos Somente para POS](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "Habilitar Produtos Somente para POS")

Habilitar produtos somente para POS exigirá uma consulta adicional ao banco de dados e, portanto, resultará em uma pequena perda de desempenho. Se você não usar Produtos Somente para POS, é recomendado que deixe essa opção desabilitada.

### Visibilidade POS & Online {#pos-online-visibility}

![Visibilidade POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "Configurações de Visibilidade do POS na página de edição do Produto")

| Configuração                     | Descrição                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------ |
| **POS & Online**<br />(padrão)  | Os produtos estarão visíveis tanto online em sua loja virtual quanto em sua loja física   |
| **Somente POS**                  | A página do produto não será visível online                                                |
| **Somente Online**               | O produto não será visível no POS                                                          |

### Visibilidade do Catálogo {#catalog-visibility}

Se você acessar diretamente uma página de Produto Somente para POS, você receberá <code>404 Página Não Encontrada</code>. No entanto, Produtos Somente para POS ainda podem aparecer na pesquisa online e nas páginas de categoria. Para ocultar os produtos completamente do seu site, você deve definir a Visibilidade do Catálogo como Oculta.

![Visibilidade do Catálogo](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "Configurações de Visibilidade do Catálogo na página de edição do Produto")

### Filtrar Produtos Somente para POS {#filter-pos-only-products}

Você pode ver rapidamente quais produtos estão visíveis/ocultos usando os links rápidos na página de administração de Produtos:

![Filtro de Visibilidade POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "Filtro de Visibilidade POS")

### Edição em Massa {#bulk-edit}

Para alterar a Visibilidade do POS de vários produtos, vá para sua lista de Produtos e selecione <code>Editar</code> nas <code>Ações em Massa</code>.

![Edição em Massa de Visibilidade POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "Edição em Massa de Visibilidade POS")
