---
title: Produtos Diversos
---

Pode haver casos em que uma loja precisa vender itens que não fazem parte do estoque do WooCommerce.
Abaixo estão duas opções para vender 'produtos diversos' no WCPOS.

### Produto Apenas para POS (recomendado)

![Produto Apenas para POS](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Criar um Produto Apenas para POS")

Usando o recurso de [Apenas para POS](./pos-only-products), você pode criar um 'produto diverso' para uso no PDV.
O título, preço de venda, preço regular, alíquota de imposto e informações de metadados podem ser alterados instantaneamente durante o checkout.

![Editar Produto Diverso](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Editar o Produto Diverso no carrinho")

### Item de linha de taxa

![Item de linha de taxa](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Editar o Item de linha de taxa")

Usar um item de linha de taxa é semelhante a criar um Produto Apenas para POS.
Os caixas podem atualizar rapidamente o título, o preço e a alíquota de imposto da taxa.
A principal desvantagem é que as taxas podem aparecer separadas dos produtos em seus relatórios.
