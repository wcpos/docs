Pode haver casos em que uma loja precisa vender itens que não fazem parte do estoque do WooCommerce.
Abaixo estão duas opções para vender 'produtos diversos' no WCPOS.

### Produto Diverso Apenas para POS (recomendada) {#pos-only-miscellaneous-product-recommended}

![Produto Diverso Apenas para POS](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Criar um Produto Diverso Apenas para POS")

Usando o recurso [Produto Apenas para POS](./pos-only-products), você pode criar um 'produto diverso' para uso no POS.
O título, preço de venda, preço regular, taxa de imposto e informações meta podem ser alterados rapidamente durante o checkout.

![Editar o Produto Diverso](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Editar o Produto Diverso no carrinho")

### Item de taxa {#fee-line-item}

![Item de taxa](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Editar o Item de taxa")

Usar um item de taxa é semelhante a criar um Produto Diverso Apenas para POS.
Os caixas podem rapidamente atualizar o título, preço e taxa de imposto da taxa.
A principal desvantagem é que as taxas podem aparecer separadas dos produtos em seus relatórios.
