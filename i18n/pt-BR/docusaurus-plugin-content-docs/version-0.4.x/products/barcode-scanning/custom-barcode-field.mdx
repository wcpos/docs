---
title: Campo de Código de Barras Personalizado
---

Por padrão, o WCPOS usará o campo SKU como código de barras.

![Por padrão, o WCPOS usará o campo SKU como código de barras.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Por padrão, o WCPOS usará o campo SKU como código de barras")

Usuários do [WCPOS Pro](http://wcpos.com/pro) podem usar qualquer campo de meta de produto selecionando um Campo de Código de Barras em `WP Admin > POS > Configurações > Geral`

![Selecionar campo de código de barras personalizado no WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Selecionar campo de código de barras personalizado no WCPOS Pro")

Campos de meta de código de barras personalizados também podem ser definidos usando o filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Campo de meta de produto personalizado para usar como código de barras
 *
 * Por padrão, o WCPOS usará o campo SKU - '_sku'
 * Você pode alterar a chave de meta usando o seguinte filtro do WordPress
 *
 * Abaixo estão campos de meta de plugins de código de barras populares
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // altere para sua chave de meta personalizada
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Um código de barras deve ser único para cada produto.
:::
