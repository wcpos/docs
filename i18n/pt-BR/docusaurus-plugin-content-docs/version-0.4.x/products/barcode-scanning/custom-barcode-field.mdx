---
title: Campo de Código de Barras Personalizado
---

Por padrão, o WCPOS usará o campo SKU como o código de barras.

![Por padrão, o WCPOS usará o campo SKU como o código de barras.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Por padrão, o WCPOS usará o campo SKU como o código de barras.")

Usuários do [WCPOS Pro](http://wcpos.com/pro) podem usar qualquer campo meta de produto selecionando um Campo de Código de Barras em `WP Admin > POS > Configurações > Geral`

![Selecione o campo de código de barras personalizado no WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Selecione o campo de código de barras personalizado no WCPOS Pro")

Campos meta personalizados para códigos de barras também podem ser definidos usando o filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Um código de barras deve ser único para cada produto.
:::
