---

O WCPOS é capaz de filtrar produtos pelo campo de SKU. Se uma busca corresponder exatamente ao campo de código de barras, o produto é adicionado ao carrinho instantaneamente e o campo de busca é reiniciado. Se você tiver um leitor de código de barras capaz de preencher o campo de busca, você poderá usar o WCPOS para escanear códigos de barras.

Para ver a busca por SKU em ação, acesse http://demo.wcpos.com/pos/

Digite (ou copie e cole) `BARCODE` e um item será adicionado ao carrinho.

Digite (ou copie e cole) `VARIATION1` e uma variação será adicionada ao carrinho.

![Escaneamento de Código de Barras no WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtrando pelo campo de SKU no WCPOS")

Os seguintes recursos estão planejados para versões futuras:

- Adicionar produtos ao WooCommerce usando leitor de código de barras. Isso será parte de um modo de inventário planejado para uma versão 0.3.x.
- Escanear códigos de barras usando a câmera do dispositivo móvel. Isso exigirá um aplicativo nativo do WCPOS para acessar o hardware do dispositivo. Aplicativos nativos estão na lista de desejos, mas adicionar recursos à versão web é a prioridade máxima no momento.
- Ativação por tecla de atalho

### Atualizações

**Desde a Versão 0.3.1:** Agora existe um modo dedicado para escaneamento de código de barras. Você pode alternar entre o modo 'Busca' e 'Escanear Código de Barras' clicando no botão à esquerda do campo de busca.
