---
title: Leitura de Código de Barras
description: O WCPOS pode filtrar produtos pelo campo SKU, permitindo adicionar produtos ao carrinho instantaneamente usando um leitor de código de barras.
---

O WCPOS pode filtrar produtos pelo campo SKU. Se uma busca corresponder exatamente ao campo de código de barras, o produto é adicionado ao carrinho instantaneamente e o campo de busca é redefinido. Se você possui um leitor de código de barras capaz de preencher o campo de busca, é possível utilizar o WCPOS para leitura de códigos de barras.

Para ver a busca por SKU em ação, acesse http://demo.wcpos.com/pos/

Digite (ou copie e cole) `BARCODE` e um item será adicionado ao carrinho.

Digite (ou copie e cole) `VARIATION1` e uma variação será adicionada ao carrinho.

![Leitura de Código de Barras no WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtering by the SKU field in WCPOS")

Os seguintes recursos estão sendo planejados para versões futuras:

- Adicionar produtos ao WooCommerce usando leitor de código de barras. Isso fará parte de um modo de inventário planejado para uma versão 0.3.x.
- Leitura de código de barras usando a câmera do dispositivo móvel. Isso exigirá um aplicativo nativo do WCPOS para acessar o hardware do dispositivo. Aplicativos nativos estão na lista de desejos, mas adicionar recursos à versão web é a maior prioridade no momento.
- Ativação por tecla de atalho

### Atualizações {#updates}

**Desde a Versão 0.3.1:** Agora existe um modo dedicado para leitura de código de barras. Você pode alternar entre o modo 'Pesquisar' e 'Ler Código de Barras' clicando no botão à esquerda do campo de pesquisa.

![Modos Pesquisar e Ler Código de Barras](https://wcpos.com/wp-content/uploads/2014/07/search-scan-mode.png)
