O WCPOS utiliza uma série de componentes de teclados numéricos para permitir a entrada rápida usando o mouse ou a tela de toque. 
Abaixo estão exemplos dos diferentes tipos de teclados numéricos:

![Teclado numérico de quantidade com entrada rápida para +1 e -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Teclado numérico de quantidade com entrada rápida para +1 e -1")

![Teclado numérico de preço do item com entrada rápida para porcentagem de desconto.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Teclado numérico de preço do item com entrada rápida para porcentagem de desconto")

![Teclado numérico de quantia recebida com entrada rápida para quantias em dinheiro prováveis.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Teclado numérico de quantia recebida com entrada rápida para quantias em dinheiro prováveis.")

![Teclado numérico de cashback com entrada rápida para quantias comuns de cashback.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Teclado numérico de cashback com entrada rápida para quantias comuns de cashback")
