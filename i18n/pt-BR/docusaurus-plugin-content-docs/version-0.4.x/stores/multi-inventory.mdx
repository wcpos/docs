---
title: "Lojas de Múltiplos Inventários"
---

:::caution
Multi-inventário atualmente não é um recurso do plugin Pro.
As informações a seguir foram projetadas para ajudar os usuários até que o WCPOS Pro suporte oficialmente múltiplos inventários.
:::

Lojas de múltiplos inventários é uma das funcionalidades mais solicitadas para o WCPOS Pro.
Esse recurso permitiria aos proprietários de lojas atribuir produtos a diferentes lojas e gerenciar o estoque entre cada loja física e a loja online.

Criar a funcionalidade de múltiplos inventários é uma tarefa não trivial.
Existem alguns plugins já trabalhando na questão, como [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646) e [WooCommerce MultiStore](http://woomultistore.com/).
Atualmente, faz sentido focar o desenvolvimento em outros recursos principais do POS e integrar-se a um desses plugins assim que eles amadurecem para soluções estáveis.

A curto prazo, você pode ser capaz de atender à maioria dos seus requisitos usando o [recurso MultiSite do WordPress](https://codex.wordpress.org/Create_A_Network).
Ao criar um site para cada loja, você pode então instalar WooCommerce e WCPOS em cada site, mantendo o inventário de cada loja completamente separado.
O WooCommerce Multisite (mencionado acima) utiliza essa abordagem com uma camada adicional para gerenciar o inventário entre os sites.
