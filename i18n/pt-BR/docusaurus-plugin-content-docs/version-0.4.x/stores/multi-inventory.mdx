:::caution
Múltiplos estoques atualmente não são um recurso do plugin Pro.
As seguintes informações destinam-se a auxiliar os usuários até que o WCPOS Pro ofereça suporte oficial a múltiplos estoques.
:::

Lojas com múltiplos estoques é um dos recursos mais solicitados para o WCPOS Pro.
Esse recurso permitiria que os proprietários de lojas atribuíssem produtos a diferentes lojas e gerenciassem o estoque entre cada loja física e a loja online.

Criar funcionalidade de múltiplos estoques não é uma tarefa simples.
Já existem alguns plugins trabalhando nesse problema, como [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646) e [WooCommerce MultiStore](http://woomultistore.com/).
Atualmente, faz mais sentido focar o desenvolvimento em outros recursos principais do POS e integrar com um desses plugins quando eles se tornarem soluções estáveis.

A curto prazo, você pode ser capaz de atender à maioria dos seus requisitos usando o [recurso de MultiSite do WordPress](https://codex.wordpress.org/Create_A_Network).
Ao criar um site para cada loja, você pode instalar o WooCommerce e o WCPOS em cada site, mantendo o estoque de cada loja completamente separado.
O WooCommerce Multisite (mencionado acima) usa essa abordagem com uma camada adicional para gerenciar o estoque entre os sites.
