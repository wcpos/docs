---
title: "Lojas"
---

:::important
Este recurso requer um upgrade para [WCPOS Pro](http://wcpos.com/pro).
:::

![Exemplo de lista de lojas](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Exemplo de lista de lojas")

WCPOS Pro adiciona a capacidade de criar lojas separadas para suas vendas físicas.
As lojas podem ter seu próprio endereço, logotipo e outros detalhes para uso no [modelo de recibo](../receipts).
As lojas podem usar diferentes taxas de imposto e as vendas podem ser separadas por loja para [relatórios](../reports#sales-by-store).

![Exemplo de página de edição de loja](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Exemplo de página de edição de loja")

### Recursos futuros {#coming-features}

O recurso Lojas foi adicionado na versão 0.4.
Existem vários _recursos frequentemente solicitados_, como [lojas de multi-inventário](./multi-inventory) e a capacidade de atribuir caixas a lojas.
Esses recursos serão adicionados ao plugin Pro conforme o plugin se aproxima da versão 1.0.
