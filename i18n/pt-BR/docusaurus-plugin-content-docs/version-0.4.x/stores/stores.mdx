---
title: Lojas
---

:::important
Este recurso requer uma atualização para o [WCPOS Pro](http://wcpos.com/pro).
:::

![Exemplo de lista de lojas](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Exemplo de lista de lojas")

O WCPOS Pro adiciona a capacidade de criar _lojas_ separadas para suas vendas físicas.
Lojas podem ter seu próprio endereço, logotipo e outros detalhes para uso no [modelo de recibo](../receipts).
Lojas podem usar diferentes taxas de imposto e as vendas podem ser separadas por loja para [relatórios](../reports#sales-by-store).

![Exemplo de página de edição de loja](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Exemplo de página de edição de loja")

### Recursos futuros

O recurso de Lojas foi adicionado na versão 0.4.
Existem vários _recursos frequentemente solicitados_, como [lojas de multi-inventário](./multi-inventory) e a capacidade de atribuir caixas a lojas.
Esses recursos serão adicionados ao plugin Pro à medida que o plugin se aproxima da versão 1.0.
