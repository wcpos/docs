:::warning Importante
Na carga inicial, o POS obterá as configurações de tributo padrão do WooCommerce. Em geral, os usuários devem manter as mesmas configurações do WooCommerce, a menos que tenham uma necessidade específica. Ter configurações de tributo diferentes do servidor quase certamente fará com que o POS e o WooCommerce calculem valores de tributos diferentes, o que pode levar a discrepâncias e problemas com a sincronização de pedidos.
:::

## Visualizar Todas as Taxas de Tributo {#view-all-tax-rates}

Clique no botão **Visualizar todas as taxas de tributo** para ver todas as taxas de tributo que foram baixadas para o POS. Isso mostra:

- Nomes das taxas de tributo
- Percentuais das taxas de tributo
- Regiões/localizações aplicáveis

Isso é útil para verificar se as taxas de tributo corretas estão carregadas e para depurar problemas de cálculo de tributo.

## Configurações de Cálculo de Tributo {#tax-calculation-settings}

### Preços Informados Com Tributo {#prices-entered-with-tax}

Determina se os preços dos produtos incluem tributo:

| Opção | Descrição |
|-------|-----------|
| **Sim** | Os preços dos produtos incluem tributo. O POS calcula o tributo trabalhando para trás a partir do preço total |
| **Não** | Os preços dos produtos não incluem tributo. O POS adiciona o tributo ao preço informado |

### Calcular Tributo Com Base Em {#calculate-tax-based-on}

Determina o endereço usado para o cálculo do tributo:

| Opção | Descrição |
|-------|-----------|
| **Endereço de entrega do cliente** | Taxas de tributo com base em onde o pedido será enviado |
| **Endereço de cobrança do cliente** | Taxas de tributo com base no endereço de cobrança do cliente |
| **Endereço base da loja** | Taxas de tributo com base na localização física da sua loja |

### Classe de Tributo de Envio {#shipping-tax-class}

Determina como os custos de envio são taxados:

| Opção | Descrição |
|-------|-----------|
| **Classe de tributo de envio com base nos itens do carrinho** | O tributo de envio é calculado com base na classe de tributo dos itens no carrinho |
| **Padrão** | O envio é taxado à taxa de tributo padrão |
| **Taxa reduzida** | O envio é taxado a uma taxa de tributo reduzida |
| **Taxa zero** | Nenhum tributo é aplicado aos custos de envio |

### Arredondamento {#rounding}

| Opção | Descrição |
|-------|-----------|
| **Arredondar tributo no nível de subtotal** | O tributo é calculado para o subtotal completo e depois arredondado, em vez de arredondar o tributo para cada item individual |

## Configurações de Exibição {#display-settings}

### Exibir Preços na Loja {#display-prices-in-the-shop}

| Opção | Descrição |
|-------|-----------|
| **Incluindo tributo** | Os preços exibidos incluem os valores do tributo |
| **Excluindo tributo** | Os preços exibidos não incluem o tributo |

### Exibir Preços Durante o Carrinho e Checkout {#display-prices-during-cart-and-checkout}

| Opção | Descrição |
|-------|-----------|
| **Incluindo tributo** | Os preços do carrinho e checkout incluem tributo |
| **Excluindo tributo** | Os preços do carrinho e checkout excluem tributo |

### Sufixo de Exibição do Preço {#price-display-suffix}

Texto que aparece após os preços para indicar o status do tributo (por exemplo, "incl. IVA" ou "excl. IVA").

### Exibir Totais de Tributo {#display-tax-totals}

| Opção | Descrição |
|-------|-----------|
| **Como um total único** | Todos os tributos são combinados em uma linha total |
| **Detalhado** | Cada taxa de tributo é exibida como um item separado |

## Restaurar Configurações do Servidor {#restore-server-settings}

Clique para buscar e aplicar as configurações de tributo atuais do WooCommerce do seu servidor.

:::tip Melhor Prática
Use "Restaurar Configurações do Servidor" regularmente para manter a configuração de tributo do seu POS em sincronia com sua loja WooCommerce, especialmente após fazer alterações nas configurações de tributo no seu admin do WordPress.
:::
