## Atalhos Disponíveis

| Combinação de Teclas | Ação | Descrição |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | Configurações | Abre o painel de configurações do POS |
| `Ctrl + Shift + A` | POS | Navega para a tela principal do POS |
| `Ctrl + Shift + P` | Produtos | Abre a tela de gerenciamento de produtos |
| `Ctrl + Shift + O` | Pedidos | Abre a tela de gerenciamento de pedidos |
| `Ctrl + Shift + C` | Clientes | Abre a tela de gerenciamento de clientes |
| `Ctrl + Shift + ?` | Suporte | Abre a seção de suporte/ajuda |
| `Ctrl + Shift + L` | Sair | Faz logout da sessão atual |

## Usando Atalhos de Teclado

Para usar qualquer atalho de teclado:

1. **Mantenha pressionadas as teclas modificadoras** (`Ctrl + Shift`)
2. **Pressione a tecla de ação** (a letra correspondente à ação desejada)
3. **Solte todas as teclas** para executar o comando

## Dicas para Uso Eficiente

- **Memorize atalhos frequentemente usados**: Concentre-se em aprender os atalhos para ações que você realiza com mais frequência
- **Pratique as combinações de teclas**: O uso regular ajudará a criar memória muscular
- **Use atalhos durante períodos movimentados**: Atalhos de teclado podem acelerar significativamente as operações durante horários de pico

## Personalização

:::info Solicitação de Recurso
A personalização de atalhos de teclado não está disponível no momento. Os atalhos listados acima são fixos e não podem ser modificados pela interface de configurações.

Se você gostaria de personalizar os atalhos de teclado, compartilhe seu feedback no [Discord](https://wcpos.com/discord).
