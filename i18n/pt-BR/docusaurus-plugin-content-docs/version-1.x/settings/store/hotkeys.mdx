## Atalhos Disponíveis

| Combinação de Teclas | Ação | Descrição |
|----------------------|------|-----------|
| `Ctrl + Shift + S`   | Configurações | Abre o painel de configurações do POS |
| `Ctrl + Shift + A`   | POS | Navega para a tela principal do POS |
| `Ctrl + Shift + P`   | Produtos | Abre a tela de gerenciamento de produtos |
| `Ctrl + Shift + O`   | Pedidos | Abre a tela de gerenciamento de pedidos |
| `Ctrl + Shift + C`   | Clientes | Abre a tela de gerenciamento de clientes |
| `Ctrl + Shift + ?`   | Suporte | Abre a seção de suporte/ajuda |
| `Ctrl + Shift + L`   | Sair | Desconecta da sessão atual |

## Usando Atalhos de Teclado

Para usar qualquer atalho de teclado:

1. **Aperte as teclas modificadoras** (`Ctrl + Shift`)
2. **Pressione a tecla de ação** (a letra correspondente à sua ação desejada)
3. **Solte todas as teclas** para executar o comando

## Dicas para Uso Eficiente

- **Memorize os atalhos frequentemente usados**: Concentre-se em aprender os atalhos para ações que você realiza com mais frequência
- **Pratique as combinações de teclas**: O uso regular ajudará a construir a memória muscular
- **Use atalhos durante períodos de alta movimentação**: Atalhos de teclado podem acelerar significativamente as operações durante horários de pico

## Personalização

:::info Solicitação de Funcionalidade
A personalização de atalhos de teclado atualmente não está disponível. Os atalhos listados acima são fixos e não podem ser modificados através da interface de configurações.

Se você gostaria de personalizar os atalhos de teclado, por favor, compartilhe seu feedback no [Discord](https://wcpos.com/discord).
:::
