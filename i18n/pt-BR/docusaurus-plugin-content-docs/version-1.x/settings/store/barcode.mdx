## Como Funciona a Detecção de Códigos de Barras {#how-barcode-detection-works}

A maioria dos scanners de código de barras se comporta como um teclado conectado ao seu dispositivo. Quando você escaneia um código de barras, o POS detecta que os caracteres foram inseridos mais rapidamente do que a digitação normal. Ele usa essas "teclas pressionadas rapidamente" para identificar a entrada como um escaneamento de código de barras.

## Configurações {#settings}

| Configuração | Descrição |
|--------------|-----------|
| **Tempo médio de entrada** | Quão rápido a entrada deve ser (em milissegundos) para contar como um código de barras. Valores mais baixos = entrada mais rápida necessária |
| **Comprimento mínimo** | Quão longa a sequência contínua de caracteres deve ser para ser tratada como um código de barras |
| **Remoção de prefixo** | Caracteres a serem removidos do início dos códigos de barras escaneados |
| **Remoção de sufixo** | Caracteres a serem removidos do final dos códigos de barras escaneados |

## Remoção de Prefixo/Sufixo {#prefixsuffix-removal}

Se o seu scanner adicionar caracteres extras (como um prefixo ou sufixo), você pode removê-los automaticamente para que apenas o código de barras principal permaneça.

**Exemplo:** Se o seu scanner adicionar "A" antes de cada código de barras e "Z" depois, configure:
- Remoção de prefixo: `A`
- Remoção de sufixo: `Z`

## Solução de Problemas {#troubleshooting}

<Accordion>
  <AccordionItem question="Código de barras não detectado">

  - **Scanner muito lento**: Tente aumentar o valor do "Tempo médio de entrada"
  - **Código de barras muito curto**: Tente diminuir o valor do "Comprimento mínimo"
  - **Caracteres extras**: Verifique se o seu scanner está adicionando prefixes/suffixes

  </AccordionItem>

  <AccordionItem question="Digitação normal detectada como código de barras">

  - **Digitando muito rápido**: Tente diminuir o valor do "Tempo médio de entrada"
  - **Entrada muito longa**: Tente aumentar o valor do "Comprimento mínimo"

  </AccordionItem>
</Accordion>
