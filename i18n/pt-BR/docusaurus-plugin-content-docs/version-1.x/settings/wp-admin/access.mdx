---
title: Acessando o PDV
sidebar_label: Acesso
---

Para controlar o acesso à funcionalidade do WCPOS, você pode utilizar as permissões e funções de usuário do WordPress. Ao atribuir funções específicas aos usuários e configurar suas permissões, você pode adaptar os níveis de acesso para diferentes usuários do PDV. O plugin WCPOS se integra às funções existentes do WordPress e introduz uma função específica para caixas.

A tabela a seguir apresenta as principais funções de usuário no PDV e suas respectivas permissões:

| Função         | Criada Por     | Descrição                                                                                                                                                                                                                                                                                                                                                    |
| ------------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Administrador | WordPress       | O proprietário do site que tem todas as permissões habilitadas por padrão.                                                                                                                                                                                                                                                                                    |
| Gerente de Loja  | WooCommerce     | Esta função é projetada para usuários que precisam de acesso ao PDV e à capacidade de gerenciar configurações do PDV. Gerentes de Loja podem não ter permissão para instalar plugins ou realizar outras tarefas administrativas.                                                                                                                                                             |
| Caixa       | WCPOS | Esta função é adicionada pelo plugin WCPOS e destina-se a usuários que utilizarão principalmente o PDV. Caixas geralmente têm permissões limitadas e podem não ter acesso ao WP Admin para modificar configurações do PDV, instalar plugins ou criar postagens no blog. Em alguns casos, você pode até querer restringir um caixa de adicionar ou editar clientes no PDV. |

Para atribuir e gerenciar funções e permissões de usuário, você pode navegar até **`WP Admin > POS > Acesso`**.

A tabela a seguir fornece uma visão geral das permissões do WordPress associadas ao plugin WCPOS:

| Permissão                  | Descrição                                                            | Exemplo de Uso                                                                                                                                    |
| --------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `access_woocommerce_pos`    | Permite acesso à interface do WCPOS.                        | Habilitar esta opção concederia ao caixa permissão para acessar e usar o PDV, permitindo realizar transações de ponto de venda.          |
| `manage_woocommerce_pos`    | Permite o gerenciamento de configurações e definições do WCPOS. | Habilitar esta opção daria ao caixa a capacidade de modificar configurações relacionadas ao PDV, como métodos de pagamento ou modelos de recibo.     |
| `create_users`              | Permite ao caixa criar novas contas de usuário do WordPress.              | Desabilitar esta opção impediria o caixa de criar novas contas de cliente no PDV.                                                  |
| `edit_others_shop_orders`   | Permite editar pedidos de loja criados por outros caixas.              | Desabilitar esta opção impediria um caixa de modificar pedidos de loja criados por um caixa diferente no PDV.                              |
| `edit_product`              | Permite ao caixa editar detalhes de produtos.                            | Desabilitar esta opção impediria o caixa de modificar os detalhes dos produtos disponíveis no PDV.                                     |
| `edit_published_products`   | Permite ao caixa editar produtos publicados.                         | Desabilitar esta opção impediria o caixa de fazer alterações em produtos já publicados no PDV.                           |
| `edit_shop_orders`          | Permite ao caixa editar pedidos de loja.                                | Desabilitar esta opção impediria o caixa de modificar os detalhes dos pedidos de loja, como status ou detalhes do cliente, no PDV. |
| `edit_users`                | Permite ao caixa editar contas de usuário existentes.                     | Desabilitar esta opção impediria o caixa de modificar os detalhes das contas de clientes, como endereço de cobrança, no PDV.            |
| `list_users`                | Permite ao caixa visualizar uma lista de contas de usuário.                    | Desabilitar esta opção impediria o caixa de pesquisar ou selecionar clientes no PDV.                                                |
| `manage_product_terms`      | Permite ao caixa gerenciar categorias e tags de produtos.              | Desabilitar esta opção impediria o caixa de criar, editar ou excluir categorias ou tags de produtos no PDV.                       |
| `publish_shop_orders`       | Permite ao caixa publicar pedidos de loja.                             | Desabilitar esta opção impediria o caixa de finalizar e processar novos pedidos de loja no PDV.                                       |
| `read_private_products`     | Permite ao caixa acessar os detalhes de produtos privados.          | Desabilitar esta opção impediria o caixa de visualizar os detalhes de produtos privados disponíveis no PDV.                               |
| `read_private_shop_coupons` | Permite ao caixa acessar os detalhes de cupons de loja privados.      | Desabilitar esta opção impediria o caixa de visualizar os detalhes de cupons de loja privados no PDV.                                     |
| `read_private_shop_orders`  | Permite ao caixa acessar os detalhes de pedidos de loja privados.       | Desabilitar esta opção impediria o caixa de visualizar os detalhes de pedidos de loja privados no PDV.                                      |
| `read`                      | Permite acesso ao Administrador do WordPress.                                  | Desabilitar esta opção impede o caixa de acessar o Painel do WordPress e a página de perfil do usuário.                                                   |

Ao atribuir as permissões apropriadas a cada função, você pode ajustar finamente o acesso e as permissões para diferentes usuários no PDV. Isso permite controlar quem pode realizar certas ações, visualizar informações sensíveis e fazer alterações na sua loja.

Para informações mais detalhadas sobre como gerenciar funções e permissões de usuário no WordPress, você pode consultar o [WordPress Codex](https://codex.wordpress.org/Roles_and_Capabilities).
