---
title: Acessando o POS
sidebar_label: Acesso
description: Configure funções de usuário e capacidades para acesso ao WCPOS, incluindo permissões de Administrador, Gerente da Loja e Caixa.
---

Para controlar o acesso à funcionalidade do WCPOS, você pode utilizar as capacidades e funções de usuário do WordPress. Atribuindo funções específicas aos usuários e configurando suas capacidades, você pode personalizar os níveis de acesso para diferentes usuários do POS. O plugin WCPOS se integra com as funções existentes do WordPress e introduz uma função específica para caixas.

A tabela a seguir descreve as principais funções de usuário no POS e suas respectivas capacidades:

| Função          | Criado Por      | Descrição                                                                                                                                                                                                                                                                                                                                                    |
| --------------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Administrador   | WordPress       | O proprietário do site que tem todas as capacidades habilitadas por padrão.                                                                                                                                                                                                                                                                                                    |
| Gerente da Loja | WooCommerce     | Esta função é projetada para usuários que precisam de acesso ao POS e a capacidade de gerenciar configurações do POS. Os Gerentes da Loja podem não ter permissão para instalar plugins ou realizar outras tarefas administrativas.                                                                                                                                                             |
| Caixa           | WCPOS           | Esta função é adicionada pelo plugin WCPOS e é destinada a usuários que usarão principalmente o POS. Os caixeiros geralmente têm capacidades limitadas e podem não ter acesso ao WP Admin para modificar configurações do POS, instalar plugins ou criar posts no blog. Em alguns casos, você pode até querer restringir um caixa de adicionar ou editar clientes no POS. |

Para atribuir e gerenciar funções e capacidades de usuário, você pode navegar para **`WP Admin > POS > Acesso`**.

A tabela a seguir fornece uma visão geral das capacidades do WordPress associadas ao plugin WCPOS:

| Capacidade                  | Descrição                                                            | Exemplo de Uso                                                                                                                                    |
| --------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `access_woocommerce_pos`    | Permite acesso à interface do WCPOS.                        | Habilitar esta opção concederia ao caixa permissão para acessar e usar o POS, permitindo que ele realizasse transações de ponto de venda.          |
| `manage_woocommerce_pos`    | Habilita a gestão das configurações e configurações do WCPOS. | Habilitar esta opção daria ao caixa a capacidade de modificar configurações relacionadas ao POS, como métodos de pagamento ou modelos de recibos.     |
| `create_users`              | Permite ao caixa criar novas contas de usuário do WordPress.              | Desabilitar esta opção impedirá que o caixa crie novas contas de clientes no POS.                                                  |
| `edit_others_shop_orders`   | Habilita a edição de pedidos de loja criados por outros caixas.              | Desabilitar esta opção impediria que um caixa modificasse pedidos de loja criados por outro caixa no POS.                              |
| `edit_product`              | Permite ao caixa editar detalhes de produtos.                            | Desabilitar esta opção impediria que o caixa modificasse os detalhes dos produtos disponíveis no POS.                                     |
| `edit_published_products`   | Permite ao caixa editar produtos publicados.                         | Desabilitar esta opção impediria que o caixa fizesse alterações em produtos que já estão publicados no POS.                           |
| `edit_shop_orders`          | Permite ao caixa editar pedidos de loja.                                | Desabilitar esta opção impediria que o caixa modificasse os detalhes dos pedidos de loja, como status do pedido ou detalhes do cliente, no POS. |
| `edit_users`                | Permite ao caixa editar contas de usuário existentes.                     | Desabilitar esta opção impediria que o caixa modificasse os detalhes das contas de clientes, como endereço de cobrança, no POS.            |
| `list_users`                | Permite ao caixa visualizar uma lista de contas de usuário.                    | Desabilitar esta opção impediria que o caixa pesquisasse ou selecionasse clientes no POS.                                                |
| `manage_product_terms`      | Permite ao caixa gerenciar categorias e etiquetas de produtos.              | Desabilitar esta opção impediria que o caixa criasse, editasse ou excluísse categorias ou etiquetas de produtos no POS.                       |
| `publish_shop_orders`       | Permite ao caixa publicar pedidos de loja.                             | Desabilitar esta opção impediria que o caixa finalizasse e processasse novos pedidos de loja no POS.                                       |
| `read_private_products`     | Permite ao caixa acessar os detalhes de produtos privados.          | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de produtos privados disponíveis no POS.                               |
| `read_private_shop_coupons` | Permite ao caixa acessar os detalhes de cupons de loja privados.      | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de cupons de loja privados no POS.                                     |
| `read_private_shop_orders`  | Permite ao caixa acessar os detalhes de pedidos de loja privados.       | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de pedidos de loja privados no POS.                                      |
| `read`                      | Permite o acesso ao WordPress Admin.                                  | Desabilitar isso impede que o caixa tenha acesso ao painel do WordPress e à página de perfil do usuário.                                                   |

Ao atribuir as capacidades apropriadas a cada função, você pode ajustar o acesso e as permissões para diferentes usuários no POS. Isso permite que você controle quem pode realizar certas ações, visualizar informações sensíveis e fazer alterações na sua loja.

Para mais informações detalhadas sobre como gerenciar funções de usuário e capacidades no WordPress, você pode consultar o [Codex do WordPress](https://codex.wordpress.org/Roles_and_Capabilities).
