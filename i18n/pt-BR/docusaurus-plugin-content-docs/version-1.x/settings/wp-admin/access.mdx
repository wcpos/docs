---
title: Acessando o POS
sidebar_label: Acesso
---

Para controlar o acesso à funcionalidade do WCPOS, você pode utilizar as capacidades e funções do WordPress. Ao atribuir funções específicas aos usuários e configurar suas capacidades, você pode personalizar os níveis de acesso para diferentes usuários do POS. O plugin WCPOS se integra com as funções existentes do WordPress e introduz uma função específica para caixas.

A tabela a seguir descreve as principais funções de usuário no POS e suas respectivas capacidades:

| Função          | Criado Por      | Descrição                                                                                                                                                                                                                                                                                                                                                    |
| --------------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Administrador   | WordPress       | O proprietário do site que tem todas as capacidades ativadas por padrão.                                                                                                                                                                                                                                                                                                    |
| Gerente de Loja | WooCommerce     | Esta função é projetada para usuários que precisam de acesso ao POS e à capacidade de gerenciar as configurações do POS. Os Gerentes de Loja podem não ter permissão para instalar plugins ou realizar outras tarefas administrativas.                                                                                                                                                             |
| Caixa           | WCPOS           | Esta função é adicionada pelo plugin WCPOS e é destinada a usuários que usarão principalmente o POS. Os caixas geralmente têm capacidades limitadas e podem não ter acesso ao WP Admin para modificar configurações do POS, instalar plugins ou criar postagens no blog. Em alguns casos, você pode querer restringir um caixa de adicionar ou editar clientes no POS. |

Para atribuir e gerenciar funções e capacidades de usuários, você pode navegar para **`WP Admin > POS > Acesso`**.

A tabela a seguir fornece uma visão geral das capacidades do WordPress associadas ao plugin WCPOS:

| Capacidade                  | Descrição                                                            | Exemplo de Uso                                                                                                                                    |
| --------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `access_woocommerce_pos`    | Permite acesso à interface do WCPOS.                              | Habilitar esta opção concederia ao caixa permissão para acessar e usar o POS, permitindo que ele realizasse transações no ponto de venda.          |
| `manage_woocommerce_pos`    | Permite a gestão das configurações e configurações do WCPOS.   | Habilitar esta opção daria ao caixa a capacidade de modificar configurações relacionadas ao POS, como métodos de pagamento ou modelos de recibo.     |
| `create_users`              | Permite que o caixa crie novas contas de usuários do WordPress.              | Desabilitar esta opção impediria que o caixa criasse novas contas de clientes no POS.                                                  |
| `edit_others_shop_orders`   | Permite editar pedidos de loja criados por outros caixas.              | Desabilitar esta opção impediria que um caixa modificasse pedidos de loja criados por outro caixa no POS.                              |
| `edit_product`              | Permite que o caixa edite os detalhes do produto.                            | Desabilitar esta opção impediria que o caixa modificasse os detalhes dos produtos disponíveis no POS.                                     |
| `edit_published_products`   | Permite que o caixa edite produtos publicados.                         | Desabilitar esta opção impediria que o caixa fizesse alterações em produtos que já estão publicados no POS.                           |
| `edit_shop_orders`          | Permite que o caixa edite pedidos de loja.                                | Desabilitar esta opção impediria que o caixa modificasse os detalhes dos pedidos de loja, como status do pedido ou detalhes do cliente, no POS. |
| `edit_users`                | Permite que o caixa edite contas de usuários existentes.                     | Desabilitar esta opção impediria que o caixa modificasse os detalhes das contas de clientes, como endereço de cobrança, no POS.            |
| `list_users`                | Permite que o caixa visualize uma lista de contas de usuários.                    | Desabilitar esta opção impediria que o caixa pesquisasse ou selecionasse clientes no POS.                                                |
| `manage_product_terms`      | Permite que o caixa gerencie categorias e tags de produtos.              | Desabilitar esta opção impediria que o caixa criasse, editasse ou excluísse categorias ou tags de produtos no POS.                       |
| `publish_shop_orders`       | Permite que o caixa publique pedidos de loja.                             | Desabilitar esta opção impediria que o caixa finalizasse e processasse novos pedidos de loja no POS.                                       |
| `read_private_products`     | Permite que o caixa acesse os detalhes de produtos privados.          | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de produtos privados disponíveis no POS.                               |
| `read_private_shop_coupons` | Permite que o caixa acesse os detalhes de cupons de loja privados.      | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de cupons de loja privados no POS.                                     |
| `read_private_shop_orders`  | Permite que o caixa acesse os detalhes de pedidos de loja privados.       | Desabilitar esta opção impediria que o caixa visualizasse os detalhes de pedidos de loja privados no POS.                                      |
| `read`                      | Permite acesso ao Admin do WordPress.                                  | Desabilitar isso impede que o caixa acesse o Painel do WordPress e a página de perfil do usuário.                                                   |

Ao atribuir as capacidades apropriadas a cada função, você pode ajustar o acesso e as permissões para diferentes usuários no POS. Isso permite que você controle quem pode realizar certas ações, visualizar informações sensíveis e fazer alterações em sua loja.

Para obter informações mais detalhadas sobre como gerenciar funções e capacidades de usuários no WordPress, você pode consultar o [WordPress Codex](https://codex.wordpress.org/Roles_and_Capabilities).
