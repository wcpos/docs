A tela **Sessões** permite ver quem está atualmente conectado ao POS, em quais dispositivos e desconectar dispositivos individuais remotamente. É o lugar para ir quando um dispositivo é perdido ou roubado, quando um caixa sai ou quando você simplesmente deseja revisar logins ativos.

Você o encontrará em **`WP Admin > POS > Settings > Sessions`**.

> _Gerencie sessões de usuários ativos. Você pode visualizar todos os usuários logados, ver seus dispositivos e encerrar sessões, se necessário._

## Como funciona {#how-it-works}

A tela possui dois painéis:

- **Usuários** (à esquerda) — todos os usuários com pelo menos uma sessão ativa, com um ponto verde em qualquer pessoa ativa nos últimos minutos, um chip **Você** para sua própria conta e uma contagem de quantas sessões cada usuário tem. Uma caixa de filtro permite pesquisar por nome ou nome de usuário.
- **Sessões** (à direita) — selecione um usuário para ver um cartão por sessão ativa.

Cada cartão de sessão mostra:

| Campo | O que isso significa |
| --- | --- |
| **Inscrição** | Onde o usuário está conectado — **Aplicativo iOS**, **Aplicativo Android**, **Aplicativo Desktop** ou **Aplicativo Web** — mais o navegador/versão |
| **Dispositivo atual** | Um chip marcando a sessão que você está usando no momento |
| **Último ativo** | Há quanto tempo a sessão fez uma solicitação (por exemplo, _agora mesmo_, _5 minutos atrás_) |
| **IP** | O endereço IP do qual a sessão foi conectada (ou _N/A_) |
| **Criado** | Quando a sessão foi conectada pela primeira vez |
| **Expira** | Quando o token de sessão expirar |
| **Agente do usuário** | Expanda para ver a string bruta do agente do usuário do navegador/dispositivo |

Uma sessão é mostrada como **Ativa agora** quando fez uma solicitação nos últimos 5 minutos.

## Desconectando dispositivos {#signing-devices-out}

Três ações estão disponíveis, cada uma atrás de uma caixa de diálogo de confirmação:

- **Encerrar** — saia de uma única sessão. Use o botão no cartão dessa sessão.
- **Sair de outros dispositivos** – desconecte o usuário selecionado de todos os dispositivos **exceto** aquele que ele está usando no momento. Mostrado quando o usuário possui mais de uma sessão.
- **Sair de todos** — desconecte o usuário selecionado de **todos** dispositivos, incluindo o atual.

:::tip Dispositivo perdido ou roubado
Se um telefone, tablet ou laptop com o POS desaparecer, abra Sessões, selecione o usuário afetado e use **Encerrar** na sessão desse dispositivo (ou **Sair de todos** por segurança). O dispositivo perde acesso na próxima solicitação.
:::

:::note
Encerrar sua própria sessão atual desconecta você do dispositivo em que está agora - você será solicitado a confirmar antes que isso aconteça.
:::
