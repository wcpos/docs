A tela **Sessões** permite ver quem está conectado ao POS no momento, em quais dispositivos, e desconectar dispositivos individuais remotamente. É o local a acessar quando um dispositivo é perdido ou roubado, quando um operador de caixa sai da empresa, ou quando você simplesmente quer revisar logins ativos.

Ela fica em **`WP Admin > POS > Configurações > Sessions`**.

> _Gerencie sessões de usuário ativas. Você pode ver todos os usuários conectados, consultar seus dispositivos e encerrar sessões se necessário._

## Como funciona {#how-it-works}

A tela tem dois painéis:

- **Usuários** (à esquerda) — todos os usuários com pelo menos uma sessão ativa, com um ponto verde em qualquer pessoa ativa nos últimos minutos, um marcador **Você** para a sua própria conta e uma contagem de quantas sessões cada usuário tem. Uma caixa de filtro permite pesquisar por nome ou nome de usuário.
- **Sessões** (à direita) — selecione um usuário para ver um cartão por sessão ativa.

Cada cartão de sessão mostra:

| Campo | O que significa |
| --- | --- |
| **Aplicativo** | Onde o usuário está conectado — **Aplicativo iOS**, **Aplicativo Android**, **Aplicativo para desktop** ou **Aplicativo web** — mais o navegador/versão |
| **Dispositivo atual** | Um selo que marca a sessão que você está usando agora |
| **Última atividade** | Há quanto tempo a sessão fez uma solicitação (por exemplo, _Agora mesmo_, _5 minutos atrás_) |
| **IP** | O endereço IP de onde a sessão se conectou (ou _N/D_) |
| **Criada** | Quando a sessão fez login pela primeira vez |
| **Expira** | Quando o token da sessão expira |
| **Agente do usuário** | Expanda para ver a string bruta do agente do usuário do navegador/dispositivo |

Uma sessão é exibida como **Ativa agora** quando fez uma solicitação nos últimos 5 minutos.

## Desconectar dispositivos {#signing-devices-out}

Há três ações disponíveis, cada uma protegida por uma caixa de diálogo de confirmação:

- **Encerrar** — desconecta uma única sessão. Use o botão no cartão dessa sessão.
- **Sair dos outros dispositivos** — desconecta o usuário selecionado de todos os dispositivos, **exceto** aquele que ele está usando no momento. Exibido quando o usuário tem mais de uma sessão.
- **Sair de todos** — desconecta o usuário selecionado de **todos** os dispositivos, incluindo o atual.

:::tip Dispositivo perdido ou roubado
Se um telefone, tablet ou notebook com o POS for perdido, abra Sessões, selecione o usuário afetado e use **Encerrar** na sessão desse dispositivo (ou **Sair de todos** por segurança). O dispositivo perde o acesso na próxima solicitação.
:::

:::note
Encerrar sua própria sessão atual desconecta você do dispositivo em uso agora — será solicitada uma confirmação antes que isso aconteça.
:::
