A página de Configurações de Checkout (`WP Admin > POS > Settings > Checkout`) controla quais gateways de pagamento são exibidos no POS e como os pedidos são processados.

## Gateways de Pagamento {#payment-gateways}

Esta seção lista todos os gateways de pagamento do WooCommerce instalados em seu site. Cada gateway pode ser ativado ou desativado de forma independente para o POS.

| Coluna | Descrição |
|--------|-------------|
| **Ativado** | Ativa ou desativa o gateway para o checkout do POS |
| **Gateway** | O nome do gateway de pagamento |
| **ID do Gateway** | O identificador do gateway no WooCommerce (útil para desenvolvedores) |

### Habilitando um gateway {#enabling-a-gateway}

1. Localize o gateway na lista
2. Ative o seletor **Habilitado**
3. O gateway aparecerá na finalização de compra do POS no próximo carregamento da página

### Reordenando gateways {#reordering-gateways}

Arraste os gateways para alterar a ordem de exibição na finalização de compra do POS. O primeiro gateway habilitado é selecionado por padrão quando um caixa abre a janela de finalização.

### POS vs. loja online {#pos-vs-online-store}

As configurações de gateway aqui são **independentes** das configurações de pagamento do WooCommerce. Um gateway pode estar:

- **Habilitado para POS, desabilitado online** — útil para pagamentos em dinheiro, que não se aplicam a pedidos web
- **Habilitado online, desabilitado para POS** — útil para gateways como o PayPal, que não são práticos para vendas presenciais
- **Habilitado para ambos** — o gateway funciona em todos os lugares

:::note
Um gateway deve primeiro estar instalado e ativo no `WP Admin > WooCommerce > Settings > Payments` antes de aparecer na lista de configurações de checkout do POS. Se um gateway estiver faltando, verifique se o plugin está instalado e se o gateway está habilitado no WooCommerce.
:::

## Status do Pedido {#order-status}

Configure o status padrão dos pedidos para pedidos do POS concluídos. Por padrão, os pedidos do POS são definidos como **Concluído**, já que o pagamento é recebido no ponto de venda.
