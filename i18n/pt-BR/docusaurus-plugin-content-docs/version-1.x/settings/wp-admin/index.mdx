Essas configurações configuram a funcionalidade do backend da aplicação POS, incluindo configurações de PHP e WooCommerce. 
Você pode acessá-las navegando para: `WP Admin > POS > Configurações`

As configurações principais incluem:

- Cliente de carrinho padrão para todos os caixas.
- Campo de código de barras do produto.
- Configurações de e-mail de pedido.
- Permissões de função do usuário.
