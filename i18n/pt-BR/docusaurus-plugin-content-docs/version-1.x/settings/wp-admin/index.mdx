Estas configurações definem a funcionalidade backend do aplicativo POS, incluindo configurações de PHP e WooCommerce. 
Você pode acessá-las navegando para: `WP Admin > POS > Configurações`

Configurações principais incluem:

- Cliente padrão do carrinho para todos os caixas.
- Campo de código de barras do produto.
- Configurações de e-mail de pedidos.
- Permissões de função de usuário.
