O WCPOS possui três tipos de configurações organizadas de acordo com onde são configuradas.

## Onde Encontrar Configurações

### 1. Configurações do WordPress Admin

Configurações do backend configuradas no seu painel do WordPress:

**Localização:** `WP Admin > POS > Configurações`

Estas controlam:
- Cliente padrão do carrinho para todos os caixas
- Configuração do campo de código de barras do produto
- Configurações de e-mail do pedido
- Permissões de função do usuário (quem pode acessar o POS)
- Disponibilidade do gateway de pagamento

Veja [Configurações do WordPress Admin](/settings/wp-admin) para mais detalhes.

### 2. Configurações da Loja POS

Configurações globais do aplicativo configuradas dentro do aplicativo POS:

**Localização:** Menu do usuário (canto superior direito) → Configurações, ou `Ctrl + Shift + S`

Estas controlam:
- Opções de cálculo de impostos
- Configuração do scanner de código de barras
- Atalhos de teclado
- Tema/aparência

Veja [Configurações da Loja](/settings/store) para mais detalhes.

### 3. Configurações de Exibição (Por Tela)

Cada tela no POS possui suas próprias configurações de exibição para personalizar a tabela de dados:

**Localização:** Clique no ícone de deslizadores (<Icon name="sliders" />) em qualquer tela

Estas controlam:
- Quais colunas são visíveis
- Opções de exibição de colunas
- Recursos específicos da tela

As configurações de exibição são documentadas dentro da documentação de cada tela:
- [Painel de Produtos POS](/pos/product-panel#display-settings)
- [Painel do Carrinho](/pos/cart#display-settings)
- [Tela de Produtos](/products#display-settings)
- [Tela de Pedidos](/orders#display-settings)
- [Tela de Clientes](/customers#display-settings)
- [Tela de Relatórios](/reports#display-settings)
