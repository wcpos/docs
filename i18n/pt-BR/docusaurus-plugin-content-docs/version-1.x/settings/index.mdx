O WCPOS possui três tipos de configurações organizadas por onde são configuradas.

## Onde Encontrar Configurações

### 1. Configurações de Administração do WordPress

Configurações de backend configuradas no seu painel do WordPress:

**Localização:** `WP Admin > POS > Configurações`

Essas controlam:
- Cliente padrão do carrinho para todos os caixas
- Configuração de campo de código de barras do produto
- Configurações de e-mail de pedido
- Permissões de função de usuário (quem pode acessar o POS)
- Disponibilidade de gateway de pagamento

Consulte [Configurações de Administração do WordPress](/settings/wp-admin) para obter detalhes.

### 2. Configurações de Loja do POS

Configurações de todo o aplicativo configuradas dentro do aplicativo POS:

**Localização:** Menu de usuário (superior direito) → Configurações, ou `Ctrl + Shift + S`

Essas controlam:
- Opções de cálculo de imposto
- Configuração de leitor de código de barras
- Atalhos de teclado
- Tema/aparência

Consulte [Configurações de Loja](/settings/store) para obter detalhes.

### 3. Configurações de Exibição (Por Tela)

Cada tela no POS possui suas próprias configurações de exibição para personalizar a tabela de dados:

**Localização:** Clique no ícone de controles deslizantes (<Icon name="sliders" />) em qualquer tela

Essas controlam:
- Quais colunas são visíveis
- Opções de exibição de colunas
- Recursos específicos da tela

As configurações de exibição são documentadas na documentação de cada tela:
- [Painel de Produtos do POS](/pos/product-panel#display-settings)
- [Painel do Carrinho](/pos/cart#display-settings)
- [Tela de Produtos](/products#display-settings)
- [Tela de Pedidos](/orders#display-settings)
- [Tela de Clientes](/customers#display-settings)
- [Tela de Relatórios](/reports#display-settings)
