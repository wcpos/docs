O WCPOS tem três tipos de configurações organizadas por onde são configuradas.

## Onde Encontrar as Configurações

### 1. Configurações do Admin do WordPress

Configurações de backend configuradas no painel do WordPress:

**Localização:** `WP Admin > POS > Configurações`

Essas controlam:
- Cliente padrão do carrinho para todos os caixas
- Configuração do campo de código de barras do produto
- Configurações de e-mail de pedido
- Permissões de função de usuário (quem pode acessar o POS)
- Disponibilidade do gateway de pagamento

Consulte [Configurações do Admin do WordPress](/settings/wp-admin) para detalhes.

### 2. Configurações da Loja POS

Configurações abrangentes do aplicativo configuradas dentro do aplicativo POS:

**Localização:** Menu do usuário (canto superior direito) → Configurações, ou `Ctrl + Shift + S`

Essas controlam:
- Opções de cálculo de impostos
- Configuração do scanner de código de barras
- Atalhos de teclado
- Tema/aparência

Consulte [Configurações da Loja](/settings/store) para detalhes.

### 3. Configurações de Exibição (Por Tela)

Cada tela no POS possui suas próprias configurações de exibição para personalizar a tabela de dados:

**Localização:** Clique no ícone de controle deslizante (<Icon name="sliders" />) em qualquer tela

Essas controlam:
- Quais colunas são visíveis
- Opções de exibição de coluna
- Recursos específicos da tela

As configurações de exibição estão documentadas na documentação de cada tela:
- [Painel de Produtos do POS](/pos/product-panel#display-settings)
- [Painel do Carrinho](/pos/cart#display-settings)
- [Tela de Produtos](/products#display-settings)
- [Tela de Pedidos](/orders#display-settings)
- [Tela de Clientes](/customers#display-settings)
- [Tela de Relatórios](/reports#display-settings)
