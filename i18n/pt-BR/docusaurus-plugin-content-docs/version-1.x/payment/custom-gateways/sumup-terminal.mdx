# Gateway do Terminal SumUp {#sumup-terminal-gateway}

O gateway do Terminal SumUp permite que você aceite pagamentos com cartão usando leitores de cartão SumUp diretamente dentro do WooCommerce POS. Este gateway proporciona uma integração perfeita com o sistema de processamento de pagamentos da SumUp para transações presenciais.

## Instalação {#installation}

<Steps>
  <StepCard number="1" title="Baixar e Instalar">

1. Visite a [página de lançamentos do Terminal SumUp para WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Baixe o arquivo mais recente **sumup-terminal-for-woocommerce.zip**
3. No seu admin do WordPress, vá para `Plugins > Adicionar Novo > Fazer Upload do Plugin`
4. Faça o upload do arquivo zip e ative o plugin

  </StepCard>
  <StepCard number="2" title="Configurar as Configurações da SumUp">

1. Navegue até `WP Admin > WooCommerce > Configurações > Pagamentos`
2. Encontre **Terminal SumUp** na lista de métodos de pagamento
3. Clique em **Terminal SumUp** para acessar as configurações
4. Insira sua **Chave da API SumUp** (disponível no seu painel de comerciante SumUp)
5. Salve as configurações

:::note
Você não precisa habilitar o gateway do Terminal SumUp nas configurações do WooCommerce. Ele será habilitado especificamente para o POS em uma etapa posterior.
:::

  </StepCard>
  <StepCard number="3" title="Emparelhe Seu Terminal SumUp">

1. Na mesma página de configurações, localize a seção **Emparelhar Leitor**
2. No seu dispositivo SumUp, navegue até a tela de emparelhamento para exibir o código de emparelhamento
3. Insira o código de emparelhamento exibido no seu dispositivo SumUp
4. Clique em **"Emparelhar Leitor"** para estabelecer a conexão
5. Aguarde a confirmação de que o leitor foi emparelhado com sucesso

:::warning Importante
O leitor deve estar emparelhado com sucesso antes que você possa processar pagamentos. Certifique-se de que o processo de emparelhamento esteja concluído antes de prosseguir.
:::

  </StepCard>
  <StepCard number="4" title="Ativar no WooCommerce POS" isLast>

1. Vá para `WP Admin > WooCommerce POS > Configurações > Checkout`
2. Encontre o gateway **Terminal SumUp** na lista
3. Ative o gateway para uso no POS
4. Salve suas configurações

  </StepCard>
</Steps>

## Uso {#usage}

### Processando Pagamentos {#processing-payments}

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Terminal SumUp" como método de pagamento
3. **Iniciar Pagamento**: Clique para iniciar um novo pagamento no seu dispositivo SumUp
4. **Pagamento do Cliente**: O cliente completa o pagamento no terminal SumUp
5. **Conclusão Automática**: O pedido é automaticamente concluído quando o pagamento é detectado

### Controles de Pagamento {#payment-controls}

Ao usar o gateway do Terminal SumUp, você tem as seguintes opções:

- **Iniciar Novo Pagamento**: Inicie uma solicitação de pagamento no terminal conectado
- **Cancelar Pagamento**: Cancele um pagamento que está atualmente em processo
- **Status do Pagamento**: Monitore o status atual do processo de pagamento

### Gerenciamento de Pedidos {#order-management}

- **Atualizações Automáticas**: Os pedidos são automaticamente marcados como concluídos após o pagamento bem-sucedido
- **Rastreamento de Pagamento**: Todos os detalhes do pagamento são registrados nas notas do pedido
- **Geração de Recibo**: Recibos padrão do POS são gerados após pagamentos bem-sucedidos

## Compatibilidade de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividade
Esta implementação utiliza o SDK JavaScript da SumUp, o que significa que funciona através de aplicações web, mas requer **terminais conectados à internet apenas**. Terminais Bluetooth não são suportados atualmente.
:::

### Terminais Suportados (Conectados à Internet) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - Terminal conectado via WiFi/Ethernet

### Terminais Não Suportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - Apenas Bluetooth
- **SumUp 3G** ❌ - Apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado aos aplicativos iOS e Android em **Q1 2026**. Isso permitirá suporte a todos os terminais certificados pela SumUp, incluindo os modelos Air e 3G.
:::

## Solução de Problemas {#troubleshooting}

### Problemas Comuns {#common-issues}

<Accordion>
  <AccordionItem question="O leitor não emparelha">

  - Certifique-se de que está usando um terminal conectado à internet suportado (apenas Solo)
  - Verifique se o seu dispositivo SumUp Solo está conectado ao WiFi e online
  - Verifique se o dispositivo está no modo de emparelhamento e se o código de emparelhamento foi inserido corretamente
  - Confirme que sua conta SumUp tem acesso à API habilitado
  - Certifique-se de que a conexão com a internet esteja estável durante o emparelhamento

  </AccordionItem>

  <AccordionItem question="Pagamento não processando">

  - Verifique se o leitor está emparelhado corretamente
  - Confira se sua conta SumUp está ativa e em boas condições
  - Certifique-se de que o dispositivo tem bateria suficiente e conectividade
  - Confirme se a chave da API está correta e ativa

  </AccordionItem>

  <AccordionItem question="O pedido não está completando">

  - Verifique a estabilidade da conexão com a internet
  - Verifique as configurações de webhook na sua conta SumUp
  - Certifique-se de que o pagamento foi bem-sucedido no terminal SumUp
  - Verifique os logs de erro do WordPress para problemas de comunicação da API

  </AccordionItem>
</Accordion>

### Limites de Taxa da API {#api-rate-limits}

O gateway está otimizado para reduzir chamadas à API e evitar limites de taxa:
- O status do pagamento é verificado de forma eficiente
- Solicitações desnecessárias à API são minimizadas
- A lógica de re-tentativa automática lida com falhas temporárias
