# Gateway de Terminal SumUp

O gateway de Terminal SumUp permite que você aceite pagamentos com cartão usando leitores de cartão SumUp diretamente no WooCommerce POS. Este gateway oferece integração direta com o sistema de processamento de pagamentos da SumUp para transações presenciais.

## Recursos

- **Integração de Hardware**: Conecte leitores de cartão SumUp ao seu sistema de PDV via conexão com a internet
- **Processamento em Tempo Real**: Processamento de pagamento instantâneo e conclusão de pedido
- **Detecção Automática**: Pedidos completados automaticamente quando o pagamento é detectado
- **Transações Seguras**: Processamento de pagamento compatível com PCI através da SumUp
- **Emparelhamento Fácil**: Processo simples de emparelhamento de dispositivos com códigos de emparelhamento
- **SDK Web**: Utiliza o SDK JavaScript da SumUp para compatibilidade com aplicações web

## Instalação

### Passo 1: Baixar e Instalar

1. Visite a [página de lançamentos do SumUp Terminal para WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Baixe o arquivo mais recente **sumup-terminal-for-woocommerce.zip**
3. No seu admin do WordPress, vá para `Plugins > Adicionar Novo > Enviar Plugin`
4. Envie o arquivo zip e ative o plugin

### Passo 2: Configurar Definições SumUp

1. Navegue para `Admin WP > WooCommerce > Configurações > Pagamentos`
2. Encontre **Terminal SumUp** na lista de métodos de pagamento
3. Clique em **Terminal SumUp** para acessar as configurações
4. Insira sua **Chave de API SumUp** (disponível no seu painel de merchant SumUp)
5. Salve as configurações

:::note
Você não precisa habilitar o gateway de Terminal SumUp nas configurações do WooCommerce. Ele será habilitado especificamente para o PDV em uma etapa posterior.
:::

### Passo 3: Emparelhar Seu Terminal SumUp

1. Na mesma página de configurações, localize a seção **Emparelhar Leitor**
2. No seu dispositivo SumUp, navegue até a tela de emparelhamento para exibir o código de emparelhamento
3. Digite o código de emparelhamento exibido no seu dispositivo SumUp
4. Clique em **"Emparelhar Leitor"** para estabelecer a conexão
5. Aguarde a confirmação de que o leitor foi emparelhado com sucesso

:::warning Importante
O leitor deve ser emparelhado com sucesso antes de processar pagamentos. Certifique-se de concluir o processo de emparelhamento antes de continuar.
:::

### Passo 4: Habilitar no WooCommerce POS

1. Vá para `Admin WP > WooCommerce POS > Configurações > Checkout`
2. Encontre o gateway **Terminal SumUp** na lista
3. Habilite o gateway para uso no PDV
4. Salve suas configurações

## Uso

### Processando Pagamentos

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no PDV
2. **Selecionar Gateway**: Escolha "Terminal SumUp" como método de pagamento
3. **Iniciar Pagamento**: Clique para iniciar um novo pagamento no seu dispositivo SumUp
4. **Pagamento do Cliente**: Cliente conclui o pagamento no terminal SumUp
5. **Conclusão Automática**: O pedido é automaticamente concluído quando o pagamento é detectado

### Controles de Pagamento

Ao usar o gateway de Terminal SumUp, você tem as seguintes opções:

- **Iniciar Novo Pagamento**: Inicie uma solicitação de pagamento no terminal conectado
- **Cancelar Pagamento**: Cancele um pagamento em processo
- **Status de Pagamento**: Monitore o status atual do processo de pagamento

### Gerenciamento de Pedidos

- **Atualizações Automáticas**: Pedidos são marcados como concluídos automaticamente após pagamento bem-sucedido
- **Rastreamento de Pagamento**: Todos os detalhes de pagamento são registrados nas notas do pedido
- **Geração de Recibo**: Recibos padrão de PDV são gerados após pagamentos bem-sucedidos

## Requisitos

- **Conta SumUp**: Conta de merchant SumUp ativa
- **Acesso à API**: Chave de API SumUp do seu painel de merchant
- **Hardware Compatível**: Dispositivo de leitor de cartão SumUp
- **WooCommerce POS**: Versão Free ou Pro instalada e ativada
- **Conexão Estável**: Conexão com internet confiável para comunicação de API

## Compatibilidade de Hardware

:::important Requisitos de Conectividade
Esta implementação usa o SDK JavaScript da SumUp, o que significa que funciona através de aplicações web, mas requer apenas **terminais conectados à internet**. Terminais Bluetooth não são atualmente suportados.
:::

### Terminais Suportados (Conectados à Internet)

- **SumUp Solo** ✅ - Terminal conectado por WiFi/Ethernet

### Terminais Não Suportados (Bluetooth)

- **SumUp Air** ❌ - Apenas Bluetooth
- **SumUp 3G** ❌ - Apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado aos aplicativos iOS e Android no **Q1 2026**. Isso permitirá suporte para todos os terminais certificados SumUp, incluindo os modelos Air e 3G.
:::

## Resolução de Problemas

### Problemas Comuns

**Leitor Não Emparelha**
- Certifique-se de estar usando um terminal conectado à internet compatível (apenas Solo)
- Verifique se seu dispositivo SumUp Solo está conectado ao WiFi e online
- Verifique se o dispositivo está no modo de emparelhamento e o código de emparelhamento foi inserido corretamente
- Confirme que sua conta SumUp tem o acesso à API habilitado
- Assegure uma conexão com internet estável durante o emparelhamento

**Pagamento Não Processando**
- Verifique se o leitor está emparelhado corretamente
- Verifique se sua conta SumUp está ativa e em bom estado
- Certifique-se de que o dispositivo tenha bateria e conectividade suficientes
- Confirme que a chave de API está correta e ativa

**Pedido Não Concluindo**
- Verifique a estabilidade da conexão com internet
- Verifique as configurações de webhook na sua conta SumUp
- Certifique-se de que o pagamento foi bem-sucedido no terminal SumUp
- Verifique os logs de erro do WordPress para problemas de comunicação de API

### Limites de Taxa de API

O gateway é otimizado para reduzir chamadas de API e evitar limitações:
- Status de pagamento é verificado de forma eficiente
- Solicitações de API desnecessárias são minimizadas
- Lógica de nova tentativa automática lida com falhas temporárias
