# Gateways de Pagamento Personalizados

O WCPOS suporta diversos gateways de pagamento personalizados que estendem a funcionalidade do seu sistema de Ponto de Venda. Esses gateways são projetados especificamente para ambientes de PDV e fornecem integração perfeita com vários métodos e serviços de pagamento.

## Gateways Disponíveis

### Gateways de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[Terminal SumUp](./sumup-terminal)** - Processe pagamentos através de leitores de cartão SumUp

### Gateways de Pagamento Digital

- **[Fatura por E-mail](./email-invoice)** - Envie faturas de pagamento para clientes por e-mail
- **[Checkout Web](./web-checkout)** - Redirecione clientes para completar pagamentos online

## Criando Seu Próprio Gateway

Se você precisar de um gateway de pagamento personalizado para seus requisitos específicos, pode usar nosso **[Modelo de Gateway](./gateway-template)** para criar seu próprio gateway de pagamento WooCommerce POS.

## Visão Geral da Instalação

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do repositório GitHub correspondente
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar Novo > Carregar Plugin`
3. **Configure** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** o gateway em `WP Admin > WooCommerce POS > Configurações > Checkout`

## Requisitos

- Plugin WooCommerce POS (versão gratuita ou pro)
- WordPress com WooCommerce instalado
- Chaves de API ou credenciais apropriadas para o serviço de pagamento específico

## Suporte

Para problemas com gateways personalizados, visite o repositório GitHub correspondente e crie um issue com informações detalhadas sobre seu problema.
