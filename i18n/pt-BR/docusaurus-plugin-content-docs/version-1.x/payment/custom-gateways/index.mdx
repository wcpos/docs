# Gateways de Pagamento Personalizados {#custom-payment-gateways}

O WCPOS oferece suporte a vários gateways de pagamento personalizados que ampliam a funcionalidade do seu sistema de Ponto de Venda. Esses gateways são projetados especificamente para ambientes POS e proporcionam integração perfeita com diversos métodos e serviços de pagamento.

## Gateways Personalizados Disponíveis {#available-custom-gateways}

### Gateways de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processe pagamentos através de leitores de cartão SumUp

### Gateways de Pagamento Móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos pelo telefone via código QR ou notificação push

### Gateways de Pagamento Digital {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Envie faturas de pagamento aos clientes por e-mail
- **[Checkout Web](./web-checkout)** - Redireciona os clientes para concluir os pagamentos online

## Criando seu próprio gateway {#creating-your-own-gateway}

Se você precisar de um gateway de pagamento personalizado para suas necessidades específicas, pode usar nosso **[Modelo de Gateway](./gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão geral da instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente no repositório do GitHub correspondente
2. **Instale** o plugin via `WP Admin > Plugins > Add New > Upload Plugin`
3. **Configure** as opções do gateway em `WP Admin > WooCommerce > Settings > Payments`
4. **Habilite** o gateway em `WP Admin > POS > Settings > Checkout`
