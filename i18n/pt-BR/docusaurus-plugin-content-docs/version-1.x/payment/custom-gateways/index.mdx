# Passarelas de Pagamento Personalizadas {#custom-payment-gateways}

WCPOS suporta várias passarelas de pagamento personalizadas que expandem a funcionalidade do seu sistema de Ponto de Venda. Essas passarelas são projetadas especificamente para ambientes de POS e fornecem integração perfeita com vários métodos e serviços de pagamento.

## Passarelas Personalizadas Disponíveis {#available-custom-gateways}

### Passarelas de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos utilizando leitores de hardware do Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processar pagamentos através de leitores de cartões SumUp

### Passarelas de Pagamento Móvel {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos via celular através de código QR ou notificação por push

### Passarelas de Pagamento Digital

- **[Fatura por E-mail](./email-invoice)** - Envie faturas de pagamento para clientes por e-mail
- **[Web Checkout](./web-checkout)** - Redirecione clientes para completar pagamentos online

## Criando Sua Própria Passarela {#creating-your-own-gateway}

Se você precisar de uma passarela de pagamento personalizada para seus requisitos específicos, pode usar nosso **[Modelo de Passarela](./gateway-template)** para criar sua própria passarela de pagamento WooCommerce POS.

## Visão Geral da Instalação {#installation-overview}

Todas as passarelas personalizadas seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório GitHub
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar Novo > Fazer Upload do Plugin`
3. **Configure** as configurações da passarela em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** a passarela em `WP Admin > WooCommerce POS > Configurações > Checkout`
