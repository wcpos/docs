# Gateways de Pagamento Personalizados {#custom-payment-gateways}

WCPOS suporta vários gateways de pagamento personalizados que estendem a funcionalidade do seu sistema de Ponto de Venda. Esses gateways são progettados especificamente para ambientes de POS e fornecem integração perfeita com vários métodos e serviços de pagamento.

## Gateways Personalizados Disponíveis {#available-custom-gateways}

### Gateways para Terminais de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processar pagamentos através de leitores de cartão SumUp

### Gateways de Pagamento Móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos baseados em telefone via código QR ou notificação push

### Gateways de Pagamento Digital {#digital-payment-gateways}

- **[Fatura por E-mail](./email-invoice)** - Enviar faturas de pagamento para clientes por e-mail
- **[Checkout na Web](./web-checkout)** - Redirecionar clientes para completar pagamentos online

## Criando Seu Próprio Gateway {#creating-your-own-gateway}

Se você precisa de um gateway de pagamento personalizado para suas necessidades específicas, pode usar nosso **[Modelo de Gateway](./gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão Geral da Instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixar** a versão mais recente do respectivo repositório do GitHub
2. **Instalar** o plugin via `WP Admin > Plugins > Adicionar Novo > Fazer Upload do Plugin`
3. **Configurar** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ativar** o gateway em `WP Admin > POS > Configurações > Checkout`
