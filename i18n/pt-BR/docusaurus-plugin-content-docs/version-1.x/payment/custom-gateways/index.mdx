# Gateways de Pagamento Personalizados {#custom-payment-gateways}

O WCPOS oferece suporte a diversos gateways de pagamento personalizados que ampliam a funcionalidade do seu sistema de Ponto de Venda. Esses gateways foram projetados especificamente para ambientes de PDV e proporcionam integração perfeita com vários métodos e serviços de pagamento.

## Gateways Personalizados Disponíveis {#available-custom-gateways}

### Gateways de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando os leitores de hardware do Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processe pagamentos através das leitoras de cartão SumUp

### Gateways de Pagamento Móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos via celular através de código QR ou notificação push

### Gateways de Pagamento Digital {#digital-payment-gateways}

- **[Fatura por E-mail](./email-invoice)** - Envie faturas de pagamento aos clientes por e-mail
- **[Checkout Web](./web-checkout)** - Redirecione os clientes para concluir os pagamentos online

## Criando seu próprio gateway {#creating-your-own-gateway}

Se você precisa de um gateway de pagamento personalizado para seus requisitos específicos, pode usar nosso **[Modelo de Gateway](./gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão geral da instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório do GitHub
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar novo > Enviar plugin`
3. **Configure** as opções do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** o gateway em `WP Admin > PDV > Configurações > Checkout`
