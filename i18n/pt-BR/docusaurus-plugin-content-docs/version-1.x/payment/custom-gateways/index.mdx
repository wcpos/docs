# Passarelas de Pagamento Personalizadas

O WCPOS suporta várias passarelas de pagamento personalizadas que ampliam a funcionalidade do seu sistema de Ponto de Venda. Essas passarelas são projetadas especificamente para ambientes de PDV e proporcionam integração perfeita com diversos métodos e serviços de pagamento.

## Passarelas Personalizadas Disponíveis

### Passarelas de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processar pagamentos através de leitores de cartão SumUp

### Passarelas de Pagamento Digital

- **[Fatura por E-mail](./email-invoice)** - Enviar faturas de pagamento para clientes via e-mail
- **[Checkout Web](./web-checkout)** - Redirecionar clientes para completar pagamentos online

## Criando Sua Própria Passarela

Se você precisar de uma passarela de pagamento personalizada para suas necessidades específicas, pode usar nosso **[Modelo de Passarela](./gateway-template)** para criar sua própria passarela de pagamento WooCommerce POS.

## Visão Geral da Instalação

Todas as passarelas personalizadas seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório do GitHub
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar Novo > Carregar Plugin`
3. **Configure** as configurações da passarela em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** a passarela em `WP Admin > WooCommerce POS > Configurações > Checkout`

## Requisitos

- Plugin WooCommerce POS (versão gratuita ou pro)
- WordPress com WooCommerce instalado
- Chaves API ou credenciais apropriadas para o serviço de pagamento específico

## Suporte

Para problemas com passarelas personalizadas, visite o respectivo repositório do GitHub e crie um problema com informações detalhadas sobre o seu problema.
