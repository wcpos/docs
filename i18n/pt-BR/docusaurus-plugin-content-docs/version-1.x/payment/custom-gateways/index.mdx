# Gateways de Pagamento Personalizados

WCPOS suporta vários gateways de pagamento personalizados que expandem a funcionalidade do seu sistema de Ponto de Venda. Esses gateways são projetados especificamente para ambientes de POS e fornecem integração perfeita com vários métodos e serviços de pagamento.

## Gateways Personalizados Disponíveis

### Gateways de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processar pagamentos através de leitores de cartão SumUp

### Gateways de Pagamento Digital

- **[Fatura por Email](./email-invoice)** - Envie faturas de pagamento para clientes via e-mail
- **[Finalização na Web](./web-checkout)** - Redirecione clientes para completar pagamentos online

## Criando Seu Próprio Gateway

Se você precisar de um gateway de pagamento personalizado para suas necessidades específicas, você pode usar nosso **[Modelo de Gateway](./gateway-template)** para criar seu próprio gateway de pagamento WooCommerce POS.

## Visão Geral da Instalação

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a última versão do respectivo repositório do GitHub
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar Novo > Enviar Plugin`
3. **Configure** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Habilite** o gateway em `WP Admin > WooCommerce POS > Configurações > Checkout`
