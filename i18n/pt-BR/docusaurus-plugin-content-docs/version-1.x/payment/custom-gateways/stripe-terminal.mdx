# Gateway do Terminal Stripe

O gateway do Terminal Stripe permite que você aceite pagamentos presenciais usando leitores de hardware do Stripe Terminal diretamente no WooCommerce POS. Este gateway oferece suporte a leitores físicos de cartão e modo de simulação para testes.

## Recursos

- **Integração de Hardware**: Conecte leitores físicos do Terminal Stripe via conexão de internet
- **Modo Simulador**: Teste pagamentos sem hardware usando o simulador do Stripe
- **Múltiplos Métodos de Pagamento**: Suporte para diversos tipos de cartão e métodos de pagamento
- **Processamento em Tempo Real**: Processamento e confirmação instantâneos de pagamento
- **Transações Seguras**: Processamento de pagamento compatível com PCI através do Stripe
- **SDK Web**: Utiliza o SDK JavaScript do Stripe para compatibilidade com aplicações web

## Instalação

### Passo 1: Baixar e Instalar

1. Visite a [página de lançamentos do Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Baixe o arquivo mais recente **stripe-terminal-for-woocommerce.zip**
3. No admin do WordPress, vá para `Plugins > Adicionar Novo > Carregar Plugin`
4. Carregue o arquivo zip e ative o plugin

### Passo 2: Configurar Configurações do Stripe

1. Navegue para `Admin do WP > WooCommerce > Configurações > Pagamentos`
2. Encontre **Terminal Stripe** na lista de métodos de pagamento
3. Clique em **Terminal Stripe** para acessar as configurações
4. Insira sua **Chave Secreta do Stripe** (você pode obtê-la no Painel do Stripe)
5. Salve as configurações

:::note
Você não precisa habilitar o gateway do Terminal Stripe nas configurações do WooCommerce. Ele será habilitado especificamente para o POS no próximo passo.
:::

### Passo 3: Habilitar no WooCommerce POS

1. Vá para `Admin do WP > WooCommerce POS > Configurações > Checkout`
2. Encontre o gateway **Terminal Stripe** na lista
3. Habilite o gateway para uso no POS
4. Salve suas configurações

## Uso

### Conectando um Leitor

Quando você seleciona o gateway do Terminal Stripe durante o checkout no POS:

1. **Escolha o Método de Conexão**: Você pode conectar um leitor físico ou usar o simulador
2. **Leitor Físico**: Siga as instruções na tela para conectar seu dispositivo Terminal Stripe
3. **Simulador**: Selecione o modo simulador para testar diversos cenários de pagamento sem hardware

### Processando Pagamentos

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Terminal Stripe" como método de pagamento
3. **Conectar Leitor**: Conecte seu leitor ou escolha o modo simulador
4. **Processar Pagamento**: Siga os prompts para concluir a transação
5. **Confirmação**: O pedido será concluído automaticamente após pagamento bem-sucedido

### Testando com Simulador

O simulador permite testar diversos métodos de pagamento e cenários:

- **Pagamentos com Cartão**: Teste diferentes tipos de cartão (Visa, Mastercard, etc.)
- **Pagamentos Sem Contato**: Simule transações por aproximação
- **Cenários de Erro**: Teste pagamentos recusados e outras condições de erro
- **Diferentes Valores**: Teste diversos valores de transação

## Requisitos

- **Conta Stripe**: Conta Stripe ativa com Terminal habilitado
- **Chaves de API**: Chave secreta do Stripe do seu painel
- **WooCommerce POS**: Versão gratuita ou profissional instalada e ativada
- **HTTPS**: Seu site deve usar SSL/HTTPS para segurança

## Compatibilidade de Hardware

:::important Requisitos de Conectividade
Esta implementação usa o SDK JavaScript do Stripe, o que significa que funciona através de aplicações web, mas requer **terminais conectados à internet apenas**. Terminais Bluetooth não são suportados atualmente.
:::

### Terminais Suportados (Conectados à Internet)

- **Stripe Reader S700** ✅ - Terminal conectado por Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado por WiFi

### Terminais Não Suportados (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Apenas Bluetooth
- **BBPOS WisePad 3** ❌ - Apenas Bluetooth  
- **Verifone P400** ❌ - Apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado aos aplicativos iOS e Android no **Q1 2026**. Isso permitirá suporte para todos os leitores certificados do Terminal Stripe, incluindo o M2 e WisePad 3.
:::

## Solução de Problemas

### Problemas Comuns

**Leitor Não Conecta**
- Certifique-se de usar um terminal conectado à internet suportado (S700 ou WisePOS E)
- Verifique se o terminal está conectado ao WiFi/Ethernet e online
- Verifique se sua conta Stripe tem o Terminal habilitado
- Confirme que o terminal está registrado no seu Painel do Stripe

**Pagamento Recusado**
- Verifique se sua conta Stripe está ativa e em boas condições
- Confirme que o cartão sendo usado é válido
- Assegure-se de que há fundos suficientes disponíveis

**Erros de Certificado SSL**
- O Terminal Stripe requer HTTPS - certifique-se de que seu site tenha um certificado SSL válido
- Verifique se seu certificado SSL está configurado corretamente
