# Gateway Stripe Terminal

O gateway Stripe Terminal permite que você aceite pagamentos presenciais usando leitores de hardware Stripe Terminal diretamente no WooCommerce POS. Este gateway suporta tanto leitores de cartão físicos quanto modo simulador para testes.

## Recursos

- **Integração de Hardware**: Conecte leitores físicos Stripe Terminal via conexão à internet
- **Modo Simulador**: Teste pagamentos sem hardware usando o simulador da Stripe
- **Múltiplos Métodos de Pagamento**: Suporte para vários tipos de cartão e métodos de pagamento
- **Processamento em Tempo Real**: Processamento instantâneo de pagamentos e confirmação
- **Transações Seguras**: Processamento de pagamentos em conformidade com PCI pela Stripe
- **SDK Baseado na Web**: Usa o SDK JavaScript da Stripe para compatibilidade com aplicativos web

## Instalação

### Passo 1: Baixar e Instalar

1. Visite a [página de lançamentos do Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Baixe o arquivo **stripe-terminal-for-woocommerce.zip** mais recente
3. No seu WordPress admin, vá para `Plugins > Adicionar Novo > Enviar Plugin`
4. Faça o upload do arquivo zip e ative o plugin

### Passo 2: Configurar as Configurações da Stripe

1. Navegue até `WP Admin > WooCommerce > Configurações > Pagamentos`
2. Encontre **Stripe Terminal** na lista de métodos de pagamento
3. Clique em **Stripe Terminal** para acessar as configurações
4. Insira sua **Chave Secreta da Stripe** (você pode obtê-la no seu Dashboard da Stripe)
5. Salve as configurações

:::note
Você não precisa habilitar o gateway Stripe Terminal nas configurações do WooCommerce. Ele será habilitado especificamente para o POS no próximo passo.
:::

### Passo 3: Habilitar no WooCommerce POS

1. Vá para `WP Admin > WooCommerce POS > Configurações > Checkout`
2. Encontre o gateway **Stripe Terminal** na lista
3. Habilite o gateway para uso no POS
4. Salve suas configurações

## Uso

### Conectando um Leitor

Quando você selecionar o gateway Stripe Terminal durante o checkout no POS:

1. **Escolha o Método de Conexão**: Você pode conectar um leitor físico ou usar o simulador
2. **Leitor Físico**: Siga as instruções na tela para conectar seu dispositivo Stripe Terminal
3. **Simulador**: Selecione o modo simulador para testar vários cenários de pagamento sem hardware

### Processando Pagamentos

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Stripe Terminal" como método de pagamento
3. **Conectar Leitor**: Conecte seu leitor ou escolha o modo simulador
4. **Processar Pagamento**: Siga as instruções para concluir a transação
5. **Confirmação**: O pedido será automaticamente concluído após o pagamento bem-sucedido

### Testando com o Simulador

O simulador permite que você teste vários métodos e cenários de pagamento:

- **Pagamentos com Cartão**: Teste diferentes tipos de cartão (Visa, Mastercard, etc.)
- **Pagamentos Sem Contato**: Simule transações de pagamento por aproximação
- **Cenários de Erro**: Teste pagamentos recusados e outras condições de erro
- **Valores Diferentes**: Teste vários valores de transação

## Requisitos

- **Conta Stripe**: Conta Stripe ativa com Terminal habilitado
- **Chaves de API**: Chave secreta da Stripe do seu dashboard
- **WooCommerce POS**: Versão gratuita ou Pro instalada e ativada
- **HTTPS**: Seu site deve usar SSL/HTTPS para segurança

## Compatibilidade de Hardware

:::important Requisitos de Conectividade
Esta implementação usa o SDK JavaScript da Stripe, o que significa que funciona através de aplicativos da web, mas requer **terminais conectados à internet apenas**. Terminais Bluetooth não são atualmente suportados.
:::

### Terminais Suportados (Conectados à Internet)

- **Stripe Reader S700** ✅ - Terminal conectado via Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado via WiFi

### Terminais Não Suportados (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Apenas Bluetooth
- **BBPOS WisePad 3** ❌ - Apenas Bluetooth  
- **Verifone P400** ❌ - Apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado aos aplicativos iOS e Android no **Q1 2026**. Isso possibilitará suporte a todos os leitores certificados do Stripe Terminal, incluindo o M2 e WisePad 3.
:::

## Solução de Problemas

### Problemas Comuns

**Leitor Não Conecta**
- Certifique-se de que você está usando um terminal conectado à internet suportado (S700 ou WisePOS E)
- Verifique se o terminal está conectado ao WiFi/Ethernet e online
- Confirme que sua conta Stripe tem o Terminal habilitado
- Confirme que o terminal está registrado no seu Dashboard da Stripe

**Pagamento Recusado**
- Verifique se sua conta Stripe está ativa e em situação regular
- Verifique se o cartão utilizado é válido
- Certifique-se de que há fundos suficientes disponíveis

**Erros de Certificado SSL**
- O Stripe Terminal requer HTTPS - certifique-se de que seu site possui um certificado SSL válido
- Verifique se seu certificado SSL está configurado corretamente
