# Gateway do Stripe Terminal {#stripe-terminal-gateway}

O gateway do Stripe Terminal permite que você aceite pagamentos presenciais usando leitores de hardware do Stripe Terminal diretamente no WooCommerce POS. Este gateway suporta tanto leitores de cartão físicos quanto modo simulador para testes.

## Instalação {#installation}

<Steps>
  <StepCard number="1" title="Baixar e Instalar">

1. Visite a [página de lançamentos do Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Baixe o arquivo mais recente **stripe-terminal-for-woocommerce.zip**
3. No seu admin do WordPress, vá para `Plugins > Adicionar Novo > Fazer Upload do Plugin`
4. Faça o upload do arquivo zip e ative o plugin

  </StepCard>
  <StepCard number="2" title="Configurar Configurações do Stripe">

1. Navegue até `Admin WP > WooCommerce > Configurações > Pagamentos`
2. Encontre **Stripe Terminal** na lista de métodos de pagamento
3. Clique em **Stripe Terminal** para acessar as configurações
4. Insira sua **Chave Secreta do Stripe** (você pode obter isso no seu Painel da Stripe)
5. Salve as configurações

:::note
Você não precisa habilitar o gateway do Stripe Terminal nas configurações do WooCommerce. Ele será habilitado especificamente para o POS na próxima etapa.
:::

  </StepCard>
  <StepCard number="3" title="Habilitar no WooCommerce POS" isLast>

1. Vá para `Admin WP > WooCommerce POS > Configurações > Finalização`
2. Encontre o gateway **Stripe Terminal** na lista
3. Habilite o gateway para uso no POS
4. Salve suas configurações

  </StepCard>
</Steps>

## Uso {#usage}

### Conectando um Leitor {#connecting-a-reader}

Quando você seleciona o gateway do Stripe Terminal durante a finalização no POS:

1. **Escolha o Método de Conexão**: Você pode conectar um leitor físico ou usar o simulador
2. **Leitor Físico**: Siga as instruções na tela para conectar seu dispositivo do Stripe Terminal
3. **Simulador**: Selecione o modo simulador para testar vários cenários de pagamento sem hardware

### Processamento de Pagamentos {#processing-payments}

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Stripe Terminal" como método de pagamento
3. **Conectar Leitor**: Conecte seu leitor ou escolha o modo simulador
4. **Processar Pagamento**: Siga as instruções para concluir a transação
5. **Confirmação**: O pedido será automaticamente concluído após o pagamento bem-sucedido

### Testando com o Simulador {#testing-with-simulator}

O simulador permite que você teste vários métodos e cenários de pagamento:

- **Pagamentos com Cartão**: Teste diferentes tipos de cartões (Visa, Mastercard, etc.)
- **Pagamentos Sem Contato**: Simule transações de aproximação para pagamento
- **Cenários de Erro**: Teste pagamentos recusados e outras condições de erro
- **Montantes Diferentes**: Teste vários valores de transação

## Compatibilidade de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividade
Esta implementação usa o SDK JavaScript da Stripe, o que significa que funciona através de aplicações web, mas requer **terminais conectados à internet apenas**. Terminais Bluetooth não são suportados no momento.
:::

### Terminais Suportados (Conectados à Internet) {#supported-terminals-internet-connected}

- **Leitor Stripe S700** ✅ - terminal conectado por Ethernet/WiFi
- **WisePOS E** ✅ - terminal conectado por WiFi

### Terminais Não Suportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - apenas Bluetooth
- **BBPOS WisePad 3** ❌ - apenas Bluetooth  
- **Verifone P400** ❌ - apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado às aplicações iOS e Android no **Q1 2026**. Isso habilitará suporte para todos os leitores certificados do Stripe Terminal, incluindo o M2 e WisePad 3.
:::

### Problemas Comuns {#common-issues}

<Accordion>
  <AccordionItem question="Leitor não conecta">

  - Certifique-se de que você está usando um terminal suportado conectado à internet (S700 ou WisePOS E)
  - Verifique se o terminal está conectado ao WiFi/Ethernet e online
  - Confira se sua conta Stripe tem o Terminal habilitado
  - Confirme se o terminal está registrado no seu Painel Stripe

  </AccordionItem>

  <AccordionItem question="Pagamento recusado">

  - Verifique se sua conta Stripe está ativa e em bom estado
  - Verifique se o cartão utilizado é válido
  - Assegure-se de que há fundos suficientes disponíveis

  </AccordionItem>

  <AccordionItem question="Erros de certificado SSL">

  - O Stripe Terminal requer HTTPS - certifique-se de que seu site possui um certificado SSL válido
  - Verifique se seu certificado SSL está corretamente configurado

  </AccordionItem>
</Accordion>
