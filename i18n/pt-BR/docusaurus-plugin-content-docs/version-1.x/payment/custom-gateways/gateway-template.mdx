# Modelo de Gateway

O Modelo de Gateway WooCommerce fornece um ponto de partida para criar seus próprios gateways de pagamento personalizados para WooCommerce POS. Este modelo inclui todo o código padrão e estrutura necessários para construir um gateway de pagamento totalmente funcional.

## Recursos

- **Modelo Completo**: Estrutura de gateway pronta para uso com todos os métodos necessários
- **Integração POS**: Pré-configurado para compatibilidade com WooCommerce POS
- **Configuração Automatizada**: Personalização de modelo baseada em script
- **Melhores Práticas**: Segue os padrões de codificação do WordPress e WooCommerce
- **Extensível**: Fácil de modificar e estender para provedores de pagamento específicos

## Primeiros Passos

### Opção 1: Geração Automatizada de Modelo

O modelo inclui um script que personaliza automaticamente o modelo para seu gateway específico:

1. **Clonar o Repositório**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **Executar o Script de Configuração**:
   ```bash
   ./create-gateway.sh
   ```

3. **Seguir os Prompts**:
   - Digite o nome do gateway (por exemplo, "Meu Gateway de Pagamento")
   - Digite um slug do gateway (por exemplo, "meu-pagamento")
   - Forneça uma descrição
   - O script gerará um plugin personalizado

### Opção 2: Uso Manual do Modelo

Se preferir personalização manual:

1. **Baixar o Modelo**:
   - Visite o [repositório do Modelo de Gateway](https://github.com/wcpos/woocommerce-gateway-template)
   - Baixe a versão mais recente ou clone o repositório

2. **Personalizar o Modelo**:
   - Substitua `{{GATEWAY_NAME}}` pelo nome de exibição do seu gateway
   - Substitua `{{GATEWAY_SLUG}}` pelo identificador único do seu gateway
   - Substitua `{{GATEWAY_DESCRIPTION}}` pela descrição do seu gateway

3. **Renomear Arquivos**:
   - Renomeie `wcpos-{{GATEWAY_SLUG}}.php` para corresponder ao slug do gateway
   - Atualize os cabeçalhos do arquivo e as informações do plugin
