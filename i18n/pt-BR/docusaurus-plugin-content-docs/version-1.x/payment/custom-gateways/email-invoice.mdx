# Gateway de Fatura por Email

O gateway de Fatura por Email permite enviar faturas de pagamento para clientes por email diretamente do WooCommerce POS. Isso é perfeito para situações em que os clientes preferem pagar depois ou precisam concluir o pagamento online usando os métodos de pagamento da sua loja web.

## Recursos

- **Integração de Email**: Envie faturas diretamente da interface do POS
- **Gerenciamento de Clientes**: Preencha automaticamente o email a partir dos dados existentes do cliente ou adicione novos endereços de email
- **Pedidos de Convidados**: Suporte para clientes convidados com atualização opcional de endereço de cobrança
- **Pagamento Online**: Clientes recebem um link "Pagar por este pedido" para concluir o pagamento online
- **Fluxo de Trabalho Flexível**: Conclua pedidos na loja ou permita a conclusão de pagamento remoto

## Instalação

### Passo 1: Baixar e Instalar

1. Visite a [página de lançamentos do Gateway de Fatura por Email](https://github.com/wcpos/email-invoice-gateway/releases)
2. Baixe o arquivo **woocommerce-pos-email-invoice-gateway.zip** mais recente
3. No admin do WordPress, vá para `Plugins > Adicionar Novo > Enviar Plugin`
4. Faça upload do arquivo zip e ative o plugin

### Passo 2: Ativar no WooCommerce POS

1. Vá para `Admin do WP > WooCommerce POS > Configurações > Checkout`
2. Localize o **Gateway de Pagamento de Fatura do WooCommerce POS** na lista
3. Ative o gateway para uso no POS
4. Salve suas configurações

:::note
Este gateway não requer chaves de API adicionais ou configuração de serviço externo. Ele usa o sistema de email do WordPress existente e os métodos de pagamento do WooCommerce.
:::

## Uso

### Enviando Faturas

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Fatura por Email" como método de pagamento
3. **Endereço de Email**: 
   - Para clientes existentes: O email é preenchido automaticamente a partir dos dados do cliente
   - Para pedidos de convidados: Insira o endereço de email do cliente manualmente
4. **Opcional**: Marque "Salvar no endereço de cobrança" para atualizar as informações de cobrança do cliente
5. **Enviar Fatura**: Conclua o processo para enviar o email de fatura

### Experiência do Cliente

Quando você envia uma fatura, o cliente recebe um email contendo:

- **Detalhes do Pedido**: Descrição completa dos itens, quantidades e preços
- **Total do Pedido**: Valor final devido, incluindo impostos e taxas
- **Link de Pagamento**: Link direto para concluir o pagamento online
- **Informações do Pedido**: Número do pedido e detalhes de referência

### Conclusão do Pagamento

Os clientes podem concluir o pagamento de duas maneiras:

1. **Pagamento Online**: Clicar no link "Pagar por este pedido" no email
2. **Retorno à Loja**: Retornar para concluir o pagamento usando outros métodos de pagamento do POS

O link "Pagar por este pedido" leva os clientes a uma página de pagamento dedicada no seu site, onde podem usar qualquer um dos seus gateways de pagamento web habilitados (Stripe, PayPal, etc.).

## Gerenciamento de Email

### Clientes Existentes

- Endereços de email são recuperados automaticamente dos perfis dos clientes
- As informações de cobrança do cliente são preservadas
- O histórico de pedidos é mantido na conta do cliente

### Clientes Convidados

- Insira manualmente endereços de email para clientes ocasionais
- Opção de salvar email no endereço de cobrança para referência futura
- Pedidos de convidados são rastreados adequadamente no WooCommerce

### Modelos de Email

O gateway usa os modelos de email padrão do WooCommerce:

- **Personalizável**: Modifique a aparência do email através das configurações de email do WooCommerce
- **Marca**: Inclua o logotipo e a identidade visual da sua loja
- **Profissional**: Formato de fatura limpo e profissional
- **Compatível com Dispositivos Móveis**: Design responsivo para todos os dispositivos

## Casos de Uso

### Perfeito Para

- **Vendas B2B**: Clientes empresariais que precisam processar pagamentos através de seus sistemas contábeis
- **Pedidos Grandes**: Transações de alto valor que requerem aprovação ou tempo de processamento
- **Clientes Remotos**: Clientes que precisam concluir o pagamento após deixar a loja
- **Clientes com Conta**: Clientes regulares com termos de pagamento estabelecidos
- **Cotação para Pedido**: Converter cotações ou estimativas em faturas pagáveis

### Exemplos de Fluxo de Trabalho

**Loja Varejista**
1. Cliente seleciona itens, mas precisa de aprovação para compra
2. Envie fatura por email com todos os detalhes
3. Cliente conclui pagamento online quando estiver pronto
4. Pedido atualizado automaticamente no seu sistema

**Negócio de Serviços**
1. Conclua o trabalho de serviço e adicione itens ao POS
2. Envie fatura para o departamento contábil do cliente
3. Cliente paga online usando o método de pagamento preferido
4. Receba confirmação de pagamento e conclua o pedido

## Requisitos

- **WooCommerce POS**: Versão Gratuita ou Pro instalada e ativada
- **Sistema de Email**: Configuração de email do WordPress funcionando
- **Gateways de Pagamento Web**: Pelo menos um método de pagamento online habilitado no WooCommerce
- **Certificado SSL**: HTTPS necessário para processamento de pagamento seguro

## Solução de Problemas

### Problemas Comuns

**Emails Não Enviando**
- Verifique a configuração de email do WordPress
- Confirme as configurações de SMTP se estiver usando serviço de email personalizado
- Teste com plugins de teste de email do WordPress
- Verifique pastas de spam para emails de teste

**Links de Pagamento Não Funcionando**
- Certifique-se de que o WooCommerce esteja configurado corretamente
- Verifique se pelo menos um gateway de pagamento está habilitado para checkout web
- Verifique se o certificado SSL está instalado corretamente
- Confirme que o status do pedido permite pagamento

**Cliente Não Consegue Concluir Pagamento**
- Verifique se os gateways de pagamento web estão ativos e configurados
- Verifique se o pedido não expirou ou foi cancelado
- Certifique-se de que o cliente tenha limites suficientes no método de pagamento
- Teste o processo de pagamento você mesmo

### Entrega de Email

Para entrega de email confiável, considere:

- **Serviço SMTP**: Use serviços como SendGrid, Mailgun ou Amazon SES
- **Plugins de Email**: Instale plugins SMTP do WordPress para melhor entrega
- **Autenticação de Domínio**: Configure registros SPF, DKIM e DMARC
- **Monitoramento**: Acompanhe a entrega e taxas de abertura de emails
