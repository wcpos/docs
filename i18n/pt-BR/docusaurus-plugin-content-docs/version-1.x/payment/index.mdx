O WCPOS suporta múltiplos métodos de pagamento para atender diferentes necessidades de negócios.

## Métodos de Pagamento Padrão

A versão gratuita inclui dois gateways de pagamento integrados:

### Dinheiro

- Digite o valor recebido
- Cálculo automático de troco
- Nenhuma configuração adicional necessária

### Cartão

- Para uso com terminais de cartão externos
- Simplesmente marque o pagamento como concluído após processar no seu terminal
- Nenhuma integração direta necessária

## Gateways de Pagamento Adicionais (Pro)

:::info Recurso Pro
Gateways de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode habilitar gateways de pagamento adicionais do WooCommerce no checkout do POS:

- **Terminal Stripe** - Integração direta com leitores de cartão Stripe
- **Terminal SumUp** - Integração com leitores de cartão SumUp
- **Qualquer Gateway WooCommerce** - Habilite qualquer gateway instalado na sua loja

## Gateways Personalizados

Crie suas próprias integrações de pagamento usando o sistema de [Gateways Personalizados](/payment/custom-gateways).
