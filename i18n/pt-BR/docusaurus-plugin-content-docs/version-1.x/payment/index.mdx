O WCPOS suporta múltiplos métodos de pagamento para atender diferentes necessidades de negócio.

## Métodos de Pagamento Padrão

A versão gratuita inclui dois gateways de pagamento incorporados:

### Dinheiro

- Insira o valor recebido
- Cálculo automático de troco
- Nenhuma configuração adicional é necessária

### Cartão

- Para uso com terminais de cartão externos
- Basta marcar o pagamento como completo após o processamento em seu terminal
- Nenhuma integração direta é necessária

## Gateways de Pagamento Adicionais (Pro)

:::info Recurso Pro
Gateways de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode habilitar gateways de pagamento WooCommerce adicionais na finalização da venda no POS:

- **Stripe Terminal** - Integração direta com leitores de cartão Stripe
- **SumUp Terminal** - Integração com leitores de cartão SumUp
- **Qualquer Gateway WooCommerce** - Habilite qualquer gateway instalado em sua loja

## Gateways Personalizados

Crie suas próprias integrações de pagamento usando o sistema [Gateways Personalizados](/payment/custom-gateways).
