O WCPOS suporta múltiplos métodos de pagamento para atender diferentes necessidades empresariais.

## Métodos de Pagamento Padrão {#default-payment-methods}

A versão gratuita inclui dois gateways de pagamento integrados:

### Dinheiro {#cash}

- Insira o valor entregue
- Cálculo automático de troco
- Nenhuma configuração adicional necessária

### Cartão {#card}

- Para uso com terminais de cartão externos
- Simplesmente marque o pagamento como completo após processá-lo no seu terminal
- Nenhuma integração direta necessária

## Gateways de Pagamento Adicionais (Pro) {#additional-payment-gateways-pro}

:::info Recurso Pro
Gateways de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode ativar gateways de pagamento adicionais do WooCommerce no checkout do POS:

- **Stripe Terminal** - Integração direta com leitores de cartão Stripe
- **SumUp Terminal** - Integração com leitores de cartão SumUp
- **Qualquer Gateway WooCommerce** - Ative qualquer gateway instalado em sua loja

## Gateways Personalizados {#custom-gateways}

Crie suas próprias integrações de pagamento usando o sistema de [Gateways Personalizados](/payment/custom-gateways).
