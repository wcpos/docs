# Gateway do Terminal Stripe {#stripe-terminal-gateway}

O gateway do Terminal Stripe permite que você aceite pagamentos presenciais usando leitores de hardware do Terminal Stripe diretamente dentro do WCPOS. Este gateway suporta tanto leitores de cartão físicos quanto o modo simulador para testes.

## Instalação {#installation}

<Steps>
  <StepCard number="1" title="Baixar e Instalar">

1. Visite a [página de lançamentos do Terminal Stripe para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Baixe o arquivo **stripe-terminal-for-woocommerce.zip** mais recente
3. No seu painel do WordPress, vá para `Plugins > Adicionar Novo > Fazer Upload do Plugin`
4. Faça o upload do arquivo zip e ative o plugin

  </StepCard>
  <StepCard number="2" title="Configurar Configurações da Stripe">

1. Navegue até `Admin do WP > WooCommerce > Configurações > Pagamentos`
2. Encontre **Terminal Stripe** na lista de métodos de pagamento
3. Clique em **Terminal Stripe** para acessar as configurações
4. Insira sua **Chave Secreta da Stripe** (você pode obtê-la no seu Painel da Stripe)
5. Salve as configurações

:::note
Você não precisa ativar o gateway do Terminal Stripe nas configurações do WooCommerce. Ele será ativado especificamente para o POS no próximo passo.
:::

  </StepCard>
  <StepCard number="3" title="Ativar no WCPOS" isLast>

1. Vá para `Admin do WP > POS > Configurações > Checkout`
2. Encontre o gateway **Terminal Stripe** na lista
3. Ative o gateway para uso no POS
4. Salve suas configurações

  </StepCard>
</Steps>

## Uso {#usage}

### Conectando um Leitor {#connecting-a-reader}

Quando você seleciona o gateway do Terminal Stripe durante o checkout no POS:

1. **Escolha o Método de Conexão**: Você pode conectar um leitor físico ou usar o simulador
2. **Leitor Físico**: Siga as instruções na tela para conectar seu dispositivo do Terminal Stripe
3. **Simulador**: Selecione o modo simulador para testar vários cenários de pagamento sem hardware

### Processando Pagamentos {#processing-payments}

1. **Adicionar Itens**: Adicione produtos ao seu carrinho no POS
2. **Selecionar Gateway**: Escolha "Terminal Stripe" como método de pagamento
3. **Conectar Leitor**: Conecte seu leitor ou escolha o modo simulador
4. **Processar Pagamento**: Siga as instruções para completar a transação
5. **Confirmação**: O pedido será automaticamente concluído após o pagamento bem-sucedido

### Testando com Simulador {#testing-with-simulator}

O simulador permite que você teste vários métodos e cenários de pagamento:

- **Pagamentos com Cartão**: Teste diferentes tipos de cartão (Visa, Mastercard, etc.)
- **Pagamentos Sem Contato**: Simule transações de pagamento por aproximação
- **Cenários de Erro**: Teste pagamentos recusados e outras condições de erro
- **Valores Diferentes**: Teste vários valores de transação

## Compatibilidade de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividade
Esta implementação usa o SDK JavaScript da Stripe, o que significa que funciona através de aplicativos web, mas requer **terminais conectados à internet apenas**. Terminais Bluetooth não são suportados atualmente.
:::

### Terminais Suportados (Conectados à Internet) {#supported-terminals-internet-connected}

- **Leitor Stripe S700** ✅ - Terminal conectado via Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado via WiFi

### Terminais Não Suportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Apenas Bluetooth
- **BBPOS WisePad 3** ❌ - Apenas Bluetooth  
- **Verifone P400** ❌ - Apenas Bluetooth

:::note Suporte Futuro
O suporte a terminais Bluetooth será adicionado aos aplicativos iOS e Android no **1º trimestre de 2026**. Isso permitirá suporte para todos os leitores certificados do Terminal Stripe, incluindo o M2 e WisePad 3.
:::

### Problemas Comuns {#common-issues}

<Accordion>
  <AccordionItem question="Leitor não conecta">

  - Certifique-se de que você está usando um terminal conectado à internet suportado (S700 ou WisePOS E)
  - Verifique se o terminal está conectado ao WiFi/Ethernet e online
  - Confirme que sua conta Stripe tem o Terminal habilitado
  - Confirme que o terminal está registrado no seu Painel da Stripe

  </AccordionItem>

  <AccordionItem question="Pagamento recusado">

  - Verifique se sua conta Stripe está ativa e em bom estado
  - Verifique se o cartão utilizado é válido
  - Certifique-se de que há fundos suficientes disponíveis

  </AccordionItem>

  <AccordionItem question="Erros de certificado SSL">

  - O Terminal Stripe requer HTTPS - certifique-se de que seu site tem um certificado SSL válido
  - Verifique se seu certificado SSL está configurado corretamente

  </AccordionItem>
</Accordion>
