# Gateways de pagamento {#payment-gateways}

O WCPOS oferece suporte a vários gateways de pagamento personalizados que ampliam a funcionalidade do seu sistema de ponto de venda. Esses gateways foram projetados especificamente para ambientes de POS e oferecem integração fluida com diversos métodos e serviços de pagamento.

## Gateways personalizados disponíveis {#available-custom-gateways}

### Gateways de terminal de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceitar pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processar pagamentos por meio de leitores de cartão SumUp

### Gateways de pagamento móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceitar pagamentos por telefone via código QR ou notificação por push

### Gateways de pagamento digital {#digital-payment-gateways}

- **[Fatura por e-mail](./email-invoice)** - Envie faturas de pagamento aos clientes por e-mail
- **[Checkout na web](./web-checkout)** - Redirecione os clientes para concluir pagamentos online

## Criando seu próprio gateway {#creating-your-own-gateway}

Se você precisa de um gateway de pagamento personalizado para seus requisitos específicos, use nosso **[Modelo de gateway](/reference/gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão geral da instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório do GitHub
2. **Instale** o plugin em `WP Admin > Plugins > Adicionar novo > Enviar plugin`
3. **Configure** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** o gateway em `WP Admin > POS > Configurações > Finalização de compra`
