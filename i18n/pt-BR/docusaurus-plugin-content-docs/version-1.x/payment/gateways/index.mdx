# Métodos de pagamento {#payment-gateways}

O WCPOS oferece suporte a vários métodos de pagamento personalizados que ampliam a funcionalidade do sistema de ponto de venda. Esses métodos foram criados especificamente para ambientes de POS e oferecem integração direta com diferentes formas de pagamento e serviços.

## Métodos personalizados disponíveis {#available-custom-gateways}

### Métodos por terminal de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processe pagamentos por meio de leitores de cartão SumUp
- **[PayPal Reader (Zettle)](./paypal-reader)** - Receba pagamentos presenciais com cartão em um terminal PayPal Reader (Zettle)

### Métodos de pagamento móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos por telefone via código QR ou notificação push

### Gateways de pagamento digitais {#digital-payment-gateways}

- **[Fatura por e-mail](./email-invoice)** - Envie faturas de pagamento aos clientes por e-mail
- **[Finalização de compra pela Web](./web-checkout)** - Redirecione os clientes para concluir os pagamentos online

## Criar seu próprio gateway {#creating-your-own-gateway}

Se você precisa de um gateway de pagamento personalizado para seus requisitos específicos, pode usar nosso **[Modelo de gateway](/reference/gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão geral da instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório do GitHub
2. **Instale** o plugin via `WP Admin > Plugins > Adicionar novo > Enviar plugin`
3. **Configure** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** o gateway em `WP Admin > POS > Configurações > Finalização de compra`
