# Gateways de Pagamento {#payment-gateways}

O WCPOS oferece suporte a diversos gateways de pagamento personalizados que estendem a funcionalidade do seu sistema de Ponto de Venda. Esses gateways foram projetados especificamente para ambientes de PDV e proporcionam integração perfeita com diversos métodos e serviços de pagamento.

## Gateways Personalizados Disponíveis {#available-custom-gateways}

### Gateways de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceite pagamentos usando leitores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Processe pagamentos por meio de leitores de cartão SumUp
- **[Square Terminal](./square-terminal)** - Receba pagamentos presenciais em dispositivos Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Aceite pagamentos presenciais com cartão em um terminal PayPal Reader (Zettle)

### Gateways de Pagamento Móvel {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceite pagamentos por telefone via código QR ou notificação push

### Gateways de Pagamento Digital {#digital-payment-gateways}

- **[Fatura por E-mail](./email-invoice)** - Envie faturas de pagamento aos clientes por e-mail
- **[Checkout Web](./web-checkout)** - Redirecione os clientes para concluir pagamentos online

## Criando Seu Próprio Gateway {#creating-your-own-gateway}

Se você precisa de um gateway de pagamento personalizado para seus requisitos específicos, pode usar nosso **[Modelo de Gateway](/reference/gateway-template)** para criar seu próprio gateway de pagamento WCPOS.

## Visão Geral da Instalação {#installation-overview}

Todos os gateways personalizados seguem um processo de instalação semelhante:

1. **Baixe** a versão mais recente do respectivo repositório no GitHub
2. **Instale** o plugin em `WP Admin > Plugins > Adicionar novo > Enviar plugin`
3. **Configure** as configurações do gateway em `WP Admin > WooCommerce > Configurações > Pagamentos`
4. **Ative** o gateway em `WP Admin > POS > Configurações > Finalização de compra`
