:::info Recurso Pro
A tela de Produtos requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem visualizar e adicionar produtos ao carrinho a partir do [Painel de Produtos POS](/pos/product-panel), mas não podem editar estoques ou preços.
:::

A tela de Produtos é uma interface dedicada à gestão de inventário. Ao contrário do [Painel de Produtos POS](/pos/product-panel) (que é para vendas), esta tela é projetada para gerenciar seu catálogo de produtos.

{/* Screenshot: products-screen.png */}

## Tela de Produtos vs Painel de Produtos POS

| Recurso | Painel de Produtos POS | Tela de Produtos |
|---------|------------------------|-----------------|
| **Propósito** | Adicionar produtos ao carrinho | Gerenciar inventário |
| **Estoque** | Visualização apenas | Edição em linha |
| **Preços** | Visualização apenas | Edição em linha |
| **Ações** | Adicionar ao carrinho | Editar, Sincronizar, Excluir |
| **Disponível em** | Grátis + Pro | Apenas Pro |

## Visão Geral da Interface

### Busca & Filtros

Na parte superior da tela:

- **Barra de busca** - Encontre produtos por nome, SKU ou código de barras
- **Botões de filtro** - Status do Estoque, Destaque, Em Promoção, Categoria, Tag, Marca
- **Configurações de exibição** (<Icon name="sliders" />) - Configure as colunas visíveis

### Lista de Produtos

A área principal exibe seus produtos em uma tabela rolável:

- **Imagem** - Miniatura do produto
- **Nome do produto** - Com SKU, código de barras e atributos opcionais
- **Estoque** - Quantidade de estoque editável com o botão "Gerenciar"
- **Status do Estoque** - Em Estoque (verde), Fora de Estoque (vermelho), Pré-venda
- **Categorias** - Badges de categorias de produtos
- **Preços** - Preço atual, preço regular, preço em promoção (todos editáveis)
- **Ações** - Menu de três pontos com Editar, Sincronizar, Excluir

### Produtos Variáveis

Produtos variáveis mostram:
- Opções de atributos (Cor, Tamanho, etc.)
- Link "Expandir" para ver todas as variações
- Linhas de variações individuais com estoque/preços separados

### Rodapé

- **Status fiscal** - Base atual de cálculo de impostos
- **Contagem de produtos** - "Exibindo X de Y" com botão de sincronização (<Icon name="arrow-rotate-right" />). **Pressione longamente** o botão de sincronização para opção de Limpar e Atualizar

## Principais Recursos

### Edição de Estoque em Linha

Clique diretamente na quantidade de estoque para editá-la:

1. Clique no número do estoque
2. Digite a nova quantidade
3. Pressione Enter para salvar

As mudanças são sincronizadas com o WooCommerce automaticamente.

### Alternar Gerenciamento de Estoque

O botão "Gerenciar" controla se o WooCommerce rastreia o inventário deste produto:

- **Ativado** - Os níveis de estoque são rastreados e decrementados nas vendas
- **Desativado** - O produto está sempre disponível (estoque infinito)

### Edição de Preço em Linha

Clique em qualquer campo de preço para editar:

- **Preço** - Preço de venda atual
- **Preço Regular** - Preço sem desconto
- **Preço Promocional** - Preço com desconto (quando em promoção)

### Ações do Produto

Clique no menu de três pontos (⋮) para cada produto:

- **Editar** - Abrir o modal de edição do produto
- **Sincronizar** - Atualizar este produto do WooCommerce
- **Excluir** - Remover produto do banco de dados local

:::note
Excluir um produto do POS apenas o remove localmente. O produto permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

### Colunas Disponíveis

| Coluna | Descrição | Opções de Exibição |
|--------|-----------|--------------------|
| **Imagem** | Miniatura do produto | - |
| **ID** | ID do produto WooCommerce | - |
| **Produto** | Nome do produto | SKU, Código de Barras, Atributos |
| **Tipo** | Simples, variável, agrupado, etc. | - |
| **SKU** | Unidade de Manutenção de Estoque | - |
| **Código de Barras** | Código de barras do produto | - |
| **Estoque** | Quantidade atual (editável) | - |
| **Status do Estoque** | Em Estoque / Fora de Estoque / Pré-venda | - |
| **Categorias** | Categorias de produtos | - |
| **Tags** | Tags de produtos | - |
| **Marcas** | Informações sobre a marca (se o plugin estiver ativo) | - |
| **Custo das Mercadorias Vendidas** | Preço de custo do produto | - |
| **Preço** | Preço de venda atual (editável) | Informações fiscais |
| **Preço Regular** | Preço sem desconto (editável) | Informações fiscais |
| **Preço Promocional** | Preço com desconto (editável) | Informações fiscais |
| **Data de Criação** | Quando o produto foi criado | - |
| **Data de Modificação** | Data da última modificação | - |
| **Ações** | Botões Editar, Sincronizar, Excluir | - |

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição para seus padrões originais.

## Filtrando Produtos

Use os botões de filtro para limitar sua visualização:

- **Status do Estoque** - Em Estoque, Fora de Estoque, Pré-venda
- **Destaque** - Produtos marcados como destaque
- **Em Promoção** - Produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Tag** - Filtrar por tag de produto
- **Marca** - Filtrar por marca (se estiver usando o plugin de marca)

## Ordenação

Clique em qualquer cabeçalho de coluna para ordenar por essa coluna. Clique novamente para inverter a ordem da classificação.
