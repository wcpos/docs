:::info Recurso Pro
A tela de Produtos requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem visualizar e adicionar produtos ao carrinho a partir do [Painel de Produtos POS](/pos/product-panel), mas não podem editar o estoque ou preços.
:::

A tela de Produtos é uma interface dedicada ao gerenciamento de inventário. Ao contrário do [Painel de Produtos POS](/pos/product-panel) (que é para vendas), esta tela é projetada para gerenciar seu catálogo de produtos.

{/* Screenshot: products-screen.png */}

## Tela de Produtos vs Painel de Produtos POS

| Recurso | Painel de Produtos POS | Tela de Produtos |
|---------|-----------------------|-----------------|
| **Propósito** | Adicionar produtos ao carrinho | Gerenciar inventário |
| **Estoque** | Visualizar apenas | Editar inline |
| **Preços** | Visualizar apenas | Editar inline |
| **Ações** | Adicionar ao carrinho | Editar, Sincronizar, Excluir |
| **Disponível em** | Grátis + Pro | Apenas Pro |

## Visão Geral da Interface

### Pesquisa e Filtros

Na parte superior da tela:

- **Barra de pesquisa** - Encontre produtos por nome, SKU ou código de barras
- **Botões de filtro** - Status do Estoque, Em Destaque, Em Promoção, Categoria, Tag, Marca
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas visíveis

### Lista de Produtos

A área principal exibe seus produtos em uma tabela rolável:

- **Imagem** - Miniatura do produto
- **Nome do produto** - Com SKU, código de barras, atributos opcionais
- **Estoque** - Quantidade editável em estoque com alternância "Gerenciar"
- **Status do Estoque** - Em Estoque (verde), Fora de Estoque (vermelho), Pré-venda
- **Categorias** - Distintivos de categoria do produto
- **Preços** - Preço atual, preço regular, preço promocional (todos editáveis)
- **Ações** - Menu de três pontos com Editar, Sincronizar, Excluir

### Produtos Variáveis

Produtos variáveis mostram:
- Opções de atributos (Cor, Tamanho, etc.)
- Link "Expandir" para visualizar todas as variações
- Linhas de variações individuais com estoque/preços separados

### Rodapé

- **Status do imposto** - Base de cálculo do imposto atual
- **Contagem de produtos** - "Mostrando X de Y" com botão de sincronização

## Principais Recursos

### Edição de Estoque Inline

Clique diretamente na quantidade de estoque para editá-la:

1. Clique no número do estoque
2. Insira a nova quantidade
3. Pressione Enter para salvar

As alterações são sincronizadas com o WooCommerce automaticamente.

### Alternância de Gerenciamento de Estoque

A alternância "Gerenciar" controla se o WooCommerce rastreia o inventário para este produto:

- **Ativado** - Níveis de estoque são rastreados e diminuídos nas vendas
- **Desativado** - O produto está sempre disponível (estoque infinito)

### Edição de Preço Inline

Clique em qualquer campo de preço para editar:

- **Preço** - Preço atual de venda
- **Preço Regular** - Preço não descontado
- **Preço Promocional** - Preço com desconto (quando em promoção)

### Ações do Produto

Clique no menu de três pontos (⋮) para cada produto:

- **Editar** - Abrir o modal de edição do produto
- **Sincronizar** - Atualizar este produto do WooCommerce
- **Excluir** - Remover o produto do banco de dados local

:::note
Excluir um produto do POS remove-o apenas localmente. O produto permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

### Colunas Disponíveis

| Coluna | Descrição | Opções de Exibição |
|--------|-----------|--------------------|
| **Imagem** | Miniatura do produto | - |
| **ID** | ID do produto no WooCommerce | - |
| **Produto** | Nome do produto | SKU, Código de Barras, Atributos |
| **Tipo** | Simples, variável, agrupado, etc. | - |
| **SKU** | Unidade de Manutenção de Estoque | - |
| **Código de Barras** | Código de barras do produto | - |
| **Estoque** | Quantidade atual (editável) | - |
| **Status do Estoque** | Em Estoque / Fora de Estoque / Pré-venda | - |
| **Categorias** | Categorias do produto | - |
| **Tags** | Tags do produto | - |
| **Marcas** | Informações da marca (se o plugin estiver ativo) | - |
| **Custo das Mercadorias Vendidas** | Preço de custo do produto | - |
| **Preço** | Preço atual de venda (editável) | Informações fiscais |
| **Preço Regular** | Preço não descontado (editável) | Informações fiscais |
| **Preço Promocional** | Preço com desconto (editável) | Informações fiscais |
| **Data de Criação** | Quando o produto foi criado | - |
| **Data de Modificação** | Última data de modificação | - |
| **Ações** | Botões Editar, Sincronizar, Excluir | - |

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição para seus padrões originais.

## Filtrando Produtos

Use os botões de filtro para restringir sua visualização:

- **Status do Estoque** - Em Estoque, Fora de Estoque, Pré-venda
- **Em Destaque** - Produtos marcados como em destaque
- **Em Promoção** - Produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Tag** - Filtrar por tag de produto
- **Marca** - Filtrar por marca (se estiver usando o plugin de marcas)

## Classificação

Clique em qualquer cabeçalho de coluna para classificar por essa coluna. Clique novamente para inverter a ordem da classificação.
