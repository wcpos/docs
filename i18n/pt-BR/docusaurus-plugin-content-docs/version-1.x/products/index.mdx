:::info Recurso Pro
A tela de Produtos requer o [WCPOS Pro](/getting-started/pro-license). Usuários do plano gratuito podem visualizar e adicionar produtos ao carrinho no [Painel de Produtos do POS](/pos/product-panel), mas não podem editar estoque ou preços.
:::

A tela de Produtos é uma interface dedicada ao gerenciamento de inventário. Diferente do [Painel de Produtos do POS](/pos/product-panel) (que é para vender), esta tela é projetada para gerenciar seu catálogo de produtos.

{/* Screenshot: products-screen.png */}

## Tela de Produtos vs Painel de Produtos do POS

| Recurso | Painel de Produtos do POS | Tela de Produtos |
|---------|-------------------|-----------------|
| **Propósito** | Adicionar produtos ao carrinho | Gerenciar inventário |
| **Estoque** | Apenas visualizar | Editar inline |
| **Preços** | Apenas visualizar | Editar inline |
| **Ações** | Adicionar ao carrinho | Editar, Sincronizar, Excluir |
| **Disponível em** | Gratuito + Pro | Apenas Pro |

## Visão Geral da Interface

### Busca e Filtros

No topo da tela:

- **Barra de busca** - Encontrar produtos por nome, SKU ou código de barras
- **Botões de filtro** - Status de Estoque, Destacados, Em Promoção, Categoria, Tag, Marca
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas visíveis

### Lista de Produtos

A área principal exibe seus produtos em uma tabela rolável:

- **Imagem** - Miniatura do produto
- **Nome do produto** - Com SKU, código de barras, atributos opcionais
- **Estoque** - Quantidade de estoque editável com alternância de "Gerenciar"
- **Status de Estoque** - Em Estoque (verde), Fora de Estoque (vermelho), Em Backorder
- **Categorias** - Badges de categoria do produto
- **Preços** - Preço atual, preço regular, preço promocional (todos editáveis)
- **Ações** - Menu de três pontos com Editar, Sincronizar, Excluir

### Produtos Variáveis

Produtos variáveis mostram:
- Opções de atributos (Cor, Tamanho, etc.)
- Link "Expandir" para ver todas as variações
- Linhas de variação individuais com estoque/preços separados

### Rodapé

- **Status de imposto** - Base de cálculo de imposto atual
- **Contagem de produtos** - "Mostrando X de Y" com botão de sincronização

## Recursos Principais

### Edição de Estoque Inline

Clique diretamente na quantidade de estoque para editá-la:

1. Clique no número de estoque
2. Digite a nova quantidade
3. Pressione Enter para salvar

As alterações são sincronizadas automaticamente com o WooCommerce.

### Alternância de Gerenciamento de Estoque

A alternância de "Gerenciar" controla se o WooCommerce rastreia o inventário para este produto:

- **Ativado** - Níveis de estoque são rastreados e decrementados nas vendas
- **Desativado** - Produto sempre disponível (estoque infinito)

### Edição de Preço Inline

Clique em qualquer campo de preço para editar:

- **Preço** - Preço de venda atual
- **Preço Regular** - Preço sem desconto
- **Preço Promocional** - Preço com desconto (quando em promoção)

### Ações do Produto

Clique no menu de três pontos (⋮) para cada produto:

- **Editar** - Abrir modal de edição do produto
- **Sincronizar** - Atualizar este produto do WooCommerce
- **Excluir** - Remover produto do banco de dados local

:::note
Excluir um produto do POS apenas o remove localmente. O produto permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

### Colunas Disponíveis

| Coluna | Descrição | Opções de Exibição |
|--------|-------------|-----------------|
| **Imagem** | Miniatura do produto | - |
| **ID** | ID do produto WooCommerce | - |
| **Produto** | Nome do produto | SKU, Código de Barras, Atributos |
| **Tipo** | Simples, variável, agrupado, etc. | - |
| **SKU** | Unidade de Manutenção de Estoque | - |
| **Código de Barras** | Código de barras do produto | - |
| **Estoque** | Quantidade atual (editável) | - |
| **Status de Estoque** | Em Estoque / Fora de Estoque / Backorder | - |
| **Categorias** | Categorias do produto | - |
| **Tags** | Tags do produto | - |
| **Marcas** | Informações de marca (se plugin ativo) | - |
| **Custo dos Bens Vendidos** | Preço de custo do produto | - |
| **Preço** | Preço de venda atual (editável) | Informações de imposto |
| **Preço Regular** | Preço sem desconto (editável) | Informações de imposto |
| **Preço Promocional** | Preço com desconto (editável) | Informações de imposto |
| **Data de Criação** | Quando o produto foi criado | - |
| **Data de Modificação** | Data da última modificação | - |
| **Ações** | Botões de Editar, Sincronizar, Excluir | - |

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição para seus padrões originais.

## Filtrando Produtos

Use os botões de filtro para restringir sua visualização:

- **Status de Estoque** - Em Estoque, Fora de Estoque, Backorder
- **Destacados** - Produtos marcados como destacados
- **Em Promoção** - Produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Tag** - Filtrar por tag de produto
- **Marca** - Filtrar por marca (se usando plugin de marca)

## Ordenação

Clique no cabeçalho de qualquer coluna para ordenar por essa coluna. Clique novamente para inverter a ordem de classificação.
