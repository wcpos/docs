:::info Recurso Pro
A tela de Produtos requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem visualizar e adicionar produtos ao carrinho a partir do [Painel de Produtos POS](/pos/product-panel), mas não podem editar inventário ou preços.
:::

A tela de Produtos é uma interface dedicada à gestão de inventário. Ao contrário do [Painel de Produtos POS](/pos/product-panel) (que é para vendas), esta tela é projetada para gerenciar seu catálogo de produtos.

{/* Screenshot: products-screen.png */}

## Tela de Produtos vs Painel de Produtos POS {#products-screen-vs-pos-product-panel}

| Recurso | Painel de Produtos POS | Tela de Produtos |
|---------|------------------------|-----------------|
| **Propósito** | Adicionar produtos ao carrinho | Gerenciar inventário |
| **Estoque** | Somente visualização | Edição em linha |
| **Preços** | Somente visualização | Edição em linha |
| **Ações** | Adicionar ao carrinho | Editar, Sincronizar, Excluir |
| **Disponível em** | Gratuito + Pro | Somente Pro |

## Visão Geral da Interface {#interface-overview}

### Pesquisa e Filtros {#search-filters}

Na parte superior da tela:

- **Barra de pesquisa** - Encontre produtos pelo nome, SKU ou código de barras
- **Botões de filtro** - Status do Estoque, Em Destaque, Em Promoção, Categoria, Tag, Marca
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas visíveis

### Lista de Produtos {#product-list}

A área principal exibe seus produtos em uma tabela rolável:

- **Imagem** - Miniatura do produto
- **Nome do produto** - Com SKU, código de barras e atributos opcionais
- **Estoque** - Quantidade de estoque editável com alternância "Gerenciar"
- **Status do Estoque** - Em Estoque (verde), Esgotado (vermelho), Sob Encomenda
- **Categorias** - Insígnias de categoria do produto
- **Preços** - Preço atual, preço regular, preço promocional (todos editáveis)
- **Ações** - Menu de três pontos com Editar, Sincronizar, Excluir

### Produtos Variáveis {#variable-products}

Produtos variáveis mostram:
- Opções de atributo (Cor, Tamanho, etc.)
- Link "Expandir" para visualizar todas as variações
- Linhas de variações individuais com estoque/preços separados

### Rodapé {#footer}

- **Status de Imposto** - Base atual de cálculo de impostos
- **Contagem de produtos** - "Exibindo X de Y" com botão de sincronização (<Icon name="arrow-rotate-right" />). **Pressione longamente** o botão de sincronização para a opção Limpar e Atualizar

## Principais Recursos {#key-features}

### Edição de Estoque em Linha {#inline-stock-editing}

Clique diretamente na quantidade do estoque para editá-la:

1. Clique no número do estoque
2. Insira a nova quantidade
3. Pressione Enter para salvar

As alterações são sincronizadas automaticamente com o WooCommerce.

### Alternância de Gerenciamento de Estoque {#manage-stock-toggle}

A alternância "Gerenciar" controla se o WooCommerce rastreia o inventário para este produto:

- **Ativado** - Os níveis de estoque são rastreados e diminuídos em vendas
- **Desativado** - O produto está sempre disponível (estoque infinito)

### Edição de Preços em Linha {#inline-price-editing}

Clique em qualquer campo de preço para editar:

- **Preço** - Preço atual de venda
- **Preço Regular** - Preço não descontado
- **Preço Promocional** - Preço com desconto (quando em promoção)

### Ações do Produto {#product-actions}

Clique no menu de três pontos (⋮) para cada produto:

- **Editar** - Abra o modal de edição do produto
- **Sincronizar** - Atualize este produto a partir do WooCommerce
- **Excluir** - Remova o produto do banco de dados local

:::note
Excluir um produto do POS apenas o remove localmente. O produto permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

### Colunas Disponíveis {#available-columns}

| Coluna | Descrição | Opções de Exibição |
|--------|-----------|--------------------|
| **Imagem** | Miniatura do produto | - |
| **ID** | ID do produto do WooCommerce | - |
| **Produto** | Nome do produto | SKU, Código de Barras, Atributos |
| **Tipo** | Simples, variável, agrupado, etc. | - |
| **SKU** | Unidade de Manutenção de Estoque | - |
| **Código de Barras** | Código de barras do produto | - |
| **Estoque** | Quantidade atual (editável) | - |
| **Status do Estoque** | Em Estoque / Esgotado / Sob Encomenda | - |
| **Categorias** | Categorias do produto | - |
| **Tags** | Tags do produto | - |
| **Marcas** | Informações da marca (se o plugin estiver ativo) | - |
| **Custo das Mercadorias Vendidas** | Preço de custo do produto | - |
| **Preço** | Preço atual de venda (editável) | Informações fiscais |
| **Preço Regular** | Preço não descontado (editável) | Informações fiscais |
| **Preço Promocional** | Preço com desconto (editável) | Informações fiscais |
| **Data de Criação** | Quando o produto foi criado | - |
| **Data de Modificação** | Data da última modificação | - |
| **Ações** | Botões Editar, Sincronizar, Excluir | - |

### Restaurar Configurações Padrão {#restore-default-settings}

Clique para redefinir todas as configurações de exibição para os padrões originais.

## Filtrando Produtos {#filtering-products}

Use os botões de filtro para restringir sua visualização:

- **Status do Estoque** - Em Estoque, Esgotado, Sob Encomenda
- **Em Destaque** - Produtos marcados como em destaque
- **Em Promoção** - Produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Tag** - Filtrar por tag de produto
- **Marca** - Filtrar por marca (se estiver usando o plugin de marcas)

## Classificação {#sorting}

Clique em qualquer cabeçalho de coluna para classificar por essa coluna. Clique novamente para reverter a ordem da classificação.
