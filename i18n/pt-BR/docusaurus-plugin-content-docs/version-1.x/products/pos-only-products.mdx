Produtos Apenas para POS é um recurso que permite que você controle se os produtos aparecem em sua loja online, no seu POS ou em ambos. Isso é útil quando você tem catálogos de produtos diferentes para vendas online e em loja.

## Visão Geral {#overview}

Quando habilitado, cada produto recebe uma configuração de **Visibilidade do POS** com três opções:

| Visibilidade | Loja Online | POS |
|--------------|-------------|-----|
| **POS & Online** (padrão) | Visível | Visível |
| **Apenas POS** | Oculto | Visível |
| **Apenas Online** | Visível | Oculto |

## Quando Usar Este Recurso {#when-to-use-this-feature}

Habilite Produtos Apenas para POS se você precisar de:

- **Vender exclusivos na loja** - Produtos disponíveis apenas quando os clientes visitam sua loja física
- **Ocultar itens apenas online do POS** - Produtos digitais, pré-encomendas ou itens que não devem ser vendidos na loja
- **Gerenciar catálogos diferentes** - Linhas de produtos separadas para canais online e de varejo
- **Controlar itens promocionais** - Ofertas exclusivas para a loja ou promoções apenas online

:::tip
Se todos os seus produtos devem aparecer em ambos os canais, você não precisa deste recurso. Deixe desabilitado para melhor desempenho.
:::

## Habilitando o Recurso {#enabling-the-feature}

1. Vá para **WP Admin → WCPOS → Configurações → Geral**
2. Habilite **"Habilitar Produtos Apenas para POS"**
3. Salve as alterações

Para mais detalhes sobre Configurações Gerais, veja [Configurações Gerais do WP Admin](/settings/wp-admin/general).

:::warning Impacto no Desempenho
Habilite esta configuração apenas se necessário. Quando habilitada, ela adiciona uma pesquisa adicional no banco de dados para cada solicitação de produto, o que pode impactar o desempenho em lojas com grandes catálogos.
:::

## Definindo a Visibilidade do Produto {#setting-product-visibility}

### Produtos Individuais {#individual-products}

Para definir a visibilidade de um único produto:

1. Vá para **WP Admin → Produtos** e edite um produto
2. No painel **Dados do produto**, procure **Visibilidade do POS**
3. Selecione sua opção desejada:
   - **POS & Online** - Produto aparece em todos os lugares (padrão)
   - **Apenas POS** - Produto oculto da loja online
   - **Apenas Online** - Produto oculto do POS
4. Clique em **Atualizar** para salvar

### Edição em Massa {#bulk-editing}

Para alterar a visibilidade de vários produtos ao mesmo tempo:

1. Vá para **WP Admin → Produtos**
2. Selecione os produtos que deseja modificar usando as caixas de seleção
3. No dropdown **Ações em massa**, selecione **Editar**
4. Clique em **Aplicar**
5. No painel de edição em massa, encontre o dropdown **Visibilidade do POS**
6. Selecione sua opção de visibilidade desejada
7. Clique em **Atualizar**

### Produtos Variáveis {#variable-products}

Para produtos variáveis, você pode definir a visibilidade tanto no nível do produto pai quanto para cada variação individual. Isso dá a você um controle mais detalhado - por exemplo, você poderia mostrar todas as variações online, mas apenas certas cores ou tamanhos na loja.

## Como Isso Afeta o POS {#how-it-affects-the-pos}

Quando um produto é definido como **Apenas Online**:

- Ele não aparecerá no [Painel de Produtos](/pos/product-panel)
- Não será encontrado por meio de busca ou leitura de código de barras
- Não pode ser adicionado ao carrinho no POS

Quando um produto é definido como **Apenas POS**:

- Ele não aparecerá na frente de sua loja WooCommerce
- Não será incluído nos resultados de busca online
- Os clientes não podem comprá-lo online

:::note
Se você recentemente alterou a visibilidade de um produto e ele ainda está aparecendo (ou não aparecendo) no POS, tente uma sincronização. Pressione e segure o botão de sincronização e selecione **"Limpar e Atualizar"** para recarregar todos os dados do produto.
:::

## Casos de Uso Comuns {#common-use-cases}

<UseCases>
  <UseCaseCard icon="gift" title="Cartões Presente Apenas em Loja">

  Cartões presente físicos que precisam ser ativados no caixa:

  1. Crie o produto do cartão-presente
  2. Defina a visibilidade como **Apenas POS**
  3. O cartão não aparecerá online, mas a equipe pode vendê-lo na loja

  </UseCaseCard>

  <UseCaseCard icon="globe" title="Pré-Encomendas Online">

  Produtos disponíveis para pré-encomenda online, mas ainda não em estoque:

  1. Crie o produto de pré-encomenda
  2. Defina a visibilidade como **Apenas Online**
  3. Os clientes podem pré-encomendar online, mas isso não polui o POS

  </UseCaseCard>

  <UseCaseCard icon="building" title="Atacado vs Varejo">

  Se você usa a mesma loja WooCommerce para atacado e varejo:

  1. Defina produtos apenas para atacado como **Apenas Online** (para acesso ao portal B2B)
  2. Defina exclusividades do varejo como **Apenas POS**
  3. Mantenha os produtos regulares como **POS & Online**

  </UseCaseCard>
</UseCases>

## Solução de Problemas {#troubleshooting}

<Accordion>
  <AccordionItem question="Produtos ainda aparecendo após configuração como Apenas Online">

  1. Abra o POS
  2. Pressione e segure o botão de sincronização na parte inferior do Painel de Produtos
  3. Selecione **"Limpar e Atualizar"**
  4. Aguarde os produtos serem recarregados

  </AccordionItem>

  <AccordionItem question="Configuração não aparecendo nos produtos">

  Certifique-se de que o recurso esteja habilitado:

  1. Vá para **WP Admin → WCPOS → Configurações → Geral**
  2. Verifique se **"Habilitar Produtos Apenas para POS"** está marcado
  3. Salve as configurações se você fizer alterações

  </AccordionItem>
</Accordion>
