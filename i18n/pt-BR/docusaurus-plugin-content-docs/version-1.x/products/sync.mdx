Um dos aspectos significativos do sistema WCPOS (Ponto de Venda) é sua abordagem à gestão de dados. Ele fornece uma maneira eficiente de lidar com um grande volume de dados de produtos. Este guia explica como os produtos são baixados e sincronizados dentro do WCPOS.

## Armazenamento Local de Produtos

O WCPOS busca produtos da sua loja WooCommerce e os armazena localmente no seu dispositivo. Essa abordagem significa que, uma vez que um produto é baixado, não precisa ser buscado repetidamente. À medida que você continua utilizando o POS, ele fará o download progressivo de todos os produtos da sua loja WooCommerce, criando um banco de dados local dos seus produtos.

## Download em Lote de Produtos

Para a maioria das lojas, tentar baixar mais de 50 produtos de uma só vez pode levar a uma alta carga no servidor, tempos de resposta mais lentos ou até mesmo falhas no servidor. Para evitar isso, o WCPOS foi projetado para baixar produtos em pequenos lotes, geralmente cerca de 50 produtos por vez. Esse processo continua até que todo o inventário tenha sido armazenado localmente, garantindo que um grande volume de dados possa ser gerenciado sem causar estresse indevido no seu servidor.

## Utilizando a Busca de Produtos

Uma maneira eficaz de iniciar o download em lote de produtos é usando a função de busca de produtos. Cada vez que você realiza uma busca de produto, o sistema POS busca e armazena mais um lote de produtos. Essa estratégia não só ajuda você a baixar novos produtos gradualmente, mas também melhora a velocidade e a eficiência da função de busca à medida que mais produtos são armazenados localmente.

## Conclusão

É importante entender que todos os seus produtos não aparecerão de uma só vez no seu sistema POS, e isso é completamente normal. Com o tempo, utilizando a busca de produtos e outras operações do POS, você fará o download de todos os seus produtos. Para informações mais detalhadas sobre como o WCPOS opera, consulte [Arquitetura](/reference/architecture).

O WCPOS é projetado para otimizar a carga do servidor e a eficiência operacional. O recurso de download progressivo de produtos garante uma experiência de gestão de varejo suave, eficiente e amigável ao usuário.

## F.A.Q.

<details>

<summary>Como eu faço para carregar mais produtos?</summary>

A sincronização de produtos no POS pode ser acionada de várias maneiras:

- Quando o POS é carregado pela primeira vez.
- Cada vez que você executa uma busca de produtos.
- Aplicando um filtro, por exemplo, por categoria ou tag.
- Rolando até o fundo da lista de produtos.
- Pressionando o botão de sincronização manualmente.
- Além disso, o POS sincronizará automaticamente a cada 10 minutos.

Uma vez que todos os produtos sejam baixados, as futuras sincronizações apenas buscarão os produtos que foram atualizados desde a última sincronização.

</details>

<details>

<summary>
  Eu rolei até o fundo e ainda não está mostrando todos os produtos
</summary>

Pode haver algumas razões pelas quais nem todos os produtos estão sendo exibidos:

- O POS ocultará itens fora de estoque por padrão. Essa configuração pode ser alterada nas configurações de exibição de produto se você desejar mostrar itens fora de estoque.

- A API REST do WooCommerce é compatível apenas com os tipos de produtos padrão do WooCommerce, que são simples, variáveis, agrupados e externos. Se você estiver usando um tipo de produto personalizado, eles podem não ser exibidos no POS.

</details>
