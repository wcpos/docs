O WCPOS funciona com os dispositivos físicos que você já usa no balcão. Esta página é o ponto de partida para conectar hardware — impressoras de recibos, leitores de código de barras, leitores de cartão e gavetas de dinheiro. Cada tipo de dispositivo tem sua própria página de configuração aqui ou aponta para a página onde a configuração realmente fica.

:::note Uma página principal por dispositivo
A central de Hardware direciona para a configuração de cada dispositivo, mas as etapas reais ficam em um único lugar. O fluxo de conta e pareamento de um leitor de cartão fica em sua página Gateway de pagamento; a configuração de abertura de uma gaveta de dinheiro fica na página Impressoras de recibos.
:::
