O WCPOS funciona com os dispositivos físicos que você já utiliza no balcão. Esta página é o ponto de partida para conectar hardware — impressoras de recibos, leitores de código de barras, leitores de cartão e gavetas de dinheiro. Cada tipo de dispositivo possui sua própria página de configuração aqui ou aponta para a página onde a configuração é realizada.

:::note Um local por dispositivo
O hub de Hardware direciona você à configuração de cada dispositivo, mas as etapas reais ficam em um único lugar. A conta e o fluxo de pareamento de um leitor de cartão ficam na página do Gateway de Pagamento correspondente; a configuração de abertura de uma gaveta de dinheiro fica na página Impressoras de Recibos.
:::

## Compatibilidade de leitores de cartão {#card-reader-compatibility}

Quais leitores de cartão funcionam depende de **como você utiliza o WCPOS** e do tipo de conexão do leitor:

- **O aplicativo web** só pode operar leitores que oferecem um **SDK web (navegador)** — normalmente terminais de balcão conectados à internet. Leitores **somente Bluetooth** (ex.: Stripe M2 / WisePad 3, SumUp Air) e **Tap-to-Pay** por celular **não são compatíveis** com o aplicativo web.
- A compatibilidade é **por gateway**, não universal — verifique o leitor em relação ao gateway que você pretende usar ([Stripe Terminal](/payment/gateways/stripe-terminal), [SumUp](/payment/gateways/sumup-terminal), [PayPal Reader](/payment/gateways/paypal-reader)) antes de comprar.

Cada página de gateway lista os leitores exatos compatíveis.
