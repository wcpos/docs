:::info Recurso Pro
A tela de Clientes requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem selecionar clientes existentes para pedidos, mas não podem visualizar a lista completa de clientes ou editar os detalhes dos clientes.
:::

A tela de Clientes fornece gerenciamento abrangente de clientes diretamente dentro do POS. Visualize, edite e crie clientes sem mudar para o painel do WooCommerce.

{/* Screenshot: customers-screen.png */}

## Visão Geral da Interface

### Ações do Cabeçalho

Na parte superior da tela:

- **Barra de busca** - Encontre clientes por nome, e-mail, etc.
- **Adicionar Cliente** (<Icon name="plus" />) - Crie um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configure as colunas visíveis

### Tabela de Clientes

A área principal exibe os clientes com:

- **Avatar** - Imagem de perfil do cliente (ou espaço reservado)
- **Primeiro Nome** - Primeiro nome do cliente
- **Sobrenome** - Sobrenome do cliente (ordenável)
- **E-mail** - Endereço de e-mail para contato
- **Endereço de Cobrança** - Endereço de cobrança completo
- **Data de Criação** - Quando o cliente foi adicionado
- **Ações** - Menu de três pontos

### Rodapé

- Contagem de clientes com botão de sincronização

## Recursos Principais

### Busca de Clientes

Encontre clientes rapidamente por:

- Primeiro ou sobrenome
- Endereço de e-mail
- Informações de endereço

### Adicionar Novo Cliente

Crie clientes diretamente do POS:

1. Clique no ícone <Icon name="plus" /> no cabeçalho
2. Preencha os detalhes do cliente
3. Salve o novo cliente

O cliente é criado no WooCommerce e está imediatamente disponível para pedidos.

### Editar Cliente

Atualize as informações do cliente:

1. Clique no menu de três pontos de um cliente
2. Selecione **Editar**
3. Modifique os detalhes (nome, e-mail, endereços)
4. Salve as alterações

As alterações são sincronizadas com o WooCommerce automaticamente.

## Configurações de Exibição

Clique no ícone de deslizadores (<Icon name="sliders" />) para personalizar as colunas visíveis.

### Colunas Disponíveis

| Coluna | Descrição |
|--------|-------------|
| **Imagem** | Avatar do cliente |
| **ID** | ID do cliente no WooCommerce |
| **Primeiro Nome** | Primeiro nome do cliente |
| **Sobrenome** | Sobrenome do cliente |
| **E-mail** | Endereço de e-mail |
| **Função** | Papel do usuário (Cliente, Inscrito, etc.) |
| **Nome de Usuário** | Nome de usuário do WordPress |
| **Endereço de Cobrança** | Informações de cobrança |
| **Endereço de Envio** | Informações de envio |
| **Data de Criação** | Quando o cliente foi adicionado |
| **Data de Modificação** | Última atualização |
| **Ações** | Editar, Sincronizar, Deletar |

## Ações do Cliente

Clique no menu de três pontos (⋮) para opções:

- **Editar** - Modificar detalhes do cliente
- **Sincronizar** - Atualizar cliente do servidor
- **Deletar** - Remover do banco de dados local

:::note
Deletar um cliente do POS remove-o apenas localmente. O cliente permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

## Usando Clientes em Pedidos

Para atribuir um cliente a um pedido:

1. No [Painel do Carrinho](/pos/cart), clique no distintivo do cliente
2. Pesquise o cliente pelo nome ou e-mail
3. Selecione o cliente
4. O cliente agora está associado ao pedido

As informações do cliente (endereços de cobrança/envio) são usadas automaticamente para o pedido.

## Sincronização

Os dados dos clientes são sincronizados entre o POS e o WooCommerce:

- **Atualizações em tempo real** - As mudanças são sincronizadas em todos os dispositivos
- **Integração com WooCommerce** - Os clientes são armazenados no WooCommerce
- **Capacidade Offline** - Visualize clientes mesmo offline
