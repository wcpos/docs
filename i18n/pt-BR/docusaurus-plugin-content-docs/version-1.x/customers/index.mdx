:::info Recurso Pro
A tela de Clientes requer [WCPOS Pro](/getting-started/pro-license). Usuários da versão gratuita podem selecionar clientes existentes para pedidos, mas não podem visualizar a lista completa de clientes ou editar detalhes do cliente.
:::

A tela de Clientes oferece gerenciamento abrangente de clientes diretamente no POS. Visualize, edite e crie clientes sem precisar acessar o admin do WooCommerce.

{/* Screenshot: customers-screen.png */}

## Visão Geral da Interface

### Ações do Cabeçalho

No topo da tela:

- **Barra de pesquisa** - Encontre clientes por nome, email, etc.
- **Adicionar Cliente** (<Icon name="plus" />) - Criar um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas visíveis

### Tabela de Clientes

A área principal exibe clientes com:

- **Avatar** - Foto de perfil do cliente (ou espaço reservado)
- **Primeiro Nome** - Primeiro nome do cliente
- **Sobrenome** - Sobrenome do cliente (classificável)
- **Email** - Endereço de email de contato
- **Endereço de Cobrança** - Endereço completo de cobrança
- **Data de Criação** - Quando o cliente foi adicionado
- **Ações** - Menu de três pontos

### Rodapé

- Contagem de clientes com botão de sincronização

## Recursos Principais

### Pesquisa de Cliente

Encontre clientes rapidamente por:

- Primeiro ou último nome
- Endereço de email
- Informações de endereço

### Adicionar Novo Cliente

Crie clientes diretamente do POS:

1. Clique no ícone <Icon name="plus" /> no cabeçalho
2. Preencha os detalhes do cliente
3. Salve o novo cliente

O cliente é criado no WooCommerce e imediatamente disponível para pedidos.

### Editar Cliente

Atualize informações do cliente:

1. Clique no menu de três pontos em um cliente
2. Selecione **Editar**
3. Modifique detalhes (nome, email, endereços)
4. Salve as alterações

As alterações são sincronizadas automaticamente com o WooCommerce.

## Configurações de Exibição

Clique no ícone de controles deslizantes (<Icon name="sliders" />) para personalizar as colunas visíveis.

### Colunas Disponíveis

| Coluna | Descrição |
|--------|-------------|
| **Imagem** | Avatar do cliente |
| **ID** | ID do cliente no WooCommerce |
| **Primeiro Nome** | Primeiro nome do cliente |
| **Sobrenome** | Sobrenome do cliente |
| **Email** | Endereço de email |
| **Função** | Função de usuário (Cliente, Assinante, etc.) |
| **Nome de Usuário** | Nome de usuário do WordPress |
| **Endereço de Cobrança** | Informações de cobrança |
| **Endereço de Entrega** | Informações de entrega |
| **Data de Criação** | Quando o cliente foi adicionado |
| **Data de Modificação** | Última atualização |
| **Ações** | Editar, Sincronizar, Excluir |

## Ações do Cliente

Clique no menu de três pontos (⋮) para opções:

- **Editar** - Modificar detalhes do cliente
- **Sincronizar** - Atualizar cliente do servidor
- **Excluir** - Remover do banco de dados local

:::note
Excluir um cliente do POS apenas o remove localmente. O cliente permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

## Usando Clientes em Pedidos

Para atribuir um cliente a um pedido:

1. No [Painel do Carrinho](/pos/cart), clique no emblema do cliente
2. Pesquise o cliente por nome ou email
3. Selecione o cliente
4. O cliente agora está associado ao pedido

As informações do cliente (endereços de cobrança/entrega) são automaticamente usadas para o pedido.

## Sincronização

Os dados do cliente são sincronizados entre o POS e o WooCommerce:

- **Atualizações em tempo real** - Alterações são sincronizadas em todos os dispositivos
- **Integração com WooCommerce** - Clientes são armazenados no WooCommerce
- **Capacidade offline** - Visualize clientes mesmo sem conexão
