:::info Recurso Pro
A tela de Clientes requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem selecionar clientes existentes para pedidos, mas não conseguem visualizar a lista completa de clientes ou editar os detalhes dos clientes.
:::

A tela de Clientes fornece um gerenciamento abrangente de clientes diretamente no POS. Visualize, edite e crie clientes sem alternar para o administrador do WooCommerce.

{/* Screenshot: customers-screen.png */}

## Visão Geral da Interface

### Ações do Cabeçalho

Na parte superior da tela:

- **Barra de pesquisa** - Encontre clientes por nome, email, etc.
- **Adicionar Cliente** (<Icon name="plus" />) - Crie um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configure as colunas visíveis

### Tabela de Clientes

A área principal exibe clientes com:

- **Avatar** - Foto de perfil do cliente (ou espaço reservado)
- **Primeiro Nome** - Primeiro nome do cliente
- **Sobrenome** - Sobrenome do cliente (ordenável)
- **Email** - Endereço de email para contato
- **Endereço de Cobrança** - Endereço de cobrança completo
- **Data de Criação** - Quando o cliente foi adicionado
- **Ações** - Menu de três pontos

### Rodapé

- Contagem de clientes com botão de sincronização (<Icon name="arrow-rotate-right" />). **Pressione longamente** para a opção Limpar e Atualizar

## Principais Recursos

### Pesquisa de Clientes

Encontre clientes rapidamente por:

- Nome ou sobrenome
- Endereço de email
- Informações de endereço

### Adicionar Novo Cliente

Crie clientes diretamente do POS:

1. Clique no ícone <Icon name="plus" /> no cabeçalho
2. Preencha os detalhes do cliente
3. Salve o novo cliente

O cliente é criado no WooCommerce e está imediatamente disponível para pedidos.

### Editar Cliente

Atualize as informações do cliente:

1. Clique no menu de três pontos em um cliente
2. Selecione **Editar**
3. Modifique os detalhes (nome, email, endereços)
4. Salve as alterações

As alterações são sincronizadas automaticamente com o WooCommerce.

## Configurações de Exibição

Clique no ícone de deslizadores (<Icon name="sliders" />) para personalizar as colunas visíveis.

### Colunas Disponíveis

| Coluna | Descrição |
|--------|-------------|
| **Imagem** | Avatar do cliente |
| **ID** | ID do cliente no WooCommerce |
| **Primeiro Nome** | Primeiro nome do cliente |
| **Sobrenome** | Sobrenome do cliente |
| **Email** | Endereço de email |
| **Cargo** | Cargo do usuário (Cliente, Assinante, etc.) |
| **Nome de Usuário** | Nome de usuário do WordPress |
| **Endereço de Cobrança** | Informações de cobrança |
| **Endereço de Entrega** | Informações de entrega |
| **Data de Criação** | Quando o cliente foi adicionado |
| **Data de Modificação** | Última atualização |
| **Ações** | Editar, Sincronizar, Deletar |

## Ações do Cliente

Clique no menu de três pontos (⋮) para opções:

- **Editar** - Modificar os detalhes do cliente
- **Sincronizar** - Atualizar cliente do servidor
- **Deletar** - Remover do banco de dados local

:::note
Deletar um cliente no POS apenas o remove localmente. O cliente permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

## Usando Clientes em Pedidos

Para atribuir um cliente a um pedido:

1. No [Painel do Carrinho](/pos/cart), clique no crachá do cliente
2. Pesquise o cliente por nome ou email
3. Selecione o cliente
4. O cliente agora está associado ao pedido

As informações do cliente (endereços de cobrança e entrega) são usadas automaticamente para o pedido.

## Sincronização

Os dados do cliente são sincronizados entre o POS e o WooCommerce:

- **Atualizações em tempo real** - Alterações são sincronizadas em todos os dispositivos
- **Integração com WooCommerce** - Clientes são armazenados no WooCommerce
- **Capacidade offline** - Veja clientes mesmo quando estiver offline
