:::info Recurso Pro
A tela de Clientes requer [WCPOS Pro](/getting-started/pro-license). Usuários gratuitos podem selecionar clientes existentes para pedidos, mas não podem visualizar a lista completa de clientes ou editar os detalhes dos clientes.
:::

A tela de Clientes fornece um gerenciamento abrangente de clientes diretamente no POS. Visualize, edite e crie clientes sem alternar para a administração do WooCommerce.

{/* Screenshot: customers-screen.png */}

## Visão Geral da Interface {#interface-overview}

### Ações do Cabeçalho {#header-actions}

Na parte superior da tela:

- **Barra de Pesquisa** - Encontre clientes pelo nome, e-mail, etc.
- **Adicionar Cliente** (<Icon name="plus" />) - Crie um novo cliente
- **Configurações de Exibição** (<Icon name="sliders" />) - Configure as colunas visíveis

### Tabela de Clientes {#customer-table}

A área principal exibe clientes com:

- **Avatar** - Foto de perfil do cliente (ou espaço reservado)
- **Primeiro Nome** - Primeiro nome do cliente
- **Último Nome** - Último nome do cliente (ordenável)
- **E-mail** - Endereço de e-mail de contato
- **Endereço de Cobrança** - Endereço de cobrança completo
- **Data de Criação** - Quando o cliente foi adicionado
- **Ações** - Menu de três pontos

### Rodapé {#footer}

- Contagem de clientes com botão de sincronização (<Icon name="arrow-rotate-right" />). **Pressione longamente** para a opção Limpar e Atualizar

## Principais Recursos {#key-features}

### Pesquisa de Clientes {#customer-search}

Encontre clientes rapidamente por:

- Primeiro ou último nome
- Endereço de e-mail
- Informações de endereço

### Adicionar Novo Cliente {#add-new-customer}

Crie clientes diretamente do POS:

1. Clique no ícone <Icon name="plus" /> no cabeçalho
2. Preencha os detalhes do cliente
3. Salve o novo cliente

O cliente é criado no WooCommerce e está imediatamente disponível para pedidos.

### Editar Cliente {#edit-customer}

Atualize as informações do cliente:

1. Clique no menu de três pontos em um cliente
2. Selecione **Editar**
3. Modifique os detalhes (nome, e-mail, endereços)
4. Salve as alterações

As mudanças são sincronizadas automaticamente com o WooCommerce.

## Configurações de Exibição {#display-settings}

Clique no ícone de controles deslizantes (<Icon name="sliders" />) para personalizar as colunas visíveis.

### Colunas Disponíveis {#available-columns}

| Coluna | Descrição |
|--------|-------------|
| **Imagem** | Avatar do cliente |
| **ID** | ID do cliente no WooCommerce |
| **Primeiro Nome** | Primeiro nome do cliente |
| **Último Nome** | Último nome do cliente |
| **E-mail** | Endereço de e-mail |
| **Papel** | Papel do usuário (Cliente, Assinante, etc.) |
| **Nome de Usuário** | Nome de usuário do WordPress |
| **Endereço de Cobrança** | Informações de cobrança |
| **Endereço de Entrega** | Informações de entrega |
| **Data de Criação** | Quando o cliente foi adicionado |
| **Data de Modificação** | Última atualização |
| **Ações** | Editar, Sincronizar, Excluir |

## Ações do Cliente {#customer-actions}

Clique no menu de três pontos (⋮) para opções:

- **Editar** - Modificar detalhes do cliente
- **Sincronizar** - Atualizar cliente do servidor
- **Excluir** - Remover do banco de dados local

:::note
Excluir um cliente do POS remove apenas localmente. O cliente permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

## Usando Clientes em Pedidos {#using-customers-in-orders}

Para atribuir um cliente a um pedido:

1. No [Painel do Carrinho](/pos/cart), clique no crachá do cliente
2. Busque o cliente pelo nome ou e-mail
3. Selecione o cliente
4. O cliente agora está associado ao pedido

As informações do cliente (endereços de cobrança/entrega) são automaticamente usadas para o pedido.

## Sincronização {#synchronization}

Os dados dos clientes são sincronizados entre o POS e o WooCommerce:

- **Atualizações em tempo real** - As mudanças são sincronizadas em todos os dispositivos
- **Integração com o WooCommerce** - Os clientes são armazenados no WooCommerce
- **Capacidade Offline** - Visualize clientes mesmo offline
