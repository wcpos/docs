:::info Recurso Pro
A tela de Clientes requer o [WCPOS Pro](/getting-started/pro-license). Usuários da versão gratuita podem selecionar clientes existentes para pedidos, mas não podem visualizar a lista completa de clientes nem editar os dados dos clientes.
:::

A tela de Clientes oferece gerenciamento completo de clientes diretamente no POS. Visualize, edite e crie clientes sem precisar acessar o painel administrativo do WooCommerce.

{/* Screenshot: customers-screen.png */}

## Visão Geral da Interface {#interface-overview}

### Ações do Cabeçalho {#header-actions}

Na parte superior da tela:

- **Barra de pesquisa** - Encontre clientes por nome, e-mail, etc.
- **Adicionar Cliente** (<Icon name="plus" />) - Criar um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas visíveis

### Tabela de Clientes {#customer-table}

A área principal exibe os clientes com:

- **Avatar** - Foto de perfil do cliente (ou imagem padrão)
- **Nome** - Primeiro nome do cliente
- **Sobrenome** - Sobrenome do cliente (ordenável)
- **E-mail** - Endereço de e-mail de contato
- **Endereço de Cobrança** - Endereço de cobrança completo
- **Data de Criação** - Quando o cliente foi adicionado
- **Ações** - Menu de três pontos

### Rodapé {#footer}

- Contagem de clientes com botão de sincronização (<Icon name="arrow-rotate-right" />). **Pressão longa** para opção de Limpar e Atualizar

## Recursos Principais {#key-features}

### Busca de Clientes {#customer-search}

Encontre clientes rapidamente por:

- Nome ou sobrenome
- Endereço de e-mail
- Informações de endereço

### Adicionar Novo Cliente {#add-new-customer}

Crie clientes diretamente do POS:

1. Clique no <Icon name="plus" /> ícone no cabeçalho
2. Preencha os dados do cliente
3. Salve o novo cliente

O cliente é criado no WooCommerce e fica imediatamente disponível para pedidos.

### Editar Cliente {#edit-customer}

Atualize as informações do cliente:

1. Clique no menu de três pontos em um cliente
2. Selecione **Editar**
3. Modifique os dados (nome, e-mail, endereços)
4. Salve as alterações

As alterações são sincronizadas com o WooCommerce automaticamente.

## Configurações de Exibição {#display-settings}

Clique no ícone de controles deslizantes (<Icon name="sliders" />) para personalizar as colunas visíveis.

## Ações do cliente {#customer-actions}

Clique no menu de três pontos (⋮) para ver as opções:

- **Editar** - Modificar os dados do cliente
- **Sincronizar** - Atualizar os dados do cliente a partir do servidor
- **Excluir** - Remover do banco de dados local

:::note
Excluir um cliente do POS apenas o remove localmente. O cliente permanece no WooCommerce e reaparecerá na próxima sincronização.
:::

## Cliente Padrão (Visitante) {#default-customer-guest}

Quando nenhum cliente é selecionado, os pedidos são feitos como pedidos de **Visitante**. Pedidos de visitante:

- Não possuem nome, e-mail ou endereço de cliente vinculados
- Não podem ser consultados por cliente no histórico de pedidos
- Ainda são visíveis na tela de Pedidos e no painel administrativo do WooCommerce

Para evitar pedidos de visitante, selecione um cliente no Painel do Carrinho antes de finalizar a compra. Se o seu negócio exige um cliente em cada pedido, considere treinar os operadores de caixa para sempre atribuir um — não há uma configuração integrada para impor isso.

## Usando Clientes nos Pedidos {#using-customers-in-orders}

Para atribuir um cliente a um pedido:

1. No [Painel do Carrinho](/pos/cart), clique no emblema do cliente
2. Pesquise o cliente pelo nome, e-mail ou número de telefone
3. Selecione o cliente
4. O cliente agora está associado ao pedido

As informações do cliente (endereços de cobrança/entrega) são utilizadas automaticamente no pedido.

## Sincronização {#synchronization}

Os dados dos clientes são sincronizados entre o POS e o WooCommerce:

- **Atualizações em tempo real** - As alterações são sincronizadas em todos os dispositivos
- **Integração com WooCommerce** - Os clientes são armazenados no WooCommerce
- **Funcionalidade offline** - Visualize os clientes mesmo sem conexão
