Quando você estiver pronto para completar uma venda, clique no botão **Checkout** para abrir o modal de checkout. É aqui que você processa o pagamento e completa o pedido.

{/* Screenshot: checkout-modal.png */}

## Visão Geral do Modal de Checkout

O modal de checkout exibe:

- **Número do pedido** - O ID do pedido do WooCommerce
- **Valor a pagar** - Total devido
- **Botão de Configurações do Checkout** - Resolver problemas de exibição
- **Caixa** - Quem está processando o pedido
- **Cliente** - O cliente para este pedido (link clicável)
- **Campo de código do cupom** - Aplique códigos de desconto
- **Resumo do pedido** - Produtos, quantidades e totais
- **Métodos de pagamento** - Opções de pagamento disponíveis
- **Botões Cancelar / Processar Pagamento**

## Métodos de Pagamento

### Disponível na Versão Gratuita

A versão gratuita do WCPOS inclui dois gateways de pagamento:

- **Dinheiro** - Com valor recebido e calculadora de troco
- **Cartão** - Para terminais de cartão externos

### Gateways Adicionais (Pro)

:::info Recurso Pro
Gateways de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode habilitar:

- **Stripe Terminal** - Integração direta com leitores de cartão Stripe
- **SumUp Terminal** - Integração com leitores de cartão SumUp
- **Gateways Personalizados** - Crie suas próprias integrações de pagamento

Veja [Pagamento](/payment) para detalhes sobre como configurar gateways.

### Selecionando um Método de Pagamento

Clique em um método de pagamento para selecioná-lo. O formulário é atualizado para mostrar os campos relevantes:

**Dinheiro:**
- **Valor Recebido** - Insira o valor que o cliente lhe dá
- **Troco** - Troco calculado automaticamente para devolver

{/* Screenshot: checkout-cash-payment.png */}

**Cartão:**
- Processar pagamento em seu terminal de cartão externo
- Clique em Processar Pagamento para completar

## Códigos de Cupom

Para aplicar um desconto:

1. Insira o código do cupom no campo
2. Clique em **Aplicar cupom**
3. O desconto aparece no resumo do pedido

Os cupons devem ser configurados no WooCommerce (`WP Admin > Marketing > Cupons`).

## Processando Pagamento

1. Selecione um método de pagamento
2. Para dinheiro, insira o valor recebido
3. Clique em **Processar Pagamento**
4. O pedido é completado e o [recibo](/pos/checkout/receipts) é exibido

## Configurações do Checkout (Solução de Problemas)

O modal de checkout usa um iframe/webview para exibir a página de pagamento do pedido do WooCommerce. Isso aproveita a infraestrutura de pagamento existente do WooCommerce, o que significa que qualquer gateway de pagamento que funcione com o WooCommerce deve funcionar no POS.

No entanto, scripts de tema e plugin podem às vezes interferir. Clique em **Configurações do Checkout** para solucionar problemas:

{/* Screenshot: checkout-settings.png */}

### Desabilitar Todos os Estilos e Scripts

- **Desabilitar wp_head** - Remove todos os scripts/estilos do cabeçalho do WordPress
- **Desabilitar wp_footer** - Remove todos os scripts/estilos do rodapé do WordPress

### Desabilitar Estilos Selecionados

Desabilite seletivamente o CSS que pode causar problemas de exibição:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- etc.

### Desabilitar Scripts Selecionados

Desabilite seletivamente o JavaScript que pode interferir com gateways de pagamento:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- etc.

:::tip
Se um gateway de pagamento não for exibido corretamente:
1. Tente desabilitar os estilos do tema primeiro
2. Depois tente desabilitar os scripts do WooCommerce que não são necessários
3. Tenha cuidado para não desabilitar scripts exigidos pelo seu gateway de pagamento
:::

## Cancelar

Clique em **Cancelar** para fechar o modal de checkout sem completar o pedido. O pedido permanece como um carrinho aberto.
