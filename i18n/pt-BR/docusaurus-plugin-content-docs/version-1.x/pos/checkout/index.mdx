Quando estiver pronto para concluir uma venda, clique no botão **Checkout** para abrir o modal de checkout. É aqui que você processa o pagamento e conclui o pedido.

{/* Screenshot: checkout-modal.png */}

## Visão Geral do Modal de Checkout

O modal de checkout exibe:

- **Número do pedido** - O ID do pedido WooCommerce
- **Valor a Pagar** - Total devido
- **Botão de Configurações do Checkout** - Solucionar problemas de exibição
- **Caixa** - Quem está processando o pedido
- **Cliente** - O cliente deste pedido (link clicável)
- **Campo de código de cupom** - Aplique códigos de desconto
- **Resumo do pedido** - Produtos, quantidades e totais
- **Métodos de pagamento** - Opções de pagamento disponíveis
- **Botões Cancelar / Processar Pagamento**

## Métodos de Pagamento

### Disponível na Versão Gratuita

A versão gratuita do WCPOS inclui dois gateways de pagamento:

- **Dinheiro** - Com valor recebido e calculadora de troco
- **Cartão** - Para terminais de cartão externos

### Gateways Adicionais (Pro)

:::info Recurso Pro
Gateways de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode habilitar:

- **Terminal Stripe** - Integração direta com leitores de cartão Stripe
- **Terminal SumUp** - Integração com leitores de cartão SumUp
- **Gateways Personalizados** - Crie suas próprias integrações de pagamento

Veja [Pagamento](/payment) para detalhes sobre a configuração de gateways.

### Selecionando um Método de Pagamento

Clique em um método de pagamento para selecioná-lo. O formulário é atualizado para mostrar os campos relevantes:

**Dinheiro:**
- **Valor Recebido** - Insira o valor que o cliente lhe entrega
- **Troco** - Troco calculado automaticamente para devolver

{/* Screenshot: checkout-cash-payment.png */}

**Cartão:**
- Procesar o pagamento no seu terminal de cartão externo
- Clique em Processar Pagamento para concluir

## Códigos de Cupom

Para aplicar um desconto:

1. Insira o código do cupom no campo
2. Clique em **Aplicar cupom**
3. O desconto aparece no resumo do pedido

Os cupons devem ser configurados no WooCommerce (`WP Admin > Marketing > Coupons`).

## Processando Pagamento

1. Selecione um método de pagamento
2. Para dinheiro, insira o valor recebido
3. Clique em **Processar Pagamento**
4. O pedido é concluído e o [recibo](/pos/checkout/receipts) é mostrado

## Configurações do Checkout (Resolução de Problemas)

O modal de checkout utiliza um iframe/webview para exibir a página de pagamento do pedido WooCommerce. Isso aproveita a infraestrutura de pagamento existente do WooCommerce, o que significa que qualquer gateway de pagamento que funcione com o WooCommerce deve funcionar no POS.

No entanto, scripts de tema e plugins podem às vezes interferir. Clique em **Configurações do Checkout** para solucionar problemas:

### Desativar Todos os Estilos e Scripts

- **Desativar wp_head** - Remove todos os scripts/estilos do cabeçalho do WordPress
- **Desativar wp_footer** - Remove todos os scripts/estilos do rodapé do WordPress

### Desativar Estilos Selecionados

Desative seletivamente o CSS que pode causar problemas de exibição:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- etc.

### Desativar Scripts Selecionados

Desative seletivamente o JavaScript que pode interferir com os gateways de pagamento:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- etc.

:::tip
Se um gateway de pagamento não for exibido corretamente:
1. Tente desativar os estilos do tema primeiro
2. Depois tente desativar scripts do WooCommerce que não são necessários
3. Cuidado para não desativar scripts exigidos pelo seu gateway de pagamento
:::

## Cancelar

Clique em **Cancelar** para fechar o modal de checkout sem concluir o pedido. O pedido permanece como um carrinho aberto.
