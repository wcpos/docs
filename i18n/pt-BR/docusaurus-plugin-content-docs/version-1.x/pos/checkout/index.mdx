Quando estiver pronto para concluir uma venda, clique no botão **Finalizar Compra** para abrir o modal de checkout. É aqui onde você processa o pagamento e conclui o pedido.

{/* Screenshot: checkout-modal.png */}

## Visão Geral do Modal de Checkout

O modal de checkout exibe:

- **Número do Pedido** - O ID do pedido do WooCommerce
- **Valor a Pagar** - Valor total devido
- **Botão de Configurações de Checkout** - Solucionar problemas de exibição
- **Caixa** - Quem está processando o pedido
- **Cliente** - O cliente deste pedido (link clicável)
- **Campo de código de cupom** - Aplicar códigos de desconto
- **Resumo do pedido** - Produtos, quantidades e totais
- **Métodos de pagamento** - Opções de pagamento disponíveis
- **Botões Cancelar / Processar Pagamento**

## Métodos de Pagamento

### Disponíveis na Versão Gratuita

A versão gratuita do WCPOS inclui duas formas de pagamento:

- **Dinheiro** - Com valor recebido e calculadora de troco
- **Cartão** - Para terminais de cartão externos

### Formas de Pagamento Adicionais (Pro)

:::info Recurso Pro
Formas de pagamento adicionais requerem [WCPOS Pro](/getting-started/pro-license).
:::

Com o Pro, você pode habilitar:

- **Terminal Stripe** - Integração direta com leitores de cartão Stripe
- **Terminal SumUp** - Integração com leitores de cartão SumUp
- **Formas de Pagamento Personalizadas** - Criar integrações de pagamento próprias

Consulte [Pagamento](/payment) para detalhes sobre configuração de formas de pagamento.

### Selecionando um Método de Pagamento

Clique em um método de pagamento para selecioná-lo. O formulário é atualizado para mostrar os campos relevantes:

**Dinheiro:**
- **Valor Recebido** - Digite o valor entregue pelo cliente
- **Troco** - Troco calculado automaticamente

{/* Screenshot: checkout-cash-payment.png */}

**Cartão:**
- Processar pagamento no terminal de cartão externo
- Clique em Processar Pagamento para concluir

## Códigos de Cupom

Para aplicar um desconto:

1. Digite o código do cupom no campo
2. Clique em **Aplicar cupom**
3. O desconto aparece no resumo do pedido

Os cupons devem ser configurados no WooCommerce (`WP Admin > Marketing > Cupons`).

## Processando Pagamento

1. Selecione um método de pagamento
2. Para dinheiro, digite o valor recebido
3. Clique em **Processar Pagamento**
4. O pedido é concluído e o [recibo](/pos/checkout/receipts) é exibido

## Configurações de Checkout (Solução de Problemas)

O modal de checkout usa um iframe/webview para exibir a página de Pagamento do Pedido do WooCommerce. Isso utiliza a infraestrutura de pagamento existente do WooCommerce, o que significa que qualquer gateway de pagamento que funcione com WooCommerce deve funcionar no POS.

No entanto, scripts de temas e plugins podem às vezes interferir. Clique em **Configurações de Checkout** para solucionar problemas:

{/* Screenshot: checkout-settings.png */}

### Desabilitar Todos os Estilos e Scripts

- **Desabilitar wp_head** - Remove todos os scripts/estilos do cabeçalho do WordPress
- **Desabilitar wp_footer** - Remove todos os scripts/estilos do rodapé do WordPress

### Desabilitar Estilos Selecionados

Desabilite seletivamente CSS que pode causar problemas de exibição:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- etc.

### Desabilitar Scripts Selecionados

Desabilite seletivamente JavaScript que pode interferir com gateways de pagamento:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- etc.

:::tip
Se um gateway de pagamento não for exibido corretamente:
1. Tente desabilitar os estilos do tema primeiro
2. Depois tente desabilitar scripts do WooCommerce que não são necessários
3. Tenha cuidado para não desabilitar scripts necessários para o seu gateway de pagamento
:::

## Cancelar

Clique em **Cancelar** para fechar o modal de checkout sem concluir o pedido. O pedido permanece como um carrinho em aberto.
