- Abre um diálogo para adicionar uma nota do cliente ao pedido.
- Notas do cliente são visíveis para o cliente e aparecem no recibo por padrão, na área Minha Conta do cliente e em `WP Admin > WooCommerce > Pedidos > Pedido` sob o endereço de entrega.

:::note
Notas de pedido são visíveis para os clientes.
:::

- Abre um modal para gerenciar metadados adicionais para o pedido.
- Os recursos incluem:
  - Alterar a moeda do pedido.
  - Adicionar um ID de transação (por exemplo, de um terminal de pagamento externo).
  - Adicionar dados meta personalizados para o pedido.

:::note
Recursos adicionais serão adicionados a este modal ao longo do tempo, incluindo integração com outros plugins do WooCommerce.
:::

### O que é Meta Dados no WooCommerce?

Os metadados fornecem informações extras sobre os pedidos e são acessíveis via REST API do WooCommerce. Desenvolvedores e plugins usam esses dados para estender a funcionalidade, como integrar serviços de terceiros.

### Visualização JSON

- Visualize a representação JSON bruta do pedido, útil para fins de depuração ou integração.

## Salvar no Servidor

- Permite salvar carrinhos abertos no servidor, para que sejam armazenados com segurança e acessíveis para uso futuro.
- Cria um pedido do WooCommerce com status `pos-open`.

### Por que salvar?

Quando um novo pedido é criado no POS, ele só existe localmente. 
Salvar o pedido garante que ele persista, mesmo que o banco de dados local seja limpo.

Veja [Pedidos Abertos](/pos/cart/open-orders) para mais detalhes sobre como gerenciar pedidos salvos.

## Anular

- Dois Cenários:
  - Pedidos Não Salvos:
    - Se o pedido não tiver sido salvo no servidor, ele será excluído permanentemente do banco de dados local.
  - Pedidos Salvos:
    - Se o pedido estiver salvo no servidor, ele será movido para a pasta Lixeira no WooCommerce e excluído do banco de dados local.
- Para recuperar um pedido anulado:
  - Vá para `WP Admin > WooCommerce > Pedidos > Lixeira`.

## Finalizar Compra

- Salva o pedido no servidor e inicia o processo de finalização da compra.
- Abre a [tela de finalização da compra](/pos/checkout) para completar o pagamento.
