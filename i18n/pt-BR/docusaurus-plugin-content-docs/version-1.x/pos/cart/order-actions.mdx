- Abre um diálogo para adicionar uma nota de cliente ao pedido.
- Notas de cliente são visíveis para o cliente e aparecem no recibo por padrão, na área Minha Conta do cliente e em `WP Admin > WooCommerce > Pedidos > Pedido` abaixo do endereço de entrega.

:::note
Notas de pedido são visíveis para clientes.
:::

- Abre um modal para gerenciar metadados adicionais do pedido.
- Recursos incluem:
  - Alterar a moeda do pedido.
  - Adicionar um ID de transação (por exemplo, de um terminal de pagamento externo).
  - Adicionar metadados personalizados para o pedido.

:::note
Recursos adicionais serão adicionados a este modal ao longo do tempo, incluindo integração com outros plugins do WooCommerce.
:::

### O que são Metadados no WooCommerce?

Metadados fornecem informações adicionais sobre pedidos e são acessíveis pela API REST do WooCommerce. Desenvolvedores e plugins usam esses dados para estender a funcionalidade, como integrar serviços de terceiros.

### Visualização JSON

- Visualize a representação JSON bruta do pedido, útil para depuração ou fins de integração.

## Salvar no Servidor

- Permite salvar carrinhos abertos no servidor, para que sejam armazenados com segurança e acessíveis para uso futuro.
- Cria um pedido WooCommerce com status `pos-open`.

### Por que salvar?

Quando um novo pedido é criado no POS, ele existe apenas localmente. 
Salvar o pedido garante que ele persista, mesmo se o banco de dados local for limpo.

Consulte [Pedidos Abertos](/pos/cart/open-orders) para mais detalhes sobre gerenciamento de pedidos salvos.

## Anular

- Dois Cenários:
  - Pedidos Não Salvos:
    - Se o pedido não foi salvo no servidor, será excluído permanentemente do banco de dados local.
  - Pedidos Salvos:
    - Se o pedido está salvo no servidor, será movido para a pasta de Lixeira no WooCommerce e excluído do banco de dados local.
- Para recuperar um pedido anulado:
  - Vá para `WP Admin > WooCommerce > Pedidos > Lixeira`.

## Finalizar Compra

- Salva o pedido no servidor e inicia o processo de checkout.
- Abre a [tela de checkout](/pos/checkout) para concluir o pagamento.
