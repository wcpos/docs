Cada item no carrinho é um item de linha que pode ser editado diretamente. O WCPOS oferece edição flexível em linha para quantidades, preços e outros detalhes.

## Tipos de Itens de Linha

O WooCommerce utiliza três tipos de itens de linha:

| Tipo | Descrição | Exemplos |
|------|-------------|----------|
| **line_item** | Produtos do seu catálogo | Camiseta, Caneca de café |
| **fee_line** | Taxas adicionais | Embalagem de presente, Taxa de serviço |
| **shipping_line** | Custos de envio | Envio padrão, Entrega expressa |

:::note
Os custos de envio são inseridos manualmente. O POS não calcula taxas de envio automaticamente.
:::

## Editando Itens de Linha

### Quantidade

Clique no campo de quantidade para editá-lo diretamente. Digite um novo número ou use os controles de incremento.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nome do Produto

Para produtos e taxas diversos, você pode editar o nome diretamente clicando nele.

### Preço

Clique no campo de preço para alterar o preço unitário. Isso é útil para:

- Aplicar descontos manuais
- Igualar preços
- Corrigir erros de preços

O total é atualizado automaticamente com base em quantidade × preço.

### Menu de Três Pontos

Clique no **⋮** (três pontos) em qualquer item de linha para acessar opções adicionais:

{/* Screenshot: cart-line-item-menu.png */}

#### Formulário de Edição

Abre um formulário de edição detalhado para o item de linha, permitindo que você modifique todos os campos.

#### Visualização JSON Bruto

Exibe os dados JSON brutos para o item de linha. Isso é extremamente útil para:

- **Depuração** - Veja exatamente quais dados estão sendo enviados para o WooCommerce
- **Solução de Problemas** - Identifique problemas com campos personalizados ou metadados
- **Desenvolvimento** - Entenda a estrutura de dados para integrações

### Remover Item

Clique no botão vermelho **×** para remover um item do carrinho.

## Atributos de Variação

Para produtos variáveis, os atributos de variação selecionados são exibidos abaixo do nome do produto:

```
Hoodie
Colour: Green
Size: Large
```

Esses atributos fazem parte dos dados do item de linha e estão incluídos no pedido.

## Metadados

Se você configurou [Chaves de Metadados](/pos/product-panel#meta-data-keys) nas configurações do Painel de Produtos, esses dados são copiados automaticamente para o item de linha quando o produto é adicionado ao carrinho.

Você pode visualizar esses metadados na visualização JSON bruta.

## Dividindo Itens de Linha

Se habilitado nas [Configurações de Exibição](/pos/cart#display-settings), você pode dividir um item de linha em várias linhas. Isso é útil quando um cliente quer aplicar descontos diferentes a porções do mesmo produto.

## Dicas

### Ajustes Rápidos de Preço

Para conceder um desconto rapidamente:

1. Clique no campo de preço
2. Insira o novo preço
3. Pressione Enter

### Navegação pelo Teclado

- **Tab** - Mover entre campos editáveis
- **Enter** - Confirmar edição
- **Escape** - Cancelar edição

### Edições em Lote

Para pedidos complexos, considere usar o menu de três pontos para acessar o formulário de edição completo, que mostra todos os campos de uma vez.
