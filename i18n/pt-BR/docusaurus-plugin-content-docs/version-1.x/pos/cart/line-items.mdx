Cada item no carrinho é um item de linha que pode ser editado diretamente. O WCPOS oferece edição flexível de quantidade, preços e outros detalhes.

## Tipos de Itens de Linha

O WooCommerce usa três tipos de itens de linha:

| Tipo | Descrição | Exemplos |
|------|-------------|----------|
| **line_item** | Produtos do seu catálogo | Camiseta, Caneca de café |
| **fee_line** | Taxas adicionais | Embrulho para presente, Taxa de serviço |
| **shipping_line** | Custos de envio | Envio padrão, Entrega expressa |

:::note
Os custos de envio são inseridos manualmente. O POS não calcula automaticamente as taxas de envio.
:::

## Editando Itens de Linha

### Quantidade

Clique no campo de quantidade para editá-lo diretamente. Digite um novo número ou use os controles de incremento.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nome do Produto

Para produtos diversos e taxas, você pode editar o nome diretamente clicando nele.

### Preço

Clique no campo de preço para alterar o preço unitário. Isso é útil para:

- Aplicar descontos manuais
- Igualar preços
- Corrigir erros de precificação

O total é atualizado automaticamente com base em quantidade × preço.

### Menu de Três Pontos

Clique no **⋮** (três pontos) em qualquer item de linha para acessar opções adicionais:

{/* Screenshot: cart-line-item-menu.png */}

#### Formulário de Edição

Abre um formulário de edição detalhado para o item de linha, permitindo modificar todos os campos.

#### Visualização JSON Bruta

Exibe os dados JSON brutos do item de linha. Isso é extremamente útil para:

- **Depuração** - Ver exatamente quais dados estão sendo enviados para o WooCommerce
- **Solução de problemas** - Identificar problemas com campos personalizados ou meta dados
- **Desenvolvimento** - Compreender a estrutura de dados para integrações

### Remover Item

Clique no botão vermelho **×** para remover um item do carrinho.

## Atributos de Variação

Para produtos variáveis, os atributos de variação selecionados são exibidos abaixo do nome do produto:

```
Moletom
Cor: Verde
Tamanho: Grande
```

Esses atributos fazem parte dos dados do item de linha e são incluídos no pedido.

## Meta Dados

Se você configurou as [Chaves de Meta Dados](/pos/product-panel#meta-data-keys) nas configurações do Painel de Produto, esses dados são automaticamente copiados para o item de linha quando o produto é adicionado ao carrinho.

Você pode visualizar esses meta dados na Visualização JSON Bruta.

## Dividindo Itens de Linha

Se habilitado nas [Configurações de Exibição](/pos/cart#display-settings), você pode dividir um item de linha em múltiplas linhas. Isso é útil quando um cliente deseja aplicar descontos diferentes a partes do mesmo produto.

## Dicas

### Ajustes Rápidos de Preço

Para dar um desconto rapidamente:

1. Clique no campo de preço
2. Digite o novo preço
3. Pressione Enter

### Navegação por Teclado

- **Tab** - Mover entre campos editáveis
- **Enter** - Confirmar edição
- **Escape** - Cancelar edição

### Edições em Lote

Para pedidos complexos, considere usar o menu de três pontos para acessar o formulário de edição completo, que mostra todos os campos de uma vez.
