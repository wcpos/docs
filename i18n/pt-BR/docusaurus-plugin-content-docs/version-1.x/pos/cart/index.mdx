O Painel do Carrinho é o lado direito da tela do POS onde você gerencia o pedido atual. Aqui você pode ver itens, editar quantidades e preços, selecionar clientes e prosseguir para o checkout.

{/* Screenshot: pos-cart-panel.png */}

## Visão Geral da Interface {#interface-overview}

### Seletor de Clientes {#customer-selector}

Na parte superior do Painel do Carrinho:

- **Emblema do Cliente** - Mostra o cliente atual (por exemplo, "Convidado" ou nome do cliente)
- **Ícone de Adicionar Cliente** (<Icon name="plus" />) - Criar um novo cliente
- **Configurações de Exibição** (<Icon name="sliders" />) - Configurar colunas e opções do carrinho

Clique no emblema do cliente para procurar e selecionar um cliente diferente.

### Itens do Pedido {#line-items}

A área principal exibe itens no pedido atual:

- **QTD** - Quantidade (editável)
- **Nome** - Nome do produto com atributos de variação
- **Preço** - Preço unitário (editável)
- **Total** - Total da linha

Veja [Itens do Pedido](/pos/cart/line-items) para detalhes sobre edição.

### Opções de Adicionar ao Carrinho {#add-to-cart-options}

Abaixo dos itens do pedido, você pode adicionar itens especiais:

- **Adicionar Produto Diverso** - Adicionar um item personalizado com entrada de preço manual
- **Adicionar Taxa** - Adicionar uma linha de taxa (por exemplo, embrulho para presente, taxa de serviço)
- **Adicionar Frete** - Adicionar uma linha de frete com entrada de custo manual

:::note Tipos de Itens do Pedido
O WooCommerce utiliza três tipos de itens do pedido: `line_item` (produtos), `fee_line` (taxas) e `shipping_line` (frete). Atualmente, apenas a entrada manual é suportada para frete—o POS não calcula custos de frete automaticamente.
:::

### Resumo do Pedido {#order-summary}

- **Subtotal** - Total antes de impostos e taxas
- **Imposto** - Valor do imposto calculado (se aplicável)
- **Total** - Total final do pedido

### Ações do Pedido {#order-actions}

Na parte inferior do Painel do Carrinho:

- **Nota do Pedido** - Adicionar uma nota visível ao cliente
- **Meta do Pedido** - Adicionar metadado personalizado ou ver JSON
- **Salvar no Servidor** - Salvar o pedido no WooCommerce com status `pos-open`

Veja [Ações do Pedido](/pos/cart/order-actions) para detalhes.

### Botões Principais {#main-buttons}

- **Anular** (vermelho) - Cancelar/excluir o pedido atual
- **Checkout** (verde) - Prosseguir para o pagamento

### Pedidos Abertos {#open-orders}

Na parte inferior, um carrossel mostra todos os pedidos abertos:

- Cada carrinho exibe seu total
- Clique em um carrinho para alternar para ele
- O carrinho atual é destacado

Veja [Pedidos Abertos](/pos/cart/open-orders) para detalhes.

### Mostrar Comprovante Automaticamente Após o Checkout {#automatically-show-receipt-after-checkout}

Se habilitado, a tela de comprovante é exibida automaticamente após a conclusão de uma venda.

### Imprimir Comprovante Automaticamente Após o Checkout {#automatically-print-receipt-after-checkout}

Se habilitado, o comprovante é impresso automaticamente quando exibido. Isso economiza tempo ao reduzir etapas manuais de impressão.

### Descontos Rápidos {#quick-discounts}

Uma lista separada por vírgulas de porcentagens de desconto que aparecem como botões de atalho rápidos.

Por exemplo, inserir `5,10,15,20` adiciona quatro botões que aplicam descontos percentuais ao pedido inteiro com um único clique.

### Colunas {#columns}

Configure quais colunas aparecem no carrinho:

| Coluna | Descrição | Opções de Exibição |
|--------|-----------|--------------------|
| **Qtd** | Quantidade de cada item | Dividir (permitir divisão de itens) |
| **Nome** | Nome do produto | SKU |
| **Preço** | Preço por unidade | Indicador de Oferta |
| **Preço Regular** | Preço não descontado | - |
| **Subtotal** | Subtotal da linha (qtd × preço) | Valor do imposto |
| **Total** | Total da linha | Valor do imposto, Indicador de Oferta |
| **Ações** | Botão remover item | - |

### Restaurar Configurações Padrão {#restore-default-settings}

Clique para redefinir todas as configurações de exibição do carrinho para os padrões originais.

## Adicionando Itens ao Carrinho {#adding-items-to-cart}

### A partir do Painel de Produtos {#from-product-panel}

Clique em um produto no [Painel de Produtos](/pos/product-panel) para adicioná-lo ao carrinho. Para produtos variáveis, selecione a variação primeiro.

### Produtos Diversos {#miscellaneous-products}

Para itens que não estão em seu catálogo:

1. Clique em **Adicionar Produto Diverso**
2. Insira um nome e preço
3. O item é adicionado ao carrinho

### Taxas {#fees}

Para adicionar uma taxa (por exemplo, embrulho para presente):

1. Clique em **Adicionar Taxa**
2. Insira um nome e valor
3. A taxa aparece como uma linha separada de item

### Frete {#shipping}

Para adicionar frete:

1. Clique em **Adicionar Frete**
2. Insira o nome e custo do método de frete
3. O frete é adicionado ao pedido

## Editando Itens do Carrinho {#editing-cart-items}

Todos os itens do carrinho são editáveis diretamente no carrinho. Veja [Itens do Pedido](/pos/cart/line-items) para detalhes sobre:

- Edição de quantidades
- Alteração de preços
- Visualização e edição de dados JSON brutos
- Remover itens
