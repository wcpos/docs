O Painel do Carrinho é o lado direito da tela do POS onde você gerencia o pedido atual. Aqui você pode visualizar itens, editar quantidades e preços, selecionar clientes e prosseguir para o pagamento.

{/* Screenshot: pos-cart-panel.png */}

## Visão Geral da Interface

### Seletor de Cliente

No topo do Painel do Carrinho:

- **Insígnia do cliente** - Mostra o cliente atual (por exemplo, "Convidado" ou nome do cliente)
- **Ícone adicionar cliente** (<Icon name="plus" />) - Criar um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas e opções do carrinho

Clique na insígnia do cliente para buscar e selecionar um cliente diferente.

### Itens da Linha

A área principal exibe itens no pedido atual:

- **QTD** - Quantidade (editável)
- **Nome** - Nome do produto com atributos de variação
- **Preço** - Preço unitário (editável)
- **Total** - Total da linha

Veja [Itens da Linha](/pos/cart/line-items) para detalhes sobre edição.

### Opções para Adicionar ao Carrinho

Abaixo dos itens da linha, você pode adicionar itens especiais:

- **Adicionar Produto Diverso** - Adicionar um item personalizado com entrada de preço manual
- **Adicionar Taxa** - Adicionar uma linha de taxa (por exemplo, embrulho para presente, taxa de serviço)
- **Adicionar Frete** - Adicionar uma linha de frete com entrada de custo manual

:::note Tipos de Itens da Linha
WooCommerce utiliza três tipos de itens da linha: `line_item` (produtos), `fee_line` (taxas) e `shipping_line` (frete). Atualmente, apenas a entrada manual é suportada para frete—o POS não calcula custos de frete automaticamente.
:::

### Resumo do Pedido

- **Subtotal** - Total antes de impostos e taxas
- **Imposto** - Valor do imposto calculado (se aplicável)
- **Total** - Total final do pedido

### Ações do Pedido

Na parte inferior do Painel do Carrinho:

- **Nota do Pedido** - Adicionar uma nota visível ao cliente
- **Meta do Pedido** - Adicionar metadados personalizados ou visualizar JSON
- **Salvar no Servidor** - Salvar o pedido no WooCommerce com status `pos-open`

Veja [Ações do Pedido](/pos/cart/order-actions) para detalhes.

### Botões Principais

- **Anular** (vermelho) - Cancelar/excluir o pedido atual
- **Finalizar Compra** (verde) - Prosseguir para o pagamento

### Pedidos Abertos

Na parte inferior, um carrossel mostra todos os pedidos abertos:

- Cada carrinho exibe seu total
- Clique em um carrinho para alternar para ele
- O carrinho atual está destacado

Veja [Pedidos Abertos](/pos/cart/open-orders) para detalhes.

### Mostrar Recebido Automaticamente Após a Finalização da Compra

Se ativado, a tela do recibo é exibida automaticamente após a conclusão de uma venda.

### Imprimir Recebido Automaticamente Após a Finalização da Compra

Se ativado, o recibo é impresso automaticamente quando exibido. Isso economiza tempo reduzindo etapas manuais de impressão.

### Descontos Rápidos

Uma lista separada por vírgulas de porcentagens de desconto que aparecem como botões de atalho rápidos.

Por exemplo, inserir `5,10,15,20` adiciona quatro botões que aplicam descontos percentuais ao pedido inteiro com um único clique.

### Colunas

Configure quais colunas aparecem no carrinho:

| Coluna | Descrição | Opções de Exibição |
|--------|-------------|-----------------|
| **Qtd** | Quantidade de cada item | Dividir (permitir divisão de itens) |
| **Nome** | Nome do produto | SKU |
| **Preço** | Preço por unidade | Indicador em Promoção |
| **Preço Normal** | Preço sem desconto | - |
| **Subtotal** | Subtotal da linha (qtd × preço) | Valor do imposto |
| **Total** | Total da linha | Valor do imposto, Indicador em Promoção |
| **Ações** | Botão remover item | - |

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição do carrinho para os padrões originais.

## Adicionando Itens ao Carrinho

### Do Painel de Produtos

Clique em um produto no [Painel de Produtos](/pos/product-panel) para adicioná-lo ao carrinho. Para produtos variáveis, selecione a variação primeiro.

### Produtos Diversos

Para itens que não estão no seu catálogo:

1. Clique em **Adicionar Produto Diverso**
2. Digite um nome e preço
3. O item é adicionado ao carrinho

### Taxas

Para adicionar uma taxa (por exemplo, embrulho para presente):

1. Clique em **Adicionar Taxa**
2. Digite um nome e valor
3. A taxa aparece como um item de linha separado

### Frete

Para adicionar frete:

1. Clique em **Adicionar Frete**
2. Digite o nome do método de frete e custo
3. O frete é adicionado ao pedido

## Editando Itens do Carrinho

Todos os itens da linha são editáveis diretamente no carrinho. Veja [Itens da Linha](/pos/cart/line-items) para detalhes sobre:

- Edição de quantidades
- Alteração de preços
- Visualização e edição de dados brutos em JSON
- Remoção de itens
