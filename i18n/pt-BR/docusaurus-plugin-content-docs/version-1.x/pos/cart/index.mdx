O Painel do Carrinho é o lado direito da tela do POS onde você gerencia o pedido atual. Aqui você pode visualizar itens, editar quantidades e preços, selecionar clientes e prosseguir para o checkout.

{/* Screenshot: pos-cart-panel.png */}

## Visão Geral da Interface

### Seletor de Cliente

No topo do Painel do Carrinho:

- **Emblema do cliente** - Mostra o cliente atual (por exemplo, "Visitante" ou nome do cliente)
- **Ícone de adicionar cliente** (<Icon name="plus" />) - Criar um novo cliente
- **Configurações de exibição** (<Icon name="sliders" />) - Configurar colunas e opções do carrinho

Clique no emblema do cliente para pesquisar e selecionar um cliente diferente.

### Itens de Linha

A área principal exibe os itens no pedido atual:

- **QTD** - Quantidade (editável)
- **Nome** - Nome do produto com atributos de variação
- **Preço** - Preço unitário (editável)
- **Total** - Total da linha

Consulte [Itens de Linha](/pos/cart/line-items) para detalhes de edição.

### Opções de Adicionar ao Carrinho

Abaixo dos itens de linha, você pode adicionar itens especiais:

- **Adicionar Produto Diversos** - Adicionar um item personalizado com entrada de preço manual
- **Adicionar Taxa** - Adicionar uma linha de taxa (por exemplo, embrulho para presente, taxa de serviço)
- **Adicionar Envio** - Adicionar uma linha de envio com entrada de custo manual

:::note Tipos de Itens de Linha
O WooCommerce usa três tipos de itens de linha: `line_item` (produtos), `fee_line` (taxas) e `shipping_line` (envio). Atualmente, apenas entrada manual é suportada para envio—o POS não calcula custos de envio automaticamente.
:::

### Resumo do Pedido

- **Subtotal** - Total antes de impostos e taxas
- **Imposto** - Valor de imposto calculado (se aplicável)
- **Total** - Total final do pedido

### Ações do Pedido

Na parte inferior do Painel do Carrinho:

- **Nota do Pedido** - Adicionar uma nota visível ao cliente
- **Metadados do Pedido** - Adicionar metadados personalizados ou visualizar JSON
- **Salvar no Servidor** - Salvar o pedido no WooCommerce com status `pos-open`

Consulte [Ações do Pedido](/pos/cart/order-actions) para detalhes.

### Botões Principais

- **Anular** (vermelho) - Cancelar/excluir o pedido atual
- **Checkout** (verde) - Prosseguir para o pagamento

### Pedidos Abertos

Na parte inferior, um carrossel mostra todos os pedidos abertos:

- Cada carrinho exibe seu total
- Clique em um carrinho para alternar para ele
- O carrinho atual é destacado

Consulte [Pedidos Abertos](/pos/cart/open-orders) para detalhes.

### Mostrar Recibo Automaticamente Após o Checkout

Se habilitado, a tela de recibo é exibida automaticamente após concluir uma venda.

### Imprimir Recibo Automaticamente Após o Checkout

Se habilitado, o recibo é impresso automaticamente quando exibido. Isso economiza tempo reduzindo etapas de impressão manual.

### Descontos Rápidos

Uma lista separada por vírgulas de percentuais de desconto que aparecem como botões de atalho.

Por exemplo, inserir `5,10,15,20` adiciona quatro botões que aplicam descontos percentuais ao pedido inteiro com um único clique.

### Colunas

Configure quais colunas aparecem no carrinho:

| Coluna | Descrição | Opções de Exibição |
|--------|-----------|---------------------|
| **Qtd** | Quantidade de cada item | Dividir (permitir dividir itens) |
| **Nome** | Nome do produto | SKU |
| **Preço** | Preço por unidade | Indicador de Promoção |
| **Preço Regular** | Preço não descontado | - |
| **Subtotal** | Subtotal da linha (qtd × preço) | Valor do Imposto |
| **Total** | Total da linha | Valor do Imposto, Indicador de Promoção |
| **Ações** | Botão de remover item | - |

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição do carrinho para seus padrões originais.

## Adicionando Itens ao Carrinho

### Do Painel de Produtos

Clique em um produto no [Painel de Produtos](/pos/product-panel) para adicioná-lo ao carrinho. Para produtos variáveis, selecione a variação primeiro.

### Produtos Diversos

Para itens não no seu catálogo:

1. Clique em **Adicionar Produto Diversos**
2. Digite um nome e preço
3. O item é adicionado ao carrinho

### Taxas

Para adicionar uma taxa (por exemplo, embrulho para presente):

1. Clique em **Adicionar Taxa**
2. Digite um nome e valor
3. A taxa aparece como um item de linha separado

### Envio

Para adicionar envio:

1. Clique em **Adicionar Envio**
2. Digite o nome do método de envio e custo
3. O envio é adicionado ao pedido

## Editando Itens do Carrinho

Todos os itens de linha são editáveis diretamente no carrinho. Consulte [Itens de Linha](/pos/cart/line-items) para detalhes sobre:

- Editar quantidades
- Alterar preços
- Visualizar e editar dados JSON brutos
- Remover itens
