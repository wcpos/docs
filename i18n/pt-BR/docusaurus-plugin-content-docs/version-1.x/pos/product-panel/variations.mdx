Produtos variáveis no WooCommerce são produtos que possuem múltiplas opções, como diferentes tamanhos, cores ou materiais. O WCPOS fornece várias maneiras de trabalhar com produtos variáveis de forma eficiente.

## Identificando Produtos Variáveis

No Painel de Produtos, os produtos variáveis são distinguidos por:

- **Botão de seta** (<Icon name="circle-arrow-right" color="green" />) em vez do botão de mais (<Icon name="circle-plus" color="green" />) usado para produtos simples
- **Opções de atributos** exibidas abaixo do nome do produto (por exemplo, "Cor: Azul, Verde, Vermelho")
- **Link "Expandir"** para visualizar todas as variações em linha

## Adicionando Produtos Variáveis ao Carrinho

### Método 1: Popover Rápido

Clique no botão de seta verde <Icon name="circle-arrow-right" color="green" /> em um produto variável para abrir um popover com menus suspensos para cada atributo:

1. Clique no botão <Icon name="circle-arrow-right" color="green" />
2. Selecione opções de cada menu suspenso (por exemplo, Tamanho: Grande, Cor: Azul)
3. A variação correspondente é adicionada ao carrinho

Este é o método mais rápido quando você sabe exatamente qual variação precisa.

{/* Screenshot: variation-popover.png */}

### Método 2: Expandir em Linha

Clique no link **"Expandir"** em um produto variável para mostrar todas as suas variações diretamente na lista de produtos:

1. Clique em **Expandir** abaixo do nome do produto
2. Todas as variações aparecem como linhas separadas abaixo do produto principal
3. Clique no botão <Icon name="circle-plus" color="green" /> em qualquer variação para adicioná-la ao carrinho
4. Clique em **Colapsar** para esconder as variações novamente

Este método é útil quando você quer ver todas as opções disponíveis, seus níveis de estoque individuais e preços.

{/* Screenshot: variations-expanded.png */}

### Método 3: Leitura de Código de Barras

Cada variação de produto pode ter seu próprio código de barras único. Quando você escaneia o código de barras de uma variação:

- Se exatamente uma variação corresponder, ela é adicionada diretamente ao carrinho
- Se múltiplas variações compartilharem o código de barras, a barra de busca mostra os resultados correspondentes

Veja [Leitura de Código de Barras](/pos/product-panel/barcode-scanning) para mais detalhes.

## Informações da Variação

Cada variação pode exibir:

- **Atributos da variação** - As opções específicas para esta variação (por exemplo, "Tamanho: Grande, Cor: Azul")
- **SKU** - Identificador único para a variação (pode diferir do produto principal)
- **Quantidade em estoque** - Nível de estoque individual para a variação
- **Preço** - As variações podem ter preços diferentes do produto principal ou entre si

## Variações no Carrinho

Quando uma variação é adicionada ao carrinho, ela exibe:

- **Nome do produto** do produto principal
- **Atributos da variação** abaixo do nome
- **Preço** para essa variação específica

Você pode editar o item da linha como qualquer outro produto no carrinho.

## Dicas para Seleção Eficiente de Variações

### Use Leitura de Código de Barras

Se suas variações têm códigos de barras individuais (recomendado), a leitura é a maneira mais rápida de adicioná-las ao carrinho.

### Configure Opções de Exibição

No Painel de Produtos [Configurações de Exibição](/pos/product-panel#display-settings), ative a opção **Atributos** para mostrar os atributos de variação diretamente na lista de produtos sem expandir.

### Navegação pelo Teclado

Quando o popover de variação está aberto, você pode usar as teclas de seta do teclado para navegar entre as opções e Enter para selecionar.

## F.A.Q.

<details>
<summary>Por que não consigo ver uma variação específica?</summary>

Verifique se a variação:
- Está publicada e não está em status de rascunho
- Está definida como "Em Estoque" ou tem quantidade em estoque > 0 (se a gestão de estoque estiver ativada)
- Tem todos os atributos obrigatórios definidos

Além disso, certifique-se de que você não filtrou o produto com o filtro "Em Estoque" se a variação estiver fora de estoque.

</details>

<details>
<summary>Posso adicionar uma variação sem selecionar todos os atributos?</summary>

Não, você deve selecionar uma opção para cada atributo para identificar uma variação específica. Se um produto tem atributos de Tamanho e Cor, você precisa escolher ambos antes de adicionar ao carrinho.

</details>

<details>
<summary>Como adiciono códigos de barras a variações individuais?</summary>

No WooCommerce, edite o produto variável e expanda cada variação. Você pode adicionar um código de barras único ao campo de código de barras de cada variação (geralmente `_sku` ou `_global_unique_id` dependendo de sua configuração).

</details>
