Produtos variáveis no WooCommerce são produtos que têm múltiplas opções, como diferentes tamanhos, cores ou materiais. O WCPOS fornece várias maneiras de trabalhar com produtos variáveis de forma eficiente.

## Identificando Produtos Variáveis {#identifying-variable-products}

No Painel de Produtos, os produtos variáveis são distinguidos por:

- **Botão de seta** (<Icon name="circle-arrow-right" color="green" />) em vez do botão de mais (<Icon name="circle-plus" color="green" />) usado para produtos simples
- **Opções de atributo** exibidas abaixo do nome do produto (por exemplo, "Colour: Azul, Verde, Vermelho")
- **Link "Expandir"** para visualizar todas as variações inline

{/* Screenshot: variable-product-row.png */}

## Adicionando Produtos Variáveis ao Carrinho {#adding-variable-products-to-cart}

### Método 1: Popover Rápido {#method-1-quick-popover}

Clique no botão de seta verde <Icon name="circle-arrow-right" color="green" /> em um produto variável para abrir um popover com menus suspensos para cada atributo:

1. Clique no botão <Icon name="circle-arrow-right" color="green" />
2. Selecione opções de cada menu suspenso (por exemplo, Tamanho: Grande, Colour: Azul)
3. A variação correspondente é adicionada ao carrinho

Este é o método mais rápido quando você sabe exatamente qual variação precisa.

{/* Screenshot: variation-popover.png */}

### Método 2: Expandir Inline {#method-2-expand-inline}

Clique no link **"Expandir"** em um produto variável para mostrar todas as suas variações diretamente na lista de produtos:

1. Clique em **Expandir** abaixo do nome do produto
2. Todas as variações aparecem como linhas separadas abaixo do produto pai
3. Clique no botão <Icon name="circle-plus" color="green" /> em qualquer variação para adicioná-la ao carrinho
4. Clique em **Colapsar** para ocultar as variações novamente

Este método é útil quando você deseja ver todas as opções disponíveis, seus níveis de estoque individuais e preços.

{/* Screenshot: variations-expanded.png */}

### Método 3: Escaneamento de Código de Barras {#method-3-barcode-scanning}

Cada variação de produto pode ter seu próprio código de barras exclusivo. Quando você escaneia o código de barras de uma variação:

- Se exatamente uma variação corresponder, ela é adicionada diretamente ao carrinho
- Se várias variações compartilharem o código de barras, a barra de pesquisa mostrará os resultados correspondentes

Veja [Escaneamento de Código de Barras](/pos/product-panel/barcode-scanning) para mais detalhes.

## Informações da Variação {#variation-information}

Cada variação pode exibir:

- **Atributos da variação** - As opções específicas para esta variação (por exemplo, "Tamanho: Grande, Colour: Azul")
- **SKU** - Identificador exclusivo para a variação (pode diferir do produto pai)
- **Quantidade em estoque** - Nível de estoque individual para a variação
- **Preço** - Variações podem ter preços diferentes do produto pai ou entre si

## Variações no Carrinho {#variations-in-the-cart}

Quando uma variação é adicionada ao carrinho, ela exibe:

- **Nome do produto** do produto pai
- **Atributos da variação** abaixo do nome
- **Preço** para aquela variação específica

Você pode editar o item da linha como qualquer outro produto no carrinho.

## Dicas para Seleção Eficiente de Variações {#tips-for-efficient-variation-selection}

### Use Escaneamento de Código de Barras {#use-barcode-scanning}

Se suas variações têm códigos de barras individuais (recomendado), o escaneamento é a maneira mais rápida de adicioná-las ao carrinho.

### Configure Opções de Exibição {#configure-display-options}

No Painel de Produtos [Configurações de Exibição](/pos/product-panel#display-settings), habilite a opção **Atributos** para mostrar os atributos da variação diretamente na lista de produtos sem expandir.

### Navegação pelo Teclado {#keyboard-navigation}

Quando o popover da variação está aberto, você pode usar as teclas de seta do teclado para navegar pelas opções e Enter para selecionar.

## F.A.Q.

<Accordion>
  <AccordionItem question="Por que não consigo ver uma variação específica?">

Verifique se a variação:
- Está publicada e não está em status de rascunho
- Está definida como "Em Estoque" ou tem quantidade em estoque > 0 (se a gestão de estoque estiver habilitada)
- Tem todos os atributos obrigatórios definidos

Certifique-se também de que você não filtrou o produto com o filtro "Em Estoque" se a variação estiver fora de estoque.

  </AccordionItem>

  <AccordionItem question="Como adiciono códigos de barras a variações individuais?">

No WooCommerce, edite o produto variável e expanda cada variação. Você pode adicionar um código de barras exclusivo ao campo de código de barras de cada variação (tipicamente `_sku` ou `_global_unique_id`, dependendo da sua configuração).

  </AccordionItem>
