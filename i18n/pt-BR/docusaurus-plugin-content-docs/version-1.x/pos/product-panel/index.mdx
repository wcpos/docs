O Painel de Produtos é o lado esquerdo da tela do POS onde você pesquisa, navega e seleciona produtos para adicionar ao carrinho.

{/* Screenshot: pos-product-panel.png */}

## Visão Geral da Interface

### Barra de Pesquisa

No topo do painel, a barra de pesquisa permite encontrar produtos rapidamente por:

- **Nome do produto** - Digite qualquer parte do nome do produto
- **SKU** - Pesquise por Unidade de Manutenção de Estoque
- **Código de barras** - Escaneie ou digite um número de código de barras

Consulte [Pesquisa e Filtragem](/pos/product-panel/search-filtering) para informações detalhadas.

### Configurações de Exibição

Clique no **ícone de controles deslizantes** (<Icon name="sliders" />) ao lado da barra de pesquisa para abrir as Configurações de Exibição. Isso permite personalizar quais colunas e informações são mostradas na lista de produtos.

### Botões de Filtro

Abaixo da barra de pesquisa, os botões de filtro permitem refinar os produtos:

- **Em Estoque** - Mostrar apenas produtos com estoque disponível
- **Destacados** - Mostrar produtos marcados como destacados no WooCommerce
- **Em Promoção** - Mostrar produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Etiqueta** - Filtrar por etiqueta de produto
- **Marca** - Filtrar por marca (se estiver usando um plugin de marca)

Filtros ativos aparecem destacados. Clique em um filtro novamente para removê-lo.

### Lista de Produtos

A área principal exibe seus produtos em uma lista rolável. Cada linha mostra:

- **Imagem do produto** - Miniatura do produto
- **Nome do produto** - Com detalhes opcionais (estoque, SKU, categorias, etc.)
- **Preço** - Preço de venda atual (com preços promocionais mostrados quando aplicável)
- **Botão de ação** - Adicionar ao carrinho (<Icon name="circle-plus" color="green" />) ou selecionar variação (<Icon name="circle-arrow-right" color="green" />)

### Tipos de Produtos

- **Produtos simples** mostram um botão verde <Icon name="circle-plus" color="green" /> para adicionar diretamente ao carrinho
- **Produtos variáveis** mostram uma seta verde <Icon name="circle-arrow-right" color="green" /> para abrir o seletor de variações

Consulte [Produtos Variáveis](/pos/product-panel/variations) para mais detalhes.

### Rodapé

Na parte inferior do Painel de Produtos:

- **Status de imposto** - Mostra a base de cálculo de imposto atual (por exemplo, "Imposto baseado em: Endereço base da loja"). Clique para ver as taxas de imposto ativas.
- **Contagem de produtos** - Mostra quantos produtos estão sendo exibidos (por exemplo, "Mostrando 10 de 17")
- **Botão de sincronização** - Atualizar produtos do servidor

## Configurações de Exibição

### Mostrar Produtos Sem Estoque

Controla se os produtos atualmente sem estoque são exibidos.

- **Habilitado**: Produtos sem estoque permanecem visíveis, mas não podem ser adicionados ao carrinho
- **Desabilitado**: Produtos sem estoque são ocultados da lista

### Colunas

Configure quais colunas aparecem na lista de produtos:

| Coluna | Descrição | Opções de Exibição |
|--------|-------------|-----------------|
| **Imagem** | Miniatura do produto | - |
| **Produto** | Nome do produto e detalhes | Estoque, SKU, Código de barras, Categorias, Etiquetas, Marcas, Atributos, Metadados |
| **SKU** | Unidade de Manutenção de Estoque (coluna separada) | - |
| **Código de Barras** | Código de barras do produto (coluna separada) | - |
| **Tipo** | Tipo de produto (simples, variável, etc.) | - |
| **Estoque** | Quantidade atual em estoque | - |
| **Custo dos Produtos Vendidos** | Preço de custo do produto | - |
| **Preço** | Preço de venda | Informações de imposto, Indicador de Promoção |
| **Ações** | Botão de adicionar ao carrinho | - |

### Chaves de Metadados

Especifique as chaves de metadados do produto que devem ser transferidas para os itens de linha do pedido quando os produtos são adicionados ao carrinho.

Digite uma lista de chaves de metadados separadas por vírgula:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Casos de uso:**
- Plugins de depósito de garrafas
- Campos personalizados de produtos
- Dados de integração de terceiros
- Informações de conformidade/rastreamento

:::note
As chaves de metadados diferenciam maiúsculas de minúsculas e devem corresponder exatamente como armazenadas nos metadados do produto.
:::

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição para seus padrões originais.

## Adicionando Produtos ao Carrinho

### Produtos Simples

Clique no botão verde <Icon name="circle-plus" color="green" /> para adicionar um produto simples ao carrinho. Cada clique adiciona mais uma unidade.

### Produtos Variáveis

Produtos variáveis (por exemplo, uma camiseta com opções de tamanho e cor) mostram uma seta verde <Icon name="circle-arrow-right" color="green" />. Clique para:

1. **Popover rápido** - Selecionar variação em um menu suspenso
2. **Expansão inline** - Clique em "Expandir" para ver todas as variações na lista

Consulte [Produtos Variáveis](/pos/product-panel/variations) para mais detalhes.

### Escaneamento de Código de Barras

Conecte um scanner de código de barras USB ou Bluetooth para adicionar produtos rapidamente. Quando você escaneia um código de barras, o WCPOS procura e adiciona automaticamente o produto correspondente.

Consulte [Escaneamento de Código de Barras](/pos/product-panel/barcode-scanning) para instruções de configuração.
