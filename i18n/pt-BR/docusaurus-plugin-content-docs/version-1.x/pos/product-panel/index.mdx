O Painel de Produtos é o lado esquerdo da tela do POS onde você pesquisa, navega e seleciona produtos para adicionar ao carrinho.

{/* Screenshot: pos-product-panel.png */}

## Visão Geral da Interface

### Barra de Pesquisa

Na parte superior do painel, a barra de pesquisa permite que você encontre rapidamente produtos por:

- **Nome do produto** - Digite qualquer parte do nome do produto
- **SKU** - Pesquisar pelo Código de Item de Estoque
- **Código de Barras** - Escaneie ou digite um número de código de barras

Consulte [Pesquisa e Filtragem](/pos/product-panel/search-filtering) para informações detalhadas.

### Configurações de Exibição

Clique no **ícone de controles deslizantes** (<Icon name="sliders" />) ao lado da barra de pesquisa para abrir as Configurações de Exibição. Isso permite que você customize quais colunas e informações são mostradas na lista de produtos.

### Botões de Filtro

Abaixo da barra de pesquisa, os botões de filtro permitem que você refine os produtos:

- **Em Estoque** - Mostrar apenas produtos com estoque disponível
- **Destaque** - Mostrar produtos marcados como destaque no WooCommerce
- **Em Promoção** - Mostrar produtos atualmente em promoção
- **Categoria** - Filtrar por categoria de produto
- **Tag** - Filtrar por tag de produto
- **Marca** - Filtrar por marca (se estiver usando um plugin de marca)

Os filtros ativos aparecem destacados. Clique em um filtro novamente para removê-lo.

### Lista de Produtos

A área principal exibe seus produtos em uma lista rolável. Cada linha mostra:

- **Imagem do produto** - Miniatura do produto
- **Nome do produto** - Com detalhes opcionais (estoque, SKU, categorias, etc.)
- **Preço** - Preço de venda atual (com preços promocionais mostrados quando aplicável)
- **Botão de ação** - Adicionar ao carrinho (<Icon name="circle-plus" color="green" />) ou selecionar variação (<Icon name="circle-arrow-right" color="green" />)

### Tipos de Produtos

- **Produtos Simples** mostram um botão verde <Icon name="circle-plus" color="green" /> para adicionar diretamente ao carrinho
- **Produtos Variáveis** mostram uma seta verde <Icon name="circle-arrow-right" color="green" /> para abrir o seletor de variações

Consulte [Produtos Variáveis](/pos/product-panel/variations) para mais detalhes.

### Rodapé

Na parte inferior do Painel de Produtos:

- **Status fiscal** - Mostra a base de cálculo de imposto atual (por exemplo, "Imposto baseado em: Endereço base da loja"). Clique para ver as taxas de imposto ativas.
- **Contagem de produtos** - Mostra quantos produtos estão exibidos (por exemplo, "Mostrando 10 de 17")
- **Botão de Sincronização** (<Icon name="arrow-rotate-right" />) - Atualizar produtos do servidor. **Pressione longamente** para opções adicionais:
  - **Sincronizar** - Atualização padrão do servidor
  - **Limpar e Atualizar** - Limpar dados locais e recarregar tudo

### Mostrar Produtos Fora de Estoque

Controla se os produtos que estão atualmente fora de estoque são exibidos.

- **Ativado**: Produtos fora de estoque permanecem visíveis, mas não podem ser adicionados ao carrinho
- **Desativado**: Produtos fora de estoque são ocultados da lista

### Colunas

Configure quais colunas aparecem na lista de produtos:

| Coluna | Descrição | Opções de Exibição |
|--------|-------------|-----------------|
| **Imagem** | Miniatura do produto | - |
| **Produto** | Nome e detalhes do produto | Estoque, SKU, Código de Barras, Categorias, Tags, Marcas, Atributos, Metadados |
| **SKU** | Código de Item de Estoque (coluna separada) | - |
| **Código de Barras** | Código de barras do produto (coluna separada) | - |
| **Tipo** | Tipo de produto (simples, variável, etc.) | - |
| **Estoque** | Quantidade atual em estoque | - |
| **Custo das Mercadorias Vendidas** | Preço de custo do produto | - |
| **Preço** | Preço de venda | Informações fiscais, Indicador de Promoção |
| **Ações** | Botão Adicionar ao carrinho | - |

### Chaves de Metadados

Especifique as chaves de metadados do produto que devem ser transferidas para os itens da linha de pedido quando os produtos são adicionados ao carrinho.

Digite uma lista de chaves de metadados separadas por vírgula:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Casos de uso:**
- Plugins de depósito de garrafa
- Campos de produtos personalizados
- Dados de integração de terceiros
- Informações de conformidade/rastreamento

:::note
As chaves de metadados são sensíveis a maiúsculas e minúsculas e devem corresponder exatamente ao que está armazenado nos metadados do produto.
:::

### Restaurar Configurações Padrão

Clique para redefinir todas as configurações de exibição para seus padrões originais.

## Adicionando Produtos ao Carrinho

### Produtos Simples

Clique no botão verde <Icon name="circle-plus" color="green" /> para adicionar um produto simples ao carrinho. Cada clique adiciona mais uma unidade.

### Produtos Variáveis

Produtos variáveis (por exemplo, uma camiseta com opções de tamanho e cor) mostram uma seta verde <Icon name="circle-arrow-right" color="green" />. Clique para:

1. **Popover rápido** - Selecionar variação de um menu suspenso
2. **Expandir inline** - Clique em "Expandir" para ver todas as variações na lista

Consulte [Produtos Variáveis](/pos/product-panel/variations) para mais detalhes.

### Leitura de Código de Barras

Conecte um scanner de código de barras USB ou Bluetooth para adicionar produtos rapidamente. Quando você escaneia um código de barras, o WCPOS pesquisa automaticamente e adiciona o produto correspondente.

Consulte [Leitura de Código de Barras](/pos/product-panel/barcode-scanning) para instruções de configuração.

## Por Que Não Consigo Ver Alguns Produtos?

Se os produtos estiverem faltando no seu Painel de Produtos, verifique:

- **Visibilidade do POS** - Produtos definidos como "Apenas Online" não aparecerão no POS. Consulte [Produtos Apenas POS](/products/pos-only-products).
- **Configurações de Estoque** - Se "Mostrar Produtos Fora de Estoque" estiver desativado nas Configurações de Exibição, os itens fora de estoque são ocultados.
- **Status de Sincronização** - Produtos novos ou recentemente atualizados podem precisar de sincronização. Pressione longamente o botão de sincronização e selecione **"Limpar e Atualizar"**.
- **Filtros** - Verifique se você tem filtros ativos (Categoria, Tag, etc.) que podem estar ocultando produtos.
