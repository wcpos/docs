Precisa devolver dinheiro no caixa? O WCPOS permite reembolsar um pedido do WooCommerce sem sair do POS — total ou parcial, de volta para a forma de pagamento original ou em dinheiro do caixa.

:::info Recurso Pro
Emitir reembolsos pelo POS requer o [WCPOS Pro](/getting-started/pro-license). Sem o Pro, faça o reembolso em `WP Admin → WooCommerce → Orders`.
:::

## Versão resumida {#the-short-version}

1. **Abra o formulário de reembolso** — na lista de Pedidos, clique no menu de três pontos do pedido e escolha **Reembolsar**; ou abra o pedido e clique em **Reembolsar** no rodapé.
2. **Defina as quantidades** — insira uma **Qtd. do reembolso** para cada linha (defina todas as linhas com a quantidade total para reembolsar o pedido inteiro, ou apenas algumas linhas para um reembolso parcial). Adicione um **Valor personalizado** e um **Motivo** opcionais, se necessário.
3. **Escolha para onde o dinheiro vai:**
   - **Reembolso para *(gateway)*** — envia os valores de volta para o cartão/carteira original, para gateways compatíveis (ex.: Stripe Terminal, Vipps MobilePay).
   - **Reembolso em dinheiro** — registra o dinheiro devolvido pelo caixa. Usado automaticamente para vendas em dinheiro.
4. **Confirmar** — pressione **Processar reembolso** e confirme o valor. O pedido é atualizado imediatamente.

Os reembolsos precisam de uma conexão ativa com o servidor — ao contrário da finalização da compra, eles não podem ser colocados em fila offline.

## Quer todos os detalhes? {#want-the-full-details}

O guia completo aborda quais status de pedido podem ser reembolsados, como o formulário de reembolso calcula impostos, as regras de decisão entre gateway e dinheiro, o que acontece depois com os recibos e o status do pedido, e a trilha de auditoria do operador de caixa/loja.
