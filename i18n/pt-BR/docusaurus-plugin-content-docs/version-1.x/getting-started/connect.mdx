## Visão Geral da Tela de Conexão

Quando você abre o aplicativo Desktop ou Mobile do WCPOS, verá a tela de Conexão. É aqui que você gerencia as conexões da sua loja e logins de usuários.

{/* Screenshot: connect-screen.png */}

## Adicionando uma Nova Loja

1. Digite a URL da sua loja WooCommerce no campo de texto (ex.: `https://minhaloja.com`)
2. Clique em **Conectar**
3. Você será redirecionado para fazer login com suas credenciais do WordPress
4. Após login bem-sucedido, você retornará ao aplicativo

## Múltiplas Lojas

Você pode se conectar a quantas lojas WooCommerce precisar. Cada loja aparece como um cartão separado na tela de Conexão, mostrando:

- **Nome da loja** e favicon
- **URL da loja**
- **Usuários logados** para essa loja

Isso é útil se você administra múltiplas localizações ou negócios.

## Múltiplos Usuários por Loja

Cada loja pode ter vários usuários logados simultaneamente. Isso é útil para:

- **Mudanças de turno** - Novo caixa pode fazer login antes do anterior sair
- **Configurações multi-caixa** - Diferentes caixas em diferentes dispositivos
- **Troca rápida** - Alternar facilmente entre contas de usuário

### Adicionando um Usuário

Clique no botão <Icon name="plus" /> ao lado de "Usuários logados" para adicionar outro usuário à loja.

### Alternando Usuários

Clique em uma etiqueta de usuário (ex.: "Brenda") para abrir o POS como esse usuário.

### Removendo um Usuário

Clique no **×** na etiqueta do usuário para desconectá-lo da loja.

## Removendo uma Loja

Clique no botão vermelho **×** no cartão da loja para removê-lo da sua lista. Isso desconecta todos os usuários e remove a conexão da loja do aplicativo.

## Loja de Demonstração

Na parte inferior da tela, você pode ver um link "Entrar na Loja de Demonstração". Isso conecta você a uma loja WooCommerce de demonstração para experimentar recursos do WCPOS sem afetar os dados da sua própria loja.

## Solução de Problemas

### Não consigo conectar à minha loja?

- Verifique se o plugin WCPOS está instalado e ativado no seu site WordPress
- Verifique se está usando a URL correta (inclua `https://` se seu site usar SSL)
- Confirme se a API REST do WooCommerce está acessível
- Verifique se sua conta de usuário tem permissões de acesso ao POS

### A conexão continua falhando?

- Tente acessar `seudominio.com/pos` em um navegador primeiro para verificar se o plugin está funcionando
- Verifique os logs de erro do seu site
- Certifique-se de que seu servidor atenda aos [requisitos mínimos](/getting-started/installation)
