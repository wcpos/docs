## Visão Geral da Tela de Conectar {#connect-screen-overview}

Quando você abre o aplicativo WCPOS Desktop ou Celular, você verá a tela de Conectar. É aqui que você gerencia suas conexões de loja e logins de usuários.

{/* Screenshot: connect-screen.png */}

## Adicionando uma Nova Loja {#adding-a-new-store}

1. Insira a URL da sua loja WooCommerce no campo de texto (por exemplo, `https://minhaloja.com`)
2. Clique em **Conectar**
3. Você será redirecionado para fazer login com suas credenciais do WordPress
4. Após o login bem-sucedido, você será retornado ao aplicativo

## Múltiplas Lojas {#multiple-stores}

Você pode se conectar a quantas lojas WooCommerce precisar. Cada loja aparece como um cartão separado na tela de Conectar, mostrando:

- **Nome da loja** e favicon
- **URL da loja**
- **Usuários logados** para essa loja

Isso é útil se você gerencia vários locais ou negócios.

## Múltiplos Usuários Por Loja {#multiple-users-per-store}

Cada loja pode ter múltiplos usuários conectados simultaneamente. Isso é útil para:

- **Mudanças de turno** - O novo caixa pode fazer login antes que o anterior saia
- **Configurações de múltiplos caixas** - Diferentes caixas em diferentes dispositivos
- **Troca rápida** - Mudar facilmente entre contas de usuário

### Adicionando um Usuário {#adding-a-user}

Clique no botão <Icon name="plus" /> ao lado de "Usuários logados" para adicionar outro usuário a essa loja.

### Mudando de Usuário {#switching-users}

Clique em um crachá de usuário (por exemplo, "Brenda") para abrir o POS como esse usuário.

### Removendo um Usuário {#removing-a-user}

Clique no **×** em um crachá de usuário para desconectar aquele usuário da loja.

## Removendo uma Loja {#removing-a-store}

Clique no botão vermelho **×** no cartão da loja para removê-la da sua lista. Isso desconecta todos os usuários e remove a conexão da loja do aplicativo.

## Loja de Demonstração {#demo-store}

Na parte inferior da tela, você pode ver um link "Entrar na Loja de Demonstração". Isso conecta você a uma loja WooCommerce de demonstração para experimentar os recursos do WCPOS sem afetar os dados da sua própria loja.

## Solução de Problemas {#troubleshooting}

<Accordion>
  <AccordionItem question="Não consegue conectar à sua loja?">

  - Verifique se o plugin WCPOS está instalado e ativado no seu site WordPress
  - Verifique se você está usando a URL correta (inclua `https://` se o seu site usar SSL)
  - Verifique se a API REST do WooCommerce está acessível
  - Verifique se sua conta de usuário tem permissões de acesso ao POS

  </AccordionItem>

  <AccordionItem question="A conexão continua falhando?">

  - Tente acessar `yourdomain.com/pos` em um navegador da web primeiro para verificar se o plugin está funcionando
  - Verifique os registros de erro do seu site para qualquer problema
  - Assegure-se de que seu servidor atenda aos [requisitos mínimos](/getting-started/installation)

  </AccordionItem>
</Accordion>
