## Visão Geral da Tela de Conexão

Quando você abre o aplicativo WCPOS Desktop ou Móvel, verá a tela de Conexão. É aqui que você gerencia suas conexões de loja e logins de usuários.

{/* Screenshot: connect-screen.png */}

## Adicionando uma Nova Loja

1. Insira a URL da sua loja WooCommerce no campo de texto (ex.: `https://minhaloja.com`)
2. Clique em **Conectar**
3. Você será redirecionado para fazer login com suas credenciais do WordPress
4. Após o login bem-sucedido, você retornará ao aplicativo

## Múltiplas Lojas

Você pode se conectar a quantas lojas WooCommerce precisar. Cada loja aparece como um cartão separado na tela de Conexão, mostrando:

- **Nome da loja** e favicon
- **URL da loja**
- **Usuários logados** naquela loja

Isso é útil se você gerencia várias localizações ou negócios.

## Múltiplos Usuários Por Loja

Cada loja pode ter vários usuários logados simultaneamente. Isso é útil para:

- **Mudanças de turno** - O novo caixa pode fazer login antes que o anterior faça logout
- **Configurações de múltiplos registros** - Diferentes caixas em diferentes dispositivos
- **Troca rápida** - Fácil alternância entre contas de usuário

### Adicionando um Usuário

Clique no botão <Icon name="plus" /> ao lado de "Usuários logados" para adicionar outro usuário a essa loja.

### Alternando Usuários

Clique em um crachá de usuário (ex.: "Brenda") para abrir o POS como aquele usuário.

### Removendo um Usuário

Clique no **×** em um crachá de usuário para fazer logout desse usuário da loja.

## Removendo uma Loja

Clique no botão **×** vermelho no cartão da loja para removê-la da sua lista. Isso fará logout de todos os usuários e removerá a conexão da loja do aplicativo.

## Loja de Demonstração

Na parte inferior da tela, você pode ver um link "Entrar na Loja de Demonstração". Isso conecta você a uma loja WooCommerce de demonstração para experimentar os recursos do WCPOS sem afetar os dados da sua própria loja.

## Resolução de Problemas

### Não consigo conectar à minha loja?

- Certifique-se de que o plugin WCPOS está instalado e ativado em seu site WordPress
- Verifique se você está usando a URL correta (inclua `https://` se seu site usar SSL)
- Verifique se a API REST do WooCommerce está acessível
- Verifique se sua conta de usuário tem permissões de acesso ao POS

### A conexão continua falhando?

- Tente acessar `seusite.com/pos` em um navegador da web primeiro para verificar se o plugin está funcionando
- Verifique os registros de erro do seu site para quaisquer problemas
- Certifique-se de que seu servidor atende aos [requisitos mínimos](/getting-started/installation)
