## O que é o WCPOS?

WCPOS é um sistema de Ponto de Venda (PDV) para WooCommerce. Ele permite que você venda produtos da sua loja WooCommerce pessoalmente, em um local físico, usando um navegador web ou um aplicativo dedicado.

**Disponível em:**

- **Web:** Acesso instantâneo em `seudominio.com/pos`
- **Desktop:** Baixe para Windows, macOS e Linux em [wcpos.com](https://wcpos.com)
- **Mobile:** Aplicativos para iOS e Android (em breve)

## Comece em 5 Minutos

### 1. Instale o Plugin

Instale o plugin gratuito [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) no seu site WordPress:

1. Vá para `WP Admin > Plugins > Adicionar Novo`
2. Procure por "WooCommerce POS"
3. Clique em **Instalar Agora**, depois **Ativar**

Veja [Instalação](/getting-started/installation) para requisitos detalhados.

### 2. Acessar o PDV

**Usuários Web:**
Navegue para `seudominio.com/pos` no seu navegador. Será solicitado que você faça login com suas credenciais do WordPress.

**Usuários Desktop/Mobile:**
1. Baixe o aplicativo em [wcpos.com](https://wcpos.com)
2. Insira a URL da sua loja e clique em **Conectar**
3. Faça login com suas credenciais do WordPress

Veja [Conectar](/getting-started/connect) para detalhes sobre gerenciamento de múltiplas lojas e usuários.

### 3. Aguarde a Sincronização Inicial

O WCPOS baixa produtos em pequenos lotes (cerca de 50 por vez) para evitar sobrecarregar o servidor. Os produtos são baixados progressivamente conforme você pesquisa, rola ou filtra—não é necessário esperar que tudo sincronize antes de começar a vender. Veja [Sincronização de Produtos](/products/sync) para detalhes.

### 4. Comece a Vender

- Pesquise ou escaneie produtos para adicioná-los ao carrinho
- Selecione um cliente ou use "Visitante"
- Clique em **Checkout** para processar o pagamento
- Imprima ou envie o recibo por e-mail

## Recursos Principais

- **Busca local rápida** - Produtos são armazenados localmente para resultados instantâneos
- **Navegação offline** - Pesquise produtos e clientes sem internet (concluir pedidos requer conectividade)
- **Leitura de código de barras** - Use qualquer leitor de código de barras USB ou Bluetooth
- **Múltiplos métodos de pagamento** - Dinheiro, cartão e gateways personalizados
- **Impressão de recibo** - Imprima ou envie recibos personalizáveis por e-mail
- **Suporte a múltiplas lojas** - Conecte-se a múltiplas lojas WooCommerce

## Recursos Pro

[WCPOS Pro](https://wcpos.com/pro) adiciona recursos adicionais:

- **Gerenciamento de produtos** - Edite estoque e preços diretamente no PDV
- **Histórico de pedidos** - Visualize e gerencie pedidos anteriores
- **Gerenciamento de clientes** - Adicione e edite detalhes do cliente
- **Relatórios de vendas** - Relatórios de fim de dia com opção de impressão
- **Gateways de pagamento adicionais** - Stripe Terminal, SumUp e mais
