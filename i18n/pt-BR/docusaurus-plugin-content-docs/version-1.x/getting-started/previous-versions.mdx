## Reinstalando uma Versão Anterior

Se você precisa reverter para uma versão anterior do plugin WCPOS, siga estas etapas:

1. Visite a página do WordPress.org [Visão Avançada](https://wordpress.org/plugins/woocommerce-pos/advanced/) para o WCPOS.

2. Role para baixo até a seção **Opções Avançadas** e localize o menu de seleção **Versões Anteriores**.

3. Abra o menu de seleção e você verá uma lista de versões disponíveis do plugin. Encontre a versão que deseja instalar e clique no botão **Baixar** ao lado dela. Salve o arquivo baixado na sua área de trabalho ou em um local conveniente.

4. Faça login no seu painel do WordPress Admin dashboard.

5. Navegue até a página **Plugins** clicando em **Plugins** no menu à esquerda.

6. Desative e exclua a versão atual do plugin WCPOS. Isso removerá o plugin da sua instalação do WordPress. Não se preocupe, seus dados não serão perdidos.

7. Clique no botão **Adicionar Novo** no topo da página.

8. Na página **Adicionar Plugins**, clique no botão **Carregar Plugin**.

9. Escolha o arquivo do plugin previamente baixado da sua área de trabalho ou do local onde você o salvou.

<div class="image-container">
  <Image
    alt="Carregar Plugin"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">Carregar plugin no Admin do WordPress</p>
</div>

10. Clique no botão **Instalar Agora** e aguarde a instalação ser concluída.

11. Após a instalação ser concluída, clique no botão **Ativar** para ativar a versão anterior do plugin WCPOS.

É isso! Você reinstalou com sucesso uma versão anterior do plugin WCPOS.

Se você encontrar algum problema durante o processo de reinstalação ou precisar de mais assistência, entre em contato com nossa equipe de suporte pelo e-mail support@wcpos.com.
