Para garantir a funcionalidade adequada e o desempenho ideal do plugin WCPOS, certifique-se de que seu ambiente atenda aos seguintes requisitos mínimos:

## Requisitos do Servidor {#server-requirements}

- **WordPress:** Versão 5.6 ou superior
- **WooCommerce:** Versão 5.3 ou superior
- **PHP:** Versão 7.2 ou superior

Atender a esses requisitos mínimos garante a compatibilidade e estabilidade entre o plugin WCPOS e seu ambiente WordPress. Isso também assegura que você possa beneficiar-se dos recursos e melhorias mais recentes fornecidos pelo plugin.

## Requisitos do Navegador {#browser-requirements}

- **Navegador web moderno:** O plugin WCPOS é projetado para funcionar com navegadores web modernos, incluindo Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Certifique-se de que você está utilizando a versão estável mais recente do seu navegador preferido.
- **JavaScript ativado:** O plugin WCPOS depende do JavaScript para seus recursos interativos. Certifique-se de que o JavaScript está ativado nas configurações do seu navegador.
- **Sem modo de navegação privada:** O navegador não deve estar no modo de navegação privada, pois isso pode restringir o acesso ao banco de dados IndexedDB (IDB) do navegador. O modo de navegação privada pode impedir o funcionamento adequado de certos recursos no plugin WCPOS.

## Instalando o Plugin {#installing-the-plugin}

1. Vá para `WP Admin > Plugins > Adicionar Novo`
2. Pesquise por "WooCommerce POS"
3. Clique em **Instalar Agora**, depois em **Ativar**

Alternativamente, você pode baixar o plugin do [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e carregá-lo manualmente.

## Acessando o POS {#accessing-the-pos}

Após a ativação, você pode acessar o POS em:

- **Web:** `seudominio.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvel:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Atendendo aos requisitos mínimos e utilizando um navegador web compatível, você pode ter uma experiência sem interrupções usando o plugin WCPOS e aproveitar seu pleno potencial para gerenciar suas operações de ponto de venda de forma eficiente.
