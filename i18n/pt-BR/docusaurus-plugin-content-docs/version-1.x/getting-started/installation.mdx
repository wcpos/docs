Para garantir o funcionamento adequado e o desempenho ideal do plugin WCPOS, certifique-se de que seu ambiente atende aos seguintes requisitos mínimos:

## Requisitos do Servidor {#server-requirements}

O plugin WCPOS é executado no seu site WordPress, que deve atender a estes mínimos:

| Componente  | Versão mínima   | Notas           |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x recomendado |

Atender a esses mínimos garante compatibilidade e estabilidade com seu ambiente WordPress, além de acesso aos recursos mais recentes do WCPOS.

## Requisitos do Navegador {#browser-requirements}

A versão web do POS funciona em qualquer navegador moderno. Use a versão estável mais recente:

| Navegador       | Versão suportada  |
| --------------- | ----------------- |
| Google Chrome   | Estável mais recente |
| Mozilla Firefox | Estável mais recente |
| Safari          | Estável mais recente |
| Microsoft Edge  | Estável mais recente |

:::warning Evite o modo privado / anônimo
A navegação privada pode bloquear o acesso ao banco de dados IndexedDB do navegador, que o WCPOS utiliza para armazenar dados offline — alguns recursos não funcionarão corretamente. O JavaScript também deve estar habilitado.
:::

## Requisitos do Aplicativo {#app-requirements}

O WCPOS também está disponível como aplicativo de desktop e móvel. Cada um é desenvolvido para os seguintes sistemas operacionais mínimos:

| Plataforma          | SO mínimo                  | Arquiteturas               |
| ------------------- | -------------------------- | -------------------------- |
| Windows (desktop)   | Windows 10                 | x64, ARM64                 |
| macOS (desktop)     | macOS 12 (Monterey)        | Intel (x64), Apple Silicon |
| iOS / iPadOS        | iOS 16.4                   | —                          |
| Android             | Android 10 (nível de API 29) | —                       |

:::note Permissões de impressão
A impressão via Bluetooth e em rede requer a concessão ao aplicativo das permissões relevantes de Bluetooth e rede local quando solicitado.
:::

## Instalando o Plugin {#installing-the-plugin}

1. Acesse `WP Admin > Plugins > Add Plugin`
2. Pesquise por "wcpos"
3. Clique em **Install Now** e depois em **Activate**

Alternativamente, você pode baixar o plugin em [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e enviá-lo manualmente.

## Acessando o POS {#accessing-the-pos}

Após a ativação, você pode acessar o POS em:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvel:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Ao atender aos requisitos mínimos e utilizar um navegador web compatível, você terá uma experiência fluida ao usar o plugin WCPOS e poderá aproveitar todo o seu potencial para gerenciar suas operações de ponto de venda com eficiência.

Se você tiver mais dúvidas ou precisar de ajuda, não hesite em entrar em contato com nossa equipe de suporte publicando suas perguntas no [chat do Discord](https://wcpos.com/discord) ou enviando um e-mail para support@wcpos.com.
