Para garantir o funcionamento adequado e o desempenho ideal do plugin WCPOS, certifique-se de que seu ambiente atenda aos seguintes requisitos mínimos:

## Requisitos do Servidor

- **WordPress:** Versão 5.6 ou superior
- **WooCommerce:** Versão 5.3 ou superior
- **PHP:** Versão 7.2 ou superior

Atender a esses requisitos mínimos garante a compatibilidade e a estabilidade entre o plugin WCPOS e seu ambiente WordPress. Também permite que você aproveite os recursos e melhorias mais recentes fornecidos pelo plugin.

## Requisitos do Navegador

- **Navegador web moderno:** O plugin WCPOS é projetado para funcionar com navegadores web modernos, incluindo Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Certifique-se de estar usando a versão estável mais recente do seu navegador preferido.
- **JavaScript habilitado:** O plugin WCPOS depende do JavaScript para seus recursos interativos. Certifique-se de que o JavaScript esteja habilitado nas configurações do seu navegador.
- **Sem modo de navegação privada:** O navegador não deve estar no modo de navegação privada, pois isso pode restringir o acesso ao banco de dados IndexedDB (IDB) do navegador. O modo de navegação privada pode impedir o funcionamento adequado de certos recursos no plugin WCPOS.

## Instalando o Plugin

1. Acesse `WP Admin > Plugins > Adicionar Novo`
2. Pesquise por "WooCommerce POS"
3. Clique em **Instalar Agora** e depois em **Ativar**

Como alternativa, você pode baixar o plugin do [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e fazer o upload manualmente.

## Acessando o POS

Após a ativação, você pode acessar o POS em:

- **Web:** `seudominio.com/pos`
- **Desktop/Mobile:** Baixe o aplicativo em [wcpos.com](https://wcpos.com) e conecte-se à sua loja

Ao atender aos requisitos mínimos e usar um navegador web compatível, você pode ter uma experiência perfeita usando o plugin WCPOS e aproveitar todo o seu potencial para gerenciar suas operações de ponto de venda com eficiência.
