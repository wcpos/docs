Para garantir o funcionamento adequado e o desempenho ideal do plugin WCPOS, certifique-se de que seu ambiente atende aos seguintes requisitos mínimos:

## Requisitos do Servidor {#server-requirements}

- **WordPress:** Versão 5.6 ou superior
- **WooCommerce:** Versão 5.3 ou superior
- **PHP:** Versão 7.4 ou superior (8.x recomendado)

Atender a esses requisitos mínimos garante a compatibilidade e a estabilidade entre o plugin WCPOS e seu ambiente WordPress. Isso também assegura que você possa aproveitar os recursos e melhorias mais recentes fornecidos pelo plugin.

## Requisitos do Navegador {#browser-requirements}

- **Navegador web moderno:** O plugin WCPOS foi projetado para funcionar com navegadores web modernos, incluindo Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Certifique-se de estar utilizando a versão estável mais recente do seu navegador preferido.
- **JavaScript habilitado:** O plugin WCPOS utiliza JavaScript para seus recursos interativos. Certifique-se de que o JavaScript esteja habilitado nas configurações do seu navegador.
- **Sem modo de navegação privada:** O navegador não deve estar em modo de navegação privada, pois isso pode restringir o acesso ao banco de dados IndexedDB (IDB) do navegador. O modo de navegação privada pode impedir o funcionamento adequado de determinados recursos do plugin WCPOS.

## Instalando o Plugin {#installing-the-plugin}

1. Acesse `WP Admin > Plugins > Add Plugin`
2. Pesquise por "wcpos"
3. Clique em **Instalar Agora** e depois em **Ativar**

Alternativamente, você pode baixar o plugin em [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e enviá-lo manualmente.

## Acessando o POS {#accessing-the-pos}

Após a ativação, você pode acessar o POS em:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Ao atender aos requisitos mínimos e utilizar um navegador web compatível, você terá uma experiência fluida ao usar o plugin WCPOS e poderá aproveitar todo o seu potencial para gerenciar suas operações de ponto de venda com eficiência.
