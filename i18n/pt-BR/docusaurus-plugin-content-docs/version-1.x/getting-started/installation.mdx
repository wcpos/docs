Para garantir o funcionamento adequado e o desempenho ideal do plugin WCPOS, certifique-se de que seu ambiente atenda aos seguintes requisitos mínimos:

## Requisitos do Servidor {#server-requirements}

- **WordPress:** Versão 5.6 ou superior
- **WooCommerce:** Versão 5.3 ou superior
- **PHP:** Versão 7.2 ou superior

Atender a esses requisitos mínimos garante compatibilidade e estabilidade entre o plugin WCPOS e seu ambiente WordPress. Também garante que você possa se beneficiar dos últimos recursos e melhorias fornecidas pelo plugin.

## Requisitos do Navegador {#browser-requirements}

- **Navegador web moderno:** O plugin WCPOS foi projetado para funcionar com navegadores web modernos, incluindo Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Certifique-se de que você está usando a versão estável mais recente do seu navegador preferido.
- **JavaScript habilitado:** O plugin WCPOS depende do JavaScript para suas funções interativas. Certifique-se de que o JavaScript está habilitado nas configurações do seu navegador.
- **Sem modo de navegação privada:** O navegador não deve estar em modo de navegação privada, pois isso pode restringir o acesso ao banco de dados IndexedDB (IDB) do navegador. O modo de navegação privada pode impedir o funcionamento adequado de certos recursos no plugin WCPOS.

## Instalando o Plugin {#installing-the-plugin}

1. Vá para `WP Admin > Plugins > Adicionar Plugin`
2. Pesquise por "wcpos"
3. Clique em **Instalar Agora**, depois **Ativar**

Alternativamente, você pode baixar o plugin do [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e enviá-lo manualmente.

## Acessando o POS {#accessing-the-pos}

Após a ativação, você pode acessar o POS em:

- **Web:** `seudominio.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvel:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Atendendo aos requisitos mínimos e usando um navegador web compatível, você pode ter uma experiência tranquila usando o plugin WCPOS e aproveitar todo o seu potencial para gerenciar suas operações de ponto de venda de forma eficiente.
