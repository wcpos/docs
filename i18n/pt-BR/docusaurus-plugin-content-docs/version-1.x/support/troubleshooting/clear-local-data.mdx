O WCPOS armazena uma cópia local de seus produtos, pedidos e clientes para que o POS funcione rapidamente e tenha funcionalidade offline. Ocasionalmente, esses dados locais podem ficar fora de sincronia com sua loja WooCommerce. Limpar todos os dados locais remove o banco de dados local e aciona uma nova sincronização do seu servidor.

:::warning
Esta é a opção nuclear. Limpar dados locais exclui tudo armazenado no dispositivo e força uma ressincronização completa. Dependendo do tamanho da sua loja, isso pode levar algum tempo. Use apenas quando outras opções — como uma [sync](/products/sync) normal — não tenham resolvido o problema.
:::

## Quando Usá-lo {#when-to-use-it}

Você pode precisar limpar os dados locais quando:

- **Problemas de sync** — produtos, pedidos ou clientes não estão sendo atualizados corretamente
- **Após uma atualização de plugin** — para garantir compatibilidade com novas estruturas de dados
- **Dados desatualizados** — preços, níveis de estoque ou outras informações parecem estar desatualizadas

## O Que é Limpo {#what-gets-cleared}

| Limpo   | Não Limpo     |
|---------|---------------|
| Banco de dados local de produtos | Dados no lado do servidor |
| Banco de dados local de pedidos  | Configurações do WooCommerce |
| Banco de dados local de clientes | |
| Sessão de login | |

Os dados da sua loja WooCommerce não serão afetados — apenas a cópia local em seu dispositivo será removida. Uma nova sync começa automaticamente após a limpeza.

:::info Diferença entre plataformas
- **Aplicativo desktop:** Você será desconectado e precisará reconectar sua loja.
- **Aplicativo web:** Os dados de login são limpos, mas como você ainda está conectado ao WP Admin, a sessão é re-hidratada automaticamente — então você não notará que foi desconectado.
:::

## Como Limpar Dados Locais {#how-to-clear-local-data}

### Web {#web}

1. Clique no seu **menu de usuário** no canto superior direito para abrir o menu
2. Clique em **Limpar Todos os Dados Locais**
3. O banco de dados local será excluído e uma nova sync começará automaticamente

### Desktop {#desktop}

1. Abra o menu do **WCPOS** na barra de menu superior
2. Passe o mouse sobre **Avançado**
3. Clique em **Limpar Dados do App**
4. O banco de dados local será excluído e uma nova sync começará automaticamente

:::note
O aplicativo desktop usa um banco de dados SQLite, enquanto o aplicativo web usa IndexedDB. Ambos os bancos de dados são totalmente limpos, mas o comportamento de login difere — veja a nota sobre a diferença entre plataformas acima.
:::
