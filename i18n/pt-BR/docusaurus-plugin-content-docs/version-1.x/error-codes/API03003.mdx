# API03003: Valor de Parâmetro Inválido {#api03003-invalid-parameter-value}

## O Que Isso Significa {#what-this-means}

Um dos valores em sua solicitação é inválido. O formato, tipo ou valor dos dados não correspondem ao que o servidor espera.

## Causas Comuns {#common-causes}

- **Tipo de dado errado** — Texto onde um número é esperado (ou vice-versa)
- **Fora do intervalo** — O valor excede os limites permitidos
- **Formato inválido** — Email, telefone ou outros campos formatados estão incorretos
- **Referência inválida** — Referenciando um ID que não existe

## Mapeamento de Erro do Servidor {#server-error-mapping}

Este código de erro é acionado quando o servidor retorna:

| Código do Servidor | Fonte |
|-------------------|-------|
| `rest_invalid_param` | WordPress REST API |
| `woocommerce_rest_invalid_id` | WooCommerce REST API |

## Como Corrigir {#how-to-fix}

### 1. Verifique os Valores de Entrada {#1-check-input-values}

Revise os dados que você está enviando:
- **Preços** — Devem ser números válidos
- **Quantidades** — Devem ser inteiros positivos
- **Emails** — Devem estar no formato de email válido
- **IDs** — Devem referenciar registros existentes

### 2. Procure Erros em Campos Específicos {#2-look-for-specific-field-errors}

O erro pode indicar qual campo é inválido:
- Verifique os detalhes da mensagem de erro
- Corrija o campo específico
- Tente a ação novamente

### 3. Verifique se os Dados Referenciados Existem {#3-verify-referenced-data-exists}

Se estiver referenciando produtos, clientes ou outros registros:
- Certifique-se de que o item existe no WooCommerce
- Verifique se foi excluído recentemente
- Sincronize os dados para atualizar os registros locais

### 4. Verifique as Restrições de Campo {#4-check-field-constraints}

O WooCommerce pode ter restrições:
- Valores máximos/mínimos
- Formatos necessários
- Opções permitidas para campos de seleção

### 5. Limpar e Reinserir {#5-clear-and-re-enter}

Às vezes, os dados se corrompem:
- Limpe o campo problemático
- Reinserir o valor do zero
- Evite copiar e colar de outras fontes
