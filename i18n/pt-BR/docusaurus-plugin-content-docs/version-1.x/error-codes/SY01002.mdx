# SY01002: Disco Cheio {#sy01002-disk-full}

## O Que Isso Significa {#what-this-means}

O armazenamento do seu dispositivo está cheio. O POS não pode salvar dados ou continuar operando sem espaço disponível no disco.

## Causas Comuns {#common-causes}

- **Armazenamento cheio** — Nenhum espaço livre no drive
- **Banco de dados local grande** — Dados do POS consumindo espaço significativo
- **Arquivos temporários** — Arquivos temporários consumindo espaço
- **Outros aplicativos** — Outros apps preenchendo o disco

## Como Resolver {#how-to-fix}

### 1. Liberar Espaço em Disco {#1-free-up-disk-space}

Exclua arquivos desnecessários:
- Esvazie a lixeira
- Remova downloads antigos
- Exclua aplicativos não utilizados
- Limpe arquivos temporários

### 2. Verifique o Espaço Disponível {#2-check-available-space}

**Windows:**
- Abra o Explorador de Arquivos
- Clique com o botão direito no drive → Propriedades
- Visualize o espaço usado e livre

**macOS:**
- Menu Apple → Sobre Este Mac
- Clique em Armazenamento

### 3. Limpar Dados do Navegador (Versão Web) {#3-clear-browser-data-web-version}

Os navegadores armazenam dados localmente:
- Limpe cache e cookies
- Remova dados antigos do site
- Verifique o uso de armazenamento do navegador

### 4. Mover ou Excluir Arquivos Grandes {#4-move-or-delete-large-files}

Encontre e gerencie arquivos grandes:
- Vídeos ou fotos antigos
- Downloads grandes
- Software não utilizado
- Arquivos de backup que podem ser movidos para armazenamento externo

### 5. Verificar Dados do POS {#5-check-pos-data}

Se os dados do POS forem muito grandes:
- O banco de dados local pode precisar de limpeza
- Considere limpar e re-sincronizar
- Verifique se dados antigos podem ser excluídos

## Espaço Livre Recomendado {#recommended-free-space}

Mantenha pelo menos:
- **10GB** livres para operação normal
- **20GB+** para catálogos grandes
- Mais espaço é sempre melhor

## Prevenindo Problemas Futuros {#preventing-future-issues}

- Limpe regularmente arquivos desnecessários
- Configure a limpeza automática de arquivos temporários
- Monitore o uso do espaço em disco
- Considere um armazenamento maior se frequentemente cheio
