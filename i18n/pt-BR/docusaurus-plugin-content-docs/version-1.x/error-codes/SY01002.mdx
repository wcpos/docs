# SY01002: Disco Cheio

## O Que Isso Significa

O armazenamento do seu dispositivo está cheio. O POS não pode salvar dados ou continuar operando sem espaço em disco disponível.

## Causas Comuns

- **Armazenamento cheio** — Sem espaço livre no drive
- **Base de dados local grande** — Dados do POS ocupando muito espaço
- **Arquivos temporários** — Arquivos temporários consumindo espaço
- **Outros aplicativos** — Outros apps preenchendo o disco

## Como Corrigir

### 1. Liberar Espaço em Disco

Excluir arquivos desnecessários:
- Esvaziar a lixeira
- Remover downloads antigos
- Deletar aplicativos não utilizados
- Limpar arquivos temporários

### 2. Verificar Espaço Disponível

**Windows:**
- Abrir Explorador de Arquivos
- Botão direito no drive → Propriedades
- Visualizar espaço usado e livre

**macOS:**
- Menu Apple → Sobre Este Mac
- Clicar em Armazenamento

### 3. Limpar Dados do Navegador (Versão Web)

Navegadores armazenam dados localmente:
- Limpar cache e cookies
- Remover dados antigos de sites
- Verificar uso de armazenamento do navegador

### 4. Mover ou Excluir Arquivos Grandes

Encontrar e gerenciar arquivos grandes:
- Vídeos ou fotos antigos
- Downloads grandes
- Softwares não utilizados
- Arquivos de backup que podem ser movidos para armazenamento externo

### 5. Verificar Dados do POS

Se os dados do POS são muito grandes:
- A base de dados local pode precisar de limpeza
- Considerar limpar e ressincronizar
- Verificar se dados antigos podem ser removidos

## Espaço Livre Recomendado

Manter pelo menos:
- **10GB** livres para operação normal
- **20GB+** para catálogos grandes
- Mais espaço é sempre melhor

## Prevenindo Problemas Futuros

- Limpar regularmente arquivos desnecessários
- Configurar limpeza automática de arquivos temporários
- Monitorar uso de espaço em disco
- Considerar armazenamento maior se frequentemente cheio
