# Erros de Banco de Dados

Erros de banco de dados ocorrem ao armazenar ou recuperar dados no banco de dados local. O WCPOS usa um banco de dados local para armazenar produtos, clientes e outros dados para acesso rápido e funcionalidade offline. Esses erros são prefixados com `BD`.

## Categorias

| Categoria | Faixa de Código | Descrição |
|----------|------------|-------------|
| [Conexão](#erros-de-conexão) | BD01xxx | Problemas de conexão e transação do banco de dados |
| [Dados](#erros-de-dados) | BD02xxx | Problemas de registros e restrições |
| [Consulta](#erros-de-consulta) | BD03xxx | Problemas de sintaxe de consulta e tipos de dados |

---

## Erros de Conexão

Problemas de conexão ou operação no banco de dados local.

| Código | Nome | Descrição |
|------|------|-------------|
| [DB01001](./DB01001) | Falha de Conexão | Não foi possível conectar ao banco de dados local |
| [DB01002](./DB01002) | Tempo Limite de Consulta | Consulta do banco de dados demorou demais |
| [DB01003](./DB01003) | Falha na Transação | Transação do banco de dados não pôde ser concluída |

## Erros de Dados

Problemas com registros, duplicatas e restrições.

| Código | Nome | Descrição |
|------|------|-------------|
| [DB02001](./DB02001) | Registro Duplicado | Um registro com este ID já existe |
| [DB02002](./DB02002) | Registro Não Encontrado | O registro solicitado não existe |
| [DB02003](./DB02003) | Violação de Restrição | Dados violam as restrições do banco de dados |

## Erros de Consulta

Problemas de sintaxe de consulta e tipos de dados.

| Código | Nome | Descrição |
|------|------|-------------|
| [DB03001](./DB03001) | Erro de Sintaxe de Consulta | A consulta do banco de dados contém erros de sintaxe |
| [DB03002](./DB03002) | Tipo de Dado Inválido | Tipo de dado não corresponde ao tipo esperado |
| [DB03003](./DB03003) | Campo Obrigatório Ausente | Um campo obrigatório está vazio |
