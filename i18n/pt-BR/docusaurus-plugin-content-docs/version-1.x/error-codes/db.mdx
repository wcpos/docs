# Erros de Banco de Dados {#database-errors}

Erros de banco de dados ocorrem ao armazenar ou recuperar dados no banco de dados local. O WCPOS usa um banco de dados local para armazenar produtos, clientes e outros dados para acesso rápido e funcionalidade offline. Esses erros são prefixados com `DB`.

## Categorias {#categories}

| Categoria | Faixa de Código | Descrição |
|----------|-----------------|-----------|
| [Conexão](#connection-errors) | DB01xxx | Problemas de conexão e transação do banco de dados |
| [Dados](#data-errors) | DB02xxx | Problemas de registro e restrições |
| [Consulta](#query-errors) | DB03xxx | Problemas de sintaxe e tipo de dados da consulta |

---

## Erros de Conexão {#connection-errors}

Problemas ao conectar ou operar no banco de dados local.

| Código | Nome | Descrição |
|--------|------|-----------|
| [DB01001](./DB01001) | Conexão Falhou | Não foi possível conectar ao banco de dados local |
| [DB01002](./DB01002) | Tempo Limite da Consulta | A consulta ao banco de dados levou muito tempo |
| [DB01003](./DB01003) | Transação Falhou | A transação do banco de dados não pôde ser concluída |

## Erros de Dados {#data-errors}

Problemas com registros, duplicatas e restrições.

| Código | Nome | Descrição |
|--------|------|-----------|
| [DB02001](./DB02001) | Registro Duplicado | Já existe um registro com este ID |
| [DB02002](./DB02002) | Registro Não Encontrado | O registro solicitado não existe |
| [DB02003](./DB02003) | Violação de Restrição | Dados violam as restrições do banco de dados |

## Erros de Consulta {#query-errors}

Problemas com sintaxe de consulta e tipos de dados.

| Código | Nome | Descrição |
|--------|------|-----------|
| [DB03001](./DB03001) | Erro de Sintaxe da Consulta | A consulta ao banco de dados contém erros de sintaxe |
| [DB03002](./DB03002) | Tipo de Dado Inválido | O tipo de dado não corresponde ao tipo esperado |
| [DB03003](./DB03003) | Campo Obrigatório Ausente | Um campo obrigatório está vazio |
