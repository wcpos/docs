# SY02002: Serviço Indisponível

## O Que Isso Significa

Um serviço ou componente necessário não está disponível. O POS depende de vários serviços para funcionar, e um deles está atualmente inacessível.

## Causas Comuns

- **Serviço em segundo plano parado** — Um serviço necessário não está em execução
- **Problema de recurso do sistema** — Recursos insuficientes para iniciar o serviço
- **Falha de dependência** — Um serviço do qual este depende falhou
- **Problema de inicialização** — O serviço falhou ao iniciar corretamente
- **Erro de servidor** — O servidor WooCommerce retornou um erro 5xx

## Mapeamento de Erro do Servidor

Este código de erro é acionado quando o servidor retorna:

| Código do Servidor | Fonte |
|--------------------|-------|
| HTTP 5xx           | Qualquer resposta de erro do servidor (500, 502, 503, 504, etc.) |

## Como Corrigir

### 1. Reinicie o Aplicativo

Feche e reabra o POS:
1. Feche completamente o aplicativo
2. Aguarde alguns segundos
3. Inicie-o novamente

### 2. Reinicie Seu Dispositivo

Uma reinicialização completa frequentemente resolve problemas de serviço:
1. Salve qualquer trabalho importante
2. Reinicie seu computador
3. Tente o POS novamente

### 3. Verifique os Recursos do Sistema

Assegure-se de que os recursos do sistema estejam disponíveis:
- Feche aplicativos desnecessários
- Verifique o uso da memória
- Verifique o espaço em disco

### 4. Atualize o Aplicativo

Assegure-se de que você está na versão mais recente:
- Verifique se há atualizações disponíveis
- Instale quaisquer atualizações pendentes
- Reinicie após a atualização

### 5. Reinstale se Necessário

Se o serviço continuar indisponível:
1. Desinstale o aplicativo
2. Reinicie seu dispositivo
3. Baixe a versão mais recente
4. Instale a cópia nova

## Específico do Navegador (Versão Web)

Para a versão web:
- Assegure-se de que o JavaScript está ativado
- Verifique se os recursos necessários do navegador não estão bloqueados
- Tente um navegador diferente
- Desative as extensões do navegador que podem interferir

## Quando Contatar o Suporte

Se este erro persistir:
- Anote quando ocorre
- Verifique se ações específicas o desencadeiam
- Faça um relatório com detalhes para investigação
