# API05003: Plugin WCPOS Desatualizado

## O Que Isso Significa

O plugin WCPOS instalado no seu servidor está desatualizado e não é compatível com sua versão do aplicativo POS. Você precisa atualizar o plugin.

## Causas Comuns

- **Plugin não atualizado** — Atualizações automáticas podem estar desativadas
- **Atualização falhou** — Uma tentativa anterior de atualização falhou
- **Incompatibilidade de versão** — Aplicativo POS atualizado, mas plugin não
- **Site de staging/desenvolvimento** — Testando em uma versão mais antiga

## Como Corrigir

### 1. Atualizar o Plugin

No Administrador do WordPress:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Se uma atualização estiver disponível, clique em "Atualizar Agora"

### 2. Ativar Atualizações Automáticas

Para prevenir incompatibilidades futuras:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Clique em "Ativar atualizações automáticas"

### 3. Atualização Manual

Se a atualização automática falhar:
1. Baixe a versão mais recente de [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Desative o plugin atual
3. Exclua os arquivos do plugin antigo
4. Faça upload e ative a nova versão

### 4. Verificar Erros de Atualização

Se as atualizações estiverem falhando:
- Verifique as permissões de arquivo em wp-content/plugins
- Confirme se há espaço em disco
- Verifique problemas de memória PHP
- Consulte logs de erro para detalhes

### 5. Fazer Downgrade do Aplicativo POS (Temporário)

Se não puder atualizar o plugin imediatamente:
- Use uma versão mais antiga do aplicativo POS
- Esta é apenas uma solução temporária
- Planeje atualizar o plugin em breve

## Compatibilidade de Versão

Mantenha sempre ambos os componentes atualizados:
- O aplicativo POS
- O plugin WordPress do WCPOS

Verifique as notas de lançamento para quaisquer mudanças significativas ao atualizar.
