# API05003: Plugin WCPOS Desatualizado {#api05003-wcpos-plugin-outdated}

## O Que Isso Significa {#what-this-means}

O plugin WCPOS instalado no seu servidor está desatualizado e não é compatível com sua versão do aplicativo POS. Você precisa atualizar o plugin.

## Causas Comuns {#common-causes}

- **Plugin não atualizado** — Atualizações automáticas podem estar desativadas
- **Atualização falhou** — Uma tentativa de atualização anterior falhou
- **Incompatibilidade de versão** — Aplicativo POS atualizado, mas o plugin não
- **Site de teste/desenvolvimento** — Testando uma versão mais antiga

## Como Corrigir {#how-to-fix}

### 1. Atualizar o Plugin {#1-update-the-plugin}

No Admin do WordPress:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Se uma atualização estiver disponível, clique em "Atualizar Agora"

### 2. Habilitar Atualizações Automáticas {#2-enable-auto-updates}

Para evitar incompatibilidades futuras:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Clique em "Habilitar atualizações automáticas"

### 3. Atualização Manual {#3-manual-update}

Se a atualização automática falhar:
1. Baixe a versão mais recente de [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Desative o plugin atual
3. Exclua os arquivos antigos do plugin
4. Faça o upload e ative a nova versão

### 4. Verifique Erros de Atualização {#4-check-for-update-errors}

Se as atualizações estiverem falhando:
- Verifique as permissões de arquivo em wp-content/plugins
- Verifique se você tem espaço em disco
- Procure por problemas de memória PHP
- Verifique os logs de erro para detalhes

### 5. Rebaixar o Aplicativo POS (Temporariamente) {#5-downgrade-pos-app-temporary}

Se você não puder atualizar o plugin imediatamente:
- Use uma versão mais antiga do aplicativo POS
- Esta é apenas uma solução temporária
- Planeje atualizar o plugin em breve

## Compatibilidade de Versão {#version-compatibility}

Mantenha sempre ambos os componentes atualizados:
- O aplicativo POS
- O plugin WCPOS para WordPress

Verifique as notas de versão para quaisquer alterações que possam causar problemas ao atualizar.
