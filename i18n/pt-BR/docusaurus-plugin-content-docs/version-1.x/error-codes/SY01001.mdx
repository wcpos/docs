# SY01001: Memória Insuficiente

## O Que Isso Significa

Seu dispositivo está com pouca memória (RAM) e não consegue completar a operação solicitada. O POS precisa de memória suficiente para funcionar corretamente.

## Causas Comuns

- **Muitos aplicativos abertos** — Outros aplicativos usando memória
- **Conjunto de dados grande** — Muitos produtos/clientes carregados
- **Vazamento de memória** — Aplicativo não liberando memória adequadamente
- **RAM insuficiente** — Dispositivo não tem memória suficiente

## Como Corrigir

### 1. Fechar Outros Aplicativos

Liberar memória:
- Fechar abas de navegador não utilizadas
- Fechar outros aplicativos
- Fechar programas em segundo plano

### 2. Reiniciar o POS

Atualizar a memória do aplicativo:
1. Fechar completamente o POS
2. Aguardar alguns segundos
3. Reabrir o aplicativo

### 3. Reiniciar Seu Dispositivo

Uma reinicialização completa limpa toda a memória:
1. Salvar qualquer trabalho em aberto
2. Reiniciar seu computador/dispositivo
3. Abrir apenas o POS ao iniciar novamente

### 4. Limpar Dados do Navegador (Versão Web)

Se estiver usando no navegador:
- Limpar cache do navegador
- Fechar e reabrir o navegador
- Usar menos abas de navegador

### 5. Reduzir Carga de Dados

Se você tem um catálogo muito grande:
- Usar filtros para carregar menos dados de uma vez
- Considerar paginação de resultados
- Limitar o escopo de sincronização, se possível

## Requisitos Mínimos

Garantir que seu dispositivo atenda as especificações mínimas:
- **RAM:** 4GB mínimo, 8GB recomendado
- **Navegador:** Navegador moderno (Chrome, Firefox, Edge)
- **Aplicativo Desktop:** Verificar notas de lançamento para requisitos

## Problemas Contínuos

Se você frequentemente fica sem memória:
- Considerar atualizar a RAM do seu dispositivo
- Usar um dispositivo com mais recursos
- Reportar o problema se parecer excessivo
