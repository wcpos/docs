# SY01001: Falta de Memória

## O Que Isso Significa

Seu dispositivo está com pouca memória (RAM) e não consegue completar a operação solicitada. O POS precisa de memória suficiente para funcionar corretamente.

## Causas Comuns

- **Muitos aplicativos abertos** — Outros aplicativos consumindo memória
- **Conjunto de dados grande** — Muitos produtos/clientes carregados
- **Vazamento de memória** — Aplicativo não liberando memória corretamente
- **RAM insuficiente** — Dispositivo não tem memória suficiente

## Como Resolver

### 1. Fechar Outros Aplicativos

Liberar memória:
- Feche as abas do navegador não utilizadas
- Feche outros aplicativos
- Feche programas em segundo plano

### 2. Reiniciar o POS

Atualizar a memória do aplicativo:
1. Feche o POS completamente
2. Espere alguns segundos
3. Reabra o aplicativo

### 3. Reiniciar Seu Dispositivo

Um reinício completo limpa toda a memória:
1. Salve qualquer trabalho aberto
2. Reinicie seu computador/dispositivo
3. Abra apenas o POS ao iniciar novamente

### 4. Limpar Dados do Navegador (Versão Web)

Se estiver usando em um navegador:
- Limpe o cache do navegador
- Feche e reabra o navegador
- Use menos abas do navegador

### 5. Reduzir a Carga de Dados

Se você tem um catálogo muito grande:
- Use filtros para carregar menos dados de uma vez
- Considere paginar os resultados
- Limite o escopo da sincronização, se possível

## Requisitos Mínimos

Certifique-se de que seu dispositivo atende às especificações mínimas:
- **RAM:** 4GB mínimo, 8GB recomendado
- **Navegador:** Navegador moderno (Chrome, Firefox, Edge)
- **Aplicativo de Desktop:** Verifique as notas de versão para requisitos

## Problemas Contínuos

Se você frequentemente fica sem memória:
- Considere atualizar a RAM do seu dispositivo
- Use um dispositivo com mais recursos
- Relate o problema se parecer excessivo
