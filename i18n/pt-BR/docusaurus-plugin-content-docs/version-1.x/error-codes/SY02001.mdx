# SY02001: Configuração Inválida

## O Que Isso Significa

A configuração do sistema é inválida ou corrompida. O aplicativo POS encontrou configurações que não fazem sentido ou estão incompletas.

## Causas Comuns

- **Configurações corrompidas** — Arquivo de configuração foi corrompido
- **Configuração incompleta** — Processo de configuração não foi concluído
- **Erro de edição manual** — Arquivos de configuração foram modificados incorretamente
- **Problema de atualização de versão** — Configuração antiga incompatível com nova versão

## Como Corrigir

### 1. Restaurar Configurações Padrão

Se disponível no aplicativo:
- Procure por "Redefinir Configurações" ou "Restaurar Padrões"
- Isso limpa configurações personalizadas e corrige a corrupção

### 2. Concluir a Configuração

Se a configuração não foi finalizada:
1. Inicie o processo de configuração novamente
2. Complete todas as etapas necessárias
3. Verifique a configuração antes de finalizar

### 3. Limpar Dados do Aplicativo

Remova a configuração corrompida:
- **Desktop:** Exclua a pasta de dados do aplicativo, reinicie o aplicativo
- **Navegador:** Limpe os dados do site, recarregue

### 4. Reinstalar o Aplicativo

Para problemas persistentes:
1. Desinstale completamente
2. Remova quaisquer pastas de dados restantes
3. Reinstale do zero
4. Complete a configuração do início

### 5. Verificar Atualizações

Formatos de configuração podem mudar:
- Atualize o aplicativo POS
- Atualize o plugin WCPOS
- Reconfigure após a atualização

## O Que Será Redefinido

Ao redefinir a configuração:
- Será necessário fazer login novamente
- Configurações personalizadas serão perdidas
- Dados precisarão ser sincronizados novamente
- Mas seus dados do WooCommerce estão seguros no servidor
