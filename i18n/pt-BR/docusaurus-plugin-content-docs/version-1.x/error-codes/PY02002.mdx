# PY02002: Tempo Limite de Pagamento

## O que Isso Significa

O processamento do pagamento demorou demais e atingiu o tempo limite. O gateway de pagamento não respondeu dentro do tempo esperado.

## Causas Comuns

- **Sobrecarga do gateway** — O processador de pagamentos está enfrentando alto volume
- **Latência de rede** — Conexão lenta com o serviço de pagamento
- **Atraso no processamento** — Verificações complexas de fraudes ou verificação
- **Problemas no servidor** — Gateway de pagamento com problemas de desempenho

## Como Resolver

### 1. Verifique se o Pagamento foi Bem-Sucedido

**Importante:** Antes de tentar novamente, verifique o status do pagamento:
- Verifique o painel do seu gateway de pagamento
- Procure pela transação no WooCommerce
- Não processe novamente se realmente foi bem-sucedido

### 2. Tente Novamente com Cuidado

Se o pagamento definitivamente não foi processado:
- Aguarde 30-60 segundos
- Tente novamente
- Fique atento a cobranças duplicadas

### 3. Verifique Sua Conexão

Assegure uma conectividade estável:
- Teste sua conexão com a internet
- Tente uma conexão com fio se estiver usando WiFi
- Verifique se há problemas de rede

### 4. Tente em um Horário Menos Movimentado

Se o gateway estiver sobrecarregado:
- Horários de pico no varejo podem causar lentidão
- Tente novamente em breve
- Use um método de pagamento alternativo

### 5. Contate o Provedor de Pagamento

Se os tempos limite persistirem:
- Verifique a página de status deles
- Contate o suporte deles
- Relate problemas contínuos

## Evitando Cobranças Duplicadas

Quando ocorre um tempo limite:
1. **Não tente novamente imediatamente** — Verifique o status primeiro
2. **Revise as transações** — Verifique tanto no gateway quanto no WooCommerce
3. **Reembolso de duplicatas** — Processar reembolsos para quaisquer duplicatas acidentais
4. **Informe o cliente** — Explique a situação se houver um atraso

## Ações Alternativas

Se o pagamento continuar atingindo o tempo limite:
- Aceite pagamento em dinheiro
- Registre como "pagar depois" se permitido
- Use um método de pagamento diferente
