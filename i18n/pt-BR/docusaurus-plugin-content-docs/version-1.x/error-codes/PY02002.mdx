# PY02002: Tempo Limite de Pagamento Excedido

## O Que Isso Significa

O processamento do pagamento demorou demais e atingiu o tempo limite. O gateway de pagamento não respondeu dentro do tempo esperado.

## Causas Comuns

- **Sobrecarga do gateway** — Processador de pagamento está experimentando alto volume
- **Latência de rede** — Conexão lenta com o serviço de pagamento
- **Atraso no processamento** — Verificações complexas de fraude ou autenticação
- **Problemas de servidor** — Gateway de pagamento com problemas de desempenho

## Como Corrigir

### 1. Verificar se o Pagamento Foi Bem-Sucedido

**Importante:** Antes de tentar novamente, verifique o status do pagamento:
- Verifique o painel do seu gateway de pagamento
- Procure a transação no WooCommerce
- Não processe novamente se o pagamento realmente foi concluído

### 2. Tentar Novamente com Cautela

Se o pagamento definitivamente não foi processado:
- Espere 30-60 segundos
- Tente novamente
- Fique atento a cobranças duplicadas

### 3. Verificar a Conexão

Garanta uma conectividade estável:
- Teste sua conexão de internet
- Tente uma conexão cabeada se estiver no WiFi
- Verifique problemas de rede

### 4. Tente em um Horário Menos Movimentado

Se o gateway estiver sobrecarregado:
- Horários de pico no varejo podem causar lentidão
- Tente novamente em breve
- Use um método de pagamento alternativo

### 5. Contatar o Provedor de Pagamento

Se os tempos limite persistirem:
- Verifique a página de status deles
- Entre em contato com o suporte
- Relate problemas recorrentes

## Evitando Cobranças Duplicadas

Quando ocorre um tempo limite:
1. **Não tente novamente imediatamente** — Verifique o status primeiro
2. **Revise as transações** — Verifique o gateway e o WooCommerce
3. **Reembolse duplicatas** — Processe reembolsos para qualquer duplicata acidental
4. **Informe o cliente** — Explique a situação se houver atraso

## Ações Alternativas

Se o pagamento continuar excedendo o tempo limite:
- Aceite pagamento em dinheiro
- Registre como "pagar depois" se permitido
- Use um método de pagamento diferente
