# API04001: Formato de Resposta Inválido

## O Que Isso Significa

O servidor respondeu, mas o formato da resposta não era o esperado pelo PDV. O servidor deveria retornar dados JSON, mas algo diferente foi recebido.

## Causas Comuns

- **Erro de PHP exibido** — Um erro de PHP está sendo exibido antes do JSON
- **Conflito de plugin** — Outro plugin está gerando conteúdo
- **Modo de manutenção** — O site está mostrando uma página de manutenção
- **Tipo de conteúdo incorreto** — Servidor enviando HTML em vez de JSON
- **Problema de cache** — Uma página de erro em cache está sendo servida

## Como Corrigir

### 1. Verifique Seu Site

Visite seu site WordPress em um navegador:
- Está exibindo normalmente?
- Há algum erro visível?
- Está em modo de manutenção?

### 2. Verifique Erros de PHP

No `wp-config.php`, habilite temporariamente a depuração:

```php
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
define('WP_DEBUG_DISPLAY', false);
```

Verifique `wp-content/debug.log` em busca de erros.

### 3. Teste a REST API Diretamente

Visite `https://seusite.com/wp-json/` no seu navegador:
- Deve retornar dados JSON
- Se você ver HTML ou erros, há um problema
- Verifique a saída relacionada a plugins

### 4. Desative o Cache Temporariamente

Plugins de cache podem servir respostas desatualizadas:
- Limpe todos os caches
- Desative temporariamente plugins de cache
- Exclua a REST API do cache

### 5. Verifique Conflitos de Plugin

Se um plugin gera conteúdo em todas as páginas:
1. Desative todos os plugins não essenciais
2. Teste o PDV
3. Reative os plugins um por um
