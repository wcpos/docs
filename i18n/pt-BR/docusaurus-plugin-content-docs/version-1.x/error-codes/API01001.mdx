# API01001: Tempo de Conexão Esgotado

## O Que Isso Significa

O aplicativo POS tentou conectar-se ao seu servidor WooCommerce, mas o servidor demorou demais para responder. Isso geralmente indica problemas de latência de rede ou sobrecarga do servidor.

## Causas Comuns

- **Conexão de internet lenta** — Sua conexão de rede pode estar experimentando alta latência
- **Sobrecarga do servidor** — Seu servidor WordPress pode estar processando solicitações demais
- **Solicitações de dados grandes** — Solicitar dados em excesso pode causar tempos de espera
- **Atrasos de firewall ou proxy** — Dispositivos de segurança de rede podem adicionar latência

## Como Corrigir

### 1. Verifique Sua Conexão de Internet

Teste a velocidade e estabilidade da sua internet. Tente carregar o painel de administração do WooCommerce em um navegador para ver se responde lentamente.

### 2. Reduza a Carga do Servidor

Se o seu servidor estiver sobrecarregado:
- Considere atualizar seu plano de hospedagem
- Desative plugins desnecessários temporariamente
- Verifique tarefas cron ou processos em segundo plano pesados

### 3. Otimize o Tempo de Resposta do Servidor

Entre em contato com seu provedor de hospedagem sobre:
- Ativar opcache do PHP
- Aumentar limites de memória PHP
- Usar cache do lado do servidor

### 4. Aumentar Configurações de Tempo de Espera

Se o seu servidor realmente precisar de mais tempo:
- Verifique se o provedor de hospedagem permite aumentar o tempo de execução do PHP
- Considere usar uma CDN para reduzir a latência
