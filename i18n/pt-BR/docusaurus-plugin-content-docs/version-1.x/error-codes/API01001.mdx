# API01001: Timeout de Conexão

## O Que Isso Significa

O aplicativo POS tentou se conectar ao seu servidor WooCommerce, mas o servidor demorou muito para responder. Isso geralmente indica problemas de latência na rede ou um servidor sobrecarregado.

## Causas Comuns

- **Conexão de internet lenta** — Sua conexão de rede pode estar passando por alta latência
- **Sobrecarga do servidor** — Seu servidor WordPress pode estar processando muitas solicitações
- **Solicitações de dados grandes** — Solicitar muitos dados de uma só vez pode causar timeouts
- **Atrasos de firewall ou proxy** — Dispositivos de segurança de rede podem estar aumentando a latência

## Como Corrigir

### 1. Verifique Sua Conexão de Internet

Teste a velocidade e estabilidade da sua internet. Tente carregar o painel de administração do WooCommerce em um navegador para ver se ele responde lentamente.

### 2. Reduza a Carga do Servidor

Se seu servidor estiver sobrecarregado:
- Considere atualizar seu plano de hospedagem
- Desative temporariamente plugins desnecessários
- Verifique se há trabalhos cron pesados ou processos em segundo plano

### 3. Optimize o Tempo de Resposta do Servidor

Entre em contato com seu provedor de hospedagem sobre:
- Habilitar o opcache do PHP
- Aumentar os limites de memória do PHP
- Usar caching do lado do servidor

### 4. Aumente as Configurações de Timeout

Se seu servidor realmente precisa de mais tempo:
- Verifique se seu provedor de hospedagem permite aumentar o tempo de execução do PHP
- Considere usar uma CDN para reduzir a latência
