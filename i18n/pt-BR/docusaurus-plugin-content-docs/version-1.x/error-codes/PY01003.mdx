# PY01003: Cartão Expirado

## O Que Isso Significa

O cartão de pagamento ultrapassou sua data de validade e não pode mais ser usado para transações.

## Causas Comuns

- **Cartão expirado** — A data de validade do cartão já passou
- **Data de expiração digitada incorretamente** — Data de validade foi inserida errada
- **Novo cartão não ativado** — Cartão de substituição recebido mas ainda não ativado

## Como Corrigir

### 1. Verificar a Data de Validade

Olhe no cartão:
- Encontre a data de expiração (formato MM/AA)
- Compare com a data atual
- Verifique se a data inserida corresponde ao cartão

### 2. Usar um Cartão Diferente

Se o cartão realmente expirou:
- Use um cartão diferente e válido
- Use dinheiro ou pagamento alternativo
- Pergunte se o cliente tem um cartão de substituição

### 3. Obter o Cartão Atualizado

O titular do cartão deve:
- Verificar se recebeu um cartão de substituição pelo correio
- Ativar o novo cartão, se recebido
- Entrar em contato com seu banco se nenhum cartão de substituição chegou

### 4. Verificar a Entrada

Verifique novamente a entrada de expiração:
- Formato MM/AA
- Mês e ano corretos
- Sem erros de digitação

## Dicas de Atendimento ao Cliente

Quando um cartão está expirado:
- Informe o cliente de forma educada
- Sugira que verifique se tem um cartão de substituição
- Ofereça opções alternativas de pagamento
- Seja compreensivo — isso acontece com todos
