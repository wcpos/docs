# PY01003: Cartão Expirado

## O Que Isso Significa

O cartão de pagamento passou da data de validade e não pode mais ser utilizado para transações.

## Causas Comuns

- **Cartão expirado** — A data de validade do cartão já passou
- **Data de expiração incorreta informada** — Data de expiração foi inserida de forma errada
- **Novo cartão não ativado** — Cartão de substituição recebido, mas ainda não ativado

## Como Resolver

### 1. Verifique a Data de Validade

Olhe para o cartão:
- Encontre a data de validade (formato MM/AA)
- Compare com a data atual
- Verifique se a data informada corresponde ao cartão

### 2. Use um Cartão Diferente

Se o cartão estiver realmente expirado:
- Use um cartão diferente e válido
- Use dinheiro ou método de pagamento alternativo
- Pergunte se o cliente tem um cartão de substituição

### 3. Obtenha o Cartão Atualizado

O portador do cartão deve:
- Verificar o correio em busca de um cartão de substituição
- Ativar o novo cartão se recebido
- Contatar o banco se nenhum cartão de substituição tiver chegado

### 4. Verifique a Entrada

Revise a entrada da data de expiração:
- Formato MM/AA
- Mês e ano corretos
- Sem erros de digitação

## Dicas de Atendimento ao Cliente

Quando um cartão está expirado:
- Informe educadamente o cliente
- Sugira que verifique por um cartão de substituição
- Ofereça opções de pagamento alternativas
- Seja compreensivo — isso acontece com todo mundo
