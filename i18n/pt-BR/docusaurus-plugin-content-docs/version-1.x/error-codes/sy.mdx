# Erros de Sistema

Erros de sistema estão relacionados a recursos do dispositivo e configuração do sistema. Esses erros são prefixados com `SY` e indicam problemas com seu dispositivo ou com o próprio aplicativo POS.

## Categorias

| Categoria | Faixa de Código | Descrição |
|----------|------------|-------------|
| [Recurso](#erros-de-recurso) | SY01xxx | Problemas de memória, disco e permissões |
| [Serviço](#erros-de-serviço) | SY02xxx | Configuração e disponibilidade de serviços |

---

## Erros de Recurso

Problemas com recursos do dispositivo como memória, armazenamento e permissões.

| Código | Nome | Descrição |
|------|------|-------------|
| [SY01001](./SY01001) | Memória Esgotada | Dispositivo está com pouca memória |
| [SY01002](./SY01002) | Disco Cheio | Armazenamento do dispositivo está lotado |
| [SY01003](./SY01003) | Permissão Negada | Permissão do sistema foi negada |

## Erros de Serviço

Problemas com configuração do sistema e disponibilidade de serviços.

| Código | Nome | Descrição |
|------|------|-------------|
| [SY02001](./SY02001) | Configuração Inválida | Configuração do sistema é inválida |
| [SY02002](./SY02002) | Serviço Indisponível | Um serviço necessário não está disponível |
