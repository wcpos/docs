# DB01001: Conexão Falhou {#db01001-connection-failed}

## O Que Isso Significa {#what-this-means}

O POS não conseguiu se conectar ao seu banco de dados local. O WCPOS usa um banco de dados local para armazenar produtos, clientes e outros dados para acesso rápido e funcionalidade offline.

## Causas Comuns {#common-causes}

- **Armazenamento cheio** — O dispositivo não tem espaço de armazenamento disponível
- **Banco de dados corrompido** — Os arquivos do banco de dados local estão danificados
- **Problema de permissão** — O aplicativo não tem permissões de armazenamento
- **Armazenamento do navegador desativado** — IndexedDB está desativado (versão web)

## Como Resolver {#how-to-fix}

### 1. Verifique o Espaço de Armazenamento {#1-check-storage-space}

Certifique-se de que seu dispositivo tem espaço de armazenamento disponível:
- Libere espaço se o armazenamento estiver cheio
- Exclua aplicativos ou arquivos não utilizados
- Verifique o espaço em disco disponível

### 2. Limpar Dados do Aplicativo {#2-clear-app-data}

Redefina o banco de dados local:
- **Aplicativo desktop:** Procure por "Limpar Dados" nas configurações
- **Navegador web:** Limpe os dados do site para o domínio do POS
- **Observação:** Isso exigirá a re-sincronização de todos os dados

### 3. Verifique as Configurações do Navegador (Web) {#3-check-browser-settings-web}

Se estiver usando a versão web:
1. Certifique-se de que o IndexedDB está habilitado
2. Não use o modo privado/invisível (armazenamento limitado)
3. Permita que o site armazene dados

### 4. Verifique as Permissões {#4-check-permissions}

No desktop:
- Certifique-se de que o aplicativo tem permissão para gravar em seu diretório de dados
- Verifique se o antivírus está bloqueando o acesso ao banco de dados

### 5. Reinstalar o Aplicativo {#5-reinstall-the-app}

Se outras soluções falharem:
1. Desinstale o aplicativo POS
2. Reinicie seu dispositivo
3. Reinstale o aplicativo
4. Faça login e sincronize os dados

## Recuperação de Dados {#data-recovery}

Se você tiver dados não salvos:
- O aplicativo pode ter alterações pendentes que não puderam ser sincronizadas
- Entre em contato com o suporte se precisar de ajuda para recuperar dados
- Na maioria dos casos, os dados podem ser re-sincronizados a partir do servidor
