# Erros de API {#api-errors}

Erros de API ocorrem durante a comunicação com seu servidor WooCommerce. Esses erros são prefixados com `API` e estão organizados nas seguintes categorias:

## Categorias {#categories}

| Categoria | Faixa de Código | Descrição |
|-----------|------------------|-----------|
| [Conexão](#connection-errors) | API01xxx | Problemas de rede e conectividade |
| [Autenticação](#authentication-errors) | API02xxx | Problemas de login, tokens e permissões |
| [Requisição](#request-errors) | API03xxx | Problemas com requisições de saída |
| [Resposta](#response-errors) | API04xxx | Problemas com respostas do servidor |
| [Plugin/WordPress](#pluginwordpress-errors) | API05xxx | Problemas com plugins do WordPress ou WooCommerce |
| [Configuração](#configuration-errors) | API06xxx | Problemas de configuração e instalação |

---

## Erros de Conexão {#connection-errors}

Problemas de rede e conectividade entre o POS e seu servidor.

| Código | Nome | Descrição |
|--------|------|-----------|
| [API01001](./API01001) | Timeout de Conexão | O servidor demorou muito para responder |
| [API01002](./API01002) | Conexão Recusada | O servidor recusou a conexão |
| [API01003](./API01003) | Conexão Reiniciada | A conexão foi fechada inesperadamente |
| [API01004](./API01004) | Falha na Resolução de DNS | Não foi possível resolver o endereço do servidor |
| [API01005](./API01005) | Erro de Certificado SSL | Problema com o certificado de segurança do site |
| [API01006](./API01006) | Rede Inacessível | Não é possível acessar a rede |
| [API01007](./API01007) | Dispositivo Offline | Seu dispositivo não está conectado à internet |
| [API01008](./API01008) | Site Indisponível | O site não está respondendo |

## Erros de Autenticação {#authentication-errors}

Problemas com login, sessões e permissões.

| Código | Nome | Descrição |
|--------|------|-----------|
| [API02001](./API02001) | Credenciais Inválidas | Nome de usuário ou senha estão incorretos |
| [API02002](./API02002) | Token Expirado | Sua sessão expirou |
| [API02003](./API02003) | Token Inválido | O token de autenticação não é válido |
| [API02004](./API02004) | Usuário Não Autorizado | Você não tem permissão para realizar esta ação |
| [API02005](./API02005) | Permissões Insuficientes | Seu papel de usuário não possui as permissões necessárias |
| [API02006](./API02006) | Chave da API Inválida | A chave da API do WooCommerce não é válida |
| [API02007](./API02007) | Falha ao Atualizar Token | Não foi possível atualizar sua sessão |
| [API02008](./API02008) | Token de Atualização Inválido | O token de atualização não é válido |
| [API02009](./API02009) | Token de Atualização Expirado | O token de atualização expirou |
| [API02010](./API02010) | Autenticação Requerida | A autenticação é necessária para esta ação |

| Código | Nome | Descrição |
|--------|------|-----------|

| [API03003](./API03003) | Valor de Parâmetro Inválido | Um parâmetro tem um valor inválido |
| [API03004](./API03004) | Requisição Muito Grande | A requisição excede os limites de tamanho |
| [API03005](./API03005) | Limite de Taxa Excedido | Muitas requisições em pouco tempo |
| [API03006](./API03006) | Método Não Suportado | O método HTTP não é suportado |
| [API03007](./API03007) | Fila de Requisição Cheia | Muitas requisições pendentes |
