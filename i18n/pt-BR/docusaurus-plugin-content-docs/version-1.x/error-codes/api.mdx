# Erros de API

Erros de API ocorrem ao se comunicar com o servidor WooCommerce. Esses erros são prefixados com `API` e estão organizados nas seguintes categorias:

## Categorias

| Categoria | Faixa de Código | Descrição |
|----------|------------|-------------|
| [Conexão](#erros-de-conexão) | API01xxx | Problemas de rede e conectividade |
| [Autenticação](#erros-de-autenticação) | API02xxx | Problemas de login, tokens e permissões |
| [Requisição](#erros-de-requisição) | API03xxx | Problemas com solicitações de saída |
| [Resposta](#erros-de-resposta) | API04xxx | Problemas com respostas do servidor |
| [Plugin/WordPress](#erros-de-pluginwordpress) | API05xxx | Problemas com plugins do WordPress ou WooCommerce |
| [Configuração](#erros-de-configuração) | API06xxx | Problemas de configuração e instalação |

---

## Erros de Conexão

Problemas de rede e conectividade entre o PDV e o servidor.

| Código | Nome | Descrição |
|------|------|-------------|
| [API01001](./API01001) | Tempo de Conexão Esgotado | O servidor demorou muito para responder |
| [API01002](./API01002) | Conexão Recusada | O servidor recusou a conexão |
| [API01003](./API01003) | Conexão Reiniciada | A conexão foi fechada inesperadamente |
| [API01004](./API01004) | Falha na Resolução de DNS | Não foi possível resolver o endereço do servidor |
| [API01005](./API01005) | Erro de Certificado SSL | Problema com o certificado de segurança do site |
| [API01006](./API01006) | Rede Inacessível | Não é possível acessar a rede |
| [API01007](./API01007) | Dispositivo Offline | Seu dispositivo não está conectado à internet |
| [API01008](./API01008) | Site Indisponível | O site não está respondendo |

## Erros de Autenticação

Problemas de login, sessões e permissões.

| Código | Nome | Descrição |
|------|------|-------------|
| [API02001](./API02001) | Credenciais Inválidas | Usuário ou senha incorretos |
| [API02002](./API02002) | Token Expirado | Sua sessão expirou |
| [API02003](./API02003) | Token Inválido | O token de autenticação não é válido |
| [API02004](./API02004) | Usuário Não Autorizado | Você não tem permissão para realizar esta ação |
| [API02005](./API02005) | Permissões Insuficientes | Seu nível de usuário não possui as permissões necessárias |
| [API02006](./API02006) | Chave de API Inválida | A chave de API do WooCommerce não é válida |
| [API02007](./API02007) | Falha na Atualização do Token | Não foi possível atualizar sua sessão |
| [API02008](./API02008) | Token de Atualização Inválido | O token de atualização não é válido |
| [API02009](./API02009) | Token de Atualização Expirado | O token de atualização expirou |
| [API02010](./API02010) | Autenticação Necessária | Autenticação é necessária para esta ação |
