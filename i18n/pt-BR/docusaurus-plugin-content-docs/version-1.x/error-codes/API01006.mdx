# API01006: Rede Inacessível

## O Que Isso Significa

O POS não consegue estabelecer uma rota para o servidor. Este é um problema de nível de rede — os pacotes de dados não conseguem encontrar um caminho para o seu servidor.

## Causas Comuns

- **Problemas de configuração de rede** — Configurações incorretas de gateway ou roteamento
- **Problemas de ISP** — Seu provedor de internet está enfrentando problemas
- **Interferência de VPN** — Uma VPN pode estar roteando o tráfego incorretamente
- **Problemas de rede do servidor** — A rede do servidor pode estar inativa

## Como Corrigir

### 1. Verifique Sua Conexão com a Internet

- Tente acessar outros sites
- Reinicie seu roteador/modem
- Verifique se você está conectado à rede

### 2. Desative a VPN Temporariamente

Se estiver usando uma VPN:
- Desconecte e tente novamente
- Tente um servidor VPN diferente
- Verifique se a VPN permite o acesso à localização do seu servidor

### 3. Consulte Seu ISP

Entre em contato com seu provedor de internet se:
- Outros sites também estiverem inacessíveis
- O problema persistir após reiniciar os equipamentos de rede

### 4. Verifique o Status do Servidor

- Verifique se o seu provedor de hospedagem reporta algum problema de rede
- Tente acessar seu site a partir de uma rede diferente (por exemplo, dados móveis)
- Use ferramentas online para verificar se o site está fora do ar para todos

### 5. Verifique as Configurações do Firewall

Certifique-se de que o firewall do seu dispositivo não esteja bloqueando conexões de saída para o seu servidor.
