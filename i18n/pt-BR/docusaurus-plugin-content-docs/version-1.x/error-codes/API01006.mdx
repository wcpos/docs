# API01006: Rede Inacessível {#api01006-network-unreachable}

## O Que Isso Significa {#what-this-means}

O POS não consegue estabelecer uma rota para o servidor. Este é um problema ao nível da rede — os pacotes de dados não conseguem encontrar um caminho até o seu servidor.

## Causas Comuns {#common-causes}

- **Problemas de configuração de rede** — Configurações incorretas de gateway ou roteamento
- **Problemas com o ISP** — Seu provedor de serviços de internet está com problemas
- **Interferência de VPN** — Uma VPN pode estar roteando o tráfego incorretamente
- **Problemas de rede do servidor** — A rede do servidor pode estar fora do ar

## Como Resolver {#how-to-fix}

### 1. Verifique Sua Conexão com a Internet {#1-check-your-internet-connection}

- Tente acessar outros sites
- Reinicie seu roteador/modem
- Verifique se você está conectado à rede

### 2. Desative a VPN Temporariamente {#2-disable-vpn-temporarily}

Se estiver usando uma VPN:
- Desconecte e tente novamente
- Tente um servidor VPN diferente
- Verifique se a VPN permite acesso à localização do seu servidor

### 3. Verifique com Seu ISP {#3-check-with-your-isp}

Entre em contato com seu provedor de serviços de internet se:
- Outros sites também estiverem inacessíveis
- O problema persistir após reiniciar os equipamentos de rede

### 4. Verifique o Status do Servidor {#4-verify-server-status}

- Verifique se seu provedor de hospedagem relata problemas de rede
- Tente acessar seu site de uma rede diferente (por exemplo, dados móveis)
- Use ferramentas online para checar se o site está fora do ar para todos

### 5. Verifique as Configurações do Firewall {#5-check-firewall-settings}

Certifique-se de que o firewall do seu dispositivo não esteja bloqueando conexões de saída para o seu servidor.
