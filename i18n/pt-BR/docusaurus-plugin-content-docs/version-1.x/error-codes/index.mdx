# Códigos de Erro

Quando o WCPOS encontra um problema, ele exibe um código de erro que ajuda a identificar o problema. Cada código de erro segue o formato `[DOMÍNIO][CATEGORIA][CÓDIGO_ESPECÍFICO]` para ajudar você a entender rapidamente o tipo e a natureza do erro.

## Formato do Código de Erro

| Componente | Descrição | Exemplo |
|-----------|-------------|---------|
| **Domínio** | A área do sistema | `API`, `BD`, `PG`, `SIS` |
| **Categoria** | A categoria do erro | `01`, `02`, `03`, etc. |
| **Código Específico** | O identificador único | `001`, `002`, `003`, etc. |

**Exemplo:** `API04001` = API (domínio) + 04 (erros de resposta) + 001 (formato de resposta inválido)

## Domínios de Erro

### [Erros de API](./api)

Erros de API ocorrem ao se comunicar com o servidor WooCommerce.

| Categoria | Intervalo de Código | Descrição |
|----------|------------|-------------|
| Conexão | API01xxx | Problemas de rede e conectividade |
| Autenticação | API02xxx | Problemas de login, tokens e permissões |
| Solicitação | API03xxx | Problemas com solicitações de saída |
| Resposta | API04xxx | Problemas com respostas do servidor |
| Plugin/WordPress | API05xxx | Problemas com plugins do WordPress ou WooCommerce |
| Configuração | API06xxx | Problemas de configuração e instalação |

### [Erros de Banco de Dados](./db)

Erros de banco de dados ocorrem ao armazenar ou recuperar dados localmente.

| Categoria | Intervalo de Código | Descrição |
|----------|------------|-------------|
| Conexão | BD01xxx | Problemas de conexão e transação do banco de dados |
| Dados | BD02xxx | Problemas de registros e restrições |
| Consulta | BD03xxx | Problemas de sintaxe de consulta e tipos de dados |

### [Erros de Pagamento](./py)

Erros de pagamento ocorrem durante o checkout e o processamento de pagamento.

| Categoria | Intervalo de Código | Descrição |
|----------|------------|-------------|
| Cartão | PG01xxx | Problemas com o cartão de pagamento |
| Gateway | PG02xxx | Problemas de comunicação com gateway de pagamento |

### [Erros de Sistema](./sy)

Erros de sistema estão relacionados a recursos do dispositivo e configuração do sistema.

| Categoria | Intervalo de Código | Descrição |
|----------|------------|-------------|
| Recurso | SIS01xxx | Problemas de memória, disco e permissões |
| Serviço | SIS02xxx | Configuração e disponibilidade de serviços |
