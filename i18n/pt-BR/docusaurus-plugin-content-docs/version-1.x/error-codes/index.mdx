# Códigos de Erro {#error-codes}

Quando o WCPOS encontra um problema, ele exibe um código de erro que ajuda a identificar a questão. Cada código de erro segue o formato `[DOMÍNIO][CATEGORIA][CÓDIGO_ESPECÍFICO]` para ajudá-lo a entender rapidamente o tipo e a natureza do erro.

## Formato do Código de Erro {#error-code-format}

| Componente | Descrição | Exemplo |
|------------|-----------|---------|
| **Domínio** | A área do sistema | `API`, `DB`, `PY`, `SY` |
| **Categoria** | A categoria do erro | `01`, `02`, `03`, etc. |
| **Código Específico** | O identificador único | `001`, `002`, `003`, etc. |

**Exemplo:** `API04001` = API (domínio) + 04 (erros de resposta) + 001 (formato de resposta inválido)

## Domínios de Erro {#error-domains}

### [Erros da API](./api) {#api-errorsapi}

Erros da API ocorrem ao se comunicar com seu servidor WooCommerce.

| Categoria | Faixa de Código | Descrição |
|-----------|-----------------|-----------|
| Conexão | API01xxx | Problemas de rede e conectividade |
| Autenticação | API02xxx | Problemas de login, tokens e permissões |
| Solicitação | API03xxx | Problemas com solicitações de saída |
| Resposta | API04xxx | Problemas com respostas do servidor |
| Plugin/WordPress | API05xxx | Problemas com plugins do WordPress ou WooCommerce |
| Configuração | API06xxx | Problemas de configuração e setup |

### [Erros de Banco de Dados](./db) {#database-errorsdb}

Erros de banco de dados ocorrem ao armazenar ou recuperar dados localmente.

| Categoria | Faixa de Código | Descrição |
|-----------|-----------------|-----------|
| Conexão | DB01xxx | Problemas de conexão e transação do banco de dados |
| Dados | DB02xxx | Problemas com registros e restrições |
| Consulta | DB03xxx | Problemas de sintaxe de consulta e tipo de dado |

### [Erros de Pagamento](./py) {#payment-errorspy}

Erros de pagamento ocorrem durante o checkout e processamento de pagamentos.

| Categoria | Faixa de Código | Descrição |
|-----------|-----------------|-----------|
| Cartão | PY01xxx | Problemas com o cartão de pagamento |
| Gateway | PY02xxx | Problemas de comunicação com o gateway de pagamento |

### [Erros de Sistema](./sy) {#system-errorssy}

Erros de sistema estão relacionados aos recursos do dispositivo e configuração do sistema.

| Categoria | Faixa de Código | Descrição |
|-----------|-----------------|-----------|
| Recurso | SY01xxx | Problemas de memória, disco e permissões |
| Serviço | SY02xxx | Configuração e disponibilidade do serviço |
