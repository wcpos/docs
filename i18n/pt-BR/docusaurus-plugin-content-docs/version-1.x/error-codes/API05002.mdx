# API05002: Plugin WCPOS Não Encontrado

## O Que Isso Significa

O plugin WCPOS (WooCommerce POS) não está instalado ou não está ativo em seu site WordPress. O aplicativo POS requer este plugin para se comunicar com o WooCommerce.

## Causas Comuns

- **Plugin não instalado** — O plugin nunca foi instalado
- **Plugin desativado** — O plugin foi desativado
- **Plugin excluído** — Os arquivos do plugin foram removidos
- **Site errado** — Conectado a um site sem WCPOS

## Como Corrigir

### 1. Instalar o Plugin

Se não estiver instalado:
1. Vá para Admin do WordPress → Plugins → Adicionar Novo
2. Procure por "WooCommerce POS"
3. Instale e ative o plugin

Ou baixe do [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Ativar o Plugin

Se instalado, mas não ativo:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Clique em "Ativar"

### 3. Verificar Erros

Se o plugin não ativar:
- Verifique se há erros PHP no seu log de erros
- Verifique se o WooCommerce está instalado e ativo
- Verifique os requisitos de versão do PHP
- Procure por conflitos de plugin

### 4. Verifique a URL

Certifique-se de que o POS esteja configurado para se conectar ao site WordPress correto:
- Verifique a URL nas configurações do POS
- Verifique se corresponde à sua instalação do WordPress

### 5. Reinstale se Necessário

Se o plugin estiver corrompido:
1. Desative e exclua o plugin
2. Reinstale a partir do WordPress.org
3. Ative o plugin
