# API05002: Plugin WCPOS Não Encontrado

## O Que Isso Significa

O plugin WCPOS (WooCommerce POS) não está instalado ou não está ativo no seu site WordPress. O aplicativo de PDV requer este plugin para se comunicar com o WooCommerce.

## Causas Comuns

- **Plugin não instalado** — O plugin nunca foi instalado
- **Plugin desativado** — O plugin foi desabilitado
- **Plugin excluído** — Os arquivos do plugin foram removidos
- **Site incorreto** — Conectado a um site sem WCPOS

## Como Corrigir

### 1. Instalar o Plugin

Se não estiver instalado:
1. Vá para Administração WordPress → Plugins → Adicionar Novo
2. Procure por "WooCommerce POS"
3. Instale e ative o plugin

Ou baixe do [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Ativar o Plugin

Se estiver instalado, mas não ativo:
1. Vá para Plugins → Plugins Instalados
2. Encontre "WooCommerce POS"
3. Clique em "Ativar"

### 3. Verificar Erros

Se o plugin não ativar:
- Verifique erros de PHP no log de erros
- Confirme que o WooCommerce está instalado e ativo
- Verifique os requisitos de versão do PHP
- Procure conflitos de plugin

### 4. Verificar o URL

Certifique-se de que o PDV está configurado para se conectar ao site WordPress correto:
- Verifique o URL nas configurações do PDV
- Confirme que corresponde à sua instalação WordPress

### 5. Reinstalar se Necessário

Se o plugin estiver corrompido:
1. Desative e exclua o plugin
2. Reinstale do WordPress.org
3. Ative o plugin

## Requisitos

O WCPOS requer:
- WordPress (versão recente)
- WooCommerce (versão recente)
- PHP 7.4 ou superior
