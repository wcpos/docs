# PY02001: Erro de Gateway de Pagamento

## O Que Isso Significa

Ocorreu um erro ao se comunicar com o gateway de pagamento. O processador de pagamento não conseguiu concluir a solicitação.

## Causas Comuns

- **Indisponibilidade do gateway** — O serviço de pagamento está enfrentando problemas
- **Erro de configuração** — As configurações do gateway estão incorretas
- **Credenciais de API** — Chaves de API inválidas ou expiradas
- **Problemas de rede** — Problemas de conexão com o gateway

## Como Corrigir

### 1. Tente Novamente a Transação

Às vezes, os gateways têm problemas temporários:
- Aguarde um momento
- Tente processar o pagamento novamente
- O problema pode se resolver sozinho

### 2. Verifique o Status do Gateway

Visite a página de status do seu provedor de pagamento:
- Stripe: status.stripe.com
- PayPal: Verifique a comunidade PayPal ou o status
- Square: status.squareup.com
- Verifique interrupções conhecidas

### 3. Verificar Configuração do Gateway

No Administrador do WooCommerce:
1. Vá para WooCommerce → Configurações → Pagamentos
2. Clique no seu gateway de pagamento
3. Verifique se as chaves de API estão corretas
4. Verifique se o modo de teste/produção corresponde às suas chaves

### 4. Verificar Credenciais de API

Certifique-se de que as credenciais são válidas:
- Chaves de API não expiraram
- Usando chaves de produção para transações reais
- Usando chaves de teste para transações de teste
- Credenciais correspondem à conta correta

### 5. Testar com um Gateway Diferente

Se disponível:
- Tente processar com um gateway alternativo
- Use pagamento em dinheiro como alternativa
- Conclua a venda e concilie posteriormente

## Para Proprietários de Lojas

Problemas comuns de configuração:
- Mistura de chaves de API de teste e produção
- Chaves de API de contas diferentes
- Métodos de pagamento desativados
- Credenciais expiradas ou revogadas

Entre em contato com o suporte do seu provedor de pagamento se o problema persistir.
