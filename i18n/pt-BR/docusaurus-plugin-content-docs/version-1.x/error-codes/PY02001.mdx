# PY02001: Erro do Gateway de Pagamento

## O que Isso Significa

Ocorreu um erro ao se comunicar com o gateway de pagamento. O processador de pagamentos não conseguiu concluir a solicitação.

## Causas Comuns

- **Tempo de inatividade do gateway** — O serviço de pagamento está enfrentando problemas
- **Erro de configuração** — As configurações do gateway estão incorretas
- **Credenciais da API** — Chaves da API inválidas ou expiradas
- **Problemas de rede** — Problemas de conexão com o gateway

## Como Corrigir

### 1. Tente Novamente a Transação

Às vezes, os gateways têm problemas temporários:
- Espere um momento
- Tente processar o pagamento novamente
- O problema pode se resolver sozinho

### 2. Verifique o Status do Gateway

Visite a página de status do seu provedor de pagamento:
- Stripe: status.stripe.com
- PayPal: Verifique a comunidade do PayPal ou o status
- Square: status.squareup.com
- Verifique se há interrupções conhecidas

### 3. Verifique a Configuração do Gateway

No WooCommerce Admin:
1. Vá para WooCommerce → Configurações → Pagamentos
2. Clique no seu gateway de pagamento
3. Verifique se as chaves da API estão corretas
4. Verifique se o modo de teste/ao vivo corresponde às suas chaves

### 4. Verifique as Credenciais da API

Garanta que as credenciais sejam válidas:
- As chaves da API não expiraram
- Usando chaves ao vivo para transações ao vivo
- Usando chaves de teste para transações de teste
- As credenciais correspondem à conta correta

### 5. Teste com um Gateway Diferente

Se disponível:
- Tente processar com um gateway alternativo
- Use pagamento em dinheiro como fallback
- Complete a venda e concilie depois

## Para Proprietários de Lojas

Problemas comuns de configuração:
- Misturando chaves da API de teste e ao vivo
- Chaves da API de contas diferentes
- Métodos de pagamento desativados
- Credenciais expiradas ou revogadas

Entre em contato com o suporte do seu provedor de pagamento se o problema persistir.
