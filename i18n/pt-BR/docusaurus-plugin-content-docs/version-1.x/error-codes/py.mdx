# Erros de Pagamento {#payment-errors}

Erros de pagamento ocorrem durante o checkout e o processamento de pagamentos. Esses erros são prefixados com `PY` e estão relacionados a problemas com cartões de pagamento, gateways e transações.

## Categorias {#categories}

| Categoria | Faixa de Código | Descrição |
|-----------|------------------|-----------|
| [Cartão](#card-errors) | PY01xxx | Problemas com o cartão de pagamento |
| [Gateway](#gateway-errors) | PY02xxx | Problemas de comunicação com o gateway de pagamento |

---

## Erros de Cartão {#card-errors}

Problemas relacionados ao cartão de pagamento do cliente.

| Código | Nome | Descrição |
|--------|------|-----------|
| [PY01001](./PY01001) | Pagamento Recusado | O pagamento foi recusado |
| [PY01002](./PY01002) | Fundos Insuficientes | Não há fundos suficientes para o pagamento |
| [PY01003](./PY01003) | Cartão Expirado | O cartão de pagamento expirou |
| [PY01004](./PY01004) | Número do Cartão Inválido | O número do cartão não é válido |

## Erros de Gateway {#gateway-errors}

Problemas de comunicação com o gateway de pagamento.

| Código | Nome | Descrição |
|--------|------|-----------|
| [PY02001](./PY02001) | Erro no Gateway de Pagamento | Erro ao comunicar-se com o gateway de pagamento |
| [PY02002](./PY02002) | Tempo de Espera do Pagamento | O processamento do pagamento excedeu o tempo limite |
