# Erros de Pagamento

Erros de pagamento ocorrem durante o checkout e processamento de pagamento. Esses erros são prefixados com `PY` e estão relacionados a problemas com cartões de pagamento, gateways e transações.

## Categorias

| Categoria | Faixa de Código | Descrição |
|----------|------------|-------------|
| [Cartão](#erros-de-cartão) | PY01xxx | Problemas com o cartão de pagamento |
| [Gateway](#erros-de-gateway) | PY02xxx | Problemas de comunicação com o gateway de pagamento |

---

## Erros de Cartão

Problemas relacionados ao cartão de pagamento do cliente.

| Código | Nome | Descrição |
|------|------|-------------|
| [PY01001](./PY01001) | Pagamento Recusado | O pagamento foi recusado |
| [PY01002](./PY01002) | Fundos Insuficientes | Não há fundos suficientes para o pagamento |
| [PY01003](./PY01003) | Cartão Expirado | O cartão de pagamento está vencido |
| [PY01004](./PY01004) | Número de Cartão Inválido | O número do cartão não é válido |

## Erros de Gateway

Problemas de comunicação com o gateway de pagamento.

| Código | Nome | Descrição |
|------|------|-------------|
| [PY02001](./PY02001) | Erro de Gateway de Pagamento | Erro ao comunicar com o gateway de pagamento |
| [PY02002](./PY02002) | Tempo Limite de Pagamento | Processamento de pagamento atingiu o tempo limite |
