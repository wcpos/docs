# API01007: Dispositivo Offline {#api01007-device-offline}

## O Que Isso Significa {#what-this-means}

Seu dispositivo não está conectado à internet. O POS detectou que não há uma conexão de rede ativa.

## Causas Comuns {#common-causes}

- **WiFi desconectado** — Seu dispositivo perdeu a conexão WiFi
- **Ethernet desconectado** — O cabo de rede está desconectado
- **Modo avião ativado** — Todas as conexões sem fio estão desativadas
- **Adaptador de rede desativado** — A interface de rede está desligada

## Como Corrigir {#how-to-fix}

### 1. Verifique a Conexão WiFi {#1-check-wifi-connection}

- Verifique se o WiFi está ativado em seu dispositivo
- Certifique-se de que você está conectado à rede correta
- Verifique a intensidade do sinal WiFi

### 2. Verifique Conexões Físicas {#2-check-physical-connections}

Se usar ethernet:
- Verifique se o cabo está conectado de forma segura
- Tente um cabo ou porta diferente
- Verifique as luzes do switch/roteador de rede

### 3. Desative o Modo Avião {#3-disable-airplane-mode}

Verifique se o modo avião está ativado acidentalmente e desative-o.

### 4. Reinicie o Adaptador de Rede {#4-restart-network-adapter}

**Windows:**
- Abra Conexões de Rede
- Clique com o botão direito em seu adaptador → Desativar, depois Ativar

**macOS:**
- Desative e ative o WiFi pela barra de menu
- Ou: Preferências do Sistema → Rede → Desligar/Ligar

### 5. Reinicie Seu Dispositivo {#5-restart-your-device}

Às vezes, uma simples reinicialização resolve problemas de detecção de rede.

## Modo Offline {#offline-mode}

Enquanto offline, o WCPOS ainda pode:
- Navegar e pesquisar produtos armazenados em cache localmente
- Navegar e pesquisar clientes armazenados em cache localmente
- Iniciar novos pedidos e adicionar itens ao carrinho
- Visualizar dados sincronizados anteriormente

**Requer conectividade:**
- Concluir/fechar pedidos
- Criar novos clientes

Quando a conectividade for restaurada, o POS retomará automaticamente as operações normais.
