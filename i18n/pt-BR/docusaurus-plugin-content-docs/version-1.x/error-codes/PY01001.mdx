# PY01001: Pagamento Recusado

## O Que Isso Significa

O pagamento foi recusado pelo processador de pagamento ou pelo emissor do cartão. A transação não pôde ser concluída.

## Causas Comuns

- **Fundos insuficientes** — Não há dinheiro suficiente na conta
- **Limite do cartão excedido** — Transação ultrapassa os limites do cartão
- **Proteção contra fraude** — Banco sinalizou como suspeito
- **Dados do cartão incorretos** — Número do cartão, CVV ou validade errados
- **Restrições do cartão** — Cartão não habilitado para este tipo de compra

## Como Corrigir

### 1. Verificar Dados do Cartão

Verifique se as informações estão corretas:
- Número do cartão (todos os dígitos)
- Data de validade
- Código de segurança (CVV)
- Endereço de cobrança (se necessário)

### 2. Tentar Novamente

Às vezes ocorrem problemas temporários:
- Espere um momento
- Redigite os dados do cartão
- Tente a transação novamente

### 3. Usar Outro Método de Pagamento

Se o cartão continuar sendo recusado:
- Tente outro cartão
- Use pagamento em dinheiro
- Use um método de pagamento alternativo

### 4. Contatar o Banco

O titular do cartão pode precisar:
- Verificar a transação com o banco
- Remover bloqueios de fraude
- Confirmar que o cartão está ativo
- Verificar limites de gastos

### 5. Verificar Restrições

Alguns cartões têm restrições:
- Transações internacionais desativadas
- Transações online/POS bloqueadas
- Categorias de comerciantes bloqueadas

## Para Proprietários de Lojas

Se você estiver vendo recusas frequentes:
- Verifique se sua conta de comerciante está em dia
- Verifique a configuração do gateway de pagamento
- Revise as configurações de prevenção de fraudes
- Considere contatar seu processador de pagamento
