# PY01001: Pagamento Recusado

## O Que Isso Significa

O pagamento foi recusado pelo processador de pagamento ou emissor do cartão. A transação não pôde ser concluída.

## Causas Comuns

- **Fundos insuficientes** — Dinheiro insuficiente na conta
- **Limite do cartão excedido** — Transação excede os limites do cartão
- **Proteção contra fraudes** — Banco sinalizado como suspeito
- **Detalhes do cartão incorretos** — Número do cartão, CVV ou data de validade errados
- **Restrições do cartão** — Cartão não habilitado para este tipo de compra

## Como Corrigir

### 1. Verifique os Detalhes do Cartão

Verifique se as informações inseridas estão corretas:
- Número do cartão (todos os dígitos)
- Data de validade
- Código CVV/código de segurança
- Endereço de cobrança (se necessário)

### 2. Tente Novamente

Às vezes, ocorrem problemas temporários:
- Espere um momento
- Insira novamente os detalhes do cartão
- Tente a transação novamente

### 3. Use um Método de Pagamento Diferente

Se o cartão continuar a ser recusado:
- Tente um cartão diferente
- Use pagamento em dinheiro
- Use um método de pagamento alternativo

### 4. Contate o Banco

O titular do cartão pode precisar:
- Verificar a transação com seu banco
- Liberar quaisquer bloqueios de fraude
- Confirmar que o cartão está ativo
- Verificar se existem limites de gastos

### 5. Verifique as Restrições

Alguns cartões têm restrições:
- Transações internacionais desabilitadas
- Transações online/POS bloqueadas
- Certas categorias de comerciantes bloqueadas

## Para Proprietários de Lojas

Se você está vendo recusa frequente:
- Verifique se sua conta de comerciante está em boa situação
- Verifique a configuração do gateway de pagamento
- Revise as configurações de prevenção a fraudes
- Considere contatar seu processador de pagamento
