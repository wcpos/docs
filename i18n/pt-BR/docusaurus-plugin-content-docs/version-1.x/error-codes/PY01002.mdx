# PY01002: Fundos Insuficientes

## O que Isso Significa

O pagamento foi recusado porque não há dinheiro suficiente na conta para cobrir a transação.

## Causas Comuns

- **Saldo baixo na conta** — Fundos insuficientes disponíveis
- **Transações pendentes** — Outras transações pendentes reduzindo o saldo disponível
- **Limite de crédito alcançado** — Para cartões de crédito, o limite está esgotado
- **Retenção de fundos** — Fundos estão retidos para outras transações

## Como Resolver

### 1. Use um Método de Pagamento Diferente

A solução mais rápida:
- Tente um cartão diferente
- Use dinheiro
- Divida o pagamento entre vários métodos (se suportado)

### 2. Reduza o Valor do Pedido

Se o pagamento parcial for possível:
- Remova alguns itens
- Aplique descontos ou cupons
- Complete uma transação menor

### 3. Ações do Cliente

O cliente pode precisar:
- Transferir fundos para a conta
- Esperar as transações pendentes serem processadas
- Pagar o saldo do cartão de crédito
- Solicitar um aumento no limite de crédito

### 4. Divida a Transação

Se o seu POS suporta pagamentos divididos:
- Pague parte com um cartão
- Pague o restante com outro método

## Nota de Privacidade

Ao comunicar isso aos clientes:
- Seja discreto sobre o motivo específico da recusa
- Diga simplesmente "o cartão foi recusado"
- Permita que eles tentem outro método de pagamento em particular
