# PY01002: Fundos Insuficientes

## O Que Isso Significa

O pagamento foi recusado porque não há dinheiro suficiente na conta para cobrir a transação.

## Causas Comuns

- **Saldo baixo** — Fundos disponíveis insuficientes
- **Transações pendentes** — Outras transações pendentes reduzindo o saldo disponível
- **Limite de crédito atingido** — Para cartões de crédito, o limite está no máximo
- **Bloqueio de fundos** — Fundos estão bloqueados para outras transações

## Como Corrigir

### 1. Usar um Método de Pagamento Diferente

A solução mais rápida:
- Tentar outro cartão
- Usar dinheiro
- Dividir o pagamento em múltiplos métodos (se suportado)

### 2. Reduzir o Valor do Pedido

Se o pagamento parcial for possível:
- Remover alguns itens
- Aplicar descontos ou cupons
- Concluir uma transação menor

### 3. Ações do Cliente

O cliente pode precisar:
- Transferir fundos para a conta
- Aguardar a liberação de transações pendentes
- Pagar o saldo do cartão de crédito
- Solicitar aumento do limite de crédito

### 4. Dividir a Transação

Se o POS suportar pagamentos divididos:
- Pagar parte com um cartão
- Pagar o restante com outro método

## Nota de Privacidade

Ao comunicar isso aos clientes:
- Ser discreto sobre o motivo específico da recusa
- Simplesmente dizer que o cartão foi recusado
- Permitir que tentem outro método de pagamento reservadamente
