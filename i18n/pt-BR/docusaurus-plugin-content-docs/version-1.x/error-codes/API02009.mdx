# API02009: Token de Atualização Expirado

## O Que Isso Significa

Seu token de atualização excedeu seu tempo de vida e não pode mais ser usado para obter novos tokens de acesso. Diferentemente dos tokens de acesso (que expiram rapidamente), os tokens de atualização normalmente duram dias ou semanas, mas eventualmente expiram.

## Causas Comuns

- **Inatividade prolongada** — Você não usou o POS por muito tempo
- **Tempo de vida curto do token** — Servidor configurado com expiração rápida de token de atualização
- **Dessincronização de relógio** — Hora do dispositivo ou do servidor está significativamente diferente

## Como Corrigir

### 1. Faça Login Novamente

Quando o token de atualização expira, você deve se autenticar novamente:
1. Saia do POS
2. Insira suas credenciais
3. Um novo token de atualização será emitido

### 2. Use o POS Regularmente

Para evitar isso no futuro:
- Use o POS regularmente para manter os tokens atualizados
- Tokens são atualizados automaticamente quando você está ativo
- Períodos prolongados de inatividade exigirão novo login

### 3. Verifique as Configurações de Tempo

Garanta tempo preciso em todos os sistemas:
- Ative a sincronização automática de tempo no seu dispositivo
- Verifique se o tempo do servidor está correto
- Diferenças de tempo podem causar expiração prematura

### 4. Ajuste o Tempo de Vida do Token (Avançado)

Se os tokens expiram muito rapidamente para suas necessidades, um administrador do servidor pode ajustar o tempo de vida do token de atualização nas configurações de autenticação.
