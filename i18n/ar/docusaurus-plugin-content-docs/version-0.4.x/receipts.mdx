### قالب الإيصال الاحترافي

يسمح قالب الإيصال الاحترافي بإضافة معلومات إضافية من [المتاجر](./stores) الإدارية مثل الشعار وعنوان المتجر وساعات العمل والرسائل الخاصة.

<Gist id="c9485366a73ceda12041" />

### تخصيص تاريخ الإيصال

يستخدم WCPOS [moment.js](http://momentjs.com/) لتوطين سلاسل التاريخ. التنسيق الافتراضي للتاريخ هو `"MMMM Do YYYY, h:mm:ss a"`, مثل: 31 مايو 2015، 7:20:44 م. يرجى الرجوع إلى [وثائق moment.js](http://momentjs.com/docs/#/parsing/string-format/) للمزيد من المعلومات حول تخصيص تنسيق التاريخ.

### خصائص الطلب

يستخدم WCPOS الإخراج JSON من واجهة برمجة التطبيقات REST الخاصة بـ WC لملء قالب إيصال الطلب. [توضح وثائق واجهة برمجة التطبيقات REST الخاصة بـ WC مثالًا](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) للإخراج JSON القياسي. تمت إضافة بعض الخصائص الإضافية بواسطة WCPOS باستخدام الفلتر `woocommerce_api_order_response`.

| الخاصية                           | الوصف                                                         |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | الجزء الضريبي من الخصم في السلة                               |
| cashier.id                        | معرف المستخدم للكاشير                                         |
| cashier.first_name                | الاسم الأول للكاشير                                            |
| cashier.last_name                 | الاسم الأخير للكاشير                                           |
| payment_details.result            | نجاح أو فشل بوابة الدفع                                        |
| payment_details.message           | أي رسائل من بوابة الدفع                                        |
| payment_details.redirect          | عنوان URL لإعادة التوجيه للمدفوعات خارج الموقع، مثل: PayPal Standard |
| payment_details.method\_`gateway` | أي رسائل خاصة بالبوابة، مثل: المبلغ المدفوع والباقي             |
| shipping_lines[i].total_tax       | إضافة مبلغ الضريبة لكل سطر شحن                                 |
| subtotal_tax                      | الجزء الضريبي من المجموع الفرعي                                |
