title: "حقول الباركود المخصصة"
---

بشكل افتراضي، ستستخدم WCPOS حقل SKU كـ باركود.

![بشكل افتراضي، ستستخدم WCPOS حقل SKU كـ باركود.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "بشكل افتراضي، ستستخدم WCPOS حقل SKU كـ باركود")

يمكن لمستخدمي [WCPOS Pro](http://wcpos.com/pro) استخدام أي حقل ميتا للمنتج عن طريق اختيار حقل باركود في `WP Admin > POS > Settings > General`

![اختر حقل باركود مخصص في WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "اختر حقل باركود مخصص في WCPOS Pro")

يمكن أيضًا تعيين حقول ميتا باركود مخصصة باستخدام مرشح `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
يجب أن يكون لكل منتج باركود فريد.
:::
