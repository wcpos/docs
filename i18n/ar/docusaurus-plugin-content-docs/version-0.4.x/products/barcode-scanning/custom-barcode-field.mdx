بشكل افتراضي، سيستخدم WCPOS حقل SKU كباركود.

![سيستخدم WCPOS بشكل افتراضي حقل SKU كباركود](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "سيستخدم WCPOS بشكل افتراضي حقل SKU كباركود")

يمكن لمستخدمي [WCPOS Pro](http://wcpos.com/pro) استخدام أي حقل بيانات وصفية للمنتج عن طريق اختيار حقل الباركود في `WP Admin > POS > الإعدادات > عام`

![اختيار الحقل المخصص للباركود في WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "اختيار الحقل المخصص للباركود في WCPOS Pro")

يمكن أيضًا تعيين حقول بيانات وصفية مخصصة للباركود باستخدام المرشح `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * حقل بيانات وصفية مخصص للمنتج للاستخدام كباركود
 *
 * بشكل افتراضي سيستخدم WCPOS حقل SKU - '_sku'
 * يمكنك تغيير مفتاح البيانات الوصفية باستخدام مرشح WordPress التالي
 *
 * أدناه حقول البيانات الوصفية من برامج الباركود الشائعة
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // غيّر إلى مفتاح البيانات الوصفية المخصص
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
يجب أن يكون الباركود فريدًا لكل منتج.
:::
