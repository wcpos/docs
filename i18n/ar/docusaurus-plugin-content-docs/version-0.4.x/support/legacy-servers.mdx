يُعد HTTP (و HTTPS) البروتوكولات المستخدمة من قبل كل متصفح للتحدث مع تطبيقات الويب.
جزء رئيسي من البروتوكول هو [أسلوب HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
تقليديًا، استخدمت تطبيقات الويب مجموعة محدودة للغاية من أساليب HTTP مثل GET (استرداد صفحة ويب) و POST (إرسال نموذج ويب).

تستخدم التطبيقات الحديثة، مثل WCPOS، نهج REST/HTTP مع أساليب HTTP التالية:

- GET
- POST
- PUT
- PATCH
- DELETE

في بعض الحالات النادرة، قد يختار مضيف الويب حظر بعض أساليب HTTP هذه.
إذا كان مضيف الويب الخاص بك يحظر أساليب HTTP الحديثة، ستحتاج إلى تمكين دعم الخوادم القديمة.
