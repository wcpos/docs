---

HTTP (و HTTPS) هي البروتوكولات المستخدمة من قبل كل متصفح للتواصل مع تطبيقات الويب. جزء أساسي من البروتوكول هو [طريقة HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1). تقليديًا، استخدمت تطبيقات الويب عددًا محدودًا جدًا من طرق HTTP مثل GET (استرجاع صفحة ويب) و POST (إرسال نموذج ويب).

تستخدم تطبيقات الويب الحديثة، مثل WCPOS، نهج REST/HTTP مع طرق HTTP التالية:

- GET
- POST
- PUT
- PATCH
- DELETE

في بعض الحالات النادرة، قد يختار مضيف الويب حظر بعض من هذه الطرق. إذا كان مضيف الويب الخاص بك يحظر طرق HTTP RESTful الحديثة، فستحتاج إلى تمكين دعم الخادم القديم.
