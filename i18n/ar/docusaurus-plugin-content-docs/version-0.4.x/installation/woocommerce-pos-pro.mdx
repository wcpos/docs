:::important

[ووكومرس بوس برو](http://wcpos.com/pro) هي إضافة مدفوعة تضيف وظائف إلى [إضافة ووكومرس بوس المجانية](http://wordpress.org/plugins/woocommerce-pos/). 

:::

إذا كنت قد اشتريت ترخيصًا لإضافة ووكومرس بوس برو، يرجى اتباع الخطوات أدناه لتثبيت وتفعيل الإضافة:

1.  انتقل إلى: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  تحت **التنزيلات**، انقر على رابط التنزيل واحفظ الإضافة على سطح المكتب الخاص بك.
3.  ثم انتقل إلى موقعك، سجل الدخول واذهب إلى `WP Admin > الإضافات > إضافة جديدة > تحميل الإضافة`
4.  قم بتحميل ملف zip للإضافة من سطح المكتب الخاص بك وقم بتفعيلها.
5.  بعد ذلك، انتقل إلى `WP Admin > بوس > الإعدادات > الترخيص` وأدخل مفتاح الترخيص وعنوان البريد الإلكتروني لإكمال التفعيل.

![تنزيلاتي](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "يمكنك تنزيل ووكومرس بوس برو على صفحة حسابك")

عندما تكون النسخ الجديدة جاهزة، ستظهر في لوحة التحكم الخاصة بالتحديثات لديك.
للحصول على معلومات حول ما أعمل عليه حاليًا، يرجى متابعة [خارطة الطريق](http://wcpos.com/roadmap/). 
كما يجب متابعة سجل التغييرات الذي سيعرض الميزات الجديدة لكل إصدار، وسيتم تحديث [صفحة الوثائق](http://docs.wcpos.com) بمعلومات حول كيفية استخدام الميزات الجديدة.
