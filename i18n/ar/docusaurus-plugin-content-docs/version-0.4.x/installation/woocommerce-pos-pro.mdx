```mdx
:::important

[WCPOS Pro](http://wcpos.com/pro) هو برنامج مساعد مدفوع يضيف وظائف إلى [البرنامج المساعد المجاني لـ WCPOS](http://wordpress.org/plugins/woocommerce-pos/).

:::

إذا كنت قد اشتريت ترخيصًا لـ WCPOS Pro، يرجى اتباع الخطوات التالية لتثبيت وتفعيل البرنامج المساعد:

1.  اذهب إلى: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  تحت **التنزيلات**، انقر على رابط التنزيل واحفظ البرنامج المساعد على سطح المكتب.
3.  ثم اذهب إلى موقعك، سجل الدخول واذهب إلى `WP Admin > Plugins > Add New > Upload Plugin`
4.  قم بتحميل ملف zip البرنامج المساعد من سطح المكتب وقم بتنشيطه.
5.  بعد ذلك، اذهب إلى `WP Admin > POS > Settings > License` وأدخل مفتاح الترخيص والبريد الإلكتروني للترخيص لإكمال التنشيط.

![التنزيلات الخاصة بي](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "يمكنك تنزيل WCPOS Pro على صفحة حسابك")

عندما تكون الإصدارات الجديدة جاهزة، ستظهر في لوحة التحديثات الخاصة بك.
للحصول على معلومات حول ما أعمل عليه حاليًا، يرجى متابعة [خارطة الطريق](http://wcpos.com/roadmap/).
كما يرجى متابعة سجل التغييرات الذي سيسرد الميزات الجديدة لكل إصدار وستيم تحديث [صفحة الوثائق](http://docs.wcpos.com) بمعلومات حول كيفية استخدام الميزات الجديدة.
```
