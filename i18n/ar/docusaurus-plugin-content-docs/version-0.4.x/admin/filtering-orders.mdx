---
title: تصفية الطلبات حسب مبيعات نقاط البيع أو المبيعات عبر الإنترنت
description: فصل طلبات نقاط البيع والطلبات عبر الإنترنت باستخدام الروابط السريعة في لوحة إدارة WooCommerce.
---

يمكنك فصل طلبات نقاط البيع والطلبات عبر الإنترنت بسرعة في لوحة إدارة WooCommerce باستخدام الروابط السريعة في أعلى اليسار.
**يُرجى ملاحظة:** سيتم تصفية طلبات نقاط البيع فقط بعد الإصدار 0.3.1، أما الطلبات السابقة للإصدار 0.3.1 فستُصنَّف افتراضيًا كطلبات عبر الإنترنت.

![تصفية الطلبات](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

بالإضافة إلى ذلك، إذا أردت تصفية جميع الطلبات المكتملة من منطقة الإدارة للتركيز على الطلبات المعلقة/قيد المعالجة، يمكنك إضافة الكود التالي إلى ملف [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
