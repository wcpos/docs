---
title: تصفية الطلبات حسب نقطة البيع أو المبيعات عبر الإنترنت
description: فصل طلبات نقطة البيع والطلبات عبر الإنترنت باستخدام الارتباطات السريعة في إدارة WooCommerce.
---

يمكنك فصل طلبات نقطة البيع والطلبات عبر الإنترنت بسرعة في إدارة WooCommerce باستخدام الارتباطات السريعة في الأعلى يسار الصفحة.
**يرجى الملاحظة:** سيتم تصفية طلبات نقطة البيع فقط بعد الإصدار 0.3.1، والطلبات قبل الإصدار 0.3.1 ستكون افتراضيًا عبر الإنترنت.

![تصفية الطلبات](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "تصفية الطلبات")

بالإضافة إلى ذلك، إذا كنت ترغب في تصفية جميع الطلبات المكتملة من منطقة الإدارة الخاصة بك حتى تركز على الطلبات المعلقة/قيد المعالجة، يمكنك إضافة الكود التالي إلى ملف [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * تغيير التصفية الافتراضية للطلبات في إدارة WordPress
 * أضف هذا إلى ملف functions.php الخاص بك
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
