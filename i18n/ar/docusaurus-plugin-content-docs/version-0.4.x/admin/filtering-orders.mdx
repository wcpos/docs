---

يمكنك فصل طلبات POS والطلبات عبر الإنترنت بسرعة في إدارة WooCommerce من خلال استخدام الروابط السريعة في الجزء العلوي الأيسر.
**يرجى ملاحظة:** ستقوم هذه الميزة بتصفية طلبات POS فقط بعد الإصدار 0.3.1، الطلبات التي تسبق الإصدار 0.3.1 ستعود افتراضيًا إلى الطلبات عبر الإنترنت.

![تصفية الطلبات](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "تصفية الطلبات")

بالإضافة إلى ذلك، إذا كنت ترغب في تصفية جميع الطلبات المكتملة من منطقة الإدارة الخاصة بك حتى تتمكن من التركيز على الطلبات المعلقة/المعالجة، يمكنك إضافة الكود التالي إلى ملف [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
