title: تصفية الطلبات حسب مبيعات POS أو المبيعات عبر الإنترنت

---

يمكنك فصل طلبات POS والطلبات عبر الإنترنت بسرعة في لوحة إدارة WooCommerce باستخدام الروابط السريعة في أعلى اليسار.
**يُرجى ملاحظة:** سيتم تصفية طلبات POS فقط بعد الإصدار 0.3.1، وستُصنَّف الطلبات السابقة للإصدار 0.3.1 كطلبات عبر الإنترنت افتراضيًا.

![تصفية الطلبات](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "تصفية الطلبات")

بالإضافة إلى ذلك، إذا كنت ترغب في تصفية جميع الطلبات المكتملة من منطقة الإدارة للتركيز على الطلبات المعلّقة/قيد المعالجة، يمكنك إضافة الكود التالي إلى ملف [functions.php](http://codex.wordpress.org/Functions_File_Explained) الخاص بك.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
