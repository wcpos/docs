### العميل الافتراضي لنقاط البيع

ستبدأ الطلبية الجديدة مع العميل الافتراضي لنقاط البيع. الخيارات الشائعة هي:

- **ضيف:** طلبات نقاط البيع ليست مرتبطة بأي معرف مستخدم
- **أمين الصندوق:** طلبات نقاط البيع مرتبطة بحساب المستخدم المسجل الدخول
- **عميل عابر:** طلبات نقاط البيع مرتبطة بحساب مستخدم مخصص (_ملحوظة: يجب إنشاء هذا الحساب_)
