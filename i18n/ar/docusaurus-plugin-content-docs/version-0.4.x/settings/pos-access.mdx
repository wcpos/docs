---

باستخدام إعدادات الوصول لنظام نقطة البيع، يمكن لأصحاب المتاجر منح أو إزالة صلاحيات المستخدمين للنظام.
بشكل افتراضي، يتم تمكين الأدوار **المسؤول** و**مدير المتجر** مع جميع الصلاحيات لاستخدام نظام نقطة البيع.
يُوصى بالحفاظ على الإعدادات الافتراضية ما لم تكن على دراية بالعواقب.
على سبيل المثال، منح صلاحية `publish_shop_orders` للدور `المشتركين` سيكون فكرة _سيئة_ للغاية.

![مثال على إعدادات الوصول لنظام نقطة البيع](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "مثال على إعدادات الوصول لنظام نقطة البيع")

| الصلاحية                                                                | الوصول                                        |
| ----------------------------------------------------------------------- | --------------------------------------------- |
| access_woocommerce_pos                                                  | مطلوب للوصول إلى واجهة نظام نقطة البيع       |
| manage_woocommerce_pos                                                  | مطلوب للوصول إلى إعدادات إدارة نظام نقطة البيع |
| read_private_products                                                   | مطلوب لعرض المنتجات                           |
| edit_product <br /> edit_published_products <br /> edit_others_products | مطلوب لتعديل المنتجات والمتغيرات              |
| read_private_shop_orders                                                | مطلوب لعرض الطلبات والإيصالات                 |
| publish_shop_orders                                                     | مطلوب لإنشاء طلبات جديدة                      |
| list_users                                                              | مطلوب لعرض العملاء                            |
| create_users                                                            | مطلوب لإنشاء عملاء جدد                        |
| edit_users                                                              | مطلوب لتحديث العملاء الحاليين                 |
| read_private_shop_coupons                                               | مطلوب لعرض القسائم                            |

**المصادر:**

- مزيد من المعلومات حول [أدوار وصلاحيات WordPress](https://codex.wordpress.org/Roles_and_Capabilities)
- مزيد من المعلومات حول [أدوار وصلاحيات WooCommerce](http://docs.woothemes.com/document/roles-capabilities/)
- [محرر أدوار المستخدم](https://wordpress.org/plugins/user-role-editor/) - إضافة رائعة لإدارة الأدوار والصلاحيات
- [الأعضاء](https://wordpress.org/plugins/members/) - إضافة أخرى لإدارة الأدوار والصلاحيات
