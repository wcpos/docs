باستخدام إعدادات وصول نظام نقاط البيع، يمكن لمالكي المتاجر منح أو إزالة امتيازات المستخدمين إلى نظام نقاط البيع. 
بشكل افتراضي، يتم تمكين الأدوار **Administrator** و**Shop Manager** مع جميع القدرات لاستخدام نظام نقاط البيع. 
يوصى بالاحتفاظ بالإعدادات الافتراضية ما لم تكن على دراية بالعواقب. 
على سبيل المثال، منح `publish_shop_orders` لدور `Subscribers` سيكون فكرة _سيئة_ جداً.

![مثال على إعدادات وصول نظام نقاط البيع](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "مثال على إعدادات وصول نظام نقاط البيع")

| القدرة                                                                 | الوصول                                    |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | مطلوب للوصول إلى واجهة نظام نقاط البيع       |
| manage_woocommerce_pos                                                  | مطلوب للوصول إلى إعدادات إدارة نظام نقاط البيع |
| read_private_products                                                   | مطلوب لعرض المنتجات                      |
| edit_product <br /> edit_published_products <br /> edit_others_products | مطلوب لتعديل المنتجات والتنوعات         |
| read_private_shop_orders                                                | مطلوب لعرض الطلبات والإيصالات          |
| publish_shop_orders                                                     | مطلوب لإنشاء طلبات جديدة                |
| list_users                                                              | مطلوب لعرض العملاء                       |
| create_users                                                            | مطلوب لإنشاء عملاء جدد                  |
| edit_users                                                              | مطلوب لتحديث العملاء الحاليين           |
| read_private_shop_coupons                                               | مطلوب لعرض القسائم                      |
