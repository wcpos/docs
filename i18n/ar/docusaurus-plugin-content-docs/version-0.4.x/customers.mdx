### الصلاحيات

سيحتاج الصراف إلى صلاحيات _list_users_ و _create_users_ و _edit_users_ لاستخدام جميع ميزات العملاء.

### الموقع المتعدد WordPress

عند استخدام [WordPress multisite](https://codex.wordpress.org/Create_A_Network) هناك بعض [التحققات الإضافية](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) لمنع غير [المسؤولين العامين](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) من إنشاء وتحرير العملاء.

أولاً، لتمكين المسؤولين غير العامين من **إنشاء** عملاء جدد، يرجى تمكين الخيار التالي في `إدارة الشبكة > الإعدادات`.

![خيار الموقع المتعدد لإنشاء مستخدمين جدد](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "خيار الموقع المتعدد لإنشاء مستخدمين جدد")

بالإضافة إلى ذلك، لتمكين المسؤولين غير العامين من **تحرير** تفاصيل العملاء، ستحتاج إلى إضافة صلاحية `manage_network_users` للصراف.

```php title="functions.php"
// قم بتغيير CASHIER_ID،
// وأضف إلى ملف functions.php الخاص بك

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
