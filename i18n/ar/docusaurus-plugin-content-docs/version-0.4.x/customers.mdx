### القدرات {#capabilities}

سيحتاج الأمناء إلى قدرات _list_users_، _create_users_ و _edit_users_ لاستخدام جميع ميزات الزبائن.

### ووردبريس مالتي سايت {#wordpress-multisite}

عند استخدام [ووردبريس مالتي سايت](https://codex.wordpress.org/Create_A_Network)، هناك بعض [التحقق الإضافي](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) لمنع غير [المسؤولين الفائقين](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) من إنشاء وتحرير الزبائن.

أولاً، للسماح لغير المسؤولين الفائقين **بإنشاء** زبائن جدد، يرجى تفعيل الخيار التالي في `Network Admin > Settings`.

![خيار مالتي سايت لإنشاء مستخدمين جدد](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "خيار مالتي سايت لإنشاء مستخدمين جدد")

بالإضافة إلى ذلك، للسماح لغير المسؤولين الفائقين **بتحرير** تفاصيل الزبائن، ستحتاج إلى إضافة قدرة `manage_network_users` إلى الأمين.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
