يعتبر تحسين الأداء أمرًا حيويًا لتوفير تجربة سداد سلسة في WCPOS. تتناول هذه section القضايا الشائعة المتعلقة بالأداء وكيفية حلها.

## القضايا الشائعة المتعلقة بالأداء

تتعلق أكثر الشكاوى الشائعة حول الأداء بما يلي:

- **عملية سداد بطيئة** - يستغرق نموذج الدفع وقتًا طويلاً للتحميل أو معالجة المدفوعات
- **واجهة بطيئة** - تبدو واجهة نقاط البيع غير مستجيبة
- **أوقات مزامنة طويلة** - يستغرق مزامنة المنتجات والطلبات وقتًا طويلاً جدًا

## فئات الأداء

### [أداء الدفع](/support/performance/checkout)
حسن عملية الدفع لمعالجة المدفوعات بشكل أسرع وتوفير تجربة أفضل للعملاء.

### [أداء الخادم](/support/performance/server)
راقب وقم بتحسين خادم WordPress/WooCommerce الخاص بك باستخدام المقاييس المدمجة وتقنيات استكشاف الأخطاء وإصلاحها.

### أداء الشبكة (قريبًا)
استكشف مشكلات الأداء المتعلقة بالشبكة وحسن اتصالات API.
