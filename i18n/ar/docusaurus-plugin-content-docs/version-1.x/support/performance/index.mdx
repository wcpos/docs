تحسين الأداء أمر بالغ الأهمية لتوفير تجربة خروج سلسة في WCPOS. تغطي هذه القسم مشكلات الأداء الشائعة وكيفية حلها.

## مشكلات الأداء الشائعة {#common-performance-issues}

تشير أكثر الشكاوى شيوعًا حول الأداء إلى:

- **عملية خروج بطيئة** - يستغرق نموذج الدفع وقتًا طويلاً للتحميل أو معالجة المدفوعات
- **واجهة غير سلسة** - تشعر واجهة نقطة البيع بأنها غير مستجيبة
- **أوقات مزامنة طويلة** - تستغرق مزامنة المنتجات والطلبات وقتًا طويلاً جدًا

## فئات الأداء {#performance-categories}

### [أداء الخروج](/support/performance/checkout) {#checkout-performancesupportperformancecheckout}
قم بتحسين عملية الخروج لتسريع معالجة المدفوعات وتحسين تجربة العملاء.

### [أداء الخادم](/support/performance/server) {#server-performancesupportperformanceserver}
راقب وقم بتحسين خادم WordPress/WooCommerce الخاص بك باستخدام المقاييس المدمجة وتقنيات استكشاف الأخطاء وإصلاحها.

### أداء الشبكة (قريبًا) {#network-performance-coming-soon}
استكشف مشكلات الأداء المتعلقة بالشبكة و قم بتحسين اتصال API.
