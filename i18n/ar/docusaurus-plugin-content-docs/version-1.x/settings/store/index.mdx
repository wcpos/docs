إعدادات المتجر هي إعدادات عامة للتطبيق تؤثر في طريقة عمل POS. تنطبق هذه الإعدادات على المتجر بالكامل، وليس على الشاشات الفردية.

## الوصول إلى إعدادات المتجر {#accessing-store-settings}

افتح إعدادات المتجر من خلال:

- النقر على **قائمة المستخدم** (الزاوية العلوية اليمنى) واختيار **الإعدادات**
- استخدام اختصار لوحة المفاتيح `Ctrl + Shift + S`

## علامات تبويب الإعدادات {#settings-tabs}

تحتوي نافذة إعدادات المتجر على علامات التبويب التالية:

| علامة التبويب | الوصف |
|-----|-------------|
| [عام](./general) | معلومات المتجر، اللغة، العملة، العميل الافتراضي |
| [الضريبة](./tax) | خيارات حساب الضريبة وعرضها |
| [الرمز الشريطي](./barcode) | تكوين ماسح الرمز الشريطي |
| [اختصارات لوحة المفاتيح](./hotkeys) | اختصارات لوحة المفاتيح المتاحة |
| [السمة](./theme) | المظهر ونظام الألوان |
| [الطابعة](/hardware/printers) | اتصال طابعة الإيصالات وتكوينها |
