تقوم هذه الإعدادات بتكوين وظيفة الخلفية لتطبيق نقاط البيع، بما في ذلك إعدادات PHP وWooCommerce. 
يمكنك الوصول إليها من خلال الانتقال إلى: `WP Admin > POS > Settings`

تشمل الإعدادات الأساسية:

- عميل سلة التسوق الافتراضي لجميع أمين الصندوق.
- حقل رمز المنتج الشريطي.
- إعدادات بريد الطلبات.
- أذونات دور المستخدم.
