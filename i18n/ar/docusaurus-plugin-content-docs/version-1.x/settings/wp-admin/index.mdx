هذه الإعدادات تهيئ وظائف الواجهة الخلفية للتطبيق نقطة البيع، بما في ذلك تهيئات PHP ووووكوميرس. 
يمكنك الوصول إليها عبر: `إدارة وورد بريس > نقطة البيع > الإعدادات`

تشمل التهيئات الرئيسية:

- العميل الافتراضي للعربة لجميع موظفي الكاشير.
- حقل الباركود للمنتج.
- إعدادات بريد الطلبات.
- أذونات دور المستخدم.
