تقوم هذه الإعدادات بتكوين وظائف الخلفية لتطبيق نقطة البيع، بما في ذلك إعدادات PHP وووردبريس.
يمكنك الوصول إليها عن طريق الانتقال إلى: `WP Admin > POS > Settings`

تشمل التكوينات الرئيسية:

- العميل الافتراضي لعربة التسوق لجميع الصرافين.
- حقل رمز المنتج.
- إعدادات بريد الطلبات.
- أذونات دور المستخدم.
