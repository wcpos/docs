تقوم هذه الإعدادات بتكوين الوظائف الخلفية لتطبيق نقاط البيع، بما في ذلك إعدادات PHP وWooCommerce. 
يمكنك الوصول إليها من خلال الانتقال إلى: `WP Admin > POS > Settings`

تشمل الإعدادات الأساسية:

- عميل سلة التسوق الافتراضي لجميع الصرافين.
- حقل رمز الشريط الخاص بالمنتج.
- إعدادات البريد الإلكتروني للطلبات.
- أذونات دور المستخدم.
