يسمح محرر قالب الإيصالات بتخصيص مظهر ومحتوى الإيصالات المطبوعة من نظام نقطة البيع. يمكنك الوصول إلى محرر القالب من خلال: `لوحة تحكم WordPress > نقطة البيع > القوالب`

## نظرة عامة

يوفر نظام القوالب نوعين من القوالب:

- **القوالب الافتراضية**: يتم اكتشافها تلقائيًا من ملفات الإضافة والقالب الخاص بك. لا يمكن حذفها ولكن يمكن نسخها لإنشاء نسخ مخصصة.
- **القوالب المخصصة**: قوالب أنشأها المستخدم يمكنك تخصيصها بالكامل وتحريرها وإدارتها.

## الوصول إلى محرر القالب

1. انتقل إلى لوحة التحكم الخاصة بـ WordPress
2. اذهب إلى **نقطة البيع > القوالب**
3. ستظهر لك قائمة بالقوالب المتاحة مقسمة إلى قسمين:
   - **القوالب الافتراضية** (من ملفات الإضافة/القالب)
   - **القوالب المخصصة** (إبداعاتك المخصصة)

## القوالب الافتراضية

يكتشف النظام تلقائيًا قوالب الإيصالات من ثلاثة مصادر:

### مصادر القوالب

| المصدر           | الوصف                                                | الحالة                                |
| ---------------- | ----------------------------------------------------- | -------------------------------------- |
| **الإضافة**      | القالب الأساسي المضمن مع WooCommerce نقطة البيع       | متاح دائمًا                            |
| **الإضافة المميزة** | قالب محسّن من WooCommerce نقطة البيع المميزة          | متاح مع ترخيص نشط                    |
| **القالب**        | قالب مخصص من القالب النشط الخاص بك                  | متاح إذا تضمن القالب قالبًا            |

### إجراءات القالب

لكل قالب افتراضي، يمكنك:

- **معاينة**: رؤية كيف يبدو القالب باستخدام بيانات نموذجية
- **نسخ**: إنشاء نسخة مخصصة يمكنك تحريرها
- **تنشيط**: تعيينه كقالب نشط لنقطة البيع

## إنشاء قوالب مخصصة

### الطريقة 1: النسخ من القالب الافتراضي

1. في قسم **القوالب الافتراضية**، ابحث عن القالب الذي تريد تخصيصه
2. انقر على زر **نسخ**
3. أدخل اسمًا للقالب المخصص
4. انقر على **نسخ القالب**
5. حرر القالب المنشأ حديثًا

### الطريقة 2: الإنشاء من الصفر

1. انقر على **إضافة قالب جديد** في أعلى الصفحة
2. أدخل اسم القالب
3. حدد **إيصال** كنوع القالب
4. ابدأ بكتابة الكود في المحرر

## ميزات محرر القالب

يوفر محرر القالب:

### محرر الكود

- **إبراز بناء الجملة**: إبراز بناء جملة PHP لقراءة الكود بشكل أفضل
- **أرقام السطور**: تنقل وتصحيح سهل
- **التحرير بملء الشاشة**: بيئة تحرير خالية من التشتيت

### لوحة إعدادات القالب

- **اللغة**: ضبط على PHP للحصول على إبراز بناء الجملة المناسب
- **نوع القالب**: الاختيار بين قوالب الإيصالات أو التقارير
- **الحالة**: إدارة حالة النشر/المسودة
- **الرؤية**: التحكم في رؤية القالب

### إجراءات القالب

- **معاينة**: رؤية كيفية عرض القالب باستخدام بيانات نموذجية
- **تعيين كنشط**: جعل هذا القالب الافتراضي لنقطة البيع
- **تحديث**: حفظ التغييرات
- **نقل إلى سلة المهملات**: حذف القوالب المخصصة

قوالب الإيصالات هي ملفات PHP تنشئ مخرجات HTML. إليك البنية الأساسية:

```php
 * قالب إيصال مخصص
 */

if ( ! defined( 'ABSPATH' ) ) {
    exit; // الخروج إذا تم الوصول مباشرة
}
?>
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
    <meta charset="<?php bloginfo( 'charset' ); ?>">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <style>
        /* أنماط CSS مخصصة الخاصة بك */
        body { font-family: sans-serif; font-size: 14px; }
        .receipt-header { text-align: center; margin-bottom: 20px; }
        /* أضف المزيد من الأنماط حسب الحاجة */
    </style>
</head>
<body>
    <!-- محتوى الإيصال الخاص بك -->
    <div class="receipt-header">
        <h1><?php echo get_bloginfo( 'name' ); ?></h1>
        <p><?php echo get_bloginfo( 'description' ); ?></p>
    </div>

    <!-- تفاصيل الطلب، العناصر، المجاميع، إلخ. -->

</body>
</html>
```
