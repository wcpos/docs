# بوابة SumUp Terminal

تمكّنك بوابة SumUp Terminal من قبول المدفوعات بواسطة بطاقات باستخدام أجهزة قراءة بطاقات SumUp مباشرة ضمن WooCommerce POS. توفر هذه البوابة تكاملًا سلسًا مع نظام معالجة المدفوعات من SumUp للمعاملات الحضورية.

## المميزات

- **تكامل الأجهزة**: قم بتوصيل أجهزة قراءة بطاقات SumUp بنظام POS الخاص بك عبر اتصال الإنترنت
- **معالجة فورية**: معالجة المدفوعات الفورية وإكمال الطلبات
- **الكشف التلقائي**: تكتمل الطلبات تلقائيًا عند الكشف عن الدفع
- **معاملات آمنة**: معالجة المدفوعات متوافقة مع PCI عبر SumUp
- **اقتران سهل**: عملية اقتران بسيطة مع أكواد اقتران
- **SDK قائم على الويب**: يستخدم SDK JavaScript من SumUp لملاءمة تطبيقات الويب

## التثبيت

### الخطوة 1: تحميل وتثبيت

1. انتقل إلى [صفحة إصدارات SumUp Terminal لـ WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. قم بتنزيل أحدث ملف **sumup-terminal-for-woocommerce.zip**
3. في لوحة إدارة WordPress الخاصة بك، انتقل إلى `Plugins > Add New > Upload Plugin`
4. قم بتحميل ملف الـ zip وقم بتفعيل الإضافة

### الخطوة 2: تكوين إعدادات SumUp

1. انتقل إلى `WP Admin > WooCommerce > Settings > Payments`
2. ابحث عن **SumUp Terminal** في قائمة طرق الدفع
3. انقر على **SumUp Terminal** للوصول إلى الإعدادات
4. أدخل **مفتاح API من SumUp** الخاص بك (متوفر من لوحة تحكم تاجر SumUp الخاصة بك)
5. احفظ الإعدادات

:::note
لا تحتاج إلى تفعيل بوابة SumUp Terminal في إعدادات WooCommerce. سيتم تفعيلها خصيصًا لنظام POS في خطوة لاحقة.
:::

### الخطوة 3: اقتران SumUp Terminal الخاص بك

1. في نفس صفحة الإعدادات، حدد موقع قسم **Pair Reader**
2. على جهاز SumUp الخاص بك، انتقل إلى شاشة الاقتران لعرض رمز الاقتران
3. أدخل رمز الاقتران المعروض على جهاز SumUp الخاص بك
4. انقر على **"Pair Reader"** لإقامة الاتصال
5. انتظر لتأكيد اقتران القارئ بنجاح

:::warning مهم
يجب أن يتم اقتران القارئ بنجاح قبل أن تتمكن من معالجة المدفوعات. تأكد من إكمال عملية الاقتران قبل المتابعة.
:::

### الخطوة 4: تمكين في WooCommerce POS

1. انتقل إلى `WP Admin > WooCommerce POS > Settings > Checkout`
2. ابحث عن بوابة **SumUp Terminal** في القائمة
3. قم بتمكين البوابة للاستخدام في نظام POS
4. احفظ إعداداتك

## الاستخدام

### معالجة المدفوعات

1. **إضافة عناصر**: أضف منتجات إلى سلة التسوق الخاصة بك في نظام POS
2. **اختيار البوابة**: اختر "SumUp Terminal" كطريقة للدفع
3. **بدء الدفع**: انقر لبدء دفع جديد على جهاز SumUp الخاص بك
4. **دفع العميل**: يكمل العميل الدفع على جهاز SumUp Terminal
5. **الإكمال التلقائي**: يكتمل الطلب تلقائيًا عند الكشف عن الدفع

### أدوات التحكم في الدفع

عند استخدام بوابة SumUp Terminal، لديك الخيارات التالية:

- **بدء دفع جديد**: بدء طلب دفع على الجهاز المتصل
- **إلغاء الدفع**: إلغاء دفع قيد التنفيذ حاليًا
- **حالة الدفع**: مراقبة الحالة الحالية لعملية الدفع

### إدارة الطلبات

- **التحديثات التلقائية**: يتم وسم الطلبات تلقائيًا على أنها مكتملة بعد الدفع الناجح
- **تتبع الدفع**: يتم تسجيل جميع تفاصيل الدفع في ملاحظات الطلب
- **توليد الإيصالات**: يتم توليد إيصالات POS القياسية بعد المدفوعات الناجحة

## المتطلبات

- **حساب SumUp**: حساب تاجر نشط مع SumUp
- **الوصول إلى API**: مفتاح API من SumUp من لوحة التحكم الخاصة بك
- **الأجهزة المتوافقة**: جهاز قراءة بطاقات SumUp
- **WooCommerce POS**: النسخة المجانية أو الاحترافية مثبتة ومفعلة
- **اتصال مستقر**: اتصال إنترنت موثوق للتواصل عبر API

## توافق الأجهزة

:::important متطلبات الاتصال
يستخدم هذا التنفيذ SDK JavaScript من SumUp، مما يعني أنه يعمل عبر تطبيقات الويب ولكن يتطلب **أجهزة متصلة بالإنترنت فقط**. أجهزة البلوتوث غير مدعومة حاليًا.
:::

### الأجهزة المدعومة (متصلة بالإنترنت)

- **SumUp Solo** ✅ - جهاز متصل عبر WiFi/Ethernet

### الأجهزة غير المدعومة (بلوتوث)

- **SumUp Air** ❌ - بلوتوث فقط
- **SumUp 3G** ❌ - بلوتوث فقط

:::note دعم مستقبلي
سيتم إضافة دعم أجهزة البلوتوث إلى تطبيقات iOS وAndroid في **الربع الأول من عام 2026**. سيمكن هذا دعم جميع أجهزة SumUp المعتمدة بما في ذلك موديلات Air و3G.
:::

## استكشاف الأخطاء وإصلاحها

### المشكلات الشائعة

**القارئ لا يقترن**
- تأكد من أنك تستخدم جهازًا متصلًا بالإنترنت مدعومًا (Solo فقط)
- تحقق من أن جهاز SumUp Solo متصل بشبكة WiFi ومتصل بالإنترنت
- تحقق من أن الجهاز في وضع الاقتران وأن رمز الاقتران تم إدخاله بشكل صحيح
- تأكد من أن حساب SumUp لديك لديه وصول إلى API مفعل
- تأكد من وجود اتصال إنترنت مستقر أثناء الاقتران

**الدفع لا يتم معالجته**
- تحقق من أن القارئ مقترن بشكل صحيح
- تحقق من أن حساب SumUp لديك نشط وفي وضع جيد
- تأكد من أن الجهاز لديه بطارية كافية واتصال
- تأكد من صحة مفتاح API وأنه نشط

**الطلب لا يكتمل**
- تحقق من استقرار اتصال الإنترنت
- تحقق من إعدادات webhook في حساب SumUp لديك
- تأكد من أن الدفع تم بنجاح على جهاز SumUp Terminal
- تحقق من سجلات أخطاء WordPress لمشكلات التواصل مع API

### حدود معدل API

تم تحسين البوابة لتقليل استدعاءات API وتجنب حدود المعدل:
- يتم التحقق من حالة الدفع بكفاءة
- يتم تقليل طلبات API غير الضرورية
- معالجة منطق إعادة المحاولة التلقائية للتعامل مع الفشلات المؤقتة

- تمكين البوابة في إعدادات WooCommerce POS
- سير عمل معالجة الدفع في سلة الدفع لنظام POS
