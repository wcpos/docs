# نموذج البوابة

يوفر نموذج بوابة WooCommerce نقطة بداية لإنشاء بوابات دفع مخصصة لـ WooCommerce POS. يتضمن هذا النموذج جميع التعليمات البرمجية الأساسية والهيكل اللازم لبناء بوابة دفع كاملة الوظائف.

## المميزات

- **نموذج كامل**: هيكل بوابة جاهز للاستخدام مع جميع الأساليب المطلوبة
- **تكامل نقطة البيع**: معد مسبقًا للتوافق مع WooCommerce POS
- **إعداد تلقائي**: تخصيص القالب باستخدام البرنامج النصي
- **أفضل الممارسات**: يتبع معايير البرمجة الخاصة بـ WordPress و WooCommerce
- **قابل للتوسعة**: سهل التعديل والتوسيع لمزودي الدفع المحددين

## البدء

### الخيار 1: توليد القالب التلقائي

يتضمن القالب برنامجًا نصيًا يخصص القالب تلقائيًا لبوابة الدفع الخاصة بك:

1. **استنساخ المستودع**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **تشغيل البرنامج النصي للإعداد**:
   ```bash
   ./create-gateway.sh
   ```

3. **اتبع الموجهات**:
   - أدخل اسم البوابة (مثل "بوابة الدفع الخاصة بي")
   - أدخل معرف البوابة (مثل "دفعي")
   - قدم وصفًا
   - سيقوم البرنامج النصي بإنشاء إضافة مخصصة

### الخيار 2: استخدام القالب يدويًا

إذا كنت تفضل التخصيص اليدوي:

1. **تنزيل القالب**:
   - قم بزيارة [مستودع نموذج البوابة](https://github.com/wcpos/woocommerce-gateway-template)
   - قم بتنزيل الإصدار الأخير أو استنساخ المستودع

2. **تخصيص القالب**:
   - استبدل `{{GATEWAY_NAME}}` باسم عرض البوابة الخاصة بك
   - استبدل `{{GATEWAY_SLUG}}` بالمعرف الفريد للبوابة
   - استبدل `{{GATEWAY_DESCRIPTION}}` بوصف بوابتك

3. **إعادة تسمية الملفات**:
   - أعد تسمية `wcpos-{{GATEWAY_SLUG}}.php` ليتوافق مع معرف البوابة
   - قم بتحديث رؤوس الملفات ومعلومات الإضافة
