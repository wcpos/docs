# بوابة الفاتورة عبر البريد الإلكتروني

تسمح بوابة الفاتورة عبر البريد الإلكتروني بإرسال فواتير الدفع مباشرة للعملاء عبر البريد الإلكتروني من خلال WooCommerce POS. هذا مثالي للحالات التي يفضل فيها العملاء الدفع لاحقًا أو يحتاجون لإكمال الدفع عبر الإنترنت باستخدام طرق الدفع الخاصة بمتجرك الإلكتروني.

## الميزات

- **التكامل البريدي**: إرسال الفواتير مباشرة من واجهة نقطة البيع
- **إدارة العملاء**: ملء البريد الإلكتروني تلقائيًا من بيانات العملاء الموجودين أو إضافة عناوين بريد إلكتروني جديدة
- **طلبات الزوار**: دعم للعملاء الزوار مع إمكانية تحديث عنوان الفاتورة اختياريًا
- **الدفع عبر الإنترنت**: يتلقى العملاء رابط "الدفع لهذا الطلب" لإكمال الدفع عبر الإنترنت
- **سير عمل مرن**: إكمال الطلبات داخل المتجر أو السماح بإكمال الدفع عن بعد

## التثبيت

### الخطوة 1: تنزيل وتثبيت

1. قم بزيارة [صفحة إصدارات بوابة الفاتورة البريدية](https://github.com/wcpos/email-invoice-gateway/releases)
2. قم بتنزيل أحدث ملف **woocommerce-pos-email-invoice-gateway.zip**
3. في لوحة تحكم WordPress الخاصة بك، انتقل إلى `الإضافات > إضافة جديد > تحميل إضافة`
4. قم بتحميل الملف المضغوط وتفعيل الإضافة

### الخطوة 2: التفعيل في WooCommerce POS

1. انتقل إلى `لوحة تحكم WP > WooCommerce POS > الإعدادات > الدفع`
2. ابحث عن **بوابة دفع فواتير WooCommerce POS** في القائمة
3. قم بتفعيل البوابة للاستخدام في نقطة البيع
4. احفظ إعداداتك

:::note
هذه البوابة لا تتطلب مفاتيح API إضافية أو إعداد خدمة خارجية. فهي تستخدم نظام البريد الإلكتروني الخاص بـ WordPress وطرق الدفع في WooCommerce.
:::
