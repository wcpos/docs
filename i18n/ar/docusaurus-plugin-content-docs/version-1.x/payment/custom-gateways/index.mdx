# بوابات الدفع المخصصة

يدعم WCPOS العديد من بوابات الدفع المخصصة التي تمد من وظائف نظام نقاط البيع الخاص بك. تم تصميم هذه البوابات خصيصًا لبيئات نقاط البيع وتوفر تكاملًا سلسًا مع طرق الدفع والخدمات المختلفة.

## بوابات مخصصة متاحة

### بوابات الأجهزة الطرفية

- **[Stripe Terminal](./stripe-terminal)** - قبول المدفوعات باستخدام أجهزة قراءة أجهزة Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - معالجة المدفوعات عبر أجهزة قراءة بطاقة SumUp

### بوابات الدفع الرقمي

- **[Email Invoice](./email-invoice)** - إرسال فواتير الدفع للعملاء عبر البريد الإلكتروني
- **[Web Checkout](./web-checkout)** - إعادة توجيه العملاء لإكمال المدفوعات عبر الإنترنت

## إنشاء بوابتك الخاصة

إذا كنت بحاجة إلى بوابة دفع مخصصة لمتطلباتك المحددة، يمكنك استخدام **[قالب البوابة](./gateway-template)** لإنشاء بوابة الدفع الخاصة بك لـ WooCommerce POS.

## نظرة عامة على التثبيت

تتبع جميع البوابات المخصصة عملية تثبيت مماثلة:

1. **قم بتنزيل** الإصدار الأخير من المستودع المناسب على GitHub
2. **قم بتثبيت** المكون الإضافي عبر `WP Admin > Plugins > Add New > Upload Plugin`
3. **قم بتكوين** إعدادات البوابة في `WP Admin > WooCommerce > Settings > Payments`
4. **قم بتمكين** البوابة في `WP Admin > WooCommerce POS > Settings > Checkout`

## المتطلبات

- مكون إضافي لـ WooCommerce POS (إصدار مجاني أو احترافي)
- WordPress مع تثبيت WooCommerce
- مفاتيح API أو بيانات اعتماد مناسبة للخدمة المحددة للدفع

## الدعم

في حالة حدوث مشكلات مع البوابات المخصصة، يرجى زيارة المستودع المناسب على GitHub وإنشاء تقرير بمعلومات مفصلة حول مشكلتك.
