# بوابات الدفع المخصصة {#custom-payment-gateways}

يدعم WCPOS العديد من بوابات الدفع المخصصة التي تزيد من وظائف نظام نقطة البيع الخاص بك. تم تصميم هذه البوابات خصيصًا لبيئات نقاط البيع وتوفر تكاملاً سلسًا مع طرق الدفع والخدمات المختلفة.

## البوابات المخصصة المتاحة {#available-custom-gateways}

### بوابات الأجهزة الطرفية {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - قبول المدفوعات باستخدام أجهزة قراءة Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - معالجة المدفوعات من خلال أجهزة قراءة بطاقات SumUp

### بوابات الدفع الرقمية {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - إرسال فواتير الدفع للعملاء عبر البريد الإلكتروني
- **[Web Checkout](./web-checkout)** - إعادة توجيه العملاء لإكمال المدفوعات عبر الإنترنت

## إنشاء بوابتك الخاصة {#creating-your-own-gateway}

إذا كنت بحاجة إلى بوابة دفع مخصصة لتلبية متطلباتك الخاصة، يمكنك استخدام **[نموذج البوابة](./gateway-template)** لإنشاء بوابة الدفع الخاصة بك لـ WooCommerce POS.

## نظرة عامة على التثبيت {#installation-overview}

تتبع جميع البوابات المخصصة عملية تثبيت مشابهة:

1. **تحميل** الإصدار الأحدث من مستودع GitHub المعني
2. **تثبيت** المكون الإضافي عبر `WP Admin > Plugins > Add New > Upload Plugin`
3. **تكوين** إعدادات البوابة في `WP Admin > WooCommerce > Settings > Payments`
4. **تمكين** البوابة في `WP Admin > WooCommerce POS > Settings > Checkout`
