# بوابات الدفع المخصصة

يدعم WCPOS العديد من بوابات الدفع المخصصة التي توسع وظائف نظام نقطة البيع الخاص بك. تم تصميم هذه البوابات خصيصًا لبيئات نقطة البيع وتوفر تكاملًا سلسًا مع مختلف طرق وخدمات الدفع.

## البوابات المتاحة

### بوابات الطرفيات المادية

- **[طرفية Stripe](./stripe-terminal)** - قبول المدفوعات باستخدام قارئات طرفية Stripe
- **[طرفية SumUp](./sumup-terminal)** - معالجة المدفوعات من خلال أجهزة قراءة البطاقات الخاصة بـ SumUp

### بوابات الدفع الرقمية

- **[فاتورة البريد الإلكتروني](./email-invoice)** - إرسال فواتير الدفع للعملاء عبر البريد الإلكتروني
- **[الدفع عبر الويب](./web-checkout)** - توجيه العملاء لإكمال الدفع عبر الإنترنت

## إنشاء بوابة خاصة بك

إذا كنت بحاجة إلى بوابة دفع مخصصة لمتطلباتك الخاصة، يمكنك استخدام **[نموذج البوابة](./gateway-template)** لإنشاء بوابة دفع خاصة بنقطة البيع في WooCommerce.

## نظرة عامة على التثبيت

تتبع جميع البوابات المخصصة عملية تثبيت مماثلة:

1. **تنزيل** الإصدار الأحدث من المستودع الخاص GitHub
2. **تثبيت** البرنامج المساعد عبر `WP Admin > Plugins > Add New > Upload Plugin`
3. **تكوين** إعدادات البوابة في `WP Admin > WooCommerce > Settings > Payments`
4. **تفعيل** البوابة في `WP Admin > WooCommerce POS > Settings > Checkout`

## المتطلبات

- برنامج WooCommerce POS (النسخة المجانية أو المدفوعة)
- WordPress مع تثبيت WooCommerce
- مفاتيح API أو بيانات اعتماد مناسبة للخدمة الدفع المحددة

## الدعم

للمشاكل المتعلقة بالبوابات المخصصة، يرجى زيارة المستودع الخاص بـ GitHub وإنشاء مشكلة مع معلومات مفصلة عن المشكلة الخاصة بك.
