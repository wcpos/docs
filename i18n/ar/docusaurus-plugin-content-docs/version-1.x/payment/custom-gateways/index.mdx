# بوابات الدفع المخصصة {#custom-payment-gateways}

تدعم WCPOS العديد من بوابات الدفع المخصصة التي توسع من وظائف نظام نقاط البيع الخاص بك. تم تصميم هذه البوابات خصيصًا لبيئات نقاط البيع وتوفر تكاملًا سلسًا مع طرق الدفع والخدمات المختلفة.

## بوابات مخصصة متاحة {#available-custom-gateways}

### بوابات الأجهزة الطرفية {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - قبول المدفوعات باستخدام أجهزة قراءة Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - معالجة المدفوعات من خلال أجهزة قراءة بطاقات SumUp

### بوابات الدفع عبر الهاتف المحمول {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - قبول المدفوعات القائمة على الهاتف عبر رمز الاستجابة السريعة أو إشعار الدفع

### بوابات الدفع الرقمية

- **[Email Invoice](./email-invoice)** - إرسال فواتير الدفع إلى العملاء عبر البريد الإلكتروني
- **[Web Checkout](./web-checkout)** - إعادة توجيه العملاء لإكمال المدفوعات عبر الإنترنت

## إنشاء بوابتك الخاصة {#creating-your-own-gateway}

إذا كنت بحاجة إلى بوابة دفع مخصصة لمتطلباتك الخاصة، يمكنك استخدام **[نموذج البوابة](./gateway-template)** لإنشاء بوابة الدفع الخاصة بك في WooCommerce POS.

## نظرة عامة على التثبيت {#installation-overview}

تتبع جميع البوابات المخصصة عملية تثبيت مشابهة:

1. **تنزيل** أحدث إصدار من مستودع GitHub الخاص
2. **تثبيت** المكون الإضافي من خلال `WP Admin > المكونات الإضافية > إضافة جديدة > رفع المكون الإضافي`
3. **تكوين** إعدادات البوابة في `WP Admin > WooCommerce > الإعدادات > المدفوعات`
4. **تمكين** البوابة في `WP Admin > WooCommerce POS > الإعدادات > الدفع`
