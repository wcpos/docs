# بوابة الدفع عبر الموقع الإلكتروني

تسمح بوابة الدفع عبر الموقع الإلكتروني للعملاء بإكمال معاملات نقطة البيع (POS) باستخدام نظام الدفع بالموقع الإلكتروني. توفر هذه البوابة جسرًا سلسًا بين نقطة البيع في المتجر والطرق الدفع عبر الإنترنت، مثالية للحالات التي يفضل فيها العملاء الدفع باستخدام أجهزتهم الخاصة أو عندما تحتاج إلى الوصول إلى طرق الدفع المتاحة عبر الموقع فقط.

## المميزات

- **تكامل سلس**: رابط مباشر من نقطة البيع إلى الدفع عبر الموقع الإلكتروني
- **جميع طرق الدفع**: الوصول إلى جميع بوابات الدفع المفعلة في متجرك الإلكتروني
- **تحكم العميل**: يمكن للعملاء الدفع باستخدام أجهزتهم وطرقهم المفضلة
- **مزامنة الطلبات**: تحديثات تلقائية للطلبات بين نقطة البيع والموقع الإلكتروني
- **سير عمل مرن**: إكمال المعاملات عبر الإنترنت مع الحفاظ على سير عمل نقطة البيع

## التثبيت

### الخطوة 1: تنزيل وتثبيت

1. زيارة [صفحة إصدارات بوابة الدفع عبر الموقع الإلكتروني](https://github.com/wcpos/web-checkout-gateway/releases)
2. تنزيل ملف **woocommerce-pos-web-checkout-gateway.zip** الأحدث
3. في لوحة تحكم WordPress الخاصة بك، انتقل إلى `الإضافات > إضافة جديدة > رفع إضافة`
4. قم بتحميل الملف المضغوط وتفعيل الإضافة

### الخطوة 2: التفعيل في WooCommerce POS

1. اذهب إلى `لوحة تحكم WP > WooCommerce POS > الإعدادات > الدفع`
2. ابحث عن **بوابة الدفع عبر الموقع الإلكتروني لـ WooCommerce POS** في القائمة
3. قم بتفعيل البوابة للاستخدام في نقطة البيع
4. احفظ إعداداتك

:::note
تعتمد هذه البوابة على طرق الدفع الموجودة في WooCommerce، لذا تأكد من تكوين بوابة دفع واحدة على الأقل بشكل صحيح (Stripe، PayPal، إلخ).
:::

## الاستخدام

### بدء الدفع عبر الموقع الإلكتروني

1. **إضافة المنتجات**: أضف المنتجات إلى سلة التسوق في نقطة البيع
2. **اختيار البوابة**: اختر "الدفع عبر الموقع الإلكتروني" كطريقة الدفع
3. **إنشاء الرابط**: ينشئ النظام رابط دفع فريد للطلب
4. **وصول العميل**: يقوم العميل بالنقر على الرابط للوصول إلى صفحة الدفع
5. **الدفع عبر الإنترنت**: يكمل العميل الدفع باستخدام طرق الدفع بالموقع الإلكتروني
