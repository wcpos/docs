# بوابات الدفع المخصصة {#custom-payment-gateways}

يدعم WCPOS العديد من بوابات الدفع المخصصة التي توسع من وظائف نظام نقطة البيع الخاص بك. تم تصميم هذه البوابات خصيصًا لبيئات نقاط البيع وتوفر تكاملًا سلسًا مع طرق وخدمات الدفع المختلفة.

## بوابات مخصصة متاحة {#available-custom-gateways}

### بوابات أجهزة الطرفية {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - قبول المدفوعات باستخدام أجهزة قراءة Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - معالجة المدفوعات من خلال أجهزة قراءة بطاقات SumUp

### بوابات الدفع عبر الهاتف المحمول {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - قبول المدفوعات المستندة إلى الهاتف عبر رمز QR أو إشعار دفع

### بوابات الدفع الرقمية {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - إرسال فواتير الدفع للعملاء عبر البريد الإلكتروني
- **[Web Checkout](./web-checkout)** - إعادة توجيه العملاء لإكمال المدفوعات عبر الإنترنت

## إنشاء بوابتك الخاصة {#creating-your-own-gateway}

إذا كنت بحاجة إلى بوابة دفع مخصصة لاحتياجاتك المحددة، يمكنك استخدام **[نموذج البوابة](./gateway-template)** لإنشاء بوابة دفع WCPOS الخاصة بك.

## نظرة عامة على التثبيت {#installation-overview}

تتبع جميع البوابات المخصصة عملية تثبيت مشابهة:

1. **تنزيل** الإصدار الأحدث من مستودع GitHub المعني
2. **تثبيت** الإضافة عبر `WP Admin > Plugins > Add New > Upload Plugin`
3. **تكوين** إعدادات البوابة في `WP Admin > WooCommerce > Settings > Payments`
4. **تفعيل** البوابة في `WP Admin > POS > Settings > Checkout`
