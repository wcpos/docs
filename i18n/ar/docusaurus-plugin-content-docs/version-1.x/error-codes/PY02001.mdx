# PY02001: خطأ في بوابة الدفع

## ما معنى هذا

حدث خطأ أثناء الاتصال ببوابة الدفع. لم يتمكن معالج الدفع من إكمال الطلب.

## الأسباب الشائعة

- **تعطيل البوابة** — تواجه خدمة الدفع مشكلات
- **خطأ في التكوين** — إعدادات البوابة غير صحيحة
- **بيانات اعتماد API** — مفاتيح API غير صالحة أو منتهية
- **مشكلات في الشبكة** — مشاكل في الاتصال بالبوابة

## كيفية الإصلاح

### 1. حاول إعادة المعاملة

أحيانًا تواجه البوابات مشكلات مؤقتة:
- انتظر لحظة
- حاول معالجة الدفع مرة أخرى
- قد يتم حل المشكلة بنفسها

### 2. تحقق من حالة البوابة

قم بزيارة صفحة حالة مزود الدفع الخاص بك:
- سترايب: status.stripe.com
- باي بال: تحقق من مجتمع باي بال أو الحالة
- سكوير: status.squareup.com
- تحقق من الانقطاعات المعروفة

### 3. تحقق من تكوين البوابة

في إدارة WooCommerce:
1. اذهب إلى WooCommerce → الإعدادات → المدفوعات
2. انقر على بوابة الدفع الخاصة بك
3. تحقق من صحة مفاتيح API
4. تأكد من أن وضع الاختبار / الوضع المباشر يتطابق مع مفاتيحك

### 4. تحقق من بيانات اعتماد API

تأكد من أن بيانات الاعتماد صالحة:
- لم تنتهِ صلاحية مفاتيح API
- استخدام مفاتيح مباشرة للمعاملات المباشرة
- استخدام مفاتيح اختبار للمعاملات التجريبية
- تطابق بيانات الاعتماد مع الحساب الصحيح

### 5. اختبر مع بوابة مختلفة

إذا كانت متاحة:
- حاول المعالجة باستخدام بوابة بديلة
- استخدم الدفع النقدي كخيار احتياطي
- أكمل البيع ووافق عليه لاحقًا

## لأصحاب المتاجر

مشكلات التكوين الشائعة:
- خلط بين مفاتيح API للاختبار والمباشر
- مفاتيح API من حسابات مختلفة
- طرق الدفع المعطلة
- بيانات الاعتماد المنتهية أو الملغاة

اتصل بدعم مزود الدفع الخاص بك إذا استمرت المشكلة.
