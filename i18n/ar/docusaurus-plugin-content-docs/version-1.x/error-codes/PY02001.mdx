# PY02001: خطأ في بوابة الدفع

## ما يعنيه هذا

حدث خطأ أثناء التواصل مع بوابة الدفع. لم يتمكن معالج الدفع من إكمال الطلب.

## الأسباب الشائعة

- **توقف البوابة** — تواجه خدمة الدفع مشاكل
- **خطأ في الإعدادات** — إعدادات البوابة غير صحيحة
- **اعتماد API** — مفاتيح API غير صالحة أو منتهية الصلاحية
- **مشاكل الشبكة** — مشاكل في الاتصال بالبوابة

## كيفية الإصلاح

### 1. إعادة محاولة المعاملة

أحيانًا تكون البوابات تواجه مشاكل مؤقتة:
- انتظر لحظة
- حاول معالجة الدفع مرة أخرى
- قد يتم حل المشكلة من تلقاء نفسها

### 2. التحقق من حالة البوابة

قم بزيارة صفحة حالة مزود الدفع الخاص بك:
- Stripe: status.stripe.com
- PayPal: تحقق من مجتمع PayPal أو الحالة
- Square: status.squareup.com
- تحقق من الانقطاعات المعروفة

### 3. التحقق من إعداد البوابة

في إدارة WooCommerce:
1. اذهب إلى WooCommerce → الإعدادات → الدفعات
2. انقر على بوابة الدفع الخاصة بك
3. تحقق من صحة مفاتيح API
4. تأكد من تطابق وضع الاختبار/المباشر مع مفاتيحك

### 4. التحقق من اعتماد API

تأكد من صحة الاعتمادات:
- لم تنته صلاحية مفاتيح API
- استخدم المفاتيح المباشرة للمعاملات المباشرة
- استخدم مفاتيح الاختبار للاختبارات
- تطابق الاعتمادات مع الحساب الصحيح

### 5. اختبار بوابة دفع مختلفة

إذا كان متاحًا:
- حاول معالجة الدفع باستخدام بوابة بديلة
- استخدم الدفع النقدي كحل بديل
- أكمل البيع وقم بالتسوية لاحقًا

## لأصحاب المتاجر

مشاكل الإعداد الشائعة:
- خلط مفاتيح API الاختبارية والمباشرة
- مفاتيح API من حسابات مختلفة
- طرق الدفع المعطلة
- اعتمادات منتهية الصلاحية أو ملغاة

تواصل مع دعم مزود الدفع إذا استمرت المشكلة.
