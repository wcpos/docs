# API06002: رابط API مفقود

## ماذا يعني هذا

لم يتم تكوين أي رابط للاتصال بمتجر WooCommerce الخاص بك. يحتاج نظام نقاط البيع إلى معرفة مكان وجود متجرك.

## الأسباب الشائعة

- **إعداد للمرة الأولى** — لم يتم إدخال الرابط أبداً
- **تم مسح الإعدادات** — تم إعادة تعيين أو مسح الإعدادات
- **تم مسح بيانات التطبيق** — تم حذف بيانات التطبيق
- **مشكلة في الترخيص** — فقد الرابط أثناء التحديث أو الانتقال

## كيفية الإصلاح

### 1. أدخل رابط متجرك

في تطبيق نقاط البيع:
1. افتح الإعدادات أو شاشة تسجيل الدخول
2. ابحث عن حقل تكوين URL / الموقع
3. أدخل رابط موقع WordPress الخاص بك
4. مثال: `https://yourstore.com`

### 2. ابحث عن رابط WordPress الخاص بك

إذا لم تكن متأكداً من رابطك:
1. قم بتسجيل الدخول إلى إدارة WordPress
2. انتقل إلى الإعدادات → عام
3. انظر إلى "عنوان WordPress (URL)"
4. استخدم هذا الرابط في نظام نقاط البيع

### 3. تحقق من الإعدادات المخزنة

إذا تم تكوين الرابط مسبقًا:
- تحقق مما إذا كانت بيانات التطبيق قد تم مسحها عرضياً
- ابحث عن خيارات النسخ الاحتياطي / التصدير للإعدادات
- أعد إدخال الرابط إذا لزم الأمر

### 4. تحقق من أن الرابط صحيح

قبل الحفظ:
1. اختبر الرابط في متصفح
2. تأكد من أنه يفتح موقع WordPress الخاص بك
3. تأكد من تضمين `https://` أو `http://`

## قائمة التحقق من الإعداد

عند تكوين WCPOS للمرة الأولى:

1. ☐ أدخل رابط موقع WordPress الخاص بك
2. ☐ تحقق من أن مكون WCPOS مثبت ومفعل
3. ☐ سجل الدخول باستخدام بيانات اعتماد WordPress الخاصة بك
4. ☐ اسمح بإكمال مزامنة البيانات الأولية
