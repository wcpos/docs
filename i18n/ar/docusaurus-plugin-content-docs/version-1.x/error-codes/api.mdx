# أخطاء API

تحدث أخطاء API عند التواصل مع خادم WooCommerce الخاص بك. يتم تمييز هذه الأخطاء بالبادئة `API` وهي منظمة في الفئات التالية:

## الفئات

| الفئة | نطاق الكود | الوصف |
|----------|------------|-------------|
| [الاتصال](#أخطاء-الاتصال) | API01xxx | مشكلات الشبكة والاتصال |
| [المصادقة](#أخطاء-المصادقة) | API02xxx | مشاكل تسجيل الدخول والرموز والأذونات |
| [الطلب](#أخطاء-الطلب) | API03xxx | مشاكل مع الطلبات الصادرة |
| [الاستجابة](#أخطاء-الاستجابة) | API04xxx | مشاكل مع استجابات الخادم |
| [البرنامج المساعد/WordPress](#أخطاء-البرنامج-المساعد-أو-wordpress) | API05xxx | مشاكل مع WordPress أو البرامج المساعدة لـ WooCommerce |
| [الإعداد](#أخطاء-الإعداد) | API06xxx | مشاكل الإعداد والتكوين |
