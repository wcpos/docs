# أخطاء API

تحدث أخطاء API عند التواصل مع خادم WooCommerce الخاص بك. تُضاف بادئة `API` لهذه الأخطاء وتنظم في الفئات التالية:

## الفئات

| الفئة | نطاق الكود | الوصف |
|----------|------------|-------------|
| [أخطاء الاتصال](#connection-errors) | API01xxx | مشاكل في الشبكة والاتصال |
| [أخطاء المصادقة](#authentication-errors) | API02xxx | مشاكل في تسجيل الدخول، الرموز، والإذن |
| [أخطاء الطلب](#request-errors) | API03xxx | مشاكل في الطلبات الصادرة |
| [أخطاء الاستجابة](#response-errors) | API04xxx | مشاكل في استجابات الخادم |
| [أخطاء الإضافات/ووردبريس](#pluginwordpress-errors) | API05xxx | مشاكل مع إضافات ووردبريس أو WooCommerce |
| [أخطاء التكوين](#configuration-errors) | API06xxx | مشاكل الإعداد والتكوين |
