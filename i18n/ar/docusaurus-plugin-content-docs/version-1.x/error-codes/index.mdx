# رموز الخطأ

عندما يواجه WCPOS مشكلة، فإنه يعرض رمز خطأ يساعد في تحديد المشكلة. يتبع كل رمز خطأ التنسيق `[DOMAIN][CATEGORY][SPECIFIC_CODE]` لمساعدتك على سرعة فهم نوع وطبيعة الخطأ.

## تنسيق رمز الخطأ

| المكون | الوصف | المثال |
|--------|-------|--------|
| **النطاق** | منطقة النظام | `API`، `DB`، `PY`، `SY` |
| **الفئة** | فئة الخطأ | `01`، `02`، `03`، إلخ. |
| **رمز محدد** | المعرف الفريد | `001`، `002`، `003`، إلخ. |

**مثال:** `API04001` = API (النطاق) + 04 (أخطاء الاستجابة) + 001 (تنسيق استجابة غير صالح)

---

## مجالات الخطأ

### [أخطاء API](./api)

تحدث أخطاء API عند التواصل مع خادم WooCommerce الخاص بك.

| الفئة | نطاق الرموز | الوصف |
|-------|------------|-------|
| الاتصال | API01xxx | مشاكل الشبكة والاتصال |
| المصادقة | API02xxx | مشاكل تسجيل الدخول، الرموز، والأذونات |
| الطلب | API03xxx | مشاكل في الطلبات الصادرة |
| الاستجابة | API04xxx | مشاكل في استجابات الخادم |
| المكونات الإضافية/ووردبريس | API05xxx | مشكلات مع المكونات الإضافية لووردبريس أو WooCommerce |
| التكوين | API06xxx | مشاكل الإعداد والتكوين |

### [أخطاء قاعدة البيانات](./db)

تحدث أخطاء قاعدة البيانات عند تخزين أو استرجاع البيانات محليًا.

| الفئة | نطاق الرموز | الوصف |
|-------|------------|-------|
| الاتصال | DB01xxx | مشاكل الاتصال بقاعدة البيانات والمعاملات |
| البيانات | DB02xxx | مشاكل السجلات والقيود |
| الاستعلام | DB03xxx | مشاكل في تركيب الاستعلام ونوع البيانات |

### [أخطاء الدفع](./py)

تحدث أخطاء الدفع أثناء عملية الدفع وعمليات المعالجة.

| الفئة | نطاق الرموز | الوصف |
|-------|------------|-------|
| البطاقة | PY01xxx | مشكلات مع بطاقة الدفع |
| البوابة | PY02xxx | مشاكل في التواصل مع بوابة الدفع |

### [أخطاء النظام](./sy)

تتعلق أخطاء النظام بموارد الجهاز وتكوين النظام.

| الفئة | نطاق الرموز | الوصف |
|-------|------------|-------|
| المورد | SY01xxx | مشاكل الذاكرة، القرص، والأذونات |
| الخدمة | SY02xxx | مشاكل في التكوين وتوفر الخدمة |

## مرجع سريع

هل تبحث عن رمز خطأ محدد؟ استخدم الشريط الجانبي للتنقل إلى النطاق، أو ابحث عن الرمز مباشرة.

| النطاق | إجمالي الرموز | المشاكل الشائعة |
|--------|---------------|-----------------|
| [API](./api) | 39 رمز | مشاكل في التواصل مع الخادم، المصادقة، مشكلات المكونات الإضافية |
| [DB](./db) | 9 رموز | التخزين المحلي واسترجاع البيانات |
| [PY](./py) | 6 رموز | معالجة الدفع |
| [SY](./sy) | 5 رموز | موارد الجهاز والتكوين |

## تحتاج إلى مزيد من المساعدة؟

إذا كنت لا تزال تواجه مشكلات بعد اتباع خطوات حل المشاكل:

1. تحقق من قسم [حل المشكلات](/support/troubleshooting/response-error) للحصول على أدلة أكثر تفصيلاً
2. انضم إلى [مجتمع Discord](https://wcpos.com/discord) الخاص بنا للحصول على الدعم
3. أبلغ عن المشكلات على [GitHub](https://github.com/wcpos)
