# أخطاء قاعدة البيانات

تحدث أخطاء قاعدة البيانات عند تخزين أو استرجاع البيانات في قاعدة البيانات المحلية. يستخدم WCPOS قاعدة بيانات محلية لتخزين المنتجات والعملاء وبيانات أخرى للوصول السريع والوظائف غير المتصلة بالإنترنت. تُسبق هذه الأخطاء بـ `DB`.

## الفئات

| الفئة | نطاق الشيفرة | الوصف |
|----------|------------|-------------|
| [الاتصال](#connection-errors) | DB01xxx | مشكلات الاتصال بقاعدة البيانات والمعاملات |
| [البيانات](#data-errors) | DB02xxx | مشكلات السجلات والقيود |
| [الاستعلام](#query-errors) | DB03xxx | مشكلات بناء جملة الاستعلام ونوع البيانات |

---

## أخطاء الاتصال

مشكلات في الاتصال بقاعدة البيانات المحلية أو العمل عليها.

| الشيفرة | الاسم | الوصف |
|------|------|-------------|
| [DB01001](./DB01001) | فشل الاتصال | تعذر الاتصال بقاعدة البيانات المحلية |
| [DB01002](./DB01002) | انتهاء مهلة الاستعلام | استغرق استعلام قاعدة البيانات وقتاً طويلاً |
| [DB01003](./DB01003) | فشل المعاملة | لم تتمكن معاملة قاعدة البيانات من الاكتمال |

## أخطاء البيانات

مشكلات في السجلات، والتكرارات، والقيود.

| الشيفرة | الاسم | الوصف |
|------|------|-------------|
| [DB02001](./DB02001) | سجل مكرر | يوجد سجل بهذا الـ ID بالفعل |
| [DB02002](./DB02002) | السجل غير موجود | السجل المطلوب غير موجود |
| [DB02003](./DB02003) | انتهاك القيد | البيانات تنتهك قيود قاعدة البيانات |

## أخطاء الاستعلام

مشكلات في بناء جملة الاستعلام وأنواع البيانات.

| الشيفرة | الاسم | الوصف |
|------|------|-------------|
| [DB03001](./DB03001) | خطأ في بناء جملة الاستعلام | يحتوي استعلام قاعدة البيانات على أخطاء في بناء الجملة |
| [DB03002](./DB03002) | نوع بيانات غير صالح | نوع البيانات لا يتطابق مع النوع المتوقع |
| [DB03003](./DB03003) | حقل مطلوب مفقود | الحقل المطلوب فارغ |
