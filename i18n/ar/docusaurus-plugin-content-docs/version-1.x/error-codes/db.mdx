# أخطاء قاعدة البيانات

تحدث أخطاء قاعدة البيانات عند تخزين أو استرجاع البيانات في قاعدة البيانات المحلية. يستخدم WCPOS قاعدة بيانات محلية لتخزين المنتجات والعملاء وبيانات أخرى للوصول السريع والوظائف بدون اتصال. يتم تمييز هذه الأخطاء بالبادئة `DB`.

## الفئات

| الفئة | نطاق الكود | الوصف |
|----------|------------|-------------|
| [الاتصال](#أخطاء-الاتصال) | DB01xxx | مشاكل الاتصال والمعاملات في قاعدة البيانات |
| [البيانات](#أخطاء-البيانات) | DB02xxx | مشاكل السجلات والقيود |
| [الاستعلام](#أخطاء-الاستعلام) | DB03xxx | مشاكل بناء الاستعلام وأنواع البيانات |

---

## أخطاء الاتصال

مشاكل الاتصال بقاعدة البيانات المحلية أو التشغيل عليها.

| الكود | الاسم | الوصف |
|------|------|-------------|
| [DB01001](./DB01001) | فشل الاتصال | تعذر الاتصال بقاعدة البيانات المحلية |
| [DB01002](./DB01002) | انتهاء وقت الاستعلام | استغرق استعلام قاعدة البيانات وقتًا طويلًا |
| [DB01003](./DB01003) | فشل المعاملة | تعذر إكمال معاملة قاعدة البيانات |

## أخطاء البيانات

مشاكل السجلات والتكرارات والقيود.

| الكود | الاسم | الوصف |
|------|------|-------------|
| [DB02001](./DB02001) | سجل مكرر | يوجد بالفعل سجل بهذا المعرف |
| [DB02002](./DB02002) | السجل غير موجود | السجل المطلوب غير موجود |
| [DB02003](./DB02003) | انتهاك القيد | البيانات تنتهك قيود قاعدة البيانات |

## أخطاء الاستعلام

مشاكل بناء الاستعلام وأنواع البيانات.

| الكود | الاسم | الوصف |
|------|------|-------------|
| [DB03001](./DB03001) | خطأ بناء الاستعلام | يحتوي استعلام قاعدة البيانات على أخطاء بنائية |
| [DB03002](./DB03002) | نوع بيانات غير صالح | نوع البيانات لا يتطابق مع النوع المتوقع |
| [DB03003](./DB03003) | الحقل المطلوب مفقود | الحقل المطلوب فارغ |
