# API01004: فشل في حل DNS {#api01004-dns-resolution-failed}

## ما يعنيه ذلك {#what-this-means}

لم يتمكن نظام نقاط البيع من تحويل اسم نطاق موقعك (مثل: `yourstore.com`) إلى عنوان IP. يُعتبر DNS (نظام أسماء النطاقات) مثل دليل الهاتف الخاص بالإنترنت — إذا لم يتمكن من العثور على الإدراج، فلن يتمكن من إنشاء الاتصال.

## الأسباب الشائعة {#common-causes}

- **خطأ مطبعي في اسم النطاق** — قد يحتوي عنوان URL على خطأ إملائي
- **مشكلات في خادم DNS** — خادم DNS الخاص بك لا يستجيب
- **انتهاء صلاحية النطاق** — قد يكون تسجيل النطاق قد انتهى
- **انتشار DNS** — التغييرات الأخيرة في DNS لم يتم انتشارها بعد
- **ذاكرة تخزين DNS المحلية** — معلومات DNS قديمة مخزنة على جهازك

## كيفية الإصلاح {#how-to-fix}

### 1. تحقق من اسم النطاق {#1-verify-the-domain-name}

راجع عنوان URL مرة أخرى للتحقق من الأخطاء المطبعية. حاول الوصول إلى الموقع في متصفح.

### 2. تحقق من حالة النطاق {#2-check-domain-status}

استخدم WHOIS للتحقق من:
- أن النطاق لا يزال مسجلاً
- أنه لم ينتهِ
- أن سجلات DNS مُعدة بشكل صحيح

### 3. مسح ذاكرة تخزين DNS {#3-clear-dns-cache}

**على جهازك:**

macOS:
```bash
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```

Windows:
```bash
ipconfig /flushdns
```

### 4. جرب DNS بديل {#4-try-alternative-dns}

قم بالتبديل إلى خادم DNS عام مثل:
- Google: `8.8.8.8` و `8.8.4.4`
- Cloudflare: `1.1.1.1`

### 5. انتظر انتشار DNS {#5-wait-for-dns-propagation}

إذا قمت مؤخرًا بتغيير إعدادات DNS، انتظر من 24 إلى 48 ساعة للانتشار الكامل.
