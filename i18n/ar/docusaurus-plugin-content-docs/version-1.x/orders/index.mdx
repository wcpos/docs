# الطلبات <span class="badge badge--primary">Pro</span>

:::info ميزة Pro
تتطلب شاشة الطلبات [WCPOS Pro](/getting-started/pro-license).
:::

تتيح لك شاشة الطلبات الوصول إلى سجل الطلبات الخاص بك مباشرة داخل نقطة البيع. عرض الطلبات السابقة، إعادة طباعة الإيصالات، وإدارة تفاصيل الطلب دون الحاجة للتبديل إلى لوحة إدارة WooCommerce.

{/* لقطة شاشة: orders-screen.png */}

## نظرة عامة على الواجهة {#interface-overview}

### البحث والفلاتر {#search-filters}

في أعلى الشاشة:

- **شريط البحث** - ابحث عن الطلبات برقم، اسم العميل، إلخ.
- **مرشح الحالة** - تصفية حسب حالة الطلب (مكتمل، قيد المعالجة، إلخ.)
- **مرشح العميل** - تصفية حسب العميل
- **مرشح الصندوق** - تصفية حسب من عالج الطلب
- **مرشح المتجر** - تصفية حسب المتجر (لإعدادات متعددة المتاجر)
- **نطاق التاريخ** - تصفية حسب التاريخ

تظهر الفلاتر النشطة كشارة زرقاء يمكن إزالتها بالنقر على ×.

### جدول الطلبات {#orders-table}

تظهر المنطقة الرئيسية الطلبات مع:

- **أيقونة الحالة** - مؤشر بصري (✓ مكتمل، 🕐 معلق، 🛒 مفتوح)
- **رقم الطلب** - معرف الطلب الفريد
- **العميل** - اسم العميل (أو "ضيف")
- **عنوان الفاتورة** - معلومات الفاتورة للعميل
- **مؤشر الملاحظات** - فقاعه الكلام عندما يكون للطلب ملاحظات
- **تاريخ الإنشاء** - الطوابع الزمنية بالنسبة للوقت الحاضر ("منذ ساعتين")
- **الصندوق** - من عالج الطلب
- **طريقة الدفع** - نقدي، بطاقة، إلخ.
- **الإجمالي** - المبلغ الإجمالي للطلب
- **الإجراءات** - قائمة ثلاثية النقاط

### الفوتر {#footer}

- عدد الطلبات مع زر المزامنة (<Icon name="arrow-rotate-right" />). **اضغط لفترة طويلة** للحصول على خيار المسح والتحديث

## السمات الرئيسية {#key-features}

### البحث عن الطلبات {#order-lookup}

يمكنك العثور بسرعة على الطلبات باستخدام:

- **بحث برقم الطلب**
- **بحث باسم العميل**
- **تصفية بنطاق التاريخ**
- **تصفية حسب الحالة**
- **تصفية حسب الصندوق**

### إعادة طباعة الإيصالات {#receipt-reprinting}

بالنسبة لأي طلب، يمكنك:

1. النقر على قائمة الثلاث نقاط
2. اختيار خيار الإيصال
3. عرض، طباعة، أو إرسال الإيصال عبر البريد

### تفاصيل الطلب {#order-details}

انقر على طلب لعرض:

- تفاصيل العناصر بالكامل
- معلومات العميل
- تفاصيل الدفع
- ملاحظات الطلب
- بيانات التعريف

| العمود | الوصف |
|--------|-------------|
| **الحالة** | أيقونة حالة الطلب |
| **رقم الطلب** | معرف الطلب الفريد |
| **العميل** | اسم العميل وتفاصيله |
| **عنوان الفاتورة** | معلومات الفاتورة |
| **عنوان الشحن** | معلومات الشحن |
| **ملاحظة العميل** | ملاحظات من العميل |
| **تاريخ الإنشاء** | متى تم تقديم الطلب |
| **تاريخ التعديل** | آخر تعديل |
| **تاريخ الاكتمال** | متى اكتمل الطلب |
| **تاريخ الدفع** | متى تم استلام الدفع |
| **أنشئ عبر** | مصدر الطلب (نقطة البيع، عبر الإنترنت، إلخ.) |
| **الصندوق** | من عالج الطلب |
| **طريقة الدفع** | نوع الدفع المستخدم |
| **الإجمالي** | الإجمالي للطلب |
| **الإيصال** | وصول سريع إلى الإيصال |
| **الإجراءات** | تعديل، عرض، إلخ. |

## إجراءات الطلب {#order-actions}

انقر على قائمة الثلاث نقاط (⋮) للاختيارات:

- **عرض** - رؤية تفاصيل الطلب بالكامل
- **تعديل** - تعديل معلومات الطلب
- **إيصال** - عرض/طباعة/إرسال الإيصال عبر البريد
- **مزامنة** - تحديث الطلب من الخادم
