:::note مكان واحد لكل جهاز
يوجهك مركز الأجهزة إلى إعداد كل جهاز، لكن الخطوات الفعلية موجودة في مكان واحد. يوجد حساب قارئ البطاقات وتدفق الاقتران في صفحة بوابة الدفع الخاصة به؛ ويوجد إعداد فتح درج النقد في صفحة طابعات الإيصالات.
:::

## توافق قارئات البطاقات {#card-reader-compatibility}

تعتمد قارئات البطاقات التي تعمل على **طريقة تشغيل WCPOS** وعلى نوع اتصال القارئ:

- يمكن لـ **تطبيق الويب** تشغيل القارئات التي توفر **SDK للويب (المتصفح)** فقط — وهي عادةً أجهزة طرفية مكتبية متصلة بالإنترنت. القارئات التي تعمل عبر **Bluetooth فقط** (مثل Stripe M2 / WisePad 3 وSumUp Air) وميزة **الدفع باللمس** عبر الهاتف **غير مدعومة** في تطبيق الويب.
- الدعم **مرتبط بكل بوابة** وليس عامًا — تحقق من توافق القارئ مع البوابة التي تخطط لاستخدامها ([Stripe Terminal](/payment/gateways/stripe-terminal)، [SumUp](/payment/gateways/sumup-terminal)، [PayPal Reader](/payment/gateways/paypal-reader)) قبل الشراء.

تعرض كل صفحة بوابة القارئات الدقيقة التي تدعمها.
