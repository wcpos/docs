يعمل WCPOS مع الأجهزة الفعلية التي تستخدمها بالفعل عند نقطة البيع. هذه الصفحة هي نقطة البداية لتوصيل الأجهزة — طابعات الإيصالات، وماسحات الباركود، وقارئات البطاقات، وأدراج النقد. لكل نوع من الأجهزة إما صفحة إعداد خاصة به هنا، أو يشير إلى الصفحة التي توجد فيها خطوات إعداده فعليًا.

:::note موضع واحد لكل جهاز
يوجهك مركز الأجهزة إلى إعداد كل جهاز، لكن الخطوات الفعلية موجودة في موضع واحد. يوجد حساب قارئ البطاقات ومسار إقرانه في صفحة بوابة الدفع الخاصة به؛ ويوجد إعداد فتح درج النقد في صفحة طابعات الإيصالات.
:::
