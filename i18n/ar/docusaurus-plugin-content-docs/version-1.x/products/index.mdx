:::info ميزة احترافية
تتطلب شاشة المنتجات [WCPOS Pro](/getting-started/pro-license). يمكن للمستخدمين المجانيين عرض وإضافة المنتجات إلى سلة التسوق من [لوحة منتجات نقطة البيع](/pos/product-panel)، ولكن لا يمكنهم تعديل المخزون أو الأسعار.
:::

شاشة المنتجات هي واجهة مخصصة لإدارة المخزون. على عكس [لوحة منتجات نقطة البيع](/pos/product-panel) (التي تُستخدم للبيع)، تم تصميم هذه الشاشة لإدارة كتالوج المنتجات الخاص بك.

{/* لقطة شاشة: products-screen.png */}

## شاشة المنتجات مقابل لوحة منتجات نقطة البيع

| الميزة | لوحة منتجات نقطة البيع | شاشة المنتجات |
|--------|----------------------|-----------------|
| **الغرض** | إضافة المنتجات إلى السلة | إدارة المخزون |
| **المخزون** | عرض فقط | تعديل مباشر |
| **الأسعار** | عرض فقط | تعديل مباشر |
| **الإجراءات** | إضافة إلى السلة | تعديل، مزامنة، حذف |
| **متاح في** | مجاني + احترافي | احترافي فقط |

## نظرة عامة على الواجهة

### البحث والفلاتر

في أعلى الشاشة:

- **شريط البحث** - ابحث عن المنتجات بالاسم أو SKU أو الرمز الشريطي
- **أزرار الفلتر** - حالة المخزون، بارز، في العرض، الفئة، العلامة، العلامة التجارية
- **إعدادات العرض** (<Icon name="sliders" />) - تكوين الأعمدة المرئية

### قائمة المنتجات

تظهر المنطقة الرئيسية المنتجات في جدول يمكن التمرير فيه:

- **الصورة** - صورة مصغرة للمنتج
- **اسم المنتج** - مع SKU، الرمز الشريطي، والسمات (اختياري)
- **المخزون** - كمية المخزون القابلة للتعديل مع زر "إدارة"
- **حالة المخزون** - متوفر (أخضر)، غير متوفر (أحمر)، طلب مسبق
- **الفئات** - شارات فئة المنتج
- **الأسعار** - سعر البيع الحالي، السعر العادي، سعر التخفيض (جميعها قابلة للتعديل)
- **الإجراءات** - قائمة ثلاثية النقاط مع تعديل، مزامنة، حذف

### المنتجات المتغيرة

تظهر المنتجات المتغيرة:
- خيارات السمات (اللون، الحجم، إلخ.)
- رابط "توسيع" لعرض جميع الاختلافات
- صفوف اختلاف فردية مع مخزون/أسعار منفصلة

### التذييل

- **حالة الضريبة** - أساس حساب الضريبة الحالي
- **عدد المنتجات** - "عرض X من Y" مع زر مزامنة

## الميزات الرئيسية

### تحرير المخزون بشكل مباشر

انقر مباشرة على كمية المخزون لتعديلها:

1. انقر على رقم المخزون
2. أدخل الكمية الجديدة
3. اضغط Enter للحفظ

تتم مزامنة التغييرات مع WooCommerce تلقائيًا.

### مفتاح إدارة المخزون

يتحكم مفتاح "إدارة" في ما إذا كان WooCommerce يتتبع المخزون لهذا المنتج:

- **تشغيل** - يتم تتبع مستويات المخزون وتخفيضها عند البيع
- **إيقاف** - المنتج متاح دائمًا (مخزون غير محدود)

### تحرير السعر بشكل مباشر

انقر على أي حقل سعر لتعديله:

- **السعر** - سعر البيع الحالي
- **السعر العادي** - السعر غير المخصوم
- **سعر التخفيض** - السعر المخفض (عند الخصم)

### إجراءات المنتج

انقر على قائمة النقاط الثلاث (⋮) لكل منتج:

- **تعديل** - فتح نموذج تعديل المنتج
- **مزامنة** - تحديث هذا المنتج من WooCommerce
- **حذف** - إزالة المنتج من قاعدة البيانات المحلية

:::note
حذف منتج من نقطة البيع يزيله محليًا فقط. يبقى المنتج في WooCommerce وسيظهر مرة أخرى في المزامنة التالية.
:::

### الأعمدة المتاحة

| العمود | الوصف | خيارات العرض |
|--------|-------|--------------|
| **الصورة** | صورة مصغرة للمنتج | - |
| **ID** | معرف المنتج في WooCommerce | - |
| **المنتج** | اسم المنتج | SKU، الرمز الشريطي، السمات |
| **النوع** | بسيط، متغير، مجمع، إلخ. | - |
| **SKU** | وحدة حفظ المخزون | - |
| **الرمز الشريطي** | الرمز الشريطي للمنتج | - |
| **المخزون** | الكمية الحالية (قابلة للتعديل) | - |
| **حالة المخزون** | متوفر / غير متوفر / طلب مسبق | - |
| **الفئات** | فئات المنتج | - |
| **العلامات** | علامات المنتج | - |
| **العلامات التجارية** | معلومات العلامة التجارية (إذا كان المكون النمطي مفعلًا) | - |
| **تكلفة البضائع المباعة** | سعر تكلفة المنتج | - |
| **السعر** | سعر البيع الحالي (قابل للتعديل) | معلومات الضرائب |
| **السعر العادي** | السعر غير المخصوم (قابل للتعديل) | معلومات الضرائب |
| **سعر التخفيض** | السعر المخفض (قابل للتعديل) | معلومات الضرائب |
| **تاريخ الإنشاء** | متى تم إنشاء المنتج | - |
| **تاريخ التعديل** | تاريخ آخر تعديل | - |
| **الإجراءات** | أزرار تعديل، مزامنة، حذف | - |

### استعادة الإعدادات الافتراضية

انقر لإعادة تعيين جميع إعدادات العرض إلى الوضع الافتراضي الأصلي.

## تصفية المنتجات

استخدم أزرار الفلتر لتضييق عرضك:

- **حالة المخزون** - متوفر، غير متوفر، طلب مسبق
- **بارز** - المنتجات المعلمة كمنتجات بارزة
- **في العرض** - المنتجات المتاحة حاليًا في العرض
- **الفئة** - تصفية حسب فئة المنتج
- **العلامة** - تصفية حسب علامة المنتج
- **العلامة التجارية** - تصفية حسب العلامة التجارية (إذا كنت تستخدم مكونًا إضافيًا للعلامة التجارية)

## الفرز

انقر على أي رأس عمود لترتيب ذلك العمود. انقر مرة أخرى لعكس ترتيب الفرز.
