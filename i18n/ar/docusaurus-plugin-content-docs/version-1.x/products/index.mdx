:::info ميزة احترافية
تتطلب شاشة المنتجات [WCPOS Pro](/getting-started/pro-license). يمكن للمستخدمين المجانيين عرض المنتجات وإضافتها إلى السلة من [لوحة منتجات نقاط البيع](/pos/product-panel)، لكن لا يمكنهم تعديل المخزون أو الأسعار.
:::

شاشة المنتجات هي واجهة مخصصة لإدارة المخزون. على عكس [لوحة منتجات نقاط البيع](/pos/product-panel) (التي تُستخدم للبيع)، تم تصميم هذه الشاشة لإدارة كتالوج منتجاتك.

## شاشة المنتجات مقابل لوحة منتجات نقاط البيع

| الميزة | لوحة منتجات نقاط البيع | شاشة المنتجات |
|---------|-------------------|-----------------|
| **الغرض** | إضافة منتجات إلى السلة | إدارة المخزون |
| **المخزون** | عرض فقط | تعديل على الفور |
| **الأسعار** | عرض فقط | تعديل على الفور |
| **الإجراءات** | إضافة إلى السلة | تعديل، مزامنة، حذف |
| **متوفر في** | مجاني + برو | برو فقط |

## نظرة عامة على الواجهة

### البحث والفلاتر

في أعلى الشاشة:

- **شريط البحث** - ابحث عن المنتجات حسب الاسم أو SKU أو الرمز الشريطي
- **أزرار التصفية** - حالة المخزون، مميز، مخفض، فئة، علامة، علامة تجارية
- **إعدادات العرض** (<Icon name="sliders" />) - تكوين الأعمدة المرئية

### قائمة المنتجات

تعرض المنطقة الرئيسية منتجاتك في جدول قابل للتمرير:

- **الصورة** - صورة مصغرة للمنتج
- **اسم المنتج** - مع SKU اختياري، رمز شريطي، سمات
- **المخزون** - كمية المخزون القابلة للتعديل مع زر "إدارة"
- **حالة المخزون** - متوفر (أخضر)، غير متوفر (أحمر)، طلب مسبق
- **الفئات** - شارات فئة المنتج
- **الأسعار** - السعر الحالي، السعر العادي، سعر الخصم (جميعها قابلة للتعديل)
- **الإجراءات** - قائمة ثلاثية النقاط مع تعديل، مزامنة، حذف

### المنتجات المتغيرة

تظهر المنتجات المتغيرة:
- خيارات السمات (اللون، الحجم، إلخ)
- رابط "توسيع" لعرض جميع الاختلافات
- صفوف الاختلافات الفردية مع مخزون/أسعار منفصلة

###footer

- **حالة الضريبة** - أساس حساب الضريبة الحالي
- **عدد المنتجات** - "يظهر X من Y" مع زر المزامنة (<Icon name="arrow-rotate-right" />). **اضغط طويلاً** على زر المزامنة لخيارات مسح وتحديث

## الميزات الرئيسية

### تعديل المخزون على الفور

انقر مباشرة على كمية المخزون لتعديلها:

1. انقر على رقم المخزون
2. ادخل الكمية الجديدة
3. اضغط على Enter للحفظ

تتزامن التغييرات مع WooCommerce تلقائيًا.

### تبديل إدارة المخزون

يحدد زر "إدارة" ما إذا كان WooCommerce يتتبع المخزون لهذا المنتج:

- **تشغيل** - يتم تتبع مستويات المخزون وتقليلها عند البيع
- **إيقاف** - المنتج متاح دائمًا (مخزون غير محدود)

### تعديل السعر على الفور

انقر على أي حقل سعر لتعديله:

- **السعر** - السعر الحالي للبيع
- **السعر العادي** - السعر غير المخفض
- **سعر البيع** - السعر المخفض (عند وجود تخفيض)

### إجراءات المنتج

انقر على قائمة النقاط الثلاث (⋮) لكل منتج:

- **تعديل** - فتح نموذج تعديل المنتج
- **مزامنة** - تحديث هذا المنتج من WooCommerce
- **حذف** - إزالة المنتج من قاعدة البيانات المحلية

:::note
حذف منتج من نظام نقاط البيع يزيله فقط محليًا. يبقى المنتج في WooCommerce وسيظهر مرة أخرى في المزامنة التالية.
:::

### الأعمدة المتاحة

| العمود | الوصف | خيارات العرض |
|--------|-------------|-----------------|
| **الصورة** | صورة مصغرة للمنتج | - |
| **المعرف** | معرّف منتج WooCommerce | - |
| **المنتج** | اسم المنتج | SKU، رمز شريطي، سمات |
| **النوع** | بسيط، متغير، مجمع، إلخ. | - |
| **SKU** | وحدة حفظ المخزون | - |
| **الرمز الشريطي** | رمز شريطي للمنتج | - |
| **المخزون** | الكمية الحالية (قابلة للتعديل) | - |
| **حالة المخزون** | متوفر / غير متوفر / طلب مسبق | - |
| **الفئات** | فئات المنتجات | - |
| **العلامات** | علامات المنتجات | - |
| **العلامات التجارية** | معلومات العلامة التجارية (إذا كانت الإضافة نشطة) | - |
| **تكاليف السلع المباعة** | سعر تكلفة المنتج | - |
| **السعر** | السعر الحالي للبيع (قابل للتعديل) | معلومات الضريبة |

| **سعر البيع** | السعر المخفض (قابل للتعديل) | معلومات الضريبة |
| **تاريخ الإنشاء** | متى تم إنشاء المنتج | - |
| **تاريخ التعديل** | تاريخ آخر تعديل | - |
| **الإجراءات** | أزرار تعديل، مزامنة، حذف | - |

### استعادة إعدادات الافتراضي

انقر لإعادة تعيين جميع إعدادات العرض إلى القيم الافتراضية الأصلية.

## تصفية المنتجات

استخدم أزرار التصفية لتضييق نطاق عرضك:

- **حالة المخزون** - متوفر، غير متوفر، طلب مسبق
- **مميز** - المنتجات المميزة
- **مخفضة** - المنتجات المخفضة حاليًا
- **فئة** - تصفية حسب فئة المنتج
- **علامة** - تصفية حسب علامة المنتج
- **علامة تجارية** - تصفية حسب العلامة التجارية (إذا كنت تستخدم إضافة العلامات التجارية)

## الفرز

انقر على أي رأس عمود لتصنيف حسب ذلك العمود. انقر مرة أخرى لعكس ترتيب الفرز.
