لضمان الأداء السليم والأمثل لإضافة WCPOS، يُرجى التأكد من أن بيئتك تستوفي الحد الأدنى من المتطلبات التالية:

## متطلبات الخادم {#server-requirements}

تعمل إضافة WCPOS على موقع WordPress الخاص بك، والذي يجب أن يستوفي الحد الأدنى التالي:

| المكوّن      | الحد الأدنى للإصدار | ملاحظات         |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | يُوصى بالإصدار 8.x |

يضمن استيفاء هذا الحد الأدنى التوافق والاستقرار مع بيئة WordPress الخاصة بك، والوصول إلى أحدث ميزات WCPOS.

## متطلبات المتصفح {#browser-requirements}

يعمل إصدار الويب من نقطة البيع في أي متصفح حديث. استخدم أحدث إصدار مستقر:

| المتصفح         | الإصدار المدعوم |
| --------------- | ----------------- |
| جوجل كروم   | أحدث إصدار مستقر     |
| موزيلا فايرفوكس | أحدث إصدار مستقر     |
| سفاري          | أحدث إصدار مستقر     |
| مايكروسوفت إيدج  | أحدث إصدار مستقر     |

:::warning تجنّب وضع التصفح الخاص / المتخفي
قد يمنع التصفح الخاص الوصول إلى قاعدة بيانات IndexedDB في المتصفح، التي يستخدمها WCPOS لتخزين البيانات دون اتصال — وقد لا تعمل بعض الميزات بشكل صحيح. يجب أيضًا تفعيل JavaScript.
:::

## متطلبات التطبيق {#app-requirements}

يتوفر WCPOS أيضًا كتطبيق لسطح المكتب والأجهزة المحمولة. كل تطبيق مصمم لأنظمة التشغيل الدنيا التالية:

| المنصة        | الحد الأدنى لنظام التشغيل                | البنيات المعمارية              |
| --------------- | ------------------------- | -------------------------- |
| سطح مكتب Windows | Windows 10                | x64, ARM64                 |
| سطح مكتب macOS   | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Android         | Android 10 (API level 29) | —                          |

:::note أذونات الطباعة
تتطلب الطباعة عبر البلوتوث والشبكة منح التطبيق أذونات البلوتوث والشبكة المحلية ذات الصلة عند ظهور الطلب.
:::

## تثبيت الإضافة {#installing-the-plugin}

1. انتقل إلى `WP Admin > إضافات > Add Plugin`
2. ابحث عن "wcpos"
3. انقر على **التثبيت الآن**، ثم **تفعيل**

بدلاً من ذلك، يمكنك تنزيل الإضافة من [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) ورفعها يدوياً.

## الوصول إلى نقطة البيع {#accessing-the-pos}

بعد التفعيل، يمكن الوصول إلى نقطة البيع عبر:

- **الويب:** `yourdomain.com/pos`
- **سطح المكتب:** [Windows](https://updates.wcpos.com/electron/download/win32-x64)، [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)، [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **الأجهزة المحمولة:** [iOS](https://testflight.apple.com/join/JGBdVRrW)، [Android](https://play.google.com/apps/testing/com.wcpos.main)

من خلال استيفاء الحد الأدنى من المتطلبات واستخدام متصفح ويب متوافق، يمكن الحصول على تجربة سلسة في استخدام إضافة WCPOS والاستفادة من كامل إمكانياتها لإدارة عمليات نقطة البيع بكفاءة.

في حال وجود أي استفسارات أو الحاجة إلى مساعدة، يُرجى التواصل مع فريق الدعم عبر نشرها في [محادثة Discord](https://wcpos.com/discord) أو إرسالها بالبريد الإلكتروني إلى support@wcpos.com.
