## ما هو WCPOS؟

WCPOS هو نظام نقطة البيع (POS) لـ WooCommerce. يتيح لك بيع المنتجات من متجر WooCommerce الخاص في الموقع الفعلي، باستخدام متصفح الويب أو تطبيق مخصص.

**متوفر على:**

- **الويب:** الوصول الفوري على `اسم-المجال.com/pos`
- **سطح المكتب:** التنزيل لـ Windows وmacOS وLinux من [wcpos.com](https://wcpos.com)
- **الجوال:** تطبيقات iOS وAndroid (قريبًا)

## ابدأ في 5 دقائق

### 1. تثبيت الإضافة

قم بتثبيت [إضافة WCPOS](https://wordpress.org/plugins/woocommerce-pos/) المجانية على موقع WordPress الخاص بك:

1. اذهب إلى `لوحة تحكم WP > الإضافات > إضافة جديدة`
2. ابحث عن "WooCommerce POS"
3. انقر على **تثبيت الآن**، ثم **تفعيل**

راجع [التثبيت](/getting-started/installation) للمتطلبات التفصيلية.

### 2. الوصول إلى نقطة البيع

**مستخدمو الويب:**
انتقل إلى `اسم-المجال.com/pos` في متصفحك. سيُطلب منك تسجيل الدخول باستخدام بيانات اعتماد WordPress الخاصة بك.

**مستخدمو سطح المكتب/الجوال:**
1. قم بتنزيل التطبيق من [wcpos.com](https://wcpos.com)
2. أدخل عنوان متجرك واضغط على **اتصال**
3. سجل الدخول باستخدام بيانات اعتماد WordPress الخاصة بك

راجع [الاتصال](/getting-started/connect) للتفاصيل حول إدارة المتاجر والمستخدمين المتعددين.

### 3. انتظر المزامنة الأولية

يقوم WCPOS بتنزيل المنتجات على دفعات صغيرة (حوالي 50 في كل مرة) لتجنب إرهاق الخادم. يتم تنزيل المنتجات تدريجيًا أثناء البحث أو التمرير أو التصفية - لا تحتاج إلى الانتظار حتى تتم مزامنة كل شيء قبل البدء في البيع. راجع [مزامنة المنتجات](/products/sync) للتفاصيل.

### 4. ابدأ البيع

- ابحث أو امسح المنتجات لإضافتها إلى عربة التسوق
- اختر عميلًا أو استخدم "زائر"
- انقر على **الدفع** لمعالجة الدفع
- اطبع أو أرسل إيصالًا عبر البريد الإلكتروني

## الميزات الأساسية

- **بحث محلي سريع** - يتم تخزين المنتجات محليًا للحصول على نتائج فورية
- **تصفح بدون اتصال** - البحث عن المنتجات والعملاء بدون إنترنت (يتطلب إكمال الطلبات اتصالًا)
- **مسح الباركود** - استخدم أي ماسح ضوئي للباركود عبر USB أو Bluetooth
- **طرق دفع متعددة** - نقدًا، بطاقة، وبوابات دفع مخصصة
- **طباعة الإيصالات** - طباعة أو إرسال إيصالات قابلة للتخصيص عبر البريد الإلكتروني
- **دعم متاجر متعددة** - الاتصال بمتاجر WooCommerce متعددة

## ميزات النسخة المتقدمة

تضيف [WCPOS Pro](https://wcpos.com/pro) قدرات إضافية:

- **إدارة المنتجات** - تحرير المخزون والأسعار مباشرة في نقطة البيع
- **سجل الطلبات** - عرض وإدارة الطلبات السابقة
- **إدارة العملاء** - إضافة وتحرير تفاصيل العملاء
- **تقارير المبيعات** - تقارير نهاية اليوم مع خيار الطباعة
- **بوابات دفع إضافية** - Stripe Terminal، SumUp، والمزيد
