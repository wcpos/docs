주문 화면에서는 POS 내에서 직접 주문 기록에 접근할 수 있습니다. 과거의 주문을 조회하고, 영수증을 재인쇄하거나, WooCommerce 관리자에 전환하지 않고도 주문 세부 사항을 관리할 수 있습니다.

{/* 스크린샷: orders-screen.png */}

## 인터페이스 개요

### 검색 및 필터

화면 상단에:

- **검색 바** - 주문 번호, 고객 이름 등으로 주문 찾기
- **상태 필터** - 주문 상태로 필터링 (완료, 처리 중 등)
- **고객 필터** - 고객으로 필터링
- **캐셔 필터** - 주문을 처리한 사람으로 필터링
- **상점 필터** - 상점으로 필터링 (다중 상점 설정의 경우)
- **날짜 범위** - 날짜로 필터링

활성 필터는 파란색 배지로 표시되며, ×를 클릭하여 제거할 수 있습니다.

### 주문 테이블

주요 영역은 다음과 함께 주문을 표시합니다:

- **상태 아이콘** - 시각적 표시기 (✓ 완료, 🕐 보류 중, 🛒 열림)
- **주문 번호** - 고유 주문 ID
- **고객** - 고객 이름 (또는 "게스트")
- **청구 주소** - 고객 청구 정보
- **메모 표시기** - 주문에 메모가 있을 때의 말풍선
- **생성된 날짜** - 상대적 타임스탬프 ("2시간 전")
- **캐셔** - 주문을 처리한 사람
- **결제 방법** - 현금, 카드 등
- **총액** - 주문 총액
- **작업** - 세 개의 점 메뉴

### 바닥글

- 동기화 버튼이 있는 주문 수 (<Icon name="arrow-rotate-right" />). **길게 누르기**로 Clear 및 Refresh 옵션

## 주요 기능

### 주문 조회

다음 방법으로 주문을 빠르게 찾을 수 있습니다:

- **주문 번호 검색**
- **고객 이름 검색**
- **날짜 범위 필터링**
- **상태 필터링**
- **캐셔 필터링**

### 영수증 재인쇄

모든 주문에 대해 다음을 수행할 수 있습니다:

1. 세 개의 점 메뉴를 클릭합니다.
2. 영수증 옵션을 선택합니다.
3. 영수증을 보고, 인쇄하거나 이메일로 보냅니다.

### 주문 세부사항

주문을 클릭하여 다음을 확인합니다:

- 완전한 항목 세부사항
- 고객 정보
- 결제 세부 정보
- 주문 메모
- 메타 데이터

## 표시 설정

슬라이더 아이콘(<Icon name="sliders" />)을 클릭하여 표시할 열을 사용자 정의할 수 있습니다.

### 사용 가능한 열

| 열 | 설명 |
|--------|-------------|
| **상태** | 주문 상태 아이콘 |
| **주문 번호** | 고유 주문 ID |
| **고객** | 고객 이름 및 세부정보 |
| **청구 주소** | 청구 정보 |
| **배송 주소** | 배송 정보 |
| **고객 메모** | 고객의 메모 |
| **생성된 날짜** | 주문이 이루어진 날짜 |
| **수정된 날짜** | 마지막 수정 |
| **완료된 날짜** | 주문 완료 날짜 |
| **결제 날** | 결제가 이루어진 날짜 |
| **생성 경로** | 주문 소스 (POS, 온라인 등) |
| **캐셔** | 주문을 처리한 사람 |
| **결제 방법** | 사용된 결제 유형 |
| **총액** | 주문 총액 |
| **영수증** | 빠른 영수증 접근 |
| **작업** | 편집, 보기 등 |

## 주문 작업

세 개의 점 메뉴(⋮)를 클릭하여 옵션을 선택합니다:

- **보기** - 전체 주문 세부사항 보기
- **편집** - 주문 정보 수정
- **영수증** - 영수증 보기/인쇄/이메일
- **동기화** - 서버에서 주문 새로 고침

## 사용 사례

### 고객 서비스

- 반품 또는 문의를 지원하기 위해 과거 주문 조회
- 고객이 구입한 항목 확인
- 요청 시 영수증 재인쇄

### 일일 마감

- 하루 동안의 모든 주문 검토
- 캐셔별로 필터링하여 개인 총액 확인
- [보고서](/reports)와 교차 확인하여 조정

### 주문 추적

- 주문 상태 모니터링
- 보류 중인 주문과 완료된 주문 추적
- 주의가 필요한 주문 확인
