주문 화면에서 POS 내에서 직접 주문 이력에 접근할 수 있습니다. WooCommerce 관리자로 전환하지 않고도 과거 주문을 보고, 영수증을 다시 인쇄하며, 주문 세부 정보를 관리할 수 있습니다.

{/* 스크린샷: orders-screen.png */}

## 인터페이스 개요

### 검색 & 필터

화면 상단에서:

- **검색 창** - 주문 번호, 고객 이름 등으로 주문 찾기
- **상태 필터** - 주문 상태로 필터링 (완료, 처리 중 등)
- **고객 필터** - 고객별 필터링
- **계산원 필터** - 주문을 처리한 사람별 필터링
- **매장 필터** - 매장별 필터링 (다중 매장 설정의 경우)
- **날짜 범위** - 날짜별 필터링

활성 필터는 파란색 배지로 표시되며 × 버튼을 클릭하여 제거할 수 있습니다.

### 주문 테이블

주요 영역에는 다음과 같은 주문 정보가 표시됩니다:

- **상태 아이콘** - 시각적 표시기 (✓ 완료, 🕐 대기 중, 🛒 열림)
- **주문 번호** - 고유 주문 ID
- **고객** - 고객 이름 (또는 "손님")
- **청구 주소** - 고객 청구 정보
- **메모 표시기** - 주문에 메모가 있을 때 말풍선
- **생성 날짜** - 상대 시간 표시 ("2시간 전")
- **계산원** - 주문을 처리한 사람
- **결제 방법** - 현금, 카드 등
- **총액** - 주문 총 금액
- **작업** - 3점 메뉴

### 하단

- 동기화 버튼이 있는 제품 수

## 주요 기능

### 주문 조회

다음을 사용하여 주문을 빠르게 찾기:

- **주문 번호 검색**
- **고객 이름 검색**
- **날짜 범위 필터링**
- **상태 필터링**
- **계산원 필터링**

### 영수증 재인쇄

모든 주문에 대해 다음과 같이 할 수 있습니다:

1. 3점 메뉴 클릭
2. 영수증 옵션 선택
3. 영수증 보기, 인쇄 또는 이메일 발송

### 주문 상세 정보

주문을 클릭하면 다음을 볼 수 있습니다:

- 전체 품목 세부 정보
- 고객 정보
- 결제 세부 정보
- 주문 메모
- 메타 데이터

| **작업** | 편집, 보기 등 |

## 주문 작업

3점 메뉴(⋮)를 클릭하여 다음 옵션 선택:

- **보기** - 전체 주문 세부 정보 확인
- **편집** - 주문 정보 수정
- **영수증** - 영수증 보기/인쇄/이메일 발송
- **동기화** - 서버에서 주문 새로고침

## 사용 사례

### 고객 서비스

- 반품 또는 문의에 대해 과거 주문 조회
- 고객이 구매한 내용 확인
- 요청 시 영수증 재인쇄

### 하루 마감

- 하루 동안의 모든 주문 검토
- 계산원별로 필터링하여 개인 총액 확인
- [보고서](/reports)와 대조하여 정산

### 주문 추적

- 주문 상태 모니터링
- 대기 중 vs 완료된 주문 식별
- 주의가 필요한 주문 확인
