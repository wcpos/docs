주문 화면은 직접적으로 POS 내에서 주문 내역에 접근합니다. 지난 주문을 보고, 영수증을 재인쇄하고, WooCommerce 관리자 화면으로 전환하지 않고 주문 세부 정보를 관리하세요.

{/* 스크린샷: orders-screen.png */}

## 인터페이스 개요

### 검색 및 필터

화면 상단에서:

- **검색창** - 주문 번호, 고객 이름 등으로 주문 조회
- **상태 필터** - 주문 상태별 필터 (완료, 처리 중 등)
- **고객 필터** - 고객별 필터
- **점원 필터** - 주문을 처리한 점원에 따라 필터
- **매장 필터** - 매장별 필터 (다중 매장 설정 시)
- **날짜 범위** - 날짜별 필터

활성화된 필터는 클릭하여 제거할 수 있는 파란색 배지로 표시됩니다.

### 주문 테이블

주요 영역은 다음과 같은 주문을 표시합니다:

- **상태 아이콘** - 시각적 표시자 (✓ 완료, 🕐 보류 중, 🛒 열림)
- **주문 번호** - 고유 주문 ID
- **고객** - 고객 이름 (또는 "게스트")
- **청구 주소** - 고객 청구 정보
- **메모 표시** - 주문에 메모가 있을 때 말풍선
- **생성일** - 상대적 타임스탬프 ("2시간 전")
- **점원** - 주문을 처리한 사람
- **결제 방법** - 현금, 카드 등
- **총액** - 주문 총액
- **작업** - 세 점짜리 메뉴

### 바닥글

- 동기화 버튼과 함께 제품 수

## 주요 기능

### 주문 조회

빠르게 주문을 찾는 방법:

- **주문 번호 검색**
- **고객 이름 검색**
- **날짜 범위 필터링**
- **상태 필터링**
- **점원 필터링**

### 영수증 재인쇄

모든 주문에 대해 다음과 같이 할 수 있습니다:

1. 세 점짜리 메뉴 클릭
2. 영수증 옵션 선택
3. 영수증 보기, 인쇄 또는 이메일 전송

### 주문 세부정보

주문을 클릭하여 볼 수 있습니다:

- 전체 품목 세부 정보
- 고객 정보
- 결제 세부 정보
- 주문 메모
- 메타 데이터

## 표시 설정

슬라이더 아이콘 (<Icon name="sliders" />) 클릭하여 표시할 열을 사용자 정의합니다.

## 주문 작업

세 점짜리 메뉴 (⋮) 클릭하여 옵션을 선택합니다:

- **보기** - 전체 주문 세부정보 보기
- **수정** - 주문 정보 수정
- **영수증** - 영수증 보기/인쇄/이메일 전송
- **동기화** - 서버에서 주문 새로 고침

## 사용 사례

### 고객 서비스

- 반품이나 문의를 돕기 위해 이전 주문 조회
- 고객이 구매한 내역 확인
- 요청 시 영수증 재인쇄

### 하루의 마감

- 하루 동안의 모든 주문 검토
- 점원별로 필터링하여 개별 총액 검증
- 조정을 위해 [보고서](/reports)와 교차 확인

### 주문 추적

- 주문 상태 모니터링
- 보류 중인 주문과 완료된 주문 추적
- 주의가 필요한 주문 식별
