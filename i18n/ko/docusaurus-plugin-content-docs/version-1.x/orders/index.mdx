:::info 전문 기능
주문 화면은 [WCPOS Pro](/getting-started/pro-license)를 요구합니다.
:::

주문 화면은 POS 내에서 직접 주문 히스토리에 접근할 수 있도록 합니다. 과거 주문을 보고, 영수증을 재출력하고, WooCommerce 관리자로 전환하지 않고도 주문 세부사항을 관리할 수 있습니다.

{/* 스크린샷: orders-screen.png */}

## 인터페이스 개요

### 검색 및 필터

화면 상단:

- **검색 바** - 주문 번호, 고객 이름 등으로 주문을 찾습니다.
- **상태 필터** - 주문 상태로 필터링 (완료, 처리 중 등)
- **고객 필터** - 고객으로 필터링
- **캐셔 필터** - 주문을 처리한 사람으로 필터링
- **상점 필터** - 상점으로 필터링 (다중 상점 설정의 경우)
- **날짜 범위** - 날짜로 필터링

활성 필터는 클릭하여 제거할 수 있는 파란색 배지로 표시됩니다.

### 주문 테이블

주요 영역에서는 다음과 같은 주문을 표시합니다:

- **상태 아이콘** - 시각적 표시기 (✓ 완료, 🕐 보류 중, 🛒 열려 있음)
- **주문 번호** - 고유 주문 ID
- **고객** - 고객 이름 (또는 "손님")
- **청구지 주소** - 고객 청구 정보
- **메모 표시기** - 주문에 메모가 있을 때의 말풍선
- **생성일** - 상대적 타임스탬프 ("2시간 전")
- **캐셔** - 주문을 처리한 사람
- **결제 방법** - 현금, 카드 등
- **합계** - 주문 총액
- **작업** - 점3개 메뉴

### 바닥글

- 동기화 버튼이 있는 주문 수 (<Icon name="arrow-rotate-right" />). **길게 누르기**로 지우기 및 새로고침 옵션

## 주요 기능

### 주문 조회

다음 방법으로 주문을 빠르게 찾을 수 있습니다:

- **주문 번호 검색**
- **고객 이름 검색**
- **날짜 범위 필터링**
- **상태 필터링**
- **캐셔 필터링**

### 영수증 재출력

어떤 주문이든 다음을 수행할 수 있습니다:

1. 점3개 메뉴를 클릭합니다.
2. 영수증 옵션을 선택합니다.
3. 영수증을 보고, 인쇄하거나 이메일로 전송합니다.

### 주문 세부정보

주문을 클릭하여 다음을 볼 수 있습니다:

- 완전한 품목 세부정보
- 고객 정보
- 결제 세부정보
- 주문 메모
- 메타 데이터

## 표시 설정

슬라이더 아이콘(<Icon name="sliders" />)을 클릭하여 표시할 열을 사용자 정의할 수 있습니다.

### 사용 가능한 열

| 열 | 설명 |
|--------|-------------|
| **상태** | 주문 상태 아이콘 |
| **주문 번호** | 고유 주문 ID |
| **고객** | 고객 이름 및 세부정보 |
| **청구지 주소** | 청구 정보 |
| **배송 주소** | 배송 정보 |
| **고객 메모** | 고객의 메모 |
| **생성일** | 주문이 접수된 일자 |
| **수정일** | 마지막 수정일 |
| **완료일** | 주문이 완료된 일자 |
| **지급일** | 결제가 접수된 일자 |
| **생성 경로** | 주문 출처 (POS, 온라인 등) |
| **캐셔** | 주문을 처리한 사람 |
| **결제 방법** | 사용된 결제 유형 |
| **합계** | 주문 총액 |
| **영수증** | 빠른 영수증 접근 |
| **작업** | 편집, 보기 등 |

## 주문 작업

점3개 메뉴(⋮)를 클릭하여 옵션을 선택합니다:

- **보기** - 전체 주문 세부정보 보기
- **편집** - 주문 정보 수정
- **영수증** - 영수증 보기/인쇄/이메일 전송
- **동기화** - 서버에서 주문 새로 고침
