# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/)으로 생성한 매장 크레딧에 WCPOS 호환성을 추가합니다. POS 주문에서 StoreApps 매장 크레딧을 사용할 수 있고, StoreApps 잔액을 동기화 상태로 유지하며, 직원 확인용 주문 메모 감사 추적을 남깁니다.

이 확장 기능은 StoreApps / WooCommerce.com Smart Coupons 플러그인 전용입니다. “Smart Coupons”와 유사한 이름의 다른 플러그인은 별도 제품이며, 별도 통합이 필요할 수 있습니다.

:::info Pro 기능
디렉터리에서 확장 기능을 설치하고 관리하려면 [WCPOS Pro](/getting-started/pro-license)가 필요합니다.
:::

## 기능 {#what-it-does}

- 일반 결제 장바구니 컨텍스트를 사용할 수 없을 때 POS에서 생성한 주문에 StoreApps `smart_coupons_contribution` 메타데이터를 기록합니다.
- POS 결제 후 StoreApps가 쿠폰 잔액에서 정확한 부분 매장 크레딧 금액을 차감할 수 있게 합니다.
- POS 주문이 WooCommerce의 일반 주문 라이프사이클을 통해 취소되거나 환불되면 StoreApps 스토어 크레딧 잔액을 복원합니다.
- 사용 후 쿠폰 코드, 사용 금액, 현재 잔액을 표시하는 비공개 주문 메모를 추가합니다.
- 기존 영수증 할인 행에서 쿠폰 설명/할인 라벨에 남은 스토어 크레딧 잔액을 덧붙여 표시합니다.

이 확장 기능은 일반 WooCommerce 쿠폰 필드와 메타데이터를 사용합니다. 사용자 지정 WCPOS 쿠폰 응답 필드를 추가하거나 사용자 지정 영수증 템플릿을 요구하지 않습니다.

## 설치 {#installation}

<Steps>
  <StepCard number="1" title="StoreApps Smart Coupons 설치">

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/)을 설치하고 활성화합니다. WooCommerce에서 평소처럼 스토어 크레딧 쿠폰이나 기프트 카드를 생성합니다.

  </StepCard>
  <StepCard number="2" title="WCPOS StoreApps Smart Coupons 설치">

다음 위치에서 설치합니다: `WP Admin > POS > 설정 > 확장`, 또는 [GitHub 릴리스 페이지](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases)에서 최신 릴리스를 다운로드한 뒤 다음 위치를 통해 업로드합니다: `플러그인 > 새로 추가 > 플러그인 업로드`.

  </StepCard>
  <StepCard number="3" title="쿠폰을 POS에 동기화" isLast>

POS를 열고 쿠폰을 동기화합니다. 이후 계산원은 일반 **쿠폰 추가** 흐름에서 StoreApps 스토어 크레딧 쿠폰을 적용할 수 있습니다.

  </StepCard>
</Steps>

## 영수증 출력 {#receipt-output}

WCPOS 영수증 템플릿은 이미 할인 행 레이블을 사용해 쿠폰 할인을 출력합니다. 영수증을 렌더링하는 동안 이 확장 기능은 기존 레이블에 현재 StoreApps 잔액을 추가합니다.

예를 들어 쿠폰 설명이 **기프트 카드**인 경우 다음과 같이 출력될 수 있습니다.

```text
Gift card — Store credit balance: £61.50
```

쿠폰에 설명이 없으면 잔액 텍스트가 할인 레이블로 사용됩니다. 기존 영수증 템플릿은 변경할 필요가 없습니다.

## 주문 감사 추적 {#order-audit-trail}

StoreApps가 POS 매장 크레딧 사용을 처리한 후, 이 확장 기능은 다음과 유사한 비공개 주문 메모를 추가합니다.

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

이 메모를 사용해 어떤 매장 크레딧 쿠폰이 사용되었는지, 얼마가 사용되었는지, 판매 후 남은 잔액이 얼마인지 확인할 수 있습니다.

## 호환성 참고 사항 {#compatibility-notes}

- **StoreApps 전용:** 이 확장 기능은 StoreApps / WooCommerce.com Smart Coupons를 대상으로 합니다. WebToffee 또는 다른 스마트 쿠폰 플러그인은 대상으로 하지 않습니다.
- **쿠폰 데이터:** WCPOS는 일반 WooCommerce 쿠폰 응답을 계속 사용합니다. StoreApps 전용 상태는 `smart_coupons_contribution` 및 쿠폰 잔액 필드와 같은 WooCommerce 메타데이터에 유지됩니다.
- **영수증:** 잔액 텍스트는 영수증 데이터나 템플릿을 변경하는 방식이 아니라, 기존 쿠폰 설명/할인 레이블 경로를 통해 추가됩니다.
- **비활성 StoreApps 플러그인:** StoreApps Smart Coupons가 활성화되어 있지 않으면 이 확장 기능은 아무 작업도 하지 않습니다.
