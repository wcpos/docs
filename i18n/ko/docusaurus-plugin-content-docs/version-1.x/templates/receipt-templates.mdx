영수증 템플릿 편집기를 사용하면 POS 시스템에서 인쇄되는 영수증의 모양과 내용을 사용자 지정할 수 있습니다. 템플릿 편집기에 접근하려면 다음으로 이동하세요: `WP 관리자 > POS > 템플릿`

## 개요

템플릿 시스템은 두 가지 유형의 템플릿을 제공합니다:

- **기본 템플릿**: 플러그인 및 테마 파일에서 자동으로 감지됩니다. 삭제할 수 없지만 사용자 지정 버전을 만들기 위해 복사할 수 있습니다.
- **사용자 지정 템플릿**: 완전히 사용자 지정, 편집 및 관리할 수 있는 사용자가 만든 템플릿입니다.

## 템플릿 편집기 접근

1. WordPress 관리자 대시보드로 이동
2. **POS > 템플릿**으로 이동
3. 다음 두 섹션으로 구성된 사용 가능한 템플릿 목록을 볼 수 있습니다:
   - **기본 템플릿** (플러그인/테마 파일에서)
   - **사용자 지정 템플릿** (사용자가 만든 템플릿)

## 기본 템플릿

시스템은 세 가지 소스에서 영수증 템플릿을 자동으로 감지합니다:

### 템플릿 소스

| 소스           | 설명                                         | 상태                               |
| -------------- | --------------------------------------------------- | ------------------------------------ |
| **플러그인**     | WooCommerce POS와 함께 포함된 핵심 영수증 템플릿 | 항상 사용 가능                     |
| **Pro 플러그인** | WooCommerce POS Pro의 향상된 템플릿          | 활성 라이선스와 함께 사용 가능        |
| **테마**      | 활성 테마의 사용자 지정 템플릿              | 테마에 템플릿이 포함된 경우 사용 가능 |

### 템플릿 작업

각 기본 템플릿에 대해 다음을 수행할 수 있습니다:

- **미리보기**: 샘플 데이터로 템플릿 모양 보기
- **복사**: 편집할 수 있는 사용자 지정 복사본 만들기
- **활성화**: POS의 기본 템플릿으로 설정

## 사용자 지정 템플릿 생성

### 방법 1: 기본 템플릿에서 복사

1. **기본 템플릿** 섹션에서 사용자 지정하려는 템플릿 찾기
2. **복사** 버튼 클릭
3. 사용자 지정 템플릿의 이름 입력
4. **템플릿 복사** 클릭
5. 새로 생성된 사용자 지정 템플릿 편집

### 방법 2: 처음부터 생성

1. 페이지 상단의 **새 템플릿 추가** 클릭
2. 템플릿 이름 입력
3. 템플릿 유형으로 **영수증** 선택
4. 편집기에서 템플릿 코딩 시작

## 템플릿 편집기 기능

템플릿 편집기는 다음을 제공합니다:

### 코드 편집기

- **구문 강조**: 더 나은 코드 가독성을 위한 PHP 구문 강조
- **줄 번호**: 쉬운 탐색 및 디버깅
- **전체 화면 편집**: 방해 없는 편집 환경

### 템플릿 설정 패널

- **언어**: 적절한 구문 강조를 위해 PHP로 설정
- **템플릿 유형**: 영수증 또는 보고서 템플릿 중 선택
- **상태**: 게시/초안 상태 관리
- **가시성**: 템플릿 가시성 제어

### 템플릿 작업

- **미리보기**: 샘플 데이터로 템플릿 렌더링 방식 확인
- **활성으로 설정**: POS의 기본 템플릿으로 지정
- **업데이트**: 변경 사항 저장
- **휴지통으로 이동**: 사용자 지정 템플릿 삭제

## 템플릿 구조

영수증 템플릿은 HTML 출력을 생성하는 PHP 파일입니다. 기본 구조는 다음과 같습니다:

```php
<?php
/**
 * 사용자 지정 영수증 템플릿
 */

if ( ! defined( 'ABSPATH' ) ) {
    exit; // 직접 액세스하면 종료
}
?>
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
    <meta charset="<?php bloginfo( 'charset' ); ?>">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <style>
        /* 사용자 지정 CSS 스타일 */
        body { font-family: sans-serif; font-size: 14px; }
        .receipt-header { text-align: center; margin-bottom: 20px; }
        /* 필요에 따라 더 많은 스타일 추가 */
    </style>
</head>
<body>
    <!-- 영수증 내용 -->
    <div class="receipt-header">
        <h1><?php echo get_bloginfo( 'name' ); ?></h1>
        <p><?php echo get_bloginfo( 'description' ); ?></p>
    </div>

    <!-- 주문 세부 정보, 항목, 총계 등 -->

</body>
</html>
```

## 사용 가능한 템플릿 변수

사용자 지정 템플릿을 만들 때 다양한 WordPress 및 WooCommerce 함수와 데이터에 액세스할 수 있습니다:

### 스토어 정보

- `get_bloginfo( 'name' )` - 스토어 이름
- `get_bloginfo( 'description' )` - 스토어 태그라인
- `get_bloginfo( 'url' )` - 스토어 URL

### 주문 데이터

템플릿이 렌더링될 때 표준 WooCommerce 주문 함수를 통해 현재 주문 데이터를 사용할 수 있습니다.

### WCPOS 함수

WCPOS 관련 데이터 및 기능에 액세스하기 위해 WCPOS별 함수(접두사 `wcpos_`)를 사용할 수 있습니다.

## 모범 사례

### 템플릿 개발

1. **복사본으로 시작**: 처음부터 생성하기보다 항상 기본 템플릿을 복사하여 시작
2. **철저히 테스트**: 활성화하기 전에 미리보기 기능을 사용하여 테스트
3. **백업 유지**: 사용자 지정 템플릿 내보내기 또는 코드 복사본 유지
4. **모바일 친화적**: 다양한 화면 크기에서 템플릿이 잘 작동하도록 보장

### 코드 품질

1. **HTML 검증**: HTML이 올바르게 구성되었는지 확인
2. **출력 이스케이프**: 보안을 위해 WordPress 이스케이프 함수 사용
3. **오류 처리**: PHP 코드에 적절한 오류 검사 포함
4. **주석**: 향후 참조를 위해 사용자 지정 내용 문서화

### 성능

1. **CSS 최적화**: 스타일을 최소하고 효율적으로 유지
2. **PHP 최소화**: 템플릿에서 복잡한 로직 방지
3. **인쇄 테스트**: 영수증 프린터에서 잘 작동하는지 확인

## 문제 해결

### 일반적인 문제

**템플릿 변경 사항 미표시**

- 변경 사항을 저장하기 위해 "업데이트"를 클릭했는지 확인
- 템플릿이 "활성"으로 설정되었는지 확인
- 캐싱 플러그인 지우기

**구문 오류**

- 문제를 식별하려면 구문 강조 사용
- 세미콜론, 괄호 또는 따옴표 누락 확인
- PHP 구문 검증

**스타일링 문제**

- 미리보기 모드에서 CSS 테스트
- 영수증 프린터와 호환되는 스타일 확인
- 인쇄 친화적인 디자인 원칙 유지

**템플릿 사용 불가**

- 템플릿 유형이 "영수증"으로 설정되었는지 확인
- 템플릿 상태가 "게시됨"인지 확인
- 적절한 권한이 있는지 확인

### 도움 받기

템플릿 사용자 지정과 관련된 문제가 발생하면:

1. 오류 메시지에 대해 템플릿 미리보기 확인
2. PHP 오류에 대해 WordPress 디버그 로그 검토
3. 먼저 기본 템플릿 복사본으로 테스트
4. **AI 지원**: Gemini, ChatGPT, Claude와 같은 최신 AI 도구는 이러한 간단한 WordPress 템플릿에 도움을 줄 수 있습니다. 템플릿 코드를 붙여넣고 원하는 내용을 설명할 수 있습니다.
5. 고급 사용자 지정을 위해 WooCommerce POS 문서 참조

## 고급 사용자 지정

고급 사용자를 위한 영수증 템플릿 추가 제어:

### 테마 통합

테마의 사용자 지정 `receipt.php` 파일을 배치하여 기본 템플릿을 재정의할 수 있습니다:

```
your-theme/woocommerce-pos/receipt.php
```

### 후크 및 필터

WooCommerce POS는 템플릿 사용자 지정을 위한 다양한 후크 및 필터를 제공합니다. 사용 가능한 옵션은 개발자 문서를 참조하세요.

### 사용자 지정 필드

적절한 WordPress 및 WooCommerce 함수를 사용하여 영수증 템플릿에 사용자 지정 주문 필드, 제품 속성 및 기타 메타데이터를 포함할 수 있습니다.
