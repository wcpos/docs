WCPOS에는 영수증, 송장, 선물 영수증, 포장 명세서, 주방 티켓 등을 위한 전체 템플릿 시스템이 포함되어 있으며, 모두 **WP Admin → POS → 템플릿** 에서 관리됩니다.

:::tip 영수증만 바꾸고 싶으신가요?
먼저 **[영수증 사용자 지정](/templates/customise)** 을 확인하세요. 다른 기본 템플릿 선택, AI에 수정 요청, 직접 편집이라는 세 가지 가장 쉬운 방법을 안내합니다.
:::

## 템플릿 엔진

WCPOS는 세 가지 템플릿 엔진을 지원합니다. 템플릿을 만들거나 갤러리 카드에서 **템플릿 사용** 을 클릭할 때 엔진이 선택되며, **나중에 변경할 수 없습니다**. 엔진을 바꾸려면 새 템플릿을 시작해야 합니다.

## 템플릿 갤러리

갤러리에는 그대로 사용하거나 사용자 지정할 수 있는 준비된 템플릿이 포함되어 있습니다. **카테고리**(영수증, 송장, 선물 영수증, 주방 티켓, 견적 / 구매 주문), **형식**(HTML / ESC/POS), **방향**(LTR / RTL)으로 필터링하고 실제 매장 데이터로 미리볼 수 있습니다. 각 카드 하단에는 **미리보기** 링크와 기본 **템플릿 사용** 버튼이 있으며, 클릭하면 *내 템플릿* 에 편집 가능한 복사본이 생성됩니다.

| 템플릿 | 엔진 | 설명 |
|----------|--------|-------------|
| 표준 영수증 | HTML | 로고, 매장 정보, 상품, 합계, 결제가 포함된 기본 영수증 |
| 표준 영수증(RTL) | HTML | 아랍어, 히브리어, 페르시아어, 우르두어용 오른쪽에서 왼쪽 버전 |
| 미니멀 / 모던 | HTML | 표준과 같은 핵심 정보를 더 작은 세로 공간에 표시 |
| 상세 영수증 | HTML | SKU 열, 단가, 세율별 세금 내역, 주소가 있는 전체 세금계산서 |
| 선물 영수증 | HTML | 상품만 표시하고 가격은 숨김. 선물 메시지와 반품 정책 포함 |
| 송장 | HTML | 미결제 주문용 선택적 "결제 방법" 패널이 있는 A4/Letter 전체 페이지 송장 |
| 포장 명세서 | HTML | 창고용 문서: 상품 + 수량, 배송지, 가격 없음 |
| 견적 / 예상서 | HTML | 가격과 조건이 있는 판매 전 문서, 결제 섹션 없음 |
| 좁은 영수증 | HTML | 좁은 용지나 HTML 지원 열전사 프린터용 고정폭 글꼴 영수증 |
| 간단한 열전사 영수증(58 mm) | 열전사 | 깔끔한 58 mm 열전사 레이아웃 |
| 간단한 열전사 영수증(80 mm) | 열전사 | 가장 일반적인 깔끔한 80 mm 열전사 레이아웃 |
| 간단한 열전사 영수증 80 mm(RTL) | 열전사 | 80 mm용 RTL 버전. 아랍어 코드페이지(CP864 / Windows-1256)를 지원하는 프린터 필요 |
| 상세 열전사 영수증(58 mm) | 열전사 | 주소, 세금 내역, 환불, 결제, 약관, 바코드가 포함된 58 mm 전체형 |
| 상세 열전사 영수증(80 mm) | 열전사 | 주소, 세금 내역, 환불, 결제, 약관, 바코드가 포함된 80 mm 전체형 |
| 주방 티켓 | 열전사 | 상품만 표시, 큰 글꼴, 가격 없음. 조리 스테이션용 |

### 타사 문서 템플릿

WP Overnight의 **PDF Invoices & Packing Slips for WooCommerce** 가 활성화되어 있으면 WCPOS는 영수증 템플릿 목록에 **송장(WP Overnight)** 및 **포장 명세서(WP Overnight)** 도 표시합니다. 이 서버 렌더링 HTML 템플릿은 WP Overnight 문서 API에 위임하여 송장 및 포장 명세서 번호, 레이아웃, 브랜딩, 법률/세금 필드, 해당 플러그인에 설정된 템플릿 사용자 지정을 재사용합니다.

이 템플릿은 WP Overnight 플러그인이 활성화된 동안에만 사용할 수 있으며 렌더링하려면 서버 연결이 필요합니다. 오프라인 인쇄가 필요하면 WCPOS에 포함된 HTML 또는 열전사 템플릿을 사용하세요.

### 적응형 세금 표시

대부분의 기본 템플릿(표준, 표준 RTL, 미니멀/모던, 좁은 영수증, 송장, 견적, 간단한 열전사 변형)은 **WooCommerce 세금 표시 설정에 자동으로 맞춰집니다**.

- **세금 포함 매장**(EU/영국/AU)은 총액 가격과 "세금 포함" 줄을 봅니다.
- **세금 별도 매장**(미국/캐나다)은 순가격과 별도 줄로 추가된 세금을 봅니다.
- 최종 합계는 항상 실제 청구된 총액을 표시합니다.

**상세** 계열은 공식 세금계산서이며, 매장 설정과 관계없이 세금 별도 항목과 명시적 내역으로 세금을 항상 표시합니다.

## 작동 방식

1. **템플릿 추가**: 갤러리 카드에서 **템플릿 사용** 을 클릭하거나 처음부터 직접 만듭니다.
2. **사용자 지정**: 실시간 미리보기가 있는 앱 내 편집기를 사용하거나 ChatGPT / Claude에 붙여넣고 원하는 변경을 요청합니다.
3. **활성화**: *내 템플릿* 에서 **활성** 토글을 켭니다. 활성 영수증 템플릿은 POS의 영수증 화면 드롭다운에 표시됩니다.
4. **인쇄 또는 표시**: 결제 후 영수증이 열리고 인쇄, 이메일 전송, 화면 표시 옵션을 제공합니다.

템플릿은 매장 정보, 상품 항목, 합계, 세금, 결제, 환불 등을 위한 섹션이 있는 표준화된 [영수증 데이터 페이로드](/templates/receipt-data)에 대해 렌더링됩니다. 모든 통화 필드는 `29.99` 대신 `$29.99`처럼 미리 형식화된 `_display` 변형을 포함합니다.

## Pro 기능

[WCPOS Pro](/getting-started/pro-license)를 사용하면 각 매장이 자체 템플릿 할당, 브랜딩(로고, 주소, 연락처), 템플릿 순서를 가질 수 있습니다. 여러 지점에서 각 매장이 자체 레터헤드를 인쇄해야 할 때 유용합니다.
