템플릿을 사용하면 POS 시스템의 다양한 출력의 모양과 내용을 사용자 정의할 수 있습니다. 템플릿 편집기에는 다음 경로로 접근할 수 있습니다: `WP Admin > POS > Templates`

템플릿 시스템은 다음을 사용자 정의하는 유연성을 제공합니다:

- **영수증 템플릿** - 인쇄 및 디지털 영수증 사용자 정의
- **보고서 템플릿** - 보고서 레이아웃 사용자 정의 _(곧 출시 예정)_
- **이메일 템플릿** - 이메일 알림 사용자 정의 _(곧 출시 예정)_

## 템플릿 유형

### 기본 템플릿

플러그인 및 테마 파일에서 자동으로 감지됩니다. 삭제할 수는 없지만 사용자 정의 버전을 만들기 위해 복사할 수 있습니다.

### 사용자 정의 템플릿

사용자가 만든 템플릿으로, 워드프레스 관리 인터페이스를 통해 완전히 사용자 정의하고, 편집하고, 관리할 수 있습니다.
