템플릿을 사용하면 POS 시스템에서 다양한 출력의 모양과 내용을 사용자 정의할 수 있습니다. 템플릿 편집기에 접속하려면 다음으로 이동하세요: `WP Admin > POS > 템플릿`

템플릿 시스템은 사용자 정의에 대한 유연성을 제공합니다:

- **영수증 템플릿** - 인쇄 및 디지털 영수증 사용자 정의
- **보고서 템플릿** - 보고서 레이아웃 사용자 정의 _(곧 제공 예정)_
- **이메일 템플릿** - 이메일 알림 사용자 정의 _(곧 제공 예정)_

## 템플릿 유형 {#template-types}

### 기본 템플릿 {#default-templates}

플러그인 및 테마 파일에서 자동으로 감지됩니다. 이러한 템플릿은 삭제할 수 없지만 사용자 정의 버전을 만들기 위해 복사할 수 있습니다.

### 사용자 정의 템플릿 {#custom-templates}

사용자가 만든 템플릿으로, WordPress 관리자 인터페이스를 통해 완전히 사용자 정의, 편집 및 관리할 수 있습니다.
