프린터 설정은 POS의 **설정 > 프린터**에 있습니다. 각 장치는 자체 프린터 구성을 관리합니다. 프린터는 로컬에 저장되며 장치 간에 동기화되지 않습니다.

## 플랫폼별 지원 프린터 {#supported-printers-by-platform}

**프린터 추가** 양식은 사용 중인 앱에 맞게 바뀝니다. 각 플랫폼은 실제로 사용할 수 있는 연결과 옵션만 표시합니다.

| 플랫폼 | 지원 프린터 | 연결 |
|----------|--------------------|-------------|
| **웹 앱** | 내장 웹 서버가 있는 Epson 및 Star 프린터 | 네트워크(HTTP) |
| **데스크톱 앱** | 모든 네트워크 영수증 프린터 | 네트워크(raw TCP) |
| **iOS 앱** | Epson 및 Star | 네트워크, Bluetooth |
| **Android 앱** | Epson 및 Star | 네트워크, Bluetooth, USB |

:::info 웹에서는 왜 Epson과 Star만 지원하나요?
웹 브라우저는 raw TCP 연결을 열 수 없습니다. Epson 및 Star 영수증 프린터에는 HTTP로 인쇄 작업을 받는 내장 웹 서버가 있어, 브라우저에서 ESC/POS 명령을 보내는 유일한 방법입니다. 데스크톱 앱은 TCP로 직접 연결할 수 있으므로 이 제한이 없습니다.

**모바일 앱**에서는 Bluetooth와 USB 인쇄가 Epson 및 Star의 공식 SDK를 통해 이루어집니다. 따라서 이 연결에도 Epson 또는 Star 프린터가 필요합니다. 일반 프린터는 네트워크 연결에서만 작동합니다.
:::

## 프린터 추가 {#adding-a-printer}

<Steps>
  <StepCard number="1" title="프린터 설정 열기">

POS에서 **설정 > 프린터**로 이동합니다. 아직 프린터가 설정되지 않았다면 **프린터 추가** 버튼이 있는 빈 상태가 표시됩니다. 모든 프린터에는 **프린터 이름**이 지정됩니다. 이는 단순한 레이블이며 "영수증 프린터"("영수증 프린터 2" 등)로 자동 입력됩니다. 프린터를 구분하기 쉬운 이름으로 바꿀 수 있습니다.

  </StepCard>
  <StepCard number="2" title="프린터 연결 방식 선택">

이 단계는 플랫폼에 따라 달라집니다.

<ul>
  <li><strong>웹 앱</strong> — <strong>제조사</strong>로 Epson 또는 Star를 선택합니다. 브라우저는 이 두 브랜드의 내장 웹 서버에만 접근할 수 있으므로 웹에서는 첫 번째 선택 항목입니다.</li>
  <li><strong>데스크톱 앱</strong> — 프린터는 네트워크로 연결됩니다. 여기서 추가로 선택할 항목은 없습니다.</li>
  <li><strong>모바일 앱</strong> — <strong>연결 유형</strong>으로 <strong>네트워크</strong>, <strong>Bluetooth</strong>, 또는 <strong>USB</strong>를 선택합니다(USB는 Android만 해당). 아래 필드는 선택에 맞게 바뀝니다.</li>
</ul>

  </StepCard>
  <StepCard number="3" title="프린터 식별">

**네트워크 프린터** — **IP 주소**를 입력합니다(예: `192.168.1.100`). WCPOS는 이 주소를 확인해 Epson 또는 Star 장치인지 감지하고 포트, 명령 언어, 제조사를 자동으로 채웁니다. 감지가 완료되면 "감지됨: Epson" 또는 "감지됨: Star" 같은 레이블이 표시됩니다. 웹 앱에서는 연결할 정확한 엔드포인트 URL도 보여 줍니다.

**Bluetooth 또는 USB 프린터**(모바일) — 검색된 장치 목록에서 프린터를 선택합니다. 장치를 선택하면 이름과 제조사가 자동으로 채워집니다. 프린터가 아직 보이지 않으면 Bluetooth에서는 **프린터 스캔**, USB에서는 **새로 고침**을 누릅니다.

자동 감지가 올바르지 않으면 **고급 설정**에서 모두 조정할 수 있습니다.

  </StepCard>
  <StepCard number="4" title="저장 및 테스트" isLast>

**저장**을 클릭합니다. WCPOS는 먼저 **테스트 인쇄**를 보내며, 성공한 경우에만 프린터를 저장합니다.

테스트 인쇄는 짧은 진단 출력입니다. 번호가 있는 열 눈금자, 가운데 정렬된 일반 및 두 배 크기 텍스트, 좌우 정렬 행, 용지 절단이 포함됩니다. 프린터에 연결되는지만이 아니라 용지 너비, 정렬, 크기, 절단이 올바르게 설정되었는지 확인하도록 설계되었습니다.

테스트가 실패하면 오류와 함께 **테스트 없이 저장** 옵션이 표시되어 프로필을 그대로 저장할 수 있습니다.

  </StepCard>
</Steps>

## 연결 유형(모바일 앱) {#connection-types-mobile-apps}

iOS 및 Android 앱에서는 프린터를 세 가지 방식으로 연결할 수 있습니다. 프린터 추가 양식 상단에서 **연결 유형**을 선택하면 연결 필드가 선택에 맞게 업데이트됩니다.

### 네트워크 {#network}

프린터가 장치와 같은 Wi-Fi 네트워크에 있습니다. IP 주소를 입력합니다. [프린터 IP 주소 찾기](#finding-your-printers-ip-address)를 참고하세요. 네트워크 프린터는 Epson, Star 또는 일반 프린터일 수 있습니다.

### Bluetooth {#bluetooth}

프린터가 Bluetooth로 장치에 페어링되어 있습니다. **프린터 스캔**을 누르고 목록에서 선택합니다. 이름과 제조사가 자동으로 채워집니다. Bluetooth 인쇄는 Epson 및 Star의 네이티브 SDK를 사용하므로 프린터는 **Epson 또는 Star** 모델이어야 합니다.

### USB(Android 전용) {#usb-android-only}

프린터가 USB 케이블로 연결되어 있습니다. **새로 고침**을 누르고 연결된 장치 목록에서 선택합니다. Bluetooth와 마찬가지로 USB 인쇄에도 **Epson 또는 Star** 프린터가 필요합니다. iOS는 일반 USB 주변기기를 지원하지 않으므로 이 옵션은 Android에만 표시됩니다.

:::note
**포트** 설정은 네트워크 프린터에만 적용되며 Bluetooth 및 USB 연결에서는 숨겨집니다.
:::

## 고급 설정 {#advanced-settings}

**고급 설정**을 펼쳐 프린터 구성을 세부 조정합니다. 대부분의 값은 제조사가 확인되면 자동으로 채워집니다.

| 설정 | 옵션 | 설명 |
|---------|---------|-------------|
| **제조사** | Epson, Star, 일반 | 프린터 제조사입니다. **웹 앱**에서는 제조사가 고급 설정이 아니라 양식 상단의 기본 선택 항목입니다(Epson 또는 Star만 가능). 일반은 데스크톱 및 모바일 앱의 **네트워크** 프린터에서 사용할 수 있지만 Bluetooth 또는 USB 연결에서는 사용할 수 없습니다. |
| **포트** | 자동 입력 | 프린터의 네트워크 포트입니다. 네트워크 전용이며 Bluetooth와 USB에서는 숨겨집니다. |
| **언어** | ESC/POS, StarPRNT, Star Line Mode | 프린터가 사용하는 명령 프로토콜입니다. 감지된 제조사에 따라 자동 입력됩니다. |
| **프린터 텍스트 너비** | 58mm(32자), 80mm 표준(42자), 80mm 와이드(48자) | 한 줄에 들어가는 문자 수입니다. 용지와 프린터에 맞추세요. 대부분의 80mm 프린터는 한 줄 42자이고 일부는 48자입니다. |
| **전체 영수증 래스터** | 끄기 / 켜기 | 영수증 전체를 텍스트 대신 이미지로 인쇄합니다. 아래 [전체 영수증 래스터](#full-receipt-raster)를 참고하세요. |

## 프린터 옵션 {#printer-options}

이 토글은 프린터 추가 양식 하단에 있습니다.

| 옵션 | 설명 |
|--------|-------------|
| **용지 자동 절단** | 각 영수증 후 용지를 자동으로 자릅니다 |
| **금전함 자동 열기** | 인쇄 후 연결된 금전함을 여는 킥 명령을 보냅니다 |
| **기본값으로 설정** | 특정 프린터로 라우팅되지 않은 영수증에 사용할 기본 프린터로 지정합니다 |

:::note "결제 후 자동 인쇄"를 찾고 있나요?
판매 완료 시 영수증을 자동으로 인쇄하는 것은 프린터 설정이 아니라 **카트 설정**입니다. POS 카트 설정에서 **영수증 자동 인쇄**를 켜세요. *어떤* 프린터를 사용할지는 기본 프린터와 템플릿별 프린터 라우팅으로 결정됩니다.
:::

## 인쇄 라우팅 {#print-routing}

열전사 영수증 **및** A4 송장처럼 두 개 이상의 템플릿을 사용하면, 인쇄 라우팅이 각 템플릿을 어떤 프린터로 보낼지 결정합니다. 라우팅은 다음 순서의 세 단계로 확인됩니다.

1. **작업별 재정의.** 영수증 화면에서 템플릿 전환기 옆에 프린터 드롭다운이 있습니다. 여기서 프린터를 선택하면 해당 인쇄 작업 하나에만 모든 설정을 재정의합니다. 템플릿을 바꾸면 다시 **자동**으로 초기화됩니다.
2. **설정 재정의.** **설정 → 인쇄 라우팅**으로 이동해 각 템플릿에 특정 프린터를 지정합니다. 예를 들어 열전사 영수증은 Epson 네트워크 프린터로, HTML 송장은 시스템 인쇄 대화상자로 보낼 수 있습니다. 재정의를 제거하려면 템플릿을 **자동**으로 되돌립니다.
3. **자동 매칭.** 재정의가 없으면 WCPOS가 자동으로 매칭합니다.
   - **열전사 템플릿**은 문자 너비가 맞는 열전사 프린터로 라우팅됩니다. 58mm 템플릿은 32열 프린터를 선호하고, 80mm 템플릿은 42열 또는 48열 프린터를 선호합니다.
   - **HTML 템플릿**은 시스템 인쇄 대화상자로 라우팅됩니다.
   - 여러 프린터가 일치하면 **기본** 프린터가 선택됩니다.

호환되지 않는 프린터로 템플릿을 수동 전송하면, 예를 들어 열전사 템플릿을 시스템 대화상자로 보내면, 영수증 화면에 황색 **불일치 경고**가 나타납니다. 인쇄는 계속되지만 출력이 올바르게 렌더링되지 않을 수 있습니다.

라우팅 재정의는 **장치별**로 저장됩니다. 각 iPad, 휴대폰, 컴퓨터가 자체 라우팅을 관리하며 서버 측 동기화는 없습니다.

:::note
인쇄 라우팅 설정 페이지는 최소 하나의 프린터를 추가한 후에만 표시됩니다. 프린터가 설정되어 있지 않으면 모든 템플릿은 암묵적으로 시스템 인쇄 대화상자를 사용합니다.
:::

## 전체 영수증 래스터 {#full-receipt-raster}

기본적으로 WCPOS는 열전사 프린터에 영수증을 **텍스트**로 보냅니다. 빠르고 작고 선명하며 프린터 내장 글꼴을 사용합니다. 단점은 열전사 프린터 글꼴이 제한된 문자만 지원한다는 점입니다. 프린터에 글꼴이 없는 문자 체계, 예를 들어 많은 비라틴 문자, 오른쪽에서 왼쪽으로 쓰는 텍스트, 일부 기호는 빈 사각형이나 깨진 문자로 나올 수 있습니다.

**전체 영수증 래스터**는 이를 해결합니다. 켜면 WCPOS가 영수증 전체를 이미지로 렌더링해 그 이미지를 프린터로 보냅니다. 그래서 어떤 언어나 문자 체계에서도 인쇄물이 화면과 정확히 일치합니다.

프린터별로 **고급 설정**에서 켭니다. 알아둘 점:

- **더 느립니다.** 이미지는 텍스트 한 줄보다 훨씬 커서 전송과 인쇄 시간이 더 걸립니다.
- **필요할 때만 사용하세요.** 영수증이 텍스트로 잘 인쇄된다면 꺼 둡니다.
- 네트워크, Bluetooth, USB 등 모든 연결의 열전사 프린터에 적용됩니다. 이미 전체 페이지 렌더링으로 인쇄하는 시스템 인쇄 대화상자에는 적용되지 않습니다.

## 프린터 검색 {#discovering-printers}

**모바일 앱**에서는 프린터 추가 양식이 **Bluetooth**(iOS 및 Android)와 **USB**(Android만) 연결용 주변 프린터를 나열합니다. **프린터 스캔** 또는 **새로 고침**을 누르고 목록에서 선택하세요. 이름과 제조사가 자동 입력됩니다.

모든 플랫폼의 **네트워크** 프린터는 IP 주소를 직접 입력합니다. 아래 [프린터 IP 주소 찾기](#finding-your-printers-ip-address)를 참고하세요. 웹 앱에는 네트워크 검색 기능이 없습니다. 브라우저는 네트워크를 스캔할 수 없으므로 항상 수동 IP 입력을 사용합니다.

## 프린터 IP 주소 찾기 {#finding-your-printer-s-ip-address}

대부분의 영수증 프린터는 IP 주소가 포함된 자체 테스트 페이지를 인쇄할 수 있습니다.

- **Epson**: 전원을 켤 때 급지 버튼을 길게 누릅니다
- **Star**: 전원이 켜진 상태에서 급지 버튼을 5초 동안 길게 누릅니다

또는 라우터의 연결된 장치 목록이나 프린터 구성 유틸리티를 확인하세요.

### 고정 IP 권장 {#static-ip-recommended}

영수증 프린터는 재시작 시 주소가 바뀌지 않도록 **고정 IP 주소**를 사용하는 것이 좋습니다. 다음 중 하나로 설정합니다.

- 프린터의 내장 웹 인터페이스에서 설정합니다. 일반적으로 `http://<printer-ip>`에서 접근할 수 있습니다
- 라우터의 DHCP 설정에서 프린터 MAC 주소에 예약 IP를 할당합니다

## Epson 프린터 {#epson-printers}

ePOS를 지원하는 Epson 프린터는 HTTP로 통신합니다. WCPOS는 프린터의 내장 웹 서버로 SOAP/XML 요청을 보냅니다.

**연결 세부 정보:**
- 포트 `8008`(HTTP) 또는 `8043`(HTTPS)
- 엔드포인트: `/cgi-bin/epos/service.cgi`

**동작 확인된 모델:**
- Epson TM-T70-i 2(네트워크)
- Epson TM-m30iii(Printus 미들웨어 사용)

:::tip
Epson 프린터 설정에서 ePOS가 활성화되어 있는지 확인하세요. `http://<printer-ip>`에서 프린터의 웹 인터페이스에 접속해 ePOS 설정을 확인하고 활성화합니다.
:::

**모바일 앱**에서는 Epson 프린터가 Epson 네이티브 SDK를 통해 Bluetooth 또는 USB로도 연결될 수 있습니다. IP 주소나 ePOS 웹 서버가 필요 없습니다.

## Star 프린터 {#star-printers}

WebPRNT를 지원하는 Star 프린터는 HTTP로 통신합니다. WCPOS는 Star WebPRNT 엔드포인트로 명령을 보냅니다.

**연결 세부 정보:**
- HTTPS 또는 HTTP
- 엔드포인트: `/StarWebPRNT/SendMessage`

**모바일 앱**에서는 Star 프린터도 Star 네이티브 SDK를 통해 Bluetooth 또는 USB로 연결될 수 있습니다.

## 일반 프린터 {#generic-printers}

**데스크톱 앱**은 raw TCP를 통해 모든 네트워크 영수증 프린터를 지원합니다. 프린터의 IP 주소와 포트(보통 `9100`)를 입력하세요. ESC/POS 명령을 받을 수 있다면 제조사와 관계없이 대부분의 열전사 프린터에서 작동합니다.

**모바일 앱**에서도 일반 프린터가 지원되지만 **네트워크** 연결에서만 가능합니다. Bluetooth 또는 USB는 Epson 또는 Star 프린터가 필요합니다. **웹 앱**은 일반 프린터를 전혀 지원하지 않습니다.

## 금전함 {#cash-drawers}

영수증 프린터에 연결된 금전함은 각 판매 후 자동으로 작동할 수 있습니다.

1. 프린터 옵션에서 **금전함 자동 열기**를 활성화합니다
2. POS가 인쇄 후 ESC/POS 킥 명령을 보냅니다

:::note
금전함 제어에는 열전사 프린터 프로필이 필요합니다. 브라우저 인쇄 대화상자는 금전함을 작동시킬 수 없습니다.
:::

## 문제 해결 {#troubleshooting}

<Accordion>
  <AccordionItem question="설정 중 프린터가 감지되지 않음">

  - 프린터 전원이 켜져 있고 장치와 같은 네트워크에 연결되어 있는지 확인합니다
  - 네트워크 프린터: IP 주소가 올바른지 확인합니다(자체 테스트 페이지 인쇄)
  - 웹 앱: 프린터가 ePOS/WebPRNT를 지원하는 Epson 또는 Star 모델인지 확인합니다
  - 데스크톱 앱: 포트 9100으로 일반 제조사 옵션을 시도합니다
  - Bluetooth: 먼저 장치의 Bluetooth 설정에서 프린터를 페어링한 다음 **프린터 스캔**을 누릅니다
  - USB(Android): 케이블을 확인하고 USB 권한 요청이 나타나면 승인합니다
  - 방화벽이 연결을 차단하지 않는지 확인합니다

  </AccordionItem>

  <AccordionItem question="프린터가 감지되지만 아무것도 인쇄되지 않음">

  - 용지 롤을 확인합니다. 비어 있거나 걸렸을 수 있습니다
  - **프린터 텍스트 너비** 설정이 용지(58mm 또는 80mm)와 맞는지 확인합니다
  - 프린터 자체에서 테스트 인쇄를 실행해 작동을 확인합니다
  - 명령 언어가 프린터와 맞는지 확인합니다(Epson은 ESC/POS, Star는 StarPRNT 또는 Star Line Mode)

  </AccordionItem>

  <AccordionItem question="영수증이 깨지거나 잘못된 문자가 표시됨">

  - 명령 언어 설정이 잘못되었을 수 있습니다. ESC/POS, StarPRNT, Star Line Mode를 전환해 보세요
  - **프린터 텍스트 너비** 설정이 올바른지 확인합니다
  - 영수증이 비라틴 문자나 오른쪽에서 왼쪽으로 쓰는 언어라면 고급 설정에서 **전체 영수증 래스터**를 켭니다. 영수증을 이미지로 인쇄해 모든 언어가 올바르게 표시됩니다

  </AccordionItem>

  <AccordionItem question="금전함이 열리지 않음">

  - 프린터 옵션에서 **금전함 자동 열기**가 활성화되어 있는지 확인합니다
  - 금전함이 별도 전원이 아니라 프린터의 DK 포트에 연결되어 있는지 확인합니다
  - 모든 프린터가 금전함 킥 명령을 지원하는 것은 아닙니다. 프린터 문서를 확인하세요

  </AccordionItem>

  <AccordionItem question="웹 앱에서 연결할 수 없음">

  - 웹 앱은 내장 웹 서버가 있는 Epson 및 Star 프린터에만 연결할 수 있습니다
  - 일반 및 USB 프린터는 웹에서 지원되지 않습니다. 대신 데스크톱 또는 모바일 앱을 사용하세요
  - 사이트가 HTTPS를 사용하지만 프린터가 HTTP만 지원하면 브라우저가 혼합 콘텐츠로 연결을 차단할 수 있습니다

  </AccordionItem>

  <AccordionItem question="Bluetooth 프린터가 연결되지 않음(모바일)">

  - WCPOS에 추가하기 전에 장치의 시스템 Bluetooth 설정에서 프린터를 페어링합니다
  - Bluetooth 인쇄는 Epson 및 Star 프린터만 지원합니다. 일반 Bluetooth 프린터는 지원되지 않습니다
  - 프린터가 목록에 나타나지 않으면 **프린터 스캔**을 다시 누르고 전원이 켜져 있으며 범위 안에 있는지 확인합니다

  </AccordionItem>
</Accordion>
