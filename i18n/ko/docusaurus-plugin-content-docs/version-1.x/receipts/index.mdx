WCPOS에는 영수증, 인보이스, 선물 영수증, 포장 명세서, 주방 주문서 등을 위한 완전한 템플릿 시스템이 포함되어 있으며, 모두 **WP Admin → POS → Templates**에서 관리됩니다.

:::tip 영수증만 변경하고 싶으신가요?
**[영수증 맞춤 설정](/receipts/customise)**에서 시작하세요. 번들 템플릿 변경, AI를 통한 수정, 직접 편집 등 가장 쉬운 세 가지 방법을 안내합니다.
:::

## 템플릿 엔진 {#template-engines}

WCPOS는 세 가지 템플릿 엔진을 지원합니다. 엔진은 템플릿을 생성하거나 갤러리 카드에서 **Use Template**을 클릭할 때 선택되며, **이후에는 변경할 수 없습니다**. 엔진을 전환하려면 새 템플릿을 생성하세요.

<FeatureGrid>
  <FeatureCard icon="code" title="로직리스 HTML (권장)">
    일반 HTML 내부의 Mustache 스타일 <code>{"{{variable}}"}</code> 플레이스홀더입니다. 클라이언트 측에서 렌더링되므로 영수증이 **오프라인에서도 작동**합니다. 브라우저 인쇄 및 PDF 영수증에 사용합니다.
  </FeatureCard>
  <FeatureCard icon="print" title="열전사 XML">
    화면 미리보기와 열전사 영수증 프린터용 **ESC/POS 명령**을 동시에 생성하는 XML 템플릿입니다. 하나의 템플릿으로 두 가지 출력을 제공합니다. Epson, Star 및 기타 열전사 프린터에 사용합니다.
  </FeatureCard>
  <FeatureCard icon="terminal" title="레거시 PHP">
    WooCommerce 함수를 사용하는 서버 측 PHP 템플릿입니다. 기존 <code>yourtheme/woocommerce-pos/receipt.php</code> 오버라이드와의 하위 호환성을 위해 유지됩니다. 렌더링하려면 서버 연결이 필요합니다.
  </FeatureCard>
</FeatureGrid>

## 템플릿 갤러리 {#template-gallery}

갤러리에는 그대로 사용하거나 맞춤 설정할 수 있는 기성 템플릿이 포함되어 있습니다. **카테고리**(영수증, 인보이스, 선물 영수증, 주방 티켓, 견적서 / 발주서), **형식**(HTML / ESC/POS), **방향**(LTR / RTL)별로 필터링하고 매장의 실제 데이터로 미리보기할 수 있습니다. 각 카드 하단에는 **Preview** 링크와 기본 **Use Template** 버튼이 있으며, Use Template을 클릭하면 *Your Templates*에 편집 가능한 사본이 생성됩니다.

### 서드파티 문서 템플릿 {#third-party-document-templates}

**PDF Invoices & Packing Slips for WooCommerce**(WP Overnight 제작)가 활성화되어 있으면, WCPOS는 영수증 템플릿 목록에 **Invoice (WP Overnight)** 및 **Packing Slip (WP Overnight)**도 표시합니다. 이 서버 렌더링 HTML 템플릿은 WP Overnight의 문서 API에 위임되며, 해당 플러그인에서 구성한 인보이스 및 포장 명세서 번호 매기기, 레이아웃, 브랜딩, 법적/세금 필드, 템플릿 사용자 정의를 재사용합니다.

WP Overnight 플러그인이 활성화되어 있는 동안에만 사용할 수 있으며 렌더링하려면 서버 연결이 필요합니다. 오프라인 인쇄가 필요한 경우 WCPOS의 번들 HTML 또는 열전사 템플릿을 사용하십시오.

### 적응형 세금 표시 {#adaptive-tax-display}

대부분의 번들 템플릿(Standard, Standard RTL, Minimal/Modern, Narrow, Invoice, Quote, Thermal Simple 변형)은 **WooCommerce 세금 표시 설정에 자동으로 적응합니다**:

- **세금 포함 매장**(EU/UK/AU)에서는 총액 가격과 "세금 포함" 항목이 표시됩니다.
- **세금 별도 매장**(US/CA)에서는 순 가격에 세금이 별도 항목으로 추가됩니다.
- 합계 금액에는 항상 실제 청구된 총액이 표시됩니다.

**Detailed** 계열은 공식 세금계산서이며, 매장 설정과 관계없이 항상 세금 별도 항목과 명시적 내역으로 세금을 항목별 표시합니다.

## 작동 방식 {#how-it-works}

1. **템플릿 추가** — 갤러리 카드에서 **Use Template**을 클릭하거나 처음부터 직접 만들 수 있습니다.
2. **커스터마이즈** — 실시간 미리보기가 포함된 인앱 편집기를 사용하거나, ChatGPT / Claude에 붙여넣고 원하는 변경 사항을 요청할 수 있습니다.
3. **활성화** — *Your Templates*에서 **Active** 토글을 켭니다. 활성화된 모든 영수증 템플릿은 계산대의 영수증 화면 드롭다운에 표시됩니다.
4. **인쇄 또는 표시** — 결제 완료 후 영수증이 열리며 인쇄, 이메일 전송, 화면 보기 옵션을 제공합니다.

템플릿은 매장 정보, 품목, 합계, 세금, 결제, 환불 등의 섹션이 포함된 표준화된 [영수증 데이터 페이로드](/receipts/receipt-data)를 기반으로 렌더링됩니다. 모든 통화 필드에는 사전 포맷된 `_display` 변형이 포함되어 있습니다(예: `29.99` 대신 `$29.99`).

## Pro 기능 {#pro-features}

[WCPOS Pro](/getting-started/pro-license)를 사용하면 각 매장별로 고유한 템플릿 할당, 브랜딩(로고, 주소, 연락처 정보) 및 템플릿 순서를 설정할 수 있어, 각 지점에서 자체 레터헤드를 인쇄하는 다중 매장 운영에 유용합니다.
