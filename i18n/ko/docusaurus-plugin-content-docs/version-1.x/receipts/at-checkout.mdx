결제가 성공적으로 처리되면 영수증 모달이 표시됩니다. 완료된 주문의 요약을 보여 주며, 영수증을 인쇄하거나 이메일로 보낼 수 있는 옵션을 제공합니다.

{/* Screenshot: receipt-modal.png */}

## 영수증 내용 {#receipt-contents}

영수증에는 다음이 표시됩니다.

### 매장 정보 {#store-information}
- **매장 이름**
- **주소**
- **전화번호**
- **이메일**
- **웹사이트**
- **로고** (구성된 경우)

### 영수증 헤더 {#receipt-header}
- **영수증 유형** (예: "세금 영수증")
- **주문 번호**
- **날짜 및 시간**
- **계산원 이름**
- **결제 방법**

### 주문 상세 정보 {#order-details}
- **품목**에는 다음이 포함됩니다:
  - 제품명
  - 변형 속성(해당하는 경우)
  - 수량
  - 가격
  - 합계

### 합계 {#totals}
- **소계**
- **세금** (해당하는 경우)
- **총계**
- **받은 금액** (현금 결제 시)
- **거스름돈** (현금 결제 시)

### 고객 정보 {#customer-information}
- **청구 주소**
- **배송 주소** (다른 경우)

## 영수증 작업 {#receipt-actions}

### 닫기 {#close}
영수증 모달을 닫고 POS로 돌아갑니다.

### 영수증 이메일 보내기 {#email-receipt}
고객의 이메일 주소로 영수증을 보냅니다:

1. **영수증 이메일 보내기**를 클릭합니다
2. 이메일 주소를 확인합니다
3. 영수증은 WooCommerce 이메일 시스템을 통해 전송됩니다

고객에게 주문과 연결된 이메일 주소가 있어야 합니다.

### 영수증 인쇄 {#print-receipt}
영수증의 실제 사본을 인쇄합니다:

1. **영수증 인쇄**를 클릭합니다
2. 브라우저의 인쇄 대화 상자가 열립니다
3. 영수증 프린터를 선택하고 인쇄합니다

:::tip 영수증 프린터
WCPOS는 브라우저에서 접근할 수 있는 모든 프린터와 함께 작동합니다. 최상의 결과를 위해 다음을 사용하세요:
- 전용 영수증 프린터(감열식 프린터가 일반적임)
- 프린터 설정에서 용지 크기를 구성합니다
- 빠른 인쇄를 위해 인쇄 프리셋을 설정합니다
:::

## 자동 영수증 옵션 {#automatic-receipt-options}

[카트 표시 설정](/pos/cart#display-settings)에서 다음을 구성할 수 있습니다:

- **결제 후 영수증 자동 표시** - 결제 직후 영수증 모달을 엽니다
- **결제 후 영수증 자동 인쇄** - 영수증이 표시될 때 인쇄를 자동으로 실행합니다

이 옵션은 대량 결제 워크플로를 더 빠르게 처리하는 데 도움이 됩니다.

## 영수증 사용자 지정 {#customising-receipts}

영수증 모양은 WordPress의 템플릿에서 제어됩니다.

`WP Admin > POS > Templates`

다음 사용자 지정에 대한 자세한 내용은 [영수증 템플릿](/receipts)을 참조하세요.

- 매장 정보 표시
- 로고 및 브랜딩
- 레이아웃 및 서식
- 포함할 정보

## 영수증 재인쇄 {#reprinting-receipts}

이전 주문의 영수증을 재인쇄하려면:

1. [주문 화면](/orders)으로 이동합니다(프로 기능).
2. 주문 찾기
3. 클릭하여 주문 열기
4. 영수증 옵션을 사용하여 다시 인쇄
