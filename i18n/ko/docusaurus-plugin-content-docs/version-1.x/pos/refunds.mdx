계산대에서 금액을 돌려줘야 하나요? WCPOS에서는 POS를 벗어나지 않고 WooCommerce 주문을 환불할 수 있습니다 — 전체 또는 일부 금액을 원래 결제 수단으로 환불하거나 계산대 현금으로 지급할 수 있습니다.

:::info Pro 기능
POS에서 환불을 처리하려면 [WCPOS Pro](/getting-started/pro-license)가 필요합니다. Pro가 없으면 `WP Admin → WooCommerce → Orders` 경로에서 환불하세요.
:::

## 요약 {#the-short-version}

1. **환불 양식 열기** — 주문 목록에서 해당 주문의 점 세 개 메뉴를 클릭하고 **환불**을 선택하거나, 주문을 열고 푸터에서 **환불**을 클릭합니다.
2. **수량 설정** — 각 항목에 **환불 수량**을 입력합니다(전체 주문 환불은 모든 항목을 전체 수량으로 설정하고, 부분 환불은 일부 항목만 설정). 필요한 경우 선택적으로 **사용자 지정 금액**과 **사유**를 추가합니다.
3. **금액이 반환될 위치 선택:**
   - ***(게이트웨이)*로 환불** — 지원되는 게이트웨이의 경우 금액을 원래 카드/지갑으로 돌려보냅니다(예: Stripe Terminal, Vipps MobilePay).
   - **현금 환불** — 금전함에서 고객에게 돌려준 현금을 기록합니다. 현금 판매에 자동으로 사용됩니다.
4. **확인** — **환불 처리**를 누르고 금액을 확인합니다. 주문은 즉시 업데이트됩니다.

환불에는 실시간 서버 연결이 필요합니다. 결제와 달리 오프라인 대기열에 넣을 수 없습니다.

## 전체 세부 정보가 필요하신가요? {#want-the-full-details}

전체 가이드에서는 환불 가능한 주문 상태, 환불 양식의 세금 계산 방식, 게이트웨이와 현금 간 결정 규칙, 이후 영수증과 주문 상태에 발생하는 변경 사항, 계산원/매장 감사 추적을 다룹니다.
