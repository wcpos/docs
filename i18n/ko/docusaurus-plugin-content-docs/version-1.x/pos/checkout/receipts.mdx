결제가 성공적으로 처리되면 영수증 모달이 표시됩니다. 이 모달은 완료된 주문의 요약을 표시하고 영수증을 인쇄하거나 이메일로 전송할 수 있는 옵션을 제공합니다.

{/* 스크린샷: receipt-modal.png */}

## 영수증 내용 {#receipt-contents}

영수증에는 다음이 표시됩니다:

### 매장 정보 {#store-information}
- **매장 이름**
- **주소**
- **전화번호**
- **이메일**
- **웹사이트**
- **로고** (설정된 경우)

### 영수증 헤더 {#receipt-header}
- **영수증 유형** (예: "세금 영수증")
- **주문 번호**
- **날짜 및 시간**
- **계산원 이름**
- **결제 방법**

### 주문 세부사항 {#order-details}
- **라인 항목**이 포함됩니다:
  - 제품 이름
  - 변형 속성 (해당하는 경우)
  - 수량
  - 가격
  - 총액

### 합계 {#totals}
- **소계**
- **세금** (해당하는 경우)
- **총계**
- **금액 수령** (현금의 경우)
- **거스름돈** (현금의 경우)

### 고객 정보 {#customer-information}
- **청구 주소**
- **배송 주소** (다른 경우)

## 영수증 동작 {#receipt-actions}

### 닫기 {#close}
영수증 모달을 닫고 POS로 돌아갑니다.

### 이메일 영수증 {#email-receipt}
고객의 이메일 주소로 영수증을 전송합니다:

1. **이메일 영수증** 클릭
2. 이메일 주소 확인
3. 영수증이 WooCommerce의 이메일 시스템을 통해 전송됩니다

고객은 주문과 연관된 이메일 주소를 가지고 있어야 합니다.

### 영수증 인쇄 {#print-receipt}
영수증의 물리적 사본을 인쇄합니다:

1. **영수증 인쇄** 클릭
2. 브라우저의 인쇄 대화상자가 열립니다
3. 영수증 프린터를 선택하고 인쇄합니다

:::tip 영수증 프린터
WCPOS는 브라우저에서 접근 가능한 모든 프린터와 함께 작동합니다. 최상의 결과를 위해 다음을 사용하세요:
- 전용 영수증 프린터 (열전사 프린터가 일반적입니다)
- 프린터 설정에서 용지 크기 구성
- 빠른 인쇄를 위한 인쇄 프리셋 설정
:::

## 자동 영수증 옵션 {#automatic-receipt-options}

[장바구니 표시 설정](/pos/cart#display-settings)에서 다음을 구성할 수 있습니다:

- **체크아웃 후 자동으로 영수증 표시** - 결제 후 즉시 영수증 모달을 엽니다
- **체크아웃 후 자동으로 영수증 인쇄** - 영수증이 표시될 때 자동으로 인쇄를 트리거합니다

이 옵션은 대량 체크아웃 작업 흐름을 가속화합니다.

## 영수증 사용자 정의 {#customising-receipts}

영수증의 모양은 WordPress의 템플릿에 의해 제어됩니다:

`WP Admin > POS > 템플릿`

[영수증 템플릿](/templates/receipt-templates)에서 사용자 정의에 대한 세부정보를 참조하세요:

- 매장 정보 표시
- 로고 및 브랜딩
- 레이아웃 및 형식
- 포함할 정보

## 영수증 재인쇄 {#reprinting-receipts}

이전 주문에 대한 영수증을 재인쇄하려면:

1. [주문 화면](/orders)로 이동 (Pro 기능)
2. 주문 찾기
3. 주문을 열기 위해 클릭
4. 영수증 옵션을 사용하여 다시 인쇄
