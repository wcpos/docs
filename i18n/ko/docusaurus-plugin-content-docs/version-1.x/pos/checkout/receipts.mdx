결제가 성공적으로 처리된 후, 영수증 모달이 표시됩니다. 여기서는 완료된 주문의 요약을 보여주며, 영수증을 인쇄하거나 이메일로 보낼 수 있는 옵션을 제공합니다.

{/* 스크린샷: receipt-modal.png */}

## 영수증 내용

영수증에는 다음 정보가 표시됩니다:

### 상점 정보
- **상점 이름**
- **주소**
- **전화번호**
- **이메일**
- **웹사이트**
- **로고** (구성된 경우)

### 영수증 헤더
- **영수증 유형** (예: "세금 영수증")
- **주문 번호**
- **날짜 및 시간**
- **캐셔 이름**
- **지불 방법**

### 주문 세부 사항
- **항목 내역**:
  - 제품 이름
  - 변형 속성 (해당되는 경우)
  - 수량
  - 가격
  - 총액

### 합계
- **소계**
- **세금** (해당되는 경우)
- **총액**
- **지급 금액** (현금의 경우)
- **거스름돈** (현금의 경우)

### 고객 정보
- **청구 주소**
- **배송 주소** (다른 경우)

## 영수증 작업

### 닫기
영수증 모달을 종료하고 POS로 돌아갑니다.

### 이메일 영수증
고객의 이메일 주소로 영수증을 보냅니다:

1. **이메일 영수증** 클릭
2. 이메일 주소 확인
3. WooCommerce의 이메일 시스템을 통해 영수증이 전송됩니다

고객은 주문과 연관된 이메일 주소가 있어야 합니다.

### 영수증 인쇄
영수증의 물리적 사본을 인쇄합니다:

1. **영수증 인쇄** 클릭
2. 브라우저의 인쇄 대화 상자가 열립니다
3. 영수증 프린터를 선택하고 인쇄합니다

:::tip 영수증 프린터
WCPOS는 브라우저에서 접근 가능한 모든 프린터와 함께 작업합니다. 최상의 결과를 위해 다음을 사용하세요:
- 전용 영수증 프린터 (열전사 프린터가 일반적입니다)
- 프린터 설정에서 용지 크기 구성
- 빠른 인출을 위한 인쇄 프리셋 설정
:::

## 자동 영수증 옵션

[장바구니 표시 설정](/pos/cart#display-settings)에서 다음을 구성할 수 있습니다:

- **결제 후 자동으로 영수증 표시** - 결제 후 즉시 영수증 모달을 엽니다.
- **결제 후 자동으로 영수증 인쇄** - 영수증이 표시될 때 자동으로 인쇄를 트리거합니다.

이 옵션은 대량 체크아웃 워크플로우를 빠르게 합니다.

## 영수증 사용자 정의

영수증 외관은 WordPress의 템플릿에 의해 제어됩니다:

`WP 관리 > POS > 템플릿`

[영수증 템플릿](/templates/receipt-templates)에서 사용자 정의에 대한 세부정보를 확인하세요:

- 상점 정보 표시
- 로고 및 브랜딩
- 레이아웃 및 서식
- 포함할 정보

## 영수증 재인쇄

이전 주문의 영수증을 재인쇄하려면:

1. [주문 화면](/orders)으로 이동 (Pro 기능)
2. 주문을 찾습니다
3. 주문을 열기 위해 클릭합니다
4. 영수증 옵션을 사용하여 다시 인쇄합니다
