결제가 성공적으로 처리된 후, 영수증 모달이 표시됩니다. 완료된 주문의 요약과 영수증을 인쇄하거나 이메일로 보내는 옵션을 제공합니다.

{/* Screenshot: receipt-modal.png */}

## 영수증 내용

영수증에는 다음이 표시됩니다:

### 매장 정보
- **매장 이름**
- **주소**
- **전화번호**
- **이메일**
- **웹사이트**
- **로고** (구성된 경우)

### 영수증 헤더
- **영수증 유형** (예: "세금 영수증")
- **주문 번호**
- **날짜 및 시간**
- **계산원 이름**
- **결제 방법**

### 주문 상세 정보
- **품목**:
  - 제품명
  - 변형 속성 (해당되는 경우)
  - 수량
  - 가격
  - 총액

### 합계
- **소계**
- **세금** (해당되는 경우)
- **총액**
- **받은 금액** (현금 결제 시)
- **거스름돈** (현금 결제 시)

### 고객 정보
- **청구 주소**
- **배송 주소** (다른 경우)

## 영수증 작업

### 닫기
영수증 모달을 닫고 POS로 돌아갑니다.

### 영수증 이메일 발송
고객의 이메일 주소로 영수증을 보냅니다:

1. **영수증 이메일 발송** 클릭
2. 이메일 주소 확인
3. WooCommerce의 이메일 시스템을 통해 영수증 발송

고객은 주문과 연결된 이메일 주소가 있어야 합니다.

### 영수증 인쇄
영수증의 물리적 사본을 인쇄합니다:

1. **영수증 인쇄** 클릭
2. 브라우저의 인쇄 대화 상자가 열림
3. 영수증 프린터 선택 및 인쇄

:::tip 영수증 프린터
WCPOS는 브라우저에서 접근 가능한 모든 프린터와 호환됩니다. 최상의 결과를 위해:
- 전용 영수증 프린터 사용 (열 프린터가 일반적)
- 프린터 설정에서 용지 크기 구성
- 빠른 인쇄를 위한 인쇄 사전 설정 구성
:::

## 자동 영수증 옵션

[카트 표시 설정](/pos/cart#display-settings)에서 다음을 구성할 수 있습니다:

- **결제 후 자동으로 영수증 표시** - 결제 즉시 영수증 모달 열기
- **결제 후 자동으로 영수증 인쇄** - 영수증 표시 시 자동으로 인쇄 트리거

이러한 옵션은 고볼륨 계산대 워크플로우를 가속화합니다.

## 영수증 사용자 정의

영수증 모양은 WordPress의 템플릿으로 제어됩니다:

`WP 관리자 > POS > 템플릿`

다음 사항을 사용자 정의하는 방법은 [영수증 템플릿](/templates/receipt-templates)을 참조하세요:

- 매장 정보 표시
- 로고 및 브랜딩
- 레이아웃 및 서식
- 포함할 정보

## 영수증 재인쇄

과거 주문의 영수증을 다시 인쇄하려면:

1. [주문 화면](/orders)으로 이동 (Pro 기능)
2. 주문 찾기
3. 주문 열기
4. 영수증 옵션을 사용하여 다시 인쇄
