판매를 완료할 준비가 되면 **체크아웃** 버튼을 클릭하여 체크아웃 모달을 엽니다. 여기서 결제를 처리하고 주문을 완료합니다.

{/* 스크린샷: checkout-modal.png */}

## 체크아웃 모달 개요

체크아웃 모달에는 다음이 표시됩니다:

- **주문 번호** - WooCommerce 주문 ID
- **지불할 금액** - 총 청구 금액
- **체크아웃 설정** 버튼 - 표시 문제 해결
- **계산원** - 주문을 처리하는 사람
- **고객** - 이 주문의 고객 (클릭 가능한 링크)
- **쿠폰 코드** 필드 - 할인 코드 적용
- **주문 요약** - 제품, 수량 및 총액
- **결제 방법** - 사용 가능한 결제 옵션
- **취소 / 결제 처리** 버튼

## 결제 방법

### 무료 버전에서 사용 가능

WCPOS의 무료 버전에는 두 가지 결제 게이트웨이가 포함되어 있습니다:

- **현금** - 지불된 금액 및 거스름돈 계산기
- **카드** - 외부 카드 단말기용

### 추가 게이트웨이 (Pro)

:::info Pro 기능
추가 결제 게이트웨이는 [WCPOS Pro](/getting-started/pro-license)를 요구합니다.
:::

Pro를 사용하면 다음을 활성화할 수 있습니다:

- **Stripe 단말기** - Stripe 카드 리더기와의 직접 통합
- **SumUp 단말기** - SumUp 카드 리더기와의 통합
- **사용자 지정 게이트웨이** - 자신만의 결제 통합 생성

게이트웨이 구성에 대한 자세한 내용은 [결제](/payment)를 참조하십시오.

### 결제 방법 선택하기

결제 방법을 클릭하여 선택합니다. 관련 필드를 보여 주도록 양식이 업데이트됩니다:

**현금:**
- **지불된 금액** - 고객이 제공하는 금액을 입력합니다
- **거스름돈** - 자동으로 계산된 거스름돈 반환

{/* 스크린샷: checkout-cash-payment.png */}

**카드:**
- 외부 카드 단말기에서 결제를 처리합니다
- 결제를 완료하려면 결제 처리 클릭

## 쿠폰 코드

할인을 적용하려면:

1. 필드에 쿠폰 코드를 입력합니다
2. **쿠폰 적용** 클릭
3. 할인 금액이 주문 요약에 나타납니다

쿠폰은 WooCommerce에서 구성해야 합니다 (`WP Admin > 마케팅 > 쿠폰`).

## 결제 처리

1. 결제 방법을 선택합니다
2. 현금을 사용하는 경우 지불된 금액을 입력합니다
3. **결제 처리** 클릭
4. 주문이 완료되고 [영수증](/pos/checkout/receipts)이 표시됩니다

## 체크아웃 설정 (문제 해결)

체크아웃 모달은 WooCommerce 주문 결제 페이지를 표시하기 위해 iframe/webview를 사용합니다. 이것은 WooCommerce의 기존 결제 인프라를 활용하므로 WooCommerce와 호환되는 모든 결제 게이트웨이가 POS에서 작동해야 합니다.

그러나 테마 및 플러그인 스크립트가 간섭할 수 있습니다. 문제를 해결하려면 **체크아웃 설정**을 클릭하십시오:

### 모든 스타일 및 스크립트 비활성화

- **wp_head 비활성화** - WordPress 헤더에서 모든 스크립트/스타일 제거
- **wp_footer 비활성화** - WordPress 바닥글에서 모든 스크립트/스타일 제거

### 선택된 스타일 비활성화

표시 문제를 일으킬 수 있는 CSS를 선택적으로 비활성화합니다:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- 기타.

### 선택된 스크립트 비활성화

결제 게이트웨이에 간섭할 수 있는 JavaScript를 선택적으로 비활성화합니다:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- 기타.

:::tip
결제 게이트웨이가 올바르게 표시되지 않는 경우:
1. 먼저 테마 스타일을 비활성화해 보십시오
2. 필요한 WooCommerce 스크립트를 비활성화해 보십시오
3. 결제 게이트웨이에 필요한 스크립트를 비활성화하지 않도록 주의하십시오
:::

## 취소

주문을 완료하지 않고 체크아웃 모달을 닫으려면 **취소**를 클릭합니다. 주문은 열린 장바구니로 남아 있습니다.
