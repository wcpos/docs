판매를 완료할 준비가 되면 **결제** 버튼을 클릭하여 결제 모달을 엽니다. 여기서 결제를 처리하고 주문을 완료합니다.

{/* Screenshot: checkout-modal.png */}

## 결제 모달 개요

결제 모달에는 다음이 표시됩니다:

- **주문 번호** - WooCommerce 주문 ID
- **결제 금액** - 총 지불 금액
- **결제 설정** 버튼 - 표시 문제 해결
- **계산원** - 주문을 처리하는 사람
- **고객** - 이 주문의 고객 (클릭 가능한 링크)
- **쿠폰 코드** 필드 - 할인 코드 적용
- **주문 요약** - 상품, 수량 및 총계
- **결제 방법** - 사용 가능한 결제 옵션
- **취소 / 결제 처리** 버튼

## 결제 방법

### 무료 버전에서 사용 가능

WCPOS 무료 버전에는 두 가지 결제 게이트웨이가 포함되어 있습니다:

- **현금** - 현금 입력 및 거스름돈 계산기 포함
- **카드** - 외부 카드 단말기용

### 추가 게이트웨이 (Pro)

:::info Pro 기능
추가 결제 게이트웨이는 [WCPOS Pro](/getting-started/pro-license)가 필요합니다.
:::

Pro를 사용하면 다음을 활성화할 수 있습니다:

- **Stripe 단말기** - Stripe 카드 리더기와 직접 통합
- **SumUp 단말기** - SumUp 카드 리더기 통합
- **맞춤 게이트웨이** - 자체 결제 통합 생성

게이트웨이 구성에 대한 자세한 내용은 [결제](/payment)를 참조하세요.

### 결제 방법 선택

결제 방법을 클릭하여 선택합니다. 양식이 업데이트되어 관련 필드가 표시됩니다:

**현금:**
- **현금 입력** - 고객이 제공한 금액 입력
- **거스름돈** - 자동으로 계산된 반환할 거스름돈

{/* Screenshot: checkout-cash-payment.png */}

**카드:**
- 외부 카드 단말기에서 결제 처리
- 결제 처리를 클릭하여 완료

## 쿠폰 코드

할인을 적용하려면:

1. 쿠폰 코드 필드에 코드 입력
2. **쿠폰 적용** 클릭
3. 할인이 주문 요약에 표시됨

쿠폰은 WooCommerce(`WP 관리자 > 마케팅 > 쿠폰`)에서 구성해야 합니다.

## 결제 처리

1. 결제 방법 선택
2. 현금의 경우, 입력된 금액 입력
3. **결제 처리** 클릭
4. 주문 완료 후 [영수증](/pos/checkout/receipts) 표시

## 결제 설정 (문제 해결)

결제 모달은 WooCommerce 주문 결제 페이지를 iframe/웹뷰로 표시합니다. 이는 WooCommerce의 기존 결제 인프라를 활용하므로 WooCommerce에서 작동하는 모든 결제 게이트웨이가 POS에서도 작동해야 합니다.

그러나 테마와 플러그인 스크립트가 때때로 방해할 수 있습니다. **결제 설정**을 클릭하여 문제를 해결하세요:

{/* Screenshot: checkout-settings.png */}

### 모든 스타일 및 스크립트 비활성화

- **wp_head 비활성화** - WordPress 헤더의 모든 스크립트/스타일 제거
- **wp_footer 비활성화** - WordPress 바닥글의 모든 스크립트/스타일 제거

### 선택한 스타일 비활성화

표시 문제를 일으킬 수 있는 CSS를 선택적으로 비활성화:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- 기타

### 선택한 스크립트 비활성화

결제 게이트웨이를 방해할 수 있는 JavaScript를 선택적으로 비활성화:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- 기타

:::tip
결제 게이트웨이가 제대로 표시되지 않는 경우:
1. 먼저 테마 스타일 비활성화 시도
2. 그런 다음 필요하지 않은 WooCommerce 스크립트 비활성화 시도
3. 결제 게이트웨이에 필요한 스크립트는 비활성화하지 않도록 주의
:::

## 취소

**취소**를 클릭하여 주문을 완료하지 않고 결제 모달을 닫습니다. 주문은 열린 장바구니로 남습니다.
