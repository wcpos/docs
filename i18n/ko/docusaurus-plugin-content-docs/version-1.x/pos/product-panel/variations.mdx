WooCommerce의 가변 상품은 다양한 옵션(크기, 색상, 재질 등)을 가진 상품입니다. WCPOS는 가변 상품을 효율적으로 작업할 수 있는 여러 방법을 제공합니다.

## 가변 상품 식별하기

제품 패널에서 가변 상품은 다음으로 구분됩니다:

- **화살표 버튼** (<Icon name="circle-arrow-right" color="green" />) 단순 상품에 사용되는 플러스 버튼 (<Icon name="circle-plus" color="green" />) 대신 표시
- 상품명 아래에 표시되는 **속성 옵션** (예: "색상: 파랑, 초록, 빨강")
- 모든 옵션을 인라인으로 보기 위한 **"확장" 링크**

## 가변 상품을 장바구니에 추가하기

### 방법 1: 빠른 팝오버

가변 상품의 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표 버튼을 클릭하여 각 속성에 대한 드롭다운 메뉴가 있는 팝오버를 엽니다:

1. <Icon name="circle-arrow-right" color="green" /> 버튼 클릭
2. 각 드롭다운에서 옵션 선택 (예: 크기: 대형, 색상: 파랑)
3. 일치하는 옵션 상품이 장바구니에 추가됨

정확히 필요한 옵션을 알고 있을 때 가장 빠른 방법입니다.

{/* Screenshot: variation-popover.png */}

### 방법 2: 인라인으로 확장

제품 목록에서 가변 상품의 **"확장"** 링크를 클릭하여 모든 옵션을 직접 표시:

1. 상품명 아래 **확장** 클릭
2. 모든 옵션이 상위 상품 아래 별도의 행으로 표시
3. 원하는 옵션의 <Icon name="circle-plus" color="green" /> 버튼을 클릭하여 장바구니에 추가
4. **접기**를 클릭하여 옵션 다시 숨기기

모든 사용 가능한 옵션, 개별 재고 수준 및 가격을 보고 싶을 때 유용합니다.

{/* Screenshot: variations-expanded.png */}

### 방법 3: 바코드 스캔

각 상품 옵션에는 고유한 바코드가 있을 수 있습니다. 옵션의 바코드를 스캔하면:

- 정확히 한 옵션과 일치하면 해당 옵션이 직접 장바구니에 추가됨
- 여러 옵션이 바코드를 공유하면 검색 창에 일치하는 결과가 표시됨

자세한 내용은 [바코드 스캔](/pos/product-panel/barcode-scanning)을 참조하세요.

## 옵션 정보

각 옵션은 다음을 표시할 수 있습니다:

- **옵션 속성** - 이 옵션의 특정 선택 사항 (예: "크기: 대형, 색상: 파랑")
- **SKU** - 옵션의 고유 식별자 (상위 상품과 다를 수 있음)
- **재고 수량** - 옵션의 개별 재고 수준
- **가격** - 옵션은 상위 상품과 다른 가격을 가질 수 있음

## 장바구니의 옵션

옵션이 장바구니에 추가되면 다음이 표시됩니다:

- 상위 상품의 **상품명**
- 이름 아래의 **옵션 속성**
- 해당 옵션의 **가격**

장바구니의 다른 상품과 마찬가지로 라인 항목을 편집할 수 있습니다.

## 옵션 선택을 위한 팁

### 바코드 스캔 사용

개별 바코드가 있는 옵션이 있다면(권장), 스캔이 장바구니에 추가하는 가장 빠른 방법입니다.

### 표시 옵션 구성

제품 패널 [표시 설정](/pos/product-panel#display-settings)에서 **속성** 옵션을 활성화하여 옵션 확장 없이 상품 목록에 직접 옵션 속성을 표시할 수 있습니다.

### 키보드 탐색

옵션 팝오버가 열려 있을 때 키보드 화살표 키로 옵션을 탐색하고 Enter로 선택할 수 있습니다.

## 자주 묻는 질문

<details>
<summary>특정 옵션을 볼 수 없는 이유는?</summary>

다음 사항을 확인하세요:
- 발행되었고 초안 상태가 아닌지
- "재고 있음"으로 설정되거나 재고 수량 > 0인지 (재고 관리가 활성화된 경우)
- 필요한 모든 속성이 정의되어 있는지

또한 옵션이 재고 부족인 경우 "재고 있음" 필터로 상품이 제외되지 않았는지 확인하세요.

</details>

<details>
<summary>모든 속성을 선택하지 않고 옵션을 추가할 수 있나요?</summary>

아니오, 특정 옵션을 식별하려면 각 속성에 대한 옵션을 선택해야 합니다. 상품에 크기와 색상 속성이 있다면, 장바구니에 추가하기 전에 두 가지를 모두 선택해야 합니다.

</details>

<details>
<summary>개별 옵션에 바코드를 추가하려면 어떻게 하나요?</summary>

WooCommerce에서 가변 상품을 편집하고 각 옵션을 확장합니다. 각 옵션의 바코드 필드(구성에 따라 `_sku` 또는 `_global_unique_id`)에 고유한 바코드를 추가할 수 있습니다.

</details>
