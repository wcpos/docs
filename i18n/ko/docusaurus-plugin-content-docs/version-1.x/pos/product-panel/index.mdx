제품 패널은 POS 화면의 왼쪽 부분으로, 장바구니에 추가할 제품을 검색하고 탐색하며 선택하는 곳입니다.

{/* 스크린샷: pos-product-panel.png */}

## 인터페이스 개요

### 검색 바

패널 상단에 있는 검색 바를 통해 다음과 같은 방법으로 제품을 신속하게 찾을 수 있습니다.

- **제품 이름** - 제품 이름의 일부를 입력합니다.
- **SKU** - 재고 관리 단위로 검색합니다.
- **바코드** - 바코드 번호를 스캔하거나 입력합니다.

자세한 정보는 [검색 및 필터링](/pos/product-panel/search-filtering)을 참조하세요.

### 디스플레이 설정

검색 바 옆의 **슬라이더 아이콘** (<Icon name="sliders" />)을 클릭하여 디스플레이 설정을 엽니다. 이를 통해 제품 목록에 표시할 열과 정보를 사용자 정의할 수 있습니다.

### 필터 버튼

검색 바 아래의 필터 버튼을 통해 제품을 세분화할 수 있습니다.

- **재고 있음** - 사용 가능한 재고가 있는 제품만 표시
- **추천 제품** - WooCommerce에서 추천 표시된 제품만 표시
- **세일 중** - 현재 세일 중인 제품만 표시
- **카테고리** - 제품 카테고리로 필터링
- **태그** - 제품 태그로 필터링
- **브랜드** - 브랜드(브랜드 플러그인을 사용하는 경우)로 필터링

활성화된 필터는 강조 표시됩니다. 필터를 다시 클릭하면 제거됩니다.

### 제품 목록

주요 영역은 스크롤 가능한 목록으로 제품을 표시합니다. 각 행은 다음을 보여줍니다.

- **제품 이미지** - 제품의 썸네일
- **제품 이름** - 선택 사항인 세부 정보(재고, SKU, 카테고리 등)와 함께
- **가격** - 현재 판매 가격(적용 가능한 경우 세일 가격 포함)
- **작업 버튼** - 장바구니에 추가하기 (<Icon name="circle-plus" color="green" />) 또는 변형 선택하기 (<Icon name="circle-arrow-right" color="green" />)

### 제품 유형

- **단순 제품**은 장바구니에 직접 추가할 수 있는 녹색 <Icon name="circle-plus" color="green" /> 버튼을 표시합니다.
- **변형 제품**은 변형 선택기를 열 수 있는 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시합니다.

자세한 내용은 [변형 제품](/pos/product-panel/variations)을 참조하세요.

### 바닥글

제품 패널 하단:

- **세금 상태** - 현재 세금 계산 기준을 보여줍니다(예: "세금 기준: 상점 기본 주소"). 클릭하여 활성 세금률을 볼 수 있습니다.
- **제품 수** - 표시되는 제품 수를 보여줍니다(예: "17개 중 10개 표시")
- **동기화 버튼** - 서버에서 제품을 새로 고침합니다.

## 디스플레이 설정

슬라이더 아이콘(<Icon name="sliders" />)을 클릭하여 제품 패널 디스플레이를 사용자 정의합니다.

### 재고 없는 제품 표시

현재 재고가 없는 제품이 표시될지 여부를 제어합니다.

- **활성화**: 재고가 없는 제품이 여전히 보이지만 장바구니에 추가할 수 없습니다.
- **비활성화**: 재고가 없는 제품이 목록에서 숨겨집니다.

### 열

제품 목록에 표시할 열을 구성합니다:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **이미지** | 제품 썸네일 | - |
| **제품** | 제품 이름 및 세부 정보 | 재고, SKU, 바코드, 카테고리, 태그, 브랜드, 속성, 메타 데이터 |
| **SKU** | 재고 관리 단위 (별도 열) | - |
| **바코드** | 제품 바코드 (별도 열) | - |
| **유형** | 제품 유형 (단순, 변형 등) | - |
| **재고** | 현재 재고 수량 | - |
| **판매 비용** | 제품 원가 | - |
| **가격** | 판매 가격 | 세금 정보, 세일 표시기 |
| **작업** | 장바구니 추가 버튼 | - |

### 메타 데이터 키

제품이 장바구니에 추가될 때 주문 항목으로 전송해야 할 제품 메타 데이터 키를 지정합니다.

쉼표로 구분된 메타 키 목록을 입력하세요:

```
_bottle_deposit,_custom_field,_tracking_code
```

**용도:**
- 병 보증금 플러그인
- 사용자 정의 제품 필드
- 제3자 통합 데이터
- 규정 준수/추적 정보

:::note
메타 키는 대소문자를 구분하며 제품의 메타 데이터에 저장된 정확한 형식과 일치해야 합니다.
:::

### 기본 설정 복원

모든 디스플레이 설정을 기본값으로 재설정하려면 클릭합니다.

## 장바구니에 제품 추가

### 단순 제품

녹색 <Icon name="circle-plus" color="green" /> 버튼을 클릭하여 단순 제품을 장바구니에 추가합니다. 각 클릭은 한 단위가 추가됩니다.

### 변형 제품

변형 제품(예: 사이즈 및 색상 옵션이 있는 티셔츠)은 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시합니다. 클릭하여:

1. **빠른 팝오버** - 드롭다운에서 변형 선택
2. **인라인 확장** - "확장"을 클릭하여 목록의 모든 변형 보기

자세한 내용은 [변형 제품](/pos/product-panel/variations)을 참조하세요.

### 바코드 스캐닝

USB 또는 블루투스 바코드 스캐너를 연결하여 제품을 빠르게 추가합니다. 바코드를 스캔하면 WCPOS가 자동으로 일치하는 제품을 검색하여 추가합니다.

설정 지침은 [바코드 스캐닝](/pos/product-panel/barcode-scanning)을 참조하세요.
