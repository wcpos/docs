제품 패널은 POS 화면의 왼쪽 부분으로, 제품을 검색, 탐색, 선택하여 장바구니에 추가할 수 있는 영역입니다.

{/* Screenshot: pos-product-panel.png */}

## 인터페이스 개요

### 검색 창

패널 상단의 검색 창을 통해 다음과 같이 빠르게 제품을 찾을 수 있습니다:

- **제품명** - 제품명의 일부를 입력
- **SKU** - 재고 관리 코드로 검색
- **바코드** - 바코드 번호 스캔 또는 입력

자세한 내용은 [검색 & 필터링](/pos/product-panel/search-filtering)을 참조하세요.

### 디스플레이 설정

검색 창 옆의 **슬라이더 아이콘**(<Icon name="sliders" />)을 클릭하여 디스플레이 설정을 엽니다. 이를 통해 제품 목록에 표시되는 열과 정보를 사용자 지정할 수 있습니다.

### 필터 버튼

검색 창 아래의 필터 버튼을 사용하여 제품을 좁힐 수 있습니다:

- **재고 있음** - 사용 가능한 재고가 있는 제품만 표시
- **추천 상품** - WooCommerce에서 추천으로 표시된 제품 표시
- **할인 중** - 현재 할인 중인 제품 표시
- **카테고리** - 제품 카테고리로 필터링
- **태그** - 제품 태그로 필터링
- **브랜드** - 브랜드 플러그인 사용 시 브랜드로 필터링

활성화된 필터는 강조 표시됩니다. 필터를 다시 클릭하면 제거됩니다.

### 제품 목록

주요 영역에는 스크롤 가능한 제품 목록이 표시됩니다. 각 행에는 다음이 표시됩니다:

- **제품 이미지** - 제품 썸네일
- **제품명** - 선택적 세부 정보 포함 (재고, SKU, 카테고리 등)
- **가격** - 현재 판매 가격 (해당되는 경우 할인 가격 표시)
- **작업 버튼** - 장바구니에 추가 (<Icon name="circle-plus" color="green" />) 또는 옵션 선택 (<Icon name="circle-arrow-right" color="green" />)

### 제품 유형

- **단순 제품**은 장바구니에 직접 추가하기 위한 녹색 <Icon name="circle-plus" color="green" /> 버튼을 표시
- **가변 제품**은 옵션 선택기를 열기 위한 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시

자세한 내용은 [가변 제품](/pos/product-panel/variations)을 참조하세요.

### 바닥글

제품 패널 하단에는:

- **세금 상태** - 현재 세금 계산 기준 표시 (예: "세금 기준: 상점 기본 주소"). 클릭하여 활성 세금 요율 확인
- **제품 수** - 표시된 제품 수 표시 (예: "17개 중 10개 표시")
- **동기화 버튼** - 서버에서 제품 새로고침

## 디스플레이 설정

슬라이더 아이콘(<Icon name="sliders" />)을 클릭하여 제품 패널 디스플레이를 사용자 지정합니다.

### 품절 제품 표시

현재 품절된 제품의 표시 여부를 제어합니다.

- **활성화**: 품절 제품이 계속 표시되지만 장바구니에 추가할 수 없음
- **비활성화**: 품절 제품이 목록에서 숨겨짐

### 열

제품 목록에 표시되는 열 구성:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **이미지** | 제품 썸네일 | - |
| **제품** | 제품명 및 세부 정보 | 재고, SKU, 바코드, 카테고리, 태그, 브랜드, 속성, 메타 데이터 |
| **SKU** | 재고 관리 코드 (별도 열) | - |
| **바코드** | 제품 바코드 (별도 열) | - |
| **유형** | 제품 유형 (단순, 가변 등) | - |
| **재고** | 현재 재고 수량 | - |
| **매출원가** | 제품 원가 | - |
| **가격** | 판매 가격 | 세금 정보, 할인 표시기 |
| **작업** | 장바구니 추가 버튼 | - |

### 메타 데이터 키

제품을 장바구니에 추가할 때 주문 항목으로 전송되어야 하는 제품 메타 데이터 키를 지정합니다.

쉼표로 구분된 메타 키 목록을 입력하세요:

```
_bottle_deposit,_custom_field,_tracking_code
```

**사용 사례:**
- 보증금 플러그인
- 맞춤 제품 필드
- 타사 통합 데이터
- 규정/추적 정보

:::note
메타 키는 대소문자를 구분하며 제품의 메타 데이터에 저장된 내용과 정확히 일치해야 합니다.
:::

### 기본 설정 복원

모든 디스플레이 설정을 원래 기본값으로 재설정하려면 클릭하세요.

## 장바구니에 제품 추가

### 단순 제품

단순 제품을 장바구니에 추가하려면 녹색 <Icon name="circle-plus" color="green" /> 버튼을 클릭하세요. 클릭할 때마다 한 개의 단위가 추가됩니다.

### 가변 제품

가변 제품(예: 크기와 색상 옵션이 있는 티셔츠)은 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시합니다. 클릭하여:

1. **빠른 팝오버** - 드롭다운에서 옵션 선택
2. **인라인 확장** - "확장"을 클릭하여 목록에서 모든 옵션 확인

자세한 내용은 [가변 제품](/pos/product-panel/variations)을 참조하세요.

### 바코드 스캔

USB 또는 블루투스 바코드 스캐너를 연결하여 빠르게 제품을 추가하세요. 바코드를 스캔하면 WCPOS가 자동으로 일치하는 제품을 검색하고 추가합니다.

설정 지침은 [바코드 스캔](/pos/product-panel/barcode-scanning)을 참조하세요.
