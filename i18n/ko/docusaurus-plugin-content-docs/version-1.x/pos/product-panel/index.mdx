제품 패널은 POS 화면의 왼쪽에 위치하여 제품을 검색하고 찾아보고 선택하여 장바구니에 추가할 수 있는 공간입니다.

{/* Screenshot: pos-product-panel.png */}

## 인터페이스 개요 {#interface-overview}

### 검색 바 {#search-bar}

패널 상단에 있는 검색 바를 사용하여 다음을 통해 제품을 빠르게 찾을 수 있습니다:

- **제품 이름** - 제품 이름의 일부를 입력하세요.
- **SKU** - Stock Keeping Unit으로 검색합니다.
- **바코드** - 바코드 번호를 스캔하거나 입력합니다.

자세한 정보는 [검색 및 필터링](/pos/product-panel/search-filtering)을 참조하세요.

### 디스플레이 설정 {#display-settings}

검색 바 옆에 있는 **슬라이더 아이콘**(<Icon name="sliders" />)을 클릭하여 디스플레이 설정을 엽니다. 이를 통해 제품 목록에 표시될 열과 정보를 사용자화할 수 있습니다.

### 필터 버튼 {#filter-buttons}

검색 바 아래의 필터 버튼을 사용하여 제품을 좁힐 수 있습니다:

- **재고 있음** - 사용 가능한 재고가 있는 제품만 표시
- **특별한 제품** - WooCommerce에서 표시된 특별한 제품 표시
- **할인 중** - 현재 할인 중인 제품 표시
- **카테고리** - 제품 카테고리로 필터링
- **태그** - 제품 태그로 필터링
- **브랜드** - 브랜드(브랜드 플러그인을 사용하는 경우)로 필터링

활성 필터는 강조 표시됩니다. 필터를 다시 클릭하면 제거됩니다.

### 제품 목록 {#product-list}

주요 영역은 스크롤 가능한 목록으로 제품을 표시합니다. 각 행에는 다음이 표시됩니다:

- **제품 이미지** - 제품의 썸네일
- **제품 이름** - 선택적인 세부정보(재고, SKU, 카테고리 등) 포함
- **가격** - 현재 판매 가격(해당하는 경우 할인 가격 포함)
- **작업 버튼** - 장바구니에 추가(<Icon name="circle-plus" color="green" />) 또는 변형 선택(<Icon name="circle-arrow-right" color="green" />)

### 제품 유형 {#product-types}

- **단순 제품**은 장바구니에 직접 추가할 수 있도록 녹색 <Icon name="circle-plus" color="green" /> 버튼을 표시합니다.
- **변경 가능한 제품**은 변형 선택을 열기 위해 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시합니다.

자세한 내용은 [변경 가능한 제품](/pos/product-panel/variations)을 참조하세요.

### 바닥글 {#footer}

제품 패널의 하단에는 다음이 표시됩니다:

- **세금 상태** - 현재 세금 계산 기준을 표시합니다(예: "세금 기준: 가게 기본 주소"). 클릭하여 활성 세율을 확인합니다.
- **제품 수** - 표시된 제품 수를 보여 줍니다(예: "17개 중 10개 표시").
- **동기화 버튼**(<Icon name="arrow-rotate-right" />) - 서버에서 제품을 새로 고칩니다. **길게 누르기**하여 추가 옵션을 선택합니다:
  - **동기화** - 서버에서 표준 새로 고침
  - **지우고 새로 고침** - 로컬 데이터를 지우고 모든 것을 다시 불러옵니다.

### 품절 제품 표시 {#show-out-of-stock-products}

현재 품절인 제품이 표시될지 여부를 제어합니다.

- **사용**: 품절 제품은 여전히 보이지만 장바구니에 추가할 수는 없습니다.
- **사용 안 함**: 품절 제품은 목록에서 숨겨집니다.

### 열 {#columns}

제품 목록에 표시될 열을 구성합니다:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **이미지** | 제품 썸네일 | - |
| **제품** | 제품 이름 및 세부정보 | 재고, SKU, 바코드, 카테고리, 태그, 브랜드, 속성, 메타 데이터 |
| **SKU** | Stock Keeping Unit (별도의 열) | - |
| **바코드** | 제품 바코드(별도의 열) | - |
| **유형** | 제품 유형(단순, 변경 가능 등) | - |
| **재고** | 현재 재고 수량 | - |
| **매출 원가** | 판매된 상품 원가 | - |
| **가격** | 판매 가격 | 세금 정보, 할인 중 지시 |

### 메타 데이터 키 {#meta-data-keys}

제품이 장바구니에 추가될 때 주문 항목으로 전송되어야 하는 제품 메타 데이터 키를 지정합니다.

콤마로 구분된 메타 키 목록을 입력하세요:

```
_bottle_deposit,_custom_field,_tracking_code
```

**사용 사례:**
- 병 보증금 플러그인
- 사용자 정의 제품 필드
- 타사 통합 데이터
- 규정/추적 정보

:::note
메타 키는 대소문자를 구분하며 제품의 메타 데이터에 저장된 것과 정확하게 일치해야 합니다.
:::

### 기본 설정 복원 {#restore-default-settings}

모든 디스플레이 설정을 원래 기본값으로 재설정하려면 클릭합니다.

## 장바구니에 제품 추가 {#adding-products-to-cart}

### 단순 제품 {#simple-products}

녹색 <Icon name="circle-plus" color="green" /> 버튼을 클릭하여 단순 제품을 장바구니에 추가합니다. 클릭할 때마다 하나의 유닛이 추가됩니다.

### 변경 가능한 제품 {#variable-products}

변경 가능한 제품(예: 사이즈 및 색상 옵션이 있는 티셔츠)은 녹색 <Icon name="circle-arrow-right" color="green" /> 화살표를 표시합니다. 클릭하여:

1. **빠른 팝오버** - 드롭다운에서 변형 선택
2. **인라인 확장** - "확장"을 클릭하여 목록에 있는 모든 변형 보기

자세한 내용은 [변경 가능한 제품](/pos/product-panel/variations)을 참조하세요.

### 바코드 스캐닝 {#barcode-scanning}

USB 또는 블루투스 바코드 스캐너를 연결하여 제품을 빠르게 추가합니다. 바코드를 스캔하면 WCPOS가 자동으로 일치하는 제품을 검색하고 추가합니다.

설정 지침은 [바코드 스캐닝](/pos/product-panel/barcode-scanning)을 참조하세요.

## 일부 제품을 볼 수 없는 이유는 무엇인가요? {#why-cant-i-see-some-products}

제품 패널에서 제품이 누락되었으면 다음을 확인하세요:

- **POS 가시성** - "온라인 전용"으로 설정된 제품은 POS에 나타나지 않습니다. [POS 전용 제품](/products/pos-only-products)을 참조하세요.
- **재고 설정** - 디스플레이 설정에서 "품절 제품 표시"가 사용 안 함으로 설정되면 품절 품목이 숨겨집니다.
- **동기화 상태** - 새로 추가된 또는 최근 업데이트된 제품은 동기화가 필요할 수 있습니다. 동기화 버튼을 길게 눌러 **"지우고 새로 고침"**을 선택하세요.
- **필터** - 제품을 숨길 수 있는 활성 필터(카테고리, 태그 등)가 있는지 확인하세요.
