## 상품 메타데이터란? {#what-is-product-meta-data}

모든 WooCommerce 상품에는 이름과 가격 외에도 추가 필드를 포함할 수 있습니다. 예를 들면
크기, 각인 메시지, 공급업체 코드 또는 다른 플러그인에서 추가한 데이터가 있습니다.
WooCommerce는 이를 **메타데이터**로 저장합니다. 각 항목에는 **키**(필드 이름,
예: `engraving`)와 **값**(예: `"Happy Birthday"`)이 있습니다.

## 메타데이터 키 설정의 기능 {#what-the-meta-data-keys-setting-does}

기본적으로 상품을 장바구니에 추가하면 표준 세부 정보만 복사됩니다.
**메타데이터 키** 설정(상품 패널 → 설정)에서 어떤 상품
메타 키를 **장바구니 라인 항목에 복사할지** 선택할 수 있으며, 따라서 해당 메타 키가 주문에 저장됩니다.

예를 들어 목록에 `engraving` 항목을 추가하면, 각인 가능한
상품을 장바구니에 추가할 때마다 해당 각인 값이 주문으로 함께 전달됩니다.

## 키 선택 {#choosing-keys}

메타데이터 키 필드에 입력을 시작합니다:

- **제안**은 이 기기에 이미 동기화된 상품에서 발견된 메타 키입니다.
- 필요한 키가 목록에 없는 경우(예: 아직 동기화되지 않은 상품에만
  있는 경우) 정확한 키 이름을 입력하고 **"…"을 사용자 지정 키로 추가**를 선택합니다.
- 선택한 키는 칩으로 표시됩니다. 해당 ✕를 탭하여 제거합니다.

선택한 키는 다른 패널 설정과 함께 저장되며
테이블 및 그리드(타일) 보기에 모두 적용됩니다.

## 팁 {#tips}

- 메타 키는 대소문자를 구분하며 상품에 저장된 키와 정확히 일치해야 합니다.
- 밑줄로 시작하는 키(예: `_size`)는 WooCommerce의 "숨김"/내부
  메타데이터입니다. 데이터가 해당 위치에 있는 경우 여기에서도 계속 작동합니다.
