POS 화면은 제품 판매를 위한 주요 인터페이스입니다. WCPOS를 사용할 때 대부분의 시간을 보내는 곳입니다.

{/* Screenshot: pos-desktop-layout.png */}

## 화면 레이아웃 {#screen-layout}

### 헤더 {#header}

헤더 바에는 다음이 표시됩니다:

- **상점 이름** - 연결된 WooCommerce 상점의 이름
- **연결 상태 표시기** - 서버에 연결되면 녹색 점, 오프라인일 때는 노란색/빨간색
- **사용자 메뉴** - 클릭하여 설정에 액세스하거나, 사용자 전환 또는 로그아웃

### 탐색 드로어 {#navigation-drawer}

왼쪽 사이드바는 모든 주요 화면에 빠르게 접근할 수 있습니다:

| 아이콘 | 화면 | 설명 |
|------|--------|-------------|
| <Icon name="cash-register" /> | **POS** | 주요 판매 인터페이스 (현재 위치) |
| <Icon name="gift" /> | **제품** | 재고 관리 (Pro) |
| <Icon name="receipt" /> | **주문** | 주문 이력 (Pro) |
| <Icon name="users" /> | **고객** | 고객 관리 (Pro) |
| <Icon name="chart-line" /> | **보고서** | 판매 보고서 (Pro) |
| <Icon name="heart-pulse" /> | **로그** | 디버깅을 위한 시스템 로그 |
| <Icon name="circle-question" /> | **지원** | Discord 지원 채널 |

버전 번호는 드로어 하단에 표시됩니다.

## 반응형 레이아웃 {#responsive-layout}

WCPOS는 다양한 화면 크기에 적응합니다:

### 데스크탑 / 태블릿 (큰 화면) {#desktop-tablet-large-screens}

더 큰 화면에서 POS는 **두 개의 열 레이아웃**을 표시합니다:

- **왼쪽 열:** 제품 패널 - 제품 검색 및 탐색
- **오른쪽 열:** 장바구니 패널 - 현재 주문 보기 및 관리

열은 **크기 조절 가능** - 열 사이의 구분선을 드래그하여 비율을 조정합니다.

{/* Screenshot: pos-desktop-layout.png */}

### 모바일 (작은 화면) {#mobile-small-screens}

작은 화면에서는 POS가 **탭 기반 레이아웃**을 사용합니다:

- **제품 탭:** 제품을 탐색하고 장바구니에 추가
- **장바구니 탭:** 장바구니 보기, 품목 관리 및 결제

화면 하단의 탐색 바를 사용하여 탭 간에 전환합니다.

{/* Screenshot: pos-mobile-layout.png */}

## 주요 구성 요소 {#main-components}

### 제품 패널 {#product-panel}

POS의 왼쪽쪽은 [제품 패널](/pos/product-panel)로, 여기에서:

- 이름, SKU 또는 바코드로 제품 검색
- 재고 상태, 카테고리, 태그 등에 따라 제품 필터링
- 장바구니에 제품 추가

[제품 패널에 대해 더 알아보기 →](/pos/product-panel)

### 장바구니 패널 {#cart-panel}

POS의 오른쪽은 [장바구니 패널](/pos/cart)로, 여기에서:

- 현재 주문의 품목 보기
- 수량 및 가격 편집
- 고객 선택 또는 추가
- 주문 작업(메모, 무효, 체크아웃) 액세스

[장바구니 패널에 대해 더 알아보기 →](/pos/cart)

### 체크아웃 {#checkout}

판매를 완료할 준비가 되면 **체크아웃**을 클릭하여 [체크아웃 모달](/pos/checkout)을 엽니다:

- 쿠폰 코드 적용
- 결제 방법 선택
- 결제 처리
- 영수증 인쇄 또는 이메일

[체크아웃에 대해 더 알아보기 →](/pos/checkout)

## 키보드 단축키 {#keyboard-shortcuts}

WCPOS는 빠른 작동을 위한 키보드 단축키를 지원합니다. 설정 모달을 통해 전체 목록에 접근하거나 [키보드 단축키](/settings/store/hotkeys)를 확인하세요.

일반적인 단축키:

| 단축키 | 동작 |
|----------|--------|
| `Ctrl + F` | 검색 바 포커스 |
| `Ctrl + Shift + S` | 설정 열기 |
| `Escape` | 모달/대화 상자 닫기 |

## 연결 상태 {#connectivity}

헤더의 녹색 표시기는 연결 상태를 보여줍니다:

- **녹색** - WooCommerce 서버에 연결됨
- **노란색** - 연결 문제, 재시도 중
- **빨간색** - 오프라인 모드

오프라인 상태에서도 캐시된 제품 및 고객을 탐색하고 새로운 주문을 시작할 수 있습니다. 주문을 완료하려면 연결이 필요합니다.
