교대가 끝나면 금전함을 마감합니다. 현금을 계산하고, POS에 기록된 수취 금액과 비교하고, 카드 합계를 확인한 뒤 기록을 보관합니다. WCPOS는 금전함에 *있어야 할* 금액을 알려주며, 계산원이 직접 계산하고 수동으로 정산합니다.

:::info Pro 기능
정산에 사용하는 보고서 화면에는 [WCPOS Pro](/getting-started/pro-license)가 필요합니다. Pro가 없으면 `WP Admin → WooCommerce → Analytics` 에서 정산하세요.
:::

## 요약 {#the-short-version}

1. **판매 접수를 중지**하여 계산 중에 새 판매가 들어오지 않게 합니다.
2. **금전함을 계산**합니다 — 모든 권종을 확인합니다. 이것이 *계산된 현금*입니다.
3. **보고서를 열고**, 오늘(및 해당 계산원/매장)로 필터링한 다음 판매 요약에서 **현금** 및 **카드** 금액을 확인합니다.
4. **현금을 정산합니다:** *시작 시재 + 현금 판매 − 현금 환불 − 현금 회수액*은 계산된 현금과 일치해야 합니다.
5. **카드 정산:** 카드 단말기의 배치 합계가 POS 카드 합계와 일치해야 합니다. 단말기가 수동 방식이면 배치를 정산하세요.
6. **보고서를 인쇄하고**, 현금을 회수한 뒤 내일 사용할 거스름돈 준비금을 다시 넣고 서명해 마감합니다.

:::note 내장 금전함 세션 없음
WCPOS에는 시작 거스름돈 필드, 금전함 계산, 자동 차액 계산 기능이 없습니다 — 거스름돈 준비금과 현금 회수 내역은 계산대의 종이나 스프레드시트에 기록하세요.
:::
