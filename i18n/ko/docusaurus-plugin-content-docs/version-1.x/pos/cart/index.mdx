장바구니 패널은 현재 주문을 관리하는 POS 화면의 오른쪽 부분입니다. 여기에서 품목을 보고, 수량과 가격을 편집하고, 고객을 선택하고, 결제 진행을 할 수 있습니다.

{/* Screenshot: pos-cart-panel.png */}

## 인터페이스 개요

### 고객 선택기

장바구니 패널 상단에:

- **고객 배지** - 현재 고객을 표시합니다 (예: "게스트" 또는 고객 이름)
- **고객 추가 아이콘** (<Icon name="plus" />) - 새 고객을 생성합니다
- **표시 설정** (<Icon name="sliders" />) - 장바구니 열 및 옵션을 구성합니다

고객 배지를 클릭하여 다른 고객을 검색하고 선택합니다.

### 품목

주요 공간은 현재 주문의 품목을 표시합니다:

- **QTY** - 수량 (편집 가능)
- **이름** - 변형 속성이 포함된 제품 이름
- **가격** - 단가 (편집 가능)
- **총계** - 품목 총계

편집에 대한 자세한 내용은 [품목](/pos/cart/line-items)을 참조하십시오.

### 장바구니 추가 옵션

품목 아래에서 특수 품목을 추가할 수 있습니다:

- **기타 제품 추가** - 수동 가격 입력으로 사용자 정의 품목을 추가합니다
- **수수료 추가** - 수수료 항목을 추가합니다 (예: 선물 포장, 서비스 요금)
- **배송 추가** - 수동 비용 입력으로 배송 항목을 추가합니다

:::note 품목 유형
WooCommerce는 세 가지 유형의 품목을 사용합니다: `line_item` (제품), `fee_line` (수수료), `shipping_line` (배송). 현재 배송에 대해서는 수동 입력만 지원되며, POS는 배송 비용을 자동으로 계산하지 않습니다.
:::

### 주문 요약

- **소계** - 세금 및 수수료 전 총액
- **세금** - 계산된 세금 금액 (해당하는 경우)
- **총계** - 최종 주문 총액

### 주문 작업

장바구니 패널 하단에:

- **주문 노트** - 고객이 볼 수 있는 노트를 추가합니다
- **주문 메타** - 사용자 정의 메타데이터를 추가하거나 JSON을 봅니다
- **서버에 저장** - 상태 `pos-open`으로 WooCommerce에 주문을 저장합니다

자세한 내용은 [주문 작업](/pos/cart/order-actions)을 참조하십시오.

### 주요 버튼

- **무효화** (빨간색) - 현재 주문을 취소/삭제합니다
- **결제** (녹색) - 결제로 진행합니다

### 열린 주문

가장 하단에는 모든 열린 주문을 보여주는 캐러셀:

- 각 장바구니는 총액을 표시합니다
- 장바구니를 클릭하여 전환합니다
- 현재 장바구니는 강조 표시됩니다

자세한 내용은 [열린 주문](/pos/cart/open-orders)을 참조하십시오.

### 결제 후 영수증 자동 표시

활성화되면, 판매 완료 후 영수증 화면이 자동으로 표시됩니다.

### 결제 후 영수증 자동 인쇄

활성화되면, 표시될 때 자동으로 영수증이 인쇄됩니다. 이는 수동 인쇄 단계를 줄여 시간을 절약합니다.

### 빠른 할인

콤마로 구분된 할인 비율 목록이 빠른 바로가기 버튼으로 표시됩니다.

예를 들어, `5,10,15,20`을 입력하면, 전체 주문에 대해 한 번의 클릭으로 적용되는 비율 할인을 위한 네 개의 버튼이 추가됩니다.

### 열

장바구니에 표시할 열을 구성합니다:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **QTY** | 각 품목의 수량 | 분할 (품목 분할 허용) |
| **이름** | 제품 이름 | SKU |
| **가격** | 단위 가격 | 세일 중 표시기 |
| **정상 가격** | 할인되지 않은 가격 | - |
| **소계** | 품목 소계 (수량 × 가격) | 세금 금액 |
| **총계** | 품목 총계 | 세금 금액, 세일 중 표시기 |
| **작업** | 품목 제거 버튼 | - |

### 기본 설정 복원

모든 장바구니 표시 설정을 원래 기본값으로 재설정하려면 클릭합니다.

## 장바구니에 품목 추가하기

### 제품 패널에서

[제품 패널](/pos/product-panel)에서 제품을 클릭하여 장바구니에 추가합니다. 변형 제품의 경우, 먼저 변형을 선택합니다.

### 기타 제품

카탈로그에 없는 품목의 경우:

1. **기타 제품 추가** 클릭
2. 이름과 가격을 입력합니다
3. 품목이 장바구니에 추가됩니다

### 수수료

수수료 (예: 선물 포장)를 추가하려면:

1. **수수료 추가** 클릭
2. 이름과 금액을 입력합니다
3. 수수료가 별도의 품목으로 표시됩니다

### 배송

배송을 추가하려면:

1. **배송 추가** 클릭
2. 배송 방법 이름과 비용을 입력합니다
3. 배송이 주문에 추가됩니다

## 장바구니 품목 편집

모든 품목은 장바구니 내에서 직접 편집 가능합니다. 편집에 대한 자세한 내용은 [품목](/pos/cart/line-items)을 참조하십시오:

- 수량 편집
- 가격 변경
- 원시 JSON 데이터 보기 및 편집
- 품목 제거
