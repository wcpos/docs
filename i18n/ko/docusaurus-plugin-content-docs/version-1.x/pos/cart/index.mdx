장바구니 패널은 현재 주문을 관리하는 POS 화면의 오른쪽 영역입니다. 여기에서 품목을 보고, 수량과 가격을 편집하고, 고객을 선택한 다음 결제로 진행할 수 있습니다.

{/* Screenshot: pos-cart-panel.png */}

## 인터페이스 개요 {#interface-overview}

### 고객 선택기 {#customer-selector}

장바구니 패널 상단:

- **고객 배지** - 현재 고객(예: "게스트" 또는 고객 이름)을 표시합니다
- **고객 추가 아이콘** (<Icon name="plus" />) - 새 고객 생성
- **표시 설정** (<Icon name="sliders" />) - 장바구니 열 및 옵션 구성

고객 배지를 클릭하여 다른 고객을 검색하고 선택합니다.

### 품목 {#line-items}

기본 영역에는 현재 주문의 항목이 표시됩니다.

- **수량** - 수량(편집 가능)
- **이름** - 변형 속성이 포함된 상품명
- **가격** - 단가(편집 가능)
- **합계** - 품목 합계

편집에 대한 자세한 내용은 [품목](/pos/cart/line-items)을 참조하세요.

### 장바구니에 추가 옵션 {#add-to-cart-options}

품목 아래에서 특수 항목을 추가할 수 있습니다.

- **기타 상품 추가** - 수동 가격 입력으로 사용자 지정 항목 추가
- **수수료 추가** - 수수료 항목을 추가합니다(예: 선물 포장, 서비스 요금)
- **배송 추가** - 수동 비용 입력으로 배송 항목을 추가합니다

:::note 라인 항목 유형
WooCommerce는 세 가지 유형의 라인 항목을 사용합니다: `line_item`(상품), `fee_line`(수수료), `shipping_line`(배송). 현재 배송에는 수동 입력만 지원됩니다. POS는 배송비를 자동으로 계산하지 않습니다.
:::

### 주문 요약 {#order-summary}

- **소계** - 세금 및 수수료 적용 전 합계
- **세금** - 계산된 세액(해당하는 경우)
- **합계** - 최종 주문 합계

### 주문 작업 {#order-actions}

장바구니 패널 하단:

- **주문 메모** - 고객에게 표시되는 메모를 추가합니다
- **주문 메타** - 사용자 지정 메타데이터를 추가하거나 JSON 보기
- **서버에 저장** - 주문을 `pos-open` 상태로 WooCommerce에 저장

자세한 내용은 [주문 작업](/pos/cart/order-actions)을 참조하세요.

### 주요 버튼 {#main-buttons}

- **무효 처리**(빨간색) - 현재 주문을 취소/삭제
- **결제**(초록색) - 결제로 진행

### 미결 주문 {#open-orders}

맨 아래에는 모든 미결 주문을 표시하는 캐러셀이 있습니다.

- 각 장바구니에는 합계가 표시됩니다.
- 장바구니를 클릭하여 해당 장바구니로 전환합니다.
- 현재 장바구니가 강조 표시됩니다.

자세한 내용은 [미결 주문](/pos/cart/open-orders)을 참조하세요.

### 결제 완료 후 영수증 자동 표시 {#automatically-show-receipt-after-checkout}

활성화하면 판매 완료 후 영수증 화면이 자동으로 표시됩니다.

### 결제 완료 후 영수증 자동 인쇄 {#automatically-print-receipt-after-checkout}

활성화하면 영수증이 표시될 때 자동으로 인쇄됩니다. 수동 인쇄 단계를 줄여 시간을 절약할 수 있습니다.

### 빠른 할인 {#quick-discounts}

빠른 바로가기 버튼으로 표시되는 할인율의 쉼표로 구분된 목록입니다.

예를 들어 `5,10,15,20` 을 입력하면 한 번의 클릭으로 전체 주문에 백분율 할인을 적용하는 네 개의 버튼이 추가됩니다.

### 열 {#columns}

카트에 표시할 열을 구성합니다:

| 열 | 설명 |
|--------|-------------|
| **수량** | 각 항목의 수량 |
| **이름** | 상품 이름 |
| **가격** | 단위당 가격 |
| **정가** | 할인 전 가격 |
| **소계** | 라인 소계 (수량 × 가격) |
| **합계** | 라인 합계 |
| **작업** | 항목 제거 버튼 |

일부 열에는 추가 표시 옵션이 있습니다:

| 열 | 옵션 |
|--------|--------|
| **수량** | 분할(항목 분할 허용) |
| **이름** | SKU |
| **가격** | 할인 중 표시 |
| **소계** | 세금 금액 |
| **합계** | 세금 금액, 할인 중 표시 |

### 기본 설정 복원 {#restore-default-settings}

모든 장바구니 표시 설정을 원래 기본값으로 재설정하려면 클릭합니다.

## 장바구니에 항목 추가 {#adding-items-to-cart}

### 제품 패널에서 {#from-product-panel}

[제품 패널](/pos/product-panel)에서 상품을 클릭하여 장바구니에 추가합니다. 옵션 상품의 경우 먼저 옵션을 선택합니다.

### 기타 상품 {#miscellaneous-products}

카탈로그에 없는 항목의 경우:

1. **기타 제품 추가**를 클릭합니다
2. 이름과 가격을 입력합니다
3. 항목이 카트에 추가됩니다

### 수수료 {#fees}

수수료(예: 선물 포장)를 추가하려면:

1. **수수료 추가**를 클릭합니다
2. 이름과 금액을 입력합니다
3. 수수료가 별도의 항목 줄로 표시됩니다

### 배송 {#shipping}

배송을 추가하려면:

1. **배송 추가**를 클릭합니다
2. 배송 방법 이름과 비용을 입력합니다
3. 배송비가 주문에 추가됩니다

## 카트 항목 편집 {#editing-cart-items}

모든 라인 항목은 카트에서 직접 편집할 수 있습니다. 자세한 내용은 [라인 항목](/pos/cart/line-items)을 참조하세요.

- 수량 편집
- 가격 변경
- 원시 JSON 데이터 보기 및 편집
- 항목 제거
