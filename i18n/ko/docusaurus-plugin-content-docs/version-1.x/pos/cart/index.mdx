장바구니 패널은 현재 주문을 관리하는 POS 화면의 오른쪽에 있습니다. 여기에서 항목을 보고, 수량 및 가격을 편집하고, 고객을 선택하고, 결제로 진행할 수 있습니다.

{/* Screenshot: pos-cart-panel.png */}

## 인터페이스 개요

### 고객 선택기

장바구니 패널의 상단에:

- **고객 배지** - 현재 고객 표시 (예: "손님" 또는 고객 이름)
- **고객 추가 아이콘** (<Icon name="plus" />) - 새 고객 생성
- **디스플레이 설정** (<Icon name="sliders" />) - 장바구니 열 및 옵션 구성

고객 배지를 클릭하여 다른 고객을 검색하고 선택하세요.

### 라인 항목

주요 영역에는 현재 주문의 항목이 표시됩니다:

- **수량** - 수량 (편집 가능)
- **이름** - 변형 속성이 포함된 제품 이름
- **가격** - 단가 (편집 가능)
- **총액** - 항목 총액

편집에 대한 자세한 내용은 [라인 항목](/pos/cart/line-items)을 참조하세요.

### 장바구니 추가 옵션

라인 항목 아래에서 특별한 항목을 추가할 수 있습니다:

- **기타 제품 추가** - 수동 가격 입력으로 사용자 정의 항목 추가
- **수수료 추가** - 수수료 항목 추가 (예: 선물 포장, 서비스 요금)
- **배송 추가** - 수동 비용 입력으로 배송 항목 추가

:::note 항목 유형
WooCommerce는 세 가지 유형의 라인 항목을 사용합니다: `line_item` (제품), `fee_line` (수수료), `shipping_line` (배송). 현재 배송에 대해서는 수동 입력만 지원되며, POS가 배송 비용을 자동으로 계산하지 않습니다.
:::

### 주문 요약

- **소계** - 세금 및 수수료가 적용되기 전 총액
- **세금** - 계산된 세금 금액 (해당되는 경우)
- **총액** - 최종 주문 총액

### 주문 작업

장바구니 패널 하단에:

- **주문 메모** - 고객이 볼 수 있는 메모 추가
- **주문 메타** - 사용자 정의 메타데이터 추가 또는 JSON 보기
- **서버에 저장** - `pos-open` 상태로 WooCommerce에 주문 저장

자세한 내용은 [주문 작업](/pos/cart/order-actions)을 참조하세요.

### 주요 버튼

- **무효화** (빨간색) - 현재 주문 취소/삭제
- **결제** (녹색) - 결제로 진행

### 열려 있는 주문

맨 아래에는 모든 열린 주문이 표시되는 캐러셀:

- 각 장바구니는 총액을 표시
- 장바구니를 클릭하여 전환
- 현재 장바구니는 강조 표시

자세한 내용은 [열려 있는 주문](/pos/cart/open-orders)을 참조하세요.

### 결제 후 자동 영수증 표시

활성화하면 판매 완료 후 자동으로 영수증 화면이 표시됩니다.

### 결제 후 자동 영수증 인쇄

활성화하면 표시되는 영수증이 자동으로 인쇄됩니다. 이렇게 하면 수동 인쇄 단계를 줄여 시간을 절약할 수 있습니다.

### 빠른 할인

콤마로 구분된 할인 비율 목록이 빠른 바로 가기 버튼으로 나타납니다.

예를 들어, `5,10,15,20`을 입력하면 전체 주문에 대해 단 한 번의 클릭으로 비율 할인을 적용하는 네 개의 버튼이 추가됩니다.

### 열

장바구니에 나타날 열을 구성합니다:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **수량** | 각 항목의 수량 | 분할 (항목 분할 허용) |
| **이름** | 제품 이름 | SKU |
| **가격** | 개당 가격 | 세일 표시기 |
| **정상 가격** | 비할인 가격 | - |
| **소계** | 항목 소계 (수량 × 가격) | 세금 금액 |
| **총액** | 항목 총액 | 세금 금액, 세일 표시기 |
| **작업** | 항목 제거 버튼 | - |

### 기본 설정 복원

클릭하여 모든 장바구니 디스플레이 설정을 원래 기본값으로 재설정합니다.

## 장바구니에 항목 추가

### 제품 패널에서

[제품 패널](/pos/product-panel)에서 제품을 클릭하여 장바구니에 추가합니다. 변형 제품의 경우 먼저 변형을 선택합니다.

### 기타 제품

카탈로그에 없는 항목의 경우:

1. **기타 제품 추가** 클릭
2. 이름 및 가격 입력
3. 항목이 장바구니에 추가됩니다

### 수수료

수수료를 추가하려면 (예: 선물 포장):

1. **수수료 추가** 클릭
2. 이름 및 금액 입력
3. 수수료가 별도의 항목으로 나타납니다

### 배송

배송을 추가하려면:

1. **배송 추가** 클릭
2. 배송 방법 이름 및 비용 입력
3. 배송이 주문에 추가됩니다

## 장바구니 항목 편집

모든 항목은 장바구니에서 직접 편집할 수 있습니다. 항목의 편집에 대한 자세한 내용은 [라인 항목](/pos/cart/line-items)에서 확인하세요:

- 수량 편집
- 가격 변경
- 원시 JSON 데이터 보기 및 편집
- 항목 제거
