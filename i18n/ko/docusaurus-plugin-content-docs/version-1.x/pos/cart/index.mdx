장바구니 패널은 POS 화면의 오른쪽에 위치하며, 현재 주문을 관리하는 곳입니다. 여기서 상품을 확인하고, 수량과 가격을 편집하며, 고객을 선택하고, 결제 절차를 진행할 수 있습니다.

{/* Screenshot: pos-cart-panel.png */}

## 인터페이스 개요

### 고객 선택기

장바구니 패널 상단에는 다음이 있습니다:

- **고객 배지** - 현재 고객 표시 (예: "손님" 또는 고객 이름)
- **고객 추가 아이콘** (<Icon name="plus" />) - 새 고객 생성
- **디스플레이 설정** (<Icon name="sliders" />) - 장바구니 열 및 옵션 구성

고객 배지를 클릭하여 다른 고객을 검색하고 선택할 수 있습니다.

### 주문 항목

주요 영역에는 현재 주문의 상품이 표시됩니다:

- **수량** - 수량 (편집 가능)
- **이름** - 상품명과 옵션 속성
- **가격** - 단위 가격 (편집 가능)
- **합계** - 라인 합계

자세한 내용은 [주문 항목](/pos/cart/line-items)을 참조하세요.

### 장바구니 추가 옵션

주문 항목 아래에서 특별한 항목을 추가할 수 있습니다:

- **기타 상품 추가** - 수동으로 가격을 입력하는 맞춤 항목 추가
- **수수료 추가** - 수수료 라인 추가 (예: 선물 포장, 서비스 요금)
- **배송 추가** - 수동으로 배송 비용을 입력하는 배송 라인 추가

:::note 주문 항목 유형
WooCommerce는 세 가지 유형의 주문 항목을 사용합니다: `line_item` (상품), `fee_line` (수수료), `shipping_line` (배송). 현재는 배송에 대해 수동 입력만 지원되며, POS는 배송 비용을 자동으로 계산하지 않습니다.
:::

### 주문 요약

- **소계** - 세금 및 수수료 이전 합계
- **세금** - 계산된 세금 금액 (해당되는 경우)
- **합계** - 최종 주문 총액

### 주문 작업

장바구니 패널 하단:

- **주문 메모** - 고객에게 보이는 메모 추가
- **주문 메타데이터** - 사용자 정의 메타데이터 추가 또는 JSON 보기
- **서버에 저장** - 주문을 `pos-open` 상태로 WooCommerce에 저장

자세한 내용은 [주문 작업](/pos/cart/order-actions)을 참조하세요.

### 주요 버튼

- **무효화** (빨간색) - 현재 주문 취소/삭제
- **결제** (녹색) - 결제 절차 진행

### 열린 주문

맨 아래에는 모든 열린 주문의 캐러셀이 표시됩니다:

- 각 장바구니는 총액을 표시
- 장바구니를 클릭하여 전환
- 현재 장바구니는 강조 표시됨

자세한 내용은 [열린 주문](/pos/cart/open-orders)을 참조하세요.

### 결제 후 자동으로 영수증 표시

활성화되면 판매 완료 후 자동으로 영수증 화면이 표시됩니다.

### 결제 후 자동으로 영수증 인쇄

활성화되면 영수증이 표시될 때 자동으로 인쇄됩니다. 이를 통해 수동 인쇄 단계를 줄여 시간을 절약할 수 있습니다.

### 빠른 할인

쉼표로 구분된 할인 비율 목록으로, 빠른 단축 버튼으로 나타납니다.

예를 들어, `5,10,15,20`을 입력하면 전체 주문에 단일 클릭으로 백분율 할인을 적용하는 네 개의 버튼이 추가됩니다.

### 열

장바구니에 표시될 열을 구성합니다:

| 열 | 설명 | 표시 옵션 |
|--------|-------------|-----------------|
| **수량** | 각 항목의 수량 | 분할 (항목 분할 허용) |
| **이름** | 상품명 | SKU |
| **가격** | 단위 가격 | 할인 표시기 |
| **정상 가격** | 할인 전 가격 | - |
| **소계** | 라인 소계 (수량 × 가격) | 세금 금액 |
| **합계** | 라인 합계 | 세금 금액, 할인 표시기 |
| **작업** | 항목 제거 버튼 | - |

### 기본 설정 복원

클릭하여 모든 장바구니 디스플레이 설정을 원래 기본값으로 재설정합니다.

## 장바구니에 항목 추가

### 상품 패널에서

[상품 패널](/pos/product-panel)에서 상품을 클릭하여 장바구니에 추가합니다. 가변 상품의 경우 먼저 옵션을 선택하세요.

### 기타 상품

카탈로그에 없는 상품의 경우:

1. **기타 상품 추가**를 클릭
2. 이름과 가격 입력
3. 항목이 장바구니에 추가됨

### 수수료

수수료 추가 (예: 선물 포장):

1. **수수료 추가**를 클릭
2. 이름과 금액 입력
3. 수수료가 별도의 라인 항목으로 표시됨

### 배송

배송 추가:

1. **배송 추가**를 클릭
2. 배송 방법 이름과 비용 입력
3. 배송이 주문에 추가됨

## 장바구니 항목 편집

모든 주문 항목은 장바구니에서 직접 편집 가능합니다. 다음에 대해서는 [주문 항목](/pos/cart/line-items)을 참조하세요:

- 수량 편집
- 가격 변경
- 원시 JSON 데이터 보기 및 편집
- 항목 제거
