장바구니의 모든 항목은 직접 편집할 수 있는 라인 항목입니다. WCPOS는 수량, 가격 및 기타 세부 정보에 대한 유연한 인라인 편집 기능을 제공합니다.

## 라인 항목 유형

WooCommerce는 세 가지 유형의 라인 항목을 사용합니다:

| 유형 | 설명 | 예시 |
|------|-------------|----------|
| **line_item** | 카탈로그의 제품 | 티셔츠, 커피 머그컵 |
| **fee_line** | 추가 요금 | 선물 포장, 서비스 요금 |
| **shipping_line** | 배송 비용 | 표준 배송, 특급 배송 |

:::note
배송 비용은 수동으로 입력됩니다. POS는 배송 요금을 자동으로 계산하지 않습니다.
:::

## 라인 항목 편집

### 수량

수량 필드를 클릭하여 직접 편집합니다. 새 숫자를 입력하거나 스피너 컨트롤을 사용합니다.

{/* 스크린샷: cart-line-item-edit-qty.png */}

### 제품 이름

기타 제품 및 요금의 경우 이름을 직접 클릭하여 편집할 수 있습니다.

### 가격

가격 필드를 클릭하여 단가를 변경합니다. 다음과 같은 경우에 유용합니다:

- 수동 할인 적용
- 가격 일치
- 가격 오류 수정

수량 × 가격을 기준으로 합계가 자동으로 업데이트됩니다.

### 점 메뉴

모든 라인 항목의 **⋮** (점 세 개) 버튼을 클릭하여 추가 옵션에 액세스합니다:

{/* 스크린샷: cart-line-item-menu.png */}

#### 편집 양식

라인 항목에 대한 자세한 편집 양식을 열어 모든 필드를 수정할 수 있습니다.

#### 원시 JSON 보기

라인 항목의 원시 JSON 데이터를 표시합니다. 다음과 같은 경우 매우 유용합니다:

- **디버깅** - WooCommerce로 전송되는 정확한 데이터 확인
- **문제 해결** - 사용자 정의 필드 또는 메타 데이터의 문제 식별
- **개발** - 통합을 위한 데이터 구조 이해

### 항목 제거

장바구니에서 항목을 제거하려면 빨간색 **×** 버튼을 클릭합니다.

## 변형 속성

가변 제품의 경우, 선택된 변형 속성이 제품 이름 아래에 표시됩니다:

```
후드티
색상: 녹색
크기: 대형
```

이러한 속성은 라인 항목 데이터의 일부이며 주문에 포함됩니다.

## 메타 데이터

[제품 패널](/pos/product-panel#meta-data-keys) 설정에서 메타 데이터 키를 구성한 경우, 제품을 장바구니에 추가할 때 해당 데이터가 자동으로 복사됩니다.

원시 JSON 보기에서 이 메타 데이터를 볼 수 있습니다.

## 라인 항목 분할

[디스플레이 설정](/pos/cart#display-settings)에서 활성화된 경우, 라인 항목을 여러 줄로 분할할 수 있습니다. 고객이 동일한 제품의 일부에 다른 할인을 적용하려는 경우에 유용합니다.

## 팁

### 빠른 가격 조정

빠르게 할인을 적용하려면:

1. 가격 필드를 클릭
2. 새 가격 입력
3. Enter 키 누르기

### 키보드 탐색

- **Tab** - 편집 가능한 필드 간 이동
- **Enter** - 편집 확인
- **Escape** - 편집 취소

### 일괄 편집

복잡한 주문의 경우 점 메뉴를 사용하여 전체 편집 양식에 액세스하는 것이 좋습니다. 이 양식에서는 모든 필드를 한 번에 볼 수 있습니다.
