장바구니의 각 항목은 직접 편집할 수 있는 라인 항목입니다. WCPOS는 수량, 가격 및 기타 세부정보에 대해 유연한 인라인 편집 기능을 제공합니다.

## 라인 항목 유형 {#line-item-types}

WooCommerce는 세 가지 유형의 라인 항목을 사용합니다:

| 유형 | 설명 | 예시 |
|------|-------------|----------|
| **line_item** | 카탈로그의 제품 | 티셔츠, 커피 머그 |
| **fee_line** | 추가 요금 | 선물 포장, 서비스 요금 |
| **shipping_line** | 배송비 | 일반 배송, 특급 배송 |

:::note
배송비는 수동으로 입력합니다. POS는 배송 요금을 자동으로 계산하지 않습니다.
:::

## 라인 항목 편집하기 {#editing-line-items}

### 수량 {#quantity}

수량 필드를 클릭하여 직접 편집합니다. 새 숫자를 입력하거나 스피너 컨트롤을 사용할 수 있습니다.

{/* Screenshot: cart-line-item-edit-qty.png */}

### 제품 이름 {#product-name}

잡다한 제품과 요금의 경우, 이름을 클릭하여 직접 편집할 수 있습니다.

### 가격 {#price}

가격 필드를 클릭하여 단가를 변경합니다. 이는 다음과 같은 경우에 유용합니다:

- 수동 할인 적용
- 가격 일치
- 가격 오류 수정

총액은 수량 × 가격에 따라 자동으로 업데이트됩니다.

### 세 개 점 메뉴 {#three-dot-menu}

모든 라인 항목에서 **⋮** (세 개 점)을 클릭하여 추가 옵션에 접근합니다:

{/* Screenshot: cart-line-item-menu.png */}

#### 편집 양식 {#edit-form}

라인 항목을 위한 세부 편집 양식이 열리며, 모든 필드를 수정할 수 있습니다.

#### 원시 JSON 보기 {#raw-json-view}

라인 항목에 대한 원시 JSON 데이터를 표시합니다. 이는 다음과 같은 경우에 매우 유용합니다:

- **디버깅** - WooCommerce에 전송되는 데이터 확인
- **문제 해결** - 사용자 정의 필드 또는 메타 데이터의 문제 식별
- **개발** - 통합을 위한 데이터 구조 이해

### 항목 제거 {#remove-item}

빨간색 **×** 버튼을 클릭하여 장바구니에서 항목을 제거합니다.

## 변형 속성 {#variation-attributes}

변형 제품의 경우 선택된 변형 속성이 제품 이름 아래에 표시됩니다:

```
Hoodie
Colour: Green
Size: Large
```

이 속성은 라인 항목 데이터의 일부로 주문에 포함됩니다.

## 메타 데이터 {#meta-data}

[메타 데이터 키](/pos/product-panel#meta-data-keys)를 제품 패널 설정에서 구성한 경우, 해당 데이터는 제품이 장바구니에 추가될 때 자동으로 라인 항목에 복사됩니다.

원시 JSON 보기에서 이 메타 데이터를 확인할 수 있습니다.

## 라인 항목 분할 {#splitting-line-items}

[표시 설정](/pos/cart#display-settings)에서 활성화된 경우, 라인 항목을 여러 줄로 분할할 수 있습니다. 이는 고객이 동일한 제품의 일부에 대해 서로 다른 할인을 적용하려는 경우에 유용합니다.

## 팁 {#tips}

### 빠른 가격 조정 {#quick-price-adjustments}

빠르게 할인을 제공하려면:

1. 가격 필드를 클릭합니다
2. 새 가격을 입력합니다
3. Enter를 누릅니다

### 키보드 탐색 {#keyboard-navigation}

- **Tab** - 편집 가능한 필드 간 이동
- **Enter** - 편집 확인
- **Escape** - 편집 취소

### 일괄 편집 {#batch-edits}

복잡한 주문의 경우, 세 개 점 메뉴를 사용하여 전체 편집 양식에 접근하는 것을 고려하세요. 이 양식은 모든 필드를 한 번에 표시합니다.
