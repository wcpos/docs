장바구니의 모든 항목은 직접 편집할 수 있는 품목입니다. WCPOS는 수량, 가격 및 기타 세부정보에 대한 유연한 인라인 편집을 제공합니다.

## 품목 유형

WooCommerce는 세 가지 유형의 품목을 사용합니다:

| 유형 | 설명 | 예시 |
|------|-------------|----------|
| **line_item** | 카탈로그의 제품 | 티셔츠, 커피 머그 |
| **fee_line** | 추가 요금 | 선물 포장, 서비스 요금 |
| **shipping_line** | 배송 비용 | 일반 배송, 급행 배송 |

:::note
배송 비용은 수동으로 입력됩니다. POS는 배송 요금을 자동으로 계산하지 않습니다.
:::

## 품목 편집

### 수량

수량 필드를 클릭하여 직접 편집합니다. 새로운 숫자를 입력하거나 스피너 컨트롤을 사용합니다.

{/* 스크린샷: cart-line-item-edit-qty.png */}

### 제품 이름

기타 제품 및 요금에 대해서는 클릭하여 이름을 직접 편집할 수 있습니다.

### 가격

가격 필드를 클릭하여 단가를 변경합니다. 이는 다음에 유용합니다:

- 수동 할인 적용
- 가격 맞추기
- 가격 오류 수정

총합은 수량 × 가격에 따라 자동으로 업데이트됩니다.

### 세 점 메뉴

모든 품목에서 **⋮** (세 점)을 클릭하여 추가 옵션에 접근합니다:

{/* 스크린샷: cart-line-item-menu.png */}

#### 편집 양식

품목을 위한 상세 편집 양식을 열어 모든 필드를 수정할 수 있습니다.

#### 원시 JSON 보기

품목을 위한 원시 JSON 데이터를 표시합니다. 이는 다음에 매우 유용합니다:

- **디버깅** - WooCommerce로 전송되는 데이터가 무엇인지 정확히 확인
- **문제 해결** - 사용자 정의 필드 또는 메타 데이터 문제 식별
- **개발** - 통합을 위한 데이터 구조 이해

### 품목 제거

빨간 **×** 버튼을 클릭하여 장바구니에서 품목을 제거합니다.

## 변동 속성

변동 제품의 경우, 선택된 변동 속성이 제품 이름 아래에 표시됩니다:

```
Hoodie
Colour: Green
Size: Large
```

이 속성은 품목 데이터의 일부이며 주문에 포함됩니다.

## 메타 데이터

[메타 데이터 키](/pos/product-panel#meta-data-keys)를 제품 패널 설정에서 구성한 경우, 해당 데이터는 제품이 장바구니에 추가될 때 자동으로 품목으로 복사됩니다.

원시 JSON 보기에서 이 메타 데이터를 확인할 수 있습니다.

## 품목 분할

[디스플레이 설정](/pos/cart#display-settings)에서 활성화할 경우, 품목을 여러 줄로 분할할 수 있습니다. 이는 고객이 동일한 제품의 일부분에 대해 다양한 할인을 적용하고자 할 때 유용합니다.

## 팁

### 빠른 가격 조정

빠르게 할인을 주려면:

1. 가격 필드를 클릭합니다.
2. 새로운 가격을 입력합니다.
3. Enter를 누릅니다.

### 키보드 탐색

- **Tab** - 편집 가능한 필드 간 이동
- **Enter** - 편집 확인
- **Escape** - 편집 취소

### 배치 편집

복잡한 주문의 경우, 세 점 메뉴를 사용하여 모든 필드를 한 번에 보여주는 전체 편집 양식에 접근하는 것을 고려합니다.
