- 주문에 고객 메모를 추가하는 대화 상자를 엽니다.
- 고객 메모는 고객이 볼 수 있으며 기본적으로 영수증, 고객의 내 계정 영역 및 `WP Admin > WooCommerce > Orders > Order`의 배송지 주소에 나타납니다.

:::note
주문 메모는 고객이 볼 수 있습니다.
:::

## 주문 메타

- 주문에 대한 추가 메타데이터를 관리하기 위한 모달을 엽니다.
- 기능 포함:
  - 주문 통화 변경.
  - 거래 ID 추가(예: 외부 결제 터미널에서).
  - 주문에 대한 사용자 정의 메타 데이터 추가.

:::note
추가 기능은 시간이 지남에 따라 이 모달에 추가될 예정이며, 다른 WooCommerce 플러그인과의 통합도 포함됩니다.
:::

### WooCommerce에서 메타 데이터란 무엇인가요?

메타 데이터는 주문에 대한 추가 정보를 제공하며 WooCommerce REST API를 통해 접근할 수 있습니다. 개발자와 플러그인은 이 데이터를 사용하여 기능을 확장하고, 서드파티 서비스를 통합합니다.

### JSON 보기

- 주문의 원시 JSON 표현을 볼 수 있으며, 디버깅이나 통합 목적에 유용합니다.

## 서버에 저장

- 열려 있는 장바구니를 서버에 저장할 수 있어 안전하게 보관되고 미래에 접근할 수 있습니다.
- 상태 `pos-open`인 WooCommerce 주문을 생성합니다.

### 왜 저장하나요?

POS에서 새 주문이 생성되면 로컬에만 존재합니다. 
주문을 저장하면 로컬 데이터베이스가 지워지더라도 지속됩니다.

저장된 주문 관리를 위한 자세한 내용은 [열린 주문](/pos/cart/open-orders)을 참조하십시오.

## 무효화

- 두 가지 시나리오:
  - 저장하지 않은 주문:
    - 주문이 서버에 저장되지 않은 경우, 로컬 데이터베이스에서 영구적으로 삭제됩니다.
  - 저장된 주문:
    - 주문이 서버에 저장된 경우, WooCommerce의 휴지통 폴더로 이동하고 로컬 데이터베이스에서 삭제됩니다.
- 무효화된 주문을 복구하려면:
  - `WP Admin > WooCommerce > Orders > Trash`로 이동합니다.

## 체크아웃

- 주문을 서버에 저장하고 체크아웃 프로세스를 시작합니다.
- 결제를 완료하기 위해 [체크아웃 화면](/pos/checkout)을 엽니다.
