WCPOS는 카운터에서 이미 사용 중인 물리적 장치와 함께 작동합니다. 이 페이지는 하드웨어 연결의 시작점입니다 — 영수증 프린터, 바코드 스캐너, 카드 리더, 금전함. 각 장치 유형은 여기에서 자체 설정 페이지를 제공하거나, 실제 설정이 있는 페이지로 안내합니다.

:::

## 카드 리더기 호환성 {#card-reader-compatibility}

사용 가능한 카드 리더기는 **WCPOS 실행 방식**과 리더기의 연결 유형에 따라 달라집니다.

- **웹 앱**은 **웹(브라우저) SDK**를 제공하는 리더기만 제어할 수 있습니다. 일반적으로 인터넷에 연결된 카운터톱 단말기가 이에 해당합니다. **Bluetooth 전용** 리더기(예: Stripe M2 / WisePad 3, SumUp Air)와 휴대폰 **탭투페이**는 웹 앱에서 **지원되지 않습니다**.
- 지원 여부는 보편적으로 적용되는 것이 아니라 **게이트웨이별**로 다릅니다. 구매하기 전에 사용하려는 게이트웨이([Stripe Terminal](/payment/gateways/stripe-terminal), [SumUp](/payment/gateways/sumup-terminal), [PayPal Reader](/payment/gateways/paypal-reader))에서 해당 리더기를 지원하는지 확인하세요.

각 게이트웨이 페이지에는 지원되는 정확한 리더기 목록이 나와 있습니다.
