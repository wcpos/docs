WCPOS는 계산대에서 이미 사용 중인 물리적 장치와 함께 작동합니다. 이 페이지는 하드웨어 연결을 시작하는 곳입니다 — 영수증 프린터, 바코드 스캐너, 카드 리더, 금전함. 각 장치 유형은 이곳에 별도의 설정 페이지가 있거나, 실제 설정이 있는 페이지로 연결됩니다.

:::note 기기마다 설정 위치는 하나
하드웨어 허브는 각 기기의 설정 위치를 안내하지만, 실제 설정 단계는 한 곳에만 있습니다. 카드 리더기의 계정 및 페어링 절차는 해당 결제 게이트웨이 페이지에 있고, 금전함 열림 설정은 영수증 프린터 페이지에 있습니다.
:::
