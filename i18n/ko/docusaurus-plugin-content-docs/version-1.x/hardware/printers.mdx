프린터 설정은 POS의 **설정 > 프린터**에서 찾을 수 있습니다. 각 장치는 자체 프린터 구성을 관리합니다. 프린터는 로컬에 저장되며 장치 간에 동기화되지 않습니다.

:::tip 이 장치에 없는 프린터로 인쇄하시나요?
이 페이지에서는 USB, Bluetooth 또는 로컬 네트워크로 계산대에 연결된 프린터를 다룹니다. 다른 방이나 위치에 있는 프린터 또는 모든 장치에서 공유하려는 프린터는 [클라우드 인쇄](/receipts/cloud-printing)를 참조하세요.
:::

## 플랫폼별 지원 프린터 {#supported-printers-by-platform}

**프린터 추가** 양식은 사용 중인 앱에 맞게 조정됩니다. 각 플랫폼은 실제로 사용할 수 있는 연결 및 옵션만 표시합니다.

| 플랫폼 | 지원되는 프린터 | 연결 |
|----------|--------------------|-------------|
| **웹 앱** | 내장 웹 서버가 있는 Epson 및 Star 프린터 | 네트워크 (HTTP) |
| **데스크톱 앱** | 모든 네트워크 영수증 프린터 | 네트워크 (raw TCP) |
| **iOS 앱** | Epson 및 Star | 네트워크, Bluetooth |
| **Android 앱** | Epson 및 Star | 네트워크, Bluetooth, USB |

:::info 웹에서는 왜 Epson 및 Star만 지원되나요?
웹 브라우저는 원시 TCP 연결을 열 수 없습니다. Epson 및 Star 영수증 프린터에는 HTTP를 통해 인쇄 작업을 수락하는 내장 웹 서버가 포함되어 있습니다. 브라우저에서 ESC/POS 명령을 보내는 유일한 방법입니다. 데스크톱 앱은 TCP를 통해 직접 연결할 수 있으므로 이 제한이 없습니다.

**모바일 앱**에서는 Bluetooth 및 USB 인쇄가 Epson 및 Star의 공식 SDK를 통해 처리되므로, 이러한 연결에도 Epson 또는 Star 프린터가 필요합니다. 일반 프린터는 네트워크를 통해서만 작동합니다.
:::

## 프린터 추가 {#adding-a-printer}

<Steps>
  <StepCard number="1" title="프린터 설정 열기">

POS에서 **설정 > 프린터**로 이동합니다. 아직 구성된 프린터가 없으면 **프린터 추가** 버튼이 있는 빈 상태가 표시됩니다. 모든 프린터에는 **프린터 이름**이 지정됩니다. 이는 단순한 라벨이며 "영수증 프린터"(그다음은 "영수증 프린터 2" 등)로 자동 입력됩니다. 프린터를 구분하는 데 도움이 되는 이름으로 변경할 수 있습니다.

  </StepCard>
  <StepCard number="2" title="프린터 연결 방식 선택">

이 단계는 플랫폼마다 달라집니다.

<ul>
  <li><strong>웹 앱</strong> — 선택할 항목은 <strong>공급업체</strong>: Epson 또는 Star입니다. 브라우저는 이 두 브랜드의 내장 웹 서버에만 접근할 수 있으므로 웹에서는 이것이 첫 번째 선택 사항입니다.</li>
  <li><strong>데스크톱 앱</strong> — 프린터는 네트워크를 통해 연결되므로 여기서 추가로 선택할 항목은 없습니다.</li>
  <li><strong>모바일 앱</strong> — 선택할 항목은 <strong>연결 유형</strong>: <strong>네트워크</strong>, <strong>블루투스</strong>, 또는 <strong>USB</strong> (USB는 Android에서만 지원됩니다). 아래 필드는 선택한 항목에 맞게 변경됩니다.</li>
</ul>

  </StepCard>
  <StepCard number="3" title="프린터 식별">

**네트워크 프린터** — **IP 주소**를 입력합니다(예: `192.168.1.100`). WCPOS는 해당 주소를 확인하여 Epson 또는 Star 장치인지 감지하고, 포트, 명령 언어, 공급업체를 자동으로 입력합니다. 감지가 완료되면 "감지됨: Epson" 또는 "감지됨: Star" 레이블이 표시됩니다. 웹 앱에서는 연결할 정확한 엔드포인트 URL도 표시됩니다.

**Bluetooth 또는 USB 프린터**(모바일) — 검색된 장치 목록에서 프린터를 선택합니다. 장치를 선택하면 이름과 공급업체가 자동으로 입력됩니다. 프린터가 아직 목록에 없으면 **프린터 검색**(Bluetooth) 또는 **새로 고침**(USB)을 탭합니다.

자동 감지가 모든 항목을 올바르게 설정하지 못한 경우 **고급 설정**에서 모두 조정할 수 있습니다.

  </StepCard>
  <StepCard number="4" title="저장 및 테스트" isLast>

**저장**을 클릭합니다. WCPOS는 먼저 **테스트 인쇄**를 보내고, 성공한 경우에만 프린터를 저장합니다.

테스트 인쇄는 번호가 있는 열 눈금자, 가운데 정렬된 일반 및 두 배 크기 텍스트, 좌우 정렬 행, 용지 절단으로 구성된 짧은 진단입니다. 프린터에 연결할 수 있는지만 확인하는 것이 아니라, 용지 너비, 정렬, 크기, 절단이 모두 올바르게 구성되었는지 확인할 수 있도록 설계되었습니다.

테스트가 실패하면 오류와 함께 프로필을 그대로 저장할 수 있는 **테스트 없이 저장** 옵션이 표시됩니다.

  </StepCard>
</Steps>

## 연결 유형(모바일 앱) {#connection-types-mobile-apps}

iOS 및 Android 앱에서는 프린터를 세 가지 방식으로 연결할 수 있습니다. 프린터 추가 양식 상단에서 **연결 유형**을 선택하면 연결 필드가 그에 맞게 업데이트됩니다.

### 네트워크 {#network}

프린터가 기기와 동일한 Wi-Fi 네트워크에 있습니다. 해당 IP 주소를 입력하세요 — [프린터 IP 주소 찾기](#finding-your-printers-ip-address)를 참조하세요. 네트워크 프린터는 Epson, Star 또는 일반 프린터일 수 있습니다.

### 블루투스 {#bluetooth}

프린터가 블루투스로 기기와 페어링되어 있습니다. **프린터 검색**을 탭하고 목록에서 프린터를 선택하세요 — 이름과 공급업체가 자동으로 입력됩니다. 블루투스 인쇄는 Epson 및 Star의 네이티브 SDK를 사용하므로 프린터는 **Epson 또는 Star** 모델이어야 합니다.

### USB(Android만 해당) {#usb-android-only}

프린터가 USB 케이블로 연결되어 있습니다. **새로 고침**을 탭하고 연결된 기기 목록에서 선택하세요. 블루투스와 마찬가지로 USB 인쇄에는 **Epson 또는 Star** 프린터가 필요합니다. iOS는 일반 USB 주변기기 지원이 없으므로 이 옵션은 Android에서만 표시됩니다.

:::note
**포트** 설정은 네트워크 프린터에만 적용됩니다 — 블루투스 및 USB 연결에서는 숨겨집니다.
:::

## 고급 설정 {#advanced-settings}

**고급 설정**을 펼쳐 프린터 구성을 세부 조정하세요. 공급업체가 확인되면 대부분의 항목은 자동으로 입력됩니다.

| 설정 | 옵션 | 설명 |
|---------|---------|-------------|
| **공급업체** | Epson, Star, 일반 | 프린터 제조업체입니다. **웹 앱**에서는 공급업체가 고급 설정이 아니라 양식 상단의 기본 선택 항목입니다(Epson 또는 Star만 해당). 일반 프린터는 데스크톱 및 모바일 앱의 **네트워크** 프린터에서 사용할 수 있지만, 블루투스 또는 USB 연결에서는 사용할 수 없습니다. |
| **포트** | 자동 입력 | 프린터의 네트워크 포트입니다. 네트워크에서만 사용되며 Bluetooth 및 USB에서는 숨겨집니다. |
| **언어** | ESC/POS, StarPRNT, Star Line Mode | 프린터가 사용하는 명령 프로토콜입니다. 감지된 공급업체에 따라 자동 입력됩니다. |
| **프린터 텍스트 너비** | 58mm (32자), 80mm 표준 (42자), 80mm 와이드 (48자) | 한 줄에 들어가는 문자 수입니다. 용지와 프린터에 맞게 설정하세요. 대부분의 80mm 프린터는 한 줄에 42자이며, 일부는 48자입니다. |
| **전체 영수증 래스터** | 끄기 / 켜기 | 전체 영수증을 텍스트 대신 이미지로 인쇄합니다. 아래 [전체 영수증 래스터](#full-receipt-raster)를 참고하세요. |

## 프린터 옵션 {#printer-options}

이 토글은 프린터 추가 양식 하단에 있습니다.

| 옵션 | 설명 |
|--------|-------------|
| **용지 자동 절단** | 각 영수증 인쇄 후 용지를 자동으로 절단합니다 |
| **현금 서랍 자동 열기** | 인쇄 후 연결된 현금 서랍을 열도록 킥 명령을 보냅니다 |
| **기본값으로 설정** | 이 프린터를 기본 프린터로 설정합니다. 특정 프린터로 라우팅되지 않은 모든 영수증에 사용됩니다 |

:::note "결제 후 자동 인쇄"를 찾고 있나요?
판매가 완료될 때 영수증을 자동으로 인쇄하는 기능은 프린터 설정이 아니라 **카트 설정**입니다. POS 카트 설정에서 **영수증 자동 인쇄**를 켜세요. *어느* 프린터를 사용할지는 기본 프린터와 템플릿별 프린터 라우팅에 따라 결정됩니다.
:::

## 인쇄 라우팅 {#print-routing}

두 개 이상의 템플릿을 사용하는 경우, 예를 들어 감열 영수증**과** A4 인보이스를 함께 사용하는 경우, 인쇄 라우팅은 각 템플릿을 어느 프린터로 인쇄할지 결정합니다. 라우팅은 세 단계로 구성되며, 다음 순서대로 확인됩니다:

1. **작업별 재정의.** 영수증 화면에서 템플릿 전환기 옆에 프린터 드롭다운이 표시됩니다. 여기에서 프린터를 선택하면 해당 인쇄 작업 하나에 대해 모든 설정이 재정의됩니다. 템플릿을 전환하면 다시 **자동**으로 초기화됩니다.
2. **설정 재정의.** **설정 → 인쇄 라우팅**으로 이동하여 각 템플릿에 특정 프린터를 할당합니다. 예를 들어 감열 영수증은 Epson 네트워크 프린터로, HTML 인보이스는 시스템 인쇄 대화상자로 라우팅할 수 있습니다. 재정의를 제거하려면 템플릿을 다시 **자동**으로 설정합니다.
3. **자동 일치.** 재정의가 설정되어 있지 않으면 WCPOS가 자동으로 일치시킵니다:
   - **감열 템플릿**은 문자 너비가 일치하는 감열 프린터로 라우팅됩니다. 58mm 감열 템플릿은 32열 프린터를 우선 사용하고, 80mm 템플릿은 42열 또는 48열 프린터를 우선 사용합니다.
   - **HTML 템플릿**은 시스템 인쇄 대화상자로 라우팅됩니다.
   - 여러 프린터가 일치하면 **기본** 프린터가 사용됩니다.

호환되지 않는 프린터로 템플릿을 수동 전송하는 경우, 예를 들어 감열 템플릿을 시스템 대화상자로 보내는 경우, 영수증 화면에 주황색 **불일치 경고**가 표시됩니다. 인쇄는 계속 진행되지만 출력이 올바르게 렌더링되지 않을 수 있습니다.

라우팅 재정의는 **기기별로** 저장됩니다. 각 iPad, 휴대폰 또는 컴퓨터가 자체 라우팅을 관리하며, 서버 측 동기화는 없습니다.

:::note
인쇄 라우팅 설정 페이지는 프린터를 하나 이상 추가한 뒤에만 표시됩니다. 구성된 프린터가 없으면 모든 템플릿이 암묵적으로 시스템 인쇄 대화상자를 사용합니다.
:::

## 전체 영수증 래스터 {#full-receipt-raster}

기본적으로 WCPOS는 감열식 프린터에 영수증을 **텍스트**로 전송합니다. 프린터에 내장된 글꼴을 사용하므로 빠르고, 용량이 작으며, 선명합니다. 다만 감열식 프린터 글꼴은 제한된 문자 집합만 지원합니다. 프린터에 해당 글꼴이 없는 문자 체계(많은 비라틴 알파벳, 오른쪽에서 왼쪽으로 쓰는 텍스트, 일부 기호 등)는 빈 사각형이나 깨진 문자로 출력될 수 있습니다.

**전체 영수증 래스터**는 이 문제를 해결합니다. 이 옵션을 켜면 WCPOS가 전체 영수증을 이미지로 렌더링해 프린터로 전송하므로, 어떤 언어나 문자 체계에서도 출력물이 화면에 표시된 내용과 정확히 일치합니다.

프린터별로 **고급 설정**에서 켤 수 있습니다. 알아둘 사항:

- **속도가 더 느립니다.** 이미지는 한 줄의 텍스트보다 훨씬 크기 때문에 영수증을 전송하고 인쇄하는 데 시간이 더 오래 걸립니다.
- **필요할 때만 사용하세요.** 영수증이 텍스트로 문제없이 인쇄된다면 꺼 둡니다.
- 이 옵션은 모든 연결 방식(네트워크, Bluetooth, USB)의 감열식 프린터에 적용됩니다. 이미 전체 페이지 렌더링에서 인쇄하는 시스템 인쇄 대화상자에는 적용되지 않습니다.

## 프린터 검색 {#discovering-printers}

**모바일 앱**에서는 프린터 추가 양식에 **Bluetooth**(iOS 및 Android) 및 **USB**(Android만 해당) 연결용 근처 프린터가 표시됩니다. **프린터 검색** 또는 **새로 고침**을 탭한 뒤 목록에서 프린터를 선택합니다. 이름과 공급업체는 자동으로 입력됩니다.

모든 플랫폼의 **네트워크** 프린터는 IP 주소를 직접 입력합니다. 아래의 [프린터 IP 주소 찾기](#finding-your-printers-ip-address)를 참조하세요. 웹 앱에는 네트워크 검색 기능이 없으므로(브라우저는 네트워크를 스캔할 수 없음) 항상 IP를 수동으로 입력해야 합니다.

## 프린터의 IP 주소 찾기 {#finding-your-printers-ip-address}

대부분의 영수증 프린터는 IP 주소가 포함된 자체 테스트 페이지를 출력할 수 있습니다.

- **Epson**: 전원을 켜는 동안 급지 버튼을 길게 누르세요.
- **Star**: 전원이 켜진 상태에서 급지 버튼을 5초 동안 길게 누르세요.

또는 라우터의 연결된 장치 목록이나 프린터의 구성 유틸리티를 확인하세요.

### 고정 IP 권장 {#static-ip-recommended}

프린터가 다시 시작될 때 주소가 변경되지 않도록 영수증 프린터에는 **고정 IP 주소**를 사용해야 합니다. 다음 중 한 곳에서 구성하세요.

- 프린터의 내장 웹 인터페이스에서(일반적으로 다음 주소에서 접근 가능 `http://<printer-ip>`)
- 라우터의 DHCP 설정에서(프린터의 MAC 주소에 예약 IP 할당)

## Epson 프린터 {#epson-printers}

ePOS를 지원하는 Epson 프린터는 HTTP를 통해 통신합니다. WCPOS는 프린터의 내장 웹 서버로 SOAP/XML 요청을 보냅니다.

**연결 세부 정보:**
- 포트 `8008`(HTTP) 또는 `8043`(HTTPS)
- 엔드포인트: `/cgi-bin/epos/service.cgi`

**작동이 확인된 모델:**
- Epson TM-T70-i 2(네트워크)
- Epson TM-m30iii(Printus 미들웨어 사용)

:::tip
Epson 프린터 설정에서 ePOS가 활성화되어 있는지 확인하세요. 프린터의 웹 인터페이스에 접속하세요: `http://<printer-ip>` ePOS 설정을 확인하고 활성화합니다.
:::

**모바일 앱**에서는 Epson 프린터를 Epson 네이티브 SDK를 통해 Bluetooth 또는 USB로도 연결할 수 있습니다. IP 주소나 ePOS 웹 서버가 필요하지 않습니다.

## Star 프린터 {#star-printers}

WebPRNT를 지원하는 Star 프린터는 HTTP를 통해 통신합니다. WCPOS는 Star WebPRNT 엔드포인트로 명령을 보냅니다.

**연결 세부 정보:**
- HTTPS 또는 HTTP
- 엔드포인트: `/StarWebPRNT/SendMessage`

**모바일 앱**에서는 Star 프린터를 Star의 기본 SDK를 통해 Bluetooth 또는 USB로도 연결할 수 있습니다.

## 일반 프린터 {#generic-printers}

**데스크톱 앱**은 원시 TCP를 통해 모든 네트워크 영수증 프린터를 지원합니다. 프린터의 IP 주소와 포트(일반적으로 `9100`)를 입력하세요. ESC/POS 명령을 수신할 수 있다면 제조업체와 관계없이 대부분의 감열식 프린터에서 작동합니다.

**모바일 앱**에서도 일반 프린터가 지원되지만, Bluetooth 또는 USB가 아닌 **네트워크** 연결만 지원됩니다(해당 연결에는 Epson 또는 Star 프린터가 필요합니다). **웹 앱**은 일반 프린터를 전혀 지원하지 않습니다.

## 금전함 {#cash-drawers}

영수증 프린터에 연결된 금전함은 각 판매 후 자동으로 열리도록 할 수 있습니다.

1. 프린터 옵션에서 **금전함 자동 열기**를 활성화합니다.
2. POS는 인쇄 후 ESC/POS 킥 명령을 보냅니다.

:::note
금전함 제어에는 감열식 프린터 프로필이 필요합니다. 브라우저 인쇄 대화상자로는 금전함을 열 수 없습니다.
:::

  - 네트워크 프린터의 경우: IP 주소가 올바른지 확인합니다(자체 테스트 페이지 인쇄)

  - USB(Android)의 경우: 케이블을 확인하고 USB 권한 요청이 표시되면 승인합니다
  - 연결을 차단하는 방화벽이 없는지 확인합니다
