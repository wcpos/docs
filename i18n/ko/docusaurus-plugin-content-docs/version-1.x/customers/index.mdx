:::info Pro Feature
고객 화면은 [WCPOS Pro](/getting-started/pro-license)를 필요로 합니다. 무료 사용자들은 주문을 위해 기존 고객을 선택할 수 있지만 전체 고객 목록을 볼 수 없으며 고객 정보를 수정할 수 없습니다.
:::

고객 화면은 POS 내에서 직접 포괄적인 고객 관리 기능을 제공합니다. WooCommerce 관리로 전환하지 않고도 고객을 보고, 편집하고, 생성할 수 있습니다.

{/* Screenshot: customers-screen.png */}

## 인터페이스 개요

### 헤더 액션

화면 상단에:

- **검색 바** - 이름, 이메일 등으로 고객 찾기
- **고객 추가** (<Icon name="plus" />) - 새 고객 생성
- **디스플레이 설정** (<Icon name="sliders" />) - 보이는 열 구성

### 고객 테이블

주요 영역에서는 고객을 다음과 같이 표시합니다:

- **아바타** - 고객 프로필 사진(또는 기본 이미지)
- **이름** - 고객의 이름
- **성** - 고객의 성(정렬 가능)
- **이메일** - 연락 이메일 주소
- **청구 주소** - 전체 청구 주소
- **생성 날짜** - 고객 추가 날짜
- **액션** - 세 점 메뉴

### 바닥글

- 고객 수와 동기화 버튼

## 주요 기능

### 고객 검색

다음과 같이 고객을 빠르게 찾기:

- 이름 또는 성
- 이메일 주소
- 주소 정보

### 새 고객 추가

POS에서 직접 고객을 생성합니다:

1. 헤더에서 <Icon name="plus" /> 아이콘 클릭
2. 고객 세부 정보 입력
3. 새 고객 저장

고객은 WooCommerce에 생성되며 즉시 주문 가능 상태가 됩니다.

### 고객 편집

고객 정보를 업데이트합니다:

1. 고객의 세 점 메뉴 클릭
2. **편집** 선택
3. 세부 사항 수정(이름, 이메일, 주소)
4. 변경 사항 저장

변경 사항은 자동으로 WooCommerce에 동기화됩니다.

## 디스플레이 설정

슬라이더 아이콘(<Icon name="sliders" />)을 클릭하여 보이는 열을 사용자 정의합니다.

### 사용 가능한 열

| 열       | 설명                     |
|----------|--------------------------|
| **이미지** | 고객 아바타               |
| **ID**   | WooCommerce 고객 ID       |
| **이름** | 고객 이름                 |
| **성**   | 고객 성                  |
| **이메일** | 이메일 주소              |
| **역할** | 사용자 역할(고객, 구독자 등)|
| **사용자 이름** | WordPress 사용자 이름   |
| **청구 주소** | 청구 정보               |
| **배송 주소** | 배송 정보               |
| **생성 날짜** | 고객이 추가된 날짜      |
| **수정된 날짜** | 마지막 업데이트         |
| **액션** | 편집, 동기화, 삭제       |

## 고객 액션

세 점 메뉴(⋮)를 클릭하여 옵션을 확인합니다:

- **편집** - 고객 세부 사항 수정
- **동기화** - 서버에서 고객 새로 고침
- **삭제** - 로컬 데이터베이스에서 제거

:::note
POS에서 고객을 삭제하면 로컬에서만 제거됩니다. 고객은 WooCommerce에 남아 있으며 다음 동기화 시 다시 나타납니다.
:::

## 주문에서 고객 사용

주문에 고객을 할당하려면:

1. [카트 패널](/pos/cart)에서 고객 배지를 클릭합니다
2. 이름 또는 이메일로 고객 검색
3. 고객 선택
4. 이제 고객이 주문에 연결됩니다

고객 정보(청구/배송 주소)는 자동으로 주문에 사용됩니다.

## 동기화

고객 데이터는 POS와 WooCommerce 간에 동기화됩니다:

- **실시간 업데이트** - 모든 장치에서 변경 사항이 동기화됩니다
- **WooCommerce 통합** - 고객은 WooCommerce에 저장됩니다
- **오프라인 기능** - 오프라인에서도 고객 보기 가능
