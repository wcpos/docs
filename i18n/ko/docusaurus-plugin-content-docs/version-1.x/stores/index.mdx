:::info Pro 기능
멀티 매장 기능을 사용하려면 [WCPOS Pro](/getting-started/pro-license)가 필요합니다.
:::

WCPOS Pro를 사용하면 하나의 WooCommerce 설치 내에서 별도의 매장을 생성할 수 있습니다. 각 매장은 고유한 주소, 로고, 설정을 가질 수 있어 여러 실제 매장을 운영하는 경우에 유용합니다.

## 매장에서 제공하는 기능 {#what-stores-provide}

각 매장은 다음을 설정할 수 있습니다:

- **매장 이름 및 주소** — 영수증과 보고서에 사용됩니다
- **매장 로고** — 인쇄된 영수증에 표시됩니다
- **세율** — 지역에 따라 세금 요건이 다를 수 있습니다
- **캐셔 배정** — 어떤 캐셔가 어떤 매장에서 근무하는지 추적

## 매장 설정하기 {#setting-up-stores}

Pro를 활성화하면 WP Admin의 POS 아래에 **매장** 하위 메뉴가 추가됩니다. 이 메뉴에서 각 매장을 생성하고, 주소, 로고, 세금 ID, 로그인할 수 있는 캐셔를 설정할 수 있습니다. 전체 단계별 안내는 **[매장 설정하기](/stores/setup)**를 참조하십시오.

캐셔가 POS에 로그인하면 운영할 매장을 선택합니다. 이에 따라 영수증에 표시되는 매장 정보와 보고서에 적용되는 매장 필터가 결정됩니다.

## 매장별 필터링 {#filtering-by-store}

여러 매장이 구성되면 POS 전체에 매장 필터가 표시됩니다:

- **[주문](/orders)** — 매장 위치별로 주문 내역 필터링
- **[보고서](/reports)** — 특정 매장의 일일 마감 보고서 생성
- **[영수증](/receipts/at-checkout)** — 영수증에 해당 매장의 주소와 로고가 자동으로 표시

## 매장별 가격 설정 {#store-specific-pricing}

WCPOS Pro는 **매장 위치별 차등 가격**을 지원합니다. 상품과 옵션 모두에 대해 매장별로 별도의 정가 및 할인가를 설정할 수 있습니다.

캐셔가 특정 매장에서 운영 중일 때 POS는 자동으로 해당 매장의 가격을 적용합니다. 매장별 가격이 설정되어 있지 않으면 기본 WooCommerce 가격이 사용됩니다.

## 재고 관리 {#inventory-management}

기본적으로 모든 매장은 동일한 WooCommerce 재고 수준을 공유합니다. 매장별 재고 추적을 위해 WCPOS Pro는 [ATUM Inventory Management](https://www.atum.org/) 플러그인과 통합됩니다:

1. WooCommerce 사이트에 ATUM 플러그인을 설치하고 구성합니다
2. ATUM에서 재고 위치를 생성합니다
3. WCPOS Pro에서 각 매장을 ATUM 재고 위치에 연결합니다
4. 주문이 접수되면 해당 위치의 재고 수준이 업데이트됩니다

### 대안: WordPress 멀티사이트 {#alternative-wordpress-multisite}

ATUM 없이 완전히 분리된 재고를 관리하려면 [WordPress 멀티사이트](https://developer.wordpress.org/advanced-administration/multisite/create-network/)를 사용할 수 있습니다:

1. 각 매장 위치에 대해 별도의 WordPress 사이트를 생성합니다
2. 각 사이트에 WooCommerce와 WCPOS를 설치합니다
3. 각 사이트는 독립적인 상품 카탈로그와 재고 수준을 유지합니다

이 방법은 완전한 분리를 제공하지만 여러 WooCommerce 설치를 관리해야 합니다.

## 알려진 제한 사항 {#known-limitations}

- **단일 WooCommerce 인스턴스** — 매장은 하나의 WooCommerce 설치 내 위치이며, 별도의 쇼핑몰이 아닙니다
- **매장별 상품 표시 미지원** — 모든 매장에서 동일한 상품 카탈로그가 표시됩니다
