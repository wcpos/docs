:::info Pro 기능
다중 매장 기능을 사용하려면 [WCPOS Pro](/getting-started/pro-license)가 필요합니다.
:::

WCPOS Pro를 사용하면 하나의 WooCommerce 설치 안에서 별도의 매장 위치를 만들 수 있습니다. 각 매장은 자체 주소, 로고, 설정을 가질 수 있어 여러 오프라인 위치에서 운영하는 경우 유용합니다.

## 매장에서 제공하는 기능 {#what-stores-provide}

각 매장에는 다음 항목을 설정할 수 있습니다:

- **매장 이름 및 주소** — 영수증과 보고서에 사용됩니다
- **매장 로고** — 인쇄된 영수증에 표시됩니다
- **세율** — 위치에 따라 세금 요건이 다를 수 있습니다
- **계산원 배정** — 어느 계산원이 어느 매장에서 근무하는지 추적합니다

## 매장 설정 {#setting-up-stores}

Pro를 활성화하면 WP Admin의 POS 아래에 **매장** 하위 메뉴가 추가됩니다. 여기에서 각 매장을 만들고 주소, 로고, 세금 ID, 로그인할 수 있는 계산원을 설정할 수 있습니다. 전체 단계별 안내는 **[매장 설정](/stores/setup)**을 참조하세요.

계산원이 POS에 로그인하면 운영 중인 매장을 선택합니다. 이 선택에 따라 영수증에 표시되는 매장 정보와 보고서에 적용되는 매장 필터가 결정됩니다.

## 매장별 필터링 {#filtering-by-store}

여러 매장을 구성하면 POS 전반에 매장 필터가 표시됩니다.

- **[주문](/orders)** — 주문 내역을 매장 위치별로 필터링합니다
- **[보고서](/reports)** — 특정 매장의 영업 마감 보고서를 생성합니다
- **[영수증](/receipts/at-checkout)** — 영수증에 올바른 매장 주소와 로고가 자동으로 표시됩니다

## 매장별 가격 {#store-specific-pricing}

WCPOS Pro는 **매장 위치별로 서로 다른 가격**을 지원합니다. 상품과 옵션 모두에 대해 매장별 일반 가격과 할인 가격을 따로 설정할 수 있습니다.

계산원이 특정 매장에서 운영 중이면 POS가 해당 매장의 가격을 자동으로 사용합니다. 매장별 가격이 설정되어 있지 않으면 기본 WooCommerce 가격이 사용됩니다.

## 재고 관리 {#inventory-management}

기본적으로 모든 매장은 동일한 WooCommerce 재고 수량을 공유합니다. 위치별 재고 추적을 위해 WCPOS Pro는 [ATUM Inventory Management](https://www.atum.org/) 플러그인과 통합됩니다.

1. WooCommerce 사이트에 ATUM 플러그인을 설치하고 구성합니다
2. ATUM에서 재고 위치를 생성합니다
3. WCPOS Pro에서 각 매장을 ATUM 재고 위치와 연결합니다
4. 주문이 접수되면 올바른 위치의 재고 수량이 업데이트됩니다

### 대안: WordPress Multisite {#alternative-wordpress-multisite}

ATUM 없이 재고를 완전히 분리하려면 [WordPress Multisite](https://developer.wordpress.org/advanced-administration/multisite/create-network/)를 사용할 수 있습니다.

1. 각 매장 위치별로 별도의 WordPress 사이트를 생성합니다
2. 각 사이트에 WooCommerce와 WCPOS를 설치합니다
3. 각 사이트는 자체적인 독립 제품 카탈로그와 재고 수량을 유지합니다

이 방식은 완전한 분리를 제공하지만 여러 WooCommerce 설치를 관리해야 합니다.

## 알려진 제한 사항 {#known-limitations}

- **단일 WooCommerce 인스턴스** — 매장은 별도의 상점이 아니라 하나의 WooCommerce 설치 내 위치입니다
- **매장별 상품 표시 여부 미지원** — 모든 매장에서 동일한 상품 카탈로그를 봅니다
