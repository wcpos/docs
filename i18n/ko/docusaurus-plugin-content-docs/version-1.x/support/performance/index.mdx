WCPOS에서 원활한 결제 환경을 제공하기 위해서는 성능 최적화가 매우 중요합니다. 이 섹션에서는 일반적인 성능 문제와 해결 방법을 다룹니다.

## 일반적인 성능 문제 {#common-performance-issues}

가장 많이 보고되는 성능 관련 불만은 다음과 같습니다:

- **느린 결제 프로세스** - 결제 모달이 로드되거나 결제를 처리하는 데 오랜 시간이 걸림
- **느린 인터페이스** - POS 인터페이스의 반응이 둔함
- **긴 동기화 시간** - 상품 및 주문 동기화에 시간이 너무 오래 걸림

## 성능 카테고리 {#performance-categories}

### [결제 성능](/support/performance/checkout) {#checkout-performancesupportperformancecheckout}
더 빠른 결제 처리와 향상된 고객 경험을 위해 결제 프로세스를 최적화하세요.

### [서버 성능](/support/performance/server) {#server-performancesupportperformanceserver}
내장 메트릭과 문제 해결 기법을 활용하여 WordPress/WooCommerce 서버를 모니터링하고 최적화하세요.

### 네트워크 성능 (출시 예정) {#network-performance-coming-soon}
네트워크 관련 성능 문제를 해결하고 API 통신을 최적화합니다.
