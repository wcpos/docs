성능 최적화는 WCPOS에서 원활한 체크아웃 경험을 제공하는 데 중요합니다. 이 섹션에서는 일반적인 성능 문제와 이를 해결하는 방법을 다룹니다.

## 일반적인 성능 문제 {#common-performance-issues}

가장 일반적인 성능 불만 사항은 다음과 관련이 있습니다:

- **느린 체크아웃 프로세스** - 결제 모달이 로드되거나 결제를 처리하는 데 오랜 시간이 걸림
- **느린 인터페이스** - POS 인터페이스가 반응하지 않는 것처럼 느껴짐
- **긴 동기화 시간** - 제품 및 주문 동기화에 너무 오랜 시간이 걸림

## 성능 범주 {#performance-categories}

### [체크아웃 성능](/support/performance/checkout) {#checkout-performancesupportperformancecheckout}
더 빠른 결제 처리를 위해 체크아웃 프로세스를 최적화하고 고객 경험을 개선하세요.

### [서버 성능](/support/performance/server) {#server-performancesupportperformanceserver}
내장된 메트릭 및 문제 해결 기술을 사용하여 WordPress/WooCommerce 서버를 모니터링하고 최적화하세요.

### 네트워크 성능 (추후 제공) {#network-performance-coming-soon}
네트워크 관련 성능 문제를 해결하고 API 통신을 최적화하세요.
