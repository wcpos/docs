이 오류 메시지가 표시되면, 귀하의 POS에서 치명적인 서버 오류가 발생하여 계속 진행할 수 없다는 의미입니다.
좋은 소식은 이 오류 메시지가 귀하의 WooCommerce 로그에 기록되며, 저희 지원팀이 문제 해결을 도와줄 수 있다는 것입니다.

## 오류 찾는 방법

1. WordPress 관리 영역에 로그인합니다.
2. WordPress 사이드바 메뉴에서 "WooCommerce" > "상태" > "로그"로 이동합니다.
3. "fatal_error" 로그 섹션을 찾습니다.
4. 오류가 발생한 날짜(보통 오늘 날짜)에 해당하는 로그 항목을 클릭하여 오류의 세부 정보를 확인합니다.
5. 이 섹션에 기록된 오류의 원인을 파악합니다.
6. 관련 오류 메시지를 복사하거나 스크린샷을 찍어 [Discord 채팅](https://wcpos.com/discord)에 게시하거나 support@wcpos.com으로 이메일을 보내 저희 지원팀과 공유합니다.
