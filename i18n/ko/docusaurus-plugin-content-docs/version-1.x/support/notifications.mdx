POS 헤더의 **알림 종**은 WCPOS가 제품 공지, 플러그인 업데이트 알림, Pro 라이선스 리마인더 등의 메시지를 표시하는 위치입니다. 종의 빨간 배지는 읽지 않은 알림 수를 표시합니다(99개를 초과하면 **99+**로 표시).

## 알림 센터 열기 {#opening-the-notification-centre}

POS 상단의 **종 아이콘**을 클릭하여 알림 패널을 엽니다. 알림은 최신 항목부터 표시되며, 각 항목에는 다음이 포함됩니다.

- **읽지 않음 점** - 읽지 않은 상태에서는 채워져 표시되며 제목은 굵게 표시됩니다.
- 짧은 **제목** 및 선택적 **본문**
- **상대 시간 표시**(예: _2시간 전_)

대기 중인 알림이 없으면 패널에 **"알림 없음"** 빈 상태가 표시됩니다.

## 알림 읽기 및 지우기 {#reading-and-clearing-notifications}

- **하나를 읽음으로 표시** - 목록에서 읽지 않은 알림을 탭합니다.
- **모두 읽음으로 표시** — 패널 헤더의 **모두 읽음으로 표시** 버튼을 사용합니다(읽지 않은 항목이 있을 때만 표시됨).
- **배지 지우기** — 각 항목을 아직 열지 않았더라도 패널을 열기만 하면 종 아이콘의 읽지 않은 개수 배지가 지워집니다.

알림은 기기에 저장되므로 오프라인에서도 목록을 사용할 수 있습니다. 연결되어 있는 동안 새 알림이 실시간으로 도착합니다.

## 표시되는 알림 유형 {#what-kinds-of-notifications-appear}

알림 센터는 매장 주문이 아니라 시스템 및 계정 메시지에 사용됩니다. 다음과 같은 알림이 표시될 수 있습니다.

| 유형 | 예시 |
| --- | --- |
| **환영** | 처음 연결할 때 표시되는 일회성 환영 메시지 |
| **공지** | WCPOS에 관한 일반 소식 및 업데이트 |
| **플러그인 업데이트** | WCPOS 플러그인 업데이트를 사용할 수 있다는 알림 |
| **라이선스 만료 예정** | Pro 라이선스의 갱신일이 다가오고 있음을 알리는 알림과 **갱신** 링크 |
| **라이선스 만료됨** | Pro 라이선스가 만료되었음을 알리는 지속 알림과 **지금 갱신** 링크 |

중요한 메시지(예: 라이선스 알림)는 종 아이콘에 표시되는 것 외에도 화면 토스트로 잠시 표시될 수 있습니다.

:::note
알림 센터는 WCPOS의 메시지를 전달합니다. **새 온라인 주문**에 대한 실시간 알림은 별도로 계획된 기능입니다 — [로드맵](https://github.com/orgs/wcpos/projects/4)을 참조하세요.
:::
