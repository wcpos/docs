WCPOS 사용 중 도움이 필요하신가요? 지원을 받고 질문에 대한 답변을 찾는 방법입니다.

## 앱 내 지원

POS에는 왼쪽 탐색 서랍에서 접근할 수 있는 내장된 지원 화면이 있습니다. 이 화면은 WCPOS Discord 커뮤니티를 앱 내에 직접 포함하여 다음을 할 수 있습니다:

- 기존 토론 탐색
- 일반적인 질문에 대한 답변 검색
- #general 채널에서 질문하기
- 공지사항 확인

## Discord 커뮤니티

WCPOS의 주요 지원 채널은 [Discord 서버](https://wcpos.com/discord)입니다.

### 채널

| 채널 | 목적 |
|---------|---------|
| **#welcome-and-rules** | 커뮤니티 가이드라인 |
| **#announcements** | 업데이트 및 뉴스 |
| **#general** | 주요 지원 채널 - 여기서 질문하세요 |
| **#forum** | 심층 토론 및 기능 요청 |
| **#github-activity** | 개발 업데이트 |

### 질문 전 확인사항

최상의 도움을 받으려면:

1. **먼저 테스트** - 플러그인을 며칠 동안 다운로드하고 테스트하여 필요성을 파악하세요
2. **문서 확인** - 많은 질문이 이 문서에서 답변됩니다
3. **Discord 검색** - 질문이 이미 답변되었을 수 있습니다
4. **세부 정보 제공** - WooCommerce 버전, WCPOS 버전, 문제 재현 단계를 포함하세요

## 이메일 지원

라이선스 관련 문제 또는 개인적인 문의는 다음 이메일로 연락하세요: **support@wcpos.com**

:::note
일반적인 지원 질문은 커뮤니티가 도움을 주고 다른 사용자도 혜택을 받을 수 있도록 Discord에 올려주세요.
:::

## 로그 화면

POS에는 시스템 활동을 기록하는 [로그 화면](/support/logs)이 포함되어 있습니다. 이는 다음에 매우 유용합니다:

- 동기화 문제 디버깅
- 오류 메시지 이해
- 지원 요청 시 상세 정보 제공

도움을 요청할 때는 로그 화면의 오류 메시지를 확인하고 지원 요청에 포함하세요.

## 프로 지원

[WCPOS 프로](https://wcpos.com/pro) 고객은 Discord를 통해 동일한 커뮤니티 지원을 받습니다. 프로 라이선스는 무료 및 프로 기능 모두의 지속적인 개발을 지원합니다.
