WCPOS에 대한 도움이 필요하신가요? 지원을 받는 방법과 질문에 대한 답변을 찾는 방법은 다음과 같습니다.

## 앱 내 지원

POS에는 왼쪽 내비게이션 드로어에서 접근할 수 있는 내장 지원 화면이 포함되어 있습니다. 이 화면은 앱 내에서 WCPOS Discord 커뮤니티를 직접 임베드하여 다음과 같은 기능을 제공합니다:

- 기존 논의 검색
- 일반적인 질문에 대한 답변 검색
- #general 채널에 질문하기
- 공지사항 최신 정보 확인

## Discord 커뮤니티

WCPOS의 주요 지원 채널은 우리의 [Discord 서버](https://wcpos.com/discord)입니다.

### 채널

| 채널 | 목적 |
|---------|---------|
| **#환영과규칙** | 커뮤니티 가이드라인 |
| **#공지사항** | 업데이트 및 뉴스 |
| **#일반** | 주요 지원 채널 - 여기서 질문하세요 |
| **#포럼** | 긴 논의 및 기능 요청 |
| **#깃허브-활동** | 개발 업데이트 |

### 질문하기 전에

최고의 도움을 얻기 위해:

1. **먼저 테스트하세요** - 플러그인을 다운로드하고 며칠 동안 테스트하여 필요에 맞는지 이해하세요.
2. **문서를 확인하세요** - 많은 질문이 이 문서에서 답변됩니다.
3. **Discord 검색** - 당신의 질문은 이미 답변되었을 수 있습니다.
4. **세부정보 제공** - WooCommerce 버전, WCPOS 버전 및 문제를 재현하는 단계를 포함하세요.

## 이메일 지원

라이센스 관련 문제나 개인 문의는 다음 이메일로 보낼 수 있습니다: **support@wcpos.com**

:::note
일반 지원 질문은 커뮤니티가 도와주고 다른 사용자가 답변의 혜택을 누릴 수 있도록 Discord로 가야 합니다.
:::

## 로그 화면

POS에는 시스템 활동을 기록하는 [로그 화면](/support/logs)이 포함되어 있습니다. 이는 다음에 매우 유용합니다:

- 동기화 문제 디버깅
- 오류 메시지 이해
- 지원 요청 시 세부 정보를 제공

도움을 요청할 때는 로그 화면에서 오류 메시지를 확인하고 지원 요청에 포함하세요.

## 프로 지원

[WCPOS Pro](https://wcpos.com/pro) 고객은 Discord를 통해 동일한 커뮤니티 지원을 받습니다. Pro 라이센스는 무료 및 Pro 기능 모두의 지속적인 개발을 지원합니다.
