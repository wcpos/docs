:::info 프로 기능
보고서 화면은 [WCPOS Pro](/getting-started/pro-license)에서 사용할 수 있습니다.
:::

보고서 화면은 판매 보고 및 일일 조정 도구를 제공합니다. 이는 주로 **일일 종료 보고서**를 위한 것으로, 점원과 관리자들이 하루를 마무리하는 데 도움을 줍니다.

{/* 스크린샷: reports-overview.png */}

:::tip 종합 분석
모든 날짜 범위에 대한 상세 분석은 **WP Admin > Analytics**에서 사용할 수 있습니다. POS 보고서 화면은 신속한 일일 요약 및 일일 조정을 위해 최적화되어 있습니다.
:::

## 인터페이스 개요 {#interface-overview}

### 필터 {#filters}

화면 상단:

- **상태 필터** - 주문 상태로 필터링 (완료, 처리 중 등)
- **고객 필터** - 고객으로 필터링
- **점원 필터** - 주문을 처리한 점원으로 필터링
- **상점 필터** - 상점으로 필터링 (다중 상점 설정)
- **날짜 범위** - 보고 기간 선택 (오늘, 이번 주, 사용자 지정)

### 세 개의 패널 레이아웃 {#three-panel-layout}

보고서 화면은 세 개의 영역으로 나뉘어 있습니다:

#### 1. 판매 차트 (상단) {#1-sales-chart-top}

시간에 따른 판매를 보여주는 막대 차트:

- 일일 보고서에 대한 시간별 분석
- 판매 패턴의 시각적 표현
- 판매 피크 타임 식별에 도움

#### 2. 주문 테이블 (하단 왼쪽) {#2-orders-table-bottom-left}

보고서에 포함된 주문 테이블:

- **체크박스** - 포함할 주문 선택/해제
- **상태** - 주문 상태 아이콘
- **주문 번호** - 고유 주문 ID
- **생성 날짜** - 주문이 발생한 날짜
- **지불 날짜** - 결제가 이루어진 날짜
- **총액** - 주문 금액

표시 설정 아이콘을 클릭하여 열을 사용자 정의할 수 있습니다.

#### 3. 보고서 요약 (하단 오른쪽) {#3-report-summary-bottom-right}

계산된 보고서 요약:

- **보고서 유형** 드롭다운 (기본/오프라인)
- **상점 정보** (이름, ID)
- **보고서 기간** (시작/종료 타임스탬프)
- **점원** (필터된 경우)

**판매 요약:**
- 총 주문 수
- 총 순매출
- 총 세금 징수
- **총 판매액**
- 총 할인

**결제 방법:**
- 결제 유형별 분류 (현금, 카드 등)

**인쇄** 버튼을 클릭하여 프린트 가능한 보고서를 생성합니다.

## 보고서 유형 {#report-types}

### 기본 (오프라인) {#default-offline}

기본 보고서는 로컬에 저장된 데이터에서 생성됩니다. 이 보고서는:

- 서버 연결 없이 작동합니다.
- 빠른 보고서 생성을 제공합니다.
- 이미 장치에 동기화된 데이터를 사용합니다.

이는 모든 관련 주문이 이미 동기화된 신속한 일일 요약에 적합합니다.

## 일일 종료 워크플로우 {#end-of-day-workflow}

### 일반적인 일일 조정 {#typical-daily-reconciliation}

1. **날짜 범위 설정** - "오늘" 또는 특정 날짜 선택
2. **점원으로 필터링** (선택 사항) - 개별 점원 보고서용
3. **주문 검토** - 주문 테이블에서 정확성을 확인
4. **총액 확인** - 실제 현금 수량과 카드 단말기와 비교
5. **보고서 인쇄** - 물리적 기록 생성

### 현금 조정 {#cash-reconciliation}

1. 현금 결제만 표시되도록 필터링
2. 보고서에서 총 현금 금액을 확인
3. 서랍의 실제 현금을 계산
4. 총액 비교 (일치해야 함)
5. 불일치 사항 기록

### 카드 조정 {#card-reconciliation}

1. 카드 결제만 표시되도록 필터링
2. 보고서에서 총 카드 금액 확인
3. 카드 단말기의 배치 총액과 비교
4. 배치 결제 전에 금액을 일치시킴

## 보고서 인쇄 {#printing-reports}

**인쇄** 버튼을 클릭하여 인쇄 가능한 보고서를 생성합니다. 여기에는 다음이 포함됩니다:

- 상점 정보
- 보고서 기간
- 판매 요약
- 결제 방법 분석
- 주문 세부 정보 (선택 시)
