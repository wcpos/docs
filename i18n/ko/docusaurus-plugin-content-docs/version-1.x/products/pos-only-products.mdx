POS 전용 제품은 제품이 온라인 스토어, POS 또는 둘 다에 표시되는지 제어할 수 있는 기능입니다. 이는 온라인 및 오프라인 판매를 위한 서로 다른 제품 카탈로그가 필요할 때 유용합니다.

## 개요 {#overview}

활성화되면 각 제품은 세 가지 옵션을 가진 **POS 가시성** 설정을 갖습니다:

| 가시성 | 온라인 스토어 | POS |
|------------|--------------|-----|
| **POS 및 온라인** (기본) | 표시됨 | 표시됨 |
| **POS 전용** | 숨김 | 표시됨 |
| **온라인 전용** | 표시됨 | 숨김 |

## 이 기능을 사용할 때 {#when-to-use-this-feature}

POS 전용 제품을 활성화해야 하는 경우:

- **매장 독점 판매** - 고객이 물리적 매장을 방문할 때만 제공되는 제품
- **온라인 전용 품목을 POS에서 숨기기** - 디지털 제품, 예약 주문 또는 매장에서 판매해서는 안 되는 품목
- **상이한 카탈로그 관리** - 온라인 및 소매 채널을 위한 별개의 제품 범위
- **프로모션 품목 관리** - 매장에서만 제공되는 특별 프로모션 또는 온라인 전용 거래

:::tip
모든 제품이 두 채널에 모두 표시되어야 한다면, 이 기능이 필요하지 않습니다. 성능 향상을 위해 비활성화해 두세요.
:::

## 기능 활성화 {#enabling-the-feature}

1. **WP Admin → WCPOS → 설정 → 일반**으로 이동합니다.
2. **"POS 전용 제품 활성화"** 옵션을 활성화합니다.
3. 변경 사항을 저장합니다.

일반 설정에 대한 자세한 내용은 [WP Admin 일반 설정](/settings/wp-admin/general)을 참조하십시오.

:::warning 성능 영향
이 설정은 필요할 때만 활성화하세요. 활성화되면 모든 제품 요청에 대해 추가적인 데이터베이스 조회가 추가되어 큰 카탈로그가 있는 스토어의 성능에 영향을 줄 수 있습니다.
:::

## 제품 가시성 설정 {#setting-product-visibility}

### 개별 제품 {#individual-products}

단일 제품의 가시성을 설정하려면:

1. **WP Admin → 제품**으로 이동하여 제품을 편집합니다.
2. **제품 데이터** 패널에서 **POS 가시성**을 찾습니다.
3. 원하는 옵션을 선택합니다:
   - **POS 및 온라인** - 제품이 어디에서나 표시됨 (기본값)
   - **POS 전용** - 제품이 온라인 스토어에서 숨김
   - **온라인 전용** - 제품이 POS에서 숨김
4. **업데이트**를 클릭하여 저장합니다.

### 일괄 편집 {#bulk-editing}

한 번에 여러 제품의 가시성을 변경하려면:

1. **WP Admin → 제품**으로 이동합니다.
2. 체크박스를 사용하여 수정할 제품을 선택합니다.
3. **일괄 작업** 드롭다운에서 **편집**을 선택합니다.
4. **적용**을 클릭합니다.
5. 일괄 편집 패널에서 **POS 가시성** 드롭다운을 찾습니다.
6. 원하는 가시성 옵션을 선택합니다.
7. **업데이트**를 클릭합니다.

### 변동 제품 {#variable-products}

변동 제품의 경우, 부모 제품 수준과 각 개별 변동에 대해 가시성을 설정할 수 있습니다. 이를 통해 세밀하게 제어할 수 있습니다. 예를 들어, 모든 변동을 온라인에 표시하되 매장에서는 특정 색상이나 사이즈만 표시할 수 있습니다.

## POS에 미치는 영향 {#how-it-affects-the-pos}

제품이 **온라인 전용**으로 설정되면:

- [제품 패널](/pos/product-panel)에 나타나지 않습니다.
- 검색 또는 바코드 스캔을 통해 찾을 수 없습니다.
- POS에서 장바구니에 추가할 수 없습니다.

제품이 **POS 전용**으로 설정되면:

- WooCommerce 스토어 프런트에서 나타나지 않습니다.
- 온라인 검색 결과에 포함되지 않습니다.
- 고객이 온라인으로 구매할 수 없습니다.

:::note
최근에 제품의 가시성을 변경하고 여전히 (또는 나타나지 않는) 경우, 동기화를 시도하세요. 동기화 버튼을 길게 눌러 **"지우기 및 새로 고침"**을 선택하여 모든 제품 데이터를 새로 고칩니다.
:::

## 일반적인 사용 사례 {#common-use-cases}

<UseCases>
  <UseCaseCard icon="gift" title="매장 전용 기프트 카드">

  등록대에서 활성화해야 하는 물리적 기프트 카드:

  1. 기프트 카드 제품을 생성합니다.
  2. 가시성을 **POS 전용**으로 설정합니다.
  3. 카드는 온라인에는 나타나지 않지만 직원이 매장에서 판매할 수 있습니다.

  </UseCaseCard>

  <UseCaseCard icon="globe" title="온라인 예약 주문">

  온라인에서 예약 주문이 가능하지만 재고가 없는 제품:

  1. 예약 주문 제품을 생성합니다.
  2. 가시성을 **온라인 전용**으로 설정합니다.
  3. 고객이 온라인으로 예약할 수 있지만 POS는 어지럽히지 않습니다.

  </UseCaseCard>

  <UseCaseCard icon="building" title="도매 대 소매">

  도매와 소매를 위해 동일한 WooCommerce 스토어를 사용하는 경우:

  1. 도매 전용 제품을 **온라인 전용**으로 설정합니다 (B2B 포털 접근용).
  2. 소매 독점 제품을 **POS 전용**으로 설정합니다.
  3. 일반 제품은 **POS 및 온라인**으로 유지합니다.

  </UseCaseCard>
</UseCases>

## 문제 해결 {#troubleshooting}

<Accordion>
  <AccordionItem question="온라인 전용으로 설정한 후에도 제품이 나타남">

  1. POS를 엽니다.
  2. 제품 패널 하단의 동기화 버튼을 길게 누릅니다.
  3. **"지우기 및 새로 고침"**을 선택합니다.
  4. 제품이 다시 로드될 때까지 기다립니다.

  </AccordionItem>

  <AccordionItem question="제품에 설정이 보이지 않음">

  기능이 활성화되었는지 확인하세요:

  1. **WP Admin → WCPOS → 설정 → 일반**으로 이동합니다.
  2. **"POS 전용 제품 활성화"**가 체크되어 있는지 확인합니다.
  3. 변경 사항이 있을 경우 설정을 저장합니다.

  </AccordionItem>
</Accordion>
