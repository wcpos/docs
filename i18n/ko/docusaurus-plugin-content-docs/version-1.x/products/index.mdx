:::info Pro 기능
제품 화면에는 [WCPOS Pro](/getting-started/pro-license)가 필요합니다. 무료 사용자는 [POS 제품 패널](/pos/product-panel)에서 제품을 보고 장바구니에 추가할 수 있지만, 재고나 가격을 수정할 수 없습니다.
:::

제품 화면은 전용 재고 관리 인터페이스입니다. [POS 제품 패널](/pos/product-panel)(판매용)과 달리, 이 화면은 제품 카탈로그를 관리하기 위해 설계되었습니다.

{/* Screenshot: products-screen.png */}

## 제품 화면 vs POS 제품 패널

| 기능 | POS 제품 패널 | 제품 화면 |
|---------|-------------------|-----------------|
| **목적** | 장바구니에 제품 추가 | 재고 관리 |
| **재고** | 보기만 가능 | 인라인 수정 가능 |
| **가격** | 보기만 가능 | 인라인 수정 가능 |
| **작업** | 장바구니에 추가 | 수정, 동기화, 삭제 |
| **이용 가능한 버전** | 무료 + Pro | Pro만 해당 |

## 인터페이스 개요

### 검색 및 필터

화면 상단에 있는 기능:

- **검색 창** - 이름, SKU, 바코드로 제품 찾기
- **필터 버튼** - 재고 상태, 추천 상품, 할인 중, 카테고리, 태그, 브랜드
- **디스플레이 설정** (<Icon name="sliders" />) - 표시할 열 구성

### 제품 목록

주 영역에 스크롤 가능한 테이블로 제품 표시:

- **이미지** - 제품 썸네일
- **제품명** - 선택적 SKU, 바코드, 속성 포함
- **재고** - 편집 가능한 재고 수량과 "관리" 토글
- **재고 상태** - 재고 있음(초록), 재고 없음(빨강), 백오더
- **카테고리** - 제품 카테고리 배지
- **가격** - 현재 가격, 정가, 할인가(모두 편집 가능)
- **작업** - 3점 메뉴(편집, 동기화, 삭제)

### 변형 제품

변형 제품은 다음을 표시:
- 속성 옵션(색상, 크기 등)
- 모든 변형 보기 "확장" 링크
- 개별 변형 행에 별도 재고/가격

### 하단

- **세금 상태** - 현재 세금 계산 기준
- **제품 개수** - "X개 중 Y개 표시" 및 동기화 버튼

## 주요 기능

### 인라인 재고 수정

재고 수량을 직접 클릭하여 수정:

1. 재고 숫자 클릭
2. 새 수량 입력
3. Enter 키를 눌러 저장

변경 사항은 자동으로 WooCommerce와 동기화됩니다.

### 재고 관리 토글

"관리" 토글은 WooCommerce의 제품 재고 추적 여부를 제어합니다:

- **켜기** - 재고 수준 추적 및 판매 시 차감
- **끄기** - 제품 항상 사용 가능 (무한 재고)

### 인라인 가격 수정

모든 가격 필드를 클릭하여 수정:

- **가격** - 현재 판매 가격
- **정가** - 할인 전 가격
- **할인가** - 할인된 가격 (할인 중일 때)

### 제품 작업

각 제품의 3점 메뉴(⋮)를 클릭:

- **편집** - 제품 편집 모달 열기
- **동기화** - WooCommerce에서 제품 새로고침
- **삭제** - 로컬 데이터베이스에서 제품 제거

:::note
POS에서 제품을 삭제해도 로컬에서만 제거됩니다. 제품은 WooCommerce에 남아 있으며 다음 동기화 시 다시 나타납니다.
:::

### 사용 가능한 열

| 열 | 설명 | 디스플레이 옵션 |
|--------|-------------|-----------------|
| **이미지** | 제품 썸네일 | - |
| **ID** | WooCommerce 제품 ID | - |
| **제품** | 제품명 | SKU, 바코드, 속성 |
| **유형** | 단일, 변형, 그룹 등 | - |
| **SKU** | 재고 관리 단위 | - |
| **바코드** | 제품 바코드 | - |
| **재고** | 현재 수량 (편집 가능) | - |
| **재고 상태** | 재고 있음 / 재고 없음 / 백오더 | - |
| **카테고리** | 제품 카테고리 | - |
| **태그** | 제품 태그 | - |
| **브랜드** | 브랜드 정보 (플러그인 활성화 시) | - |
| **매출원가** | 제품 원가 | - |
| **가격** | 현재 판매 가격 (편집 가능) | 세금 정보 |
| **정가** | 할인 전 가격 (편집 가능) | 세금 정보 |
| **할인가** | 할인된 가격 (편집 가능) | 세금 정보 |
| **생성일** | 제품 생성 시기 | - |
| **수정일** | 마지막 수정 날짜 | - |
| **작업** | 편집, 동기화, 삭제 버튼 | - |

### 기본 설정 복원

클릭하여 모든 디스플레이 설정을 원래 기본값으로 초기화합니다.

## 제품 필터링

필터 버튼을 사용하여 보기 좁히기:

- **재고 상태** - 재고 있음, 재고 없음, 백오더
- **추천 상품** - 추천 상품으로 표시된 제품
- **할인 중** - 현재 할인 중인 제품
- **카테고리** - 제품 카테고리로 필터링
- **태그** - 제품 태그로 필터링
- **브랜드** - 브랜드로 필터링 (브랜드 플러그인 사용 시)

## 정렬

열 머리글을 클릭하여 해당 열로 정렬합니다. 다시 클릭하면 정렬 순서를 반대로 바꿉니다.
