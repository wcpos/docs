# API01001: 연결 시간 초과

## 이것이 의미하는 바

POS 애플리케이션이 귀하의 WooCommerce 서버에 연결을 시도했으나, 서버가 응답하는 데 너무 오랜 시간이 걸렸습니다. 이는 일반적으로 네트워크 지연 문제 또는 과부하된 서버를 나타냅니다.

## 일반적인 원인

- **느린 인터넷 연결** — 귀하의 네트워크 연결이 높은 지연을 겪고 있을 수 있습니다.
- **서버 과부하** — 귀하의 WordPress 서버가 너무 많은 요청을 처리하고 있을 수 있습니다.
- **대량 데이터 요청** — 한 번에 너무 많은 데이터를 요청하면 시간 초과가 발생할 수 있습니다.
- **방화벽 또는 프록시 지연** — 네트워크 보안 장치가 지연을 초래할 수 있습니다.

## 해결 방법

### 1. 인터넷 연결 확인

인터넷 속도와 안정성을 테스트하십시오. 브라우저에서 WooCommerce 관리 패널을 로드하여 응답 속도가 느린지 확인하십시오.

### 2. 서버 부하 줄이기

서버가 과부하된 경우:
- 호스팅 계획 업그레이드를 고려하십시오.
- 불필요한 플러그인을 일시적으로 비활성화하십시오.
- 무거운 크론 작업이나 백그라운드 프로세스를 확인하십시오.

### 3. 서버 응답 시간 최적화

호스팅 제공업체에 문의하여:
- PHP opcache 활성화
- PHP 메모리 제한 늘리기
- 서버 측 캐싱 사용

### 4. 타임아웃 설정 늘리기

서버가 합리적으로 더 많은 시간이 필요한 경우:
- 호스팅 제공업체에서 PHP 실행 시간 증가를 허용하는지 확인하십시오.
- 지연 감소를 위해 CDN 사용을 고려하십시오.
