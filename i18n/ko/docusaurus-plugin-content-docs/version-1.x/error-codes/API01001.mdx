# API01001: 연결 시간 초과

## 의미

POS 애플리케이션이 WooCommerce 서버에 연결을 시도했지만, 서버가 응답하는 데 너무 오래 걸렸습니다. 이는 주로 네트워크 대기 시간 문제 또는 서버 과부하를 나타냅니다.

## 일반적인 원인

- **느린 인터넷 연결** — 네트워크 연결이 높은 대기 시간을 경험하고 있을 수 있습니다
- **서버 과부하** — WordPress 서버가 너무 많은 요청을 처리하고 있을 수 있습니다
- **대용량 데이터 요청** — 한 번에 너무 많은 데이터를 요청하면 시간 초과가 발생할 수 있습니다
- **방화벽 또는 프록시 지연** — 네트워크 보안 장치가 대기 시간을 추가하고 있을 수 있습니다

## 해결 방법

### 1. 인터넷 연결 확인

인터넷 속도와 안정성을 테스트하세요. 브라우저에서 WooCommerce 관리자 페이지를 로드하여 느리게 응답하는지 확인하세요.

### 2. 서버 부하 줄이기

서버가 과부하 상태라면:
- 호스팅 플랜 업그레이드 고려
- 불필요한 플러그인 일시적으로 비활성화
- 무거운 크론 작업 또는 백그라운드 프로세스 확인

### 3. 서버 응답 시간 최적화

호스팅 제공업체에 문의하세요:
- PHP 옵캐시 활성화
- PHP 메모리 제한 증가
- 서버 측 캐싱 사용

### 4. 시간 초과 설정 증가

서버에 실제로 더 많은 시간이 필요한 경우:
- 호스팅 제공업체가 PHP 실행 시간 증가를 허용하는지 확인
- 대기 시간을 줄이기 위해 CDN 사용 고려
