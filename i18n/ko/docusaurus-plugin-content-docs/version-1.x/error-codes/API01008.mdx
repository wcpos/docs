# API01008: 웹사이트 사용 불가 {#api01008-website-unavailable}

## 의미 {#what-this-means}

POS는 인터넷에 연결할 수 있지만, 특정 웹사이트가 응답하지 않습니다. WooCommerce 스토어를 호스팅하는 서버가 다운되었거나 도달할 수 없는 상태입니다.

## 일반적인 원인 {#common-causes}

- **서버 유지관리** — 호스팅 제공업체가 업데이트를 수행 중일 수 있습니다.
- **서버 다운** — 웹 서버가 예기치 않게 중지되었습니다.
- **자원 한도 초과** — 호스팅 계정이 한도에 도달했습니다.
- **DDoS 공격** — 서버가 악성 트래픽으로 인해 과부하 상태입니다.
- **도메인/호스팅 만료** — 호스팅 또는 도메인이 만료되었습니다.

## 해결 방법 {#how-to-fix}

### 1. 사이트 다운 확인 {#1-verify-the-site-is-down}

브라우저에서 사이트에 직접 접근해 보세요. 로드되지 않으면 문제는 서버 측입니다.

### 2. 호스팅 제공업체 상태 확인 {#2-check-hosting-provider-status}

- 호스팅 제공업체의 상태 페이지를 방문하세요.
- 예정된 유지관리가 있는지 확인하세요.
- 보고된 중단 사항을 확인하세요.

### 3. 호스팅 계정 검토 {#3-review-hosting-account}

호스팅 관리 패널에 로그인하세요:
- 계정이 활성화되어 있는지 확인하세요.
- 자원 한도를 초과하지 않았는지 확인하세요.
- 정지 통지를 확인하세요.

### 4. 호스팅 지원에 문의 {#4-contact-hosting-support}

귀하의 쪽에서 모든 것이 정상으로 보이면, 호스팅 제공업체에 문의하세요. 그들은:
- 서버 상태를 확인할 수 있습니다.
- 서버 로그를 검토할 수 있습니다.
- 필요시 서비스를 재시작할 수 있습니다.

### 5. 대기 후 재시도 {#5-wait-and-retry}

일시적인 중단인 경우:
- 15-30분 대기하세요.
- POS는 자동으로 연결을 재시도합니다.
- 캐시된 데이터는 오프라인에서도 사용 가능합니다.
