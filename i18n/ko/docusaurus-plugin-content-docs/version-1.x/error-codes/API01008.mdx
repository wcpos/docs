# API01008: 웹사이트 접속 불가

## 의미

POS는 인터넷에 연결되어 있지만, 특정 웹사이트에 응답이 없습니다. WooCommerce 스토어를 호스팅하는 서버가 다운되었거나 접근할 수 없는 상태입니다.

## 일반적인 원인

- **서버 유지보수** — 호스팅 제공업체가 업데이트를 수행 중
- **서버 충돌** — 웹 서버가 예기치 않게 중단됨
- **리소스 한도 초과** — 호스팅 계정의 리소스 한도에 도달
- **DDoS 공격** — 서버가 악의적인 트래픽으로 과부하 상태
- **도메인/호스팅 만료** — 호스팅 또는 도메인이 만료됨

## 해결 방법

### 1. 사이트 다운 확인

브라우저에서 직접 사이트에 접속해 보세요. 로드되지 않으면 서버 측 문제입니다.

### 2. 호스팅 제공업체 상태 확인

- 호스팅 제공업체의 상태 페이지 방문
- 예정된 유지보수 확인
- 보고된 장애 여부 확인

### 3. 호스팅 계정 검토

호스팅 제어판에 로그인:
- 계정이 활성화되어 있는지 확인
- 리소스 한도를 초과하지 않았는지 확인
- 중단 통지 확인

### 4. 호스팅 지원팀 연락

자체적으로 모든 것이 정상적으로 보이면 호스팅 제공업체에 연락하세요. 다음을 수행할 수 있습니다:
- 서버 상태 확인
- 서버 로그 검토
- 필요한 경우 서비스 재시작

### 5. 기다리고 재시도

일시적인 장애의 경우:
- 15-30분 대기
- POS가 자동으로 연결 재시도
- 오프라인에서 캐시된 데이터 유지
