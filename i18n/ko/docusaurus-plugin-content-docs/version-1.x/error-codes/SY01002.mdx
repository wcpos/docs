# SY01002: 디스크 꽉 참

## 이 메시지의 의미

기기의 저장 공간이 꽉 찼습니다. POS는 사용 가능한 디스크 공간 없이 데이터를 저장하거나 계속 작동할 수 없습니다.

## 일반적인 원인

- **저장 공간 부족** — 드라이브에 여유 공간 없음
- **대용량 로컬 데이터베이스** — POS 데이터가 상당한 공간 차지
- **임시 파일** — 임시 파일이 공간 소비
- **다른 애플리케이션** — 다른 앱이 디스크를 채움

## 해결 방법

### 1. 디스크 공간 확보

불필요한 파일 삭제:
- 휴지통 비우기
- 오래된 다운로드 제거
- 사용하지 않는 애플리케이션 삭제
- 임시 파일 지우기

### 2. 사용 가능한 공간 확인

**Windows:**
- 파일 탐색기 열기
- 드라이브 우클릭 → 속성
- 사용된 공간과 여유 공간 보기

**macOS:**
- Apple 메뉴 → 이 Mac에 관하여
- 저장 공간 클릭

### 3. 브라우저 데이터 지우기 (웹 버전)

브라우저에 로컬로 저장된 데이터:
- 캐시 및 쿠키 지우기
- 오래된 사이트 데이터 제거
- 브라우저 저장 공간 사용량 확인

### 4. 대용량 파일 이동 또는 삭제

대용량 파일 찾아 처리:
- 오래된 동영상이나 사진
- 대용량 다운로드
- 사용하지 않는 소프트웨어
- 외부 저장 장치로 이동 가능한 백업 파일

### 5. POS 데이터 확인

POS 데이터가 매우 큰 경우:
- 로컬 데이터베이스 정리 필요할 수 있음
- 삭제 및 다시 동기화 고려
- 오래된 데이터 제거 가능 여부 확인

## 권장 여유 공간

최소한 다음 공간 확보:
- **정상 작동을 위해 10GB** 여유
- **대규모 카탈로그의 경우 20GB 이상**
- 더 많은 공간일수록 좋음

## 향후 문제 예방

- 불필요한 파일 정기적으로 정리
- 자동 임시 파일 삭제 설정
- 디스크 공간 사용량 모니터링
- 자주 꽉 찬다면 더 큰 저장 장치 고려
