# SY01002: 디스크 가득 참

## 이것이 의미하는 바

기기의 저장 공간이 가득 찼습니다. POS는 데이터를 저장하거나 사용 가능한 디스크 공간 없이 작동을 계속할 수 없습니다.

## 일반적인 원인

- **스토리지 가득 참** — 드라이브에 무료 공간 없음
- **대용량 로컬 데이터베이스** — 상당한 공간을 차지하는 POS 데이터
- **임시 파일** — 공간을 차지하는 임시 파일
- **다른 애플리케이션** — 디스크를 채우고 있는 다른 앱들

## 해결 방법

### 1. 디스크 공간 확보

불필요한 파일을 삭제합니다:
- 휴지통/재활용통 비우기
- 오래된 다운로드 삭제
- 사용하지 않는 애플리케이션 삭제
- 임시 파일 지우기

### 2. 사용 가능한 공간 확인

**Windows:**
- 파일 탐색기 열기
- 드라이브를 우클릭 → 속성
- 사용 중인 공간과 무료 공간 보기

**macOS:**
- 애플 메뉴 → 이 Mac에 관하여
- 저장 공간 클릭

### 3. 브라우저 데이터 지우기 (웹 버전)

브라우저는 데이터를 로컬에 저장합니다:
- 캐시 및 쿠키 지우기
- 오래된 사이트 데이터 삭제
- 브라우저 저장 공간 사용량 확인

### 4. 대용량 파일 이동 또는 삭제

대용량 파일을 찾아서 처리합니다:
- 오래된 비디오 또는 사진
- 대용량 다운로드
- 사용하지 않는 소프트웨어
- 외부 저장소로 이동할 수 있는 백업 파일

### 5. POS 데이터 확인

POS 데이터가 매우 클 경우:
- 로컬 데이터베이스 정리가 필요할 수 있음
- 지우고 재동기화할 것을 고려
- 오래된 데이터 삭제 가능 여부 확인

## 권장하는 무료 공간

최소한 유지하십시오:
- **10GB** 정상 작동을 위해 무료
- **20GB+** 대규모 카탈로그를 위해
- 여유 공간이 많을수록 좋습니다

## 향후 문제 예방

- 정기적으로 불필요한 파일을 정리
- 자동 임시 파일 정리 설정
- 디스크 공간 사용량 모니터링
- 자주 가득 찬 경우 더 큰 저장소 고려
