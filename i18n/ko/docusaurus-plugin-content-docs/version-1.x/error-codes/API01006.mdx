# API01006: 네트워크 연결 불가

## 의미

POS가 서버로 경로를 설정할 수 없습니다. 이는 네트워크 수준의 문제로, 데이터 패킷이 서버로 가는 경로를 찾을 수 없는 상황입니다.

## 일반적인 원인

- **네트워크 구성 문제** — 게이트웨이 또는 라우팅 설정 오류
- **ISP 문제** — 인터넷 서비스 제공업체에 문제 발생
- **VPN 간섭** — VPN이 트래픽을 잘못 라우팅
- **서버 네트워크 문제** — 서버의 네트워크가 다운됨

## 해결 방법

### 1. 인터넷 연결 확인

- 다른 웹사이트 접속 시도
- 라우터/모뎀 재시작
- 네트워크 연결 확인

### 2. VPN 일시적으로 비활성화

VPN 사용 시:
- 연결 해제 후 다시 시도
- 다른 VPN 서버 사용
- VPN이 서버 위치 접속 허용하는지 확인

### 3. ISP에 문의

다음과 같은 경우 인터넷 서비스 제공업체에 연락:
- 다른 사이트도 접속 불가
- 네트워크 장비 재시작 후에도 문제 지속

### 4. 서버 상태 확인

- 호스팅 업체의 네트워크 문제 보고 확인
- 다른 네트워크(예: 모바일 데이터)에서 사이트 접속 시도
- 온라인 도구로 사이트가 모든 사용자에게 다운되었는지 확인

### 5. 방화벽 설정 확인

장치의 방화벽이 서버로의 나가는 연결을 차단하지 않는지 확인하세요.
