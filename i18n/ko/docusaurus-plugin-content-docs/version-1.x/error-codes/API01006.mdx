# API01006: 네트워크 도달 불가 {#api01006-network-unreachable}

## 의미 {#what-this-means}

POS가 서버에 경로를 설정할 수 없습니다. 이는 네트워크 수준의 문제로, 데이터 패킷이 서버로 가는 경로를 찾을 수 없습니다.

## 일반적인 원인 {#common-causes}

- **네트워크 구성 문제** — 잘못된 게이트웨이 또는 라우팅 설정
- **ISP 문제** — 인터넷 서비스 제공업체에 문제가 발생 중
- **VPN 간섭** — VPN이 트래픽을 잘못 라우팅할 수 있음
- **서버 네트워크 문제** — 서버의 네트워크가 다운되었을 수 있음

## 해결 방법 {#how-to-fix}

### 1. 인터넷 연결 확인 {#1-check-your-internet-connection}

- 다른 웹사이트에 접속해 보세요
- 라우터/모뎀을 재시작하세요
- 네트워크에 연결되어 있는지 확인하세요

### 2. VPN 일시적으로 비활성화 {#2-disable-vpn-temporarily}

VPN을 사용하는 경우:
- 연결을 끊고 다시 시도하세요
- 다른 VPN 서버를 시도하세요
- VPN이 서버 위치에 접근을 허용하는지 확인하세요

### 3. ISP에 문의 {#3-check-with-your-isp}

다음 경우 인터넷 서비스 제공업체에 연락하세요:
- 다른 사이트도 도달할 수 없는 경우
- 네트워크 장비를 재시작한 후에도 문제가 지속되는 경우

### 4. 서버 상태 확인 {#4-verify-server-status}

- 호스팅 제공업체에서 네트워크 문제가 보고되었는지 확인하세요
- 다른 네트워크(예: 모바일 데이터)에서 사이트에 접근해 보세요
- 온라인 도구를 사용해 사이트가 모두에게 다운되었는지 확인하세요

### 5. 방화벽 설정 확인 {#5-check-firewall-settings}

장치의 방화벽이 서버로의 아웃고잉 연결을 차단하고 있지 않은지 확인하세요.
