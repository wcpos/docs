# API05002: WCPOS 플러그인이 발견되지 않음 {#api05002-wcpos-plugin-not-found}

## 의미 {#what-this-means}

WCPOS(우커머스 POS) 플러그인이 WordPress 사이트에 설치되지 않았거나 활성화되지 않았습니다. POS 애플리케이션은 WooCommerce와 통신하기 위해 이 플러그인이 필요합니다.

## 일반적인 원인 {#common-causes}

- **플러그인이 설치되지 않음** — 플러그인이 설치되지 않았습니다
- **플러그인이 비활성화됨** — 플러그인이 비활성화되었습니다
- **플러그인이 삭제됨** — 플러그인 파일이 제거되었습니다
- **잘못된 사이트** — WCPOS가 없는 사이트에 연결됨

## 해결 방법 {#how-to-fix}

### 1. 플러그인 설치 {#1-install-the-plugin}

설치되지 않은 경우:
1. WordPress 관리 → 플러그인 → 새로 추가
2. "WooCommerce POS" 검색
3. 플러그인 설치 및 활성화

또는 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)에서 다운로드하세요.

### 2. 플러그인 활성화 {#2-activate-the-plugin}

설치되었지만 활성화되지 않은 경우:
1. 플러그인 → 설치된 플러그인으로 이동
2. "WooCommerce POS" 찾기
3. "활성화" 클릭

### 3. 오류 확인 {#3-check-for-errors}

플러그인이 활성화되지 않는 경우:
- 오류 로그에서 PHP 오류 확인
- WooCommerce가 설치되고 활성화되었는지 확인
- PHP 버전 요구 사항 확인
- 플러그인 충돌 여부 확인

### 4. URL 확인 {#4-verify-the-url}

POS가 올바른 WordPress 사이트에 연결되도록 구성되었는지 확인하세요:
- POS 설정에서 URL 확인
- WordPress 설치와 일치하는지 확인

### 5. 필요시 재설치 {#5-reinstall-if-necessary}

플러그인이 손상된 경우:
1. 플러그인을 비활성화하고 삭제
2. WordPress.org에서 재설치
3. 플러그인을 활성화
