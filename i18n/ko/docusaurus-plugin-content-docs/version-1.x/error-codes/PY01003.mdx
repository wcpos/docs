# PY01003: 카드 만료됨

## 의미

결제 카드의 유효 기간이 지나 더 이상 거래에 사용할 수 없습니다.

## 일반적인 원인

- **만료된 카드** — 카드의 유효 기간이 지남
- **만료일 잘못 입력** — 만료 날짜를 잘못 입력함
- **새 카드 미활성화** — 교체 카드를 받았지만 아직 활성화되지 않음

## 해결 방법

### 1. 만료 날짜 확인

카드 확인:
- 만료 날짜 찾기 (MM/YY 형식)
- 현재 날짜와 비교
- 입력한 날짜가 카드의 날짜와 일치하는지 확인

### 2. 다른 카드 사용

카드가 실제로 만료된 경우:
- 유효한 다른 카드 사용
- 현금 또는 대체 결제 방법 사용
- 고객에게 대체 카드가 있는지 문의

### 3. 업데이트된 카드 받기

카드 소지자는 다음을 해야 합니다:
- 우편으로 온 교체 카드 확인
- 받은 경우 새 카드 활성화
- 교체 카드를 받지 못한 경우 은행에 문의

### 4. 입력 확인

만료 날짜 입력 다시 확인:
- MM/YY 형식
- 올바른 월과 년도
- 오타 없는지 확인

## 고객 서비스 팁

카드가 만료된 경우:
- 정중하게 고객에게 알림
- 교체 카드 확인 제안
- 대체 결제 방법 제시
- 이해심 있게 대응 — 누구에게나 발생할 수 있는 상황
