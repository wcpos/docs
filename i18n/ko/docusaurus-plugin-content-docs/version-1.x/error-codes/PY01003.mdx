# PY01003: 카드 만료 {#py01003-card-expired}

## 의미 {#what-this-means}

결제 카드의 유효기간이 만료되어 더 이상 거래에 사용할 수 없습니다.

## 일반적인 원인 {#common-causes}

- **카드 만료** — 카드의 유효 기간이 지나감
- **잘못된 만료일 입력** — 만료일이 잘못 입력됨
- **새 카드 미활성화** — 교체 카드를 받았으나 아직 활성화되지 않음

## 해결 방법 {#how-to-fix}

### 1. 만료일 확인 {#1-check-the-expiration-date}

카드를 확인하십시오:
- 만료일 찾기 (MM/YY 형식)
- 현재 날짜와 비교
- 입력된 날짜가 카드와 일치하는지 확인

### 2. 다른 카드 사용 {#2-use-a-different-card}

카드가 실제로 만료된 경우:
- 다른 유효한 카드 사용
- 현금 또는 대체 결제 수단 사용
- 고객에게 교체 카드가 있는지 문의

### 3. 업데이트된 카드 받기 {#3-get-the-updated-card}

카드 소지자는:
- 교체 카드를 위해 우편을 확인
- 새 카드를 받았다면 활성화
- 교체 카드가 도착하지 않았다면 은행에 연락

### 4. 입력 확인 {#4-verify-entry}

만료일 입력을 다시 확인:
- MM/YY 형식
- 올바른 월과 연도
- 오타 없음

## 고객 서비스 팁 {#customer-service-tips}

카드가 만료된 경우:
- 고객에게 정중하게 알리기
- 교체 카드 여부를 확인하도록 제안
- 대체 결제 옵션 제공
- 이해심을 가지세요 — 이는 모든 사람에게 발생할 수 있음
