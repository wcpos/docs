# API 오류

API 오류는 WooCommerce 서버와 통신할 때 발생합니다. 이 오류는 `API`로 접두사가 붙으며 다음과 같은 카테고리로 구성되어 있습니다:

## 카테고리

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| [연결](#connection-errors) | API01xxx | 네트워크 및 연결 문제 |
| [인증](#authentication-errors) | API02xxx | 로그인, 토큰 및 권한 문제 |
| [요청](#request-errors) | API03xxx | 발신 요청의 문제 |
| [응답](#response-errors) | API04xxx | 서버 응답의 문제 |
| [플러그인/워드프레스](#pluginwordpress-errors) | API05xxx | 워드프레스 또는 WooCommerce 플러그인 문제 |
| [구성](#configuration-errors) | API06xxx | 설정 및 구성 문제 |

## 연결 오류

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [API01001](./API01001) | 연결 시간 초과 | 서버가 응답하는 데 너무 오랜 시간이 걸렸습니다 |
| [API01002](./API01002) | 연결 거부 | 서버가 연결을 거부했습니다 |
| [API01003](./API01003) | 연결 재설정 | 연결이 예기치 않게 종료되었습니다 |
| [API01004](./API01004) | DNS 확인 실패 | 서버 주소를 확인할 수 없습니다 |

| [API01006](./API01006) | 네트워크 도달 불가 | 네트워크에 도달할 수 없습니다 |
| [API01007](./API01007) | 기기 오프라인 | 기기가 인터넷에 연결되어 있지 않습니다 |
| [API01008](./API01008) | 웹사이트 사용 불가 | 웹사이트가 응답하지 않습니다 |
