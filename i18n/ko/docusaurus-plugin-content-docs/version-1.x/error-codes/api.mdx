# API 오류

API 오류는 WooCommerce 서버와 통신할 때 발생합니다. 이러한 오류는 `API`로 시작되며 다음 카테고리로 구성됩니다:

## 카테고리

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| [연결](#연결-오류) | API01xxx | 네트워크 및 연결 문제 |
| [인증](#인증-오류) | API02xxx | 로그인, 토큰, 권한 문제 |
| [요청](#요청-오류) | API03xxx | 발신 요청 문제 |
| [응답](#응답-오류) | API04xxx | 서버 응답 문제 |
| [플러그인/워드프레스](#플러그인워드프레스-오류) | API05xxx | WordPress 또는 WooCommerce 플러그인 문제 |
| [구성](#구성-오류) | API06xxx | 설정 및 구성 문제 |

---

## 연결 오류

POS와 서버 간의 네트워크 및 연결 문제입니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [API01001](./API01001) | 연결 시간 초과 | 서버가 응답하는 데 너무 오래 걸림 |
| [API01002](./API01002) | 연결 거부 | 서버가 연결을 거부함 |
| [API01003](./API01003) | 연결 재설정 | 연결이 예기치 않게 닫힘 |
| [API01004](./API01004) | DNS 확인 실패 | 서버 주소를 확인할 수 없음 |
| [API01005](./API01005) | SSL 인증서 오류 | 사이트의 보안 인증서에 문제 있음 |
| [API01006](./API01006) | 네트워크 연결 불가 | 네트워크에 연결할 수 없음 |
| [API01007](./API01007) | 장치 오프라인 | 장치가 인터넷에 연결되어 있지 않음 |
| [API01008](./API01008) | 웹사이트 사용 불가 | 웹사이트가 응답하지 않음 |

## 인증 오류

| 코드 | 이름 | 설명 |
|------|------|-------------|

| [API02002](./API02002) | 토큰 만료 | 세션이 만료됨 |
| [API02003](./API02003) | 토큰 무효 | 인증 토큰이 유효하지 않음 |
| [API02004](./API02004) | 사용자 권한 없음 | 이 작업을 수행할 권한이 없음 |
| [API02005](./API02005) | 권한 부족 | 사용자 역할에 필요한 권한 없음 |
| [API02006](./API02006) | API 키 무효 | WooCommerce API 키가 유효하지 않음 |
| [API02007](./API02007) | 토큰 갱신 실패 | 세션을 갱신할 수 없음 |
| [API02008](./API02008) | 갱신 토큰 무효 | 갱신 토큰이 유효하지 않음 |
| [API02009](./API02009) | 갱신 토큰 만료 | 갱신 토큰이 만료됨 |
| [API02010](./API02010) | 인증 필요 | 이 작업에는 인증이 필요함 |

| [API03003](./API03003) | 잘못된 매개변수 값 | 매개변수 값이 잘못됨 |
| [API03004](./API03004) | 요청 크기 초과 | 요청이 크기 제한을 초과함 |

| [API03006](./API03006) | 지원되지 않는 방법 | HTTP 메서드가 지원되지 않음 |
| [API03007](./API03007) | 요청 대기열 꽉 참 | 보류 중인 요청이 너무 많음 |
