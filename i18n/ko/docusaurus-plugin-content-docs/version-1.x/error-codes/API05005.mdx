# API05005: 플러그인을 찾을 수 없음

## 의미는 무엇인가요

필요한 WordPress 플러그인이 설치되어 있지 않거나 활성화되어 있지 않습니다. 이는 WooCommerce 자체, 필요한 확장 프로그램 또는 통합 플러그인이 될 수 있습니다.

## 일반적인 원인

- **WooCommerce 비활성화** — WooCommerce가 비활성화되었습니다.
- **필요한 확장 프로그램 없음** — 필요한 WooCommerce 확장 프로그램이 설치되어 있지 않습니다.
- **플러그인 충돌** — 다른 플러그인으로 인해 필요한 플러그인이 비활성화되었습니다.
- **업데이트 실패** — 플러그인 업데이트가 실패하여 비활성화 상태로 남아 있습니다.

## 해결 방법

### 1. WooCommerce가 활성화되어 있는지 확인

WCPOS는 WooCommerce가 필요합니다:
1. 플러그인 → 설치된 플러그인으로 이동합니다.
2. "WooCommerce"를 찾습니다.
3. 활성화되어 있는지 확인합니다.

### 2. 필요한 확장 프로그램 확인

일부 WCPOS 기능은 확장 프로그램이 필요할 수 있습니다:
- 특정 플러그인 이름에 대한 오류 메시지를 확인합니다.
- 필요한 확장 프로그램을 설치하고 활성화합니다.

### 3. 플러그인 충돌 검토

플러그인이 충돌로 인해 비활성화된 경우:
1. WordPress 알림에 대한 이메일을 확인합니다.
2. 오류 로그를 검토합니다.
3. 충돌을 해결하고 재활성화합니다.

### 4. 플러그인 파일 확인

플러그인 파일이 누락된 경우:
1. WordPress.org에서 플러그인을 재설치합니다.
2. 또는 백업에서 복원합니다.
3. 플러그인을 활성화합니다.

### 5. 모든 플러그인 업데이트

구버전 플러그인으로 인해 문제가 발생할 수 있습니다:
1. 대시보드 → 업데이트로 이동합니다.
2. 모든 플러그인을 업데이트합니다.
3. 필요한 모든 플러그인이 활성화되어 있는지 확인합니다.

## 필수 플러그인

최소한 WCPOS에는 다음이 필요합니다:
- **WooCommerce** — 전자상거래 플랫폼
- **WCPOS** — POS 플러그인

추가 기능은 다른 플러그인이 필요할 수 있습니다.
