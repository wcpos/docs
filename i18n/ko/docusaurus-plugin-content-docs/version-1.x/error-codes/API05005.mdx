# API05005: 플러그인을 찾을 수 없음

## 이 오류의 의미

필수 WordPress 플러그인이 설치되지 않았거나 활성화되어 있지 않습니다. 이는 WooCommerce 자체, 필수 확장 프로그램 또는 통합 플러그인일 수 있습니다.

## 일반적인 원인

- **WooCommerce 비활성화** — WooCommerce가 비활성화됨
- **필수 확장 프로그램 누락** — 필요한 WooCommerce 확장 프로그램이 설치되지 않음
- **플러그인 충돌** — 다른 플러그인으로 인해 필수 플러그인이 비활성화됨
- **업데이트 실패** — 플러그인 업데이트 실패로 비활성화됨

## 해결 방법

### 1. WooCommerce 활성화 확인

WCPOS는 WooCommerce를 필요로 합니다:
1. 플러그인 → 설치된 플러그인으로 이동
2. "WooCommerce" 찾기
3. 활성화되어 있는지 확인

### 2. 필수 확장 프로그램 확인

일부 WCPOS 기능은 확장 프로그램을 필요로 할 수 있습니다:
- 오류 메시지에서 특정 플러그인 이름 확인
- 필요한 확장 프로그램 설치 및 활성화

### 3. 플러그인 충돌 검토

플러그인이 충돌로 인해 비활성화된 경우:
1. WordPress 알림 이메일 확인
2. 오류 로그 검토
3. 충돌 해결 및 재활성화

### 4. 플러그인 파일 확인

플러그인 파일이 누락된 경우:
1. WordPress.org에서 플러그인 다시 설치
2. 또는 백업에서 복원
3. 플러그인 활성화

### 5. 모든 플러그인 업데이트

오래된 플러그인이 문제를 일으킬 수 있습니다:
1. 대시보드 → 업데이트로 이동
2. 모든 플러그인 업데이트
3. 필요한 모든 플러그인이 활성화되었는지 확인

## 필수 플러그인

최소한 WCPOS는 다음을 필요로 합니다:
- **WooCommerce** — 전자상거래 플랫폼
- **WCPOS** — POS 플러그인

추가 기능은 다른 플러그인을 필요로 할 수 있습니다.
