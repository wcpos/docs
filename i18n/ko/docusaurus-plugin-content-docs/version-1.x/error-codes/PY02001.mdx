# PY02001: 결제 게이트웨이 오류

## 의미

결제 게이트웨이와 통신하는 동안 오류가 발생했습니다. 결제 프로세서가 요청을 완료할 수 없었습니다.

## 일반적인 원인

- **게이트웨이 중단** — 결제 서비스에 문제가 있음
- **구성 오류** — 게이트웨이 설정이 잘못됨
- **API 자격 증명** — 잘못되거나 만료된 API 키
- **네트워크 문제** — 게이트웨이에 대한 연결 문제

## 해결 방법

### 1. 거래 재시도

게이트웨이에 일시적인 문제가 있을 수 있습니다:
- 잠시 기다리기
- 결제 처리 다시 시도
- 문제가 자체적으로 해결될 수 있음

### 2. 게이트웨이 상태 확인

결제 제공업체의 상태 페이지 방문:
- Stripe: status.stripe.com
- PayPal: PayPal 커뮤니티 또는 상태 확인
- Square: status.squareup.com
- 알려진 장애 확인

### 3. 게이트웨이 구성 확인

WooCommerce 관리자에서:
1. WooCommerce → 설정 → 결제로 이동
2. 사용 중인 결제 게이트웨이 클릭
3. API 키가 올바른지 확인
4. 테스트/라이브 모드가 키와 일치하는지 확인

### 4. API 자격 증명 확인

자격 증명 유효성 검사:
- API 키가 만료되지 않았는지
- 라이브 거래에는 라이브 키 사용
- 테스트 거래에는 테스트 키 사용
- 자격 증명이 올바른 계정과 일치하는지

### 5. 다른 게이트웨이로 테스트

가능한 경우:
- 대체 게이트웨이로 처리 시도
- 현금 결제를 대안으로 사용
- 나중에 정산할 판매 완료

## 스토어 소유자용

일반적인 구성 문제:
- 테스트 및 라이브 API 키 혼용
- 다른 계정의 API 키
- 비활성화된 결제 방법
- 만료되거나 취소된 자격 증명

문제가 지속되면 결제 제공업체 지원팀에 문의하세요.
