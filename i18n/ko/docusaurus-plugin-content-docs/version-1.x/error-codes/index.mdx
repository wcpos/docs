# 오류 코드

WCPOS가 문제를 발견하면 문제를 식별하는 데 도움이 되는 오류 코드를 표시합니다. 각 오류 코드는 `[DOMAIN][CATEGORY][SPECIFIC_CODE]` 형식을 따르며, 오류의 유형과 성격을 신속하게 이해하는 데 도움이 됩니다.

## 오류 코드 형식

| 구성 요소 | 설명 | 예시 |
|-----------|-------------|---------|
| **도메인** | 시스템 영역 | `API`, `DB`, `PY`, `SY` |
| **카테고리** | 오류 카테고리 | `01`, `02`, `03` 등 |
| **특정 코드** | 고유 식별자 | `001`, `002`, `003` 등 |

**예시:** `API04001` = API (도메인) + 04 (응답 오류) + 001 (유효하지 않은 응답 형식)

## 오류 도메인

### [API 오류](./api)

API 오류는 WooCommerce 서버와 통신할 때 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 연결 | API01xxx | 네트워크 및 연결 문제 |
| 인증 | API02xxx | 로그인, 토큰 및 권한 문제 |
| 요청 | API03xxx | 전송 요청 문제 |
| 응답 | API04xxx | 서버 응답 문제 |
| 플러그인/워드프레스 | API05xxx | 워드프레스 또는 WooCommerce 플러그인 문제 |
| 구성 | API06xxx | 설정 및 구성 문제 |

### [데이터베이스 오류](./db)

데이터베이스 오류는 데이터를 로컬에 저장하거나 검색할 때 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 연결 | DB01xxx | 데이터베이스 연결 및 트랜잭션 문제 |
| 데이터 | DB02xxx | 레코드 및 제약 조건 문제 |
| 쿼리 | DB03xxx | 쿼리 구문 및 데이터 유형 문제 |

### [결제 오류](./py)

결제 오류는 체크아웃 및 결제 처리 중에 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 카드 | PY01xxx | 결제 카드 문제 |
| 게이트웨이 | PY02xxx | 결제 게이트웨이 통신 문제 |

### [시스템 오류](./sy)

시스템 오류는 장치 리소스 및 시스템 구성과 관련이 있습니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 리소스 | SY01xxx | 메모리, 디스크 및 권한 문제 |
| 서비스 | SY02xxx | 구성 및 서비스 가용성 |

## 도움이 더 필요하신가요?

2. 지원을 위해 [Discord 커뮤니티](https://wcpos.com/discord)에 참여하세요.
3. [GitHub](https://github.com/wcpos)에서 문제를 보고하세요.
