# 오류 코드

WCPOS에서 문제가 발생하면 문제를 식별하는 데 도움이 되는 오류 코드를 표시합니다. 각 오류 코드는 `[도메인][카테고리][특정 코드]` 형식을 따라 오류의 유형과 특성을 빠르게 이해할 수 있습니다.

## 오류 코드 형식

| 구성 요소 | 설명 | 예시 |
|-----------|-------------|---------|
| **도메인** | 시스템 영역 | `API`, `DB`, `PY`, `SY` |
| **카테고리** | 오류 카테고리 | `01`, `02`, `03` 등 |
| **특정 코드** | 고유 식별자 | `001`, `002`, `003` 등 |

**예:** `API04001` = API (도메인) + 04 (응답 오류) + 001 (잘못된 응답 형식)

## 오류 도메인

### [API 오류](./api)

API 오류는 WooCommerce 서버와 통신할 때 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 연결 | API01xxx | 네트워크 및 연결 문제 |
| 인증 | API02xxx | 로그인, 토큰 및 권한 문제 |
| 요청 | API03xxx | 송신 요청 문제 |
| 응답 | API04xxx | 서버 응답 문제 |
| 플러그인/WordPress | API05xxx | WordPress 또는 WooCommerce 플러그인 문제 |
| 구성 | API06xxx | 설정 및 구성 문제 |

### [데이터베이스 오류](./db)

데이터베이스 오류는 로컬에서 데이터를 저장하거나 검색할 때 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 연결 | DB01xxx | 데이터베이스 연결 및 트랜잭션 문제 |
| 데이터 | DB02xxx | 레코드 및 제약 조건 문제 |
| 쿼리 | DB03xxx | 쿼리 구문 및 데이터 유형 문제 |

### [결제 오류](./py)

결제 오류는 결제 및 결제 처리 중에 발생합니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 카드 | PY01xxx | 결제 카드 문제 |
| 게이트웨이 | PY02xxx | 결제 게이트웨이 통신 문제 |

### [시스템 오류](./sy)

시스템 오류는 장치 리소스 및 시스템 구성과 관련됩니다.

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| 리소스 | SY01xxx | 메모리, 디스크 및 권한 문제 |
| 서비스 | SY02xxx | 구성 및 서비스 가용성 |
