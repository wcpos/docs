# PY01004: 유효하지 않은 카드 번호

## 의미

입력된 카드 번호가 유효하지 않습니다. 카드 번호는 특정 패턴을 따르며 검증을 위한 체크 숫자를 포함합니다.

## 일반적인 원인

- **입력 오류** — 하나 이상의 숫자를 잘못 입력
- **누락된 숫자** — 모든 숫자를 입력하지 않음
- **추가 숫자** — 너무 많은 숫자 입력
- **잘못된 카드** — 실제 카드와 다른 카드 번호
- **카드 리더 오류** — 칩/스와이프가 잘못된 데이터를 읽음

## 해결 방법

### 1. 카드 번호 다시 입력

모든 숫자를 주의 깊게 입력:
- 실제 카드 확인
- 각 숫자를 천천히 입력
- 제출 전 전체 번호 확인

### 2. 카드 유형 확인

지원되는 카드 유형 확인:
- Visa (4로 시작)
- Mastercard (51-55 또는 22-27로 시작)
- Amex (34 또는 37로 시작)
- Discover (6011, 622, 644-649, 65로 시작)

### 3. 카드 리더로 다시 시도

칩/스와이프 리더 사용 시:
- 카드 칩 청소
- 다른 읽기 방법 시도 (칩 vs 스와이프)
- 리더 실패 시 수동 입력

### 4. 다른 카드 사용

카드 번호가 계속 실패하는 경우:
- 카드가 손상되었을 수 있음
- 다른 카드 시도
- 대체 결제 수단 사용

### 5. 카드 손상 확인

카드 물리적 문제:
- 긁힌 자기 띠
- 손상된 칩
- 마모되거나 흐릿한 숫자
- 카드 교체 필요

## 카드 번호 형식

| 카드 유형 | 길이 | 시작 숫자 |
|-----------|--------|-------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55, 22-27 |
| Amex | 15 | 34, 37 |
| Discover | 16 | 6011, 622, 644-649, 65 |
