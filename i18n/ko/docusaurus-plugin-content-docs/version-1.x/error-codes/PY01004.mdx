# PY01004: 유효하지 않은 카드 번호 {#py01004-invalid-card-number}

## 의미 {#what-this-means}

입력한 카드 번호가 유효하지 않습니다. 카드 번호는 특정 패턴을 따르며 유효성을 검증하기 위해 체크 자리가 포함되어 있습니다.

## 일반적인 원인 {#common-causes}

- **오타** — 하나 이상의 숫자가 잘못 입력됨
- **누락된 숫자** — 모든 숫자가 입력되지 않음
- **여분의 숫자** — 너무 많은 숫자가 입력됨
- **잘못된 카드** — 실제 카드와 다른 카드 번호
- **카드 리더기 오류** — 칩/스와이프가 잘못된 데이터를 읽음

## 해결 방법 {#how-to-fix}

### 1. 카드 번호 재입력 {#1-re-enter-the-card-number}

모든 숫자를 주의 깊게 입력하세요:
- 실제 카드를 확인하세요
- 각 숫자를 천천히 입력하세요
- 제출하기 전에 전체 번호를 확인하세요

### 2. 카드 종류 확인 {#2-check-card-type}

카드 종류가 지원되는지 확인하세요:
- 비자(Visa) (4로 시작)
- 마스터카드(Mastercard) (51-55 또는 22-27로 시작)
- 아메리칸 익스프레스(Amex) (34 또는 37로 시작)
- 디스커버(Discover) (6011, 622, 644-649, 65로 시작)

### 3. 카드 리더로 다시 시도 {#3-try-again-with-card-reader}

칩/스와이프 리더를 사용하는 경우:
- 카드 칩을 청소하세요
- 다른 읽기 방법(칩 대 스와이프)을 시도하세요
- 리더가 실패하면 수동으로 입력하세요

### 4. 다른 카드 사용 {#4-use-a-different-card}

카드 번호가 계속 실패하는 경우:
- 카드가 손상되었을 수 있음
- 다른 카드를 시도하세요
- 대체 결제 방법을 사용하세요

### 5. 카드 손상 여부 확인 {#5-check-for-card-damage}

실제 카드 문제:
- 긁힌 자기 띠
- 손상된 칩
- 마모되거나 흐릿한 숫자
- 카드 교체가 필요할 수 있음

## 카드 번호 형식 {#card-number-formats}

| 카드 종류 | 길이 | 시작 번호 |
|-----------|--------|-------------|
| 비자 | 16 | 4 |
| 마스터카드 | 16 | 51-55, 22-27 |
| 아메리칸 익스프레스 | 15 | 34, 37 |
| 디스커버 | 16 | 6011, 622, 644-649, 65 |
