# DB03003: 필수 필드 누락

## 의미

필수 필드가 제공되지 않았습니다. 데이터베이스는 필수 정보가 누락되어 레코드를 저장할 수 없습니다.

## 일반적인 원인

- **불완전한 양식** — 필수 필드가 채워지지 않음
- **데이터 동기화 문제** — 필수 데이터가 제대로 동기화되지 않음
- **유효성 검사 우회** — 유효성 검사 없이 양식 제출
- **사용자 지정 필드 필수** — 비어 있는 필수 사용자 지정 필드

## 해결 방법

### 1. 모든 필수 필드 완성

다음 사항을 확인하세요:
- 별표(*)로 표시된 필드
- 강조 표시되거나 오류 표시된 필드
- 값이 있어야 하는 빈 필드

### 2. 새로 고침 및 다시 입력

양식 상태가 잘못된 경우:
1. 페이지/화면 새로 고침
2. 모든 정보 다시 입력
3. 제출 전 모든 필드 확인

### 3. WooCommerce 설정 확인

사용자 지정 필드가 필요한 경우:
1. WooCommerce 체크아웃 설정 검토
2. 필수 사용자 지정 필드 확인
3. POS에서 해당 필드 제공 확인

### 4. 필수 데이터 동기화

관련 데이터가 누락된 경우:
- 제품/고객 등 새로 고침
- 동기화 완료 대기
- 작업 다시 시도

### 5. 필수 필드 설정 검토

WooCommerce 관리자에서:
- 필수로 표시된 필드 확인
- POS에 실제로 필요한지 고려
- 필요에 따라 요구사항 조정

## 일반적인 필수 필드

주문에 대해 일반적으로 필요한 필드:
- 최소 하나의 라인 항목
- 결제 방법
- 고객 (설정에 따라)

제품에 대해 일반적으로 필요한 필드:
- 제품 이름
- 가격 (단순 제품의 경우)
