# DB03003: 필수 필드 누락 {#db03003-missing-required-field}

## 의미 {#what-this-means}

필수 필드가 제공되지 않았습니다. 필수 정보가 누락되어 데이터베이스가 레코드를 저장할 수 없습니다.

## 일반적인 원인 {#common-causes}

- **불완전한 양식** — 필수 필드가 작성되지 않았습니다.
- **데이터 동기화 문제** — 필수 데이터가 제대로 동기화되지 않았습니다.
- **검증 우회** — 검증 없이 양식이 제출되었습니다.
- **필수 사용자 정의 필드** — 필수 사용자 정의 필드가 비어 있습니다.

## 수정 방법 {#how-to-fix}

### 1. 모든 필수 필드 작성 {#1-complete-all-required-fields}

양식을 확인하십시오:
- 별표(*)로 표시된 필드
- 강조 표시되었거나 오류가 표시된 필드
- 값이 있어야 하는 빈 필드

### 2. 새로 고치고 다시 입력 {#2-refresh-and-re-enter}

양식 상태가 잘못된 경우:
1. 페이지/화면을 새로 고치기
2. 모든 정보를 다시 입력하기
3. 제출하기 전에 모든 필드를 확인하기

### 3. WooCommerce 설정 확인 {#3-check-woocommerce-settings}

사용자 정의 필드가 필요한 경우:
1. WooCommerce 체크아웃 설정 검토
2. 필수 사용자 정의 필드 확인
3. POS에서 이러한 필드를 제공하는지 확인

### 4. 필수 데이터 동기화 {#4-sync-required-data}

관련 데이터가 누락된 경우:
- 제품/고객 등 새로 고치기
- 동기화 완료를 기다리기
- 작업을 다시 시도하기

### 5. 필수 필드 설정 검토 {#5-review-required-field-settings}

WooCommerce 어드민에서:
- 어떤 필드가 필수로 표시되어 있는지 확인
- 모든 필드가 POS에 정말 필요한지 고려
- 필요에 따라 요구 사항 조정

## 일반적인 필수 필드 {#common-required-fields}

주문 시 일반적으로 요구되는 필드:
- 최소한 하나의 품목
- 결제 방법
- 고객(설정에 따라)

제품에 대해 일반적으로 요구되는 필드:
- 제품 이름
- 가격(단순 제품의 경우)
