# 결제 오류

결제 오류는 결제 및 처리 과정에서 발생합니다. 이러한 오류는 `PY` 접두사가 붙으며, 결제 카드, 결제 게이트웨이 및 거래와 관련된 문제를 나타냅니다.

## 카테고리

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| [카드](#카드-오류) | PY01xxx | 결제 카드 관련 문제 |
| [결제 게이트웨이](#결제-게이트웨이-오류) | PY02xxx | 결제 게이트웨이 통신 문제 |

---

## 카드 오류

고객의 결제 카드와 관련된 문제입니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [PY01001](./PY01001) | 결제 거절 | 결제가 거절됨 |
| [PY01002](./PY01002) | 잔액 부족 | 결제에 필요한 자금이 부족함 |
| [PY01003](./PY01003) | 카드 만료 | 결제 카드가 만료됨 |
| [PY01004](./PY01004) | 잘못된 카드 번호 | 카드 번호가 유효하지 않음 |

## 결제 게이트웨이 오류

결제 게이트웨이와 통신 중 발생하는 문제입니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [PY02001](./PY02001) | 결제 게이트웨이 오류 | 결제 게이트웨이와 통신 중 오류 발생 |
| [PY02002](./PY02002) | 결제 시간 초과 | 결제 처리 시간 초과 |
