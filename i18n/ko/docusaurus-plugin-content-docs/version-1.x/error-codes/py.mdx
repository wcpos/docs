# 결제 오류

결제 오류는 체크아웃 및 결제 처리 중 발생합니다. 이러한 오류는 `PY`로 시작되며 결제 카드, 게이트웨이 및 거래와 관련된 문제입니다.

## 카테고리

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| [카드](#card-errors) | PY01xxx | 결제 카드 관련 문제 |
| [게이트웨이](#gateway-errors) | PY02xxx | 결제 게이트웨이 통신 문제 |

---

## 카드 오류

고객의 결제 카드와 관련된 문제입니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [PY01001](./PY01001) | 결제가 거부됨 | 결제가 거부되었습니다 |
| [PY01002](./PY01002) | 자금 부족 | 결제에 필요한 자금이 충분하지 않습니다 |
| [PY01003](./PY01003) | 카드가 만료됨 | 결제 카드가 만료되었습니다 |
| [PY01004](./PY01004) | 유효하지 않은 카드 번호 | 카드 번호가 유효하지 않습니다 |

## 게이트웨이 오류

결제 게이트웨이와의 통신 문제입니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [PY02001](./PY02001) | 결제 게이트웨이 오류 | 결제 게이트웨이와의 통신 오류 |
| [PY02002](./PY02002) | 결제 타임아웃 | 결제 처리 시간 초과 |
