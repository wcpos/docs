# API01007: 장치 오프라인 {#api01007-device-offline}

## 이것이 의미하는 바 {#what-this-means}

귀하의 장치가 인터넷에 연결되어 있지 않습니다. POS는 활성 네트워크 연결이 없음을 감지했습니다.

## 일반적인 원인 {#common-causes}

- **WiFi 연결 끊김** — 귀하의 장치가 WiFi 연결을 잃었습니다
- **이더넷 분리됨** — 네트워크 케이블이 분리되었습니다
- **비행기 모드 활성화됨** — 모든 무선 연결이 비활성화되었습니다
- **네트워크 어댑터 비활성화됨** — 네트워크 인터페이스가 꺼졌습니다

## 해결 방법 {#how-to-fix}

### 1. WiFi 연결 확인 {#1-check-wifi-connection}

- 귀하의 장치에서 WiFi가 활성화되어 있는지 확인합니다
- 올바른 네트워크에 연결되어 있는지 확인합니다
- WiFi 신호 강도를 확인합니다

### 2. 물리적 연결 확인 {#2-check-physical-connections}

이더넷 사용 시:
- 케이블이 단단히 연결되어 있는지 확인합니다
- 다른 케이블이나 포트를 사용해 보십시오
- 네트워크 스위치/라우터의 상태 표시등을 확인합니다

### 3. 비행기 모드 비활성화 {#3-disable-airplane-mode}

비행기 모드가 실수로 활성화되어 있는지 확인하고 비활성화합니다.

### 4. 네트워크 어댑터 재시작 {#4-restart-network-adapter}

**Windows:**
- 네트워크 연결을 엽니다
- 어댑터를 마우스 오른쪽 버튼으로 클릭 → 비활성화, 그런 다음 활성화

**macOS:**
- 메뉴 막대에서 WiFi를 끄고 켭니다
- 또는: 시스템 환경설정 → 네트워크 → 끄기/켜기

### 5. 장치 재시작 {#5-restart-your-device}

때때로 간단한 재시작이 네트워크 감지 문제를 해결합니다.

## 오프라인 모드 {#offline-mode}

오프라인 상태에서도 WCPOS는 다음을 수행할 수 있습니다:
- 로컬 캐시된 제품을 탐색하고 검색
- 로컬 캐시된 고객을 탐색하고 검색
- 새로운 주문을 시작하고 장바구니에 항목 추가
- 이전에 동기화된 데이터 보기

**연결 필요:**
- 주문 완료/결제
- 새로운 고객 생성

연결이 복구되면 POS는 자동으로 정상 운영을 재개합니다.
