# API01007: 장치 오프라인

## 의미

귀하의 장치가 인터넷에 연결되어 있지 않습니다. POS에서 활성 네트워크 연결이 없음을 감지했습니다.

## 일반적인 원인

- **WiFi 연결 끊김** — 장치의 WiFi 연결이 끊어졌습니다
- **이더넷 연결 해제** — 네트워크 케이블이 분리되었습니다
- **비행기 모드 활성화** — 모든 무선 연결이 비활성화되었습니다
- **네트워크 어댑터 비활성화** — 네트워크 인터페이스가 꺼졌습니다

## 해결 방법

### 1. WiFi 연결 확인

- 장치의 WiFi가 활성화되어 있는지 확인
- 올바른 네트워크에 연결되어 있는지 확인
- WiFi 신호 강도 확인

### 2. 물리적 연결 확인

이더넷 사용 시:
- 케이블이 제대로 연결되어 있는지 확인
- 다른 케이블이나 포트 시도
- 네트워크 스위치/라우터 표시등 확인

### 3. 비행기 모드 비활성화

비행기 모드가 실수로 활성화되었는지 확인하고 비활성화하세요.

### 4. 네트워크 어댑터 다시 시작

**Windows:**
- 네트워크 연결 열기
- 어댑터 마우스 오른쪽 버튼 클릭 → 비활성화, 그 후 활성화

**macOS:**
- 메뉴 표시줄에서 WiFi 끄고 켜기
- 또는: 시스템 환경설정 → 네트워크 → 끄기/켜기

### 5. 장치 다시 시작

간단한 다시 시작으로 네트워크 감지 문제가 해결될 수 있습니다.

## 오프라인 모드

오프라인 상태에서 WCPOS는 다음을 계속 수행할 수 있습니다:
- 로컬에 캐시된 제품 검색 및 찾아보기
- 로컬에 캐시된 고객 검색 및 찾아보기
- 새 주문 시작 및 장바구니에 항목 추가
- 이전에 동기화된 데이터 보기

**연결 필요:**
- 주문 완료/체크아웃
- 새 고객 생성

연결이 복구되면 POS가 자동으로 정상 작동을 재개합니다.
