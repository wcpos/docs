# SY01001: 메모리 부족

## 의미

당신의 장치의 메모리(RAM)가 부족하여 요청된 작업을 완료할 수 없습니다. POS가 제대로 작동하기 위해서는 충분한 메모리가 필요합니다.

## 일반적인 원인

- **앱이 너무 많이 열려 있음** — 메모리를 사용하는 다른 애플리케이션
- **대량 데이터 세트** — 많은 제품/고객이 로드됨
- **메모리 누수** — 애플리케이션이 메모리를 제대로 해제하지 않음
- **RAM 부족** — 장치에 메모리가 충분하지 않음

## 해결 방법

### 1. 다른 애플리케이션 닫기

메모리 확보하기:
- 사용하지 않는 브라우저 탭 닫기
- 다른 애플리케이션 닫기
- 백그라운드 프로그램 닫기

### 2. POS 재시작

애플리케이션 메모리 새로 고침:
1. POS를 완전히 닫기
2. 몇 초 기다리기
3. 애플리케이션 재오픈하기

### 3. 장치 재시작

전체 재시작은 모든 메모리를 지웁니다:
1. 열린 작업 저장하기
2. 컴퓨터/장치 재시작하기
3. 다시 시작할 때 POS만 열기

### 4. 브라우저 데이터 지우기 (웹 버전)

브라우저에서 사용하는 경우:
- 브라우저 캐시 지우기
- 브라우저 닫고 다시 열기
- 브라우저 탭 수 줄이기

### 5. 데이터 로드 줄이기

매우 큰 카탈로그가 있는 경우:
- 필터를 사용하여 한 번에 로드되는 데이터 줄이기
- 결과를 페이지로 나누는 것을 고려하기
- 가능하다면 동기화 범위 제한하기

## 최소 요구 사항

장치가 최소 사양을 충족하는지 확인하기:
- **RAM:** 최소 4GB, 권장 8GB
- **브라우저:** 최신 브라우저 (Chrome, Firefox, Edge)
- **데스크톱 앱:** 요구 사항에 대한 릴리스 노트 확인하기

## 지속적인 문제

자주 메모리가 부족한 경우:
- 장치의 RAM 업그레이드를 고려하기
- 더 많은 리소스를 가진 장치 사용하기
- 문제가 과도해 보이면 신고하기
