# SY01001: 메모리 부족

## 이 오류의 의미

기기의 메모리(RAM)가 부족하여 요청된 작업을 완료할 수 없습니다. POS는 제대로 작동하기 위해 충분한 메모리가 필요합니다.

## 일반적인 원인

- **너무 많은 앱 실행** — 다른 애플리케이션이 메모리를 사용 중
- **큰 데이터 세트** — 많은 제품/고객 로드
- **메모리 누수** — 애플리케이션이 메모리를 제대로 해제하지 않음
- **RAM 부족** — 기기의 메모리가 충분하지 않음

## 해결 방법

### 1. 다른 애플리케이션 닫기

메모리 확보:
- 사용하지 않는 브라우저 탭 닫기
- 다른 애플리케이션 닫기
- 백그라운드 프로그램 닫기

### 2. POS 다시 시작

애플리케이션 메모리 새로 고침:
1. POS 완전히 닫기
2. 몇 초 대기
3. 애플리케이션 다시 열기

### 3. 기기 다시 시작

전체 다시 시작으로 모든 메모리 지우기:
1. 열린 작업 저장
2. 컴퓨터/기기 다시 시작
3. 다시 시작 시 POS만 열기

### 4. 브라우저 데이터 지우기 (웹 버전)

브라우저에서 사용 중인 경우:
- 브라우저 캐시 지우기
- 브라우저 닫고 다시 열기
- 브라우저 탭 최소화

### 5. 데이터 로드 줄이기

매우 큰 카탈로그가 있는 경우:
- 필터를 사용하여 한 번에 적은 데이터 로드
- 결과 페이지화 고려
- 가능한 경우 동기화 범위 제한

## 최소 요구 사항

기기가 최소 사양을 충족하는지 확인:
- **RAM:** 최소 4GB, 8GB 권장
- **브라우저:** 최신 브라우저 (Chrome, Firefox, Edge)
- **데스크톱 앱:** 릴리스 노트에서 요구 사항 확인

## 지속적인 문제

메모리 부족이 자주 발생하는 경우:
- 기기의 RAM 업그레이드 고려
- 더 많은 리소스가 있는 기기 사용
- 과도해 보이는 경우 문제 보고
