# API03002: 필수 매개변수 누락

## 이 의미

서버가 작업을 완료하는 데 필요한 데이터가 요청에 누락되었습니다. 필수 필드가 요청에 포함되지 않았습니다.

## 일반적인 원인

- **불완전한 양식** — 필수 필드가 채워지지 않음
- **데이터 저장 실패** — 양식 데이터가 제대로 캡처되지 않음
- **버전 불일치** — 플러그인이 POS가 보내는 것과 다른 매개변수를 예상함
- **사용자 정의 필드** — 필수 사용자 정의 필드가 누락됨

## 해결 방법

### 1. 필수 필드 완성

모든 필수 정보가 채워졌는지 확인:
- 고객 세부 정보 (필요한 경우)
- 제품 정보
- 주문 세부 정보

### 2. 새로 고침 및 다시 시도

양식 상태가 불완전할 수 있습니다:
1. POS 새로 고침
2. 필요한 정보 다시 입력
3. 작업 다시 시도

### 3. 플러그인 버전 확인

호환성 확인:
- WCPOS 플러그인 업데이트
- POS 애플리케이션 업데이트
- 둘 다 호환 가능한 버전이어야 함

### 4. 필수 필드 설정 확인

WooCommerce에서 일부 필드가 필수로 표시될 수 있습니다:
- 체크아웃 필드 요구사항 검토
- 사용자 정의 필드 구성 확인
- 필요한 경우 요구사항 조정

### 5. 사용자 정의 통합 확인

사용자 정의 플러그인 또는 통합을 사용하는 경우:
- 추가 필드가 필요할 수 있음
- 플러그인 문서 검토
- 충돌 여부 확인
