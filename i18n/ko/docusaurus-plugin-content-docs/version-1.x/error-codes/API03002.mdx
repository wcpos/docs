# API03002: 필수 매개변수 누락

## 의미

요청에 서버가 작업을 완료하는 데 필요한 데이터가 누락되었습니다. 필수 필드가 요청에 포함되지 않았습니다.

## 일반적인 원인

- **불완전한 양식** — 필수 필드가 채워지지 않았습니다.
- **데이터 저장 실패** — 양식 데이터가 제대로 저장되지 않았습니다.
- **버전 불일치** — 플러그인이 POS에서 전송하는 것과 다른 매개변수를 예상합니다.
- **사용자 정의 필드** — 필수 사용자 정의 필드가 누락되었습니다.

## 해결 방법

### 1. 필수 필드 채우기

모든 필수 정보가 채워졌는지 확인하세요:
- 고객 세부정보 (필요한 경우)
- 제품 정보
- 주문 세부정보

### 2. 새로 고침 후 재시도

양식 상태가 불완전할 수 있습니다:
1. POS를 새로 고칩니다.
2. 필요한 정보를 다시 입력합니다.
3. 작업을 다시 시도합니다.

### 3. 플러그인 버전 확인

호환성을 보장하세요:
- WCPOS 플러그인을 업데이트합니다.
- POS 애플리케이션을 업데이트합니다.
- 두 가지 모두 호환 가능한 버전이어야 합니다.

### 4. 필수 필드 설정 확인

WooCommerce에서 일부 필드는 필수로 표시될 수 있습니다:
- 체크아웃 필드 요구 사항을 검토합니다.
- 사용자 정의 필드 구성을 확인합니다.
- 필요시 요구 사항을 조정합니다.

### 5. 사용자 정의 통합 확인

사용자 정의 플러그인이나 통합을 사용하는 경우:
- 추가 필드가 필요할 수 있습니다.
- 플러그인 문서를 검토합니다.
- 충돌 여부를 확인합니다.
