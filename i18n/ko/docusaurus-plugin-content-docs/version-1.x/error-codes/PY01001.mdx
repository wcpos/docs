# PY01001: 결제가 거부됨 {#py01001-payment-declined}

## 의미 {#what-this-means}

결제가 결제 처리업체나 카드 발급사에 의해 거부되었습니다. 거래를 완료할 수 없습니다.

## 일반적인 원인 {#common-causes}

- **잔액 부족** — 계좌에 충분한 돈이 없음
- **카드 한도 초과** — 거래가 카드 한도를 초과함
- **사기 방지** — 은행에서 의심스러운 것으로 표시됨
- **잘못된 카드 정보** — 카드 번호, CVV 또는 만료일 오류
- **카드 제한** — 이 유형의 구매에 대해 카드가 활성화되지 않음

## 해결 방법 {#how-to-fix}

### 1. 카드 정보 확인 {#1-verify-card-details}

입력된 정보가 올바른지 확인하세요:
- 카드 번호 (모든 숫자)
- 만료 날짜
- CVV/보안 코드
- 청구 주소 (필요한 경우)

### 2. 다시 시도하기 {#2-try-again}

때때로 일시적인 문제가 발생할 수 있습니다:
- 잠시 기다립니다
- 카드 정보를 다시 입력합니다
- 거래를 다시 시도합니다

### 3. 다른 결제 방법 사용 {#3-use-a-different-payment-method}

카드가 계속 거부되는 경우:
- 다른 카드를 시도합니다
- 현금 결제 사용
- 대체 결제 방법 사용

### 4. 은행에 연락하기 {#4-contact-the-bank}

카드 소유자는 다음을 수행해야 할 수 있습니다:
- 은행과 거래 확인
- 사기 차단 해제
- 카드가 활성화되어 있는지 확인
- 사용 한도 확인

### 5. 제한 사항 확인 {#5-check-for-restrictions}

일부 카드에는 제한이 있습니다:
- 국제 거래 비활성화
- 온라인/POS 거래 차단
- 특정 상인 카테고리 차단

## 상점 소유자를 위한 조언 {#for-store-owners}

빈번한 거부가 발생하는 경우:
- 상인 계정이 양호한 상태인지 확인
- 결제 게이트웨이 구성 확인
- 사기 방지 설정 검토
- 결제 처리업체에 연락 고려
