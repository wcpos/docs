# PY01001: 결제 거부됨

## 의미

결제 프로세서 또는 카드 발행사에 의해 결제가 거부되었습니다. 트랜잭션을 완료할 수 없습니다.

## 일반적인 원인

- **자금 부족** — 계좌에 충분한 돈이 없음
- **카드 한도 초과** — 트랜잭션이 카드 한도를 초과함
- **사기 방지** — 은행에서 의심으로 표시됨
- **부정확한 카드 세부 정보** — 카드 번호, CVV 또는 만료일 오류
- **카드 제한** — 해당 유형의 구매에 카드 사용 불가

## 해결 방법

### 1. 카드 정보 확인

입력된 정보가 정확한지 확인:
- 카드 번호 (모든 숫자)
- 만료 날짜
- CVV/보안 코드
- 청구 주소 (필요한 경우)

### 2. 다시 시도

일시적인 문제가 발생할 수 있습니다:
- 잠시 대기
- 카드 세부 정보 다시 입력
- 트랜잭션 다시 시도

### 3. 다른 결제 방법 사용

카드가 계속 거부되는 경우:
- 다른 카드 시도
- 현금 결제
- 대체 결제 방법 사용

### 4. 은행에 문의

카드 소지자는 다음을 해야 할 수 있습니다:
- 은행에 트랜잭션 확인
- 사기 차단 해제
- 카드 활성 상태 확인
- 지출 한도 확인

### 5. 제한 사항 확인

일부 카드에는 제한이 있습니다:
- 국제 거래 비활성화
- 온라인/POS 거래 차단
- 특정 상인 카테고리 차단

## 스토어 소유자를 위한 안내

빈번한 거절이 발생하는 경우:
- 가맹점 계정이 정상 상태인지 확인
- 결제 게이트웨이 구성 확인
- 사기 방지 설정 검토
- 결제 처리자에게 문의 고려
