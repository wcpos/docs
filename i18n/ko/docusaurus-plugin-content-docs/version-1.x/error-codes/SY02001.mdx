# SY02001: 잘못된 구성

## 의미

시스템 구성이 잘못되었거나 손상되었습니다. POS 애플리케이션은 의미가 없거나 불완전한 구성 설정을 발견했습니다.

## 일반적인 원인

- **손상된 설정** — 구성 파일이 손상됨
- **불완전한 설정** — 설정 프로세스가 완료되지 않음
- **수동 편집 오류** — 구성 파일이 잘못 수정됨
- **버전 업그레이드 문제** — 오래된 구성 파일이 새로운 버전과 호환되지 않음

## 수정 방법

### 1. 기본 설정으로 재설정

애플리케이션에서 사용할 수 있는 경우:
- "설정 재설정" 또는 "기본값 복원"을 찾으십시오
- 이는 사용자 지정을 지우지만 손상을 수정합니다

### 2. 설정 완료

설정이 완료되지 않은 경우:
1. 설정 프로세스를 다시 시작하십시오
2. 모든 필수 단계를 완료하십시오
3. 완료 전에 구성을 확인하십시오

### 3. 애플리케이션 데이터 지우기

손상된 구성을 제거하십시오:
- **데스크탑:** 애플리케이션 데이터 폴더를 삭제하고 앱을 재시작합니다
- **브라우저:** 사이트 데이터를 지우고 새로 고침합니다

### 4. 애플리케이션 재설치

문제가 지속되는 경우:
1. 완전히 제거하십시오
2. 남은 데이터 폴더를 모두 제거하십시오
3. 새로 설치하십시오
4. 처음부터 설정을 완료하십시오

### 5. 업데이트 확인

구성 형식이 변경될 수 있습니다:
- POS 애플리케이션을 업데이트하십시오
- WCPOS 플러그인을 업데이트하십시오
- 업데이트 후 재구성하십시오

## 재설정되는 항목

구성을 재설정할 때:
- 다시 로그인해야 합니다
- 사용자 정의 설정은 손실됩니다
- 데이터는 다시 동기화해야 합니다
- 그러나 WooCommerce 데이터는 서버에서 안전합니다
