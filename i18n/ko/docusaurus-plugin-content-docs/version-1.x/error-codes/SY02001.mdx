# SY02001: 잘못된 구성

## 의미

시스템 구성이 잘못되었거나 손상되었습니다. POS 애플리케이션에서 이해할 수 없거나 불완전한 구성 설정을 발견했습니다.

## 일반적인 원인

- **손상된 설정** — 구성 파일이 손상됨
- **불완전한 설정** — 설정 프로세스가 완료되지 않음
- **수동 편집 오류** — 구성 파일이 잘못 수정됨
- **버전 업그레이드 문제** — 이전 구성과 새 버전이 호환되지 않음

## 해결 방법

### 1. 기본 설정으로 초기화

앱에서 사용 가능한 경우:
- "설정 초기화" 또는 "기본값 복원" 찾기
- 이는 사용자 지정 설정을 지우지만 손상을 수정합니다

### 2. 설정 완료

설정이 완료되지 않은 경우:
1. 설정 프로세스 다시 시작
2. 모든 필수 단계 완료
3. 완료 전 구성 확인

### 3. 애플리케이션 데이터 지우기

손상된 구성 제거:
- **데스크톱:** 앱 데이터 폴더 삭제, 앱 다시 시작
- **브라우저:** 사이트 데이터 지우기, 다시 로드

### 4. 애플리케이션 재설치

지속적인 문제의 경우:
1. 완전히 제거
2. 남아있는 데이터 폴더 삭제
3. 새로 설치
4. 처음부터 설정 완료

### 5. 업데이트 확인

구성 형식이 변경될 수 있음:
- POS 애플리케이션 업데이트
- WCPOS 플러그인 업데이트
- 업데이트 후 재구성

## 초기화 시 영향

구성을 초기화하면:
- 다시 로그인해야 함
- 사용자 지정 설정이 손실됨
- 데이터를 다시 동기화해야 함
- 하지만 WooCommerce 데이터는 서버에 안전하게 보관됨
