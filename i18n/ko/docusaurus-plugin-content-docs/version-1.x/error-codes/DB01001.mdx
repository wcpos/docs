# DB01001: 연결 실패

## 의미

POS가 로컬 데이터베이스에 연결할 수 없었습니다. WCPOS는 제품, 고객 및 기타 데이터를 빠르게 액세스하고 오프라인 기능을 제공하기 위해 로컬 데이터베이스를 사용합니다.

## 일반적인 원인

- **저장 공간 부족** — 기기에 저장 공간이 남아있지 않음
- **데이터베이스 손상** — 로컬 데이터베이스 파일이 손상됨
- **권한 문제** — 앱이 저장소 권한을 가지고 있지 않음
- **브라우저 저장소 비활성화** — IndexedDB가 비활성화됨 (웹 버전)

## 해결 방법

### 1. 저장 공간 확인

기기의 사용 가능한 저장 공간을 확인하세요:
- 저장 공간이 꽉 찼다면 공간을 확보
- 사용하지 않는 앱이나 파일 삭제
- 디스크 사용 가능 공간 확인

### 2. 앱 데이터 초기화

로컬 데이터베이스를 재설정:
- **데스크톱 앱:** 설정에서 "데이터 지우기" 찾기
- **웹 브라우저:** POS 도메인의 사이트 데이터 지우기
- **참고:** 모든 데이터를 다시 동기화해야 함

### 3. 브라우저 설정 확인 (웹)

웹 버전 사용 시:
1. IndexedDB가 활성화되었는지 확인
2. 사생활 보호/시크릿 모드 사용 안 함 (제한된 저장소)
3. 사이트에서 데이터 저장 허용

### 4. 권한 확인

데스크톱에서:
- 앱이 데이터 디렉터리에 쓰기 권한이 있는지 확인
- 백신 프로그램이 데이터베이스 접근을 차단하는지 확인

### 5. 앱 재설치

다른 해결 방법이 실패한 경우:
1. POS 앱 제거
2. 기기 재시작
3. 앱 다시 설치
4. 로그인 및 데이터 동기화

## 데이터 복구

저장되지 않은 데이터가 있는 경우:
- 앱에 동기화되지 못한 보류 중인 변경 사항이 있을 수 있음
- 데이터 복구가 필요하면 지원팀에 문의
- 대부분의 경우 서버에서 데이터를 다시 동기화할 수 있음
