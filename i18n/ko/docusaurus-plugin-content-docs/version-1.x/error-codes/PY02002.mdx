# PY02002: 결제 시간 초과

## 의미

결제 처리가 너무 오래 걸려 시간 초과되었습니다. 결제 게이트웨이가 예상 시간 내에 응답하지 않았습니다.

## 일반적인 원인

- **게이트웨이 과부하** — 결제 프로세서가 높은 트래픽을 경험하고 있습니다.
- **네트워크 지연** — 결제 서비스에 대한 느린 연결
- **처리 지연** — 복잡한 사기 검사 또는 인증
- **서버 문제** — 결제 게이트웨이에 성능 문제가 발생하고 있습니다.

## 해결 방법

### 1. 결제가 성공했는지 확인

**중요:** 다시 시도하기 전에 결제 상태를 확인하세요:
- 결제 게이트웨이 대시보드를 확인하세요.
- WooCommerce에서 거래를 찾아보세요.
- 실제로 성공했다면 다시 처리하지 마세요.

### 2. 주의 깊게 재시도

결제가 분명히 진행되지 않았을 경우:
- 30-60초 기다리세요.
- 다시 시도하세요.
- 중복 청구를 주의하세요.

### 3. 연결 확인

안정적인 연결을 보장하세요:
- 인터넷 연결을 테스트하세요.
- WiFi를 사용하는 경우 유선 연결을 시도하세요.
- 네트워크 문제를 확인하세요.

### 4. 덜 바쁜 시간에 시도

게이트웨이가 과부하 상태일 경우:
- 소매 피크 시간대에 느려질 수 있습니다.
- 잠시 후 다시 시도하세요.
- 대체 결제 방법을 사용하세요.

### 5. 결제 제공자에 문의

시간 초과가 지속될 경우:
- 상태 페이지를 확인하세요.
- 지원 팀에 문의하세요.
- 지속적인 문제를 보고하세요.

## 중복 청구 방지

시간 초과가 발생할 경우:
1. **즉시 재시도하지 마세요** — 먼저 상태를 확인하세요.
2. **거래 검토** — 게이트웨이와 WooCommerce 모두 확인하세요.
3. **중복 환불** — 우발적인 중복 청구에 대한 환불을 처리하세요.
4. **고객에게 알리기** — 지연이 발생할 경우 상황을 설명하세요.

## 대체 조치

결제가 계속해서 시간 초과되는 경우:
- 현금 결제를 수락하세요.
- 허용되는 경우 "후 지급"으로 기록하세요.
- 다른 결제 방법을 사용하세요.
