# PY02002: 결제 시간 초과

## 이 메시지의 의미

결제 처리 과정에서 너무 오래 걸려 시간 초과가 발생했습니다. 결제 게이트웨이가 예상된 시간 내에 응답하지 않았습니다.

## 일반적인 원인

- **게이트웨이 과부하** — 결제 처리기가 높은 트래픽을 경험 중
- **네트워크 지연** — 결제 서비스로의 느린 연결
- **처리 지연** — 복잡한 사기 검사 또는 인증
- **서버 문제** — 결제 게이트웨이의 성능 문제

## 해결 방법

### 1. 결제 성공 여부 확인

**중요:** 재시도 전에 결제 상태를 확인하세요:
- 결제 게이트웨이 대시보드 확인
- WooCommerce에서 해당 거래 조회
- 실제로 성공한 경우 다시 처리하지 마세요

### 2. 주의하며 재시도

결제가 확실히 처리되지 않았다면:
- 30-60초 대기
- 다시 시도
- 중복 청구에 주의

### 3. 연결 상태 확인

안정적인 연결 확인:
- 인터넷 연결 테스트
- WiFi 사용 시 유선 연결 시도
- 네트워크 문제 확인

### 4. 덜 바쁜 시간대에 시도

게이트웨이가 과부하 상태라면:
- 소매점 성수기에는 지연 발생 가능
- 잠시 후 다시 시도
- 대체 결제 방법 사용

### 5. 결제 제공업체 연락

시간 초과가 지속되면:
- 상태 페이지 확인
- 고객 지원팀에 연락
- 지속적인 문제 보고

## 중복 청구 방지

시간 초과 발생 시:
1. **즉시 재시도 금지** — 상태를 먼저 확인
2. **거래 검토** — 게이트웨이와 WooCommerce 확인
3. **중복 환불** — 우발적 중복 청구 환불
4. **고객에게 알림** — 지연 시 상황 설명

## 대체 조치

결제가 계속 시간 초과되는 경우:
- 현금 결제 수락
- 허용된 경우 "나중에 지불" 로 기록
- 다른 결제 방법 사용
