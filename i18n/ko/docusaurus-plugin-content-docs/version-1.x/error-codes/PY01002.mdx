# PY01002: 자금 부족

## 의미

거래를 처리하기에 계좌의 자금이 충분하지 않아 결제가 거부되었습니다.

## 일반적인 원인

- **낮은 계좌 잔액** — 사용 가능한 자금 부족
- **보류 중인 거래** — 다른 보류 중인 거래로 인한 가용 잔액 감소
- **신용 한도 도달** — 신용카드의 경우 한도 초과
- **자금 보류** — 다른 거래를 위해 자금이 보류 상태

## 해결 방법

### 1. 다른 결제 방법 사용

가장 빠른 해결책:
- 다른 카드 시도
- 현금 사용
- 여러 결제 방법으로 분할 결제 (지원되는 경우)

### 2. 주문 금액 감소

부분 결제가 가능한 경우:
- 일부 항목 제거
- 할인 또는 쿠폰 적용
- 더 작은 거래 완료

### 3. 고객 조치

고객이 다음을 해야 할 수 있습니다:
- 계좌로 자금 이체
- 보류 중인 거래 해결 대기
- 신용카드 잔액 지불
- 신용 한도 증가 요청

### 4. 거래 분할

POS에서 분할 결제를 지원하는 경우:
- 한 카드로 일부 지불
- 다른 방법으로 나머지 지불

## 개인정보 보호 참고사항

고객에게 안내할 때:
- 구체적인 거절 이유를 자제하여 알림
- 단순히 "카드가 거절되었습니다"라고 언급
- 다른 결제 방법을 개인적으로 시도할 수 있도록 허용
