# PY01002: 자금 부족

## 의미

결제가 거부된 것은 거래를 처리할 계좌에 충분한 금액이 없기 때문입니다.

## 일반적인 원인

- **저금 잔고** — 사용 가능한 자금이 부족함
- **보류 중인 거래** — 사용 가능한 잔고를 줄이는 다른 보류 중인 거래
- **신용 한도 초과** — 신용 카드의 경우, 한도가 초과됨
- **자금 보류** — 다른 거래로 인해 자금이 보류 중임

## 해결 방법

### 1. 다른 결제 방법 사용

가장 빠른 해결책:
- 다른 카드를 시도해 보세요
- 현금을 사용하세요
- 여러 방법으로 결제를 나누세요 (지원되는 경우)

### 2. 주문 금액 줄이기

부분 결제가 가능한 경우:
- 일부 품목을 제거하세요
- 할인이나 쿠폰을 적용하세요
- 작은 거래를 완료하세요

### 3. 고객 행동

고객이 필요할 수 있는 사항:
- 계좌로 자금을 이체하세요
- 보류 중인 거래가 클리어될 때까지 기다리세요
- 신용 카드 잔액을 지불하세요
- 신용 한도 증가를 요청하세요

### 4. 거래 나누기

POS가 분할 결제를 지원하는 경우:
- 하나의 카드로 일부를 지불하세요
- 나머지는 다른 방법으로 지불하세요

## 개인 정보 보호 주의 사항

고객에게 이를 전달할 때:
- 특정 거부 사유에 대해 신중을 기하세요
- 단순히 "카드가 거부되었습니다"라고 말씀하세요
- 다른 결제 방법을 개인적으로 시도할 수 있도록 허용하세요
