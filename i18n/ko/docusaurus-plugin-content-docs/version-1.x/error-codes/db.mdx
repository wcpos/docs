# 데이터베이스 오류

데이터베이스 오류는 로컬 데이터베이스에 데이터를 저장하거나 검색할 때 발생합니다. WCPOS는 빠른 접근과 오프라인 기능을 위해 로컬 데이터베이스를 사용하여 제품, 고객 및 기타 데이터를 저장합니다. 이러한 오류는 `DB` 접두사로 시작됩니다.

## 카테고리

| 카테고리 | 코드 범위 | 설명 |
|----------|------------|-------------|
| [연결](#연결-오류) | DB01xxx | 데이터베이스 연결 및 트랜잭션 문제 |
| [데이터](#데이터-오류) | DB02xxx | 레코드 및 제약 조건 문제 |
| [쿼리](#쿼리-오류) | DB03xxx | 쿼리 구문 및 데이터 유형 문제 |

---

## 연결 오류

로컬 데이터베이스 연결 및 작동 문제.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB01001](./DB01001) | 연결 실패 | 로컬 데이터베이스에 연결할 수 없음 |
| [DB01002](./DB01002) | 쿼리 시간 초과 | 데이터베이스 쿼리가 너무 오래 걸림 |
| [DB01003](./DB01003) | 트랜잭션 실패 | 데이터베이스 트랜잭션을 완료할 수 없음 |

## 데이터 오류

레코드, 중복, 제약 조건 문제.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB02001](./DB02001) | 중복 레코드 | 이 ID를 가진 레코드가 이미 존재함 |
| [DB02002](./DB02002) | 레코드를 찾을 수 없음 | 요청된 레코드가 존재하지 않음 |
| [DB02003](./DB02003) | 제약 조건 위반 | 데이터가 데이터베이스 제약 조건을 위반함 |

## 쿼리 오류

쿼리 구문 및 데이터 유형 문제.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB03001](./DB03001) | 쿼리 구문 오류 | 데이터베이스 쿼리에 구문 오류가 있음 |
| [DB03002](./DB03002) | 잘못된 데이터 유형 | 데이터 유형이 예상된 유형과 일치하지 않음 |
| [DB03003](./DB03003) | 필수 필드 누락 | 필수 필드가 비어 있음 |
