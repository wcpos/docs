# 데이터베이스 오류 {#database-errors}

데이터베이스 오류는 로컬 데이터베이스에 데이터를 저장하거나 검색할 때 발생합니다. WCPOS는 제품, 고객 및 기타 데이터를 빠르게 접근하고 오프라인 기능을 위해 로컬 데이터베이스에 저장합니다. 이러한 오류는 `DB`로 시작합니다.

## 범주 {#categories}

| 범주 | 코드 범위 | 설명 |
|------|------------|-------------|
| [연결](#connection-errors) | DB01xxx | 데이터베이스 연결 및 트랜잭션 문제 |
| [데이터](#data-errors) | DB02xxx | 레코드 및 제약 조건 문제 |
| [쿼리](#query-errors) | DB03xxx | 쿼리 구문 및 데이터 형식 문제 |

---

## 연결 오류 {#connection-errors}

로컬 데이터베이스에 연결하거나 작업하는 데 문제가 발생했습니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB01001](./DB01001) | 연결 실패 | 로컬 데이터베이스에 연결할 수 없습니다 |
| [DB01002](./DB01002) | 쿼리 타임아웃 | 데이터베이스 쿼리가 너무 오래 걸렸습니다 |
| [DB01003](./DB01003) | 트랜잭션 실패 | 데이터베이스 트랜잭션을 완료할 수 없습니다 |

## 데이터 오류 {#data-errors}

레코드, 중복 및 제약 조건에 문제가 발생했습니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB02001](./DB02001) | 중복 레코드 | 이 ID를 가진 레코드가 이미 존재합니다 |
| [DB02002](./DB02002) | 레코드 없음 | 요청한 레코드가 존재하지 않습니다 |
| [DB02003](./DB02003) | 제약 조건 위반 | 데이터가 데이터베이스 제약 조건을 위반합니다 |

## 쿼리 오류 {#query-errors}

쿼리 구문 및 데이터 형식에 문제가 발생했습니다.

| 코드 | 이름 | 설명 |
|------|------|-------------|
| [DB03001](./DB03001) | 쿼리 구문 오류 | 데이터베이스 쿼리에 구문 오류가 있습니다 |
| [DB03002](./DB03002) | 잘못된 데이터 형식 | 데이터 형식이 예상 형식과 일치하지 않습니다 |
| [DB03003](./DB03003) | 필수 필드 누락 | 필수 필드가 비어 있습니다 |
