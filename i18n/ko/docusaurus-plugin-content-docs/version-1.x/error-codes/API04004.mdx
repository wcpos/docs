# API04004: 응답 데이터 누락

## 의미

서버가 성공적으로 응답했지만, 예상된 데이터가 누락되었습니다. JSON은 유효하지만 POS에 필요한 정보가 포함되어 있지 않습니다.

## 일반적인 원인

- **빈 결과** — 쿼리와 일치하는 데이터 없음
- **권한 제한** — 권한으로 인한 데이터 필터링
- **플러그인 필터링** — 다른 플러그인의 API 응답 필터링
- **버전 불일치** — API 버전 차이
- **데이터베이스 문제** — WooCommerce에 데이터 부재

## 해결 방법

### 1. 데이터 존재 확인

WordPress 관리자에서 확인:
- WooCommerce에 제품이 있나요?
- 로드할 고객이 있나요?
- 특정 항목이 존재하나요?

### 2. 사용자 권한 확인

사용자가 모든 데이터에 접근할 수 없을 수 있습니다:
- 사용자 역할 권한 확인
- POS 접근 설정 확인
- 관리자 계정으로 시도

### 3. API 응답 필터 확인

일부 플러그인은 REST API 응답을 필터링합니다:
- 필터링 플러그인 일시적으로 비활성화
- 테마의 사용자 정의 API 필터 확인
- 보안 플러그인 설정 검토

### 4. 구성 요소 업데이트

버전 불일치가 문제를 일으킬 수 있습니다:
- WCPOS 플러그인 업데이트
- POS 애플리케이션 업데이트
- 호환성 정보 확인

### 5. WooCommerce 데이터 확인

WooCommerce에서:
- 제품이 게시되었는지 확인 (초안 아님)
- 항목이 표시되도록 표시되었는지 확인
- 데이터가 손상되지 않았는지 확인

## 빈 데이터 vs. 누락 데이터

- **빈 응답** — 결과 없는 유효한 응답 (예상될 수 있음)
- **필드 누락** — 필요한 데이터 필드가 없는 응답 (이 오류)
