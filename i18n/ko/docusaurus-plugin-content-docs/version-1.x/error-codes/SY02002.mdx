# SY02002: 서비스 사용 불가

## 의미

필요한 서비스 또는 구성 요소를 사용할 수 없습니다. POS는 작동하기 위해 다양한 서비스에 의존하며, 그 중 하나가 현재 접근 불가능합니다.

## 일반적인 원인

- **백그라운드 서비스 중지** — 필요한 서비스가 실행되고 있지 않음
- **시스템 리소스 문제** — 서비스를 시작하기 위한 리소스 부족
- **종속성 실패** — 이 서비스가 의존하는 서비스 실패
- **시작 문제** — 서비스가 제대로 시작되지 않음
- **서버 오류** — WooCommerce 서버가 5xx 오류를 반환함

## 서버 오류 매핑

서버가 다음과 같은 응답을 반환할 때 이 오류 코드가 발생합니다:

| 서버 코드 | 출처 |
|-------------|--------|
| HTTP 5xx | 모든 서버 오류 응답 (500, 502, 503, 504 등) |

## 수정 방법

### 1. 애플리케이션 재시작

POS를 닫고 다시 엽니다:
1. 애플리케이션을 완전히 닫습니다
2. 몇 초 기다립니다
3. 다시 시작합니다

### 2. 장치 재시작

전체 재시작이 서비스 문제를 종종 해결합니다:
1. 중요한 작업을 저장합니다
2. 컴퓨터를 재시작합니다
3. POS를 다시 시도합니다

### 3. 시스템 리소스 확인

시스템 리소스가 사용 가능한지 확인합니다:
- 불필요한 애플리케이션을 닫습니다
- 메모리 사용량을 확인합니다
- 디스크 공간을 확인합니다

### 4. 애플리케이션 업데이트

최신 버전인지 확인합니다:
- 사용 가능한 업데이트를 확인합니다
- 보류 중인 업데이트를 설치합니다
- 업데이트 후 재시작합니다

### 5. 필요 시 재설치

서비스가 여전히 사용 불가한 경우:
1. 애플리케이션을 제거합니다
2. 장치를 재시작합니다
3. 최신 버전을 다운로드합니다
4. 새로 설치합니다

## 브라우저 전용 (웹 버전)

웹 버전의 경우:
- JavaScript가 활성화되어 있는지 확인합니다
- 필요한 브라우저 기능이 차단되지 않았는지 확인합니다
- 다른 브라우저를 사용해 봅니다
- 간섭할 수 있는 브라우저 확장 기능을 비활성화합니다

## 지원 팀에 연락해야 할 때

이 오류가 계속 발생하면:
- 오류 발생 시간을 기록합니다
- 특정 작업이 이를 유발하는지 확인합니다
- 조사에 대한 세부사항을 포함하여 보고합니다
