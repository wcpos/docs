# SY02002: 서비스 사용 불가

## 의미

필요한 서비스 또는 구성 요소를 사용할 수 없습니다. POS는 제대로 작동하기 위해 다양한 서비스에 의존하고 있으며, 그중 하나에 현재 접근할 수 없는 상태입니다.

## 일반적인 원인

- **백그라운드 서비스 중지** — 필요한 서비스가 실행되지 않음
- **시스템 리소스 문제** — 서비스를 시작하기에 리소스가 부족함
- **종속성 오류** — 의존하는 서비스에 오류 발생
- **시작 문제** — 서비스가 제대로 시작되지 않음
- **서버 오류** — WooCommerce 서버에서 5xx 오류 반환

## 서버 오류 매핑

다음과 같은 서버 응답으로 이 오류 코드가 트리거됩니다:

| 서버 코드 | 소스 |
|-------------|--------|
| HTTP 5xx | 모든 서버 오류 응답 (500, 502, 503, 504 등) |

## 해결 방법

### 1. 애플리케이션 재시작

POS를 완전히 닫고 다시 열기:
1. 애플리케이션을 완전히 종료
2. 몇 초 대기
3. 다시 시작

### 2. 기기 재시작

전체 재시작으로 서비스 문제 해결:
1. 중요한 작업 저장
2. 컴퓨터 재시작
3. POS 다시 시도

### 3. 시스템 리소스 확인

시스템 리소스 사용 가능 여부 확인:
- 불필요한 애플리케이션 종료
- 메모리 사용량 확인
- 디스크 공간 확인

### 4. 애플리케이션 업데이트

최신 버전인지 확인:
- 사용 가능한 업데이트 확인
- 보류 중인 업데이트 설치
- 업데이트 후 재시작

### 5. 필요한 경우 재설치

서비스를 계속 사용할 수 없는 경우:
1. 애플리케이션 제거
2. 기기 재시작
3. 최신 버전 다운로드
4. 새로 설치

## 브라우저별 (웹 버전)

웹 버전의 경우:
- JavaScript가 활성화되어 있는지 확인
- 필요한 브라우저 기능이 차단되지 않았는지 확인
- 다른 브라우저 시도
- 방해할 수 있는 브라우저 확장 프로그램 비활성화

## 지원팀에 문의해야 하는 경우

오류가 지속되는 경우:
- 발생 시점 기록
- 특정 작업으로 트리거되는지 확인
- 조사를 위해 자세한 내용과 함께 보고
