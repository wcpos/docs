이 설정은 POS 애플리케이션의 백엔드 기능을 구성하며, PHP 및 WooCommerce 설정을 포함합니다. 
다음 경로를 통해 접근할 수 있습니다: `WP Admin > POS > Settings`

주요 설정 항목은 다음과 같습니다:

- 모든 계산원에 대한 기본 카트 고객.
- 제품 바코드 필드.
- 주문 이메일 설정.
- 사용자 역할 권한.
