이러한 설정은 PHP 및 WooCommerce 구성을 포함한 POS 애플리케이션의 백엔드 기능을 구성합니다.
`WP 관리자 > POS > 설정`으로 이동하여 액세스할 수 있습니다.

주요 구성은 다음과 같습니다:

- 모든 계산원을 위한 기본 장바구니 고객.
- 제품 바코드 필드.
- 주문 이메일 설정.
- 사용자 역할 권한.
