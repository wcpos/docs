**세션** 화면에서는 현재 POS에 로그인한 사용자와 사용 중인 기기를 확인하고, 개별 기기를 원격으로 로그아웃할 수 있습니다. 기기를 분실했거나 도난당한 경우, 계산원이 퇴사한 경우, 또는 활성 로그인 상태를 검토하려는 경우에 사용하는 화면입니다.

**`WP Admin > POS > 설정 > Sessions`**에서 찾을 수 있습니다.

> _활성 사용자 세션을 관리합니다. 로그인한 모든 사용자를 보고, 사용 중인 기기를 확인하며, 필요한 경우 세션을 종료할 수 있습니다._

## 작동 방식 {#how-it-works}

이 화면은 두 개의 영역으로 구성됩니다.

- **사용자**(왼쪽) — 활성 세션이 하나 이상 있는 모든 사용자입니다. 최근 몇 분 안에 활동한 사용자에게는 녹색 점이 표시되고, 본인 계정에는 **나** 칩이 표시되며, 각 사용자가 보유한 세션 수가 함께 표시됩니다. 필터 상자에서 이름 또는 사용자명으로 검색할 수 있습니다.
- **세션**(오른쪽) — 사용자를 선택하면 활성 세션별로 카드가 하나씩 표시됩니다.

각 세션 카드에는 다음 항목이 표시됩니다.

| 필드 | 의미 |
| --- | --- |
| **애플리케이션** | 사용자가 로그인한 위치 — **iOS 애플리케이션**, **Android 애플리케이션**, **데스크톱 애플리케이션** 또는 **웹 애플리케이션** — 및 브라우저/버전 |
| **현재 기기** | 지금 사용 중인 세션을 표시하는 칩 |
| **마지막 활성** | 세션이 얼마나 최근에 요청을 보냈는지 (예: _방금 전_, _5분 전_) |
| **IP** | 세션이 연결된 IP 주소 (또는 _N/A_) |
| **생성됨** | 세션이 처음 로그인한 시점 |
| **만료됨** | 세션 토큰이 만료되는 시점 |
| **사용자 에이전트** | 원시 브라우저/기기 user-agent 문자열을 보려면 펼치세요 |

세션이 최근 5분 이내에 요청을 보낸 경우 **지금 활성**으로 표시됩니다.

## 기기 로그아웃하기 {#signing-devices-out}

세 가지 작업을 사용할 수 있으며, 각 작업에는 확인 대화 상자가 표시됩니다.

- **종료** — 단일 세션에서 로그아웃합니다. 해당 세션의 카드에 있는 버튼을 사용하세요.
- **다른 기기에서 로그아웃** — 선택한 사용자를 현재 사용 중인 기기를 **제외한** 모든 기기에서 로그아웃합니다. 사용자에게 세션이 두 개 이상 있을 때 표시됩니다.
- **모두 로그아웃** — 현재 기기를 포함하여 선택한 사용자를 **모든** 기기에서 로그아웃합니다.

:::tip 분실 또는 도난 기기
POS가 설치된 휴대폰, 태블릿 또는 노트북을 분실한 경우 세션을 열고 영향을 받은 사용자를 선택한 다음 해당 기기의 세션에서 **종료**를 사용합니다(또는 안전을 위해 **모두 로그아웃**을 사용합니다). 해당 기기는 다음 요청 시 액세스 권한을 잃습니다.
:::

:::note
현재 사용 중인 본인의 세션을 종료하면 지금 사용 중인 기기에서 로그아웃됩니다. 이 작업이 수행되기 전에 확인 요청이 표시됩니다.
:::
