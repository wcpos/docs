**세션** 화면을 사용하면 현재 POS에 로그인한 사람, 어떤 장치에 로그인되어 있는지 확인하고 개별 장치를 원격으로 로그아웃할 수 있습니다. 장치를 분실하거나 도난당했을 때, 계산원이 떠날 때 또는 단순히 활성 로그인을 검토하고 싶을 때 갈 수 있는 곳입니다.

**`WP Admin > POS > Settings > Sessions`**에서 찾을 수 있습니다.

> _활성 사용자 세션을 관리합니다. 로그인한 모든 사용자와 해당 사용자의 장치를 확인하고 필요한 경우 세션을 종료할 수 있습니다._

## 작동 원리 {#how-it-works}

화면에는 두 개의 창이 있습니다.

- **사용자**(왼쪽) — 하나 이상의 활성 세션이 있는 모든 사용자. 지난 몇 분 동안 활성 상태인 사람은 녹색 점으로 표시되며 자신의 계정에 대한 **You** 칩 및 각 사용자가 보유한 세션 수를 나타냅니다. 필터 상자를 사용하면 이름이나 사용자 이름으로 검색할 수 있습니다.
- **세션**(오른쪽) — 활성 세션당 하나의 카드를 보려면 사용자를 선택합니다.

각 세션 카드에는 다음이 표시됩니다.

| 필드 | 의미 |
| --- | --- |
| **신청** | 사용자가 로그인한 위치 — **iOS 애플리케이션**, **Android 애플리케이션**, **데스크톱 애플리케이션** 또는 **웹 애플리케이션** — 및 브라우저/버전 |
| **현재 장치** | 현재 사용하고 있는 세션을 표시하는 칩 |
| **마지막 활성** | 세션이 얼마나 최근에 요청을 했는지(예: _지금_, _5분 전_) |
| **IP** | 세션이 연결된 IP 주소(또는 _N/A_) |
| **생성됨** | 세션이 처음 로그인했을 때 |
| **만료** | 세션 토큰이 만료되는 경우 |
| **사용자 에이전트** | 원시 브라우저/장치 사용자 에이전트 문자열을 보려면 확장하세요. |

세션이 지난 5분 이내에 요청을 하면 **현재 활성**으로 표시됩니다.

## 기기 로그아웃 {#signing-devices-out}

세 가지 작업을 사용할 수 있으며 각 작업은 확인 대화 상자 뒤에 표시됩니다.

- **종료** — 단일 세션에서 로그아웃합니다. 해당 세션 카드의 버튼을 사용하세요.
- **다른 장치 로그아웃** — 선택한 사용자가 현재 사용 중인 장치를 **제외** 모든 장치에서 로그아웃합니다. 사용자가 두 개 이상의 세션을 갖고 있는 경우 표시됩니다.
- **모두 로그아웃** — 현재 장치를 포함하여 **모든** 장치에서 선택한 사용자를 로그아웃합니다.

:::tip 분실 또는 도난당한 기기
POS가 있는 휴대폰, 태블릿 또는 노트북이 사라진 경우 세션을 열고 영향을 받은 사용자를 선택한 다음 해당 장치의 세션에서 **종료**를 사용하세요(또는 안전을 위해 **모두 로그아웃**). 기기는 다음 요청 시 액세스 권한을 잃습니다.
:::

:::note
현재 세션을 종료하면 현재 사용 중인 장치에서 로그아웃됩니다. 로그아웃하기 전에 확인하라는 메시지가 표시됩니다.
:::
