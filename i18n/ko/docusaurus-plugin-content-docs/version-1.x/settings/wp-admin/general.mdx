## POS 전용 제품 사용

이 설정은 제품의 가시성 옵션을 추가하여 제품이 온라인 상점, POS 또는 둘 다에 나타날지를 제어할 수 있게 합니다.

:::warning 성능 영향
온라인 상점이나 POS에서 제품을 숨길 필요가 있는 경우에만 이 설정을 활성화하십시오. 활성화 시, 각 제품 요청마다 추가적인 데이터베이스 조회가 추가됩니다.
:::

이 기능을 사용하는 방법에 대한 자세한 정보는 개별 제품의 가시성을 설정하고 일괄 편집하는 방법을 포함하여 [POS 전용 제품](/products/pos-only-products)을 참조하십시오.

## 소수량 사용

이 설정은 고객과 직원이 제품을 장바구니에 추가할 때 소수량을 입력할 수 있도록 허용합니다.

**소수량의 예:**
- 0.25 (1/4 단위)
- 1.5 (1.5 단위)
- 2.75 (2.75 단위)

**유용한 경우:**
- 중량으로 판매되는 제품 (예: 0.5kg의 커피 원두)
- 분수 서비스 (예: 1.5시간의 상담)
- 부분 수량으로 판매할 수 있는 대량 제품

## 고객 이메일에서 자동으로 사용자 이름 생성

활성화되면, WCPOS는 체크아웃 중에 고객의 이메일 주소를 기반으로 새로운 고객의 사용자 이름을 자동으로 생성합니다. 이는 고객 생성 프로세스를 간소화하고 일관된 사용자 이름 형식을 보장합니다.

## 기본 POS 고객

새로운 POS 주문에 지정될 기본 고객을 선택합니다. 일반적인 옵션은 다음과 같습니다:

- **손님**: 주문이 사용자 계정과 연관되지 않음
- **계산원**: 주문이 로그인한 계산원의 계정과 연관됨
- **내점 고객**: 주문이 특정 고객 계정과 연관됨 (이 계정을 먼저 생성해야 합니다)

## 계산원 계정 사용

활성화되면, 계산원의 사용자 계정이 선택한 기본 POS 고객 대신 POS 주문의 기본 고객으로 사용됩니다.

## 바코드 필드

이 설정은 바코드를 스캔하거나 바코드 검색 기능을 사용할 때 WCPOS가 검색할 제품 필드를 결정합니다.

:::warning 중요: 데이터 지우기 및 새로 고침 필요
바코드 필드 설정을 변경한 후에는 변경 사항이 반영되도록 POS에서 제품 데이터를 **지우고 새로 고쳐야** 합니다.

**지우고 새로 고치는 방법:**
1. POS의 제품 화면으로 이동합니다.
2. 화면 하단의 동기화 아이콘(↻)을 **길게 누릅니다.**
3. 컨텍스트 메뉴에서 **"지우고 새로 고침"**을 선택합니다.
4. 새로운 바코드 필드 데이터로 제품이 다시 로드될 때까지 기다립니다.

이는 POS가 로컬로 제품 데이터를 캐시하기 때문에 바코드 필드 설정을 변경해도 캐시된 데이터가 자동으로 업데이트되지 않기 때문에 필요합니다.
:::

### 기본 옵션

**`_global_unique_id`** - WooCommerce의 Global Unique ID 필드로, WooCommerce 8.2(2024년 7월)에 도입되었습니다. GTIN, UPC, EAN 또는 ISBN 코드를 저장하도록 설계되었으며, 새로운 설치에 권장되는 옵션입니다.

### 기타 일반 옵션

**`_sku`** - 제품 SKU(재고 관리 단위) 필드입니다. Global Unique ID 필드를 사용하지 않는 경우 좋은 대체 옵션입니다.

### 타사 플러그인 메타 키

바코드 관리 플러그인을 사용하는 경우 특정 메타 키를 사용해야 할 수 있습니다:

| 플러그인 유형 | 일반 메타 키 | 설명 |
|-------------|------------------|-------------|
| EAN 플러그인 | `_ean`, `_ean13`, `_ean8` | 유럽 기사 번호 코드 |
| UPC 플러그인 | `_upc`, `_upc_code` | 보편 상품 코드 |
| GTIN 플러그인 | `_gtin`, `_gtin13`, `_gtin14` | 글로벌 무역 품목 번호 |
| 일반 바코드 | `_barcode`, `_product_barcode` | 일반 바코드 필드 |
| WooCommerce 제품 바코드 | `_barcode_text` | 인기 바코드 플러그인 |
| ATUM 재고 | `_atum_barcode` | ATUM 재고 관리 |

### 플러그인의 메타 키 찾기

바코드 플러그인의 올바른 메타 키를 찾으려면:

1. WP Admin에서 제품을 편집합니다.
2. 플러그인을 사용하여 바코드를 추가합니다.
3. 메타 키 이름에 대한 플러그인 문서를 확인합니다.
4. 또는 데이터베이스 `wp_postmeta` 테이블을 검사하여 생성된 메타 키를 확인합니다.

### 사용자 지정 메타 키

바코드 데이터를 저장하는 사용자 지정 메타 키를 사용할 수도 있습니다. 이 필드에 메타 키 이름(해당되는 경우 언더스코어 접두사 포함)을 입력하면 됩니다.

:::tip
이 설정을 변경한 후 바코드 스캔이 선택한 필드와 올바르게 작동하는지 테스트하십시오. 변경 사항이 반영되도록 POS에서 제품 데이터를 지우고 새로 고치십시오.
:::
