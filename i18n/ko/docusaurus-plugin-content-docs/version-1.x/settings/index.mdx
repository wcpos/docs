WCPOS에는 구성 위치에 따라 세 가지 유형의 설정이 있습니다.

## 설정 위치

### 1. WordPress 관리자 설정

WordPress 대시보드에서 구성되는 백엔드 설정:

**위치:** `WP 관리자 > POS > 설정`

이 설정은 다음을 제어합니다:
- 모든 계산원의 기본 장바구니 고객
- 상품 바코드 필드 구성
- 주문 이메일 설정
- 사용자 역할 권한 (POS 접근 권한)
- 결제 게이트웨이 가용성

자세한 내용은 [WordPress 관리자 설정](/settings/wp-admin)을 참조하세요.

### 2. POS 스토어 설정

POS 애플리케이션 내에서 구성되는 앱 전체 설정:

**위치:** 사용자 메뉴 (우측 상단) → 설정, 또는 `Ctrl + Shift + S`

이 설정은 다음을 제어합니다:
- 세금 계산 옵션
- 바코드 스캐너 구성
- 키보드 단축키
- 테마/외관

자세한 내용은 [스토어 설정](/settings/store)을 참조하세요.

### 3. 디스플레이 설정 (화면별)

POS의 각 화면에는 데이터 테이블을 사용자 지정하기 위한 고유한 디스플레이 설정이 있습니다:

**위치:** 모든 화면에서 슬라이더 아이콘(<Icon name="sliders" />)을 클릭

이 설정은 다음을 제어합니다:
- 표시되는 열
- 열 표시 옵션
- 화면별 기능

디스플레이 설정은 각 화면의 문서에 설명되어 있습니다:
- [POS 상품 패널](/pos/product-panel#display-settings)
- [장바구니 패널](/pos/cart#display-settings)
- [상품 화면](/products#display-settings)
- [주문 화면](/orders#display-settings)
- [고객 화면](/customers#display-settings)
- [보고서 화면](/reports#display-settings)
