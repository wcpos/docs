WCPOS에는 설정이 구성되는 위치에 따라 조직된 세 가지 유형의 설정이 있습니다.

## 설정 찾기

### 1. 워드프레스 관리자 설정

워드프레스 대시보드에서 구성된 백엔드 설정:

**위치:** `WP Admin > POS > Settings`

이 설정은 다음을 제어합니다:
- 모든 캐셔의 기본 장바구니 고객
- 제품 바코드 필드 구성
- 주문 이메일 설정
- 사용자 역할 권한(누가 POS에 접근할 수 있는지)
- 결제 게이트웨이 가용성

자세한 내용은 [워드프레스 관리자 설정](/settings/wp-admin)을 참조하세요.

### 2. POS 상점 설정

POS 애플리케이션 내에서 구성된 애플리케이션 전체 설정:

**위치:** 사용자 메뉴(우상단) → 설정, 또는 `Ctrl + Shift + S`

이 설정은 다음을 제어합니다:
- 세금 계산 옵션
- 바코드 스캐너 구성
- 키보드 단축키
- 테마/모양

자세한 내용은 [상점 설정](/settings/store)을 참조하세요.

### 3. 표시 설정 (화면별)

POS의 각 화면에는 데이터 테이블을 사용자 정의하기 위한 고유한 표시 설정이 있습니다:

**위치:** 모든 화면에서 슬라이더 아이콘(<Icon name="sliders" />)을 클릭

이 설정은 다음을 제어합니다:
- 어떤 열이 표시되는지
- 열 표시 옵션
- 화면별 기능

표시 설정은 각 화면의 문서 내에 문서화되어 있습니다:
- [POS 제품 패널](/pos/product-panel#display-settings)
- [장바구니 패널](/pos/cart#display-settings)
- [제품 화면](/products#display-settings)
- [주문 화면](/orders#display-settings)
- [고객 화면](/customers#display-settings)
- [보고서 화면](/reports#display-settings)
