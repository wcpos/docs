WCPOS는 설정이 구성되는 장소에 따라 세 가지 유형으로 조직됩니다.

## 설정 찾는 방법

### 1. WordPress 관리자 설정

WordPress 대시보드에서 구성된 백엔드 설정:

**위치:** `WP Admin > POS > Settings`

이 설정은 다음을 제어합니다:
- 모든 캐셔에 대한 기본 카트 고객
- 제품 바코드 필드 구성
- 주문 이메일 설정
- 사용자 역할 권한 (POS에 접근할 수 있는 사용자)
- 결제 게이트웨이 가용성

자세한 내용은 [WordPress 관리자 설정](/settings/wp-admin)을 참조하십시오.

### 2. POS 스토어 설정

POS 애플리케이션 내에서 구성된 앱 전체 설정:

**위치:** 사용자 메뉴 (우측 상단) → 설정, 또는 `Ctrl + Shift + S`

이 설정은 다음을 제어합니다:
- 세금 계산 옵션
- 바코드 스캐너 구성
- 키보드 단축키
- 테마/모양

자세한 내용은 [스토어 설정](/settings/store)을 참조하십시오.

### 3. 디스플레이 설정 (화면별)

POS의 각 화면에는 데이터 테이블을 사용자 정의하기 위한 자체 디스플레이 설정이 있습니다:

**위치:** 모든 화면에서 슬라이더 아이콘(<Icon name="sliders" />)을 클릭합니다

이 설정은 다음을 제어합니다:
- 보이는 열
- 열 표시 옵션
- 화면별 기능

디스플레이 설정은 각 화면의 문서에 기록되어 있습니다:
- [POS 제품 패널](/pos/product-panel#display-settings)
- [카트 패널](/pos/cart#display-settings)
- [제품 화면](/products#display-settings)
- [주문 화면](/orders#display-settings)
- [고객 화면](/customers#display-settings)
- [보고서 화면](/reports#display-settings)
