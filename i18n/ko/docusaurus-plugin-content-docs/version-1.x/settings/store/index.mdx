매장 설정은 POS의 작동 방식에 영향을 미치는 앱 전체 구성입니다. 이 설정은 개별 화면이 아닌 전체 매장에 적용됩니다.

## 매장 설정 접근 {#accessing-store-settings}

다음 방법으로 매장 설정을 엽니다:

- **사용자 메뉴**(우측 상단)를 클릭하고 **설정**을 선택
- 키보드 단축키 `Ctrl + Shift + S` 사용

## 설정 탭 {#settings-tabs}

매장 설정 모달에는 다음 탭이 포함되어 있습니다:

| 탭 | 설명 |
|-----|-------------|
| [일반](./general) | 매장 정보, 언어, 통화, 기본 고객 |
| [세금](./tax) | 세금 계산 및 표시 옵션 |
| [바코드](./barcode) | 바코드 스캐너 설정 |
| [키보드 단축키](./hotkeys) | 사용 가능한 키보드 단축키 |
| [테마](./theme) | 외관 및 색상 구성 |
| [프린터](/hardware/printers) | 영수증 프린터 연결 및 설정 |
