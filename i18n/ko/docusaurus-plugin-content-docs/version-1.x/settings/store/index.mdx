상점 설정은 POS 작동 방식에 영향을 미치는 앱 전체 구성입니다. 이러한 설정은 개별 화면이 아닌 전체 상점에 적용됩니다.

## 상점 설정에 접근하기

상점 설정을 열려면:

- **사용자 메뉴**(오른쪽 상단 모서리)를 클릭하고 **설정**을 선택합니다.
- 키보드 단축키 `Ctrl + Shift + S`를 사용합니다.

## 설정 탭

상점 설정 모달에는 다음과 같은 탭이 있습니다:

| 탭 | 설명 |
|-----|-------------|
| [일반](./general) | 상점 정보, 언어, 통화, 기본 고객 |
| [세금](./tax) | 세금 계산 및 표시 옵션 |
| [바코드](./barcode) | 바코드 스캐너 구성 |
| [키보드 단축키](./hotkeys) | 사용 가능한 키보드 단축키 |
| [테마](./theme) | 모양 및 색 구성 |
