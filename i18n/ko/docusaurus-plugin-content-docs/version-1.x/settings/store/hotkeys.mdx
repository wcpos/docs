## 사용 가능한 단축키

| 키 조합           | 동작    | 설명                             |
|------------------|---------|----------------------------------|
| `Ctrl + Shift + S` | 설정    | POS 설정 패널을 엽니다          |
| `Ctrl + Shift + A` | POS    | 주요 POS 화면으로 이동합니다     |
| `Ctrl + Shift + P` | 제품    | 제품 관리 화면을 엽니다         |
| `Ctrl + Shift + O` | 주문    | 주문 관리 화면을 엽니다         |
| `Ctrl + Shift + C` | 고객    | 고객 관리 화면을 엽니다         |
| `Ctrl + Shift + ?` | 지원    | 지원/도움 섹션을 엽니다         |
| `Ctrl + Shift + L` | 로그아웃 | 현재 세션에서 로그아웃합니다    |

## 키보드 단축키 사용하기

모든 키보드 단축키를 사용하려면:

1. **수정 키를 누르고 있**습니다 (`Ctrl + Shift`)
2. **동작 키를 누릅니다** (원하는 동작에 해당하는 글자)
3. **모든 키를 놓아** 명령을 실행합니다

## 효율적인 사용을 위한 팁

- **자주 사용하는 단축키를 암기하세요**: 가장 자주 수행하는 작업에 대한 단축키를 배우는 데 집중하세요
- **키 조합을 연습하세요**: 정기적인 사용은 근육 기억을 향상시킵니다
- **혼잡한 시간대에 단축키를 사용하세요**: 키보드 단축키는 피크 시간 동안 작업 속도를 크게 높일 수 있습니다

## 사용자 정의

:::info 기능 요청
현재 키보드 단축키 사용자 정의는 제공되지 않습니다. 위에 나열된 단축키는 고정되어 있으며 설정 인터페이스를 통해 수정할 수 없습니다.

키보드 단축키를 사용자 정의하고 싶으시면 [Discord](https://wcpos.com/discord)에서 피드백을 공유해 주세요.
:::
