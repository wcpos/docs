## 사용 가능한 단축키

| 키 조합 | 동작 | 설명 |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 설정 | POS 설정 패널 열기 |
| `Ctrl + Shift + A` | POS | 메인 POS 화면으로 이동 |
| `Ctrl + Shift + P` | 상품 | 상품 관리 화면 열기 |
| `Ctrl + Shift + O` | 주문 | 주문 관리 화면 열기 |
| `Ctrl + Shift + C` | 고객 | 고객 관리 화면 열기 |
| `Ctrl + Shift + ?` | 지원 | 지원/도움말 섹션 열기 |
| `Ctrl + Shift + L` | 로그아웃 | 현재 세션에서 로그아웃 |

## 키보드 단축키 사용하기

키보드 단축키를 사용하려면:

1. **보조 키 누르기** (`Ctrl + Shift`)
2. **작업 키 누르기** (원하는 작업에 해당하는 문자)
3. **모든 키 놓기**로 명령 실행

## 효율적인 사용을 위한 팁

- **자주 사용하는 단축키 암기**: 가장 자주 수행하는 작업의 단축키에 집중
- **키 조합 연습**: 정기적인 사용으로 근육 기억 형성
- **바쁜 시간에 단축키 사용**: 키보드 단축키로 피크 시간 작업 속도 크게 향상

## 사용자 정의
