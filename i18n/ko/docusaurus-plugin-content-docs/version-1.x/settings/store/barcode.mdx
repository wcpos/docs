## 바코드 감지 작동 방식

대부분의 바코드 스캐너는 장치에 연결된 키보드처럼 작동합니다. 바코드를 스캔하면 POS는 문자가 일반적인 타이핑보다 더 빠르게 입력되었음을 감지합니다. 이 "빠른 키 입력"을 사용하여 입력을 바코드 스캔으로 식별합니다.

## 설정

| 설정 | 설명 |
|---------|-------------|
| **평균 입력 시간** | 바코드로 간주되기 위해 입력이 얼마나 빨라야 하는지 (밀리초 단위). 낮은 값 = 더 빠른 입력 필요 |
| **최소 길이** | 바코드로 처리되기 위한 연속된 문자열의 최소 길이 |
| **접두사 제거** | 스캔된 바코드의 시작 부분에서 제거할 문자 |
| **접미사 제거** | 스캔된 바코드의 끝 부분에서 제거할 문자 |

## 접두사/접미사 제거

스캐너가 추가 문자(접두사 또는 접미사)를 추가하는 경우, 기본 바코드만 남도록 자동으로 제거할 수 있습니다.

**예시:** 스캐너가 모든 바코드 앞에 "A"를, 뒤에 "Z"를 추가하는 경우 다음과 같이 구성:
- 접두사 제거: `A`
- 접미사 제거: `Z`

## 문제 해결

### 바코드 감지 안 됨

- **스캐너가 너무 느림**: "평균 입력 시간" 값을 높여보세요
- **바코드가 너무 짧음**: "최소 길이" 값을 낮추어 보세요
- **추가 문자**: 스캐너가 접두사/접미사를 추가하는지 확인하세요

### 일반 타이핑이 바코드로 감지됨

- **타이핑이 너무 빠름**: "평균 입력 시간" 값을 낮추어 보세요
- **입력이 너무 김**: "최소 길이" 값을 높여 보세요
