## 사용 가능한 테마

### 시스템 테마

- **설명**: 장치 설정을 따릅니다
- **작동 방식**: 장치의 시스템 환경 설정에 따라 라이트 및 다크 모드 간에 자동으로 전환됩니다

### 라이트 테마

- **설명**: 깔끔하고 밝음
- **최고의 환경**: 밝은 환경과 전통적인 밝은 인터페이스를 선호하는 사용자에게 적합

### 다크 테마

- **설명**: 눈에 편안함
- **최고의 환경**: 저조도 환경과 장시간 사용 시 눈의 피로를 줄이는 데 적합

### 오션 테마

- **설명**: 시원한 파랑과 청록색
- **최고의 환경**: 편안하고 전문적인 분위기를 조성하는 데 적합

### 석양 테마

- **설명**: 따뜻한 오렌지와 보라색
- **최고의 환경**: 아늑하고 초대하는 분위기를 조성하는 데 적합

### 단색 테마

- **설명**: 고대비 그레이스케일
- **최고의 환경**: 접근성 요구와 높은 대비가 필요한 환경

## 테마 선택하기

1. 테마 설정 패널에서 카드로 표시된 모든 사용 가능한 테마를 확인합니다
2. 각 테마 카드는 테마 이름과 간단한 설명을 표시합니다
3. 원하는 테마 카드를 클릭하여 선택합니다
4. 현재 활성화된 테마는 파란색 테두리로 강조 표시됩니다
5. 선택 사항은 자동으로 저장되며 즉시 적용됩니다

## 테마 선택을 위한 팁

| 테마 | 최적의 환경 |
|-------|------------------|
| **시스템** | POS가 장치와 자동으로 일치하도록 할 때 |
| **라이트** | 조명이 좋은 밝은 소매 환경 |
| **다크** | 조명이 어두운 환경이나 저녁/야간 근무 |
| **오션/석양** | 개성과 브랜드 정렬 추가 |
| **단색** | 최대 대비가 필요할 때 |

선택한 테마는 모든 POS 세션에서 지속됩니다.

:::info 곧 공개
더 많은 테마와 사용자 지정 도구가 곧 출시됩니다!
