## 사용 가능한 테마

### 시스템 테마

- **설명**: 기기 설정 따라가기
- **동작**: 기기의 시스템 환경설정에 따라 자동으로 라이트와 다크 모드 간 전환

### 라이트 테마

- **설명**: 깨끗하고 밝은 느낌
- **추천 상황**: 밝은 환경과 전통적인 라이트 인터페이스를 선호하는 사용자

### 다크 테마

- **설명**: 눈에 편한
- **추천 상황**: 조명이 어두운 환경과 장시간 사용 시 눈의 피로 감소

### 오션 테마

- **설명**: 시원한 블루와 청록색
- **추천 상황**: 차분하고 전문적인 분위기 연출

### 선셋 테마

- **설명**: 따뜻한 주황색과 보라색
- **추천 상황**: 아늑하고 친근한 분위기 연출

### 모노크롬 테마

- **설명**: 고대비 그레이스케일
- **추천 상황**: 접근성 요구사항과 고대비가 필요한 환경

## 테마 선택하기

1. 테마 설정 패널에서 카드 형태로 표시된 모든 사용 가능한 테마를 확인하세요
2. 각 테마 카드는 테마 이름과 간단한 설명을 보여줍니다
3. 원하는 테마 카드를 클릭하여 선택하세요
4. 현재 활성화된 테마는 파란색 테두리로 강조 표시됩니다
5. 선택사항은 자동으로 저장되고 즉시 적용됩니다

## 테마 선택 팁

| 테마 | 최적의 환경 |
|-------|------------------|
| **시스템** | 기기와 자동으로 일치시키고 싶을 때 |
| **라이트** | 조명이 밝은 소매점 환경 |
| **다크** | 조명이 어두운 환경 또는 저녁/야간 근무 |
| **오션/선셋** | 개성과 브랜드 일치를 추가할 때 |
| **모노크롬** | 최대 대비가 필요할 때 |

선택한 테마는 모든 POS 세션에서 유지됩니다.
