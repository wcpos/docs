## 사용 가능한 테마

### 시스템 테마

- **설명**: 장치 설정을 따릅니다
- **작동 방식**: 장치의 시스템 기본 설정에 따라 자동으로 밝은 모드와 어두운 모드 간에 전환됩니다

### 라이트 테마

- **설명**: 깔끔하고 밝음
- **최적의 환경**: 조명이 충분한 환경과 전통적인 밝은 인터페이스를 선호하는 사용자

### 다크 테마

- **설명**: 눈에 편안함
- **최적의 환경**: 조명이 어두운 환경 및 장시간 사용 시 눈의 피로를 줄이는 데 적합

### 오션 테마

- **설명**: 시원한 파랑과 청록
- **최적의 환경**: 편안하고 전문적인 분위기를 만드는 데 적합

### 선셋 테마

- **설명**: 따뜻한 주황색과 보라색
- **최적의 환경**: 아늑하고 매력적인 분위기를 만드는 데 적합

### 모노크롬 테마

- **설명**: 높은 대비의 그레이 스케일
- **최적의 환경**: 접근성 필요 및 높은 대비가 필요한 환경

## 테마 선택하기

1. 테마 설정 패널에서 모든 사용 가능한 테마가 카드로 표시됩니다
2. 각 테마 카드는 테마 이름과 간단한 설명을 표시합니다
3. 선택할 테마 카드에 클릭합니다
4. 현재 활성 테마는 파란색 테두리로 강조 표시됩니다
5. 선택한 내용은 자동으로 저장되고 즉시 적용됩니다

## 테마 선택을 위한 팁

| 테마 | 최적의 환경 |
|-------|------------------|
| **시스템** | POS가 장치와 자동으로 일치하게 하려는 경우 |
| **라이트** | 조명이 좋은 밝은 소매 환경 |
| **다크** | 어두운 환경이나 저녁/야간 근무 |
| **오션/선셋** | 개성과 브랜드 정렬을 추가할 때 |
| **모노크롬** | 최대 대비가 필요할 때 |

선택한 테마는 모든 POS 세션에 지속됩니다.
