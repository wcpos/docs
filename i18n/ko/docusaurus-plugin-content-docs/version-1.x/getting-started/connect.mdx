:::

## 연결 화면 개요

WCPOS 데스크톱 또는 모바일 앱을 열면 연결 화면이 표시됩니다. 여기서 스토어 연결 및 사용자 로그인을 관리할 수 있습니다.

{/* 스크린샷: connect-screen.png */}

## 새 스토어 추가

1. 텍스트 필드에 WooCommerce 스토어 URL을 입력합니다 (예: `https://mystore.com`)
2. **연결**을 클릭합니다
3. WordPress 자격 증명으로 로그인하기 위해 리다이렉트됩니다
4. 로그인 성공 후 앱으로 돌아갑니다

## 여러 스토어

필요한 만큼 많은 WooCommerce 스토어에 연결할 수 있습니다. 각 스토어는 연결 화면에 다음을 보여주는 별도의 카드로 표시됩니다:

- **스토어 이름** 및 파비콘
- **스토어 URL**
- 해당 스토어의 **로그인된 사용자**

여러 지점이나 비즈니스를 관리할 때 유용합니다.

## 스토어당 여러 사용자

각 스토어에 여러 사용자가 동시에 로그인할 수 있습니다. 다음과 같은 경우에 유용합니다:

- **교대 근무** - 새 계산원이 이전 계산원이 로그아웃하기 전에 로그인
- **다중 계산대 설정** - 다른 장치의 다른 계산원
- **빠른 전환** - 사용자 계정 간 쉬운 전환

### 사용자 추가

"로그인된 사용자" 옆의 <Icon name="plus" /> 버튼을 클릭하여 해당 스토어에 다른 사용자를 추가합니다.

### 사용자 전환

사용자 배지(예: "브렌다")를 클릭하여 해당 사용자로 POS를 엽니다.

### 사용자 제거

사용자 배지의 **×**를 클릭하여 해당 사용자를 스토어에서 로그아웃합니다.

## 스토어 제거

스토어 카드의 빨간색 **×** 버튼을 클릭하여 목록에서 제거합니다. 이렇게 하면 모든 사용자가 로그아웃되고 앱에서 스토어 연결이 제거됩니다.

## 데모 스토어

화면 하단에 "데모 스토어 입장" 링크가 표시될 수 있습니다. 이를 통해 자신의 스토어 데이터에 영향을 주지 않고 WCPOS 기능을 체험해 볼 수 있습니다.

## 문제 해결

### 스토어에 연결할 수 없나요?

- WCPOS 플러그인이 WordPress 사이트에 설치 및 활성화되었는지 확인
- 올바른 URL을 사용하고 있는지 확인 (SSL을 사용하는 경우 `https://` 포함)
- WooCommerce REST API에 접근할 수 있는지 확인
- 사용자 계정에 POS 접근 권한이 있는지 확인

### 연결이 계속 실패하나요?

- 먼저 웹 브라우저에서 `yourdomain.com/pos`에 접속하여 플러그인이 작동하는지 확인
- 사이트의 오류 로그를 확인
- 서버가 [최소 요구 사항](/getting-started/installation)을 충족하는지 확인
