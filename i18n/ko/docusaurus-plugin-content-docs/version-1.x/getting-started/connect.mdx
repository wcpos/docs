:::

## 연결 화면 개요 {#connect-screen-overview}

WCPOS 데스크탑 또는 모바일 앱을 열면 연결 화면이 나타납니다. 이곳에서 스토어 연결 및 사용자 로그인을 관리할 수 있습니다.

{/* 스크린샷: connect-screen.png */}

## 새 스토어 추가하기 {#adding-a-new-store}

1. 텍스트 필드에 WooCommerce 스토어 URL을 입력하세요 (예: `https://mystore.com`)
2. **연결** 클릭
3. WordPress 자격 증명으로 로그인하라는 페이지로 리디렉션됩니다.
4. 로그인에 성공하면 앱으로 돌아옵니다.

## 여러 스토어 {#multiple-stores}

필요한 만큼 여러 WooCommerce 스토어에 연결할 수 있습니다. 각각의 스토어는 연결 화면에 별도의 카드로 표시되며, 다음을 보여줍니다:

- **스토어 이름**과 파비콘
- **스토어 URL**
- 해당 스토어의 **로그인한 사용자**

이는 여러 위치나 비즈니스를 관리하는 경우에 유용합니다.

## 스토어당 여러 사용자 {#multiple-users-per-store}

각 스토어는 동시에 여러 사용자가 로그인할 수 있습니다. 이는 다음과 같은 상황에 유용합니다:

- **교대 근무** - 새로운 캐셔가 이전 캐셔가 로그아웃하기 전에 로그인할 수 있습니다.
- **다중 등록 세팅** - 다른 장치에서 다른 캐셔가 사용합니다.
- **빠른 전환** - 사용자 계정 간에 쉽게 전환할 수 있습니다.

### 사용자 추가하기 {#adding-a-user}

"로그인한 사용자" 옆의 <Icon name="plus" /> 버튼을 클릭하여 해당 스토어에 다른 사용자를 추가하세요.

### 사용자 전환하기 {#switching-users}

사용자 배지를 클릭하세요 (예: "Brenda") 해당 사용자로 POS를 엽니다.

### 사용자 제거하기 {#removing-a-user}

사용자 배지에 있는 **×** 버튼을 클릭하면 해당 사용자가 스토어에서 로그아웃됩니다.

## 스토어 제거하기 {#removing-a-store}

스토어 카드의 빨간 **×** 버튼을 클릭하여 목록에서 제거합니다. 이 작업은 모든 사용자를 로그아웃시키고 앱에서 스토어 연결을 제거합니다.

## 데모 스토어 {#demo-store}

화면 하단에 "데모 스토어 들어가기" 링크가 표시될 수 있습니다. 이 링크를 통해 자신의 스토어 데이터에 영향을 주지 않고 WCPOS 기능을 체험할 수 있는 데모 WooCommerce 스토어에 연결됩니다.

## 문제 해결 {#troubleshooting}

<Accordion>
  <AccordionItem question="내 스토어에 연결할 수 없나요?">

  - WordPress 사이트에 WCPOS 플러그인이 설치되어 있고 활성화되어 있는지 확인하세요.
  - 올바른 URL을 사용하고 있는지 확인하세요 (사이트에서 SSL을 사용하는 경우 `https://` 포함).
  - WooCommerce REST API에 접근할 수 있는지 확인하세요.
  - 귀하의 사용자 계정에 POS 접근 권한이 있는지 확인하세요.

  </AccordionItem>

  <AccordionItem question="연결이 계속 실패하나요?">

  - 먼저 웹 브라우저에서 `yourdomain.com/pos`에 액세스하여 플러그인이 제대로 작동하는지 확인하세요.
  - 사이트의 오류 로그를 확인하여 문제가 있는지 확인하세요.
  - 서버가 [최소 요구 사항](/getting-started/installation)을 충족하는지 확인하세요.

  </AccordionItem>
</Accordion>
