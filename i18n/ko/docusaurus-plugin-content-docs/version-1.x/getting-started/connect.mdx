## 연결 화면 개요

WCPOS 데스크톱 또는 모바일 앱을 열면 연결 화면이 표시됩니다. 여기에서 상점 연결 및 사용자 로그인을 관리할 수 있습니다.

{/* 스크린샷: connect-screen.png */}

## 새 상점 추가하기

1. 텍스트 필드에 WooCommerce 상점 URL을 입력하세요 (예: `https://mystore.com`)
2. **연결**을 클릭하세요
3. WordPress 자격 증명으로 로그인하라는 페이지로 리디렉션됩니다
4. 로그인에 성공한 후 앱으로 돌아옵니다

## 여러 상점 연결하기

필요한 만큼 많은 WooCommerce 상점에 연결할 수 있습니다. 각 상점은 연결 화면에서 별도의 카드로 표시되며, 다음을 보여줍니다:

- **상점 이름** 및 파비콘
- **상점 URL**
- 해당 상점에 로그인한 **사용자**

이는 여러 위치나 사업체를 관리하는 데 유용합니다.

## 상점당 여러 사용자

각 상점은 동시에 여러 사용자가 로그인할 수 있습니다. 이는 다음에 유용합니다:

- **교대 근무** - 새로운 캐셔가 이전 캐셔가 로그아웃하기 전에 로그인할 수 있습니다
- **다중 등록기 설정** - 서로 다른 장치에서 서로 다른 캐셔
- **빠른 사용자 전환** - 사용자 계정 간 쉽게 전환 가능

### 사용자 추가하기

상점의 "로그인한 사용자" 옆에 있는 <Icon name="plus" /> 버튼을 클릭하여 해당 상점에 다른 사용자를 추가합니다.

### 사용자 전환하기

사용자 배지(e.g., "Brenda")를 클릭하여 그 사용자로서 POS를 엽니다.

### 사용자 제거하기

사용자 배지에서 **×**를 클릭하여 해당 사용자를 상점에서 로그아웃합니다.

## 상점 제거하기

상점 카드에서 빨간 **×** 버튼을 클릭하여 목록에서 제거합니다. 이 작업은 모든 사용자를 로그아웃시키고 앱에서 상점 연결을 제거합니다.

## 데모 상점

화면 하단에 "데모 상점 입력" 링크가 있을 수 있습니다. 이를 통해 자신의 상점 데이터에 영향을 주지 않고 WCPOS 기능을 시험해 볼 수 있는 데모 WooCommerce 상점에 연결됩니다.

## 문제 해결

### 상점에 연결할 수 없나요?

- WCPOS 플러그인이 WordPress 사이트에 설치 및 활성화되어 있는지 확인하세요
- 올바른 URL을 사용하고 있는지 확인하세요 (사이트가 SSL을 사용하는 경우 `https://`를 포함하세요)
- WooCommerce REST API에 액세스할 수 있는지 확인하세요
- 사용자 계정에 POS 액세스 권한이 있는지 확인하세요

### 연결이 계속 실패하나요?

- 먼저 웹 브라우저에서 `yourdomain.com/pos`에 접근하여 플러그인이 작동하는지 확인하세요
- 사이트의 오류 로그에서 문제를 확인하세요
- 서버가 [최소 요구사항](/getting-started/installation)을 충족하는지 확인하세요
