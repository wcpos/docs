## 이전 버전 재설치

WCPOS 플러그인의 이전 버전으로 돌아가야 하는 경우, 다음 단계를 따르십시오:

1. WCPOS의 [고급 보기](https://wordpress.org/plugins/woocommerce-pos/advanced/) 페이지를 방문하십시오.

2. **고급 옵션** 섹션으로 스크롤하여 **이전 버전** 선택 메뉴를 찾습니다.

3. 선택 메뉴를 열면 사용 가능한 플러그인 버전 목록이 표시됩니다. 설치하려는 버전을 찾아 **다운로드** 버튼을 클릭하십시오. 다운로드한 파일을 바탕 화면이나 편리한 위치에 저장하십시오.

4. WordPress 관리 대시보드에 로그인하십시오.

5. 왼쪽 메뉴에서 **플러그인**을 클릭하여 **플러그인** 페이지로 이동하십시오.

6. 현재 버전의 WCPOS 플러그인을 비활성화하고 삭제하십시오. 이렇게 하면 WordPress 설치에서 플러그인이 제거됩니다. 데이터는 손실되지 않으니 걱정하지 마십시오.

7. 페이지 상단의 **새로 추가** 버튼을 클릭하십시오.

8. **플러그인 추가** 페이지에서 **플러그인 업로드** 버튼을 클릭하십시오.

9. 바탕 화면이나 저장한 위치에서 이전에 다운로드한 플러그인 파일을 선택하십시오.

10. **지금 설치** 버튼을 클릭하고 설치가 완료될 때까지 기다리십시오.

11. 설치가 완료된 후, **활성화** 버튼을 클릭하여 이전 버전의 WCPOS 플러그인을 활성화하십시오.

이것으로 끝입니다! WCPOS 플러그인의 이전 버전을 성공적으로 재설치하였습니다.

재설치 과정 중 문제가 발생하거나 추가 지원이 필요한 경우, support@wcpos.com으로 지원 팀에 문의해 주십시오.
