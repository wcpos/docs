## 이전 버전 다시 설치하기

WCPOS 플러그인의 이전 버전으로 돌아가야 하는 경우, 다음 단계를 따르세요:

1. WordPress.org의 [고급 보기](https://wordpress.org/plugins/woocommerce-pos/advanced/) 페이지를 방문하세요.

2. 아래로 스크롤하여 **고급 옵션** 섹션에서 **이전 버전** 선택 메뉴를 찾으세요.

3. 선택 메뉴를 열면 사용 가능한 플러그인 버전 목록이 표시됩니다. 설치하려는 버전을 찾아 해당 버전 옆의 **다운로드** 버튼을 클릭하세요. 다운로드한 파일을 데스크톱이나 편리한 위치에 저장하세요.

4. WordPress 관리자 대시보드에 로그인하세요.

5. 왼쪽 메뉴에서 **플러그인**을 클릭하여 **플러그인** 페이지로 이동하세요.

6. 현재 버전의 WCPOS 플러그인을 비활성화하고 삭제하세요. 이렇게 하면 플러그인이 WordPress 설치에서 제거됩니다. 걱정하지 마세요, 데이터는 손실되지 않습니다.

7. 페이지 상단의 **새로 추가** 버튼을 클릭하세요.

8. **플러그인 추가** 페이지에서 **플러그인 업로드** 버튼을 클릭하세요.

9. 이전에 다운로드한 플러그인 파일을 데스크톱이나 저장한 위치에서 선택하세요.

10. **지금 설치** 버튼을 클릭하고 설치가 완료될 때까지 기다리세요.

11. 설치가 완료되면 **활성화** 버튼을 클릭하여 이전 버전의 WCPOS 플러그인을 활성화하세요.

이제 끝났습니다! WCPOS 플러그인의 이전 버전을 성공적으로 다시 설치했습니다.

다시 설치 과정에서 문제가 발생하거나 추가 도움이 필요한 경우, support@wcpos.com으로 지원팀에 문의하세요.
