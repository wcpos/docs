## 이전 버전 재설치 {#re-installing-a-previous-version}

WCPOS 플러그인의 이전 버전으로 되돌려야 하는 경우 다음 단계를 따르세요:

1. WCPOS의 WordPress.org [고급 보기](https://wordpress.org/plugins/woocommerce-pos/advanced/) 페이지를 방문하세요.

2. **고급 옵션** 섹션까지 아래로 스크롤하여 **이전 버전** 선택 메뉴를 찾으세요.

3. 선택 메뉴를 열면 사용 가능한 플러그인 버전 목록이 표시됩니다. 설치하려는 버전을 찾아 옆에 있는 **다운로드** 버튼을 클릭하세요. 다운로드한 파일을 바탕화면이나 편리한 위치에 저장하세요.

4. WordPress 관리자 대시보드에 로그인합니다.

5. 왼쪽 메뉴에서 **플러그인**을 클릭하여 **플러그인** 페이지로 이동합니다.

6. 현재 버전의 WCPOS 플러그인을 비활성화하고 삭제합니다. 이렇게 하면 WordPress 설치에서 플러그인이 제거됩니다. 데이터는 손실되지 않으니 걱정하지 않아도 됩니다.

7. 페이지 상단의 **플러그인 추가** 버튼을 클릭합니다.

8. **플러그인 추가** 페이지에서 **플러그인 업로드** 버튼을 클릭합니다.

9. 이전에 다운로드한 플러그인 파일을 바탕화면 또는 저장한 위치에서 선택합니다.

  <Image
    alt="플러그인 업로드"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">Upload plugin in the WordPress Admin</p>
</div>

10. **지금 설치** 버튼을 클릭하고 설치가 완료될 때까지 기다립니다.

11. 설치가 완료되면 **활성화** 버튼을 클릭하여 이전 버전의 WCPOS 플러그인을 활성화합니다.

완료되었습니다! 이전 버전의 WCPOS 플러그인이 성공적으로 재설치되었습니다.

재설치 과정에서 문제가 발생하거나 추가 지원이 필요한 경우, support@wcpos.com으로 지원팀에 문의해 주십시오.
