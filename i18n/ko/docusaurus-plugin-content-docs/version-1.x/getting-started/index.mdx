## WCPOS란 무엇인가요?

WCPOS는 WooCommerce를 위한 판매 시점(POS) 시스템입니다. 웹 브라우저 또는 전용 앱을 사용하여 실제 위치에서 WooCommerce 스토어의 제품을 판매할 수 있습니다.

**이용 가능:**

- **웹:** `yourdomain.com/pos`에서 즉시 액세스
- **데스크탑:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **모바일:** [iOS (TestFlight)](https://testflight.apple.com/join/JGBdVRrW) 및 [Android (Google Play Beta)](https://play.google.com/apps/testing/com.wcpos.main)

## 5분 안에 시작하기

### 1. 플러그인 설치

WordPress 사이트에 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)을 설치하세요:

1. `WP Admin > 플러그인 > 새로 추가`로 이동
2. "WooCommerce POS"를 검색
3. **지금 설치**를 클릭한 후, **활성화** 클릭

자세한 요구 사항은 [설치](/getting-started/installation)를 참조하세요.

### 2. POS에 액세스

**웹 사용자:**
브라우저에서 `yourdomain.com/pos`로 이동하세요. WordPress 자격 증명으로 로그인하라는 메시지가 표시됩니다.

**데스크탑/모바일 사용자:**
1. 앱 다운로드: [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64), [iOS](https://testflight.apple.com/join/JGBdVRrW), 또는 [Android](https://play.google.com/apps/testing/com.wcpos.main)
2. 스토어 URL을 입력하고 **연결** 클릭
3. WordPress 자격 증명으로 로그인

여러 스토어 및 사용자를 관리하는 방법에 대한 자세한 내용은 [연결](/getting-started/connect)을 참조하세요.

### 3. 초기 동기화 대기

WCPOS는 서버 과부하를 방지하기 위해 제품을 소량(한 번에 약 50개)으로 다운로드합니다. 검색, 스크롤, 또는 필터를 사용하면서 제품이 점진적으로 다운로드됩니다. 판매를 시작하기 전에 모든 것이 동기화될 때까지 기다릴 필요가 없습니다. 자세한 내용은 [제품 동기화](/products/sync)를 참조하세요.

### 4. 판매 시작

- 제품을 검색하거나 스캔하여 장바구니에 추가
- 고객을 선택하거나 " 게스트" 사용
- **결제** 클릭하여 결제 처리
- 영수증 인쇄 또는 이메일 발송

## 주요 기능

- **빠른 로컬 검색** - 제품이 로컬로 저장되어 즉각적인 결과 제공
- **오프라인 탐색** - 인터넷 없이 제품 및 고객 검색(주문 완료에는 연결이 필요함)
- **바코드 스캔** - USB 또는 블루투스 바코드 스캐너 사용 가능
- **다양한 결제 방법** - 현금, 카드 및 맞춤형 게이트웨이
- **영수증 인쇄** - 사용자 정의 가능한 영수증 인쇄 또는 이메일 발송
- **다양한 스토어 지원** - 여러 WooCommerce 스토어에 연결 가능

## 프로 기능

[WCPOS Pro](https://wcpos.com/pro)는 추가 기능을 제공합니다:

- **제품 관리** - POS에서 직접 재고 및 가격 수정
- **주문 기록** - 이전 주문 보기 및 관리
- **고객 관리** - 고객 세부정보 추가 및 수정
- **판매 보고서** - 인쇄 옵션이 있는 종일 보고서
- **추가 결제 게이트웨이** - Stripe Terminal, SumUp 등
