## WCPOS란? {#what-is-wcpos}

WCPOS는 WooCommerce용 POS(Point of Sale) 시스템입니다. 웹 브라우저나 전용 앱을 사용하여 실제 매장에서 WooCommerce 스토어의 상품을 직접 판매할 수 있습니다.

## 5분 만에 시작하기 {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="플러그인 설치">

WordPress 사이트에 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)을 설치합니다:

1. **WP Admin → 플러그인 → 새로 추가**로 이동합니다
2. "WCPOS"를 검색합니다
3. **지금 설치**를 클릭한 다음 **활성화**를 클릭합니다

자세한 요구 사항은 [설치](/getting-started/installation)를 참조하세요.

</StepCard>

<StepCard number={2} title="POS 접속">

**웹 사용자:** 브라우저에서 `yourdomain.com/pos` 로 이동합니다. WordPress 자격 증명으로 로그인하라는 메시지가 표시됩니다.

**데스크톱/모바일 사용자:** 앱을 다운로드하고 매장 URL을 입력한 후 **연결**을 클릭하여 WordPress 자격 증명으로 로그인합니다.

여러 매장 및 사용자 관리에 대한 자세한 내용은 [연결](/getting-started/connect)을 참조하세요.

</StepCard>

<StepCard number={3} title="초기 동기화 대기">

WCPOS는 서버에 과부하가 걸리지 않도록 상품을 소량 배치로 다운로드합니다. 검색, 스크롤 또는 필터링할 때 상품이 점진적으로 다운로드되므로 판매를 시작하기 전에 모든 항목이 동기화될 때까지 기다릴 필요가 없습니다.

자세한 내용은 [상품 동기화](/products/sync)를 참조하세요.

</StepCard>

<StepCard number={4} title="판매 시작" isLast>

- 상품을 검색하거나 스캔하여 장바구니에 추가합니다
- 고객을 선택하거나 "게스트"를 사용합니다
- **결제**를 클릭하여 결제를 처리합니다
- 영수증을 인쇄하거나 이메일로 전송합니다

</StepCard>

</Steps>
