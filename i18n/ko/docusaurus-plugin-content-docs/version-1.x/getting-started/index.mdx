## WCPOS란 무엇인가요? {#what-is-wcpos}

WCPOS는 WooCommerce를 위한 판매 시점(POS) 시스템입니다. 이를 통해 웹 브라우저나 전용 앱을 사용하여 오프라인에서 WooCommerce 상점의 제품을 판매할 수 있습니다.

## 5분 안에 시작하기 {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="플러그인 설치하기">

워드프레스 사이트에 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)을 설치하세요:

1. **WP 관리자 → 플러그인 → 새로 추가**로 이동하세요.
2. "WooCommerce POS"를 검색하세요.
3. **지금 설치**를 클릭한 다음 **활성화**하세요.

자세한 요구 사항은 [설치](/getting-started/installation)를 참조하세요.

</StepCard>

<StepCard number={2} title="POS에 접속하기">

**웹 사용자:** 브라우저에서 `yourdomain.com/pos`로 이동하세요. 워드프레스 인증 정보를 입력하라는 메시지가 표시됩니다.

**데스크탑/모바일 사용자:** 앱을 다운로드하고, 상점 URL을 입력하고, **연결**을 클릭한 후 워드프레스 인증 정보를 입력하세요.

여러 상점과 사용자를 관리하는 방법에 대한 자세한 사항은 [연결](/getting-started/connect)을 참조하세요.

</StepCard>

<StepCard number={3} title="초기 동기화 기다리기">

WCPOS는 서버 과부하를 피하기 위해 제품을 소량으로 다운로드합니다. 제품은 검색, 스크롤 또는 필터링 할 때 점진적으로 다운로드되므로 판매를 시작하기 전에 모든 것이 동기화되기를 기다릴 필요가 없습니다.

자세한 내용은 [제품 동기화](/products/sync)를 참조하세요.

</StepCard>

<StepCard number={4} title="판매 시작하기" isLast>

- 제품을 검색하거나 스캔하여 장바구니에 추가
- 고객을 선택하거나 "게스트" 사용
- 결제를 처리하려면 **결제** 클릭
- 영수증 인쇄 또는 이메일 전송

</StepCard>

</Steps>
