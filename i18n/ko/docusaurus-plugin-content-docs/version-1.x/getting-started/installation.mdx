WCPOS 플러그인의 적절한 기능과 최적의 성능을 보장하기 위해, 사용 환경이 다음 최소 요구 사항을 충족하는지 확인하세요:

## 서버 요구 사항 {#server-requirements}

WCPOS 플러그인은 WordPress 사이트에서 실행되며, 사이트는 다음 최소 사양을 충족해야 합니다:

| 구성 요소   | 최소 버전 | 참고           |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x 권장 |

이러한 최소 사양을 충족하면 WordPress 환경과의 호환성 및 안정성이 보장되고 최신 WCPOS 기능에 접근할 수 있습니다.

## 브라우저 요구 사항 {#browser-requirements}

POS의 웹 버전은 모든 최신 브라우저에서 실행됩니다. 최신 안정 버전을 사용하세요:

| 브라우저         | 지원 버전 |
| --------------- | ----------------- |
| Google Chrome   | 최신 안정 버전     |
| Mozilla Firefox | 최신 안정 버전     |
| Safari          | 최신 안정 버전     |
| Microsoft Edge  | 최신 안정 버전     |

:::warning 시크릿/비공개 모드 사용 금지
비공개 브라우징은 WCPOS가 데이터를 오프라인으로 저장하는 데 사용하는 브라우저의 IndexedDB 데이터베이스에 대한 접근을 차단할 수 있어 일부 기능이 올바르게 작동하지 않습니다. 또한 JavaScript가 활성화되어 있어야 합니다.
:::

## 앱 요구 사항 {#app-requirements}

WCPOS는 데스크톱 및 모바일 앱으로도 제공됩니다. 각 앱은 다음 최소 운영 체제용으로 빌드되었습니다:

| 플랫폼        | 최소 OS                | 아키텍처              |
| --------------- | ------------------------- | -------------------------- |
| Windows 데스크톱 | Windows 10                | x64, ARM64                 |
| macOS 데스크톱   | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Android         | Android 10 (API level 29) | —                          |

:::note 인쇄 권한
블루투스 및 네트워크 인쇄를 사용하려면 프롬프트가 표시될 때 앱에 해당 블루투스 및 로컬 네트워크 권한을 부여해야 합니다.
:::

## 플러그인 설치 {#installing-the-plugin}

1. `WP Admin > Plugins > Add Plugin`으로 이동합니다
2. "wcpos"를 검색합니다
3. **지금 설치**를 클릭한 다음 **활성화**합니다

또는 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)에서 플러그인을 다운로드하여 수동으로 업로드할 수 있습니다.

## POS 접근하기 {#accessing-the-pos}

활성화 후 다음 위치에서 POS에 접근할 수 있습니다:

- **웹:** `yourdomain.com/pos`
- **데스크톱:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **모바일:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

최소 요구 사항을 충족하고 호환되는 웹 브라우저를 사용하면 WCPOS 플러그인을 원활하게 사용할 수 있으며, 판매 시점 운영을 효율적으로 관리하기 위한 모든 잠재력을 활용할 수 있습니다.

추가 질문이 있거나 도움이 필요하면 [Discord 채팅](https://wcpos.com/discord)에 글을 올리거나 support@wcpos.com으로 이메일을 보내 주저 없이 문의해 주세요.
