WCPOS 플러그인의 올바른 기능 및 최적의 성능을 보장하기 위해, 귀하의 환경이 다음 최소 요구 사항을 충족하는지 확인해 주십시오:

## 서버 요구 사항 {#server-requirements}

- **WordPress:** 버전 5.6 이상
- **WooCommerce:** 버전 5.3 이상
- **PHP:** 버전 7.2 이상

이러한 최소 요구 사항을 충족하면 WCPOS 플러그인과 귀하의 WordPress 환경 간의 호환성과 안정성을 보장합니다. 또한 플러그인이 제공하는 최신 기능과 개선 사항의 혜택을 누릴 수 있습니다.

## 브라우저 요구 사항 {#browser-requirements}

- **현대 웹 브라우저:** WCPOS 플러그인은 Google Chrome, Mozilla Firefox, Safari 및 Microsoft Edge와 같은 현대 웹 브라우저에서 작동하도록 설계되었습니다. 귀하가 선호하는 브라우저의 최신 안정된 버전을 사용하고 있는지 확인해 주십시오.
- **JavaScript 사용 가능:** WCPOS 플러그인은 상호 작용 기능을 위해 JavaScript에 의존합니다. 브라우저 설정에서 JavaScript가 활성화되어 있는지 확인해 주십시오.
- **비공식 탐색 모드 비활성화:** 브라우저가 비공식 탐색 모드에 있지 않아야 합니다. 이 모드는 브라우저의 IndexedDB (IDB) 데이터베이스에 대한 접근을 제한할 수 있습니다. 비공식 탐색 모드는 WCPOS 플러그인 내 특정 기능의 정상 작동을 방해할 수 있습니다.

## 플러그인 설치 {#installing-the-plugin}

1. `WP Admin > 플러그인 > 새로 추가`
2. "WooCommerce POS" 검색
3. **지금 설치**를 클릭한 후, **활성화** 클릭

또는, [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)에서 플러그인을 다운로드하여 수동으로 업로드할 수 있습니다.

## POS 접근 {#accessing-the-pos}

활성화 후, 다음 주소에서 POS에 접근할 수 있습니다:

- **웹:** `yourdomain.com/pos`
- **데스크탑:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **모바일:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

최소 요구 사항을 충족하고 호환되는 웹 브라우저를 사용함으로써, WCPOS 플러그인을 원활하게 사용할 수 있으며, 효율적으로 판매 시점 운영을 관리할 수 있는 전체 잠재력을 활용할 수 있습니다.
