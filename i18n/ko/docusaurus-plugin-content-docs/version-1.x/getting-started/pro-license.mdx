:::info Pro 기능
[WCPOS Pro](https://wcpos.com/pro)는 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)에 기능을 추가하는 유료 플러그인입니다.
:::

## Pro에 포함된 내용은? {#whats-included-in-pro}

WCPOS Pro는 추가 기능을 잠금 해제합니다:

- **제품 화면** - POS에서 재고 수준 및 가격을 직접 편집
- **주문 화면** - 주문 내역을 보고 관리
- **고객 화면** - 고객 정보 추가 및 편집
- **보고서 화면** - 일일 판매 보고서 생성
- **추가 결제 게이트웨이** - Stripe Terminal, SumUp 및 커스텀 게이트웨이

## 설치 {#installation}

WCPOS Pro 라이센스를 구매하신 경우 플러그인을 설치 및 활성화하려면 아래 단계를 따르세요:

1. 다음으로 이동합니다: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **다운로드** 아래에서 다운로드 링크를 클릭하고 플러그인을 데스크탑에 저장하십시오.
3. 그런 다음 사이트에 로그인하고 `WP Admin > Plugins > Add New > Upload Plugin`으로 이동합니다.
4. 데스크탑에서 플러그인 zip 파일을 업로드하고 활성화합니다.
5. 다음으로 `WP Admin > POS > Settings > License`로 이동하여 라이센스 키와 라이센스 이메일을 입력하여 활성화를 완료합니다.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "계정 페이지에서 WCPOS Pro를 다운로드할 수 있습니다.")

## 업데이트 {#updates}

새 버전이 준비되면 다른 플러그인처럼 업데이트 대시보드에 나타납니다. 최신 버전은 [계정 페이지](https://wcpos.com/my-account/)에서 다운로드할 수도 있습니다.

## 라이센스 구매 {#purchase-a-license}

[wcpos.com/pro](https://wcpos.com/pro) 를 방문하여 Pro 라이센스를 구매하십시오.
