:::info Pro 기능
[WCPOS Pro](https://wcpos.com/pro)는 WCPOS의 관리 화면, 추가 결제 게이트웨이 및 Pro 전용 기능을 제공하는 유료 플러그인입니다.

**v1.8 이상부터 Pro는 독립형 플러그인입니다** — 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)을 함께 설치할 필요가 없습니다.
:::

## Pro에는 무엇이 포함되어 있나요? {#whats-included-in-pro}

WCPOS Pro는 추가 기능을 활성화합니다:

- **상품 화면** - POS에서 직접 재고 수량과 가격 편집
- **주문 화면** - 주문 내역 보기 및 관리
- **고객 화면** - 고객 정보 추가 및 편집
- **보고서 화면** - 일일 마감 판매 보고서 생성
- **추가 결제 게이트웨이** - Stripe Terminal, SumUp 및 사용자 지정 게이트웨이

## 설치 {#installation}

WCPOS Pro 라이선스를 구매한 경우 아래 단계에 따라 플러그인을 설치하고 활성화하세요.

:::note 이전 버전에서 업그레이드하시나요?
이전에 무료 WCPOS 플러그인과 이전 Pro 애드온을 모두 설치했다면, 새 독립형 Pro 플러그인을 활성화하기 전에 **무료 플러그인을 비활성화하고 삭제**하세요. 두 플러그인을 함께 실행하면 *"Invalid response fetching remote state"* 또는 치명적인 *"Class not found"* 오류와 같은 활성화 오류가 발생할 수 있습니다.
:::

1. 이동: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **다운로드**에서 다운로드 링크를 클릭하고 플러그인을 데스크톱에 저장합니다.
3. 그런 다음 사이트로 이동해 로그인하고 `WP Admin > 플러그인 > Add Plugin > 플러그인 업로드` 로 이동합니다.
4. 데스크톱에 있는 플러그인 zip 파일을 업로드하고 활성화합니다.
5. 다음으로 `WP Admin > POS > 설정 > License` 로 이동해 라이선스 키와 라이선스 이메일을 입력하여 활성화를 완료합니다.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "계정 페이지에서 WCPOS Pro를 다운로드할 수 있습니다")

## 업데이트 {#updates}

새 버전이 준비되면 다른 플러그인과 마찬가지로 업데이트 알림판에 표시됩니다. 최신 버전은 [계정 페이지](https://wcpos.com/my-account/)에서도 다운로드할 수 있습니다.

## 문제 해결 {#troubleshooting}

### 활성화 실패 {#activation-failures}

- **"유효하지 않은 라이선스 키"** — 붙여넣을 때 끝에 공백이 들어가지 않았는지 다시 확인하고, 이메일 주소가 구매 시 사용한 주소와 일치하는지 확인하세요.
- **"라이선스 한도에 도달했습니다"** — 기본적으로 라이선스는 **2개 도메인**에서 활성화됩니다(운영 사이트 1개와 스테이징 사이트 1개를 기준으로 함). **하위 도메인은 별도 활성화로 계산됩니다** — `example.com`, `staging.example.com`, `dev.example.com` 은 세 개로 계산됩니다. 필수 개발/테스트/운영 환경을 제공하는 Pantheon 같은 호스팅에서 자주 발생합니다. 사용하지 않는 사이트는 [계정 페이지](https://wcpos.com/my-account/)에서 비활성화하거나 **support@wcpos.com** 으로 이메일을 보내세요 — 요청하면 추가 활성화가 일반적으로 승인됩니다.

### 만료된 라이선스 {#expired-licence}

- 라이선스가 만료되어도 Pro 기능은 계속 활성 상태로 유지되지만 업데이트는 받을 수 없습니다.
- [계정 페이지](https://wcpos.com/my-account/)에서 갱신하세요 — 기존 키는 그대로 유지됩니다.

### 도메인 이전 {#domain-transfers}

- 사이트를 새 도메인으로 이전한 경우 먼저 이전 도메인에서 라이선스를 비활성화한 다음 새 도메인에서 다시 활성화하세요.
- 이전 도메인에 더 이상 접근할 수 없는 경우 **support@wcpos.com**에 문의하여 활성화를 재설정하세요.

### 라이선스 서버 연결 {#licence-server-connectivity}

- 활성화하려면 WordPress 서버가 WCPOS 라이선스 서버에 연결할 수 있어야 합니다.
- 활성화가 별도 메시지 없이 실패하는 경우 호스트의 방화벽이나 보안 플러그인이 아웃바운드 HTTPS 요청을 차단하고 있지 않은지 확인하세요.
- 일부 관리형 호스트는 기본적으로 외부 API 호출을 차단합니다. 호스트에 문의하여 `wcpos.com` 을 허용 목록에 추가하세요.

### 라이선스 상태 확인 {#checking-licence-status}

- 현재 활성화 상태를 확인하려면 `WP Admin > POS > 설정 > License` 로 이동하세요.
- [계정 페이지](https://wcpos.com/my-account/)에서 모든 활성 사이트와 만료일을 확인할 수 있습니다.

## 라이선스 구매 {#purchase-a-license}

Pro 라이선스를 구매하려면 [wcpos.com/pro](https://wcpos.com/pro)를 방문하세요.

:::note 구매 및 업그레이드는 웹사이트에서 진행됩니다
앱 안에는 "Pro로 업그레이드" 버튼이 없습니다. 구매, 요금제 업그레이드, 갱신은 모두 [wcpos.com](https://wcpos.com/pro) 및 [계정 페이지](https://wcpos.com/my-account/)에서 처리됩니다. 구매 후에는 [위 단계](#installation)에 따라 Pro를 설치하세요.
:::
