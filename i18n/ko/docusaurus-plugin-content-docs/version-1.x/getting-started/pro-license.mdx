:::info Pro 기능
[WCPOS Pro](https://wcpos.com/pro)는 WCPOS의 관리 화면, 추가 결제 게이트웨이 및 기타 Pro 전용 기능을 지원하는 유료 플러그인입니다.

**v1.8 이상부터 Pro는 독립형 플러그인입니다** — 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)을 함께 설치할 필요가 없습니다.
:::

## Pro에 포함된 기능 {#whats-included-in-pro}

WCPOS Pro는 다음과 같은 추가 기능을 제공합니다:

- **상품 화면** - POS에서 직접 재고 수량과 가격을 수정
- **주문 화면** - 주문 내역 조회 및 관리
- **고객 화면** - 고객 정보 추가 및 수정
- **보고서 화면** - 마감 매출 보고서 생성
- **추가 결제 게이트웨이** - Stripe Terminal, SumUp 및 사용자 정의 게이트웨이

## 설치 {#installation}

WCPOS Pro 라이선스를 구매하셨다면 아래 단계에 따라 플러그인을 설치하고 활성화하십시오:

:::note 이전 버전에서 업그레이드하시나요?
기존에 무료 WCPOS 플러그인과 이전 Pro 애드온을 함께 설치한 경우, 새 독립형 Pro 플러그인을 활성화하기 전에 **무료 플러그인을 비활성화하고 삭제하십시오**. 두 플러그인을 동시에 실행하면 *"Invalid response fetching remote state"* 또는 치명적인 *"Class not found"* 오류와 같은 활성화 오류가 발생할 수 있습니다.
:::

1. 다음으로 이동: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **다운로드** 아래에서 다운로드 링크를 클릭하고 플러그인을 데스크톱에 저장합니다.
3. 사이트에 로그인한 후 `WP Admin > 플러그인 > Add Plugin > 플러그인 업로드` 로 이동합니다.
4. 데스크톱에서 플러그인 zip 파일을 업로드하고 활성화합니다.
5. 다음으로 `WP Admin > POS > 설정 > License` 로 이동하여 라이선스 키와 라이선스 이메일을 입력하면 활성화가 완료됩니다.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "You can download WCPOS Pro on your account page")

## 업데이트 {#updates}

새 버전이 출시되면 다른 플러그인과 마찬가지로 업데이트 대시보드에 표시됩니다. [계정 페이지](https://wcpos.com/my-account/)에서 최신 버전을 직접 다운로드할 수도 있습니다.

## 문제 해결 {#purchase-a-license}

### 활성화 실패 {#activation-failures}

- **"Invalid licence key"** — 붙여넣을 때 뒤에 공백이 포함되지 않았는지 확인하고, 이메일 주소가 구매 시 사용한 것과 일치하는지 확인하십시오.
- **"Licence limit reached"** — 각 라이선스는 정해진 수의 사이트에 적용됩니다. [계정 페이지](https://wcpos.com/my-account/)에서 사용하지 않는 사이트를 비활성화하거나 플랜을 업그레이드하십시오.

### 라이선스 만료 {#expired-licence}

- 라이선스가 만료되어도 Pro 기능은 계속 사용할 수 있지만 업데이트는 제공되지 않습니다.
- [계정 페이지](https://wcpos.com/my-account/)에서 갱신할 수 있으며, 기존 키는 그대로 유지됩니다.

### 도메인 이전 {#domain-transfers}

- 사이트를 새 도메인으로 이전한 경우, 먼저 이전 도메인에서 라이선스를 비활성화한 후 새 도메인에서 다시 활성화하십시오.
- 이전 도메인에 더 이상 접근할 수 없는 경우, **support@wcpos.com**으로 연락하여 활성화를 초기화하세요.

### 라이선스 서버 연결 {#licence-server-connectivity}

- 활성화하려면 WordPress 서버가 WCPOS 라이선스 서버에 연결할 수 있어야 합니다.
- 활성화가 오류 없이 실패하는 경우, 호스트의 방화벽이나 보안 플러그인이 아웃바운드 HTTPS 요청을 차단하고 있지 않은지 확인하세요.
- 일부 매니지드 호스트는 기본적으로 외부 API 호출을 차단합니다. 호스트에 연락하여 `wcpos.com` 을 허용 목록에 추가하세요.

### 라이선스 상태 확인 {#checking-licence-status}

- `WP Admin > POS > 설정 > License` 로 이동하여 현재 활성화 상태를 확인하세요.
- [계정 페이지](https://wcpos.com/my-account/)에서 모든 활성화된 사이트와 만료일을 확인할 수 있습니다.

## 라이선스 구매 {#purchase-a-license}

Pro 라이선스를 구매하려면 [wcpos.com/pro](https://wcpos.com/pro)를 방문하세요.
