:::info 프로 기능
[WCPOS 프로](https://wcpos.com/pro)는 무료 [WCPOS 플러그인](https://wordpress.org/plugins/woocommerce-pos/)에 기능을 추가하는 유료 플러그인입니다.
:::

## 프로에 포함된 기능

WCPOS 프로는 추가 기능을 제공합니다:

- **상품 화면** - POS에서 직접 재고 수량과 가격 편집
- **주문 화면** - 주문 이력 보기 및 관리
- **고객 화면** - 고객 정보 추가 및 편집
- **보고서 화면** - 일일 판매 보고서 생성
- **추가 결제 게이트웨이** - 스트라이프 터미널, SumUp, 맞춤 게이트웨이

## 설치

WCPOS 프로 라이선스를 구매한 경우 다음 단계에 따라 플러그인을 설치하고 활성화하세요:

1. 이동: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **다운로드** 항목에서 다운로드 링크를 클릭하고 데스크톱에 플러그인을 저장합니다.
3. 사이트로 이동하여 로그인하고 `WP 관리자 > 플러그인 > 새로 추가 > 플러그인 업로드`로 이동합니다.
4. 데스크톱의 플러그인 zip 파일을 업로드하고 활성화합니다.
5. 다음으로 `WP 관리자 > POS > 설정 > 라이선스`로 이동하여 라이선스 키와 라이선스 이메일을 입력하여 활성화를 완료합니다.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "계정 페이지에서 WCPOS 프로를 다운로드할 수 있습니다")

## 업데이트

새 버전이 준비되면 다른 플러그인과 마찬가지로 업데이트 대시보드에 표시됩니다. [계정 페이지](https://wcpos.com/my-account/)에서 최신 버전을 다운로드할 수도 있습니다.

## 라이선스 구매

프로 라이선스를 구매하려면 [wcpos.com/pro](https://wcpos.com/pro)를 방문하세요.
