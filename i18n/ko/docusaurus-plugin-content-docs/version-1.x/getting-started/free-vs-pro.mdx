WCPOS에는 두 가지 에디션이 있습니다. **무료 플러그인**은 매장에서 결제를 받고, 영수증을 출력하고, 주문을 WooCommerce로 다시 동기화하는 데 필요한 모든 것을 제공합니다. **Pro**는 관리 화면(상품, 주문, 고객, 보고서), 더 많은 결제 게이트웨이, 쿠폰, 환불, 다중 매장 지원을 추가합니다.

주로 판매를 기록하는 계산대가 필요하다면 무료 플러그인으로 충분합니다. POS에서 매장의 일상 업무까지 처리하고 싶다면 — 재고 관리, 이전 주문 조회, 고객 환불, 통합 리더기를 통한 카드 결제 승인 — Pro가 그 용도에 맞습니다.

## 한눈에 보기 {#at-a-glance}

| 영역 | 무료 | Pro |
|---|---|---|
| 판매 처리 및 영수증 인쇄/이메일 발송 | ✓ | ✓ |
| 현금 및 외부 카드 결제 | ✓ | ✓ |
| 통합 카드 리더기(Stripe, SumUp, Vipps/MobilePay) | — | ✓ |
| 계산대에서 쿠폰 사용 | — | ✓ |
| POS에서 환불 처리 | — | ✓ |
| POS에서 재고, 가격, 상품 세부 정보 편집 | — | ✓ |
| 과거 주문 조회 및 재인쇄 | — | ✓ |
| 고객 추가 및 편집 | — | ✓ |
| 마감 보고서 | — | ✓ |
| 여러 매장 위치 | — | ✓ |
| 매장별 영수증 템플릿, 세금 ID, 브랜딩 | — | ✓ |
| 설정에서 POS 확장 기능 설치 및 관리 | — | ✓ |
| Discord 우선 지원 | — | ✓ |

## 무료 플러그인에 포함된 기능 {#what-s-in-the-free-plugin}

[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)의 무료 플러그인은 기본적인 "판매 처리" 워크플로를 지원합니다.

  </FeatureCard>
  <FeatureCard icon="cart-shopping" title="장바구니와 상품 검색">
    상품 패널 검색, 바코드 스캔, 항목 추가, 항목별 할인 적용, 카탈로그에 없는 개방 가격의 "기타" 항목 추가를 할 수 있습니다.
  </FeatureCard>
  <FeatureCard icon="receipt" title="영수증과 감열 인쇄">
    내장 [템플릿 갤러리](/templates/customise)에서 선택할 수 있습니다 — 영수증, 청구서, 견적서, 포장 명세서, 선물 영수증, 주방 티켓 — 그리고 네트워크, Bluetooth 또는 USB를 통해 58 mm 또는 80 mm 감열 프린터로 인쇄할 수 있습니다.

  <FeatureCard icon="sync" title="오프라인 저장 + 동기화">

  </FeatureCard>
  <FeatureCard icon="barcode" title="바코드 지원">
    USB 또는 Bluetooth 스캐너로 상품을 장바구니에 바로 스캔하거나 기기 카메라를 사용할 수 있습니다.
  </FeatureCard>
  <FeatureCard icon="file-invoice" title="고객 세금 ID">
    고객 기록에 내장된 세금 ID 필드 — VAT, ABN, GST, GSTIN, EIN 및 기타 지역별 형식을 지원합니다.
  </FeatureCard>
  <FeatureCard icon="globe" title="크로스 플랫폼 + 다국어">
    브라우저, 데스크톱(Windows/macOS), 모바일 앱(iOS/Android, 베타)을 지원합니다. 주요 유럽, 아시아, 라틴아메리카 언어로 번역되어 있습니다.
  </FeatureCard>
  <FeatureCard icon="puzzle-piece" title="확장 기능 카탈로그(보기 전용)">
    설정에서 POS 확장 기능 및 통합 목록을 둘러볼 수 있습니다. 설치하려면 Pro가 필요합니다.
  </FeatureCard>
</FeatureGrid>

:::note 무료 버전의 POS 전용 고객/쿠폰 접근
무료 플러그인에서도 기존 고객을 **선택**하거나 계산대에서 쿠폰 카탈로그를 **볼 수** 있습니다 — 다만 고객을 추가/편집할 수 없고 결제 시 쿠폰 코드를 적용할 수 없습니다. 이 기능들은 Pro에서 사용할 수 있습니다.
:::

## Pro에서 추가되는 기능 {#what-pro-adds}

Pro는 나머지 백오피스 기능, 더 많은 결제 옵션, 매장별 기능을 제공합니다. 아래 항목은 모두 Pro 전용입니다.

### 관리 화면 {#management-screens}

무료 플러그인은 판매 처리를 위한 상품 패널을 제공합니다. Pro는 전체 관리 영역을 추가합니다.

| 화면 | 할 수 있는 작업 |
|---|---|
| **[상품](/products)** | 재고 수준, 가격, 매출원가를 인라인으로 편집합니다. 전체 카탈로그에 대한 일괄 작업도 가능합니다. |
| **[주문](/orders)** | 과거 주문을 조회하고, 영수증을 다시 인쇄하고, 주문을 편집하며, 환불 흐름에 접근합니다. |
| **[고객](/customers)** | 새 고객을 추가하고 기존 고객 정보를 편집합니다(무료 사용자는 결제 시 선택기에서 고객을 선택만 할 수 있습니다). |
| **[보고서](/reports)** | 마감 판매 보고서 — 결제 방법, 계산원, 매장별 합계를 제공하며 현금 정산에 적합합니다. |
| **[쿠폰](/coupons)** | 계산대에서 WooCommerce 쿠폰 카탈로그를 둘러봅니다(무료 사용자는 흐릿한 미리보기만 볼 수 있습니다). |

### 결제 게이트웨이 {#payment-gateways}

무료 플러그인의 "카드" 게이트웨이는 외부 단말기에서 받은 판매를 기록하기만 하며, 리더기 자체와의 통합은 없습니다. Pro에서는 POS에서 **모든 WooCommerce 결제 게이트웨이**를 사용할 수 있고, 여러 통합 게이트웨이가 함께 제공됩니다.

| 게이트웨이 | 사용 사례 |
|---|---|
| **[Stripe Terminal](/payment/custom-gateways/stripe-terminal)** | Stripe의 S700 / WisePOS E 리더기와 직접 통합합니다. MOTO(전화 주문)와 테스트용 시뮬레이터를 지원합니다. |
| **[SumUp Terminal](/payment/custom-gateways/sumup-terminal)** | SumUp 카드 리더기를 POS에 페어링하고 계산대를 떠나지 않고 카드 결제를 완료합니다. |
| **[Vipps MobilePay](/payment/custom-gateways/vipps-mobilepay)** | QR 코드 또는 푸시 알림을 통한 휴대폰 기반 결제 — 노르웨이에서는 Vipps, 덴마크와 핀란드에서는 MobilePay입니다. |
| **[이메일 청구서](/payment/custom-gateways/email-invoice)** | 고객에게 이메일로 결제 링크를 보내며, 고객이 온라인으로 결제하면 주문이 완료됩니다. |
| **[웹 체크아웃](/payment/custom-gateways/web-checkout)** | 고객을 호스팅된 체크아웃으로 리디렉션하여 온라인 결제를 완료하게 합니다. |
| **기타 WooCommerce 게이트웨이** | [게이트웨이 템플릿](/payment/custom-gateways)을 사용하여 모든 WooCommerce 결제 플러그인을 POS 체크아웃에 연결할 수 있습니다. |

### 지원 {#support}

- **Discord 우선 지원** — 공개 커뮤니티 채널에 더해 비공개 채널에서 일대일 지원을 받을 수 있습니다.

## 가격 {#pricing}

Pro는 [wcpos.com/pro](https://wcpos.com/pro)에서 사이트별로 판매됩니다.

- **연간 라이선스** — 연 $129, 라이선스 기간 동안 업데이트와 지원이 포함됩니다.
- **평생 라이선스** — 일회성 $399, 제품 수명 동안 업데이트와 지원이 포함됩니다.

가격은 2026년 5월 기준이며 변경될 수 있습니다.

라이선스가 만료되어도 Pro 기능은 계속 작동하지만 업데이트와 지원은 더 이상 제공되지 않습니다. 활성화 단계와 라이선스 문제 해결은 [WCPOS Pro 설치](/getting-started/pro-license)를 참조하세요.

:::tip 아직 결정 중이신가요?
[demo.wcpos.com/pos](https://demo.wcpos.com/pos)에서 공개 데모를 사용해 볼 수 있습니다(로그인 `demo` / `demo`) — 전체 Pro 기능 세트가 실행되므로 구매 전에 관리 화면, 쿠폰, 환불 흐름을 확인할 수 있습니다.
:::

## 업그레이드할 준비가 되셨나요? {#ready-to-upgrade}

다운로드 및 활성화 단계는 [WCPOS Pro 설치](/getting-started/pro-license)를 참조하세요.

- **v1.8 이상 Pro** — 먼저 무료 플러그인을 비활성화하고 삭제하세요. v1.8부터 Pro는 독립형 플러그인이며 무료 버전과 함께 설치하면 안 됩니다.
- **이전 Pro(< v1.8)** — 사용 중인 Pro 버전에 필요하다면 무료 플러그인을 설치된 상태로 유지하세요.
