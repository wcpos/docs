# Stripe 터미널 게이트웨이

Stripe 터미널 게이트웨이를 통해 WooCommerce POS 내에서 Stripe 터미널 하드웨어 리더기를 사용하여 직접 대면 결제를 수락할 수 있습니다. 이 게이트웨이는 물리적인 카드 리더기와 테스트를 위한 시뮬레이터 모드를 모두 지원합니다.

## 주요 기능

- **하드웨어 통합**: 인터넷 연결을 통해 물리적 Stripe 터미널 리더기 연결
- **시뮬레이터 모드**: Stripe의 시뮬레이터를 사용하여 하드웨어 없이 결제 테스트
- **다양한 결제 방법**: 여러 카드 유형 및 결제 방법 지원
- **실시간 처리**: 즉각적인 결제 처리 및 확인
- **안전한 거래**: Stripe를 통한 PCI 준수 결제 처리
- **웹 기반 SDK**: 웹 응용 프로그램 호환성을 위한 Stripe의 JavaScript SDK 사용

## 설치

### 단계 1: 다운로드 및 설치

1. [WooCommerce용 Stripe 터미널 릴리스 페이지](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases) 방문
2. 최신 **stripe-terminal-for-woocommerce.zip** 파일 다운로드
3. WordPress 관리자에서 `플러그인 > 새로 추가 > 플러그인 업로드`로 이동
4. zip 파일을 업로드하고 플러그인 활성화

### 단계 2: Stripe 설정 구성

1. `WP 관리자 > WooCommerce > 설정 > 결제`로 이동
2. 결제 방법 목록에서 **Stripe 터미널** 찾기
3. **Stripe 터미널**을 클릭하여 설정에 액세스
4. **Stripe 비밀 키** 입력 (Stripe 대시보드에서 확인 가능)
5. 설정 저장

:::note
WooCommerce 설정에서 Stripe 터미널 게이트웨이를 활성화할 필요가 없습니다. 다음 단계에서 POS에 대해 특별히 활성화됩니다.
:::

### 단계 3: WooCommerce POS에서 활성화

1. `WP 관리자 > WooCommerce POS > 설정 > 결제`로 이동
2. 목록에서 **Stripe 터미널** 게이트웨이 찾기
3. POS에서 사용하기 위해 게이트웨이 활성화
4. 설정 저장

## 사용법

### 리더기 연결

POS에서 결제 중 Stripe 터미널 게이트웨이를 선택할 때:

1. **연결 방법 선택**: 물리적 리더기를 연결하거나 시뮬레이터를 사용할 수 있습니다.
2. **물리적 리더기**: Stripe 터미널 장치를 연결하기 위한 화면 지침 따름
3. **시뮬레이터**: 하드웨어 없이 다양한 결제 시나리오를 테스트하기 위해 시뮬레이터 모드 선택

### 결제 처리

1. **상품 추가**: POS의 장바구니에 상품 추가
2. **게이트웨이 선택**: 결제 방법으로 "Stripe 터미널" 선택
3. **리더기 연결**: 리더기를 연결하거나 시뮬레이터 모드 선택
4. **결제 처리**: 거래를 완료하기 위해 프롬프트 따르기
5. **확인**: 결제가 성공적으로 완료되면 주문이 자동으로 완료됩니다.

### 시뮬레이터로 테스트

시뮬레이터를 통해 다양한 결제 방법과 시나리오를 테스트할 수 있습니다:

- **카드 결제**: 다양한 카드 유형 테스트 (Visa, Mastercard 등)
- **비접촉 결제**: 탭하여 결제 거래 시뮬레이트
- **오류 시나리오**: 거부된 결제 및 기타 오류 조건 테스트
- **다양한 금액**: 다양한 거래 금액 테스트

## 요구 사항

- **Stripe 계정**: 터미널이 활성화된 활성 Stripe 계정
- **API 키**: 대시보드에서 가져온 Stripe 비밀 키
- **WooCommerce POS**: 무료 또는 프로 버전 설치 및 활성화
- **HTTPS**: 보안을 위해 SSL/HTTPS를 사용해야 합니다.

## 하드웨어 호환성

:::important 연결 요구 사항
이 구현은 Stripe의 JavaScript SDK를 사용하므로 웹 응용 프로그램을 통해 작동하지만 **인터넷에 연결된 터미널만** 필요합니다. Bluetooth 터미널은 현재 지원되지 않습니다.
:::

### 지원되는 터미널 (인터넷 연결)

- **Stripe 리더 S700** ✅ - 이더넷/WiFi 연결 터미널
- **WisePOS E** ✅ - WiFi 연결 터미널

### 지원되지 않는 터미널 (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Bluetooth 전용
- **BBPOS WisePad 3** ❌ - Bluetooth 전용  
- **Verifone P400** ❌ - Bluetooth 전용

:::note 향후 지원
Bluetooth 터미널 지원은 **2026년 1분기**에 iOS 및 Android 애플리케이션에 추가될 예정입니다. 이를 통해 M2 및 WisePad 3을 포함한 모든 Stripe 터미널 인증 리더기를 지원할 수 있습니다.
:::

## 문제 해결

### 일반적인 문제

**리더기가 연결되지 않음**
- 지원되는 인터넷에 연결된 터미널(S700 또는 WisePOS E)을 사용하고 있는지 확인
- 터미널이 WiFi/이더넷에 연결되어 온라인인지 확인
- Stripe 계정에서 터미널이 활성화되어 있는지 확인
- 터미널이 Stripe 대시보드에 등록되어 있는지 확인

**결제 거부됨**
- Stripe 계정이 활성화되어 있고 정상인지 확인
- 사용 중인 카드가 유효한지 확인
- 사용 가능한 자금이 충분한지 확인

**SSL 인증서 오류**
- Stripe 터미널은 HTTPS를 요구합니다 - 사이트에 유효한 SSL 인증서가 있는지 확인
- SSL 인증서가 올바르게 구성되었는지 확인
