# 스트라이프 단말기 게이트웨이 {#stripe-terminal-gateway}

스트라이프 단말기 게이트웨이를 사용하면 WCPOS 내에서 직접 스트라이프 단말기 하드웨어 리더기를 통해 대면 결제를 수락할 수 있습니다. 이 게이트웨이는 물리적 카드 리더기와 테스트를 위한 시뮬레이터 모드를 모두 지원합니다.

## 설치 {#installation}

<Steps>
  <StepCard number="1" title="다운로드 및 설치">

1. [WooCommerce용 스트라이프 단말기 릴리스 페이지](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)에 방문합니다.
2. 최신 **stripe-terminal-for-woocommerce.zip** 파일을 다운로드합니다.
3. WordPress 관리 페이지에서 `플러그인 > 새로 추가 > 플러그인 업로드`로 이동합니다.
4. zip 파일을 업로드하고 플러그인을 활성화합니다.

  </StepCard>
  <StepCard number="2" title="스트라이프 설정 구성">

1. `WP 관리자 > WooCommerce > 설정 > 결제`로 이동합니다.
2. 결제 방법 목록에서 **스트라이프 단말기**를 찾습니다.
3. **스트라이프 단말기**를 클릭하여 설정에 접근합니다.
4. **스트라이프 비밀 키**를 입력합니다(스트라이프 대시보드에서 확인할 수 있습니다).
5. 설정을 저장합니다.

:::note
워드프레스 WooCommerce 설정에서 스트라이프 단말기 게이트웨이를 활성화할 필요가 없습니다. 다음 단계에서 POS에 대해 특별히 활성화됩니다.
:::

  </StepCard>
  <StepCard number="3" title="WCPOS에서 활성화" isLast>

1. `WP 관리자 > POS > 설정 > 체크아웃`으로 이동합니다.
2. 목록에서 **스트라이프 단말기** 게이트웨이를 찾습니다.
3. POS에서 사용하기 위해 게이트웨이를 활성화합니다.
4. 설정을 저장합니다.

  </StepCard>
</Steps>

## 사용법 {#usage}

### 리더기 연결하기 {#connecting-a-reader}

POS에서 체크아웃 중 스트라이프 단말기 게이트웨이를 선택할 때:

1. **연결 방법 선택**: 물리적 리더기를 연결하거나 시뮬레이터를 사용할 수 있습니다.
2. **물리적 리더기**: 화면 지침에 따라 스트라이프 단말기 장치를 연결합니다.
3. **시뮬레이터**: 하드웨어 없이 다양한 결제 시나리오를 테스트하기 위해 시뮬레이터 모드를 선택합니다.

### 결제 처리하기 {#processing-payments}

1. **상품 추가**: POS에서 장바구니에 상품을 추가합니다.
2. **게이트웨이 선택**: 결제 방법으로 "스트라이프 단말기"를 선택합니다.
3. **리더기 연결**: 리더기를 연결하거나 시뮬레이터 모드를 선택합니다.
4. **결제 처리**: 거래를 완료하기 위해 지침을 따릅니다.
5. **확인**: 결제가 성공적으로 완료되면 주문이 자동으로 완료됩니다.

### 시뮬레이터로 테스트하기 {#testing-with-simulator}

시뮬레이터를 사용하면 다양한 결제 방법과 시나리오를 테스트할 수 있습니다:

- **카드 결제**: 다양한 카드 유형(비자, 마스터카드 등)을 테스트합니다.
- **비접촉 결제**: 탭-투-페이 거래를 시뮬레이션합니다.
- **오류 시나리오**: 결제가 거부된 경우 및 기타 오류 조건을 테스트합니다.
- **다양한 금액**: 여러 거래 금액을 테스트합니다.

## 하드웨어 호환성 {#hardware-compatibility}

:::important 연결 요구 사항
이 구현은 스트라이프의 JavaScript SDK를 사용하므로 웹 애플리케이션을 통해 작동하지만 **인터넷에 연결된 단말기만 필요합니다**. 블루투스 단말기는 현재 지원되지 않습니다.
:::

### 지원되는 단말기 (인터넷 연결됨) {#supported-terminals-internet-connected}

- **스트라이프 리더기 S700** ✅ - 이더넷/WiFi 연결 단말기
- **와이즈POS E** ✅ - WiFi 연결 단말기

### 지원되지 않는 단말기 (블루투스) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - 블루투스 전용
- **BBPOS WisePad 3** ❌ - 블루투스 전용  
- **Verifone P400** ❌ - 블루투스 전용

:::note 향후 지원
블루투스 단말기 지원이 **2026년 1분기**에 iOS 및 Android 애플리케이션에 추가됩니다. 이를 통해 M2 및 WisePad 3을 포함한 모든 스트라이프 단말기 인증 리더기에 대한 지원이 가능해집니다.
:::

### 일반적인 문제 {#common-issues}

<Accordion>
  <AccordionItem question="리더기가 연결되지 않음">

  - 지원되는 인터넷 연결 단말기(S700 또는 WisePOS E)를 사용하고 있는지 확인합니다.
  - 단말기가 WiFi/이더넷에 연결되어 있고 온라인인지 확인합니다.
  - 스트라이프 계정에서 단말기가 활성화되어 있는지 확인합니다.
  - 스트라이프 대시보드에서 단말기가 등록되어 있는지 확인합니다.

  </AccordionItem>

  <AccordionItem question="결제가 거부됨">

  - 스트라이프 계정이 활성 상태이고 양호한지 확인합니다.
  - 사용 중인 카드가 유효한지 확인합니다.
  - 충분한 자금이 있는지 확인합니다.

  </AccordionItem>

  <AccordionItem question="SSL 인증서 오류">

  - 스트라이프 단말기는 HTTPS를 요구합니다 - 사이트에 유효한 SSL 인증서가 있는지 확인합니다.
  - SSL 인증서가 제대로 구성되어 있는지 확인합니다.

  </AccordionItem>
</Accordion>

### 도움 요청하기 {#getting-help}

기술 지원을 받으려면:
- [GitHub 저장소](https://github.com/wcpos/stripe-terminal-for-woocommerce)에 방문하여 문제를 보고합니다.
- 하드웨어 관련 지침을 위해 [스트라이프 단말기 문서](https://stripe.com/docs/terminal)를 확인합니다.
- 계정 관련 문제는 스트라이프 지원에 문의합니다.
