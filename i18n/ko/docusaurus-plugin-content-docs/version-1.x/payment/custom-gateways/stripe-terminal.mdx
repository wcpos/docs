# 스트라이프 터미널 게이트웨이

스트라이프 터미널 게이트웨이를 사용하면 WooCommerce POS 내에서 스트라이프 터미널 하드웨어 리더기를 통해 대면 결제를 수락할 수 있습니다. 이 게이트웨이는 물리 카드를 위한 리더기와 테스트를 위한 시뮬레이터 모드를 모두 지원합니다.

## 설치

<Steps>
  <StepCard number="1" title="다운로드 및 설치">

1. [WooCommerce용 스트라이프 터미널 릴리스 페이지](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases) 방문
2. 최신 **stripe-terminal-for-woocommerce.zip** 파일 다운로드
3. WordPress 관리자에서 `플러그인 > 새로 추가 > 플러그인 업로드`로 이동
4. zip 파일을 업로드하고 플러그인 활성화

  </StepCard>
  <StepCard number="2" title="스트라이프 설정 구성">

1. `WP 관리자 > WooCommerce > 설정 > 결제`로 이동
2. 결제 방법 목록에서 **스트라이프 터미널** 찾기
3. **스트라이프 터미널**을 클릭하여 설정 접근
4. **스트라이프 비밀 키** 입력 (스트라이프 대시보드에서 받을 수 있음)
5. 설정 저장

:::note
WooCommerce 설정에서 스트라이프 터미널 게이트웨이를 활성화할 필요가 없습니다. 다음 단계에서 POS에 대해 특별히 활성화됩니다.
:::

  </StepCard>
  <StepCard number="3" title="WooCommerce POS에서 활성화" isLast>

1. `WP 관리자 > WooCommerce POS > 설정 > 체크아웃`으로 이동
2. 목록에서 **스트라이프 터미널** 게이트웨이 찾기
3. POS에서 사용하도록 게이트웨이 활성화
4. 설정 저장

  </StepCard>
</Steps>

## 사용 방법

### 리더기 연결

POS에서 체크아웃 시 스트라이프 터미널 게이트웨이를 선택할 때:

1. **연결 방법 선택**: 물리적 리더기를 연결하거나 시뮬레이터를 사용할 수 있습니다.
2. **물리적 리더기**: 화면 지침에 따라 스트라이프 터미널 장치 연결
3. **시뮬레이터**: 하드웨어 없이 다양한 결제 시나리오를 테스트하기 위해 시뮬레이터 모드 선택

### 결제 처리

1. **상품 추가**: POS에서 장바구니에 상품 추가
2. **게이트웨이 선택**: 결제 방법으로 "스트라이프 터미널" 선택
3. **리더기 연결**: 리더기를 연결하거나 시뮬레이터 모드 선택
4. **결제 처리**: 거래를 완료하기 위한 프롬프트 따르기
5. **확인**: 성공적인 결제 후 주문이 자동으로 완료됨

### 시뮬레이터로 테스트

시뮬레이터를 사용하면 다양한 결제 방법과 시나리오를 테스트할 수 있습니다:

- **카드 결제**: 다양한 카드 종류 테스트 (비자, 마스터카드 등)
- **비접촉 결제**: 터치 투 페이 거래 시뮬레이션
- **오류 시나리오**: 거부된 결제 및 기타 오류 조건 테스트
- **다양한 금액**: 다양한 거래 금액 테스트

## 하드웨어 호환성

:::important 연결 요건
이 구현은 스트라이프의 자바스크립트 SDK를 사용하므로 웹 애플리케이션을 통해 작동하지만 **인터넷 연결된 터미널만** 필요합니다. 블루투스 터미널은 현재 지원되지 않습니다.
:::

### 지원되는 터미널 (인터넷 연결)

- **스트라이프 리더 S700** ✅ - 이더넷/WiFi 연결 터미널
- **WisePOS E** ✅ - WiFi 연결 터미널

### 지원되지 않는 터미널 (블루투스)

- **BBPOS Chipper 2X BT** ❌ - 블루투스 전용
- **BBPOS WisePad 3** ❌ - 블루투스 전용  
- **Verifone P400** ❌ - 블루투스 전용

:::note 향후 지원
블루투스 터미널 지원은 **2026년 1분기**에 iOS 및 안드로이드 애플리케이션에 추가될 것입니다. 이를 통해 M2 및 WisePad 3를 포함한 모든 스트라이프 터미널 인증 리더기를 지원할 수 있습니다.
:::

### 일반적인 문제

<Accordion>
  <AccordionItem question="리더기가 연결되지 않음">

  - 지원되는 인터넷 연결 터미널(S700 또는 WisePOS E)을 사용하고 있는지 확인
  - 터미널이 WiFi/Ethernet에 연결되어 있고 온라인인지 확인
  - 스트라이프 계정에 터미널이 활성화되어 있는지 확인
  - 터미널이 스트라이프 대시보드에 등록되어 있는지 확인

  </AccordionItem>

  <AccordionItem question="결제 거부됨">

  - 스트라이프 계정이 활성화되어 있고 정상인지 확인
  - 사용 중인 카드가 유효한지 확인
  - 충분한 자금이 있는지 확인

  </AccordionItem>

  <AccordionItem question="SSL 인증서 오류">

  - 스트라이프 터미널은 HTTPS가 필요합니다 - 사이트에 유효한 SSL 인증서가 있는지 확인
  - SSL 인증서가 올바르게 구성되어 있는지 확인

  </AccordionItem>
</Accordion>
