# 스트라이프 터미널 게이트웨이

스트라이프 터미널 게이트웨이를 사용하면 WooCommerce POS 내에서 스트라이프 터미널 하드웨어 리더기를 통해 대면 결제를 수락할 수 있습니다. 이 게이트웨이는 물리적 카드 리더기와 테스트용 시뮬레이터 모드를 모두 지원합니다.

## 기능

- **하드웨어 통합**: 인터넷 연결을 통해 물리적 스트라이프 터미널 리더기 연결
- **시뮬레이터 모드**: 하드웨어 없이 스트라이프 시뮬레이터로 결제 테스트
- **다양한 결제 방법**: 다양한 카드 유형 및 결제 방법 지원
- **실시간 처리**: 즉시 결제 처리 및 확인
- **안전한 트랜잭션**: 스트라이프를 통한 PCI 규정 준수 결제 처리
- **웹 기반 SDK**: 웹 애플리케이션 호환성을 위한 스트라이프 JavaScript SDK 사용

## 설치

### 1단계: 다운로드 및 설치

1. [스트라이프 터미널 for WooCommerce 릴리스 페이지](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases) 방문
2. 최신 **stripe-terminal-for-woocommerce.zip** 파일 다운로드
3. WordPress 관리자에서 `플러그인 > 새로 추가 > 플러그인 업로드`로 이동
4. zip 파일 업로드 및 플러그인 활성화

### 2단계: 스트라이프 설정 구성

1. `WP 관리자 > WooCommerce > 설정 > 결제`로 이동
2. 결제 방법 목록에서 **스트라이프 터미널** 찾기
3. **스트라이프 터미널**을 클릭하여 설정 접근
4. **스트라이프 비밀 키** 입력 (스트라이프 대시보드에서 확인 가능)
5. 설정 저장

:::note
WooCommerce 설정에서 스트라이프 터미널 게이트웨이를 활성화할 필요가 없습니다. 다음 단계에서 POS에 특별히 활성화됩니다.
:::

### 3단계: WooCommerce POS에서 활성화

1. `WP 관리자 > WooCommerce POS > 설정 > 체크아웃`로 이동
2. 목록에서 **스트라이프 터미널** 게이트웨이 찾기
3. POS에서 사용할 게이트웨이 활성화
4. 설정 저장

## 사용법

### 리더기 연결

POS 체크아웃에서 스트라이프 터미널 게이트웨이를 선택할 때:

1. **연결 방법 선택**: 물리적 리더기 연결 또는 시뮬레이터 사용 가능
2. **물리적 리더기**: 화면 지침에 따라 스트라이프 터미널 장치 연결
3. **시뮬레이터**: 하드웨어 없이 다양한 결제 시나리오 테스트를 위해 시뮬레이터 모드 선택

### 결제 처리

1. **상품 추가**: POS에서 장바구니에 상품 추가
2. **게이트웨이 선택**: 결제 방법으로 "스트라이프 터미널" 선택
3. **리더기 연결**: 리더기 연결 또는 시뮬레이터 모드 선택
4. **결제 처리**: 트랜잭션 완료를 위한 프롬프트 따라가기
5. **확인**: 성공적인 결제 시 주문 자동 완료

### 시뮬레이터로 테스트

시뮬레이터를 통해 다양한 결제 방법 및 시나리오 테스트 가능:

- **카드 결제**: 다양한 카드 유형 테스트 (비자, 마스터카드 등)
- **비접촉식 결제**: 터치 결제 트랜잭션 시뮬레이션
- **오류 시나리오**: 거절된 결제 및 기타 오류 조건 테스트
- **다양한 금액**: 다양한 트랜잭션 금액 테스트

## 요구 사항

- **스트라이프 계정**: 터미널이 활성화된 스트라이프 계정
- **API 키**: 대시보드의 스트라이프 비밀 키
- **WooCommerce POS**: 무료 또는 Pro 버전 설치 및 활성화
- **HTTPS**: 보안을 위해 SSL/HTTPS 사용 필요

## 하드웨어 호환성

:::important 연결 요구 사항
이 구현은 스트라이프의 JavaScript SDK를 사용하므로 웹 애플리케이션을 통해 작동하지만 **인터넷 연결된 터미널만** 필요합니다. 현재 블루투스 터미널은 지원되지 않습니다.
:::

### 지원되는 터미널 (인터넷 연결)

- **스트라이프 리더기 S700** ✅ - 이더넷/WiFi 연결 터미널
- **WisePOS E** ✅ - WiFi 연결 터미널

### 지원되지 않는 터미널 (블루투스)

- **BBPOS Chipper 2X BT** ❌ - 블루투스 전용
- **BBPOS WisePad 3** ❌ - 블루투스 전용  
- **Verifone P400** ❌ - 블루투스 전용

:::note 향후 지원
블루투스 터미널 지원은 **2026년 1분기**에 iOS 및 Android 애플리케이션에 추가될 예정입니다. 이를 통해 M2 및 WisePad 3를 포함한 모든 스트라이프 터미널 인증 리더기 지원이 가능해집니다.
:::

## 문제 해결

### 일반적인 문제

**리더기 연결 안 됨**
- 지원되는 인터넷 연결 터미널(S700 또는 WisePOS E) 사용 확인
- 터미널이 WiFi/이더넷에 연결되어 있고 온라인 상태인지 확인
- 스트라이프 계정에 터미널이 활성화되어 있는지 확인
- 터미널이 스트라이프 대시보드에 등록되어 있는지 확인

**결제 거절**
- 스트라이프 계정이 활성화되어 있고 정상 상태인지 확인
- 사용 중인 카드가 유효한지 확인
- 충분한 자금이 있는지 확인

**SSL 인증서 오류**
- 스트라이프 터미널은 HTTPS 필요 - 유효한 SSL 인증서가 있는지 확인
- SSL 인증서가 제대로 구성되어 있는지 확인

### 도움 받기
