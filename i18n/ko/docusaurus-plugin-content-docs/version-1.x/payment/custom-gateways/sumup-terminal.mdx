# SumUp 단말기 게이트웨이

SumUp 단말기 게이트웨이를 사용하면 WooCommerce POS 내에서 SumUp 카드 리더기를 사용하여 카드 결제를 받을 수 있습니다. 이 게이트웨이는 대면 거래를 위한 SumUp의 결제 처리 시스템과 원활하게 통합됩니다.

## 기능

- **하드웨어 통합**: 인터넷 연결을 통해 SumUp 카드 리더기를 POS 시스템에 연결
- **실시간 처리**: 즉각적인 결제 처리 및 주문 완료
- **자동 감지**: 결제가 감지되면 주문이 자동으로 완료됨
- **안전한 거래**: SumUp을 통한 PCI 준수 결제 처리
- **간편한 페어링**: 페어링 코드로 간단한 장치 페어링 과정
- **웹 기반 SDK**: SumUp의 JavaScript SDK를 사용하여 웹 애플리케이션 호환성 제공

## 설치

### 1단계: 다운로드 및 설치

1. [WooCommerce용 SumUp 단말기 릴리스 페이지](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)를 방문
2. 최신 **sumup-terminal-for-woocommerce.zip** 파일 다운로드
3. WordPress 관리에서 `플러그인 > 새로 추가 > 플러그인 업로드`로 이동
4. zip 파일을 업로드하고 플러그인 활성화

### 2단계: SumUp 설정 구성

1. `WP 관리자 > WooCommerce > 설정 > 결제`로 이동
2. 결제 방법 목록에서 **SumUp 단말기** 찾기
3. **SumUp 단말기**를 클릭하여 설정에 접근
4. **SumUp API 키** 입력 (SumUp 상인 대시보드에서 확인 가능)
5. 설정 저장

:::note
WooCommerce 설정에서 SumUp 단말기 게이트웨이를 활성화할 필요는 없습니다. 나중 단계에서 POS에 대해 별도로 활성화됩니다.
:::

### 3단계: SumUp 단말기 페어링

1. 동일한 설정 페이지에서 **리더 페어링** 섹션 찾기
2. SumUp 장치에서 페어링 화면으로 이동하여 페어링 코드 표시
3. SumUp 장치에 표시된 페어링 코드 입력
4. **"리더 페어링"** 클릭하여 연결 설정
5. 리더가 성공적으로 페어링 되었음을 확인

:::warning 중요
결제를 처리하기 전에 리더가 성공적으로 페어링되어야 합니다. 진행하기 전에 페어링 과정이 완료되었는지 확인하십시오.
:::

### 4단계: WooCommerce POS에서 활성화

1. `WP 관리자 > WooCommerce POS > 설정 > 결제`로 이동
2. 목록에서 **SumUp 단말기** 게이트웨이를 찾기
3. POS에서 사용하기 위해 게이트웨이 활성화
4. 설정 저장

## 사용

### 결제 처리

1. **아이템 추가**: POS에 제품을 장바구니에 추가
2. **게이트웨이 선택**: 결제 방법으로 "SumUp 단말기" 선택
3. **결제 시작**: SumUp 장치에서 새 결제를 시작하도록 클릭
4. **고객 결제**: 고객이 SumUp 단말기에서 결제를 완료
5. **자동 완료**: 결제가 감지되면 주문이 자동으로 완료

### 결제 제어

SumUp 단말기 게이트웨이를 사용할 때 다음 옵션이 있습니다:

- **새 결제 시작**: 연결된 단말기에서 결제 요청 시작
- **결제 취소**: 현재 진행 중인 결제 취소
- **결제 상태**: 결제 프로세스의 현재 상태 모니터링

### 주문 관리

- **자동 업데이트**: 결제가 성공적으로 이루어지면 주문이 자동으로 완료로 표시됨
- **결제 추적**: 모든 결제 세부정보가 주문 메모에 기록됨
- **영수증 생성**: 성공적인 결제 후 표준 POS 영수증이 생성됨

## 요구 사항

- **SumUp 계정**: 활성 SumUp 상인 계정
- **API 접근**: 상인 대시보드에서 SumUp API 키
- **호환 가능 하드웨어**: SumUp 카드 리더기 장치
- **WooCommerce POS**: 무료 또는 Pro 버전이 설치 및 활성화되어 있어야 함
- **안정적인 연결**: API 통신을 위한 신뢰할 수 있는 인터넷 연결

## 하드웨어 호환성

:::important 연결 요구 사항
이 구현은 SumUp의 JavaScript SDK를 사용하므로 웹 애플리케이션을 통해 작동하지만 **인터넷에 연결된 단말기만** 필요합니다. Bluetooth 단말기는 현재 지원되지 않습니다.
:::

### 지원되는 단말기 (인터넷 연결)

- **SumUp Solo** ✅ - WiFi/Ethernet 연결 단말기

### 지원되지 않는 단말기 (Bluetooth)

- **SumUp Air** ❌ - Bluetooth 전용
- **SumUp 3G** ❌ - Bluetooth 전용

:::note 향후 지원
Bluetooth 단말기 지원은 **2026년 1분기** iOS 및 Android 애플리케이션에 추가될 예정입니다. 이를 통해 Air 및 3G 모델을 포함한 모든 SumUp 인증 단말기에 대한 지원이 가능해집니다.
:::

## 문제 해결

### 일반적인 문제

**리더 페어링 실패**
- 지원되는 인터넷 연결 단말기(Solo 전용)를 사용하고 있는지 확인
- SumUp Solo 장치가 WiFi에 연결되고 온라인인지 확인
- 장치가 페어링 모드에 있고 페어링 코드가 올바르게 입력되었는지 확인
- SumUp 계정에 API 접근이 활성화되어 있는지 확인
- 페어링 중 안정적인 인터넷 연결을 확보

**결제 처리 실패**
- 리더가 제대로 페어링되었는지 확인
- SumUp 계정이 활성화되어 있고 정상적으로 운영 중인지 확인
- 장치에 충분한 배터리와 연결 상태인지 확인
- API 키가 올바르고 활성화되어 있는지 확인

**주문 완료 실패**
- 인터넷 연결의 안정성을 확인
- SumUp 계정의 웹훅 설정 확인
- SumUp 단말기에서 결제가 성공적으로 이루어졌는지 확인
- API 통신 문제에 대한 WordPress 오류 로그 확인

### API 속도 제한

게이트웨이는 API 호출을 줄이고 속도 제한을 피하도록 최적화되어 있습니다:
- 결제 상태가 효율적으로 확인됨
- 불필요한 API 요청이 최소화됨
- 임시 실패를 처리하는 자동 재시도 로직
