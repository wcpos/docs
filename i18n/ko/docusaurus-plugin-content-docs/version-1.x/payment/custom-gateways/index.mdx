# 사용자 지정 결제 게이트웨이 {#custom-payment-gateways}

WCPOS는 POS 시스템의 기능을 확장하는 여러 사용자 지정 결제 게이트웨이를 지원합니다. 이러한 게이트웨이는 POS 환경에 맞게 특별히 설계되었으며 다양한 결제 수단 및 서비스와의 원활한 통합을 제공합니다.

## 사용 가능한 사용자 지정 게이트웨이 {#available-custom-gateways}

### 하드웨어 단말기 게이트웨이 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminal 하드웨어 리더기를 사용하여 결제를 받습니다
- **[SumUp Terminal](./sumup-terminal)** - SumUp 카드 리더기를 통해 결제를 처리합니다

### 모바일 결제 게이트웨이 {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QR 코드 또는 푸시 알림을 통해 휴대폰 기반 결제를 받습니다

### 디지털 결제 게이트웨이 {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - 이메일을 통해 고객에게 결제 송장을 발송합니다
- **[웹 결제](./web-checkout)** - 고객을 온라인 결제 페이지로 리디렉션합니다

## 나만의 게이트웨이 만들기 {#creating-your-own-gateway}

특정 요구 사항에 맞는 사용자 지정 결제 게이트웨이가 필요한 경우, **[게이트웨이 템플릿](./gateway-template)** 을 사용하여 자신만의 WCPOS 결제 게이트웨이를 만들 수 있습니다.

## 설치 개요 {#installation-overview}

모든 사용자 지정 게이트웨이는 유사한 설치 절차를 따릅니다:

1. 해당 GitHub 저장소에서 최신 릴리스를 **다운로드**합니다
2. `WP Admin > Plugins > Add New > Upload Plugin` 을 통해 플러그인을 **설치**합니다
3. `WP Admin > WooCommerce > Settings > Payments` 에서 게이트웨이 설정을 **구성**합니다
4. `WP Admin > POS > Settings > Checkout` 에서 게이트웨이를 **활성화**합니다
