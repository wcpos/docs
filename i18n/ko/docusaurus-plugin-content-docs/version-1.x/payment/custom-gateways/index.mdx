# 사용자 정의 결제 게이트웨이

WCPOS는 포인트 오브 세일 시스템의 기능을 확장하는 여러 사용자 정의 결제 게이트웨이를 지원합니다. 이러한 게이트웨이는 POS 환경에 특별히 설계되어 다양한 결제 방법 및 서비스와 원활하게 통합됩니다.

## 사용 가능한 사용자 정의 게이트웨이

### 하드웨어 터미널 게이트웨이

- **[Stripe 터미널](./stripe-terminal)** - Stripe 터미널 하드웨어 리더기를 사용하여 결제 수락
- **[SumUp 터미널](./sumup-terminal)** - SumUp 카드 리더기를 통해 결제 처리

### 디지털 결제 게이트웨이

- **[이메일 인보이스](./email-invoice)** - 고객에게 이메일로 결제 인보이스 발송
- **[웹 결제](./web-checkout)** - 고객을 온라인 결제 완료 페이지로 리다이렉트

## 고유한 게이트웨이 만들기

특정 요구 사항에 맞는 사용자 정의 결제 게이트웨이가 필요한 경우, **[게이트웨이 템플릿](./gateway-template)**을 사용하여 고유한 WooCommerce POS 결제 게이트웨이를 생성할 수 있습니다.

## 설치 개요

모든 사용자 정의 게이트웨이는 비슷한 설치 프로세스를 따릅니다:

1. **다운로드** 해당 GitHub 저장소에서 최신 릴리스
2. **설치** `WP 관리자 > 플러그인 > 새로 추가 > 플러그인 업로드`를 통해 플러그인
3. **구성** `WP 관리자 > WooCommerce > 설정 > 결제`에서 게이트웨이 설정
4. **활성화** `WP 관리자 > WooCommerce POS > 설정 > 체크아웃`에서 게이트웨이

## 요구 사항

- WooCommerce POS 플러그인 (무료 또는 프로 버전)
- WooCommerce가 설치된 WordPress
- 특정 결제 서비스에 대한 적절한 API 키 또는 자격 증명
