# 사용자 정의 결제 게이트웨이 {#custom-payment-gateways}

WCPOS는 판매 시점 시스템의 기능을 확장하는 여러 사용자 정의 결제 게이트웨이를 지원합니다. 이 게이트웨이는 POS 환경에 맞게 설계되어 있으며 다양한 결제 방법 및 서비스와 원활하게 통합됩니다.

## 사용 가능한 사용자 정의 게이트웨이 {#available-custom-gateways}

### 하드웨어 단말기 게이트웨이 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminal 하드웨어 리더기를 이용하여 결제 수락
- **[SumUp Terminal](./sumup-terminal)** - SumUp 카드 리더기를 통해 결제 처리

### 모바일 결제 게이트웨이 {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QR 코드 또는 푸시 알림을 통해 전화 기반 결제 수락

### 디지털 결제 게이트웨이

- **[이메일 인보이스](./email-invoice)** - 고객에게 이메일로 결제 인보이스 전송
- **[웹 체크아웃](./web-checkout)** - 고객을 온라인 결제 완료 페이지로 리디렉션

## 나만의 게이트웨이 만들기 {#creating-your-own-gateway}

특정 요구 사항에 맞는 사용자 정의 결제 게이트웨이가 필요한 경우, **[게이트웨이 템플릿](./gateway-template)**을 사용하여 나만의 WooCommerce POS 결제 게이트웨이를 만들 수 있습니다.

## 설치 개요 {#installation-overview}

모든 사용자 정의 게이트웨이는 유사한 설치 프로세스를 따릅니다:

1. **최신 릴리스를** 해당 GitHub 저장소에서 다운로드
2. **플러그인을** `WP Admin > 플러그인 > 새로 추가 > 플러그인 업로드`를 통해 설치
3. **게이트웨이 설정을** `WP Admin > WooCommerce > 설정 > 결제`에서 구성
4. **게이트웨이를** `WP Admin > WooCommerce POS > 설정 > 체크아웃`에서 활성화
