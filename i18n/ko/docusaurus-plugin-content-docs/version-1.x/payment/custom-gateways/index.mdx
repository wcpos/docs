# 사용자 정의 결제 게이트웨이 {#custom-payment-gateways}

WCPOS는 귀하의 판매 시점 시스템 기능을 확장하는 여러 사용자 정의 결제 게이트웨이를 지원합니다. 이러한 게이트웨이는 POS 환경에 맞게 설계되었으며 다양한 결제 방법 및 서비스와 원활하게 통합됩니다.

## 사용 가능한 사용자 정의 게이트웨이 {#available-custom-gateways}

### 하드웨어 터미널 게이트웨이 {#hardware-terminal-gateways}

- **[Stripe 터미널](./stripe-terminal)** - Stripe 터미널 하드웨어 리더기를 사용하여 결제를 수락합니다
- **[SumUp 터미널](./sumup-terminal)** - SumUp 카드 리더기를 통해 결제를 처리합니다

### 모바일 결제 게이트웨이 {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QR 코드 또는 푸쉬 알림을 통해 전화 기반 결제를 수락합니다

### 디지털 결제 게이트웨이 {#digital-payment-gateways}

- **[이메일 송장](./email-invoice)** - 고객에게 이메일로 결제 송장을 발송합니다
- **[웹 체크아웃](./web-checkout)** - 고객을 리디렉션하여 온라인에서 결제를 완료합니다

## 자체 게이트웨이 생성 {#creating-your-own-gateway}

특정 요구 사항에 맞는 사용자 정의 결제 게이트웨이가 필요한 경우, **[게이트웨이 템플릿](./gateway-template)** 을 사용하여 자신의 WCPOS 결제 게이트웨이를 생성할 수 있습니다.

## 설치 개요 {#installation-overview}

모든 사용자 정의 게이트웨이는 유사한 설치 프로세스를 따릅니다:

1. **최신 릴리스를** 해당 GitHub 저장소에서 다운로드합니다
2. **플러그인을** `WP Admin > Plugins > Add New > Upload Plugin`를 통해 설치합니다
3. **게이트웨이 설정을** `WP Admin > WooCommerce > Settings > Payments`에서 구성합니다
4. **게이트웨이를** `WP Admin > POS > Settings > Checkout`에서 활성화합니다
