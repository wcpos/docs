# 맞춤 결제 게이트웨이 {#custom-payment-gateways}

WCPOS는 판매 시점 시스템의 기능을 확장하는 여러 맞춤 결제 게이트웨이를 지원합니다. 이러한 게이트웨이는 POS 환경에 맞게 설계되었으며, 다양한 결제 방법 및 서비스와 원활하게 통합됩니다.

## 사용 가능한 맞춤 게이트웨이 {#available-custom-gateways}

### 하드웨어 단말기 게이트웨이 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminal 하드웨어 리더기를 사용하여 결제 수락
- **[SumUp Terminal](./sumup-terminal)** - SumUp 카드 리더기를 통해 결제 처리

### 디지털 결제 게이트웨이 {#digital-payment-gateways}

- **[이메일 송장](./email-invoice)** - 고객에게 이메일로 결제 송장 전송
- **[웹 체크아웃](./web-checkout)** - 고객을 온라인 결제 완료로 리디렉션

## 나만의 게이트웨이 만들기 {#creating-your-own-gateway}

특별한 요구 사항을 위한 맞춤 결제 게이트웨이가 필요한 경우, **[게이트웨이 템플릿](./gateway-template)**을 사용하여 나만의 WooCommerce POS 결제 게이트웨이를 만들 수 있습니다.

## 설치 개요 {#installation-overview}

모든 맞춤 게이트웨이는 유사한 설치 프로세스를 따릅니다:

1. **해당 GitHub 리포지토리**에서 최신 릴리스를 다운로드합니다.
2. `WP Admin > 플러그인 > 새로 추가 > 플러그인 업로드`를 통해 플러그인을 설치합니다.
3. `WP Admin > WooCommerce > 설정 > 결제`에서 게이트웨이 설정을 구성합니다.
4. `WP Admin > WooCommerce POS > 설정 > 체크아웃`에서 게이트웨이를 활성화합니다.

## 지원 {#support}

맞춤 게이트웨이에 문제가 발생한 경우, 해당 GitHub 리포지토리를 방문하고 문제에 대한 자세한 정보를 포함하여 이슈를 생성해 주십시오.
