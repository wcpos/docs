WCPOS는 다양한 비즈니스 요구를 수용하기 위해 여러 결제 방법을 지원합니다.

## 기본 결제 방법

무료 버전에는 두 개의 내장 결제 게이트웨이가 포함되어 있습니다:

### 현금

- 지불한 금액 입력
- 자동 잔돈 계산
- 추가 구성 불필요

### 카드

- 외부 카드 단말기와 함께 사용
- 단말기에서 처리 후 결제를 완료로 표시하기만 하면 됩니다
- 직접 통합 필요 없음

## 추가 결제 게이트웨이 (Pro)

:::info Pro 기능
추가 결제 게이트웨이는 [WCPOS Pro](/getting-started/pro-license)를 요구합니다.
:::

Pro를 사용하면 POS 체크아웃에서 추가 WooCommerce 결제 게이트웨이를 활성화할 수 있습니다:

- **Stripe Terminal** - Stripe 카드 리더와의 직접 통합
- **SumUp Terminal** - SumUp 카드 리더와의 통합
- **모든 WooCommerce 게이트웨이** - 상점에 설치된 모든 게이트웨이 활성화

## 맞춤형 게이트웨이

[맞춤형 게이트웨이](/payment/custom-gateways) 시스템을 사용하여 자신만의 결제 통합을 생성하십시오.
