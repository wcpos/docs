WCPOS는 다양한 비즈니스 요구에 맞는 여러 결제 방법을 지원합니다.

## 기본 결제 방법

무료 버전에는 다음 두 가지 기본 결제 게이트웨이가 포함되어 있습니다:

### 현금

- 지불 금액 입력
- 자동 잔돈 계산
- 추가 구성 불필요

### 카드

- 외부 카드 단말기와 함께 사용
- 단말기에서 결제 처리 후 결제 완료로 표시
- 직접 연동 불필요

## 추가 결제 게이트웨이 (Pro)

:::info Pro 기능
추가 결제 게이트웨이는 [WCPOS Pro](/getting-started/pro-license)가 필요합니다.
:::

Pro 버전에서는 POS 결제에서 다음과 같은 WooCommerce 결제 게이트웨이를 활성화할 수 있습니다:

- **Stripe 단말기** - Stripe 카드 리더와 직접 연동
- **SumUp 단말기** - SumUp 카드 리더와 연동
- **모든 WooCommerce 게이트웨이** - 스토어에 설치된 모든 게이트웨이 활성화

## 맞춤형 게이트웨이

[맞춤형 게이트웨이](/payment/custom-gateways) 시스템을 사용하여 고유한 결제 연동을 만들 수 있습니다.
