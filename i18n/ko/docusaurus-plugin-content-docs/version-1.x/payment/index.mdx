WCPOS는 다양한 비즈니스 요구를 수용하기 위해 여러 결제 방법을 지원합니다.

## 기본 결제 방법

무료 버전에는 두 개의 기본 결제 게이트웨이가 포함되어 있습니다:

### 현금

- 받을 금액 입력
- 자동 잔돈 계산
- 추가 설정 필요 없음

### 카드

- 외부 카드 단말기와 함께 사용
- 단말기에서 처리 후 결제를 완료로 표시
- 직접 통합 필요 없음

## 추가 결제 게이트웨이(프로)

:::info 프로 기능
추가 결제 게이트웨이는 [WCPOS Pro](/getting-started/pro-license)를 요구합니다.
:::

Pro 버전을 사용하면 POS 체크아웃에서 추가 WooCommerce 결제 게이트웨이를 활성화할 수 있습니다:

- **Stripe 단말기** - Stripe 카드 읽기기와의 직접 통합
- **SumUp 단말기** - SumUp 카드 읽기기와의 통합
- **모든 WooCommerce 게이트웨이** - 매장에 설치된 모든 게이트웨이 활성화

## 사용자 정의 게이트웨이

[사용자 정의 게이트웨이](/payment/custom-gateways) 시스템을 사용하여 자체 결제 통합을 생성하십시오.
