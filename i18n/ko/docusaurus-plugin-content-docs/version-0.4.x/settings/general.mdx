### 상품 표시 설정

POS 전용 상품은 온라인 또는 매장에서 특정 상품을 숨겨야 하는 경우에만 활성화해야 합니다.
자세한 내용은 [POS 전용 상품](../products/pos-only-products) 문서를 참조하세요.

### 기본 POS 고객

새 주문은 기본 POS 고객으로 초기화됩니다. 일반적인 옵션은 다음과 같습니다:

- **손님:** POS 주문이 사용자 ID와 연결되지 않음
- **계산원:** POS 주문이 로그인한 사용자 계정과 연결됨
- **방문 고객:** POS 주문이 사용자 정의 계정과 연결됨 (_참고: 이 계정을 먼저 생성해야 함_)

[WCPOS Pro](https://wcpos.com/pro) 사용자는 체크아웃 중 즉시 사용자 계정을 생성할 수 있습니다.
새 계정을 생성하려면 이메일 주소가 필요하며, 이는 WordPress의 요구 사항입니다.
**새 고객을 위한 계정 생성이 권장됩니다** 고객의 주문 이력을 추적하고 향후 프로모션 제안을 보낼 수 있기 때문입니다.
