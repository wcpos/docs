POS 접근 설정을 통해 스토어 소유자는 포인트 오브 세일 시스템에 대한 사용자 권한을 부여하거나 제거할 수 있습니다.
기본적으로, **관리자**와 **상점 관리자** 역할은 POS를 사용하는 모든 기능이 활성화되어 있습니다.
결과를 잘 알고 있지 않다면 기본 설정을 유지하는 것이 좋습니다.
예를 들어, `구독자` 역할에 `publish_shop_orders`를 부여하는 것은 _매우_ 나쁜 생각입니다.

![POS 접근 설정 예시](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "POS 접근 설정 예시")

| 기능                                                                    | 접근                                      |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | POS 프론트엔드 접근에 필요                |
| manage_woocommerce_pos                                                  | POS 관리 설정 접근에 필요                 |
| read_private_products                                                   | 상품 보기에 필요                          |
| edit_product <br /> edit_published_products <br /> edit_others_products | 상품 및 변형 편집에 필요                  |
| read_private_shop_orders                                                | 주문 및 영수증 보기에 필요                |
| publish_shop_orders                                                     | 새 주문 생성에 필요                       |
| list_users                                                              | 고객 보기에 필요                          |
| create_users                                                            | 새 고객 생성에 필요                       |
| edit_users                                                              | 기존 고객 업데이트에 필요                 |
| read_private_shop_coupons                                               | 쿠폰 보기에 필요                          |
