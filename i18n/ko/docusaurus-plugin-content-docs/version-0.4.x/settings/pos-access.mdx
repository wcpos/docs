POS 접근 설정을 사용하여 상점 소유자는 POS 시스템에 대한 사용자 권한을 부여하거나 제거할 수 있습니다.
기본적으로, **관리자** 및 **상점 관리자** 역할은 POS를 사용하는 모든 기능을 사용할 수 있도록 활성화되어 있습니다.
결과를 알고 있지 않은 경우 기본 설정을 유지하는 것이 좋습니다.
예를 들어, `publish_shop_orders`를 `구독자` 역할에 부여하는 것은 _매우_ 나쁜 생각일 수 있습니다.

![POS 접근 설정의 예](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "POS 접근 설정의 예")

| 기능                                                                  | 접근                                     |
| --------------------------------------------------------------------- | ---------------------------------------- |
| access_woocommerce_pos                                                | POS 프론트엔드에 접근하려면 필요          |
| manage_woocommerce_pos                                                | POS 관리자 설정에 접근하려면 필요         |
| read_private_products                                                 | 제품을 보기 위해 필요                    |
| edit_product <br /> edit_published_products <br /> edit_others_products | 제품 및 변형을 편집하려면 필요             |
| read_private_shop_orders                                              | 주문 및 영수증을 보기 위해 필요           |
| publish_shop_orders                                                   | 새 주문을 생성하려면 필요                 |
| list_users                                                            | 고객을 보기 위해 필요                    |
| create_users                                                          | 새로운 고객을 만들기 위해 필요           |
| edit_users                                                            | 기존 고객을 업데이트하기 위해 필요        |
| read_private_shop_coupons                                             | 쿠폰을 보기 위해 필요                    |
