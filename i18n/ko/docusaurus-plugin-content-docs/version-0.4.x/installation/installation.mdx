### 자동 설치 (권장) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "1분도 안 되어 WCPOS를 설치하고 활성화합니다")

1. `WP 관리자 > 플러그인 > 새로 추가`로 이동
2. WordPress 플러그인 디렉토리에서 `WCPOS` 검색
3. 플러그인 설치
4. 플러그인 활성화

### 수동 설치 {#manual-installation}

1. [WCPOS 다운로드](https://wordpress.org/plugins/woocommerce-pos/) WordPress.org에서
2. `WP 관리자 > 플러그인 > 새로 추가 > 플러그인 업로드`로 이동
3. 플러그인 활성화

:::info
이전 버전의 WCPOS는 WordPress.org의 [개발자](https://wordpress.org/plugins/woocommerce-pos/developers/) 탭에서 찾을 수 있습니다.
:::
