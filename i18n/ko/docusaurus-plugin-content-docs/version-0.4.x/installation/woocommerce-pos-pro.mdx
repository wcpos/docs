---

:::important

[WCPOS 프로](http://wcpos.com/pro)는 무료 [WCPOS 플러그인](http://wordpress.org/plugins/woocommerce-pos/)에 기능을 추가하는 유료 플러그인입니다. 

:::

WCPOS 프로 라이센스를 구매하셨다면 다음 단계를 따라 플러그인을 설치하고 활성화해 주세요:

1.  다음 페이지로 이동: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  **다운로드** 항목에서 다운로드 링크를 클릭하고 플러그인을 데스크톱에 저장합니다.
3.  사이트로 이동하여 로그인한 후 `WP 관리자 > 플러그인 > 새로 추가 > 플러그인 업로드`로 이동합니다.
4.  데스크톱의 플러그인 zip 파일을 업로드하고 활성화합니다.
5.  다음으로, `WP 관리자 > POS > 설정 > 라이센스`로 이동하여 라이센스 키와 라이센스 이메일을 입력하여 활성화를 완료합니다.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "계정 페이지에서 WCPOS 프로를 다운로드할 수 있습니다")

새 버전이 준비되면 업데이트 대시보드에 표시됩니다. 
현재 작업 중인 내용에 대한 정보는 [로드맵](http://wcpos.com/roadmap/)을 참고해 주세요. 
또한 각 버전의 새로운 기능을 나열하는 변경 로그를 지켜보시고, [문서 페이지](http://docs.wcpos.com)는 새로운 기능 사용 방법에 대한 정보로 업데이트될 예정입니다.
