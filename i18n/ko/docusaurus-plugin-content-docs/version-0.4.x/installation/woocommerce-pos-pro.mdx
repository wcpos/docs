:::important

[WCPOS Pro](http://wcpos.com/pro)는 무료 [WCPOS 플러그인](http://wordpress.org/plugins/woocommerce-pos/)에 기능을 추가하는 유료 플러그인입니다.

:::

WCPOS Pro 라이센스를 구매한 경우 아래 단계를 따라 플러그인을 설치하고 활성화하십시오:

1.  다음으로 이동: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  **다운로드** 섹션에서 다운로드 링크를 클릭하고 플러그인을 데스크탑에 저장하십시오.
3.  그런 다음 사이트에 로그인하고 `WP Admin > 플러그인 > 새로 추가 > 플러그인 업로드`로 이동하십시오.
4.  데스크탑에서 플러그인 zip 파일을 업로드하고 활성화하십시오.
5.  다음으로 `WP Admin > POS > 설정 > 라이센스`로 이동하여 라이센스 키와 라이센스 이메일을 입력하여 활성화를 완료하십시오.

![내 다운로드](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "계정 페이지에서 WCPOS Pro를 다운로드할 수 있습니다.")

새로운 버전이 준비되면 업데이트 대시보드에 나타납니다. 
현재 진행 중인 작업에 대한 정보는 [로드맵](http://wcpos.com/roadmap/)을 확인하십시오. 
또한 각 버전의 새로운 기능을 나열하는 변경 로그를 주시하고 [문서 페이지](http://docs.wcpos.com)는 새로운 기능 사용 방법에 대한 정보로 업데이트될 것입니다.
