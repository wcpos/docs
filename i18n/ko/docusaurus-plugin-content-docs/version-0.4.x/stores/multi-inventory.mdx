:::caution
다중 재고는 현재 Pro 플러그인의 기능이 아닙니다.
다음 정보는 WCPOS Pro가 공식적으로 다중 재고를 지원할 때까지 사용자를 지원하기 위한 것입니다.
:::

다중 재고 매장은 WCPOS Pro에서 가장 많이 요청되는 기능 중 하나입니다.
이 기능을 통해 매장 소유자는 제품을 다른 매장에 할당하고 각 물리적 매장과 온라인 매장 간의 재고를 관리할 수 있습니다.

다중 재고 기능을 만드는 것은 간단한 작업이 아닙니다.
이미 이 문제를 해결하고 있는 몇 가지 플러그인이 있습니다. 예를 들어 [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646)와 [WooCommerce MultiStore](http://woomultistore.com/)가 있습니다.
현재는 다른 핵심 POS 기능 개발에 집중하고, 이러한 플러그인이 안정적인 솔루션으로 성숙해지면 통합하는 것이 합리적입니다.

단기적으로는 [WordPress MultiSite 기능](https://codex.wordpress.org/Create_A_Network)을 사용하여 대부분의 요구 사항을 충족할 수 있습니다.
각 매장에 대해 사이트를 생성한 후 각 사이트에 WooCommerce와 WCPOS를 설치하면 각 매장의 재고를 완전히 분리할 수 있습니다.
위에서 언급한 WooCommerce Multisite는 사이트 간 재고를 관리하는 추가 계층과 함께 이 접근 방식을 사용합니다.
