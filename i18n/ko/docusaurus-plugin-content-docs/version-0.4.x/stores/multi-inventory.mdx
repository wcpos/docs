---
title: 다중 재고 매장
---

:::caution
다중 재고는 현재 Pro 플러그인의 기능이 아닙니다.
다음 정보는 WCPOS Pro가 공식적으로 다중 재고를 지원할 때까지 사용자에게 도움이 될 수 있도록 작성되었습니다.
:::

다중 재고 매장은 WCPOS Pro에서 가장 많이 요청된 기능 중 하나입니다.
이 기능은 매장 소유자가 제품을 서로 다른 매장에 할당하고 각 물리적 매장과 온라인 매장 간의 재고를 관리할 수 있게 합니다.

다중 재고 기능을 만드는 것은 간단하지 않은 작업입니다.
현재 [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646)와 [WooCommerce MultiStore](http://woomultistore.com/)와 같은 몇 가지 플러그인이 이 문제를 해결하기 위해 작업 중입니다.
현재는 다른 핵심 POS 기능에 대한 개발에 집중하고, 안정적인 솔루션으로 성숙해지면 이들 플러그인 중 하나와 통합하는 것이 합리적입니다.

단기적으로는 [WordPress 멀티사이트 기능](https://codex.wordpress.org/Create_A_Network)을 사용하여 대부분의 요구 사항을 충족할 수 있습니다.
각 매장에 대한 사이트를 생성하면 각 사이트에 WooCommerce와 WCPOS를 설치하여 각 매장의 재고를 완전히 별도로 유지할 수 있습니다.
위에서 언급한 WooCommerce Multisite는 사이트 간 재고를 관리하기 위한 추가 계층을 추가하여 이 접근 방식을 사용합니다.
