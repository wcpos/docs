### 기능

캐셔는 모든 고객 기능을 사용하기 위해 _list_users_, _create_users_ 및 _edit_users_ 기능이 필요합니다.

### 워드프레스 멀티사이트

[워드프레스 멀티사이트](https://codex.wordpress.org/Create_A_Network)를 사용할 때, 비 [슈퍼 관리자](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin)가 고객을 생성하고 편집하는 것을 방지하기 위해 몇 가지 [추가 점검](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php)이 있습니다.

먼저, 비 슈퍼 관리자가 새 고객을 **생성**하려면 `Network Admin > Settings`에서 다음 옵션을 활성화해야 합니다.

![새 사용자를 생성하는 멀티사이트 옵션](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "새 사용자를 생성하는 멀티사이트 옵션")

추가로, 비 슈퍼 관리자가 고객 세부 정보를 **편집**할 수 있도록 하려면 캐셔에게 `manage_network_users` 기능을 추가해야 합니다.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
