### 기능 권한

캐셔는 모든 고객 기능을 사용하기 위해 _list_users_, _create_users_, _edit_users_ 권한이 필요합니다.

### WordPress 다중 사이트

[WordPress 다중 사이트](https://codex.wordpress.org/Create_A_Network)를 사용할 때는 비 [슈퍼 관리자](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin)가 고객을 생성 및 수정하는 것을 방지하기 위한 [추가 확인](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php)이 있습니다.

먼저, 비 슈퍼 관리자가 새 고객을 **생성**할 수 있도록 `네트워크 관리자 > 설정`에서 다음 옵션을 활성화하세요.

![새 사용자 생성을 위한 다중 사이트 옵션](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "새 사용자 생성을 위한 다중 사이트 옵션")

또한, 비 슈퍼 관리자가 고객 세부 정보를 **수정**할 수 있도록 캐셔에게 `manage_network_users` 권한을 추가해야 합니다.

```php title="functions.php"
// CASHIER_ID를 변경하고
// functions.php 파일에 추가하세요

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
