---

WooCommerce 관리자에서 왼쪽 상단의 빠른 링크를 사용하여 POS 주문과 온라인 주문을 간편하게 구분할 수 있습니다.
**참고:** 이 기능은 버전 0.3.1 이후의 POS 주문만 필터링하며, 0.3.1 이전의 주문은 기본적으로 온라인으로 표시됩니다.

![주문 필터링](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

또한, 대기 중/처리 중인 주문에 집중할 수 있도록 관리자 영역에서 완료된 주문을 모두 필터링하려면 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 파일에 다음 코드를 추가하면 됩니다.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
