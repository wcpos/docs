---

WooCommerce 관리자의 왼쪽 상단에 있는 빠른 링크를 사용하여 POS와 온라인 주문을 빠르게 분리할 수 있습니다.
**참고:** 이 필터는 0.3.1 버전 이후의 POS 주문만 필터링하며, 0.3.1 이전의 주문은 기본적으로 온라인으로 표시됩니다.

![주문 필터](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "주문 필터")

추가로, 관리자 영역에서 완료된 주문을 필터링하여 대기 중/처리 중인 주문에 집중하고 싶다면 다음 코드를 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 파일에 추가할 수 있습니다.

```php title="functions.php"
/**
 * WP 관리자의 주문 기본 필터 변경
 * functions.php 파일에 추가
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
