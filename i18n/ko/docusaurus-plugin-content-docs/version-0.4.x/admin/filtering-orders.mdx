---

WooCommerce 관리에서 왼쪽 상단의 빠른 링크를 사용하여 POS 및 온라인 주문을 신속하게 분리할 수 있습니다.
**참고:** 이것은 버전 0.3.1 이후의 POS 주문만 필터링합니다. 0.3.1 이전의 주문은 기본적으로 온라인으로 설정됩니다.

![주문 필터](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "주문 필터")

또한, 관리 영역에서 모든 완료된 주문을 필터링하여 보류 중인/처리 중인 주문에 집중하고 싶다면 다음 코드를 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 파일에 추가할 수 있습니다.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
