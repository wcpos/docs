POS 영수증 인쇄 템플릿은 무료 및 Pro 플러그인의 `includes/views/print/tmpl-receipt.php` 파일에 있습니다. 영수증 템플릿은 테마 디렉터리에 `woocommerce-pos/print/tmpl-receipt.php` 파일을 생성하여 사용자 정의할 수 있습니다. 두 템플릿의 코드가 아래에 포함되어 있습니다.

### Pro 영수증 템플릿

Pro 영수증 템플릿은 [스토어](./stores) 관리자의 로고, 스토어 주소, 영업 시간 및 특별 메시지와 같은 추가 정보를 허용합니다.

### 영수증 날짜 사용자 정의

WCPOS는 날짜 문자열을 현지화하기 위해 [moment.js](http://momentjs.com/)를 사용합니다. 기본 날짜 형식은 `"MMMM Do YYYY, h:mm:ss a"`입니다. 예: 2015년 5월 31일, 오후 7:20:44. 날짜 형식 사용자 정의에 대한 자세한 내용은 [moment.js 문서](http://momentjs.com/docs/#/parsing/string-format/)를 참조하세요.

### 주문 속성

WCPOS는 WC REST API의 JSON 출력을 사용하여 주문 영수증 템플릿을 채웁니다. [WC REST API 문서에서 표준 JSON 출력 예제](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order)를 확인할 수 있습니다. `woocommerce_api_order_response` 필터를 사용하여 WCPOS에서 추가 속성을 추가했습니다.

| 속성                             | 설명                                                         |
| --------------------------------- | ------------------------------------------------------------ |
| cart_discount_tax                 | 장바구니 할인의 세금 부분                                   |
| cashier.id                        | 캐셔의 사용자 ID                                             |
| cashier.first_name                | 캐셔의 이름                                                 |
| cashier.last_name                 | 캐셔의 성                                                   |
| payment_details.result            | 결제 게이트웨이 성공 또는 실패                              |
| payment_details.message           | 결제 게이트웨이의 메시지                                    |
| payment_details.redirect          | 오프사이트 결제를 위한 리디렉션 URL, 예: PayPal 표준         |
| payment_details.method\_`gateway` | 게이트웨이별 메시지, 예: 지불 금액 및 거스름돈               |
| shipping_lines[i].total_tax       | 각 배송 라인의 세금 금액 추가                               |
| subtotal_tax                      | 소계의 세금 부분                                            |
