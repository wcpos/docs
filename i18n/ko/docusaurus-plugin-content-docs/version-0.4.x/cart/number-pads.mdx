![상품 가격 숫자 키패드로 할인 비율의 빠른 입력을 제공합니다.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "상품 가격 숫자 키패드로 할인 비율의 빠른 입력을 제공합니다.")

![지불 금액 숫자 키패드로 일반적인 현금 액수의 빠른 입력을 제공합니다.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "지불 금액 숫자 키패드로 일반적인 현금 액수의 빠른 입력을 제공합니다.")
