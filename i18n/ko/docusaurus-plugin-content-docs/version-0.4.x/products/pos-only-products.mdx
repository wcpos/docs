상품을 온라인 스토어나 POS 중 한쪽에서 숨겨야 하는 상황이 있을 수 있습니다.
WCPOS에서는 상품 편집 페이지의 POS 가시성 설정을 사용하여 이를 달성할 수 있습니다.

### POS 전용 상품 활성화

POS 전용 상품은 기본적으로 활성화되어 있지 않습니다.
POS 전용 상품을 활성화하려면 <code>WP 관리자 > POS > 설정 > 일반</code>으로 이동하세요.

![POS 전용 상품 활성화](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "POS 전용 상품 활성화")

POS 전용 상품을 활성화하면 추가 데이터베이스 조회가 필요하므로 약간의 성능 저하가 발생할 수 있습니다.
POS 전용 상품을 사용하지 않는 경우 이 옵션을 비활성화된 상태로 두는 것이 좋습니다.

### POS 및 온라인 가시성

![POS 가시성](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "상품 편집 페이지의 POS 가시성 설정")

| 설정                             | 설명                                                                               |
| ------------------------------- | ---------------------------------------------------------------------------------- |
| **POS & 온라인**<br />(기본값)    | 상품이 온라인 웹 스토어와 실제 매장 모두에서 표시됩니다                                |
| **POS 전용**                     | 상품 페이지가 온라인에서 표시되지 않습니다                                            |
| **온라인 전용**                   | 상품이 POS에서 표시되지 않습니다                                                     |

### 카탈로그 가시성

POS 전용 상품 페이지로 직접 이동하면 <code>404 페이지를 찾을 수 없음</code>이 표시됩니다.
그러나 POS 전용 상품은 여전히 온라인 검색 및 카테고리 페이지에 나타날 수 있습니다.
상품을 웹사이트에서 완전히 숨기려면 카탈로그 가시성을 '숨김'으로 설정해야 합니다.

![카탈로그 가시성](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "상품 편집 페이지의 카탈로그 가시성 설정")

### POS 전용 상품 필터

상품 관리 페이지의 빠른 링크를 사용하여 어떤 상품이 표시/숨김 상태인지 빠르게 확인할 수 있습니다:

![POS 가시성 필터](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "POS 가시성 필터")

### 일괄 편집

여러 상품의 POS 가시성을 변경하려면 상품 목록으로 이동하여 <code>대량 작업</code>에서 <code>편집</code>을 선택하세요.

![POS 가시성 일괄 편집](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "POS 가시성 일괄 편집")
