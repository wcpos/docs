---
title: 사용자 정의 바코드 필드
---

기본적으로 WCPOS는 SKU 필드를 바코드로 사용합니다.

![기본적으로 WCPOS는 SKU 필드를 바코드로 사용합니다.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "기본적으로 WCPOS는 SKU 필드를 바코드로 사용합니다.")

[WCPOS Pro](http://wcpos.com/pro) 사용자는 `WP Admin > POS > 설정 > 일반`에서 바코드 필드를 선택하여 모든 제품 메타 필드를 사용할 수 있습니다.

![WCPOS Pro에서 사용자 정의 바코드 필드 선택](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "WCPOS Pro에서 사용자 정의 바코드 필드 선택")

사용자 정의 바코드 메타 필드는 `woocommerce_pos_barcode_meta_key` 필터를 사용하여 설정할 수도 있습니다.

```php title="functions.php"
/**
 * 바코드로 사용할 사용자 정의 제품 메타 필드
 *
 * 기본적으로 WCPOS는 SKU 필드 '_sku'를 사용합니다.
 * 다음 WordPress 필터를 사용하여 메타 키를 변경할 수 있습니다.
 *
 * 아래는 인기 있는 바코드 플러그인의 메타 필드입니다.
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // 사용자 정의 메타 키로 변경
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
바코드는 각 제품에 대해 고유해야 합니다.
:::
