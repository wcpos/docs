WCPOS는 SKU 필드를 통해 제품을 필터링할 수 있습니다. 검색이 바코드 필드와 정확히 일치하면 제품이 즉시 장바구니에 추가되고 검색 필드는 초기화됩니다. 검색 필드를 자동으로 채울 수 있는 바코드 스캐너가 있는 경우 WCPOS를 사용하여 바코드를 스캔해야 합니다.

SKU 검색을 실제로 보려면 http://demo.wcpos.com/pos/로 가십시오.

`BARCODE`를 입력(또는 복사하여 붙여넣기)하면 항목이 장바구니에 추가됩니다.

`VARIATION1`을 입력(또는 복사하여 붙여넣기)하면 변형이 장바구니에 추가됩니다.

![WCPOS의 바코드 스캔](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "WCPOS에서 SKU 필드를 사용하여 필터링")

다음 기능이 향후 릴리스를 위해 계획되고 있습니다:

- 바코드 스캐너를 사용하여 WooCommerce에 제품 추가. 이는 0.3.x 릴리스에 계획된 재고 조사 모드의 일부가 될 것입니다.
- 모바일 장치 카메라를 사용한 바코드 스캔. 이는 장치 하드웨어에 접근하기 위해 네이티브 WCPOS 앱이 필요합니다. 네이티브 앱은 희망 목록에 있지만 현재 웹 버전의 기능 추가가 가장 높은 우선 사항입니다.
- 핫키 활성화

### 업데이트

**버전 0.3.1 이후:** 이제 바코드 스캔을 위한 전용 모드가 제공됩니다. 검색 필드 왼쪽에 있는 버튼을 클릭하여 '검색' 모드와 '바코드 스캔' 사이를 전환할 수 있습니다.
