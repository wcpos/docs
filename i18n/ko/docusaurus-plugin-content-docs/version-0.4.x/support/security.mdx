---
title: 보안
description: 고객 정보의 보안을 보장하는 것은 귀하의 책임입니다.
---

WCPOS는 설치된 [WooCommerce 게이트웨이](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/)를 사용하여 결제할 수 있습니다. 결제 게이트웨이가 개인 정보를 수집하는 경우 **고객 정보의 보안을 보장하는 것은 귀하의 책임입니다**. POS를 안전하게 유지하기 위한 몇 가지 팁은 다음과 같습니다:

### SSL 인증서 사용 {#use-a-ssl-certificate}

온라인 상점을 운영하는 경우 웹사이트에 대한 보안 인증서를 가져야 합니다. 이를 통해 로그인 및 신용 카드 번호와 같은 양식 데이터를 전송할 때 암호화됩니다. 보안 인증서를 사용하지 않으면 인터넷을 통해 전송되는 모든 정보가 가로채어져 읽힐 수 있으며, 이는 귀하의 네트워크를 공유하는 사람일 수도 있고, 컴퓨터와 서버를 연결하는 수십 개의 노드 중 한 곳일 수도 있습니다. 기본 보안 인증서는 [US$16 이하의 비용](https://www.servertastic.com/rapidssl/)이 들며 설정하는 데 하루도 안 걸립니다.. 따라서 보안 인증서를 갖지 않는 것은 변명이 되지 않습니다.

### POS를 방치하지 마십시오 {#dont-leave-your-pos-unattended}

인터넷을 통한 스누핑은 보안이 침해될 수 있는 하나의 방법일 뿐이며, 실제 피해는 악의적인 사용자가 귀하의 장치에 물리적으로 접근할 때 발생하는 경우가 더 많습니다. 이는 컴퓨터나 태블릿을 혼잡한 장소에 자주 두는 상점 소유자에게 특히 중요한 문제입니다. 누군가 귀하의 컴퓨터나 장치를 훔치는 최악의 상황을 대비하고 다음과 같은 보안 조치를 통해 위험을 최소화하십시오:

- 장치가 5분 또는 10분 동안 방치되면 잠금 화면 사용
- 장치를 떠날 때 WordPress와 WCPOS에서 로그아웃
- 장치가 도난당했을 경우 신속하게 비밀번호 재설정
- 장치가 도난당했을 경우 원격으로 장치 초기화

### 가능한 한 적은 수의 플러그인 사용 {#use-the-least-number-of-plugins-possible}

WordPress의 멋진 점 중 하나는 클릭 한 번으로 사용할 수 있는 수천 개의 플러그인이 있다는 것이지만 ... 또한 최악의 점 중 하나이기도 합니다. 플러그인을 설치할 때마다 해당 플러그인에 귀하의 웹사이트 및 이를 통해 전송되는 데이터에 대한 접근 권한을 부여하게 됩니다. 모든 플러그인으로 매장에 '장식'하고 싶어하는 유혹이 있지만, 이는 종종 사이트를 느리게 만들고 사이트를 불안전하게 만들 수 있습니다. 플러그인을 설치하기 전에 자신에게 물어보십시오:

1. 정말 필요합니까?
2. 플러그인 제작자를 신뢰합니까?

WooCommerce 상점을 안전하게 유지하기 위한 추가 정보는 WooThemes 웹사이트의 [이 훌륭한 게시물](http://www.woothemes.com/2013/09/improve-your-wordpress-security-with-these-10-tips/)을 읽어보십시오.
