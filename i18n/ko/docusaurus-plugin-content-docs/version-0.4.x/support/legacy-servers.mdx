HTTP (및 HTTPS)는 모든 브라우저가 웹 애플리케이션과 통신할 때 사용하는 프로토콜입니다.
프로토콜의 핵심 부분은 [HTTP 메서드](https://tools.ietf.org/html/rfc2616#section-5.1.1)입니다.
전통적으로 웹 애플리케이션은 GET (웹 페이지 검색) 및 POST (웹 양식 제출)와 같은 매우 제한된 HTTP 메서드 세트를 사용했습니다.

WCPOS와 같은 현대적인 웹 애플리케이션은 다음 HTTP 메서드를 사용하는 REST/HTTP 접근 방식을 사용합니다:

- GET
- POST
- PUT
- PATCH
- DELETE

드문 경우지만 웹 호스트가 이러한 HTTP 메서드 중 일부를 차단할 수 있습니다.
웹 호스트가 최신 RESTful HTTP 메서드를 차단하는 경우 레거시 서버 지원을 활성화해야 합니다.
