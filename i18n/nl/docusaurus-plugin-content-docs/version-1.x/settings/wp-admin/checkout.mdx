De pagina Afrekeninstellingen (`WP Admin > POS > Settings > Checkout`) bepaalt welke betaalgateways in de POS verschijnen, hoe bestellingen worden verwerkt en welke notificatie-e-mails worden verzonden voor POS-verkopen.

## E-mailmeldingen {#email-notifications}

Bovenaan de pagina staan drie groepen e-mailschakelaars — **Beheerders-e-mails**, **Klant-e-mails** en **Kassier-e-mails** — waarmee u bepaalt welke WooCommerce-notificatie-e-mails worden verzonden voor POS-bestellingen, los van de algemene WooCommerce-e-mailinstellingen van uw site.

Zie **[E-mailmeldingen](/settings/wp-admin/email-notifications)** voor de volledige referentie, inclusief standaardwaarden per e-mailtoggle, het model voor kassierontvangers, veelvoorkomende scenario's en ontwikkelaarshooks.

## Betaalgateways {#payment-gateways}

Deze sectie toont alle WooCommerce-betaalgateways die op uw site zijn geïnstalleerd. Elke gateway kan afzonderlijk worden in- of uitgeschakeld voor de POS.

| Kolom | Beschrijving |
|--------|-------------|
| **Standaard** | Kies welke gateway vooraf geselecteerd is wanneer een kassier de checkout opent |
| **Gateway** | De naam van de betaalgateway |
| **Gateway-ID** | De WooCommerce-gateway-ID (handig voor ontwikkelaars) |
| **Ingeschakeld** | Schakel de gateway in of uit voor POS-afrekenen |
| **Orderstatus** | De status die een POS-order krijgt wanneer deze met deze gateway is betaald (zie [Orderstatus](#order-status) hieronder) |

Elke rij heeft ook een menu (⋮) met **POS-instellingen** — wijzig de POS-specifieke **Titel** en **Beschrijving** van de gateway — en **WooCommerce-instellingen**, die rechtstreeks linken naar de volledige gateway-instellingen in WooCommerce.

### Een gateway inschakelen {#enabling-a-gateway}

1. Zoek de gateway in de lijst
2. Zet de schakelaar **Ingeschakeld** aan
3. De gateway verschijnt bij de volgende paginalading in de POS-checkout

:::note Gratis versus Pro
In de gratis versie zijn alleen de ingebouwde gateways **Cash** en **Card** bewerkbaar. Andere geïnstalleerde WooCommerce-gateways inschakelen voor de POS vereist [WCPOS Pro](/getting-started/pro-license).
:::

### Gateways herschikken {#reordering-gateways}

Sleep gateways om hun weergavevolgorde in de POS-checkout te wijzigen. Gebruik de kolom **Standaard** om te kiezen welke gateway vooraf geselecteerd is wanneer een kassier het checkoutvenster opent.

### POS versus online winkel {#pos-vs-online-store}

Gateway-instellingen hier zijn **onafhankelijk** van uw WooCommerce-betaalinstellingen. Een gateway kan zijn:

- **Ingeschakeld voor POS, uitgeschakeld online** — handig voor contante betalingen die niet gelden voor weborders
- **Ingeschakeld online, uitgeschakeld voor POS** — handig voor gateways zoals PayPal die niet praktisch zijn voor verkoop op locatie
- **Ingeschakeld voor beide** — de gateway werkt overal

:::note
Een gateway moet eerst geïnstalleerd en actief zijn in `WP Admin > WooCommerce > Settings > Payments` voordat deze in de POS-afrekeninstellingen verschijnt. Als een gateway ontbreekt, controleer dan of de plugin is geïnstalleerd en de gateway in WooCommerce is ingeschakeld.
:::

## Orderstatus {#order-status}

Orderstatus wordt **per gateway** ingesteld via de kolom **Orderstatus** in de gatewaytabel — een verkoop kan dus anders eindigen afhankelijk van hoe deze is betaald:

- **Cash** en **Card** staan standaard op **Completed**, omdat betaling aan de kassa is ontvangen.
- Gateways waarbij het geld later binnenkomt — zoals **BACS** (bankoverschrijving) of **Cheque** — staan standaard op **On hold**.

Kies elke WooCommerce-orderstatus voor elke gateway. (Oudere versies gebruikten één globale standaard; die waarde is bij de upgrade naar elke gateway gemigreerd.)
