Winkelinstellingen zijn appbrede configuraties die bepalen hoe de POS werkt. Deze instellingen gelden voor de hele winkel, niet voor afzonderlijke schermen.

## Winkelinstellingen openen {#accessing-store-settings}

Open de Winkelinstellingen door:

- Op het **gebruikersmenu** (rechterbovenhoek) te klikken en **Instellingen** te selecteren
- De sneltoets `Ctrl + Shift + S` gebruiken

## Tabbladen voor instellingen {#settings-tabs}

Het venster Winkelinstellingen bevat deze tabbladen:

| Tabblad | Beschrijving |
|-----|-------------|
| [Algemeen](./general) | Winkelinformatie, taal, valuta, standaardklant |
| [Belasting](./tax) | Opties voor belastingberekening en weergave |
| [Streepjescode](./barcode) | Configuratie van de barcodescanner |
| [Sneltoetsen](./hotkeys) | Beschikbare sneltoetsen |
| [Thema](./theme) | Vormgeving en kleurenschema |
| [Bonprinter](/hardware/printers) | Verbinding en configuratie van de bonprinter |
