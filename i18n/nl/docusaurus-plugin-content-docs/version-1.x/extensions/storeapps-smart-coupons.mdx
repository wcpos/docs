NL: # WCPOS StoreApps Smart Coupons

NL: Adds WCPOS compatibility for store credit created with [Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/). It lets POS orders redeem StoreApps store credit, keeps the StoreApps balance in sync, and leaves an order-note audit trail for staff.

NL: This extension is specifically for the StoreApps / WooCommerce.com Smart Coupons plugin. Other plugins with similar “Smart Coupons” names are separate products and may need separate integrations.

:::info Pro Feature
NL: Installing and managing extensions from the directory requires [WCPOS Pro](/getting-started/pro-license).
:::

## Vertaald: What It Does {#what-it-does}

- NL: Records the StoreApps `smart_coupons_contribution` metadata for POS-created orders when the normal checkout cart context is not available.
- NL: Lets StoreApps deduct the correct partial store-credit amount from the coupon balance after POS checkout.
- NL: Restores the StoreApps store-credit balance when the POS order is cancelled or refunded through WooCommerce's normal order lifecycle.
- NL: Adds a private order note after redemption showing the coupon code, amount used, and current balance.
- NL: Shows the remaining store-credit balance in the existing receipt discount row by appending it to the coupon description/discount label.

NL: The extension uses normal WooCommerce coupon fields and metadata. It does not add custom WCPOS coupon response fields or require custom receipt templates.

## Vertaald: Installation {#installation}

<Steps>
  <StepCard number="1" title="NL: Install StoreApps Smart Coupons">

NL: Install and activate [Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/). Create your store-credit coupons or gift cards as usual in WooCommerce.

  </StepCard>
  <StepCard number="2" title="NL: Install WCPOS StoreApps Smart Coupons">

NL: Install from `WP Admin > POS > Settings > Extensions`, or download the latest release from the [GitHub releases page](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases) and upload via `Plugins > Add New > Upload Plugin`.

  </StepCard>
  <StepCard number="3" title="NL: Sync coupons to the POS" isLast>

NL: Open the POS and sync coupons. Cashiers can then apply StoreApps store-credit coupons from the normal **Add Coupon** flow.

  </StepCard>
</Steps>

## Vertaald: Receipt Output {#receipt-output}

NL: WCPOS receipt templates already print coupon discounts using the discount row label. During receipt rendering, this extension appends the current StoreApps balance to that existing label.

NL: For example, a coupon description of **Gift card** may print as:

```text
Gift card — Store credit balance: £61.50
```

NL: If the coupon has no description, the balance text is used as the discount label. Existing receipt templates do not need to be changed.

## Vertaald: Order Audit Trail {#order-audit-trail}

NL: After StoreApps processes a POS store-credit redemption, the extension adds a private order note similar to:

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

NL: Use this note to confirm which store-credit coupon was used, how much was redeemed, and what balance remained after the sale.

## Vertaald: Compatibility Notes {#compatibility-notes}

- NL: **StoreApps only:** this extension targets StoreApps / WooCommerce.com Smart Coupons. It does not target WebToffee or other smart-coupon plugins.
- NL: **Coupon data:** WCPOS continues to use normal WooCommerce coupon responses. StoreApps-specific state remains in WooCommerce metadata such as `smart_coupons_contribution` and coupon balance fields.
- NL: **Receipts:** balance text is added through the existing coupon description/discount label path, not by changing receipt data or templates.
- NL: **Inactive StoreApps plugin:** if StoreApps Smart Coupons is not active, the extension does nothing.
