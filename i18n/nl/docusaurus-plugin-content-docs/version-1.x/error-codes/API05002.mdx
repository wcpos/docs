# API05002: WCPOS-plugin niet gevonden

## Wat dit betekent {#what-this-means}

De WCPOS-plugin is niet geïnstalleerd of niet actief op je WordPress-site. De POS-applicatie heeft deze plugin nodig om met WooCommerce te communiceren.

## Veelvoorkomende oorzaken {#common-causes}

- **Plugin niet geïnstalleerd** — De plugin is nooit geïnstalleerd
- **Plugin gedeactiveerd** — De plugin is uitgeschakeld
- **Plugin verwijderd** — De pluginbestanden zijn verwijderd
- **Verkeerde site** — Verbonden met een site zonder WCPOS

## Hoe op te lossen {#how-to-fix}

### 1. Installeer de plugin {#1-install-the-plugin}

Als deze niet is geïnstalleerd:
1. Ga naar WordPress-beheer → Plugins → Nieuwe toevoegen
2. Zoek naar "WooCommerce POS"
3. Installeer en activeer de plugin

Of download van [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Activeer de plugin {#2-activate-the-plugin}

Als deze is geïnstalleerd maar niet actief:
1. Ga naar Plugins → Geïnstalleerde plugins
2. Zoek "WooCommerce POS"
3. Klik op "Activeren"

### 3. Controleer op fouten {#3-check-for-errors}

Als de plugin niet wil activeren:
- Controleer op PHP-fouten in je foutlog
- Verifieer dat WooCommerce is geïnstalleerd en actief is
- Controleer de PHP-versievereisten
- Zoek naar pluginconflicten

### 4. Verifieer de URL {#4-verify-the-url}

Zorg dat de POS is geconfigureerd om verbinding te maken met de juiste WordPress-site:
- Controleer de URL in de POS-instellingen
- Verifieer dat deze overeenkomt met je WordPress-installatie

### 5. Herinstalleer indien nodig {#5-reinstall-if-necessary}

Als de plugin beschadigd is:
1. Deactiveer en verwijder de plugin
2. Herinstalleer vanaf WordPress.org
3. Activeer de plugin
