# API-fouten

API-fouten treden op bij de communicatie met je WooCommerce-server. Deze fouten hebben het voorvoegsel `API` en zijn ingedeeld in de volgende categorieën:

## Categorieën {#categories}

| Categorie | Codebereik | Omschrijving |
|----------|------------|-------------|
| [Verbinding](#connection-errors) | API01xxx | Netwerk- en connectiviteitsproblemen |
| [Authenticatie](#authentication-errors) | API02xxx | Problemen met inloggen, tokens en machtigingen |
| [Verzoek](#request-errors) | API03xxx | Problemen met uitgaande verzoeken |
| [Antwoord](#response-errors) | API04xxx | Problemen met serverantwoorden |
| [Plugin/WordPress](#pluginwordpress-errors) | API05xxx | Problemen met WordPress- of WooCommerce-plugins |
| [Configuratie](#configuration-errors) | API06xxx | Problemen met installatie en configuratie |

---

## Verbindingsfouten {#connection-errors}

Netwerk- en connectiviteitsproblemen tussen de POS en je server.

| Code | Naam | Omschrijving |
|------|------|-------------|
| [API01001](./API01001) | Verbinding time-out | De server deed er te lang over om te antwoorden |
| [API01002](./API01002) | Verbinding geweigerd | De server weigerde de verbinding |
| [API01003](./API01003) | Verbinding gereset | De verbinding werd onverwacht verbroken |
| [API01004](./API01004) | DNS-resolutie mislukt | Kon het serveradres niet omzetten |
| [API01005](./API01005) | SSL-certificaatfout | Probleem met het beveiligingscertificaat van de site |
| [API01006](./API01006) | Netwerk onbereikbaar | Kan het netwerk niet bereiken |
| [API01007](./API01007) | Apparaat offline | Je apparaat is niet verbonden met internet |
| [API01008](./API01008) | Website niet beschikbaar | De website reageert niet |

## Authenticatiefouten {#authentication-errors}

Problemen met inloggen, sessies en machtigingen.

| Code | Naam | Omschrijving |
|------|------|-------------|
| [API02001](./API02001) | Ongeldige inloggegevens | Gebruikersnaam of wachtwoord is onjuist |
| [API02002](./API02002) | Token verlopen | Je sessie is verlopen |
| [API02003](./API02003) | Token ongeldig | Het authenticatietoken is niet geldig |
| [API02004](./API02004) | Gebruiker niet geautoriseerd | Je hebt geen toestemming om deze actie uit te voeren |
| [API02005](./API02005) | Onvoldoende machtigingen | Je gebruikersrol mist de vereiste machtigingen |
| [API02006](./API02006) | API-sleutel ongeldig | De WooCommerce API-sleutel is niet geldig |
| [API02007](./API02007) | Token vernieuwen mislukt | Kon je sessie niet vernieuwen |
| [API02008](./API02008) | Refresh-token ongeldig | Het refresh-token is niet geldig |
| [API02009](./API02009) | Refresh-token verlopen | Het refresh-token is verlopen |
| [API02010](./API02010) | Authenticatie vereist | Authenticatie is vereist voor deze actie |

## Verzoekfouten {#request-errors}

Problemen met de verzoeken die naar de server worden gestuurd.

| Code | Naam | Omschrijving |
|------|------|-------------|
| [API03001](./API03001) | Ongeldig verzoekformaat | Het verzoekformaat is niet correct |
| [API03002](./API03002) | Ontbrekende vereiste parameters | Vereiste gegevens ontbreken in het verzoek |
| [API03003](./API03003) | Ongeldige parameterwaarde | Een parameter heeft een ongeldige waarde |
| [API03004](./API03004) | Verzoek te groot | Het verzoek overschrijdt de groottelimieten |
| [API03005](./API03005) | Snelheidslimiet overschreden | Te veel verzoeken in korte tijd |
| [API03006](./API03006) | Methode niet ondersteund | De HTTP-methode wordt niet ondersteund |
| [API03007](./API03007) | Verzoekwachtrij vol | Te veel openstaande verzoeken |

## Antwoordfouten {#response-errors}

Problemen met de antwoorden die van de server worden ontvangen.

| Code | Naam | Omschrijving |
|------|------|-------------|
| [API04001](./API04001) | Ongeldig antwoordformaat | Het serverantwoordformaat is niet geldig |
| [API04002](./API04002) | Onverwachte antwoordcode | Een onverwachte HTTP-statuscode ontvangen |
| [API04003](./API04003) | Misvormd JSON-antwoord | Het JSON-antwoord is beschadigd of ongeldig |
| [API04004](./API04004) | Ontbrekende antwoordgegevens | Verwachte gegevens ontbreken in het antwoord |
| [API04005](./API04005) | JSON-herstel geprobeerd | Geprobeerd te herstellen van misvormde JSON |
| [API04006](./API04006) | Bron niet gevonden | De gevraagde bron bestaat niet (404) |

|------|------|-------------|

| [API05003](./API05003) | WCPOS-plugin verouderd | De WCPOS-plugin moet worden bijgewerkt |
| [API05004](./API05004) | WordPress API uitgeschakeld | De WordPress REST API is uitgeschakeld |
| [API05005](./API05005) | Plugin niet gevonden | Een vereiste plugin is niet geïnstalleerd |
