# SY02002: Service niet beschikbaar

## Wat dit betekent {#what-this-means}

Een vereiste service of component is niet beschikbaar. De POS is afhankelijk van diverse services om te functioneren, en een daarvan is op dit moment ontoegankelijk.

## Veelvoorkomende oorzaken {#common-causes}

- **Achtergrondservice gestopt** — Een vereiste service draait niet
- **Probleem met systeembronnen** — Onvoldoende bronnen om de service te starten
- **Afhankelijkheidsfout** — Een service waarvan dit afhankelijk is, is mislukt
- **Opstartprobleem** — De service kon niet correct starten
- **Serverfout** — De WooCommerce-server retourneerde een 5xx-fout

## Toewijzing van serverfouten {#server-error-mapping}

Deze foutcode wordt geactiveerd wanneer de server het volgende retourneert:

| Servercode | Bron |
|-------------|--------|
| HTTP 5xx | Elke serverfoutrespons (500, 502, 503, 504, enz.) |

## Hoe op te lossen {#how-to-fix}

### 1. De applicatie opnieuw starten {#1-restart-the-application}

Sluit de POS en open deze opnieuw:
1. Sluit de applicatie volledig af
2. Wacht een paar seconden
3. Start deze opnieuw

### 2. Je apparaat opnieuw opstarten {#2-restart-your-device}

Een volledige herstart lost serviceproblemen vaak op:
1. Sla belangrijk werk op
2. Start je computer opnieuw op
3. Probeer de POS opnieuw

### 3. Systeembronnen controleren {#3-check-system-resources}

Zorg ervoor dat er systeembronnen beschikbaar zijn:
- Sluit onnodige applicaties
- Controleer het geheugengebruik
- Controleer de schijfruimte

### 4. De applicatie bijwerken {#4-update-the-application}

Zorg ervoor dat je de nieuwste versie gebruikt:
- Controleer op beschikbare updates
- Installeer eventuele openstaande updates
- Herstart na het bijwerken

### 5. Opnieuw installeren indien nodig {#5-reinstall-if-necessary}

Als de service niet beschikbaar blijft:
1. Verwijder de applicatie
2. Start je apparaat opnieuw op
3. Download de nieuwste versie
4. Installeer opnieuw vanaf nul

## Browserspecifiek (webversie) {#browser-specific-web-version}

Voor de webversie:
- Zorg ervoor dat JavaScript is ingeschakeld
- Controleer of vereiste browserfuncties niet worden geblokkeerd
- Probeer een andere browser
- Schakel browserextensies uit die mogelijk interfereren

## Wanneer contact opnemen met support {#when-to-contact-support}

Als deze fout aanhoudt:
- Noteer wanneer deze optreedt
- Controleer of specifieke acties deze veroorzaken
- Rapporteer met details voor onderzoek
