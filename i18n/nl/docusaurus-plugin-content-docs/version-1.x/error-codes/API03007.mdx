# API03007: Verzoekwachtrij vol

## Wat dit betekent {#what-this-means}

De POS heeft te veel openstaande verzoeken die wachten om verzonden te worden. De interne wachtrij is vol en kan geen verzoeken meer accepteren totdat er enkele zijn voltooid.

## Veelvoorkomende oorzaken {#common-causes}

- **Traag netwerk** — Verzoeken stapelen zich op door een trage verbinding
- **Server reageert niet** — De server doet er te lang over om te reageren
- **Snelle acties** — Veel acties sneller uitvoeren dan ze verwerkt kunnen worden
- **Synchronisatieoverbelasting** — Een grote synchronisatiebewerking blokkeert de wachtrij

## Hoe op te lossen {#how-to-fix}

### 1. Wacht tot de wachtrij leeg is {#1-wait-for-queue-to-clear}

Geef openstaande verzoeken de tijd om te voltooien:
- Wacht 30-60 seconden
- Let op de synchronisatie-indicatoren tot ze voltooid zijn
- Vermijd extra acties totdat de wachtrij is geleegd

### 2. Controleer de netwerkverbinding {#2-check-network-connection}

Een trage verbinding veroorzaakt opstapeling:
- Test je internetsnelheid
- Probeer een snellere verbinding
- Ga dichter bij de WiFi-router zitten

### 3. Controleer de reactietijd van de server {#3-check-server-response-time}

Als de server traag is:
- Test of je je site in een browser kunt openen
- Neem contact op met de hosting als de site traag is
- Wacht tot de serverbelasting afneemt

### 4. Herstart de POS {#4-restart-the-pos}

Als de wachtrij vastzit:
1. Sluit de POS-applicatie
2. Wacht even
3. Open de POS opnieuw
4. Openstaande verzoeken moeten mogelijk opnieuw worden gestart

### 5. Vermijd snelle acties {#5-avoid-rapid-actions}

Tijdens synchronisatie of trage periodes:
- Wacht tot één actie is voltooid voordat je een andere start
- Klik niet herhaaldelijk op knoppen
- Wees geduldig met netwerkbewerkingen

## Offlinegedrag {#offline-behaviour}

Wanneer de wachtrij vol raakt door offline-omstandigheden:
- De POS houdt openstaande verzoeken vast totdat je weer online bent
- Je kunt nog steeds gecachte producten en klanten doorbladeren
- Je kunt nieuwe bestellingen starten en items aan de winkelwagen toevoegen
- Het voltooien van bestellingen vereist dat de verbinding is hersteld
