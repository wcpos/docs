# Betalingsfouten

Betalingsfouten treden op tijdens het afrekenen en het verwerken van betalingen. Deze fouten hebben het voorvoegsel `PY` en hebben betrekking op problemen met betaalkaarten, gateways en transacties.

## Categorieën {#categories}

| Categorie | Codereeks | Beschrijving |
|----------|------------|-------------|
| [Kaart](#card-errors) | PY01xxx | Problemen met de betaalkaart |
| [Gateway](#gateway-errors) | PY02xxx | Communicatieproblemen met de betalingsgateway |

---

## Kaartfouten {#card-errors}

Problemen met betrekking tot de betaalkaart van de klant.

| Code | Naam | Beschrijving |
|------|------|-------------|
| [PY01001](./PY01001) | Betaling geweigerd | De betaling is geweigerd |
| [PY01002](./PY01002) | Onvoldoende saldo | Niet genoeg saldo voor de betaling |
| [PY01003](./PY01003) | Kaart verlopen | De betaalkaart is verlopen |
| [PY01004](./PY01004) | Ongeldig kaartnummer | Het kaartnummer is niet geldig |

## Gatewayfouten {#gateway-errors}

Problemen bij de communicatie met de betalingsgateway.

| Code | Naam | Beschrijving |
|------|------|-------------|
| [PY02001](./PY02001) | Fout in betalingsgateway | Fout bij de communicatie met de betalingsgateway |
| [PY02002](./PY02002) | Time-out bij betaling | De betalingsverwerking is verlopen |
