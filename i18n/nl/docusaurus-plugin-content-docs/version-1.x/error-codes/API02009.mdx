# API02009: Vernieuwingstoken verlopen

## Wat dit betekent {#what-this-means}

Je vernieuwingstoken heeft zijn levensduur overschreden en kan niet langer worden gebruikt om nieuwe toegangstokens te verkrijgen. In tegenstelling tot toegangstokens (die snel verlopen) blijven vernieuwingstokens doorgaans dagen of weken geldig, maar uiteindelijk verlopen ze toch.

## Veelvoorkomende oorzaken {#common-causes}

- **Langdurige inactiviteit** — Je hebt de POS lange tijd niet gebruikt
- **Korte levensduur van token** — De server is ingesteld met een korte verloopduur voor vernieuwingstokens
- **Tijdsverschil** — De tijd van het apparaat of de server wijkt aanzienlijk af

## Oplossen {#how-to-fix}

### 1. Opnieuw inloggen {#1-log-in-again}

Wanneer het vernieuwingstoken verloopt, moet je opnieuw authenticeren:
1. Log uit van de POS
2. Voer je gegevens in
3. Er wordt een nieuw vernieuwingstoken uitgegeven

### 2. De POS regelmatig gebruiken {#2-use-pos-regularly}

Om dit in de toekomst te voorkomen:
- Gebruik de POS regelmatig om tokens fris te houden
- Tokens worden automatisch vernieuwd terwijl je actief bent
- Langdurige inactiviteit vereist opnieuw inloggen

### 3. Tijdsinstellingen controleren {#3-check-time-settings}

Zorg voor een nauwkeurige tijd op alle systemen:
- Schakel automatische tijdsynchronisatie in op je apparaat
- Controleer of de servertijd correct is
- Tijdsverschillen kunnen voortijdig verlopen veroorzaken

### 4. Levensduur van token aanpassen (geavanceerd) {#4-adjust-token-lifetime-advanced}

Als tokens te snel verlopen voor jouw behoeften, kan een serverbeheerder de levensduur van het vernieuwingstoken aanpassen in de authenticatie-instellingen.
