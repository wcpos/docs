# API03003: Ongeldige parameterwaarde

## Wat dit betekent {#what-this-means}

Een van de waarden in je verzoek is ongeldig. De gegevensindeling, het type of de waarde komt niet overeen met wat de server verwacht.

## Veelvoorkomende oorzaken {#common-causes}

- **Verkeerd gegevenstype** — Tekst waar een getal wordt verwacht (of omgekeerd)
- **Buiten bereik** — De waarde overschrijdt de toegestane limieten
- **Ongeldige indeling** — E-mail-, telefoon- of andere opgemaakte velden zijn onjuist
- **Ongeldige verwijzing** — Verwijzen naar een ID die niet bestaat

## Toewijzing van serverfouten {#server-error-mapping}

Deze foutcode wordt geactiveerd wanneer de server het volgende retourneert:

| Servercode | Bron |
|-------------|--------|
| `rest_invalid_param` | WordPress REST API |
| `woocommerce_rest_invalid_id` | WooCommerce REST API |

## Oplossen {#how-to-fix}

### 1. Invoerwaarden controleren {#1-check-input-values}

Bekijk de gegevens die je verstuurt:
- **Prijzen** — Moeten geldige getallen zijn
- **Aantallen** — Moeten positieve gehele getallen zijn
- **E-mails** — Moeten een geldige e-mailindeling hebben
- **ID's** — Moeten naar bestaande records verwijzen

### 2. Zoeken naar specifieke veldfouten {#2-look-for-specific-field-errors}

De fout kan aangeven welk veld ongeldig is:
- Controleer de details van het foutbericht
- Corrigeer het specifieke veld
- Probeer de actie opnieuw

### 3. Controleren of de gerefereerde gegevens bestaan {#3-verify-referenced-data-exists}

Als je naar producten, klanten of andere records verwijst:
- Zorg dat het item in WooCommerce bestaat
- Controleer of het onlangs is verwijderd
- Synchroniseer de gegevens om lokale records te vernieuwen

### 4. Veldbeperkingen controleren {#4-check-field-constraints}

WooCommerce kan beperkingen hebben:
- Maximum-/minimumwaarden
- Vereiste indelingen
- Toegestane opties voor keuzevelden

### 5. Wissen en opnieuw invoeren {#5-clear-and-re-enter}

Soms raken gegevens beschadigd:
- Wis het problematische veld
- Voer de waarde helemaal opnieuw in
- Vermijd kopiëren en plakken uit andere bronnen
