# API01006: Netwerk onbereikbaar

## Wat dit betekent {#what-this-means}

De POS kan geen route naar de server tot stand brengen. Dit is een probleem op netwerkniveau — de datapakketten kunnen geen pad naar je server vinden.

## Veelvoorkomende oorzaken {#common-causes}

- **Problemen met netwerkconfiguratie** — Onjuiste gateway- of routinginstellingen
- **ISP-problemen** — Je internetprovider ondervindt problemen
- **VPN-interferentie** — Een VPN routeert het verkeer mogelijk verkeerd
- **Netwerkproblemen van de server** — Het netwerk van de server is mogelijk offline

## Hoe op te lossen {#how-to-fix}

### 1. Controleer je internetverbinding {#1-check-your-internet-connection}

- Probeer andere websites te openen
- Start je router/modem opnieuw op
- Controleer of je verbonden bent met het netwerk

### 2. Schakel VPN tijdelijk uit {#2-disable-vpn-temporarily}

Als je een VPN gebruikt:
- Verbreek de verbinding en probeer het opnieuw
- Probeer een andere VPN-server
- Controleer of de VPN toegang toestaat tot de locatie van je server

### 3. Informeer bij je ISP {#3-check-with-your-isp}

Neem contact op met je internetprovider als:
- Andere sites ook onbereikbaar zijn
- Het probleem aanhoudt na het opnieuw opstarten van de netwerkapparatuur

### 4. Verifieer de serverstatus {#4-verify-server-status}

- Controleer of je hostingprovider netwerkproblemen meldt
- Probeer je site te openen vanaf een ander netwerk (bijv. mobiele data)
- Gebruik onlinetools om te controleren of de site voor iedereen offline is

### 5. Controleer de firewall-instellingen {#5-check-firewall-settings}

Zorg ervoor dat de firewall van je apparaat geen uitgaande verbindingen naar je server blokkeert.
