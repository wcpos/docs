# Betalingsgateways

WCPOS ondersteunt verschillende aangepaste betalingsgateways die de functionaliteit van uw kassasysteem uitbreiden. Deze gateways zijn specifiek ontworpen voor POS-omgevingen en bieden naadloze integratie met verschillende betaalmethoden en diensten.

## Beschikbare aangepaste gateways {#available-custom-gateways}

### Hardwareterminal-gateways {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accepteer betalingen met behulp van Stripe Terminal-hardwarelezers
- **[SumUp Terminal](./sumup-terminal)** - Betalingen verwerken via SumUp-kaartlezers
- **[PayPal Reader (Zettle)](./paypal-reader)** - Voer persoonlijke kaartbetalingen uit op een PayPal Reader (Zettle)-terminal

### Mobiele betalingsgateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accepteer telefonische betalingen via QR-code of pushmelding

### Digitale betalingsgateways {#digital-payment-gateways}

- **[E-mailfactuur](./email-invoice)** - Stuur betalingsfacturen naar klanten via e-mail
- **[Web Checkout](./web-checkout)** - Leid klanten door om online betalingen te voltooien

## Creëer uw eigen gateway {#creating-your-own-gateway}

Als u een aangepaste betalingsgateway nodig heeft voor uw specifieke vereisten, kunt u onze **[Gateway-sjabloon](/reference/gateway-template)** gebruiken om uw eigen WCPOS-betalingsgateway te maken.

## Installatieoverzicht {#installation-overview}

Alle aangepaste gateways volgen een soortgelijk installatieproces:

1. **Download** de nieuwste release uit de betreffende GitHub-repository
2. **Installeer** de plug-in via `WP Admin > Plugins > Add New > Upload Plugin`
3. **Configureer** de gateway-instellingen in `WP Admin > WooCommerce > Settings > Payments`
4. **Schakel** de gateway in `WP Admin > POS > Settings > Checkout` in
