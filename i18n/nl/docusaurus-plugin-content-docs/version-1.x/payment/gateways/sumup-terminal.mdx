# SumUp Terminal Gateway

Met de SumUp Terminal-gateway kun je rechtstreeks binnen WCPOS kaartbetalingen accepteren met SumUp-kaartlezers. Deze gateway biedt een naadloze integratie met het betalingsverwerkingssysteem van SumUp voor persoonlijke transacties.

## Functies {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Hardware-integratie">
    Verbind SumUp-kaartlezers met je POS-systeem via een internetverbinding
  </FeatureCard>
  <FeatureCard icon="bolt" title="Realtime verwerking">
    Directe betalingsverwerking en voltooiing van de bestelling
  </FeatureCard>
  <FeatureCard icon="shield" title="Veilige transacties">
    PCI-conforme betalingsverwerking via SumUp
  </FeatureCard>
  <FeatureCard icon="link" title="Eenvoudig koppelen">
    Eenvoudig apparaatkoppelingsproces met koppelingscodes
  </FeatureCard>
</FeatureGrid>

## Installatie {#installation}

<Steps>
  <StepCard number="1" title="Installeer SumUp Terminal for WooCommerce">

Installeer via `WP Admin > POS > Settings > Extensions`, of download de nieuwste **plugin-zip-asset** (niet de GitHub broncode-zip of tarball) van de [GitHub releases-pagina](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases) en upload deze via `Plugins > Add New > Upload Plugin`.

  </StepCard>
  <StepCard number="2" title="SumUp-instellingen configureren">

1. Ga naar `WP Admin > WooCommerce > Settings > Payments`
2. Zoek **SumUp Terminal** in de lijst met betaalmethoden
3. Klik op **SumUp Terminal** om de instellingen te openen
4. Voer je **SumUp API Key** in (beschikbaar in je SumUp-merchantdashboard)
5. Sla de instellingen op

:::note
Je hoeft de SumUp Terminal-gateway niet in te schakelen in de WooCommerce-instellingen. Deze wordt in een latere stap specifiek voor de POS ingeschakeld.
:::

  </StepCard>
  <StepCard number="3" title="Je SumUp Terminal koppelen">

1. Zoek op dezelfde instellingenpagina het gedeelte **Pair Reader**
2. Navigeer op je SumUp-apparaat naar het koppelingsscherm om de koppelingscode weer te geven
3. Voer de koppelingscode in die op je SumUp-apparaat wordt weergegeven
4. Klik op **"Pair Reader"** om de verbinding tot stand te brengen
5. Wacht op de bevestiging dat de lezer succesvol is gekoppeld

:::warning Belangrijk
De lezer moet succesvol gekoppeld zijn voordat je betalingen kunt verwerken. Zorg dat het koppelingsproces is voltooid voordat je verdergaat.
:::

  </StepCard>
  <StepCard number="4" title="Inschakelen in WCPOS" isLast>

1. Ga naar `WP Admin > POS > Settings > Checkout`
2. Zoek de **SumUp Terminal**-gateway in de lijst
3. Schakel de gateway in voor gebruik in de POS
4. Sla je instellingen op

  </StepCard>
</Steps>

## Gebruik {#usage}

### Betalingen verwerken {#processing-payments}

1. **Artikelen toevoegen**: Voeg producten toe aan je winkelwagen in de POS
2. **Gateway selecteren**: Kies "SumUp Terminal" als betaalmethode
3. **Betaling starten**: Klik om een nieuwe betaling op je SumUp-apparaat te starten
4. **Klantbetaling**: De klant voltooit de betaling op de SumUp-terminal
5. **Automatische voltooiing**: De bestelling wordt automatisch voltooid wanneer de betaling wordt gedetecteerd

### Betalingsbedieningen {#payment-controls}

Wanneer je de SumUp Terminal-gateway gebruikt, heb je de volgende opties:

- **Nieuwe betaling starten**: Start een betaalverzoek op de verbonden terminal
- **Betaling annuleren**: Annuleer een betaling die momenteel wordt verwerkt
- **Betalingsstatus**: Volg de huidige status van het betalingsproces

### Bestelbeheer {#order-management}

- **Automatische updates**: Bestellingen worden automatisch als voltooid gemarkeerd na een geslaagde betaling
- **Betalingsregistratie**: Alle betalingsgegevens worden vastgelegd in de bestelnotities
- **Bonnetjes genereren**: Standaard POS-bonnetjes worden gegenereerd na geslaagde betalingen

## Hardwarecompatibiliteit {#hardware-compatibility}

:::important Connectiviteitsvereisten
Deze implementatie maakt gebruik van de JavaScript SDK van SumUp, wat betekent dat het werkt via webapplicaties, maar **alleen internetverbonden terminals** vereist. Bluetooth-terminals worden momenteel niet ondersteund.
:::

### Ondersteunde terminals (internetverbonden) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - Terminal met WiFi-/ethernetverbinding

### Niet-ondersteunde terminals (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - Alleen Bluetooth
- **SumUp 3G** ❌ - Alleen Bluetooth

:::note Toekomstige ondersteuning
Ondersteuning voor Bluetooth-terminals staat gepland voor een toekomstige release van de iOS- en Android-app. Wanneer dit beschikbaar is, maakt dit ondersteuning mogelijk voor alle door SumUp gecertificeerde terminals, waaronder de Air- en 3G-modellen.
:::

## Probleemoplossing {#troubleshooting}

### Veelvoorkomende problemen {#common-issues}

<Accordion>
  <AccordionItem question="Lezer wil niet koppelen">

  - Zorg dat je een ondersteunde internetverbonden terminal gebruikt (alleen Solo)
  - Controleer of je SumUp Solo-apparaat verbonden is met WiFi en online is
  - Controleer of het apparaat in de koppelingsmodus staat en de koppelingscode correct is ingevoerd
  - Bevestig dat API-toegang is ingeschakeld voor je SumUp-account
  - Zorg voor een stabiele internetverbinding tijdens het koppelen

  </AccordionItem>

  <AccordionItem question="Betaling wordt niet verwerkt">

  - Controleer of de lezer correct is gekoppeld
  - Controleer of je SumUp-account actief is en in goede staat verkeert
  - Zorg dat het apparaat voldoende batterij en connectiviteit heeft
  - Bevestig dat de API-sleutel correct en actief is

  </AccordionItem>

  <AccordionItem question="Bestelling wordt niet voltooid">

  - Controleer de stabiliteit van de internetverbinding
  - Controleer de webhook-instellingen in je SumUp-account
  - Zorg dat de betaling op de SumUp-terminal geslaagd is
  - Controleer de WordPress-foutlogs op problemen met API-communicatie

  </AccordionItem>
</Accordion>
