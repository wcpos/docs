---
title: Bestellingen filteren op POS- of onlineverkoop
description: Scheid POS- en onlinebestellingen door de snelkoppelingen in het WooCommerce-beheer te gebruiken.
---

U kunt uw POS- en onlinebestellingen in het WooCommerce-beheer snel scheiden door de snelkoppelingen linksboven te gebruiken.
**Let op:** dit filtert alleen POS-bestellingen vanaf versie 0.3.1; bestellingen van vóór 0.3.1 worden standaard als online weergegeven.

![Bestellingen filteren](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Bestellingen filteren")

Als u daarnaast alle voltooide bestellingen uit uw beheergebied wilt filteren zodat u zich kunt richten op de in behandeling zijnde/verwerkende bestellingen, kunt u de volgende code toevoegen aan uw [functions.php](http://codex.wordpress.org/Functions_File_Explained)-bestand.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
