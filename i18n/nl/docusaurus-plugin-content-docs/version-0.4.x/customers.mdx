### Rechten {#capabilities}

Kassiers hebben de rechten _list_users_, _create_users_ en _edit_users_ nodig om alle klantfuncties te gebruiken.

### WordPress Multisite {#wordpress-multisite}

Bij gebruik van [WordPress multisite](https://codex.wordpress.org/Create_A_Network) zijn er enkele [extra controles](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) om te voorkomen dat gebruikers die geen [Super Admins](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) zijn klanten kunnen aanmaken en bewerken.

Om te beginnen moet je de volgende optie inschakelen in `Network Admin > Settings` zodat niet-Super Admins nieuwe klanten kunnen **aanmaken**.

![Multisite-optie om nieuwe gebruikers aan te maken](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Multisite-optie om nieuwe gebruikers aan te maken")

Daarnaast moet je het recht `manage_network_users` aan de kassier toevoegen zodat niet-Super Admins klantgegevens kunnen **bewerken**.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
