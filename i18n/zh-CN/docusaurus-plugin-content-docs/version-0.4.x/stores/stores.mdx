:::important
此功能需要升级到 [WCPOS Pro](http://wcpos.com/pro)。
:::

![商店列表示例](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "商店列表示例")

WCPOS Pro 添加了为实体销售创建独立_商店_的功能。
商店可以拥有自己的地址、标志和其他详细信息，用于[收据模板](../receipts)。
商店可以使用不同的税率，销售可以按商店[分开报告](../reports#sales-by-store)。

![商店编辑页面示例](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "商店编辑页面示例")

### 即将推出的功能

商店功能已在 0.4 版本中添加。
有几个_频繁被请求的功能_，如[多库存商店](./multi-inventory)和将收银员分配给商店的能力。
这些功能将在 Pro 插件接近 1.0 版本时添加。
