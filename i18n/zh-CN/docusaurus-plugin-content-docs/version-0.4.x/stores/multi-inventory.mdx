:::caution
目前Pro插件尚未支持多仓库存功能。
以下信息旨在在WCPOS Pro正式支持多仓库存之前为用户提供帮助。
:::

多仓库存管理是WCPOS Pro最受期待的功能之一。
这个功能将使商店所有者能够将产品分配到不同的仓库，并管理各个实体店铺和在线商店之间的库存。

创建多仓库存功能并非易事。
已经有一些插件正在解决这个问题，例如[WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646)和[WooCommerce MultiStore](http://woomultistore.com/)。
目前，将重点放在开发其他核心POS功能上，并在这些插件成熟后与之集成是明智的。

在短期内，您可以通过使用[WordPress多站点功能](https://codex.wordpress.org/Create_A_Network)来满足大部分需求。
通过为每个店铺创建一个站点，然后在每个站点上安装WooCommerce和WCPOS，可以使每个店铺的库存完全分开。
上面提到的WooCommerce Multisite在此方法的基础上添加了管理站点间库存的额外层。
