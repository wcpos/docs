使用 POS 访问设置，店主可以授予或移除用户对点销系统的权限。
默认情况下，**管理员**和**店铺管理员**角色已启用所有使用 POS 的权限。
建议保留默认设置，除非您完全了解后果。
例如，将 `publish_shop_orders` 权限授予 `订阅者` 角色将是一个_非常_糟糕的主意。

![POS 访问设置示例](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "POS 访问设置示例")

| 权限                                                                    | 访问                                     |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | 访问 POS 前端所需                         |
| manage_woocommerce_pos                                                  | 访问 POS 管理设置所需                     |
| read_private_products                                                   | 查看产品所需                              |
| edit_product <br /> edit_published_products <br /> edit_others_products | 编辑产品和变体所需                        |
| read_private_shop_orders                                                | 查看订单和收据所需                        |
| publish_shop_orders                                                     | 创建新订单所需                            |
| list_users                                                              | 查看客户所需                              |
| create_users                                                            | 创建新客户所需                            |
| edit_users                                                              | 更新现有客户所需                          |
| read_private_shop_coupons                                               | 查看优惠券所需                            |
