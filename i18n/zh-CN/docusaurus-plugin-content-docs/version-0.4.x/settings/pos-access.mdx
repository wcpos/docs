使用POS访问设置，商店拥有者可以授予或撤销用户对销售点系统的权限。
默认情况下，**管理员**和**商店经理**角色具有使用POS的所有权限。
建议您保持默认设置，除非您知道后果。
例如，授予`publish_shop_orders`给`Subscribers`角色将是一个_非常_糟糕的主意。

![POS访问设置示例](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "POS访问设置示例")

| 权限                                                                     | 访问                                    |
| ----------------------------------------------------------------------- | --------------------------------------- |
| access_woocommerce_pos                                                  | 访问POS前端所必需                       |
| manage_woocommerce_pos                                                  | 访问POS管理设置所必需                   |
| read_private_products                                                   | 查看产品所必需                         |
| edit_product <br /> edit_published_products <br /> edit_others_products | 编辑产品和变体所必需                    |
| read_private_shop_orders                                                | 查看订单和收据所必需                   |
| publish_shop_orders                                                     | 创建新订单所必需                       |
| list_users                                                              | 查看客户所必需                         |
| create_users                                                            | 创建新客户所必需                       |
| edit_users                                                              | 更新现有客户所必需                     |
| read_private_shop_coupons                                               | 查看优惠券所必需                       |
