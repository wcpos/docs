### 自动安装（推荐） {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Install and activate WCPOS in less than 1 minute")

1. 前往 `WP Admin > 插件 > 新增`
2. 在 WordPress 插件目录中搜索 `WCPOS`
3. 安装插件
4. 激活插件

### 手动安装 {#manual-installation}

1. 从 WordPress.org [下载 WCPOS](https://wordpress.org/plugins/woocommerce-pos/)
2. 前往 `WP Admin > 插件 > 新增 > 上传插件`
3. 激活插件

:::info
WCPOS 的旧版本可在 WordPress.org 的 [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) 标签页中找到。
:::
