### 自动安装（推荐）

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "在不到1分钟的时间内安装和激活WCPOS")

1. 转到 `WP管理后台 > 插件 > 添加新插件`
2. 在WordPress插件目录中搜索 `WCPOS`
3. 安装插件
4. 激活插件

### 手动安装

1. 从WordPress.org [下载WCPOS](https://wordpress.org/plugins/woocommerce-pos/)
2. 转到 `WP管理后台 > 插件 > 添加新插件 > 上传插件`
3. 激活插件

:::info
可以在Wordpress.org的[开发者](https://wordpress.org/plugins/woocommerce-pos/developers/)标签下找到WCPOS的早期版本。
:::
