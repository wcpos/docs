:::important

[WCPOS Pro](http://wcpos.com/pro) 是一个付费插件，为免费的 [WCPOS 插件](http://wordpress.org/plugins/woocommerce-pos/) 增加了功能。

:::

如果您购买了 WCPOS Pro 的许可证，请按照以下步骤安装并激活插件：

1.  前往: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  在 **Downloads** 下，点击下载链接并将插件保存到您的桌面。
3.  然后前往您的网站，登录并转到 `WP Admin > Plugins > Add New > Upload Plugin`
4.  从桌面上传插件的 zip 文件并激活。
5.  接下来，前往 `WP Admin > POS > Settings > License`，输入您的 License Key 和 License Email 以完成激活。

![My Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在您的账户页面下载 WCPOS Pro")

当新版本准备好时，它们将出现在您的 Updates 仪表板中。
有关我当前正在进行的工作的信息，请密切关注 [Roadmap](http://wcpos.com/roadmap/)。
另外，请密切关注变更日志，其中将列出每个版本的新功能，并且 [documentation page](http://docs.wcpos.com) 将更新有关如何使用新功能的信息。
