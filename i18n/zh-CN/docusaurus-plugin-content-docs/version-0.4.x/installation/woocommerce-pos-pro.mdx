:::important

[WCPOS 专业版](http://wcpos.com/pro)是一个付费插件，可为免费的[WCPOS插件](http://wordpress.org/plugins/woocommerce-pos/)增加功能。

:::

如果您已购买 WCPOS 专业版许可证，请按照以下步骤安装和激活插件：

1.  转到：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  在**下载**项下，点击下载链接并将插件保存到桌面。
3.  然后转到您的站点，登录并进入 `WP Admin > 插件 > 添加新插件 > 上传插件`
4.  上传桌面上的插件 zip 文件并激活。
5.  接下来，转到 `WP Admin > POS > 设置 > 许可证`，输入您的许可证密钥和许可证电子邮件以完成激活。

![我的下载](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在账户页面下载 WCPOS 专业版")

当新版本准备就绪时，它们将出现在您的更新仪表板中。
要了解我目前正在开发的内容，请关注[路线图](http://wcpos.com/roadmap/)。
同时关注变更日志，它将列出每个版本的新功能，[文档页面](http://docs.wcpos.com)也将更新有关如何使用新功能的信息。
