可能会出现需要从您的在线商店或POS隐藏产品的情况。
通过WCPOS，可以在产品编辑页面使用POS可见性设置来实现这一点。

### 启用仅限POS产品

仅限POS产品默认情况下是未启用的。
要启用仅限POS产品，请转到<code>WP Admin > POS > 设置 > 一般</code>

![启用仅限POS产品](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "启用仅限POS产品")

启用仅限POS产品将需要额外的数据库查找，因此会略微影响性能。
如果您不使用仅限POS产品，建议您将此选项保持为禁用状态。

### POS与在线可见性

![POS可见性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "产品编辑页面上的POS可见性设置")

| 设置                           | 描述                                                                         |
| ------------------------------ | -------------------------------------------------------------------------- |
| **POS与在线**<br />(默认)     | 产品在您的网上商店和实体店均可见                                             |
| **仅限POS**                   | 产品页面在网上不可见                                                       |
| **仅限在线**                  | 产品在POS中不可见                                                          |

### 目录可见性

如果直接访问仅限POS产品页面，您将看到<code>404 页面未找到</code>。
然而，仅限POS产品仍可能出现在在线搜索和分类页面中。
要将产品完全从您的网站中隐藏，您应该将目录可见性设置为隐藏。

![目录可见性](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "产品编辑页面上的目录可见性设置")

### 过滤仅限POS产品

您可以通过在产品管理页面使用快速链接快速查看哪些产品是可见/隐藏的：

![POS可见性过滤](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "POS可见性过滤")

### 批量编辑

要更改多个产品的POS可见性，转到您的产品列表并从<code>批量操作</code>中选择<code>编辑</code>。

![批量编辑POS可见性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "批量编辑POS可见性")
