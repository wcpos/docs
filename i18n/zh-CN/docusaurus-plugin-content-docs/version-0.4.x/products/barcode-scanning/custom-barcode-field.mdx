---
title: 自定义条形码字段
---

默认情况下，WCPOS 将使用 SKU 字段作为条形码。

![默认情况下，WCPOS 将使用 SKU 字段作为条形码。](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "默认情况下，WCPOS 将使用 SKU 字段作为条形码")

[WCPOS Pro](http://wcpos.com/pro) 用户可以通过在 `WP Admin > POS > 设置 > 常规` 中选择条形码字段来使用任何产品元字段

![在 WCPOS Pro 中选择自定义条形码字段](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "在 WCPOS Pro 中选择自定义条形码字段")

还可以使用 `woocommerce_pos_barcode_meta_key` 过滤器设置自定义条形码元字段。

```php title="functions.php"
/**
 * 用作条形码的自定义产品元字段
 *
 * 默认情况下，WCPOS 将使用 SKU 字段 '_sku'
 * 您可以使用以下 WordPress 过滤器更改元键
 *
 * 以下是流行条形码插件的元字段
 * '_ywbc_barcode_value' - YITH WooCommerce 条形码高级版
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // 更改为您的自定义元键
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
每个产品的条形码必须是唯一的。
:::
