快捷键是可在WCPOS中快速触发操作的键盘快捷方式。

| 键                         | 描述      |
| --------------------------- | ---------------- |
| <div align="center">?</div> | 帮助屏幕      |
| <div align="center">B</div> | 条码模式     |
| <div align="center">V</div> | 搜索模式      |
| <div align="center">S</div> | 与服务器同步 |
