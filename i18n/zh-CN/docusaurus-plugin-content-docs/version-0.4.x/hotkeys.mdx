热键是可用于快速触发 WCPOS 中操作的键盘快捷键。

| 键                         | 描述              |
| --------------------------- | ---------------- |
| <div align="center">?</div> | 帮助页面          |
| <div align="center">B</div> | 条形码模式        |
| <div align="center">V</div> | 搜索模式          |
| <div align="center">S</div> | 与服务器同步      |
