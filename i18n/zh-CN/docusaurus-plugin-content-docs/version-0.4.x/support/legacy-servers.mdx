---
title: 传统服务器
description: 如果您的网络主机阻止了现代的RESTful HTTP方法，您需要启用传统服务器支持。
---

HTTP（和HTTPS）是每个浏览器与Web应用程序通信的协议。
协议的一个关键部分是[HTTP方法](https://tools.ietf.org/html/rfc2616#section-5.1.1)。
传统上，Web应用程序使用非常有限的HTTP方法，如GET（检索网页）和POST（提交网络表单）。

现代Web应用程序，如WCPOS，使用REST/HTTP方法，包括以下HTTP方法：

- GET
- POST
- PUT
- PATCH
- DELETE

在一些罕见的情况下，网络主机可能会选择阻止某些HTTP方法。
如果您的网络主机阻止了现代的RESTful HTTP方法，您需要启用传统服务器支持。
