title: 旧版服务器
description: 如果您的网络主机阻止现代 RESTful HTTP 方法，您将需要启用旧版服务器支持。
---

HTTP（和 HTTPS）是每个浏览器与 Web 应用程序通信所使用的协议。
该协议的一个关键部分是 [HTTP 方法](https://tools.ietf.org/html/rfc2616#section-5.1.1)。
传统上，web 应用程序使用一套非常有限的 HTTP 方法，例如 GET（检索网页）和 POST（提交网页表单）。

现代 web 应用程序，例如 WCPOS，使用 REST/HTTP 方法，支持以下 HTTP 方法：

- GET
- POST
- PUT
- PATCH
- DELETE

在一些罕见的情况下，网络主机可能会选择阻止其中一些 HTTP 方法。
如果您的网络主机阻止现代 RESTful HTTP 方法，您将需要启用旧版服务器支持。
