title: 安全
description: 确保客户信息安全是您的责任
---

WCPOS 允许使用任何已安装的 [WooCommerce 网关](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/) 进行付款。如果支付网关收集个人信息，**确保您客户的信息安全是由您负责的**。以下是一些保持您的 POS 安全的建议：

### 使用 SSL 证书 {#use-a-ssl-certificate}

如果您正在运营一个在线商店，应该为您的网站安装安全证书。这样可以确保您发送的任何表单数据（例如登录信息和信用卡号码）都是加密的。如果您没有使用安全证书，您通过互联网发送的任何信息可能会被拦截和读取，这可能是与您共享网络的人，或者可能是在连接您的计算机与服务器的众多节点中的任何一个。一个基础的安全证书 [费用低至 16 美元](https://www.servertastic.com/rapidssl/) ，设置时间不到一天……所以没有理由不安装安全证书。

### 不要让您的 POS 无人看管 {#dont-leave-your-pos-unattended}

互联网偷窥只是您的安全性可能受到威胁的方式之一，更常见的是在恶意用户能够实际访问您的设备时会造成真正的损害。这对于经常将计算机或平板电脑放在高流量区域的店主尤其相关。为最坏的情况做好准备，假设有人偷走了您的计算机或设备，并通过实行一些安全措施来降低风险，例如：

- 当设备无人看管 5 或 10 分钟时使用锁屏
- 离开设备时注销 WordPress 和 WCPOS
- 如果您的设备被偷，快速重置您的密码
- 如果设备被偷，远程擦除您的设备

### 使用尽可能少数量的插件 {#use-the-least-number-of-plugins-possible}

WordPress 的优点之一是可以轻松获取成千上万的插件……但这也是一个隐患。每次安装插件时，您实际上是在授予该插件访问您网站及其数据的权限。我知道用各种插件来“装饰”您的商店很诱人，但这通常会使您的网站变慢，甚至可能使您的网站不安全。在安装任何插件之前，您应该问自己：

1.  您真的需要它吗？
2.  您信任该插件的作者吗？

有关保护您的 WooCommerce 商店的更多信息，请阅读 WooThemes 网站上的[这篇精彩文章](http://www.woothemes.com/2013/09/improve-your-wordpress-security-with-these-10-tips/)。
