title: 安全性
description: 确保客户信息安全是您的责任
---

WCPOS 允许使用任何已安装的 [WooCommerce 网关](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/) 进行付款。如果支付网关收集个人信息，**确保客户信息安全就取决于您**。以下是一些保持您的 POS 安全的建议：

### 使用 SSL 证书

如果您正在运行在线商店，您应该为您的网站拥有安全证书。这将确保您发送的任何表单数据（例如登录信息和信用卡号）都是加密的。如果您不使用安全证书，您通过互联网发送的任何信息都可能被拦截并读取，这可能是有人共享您的网络，或者可能是在连接您的计算机与服务器的众多节点中的任何一个。一个基本的安全证书 [费用低至 16 美元](https://www.servertastic.com/rapidssl/) ，并且设置时间不到一天……所以没有理由不拥有安全证书。

### 不要让您的 POS 无人看管

网络监听只是您的安全性受到威胁的一种方式，但更常见的真正损害发生在恶意用户获得对您设备的物理访问权限时。这对经常把计算机或平板电脑放在高流量区域的商店老板尤其相关。准备应对最坏的情况，假设有人偷走了您的计算机或设备，并通过实施一些安全措施来最小化风险，例如：

- 当设备无人看管 5 或 10 分钟时使用锁屏
- 离开设备时注销 WordPress 和 WCPOS
- 如果设备被盗，迅速重置您的密码
- 如果设备被盗，远程清除设备上的所有数据

### 尽量减少插件数量

WordPress 的一个优点是可以轻松获得成千上万的插件……但这也是一个缺点。每次安装插件时，您实际是授予该插件访问您网站及其所有经过数据的权限。我知道用各种插件“美化”您的商店是很诱人的，但这通常会使您的网站变慢，并可能使您的网站不安全。在您安装任何插件之前，您应该问自己：

1.  您真的需要它吗？
2.  您信任该插件的作者吗？
