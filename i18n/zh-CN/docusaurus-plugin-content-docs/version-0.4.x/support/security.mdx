WCPOS 允许使用任何已安装的 [WooCommerce 支付网关](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/)。如果支付网关收集个人信息，**确保客户信息安全的责任在于您**。以下是保持 POS 安全的一些建议：

### 使用 SSL 证书

如果您运营在线商店，应为您的网站购买安全证书。
这将确保您发送的任何表单数据（如登录信息和信用卡号）都经过加密。
如果不使用安全证书，您通过互联网发送的任何信息都可能被拦截和阅读，这可能是与您共享网络的人，也可能是连接您的计算机和服务器的众多节点中的任何一个。
基本的安全证书[价格低至16美元](https://www.servertastic.com/rapidssl/)，设置不到一天，所以没有不购买安全证书的借口。

### 不要让 POS 无人看管

通过互联网窥探只是您安全可能被破坏的一种方式，但更常见的真正损害是当恶意用户获得设备的物理访问权限。
对于经常将计算机或平板电脑放在高流量区域的店主来说，这一点尤为重要。
为最坏的情况做准备，即有人偷走您的计算机或设备，并通过以下安全措施来最大程度地降低风险：

- 当设备无人看管 5 到 10 分钟时使用锁屏
- 离开设备时注销 WordPress 和 WCPOS
- 如果设备被盗，迅速重置密码
- 在设备被盗的情况下远程清除设备

### 尽可能少地使用插件

WordPress 的魅力之一是可以轻松点击安装成千上万的插件，但这也是最糟糕的地方。
每次安装插件，您实际上都在授予该插件访问您网站和通过它传输的任何数据的权限。
我知道用尽可能多的插件来"装饰"您的商店很诱人，但这通常会使您的网站变慢，甚至可能使其不安全。
在安装任何插件之前，您应该问自己：

1. 您真的需要它吗？
2. 您信任该插件作者吗？
