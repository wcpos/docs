WCPOS 是 [WooCommerce](https://www.woothemes.com/woocommerce/) 的销售点插件。
本文档涵盖开源插件，以及为免费插件增加功能的 Pro 插件。

- 从 WordPress.org 下载 [WCPOS](https://wordpress.org/plugins/woocommerce-pos/)
- 升级到 [WCPOS Pro](http://wcpos.com/pro)

:::danger

请确保文档的版本号与您安装的插件的版本相匹配。

:::
