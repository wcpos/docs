WCPOS 是 [WooCommerce](https://www.woothemes.com/woocommerce/) 的收银点（Point of Sale）插件。
本文档涵盖免费插件以及增加额外功能的专业版插件。

- 从 WordPress.org 下载 [WCPOS](https://wordpress.org/plugins/woocommerce-pos/)
- 升级到 [WCPOS Pro](http://wcpos.com/pro)

:::danger

请确保文档的版本号与已安装插件的版本号一致。

:::

### 其他文档

**常见问题解答**可在 [faq.wcpos.com](http://faq.wcpos.com) 找到。
**开发者文档**可在 [api.wcpos.com](http://api.wcpos.com) 获取。
