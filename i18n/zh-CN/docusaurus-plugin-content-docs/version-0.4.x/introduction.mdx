WCPOS 是 [WooCommerce](https://www.woothemes.com/woocommerce/) 的销售点插件。
本文件涵盖开源插件，以及为免费插件增加功能的 Pro 插件。

- 从 WordPress.org 下载 [WCPOS](https://wordpress.org/plugins/woocommerce-pos/)
- 升级到 [WCPOS Pro](http://wcpos.com/pro)

:::danger

请确保文档的版本号与您安装的插件的版本相匹配。

:::

### 其他文档

常见问题的列表可以在 [faq.wcpos.com](http://faq.wcpos.com) 找到。
**开发者文档** 可在 [api.wcpos.com](http://api.wcpos.com) 获取。

如果您有文档或常见问题中未覆盖的问题，您可以通过 [support@wcpos.com](mailto:support@wcpos.com) 联系我们。
