### 自定义收据日期

WCPOS 使用 [moment.js](http://momentjs.com/) 本地化日期字符串。默认日期格式为 `"MMMM Do YYYY, h:mm:ss a"`，例如：2015年5月31日，7:20:44 下午。请查阅 [moment.js 文档](http://momentjs.com/docs/#/parsing/string-format/)了解更多关于自定义日期格式的信息。

### 订单属性

WCPOS 使用 WC REST API 的 JSON 输出来填充订单收据模板。[WC REST API 文档显示了标准 JSON 输出的示例](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order)。WCPOS 使用 `woocommerce_api_order_response` 过滤器添加了一些额外的属性。

| 属性                               | 描述                                                         |
| --------------------------------- | ------------------------------------------------------------ |
| cart_discount_tax                 | 购物车折扣的税额部分                                         |
| cashier.id                        | 收银员的用户 ID                                               |
| cashier.first_name                | 收银员的名字                                                 |
| cashier.last_name                 | 收银员的姓氏                                                 |
| payment_details.result            | 支付网关成功或失败                                           |
| payment_details.message           | 来自支付网关的任何消息                                       |
| payment_details.redirect          | 站外支付的重定向 URL，如 PayPal 标准                         |
| payment_details.method\_`gateway` | 任何网关特定的消息，如实收金额和找零                         |
| shipping_lines[i].total_tax       | 为每个配送线路添加税额                                       |
| subtotal_tax                      | 小计的税额部分                                               |
