---
title: 按 POS 或在线销售筛选订单
description: 通过 WooCommerce 管理后台中的快捷链接，区分 POS 和在线订单。
---

您可以通过左上角的快捷链接，在 WooCommerce 管理后台中快速区分 POS 和在线订单。
**请注意：**此功能仅会筛选 0.3.1 版本之后的 POS 订单，0.3.1 之前的订单将默认显示为在线订单。

![筛选订单](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "筛选订单")

此外，如果您希望在管理后台中过滤掉所有已完成的订单，以便专注于待处理的订单，可以将以下代码添加到您的 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 文件中。

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
