---
title: "通过 POS 或在线销售过滤订单"
description: "通过使用 WooCommerce 管理中的快捷链接来分离 POS 和在线订单。"
---

您可以通过使用左上角的快捷链接快速分离 WooCommerce 管理中的 POS 和在线订单。
**请注意：** 这仅会过滤版本 0.3.1 之后的 POS 订单，0.3.1 之前的订单将默认视为在线订单。

![过滤订单](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "过滤订单")

此外，如果您想从管理区域过滤所有已完成的订单，以便能够专注于待处理/正在处理的订单，可以将以下代码添加到您的 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 文件中。

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
