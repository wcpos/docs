---
title: 按 POS 和线上销售筛选订单
description: 使用 WooCommerce 管理后台的快速链接分隔 POS 和线上订单。
---

你可以通过左上角的快速链接在 WooCommerce 管理后台快速分隔 POS 和线上订单。
**请注意：** 这仅适用于 0.3.1 版本之后的 POS 订单，0.3.1 版本之前的订单默认为线上订单。

![筛选订单](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "筛选订单")

此外，如果你想在管理后台筛选所有已完成的订单以便专注于待处理/处理中的订单，可以在 [functions.php](http://codex.wordpress.org/Functions_File_Explained) 文件中添加以下代码。

```php title="functions.php"
/**
 * 更改 WP 管理后台订单的默认筛选
 * 将此代码添加到你的 functions.php 文件
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
