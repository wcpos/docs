WCPOS 提供两个版本。**免费插件**包含在柜台收款、打印收据以及将订单同步回 WooCommerce 所需的一切功能。**专业版**增加了管理界面（产品、订单、客户、报表）、更多支付网关、优惠券、退款和多店铺支持。

如果您主要需要一个记录销售的收银台，免费插件就已足够。如果您还想通过 POS 管理店铺的日常运营——管理库存、查询历史订单、为客户退款、通过集成读卡器接受刷卡支付——这正是专业版的用途。

|---|---|---|

:::note 免费版中仅限 POS 端的客户/优惠券访问权限
在免费插件中，您仍然可以**选择**现有客户或在收银台**查看**优惠券目录——只是无法添加/编辑客户，也无法在结账时使用优惠券代码。这些功能在 Pro 版本中可用。
:::

## Pro 版新增功能 {#what-pro-adds}

Pro 版解锁其余后台管理功能、更多支付选项和按店铺配置的功能。以下所有项目均为 Pro 专属。

### 管理界面 {#management-screens}

免费插件提供用于收银的产品面板。Pro 版新增完整的管理界面：

| 界面 | 功能说明 |
|---|---|
| **[Products](/products)** | Edit stock levels, prices, and cost of goods sold inline. Bulk operations across the catalogue. |
| **[Orders](/orders)** | Look up historical orders, reprint receipts, edit orders, and access the refund flow. |
| **[Customers](/customers)** | Add new customers and edit existing customer details (free users can only select from a picker at checkout). |
| **[Reports](/reports)** | End-of-day sales reports — totals by payment method, cashier, and store, suitable for cash reconciliation. |
| **[Coupons](/coupons)** | Browse the WooCommerce coupon catalogue from the till (free users see a blurred preview only). |

### Payment gateways {#payment-gateways}

The free plugin's "Card" gateway just records a sale that you took on an external terminal — there's no integration with the reader itself. Pro lets you use **any WooCommerce payment gateway** at the POS, and bundles several integrated gateways:

| Gateway | Use case |
|---|---|
| **[Stripe Terminal](/payment/custom-gateways/stripe-terminal)** | Direct integration with Stripe's S700 / WisePOS E readers. Supports MOTO (phone orders) and a simulator for testing. |
| **[SumUp Terminal](/payment/custom-gateways/sumup-terminal)** | Pair a SumUp card reader to the POS and complete card payments without leaving the till. |
| **[Vipps MobilePay](/payment/custom-gateways/vipps-mobilepay)** | Phone-based payments via QR code or push notification — Vipps in Norway, MobilePay in Denmark and Finland. |
| **[邮件账单](/payment/custom-gateways/email-invoice)** | 通过电子邮件向客户发送付款链接，客户在线完成付款后订单自动完成。 |
| **[网页结账](/payment/custom-gateways/web-checkout)** | 将客户重定向到托管的结账页面，以在线完成付款。 |
| **任何其他 WooCommerce 网关** | 使用[网关模板](/payment/custom-gateways)封装任何 WooCommerce 支付插件，用于 POS 结账。 |

## 准备升级？ {#ready-to-upgrade}

前往[安装 WCPOS Pro](/getting-started/pro-license) 了解下载和激活步骤。

- **v1.8+ Pro** ——请先停用并删除免费版插件。从 v1.8 起，Pro 是一个独立插件，不应与免费版同时安装。
- **旧版 Pro（< v1.8）** ——如果您的 Pro 版本需要免费版插件，请保留其安装。
