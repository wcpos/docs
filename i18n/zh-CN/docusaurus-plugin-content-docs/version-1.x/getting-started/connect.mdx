## 连接屏幕概述

当您打开 WCPOS 桌面或移动应用时，您将看到连接屏幕。在这里，您可以管理商店连接和用户登录。

{/* Screenshot: connect-screen.png */}

## 添加新商店

1. 在文本框中输入您的 WooCommerce 商店网址（例如 `https://mystore.com`）
2. 点击**连接**
3. 您将被重定向到使用 WordPress 凭据登录
4. 成功登录后，您将返回到应用

## 多个商店

您可以连接到所需数量的 WooCommerce 商店。每个商店都会在连接屏幕上显示为单独的卡片，包括：

- **商店名称**和网站图标
- **商店网址**
- 该商店的**已登录用户**

这对于管理多个地点或业务很有帮助。

## 每个商店的多个用户

每个商店可以同时有多个用户登录。这对以下情况很有帮助：

- **轮班更换** - 新收银员可在前一个收银员退出前登录
- **多收银台设置** - 不同设备上的不同收银员
- **快速切换** - 轻松在用户账户之间切换

### 添加用户

点击"已登录用户"旁边的 <Icon name="plus" /> 按钮，向该商店添加另一个用户。

### 切换用户

点击用户徽章（例如"Brenda"）以该用户身份打开 POS。

### 删除用户

点击用户徽章上的**×**以将该用户从商店登出。

## 删除商店

点击商店卡片上的红色**×**按钮，将其从列表中删除。这将注销所有用户并从应用中移除商店连接。

## 演示商店

在屏幕底部，您可能会看到一个"进入演示商店"链接。这将连接到一个演示 WooCommerce 商店，让您在不影响自己的商店数据的情况下试用 WCPOS 功能。

## 故障排除

### 无法连接到我的商店？

- 确保在您的 WordPress 站点上已安装并激活 WCPOS 插件
- 检查您使用的是正确的网址（如果站点使用 SSL，请包含 `https://`）
- 验证 WooCommerce REST API 是否可访问
- 检查您的用户账户是否有 POS 访问权限

### 连接持续失败？

- 首先在网页浏览器中尝试访问 `yourdomain.com/pos`，验证插件是否正常工作
- 检查您站点的错误日志以查找任何问题
- 确保您的服务器符合[最低要求](/getting-started/installation)
