## 连接屏幕概述

当您打开 WCPOS 桌面或移动应用时，您将看到连接屏幕。在这里，您可以管理商店连接和用户登录。

{/* 截图：connect-screen.png */}

## 添加新商店

1. 在文本框中输入您的 WooCommerce 商店 URL（例如，`https://mystore.com`）
2. 单击 **连接**
3. 您将被重定向以使用您的 WordPress 凭据登录
4. 登录成功后，您将返回到应用

## 多个商店

您可以根据需要连接多个 WooCommerce 商店。每个商店在连接屏幕上作为单独的卡片出现，显示：

- **商店名称** 和小图标
- **商店 URL**
- 此商店的 **已登录用户**

这对于您管理多个地点或业务非常有用。

## 每个商店多个用户

每个商店可以同时有多个用户登录。这对于以下情况非常有帮助：

- **换班** - 新收银员可以在前一位收银员注销之前登录
- **多收银机设置** - 不同的收银员在不同的设备上
- **快速切换** - 轻松切换用户账户

### 添加用户

单击“已登录用户”旁边的 <Icon name="plus" /> 按钮以为该商店添加另一个用户。

### 切换用户

单击用户徽章（例如，“Brenda”）以该用户身份打开 POS。

### 移除用户

单击用户徽章上的 **×** 以注销该商店的用户。

## 移除商店

单击商店卡片上的红色 **×** 按钮以将其从您的列表中移除。这将注销所有用户并从应用中移除商店连接。

## 演示商店

在屏幕底部，您可能会看到一个“进入演示商店”链接。此链接将您连接到一个演示 WooCommerce 商店，让您在不影响自己商店数据的情况下试用 WCPOS 功能。

## 故障排除

### 无法连接到我的商店？

- 确保在您的 WordPress 网站上安装并激活了 WCPOS 插件
- 检查您使用的 URL 是否正确（如果您的网站使用 SSL，请包含 `https://`）
- 验证 WooCommerce REST API 是否可访问
- 检查您的用户账户是否具有 POS 访问权限

### 连接失败？

- 先尝试在网络浏览器中访问 `yourdomain.com/pos` 以验证插件是否正常工作
- 检查您网站的错误日志以查找任何问题
- 确保您的服务器满足 [最低要求](/getting-started/installation)
