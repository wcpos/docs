## 连接屏幕概述

当您打开 WCPOS 桌面或移动应用时，您将看到连接屏幕。这是您管理商店连接和用户登录的地方。

{/* 截图: connect-screen.png */}

## 添加新商店

1. 在文本字段中输入您的 WooCommerce 商店 URL（例如，`https://mystore.com`）
2. 点击 **连接**
3. 系统将重定向您登录您的 WordPress 凭据
4. 登录成功后，您将返回到应用程序

## 多个商店

您可以根据需要连接多个 WooCommerce 商店。每个商店在连接屏幕上显示为一个单独的卡片，显示：

- **商店名称** 和图标
- **商店 URL**
- **已登录用户** 列表

如果您管理多个地点或业务，这很有用。

## 每个商店多个用户

每个商店可以同时有多个用户登录。这对于以下情况很有帮助：

- **换班** - 新收银员可以在前一个退出之前登录
- **多收银机设置** - 不同的收银员在不同的设备上
- **快速切换** - 容易在用户帐户之间切换

### 添加用户

点击 "已登录用户" 旁边的 <Icon name="plus" /> 按钮以添加另一个用户到该商店。

### 切换用户

点击用户徽章（例如，“Brenda”）以该用户身份打开 POS。

### 移除用户

点击用户徽章上的 **×** 以将该用户从商店中登出。

## 移除商店

点击商店卡片上的红色 **×** 按钮以将其从您的列表中移除。这会登出所有用户并从应用中移除商店连接。

## 演示商店

在屏幕底部，您可能会看到一个“进入演示商店”的链接。这将连接您到一个演示 WooCommerce 商店，以尝试 WCPOS 功能而不影响您自己的商店数据。

## 故障排除

<Accordion>
  <AccordionItem question="无法连接到我的商店？">

  - 确保 WCPOS 插件已安装并在您的 WordPress 网站上激活
  - 检查您使用的 URL 是否正确（如果您的网站使用 SSL，请包含 `https://`）
  - 验证 WooCommerce REST API 是否可访问
  - 确保您的用户帐户具有 POS 访问权限

  </AccordionItem>

  <AccordionItem question="连接失败？">

  - 尝试首先在网页浏览器中访问 `yourdomain.com/pos` 以验证插件是否正常工作
  - 检查您网站的错误日志以查找任何问题
  - 确保您的服务器满足 [最低要求](/getting-started/installation)

  </AccordionItem>
</Accordion>
