## 什么是 WCPOS？ {#what-is-wcpos}

WCPOS 是一个用于 WooCommerce 的销售点 (POS) 系统。它允许您在实体位置使用网络浏览器或专用应用程序直接销售 WooCommerce 商店中的产品。

## 5 分钟开始使用 {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="安装插件">

在您的 WordPress 网站上安装免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/)：

1. 转到 **WP 管理 → 插件 → 添加新**
2. 搜索 "WooCommerce POS"
3. 点击 **立即安装**，然后 **激活**

有关详细要求，请参见 [安装](/getting-started/installation)。

</StepCard>

<StepCard number={2} title="访问 POS">

**Web 用户：** 在浏览器中导航到 `yourdomain.com/pos`。系统会提示您使用 WordPress 凭据登录。

**桌面/移动用户：** 下载应用程序，输入您的商店 URL，点击 **连接**，并使用您的 WordPress 凭据登录。

有关管理多个商店和用户的详细信息，请参见 [连接](/getting-started/connect)。

</StepCard>

<StepCard number={3} title="等待初始同步">

WCPOS 会分批下载产品以避免给您的服务器带来过重的负担。当您搜索、滚动或筛选时，产品会逐步下载——您无需等待所有内容同步完毕就可以开始销售。

有关详细信息，请参见 [产品同步](/products/sync)。

</StepCard>

<StepCard number={4} title="开始销售" isLast>

- 搜索或扫描产品以将其添加到购物车
- 选择客户或使用 "游客"
- 点击 **结账** 以处理付款
- 打印或通过电子邮件发送收据

</StepCard>

</Steps>

## 主要特性 {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="快速本地搜索">
    产品本地存储以实现即时搜索结果
  </FeatureCard>
  <FeatureCard icon="wifi" title="离线浏览">
    无需互联网搜索产品和客户（订单需要连接）
  </FeatureCard>
  <FeatureCard icon="barcode" title="条形码扫描">
    使用任何 USB 或蓝牙条形码扫描器
  </FeatureCard>
  <FeatureCard icon="credit-card" title="现金与卡片付款">
    内置支持现金和基本卡片付款
  </FeatureCard>
  <FeatureCard icon="print" title="收据打印">
    打印或通过电子邮件发送可自定义的收据
  </FeatureCard>
  <FeatureCard icon="calculator" title="自动税务">

  </FeatureCard>
</FeatureGrid>
