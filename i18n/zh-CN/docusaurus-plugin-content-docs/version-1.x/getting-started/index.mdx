## 什么是 WCPOS？ {#what-is-wcpos}

WCPOS 是一个 WooCommerce 的销售点 (POS) 系统。它允许您在一个实体位置，通过网络浏览器或专用应用程序，出售您 WooCommerce 商店中的产品。

## 5 分钟内入门 {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="安装插件">

在您的 WordPress 网站上安装免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/)：

1. 转到 **WP 管理 → 插件 → 添加新**
2. 搜索 "WooCommerce POS"
3. 点击 **立即安装**，然后 **激活**

请参阅 [安装](/getting-started/installation) 以获取详细要求。

</StepCard>

<StepCard number={2} title="访问销售点">

**网页用户：** 在您的浏览器中导航到 `yourdomain.com/pos`。系统会提示您使用 WordPress 证书登录。

**桌面/移动用户：** 下载应用程序，输入您的商店网址，点击 **连接**，并使用您的 WordPress 证书登录。

请参阅 [连接](/getting-started/connect) 以获取有关管理多个商店和用户的详细信息。

</StepCard>

<StepCard number={3} title="等待初始同步">

WCPOS 会以小批量下载产品，以避免过载您的服务器。随着您搜索、滚动或筛选，产品将逐步下载——您无需等待所有内容同步完毕即可开始销售。

有关详细信息，请参见 [产品同步](/products/sync)。

</StepCard>

<StepCard number={4} title="开始销售" isLast>

- 搜索或扫描产品以添加到购物车
- 选择客户或使用 "访客"
- 点击 **结账** 进行付款
- 打印或电邮收据

</StepCard>

</Steps>

## 主要功能 {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="快速本地搜索">
    产品本地存储以实现即时搜索结果
  </FeatureCard>
  <FeatureCard icon="wifi" title="离线浏览">
    在没有互联网的情况下搜索产品和客户（订单需要连接）
  </FeatureCard>
  <FeatureCard icon="barcode" title="条形码扫描">
    使用任何 USB 或蓝牙条形码扫描仪
  </FeatureCard>
  <FeatureCard icon="credit-card" title="现金与卡片支付">
    内置支持现金和基本卡片支付
  </FeatureCard>
  <FeatureCard icon="print" title="收据打印">
    打印或电邮可定制的收据
  </FeatureCard>
  <FeatureCard icon="calculator" title="自动税务">
    基于您的 WooCommerce 税务设置进行税务计算
  </FeatureCard>
</FeatureGrid>
