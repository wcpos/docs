:::info Pro 功能
[WCPOS Pro](https://wcpos.com/pro) 是一款付费插件，为 WCPOS 提供管理界面、额外的支付网关及其他 Pro 专属功能。

**自 v1.8+ 起，Pro 是一个独立插件** — 无需同时安装免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/)。
:::

## Pro 包含哪些功能？ {#whats-included-in-pro}

WCPOS Pro 解锁以下附加功能：

- **商品界面** - 直接在 POS 中编辑库存数量和价格
- **订单界面** - 查看和管理订单历史
- **客户界面** - 添加和编辑客户信息
- **报表界面** - 生成日终销售报表
- **更多支付网关** - Stripe Terminal、SumUp 及自定义网关

## 安装 {#installation}

如果您已购买 WCPOS Pro 许可证，请按照以下步骤安装并激活插件：

1. 前往：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. 在**下载**部分，点击下载链接并将插件保存到桌面。
3. 然后前往您的站点，登录并进入 `WP Admin > 插件 > Add Plugin > 上传插件`
4. 从桌面上传插件 zip 文件并激活。
5. 接下来，前往 `WP Admin > POS > 设置 > License`，输入您的许可证密钥和许可证邮箱以完成激活。

:::note 从旧版本升级？
如果您之前同时安装了免费的 WCPOS 插件和旧版 Pro 附加组件，请在激活新的独立 Pro 插件之前**停用并删除免费插件**。同时运行两个插件可能会导致激活错误，例如 *"Invalid response fetching remote state"* 或致命的 *"Class not found"* 错误。
:::

## 更新 {#updates}

新版本发布后，将像其他插件一样显示在更新仪表盘中。您也可以从[账户页面](https://wcpos.com/my-account/)下载最新版本。

## 故障排除 {#purchase-a-license}

### 激活失败 {#activation-failures}

- **"Invalid licence key"** — 请仔细检查粘贴时是否包含多余空格，并确认电子邮件地址与购买时使用的一致。
- **"Licence limit reached"** — 每个许可证覆盖一定数量的站点。请从[账户页面](https://wcpos.com/my-account/)停用未使用的站点，或升级您的方案。

### 许可证过期 {#expired-licence}

- 许可证过期后，Pro 功能仍保持可用，但将无法接收更新。
- 请从[账户页面](https://wcpos.com/my-account/)续订——现有密钥保持不变。

### 域名转移 {#domain-transfers}

- 如果您已将站点迁移到新域名，请先在旧域名上停用许可证，然后在新域名上重新激活。
- 如果旧域名已无法访问，请联系 **support@wcpos.com** 重置激活。

### 许可证服务器连接 {#licence-server-connectivity}

- 激活需要您的 WordPress 服务器能够访问 WCPOS 许可证服务器。
- 如果激活无提示地失败，请检查出站 HTTPS 请求是否被主机防火墙或安全插件阻止。
- 部分托管主机默认会阻止外部 API 调用——请联系您的主机服务商将 `wcpos.com` 加入白名单。

### 查看许可证状态 {#checking-licence-status}

- 前往 `WP Admin > POS > 设置 > License` 查看当前激活状态。
- 您的[账户页面](https://wcpos.com/my-account/)会显示所有已激活的站点及到期日期。

## 购买许可证 {#purchase-a-license}

访问 [wcpos.com/pro](https://wcpos.com/pro) 购买 Pro 许可证。
