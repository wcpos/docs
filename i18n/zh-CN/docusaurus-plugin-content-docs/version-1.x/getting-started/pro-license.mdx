:::info Pro 特性
[WCPOS Pro](https://wcpos.com/pro) 是一个付费插件，可为免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/) 添加功能。
:::

## Pro 中包含什么？ {#whats-included-in-pro}

WCPOS Pro 解锁了额外的功能：

- **产品页面** - 直接在 POS 中编辑库存水平和价格
- **订单页面** - 查看和管理订单历史
- **客户页面** - 添加和编辑客户信息
- **报告页面** - 生成日终销售报告
- **额外的支付网关** - Stripe Terminal、SumUp 和自定义网关

## 安装 {#installation}

如果您已购买 WCPOS Pro 的许可证，请按照以下步骤安装和激活插件：

1. 访问: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. 在 **下载** 下，点击下载链接并保存插件到您的桌面。
3. 然后访问您的网站，登录并转到 `WP Admin > 插件 > 添加插件 > 上传插件`
4. 从您的桌面上传插件 zip 文件并激活。
5. 接下来，转到 `WP Admin > POS > 设置 > 许可证`，输入您的许可证密钥和许可证电子邮件以完成激活。

![我的下载](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在您的账户页面下载 WCPOS Pro")

## 更新 {#updates}

当新版本准备好时，它们将在您的更新仪表板中像其他插件一样出现。您也可以从您的 [账户页面](https://wcpos.com/my-account/) 下载最新版本。

## 购买许可证 {#purchase-a-license}

访问 [wcpos.com/pro](https://wcpos.com/pro) 购买 Pro 许可证。
