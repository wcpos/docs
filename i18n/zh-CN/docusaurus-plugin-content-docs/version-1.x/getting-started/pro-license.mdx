:::info 专业功能
[WCPOS Pro](https://wcpos.com/pro) 是一个付费插件，为免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/) 添加了更多功能。
:::

## 专业版包含什么？

WCPOS Pro 解锁了额外的功能：

- **产品页面** - 直接在 POS 中编辑库存水平和价格
- **订单页面** - 查看和管理订单历史
- **客户页面** - 添加和编辑客户信息
- **报告页面** - 生成每日销售报告
- **额外的支付网关** - Stripe Terminal、SumUp 和自定义网关

## 安装

如果您已购买 WCPOS Pro 的许可证，请按照以下步骤安装和激活插件：

1. 转到：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. 在 **下载** 下，点击下载链接并将插件保存到桌面。
3. 然后访问您的站点，登录并转到 `WP Admin > 插件 > 添加新 > 上传插件`
4. 从桌面上传插件 zip 文件并激活。
5. 接下来，转到 `WP Admin > POS > 设置 > 许可证`，输入您的许可证密钥和许可证电子邮件以完成激活。

![我的下载](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在账户页面下载 WCPOS Pro")

## 更新

当新版本准备好时，它们将在您的更新仪表板中显示，像任何其他插件一样。您也可以从您的 [账户页面](https://wcpos.com/my-account/) 下载最新版本。
