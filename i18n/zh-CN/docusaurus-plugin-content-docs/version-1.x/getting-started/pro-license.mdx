:::info Pro 功能
[WCPOS Pro](https://wcpos.com/pro) 是一款付费插件，用于支持 WCPOS 中的管理界面、其他支付网关以及仅 Pro 可用的功能。

**自 v1.8+ 起，Pro 是独立插件** — 无需同时安装免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/)。
::: 

## Pro 包含哪些内容？ {#whats-included-in-pro}

WCPOS Pro 可解锁更多功能：

- **产品界面** - 直接在 POS 中编辑库存水平和价格
- **订单界面** - 查看和管理订单历史记录
- **客户界面** - 添加和编辑客户信息
- **报告屏幕** - 生成日终销售报告
- **其他支付网关** - Stripe Terminal、SumUp 和自定义网关

## 安装 {#installation}

如果您已购买 WCPOS Pro 许可证，请按照以下步骤安装并启用插件：

:::note 从旧版本升级？
如果您之前同时安装了免费的 WCPOS 插件和旧版 Pro 附加组件，请在启用新的独立版 Pro 插件之前，**停用并删除免费插件**。同时运行两者可能会导致激活错误，例如 *"Invalid response fetching remote state"*，或致命的 *"Class not found"* 错误。
::: 

1. 前往：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. 在 **下载** 下，点击下载链接并将插件保存到桌面。
3. 然后前往您的站点，登录并进入 `WP Admin > 插件 > Add Plugin > 上传插件`
4. 从桌面上传插件 ZIP 文件并启用。
5. 接下来，前往 `WP Admin > POS > 设置 > License`，输入您的许可证密钥和许可证邮箱以完成激活。

![我的下载](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "可在您的账户页面下载 WCPOS Pro")

## 更新 {#updates}

新版本就绪后，会像其他插件一样显示在您的“更新”仪表盘中。您也可以从[账户页面](https://wcpos.com/my-account/)下载最新版本。

## 故障排除 {#troubleshooting}

### 激活失败 {#activation-failures}

- **"许可证密钥无效"** — 粘贴时请仔细检查末尾是否有空格，并确认电子邮件地址与购买时使用的地址一致。
- **"已达到许可证限制"** — 默认情况下，一个许可证可在 **2 个域名**上激活（用于一个正式站点和一个暂存站点）。请注意，**子域名会计为单独激活** — `example.com`、`staging.example.com` 和 `dev.example.com` 计为三个。这通常会影响 Pantheon 这类强制使用开发/测试/正式环境的主机。请从[账户页面](https://wcpos.com/my-account/)停用不再使用的站点，或发送电子邮件至 **support@wcpos.com** — 通常可按请求授予额外激活次数。

### 许可证已过期 {#expired-licence}

- 许可证过期后，Pro 功能仍会保持启用，但您将不会收到更新。
- 请从[账户页面](https://wcpos.com/my-account/)续订 — 现有密钥保持不变。

### 域名转移 {#domain-transfers}

- 如果已将站点迁移到新域名，请先在旧域名上停用许可证，然后在新域名上重新激活。
- 如果旧域名已无法访问，请联系 **support@wcpos.com** 重置激活。

### 许可证服务器连接 {#licence-server-connectivity}

- 激活需要 WordPress 服务器能够访问 WCPOS 许可证服务器。
- 如果激活失败但没有显示错误，请检查主机防火墙或安全插件是否阻止了出站 HTTPS 请求。
- 某些托管主机默认会阻止外部 API 调用，请联系主机服务商将 `wcpos.com` 加入白名单。

### 检查许可证状态 {#checking-licence-status}

- 前往 `WP Admin > POS > 设置 > License` 查看当前激活状态。
- [账户页面](https://wcpos.com/my-account/)会显示所有已激活的站点和到期日期。

## 购买许可证 {#purchase-a-license}

访问 [wcpos.com/pro](https://wcpos.com/pro) 购买 Pro 许可证。

:::note 购买和升级在网站上完成
应用内没有“升级到 Pro”按钮——购买、套餐升级和续订都在 [wcpos.com](https://wcpos.com/pro) 和你的[账户页面](https://wcpos.com/my-account/)处理。购买后，请按照[上述步骤](#installation)安装 Pro。
:::
