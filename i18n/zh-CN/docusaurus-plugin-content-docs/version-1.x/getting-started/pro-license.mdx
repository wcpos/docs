:::info Pro Feature
[WCPOS Pro](https://wcpos.com/pro) 是一个付费插件，为免费的 [WCPOS 插件](https://wordpress.org/plugins/woocommerce-pos/) 添加了更多功能。
:::

## 专业版包含什么？

WCPOS Pro 解锁了额外的功能：

- **Products Screen** - 直接在 POS 中编辑库存水平和价格
- **Orders Screen** - 查看和管理订单历史
- **Customers Screen** - 添加和编辑客户信息
- **Reports Screen** - 生成日终销售报告
- **Additional Payment Gateways** - Stripe Terminal、SumUp 和自定义网关

## 安装

如果您已购买 WCPOS Pro 的许可证，请按照以下步骤安装和激活插件：

1. 转到：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. 在 **Downloads** 下，点击下载链接并将插件保存到您的桌面。
3. 然后访问您的站点，登录并转到 `WP Admin > Plugins > Add New > Upload Plugin`
4. 从桌面上传插件 zip 文件并激活。
5. 接下来，转到 `WP Admin > POS > Settings > License`，输入您的 License Key 和 License Email 以完成激活。

![My Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在账户页面下载 WCPOS Pro")

## Updates

当新版本准备好时，它们将在您的 Updates 仪表板中显示，像任何其他插件一样。您也可以从您的 [account page](https://wcpos.com/my-account/) 下载最新版本。
