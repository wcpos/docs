为确保 WCPOS 插件的正常功能和最佳性能，请确保您的环境满足以下最低要求：

## 服务器要求

- **WordPress：** 5.6 或更高版本
- **WooCommerce：** 5.3 或更高版本
- **PHP：** 7.2 或更高版本

满足这些最低要求可确保 WCPOS 插件与您的 WordPress 环境兼容且稳定。这还确保您可以受益于插件提供的最新功能和增强功能。

## 浏览器要求

- **现代网络浏览器：** WCPOS 插件设计用于与现代网络浏览器兼容，包括 Google Chrome、Mozilla Firefox、Safari 和 Microsoft Edge。请确保使用您首选浏览器的最新稳定版本。
- **启用 JavaScript：** WCPOS 插件依赖 JavaScript 实现交互功能。请确保在浏览器设置中启用 JavaScript。
- **非隐私浏览模式：** 浏览器不应处于隐私浏览模式，因为这可能限制对浏览器的 IndexedDB（IDB）数据库的访问。隐私浏览模式可能会阻止 WCPOS 插件的某些功能正常运行。

## 安装插件

1. 转到 `WP 管理后台 > 插件 > 添加新插件`
2. 搜索"WooCommerce POS"
3. 点击**立即安装**，然后**激活**

或者，您可以从 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) 下载插件并手动上传。

## 访问 POS

激活后，您可以通过以下方式访问 POS：

- **网页：** `yourdomain.com/pos`
- **桌面/移动：** 从 [wcpos.com](https://wcpos.com) 下载应用并连接到您的店铺

通过满足最低要求并使用兼容的网络浏览器，您可以流畅地使用 WCPOS 插件，并充分发挥其潜力以高效管理您的销售点业务。
