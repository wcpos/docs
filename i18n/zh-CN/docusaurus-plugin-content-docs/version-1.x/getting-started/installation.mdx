为了确保 WCPOS 插件的正常功能和最佳性能，请确保您的环境满足以下最低要求：

## 服务器要求 {#server-requirements}

- **WordPress：** 版本 5.6 或更高
- **WooCommerce：** 版本 5.3 或更高
- **PHP：** 版本 7.2 或更高

满足这些最低要求可以确保 WCPOS 插件与您的 WordPress 环境之间的兼容性和稳定性。这也确保您能够享受插件提供的最新功能和增强。

## 浏览器要求 {#browser-requirements}

- **现代网页浏览器：** WCPOS 插件旨在与现代网页浏览器兼容，包括谷歌 Chrome、莫zilla Firefox、Safari 和微软 Edge。请确保您使用最新的稳定版本的首选浏览器。
- **启用 JavaScript：** WCPOS 插件依赖 JavaScript 来实现其交互功能。请确保在您的浏览器设置中启用了 JavaScript。
- **未使用隐私浏览模式：** 浏览器不应处于隐私浏览模式，因为这可能会限制对浏览器的 IndexedDB (IDB) 数据库的访问。隐私浏览模式可能会影响 WCPOS 插件某些功能的正常运行。

## 安装插件 {#installing-the-plugin}

1. 转到 `WP Admin > Plugins > Add New`
2. 搜索 "WooCommerce POS"
3. 点击 **立即安装**，然后点击 **激活**

或者，您可以从 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) 下载插件并手动上传。

## 访问 POS {#accessing-the-pos}

激活后，您可以在以下地址访问 POS：

- **网页：** `yourdomain.com/pos`
- **桌面：** [Windows](https://updates.wcpos.com/electron/download/win32-x64)，[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)，[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **移动：** [iOS](https://testflight.apple.com/join/JGBdVRrW)，[Android](https://play.google.com/apps/testing/com.wcpos.main)

通过满足最低要求并使用兼容的网页浏览器，您可以顺利使用 WCPOS 插件，并充分发挥其全部潜力来高效管理您的销售点操作。
