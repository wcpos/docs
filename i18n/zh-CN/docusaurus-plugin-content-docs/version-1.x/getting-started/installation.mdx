为确保 WCPOS 插件正常运行并获得最佳性能，请确认环境满足以下最低要求：

## 服务器要求 {#server-requirements}

WCPOS 插件运行在 WordPress 站点上，该站点必须满足以下最低要求：

| 组件        | 最低版本        | 备注            |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 推荐使用 8.x    |

满足这些最低要求可确保与 WordPress 环境的兼容性和稳定性，并可使用最新的 WCPOS 功能。

## 浏览器要求 {#browser-requirements}

POS 的网页版可在任何现代浏览器中运行。请使用最新的稳定版本：

| 浏览器         | 支持的版本 |
| --------------- | ----------------- |
| Google Chrome 浏览器   | 最新稳定版     |
| Mozilla Firefox 浏览器 | 最新稳定版     |
| Safari 浏览器          | 最新稳定版     |
| Microsoft Edge 浏览器  | 最新稳定版     |

:::warning 避免使用隐私/无痕浏览模式
隐私浏览模式可能会阻止访问浏览器的 IndexedDB 数据库，而 WCPOS 使用该数据库离线存储数据——部分功能将无法正常工作。同时还必须启用 JavaScript。
:::

## 应用要求 {#app-requirements}

WCPOS 还提供桌面端和移动端应用。各应用所需的最低操作系统版本如下：

| 平台        | 最低操作系统                | 架构              |
| --------------- | ------------------------- | -------------------------- |
| Windows 桌面版 | Windows 10                | x64、ARM64                 |
| macOS 桌面版   | macOS 12（Monterey）       | Intel（x64）、Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| 安卓         | 安卓 10（API 级别 29） | —                          |

:::note 打印权限
蓝牙和网络打印需要在提示时授予应用相关的蓝牙和本地网络权限。
:::

## 安装插件 {#installing-the-plugin}

1. 前往 `WP Admin > 插件 > Add Plugin`
2. 搜索“wcpos”
3. 点击**立即安装**，然后点击**启用**

也可以从 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) 下载插件并手动上传。

## 访问 POS {#accessing-the-pos}

激活后，可通过以下方式访问 POS：

- **网页端：** `yourdomain.com/pos`
- **桌面端：** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac（Intel）](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac（Apple Silicon）](https://updates.wcpos.com/electron/download/darwin-arm64)
- **移动端：** [iOS](https://testflight.apple.com/join/JGBdVRrW)、[安卓](https://play.google.com/apps/testing/com.wcpos.main)

满足最低要求并使用兼容的网页浏览器，即可顺畅使用 WCPOS 插件，并充分发挥其功能，高效管理销售点运营。
