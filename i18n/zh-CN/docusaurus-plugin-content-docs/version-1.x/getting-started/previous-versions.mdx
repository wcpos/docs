## 重新安装旧版本 {#re-installing-a-previous-version}

如果您需要恢复到 WCPOS 插件的旧版本，请按照以下步骤进行：

1. 访问 WCPOS 的 WordPress.org 页面 [高级视图](https://wordpress.org/plugins/woocommerce-pos/advanced/)。

2. 向下滚动到 **高级选项** 部分，并找到 **旧版本** 选择菜单。

3. 打开选择菜单，您将看到可用插件版本的列表。找到您想要安装的版本，并单击其旁边的 **下载** 按钮。将下载的文件保存到您的桌面或方便的位置。

4. 登录到您的 WordPress 管理仪表板。

5. 通过单击左侧菜单中的 **插件** 进入 **插件** 页面。

6. 禁用并删除当前版本的 WCPOS 插件。这将从您的 WordPress 安装中移除该插件。请放心，您的数据不会丢失。

7. 单击页面顶部的 **添加新** 按钮。

8. 在 **添加插件** 页面上，单击 **上传插件** 按钮。

9. 从您的桌面或保存位置选择之前下载的插件文件。

<div class="image-container">
  <Image
    alt="上传插件"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">在 WordPress 管理中上传插件</p>
</div>

10. 单击 **立即安装** 按钮，等待读取完成。

11. 安装完成后，单击 **激活** 按钮以激活 WCPOS 插件的旧版本。

就这样！您已成功重新安装 WCPOS 插件的旧版本。

如果您在重新安装过程中遇到任何问题或需要进一步的帮助，请通过 support@wcpos.com 联系我们的支持团队。
