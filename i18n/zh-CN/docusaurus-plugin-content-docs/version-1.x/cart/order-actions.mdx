### WooCommerce 中的元数据是什么？

元数据提供有关订单的额外信息，可通过 WooCommerce REST API 访问。开发人员和插件使用这些数据来扩展功能，例如集成第三方服务。

### JSON 视图

- 查看订单的原始 JSON 表示，可用于调试或集成目的。

## 保存到服务器

- 允许您将打开的购物车保存到服务器，以便安全地存储并可供将来使用。

### 为什么要保存？

在 POS 中创建新订单时，它只存在于本地。
保存订单可确保即使清除本地数据库，订单仍会保留。

## 无效

- 两种情况：
- 未保存的订单：
- 如果订单尚未保存到服务器，它将从本地数据库中永久删除。
- 已保存的订单：
- 如果订单已保存到服务器，它将被移动到 WooCommerce 中的“垃圾”文件夹并从本地数据库中删除。
- 要恢复作废的订单：
- 转到“WP Admin > WooCommerce > 订单 > 垃圾”。

## 结帐

- 将订单保存到服务器并启动结帐流程。
- 打开结帐屏幕以完成付款。
