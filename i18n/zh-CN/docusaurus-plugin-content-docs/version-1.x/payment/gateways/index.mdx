# 支付网关 {#payment-gateways}

WCPOS 支持多种自定义支付网关，可扩展销售点系统的功能。这些网关专为 POS 环境设计，可与多种支付方式和服务无缝集成。

## 可用的自定义网关 {#available-custom-gateways}

### 硬件终端网关 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - 使用 Stripe Terminal 硬件读卡器接受付款
- **[SumUp Terminal](./sumup-terminal)** - 通过 SumUp 读卡器处理付款
- **[Square Terminal](./square-terminal)** - 在 Square Terminal 设备上收取现场付款
- **[PayPal Reader (Zettle)](./paypal-reader)** - 在 PayPal Reader（Zettle）终端上收取现场刷卡付款
- **[Mollie Terminal](./mollie-terminal)** - 在 Mollie Terminal 设备上收取现场付款

### 移动支付网关 {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - 通过二维码或推送通知接受手机支付

### 数字支付网关 {#digital-payment-gateways}

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送付款发票
- **[网页结账](./web-checkout)** - 将客户重定向至在线完成付款

## 创建自己的网关 {#creating-your-own-gateway}

如果需要满足特定要求的自定义支付网关，可以使用我们的 **[网关模板](/reference/gateway-template)** 创建自己的 WCPOS 支付网关。

## 安装概览 {#installation-overview}

所有自定义网关均遵循类似的安装流程：

1. **下载** 相应 GitHub 仓库中的最新版本
2. **安装** 通过 `WP Admin > 插件 > 新增 > 上传插件` 安装插件
3. **配置** `WP Admin > WooCommerce > 设置 > 支付` 中的网关设置
4. 在 `WP Admin > POS > 设置 > 结账` 中**启用**该网关
