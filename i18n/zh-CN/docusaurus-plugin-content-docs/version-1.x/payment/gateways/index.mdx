# 支付网关 {#payment-gateways}

WCPOS 支持多种自定义支付网关，可扩展销售点系统的功能。这些网关专为 POS 环境设计，能够与各种支付方式和服务无缝集成。

## 可用的自定义网关 {#available-custom-gateways}

### 硬件终端网关 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - 使用 Stripe Terminal 硬件读卡器收款
- **[SumUp Terminal](./sumup-terminal)** - 通过 SumUp 读卡器处理支付
- **[Square Terminal](./square-terminal)** - 在 Square Terminal 设备上收取当面付款
- **[PayPal Reader (Zettle)](./paypal-reader)** - 使用 PayPal Reader (Zettle) 终端收取当面刷卡付款

### 移动支付网关 {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - 通过二维码或推送通知接受手机支付

### 数字支付网关 {#digital-payment-gateways}

- **[电子邮件账单](./email-invoice)** - 通过电子邮件向客户发送付款账单
- **[网页结账](./web-checkout)** - 引导客户跳转至线上完成支付

## 创建自定义网关 {#creating-your-own-gateway}

如果您需要满足特定需求的自定义支付网关，可以使用我们的**[网关模板](/reference/gateway-template)**来创建自己的 WCPOS 支付网关。

## 安装概述 {#installation-overview}

所有自定义网关的安装流程类似：

1. 从相应的 GitHub 仓库**下载**最新版本
2. 通过 `WP Admin > 插件 > 新增 > 上传插件` **安装**插件
3. 在 `WP Admin > WooCommerce > 设置 > 支付` 中**配置**网关设置
4. 在 `WP Admin > POS > 设置 > 结账` 中**启用**网关
