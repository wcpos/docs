# 自定义支付网关 {#custom-payment-gateways}

WCPOS 支持多种自定义支付网关，扩展您的销售点系统的功能。这些网关专为销售点环境设计，提供与各种支付方式和服务的无缝集成。

## 可用的自定义网关 {#available-custom-gateways}

### 硬件终端网关 {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - 使用 Stripe 终端硬件读卡器接受付款
- **[SumUp Terminal](./sumup-terminal)** - 通过 SumUp 刷卡机处理支付

### 移动支付网关 {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - 通过二维码或推送通知接受手机支付

### 数字支付网关 {#digital-payment-gateways}

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送付款发票
- **[网页结账](./web-checkout)** - 重定向客户在线完成支付

## 创建您自己的网关 {#creating-your-own-gateway}

如果您需要针对特定要求的自定义支付网关，可以使用我们的 **[网关模板](./gateway-template)** 来创建您自己的 WCPOS 支付网关。

## 安装概述 {#installation-overview}

所有自定义网关遵循类似的安装过程：

1. **下载**来自相应 GitHub 存储库的最新版本
2. **安装**插件，路径为 `WP Admin > Plugins > Add New > Upload Plugin`
3. **配置**网关设置，路径为 `WP Admin > WooCommerce > Settings > Payments`
4. **启用**网关，路径为 `WP Admin > POS > Settings > Checkout`
