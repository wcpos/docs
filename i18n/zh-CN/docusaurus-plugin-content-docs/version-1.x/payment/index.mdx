WCPOS 支持多种支付方式，以满足不同业务需求。

## 默认支付方式 {#default-payment-methods}

免费版包括两个内置支付网关：

### 现金 {#cash}

- 输入支付金额
- 自动计算找零
- 无需额外配置

### 卡 {#card}

- 用于外部卡终端
- 在终端处理后，只需将支付标记为完成
- 无需直接集成

## 额外支付网关（Pro） {#additional-payment-gateways-pro}

:::info 专业功能
额外支付网关需要 [WCPOS Pro](/getting-started/pro-license)。
:::

使用 Pro，您可以在 POS 结账中启用额外的 WooCommerce 支付网关：

- **Stripe Terminal** - 与 Stripe 卡读取设备的直接集成
- **SumUp Terminal** - 与 SumUp 卡读取设备的集成
- **任意 WooCommerce 网关** - 启用您商店中安装的任何网关

## 自定义网关 {#custom-gateways}

使用 [自定义网关](/payment/custom-gateways) 系统创建您自己的支付集成。
