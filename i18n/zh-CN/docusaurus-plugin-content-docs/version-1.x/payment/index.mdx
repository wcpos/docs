WCPOS 支持多种支付方式以满足不同的业务需求。

## 默认支付方式

免费版包括两个内置支付网关：

### 现金

- 输入金额
- 自动计算找零
- 无需额外配置

### 卡

- 用于外部卡终端
- 在终端上处理完成后简单标记支付为完成
- 无需直接集成

## 额外支付网关 (Pro)

:::info Pro 功能
额外支付网关需要 [WCPOS Pro](/getting-started/pro-license)。
:::

使用 Pro，您可以在 POS 结账中启用额外的 WooCommerce 支付网关：

- **Stripe 终端** - 与 Stripe 刷卡读卡器直接集成
- **SumUp 终端** - 与 SumUp 刷卡读卡器集成
- **任何 WooCommerce 网关** - 启用您商店中安装的任何网关

## 自定义网关

使用 [自定义网关](/payment/custom-gateways) 系统创建您自己的支付集成。
