# 网关模板

WooCommerce 网关模板为创建自定义 WooCommerce POS 支付网关提供了一个起点。该模板包含构建全功能支付网关所需的所有必要样板代码和结构。

## 特点

- **完整模板**：开箱即用的网关结构，包含所有必需的方法
- **POS 集成**：预配置 WooCommerce POS 兼容性
- **自动设置**：基于脚本的模板自定义
- **最佳实践**：遵循 WordPress 和 WooCommerce 编码标准
- **可扩展**：易于修改和扩展以适应特定支付提供商

## 开始使用

### 选项 1：自动模板生成

该模板包含一个可自动为您的特定网关定制模板的脚本：

1. **克隆仓库**：
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **运行设置脚本**：
   ```bash
   ./create-gateway.sh
   ```

3. **按提示操作**：
   - 输入网关名称（例如 "我的支付网关"）
   - 输入网关标识（例如 "my-payment"）
   - 提供描述
   - 脚本将生成自定义插件

### 选项 2：手动模板使用

如果您更喜欢手动自定义：

1. **下载模板**：
   - 访问 [网关模板仓库](https://github.com/wcpos/woocommerce-gateway-template)
   - 下载最新版本或克隆仓库

2. **自定义模板**：
   - 将 `{{GATEWAY_NAME}}` 替换为您的网关显示名称
   - 将 `{{GATEWAY_SLUG}}` 替换为您的网关唯一标识符
   - 将 `{{GATEWAY_DESCRIPTION}}` 替换为您的网关描述

3. **重命名文件**：
   - 将 `wcpos-{{GATEWAY_SLUG}}.php` 重命名以匹配您的网关标识
   - 更新文件头和插件信息
