# SumUp 终端网关 {#sumup-terminal-gateway}

SumUp 终端网关使您能够在 WooCommerce POS 中直接使用 SumUp 卡读取器接受卡片支付。该网关与 SumUp 的支付处理系统无缝集成，用于面对面交易。

  <FeatureCard icon="credit-card" title="硬件集成">

## 安装 {#installation}

<Steps>
  <StepCard number="1" title="下载并安装">

1. 访问 [WooCommerce 的 SumUp 终端版本页面](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. 下载最新的 **sumup-terminal-for-woocommerce.zip** 文件
3. 在您的 WordPress 管理后台，转到 `插件 > 添加新 > 上传插件`
4. 上传 zip 文件并激活插件

  </StepCard>
  <StepCard number="2" title="配置 SumUp 设置">

1. 导航到 `WP Admin > WooCommerce > 设置 > 支付`
2. 在支付方式列表中找到 **SumUp 终端**
3. 点击 **SumUp 终端** 以访问设置
4. 输入您的 **SumUp API 密钥**（可以在您的 SumUp 商户仪表板中获取）
5. 保存设置

:::note
您无需在 WooCommerce 设置中启用 SumUp 终端网关。它将在后续步骤中专门为 POS 启用。
:::

  </StepCard>
  <StepCard number="3" title="配对您的 SumUp 终端">

1. 在同一设置页面，找到 **配对读取器** 部分
2. 在您的 SumUp 设备上，导航到配对屏幕以显示配对代码
3. 输入您 SumUp 设备上显示的配对代码
4. 点击 **"配对读取器"** 以建立连接
5. 等待确认读取器已成功配对

:::warning 重要
必须成功配对读取器才能处理支付。在继续之前，请确保配对过程已完成。
:::

  </StepCard>
  <StepCard number="4" title="在 WooCommerce POS 中启用" isLast>

1. 转到 `WP Admin > WooCommerce POS > 设置 > 结账`
2. 在列表中找到 **SumUp 终端** 网关
3. 启用该网关以供 POS 使用
4. 保存您的设置

  </StepCard>
</Steps>

## 使用 {#usage}

### 处理支付 {#processing-payments}

1. **添加商品**：在 POS 中将商品添加到购物车
2. **选择网关**：选择 "SumUp 终端" 作为支付方式
3. **开始支付**：点击以在您的 SumUp 设备上发起新支付
4. **客户支付**：客户在 SumUp 终端上完成支付
5. **自动完成**：当检测到支付时，订单会自动完成

### 支付控制 {#payment-controls}

使用 SumUp 终端网关时，您有以下选项：

- **开始新支付**：在连接的终端上发起支付请求
- **取消支付**：取消当前正在处理的支付
- **支付状态**：监控支付过程的当前状态

### 订单管理 {#order-management}

- **自动更新**：成功支付后，订单会自动标记为已完成
- **支付跟踪**：所有支付详情都会记录在订单备注中
- **收据生成**：成功支付后生成标准 POS 收据

## 硬件兼容性 {#hardware-compatibility}

:::important 连接要求
此实现使用 SumUp 的 JavaScript SDK，这意味着它通过 Web 应用程序工作，但仅需要 **互联网连接的终端**。目前不支持蓝牙终端。
:::

### 支持的终端（互联网连接） {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - 连接 WiFi/以太网的终端

### 不支持的终端（蓝牙） {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - 仅蓝牙
- **SumUp 3G** ❌ - 仅蓝牙

:::note 未来支持
蓝牙终端支持将在 **2026 年第一季度** 添加到 iOS 和 Android 应用中。这将支持所有 SumUp 认证终端，包括 Air 和 3G 型号。
:::

## 故障排除 {#troubleshooting}

### 常见问题 {#common-issues}

<Accordion>
  <AccordionItem question="读取器无法配对">

  - 确保您使用的是支持的互联网连接终端（仅限 Solo）
  - 验证您的 SumUp Solo 设备已连接到 WiFi 并在线
  - 检查设备是否处于配对模式，且配对代码输入正确
  - 确认您的 SumUp 账户已启用 API 访问
  - 确保在配对过程中有稳定的互联网连接

  </AccordionItem>

  <AccordionItem question="支付未处理">

  - 验证读取器是否已正确配对
  - 检查您的 SumUp 账户是否活跃且状态良好
  - 确保设备电池充足且连接正常
  - 确认 API 密钥是否正确且有效

  </AccordionItem>

  <AccordionItem question="订单未完成">

  - 检查互联网连接的稳定性
  - 验证您的 SumUp 账户中的 webhook 设置
  - 确保 SumUp 终端上的支付成功
  - 检查 WordPress 错误日志以寻找 API 通信问题

  </AccordionItem>
</Accordion>

### API 速率限制 {#api-rate-limits}

该网关经过优化以减小 API 调用，避免速率限制：
- 高效检查支付状态
- 最小化不必要的 API 请求
- 自动重试逻辑处理临时失败
