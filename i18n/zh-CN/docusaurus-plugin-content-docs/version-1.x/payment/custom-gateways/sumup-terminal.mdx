# SumUp 终端网关

SumUp 终端网关使您能够直接在 WooCommerce POS 中使用 SumUp 读卡器接受卡支付。该网关为线下交易提供了与 SumUp 支付处理系统的无缝集成。

## 功能

- **硬件集成**：通过互联网连接将 SumUp 读卡器连接到您的 POS 系统
- **实时处理**：即时支付处理和订单完成
- **自动检测**：检测到支付时自动完成订单
- **安全交易**：通过 SumUp 进行 PCI 合规的支付处理
- **简单配对**：使用配对码进行简单的设备配对
- **Web SDK**：使用 SumUp 的 JavaScript SDK 实现 Web 应用兼容性

## 安装

### 步骤 1：下载并安装

1. 访问 [SumUp Terminal for WooCommerce 发布页面](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. 下载最新的 **sumup-terminal-for-woocommerce.zip** 文件
3. 在您的 WordPress 管理后台，转到 `插件 > 添加新插件 > 上传插件`
4. 上传 zip 文件并激活插件

### 步骤 2：配置 SumUp 设置

1. 导航至 `WP 管理后台 > WooCommerce > 设置 > 支付`
2. 在支付方式列表中找到 **SumUp 终端**
3. 点击 **SumUp 终端** 访问设置
4. 输入您的 **SumUp API 密钥**（可在 SumUp 商户仪表板获取）
5. 保存设置

:::note
您无需在 WooCommerce 设置中启用 SumUp 终端网关。它将在稍后的步骤中专门为 POS 启用。
:::

### 步骤 3：配对您的 SumUp 终端

1. 在同一设置页面上，找到 **配对读卡器** 部分
2. 在您的 SumUp 设备上，导航到配对屏幕以显示配对码
3. 输入 SumUp 设备上显示的配对码
4. 点击 **"配对读卡器"** 建立连接
5. 等待读卡器已成功配对的确认

:::warning 重要
必须成功配对读卡器，才能处理支付。确保在继续之前完成配对过程。
:::

### 步骤 4：在 WooCommerce POS 中启用

1. 转到 `WP 管理后台 > WooCommerce POS > 设置 > 结账`
2. 在列表中找到 **SumUp 终端** 网关
3. 启用在 POS 中使用该网关
4. 保存您的设置

## 使用

### 处理支付

1. **添加商品**：在 POS 中将商品添加到购物车
2. **选择网关**：选择 "SumUp 终端" 作为支付方式
3. **开始支付**：在 SumUp 设备上启动新的支付
4. **客户支付**：客户在 SumUp 终端完成支付
5. **自动完成**：检测到支付时自动完成订单

### 支付控制

使用 SumUp 终端网关时，您有以下选项：

- **开始新支付**：在连接的终端上发起支付请求
- **取消支付**：取消当前进行的支付
- **支付状态**：监控支付过程的当前状态

### 订单管理

- **自动更新**：成功支付后订单自动标记为已完成
- **支付跟踪**：所有支付详细信息记录在订单备注中
- **收据生成**：成功支付后生成标准 POS 收据

## 要求

- **SumUp 账户**：活跃的 SumUp 商户账户
- **API 访问**：来自商户仪表板的 SumUp API 密钥
- **兼容硬件**：SumUp 读卡器设备
- **WooCommerce POS**：已安装并激活的免费或专业版本
- **稳定连接**：用于 API 通信的可靠互联网连接

## 硬件兼容性

:::important 连接要求
该实现使用 SumUp 的 JavaScript SDK，这意味着它通过 Web 应用程序工作，但需要**仅限互联网连接的终端**。目前不支持蓝牙终端。
:::

### 支持的终端（互联网连接）

- **SumUp Solo** ✅ - WiFi/以太网连接终端

### 不支持的终端（蓝牙）

- **SumUp Air** ❌ - 仅蓝牙
- **SumUp 3G** ❌ - 仅蓝牙

:::note 未来支持
蓝牙终端支持将在 **2026 年第一季度** 添加到 iOS 和 Android 应用程序中。这将支持所有 SumUp 认证终端，包括 Air 和 3G 型号。
:::

## 故障排除

### 常见问题

**读卡器无法配对**
- 确保使用支持的互联网连接终端（仅限 Solo）
- 验证您的 SumUp Solo 设备是否连接到 WiFi 并在线
- 检查设备是否处于配对模式，且配对码输入正确
- 确认您的 SumUp 账户已启用 API 访问
- 确保在配对期间有稳定的互联网连接

**支付未处理**
- 验证读卡器是否已正确配对
- 检查您的 SumUp 账户是否活跃且状态良好
- 确保设备电池充足且连接正常
- 确认 API 密钥正确且有效

**订单未完成**
- 检查互联网连接稳定性
- 验证 SumUp 账户中的 webhook 设置
- 确保 SumUp 终端上支付成功
- 检查 WordPress 错误日志中的 API 通信问题

### API 速率限制

网关经过优化，以减少 API 调用并避免速率限制：
- 高效检查支付状态
- 最小化不必要的 API 请求
- 自动重试逻辑处理临时失败
