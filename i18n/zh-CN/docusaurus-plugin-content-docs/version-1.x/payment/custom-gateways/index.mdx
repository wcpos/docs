# 自定义支付网关 {#custom-payment-gateways}

WCPOS支持多种自定义支付网关，扩展您的销售点系统的功能。这些网关专为POS环境设计，提供与各种支付方式和服务的无缝集成。

## 可用的自定义网关 {#available-custom-gateways}

### 硬件终端网关 {#hardware-terminal-gateways}

- **[Stripe终端](./stripe-terminal)** - 使用Stripe终端硬件读卡器接受支付
- **[SumUp终端](./sumup-terminal)** - 通过SumUp读卡器处理支付

### 移动支付网关 {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - 通过二维码或推送通知接受手机支付

### 数字支付网关

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送支付发票
- **[网络结账](./web-checkout)** - 重定向客户在线完成支付

## 创建您自己的网关 {#creating-your-own-gateway}

如果您需要满足特定要求的自定义支付网关，可以使用我们的**[网关模板](./gateway-template)**创建自己的WooCommerce POS支付网关。

## 安装概述 {#installation-overview}

所有自定义网关遵循类似的安装过程：

1. **下载**来自相应GitHub仓库的最新版本
2. **安装**插件，通过`WP Admin > 插件 > 添加新 > 上传插件`
3. **配置**网关设置，在`WP Admin > WooCommerce > 设置 > 支付`
4. **启用**网关，在`WP Admin > WooCommerce POS > 设置 > 结账`
