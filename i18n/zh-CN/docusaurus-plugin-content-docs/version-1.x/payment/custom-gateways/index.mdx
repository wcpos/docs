# 自定义支付网关

WCPOS 支持多个自定义支付网关，可以扩展您的销售点系统的功能。这些网关专为 POS 环境设计，并提供与各种支付方式和服务的无缝集成。

## 可用的自定义网关

### 硬件终端网关

- **[Stripe 终端](./stripe-terminal)** - 使用 Stripe 终端硬件读卡器接受支付
- **[SumUp 终端](./sumup-terminal)** - 通过 SumUp 卡读取器处理支付

### 数字支付网关

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送支付发票
- **[网页结账](./web-checkout)** - 将客户重定向到线上完成支付

## 创建您自己的网关

如果您需要针对特定要求的自定义支付网关，可以使用我们的 **[网关模板](./gateway-template)** 创建自己的 WooCommerce POS 支付网关。

## 安装概述

所有自定义网关都遵循类似的安装流程：

1. **下载** GitHub 仓库中的最新版本
2. **安装** 插件，通过 `WP 管理后台 > 插件 > 添加新插件 > 上传插件`
3. **配置** 网关设置，位于 `WP 管理后台 > WooCommerce > 设置 > 支付`
4. **启用** 网关，位于 `WP 管理后台 > WooCommerce POS > 设置 > 结账`

## 要求

- WooCommerce POS 插件（免费或专业版）
- 已安装 WooCommerce 的 WordPress
- 特定支付服务的适当 API 密钥或凭据

## 支持

对于自定义网关的问题，请访问相应的 GitHub 仓库并创建一个包含问题详细信息的 issue。
