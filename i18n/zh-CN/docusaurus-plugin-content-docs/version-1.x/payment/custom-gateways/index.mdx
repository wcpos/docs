# 自定义支付网关

WCPOS 支持多种自定义支付网关，扩展您的销售点系统的功能。这些网关专为 POS 环境设计，提供与各种支付方法和服务的无缝集成。

## 可用自定义网关

### 硬件终端网关

- **[Stripe 终端](./stripe-terminal)** - 使用 Stripe 终端硬件阅读器接受支付
- **[SumUp 终端](./sumup-terminal)** - 通过 SumUp 售卡阅读器处理支付

### 数字支付网关

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送支付发票
- **[网页结账](./web-checkout)** - 将客户重定向到在线完成支付

## 创建自定义网关

如果您需要满足特定需求的自定义支付网关，可以使用我们的 **[网关模板](./gateway-template)** 创建您自己的 WooCommerce POS 支付网关。

## 安装概述

所有自定义网关遵循类似的安装过程：

1. **下载** 来自各自 GitHub 仓库的最新版本
2. **安装** 插件通过 `WP Admin > 插件 > 添加新 > 上传插件`
3. **配置** 在 `WP Admin > WooCommerce > 设置 > 支付` 中的网关设置
4. **启用** 在 `WP Admin > WooCommerce POS > 设置 > 结账` 中的网关

## 需求

- WooCommerce POS 插件（免费或专业版）
- 安装了 WooCommerce 的 WordPress
- 特定支付服务所需的适当 API 密钥或凭证
