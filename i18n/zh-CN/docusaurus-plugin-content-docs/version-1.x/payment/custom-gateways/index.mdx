# 自定义支付网关

WCPOS 支持多个自定义支付网关，扩展您的销售点系统的功能。这些网关专为 POS 环境设计，并提供与各种支付方式和服务的无缝集成。

## 可用的自定义网关

### 硬件终端网关

- **[Stripe Terminal](./stripe-terminal)** - 使用 Stripe Terminal 硬件读取器接受支付
- **[SumUp Terminal](./sumup-terminal)** - 通过 SumUp 刷卡机处理支付

### 数字支付网关

- **[电子邮件发票](./email-invoice)** - 通过电子邮件向客户发送支付发票
- **[网页结账](./web-checkout)** - 重定向客户在线完成支付

## 创建您自己的网关

如果您需要满足特定要求的自定义支付网关，您可以使用我们的 **[网关模板](./gateway-template)** 来创建自己的 WooCommerce POS 支付网关。

## 安装概述

所有自定义网关遵循类似的安装过程：

1. **下载** 从各自的 GitHub 存储库获取最新版本
2. **安装** 通过`WP Admin > 插件 > 添加新 > 上传插件`
3. **配置** 在`WP Admin > WooCommerce > 设置 > 支付`中设置网关
4. **启用** 在`WP Admin > WooCommerce POS > 设置 > 结账`中启用网关
