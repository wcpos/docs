# Stripe 终端支付网关

Stripe 终端支付网关允许您直接在 WooCommerce POS 中使用 Stripe 终端硬件读卡器接受现场付款。此网关支持物理读卡器和模拟器模式进行测试。

## 功能

- **硬件集成**：通过互联网连接物理 Stripe 终端读卡器
- **模拟器模式**：使用 Stripe 模拟器在无硬件情况下测试付款
- **多种支付方式**：支持各种卡类型和支付方法
- **实时处理**：即时支付处理和确认
- **安全交易**：通过 Stripe 进行 PCI 合规支付处理
- **基于 Web 的 SDK**：使用 Stripe 的 JavaScript SDK 实现 Web 应用兼容性

## 安装

### 第 1 步：下载并安装

1. 访问 [Stripe 终端 WooCommerce 发布页面](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. 下载最新的 **stripe-terminal-for-woocommerce.zip** 文件
3. 在 WordPress 管理后台，转到 `插件 > 新增 > 上传插件`
4. 上传 zip 文件并激活插件

### 第 2 步：配置 Stripe 设置

1. 导航到 `WP 管理后台 > WooCommerce > 设置 > 支付`
2. 在支付方式列表中找到 **Stripe 终端**
3. 点击 **Stripe 终端**以访问设置
4. 输入您的 **Stripe 密钥**（可以在 Stripe 仪表板中获取）
5. 保存设置

:::note
您无需在 WooCommerce 设置中启用 Stripe 终端支付网关。它将在下一步专门为 POS 启用。
:::

### 第 3 步：在 WooCommerce POS 中启用

1. 转到 `WP 管理后台 > WooCommerce POS > 设置 > 结账`
2. 在列表中找到 **Stripe 终端** 网关
3. 在 POS 中启用该网关
4. 保存您的设置

## 使用

### 连接读卡器

在 POS 结账时选择 Stripe 终端支付网关：

1. **选择连接方式**：您可以连接物理读卡器或使用模拟器
2. **物理读卡器**：按照屏幕上的说明连接您的 Stripe 终端设备
3. **模拟器**：选择模拟器模式以在无硬件的情况下测试各种支付场景

### 处理支付

1. **添加商品**：在 POS 中将商品添加到购物车
2. **选择网关**：选择"Stripe 终端"作为支付方式
3. **连接读卡器**：连接您的读卡器或选择模拟器模式
4. **处理支付**：按照提示完成交易
5. **确认**：支付成功后，订单将自动完成

### 使用模拟器测试

模拟器允许您测试各种支付方法和场景：

- **卡支付**：测试不同的卡类型（Visa、Mastercard 等）
- **非接触式支付**：模拟快速支付交易
- **错误场景**：测试支付被拒绝和其他错误情况
- **不同金额**：测试各种交易金额

## 要求

- **Stripe 账户**：已启用终端的活跃 Stripe 账户
- **API 密钥**：来自仪表板的 Stripe 密钥
- **WooCommerce POS**：已安装并激活的免费或专业版
- **HTTPS**：您的站点必须使用 SSL/HTTPS 以确保安全

## 硬件兼容性

:::important 连接要求
此实现使用 Stripe 的 JavaScript SDK，这意味着它通过 Web 应用程序工作，但仅支持**联网终端**。目前不支持蓝牙终端。
:::

### 支持的终端（联网）

- **Stripe 读卡器 S700** ✅ - 以太网/WiFi 连接终端
- **WisePOS E** ✅ - WiFi 连接终端

### 不支持的终端（蓝牙）

- **BBPOS Chipper 2X BT** ❌ - 仅蓝牙
- **BBPOS WisePad 3** ❌ - 仅蓝牙
- **Verifone P400** ❌ - 仅蓝牙

:::note 未来支持
蓝牙终端支持将在 **2026 年第一季度**添加到 iOS 和 Android 应用程序中。这将实现对所有 Stripe 终端认证读卡器的支持，包括 M2 和 WisePad 3。
:::

## 故障排除

### 常见问题

**读卡器无法连接**
- 确保使用支持的联网终端（S700 或 WisePOS E）
- 验证终端是否连接到 WiFi/以太网并在线
- 检查您的 Stripe 账户是否已启用终端
- 确认终端已在 Stripe 仪表板中注册

**支付被拒绝**
- 检查您的 Stripe 账户是否处于活跃状态且信誉良好
- 验证所使用的卡是否有效
- 确保有足够的资金

**SSL 证书错误**
- Stripe 终端需要 HTTPS - 确保您的站点有有效的 SSL 证书
- 检查 SSL 证书是否正确配置
