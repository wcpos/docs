# Stripe 终端网关 {#stripe-terminal-gateway}

Stripe 终端网关允许您直接在 WCPOS 中使用 Stripe 终端硬件读卡器接受面对面的付款。此网关支持物理读卡器和用于测试的模拟模式。

## 功能 {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="硬件集成">
    通过互联网连接连接物理 Stripe 终端读卡器
  </FeatureCard>
  <FeatureCard icon="desktop" title="模拟模式">
    使用 Stripe 的模拟器测试无硬件付款
  </FeatureCard>
  <FeatureCard icon="bolt" title="实时处理">
    即时付款处理和确认
  </FeatureCard>
  <FeatureCard icon="shield" title="安全交易">
    通过 Stripe 进行 PCI 合规的付款处理
  </FeatureCard>
</FeatureGrid>

## 安装 {#installation}

<Steps>
  <StepCard number="1" title="下载并安装">

1. 访问 [Stripe Terminal for WooCommerce 发布页面](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. 下载最新的 **stripe-terminal-for-woocommerce.zip** 文件
3. 在您的 WordPress 管理中，前往 `插件 > 添加新插件 > 上传插件`
4. 上传 zip 文件并激活插件

  </StepCard>
  <StepCard number="2" title="配置 Stripe 设置">

1. 导航至 `WP 管理 > WooCommerce > 设置 > 付款`
2. 在付款方式列表中找到 **Stripe Terminal**
3. 点击 **Stripe Terminal** 以访问设置
4. 输入您的 **Stripe 秘密密钥**（您可以从您的 Stripe 控制面板获取）
5. 保存设置

:::note
您无需在 WooCommerce 设置中启用 Stripe 终端网关。它将在下一步中专门为 POS 启用。
:::

  </StepCard>
  <StepCard number="3" title="在 WCPOS 中启用" isLast>

1. 前往 `WP 管理 > POS > 设置 > 结账`
2. 在列表中找到 **Stripe Terminal** 网关
3. 启用网关以供 POS 使用
4. 保存您的设置

  </StepCard>
</Steps>

## 使用 {#usage}

### 连接读卡器 {#connecting-a-reader}

当您在 POS 的结账过程中选择 Stripe 终端网关时：

1. **选择连接方式**：您可以连接实体读卡器或使用模拟器
2. **实体读卡器**：按照屏幕上的说明连接您的 Stripe 终端设备
3. **模拟器**：选择模拟模式以测试各种付款场景，无需硬件

### 处理付款 {#processing-payments}

1. **添加商品**：在 POS 中将产品添加到购物车
2. **选择网关**：选择 "Stripe Terminal" 作为付款方式
3. **连接读卡器**：连接您的读卡器或选择模拟模式
4. **处理付款**：按照提示完成交易
5. **确认**：成功付款后订单将自动完成

### 使用模拟器测试 {#testing-with-simulator}

模拟器允许您测试各种付款方式和场景：

- **卡片付款**：测试不同的卡片类型（Visa、Mastercard 等）
- **非接触式付款**：模拟刷卡付款交易
- **错误场景**：测试被拒付款和其他错误情况
- **不同金额**：测试各种交易金额

## 硬件兼容性 {#hardware-compatibility}

:::重要 连接性要求
此实现使用 Stripe 的 JavaScript SDK，这意味着它通过 Web 应用程序工作，但仅需要 **连接互联网的终端**。当前不支持蓝牙终端。
:::

### 支持的终端（网络连接） {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - 以以太网/WiFi 连接的终端
- **WisePOS E** ✅ - 以 WiFi 连接的终端

### 不支持的终端（蓝牙） {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - 仅蓝牙
- **BBPOS WisePad 3** ❌ - 仅蓝牙  
- **Verifone P400** ❌ - 仅蓝牙

:::note 未来支持
蓝牙终端支持将在 **2026年第一季度** 添加到 iOS 和 Android 应用程序中。这将使所有经过 Stripe 终端认证的读卡器（包括 M2 和 WisePad 3）获得支持。
:::

### 常见问题 {#common-issues}

<Accordion>
  <AccordionItem question="读卡器无法连接">

  - 确保您使用的是支持的网络连接终端（S700 或 WisePOS E）
  - 验证终端是否连接到 WiFi/Ethernet 并在线
  - 检查您的 Stripe 账户是否启用了终端
  - 确认终端已在您的 Stripe 控制面板中注册

  </AccordionItem>

  <AccordionItem question="付款被拒">

  - 检查您的 Stripe 账户是否激活且状态良好
  - 验证所使用的卡片是否有效
  - 确保有足够的资金

  </AccordionItem>

  <AccordionItem question="SSL 证书错误">

  - Stripe Terminal 需要 HTTPS - 确保您网站有有效的 SSL 证书
  - 检查您的 SSL 证书是否正确配置

  </AccordionItem>
</Accordion>
