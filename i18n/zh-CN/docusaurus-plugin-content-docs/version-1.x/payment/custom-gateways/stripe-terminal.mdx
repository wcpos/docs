# Stripe 终端网关

Stripe 终端网关允许您直接在 WooCommerce POS 中使用 Stripe 终端硬件读取器接受面对面的支付。该网关支持物理卡读取器和测试的模拟模式。

## 特性

- **硬件集成**：通过互联网连接连接物理 Stripe 终端读取器
- **模拟模式**：使用 Stripe 的模拟器测试支付，无需硬件
- **多种支付方式**：支持各种卡类型和支付方式
- **实时处理**：即时支付处理和确认
- **安全交易**：通过 Stripe 进行 PCI 合规的支付处理
- **基于网络的 SDK**：使用 Stripe 的 JavaScript SDK 以确保与网络应用程序的兼容性

## 安装

### 步骤 1：下载并安装

1. 访问 [Stripe Terminal for WooCommerce 发布页面](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. 下载最新的 **stripe-terminal-for-woocommerce.zip** 文件
3. 在您的 WordPress 管理后台，转到 `插件 > 添加新 > 上传插件`
4. 上传 zip 文件并激活插件

### 步骤 2：配置 Stripe 设置

1. 导航至 `WP 管理 > WooCommerce > 设置 > 支付`
2. 在支付方式列表中找到 **Stripe Terminal**
3. 点击 **Stripe Terminal** 以访问设置
4. 输入您的 **Stripe 秘钥**（您可以从您的 Stripe 仪表板获取此信息）
5. 保存设置

:::note
您无需在 WooCommerce 设置中启用 Stripe 终端网关。它将在下一步为 POS 特别启用。
:::

### 步骤 3：在 WooCommerce POS 中启用

1. 转到 `WP 管理 > WooCommerce POS > 设置 > 结算`
2. 在列表中找到 **Stripe Terminal** 网关
3. 启用该网关以供 POS 使用
4. 保存您的设置

## 使用

### 连接读取器

当您在 POS 的结算期间选择 Stripe 终端网关时：

1. **选择连接方式**：您可以连接一个物理读取器或使用模拟器
2. **物理读取器**：按照屏幕上的说明连接您的 Stripe 终端设备
3. **模拟器**：选择模拟模式以在没有硬件的情况下测试各种支付场景

### 处理支付

1. **添加项目**：在 POS 中将产品添加到购物车
2. **选择网关**：选择 "Stripe Terminal" 作为支付方式
3. **连接读取器**：连接您的读取器或选择模拟模式
4. **处理支付**：按照提示完成交易
5. **确认**：成功支付后，订单将自动完成

### 使用模拟器进行测试

模拟器允许您测试各种支付方式和场景：

- **卡支付**：测试不同的卡类型（Visa、Mastercard 等）
- **非接触式支付**：模拟轻触支付交易
- **错误场景**：测试拒绝支付和其他错误条件
- **不同金额**：测试不同的交易金额

## 要求

- **Stripe 账户**：启用终端的有效 Stripe 账户
- **API 密钥**：来自您的仪表板的 Stripe 秘钥
- **WooCommerce POS**：已安装和激活的免费或专业版
- **HTTPS**：您的站点必须使用 SSL/HTTPS 以确保安全

## 硬件兼容性

:::important 连接要求
此实现使用 Stripe 的 JavaScript SDK，这意味着它通过网络应用程序工作，但仅需**互联网连接的终端**。目前不支持蓝牙终端。
:::

### 支持的终端（互联网连接）

- **Stripe Reader S700** ✅ - 以太网/WiFi 连接的终端
- **WisePOS E** ✅ - WiFi 连接的终端

### 不支持的终端（蓝牙）

- **BBPOS Chipper 2X BT** ❌ - 仅支持蓝牙
- **BBPOS WisePad 3** ❌ - 仅支持蓝牙  
- **Verifone P400** ❌ - 仅支持蓝牙

:::note 未来支持
蓝牙终端支持将于 **2026 年第一季度** 添加到 iOS 和 Android 应用程序中。这将支持所有 Stripe 终端认证的读取器，包括 M2 和 WisePad 3。
:::

## 故障排除

### 常见问题

**读取器无法连接**
- 确保您使用的是支持的互联网连接终端（S700 或 WisePOS E）
- 验证终端已连接到 WiFi/Ethernet 并在线
- 检查您的 Stripe 账户是否已启用终端
- 确认终端已在您的 Stripe 仪表板中注册

**支付被拒绝**
- 检查您的 Stripe 账户是否处于活动状态且良好
- 验证所用卡的有效性
- 确保有足够的资金可用

**SSL 证书错误**
- Stripe 终端要求 HTTPS - 确保您的站点有有效的 SSL 证书
- 检查您的 SSL 证书是否已正确配置
