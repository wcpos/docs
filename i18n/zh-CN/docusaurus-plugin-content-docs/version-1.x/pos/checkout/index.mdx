当您准备完成销售时，点击**结账**按钮打开结账弹窗。在这里您可以处理支付并完成订单。

{/* Screenshot: checkout-modal.png */}

## 结账弹窗概览 {#checkout-modal-overview}

结账弹窗显示以下内容：

- **订单编号** - WooCommerce 订单 ID
- **应付金额** - 应付总金额
- **结账设置**按钮 - 排查显示问题
- **收银员** - 正在处理订单的操作人员
- **顾客** - 此订单的顾客（可点击链接）
- **添加优惠券**按钮 - 应用折扣码
- **订单摘要** - 商品、数量和合计
- **支付方式** - 可用的支付选项
- **取消/处理付款**按钮

## 支付方式 {#payment-methods}

### 免费版可用 {#available-in-free-version}

WCPOS 免费版包含两种支付网关：

- **现金** - 支持输入收款金额并自动计算找零
- **银行卡** - 用于外接刷卡终端

### 更多支付网关（Pro） {#additional-gateways-pro}

:::info Pro 功能
更多支付网关需要 [WCPOS Pro](/getting-started/pro-license)。
:::

使用 Pro 版，您可以启用：

- **Stripe Terminal** - 与 Stripe 读卡器直接集成
- **SumUp Terminal** - 与 SumUp 读卡器集成
- **自定义网关** - 创建您自己的支付集成

有关配置网关的详细信息，请参阅[支付](/payment)。

### 选择支付方式 {#selecting-a-payment-method}

点击支付方式进行选择。表单会更新并显示相关字段：

**现金：**

- **收取金额** - 输入顾客支付的金额
- **找零** - 自动计算应找回的零钱

{/* Screenshot: checkout-cash-payment.png */}

**银行卡：**

- 在外部刷卡终端上处理付款
- 点击"处理付款"以完成

## 优惠券 {#coupons}

购物车在合计上方包含一个**添加优惠券**输入框（仅限专业版）。输入代码或按描述搜索；优惠券会在本地验证并显示为可移除的标签。多个优惠券按顺序叠加。

请参阅**[在收银台使用优惠券](/coupons/applying-coupons)**了解完整流程——搜索、标签、顺序折扣，以及验证错误与解决方案对照表。有关优惠券类型、验证规则和 WooCommerce 中的设置，请参阅[优惠券](/coupons)。

## 处理付款 {#processing-payment}

1. 选择支付方式
2. 对于现金支付，请输入实收金额
3. 点击**处理付款**
4. 订单已完成，并显示[收据](/receipts/at-checkout)

## 结账设置（故障排除） {#checkout-settings-troubleshooting}

结账弹窗使用 iframe/webview 来显示 WooCommerce 订单支付页面。这利用了 WooCommerce 现有的支付基础设施，意味着任何与 WooCommerce 兼容的支付网关都可以在 POS 中使用。

但是，主题和插件脚本有时会产生干扰。点击**结账设置**进行排查：

### 禁用所有样式和脚本 {#disable-all-styles-and-scripts}

:::warning 终极手段
这是终极手段，仅应在测试时使用，或在开发者清楚了解操作后果的少数情况下使用。禁用所有 wp_head 脚本将移除包括 WooCommerce 展开/收起支付网关所需脚本在内的所有脚本，可能导致支付功能异常。
:::

- **禁用 wp_head** - 移除 WordPress 页头中的所有脚本/样式
- **禁用 wp_footer** - 移除 WordPress 页脚中的所有脚本/样式

### 禁用选定样式 {#disable-selected-styles}

选择性禁用可能导致显示问题的 CSS：

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- 等等。

### 禁用选定脚本 {#disable-selected-scripts}

选择性禁用可能影响支付网关的 JavaScript：

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- 等等。

:::tip
如果支付网关显示不正确：

1. 请先尝试禁用主题样式
2. 然后尝试禁用不需要的 WooCommerce 脚本
3. 注意不要禁用支付网关所需的脚本
   :::

## 取消 {#cancel}

点击**取消**可关闭结账弹窗，订单不会完成，仍保留为未结账的购物车。
