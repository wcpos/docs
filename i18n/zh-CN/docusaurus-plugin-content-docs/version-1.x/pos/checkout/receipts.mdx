成功处理付款后，会显示收据弹窗。此弹窗显示已完成订单的摘要，并提供打印或电子邮件发送收据的选项。

{/* 截图：receipt-modal.png */}

## 收据内容 {#receipt-contents}

收据显示：

### 商店信息 {#store-information}
- **商店名称**
- **地址**
- **电话号码**
- **电子邮件**
- **网站**
- **徽标**（如果已配置）

### 收据头部 {#receipt-header}
- **收据类型**（例如，“税收收据”）
- **订单号码**
- **日期和时间**
- **收银员姓名**
- **支付方式**

### 订单详情 {#order-details}
- **商品条目**：
  - 产品名称
  - 变体属性（如适用）
  - 数量
  - 价格
  - 总计

### 总计 {#totals}
- **小计**
- **税费**（如适用）
- **总计**
- **付款金额**（现金）
- **找零**（现金）

### 客户信息 {#customer-information}
- **账单地址**
- **收货地址**（如不同）

## 收据操作 {#receipt-actions}

### 关闭 {#close}
关闭收据弹窗并返回到 POS。

### 电子邮件发送收据 {#email-receipt}
将收据发送到客户的电子邮件地址：

1. 点击 **电子邮件发送收据**
2. 确认电子邮件地址
3. 收据通过 WooCommerce 的电子邮件系统发送

客户必须有与订单关联的电子邮件地址。

### 打印收据 {#print-receipt}
打印收据的纸质副本：

1. 点击 **打印收据**
2. 浏览器的打印对话框打开
3. 选择您的收据打印机并打印

:::tip 收据打印机
WCPOS 可与任何可以从您的浏览器访问的打印机配合使用。为获得最佳效果，请使用：
- 专用收据打印机（热敏打印机较为常见）
- 在打印机设置中配置纸张大小
- 设置打印预设以便快速打印
:::

## 自动收据选项 {#automatic-receipt-options}

在 [购物车显示设置](/pos/cart#display-settings) 中，您可以配置：

- **结账后自动显示收据** - 在付款后立即打开收据弹窗
- **结账后自动打印收据** - 在显示收据时自动触发打印

这些选项加快了高流量结账工作流程。

## 自定义收据 {#customising-receipts}

收据外观由 WordPress 中的模板控制：

`WP Admin > POS > 模板`

有关自定义的详细信息，请参见 [收据模板](/templates/receipt-templates)：

- 商店信息显示
- 徽标和品牌
- 布局和格式
- 包含哪些信息

## 重新打印收据 {#reprinting-receipts}

要重新打印过去订单的收据：

1. 转到 [订单页面](/orders)（专业功能）
2. 找到订单
3. 点击打开订单
4. 使用收据选项再次打印
