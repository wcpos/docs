需要在收银台退款？WCPOS 让您无需离开 POS 即可退还 WooCommerce 订单款项——全额或部分退款，退回原支付方式或以现金方式退还。

:::info 专业版功能
从 POS 发起退款需要 [WCPOS Pro](/getting-started/pro-license)。如果没有 Pro，请从 `WP Admin → WooCommerce → Orders` 进行退款。
:::

## 简要步骤 {#the-short-version}

1. **打开退款表单**——在订单列表中，点击订单的三点菜单并选择**退款**；或打开订单，点击底部的**退款**按钮。
2. **设置数量**——为每行输入**退款数量**（将所有行设为全部数量即可全额退款，或只设置部分行进行部分退款）。如需要，可添加可选的**自定义金额**和**原因**。
3. **选择退款去向：**
   - **退款至*（网关）***——将资金退回原始卡片/钱包，适用于支持此功能的支付网关（如 Stripe Terminal、Vipps MobilePay）。
   - **现金退款** — 记录从收银台退还的现金。用于现金销售时自动使用。
4. **确认** — 点击**处理退款**并确认金额。订单将立即更新。

退款需要实时服务器连接——与结账不同，退款无法离线排队处理。

## 想了解完整详情？ {#want-the-full-details}

完整指南涵盖了哪些订单状态可以退款、退款表单如何计算税费、网关与现金的决策规则、退款后小票和订单状态的变化，以及收银员/店铺审计记录。
