在班次结束时进行收银结账：清点现金，与 POS 记录的收款金额进行比对，核对银行卡总额，并归档记录。WCPOS 会显示收银抽屉*应有*的金额——收银员负责清点并手动核对。

:::info 专业版功能
用于对账的报表页面需要 [WCPOS 专业版](/getting-started/pro-license)。如果没有专业版，请通过 `WP Admin → WooCommerce → Analytics` 进行对账。
:::

## 简要步骤 {#the-short-version}

1. **停止收银**，确保清点过程中不会有新的交易入账。
2. **清点收银抽屉**——逐一清点每种面额。这就是您的*实际现金*。
3. **打开报表**，筛选当天（以及您的收银员/店铺），然后从销售汇总中查看**现金**和**银行卡**金额。
4. **核对现金：***备用金 + 现金销售额 − 现金退款 − 现金取出*应等于您的实际现金。
5. **核对银行卡：**卡终端的批次总额应与 POS 银行卡总额一致。如果终端为手动结算，请进行批次结算。
6. **打印报表**，投放现金，放回明日备用金，然后签退。

:::note 无内置班次管理
WCPOS 没有开班备用金字段、抽屉清点或自动差异计算功能——请在收银台使用纸质表格或电子表格记录备用金和投放金额。
:::
