POS屏幕是销售产品的主要界面。在使用WCPOS时，您会在此界面花费大部分时间。

{/* Screenshot: pos-desktop-layout.png */}

## 屏幕布局 {#screen-layout}

### 标题栏 {#header}

标题栏显示：

- **商店名称** - 您的连接WooCommerce商店名称
- **连接指示器** - 连接到服务器时为绿色点，离线时为黄色/红色
- **用户菜单** - 点击以访问设置、切换用户或注销

### 导航抽屉 {#navigation-drawer}

左侧边栏提供对所有主要屏幕的快速访问：

| 图标 | 屏幕 | 描述 |
|------|--------|-------------|
| <Icon name="cash-register" /> | **POS** | 主要销售界面（您在这里） |
| <Icon name="gift" /> | **产品** | 库存管理 (Pro) |
| <Icon name="receipt" /> | **订单** | 订单历史 (Pro) |
| <Icon name="users" /> | **客户** | 客户管理 (Pro) |
| <Icon name="chart-line" /> | **报告** | 销售报告 (Pro) |
| <Icon name="heart-pulse" /> | **日志** | 用于调试的系统日志 |
| <Icon name="circle-question" /> | **支持** | Discord支持频道 |

版本号显示在抽屉的底部。

## 响应式布局 {#responsive-layout}

WCPOS适应不同的屏幕尺寸：

### 桌面/平板电脑（大屏幕） {#desktop-tablet-large-screens}

在较大屏幕上，POS显示**双列布局**：

- **左列：** 产品面板 - 搜索和浏览产品
- **右列：** 购物车面板 - 查看和管理当前订单

列是**可调整大小的** - 拖动它们之间的分隔线以调整比例。

{/* Screenshot: pos-desktop-layout.png */}

### 移动设备（小屏幕） {#mobile-small-screens}

在较小屏幕上，POS使用**标签式布局**：

- **产品标签：** 浏览并将产品添加到购物车
- **购物车标签：** 查看购物车，管理商品项，并进行结账

使用屏幕底部的导航栏在标签之间切换。

{/* Screenshot: pos-mobile-layout.png */}

## 主要组件 {#main-components}

### 产品面板 {#product-panel}

POS的左侧是[产品面板](/pos/product-panel)，您可以在这里：

- 按名称、SKU或条形码搜索产品
- 按库存状态、类别、标签等过滤产品
- 将产品添加到购物车

[了解更多关于产品面板的信息 →](/pos/product-panel)

### 购物车面板 {#cart-panel}

POS的右侧是[购物车面板](/pos/cart)，您可以在这里：

- 查看当前订单中的商品
- 编辑数量和价格
- 选择或添加客户
- 访问订单操作（备注、作废、结账）

[了解更多关于购物车面板的信息 →](/pos/cart)

### 结账 {#checkout}

当您准备完成销售时，点击**结账**以打开[结账模态](/pos/checkout)：

- 应用优惠券代码
- 选择支付方式
- 处理付款
- 打印或发送收据

[了解更多关于结账的信息 →](/pos/checkout)

## 快捷键 {#keyboard-shortcuts}

WCPOS支持快捷键以加快操作速度。通过设置模态访问完整列表或查看[快捷键](/settings/store/hotkeys)。

常用快捷键：

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl + F` | 聚焦搜索栏 |
| `Ctrl + Shift + S` | 打开设置 |
| `Escape` | 关闭模态/对话框 |

## 连接性 {#connectivity}

标题中的绿色指示器显示您的连接状态：

- **绿色** - 已连接到WooCommerce服务器
- **黄色** - 连接问题，重试中
- **红色** - 离线模式

在离线时，您仍然可以浏览缓存的产品和客户，并开始新订单。完成订单需要连接。
