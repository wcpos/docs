POS 屏幕是销售产品的主界面。这是您使用 WCPOS 时最常待的地方。

{/* 截图：pos-desktop-layout.png */}

## 屏幕布局

### 标题栏

标题栏显示：

- **店铺名称** - 您连接的 WooCommerce 店铺名称
- **连接指示器** - 连接服务器时显示绿点，离线时显示黄色/红色
- **用户菜单** - 点击访问设置、切换用户或注销

### 导航抽屉

左侧边栏快速访问所有主要屏幕：

| 图标 | 屏幕 | 描述 |
|------|------|-------------|
| <Icon name="cash-register" /> | **POS** | 主要销售界面（您当前所在） |
| <Icon name="gift" /> | **产品** | 库存管理（专业版） |
| <Icon name="receipt" /> | **订单** | 订单历史（专业版） |
| <Icon name="users" /> | **客户** | 客户管理（专业版） |
| <Icon name="chart-line" /> | **报告** | 销售报告（专业版） |
| <Icon name="heart-pulse" /> | **日志** | 用于调试的系统日志 |
| <Icon name="circle-question" /> | **支持** | Discord 支持频道 |

版本号显示在抽屉底部。

## 响应式布局

WCPOS 适应不同屏幕尺寸：

### 桌面 / 平板（大屏幕）

在大屏幕上，POS 显示**两列布局**：

- **左列：** 产品面板 - 搜索和浏览产品
- **右列：** 购物车面板 - 查看和管理当前订单

这些列是**可调整大小的** - 拖动分隔线可以调整比例。

{/* 截图：pos-desktop-layout.png */}

### 移动设备（小屏幕）

在小屏幕上，POS 使用**选项卡式布局**：

- **产品选项卡：** 浏览和添加产品到购物车
- **购物车选项卡：** 查看购物车，管理订单项目，结账

使用屏幕底部的导航栏切换选项卡。

{/* 截图：pos-mobile-layout.png */}

## 主要组件

### 产品面板

POS 的左侧是[产品面板](/pos/product-panel)，您可以在这里：

- 通过名称、SKU 或条形码搜索产品
- 按库存状态、类别、标签等筛选产品
- 将产品添加到购物车

[了解更多关于产品面板 →](/pos/product-panel)

### 购物车面板

POS 的右侧是[购物车面板](/pos/cart)，您可以在这里：

- 查看当前订单的商品
- 编辑数量和价格
- 选择或添加客户
- 访问订单操作（备注、作废、结账）

[了解更多关于购物车面板 →](/pos/cart)

### 结账

当您准备完成销售时，点击**结账**打开[结账模态框](/pos/checkout)：

- 应用优惠券代码
- 选择支付方式
- 处理支付
- 打印或发送电子收据

[了解更多关于结账 →](/pos/checkout)

## 键盘快捷键

WCPOS 支持键盘快捷键以提高操作速度。通过设置模态框访问完整列表，或查看[键盘快捷键](/settings/store/hotkeys)。

常用快捷键：

| 快捷键 | 操作 |
|--------|------|
| `Ctrl + F` | 聚焦搜索栏 |
| `Ctrl + Shift + S` | 打开设置 |
| `Escape` | 关闭模态框/对话框 |

## 连接状态

标题栏中的绿色指示器显示您的连接状态：

- **绿色** - 已连接到 WooCommerce 服务器
- **黄色** - 连接出现问题，正在重试
- **红色** - 离线模式

离线时，您仍可以浏览缓存的产品和客户，并开始新订单。完成订单需要连接。
