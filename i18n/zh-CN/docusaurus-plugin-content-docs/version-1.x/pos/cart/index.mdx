购物车面板是 POS 屏幕的右侧，用于管理当前订单。在这里，您可以查看商品、编辑数量和价格、选择客户，并进行结账。

{/* 截图: pos-cart-panel.png */}

## 界面概览

### 客户选择器

在购物车面板的顶部：

- **客户徽章** - 显示当前客户（例如，“访客”或客户姓名）
- **添加客户图标** (<Icon name="plus" />) - 创建新客户
- **显示设置** (<Icon name="sliders" />) - 配置购物车列和选项

点击客户徽章可以搜索并选择不同的客户。

### 行项目

主要区域显示当前订单中的商品：

- **数量** - 数量（可编辑）
- **名称** - 产品名称及变体属性
- **价格** - 单价（可编辑）
- **总计** - 行总计

有关编辑的详细信息，请参见 [行项目](/pos/cart/line-items)。

### 添加到购物车选项

在行项目下方，您可以添加特殊商品：

- **添加杂项产品** - 添加自定义商品并手动输入价格
- **添加费用** - 添加费用行（例如，礼品包装、服务费）
- **添加运费** - 添加运费行并手动输入费用

:::note 行项目类型
WooCommerce 使用三种类型的行项目：`line_item`（产品）、`fee_line`（费用）和 `shipping_line`（运费）。目前，仅支持手动输入运费 - POS 不会自动计算运费。
:::

### 订单摘要

- **小计** - 税费和费用之前的总计
- **税** - 计算出的税额（如适用）
- **总计** - 最终订单总额

### 订单操作

在购物车面板的底部：

- **订单备注** - 添加客户可见的备注
- **订单元数据** - 添加自定义元数据或查看 JSON
- **保存到服务器** - 将订单保存到 WooCommerce，状态为 `pos-open`

有关详细信息，请参见 [订单操作](/pos/cart/order-actions)。

### 主要按钮

- **作废**（红色） - 取消/删除当前订单
- **结账**（绿色） - 进行支付

### 打开订单

在最底部，轮播显示所有打开的订单：

- 每个购物车显示其总计
- 点击购物车切换到该购物车
- 当前购物车被高亮显示

有关详细信息，请参见 [打开订单](/pos/cart/open-orders)。

点击滑块图标 (<Icon name="sliders" />) 自定义购物车面板的显示。

### 完成结账后自动显示收据

启用后，完成销售后收据屏幕会自动显示。

### 完成结账后自动打印收据

启用后，收据在显示时会自动打印。这通过减少手动打印步骤来节省时间。

### 快速折扣

以逗号分隔的折扣百分比列表，作为快速快捷按钮显示。

例如，输入 `5,10,15,20` 会添加四个按钮，这些按钮可以通过单击将百分比折扣应用于整个订单。

### 列

配置在购物车中显示哪些列：

| 列      | 描述                | 显示选项                    |
|---------|---------------------|-----------------------------|
| **数量** | 每项商品的数量      | 拆分（允许拆分商品）        |
| **名称** | 产品名称           | SKU                         |
| **价格** | 每单位价格          | 打折指示                   |
| **常规价格** | 未折扣价格       | -                           |
| **小计** | 行小计（数量 × 价格）| 税额                       |
| **总计** | 行总计              | 税额、打折指示             |
| **操作** | 删除商品按钮        | -                           |

### 恢复默认设置

点击以重置所有购物车显示设置为默认值。

## 向购物车添加商品

### 从产品面板

点击 [产品面板](/pos/product-panel) 中的产品以将其添加到购物车。对于可变产品，请先选择变体。

### 杂项产品

对于不在您的目录中的商品：

1. 点击 **添加杂项产品**
2. 输入名称和价格
3. 商品将被添加到购物车

### 费用

要添加费用（例如，礼品包装）：

1. 点击 **添加费用**
2. 输入名称和金额
3. 费用以单独行项目的形式出现

### 运费

要添加运费：

1. 点击 **添加运费**
2. 输入运送方式名称和费用
3. 运费将被添加到订单中

## 编辑购物车项目

所有行项目均可直接在购物车中编辑。有关以下内容的详细信息，请参见 [行项目](/pos/cart/line-items)：

- 编辑数量
- 更改价格
- 查看和编辑原始 JSON 数据
- 移除商品
