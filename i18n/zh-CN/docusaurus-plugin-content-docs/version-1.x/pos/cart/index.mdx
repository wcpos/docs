购物车面板是POS屏幕右侧的区域，用于管理当前订单。在这里，您可以查看商品、编辑数量和价格、选择客户，并进行结账。

{/* Screenshot: pos-cart-panel.png */}

## 界面概述

### 客户选择器

在购物车面板顶部：

- **客户标记** - 显示当前客户（例如，"游客"或客户名称）
- **添加客户图标**（<Icon name="plus" />） - 创建新客户
- **显示设置**（<Icon name="sliders" />） - 配置购物车列和选项

点击客户标记可搜索并选择不同的客户。

### 订单明细

主要区域显示当前订单的商品：

- **数量** - 数量（可编辑）
- **名称** - 产品名称及变体属性
- **价格** - 单价（可编辑）
- **总计** - 行总计

查看[订单明细](/pos/cart/line-items)了解编辑详情。

### 添加到购物车选项

在订单明细下方，您可以添加特殊商品：

- **添加杂项产品** - 添加自定义商品并手动输入价格
- **添加费用** - 添加费用行（例如，礼品包装、服务费）
- **添加配送** - 添加配送行并手动输入费用

:::note 订单明细类型
WooCommerce使用三种类型的订单明细：`line_item`（产品）、`fee_line`（费用）和`shipping_line`（配送）。目前仅支持手动输入配送，POS不会自动计算配送费用。
:::

### 订单概要

- **小计** - 税费和费用前的总计
- **税费** - 计算的税费金额（如适用）
- **总计** - 最终订单总计

### 订单操作

在购物车面板底部：

- **订单备注** - 添加对客户可见的备注
- **订单元数据** - 添加自定义元数据或查看JSON
- **保存到服务器** - 将订单以`pos-open`状态保存到WooCommerce

查看[订单操作](/pos/cart/order-actions)了解详情。

### 主要按钮

- **作废**（红色） - 取消/删除当前订单
- **结账**（绿色） - 进入支付流程

### 打开的订单

在最底部，轮播显示所有打开的订单：

- 每个购物车显示其总计
- 点击购物车切换到该订单
- 当前购物车高亮显示

查看[打开的订单](/pos/cart/open-orders)了解详情。

### 结账后自动显示收据

如果启用，销售完成后自动显示收据屏幕。

### 结账后自动打印收据

如果启用，显示收据时自动打印。这可以通过减少手动打印步骤来节省时间。

### 快速折扣

逗号分隔的折扣百分比列表，作为快速快捷按钮显示。

例如，输入`5,10,15,20`会添加四个按钮，通过单击即可对整个订单应用百分比折扣。

### 列

配置购物车中显示的列：

| 列 | 描述 | 显示选项 |
|--------|-------------|-----------------|
| **数量** | 每个商品的数量 | 拆分（允许拆分商品） |
| **名称** | 产品名称 | SKU |
| **价格** | 单价 | 特价指示器 |
| **原价** | 未打折价格 | - |
| **小计** | 行小计（数量 × 价格） | 税费金额 |
| **总计** | 行总计 | 税费金额，特价指示器 |
| **操作** | 移除商品按钮 | - |

### 恢复默认设置

点击重置所有购物车显示设置为原始默认值。

## 添加商品到购物车

### 从产品面板

在[产品面板](/pos/product-panel)中点击一个产品将其添加到购物车。对于可变产品，请先选择变体。

### 杂项产品

对于目录中没有的商品：

1. 点击**添加杂项产品**
2. 输入名称和价格
3. 商品被添加到购物车

### 费用

添加费用（例如礼品包装）：

1. 点击**添加费用**
2. 输入名称和金额
3. 费用作为单独的订单明细出现

### 配送

添加配送：

1. 点击**添加配送**
2. 输入配送方式名称和费用
3. 配送被添加到订单

## 编辑购物车商品

所有订单明细可在购物车中直接编辑。查看[订单明细](/pos/cart/line-items)了解以下详情：

- 编辑数量
- 更改价格
- 查看和编辑原始JSON数据
- 移除商品
