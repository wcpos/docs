购物车中的每个项目都是可以直接编辑的行项目。WCPOS 提供数量、价格和其他细节的灵活内联编辑功能。

## 行项目类型 {#line-item-types}

WooCommerce 使用三种类型的行项目：

| 类型 | 描述 | 示例 |
|------|-------------|----------|
| **line_item** | 来自您目录的产品 | T-shirt, Coffee mug |
| **fee_line** | 附加费用 | Gift wrapping, Service charge |
| **shipping_line** | 运费 | Standard shipping, Express delivery |

:::note
运费是手动输入的。POS 不会自动计算运费。
:::

## 编辑行项目 {#editing-line-items}

### 数量 {#quantity}

点击数量字段即可直接编辑。输入一个新数字或使用旋转控制。

{/* Screenshot: cart-line-item-edit-qty.png */}

### 产品名称 {#product-name}

对于杂项产品和费用，您可以直接通过点击名称来编辑。

### 价格 {#price}

点击价格字段以更改单价。这对以下情况特别有用：

- 应用手动折扣
- 价格匹配
- 更正定价错误

总计会根据数量 × 价格自动更新。

### 三点菜单 {#three-dot-menu}

点击任何行项目上的 **⋮**（三个点）以访问其他选项：

{/* Screenshot: cart-line-item-menu.png */}

#### 编辑表单 {#edit-form}

打开行项目的详细编辑表单，允许您修改所有字段。

#### 原始 JSON 视图 {#raw-json-view}

显示行项目的原始 JSON 数据。这在以下方面非常有帮助：

- **调试** - 查看发送到 WooCommerce 的确切数据
- **故障排除** - 识别自定义字段或元数据的问题
- **开发** - 理解集成的数据结构

### 删除项目 {#remove-item}

点击红色 **×** 按钮以从购物车中删除一个项目。

## 变体属性 {#variation-attributes}

对于可变产品，所选的变体属性显示在产品名称下方：

```
Hoodie
Colour: Green
Size: Large
```

这些属性是行项目数据的一部分，并包含在订单中。

## 元数据 {#meta-data}

如果您在产品面板设置中配置了 [元数据键](/pos/product-panel#meta-data-keys)，这些数据会在产品添加到购物车时自动复制到行项目中。

您可以在原始 JSON 视图中查看此元数据。

## 拆分行项目 {#splitting-line-items}

如果在 [显示设置](/pos/cart#display-settings) 中启用，您可以将行项目拆分为多个行。这在客户想对同一产品的不同部分应用不同折扣时很有用。

## 提示 {#tips}

### 快速调整价格 {#quick-price-adjustments}

要快速提供折扣：

1. 点击价格字段
2. 输入新价格
3. 按 Enter

### 键盘导航 {#keyboard-navigation}

- **Tab** - 在可编辑字段之间移动
- **Enter** - 确认编辑
- **Escape** - 取消编辑

### 批量编辑 {#batch-edits}

对于复杂订单，考虑使用三点菜单访问完整编辑表单，显示所有字段。
