购物车中的每个项目都是可以直接编辑的明细项目。WCPOS 提供了灵活的行内编辑功能，可以编辑数量、价格和其他详细信息。

## 明细项目类型

WooCommerce 使用三种类型的明细项目：

| 类型 | 描述 | 示例 |
|------|-------------|----------|
| **line_item** | 来自目录的产品 | T恤，咖啡杯 |
| **fee_line** | 额外费用 | 礼品包装，服务费 |
| **shipping_line** | 配送费用 | 标准配送，快递配送 |

:::note
配送费用需要手动输入。POS 系统不会自动计算配送费率。
:::

## 编辑明细项目

### 数量

点击数量字段直接编辑。输入新数字或使用微调控件。

{/* 截图：cart-line-item-edit-qty.png */}

### 产品名称

对于零散产品和费用，可以通过点击直接编辑名称。

### 价格

点击价格字段更改单价。这对以下情况很有用：

- 应用手动折扣
- 价格匹配
- 更正定价错误

总价将根据数量 × 价格自动更新。

### 三点菜单

点击任何明细项目上的 **⋮**（三个点）可以访问其他选项：

{/* 截图：cart-line-item-menu.png */}

#### 编辑表单

打开明细项目的详细编辑表单，允许您修改所有字段。

#### 原始 JSON 视图

显示明细项目的原始 JSON 数据。这对以下情况非常有帮助：

- **调试** - 准确查看发送到 WooCommerce 的数据
- **故障排除** - 识别自定义字段或元数据的问题
- **开发** - 了解集成的数据结构

### 移除项目

点击红色的 **×** 按钮从购物车中移除项目。

## 变体属性

对于可变产品，选定的变体属性显示在产品名称下方：

```
连帽衫
颜色：绿色
尺码：大号
```

这些属性是明细项目数据的一部分，将包含在订单中。

## 元数据

如果您在"产品面板"设置中配置了[元数据键](/pos/product-panel#meta-data-keys)，将在将产品添加到购物车时自动复制该数据。

您可以在原始 JSON 视图中查看这些元数据。

## 拆分明细项目

如果在[显示设置](/pos/cart#display-settings)中启用，您可以将明细项目拆分为多行。当顾客希望对同一产品的不同部分应用不同折扣时，这很有用。

## 提示

### 快速价格调整

快速给予折扣：

1. 点击价格字段
2. 输入新价格
3. 按回车键

### 键盘导航

- **Tab** - 在可编辑字段之间移动
- **回车** - 确认编辑
- **Escape** - 取消编辑

### 批量编辑

对于复杂订单，建议使用三点菜单访问完整的编辑表单，一次性显示所有字段。
