## 什么是产品元数据？ {#what-is-product-meta-data}

每个 WooCommerce 产品除了名称和价格外，还可以包含额外字段，例如
尺寸、雕刻文字、供应商代码，或由其他插件添加的数据。
WooCommerce 会将这些字段存储为**元数据**：每个条目都有一个**键**（字段名称，
例如 `engraving`）和一个**值**（例如 `"Happy Birthday"`）。

## 元数据键设置的作用 {#what-the-meta-data-keys-setting-does}

默认情况下，将产品添加到购物车时，只会复制其标准详细信息。
**元数据键**设置（产品面板 → 设置）可用于选择哪些产品
元数据键应**复制到购物车行项目**，并因此保存到订单中。

例如，将 `engraving` 添加到列表中，意味着每当你添加一件可雕刻的
商品到购物车时，其雕刻值都会随商品一起进入订单。

## 选择键 {#choosing-keys}

在元数据键字段中开始输入：

- **建议** 是已同步到此设备的商品上找到的元键。
- 如果没有列出所需的键（例如，它只存在于尚未
  同步的商品上），只需输入准确的键名，然后选择 **将“…”添加为自定义键**。
- 选中的键会显示为标签块；轻点其 ✕ 即可移除。

选择的键会与其他面板设置一起保存，并同时应用于
表格视图和网格（磁贴）视图。

## 提示 {#tips}

- 元键区分大小写，必须与商品上存储的键完全匹配。
- 以下划线开头的键（例如 `_size`）是 WooCommerce 的“隐藏”/内部
  元数据。如果数据存放在那里，它们在此处仍然可用。
