产品面板是 POS 屏幕左侧的部分，您可以在其中搜索、浏览和选择产品以添加到购物车中。

{/* Screenshot: pos-product-panel.png */}

## 界面概述

### 搜索栏

在面板顶部，搜索栏允许您通过以下方式快速找到产品：

- **产品名称** - 输入产品名称的任何部分
- **SKU** - 按库存单位搜索
- **条形码** - 扫描或输入条形码号码

有关详细信息，请参见 [搜索与筛选](/pos/product-panel/search-filtering)。

### 显示设置

单击搜索栏旁边的 **滑块图标** (<Icon name="sliders" />) 打开显示设置。这允许您自定义在产品列表中显示的列和信息。

### 筛选按钮

在搜索栏下，筛选按钮让您缩小产品范围：

- **有库存** - 仅显示有现货的产品
- **推荐** - 显示 WooCommerce 中标记为推荐的产品
- **促销** - 显示目前在促销的产品
- **类别** - 按产品类别筛选
- **标签** - 按产品标签筛选
- **品牌** - 按品牌筛选（如果使用品牌插件）

活动筛选器将突出显示。再次单击筛选器以将其移除。

### 产品列表

主区域以可滚动的列表显示您的产品。每行显示：

- **产品图片** - 产品的缩略图
- **产品名称** - 可选详细信息（库存、SKU、类别等）
- **价格** - 当前售价（适用时显示促销价格）
- **操作按钮** - 添加到购物车 (<Icon name="circle-plus" color="green" />) 或选择变体 (<Icon name="circle-arrow-right" color="green" />)

### 产品类型

- **简单产品** 显示绿色 <Icon name="circle-plus" color="green" /> 按钮可直接添加到购物车
- **可变产品** 显示绿色 <Icon name="circle-arrow-right" color="green" /> 箭头以打开变体选择器

有关更多详细信息，请参见 [可变产品](/pos/product-panel/variations)。

### 页脚

在产品面板底部：

- **税务状态** - 显示当前的税务计算基础（例如，“基于：商店基本地址的税”）。单击可查看当前税率。
- **产品数量** - 显示当前显示的产品数量（例如，“显示 10 个，共 17 个”）
- **同步按钮** - 从服务器刷新产品

### 显示缺货产品

控制是否显示当前缺货的产品。

- **启用**：缺货的产品仍然可见，但无法添加到购物车
- **禁用**：缺货产品从列表中隐藏

### 列

配置在产品列表中显示的列：

| 列      | 描述                  | 显示选项                                 |
|--------|---------------------|--------------------------------------|
| **图片** | 产品缩略图              | -                                    |
| **产品** | 产品名称和详细信息         | 库存、SKU、条形码、类别、标签、品牌、属性、元数据 |
| **SKU** | 库存单位（单独列）        | -                                    |
| **条形码** | 产品条形码（单独列）      | -                                    |
| **类型** | 产品类型（简单、可变等）    | -                                    |
| **库存** | 当前库存数量             | -                                    |
| **销售成本** | 产品成本价格            | -                                    |
| **价格** | 销售价格                | 税务信息，促销指示                       |
| **操作** | 添加到购物车按钮        | -                                    |

### 元数据键

指定在将产品添加到购物车时应转移到订单项的产品元数据键。

输入以逗号分隔的元键列表：

```
_bottle_deposit,_custom_field,_tracking_code
```

**用例：**
- 瓶子押金插件
- 自定义产品字段
- 第三方集成数据
- 合规/跟踪信息

:::note
元键是区分大小写的，必须与存储在产品的元数据中的内容完全匹配。
:::

### 恢复默认设置

单击以将所有显示设置重置为其原始默认值。

## 添加产品到购物车

### 简单产品

单击绿色 <Icon name="circle-plus" color="green" /> 按钮，将简单产品添加到购物车。每次单击增加一个单位。

### 可变产品

可变产品（例如，有尺寸和颜色选项的 T 恤）显示绿色 <Icon name="circle-arrow-right" color="green" /> 箭头。单击以：

1. **快速弹出** - 从下拉菜单选择变体
2. **展开内联** - 单击“展开”以查看列表中的所有变体

有关更多详细信息，请参见 [可变产品](/pos/product-panel/variations)。

### 条形码扫描

连接 USB 或蓝牙条形码扫描仪以快速添加产品。当您扫描条形码时，WCPOS 会自动搜索并添加匹配的产品。

有关设置说明，请参见 [条形码扫描](/pos/product-panel/barcode-scanning)。
