商品面板是POS屏幕的左侧区域，用于搜索、浏览和选择要添加到购物车的商品。

{/* 屏幕截图：pos-product-panel.png */}

## 界面概述

### 搜索栏

在面板顶部，搜索栏可以让你快速查找商品：

- **商品名称** - 输入商品名称的任何部分
- **SKU** - 按库存单位搜索
- **条形码** - 扫描或输入条形码号

查看 [搜索和筛选](/pos/product-panel/search-filtering) 获取详细信息。

### 显示设置

点击搜索栏旁边的**滑块图标**（<Icon name="sliders" />）打开显示设置。这允许你自定义在商品列表中显示的列和信息。

### 筛选按钮

在搜索栏下方，筛选按钮可以缩小商品范围：

- **有库存** - 仅显示有可用库存的商品
- **精选** - 显示在WooCommerce中标记为精选的商品
- **促销** - 显示当前在促销的商品
- **分类** - 按商品分类筛选
- **标签** - 按标签筛选
- **品牌** - 按品牌筛选（如果使用品牌插件）

活动筛选器会高亮显示。再次点击筛选器可以取消。

### 商品列表

主区域以可滚动列表显示你的商品。每行显示：

- **商品图片** - 商品缩略图
- **商品名称** - 包含可选细节（库存、SKU、分类等）
- **价格** - 当前销售价格（显示促销价格）
- **操作按钮** - 添加到购物车（<Icon name="circle-plus" color="green" />）或选择变体（<Icon name="circle-arrow-right" color="green" />）

### 商品类型

- **简单商品**显示绿色<Icon name="circle-plus" color="green" />按钮，直接添加到购物车
- **可变商品**显示绿色<Icon name="circle-arrow-right" color="green" />箭头，打开变体选择器

查看 [可变商品](/pos/product-panel/variations) 了解更多详情。

### 底部

在商品面板底部：

- **税收状态** - 显示当前税收计算基础（例如："税收基于：商店基础地址"）。点击查看活动税率。
- **商品数量** - 显示已显示的商品数量（例如："显示 10 个，共 17 个"）
- **同步按钮** - 从服务器刷新商品

### 显示无库存商品

控制是否显示当前无库存的商品。

- **启用**：无库存商品仍然可见，但不能添加到购物车
- **禁用**：无库存商品从列表中隐藏

### 列

配置商品列表中显示的列：

| 列 | 描述 | 显示选项 |
|--------|-------------|-----------------|
| **图片** | 商品缩略图 | - |
| **商品** | 商品名称和详情 | 库存、SKU、条形码、分类、标签、品牌、属性、元数据 |
| **SKU** | 库存单位（单独列） | - |
| **条形码** | 商品条形码（单独列） | - |
| **类型** | 商品类型（简单、可变等） | - |
| **库存** | 当前库存数量 | - |
| **销售成本** | 商品成本价 | - |
| **价格** | 销售价格 | 税收信息、促销指示器 |
| **操作** | 添加到购物车按钮 | - |

### 元数据键

指定在将商品添加到购物车时应转移到订单行项目的商品元数据键。

输入以逗号分隔的元数据键列表：

```
_bottle_deposit,_custom_field,_tracking_code
```

**使用场景：**
- 瓶子押金插件
- 自定义商品字段
- 第三方集成数据
- 合规/跟踪信息

:::note
元数据键区分大小写，必须与商品元数据中存储的完全匹配。
:::

### 恢复默认设置

点击将所有显示设置重置为原始默认值。

## 添加商品到购物车

### 简单商品

点击绿色<Icon name="circle-plus" color="green" />按钮将简单商品添加到购物车。每次点击添加一个单位。

### 可变商品

可变商品（例如，带有尺码和颜色选项的T恤）显示绿色<Icon name="circle-arrow-right" color="green" />箭头。点击可以：

1. **快速弹出** - 从下拉菜单选择变体
2. **内联展开** - 点击"展开"以在列表中查看所有变体

查看 [可变商品](/pos/product-panel/variations) 了解更多详情。

### 条形码扫描

连接USB或蓝牙条形码扫描器以快速添加商品。扫描条形码时，WCPOS会自动搜索并添加匹配的商品。

查看 [条形码扫描](/pos/product-panel/barcode-scanning) 获取设置说明。
