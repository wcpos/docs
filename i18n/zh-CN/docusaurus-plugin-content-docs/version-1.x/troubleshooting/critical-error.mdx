如果您看到此错误消息，则表示您的 POS 遇到了致命的服务器错误并且无法继续。
好消息是，此错误消息已记录在您的 WooCommerce 日志中，我们的支持团队可以帮助您解决问题。

<div class="image-container">
<Image
alt="致命错误日志"
img="https://woocommerce.com/wp-content/uploads/2018/11/wc_log_fatal_errors.png"
/>
<p class="image-caption"><a target="_blank" href="https://woocommerce.com/document/finding-php-error-logs/">WooCommerce 致命错误日志</a>{" "} 示例</p>
</div>

1. 登录 WordPress 管理区。
2. 在 WordPress 侧边栏菜单中导航至“WooCommerce”>“状态”>“日志”。
3. 找到“fatal_error”日志部分。
4. 单击与发生错误日期（通常是今天的日期）相对应的日志条目以查看错误的详细信息。
5. 识别此部分中记录的错误原因。
6. 复制任何相关错误消息或截取屏幕截图，并通过将其发布到 [Discord 聊天](https://wcpos.com/discord) 或通过电子邮件发送至 support@wcpos.com 与我们的支持团队分享。

:::info
如果您是 **Pro 用户**，则有资格获得优先支持。我们的团队将为您解决这些问题。要升级到 Pro 并享受优先支持的好处，请[访问我们的 Pro 页面](https://wcpos.com/pro) 了解更多信息。
:::
