---
title: 访问POS
sidebar_label: 访问
---

要控制WCPOS功能的访问权限，你可以利用WordPress的权限和用户角色。通过为用户分配特定角色并配置其权限，你可以为不同的POS用户定制访问级别。WCPOS插件与现有的WordPress角色集成，并引入了专门针对收银员的角色。

下表概述了POS中的主要用户角色及其相应权限：

| 角色          | 创建者      | 描述                                                                                                                                                                                                                                                                                                                                                    |
| ------------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 管理员 | WordPress       | 站点所有者，默认启用所有权限。                                                                                                                                                                                                                                                                                                                                    |
| 店铺管理员  | WooCommerce     | 此角色专为需要访问POS并管理POS设置的用户设计。店铺管理员可能没有权限安装插件或执行其他管理任务。                                                                                                                                                                             |
| 收银员       | WCPOS | 由WCPOS插件添加的角色，主要用于将使用POS的用户。收银员通常具有有限的权限，可能无法访问WP管理后台修改POS设置、安装插件或创建博客文章。在某些情况下，你甚至可能希望限制收银员在POS中添加或编辑客户。 |

要分配和管理用户角色和权限，你可以导航到 **`WP管理后台 > POS > 访问`**。

下表提供了与WCPOS插件相关的WordPress权限概述：

| 权限                  | 描述                                                            | 示例用途                                                                                                                                    |
| --------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `access_woocommerce_pos`    | 允许访问WCPOS界面。                        | 启用此选项将授予收银员访问和使用POS的权限，使其能够执行销售点交易。          |
| `manage_woocommerce_pos`    | 启用WCPOS设置和配置的管理。 | 启用此选项将使收银员能够修改与POS相关的设置，如支付方式或收据模板。     |
| `create_users`              | 允许收银员创建新的WordPress用户帐户。              | 禁用此选项将阻止收银员在POS中创建新的客户帐户。                                                  |
| `edit_others_shop_orders`   | 启用编辑其他收银员创建的店铺订单。              | 禁用此选项将阻止收银员修改在POS中由其他收银员创建的店铺订单。                              |
| `edit_product`              | 允许收银员编辑产品详细信息。                            | 禁用此选项将阻止收银员修改POS中可用产品的详细信息。                                     |
| `edit_published_products`   | 允许收银员编辑已发布的产品。                         | 禁用此选项将阻止收银员对已在POS中发布的产品进行更改。                           |
| `edit_shop_orders`          | 允许收银员编辑店铺订单。                                | 禁用此选项将阻止收银员在POS中修改店铺订单的详细信息，如订单状态或客户详细信息。 |
| `edit_users`                | 允许收银员编辑现有用户帐户。                     | 禁用此选项将阻止收银员在POS中修改客户帐户的详细信息，如账单地址。            |
| `list_users`                | 允许收银员查看用户帐户列表。                    | 禁用此选项将阻止收银员在POS中搜索或选择客户。                                                |
| `manage_product_terms`      | 允许收银员管理产品类别和标签。              | 禁用此选项将阻止收银员在POS中创建、编辑或删除产品类别或标签。                       |
| `publish_shop_orders`       | 允许收银员发布店铺订单。                             | 禁用此选项将阻止收银员在POS中完成和处理新的店铺订单。                                       |
| `read_private_products`     | 允许收银员访问私有产品的详细信息。          | 禁用此选项将阻止收银员查看POS中可用的私有产品详细信息。                               |
| `read_private_shop_coupons` | 允许收银员访问私有店铺优惠券的详细信息。      | 禁用此选项将阻止收银员在POS中查看私有店铺优惠券的详细信息。                                     |
| `read_private_shop_orders`  | 允许收银员访问私有店铺订单的详细信息。       | 禁用此选项将阻止收银员在POS中查看私有店铺订单的详细信息。                                      |
| `read`                      | 允许访问WordPress管理后台。                                  | 禁用此选项将阻止收银员访问WordPress仪表板和用户配置文件页面。                                                   |

通过为每个角色分配适当的权限，你可以微调POS中不同用户的访问和权限。这使你能够控制谁可以执行某些操作、查看敏感信息并对你的店铺进行更改。

有关在WordPress中管理用户角色和权限的更多详细信息，你可以参考 [WordPress Codex](https://codex.wordpress.org/Roles_and_Capabilities)。
