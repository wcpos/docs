---
title: 访问 POS
sidebar_label: 访问
---

要控制对 WCPOS 功能的访问，您可以利用 WordPress 功能和用户角色。通过将特定角色分配给用户并配置他们的功能，您可以为不同的 POS 用户定制访问级别。WCPOS 插件与现有的 WordPress 角色集成，并为收银员引入了一个特定角色。

以下表格列出了 POS 中主要的用户角色及其各自的功能：

| 角色          | 创建者           | 描述                                                                                                                                                                                                                                                                                                                                                     |
| ------------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 管理员       | WordPress       | 拥有所有默认启用功能的网站所有者。                                                                                                                                                                                                                                                                                                                     |
| 商店经理      | WooCommerce     | 此角色专为需要访问 POS 并能够管理 POS 设置的用户设计。商店经理可能没有权限安装插件或执行其他管理任务。                                                                                                                                                                                                                                        |
| 收银员       | WCPOS           | 此角色由 WCPOS 插件添加，旨在供主要使用 POS 的用户使用。收银员通常具有有限的功能，可能无法访问 WP 管理员以修改 POS 设置、安装插件或创建博客文章。在某些情况下，您甚至可能希望限制收银员在 POS 中添加或编辑客户。                                                                                |

要分配和管理用户角色及功能，您可以导航至 **`WP Admin > POS > Access`**。

以下表格提供了与 WCPOS 插件相关的 WordPress 功能概述：

| 功能                      | 描述                                                           | 示例用法                                                                                                                                                           |
| ------------------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `access_woocommerce_pos`   | 允许访问 WCPOS 界面。                                               | 启用此选项将授予收银员访问和使用 POS 的权限，使他们能够进行销售交易。                                                                                      |
| `manage_woocommerce_pos`   | 启用管理 WCPOS 设置和配置。                                         | 启用此选项将使收银员能够修改与 POS 相关的设置，例如支付方式或收据模板。                                                                                     |
| `create_users`            | 允许收银员创建新的 WordPress 用户帐户。                                | 禁用此选项将阻止收银员在 POS 中创建新客户帐户。                                                                                                                 |
| `edit_others_shop_orders`  | 允许编辑其他收银员创建的商店订单。                                    | 禁用此选项将阻止收银员修改由其他收银员在 POS 中创建的商店订单。                                                                                                |
| `edit_product`            | 允许收银员编辑产品详细信息。                                       | 禁用此选项将阻止收银员修改在 POS 中可用的产品详细信息。                                                                                                       |
| `edit_published_products` | 允许收银员编辑已发布的产品。                                        | 禁用此选项将阻止收银员对在 POS 中已经发布的产品进行更改。                                                                                                   |
| `edit_shop_orders`        | 允许收银员编辑商店订单。                                           | 禁用此选项将阻止收银员修改订单详细信息，例如订单状态或客户详细信息，在 POS 中。                                                                                |
| `edit_users`              | 允许收银员编辑现有用户帐户。                                       | 禁用此选项将阻止收银员修改客户帐户的详细信息，例如账单地址，在 POS 中。                                                                                        |
| `list_users`              | 允许收银员查看用户帐户列表。                                       | 禁用此选项将阻止收银员在 POS 中搜索或选择客户。                                                                                                              |
| `manage_product_terms`    | 允许收银员管理产品类别和标签。                                     | 禁用此选项将阻止收银员在 POS 中创建、编辑或删除产品类别或标签。                                                                                              |
| `publish_shop_orders`     | 允许收银员发布商店订单。                                          | 禁用此选项将阻止收银员在 POS 中最终处理新商店订单。                                                                                                           |
| `read_private_products`   | 允许收银员访问私人产品的详细信息。                                 | 禁用此选项将阻止收银员查看在 POS 中可用的私人产品的详细信息。                                                                                                 |
| `read_private_shop_coupons` | 允许收银员访问私人商店优惠券的详细信息。                            | 禁用此选项将阻止收银员查看在 POS 中的私人商店优惠券的详细信息。                                                                                               |
| `read_private_shop_orders` | 允许收银员访问私人商店订单的详细信息。                             | 禁用此选项将阻止收银员查看在 POS 中的私人商店订单的详细信息。                                                                                                   |
| `read`                    | 允许访问 WordPress 管理员界面。                                    | 禁用此选项将阻止收银员访问 WordPress 控制面板和用户个人资料页面。                                                                                             |

通过为每个角色分配适当的功能，您可以微调 POS 中不同用户的访问权限。这使您能够控制谁可以执行某些操作、查看敏感信息以及对您的商店进行更改。

有关在 WordPress 中管理用户角色和功能的更多详细信息，您可以参考 [WordPress Codex](https://codex.wordpress.org/Roles_and_Capabilities)。
