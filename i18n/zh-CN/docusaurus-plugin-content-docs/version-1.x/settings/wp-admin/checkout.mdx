结账设置页面 (`WP Admin > POS > Settings > Checkout`) 用于控制哪些支付网关显示在 POS 中以及订单的处理方式。

## 支付网关 {#payment-gateways}

此部分列出了您站点上安装的所有 WooCommerce 支付网关。每个网关均可独立启用或停用以用于 POS。

| 列 | 说明 |
|--------|-------------|
| **启用** | 启用或停用该网关用于 POS 结账 |
| **网关** | 支付网关名称 |
| **网关 ID** | WooCommerce 网关标识符（对开发者有用） |

### 启用支付网关 {#enabling-a-gateway}

1. 在列表中找到对应的支付网关
2. 打开 **启用** 开关
3. 下次加载页面时，该支付网关将显示在 POS 结账界面中

### 调整支付网关顺序 {#reordering-gateways}

通过拖动支付网关可以更改其在 POS 结账界面中的显示顺序。当收银员打开结账弹窗时，默认选中第一个已启用的支付网关。

### POS 与在线商店 {#pos-vs-online-store}

此处的支付网关设置与 WooCommerce 的付款设置相互**独立**。支付网关可以：

- **在 POS 中启用，在线上禁用** — 适用于不适合网络订单的现金支付
- **在线上启用，在 POS 中禁用** — 适用于不适合线下销售的支付网关，例如 PayPal
- **两者均启用** — 该支付网关在所有渠道均可使用

:::note
网关必须先在 `WP Admin > WooCommerce > Settings > Payments` 中安装并启用，才会显示在 POS 结账设置列表中。如果缺少某个网关，请检查插件是否已安装，以及该网关是否已在 WooCommerce 中启用。
:::

## 订单状态 {#order-status}

配置已完成 POS 订单的默认订单状态。默认情况下，POS 订单设置为**已完成**，因为款项已在销售点收取。
