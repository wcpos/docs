商店税号是 WCPOS 在您的 POS 收据和发票上打印的企业税号标识符——您的 VAT 号、ABN、GSTIN、EIN，或任何本地公司注册号。

WooCommerce 没有用于这些的内置字段，而且许多地区需要不止一个。一家德国企业可能会打印一个 USt-IdNr.、一个 Steuernummer **以及**一个 HRB 号；一家意大利企业会同时打印一个 Codice Fiscale 和一个 Partita IVA；许多公司在显示 VAT ID 的同时还会显示本地注册号。商店税号字段让您可以添加所需的任意多个条目。

您可以在 **WP Admin > POS > Settings > General** 的 **Store tax IDs** 部分找到它。

{/* Screenshot: wp-admin-store-tax-ids.png */}

## 添加税号 {#adding-a-tax-id}

1. 前往 **WP Admin > POS > Settings > General** 并滚动到 **Store tax IDs**。
2. 点击 **Add tax ID**。
3. 选择一个 **Type**，输入 **Value**，并可选地设置 **Country** 和自定义 **Label**。
4. 为您需要打印的每个标识符重复此操作。
5. 使用 **Remove** 删除某一行。

:::info 行会自动保存
此部分没有保存按钮。每一行在 **Value** 字段失去焦点时保存——点击或用 Tab 离开该字段即可提交。
:::

### 第一行使用您的店铺所在国家 {#the-first-row-uses-your-store-country}

当您添加第一个税号时，WCPOS 会用您的 WooCommerce 基准国家预填 **Country**，并选择匹配的 **Type**——这样，只需要单个本地税号的企业就无需手动选择任何内容。现有行永远不会被更改。

类型和国家在您编辑时保持同步：更改 **Type** 会将 **Country** 更新为该类型的所属国家，而更改 **Country** 会将 **Type** 切换为该国家通常使用的标识符。

### 可用类型 {#available-types}

类型列表涵盖常见的面向客户的标识符，以及企业在收据上打印的公司注册号：

| 地区 | 类型 |
|--------|-------|
| 欧盟 | EU VAT |
| 英国 | GB VAT、GB Company number |
| 德国 | USt-IdNr.、Steuernummer、HRB |
| 法国 | SIRET、SIREN |
| 其他欧洲 | Netherlands KVK、Italy Codice Fiscale、Italy Partita IVA、Spain NIF、Switzerland UID |
| 美洲 | US EIN、Canada GST/HST、Brazil CPF、Brazil CNPJ、Argentina CUIT |
| 亚洲及中东 | India GSTIN、Saudi Arabia VAT |
| 澳大利亚 | ABN |
| 通用 | Other |

**Value** 字段以占位符文本形式显示格式示例（例如 US EIN 显示 `12-3456789`）。这只是提示——WCPOS 不会拒绝不符合预期形状的值，因此不寻常但有效的标识符仍能保存。

## 收据上打印的内容 {#what-prints-on-receipts}

您添加的每个税号都会按顺序打印在收据和发票上，各占一行。WCPOS 为类型使用本地化标签（“VAT ID”、“ABN”、“GSTIN”等）——或者您设置的自定义 **Label**。如果两行的值相同，重复项会被自动删除。

所有捆绑的收据模板都已经渲染完整列表。如果您维护了一个仍然引用单个 `store.tax_id` 值的**自定义模板**，它仍然可以工作，但只会打印第一个条目。请改为切换到遍历列表：

```html
{{#store.tax_ids}}{{label}}: {{value}}{{/store.tax_ids}}
```

## 按店铺税号（Pro） {#per-store-tax-ids-pro}

:::info Pro 功能
使用 [WCPOS Pro](/getting-started/pro-license)，您可以**按店铺**设置税号，而不是为整个站点设置一次——当每个地点都有自己的 VAT/EIN/GSTIN，或当一家连锁企业跨越使用不同标识符的多个地区运营时，这很有用。
:::

按店铺税号在 **WP Admin > POS > Stores > Edit Store** 的侧边栏 **Tax IDs** 部分设置。当某个店铺自己的列表为空时，它会回退到此处 General 选项卡上配置的站点级商店税号。一旦您向某个店铺添加了一个条目，该店铺就只打印它自己的列表。
