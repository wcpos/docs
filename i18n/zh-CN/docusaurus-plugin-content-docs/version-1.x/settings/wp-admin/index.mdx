这些设置配置了 POS 应用程序的后端功能，包括 PHP 和 WooCommerce 配置。 
您可以通过导航到：`WP Admin > POS > Settings` 来访问它们。

主要配置包括：

- 所有收银员的默认购物车客户。
- 产品条码字段。
- 订单电子邮件设置。
- 用户角色权限。
