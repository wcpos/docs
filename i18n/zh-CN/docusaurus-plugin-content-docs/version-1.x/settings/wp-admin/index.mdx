这些设置配置POS应用程序的后台功能，包括PHP和WooCommerce配置。  
您可以通过导航到：`WP Admin > POS > Settings`访问它们。

关键配置包括：

- 所有收银员的默认购物车客户。
- 产品条形码字段。
- 订单电子邮件设置。
- 用户角色权限。
