这些设置用于配置 POS 应用程序的后端功能，包括 PHP 和 WooCommerce 配置。你可以通过导航到 `WP Admin > POS > 设置` 来访问它们。

## 设置概览 {#settings-at-a-glance}

| 设置 | 作用 | 了解更多 |
|---|---|---|
| **常规** | 所有收银员的默认购物车客户、仅限 POS 的产品、小数数量以及产品条形码字段。 | [常规设置](./general) |
| **收银台** | POS 可用的支付网关，以及 POS 订单的处理方式。 | [收银台设置](./checkout) |
| **电子邮件通知** | 针对管理员、客户和收银员的各类电子邮件开关，用于控制 POS 订单会触发哪些 WooCommerce 邮件。 | [电子邮件通知](./email-notifications) |
| **访问权限** | 用户角色和功能权限，用于控制谁可以使用 POS，包括专用的收银员角色。 | [访问 POS](./access) |
| **会话** | 按用户和设备查看活动的 POS 登录会话，并可远程让设备退出登录。 | [会话](./sessions) |
| **商店税号** | 打印在收据和发票上的企业税务标识（VAT、ABN、GSTIN、EIN、公司注册号）。 | [商店税号](./store-tax-ids) |
| **客户税号** | 客户税号的存储方式、写入的插件字段，以及结账时如何复制到订单中。 | [客户税号](./customer-tax-ids) |
