**会话**屏幕可让您查看当前登录 POS 的人员、在哪些设备上登录，以及远程注销各个设备。当设备丢失或被盗、收银员离开或您只是想查看活动登录时，您可以去那里。

您可以在 **`WP Admin > POS > Settings > Sessions`** 找到它。

> _管理活动用户会话。您可以查看所有登录的用户，查看他们的设备，并根据需要终止会话。_

## 它是如何运作的 {#how-it-works}

屏幕有两个窗格：

- **用户**（左）- 每个用户至少有一个活跃会话，在过去几分钟内活跃的任何人上都有一个绿点，您自己的帐户有一个 **You** 筹码，以及每个用户拥有多少会话的计数。过滤框可让您按名称或用户名进行搜索。
- **会话**（右）- 选择用户以在每个活动会话中查看一张卡片。

每个会话卡显示：

| 领域 | 这意味着什么 |
| --- | --- |
| **申请** | 用户登录的位置 - **iOS 应用程序**、**Android 应用程序**、**桌面应用程序** 或 **Web 应用程序** - 加上浏览器/版本 |
|**当前设备**|标记您当前正在使用的会话的芯片|
| **最后活跃** | 会话最近多久发出了请求（例如_刚刚_、_5 分钟前_） |
| **IP** | 会话连接的 IP 地址（或_N/A_） |
| **创建** | 会话首次登录时 |
| **过期** | 当会话令牌过期时 |
| **用户代理** | 展开以查看原始浏览器/设备用户代理字符串 |

当会话在过去 5 分钟内发出请求时，它会显示为 **现在处于活动状态**。

## 注销设备 {#signing-devices-out}

可以执行三个操作，每个操作都有一个确认对话框：

- **终止** — 注销单个会话。使用该会话卡上的按钮。
- **注销其他设备** — 将所选用户从每台设备上注销**除了**他们当前使用的设备。当用户有多个会话时显示。
- **全部注销** — 将所选用户从**每台**设备中注销，包括当前设备。

:::tip 设备丢失或被盗
如果带有 POS 的手机、平板电脑或笔记本电脑丢失，请打开“会话”，选择受影响的用户，然后在该设备的会话上使用“**终止**”（或“全部注销”以确保安全）。设备在下一个请求时将失去访问权限。
:::

:::note
终止您自己的当前会话将使您退出当前使用的设备 - 在此之前系统会要求您确认。
:::
