**会话**界面可以查看当前哪些用户已登录 POS、使用了哪些设备，并支持远程登出单个设备。当设备丢失或被盗、收银员离职，或需要审查活跃登录情况时，都可以在此处操作。

可以在 **`WP Admin > POS > 设置 > Sessions`** 中找到该页面。

> _管理活跃的用户会话。可以查看所有已登录的用户及其设备，并在需要时终止会话。_

## 工作原理 {#how-it-works}

该界面分为两个面板：

- **用户**（左侧）——列出所有至少拥有一个活跃会话的用户。最近几分钟内活跃的用户会显示绿色圆点，您自己的账户会标注**您**标签，并显示每个用户的会话数量。筛选框支持按姓名或用户名搜索。
- **会话**（右侧）——选择一个用户即可查看其每个活跃会话的卡片。

每张会话卡片显示：

| 字段 | 含义 |
| --- | --- |
| **应用程序** | 用户登录的位置——**iOS 应用程序**、**Android 应用程序**、**桌面应用程序**或 **Web 应用程序**——以及浏览器/版本 |
| **当前设备** | 标记您当前正在使用的会话的标签 |
| **最后活跃** | 该会话最近一次发出请求的时间（例如 _刚刚_、_5 分钟前_） |
| **IP** | 该会话的连接 IP 地址（或 _N/A_） |
| **创建时间** | 该会话首次登录的时间 |
| **过期时间** | 会话令牌的过期时间 |
| **用户代理** | 展开可查看原始浏览器/设备用户代理字符串 |

当会话在过去 5 分钟内发出过请求时，将显示为**当前活跃**。

## 退出设备登录 {#signing-devices-out}

有三个操作可用，每个操作都需要确认：

- **终止** ——退出单个会话。使用该会话卡片上的按钮。
- **退出其他设备** ——将所选用户从所有设备退出登录，**除了**当前正在使用的设备。当用户有多个会话时显示。
- **全部登出** — 将所选用户从**所有**设备上登出，包括当前设备。

:::tip 设备丢失或被盗
如果安装了 POS 的手机、平板电脑或笔记本电脑丢失，请打开会话管理，选择受影响的用户，然后对该设备的会话使用**终止**（或使用**全部登出**以确保安全）。该设备将在下一次请求时失去访问权限。
:::

:::note
终止您当前的会话会将您从当前使用的设备上登出——在此操作执行前，系统会要求您确认。
:::
