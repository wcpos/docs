## 条形码检测的工作原理

大多数条形码扫描仪的行为就像连接到您的设备的键盘。当您扫描条形码时，POS 检测到字符的输入比正常打字要快。它使用这些“快速按键”来识别输入为条形码扫描。

## 设置

| 设置 | 描述 |
|------|------|
| **平均输入时间** | 输入应该多快（以毫秒为单位）才能被算作条形码。较低的值 = 需要更快的输入 |
| **最小长度** | 连续字符字符串必须有多长才能被视为条形码 |
| **前缀移除** | 要从扫描的条形码开头处移除的字符 |
| **后缀移除** | 要从扫描的条形码末尾处移除的字符 |

## 前缀/后缀移除

如果您的扫描仪添加了额外字符（如前缀或后缀），您可以自动移除它们，以便仅保留主要条形码。

**示例：** 如果您的扫描仪在每个条形码前添加“A”，在之后添加“Z”，请配置：
- 前缀移除: `A`
- 后缀移除: `Z`

## 故障排除

### 条形码未检测到

- **扫描仪太慢**: 尝试增加“平均输入时间”值
- **条形码太短**: 尝试减少“最小长度”值
- **额外字符**: 检查您的扫描仪是否添加了前缀/后缀

### 正常输入被检测为条形码

- **输入太快**: 尝试降低“平均输入时间”值
- **输入太长**: 尝试增加“最小长度”值
