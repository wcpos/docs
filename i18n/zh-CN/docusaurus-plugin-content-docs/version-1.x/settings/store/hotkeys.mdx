## 可用快捷键

| 按键组合 | 操作 | 描述 |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 设置 | 打开POS设置面板 |
| `Ctrl + Shift + A` | POS | 导航到主POS屏幕 |
| `Ctrl + Shift + P` | 产品 | 打开产品管理屏幕 |
| `Ctrl + Shift + O` | 订单 | 打开订单管理屏幕 |
| `Ctrl + Shift + C` | 客户 | 打开客户管理屏幕 |
| `Ctrl + Shift + ?` | 支持 | 打开支持/帮助部分 |
| `Ctrl + Shift + L` | 注销 | 退出当前会话 |

## 使用键盘快捷键

使用任何键盘快捷键：

1. **按住修饰键**（`Ctrl + Shift`）
2. **按下操作键**（对应你想要的操作的字母）
3. **释放所有键**以执行命令

## 高效使用提示

- **记忆常用快捷键**：专注于学习你最常执行的操作的快捷键
- **练习按键组合**：经常使用将帮助建立肌肉记忆
- **在繁忙时段使用快捷键**：键盘快捷键可以显著加快高峰期的操作
