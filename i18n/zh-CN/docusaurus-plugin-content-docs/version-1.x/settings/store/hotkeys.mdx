## 可用快捷键

| Key Combination | Action | Description |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 设置   | 打开 POS 设置面板 |
| `Ctrl + Shift + A` | POS    | 导航到主要的 POS 屏幕 |
| `Ctrl + Shift + P` | 产品   | 打开产品管理屏幕 |
| `Ctrl + Shift + O` | 订单   | 打开订单管理屏幕 |
| `Ctrl + Shift + C` | 客户   | 打开客户管理屏幕 |
| `Ctrl + Shift + ?` | 支持   | 打开支持/帮助部分 |
| `Ctrl + Shift + L` | 注销   | 注销当前会话 |

## 使用键盘快捷键

要使用任何键盘快捷键：

1. **按住修饰键** (`Ctrl + Shift`)
2. **按下操作键**（与您所需操作相对应的字母）
3. **释放所有按键**以执行命令

## 高效使用提示

- **记住常用快捷键**：专注于学习您最常执行操作的快捷键
- **练习键组合**：定期使用有助于建立肌肉记忆
- **在繁忙期间使用快捷键**：键盘快捷键可以显著加快高峰时段的操作

## 自定义

:::info 功能请求
当前不支持键盘快捷键自定义。上面列出的快捷键是固定的，无法通过设置界面进行修改。

如果您想自定义键盘快捷键，请在 [Discord](https://wcpos.com/discord) 上分享您的反馈。
:::
