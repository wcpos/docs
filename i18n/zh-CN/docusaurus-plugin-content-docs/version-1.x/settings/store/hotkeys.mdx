## 可用的快捷键 {#available-shortcuts}

| 键组合 | 动作 | 描述 |
|--------|------|------|
| `Ctrl + Shift + S` | 设置 | 打开 POS 设置面板 |
| `Ctrl + Shift + A` | POS | 导航到主 POS 屏幕 |
| `Ctrl + Shift + P` | 产品 | 打开产品管理屏幕 |
| `Ctrl + Shift + O` | 订单 | 打开订单管理屏幕 |
| `Ctrl + Shift + C` | 客户 | 打开客户管理屏幕 |
| `Ctrl + Shift + ?` | 支持 | 打开支持/帮助部分 |
| `Ctrl + Shift + L` | 登出 | 登出当前会话 |

## 使用键盘快捷键 {#using-keyboard-shortcuts}

要使用任何键盘快捷键：

1. **按住修饰键** (`Ctrl + Shift`)
2. **按下动作键**（对应于您所需动作的字母）
3. **释放所有键**以执行命令

## 提高使用效率的小贴士 {#tips-for-efficient-use}

- **记住常用快捷键**：专注于学习您最常执行操作的快捷键
- **练习键组合**：定期使用将帮助建立肌肉记忆
- **在繁忙时段使用快捷键**：键盘快捷键可以显著加快繁忙时段的操作

## 自定义 {#customisation}

:::info 功能请求
当前无法自定义键盘快捷键。上述列出的快捷键是固定的，无法通过设置界面进行修改。

如果您希望自定义键盘快捷键，请在 [Discord](https://wcpos.com/discord) 上分享您的反馈。
:::
