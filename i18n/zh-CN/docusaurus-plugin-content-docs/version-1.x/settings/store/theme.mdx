## 可用主题 {#available-themes}

### 系统主题 {#system-theme}

- **描述**: 遵循您的设备设置
- **行为**: 根据设备的系统偏好自动在明亮和黑暗模式之间切换

### 明亮主题 {#light-theme}

- **描述**: 干净而明亮
- **最适合**: 光线充足的环境和喜欢传统明亮界面的用户

### 黑暗主题 {#dark-theme}

- **描述**: 养眼
- **最适合**: 光线较暗的环境和减少长时间使用时的眼睛疲劳

### 海洋主题 {#ocean-theme}

- **描述**: 清凉的蓝色和青色
- **最适合**: 创建轻松、专业的氛围

### 日落主题 {#sunset-theme}

- **描述**: 温暖的橙色和紫色
- **最适合**: 创建温馨、宜人的氛围

### 单色主题 {#monochrome-theme}

- **描述**: 高对比度的灰度
- **最适合**: 无障碍需求和需要高对比度的环境

## 选择主题 {#selecting-a-theme}

1. 在主题设置面板中，您将看到所有可用主题以卡片形式显示
2. 每个主题卡片显示主题名称和简要描述
3. 点击任意主题卡片进行选择
4. 当前活动主题将以蓝色边框突出显示
5. 您的选择会自动保存并立即应用

## 主题选择提示 {#tips-for-theme-selection}

| 主题 | 最佳环境 |
|-------|------------------|
| **系统** | 当您希望 POS 自动匹配您的设备时 |
| **明亮** | 光线良好的零售环境 |
| **黑暗** | 光线暗淡的环境或晚班/夜班 |
| **海洋/日落** | 增添个性和品牌一致性 |
| **单色** | 当您需要最大对比度时 |

您选择的主题将在所有 POS 会话中持续存在。
