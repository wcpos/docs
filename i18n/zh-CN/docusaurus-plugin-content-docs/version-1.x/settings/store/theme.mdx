## 可用主题

### 系统主题

- **描述**：跟随您的设备设置
- **行为**：根据您的设备系统偏好自动切换浅色和深色模式

### 浅色主题

- **描述**：简洁明亮
- **最佳场景**：光线充足的环境和喜欢传统浅色界面的用户

### 深色主题

- **描述**：护眼模式
- **最佳场景**：光线较暗的环境和长时间使用时减轻眼部疲劳

### 海洋主题

- **描述**：清凉的蓝色和蓝绿色
- **最佳场景**：营造轻松专业的氛围

### 日落主题

- **描述**：温暖的橙色和紫色
- **最佳场景**：营造温馨宜人的氛围

### 单色主题

- **描述**：高对比度灰度
- **最佳场景**：无障碍需求和需要高对比度的环境

## 选择主题

1. 在主题设置面板中，您将看到以卡片形式展示的所有可用主题
2. 每个主题卡片显示主题名称和简要描述
3. 点击任何主题卡片进行选择
4. 当前活动主题将以蓝色边框突出显示
5. 您的选择将自动保存并立即应用

## 主题选择建议

| 主题 | 最佳环境 |
|-------|------------------|
| **系统** | 希望POS自动匹配设备 |
| **浅色** | 光线充足的零售环境 |
| **深色** | 光线昏暗的环境或夜班 |
| **海洋/日落** | 添加个性和品牌标识 |
| **单色** | 需要最大对比度时 |

您选择的主题将在所有POS会话中保持一致。
