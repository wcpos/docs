## 可用主题

### 系统主题

- **描述**：遵循您的设备设置
- **行为**：根据设备的系统偏好自动在亮模式和暗模式之间切换

### 明亮主题

- **描述**：干净明亮
- **最佳适用**：光线充足的环境和偏好传统亮界面的用户

### 暗色主题

- **描述**：对眼睛友好
- **最佳适用**：低光环境和在长时间使用中减少眼睛疲劳

### 海洋主题

- **描述**：冷静的蓝色和青色
- **最佳适用**：营造轻松、专业的氛围

### 日落主题

- **描述**：温暖的橙色和紫色
- **最佳适用**：营造舒适、温馨的氛围

### 单色主题

- **描述**：高对比度的灰度
- **最佳适用**：需要无障碍的环境和高对比度的环境

## 选择主题

1. 在主题设置面板中，您将看到所有可用主题以卡片形式显示
2. 每个主题卡片显示主题名称和简要描述
3. 点击任何主题卡片以选择它
4. 当前活动主题会用蓝色边框突出显示
5. 您的选择会自动保存并立即应用

## 主题选择提示

| 主题 | 最佳环境 |
|-------|------------------|
| **系统** | 当您希望POS自动与设备匹配时 |
| **明亮** | 照明良好的零售环境 |
| **暗色** | 光线昏暗的环境或晚上班 |
| **海洋/日落** | 增加个性和品牌一致性 |
| **单色** | 当您需要最大对比度时 |

您选择的主题将在所有POS会话中保持不变。
