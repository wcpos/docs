商店设置是影响 POS 操作的应用范围内配置。这些设置适用于整个商店，而不是单个屏幕。

## 访问商店设置 {#accessing-store-settings}

通过以下方式打开商店设置：

- 点击 **用户菜单**（右上角）并选择 **设置**
- 使用键盘快捷键 `Ctrl + Shift + S`

## 设置选项卡 {#settings-tabs}

商店设置模态包含以下选项卡：

| 选项卡 | 描述 |
|-----|-------------|
| [常规](./general) | 商店信息、语言、货币、默认客户 |
| [税收](./tax) | 税收计算和显示选项 |
| [条形码](./barcode) | 条形码扫描器配置 |
| [键盘快捷键](./hotkeys) | 可用的键盘快捷键 |
| [主题](./theme) | 外观和颜色方案 |
