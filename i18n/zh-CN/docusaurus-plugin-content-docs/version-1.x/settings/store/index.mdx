商店设置是影响POS运营方式的全应用配置。这些设置适用于整个商店，而非个别屏幕。

## 访问商店设置

通过以下方式打开商店设置：

- 点击**用户菜单**（右上角）并选择**设置**
- 使用键盘快捷键 `Ctrl + Shift + S`

## 设置选项卡

商店设置弹窗包含以下选项卡：

| 选项卡 | 描述 |
|-----|-------------|
| [常规](./general) | 商店信息、语言、货币、默认客户 |
| [税务](./tax) | 税收计算和显示选项 |
| [条形码](./barcode) | 条形码扫描器配置 |
| [键盘快捷键](./hotkeys) | 可用的键盘快捷键 |
| [主题](./theme) | 外观和颜色方案 |
