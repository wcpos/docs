:::note 每台设备一个设置位置
硬件中心会引导你前往每台设备的设置说明，但实际步骤只放在一个位置。读卡器的账户和配对流程位于其支付网关页面；现金抽屉的弹开设置位于收据打印机页面。
:::

## 读卡器兼容性 {#card-reader-compatibility}

哪些读卡器可用取决于**你运行 WCPOS 的方式**以及读卡器的连接类型：

- **Web 应用**只能驱动提供 **Web（浏览器）SDK** 的读卡器，通常是联网的柜台终端。**仅支持 Bluetooth** 的读卡器（例如 Stripe M2 / WisePad 3、SumUp Air）以及手机 **Tap-to-Pay** 在 Web 应用中**不受支持**。
- 支持情况**按网关而定**，并非通用。购买前，请根据计划使用的网关核对读卡器（[Stripe Terminal](/payment/gateways/stripe-terminal)、[SumUp](/payment/gateways/sumup-terminal)、[PayPal Reader](/payment/gateways/paypal-reader)）。

每个网关页面都会列出其支持的确切读卡器。
