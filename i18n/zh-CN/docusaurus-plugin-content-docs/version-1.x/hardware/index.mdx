WCPOS 可与你柜台现有的实体设备配合使用。本页是连接硬件的起点——包括收据打印机、条码扫描器、读卡器和钱箱。每种设备类型要么在此处有自己的设置页面，要么会指向其实际设置所在的页面。

## 设备 {#devices}

  <LinkCard

    title="钱箱"

  />
</LinkCards>

:::note 每台设备一个设置位置
硬件中心会指向各设备的设置说明，但实际步骤只放在一个位置。读卡器的账户和配对流程位于其支付网关页面；现金抽屉的弹开设置位于收据打印机页面。
:::
