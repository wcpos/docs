:::info 专业版功能
客户界面需要 [WCPOS 专业版](/getting-started/pro-license)。免费用户可以为订单选择现有客户，但无法查看完整的客户列表或编辑客户详细信息。
:::

客户界面提供了直接在POS系统中进行全面的客户管理。无需切换到WooCommerce管理后台即可查看、编辑和创建客户。

{/* Screenshot: customers-screen.png */}

## 界面概述

### 标题操作

屏幕顶部：

- **搜索栏** - 按姓名、邮箱等查找客户
- **添加客户**（<Icon name="plus" />） - 创建新客户
- **显示设置**（<Icon name="sliders" />） - 配置可见列

### 客户表格

主区域显示客户信息，包括：

- **头像** - 客户个人资料图片（或占位符）
- **名字** - 客户的名字
- **姓氏** - 客户的姓氏（可排序）
- **邮箱** - 联系邮箱地址
- **账单地址** - 完整的账单地址
- **创建日期** - 客户添加的时间
- **操作** - 三点菜单

### 底部

- 客户数量及同步按钮

## 主要功能

### 客户搜索

快速查找客户：

- 名字或姓氏
- 邮箱地址
- 地址信息

### 添加新客户

直接从POS系统创建客户：

1. 点击标题中的 <Icon name="plus" /> 图标
2. 填写客户详细信息
3. 保存新客户

客户将在WooCommerce中创建并立即可用于订单。

### 编辑客户

更新客户信息：

1. 点击客户的三点菜单
2. 选择**编辑**
3. 修改详细信息（姓名、邮箱、地址）
4. 保存更改

更改将自动同步到WooCommerce。

### 可用列

| 列 | 描述 |
|--------|-------------|
| **图片** | 客户头像 |
| **ID** | WooCommerce客户ID |
| **名字** | 客户名字 |
| **姓氏** | 客户姓氏 |
| **邮箱** | 邮箱地址 |
| **角色** | 用户角色（客户、订阅者等） |
| **用户名** | WordPress用户名 |
| **账单地址** | 账单信息 |
| **配送地址** | 配送信息 |
| **创建日期** | 客户添加时间 |
| **修改日期** | 最后更新 |
| **操作** | 编辑、同步、删除 |

## 客户操作

点击三点菜单（⋮）查看选项：

- **编辑** - 修改客户详细信息
- **同步** - 从服务器刷新客户信息
- **删除** - 从本地数据库中移除

:::note
从POS系统删除客户仅会本地移除。客户仍然存在于WooCommerce，并将在下次同步时重新出现。
:::

## 在订单中使用客户

为订单分配客户：

1. 在[购物车面板](/pos/cart)，点击客户徽章
2. 通过姓名或邮箱搜索客户
3. 选择客户
4. 客户现在与订单关联

客户信息（账单/配送地址）将自动用于订单。

## 同步

客户数据在POS系统和WooCommerce之间同步：

- **实时更新** - 更改跨所有设备同步
- **WooCommerce集成** - 客户存储在WooCommerce中
- **离线功能** - 即使离线也可查看客户
