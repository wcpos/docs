:::info 专业功能
客户界面需要 [WCPOS Pro](/getting-started/pro-license)。免费用户可以为订单选择现有客户，但无法查看完整的客户列表或编辑客户详情。
:::

客户界面提供全面的客户管理，直接在 POS 中进行。查看、编辑和创建客户时无需切换到 WooCommerce 管理后台。

{/* Screenshot: customers-screen.png */}

## 界面概述

### 顶部操作

在屏幕顶部：

- **搜索框** - 通过姓名、电子邮件等查找客户。
- **添加客户** (<Icon name="plus" />) - 创建新客户
- **显示设置** (<Icon name="sliders" />) - 配置可见列

### 客户表

主要区域显示客户信息，包括：

- **头像** - 客户头像（或占位符）
- **名** - 客户的名字
- **姓** - 客户的姓（可排序）
- **电子邮件** - 联系电子邮件地址
- **账单地址** - 完整账单地址
- **创建日期** - 客户添加的日期
- **操作** - 三点菜单

### 底部

- 客户数量与同步按钮

## 关键特性

### 客户搜索

快速查找客户，方法有：

- 名字或姓氏
- 电子邮件地址
- 地址信息

### 添加新客户

直接从 POS 创建客户：

1. 点击顶部的 <Icon name="plus" /> 图标
2. 填写客户信息
3. 保存新客户

客户将在 WooCommerce 中创建，并立即可用于订单。

### 编辑客户

更新客户信息：

1. 点击客户的三点菜单
2. 选择 **编辑**
3. 修改详情（姓名、电子邮件、地址）
4. 保存更改

更改会自动同步到 WooCommerce。

## 显示设置

点击滑块图标 (<Icon name="sliders" />) 自定义可见列。

### 可用列

| 列 | 描述 |
|--------|-------------|
| **图片** | 客户头像 |
| **ID** | WooCommerce 客户 ID |
| **名** | 客户名字 |
| **姓** | 客户姓 |
| **电子邮件** | 电子邮件地址 |
| **角色** | 用户角色（客户、订阅者等） |
| **用户名** | WordPress 用户名 |
| **账单地址** | 账单信息 |
| **发货地址** | 发货信息 |
| **创建日期** | 客户添加的日期 |
| **修改日期** | 最后更新 |
| **操作** | 编辑、同步、删除 |

## 客户操作

点击三点菜单 (⋮) 选择操作：

- **编辑** - 修改客户详情
- **同步** - 从服务器刷新客户信息
- **删除** - 从本地数据库中移除

:::note
从 POS 删除客户仅会在本地移除该客户。该客户仍然在 WooCommerce 中，并将在下次同步时重新出现。
:::

## 在订单中使用客户

要将客户分配给订单：

1. 在 [购物车面板](/pos/cart) 中，点击客户徽章
2. 通过姓名或电子邮件搜索客户
3. 选择客户
4. 该客户现在与订单关联

客户信息（账单/发货地址）会自动用于订单。

## 同步

客户数据在 POS 和 WooCommerce 之间同步：

- **实时更新** - 更改会在所有设备间同步
- **WooCommerce 集成** - 客户存储在 WooCommerce 中
- **离线功能** - 即使离线也能查看客户
