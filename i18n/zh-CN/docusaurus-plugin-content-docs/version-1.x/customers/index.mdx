:::info 专业功能
客户界面需要 [WCPOS Pro](/getting-started/pro-license)。免费用户可以为订单选择现有客户，但无法查看完整客户列表或编辑客户详情。
:::

客户界面提供了全面的客户管理，直接在 POS 中完成。查看、编辑和创建客户，无需切换到 WooCommerce 管理界面。

{/* 截图：customers-screen.png */}

## 界面概述 {#interface-overview}

### 头部操作 {#header-actions}

在屏幕顶部：

- **搜索栏** - 按名称、电子邮箱等查找客户
- **添加客户** (<Icon name="plus" />) - 创建新客户
- **显示设置** (<Icon name="sliders" />) - 配置可见列

### 客户表 {#customer-table}

主要区域显示客户信息，包括：

- **头像** - 客户头像（或占位符）
- **名** - 客户的名字
- **姓** - 客户的姓氏（可排序）
- **电子邮箱** - 联系电子邮箱地址
- **账单地址** - 完整账单地址
- **创建日期** - 客户添加时间
- **操作** - 三点菜单

### 页脚 {#footer}

- 客户数量和同步按钮 (<Icon name="arrow-rotate-right" />)。 **长按** 可选择清除和刷新选项

## 关键功能 {#key-features}

### 客户搜索 {#customer-search}

通过以下方式快速查找客户：

- 名字或姓氏
- 电子邮箱地址
- 地址信息

### 添加新客户 {#add-new-customer}

直接从 POS 创建客户：

1. 点击头部的 <Icon name="plus" /> 图标
2. 填写客户详情
3. 保存新客户

客户在 WooCommerce 中创建，并立即可用于订单。

### 编辑客户 {#edit-customer}

更新客户信息：

1. 点击客户旁的三点菜单
2. 选择 **编辑**
3. 修改详情（姓名、电子邮箱、地址）
4. 保存更改

更改会自动同步到 WooCommerce。

### 可用列 {#available-columns}

| 列 | 描述 |
|--------|-------------|
| **图片** | 客户头像 |
| **ID** | WooCommerce 客户 ID |
| **名** | 客户名字 |
| **姓** | 客户姓氏 |
| **电子邮箱** | 电子邮箱地址 |
| **角色** | 用户角色（客户、订阅者等） |
| **用户名** | WordPress 用户名 |
| **账单地址** | 账单信息 |
| **送货地址** | 送货信息 |
| **创建日期** | 客户添加时间 |
| **修改日期** | 最后更新 |
| **操作** | 编辑、同步、删除 |

## 客户操作 {#customer-actions}

点击三点菜单 (⋮) 获取选项：

- **编辑** - 修改客户详情
- **同步** - 从服务器刷新客户
- **删除** - 从本地数据库中移除

:::note
从 POS 删除客户仅会在本地移除该客户。客户仍保留在 WooCommerce 中，并将在下次同步时重新出现。
:::

## 在订单中使用客户 {#using-customers-in-orders}

要将客户分配给订单：

1. 在 [购物车面板](/pos/cart) 中，点击客户徽章
2. 按名称或电子邮件搜索客户
3. 选择客户
4. 客户现在与订单关联

客户信息（账单/送货地址）会自动用于订单。

## 同步 {#synchronization}

客户数据在 POS 和 WooCommerce 之间同步：

- **实时更新** - 变更将在所有设备上同步
- **WooCommerce 集成** - 客户存储在 WooCommerce 中
- **离线能力** - 即使离线也能查看客户
