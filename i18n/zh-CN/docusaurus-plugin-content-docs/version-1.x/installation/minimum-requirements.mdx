---
title: 最低要求
---

为了确保 WooCommerce POS 插件的正常功能和最佳性能，请确保您的环境满足以下最低要求：

### 服务器要求：

- **WordPress：** 版本 5.6 或更高版本
- **WooCommerce：** 版本 5.3 或更高版本
- **PHP：** 版本 7.2 或更高版本

满足这些最低要求可以确保 WooCommerce POS 插件与您的 WordPress 环境之间的兼容性和稳定性。它还确保您可以从插件提供的最新功能和增强功能中受益。

### 浏览器要求：

- **现代 Web 浏览器：** WooCommerce POS 插件设计用于与现代 Web 浏览器配合使用，包括 Google Chrome、Mozilla Firefox、Safari 和 Microsoft Edge。请确保您正在使用首选浏览器的最新稳定版本。
- **启用 JavaScript：** WooCommerce POS 插件依赖 JavaScript 进行交互式功能。请确保在浏览器设置中启用了 JavaScript。
- **没有私密浏览模式：** 浏览器不应处于私密浏览模式，因为它可能限制对浏览器的 IndexedDB（IDB）数据库的访问。私密浏览模式可能会阻止 WooCommerce POS 插件中某些功能的正常运行。

通过满足最低要求并使用兼容的 Web 浏览器，您可以在使用 WooCommerce POS 插件时拥有无缝的体验，并充分利用其管理您的销售点操作的潜力。
