:::important
[WCPOS Pro](http://wcpos.com/pro) 是一个付费插件，它为免费的 [WCPOS插件](http://wordpress.org/plugins/woocommerce-pos/) 添加了功能。
:::

## 安装

如果您已经购买了WCPOS Pro的许可证，请按照以下步骤安装和激活插件：

1.  访问：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  在 **Downloads** 下，点击下载链接并将插件保存到您的桌面。
3.  然后进入您的网站，登录并进入 `WP Admin > Plugins > Add New > Upload Plugin`
4.  从您的桌面上传插件zip文件并激活。
5.  接下来，进入 `WP Admin > POS > Settings > License` 并输入您的许可证密钥和许可证电子邮件以完成激活。

![My Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "您可以在您的账户页面下载WCPOS Pro")

## 更新

当新版本准备好时，它们将出现在您的更新仪表板中，就像任何其他插件一样。您也可以从您的 [账户页面](https://wcpos.com/my-account/) 下载最新版本。
