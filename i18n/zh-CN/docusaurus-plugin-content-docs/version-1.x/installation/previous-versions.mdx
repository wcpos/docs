## 重新安装旧版本

如果您需要恢复到 WCPOS 插件的旧版本，请按照以下步骤操作：

1. 访问 WCPOS 的 WordPress.org 页面 [高级视图](https://wordpress.org/plugins/woocommerce-pos/advanced/)。

2. 滚动到 **高级选项** 部分，找到 **旧版本** 选择菜单。

3. 打开选择菜单，您将看到一个可用插件版本的列表。找到您想要安装的版本，然后单击旁边的 **下载** 按钮。将下载的文件保存到桌面或方便的位置。

4. 登录您的 WordPress 管理面板。

5. 通过单击左侧菜单中的 **插件**，导航到 **插件** 页面。

6. 停用并删除当前版本的 WCPOS 插件。这将从您的 WordPress 安装中删除插件。不用担心，您的数据不会丢失。

7. 单击页面顶部的 **添加新内容** 按钮。

8. 在 **添加插件** 页面上，单击 **上传插件** 按钮。

9. 从桌面或保存位置中选择以前下载的插件文件。

<div class="image-container">
  <Image
    alt="上传插件"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">在 WordPress 管理面板中上传插件</p>
</div>

10. 单击 **立即安装** 按钮，等待安装完成。

11. 安装完成后，单击 **激活** 按钮以激活 WCPOS 插件的旧版本。

就是这样！您已成功重新安装了 WCPOS 插件的旧版本。

如果在重新安装过程中遇到任何问题或需要进一步帮助，请联系我们的支持团队 support@wcpos.com。
