:::info 专业功能
订单界面需要 [WCPOS Pro](/getting-started/pro-license)。
:::

订单界面直接提供对您的订单历史的访问。查看过去的订单，重新打印收据，以及管理订单详情，而无需切换到 WooCommerce 管理后台。

{/* 截图：orders-screen.png */}

## 界面概述 {#interface-overview}

### 搜索与过滤 {#search-filters}

在界面顶部：

- **搜索框** - 通过订单号、客户姓名等查找订单
- **状态过滤器** - 按订单状态过滤（已完成、处理中等）
- **客户过滤器** - 按客户过滤
- **收银员过滤器** - 按处理订单的人员过滤
- **商店过滤器** - 按商店过滤（适用于多商店设置）
- **日期范围** - 按日期过滤

活动过滤器显示为蓝色徽章，可以通过单击 × 移除。

### 订单表 {#orders-table}

主要区域显示订单，包括：

- **状态图标** - 视觉指示器（✓ 已完成，🕐 待处理，🛒 开放）
- **订单号** - 唯一的订单 ID
- **客户** - 客户姓名（或"访客"）
- **账单地址** - 客户账单信息
- **备注指示器** - 当订单有备注时的气泡
- **创建日期** - 相对时间戳（"2小时前"）
- **收银员** - 处理订单的人
- **支付方式** - 现金、卡等
- **总计** - 订单总金额
- **操作** - 三点菜单

### 页脚 {#footer}

- 订单数量和同步按钮 (<Icon name="arrow-rotate-right" />)。**长按**以获取清除和刷新选项

## 主要功能 {#key-features}

### 订单查找 {#order-lookup}

快速查找订单的方法：

- **订单号搜索**
- **客户姓名搜索**
- **日期范围过滤**
- **状态过滤**
- **收银员过滤**

### 收据重新打印 {#receipt-reprinting}

对于任何订单，您可以：

1. 单击三点菜单
2. 选择收据选项
3. 查看、打印或发送电子邮件收据

### 订单详情 {#order-details}

单击订单以查看：

- 完整的项目细节
- 客户信息
- 支付详情
- 订单备注
- 元数据

### 可用列 {#available-columns}

| 列       | 描述                    |
|----------|-------------------------|
| **状态** | 订单状态图标           |
| **订单号** | 唯一的订单 ID         |
| **客户** | 客户姓名和详情         |
| **账单地址** | 账单信息             |
| **送货地址** | 送货信息             |
| **客户备注** | 客户的备注           |
| **创建日期** | 订单下单时间         |
| **修改日期** | 最近修改时间         |
| **完成日期** | 订单完成时间         |
| **支付日期** | 收到付款的时间       |
| **创建来源** | 订单来源（POS、在线等） |
| **收银员** | 处理订单的人          |
| **支付方式** | 使用的支付类型       |
| **总计** | 订单总计               |
| **收据** | 快速访问收据           |
| **操作** | 编辑、查看等           |

## 订单操作 {#order-actions}

单击三点菜单 (⋮) 获取选项：

- **查看** - 查看完整的订单详情
- **编辑** - 修改订单信息
- **收据** - 查看/打印/发送电子邮件收据
- **同步** - 从服务器刷新订单
