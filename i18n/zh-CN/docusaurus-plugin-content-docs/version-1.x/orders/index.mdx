:::info Pro 功能
订单界面需要 [WCPOS Pro](/getting-started/pro-license)。
:::

订单界面直接在 POS 内提供对订单历史的访问。查看过去的订单，重新打印收据，以及管理订单详情，而无需切换到 WooCommerce 管理界面。

{/* Screenshot: orders-screen.png */}

## 界面概述 {#interface-overview}

### 搜索与过滤 {#search--filters}

在界面顶部：

- **搜索栏** - 通过订单号、客户姓名等查找订单
- **状态过滤器** - 按订单状态过滤（已完成、处理中等）
- **客户过滤器** - 按客户过滤
- **收银员过滤器** - 按处理订单的人过滤
- **商店过滤器** - 按商店过滤（适用于多商店设置）
- **日期范围** - 按日期过滤

活动过滤器显示为蓝色徽章，可以通过单击 × 移除。

### 订单表 {#orders-table}

主要区域显示订单，包括：

- **状态图标** - 视觉指示器（✓ 已完成，🕐 待处理，🛒 开放）
- **订单号** - 唯一的订单 ID
- **客户** - 客户姓名（或“访客”）
- **账单地址** - 客户账单信息
- **备注指示器** - 当订单有备注时的气泡
- **创建日期** - 相对时间戳（“2 小时前”）
- **收银员** - 处理订单的人
- **支付方式** - 现金、卡等
- **总计** - 订单总金额
- **操作** - 三点菜单

### 页脚 {#footer}

- 订单数量和同步按钮 (<Icon name="arrow-rotate-right" />)。**长按**以获取清除和刷新选项

## 主要功能 {#key-features}

### 订单查找 {#order-lookup}

快速查找订单的方法：

- **订单号搜索**
- **客户姓名搜索**
- **日期范围过滤**
- **状态过滤**
- **收银员过滤**

### 收据重新打印 {#receipt-reprinting}

对于任何订单，您可以：

1. 单击三点菜单
2. 选择收据选项
3. 查看、打印或发送电子邮件收据

### 订单详情 {#order-details}

单击订单以查看：

- 完整的行项目细节
- 客户信息
- 支付详情
- 订单备注
- 元数据

## 订单操作 {#order-actions}

单击三点菜单 (⋮) 获取选项：

- **查看** - 查看完整的订单详情
- **编辑** - 修改订单信息
- **收据** - 查看/打印/发送电子邮件收据
- **同步** - 从服务器刷新订单

### 编辑订单 {#editing-orders}

从三点菜单中选择**编辑**以修改现有订单。这会在购物车中重新打开订单，您可以在其中：

- 添加或移除行项目
- 更改数量或价格
- 更新客户
- 添加备注或元数据

进行更改后，再次完成结账流程以保存更新后的订单。

### 订单状态 {#order-statuses}

POS 订单使用标准的 WooCommerce 订单状态，外加一个自定义状态：

:::info 典型订单流程
订单通常按 **POS - Open → Pending → Processing → Completed** 推进，**Cancelled** 或 **Refunded** 为终止状态。大多数 POS 订单会直接跳到 **Completed**，因为付款是在销售点收取的。
:::

| 状态 | 描述 |
|--------|-------------|
| **POS - Open** | 已从 POS 保存但尚未完成的订单（暂存/挂起订单） |
| **Pending** | 等待付款 |
| **Processing** | 已收到付款，订单正在履行 |
| **Completed** | 订单已履行且完成 |
| **Cancelled** | 订单已取消 |
| **Refunded** | 订单已全额退款 |

大多数 POS 订单会直接进入 **Completed**，因为付款是在销售点收取的。开放订单（使用购物车中的 **Save to Server** 按钮保存）使用 **POS - Open** 状态，之后可以重新打开以继续交易。

### 退款 {#refunds}

您可以直接从 POS 发起全额或部分退款——退回原支付方式（当网关支持时）或从收银台以现金退还。在某个订单上打开三点菜单 (⋮) 并选择**退款**，或单击订单视图页脚中的**退款**按钮。

完整操作步骤请参阅[退款](/orders/refunds)，包括退款去向、部分退款，以及退款在收据上的显示方式。
