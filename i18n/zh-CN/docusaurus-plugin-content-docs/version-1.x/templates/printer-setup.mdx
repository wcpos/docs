打印机设置位于 POS 的 **设置 > 打印机**。每台设备管理自己的打印机配置。打印机会保存在本机，不会在设备之间同步。

## 按平台支持的打印机 {#supported-printers-by-platform}

**添加打印机** 表单会根据你使用的应用自动调整。每个平台只显示它实际可用的连接方式和选项。

| 平台 | 支持的打印机 | 连接 |
|----------|--------------------|-------------|
| **Web 应用** | 带内置 Web 服务器的 Epson 和 Star 打印机 | 网络 (HTTP) |
| **桌面应用** | 任意网络小票打印机 | 网络 (raw TCP) |
| **iOS 应用** | Epson 和 Star | 网络、Bluetooth |
| **Android 应用** | Epson 和 Star | 网络、Bluetooth、USB |

:::info 为什么 Web 上只支持 Epson 和 Star?
Web 浏览器无法打开 raw TCP 连接。Epson 和 Star 小票打印机带有内置 Web 服务器，可以通过 HTTP 接收打印任务，这是从浏览器发送 ESC/POS 命令的唯一方式。桌面应用可以直接通过 TCP 连接，因此没有这个限制。

在 **移动应用** 中，Bluetooth 和 USB 打印会通过 Epson 与 Star 的官方 SDK，因此这些连接也需要 Epson 或 Star 打印机。通用打印机只能通过网络使用。
:::

## 添加打印机 {#adding-a-printer}

<Steps>
  <StepCard number="1" title="打开打印机设置">

在 POS 中前往 **设置 > 打印机**。如果还没有配置打印机，你会看到一个带 **添加打印机** 按钮的空状态。每台打印机都会得到一个 **打印机名称**。这只是标签，会自动填为“Receipt Printer”（之后是“Receipt Printer 2”等），你可以改成便于区分的任何名称。

  </StepCard>
  <StepCard number="2" title="选择打印机的连接方式">

这一步会按平台变化：

<ul>
  <li><strong>Web 应用</strong> — 选择 <strong>厂商</strong>：Epson 或 Star。这是 Web 上的第一项选择，因为浏览器只能访问这两个品牌的内置 Web 服务器。</li>
  <li><strong>桌面应用</strong> — 打印机通过网络连接；这里没有额外选项。</li>
  <li><strong>移动应用</strong> — 选择 <strong>连接类型</strong>：<strong>网络</strong>、<strong>Bluetooth</strong> 或 <strong>USB</strong>（USB 仅限 Android）。下方字段会随选择而变化。</li>
</ul>

  </StepCard>
  <StepCard number="3" title="识别打印机">

**网络打印机** — 输入 **IP 地址**（例如 `192.168.1.100`）。WCPOS 会探测该地址，判断它是 Epson 还是 Star 设备，并自动填充端口、命令语言和厂商。检测完成后，你会看到“已检测到: Epson”或“已检测到: Star”标签。在 Web 应用中，还会显示将要连接的准确端点 URL。

**Bluetooth 或 USB 打印机**（移动端）— 从发现的设备列表中选择打印机。选择设备后，名称和厂商会自动填入。如果打印机尚未列出，Bluetooth 请点 **扫描打印机**，USB 请点 **刷新**。

如果自动检测没有完全正确，可以在 **高级设置** 中调整所有内容。

  </StepCard>
  <StepCard number="4" title="保存并测试" isLast>

点击 **保存**。WCPOS 会先发送一次 **测试打印**，只有成功后才保存打印机。

测试打印是一段简短诊断内容，包括带编号的列尺、居中的普通和双倍大小文本、左右对齐行以及切纸。它用于确认纸张宽度、对齐、尺寸和切纸都配置正确，而不仅仅是确认打印机可访问。

如果测试失败，你会看到错误和 **不测试直接保存** 选项，以便仍然保存该配置文件。

  </StepCard>
</Steps>

## 连接类型（移动应用） {#connection-types-mobile-apps}

在 iOS 和 Android 应用中，打印机可通过三种方式连接。在添加打印机表单顶部选择 **连接类型**，连接字段会随之更新。

### 网络 {#network}

打印机与设备在同一个 Wi-Fi 网络中。输入它的 IP 地址；请参阅 [查找打印机的 IP 地址](#finding-your-printers-ip-address)。网络打印机可以是 Epson、Star 或通用打印机。

### Bluetooth {#bluetooth}

打印机已通过 Bluetooth 与设备配对。点 **扫描打印机** 并从列表中选择。名称和厂商会自动填入。Bluetooth 打印使用 Epson 与 Star 的原生 SDK，因此打印机必须是 **Epson 或 Star** 型号。

### USB（仅 Android） {#usb-android-only}

打印机通过 USB 线连接。点 **刷新** 并从已连接设备列表中选择。与 Bluetooth 一样，USB 打印需要 **Epson 或 Star** 打印机。iOS 没有通用 USB 外设支持，因此此选项只在 Android 上显示。

:::note
**端口** 设置只适用于网络打印机；Bluetooth 和 USB 连接会隐藏此设置。
:::

## 高级设置 {#advanced-settings}

展开 **高级设置** 可微调打印机配置。厂商已知后，大多数项目会自动填充。

| 设置 | 选项 | 说明 |
|---------|---------|-------------|
| **厂商** | Epson、Star、通用 | 打印机制造商。在 **Web 应用** 中，厂商是表单顶部的主要选择（仅 Epson 或 Star），而不是高级设置。通用适用于桌面和移动应用中的 **网络** 打印机，但不适用于 Bluetooth 或 USB 连接。 |
| **端口** | 自动填充 | 打印机的网络端口。仅限网络；Bluetooth 和 USB 会隐藏。 |
| **语言** | ESC/POS、StarPRNT、Star Line Mode | 打印机使用的命令协议。会根据检测到的厂商自动填充。 |
| **打印机文本宽度** | 58mm（32 个字符）、80mm 标准（42 个字符）、80mm 宽（48 个字符） | 每行可容纳的字符数。请与纸张和打印机匹配；大多数 80mm 打印机每行 42 个字符，有些为 48 个。 |
| **整张小票栅格化** | 关闭 / 开启 | 将整张小票作为图像打印，而不是文本；见下方 [整张小票栅格化](#full-receipt-raster)。 |

## 打印机选项 {#printer-options}

这些开关位于添加打印机表单底部：

| 选项 | 说明 |
|--------|-------------|
| **自动切纸** | 每张小票后自动切纸 |
| **自动打开钱箱** | 打印后发送 kick 命令打开已连接的钱箱 |
| **设为默认** | 将此打印机设为默认打印机，用于未路由到指定打印机的任何小票 |

:::note 在找“结账后自动打印”吗?
销售完成后自动打印小票属于 **购物车设置**，不是打印机设置。请在 POS 购物车设置中开启 **自动打印小票**。使用 *哪台* 打印机由默认打印机和模板级打印机路由决定。
:::

## 打印路由 {#print-routing}

如果使用多个模板，例如热敏小票 **和** A4 发票，打印路由会决定每个模板发送到哪台打印机。路由分三层，按以下顺序检查：

1. **单次任务覆盖。** 在小票屏幕上，模板切换器旁有一个打印机下拉菜单。在这里选择打印机会仅对这一次打印任务覆盖所有设置。切换模板会将其重置为 **自动**。
2. **设置覆盖。** 前往 **设置 → 打印路由**，为每个模板指定打印机。例如，将热敏小票路由到 Epson 网络打印机，将 HTML 发票路由到系统打印对话框。将模板设回 **自动** 可移除覆盖。
3. **自动匹配。** 没有覆盖时，WCPOS 会自动匹配：
   - **热敏模板** 会路由到字符宽度匹配的热敏打印机。58mm 热敏模板优先 32 列打印机；80mm 模板优先 42 或 48 列打印机。
   - **HTML 模板** 会路由到系统打印对话框。
   - 如果多台打印机匹配，则选择 **默认** 打印机。

如果手动将模板发送到不兼容的打印机，例如把热敏模板发送到系统对话框，小票屏幕会出现琥珀色 **不匹配警告**。打印仍会继续，但输出可能无法正确渲染。

路由覆盖按 **设备** 存储。每台 iPad、手机或电脑管理自己的路由；没有服务端同步。

:::note
只有添加至少一台打印机后，打印路由设置页才会出现。没有配置打印机时，所有模板都会默认使用系统打印对话框。
:::

## 整张小票栅格化 {#full-receipt-raster}

默认情况下，WCPOS 将小票以 **文本** 形式发送到热敏打印机：快速、紧凑、清晰，并使用打印机内置字体。问题是热敏打印机字体只覆盖有限字符集。打印机没有字体的文字体系，例如许多非拉丁字母、从右到左的文本和一些符号，可能会打印成空方框或乱码。

**整张小票栅格化** 可解决这个问题。开启后，WCPOS 会把整张小票渲染为图像并发送给打印机，因此打印结果会与屏幕上看到的完全一致，适用于任何语言或文字体系。

可在每台打印机的 **高级设置** 中开启。注意事项：

- **速度更慢。** 图像比一行文本大得多，因此发送和打印小票需要更长时间。
- **仅在需要时使用。** 如果小票作为文本打印正常，请保持关闭。
- 它适用于任何连接方式的热敏打印机（网络、Bluetooth、USB）。它不适用于系统打印对话框，因为后者已经从整页渲染进行打印。

## 发现打印机 {#discovering-printers}

在 **移动应用** 中，添加打印机表单会列出附近可用于 **Bluetooth**（iOS 和 Android）和 **USB**（仅 Android）连接的打印机。点 **扫描打印机** 或 **刷新**，然后从列表中选择。名称和厂商会自动填入。

对于任何平台上的 **网络** 打印机，请直接输入 IP 地址；见下方 [查找打印机的 IP 地址](#finding-your-printers-ip-address)。Web 应用没有网络发现功能，因为浏览器无法扫描网络，所以始终使用手动 IP 输入。

## 查找打印机的 IP 地址 {#finding-your-printer-s-ip-address}

大多数小票打印机可以打印包含 IP 地址的自检页：

- **Epson**：开机时按住进纸按钮
- **Star**：开机状态下按住进纸按钮 5 秒

也可以查看路由器的已连接设备列表或打印机配置工具。

### 建议使用静态 IP {#static-ip-recommended}

小票打印机应使用 **静态 IP 地址**，防止打印机重启后地址发生变化。可通过以下方式配置：

- 在打印机的内置 Web 界面中设置，通常可通过 `http://<printer-ip>` 访问
- 在路由器的 DHCP 设置中，为打印机的 MAC 地址分配保留 IP

## Epson 打印机 {#epson-printers}

支持 ePOS 的 Epson 打印机通过 HTTP 通信。WCPOS 会向打印机的内置 Web 服务器发送 SOAP/XML 请求。

**连接详情：**
- 端口 `8008` (HTTP) 或 `8043` (HTTPS)
- 端点：`/cgi-bin/epos/service.cgi`

**已确认可用的型号：**
- Epson TM-T70-i 2（网络）
- Epson TM-m30iii（使用 Printus 中间件）

:::tip
请确认 Epson 打印机配置中已启用 ePOS。通过 `http://<printer-ip>` 访问打印机的 Web 界面，检查并启用 ePOS 设置。
:::

在 **移动应用** 中，Epson 打印机也可以通过 Epson 原生 SDK 使用 Bluetooth 或 USB 连接，不需要 IP 地址或 ePOS Web 服务器。

## Star 打印机 {#star-printers}

支持 WebPRNT 的 Star 打印机通过 HTTP 通信。WCPOS 会向 Star WebPRNT 端点发送命令。

**连接详情：**
- HTTPS 或 HTTP
- 端点：`/StarWebPRNT/SendMessage`

在 **移动应用** 中，Star 打印机也可以通过 Star 原生 SDK 使用 Bluetooth 或 USB 连接。

## 通用打印机 {#generic-printers}

**桌面应用** 通过 raw TCP 支持任意网络小票打印机。输入打印机的 IP 地址和端口（通常是 `9100`）。只要打印机接受 ESC/POS 命令，不论厂商如何，大多数热敏打印机都可以使用。

在 **移动应用** 中也支持通用打印机，但仅限 **网络** 连接，不支持 Bluetooth 或 USB（这些需要 Epson 或 Star 打印机）。**Web 应用** 完全不支持通用打印机。

## 钱箱 {#cash-drawers}

连接到小票打印机的钱箱可以在每次销售后自动触发：

1. 在打印机选项中启用 **自动打开钱箱**
2. POS 在打印后发送 ESC/POS kick 命令

:::note
钱箱控制需要热敏打印机配置文件；浏览器打印对话框无法触发钱箱。
:::

## 故障排除 {#troubleshooting}

<Accordion>
  <AccordionItem question="设置时未检测到打印机">

  - 确认打印机已开机，并连接到与设备相同的网络
  - 网络打印机：检查 IP 地址是否正确（打印自检页）
  - Web 应用：确认打印机是支持 ePOS/WebPRNT 的 Epson 或 Star 型号
  - 桌面应用：尝试使用通用厂商选项和 9100 端口
  - Bluetooth：先在设备的 Bluetooth 设置中配对打印机，然后点 **扫描打印机**
  - USB（Android）：检查线缆，并在出现 USB 权限提示时批准
  - 检查是否有防火墙阻止连接

  </AccordionItem>

  <AccordionItem question="已检测到打印机但没有打印内容">

  - 检查纸卷，可能已用完或卡纸
  - 确认 **打印机文本宽度** 设置与纸张匹配（58mm 或 80mm）
  - 尝试从打印机本身打印自检页，以确认打印机可工作
  - 检查命令语言是否匹配打印机（Epson 使用 ESC/POS，Star 使用 StarPRNT 或 Star Line Mode）

  </AccordionItem>

  <AccordionItem question="小票乱码或字符错误">

  - 命令语言设置可能不正确；尝试在 ESC/POS、StarPRNT 和 Star Line Mode 之间切换
  - 确认 **打印机文本宽度** 设置正确
  - 如果小票使用非拉丁文字或从右到左的语言，请在高级设置中开启 **整张小票栅格化**。它会把小票作为图像打印，因此任何语言都能正确渲染

  </AccordionItem>

  <AccordionItem question="钱箱打不开">

  - 确认打印机选项中已启用 **自动打开钱箱**
  - 检查钱箱是否连接到打印机的 DK 端口，而不是单独电源
  - 并非所有打印机都支持钱箱 kick 命令；请查阅打印机文档

  </AccordionItem>

  <AccordionItem question="无法从 Web 应用连接">

  - Web 应用只能连接带内置 Web 服务器的 Epson 和 Star 打印机
  - Web 不支持通用和 USB 打印机；请改用桌面或移动应用
  - 如果站点使用 HTTPS，但打印机只支持 HTTP，浏览器可能会把连接作为混合内容阻止

  </AccordionItem>

  <AccordionItem question="Bluetooth 打印机无法连接（移动端）">

  - 在 WCPOS 中添加前，先在设备系统的 Bluetooth 设置中配对打印机
  - Bluetooth 打印仅支持 Epson 和 Star 打印机；不支持通用 Bluetooth 打印机
  - 如果打印机没有出现在列表中，请再次点 **扫描打印机**，并确认打印机已开机且在范围内

  </AccordionItem>
</Accordion>
