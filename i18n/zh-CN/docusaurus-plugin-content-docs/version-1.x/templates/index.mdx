WCPOS 包含完整的模板系统，适用于小票、发票、礼品小票、装箱单、厨房票据等——全部在 **WP Admin → POS → 模板** 中管理。

:::tip 只是想修改小票？
从 **[自定义小票](/templates/customise)** 开始。该指南介绍了三种最简单的方法——选择其他内置模板、让 AI 帮助调整，或手动编辑。
:::

## 模板引擎 {#template-types}

WCPOS 支持三种模板引擎。创建模板或点击模板库卡片上的 **使用模板** 时选择引擎，且 **之后无法更改**——如需切换引擎，请新建一个模板。

<FeatureGrid>
  <FeatureCard icon="code" title="无逻辑 HTML（推荐）">
    Mustache 风格 <code>{"{{variable}}"}</code> 在纯 HTML 中使用占位符。客户端渲染，因此收据**支持离线使用**。适用于浏览器打印和 PDF 收据。
  </FeatureCard>
  <FeatureCard icon="print" title="热敏 XML">
    XML 模板可同时生成屏幕预览和用于热敏收据打印机的 **ESC/POS 指令**。一套模板，两种输出。适用于 Epson、Star 及其他热敏打印机。
  </FeatureCard>
  <FeatureCard icon="terminal" title="旧版 PHP">
    使用 WooCommerce 函数的服务器端 PHP 模板。为兼容现有的 <code>yourtheme/woocommerce-pos/receipt.php</code> 覆盖文件而保留。渲染需要服务器连接。
  </FeatureCard>
</FeatureGrid>

## 模板库 {#template-gallery}

模板库提供了多种现成模板，可直接使用或自定义。支持按**类别**（收据、发票、礼品收据、厨房小票、报价单/采购订单）、**格式**（HTML / ESC/POS）和**方向**（LTR / RTL）筛选，并可使用店铺真实数据进行预览。每张卡片底部有**预览**链接和**使用模板**主按钮——点击“使用模板”会在*您的模板*中创建一个可编辑的副本。

| 模板 | 引擎 | 描述 |
|----------|--------|-------------|
| 标准收据 | HTML | 默认收据——Logo、店铺信息、商品、合计、付款 |
| 标准收据（RTL） | HTML | 适用于阿拉伯语、希伯来语、波斯语、乌尔都语的从右到左对应版本 |
| Minimal / Modern | HTML | 与 Standard 相同的基本内容，以更紧凑的垂直空间呈现 |
| 详细收据 | HTML | 完整税务发票——SKU 列、单价、分税率明细、地址 |
| 礼品收据 | HTML | 仅显示商品，隐藏价格。包含礼品留言和退换货政策 |
| 发票 | HTML | 全页 A4/Letter 发票，包含可选的“如何付款”面板，适用于未付款订单 |
| 装箱单 | HTML | 仓库配套单据——商品 + 数量、收货地址，不含价格 |
| 报价/估价 | HTML | 售前文档，包含价格和条款——无付款部分 |
| 窄幅收据 | HTML | 适用于窄幅纸张或支持 HTML 的热敏打印机的等宽字体收据 |
| 简易热敏小票（58mm） | 热敏 | 简洁的 58mm 热敏布局 |
| 简易热敏小票（80mm） | 热敏 | 简洁的 80mm 热敏布局——最常用 |
| 简易热敏小票 80mm (RTL) | Thermal | 80mm 的 RTL 对应版本。需要支持阿拉伯语代码页（CP864 / Windows-1256）的打印机 |
| 详细热敏收据（58mm） | 热敏 | 全功能 58mm —— 地址、税费明细、退款、付款、条款、条形码 |
| 详细热敏小票（80mm） | 热敏 | 全功能 80mm —— 地址、税费明细、退款、付款、条款、条形码 |
| 厨房票据 | 热敏 | 仅商品、大字体、无价格 —— 适用于备餐台 |

### 第三方文档模板 {#default-templates}

如果 WP Overnight 的 **PDF Invoices & Packing Slips for WooCommerce** 插件已启用，WCPOS 还会在小票模板列表中显示**发票（WP Overnight）**和**装箱单（WP Overnight）**。这些由服务器渲染的 HTML 模板会调用 WP Overnight 的文档 API，复用该插件中配置的发票和装箱单编号、布局、品牌标识、法律/税务字段以及模板自定义设置。

仅在 WP Overnight 插件启用时可用，且需要服务器连接才能渲染。如需离线打印，请使用 WCPOS 自带的 HTML 或热敏模板。

### 自适应税费显示 {#custom-templates}

大多数内置模板（Standard、Standard RTL、Minimal/Modern、Narrow、Invoice、Quote、Thermal Simple 系列）会**自动适应您的 WooCommerce 税费显示设置**：

- **含税店铺**（EU/UK/AU）显示含税价格，并附带“含税”行。
- **不含税店铺**（US/CA）显示税前价格，税费作为单独一行显示。
- 总计始终显示实际收取的含税金额。

**Detailed** 系列是正式的税务发票，无论店铺设置如何，始终以不含税行项目加明确的税费明细来列示税额。

## 工作原理 {#how-it-works}

1. **添加模板** — 点击模板库卡片上的 **使用模板**，或从头创建自己的模板。
2. **自定义模板** 使用应用内编辑器进行实时预览编辑，或将模板粘贴到 ChatGPT / Claude 中，描述您需要的更改。
3. **激活模板** — 在*您的模板*中切换 **启用** 开关。每个已激活的小票模板都会出现在收银台小票界面的下拉菜单中。
4. **打印或展示** — 结账后小票会自动打开，提供打印、发送电子邮件或在屏幕上查看等选项。

模板基于标准化的[小票数据载荷](/templates/receipt-data)进行渲染，包含店铺信息、商品明细、合计、税费、付款、退款等部分。所有货币字段均包含预格式化的 `_display` 变体（例如 `$29.99` 而非 `29.99`）。

## Pro 功能 {#pro-features}

使用 [WCPOS Pro](/getting-started/pro-license)，每个店铺可以拥有独立的模板分配、品牌设置（Logo、地址、联系方式）以及模板排序——适用于多门店运营场景，每个店铺可打印自己的专属信头。
