需要 WCPOS 的帮助？以下是获取支持和查找问题答案的方法。

## 应用内支持

POS 包含一个可从左侧导航抽屉访问的内置支持屏幕。该屏幕直接嵌入 WCPOS Discord 社区，允许您：

- 浏览现有讨论
- 搜索常见问题的答案
- 在 #general 频道提问
- 随时了解最新公告

{/* Screenshot: support-screen.png */}

## Discord 社区

WCPOS 的主要支持渠道是我们的 [Discord 服务器](https://wcpos.com/discord)。

### 频道

| 频道 | 目的 |
|---------|---------|
| **#welcome-and-rules** | 社区准则 |
| **#announcements** | 更新和新闻 |
| **#general** | 主要支持频道 - 在此处提问 |
| **#forum** | 更长的讨论和功能请求 |
| **#github-activity** | 开发更新 |

### 提问前

要获得最好的帮助：

1. **先测试** - 下载并测试插件几天，了解它是否满足您的需求
2. **查看文档** - 许多问题都可以在本文档中找到答案
3. **搜索 Discord** - 您的问题可能已经得到解答
4. **提供详细信息** - 包括您的 WooCommerce 版本、WCPOS 版本和重现问题的步骤

## 邮件支持

对于许可证相关问题或私人咨询，您可以发送邮件至：**support@wcpos.com**

:::note
一般支持问题应该在 Discord 上提出，这样社区可以帮助，其他人也能从答案中受益。
:::

## 日志屏幕

POS 包含一个记录系统活动的[日志屏幕](/support/logs)。这对以下方面非常有价值：

- 调试同步问题
- 理解错误消息
- 在请求支持时提供详细信息

请求帮助时，请检查日志屏幕上的任何错误消息并将其包含在您的支持请求中。

## 专业支持

[WCPOS Pro](https://wcpos.com/pro) 客户可以通过 Discord 获得相同的社区支持。专业版许可证资助了免费和专业功能的持续开发。

## 贡献

WCPOS 是开源的！您可以通过以下方式贡献：

- 在 [GitHub](https://github.com/wcpos) 上报告 bug
- 在 Discord 上建议功能
- 翻译文档
- 与他人分享您的体验
