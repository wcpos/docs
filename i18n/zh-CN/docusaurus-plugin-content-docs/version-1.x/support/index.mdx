需要有关 WCPOS 的帮助吗？以下是获取支持和寻找问题答案的方法。

## 应用内支持

POS 包含一个内置的支持屏幕，可从左侧导航抽屉访问。此屏幕在应用中直接嵌入 WCPOS Discord 社区，允许您：

- 浏览现有讨论
- 搜索常见问题的答案
- 在 #general 频道中提问
- 及时了解公告

## Discord 社区

WCPOS 的主要支持渠道是我们的 [Discord 服务器](https://wcpos.com/discord)。

### 频道

| 频道 | 目的 |
|---------|---------|
| **#欢迎与规则** | 社区准则 |
| **#公告** | 更新和新闻 |
| **#常规** | 主要支持频道 - 在这里提问 |
| **#论坛** | 更长的讨论和功能请求 |
| **#github-活动** | 开发更新 |

### 提问前的准备

为了获得最佳帮助：

1. **先测试** - 下载并测试插件几天，以了解它是否满足您的需求
2. **查看文档** - 许多问题在本文档中都有解答
3. **搜索 Discord** - 您的问题可能已经得到解答
4. **提供细节** - 包括您的 WooCommerce 版本、WCPOS 版本和重现问题的步骤

## 电子邮件支持

有关许可证相关问题或私人咨询，您可以发送电子邮件至：**support@wcpos.com**

:::note
一般支持问题应转到 Discord，以便社区可以提供帮助，并让其他人从答案中受益。
:::

## 日志屏幕

POS 包含一个 [日志屏幕](/support/logs)，记录系统活动。这对于以下方面至关重要：

- 调试同步问题
- 理解错误消息
- 在请求支持时提供详细信息

在请求帮助时，请检查日志屏幕中的任何错误消息，并将它们包含在您的支持请求中。

## 专业支持

[WCPOS Pro](https://wcpos.com/pro) 客户通过 Discord 接收相同的社区支持。Pro 许可证为免费和 Pro 功能的持续开发提供资金。

## 贡献

WCPOS 是开源的！您可以通过以下方式贡献：

- 在 [GitHub](https://github.com/wcpos) 上报告错误
- 在 Discord 上建议功能
- 翻译文档
- 与他人分享您的经验
