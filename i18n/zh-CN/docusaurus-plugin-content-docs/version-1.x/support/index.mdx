需要 WCPOS 的帮助吗？以下是获取支持和找到问题答案的方法。

## 应用内支持

POS 包含一个内置的支持屏幕，可以从左侧导航抽屉访问。此屏幕直接在应用程序中嵌入 WCPOS Discord 社区，允许您：

- 浏览现有讨论
- 搜索常见问题的答案
- 在 #general 频道提问
- 了解最新公告

## Discord 社区

WCPOS 的主要支持渠道是我们的 [Discord 服务器](https://wcpos.com/discord)。

### 频道

| 频道 | 目的 |
|------|------|
| **#welcome-and-rules** | 社区指南 |
| **#announcements** | 更新和新闻 |
| **#general** | 主要支持频道 - 在这里提问 |
| **#forum** | 更长的讨论和功能请求 |
| **#github-activity** | 开发更新 |

### 提问前

为了获得最佳帮助：

1. **先测试** - 下载并测试插件几天，以了解它是否满足您的需求
2. **查看文档** - 许多问题在此文档中都有解答
3. **搜索 Discord** - 您的问题可能已经得到解答
4. **提供详细信息** - 包括您的 WooCommerce 版本、WCPOS 版本以及重现问题的步骤

## 电子邮件支持

如有许可证相关问题或私人咨询，可以发送电子邮件至：**support@wcpos.com**

:::note
一般支持问题应发送至 Discord，以便社区提供帮助，其他人也能从中受益。
:::

## 日志屏幕

POS 包含一个 [日志屏幕](/support/logs)，记录系统活动。这对于以下方面非常重要：

- 调试同步问题
- 理解错误信息
- 在请求支持时提供详细信息

请求帮助时，请检查日志屏幕中的错误消息并将其包含在您的支持请求中。

## 专业支持

[WCPOS Pro](https://wcpos.com/pro) 客户可以通过 Discord 获得相同的社区支持。Pro 许可证资助继续开发免费和 Pro 功能。

## 贡献

WCPOS 是开源的！您可以通过以下方式贡献：

- 在 [GitHub](https://github.com/wcpos) 上报告错误
- 在 Discord 上建议功能
- 翻译文档
- 与他人分享您的经验
