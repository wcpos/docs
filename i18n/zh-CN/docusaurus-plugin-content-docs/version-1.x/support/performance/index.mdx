性能优化对提供顺畅的 WCPOS 结账体验至关重要。本节涵盖常见的性能问题及其解决方案。

## 常见性能问题

最常见的性能投诉与以下方面有关：

- **结账过程缓慢** - 支付窗口加载或处理支付所需时间较长
- **界面迟钝** - POS 界面反应不灵敏
- **同步时间过长** - 产品和订单同步耗时过长

## 性能类别

### [结账性能](/support/performance/checkout)
优化结账过程，以加快支付处理速度并改善客户体验。

### [服务器性能](/support/performance/server)
使用内置指标和故障排除技术监控和优化您的 WordPress/WooCommerce 服务器。

### 网络性能（敬请期待）
排查与网络相关的性能问题并优化 API 通信。
