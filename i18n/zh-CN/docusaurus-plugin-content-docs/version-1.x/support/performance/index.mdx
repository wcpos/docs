性能优化对于提供顺畅的 WCPOS 结账体验至关重要。本节涵盖常见的性能问题及其解决方案。

## 常见性能问题

最常见的性能投诉与以下内容有关：

- **缓慢的结账过程** - 支付模态窗口加载或处理支付的时间过长
- **迟缓的界面** - POS 界面感觉不响应
- **长时间的同步** - 产品和订单同步耗时过长

## 性能类别

### [结账性能](/support/performance/checkout)
优化结账过程，以便更快的支付处理和更好的客户体验。

### [服务器性能](/support/performance/server)
使用内置指标和故障排除技术监控和优化您的 WordPress/WooCommerce 服务器。

### 网络性能（即将推出）
排查与网络相关的性能问题，并优化 API 通信。
