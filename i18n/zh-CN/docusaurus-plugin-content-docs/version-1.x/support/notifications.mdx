POS 顶部栏中的**通知铃铛**是 WCPOS 向您推送消息的位置——包括产品公告、插件更新通知和 Pro 许可证提醒。铃铛上的红色徽标显示未读通知数量（超过 99 条时显示为 **99+**）。

## 打开通知中心 {#opening-the-notification-centre}

点击 POS 顶部的**铃铛图标**打开通知面板。通知按时间从新到旧排列，每条通知包含：

- **未读圆点**——未读时为实心圆点，标题以粗体显示
- 简短的**标题**和可选的**正文**
- **相对时间戳**（例如 _2 小时前_）

如果没有待处理的内容，面板会显示**"暂无通知"**的空状态。

## 查看和清除通知 {#reading-and-clearing-notifications}

- **标记单条为已读** — 点击列表中任意未读通知。
- **全部标记为已读** — 使用面板顶部的**全部标记为已读**按钮（仅在有未读项目时显示）。
- **清除徽标** — 只需打开面板即可清除铃铛上的未读计数徽标，即使您尚未逐一打开每条通知。

通知存储在您的设备上，因此列表在离线状态下仍然可用。连接期间，新通知会实时到达。

## 通知类型 {#what-kinds-of-notifications-appear}

通知中心用于系统和账户消息，而非店铺订单。您可能会看到：

| 类型 | 示例 |
| --- | --- |
| **欢迎** | 首次连接时显示的一次性欢迎消息 |
| **公告** | 关于 WCPOS 的一般新闻和更新 |
| **插件更新** | WCPOS 插件有可用更新的通知 |
| **许可证即将到期** | 提醒您的 Pro 许可证即将到达续订日期，附带**续订**链接 |
| **许可证已过期** | 持续提醒您的 Pro 许可证已过期，附带**立即续订**链接 |

重要消息（如许可证提醒）除了铃铛通知外，还可能以屏幕弹窗的形式短暂显示。

:::note
通知中心用于传递来自 WCPOS 的消息。**新在线订单**的实时提醒是一项独立的计划功能——详见[路线图](https://github.com/orgs/wcpos/projects/4)。
:::
