POS 标头中的 **通知铃声** 是 WCPOS 向您显示消息的地方 - 产品公告、插件更新通知和 Pro 许可证提醒。铃声上的红色徽章显示您有多少条未读通知（计数超过 99 则显示为 **99+**）。

## 打开通知中心 {#opening-the-notification-centre}

单击 POS 顶部的 **钟形图标** 打开通知面板。它按最新顺序列出您的通知，每个通知包含：

- **未读点** — 未读时填充，标题以粗体显示
- 简短的**标题**和可选的**正文**
- **相对时间戳**（例如_2小时前_）

如果您没有等待，面板将显示 **“无通知”** 空状态。

## 读取和清除通知 {#reading-and-clearing-notifications}

- **将其标记为已读** — 点击列表中的任何未读通知。
- **将全部标记为已读** — 使用面板标题中的 **将全部标记为已读** 按钮（仅在有未读项目时显示）。
- **清除徽章** — 只需打开面板即可清除铃声上的未读计数徽章，即使您尚未打开每个项目。

通知存储在您的设备上，因此该列表仍然可以离线使用。当您连接时，新通知会实时到达。

## 出现什么类型的通知 {#what-kinds-of-notifications-appear}

通知中心用于系统和账户消息，不用于商店订单。您可能会看到：

| 类型 | 示例 |
| --- | --- |
| **欢迎** | 第一次连接时出现的一次性欢迎消息 |
| **公告** | 有关 WCPOS 的一般新闻和更新 |
| **插件更新** | WCPOS 插件更新可用的通知 |
| **许可证即将过期** | 提醒您的 Pro 许可证即将到期，并带有 **续订** 链接 |
| **许可证已过期** | 持续提醒您的 Pro 许可证已过期，并带有 **立即续订** 链接 |

除了铃声之外，重要消息（例如许可证警报）也可能会以屏幕提示的形式短暂出现。

:::note
通知中心传递来自 WCPOS 的消息。 **新在线订单**的实时警报是一项单独的、计划中的功能 - 请参阅[路线图](https://github.com/orgs/wcpos/projects/4)。
:::
