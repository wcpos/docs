# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

为通过 [Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/) 创建的店铺余额添加 WCPOS 兼容性。它允许 POS 订单兑换 StoreApps 店铺余额，保持 StoreApps 余额同步，并为工作人员留下订单备注审计记录。

此扩展专门用于 StoreApps / WooCommerce.com Smart Coupons 插件。其他名称类似“Smart Coupons”的插件属于不同产品，可能需要单独集成。

:::info 专业版功能
从目录安装和管理扩展需要 [WCPOS Pro](/getting-started/pro-license)。
:::

## 功能说明 {#what-it-does}

- 在常规结账购物车上下文不可用时，为通过 POS 创建的订单记录 StoreApps `smart_coupons_contribution` 元数据。
- 让 StoreApps 在 POS 结账后从优惠券余额中扣除正确的部分店铺余额金额。
- 当 POS 订单通过 WooCommerce 的正常订单生命周期取消或退款时，恢复 StoreApps 店铺积分余额。
- 兑换后添加一条私有订单备注，显示优惠券代码、已使用金额和当前余额。
- 通过将剩余店铺积分余额附加到优惠券描述/折扣标签，在现有收据折扣行中显示该余额。

该扩展使用标准 WooCommerce 优惠券字段和元数据。它不会添加自定义 WCPOS 优惠券响应字段，也不需要自定义收据模板。

## 安装 {#installation}

<Steps>
  <StepCard number="1" title="安装 StoreApps Smart Coupons">

安装并启用 [Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/)。像往常一样在 WooCommerce 中创建店铺积分优惠券或礼品卡。

  </StepCard>
  <StepCard number="2" title="安装 WCPOS StoreApps Smart Coupons">

从 `WP Admin > POS > 设置 > 扩展` 安装，或从 [GitHub 发布页面](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases)下载最新版本，并通过 `插件 > 新增 > 上传插件` 上传。

  </StepCard>
  <StepCard number="3" title="将优惠券同步到 POS" isLast>

打开 POS 并同步优惠券。收银员随后可以通过常规的 **添加优惠券** 流程应用 StoreApps 店铺积分优惠券。

  </StepCard>
</Steps>

## 收据输出 {#receipt-output}

WCPOS 收据模板已使用折扣行标签打印优惠券折扣。在渲染收据期间，此扩展会将当前 StoreApps 余额附加到该现有标签。

例如，优惠券描述 **Gift card** 可能会打印为：

```text
Gift card — Store credit balance: £61.50
```

如果优惠券没有描述，则余额文本会用作折扣标签。无需更改现有收据模板。

## 订单审计跟踪 {#order-audit-trail}

StoreApps 处理 POS 店铺储值余额兑换后，此扩展会添加一条类似如下的私有订单备注：

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

使用此备注可确认使用了哪张店铺储值优惠券、兑换了多少金额，以及销售完成后剩余多少余额。

## 兼容性说明 {#compatibility-notes}

- **仅限 StoreApps：**此扩展面向 StoreApps / WooCommerce.com Smart Coupons。它不面向 WebToffee 或其他智能优惠券插件。
- **优惠券数据：**WCPOS 会继续使用常规 WooCommerce 优惠券响应。StoreApps 特定状态仍保留在 WooCommerce 元数据中，例如 `smart_coupons_contribution` 和优惠券余额字段。
- **收据：**余额文本通过现有的优惠券描述/折扣标签路径添加，而不是通过更改收据数据或模板添加。
- **未启用 StoreApps 插件：**如果 StoreApps Smart Coupons 未启用，此扩展不会执行任何操作。
