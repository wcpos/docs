# API01008：网站不可用

## 这意味着什么

POS可以连接互联网，但您的特定网站无法响应。托管您的WooCommerce商店的服务器似乎已停机或无法访问。

## 常见原因

- **服务器维护** — 您的主机提供商可能正在执行更新
- **服务器崩溃** — Web服务器意外停止
- **资源限制超出** — 您的托管账户已达到限制
- **DDoS攻击** — 服务器被恶意流量压垮
- **域名/主机过期** — 您的主机或域名已到期

## 如何解决

### 1. 验证网站是否宕机

直接在浏览器中尝试访问您的网站。如果无法加载，则问题出在服务器端。

### 2. 检查主机提供商状态

- 访问您的主机提供商的状态页面
- 检查是否有计划的维护
- 查看是否有报告的故障

### 3. 检查托管账户

登录您的主机控制面板：
- 检查您的账户是否活跃
- 验证是否未超出资源限制
- 查看是否有任何暂停通知

### 4. 联系主机支持

如果您那边一切看起来正常：
- 联系您的主机提供商
- 检查服务器状态
- 审查服务器日志
- 在需要时重启服务

### 5. 等待并重试

如果是临时故障：
- 等待15-30分钟
- POS将自动重试连接
- 缓存数据仍可离线使用
