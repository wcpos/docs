# API01008: 网站不可用

## 这意味着什么

POS 可以连接到互联网，但您的特定网站没有响应。托管您 WooCommerce 商店的服务器似乎已经宕机或无法访问。

## 常见原因

- **服务器维护** — 您的托管提供商可能正在进行更新
- **服务器崩溃** — 网络服务器意外停止
- **资源限制超出** — 您的托管帐户达到了限制
- **DDoS 攻击** — 服务器受到恶意流量的影响
- **域名/托管过期** — 您的托管或域名已过期

## 如何修复

### 1. 验证网站是否宕机

尝试在浏览器中直接访问您的网站。如果页面无法加载，问题出在服务器端。

### 2. 检查托管提供商状态

- 访问您托管提供商的状态页面
- 检查是否有计划维护
- 查看是否有报告的中断

### 3. 审查托管帐户

登录到您的托管控制面板：
- 检查您的帐户是否处于活动状态
- 确保您没有超出资源限制
- 查找任何暂停通知

### 4. 联系托管支持

如果您那边一切正常，请联系您的托管提供商。他们可以：
- 检查服务器状态
- 审查服务器日志
- 如果需要，重新启动服务

### 5. 等待并重试

如果是临时中断：
- 等待 15-30 分钟
- POS 将自动重试连接
- 缓存数据将保持离线可用
