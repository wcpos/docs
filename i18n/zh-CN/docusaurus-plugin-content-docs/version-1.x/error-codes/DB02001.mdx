# DB02001：重复记录

## 含义

数据库尝试插入一个已存在ID的记录。每条记录必须有唯一的标识符。

## 常见原因

- **同步冲突** — 同一数据被重复同步
- **竞争条件** — 多个操作创建同一记录
- **数据损坏** — ID被损坏或重复
- **导入问题** — 导入已存在的数据

## 解决方法

### 1. 刷新数据

从服务器重新同步：
- 获取最新数据
- 让同步过程解决冲突
- 较新的数据应该优先

### 2. 重试操作

创建新记录时：
- 系统可能已经创建了记录
- 检查记录是否存在
- 避免多次点击提交

### 3. 清除本地缓存

重置本地数据：
1. 清除本地数据库/缓存
2. 重新登录
3. 重新同步数据

### 4. 检查WooCommerce中的重复项

如果问题仍然存在：
- 检查WooCommerce中是否有重复条目
- 查找具有相同ID的产品/订单
- 清理管理后台中的重复项

### 5. 报告持续性问题

如果频繁发生：
- 记录导致问题的操作
- 寻找模式
- 向支持团队报告详细信息

## 为什么这很重要

重复ID将导致：
- 对哪条记录正确产生混淆
- 数据覆盖
- 同步问题

此错误可防止数据损坏。
