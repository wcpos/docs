# 数据库错误

数据库错误发生在存储或检索本地数据库中的数据时。WCPOS 使用本地数据库存储产品、客户和其他数据，以便快速访问和离线功能。这些错误以 `DB` 为前缀。

## 分类

| 分类 | 代码范围 | 描述 |
|------|----------|------|
| [连接](#connection-errors) | DB01xxx | 数据库连接和事务问题 |
| [数据](#data-errors) | DB02xxx | 记录和约束问题 |
| [查询](#query-errors) | DB03xxx | 查询语法和数据类型问题 |

---

## 连接错误

连接到本地数据库或对其操作时出现的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [DB01001](./DB01001) | 连接失败 | 无法连接到本地数据库 |
| [DB01002](./DB01002) | 查询超时 | 数据库查询耗时过长 |
| [DB01003](./DB01003) | 事务失败 | 数据库事务无法完成 |

## 数据错误

与记录、重复项和约束相关的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [DB02001](./DB02001) | 重复记录 | 已存在此 ID 的记录 |
| [DB02002](./DB02002) | 找不到记录 | 请求的记录不存在 |
| [DB02003](./DB02003) | 约束违规 | 数据违反数据库约束 |

## 查询错误

与查询语法和数据类型相关的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [DB03001](./DB03001) | 查询语法错误 | 数据库查询存在语法错误 |
| [DB03002](./DB03002) | 无效数据类型 | 数据类型与预期类型不匹配 |
| [DB03003](./DB03003) | 缺少必填字段 | 必填字段为空 |
