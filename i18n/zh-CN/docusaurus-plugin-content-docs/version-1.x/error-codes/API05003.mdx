# API05003: WCPOS 插件过时 {#api05003-wcpos-plugin-outdated}

## 这意味着什么 {#what-this-means}

安装在您服务器上的 WCPOS 插件已经过时，并且与您的 POS 应用程序版本不兼容。您需要更新插件。

## 常见原因 {#common-causes}

- **插件未更新** — 自动更新可能已禁用
- **更新失败** — 之前的更新尝试失败
- **版本不匹配** — POS 应用程序已更新但插件未更新
- **暂存/开发站点** — 在旧版本上进行测试

## 如何修复 {#how-to-fix}

### 1. 更新插件 {#1-update-the-plugin}

在 WordPress Admin：
1. 转到插件 → 已安装插件
2. 找到“WooCommerce POS”
3. 如果有可用更新，点击“立即更新”

### 2. 启用自动更新 {#2-enable-auto-updates}

为了防止未来的不匹配：
1. 转到插件 → 已安装插件
2. 找到“WooCommerce POS”
3. 点击“启用自动更新”

### 3. 手动更新 {#3-manual-update}

如果自动更新失败：
1. 从 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) 下载最新版本
2. 禁用当前插件
3. 删除旧插件文件
4. 上传并激活新版本

### 4. 检查更新错误 {#4-check-for-update-errors}

如果更新失败：
- 检查 wp-content/plugins 的文件权限
- 确保您有磁盘空间
- 查找 PHP 内存问题
- 检查错误日志以获取详细信息

### 5. 降级 POS 应用（临时） {#5-downgrade-pos-app-temporary}

如果您无法立即更新插件：
- 使用旧版本的 POS 应用程序
- 这只是一个临时解决方案
- 计划尽快更新插件

## 版本兼容性 {#version-compatibility}

始终保持两个组件更新：
- POS 应用程序
- WCPOS WordPress 插件

在更新时检查发布说明以查找任何重大更改。
