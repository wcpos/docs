# API05003：WCPOS插件已过时

## 这意味着什么

安装在您服务器上的WCPOS插件已过时，与您的POS应用程序版本不兼容。您需要更新插件。

## 常见原因

- **插件未更新** — 可能禁用了自动更新
- **更新失败** — 之前的更新尝试失败
- **版本不匹配** — POS应用已更新但插件未更新
- **测试/开发站点** — 正在使用旧版本进行测试

## 如何修复

### 1. 更新插件

在WordPress管理后台：
1. 转到"插件" → "已安装插件"
2. 找到"WooCommerce POS"
3. 如果有可用更新，点击"立即更新"

### 2. 启用自动更新

为防止未来出现不匹配：
1. 转到"插件" → "已安装插件"
2. 找到"WooCommerce POS"
3. 点击"启用自动更新"

### 3. 手动更新

如果自动更新失败：
1. 从[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)下载最新版本
2. 停用当前插件
3. 删除旧的插件文件
4. 上传并激活新版本

### 4. 检查更新错误

如果更新失败：
- 检查wp-content/plugins的文件权限
- 验证是否有足够的磁盘空间
- 查看PHP内存问题
- 查看错误日志以获取详细信息

### 5. 暂时降级POS应用

如果您暂时无法更新插件：
- 使用较旧版本的POS应用
- 这只是临时解决方案
- 尽快计划更新插件

## 版本兼容性

始终保持两个组件都是最新的：
- POS应用程序
- WCPOS WordPress插件

更新时查看发行说明，了解任何重大变更。
