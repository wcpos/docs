# API02002：会话令牌已过期

## 这意味着什么

您的身份验证会话已过期。出于安全原因，登录会话不会永久有效。POS需要刷新您的身份验证，或者您可能需要重新登录。

## 常见原因

- **长时间不活动** — 您长时间未使用POS
- **服务器令牌生命周期** — 服务器的令牌过期时间设置过短
- **服务器时间不一致** — 服务器和设备时钟不同步

## 服务器错误映射

当服务器返回以下内容时，触发此错误代码：

| 服务器代码 | 来源 |
|-------------|--------|
| `jwt_auth_expired_token` | JWT身份验证插件 |

## 如何解决

### 1. 重新登录

最简单的解决方案是注销并重新登录：
1. 从POS注销
2. 重新输入您的凭据
3. 这将创建一个新的会话

### 2. 检查令牌刷新

POS应自动刷新令牌。如果无法正常工作：
- 检查您的互联网连接
- 验证服务器是否可访问
- 如果刷新失败，请查看 [API02007](./API02007)

### 3. 检查服务器时间

令牌过期取决于准确的时间：
- 确保服务器时钟正确
- 检查设备时间是否准确
- 在两者上使用自动时间同步

### 4. 检查令牌设置

如果令牌过期太快，您可能需要调整服务器设置（请咨询您的主机提供商或WordPress管理员）。
