# SY02002：服务不可用

## 这是什么意思

所需的服务或组件不可用。POS 依赖于多个服务才能正常运行，其中一个服务目前无法访问。

## 常见原因

- **后台服务停止** — 所需服务未运行
- **系统资源问题** — 启动服务的资源不足
- **依赖项失败** — 依赖的服务出现故障
- **启动问题** — 服务未能正确启动
- **服务器错误** — WooCommerce 服务器返回 5xx 错误

## 服务器错误映射

当服务器返回以下内容时触发此错误代码：

| 服务器代码 | 来源 |
|-------------|--------|
| HTTP 5xx | 任何服务器错误响应（500, 502, 503, 504 等） |

## 如何修复

### 1. 重启应用程序

关闭并重新打开 POS：
1. 完全关闭应用程序
2. 等待几秒钟
3. 重新启动

### 2. 重启您的设备

完全重启通常可以解决服务问题：
1. 保存任何重要工作
2. 重启您的电脑
3. 重新尝试 POS

### 3. 检查系统资源

确保系统资源可用：
- 关闭不必要的应用程序
- 检查内存使用情况
- 检查磁盘空间

### 4. 更新应用程序

确保使用最新版本：
- 检查可用更新
- 安装待处理的更新
- 更新后重启

### 5. 必要时重新安装

如果服务仍然不可用：
1. 卸载应用程序
2. 重启您的设备
3. 下载最新版本
4. 重新安装

## 浏览器特定（网页版）

对于网页版：
- 确保 JavaScript 已启用
- 检查必需的浏览器功能是否被阻止
- 尝试使用其他浏览器
- 禁用可能干扰的浏览器扩展

## 何时联系支持

如果此错误持续存在：
- 记录发生时间
- 检查是否有特定操作触发它
- 提供详细信息以便调查
