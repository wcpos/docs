# SY02002: 服务不可用

## 这意味着什么

所需的服务或组件不可用。POS系统依赖于各种服务才能正常运行，而其中一个服务当前无法访问。

## 常见原因

- **后台服务停止** — 所需的服务未在运行
- **系统资源问题** — 启动服务的资源不足
- **依赖项失败** — 依赖的服务出现故障
- **启动问题** — 服务未能正确启动
- **服务器错误** — WooCommerce服务器返回了5xx错误

## 服务器错误映射

当服务器返回以下情况时触发此错误代码：

| 服务器代码 | 来源 |
|-------------|--------|
| HTTP 5xx | 任何服务器错误响应（500, 502, 503, 504等） |

## 如何修复

### 1. 重启应用程序

关闭并重新打开POS：
1. 完全关闭应用程序
2. 等待几秒钟
3. 再次启动它

### 2. 重新启动设备

完全重启通常可以解决服务问题：
1. 保存任何重要的工作
2. 重启你的计算机
3. 再次尝试使用POS

### 3. 检查系统资源

确保系统资源可用：
- 关闭不必要的应用程序
- 检查内存使用情况
- 检查磁盘空间

### 4. 更新应用程序

确保你在使用最新版本：
- 检查可用更新
- 安装任何待处理的更新
- 更新后重启

### 5. 如有必要，重新安装

如果服务仍然不可用：
1. 卸载应用程序
2. 重启设备
3. 下载最新版本
4. 重新安装

## 浏览器特定（网页版本）

对于网页版本：
- 确保启用JavaScript
- 检查所需的浏览器功能是否被阻止
- 尝试不同的浏览器
- 禁用可能干扰的浏览器扩展

## 何时联系支持

如果此错误仍然存在：
- 记下发生的时间
- 检查是否有特定的操作触发此错误
- 提供详细信息以便进一步调查
