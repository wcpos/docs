# SY02002: 服务不可用 {#sy02002-service-unavailable}

## 这意味着什么 {#what-this-means}

所需的服务或组件不可用。POS依赖于各种服务来运行，而其中一个服务当前无法访问。

## 常见原因 {#common-causes}

- **后台服务停止** — 所需的服务未在运行
- **系统资源问题** — 启动服务的资源不足
- **依赖失败** — 依赖的服务未能成功运行
- **启动问题** — 服务未能正确启动
- **服务器错误** — WooCommerce服务器返回了5xx错误

## 服务器错误映射 {#server-error-mapping}

此错误代码在服务器返回以下情况时被触发：

| 服务器代码 | 来源 |
|-------------|--------|
| HTTP 5xx | 任何服务器错误响应（500, 502, 503, 504等） |

## 如何修复 {#how-to-fix}

### 1. 重启应用程序 {#1-restart-the-application}

关闭并重新打开POS：
1. 完全关闭应用程序
2. 等待几秒钟
3. 再次启动它

### 2. 重新启动您的设备 {#2-restart-your-device}

完全重启通常可解决服务问题：
1. 保存任何重要工作
2. 重启您的计算机
3. 再次尝试POS

### 3. 检查系统资源 {#3-check-system-resources}

确保系统资源可用：
- 关闭不必要的应用程序
- 检查内存使用情况
- 检查磁盘空间

### 4. 更新应用程序 {#4-update-the-application}

确保您使用的是最新版本：
- 检查可用更新
- 安装所有待处理的更新
- 更新后重启

### 5. 如有必要，重新安装 {#5-reinstall-if-necessary}

如果服务仍然不可用：
1. 卸载应用程序
2. 重启您的设备
3. 下载最新版本
4. 重新安装

## 浏览器特定（网页版本） {#browser-specific-web-version}

对于网页版本：
- 确保启用JavaScript
- 检查所需的浏览器功能是否未被阻止
- 尝试不同的浏览器
- 禁用可能干扰的浏览器扩展

## 何时联系支持 {#when-to-contact-support}

如果此错误仍然存在：
- 记下发生的时间
- 检查是否有特定操作触发它
- 提供详细信息以供调查
