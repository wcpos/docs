# SY01001：内存不足

## 这意味着什么

您的设备内存（RAM）不足，无法完成请求的操作。POS 需要足够的内存才能正常运行。

## 常见原因

- **打开的应用过多** — 其他应用正在使用内存
- **大型数据集** — 加载了大量产品/客户
- **内存泄漏** — 应用未正确释放内存
- **RAM不足** — 设备没有足够的内存

## 如何解决

### 1. 关闭其他应用程序

释放内存：
- 关闭未使用的浏览器标签
- 关闭其他应用程序
- 关闭后台程序

### 2. 重启POS

刷新应用程序内存：
1. 完全关闭POS
2. 等待几秒钟
3. 重新打开应用程序

### 3. 重启您的设备

完全重启会清除所有内存：
1. 保存所有未完成的工作
2. 重启您的电脑/设备
3. 重新启动时仅打开POS

### 4. 清除浏览器数据（Web版本）

如果在浏览器中使用：
- 清除浏览器缓存
- 关闭并重新打开浏览器
- 减少浏览器标签数量

### 5. 减少数据加载

如果您有非常大的目录：
- 使用过滤器一次加载较少的数据
- 考虑分页显示结果
- 尽可能限制同步范围

## 最低要求

确保您的设备满足最低规格：
- **RAM：** 最低4GB，推荐8GB
- **浏览器：** 现代浏览器（Chrome、Firefox、Edge）
- **桌面应用：** 查看发行说明的要求

## 持续性问题

如果您经常遇到内存不足：
- 考虑升级设备的RAM
- 使用资源更多的设备
- 如果情况异常，请报告问题
