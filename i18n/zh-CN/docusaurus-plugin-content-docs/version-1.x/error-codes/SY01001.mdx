# SY01001: 内存不足

## 这意味着什么

您的设备内存（RAM）不足，无法完成请求的操作。POS需要足够的内存才能正常工作。

## 常见原因

- **打开了太多应用程序** — 其他应用程序占用内存
- **数据集过大** — 加载了许多产品/客户
- **内存泄漏** — 应用程序未正确释放内存
- **RAM不足** — 设备内存不够

## 如何修复

### 1. 关闭其他应用程序

释放内存：
- 关闭未使用的浏览器标签
- 关闭其他应用程序
- 关闭后台程序

### 2. 重启POS

刷新应用程序内存：
1. 完全关闭POS
2. 等待几秒钟
3. 重新打开应用程序

### 3. 重启设备

完全重启可以清除所有内存：
1. 保存任何打开的工作
2. 重启您的计算机/设备
3. 重新启动时仅打开POS

### 4. 清除浏览器数据（网页版本）

如果在浏览器中使用：
- 清除浏览器缓存
- 关闭并重新打开浏览器
- 使用更少的浏览器标签

### 5. 减少数据负载

如果您的目录非常大：
- 使用过滤器一次加载更少的数据
- 考虑对结果进行分页
- 如果可能，限制同步范围

## 最低要求

确保您的设备满足最低规格：
- **RAM：** 最少4GB，推荐8GB
- **浏览器：** 现代浏览器（Chrome, Firefox, Edge）
- **桌面应用程序：** 检查发行说明以获取要求

## 持续问题

如果您经常内存不足：
- 考虑升级设备的RAM
- 使用资源更多的设备
- 如果问题似乎过于频繁，请报告该问题
