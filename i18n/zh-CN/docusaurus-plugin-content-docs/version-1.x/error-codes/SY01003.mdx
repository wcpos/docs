# SY01003：权限被拒绝

## 含义

POS应用程序无权访问所需的资源。这可能是文件、文件夹或系统功能。

## 常见原因

- **文件权限** — 无法读取或写入所需文件
- **应用权限** — 应用程序缺少所需权限
- **受保护文件夹** — 尝试访问受限位置
- **安全软件** — 防病毒软件阻止访问

## 解决方法

### 1. 以管理员身份运行（Windows）

如果需要初始设置，右键点击应用程序并选择"以管理员身份运行"。

### 2. 检查应用程序权限

**macOS：**
- 系统偏好设置 → 安全性与隐私 → 隐私
- 检查应用程序是否具有必要的权限

**Windows：**
- 设置 → 隐私
- 查看应用程序权限

### 3. 检查文件夹权限

确保应用程序可以访问其数据文件夹：
- 用户应拥有应用程序数据目录
- 启用写入权限
- 文件夹上没有只读标记

### 4. 检查防病毒设置

安全软件可能会阻止访问：
- 检查防病毒日志中的阻止记录
- 将POS应用程序添加到允许列表
- 对应用程序数据文件夹进行白名单设置

### 5. 重新安装应用程序

如果权限已损坏：
1. 卸载POS应用程序
2. 重启设备
3. 重新安装到标准位置
4. 以普通（非管理员）权限运行

## 浏览器权限（Web版本）

如果使用Web版本：
- 允许站点本地存储
- 启用站点Cookie
- 不要使用过于严格的浏览器设置
