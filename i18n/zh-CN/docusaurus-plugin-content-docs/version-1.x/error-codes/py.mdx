# 支付错误

支付错误发生在结账和支付处理期间。这些错误以 `PY` 开头，与支付卡、网关和交易相关的问题有关。

## 类别

| 类别 | 代码范围 | 描述 |
|----------|------------|-------------|
| [卡片](#卡片错误) | PY01xxx | 与支付卡相关的问题 |
| [网关](#网关错误) | PY02xxx | 支付网关通信问题 |

---

## 卡片错误

与客户支付卡相关的问题。

| 代码 | 名称 | 描述 |
|------|------|-------------|
| [PY01001](./PY01001) | 支付被拒绝 | 支付被拒绝 |
| [PY01002](./PY01002) | 资金不足 | 支付资金不足 |
| [PY01003](./PY01003) | 卡片已过期 | 支付卡片已过期 |
| [PY01004](./PY01004) | 无效卡号 | 卡号无效 |

与支付网关通信的问题。
