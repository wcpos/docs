# SY02001: 无效的配置

## 这意味着什么

系统配置无效或已损坏。POS 应用程序发现配置设置不合逻辑或不完整。

## 常见原因

- **配置损坏** — 配置文件已损坏
- **设置不完整** — 设置过程未完成
- **手动编辑错误** — 配置文件被错误修改
- **版本升级问题** — 旧配置与新版本不兼容

## 如何修复

### 1. 重置为默认设置

如果应用程序中可用：
- 查找“重置设置”或“恢复默认值”
- 这将清除自定义设置，但修复损坏

### 2. 完成设置

如果设置未完成：
1. 再次启动设置过程
2. 完成所有必需步骤
3. 在完成之前验证配置

### 3. 清除应用程序数据

删除损坏的配置：
- **桌面：** 删除应用程序数据文件夹，重启应用
- **浏览器：** 清除网站数据，重新加载

### 4. 重新安装应用程序

对于持续出现的问题：
1. 完全卸载
2. 删除任何剩余的数据文件夹
3. 重新安装最新版本
4. 从头开始完成设置

### 5. 检查更新

配置格式可能会变化：
- 更新 POS 应用程序
- 更新 WCPOS 插件
- 更新后重新配置

## 重置时会发生什么

重置配置时：
- 您需要重新登录
- 自定义设置将丢失
- 数据需要重新同步
- 但您的 WooCommerce 数据在服务器上是安全的
