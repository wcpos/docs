# API02009：刷新令牌已过期

## 这意味着什么

您的刷新令牌已超过其有效期，不能再用于获取新的访问令牌。与快速过期的访问令牌不同，刷新令牌通常可以持续几天或几周，但最终也会过期。

## 常见原因

- **长时间不活跃** — 您长时间未使用POS系统
- **令牌有效期较短** — 服务器配置了较短的刷新令牌过期时间
- **时钟偏差** — 设备或服务器时间严重不同步

## 如何解决

### 1. 重新登录

当刷新令牌过期时，您必须重新进行身份验证：
1. 登出POS系统
2. 输入您的凭据
3. 系统将颁发新的刷新令牌

### 2. 定期使用POS

为避免此问题：
- 定期使用POS系统以保持令牌有效
- 在活跃状态下，令牌会自动刷新
- 长时间不活跃将需要重新登录

### 3. 检查时间设置

确保所有系统的时间准确：
- 在设备上启用自动时间同步
- 验证服务器时间是否正确
- 时间差异可能导致令牌过早过期

### 4. 调整令牌有效期（高级）

如果令牌过快过期不能满足需求，服务器管理员可以在身份验证设置中调整刷新令牌的有效期。
