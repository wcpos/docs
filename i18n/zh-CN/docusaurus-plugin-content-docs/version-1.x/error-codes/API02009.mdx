# API02009: 刷新令牌已过期

## 这意味着什么

您的刷新令牌已超出其有效期，无法再用于获取新的访问令牌。与快速过期的访问令牌不同，刷新令牌通常可以使用几天或几周，但最终还是会过期。

## 常见原因

- **长时间未使用** — 您很长时间没有使用POS
- **短令牌有效期** — 服务器配置了短的刷新令牌过期时间
- **时钟偏差** — 设备或服务器时间明显不准确

## 如何修复

### 1. 再次登录

当刷新令牌过期时，您必须再次进行身份验证：
1. 从POS注销
2. 输入您的凭据
3. 将发放新的刷新令牌

### 2. 定期使用POS

为了避免将来的问题：
- 定期使用POS以保持令牌的有效
- 当您活跃时，令牌会自动刷新
- 长时间未使用将需要重新登录

### 3. 检查时间设置

确保所有系统准确时间：
- 在您的设备上启用自动时间同步
- 验证服务器时间是否正确
- 时间差异可能导致过早过期

### 4. 调整令牌有效期（高级）

如果令牌过期太快以满足您的需求，服务器管理员可以在身份验证设置中调整刷新令牌的有效期。
