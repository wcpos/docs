# API02009: 刷新令牌过期 {#api02009-refresh-token-expired}

## 这意味着什么 {#what-this-means}

您的刷新令牌已超过其生命周期，无法再用于获取新的访问令牌。与访问令牌（其过期速度较快）不同，刷新令牌通常会持续数天或数周，但最终会过期。

## 常见原因 {#common-causes}

- **长时间未使用** — 您很长时间没有使用POS
- **短令牌生命周期** — 服务器配置了短的刷新令牌过期时间
- **时钟偏移** — 设备或服务器时间严重不准确

## 如何修复 {#how-to-fix}

### 1. 再次登录 {#1-log-in-again}

当刷新令牌过期时，您必须再次进行身份验证：
1. 从POS注销
2. 输入您的凭据
3. 将发放新的刷新令牌

### 2. 定期使用POS {#2-use-pos-regularly}

为了避免将来的问题：
- 定期使用POS以保持令牌的新鲜度
- 当您活跃时，令牌会自动刷新
- 长时间未使用将需要重新登录

### 3. 检查时间设置 {#3-check-time-settings}

确保所有系统时间准确：
- 在您的设备上启用自动时间同步
- 验证服务器时间是否正确
- 时间差异可能导致提前过期

### 4. 调整令牌生命周期（高级） {#4-adjust-token-lifetime-advanced}

如果令牌过期速度过快，不符合您的需求，服务器管理员可以在身份验证设置中调整刷新令牌的生命周期。
