# PY01003：卡片已过期

## 这意味着什么

支付卡已超过有效期，不能再用于交易。

## 常见原因

- **卡片过期** — 卡片的有效期已经过去
- **输入错误的过期日期** — 输入的过期日期不正确
- **新卡未激活** — 收到更换的卡片但尚未激活

## 如何解决

### 1. 检查过期日期

查看卡片：
- 找到过期日期（MM/YY格式）
- 与当前日期比较
- 确认输入的日期与卡片上的日期一致

### 2. 使用其他卡片

如果卡片确实已过期：
- 使用其他有效卡片
- 使用现金或其他支付方式
- 询问客户是否有更换的卡片

### 3. 获取更新的卡片

持卡人应该：
- 查看邮件是否收到更换的卡片
- 如果已收到，激活新卡
- 如果未收到，联系银行

### 4. 验证输入

仔细检查过期日期的输入：
- MM/YY格式
- 正确的月份和年份
- 没有输入错误

## 客户服务提示

当遇到卡片过期时：
- 礼貌地通知客户
- 建议检查是否收到更换的卡片
- 提供其他支付选项
- 保持理解 — 这种情况每个人都会遇到
