# PY01003: 卡片过期

## 这意味着什么

支付卡已过期，无法再用于交易。

## 常见原因

- **卡片过期** — 卡片的有效期已过
- **输入的到期日期不正确** — 输入的到期日期错误
- **新卡未激活** — 收到的替换卡尚未激活

## 如何解决

### 1. 检查到期日期

查看卡片：
- 找到到期日期（MM/YY 格式）
- 与当前日期进行比较
- 验证输入的日期与卡片一致

### 2. 使用其他卡

如果卡片确实过期：
- 使用其他有效的卡片
- 使用现金或替代支付方式
- 询问客户是否有替换卡

### 3. 获取更新的卡

持卡人应：
- 检查邮件是否收到替换卡
- 如果已收到，激活新卡
- 如果未收到替换卡，联系他们的银行

### 4. 验证输入

仔细检查到期日期的输入：
- MM/YY 格式
- 正确的月份和年份
- 无错别字

## 客户服务提示

当卡片过期时：
- 礼貌地告知客户
- 建议他们查看是否有替换卡
- 提供替代支付选项
- 表示理解——每个人都会遇到这种情况
