# API01007: 设备离线 {#api01007-device-offline}

## 这意味着什么 {#what-this-means}

您的设备没有连接到互联网。POS 检测到没有活动的网络连接。

## 常见原因 {#common-causes}

- **WiFi 断开** — 您的设备失去了 WiFi 连接
- **以太网未插入** — 网络电缆已断开
- **启用了飞行模式** — 所有无线连接已禁用
- **网络适配器禁用** — 网络接口已关闭

## 如何修复 {#how-to-fix}

### 1. 检查 WiFi 连接 {#1-check-wifi-connection}

- 确保您的设备上已启用 WiFi
- 确保您连接到正确的网络
- 检查 WiFi 信号强度

### 2. 检查物理连接 {#2-check-physical-connections}

如果使用以太网：
- 确认电缆已牢固插入
- 尝试不同的电缆或端口
- 检查网络交换机/路由器指示灯

### 3. 禁用飞行模式 {#3-disable-airplane-mode}

检查飞行模式是否意外启用并将其禁用。

### 4. 重启网络适配器 {#4-restart-network-adapter}

**Windows:**
- 打开网络连接
- 右键单击您的适配器 → 禁用，然后启用

**macOS:**
- 从菜单栏关闭再开启 WiFi
- 或：系统偏好设置 → 网络 → 关闭/开启

### 5. 重启您的设备 {#5-restart-your-device}

有时，简单的重启可以解决网络检测问题。

## 离线模式 {#offline-mode}

在离线状态下，WCPOS 仍然可以：
- 浏览和搜索本地缓存的产品
- 浏览和搜索本地缓存的客户
- 开始新订单并将商品添加到购物车
- 查看以前同步的数据

**需要连接：**
- 完成/结账订单
- 创建新客户

当连接恢复时，POS 将自动恢复正常操作。
