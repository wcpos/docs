# API01007：设备离线

## 含义

您的设备未连接到互联网。POS检测到没有活动的网络连接。

## 常见原因

- **WiFi断开** — 您的设备失去了WiFi连接
- **以太网未插入** — 网络电缆已断开
- **飞行模式已启用** — 所有无线连接被禁用
- **网络适配器已禁用** — 网络接口已关闭

## 解决方法

### 1. 检查WiFi连接

- 确认设备上的WiFi已启用
- 确保连接到正确的网络
- 检查WiFi信号强度

### 2. 检查物理连接

如果使用以太网：
- 确认电缆已牢固插入
- 尝试使用不同的电缆或端口
- 检查网络交换机/路由器指示灯

### 3. 禁用飞行模式

检查飞行模式是否意外启用并将其禁用。

### 4. 重启网络适配器

**Windows：**
- 打开网络连接
- 右键单击您的适配器 → 禁用，然后启用

**macOS：**
- 从菜单栏关闭并打开WiFi
- 或：系统偏好设置 → 网络 → 关闭/打开

### 5. 重启您的设备

有时简单的重启可以解决网络检测问题。

## 离线模式

离线时，WCPOS仍可以：
- 浏览和搜索本地缓存的产品
- 浏览和搜索本地缓存的客户
- 开始新订单并向购物车添加商品
- 查看之前同步的数据

**需要连接：**
- 完成/结账订单
- 创建新客户

当连接恢复时，POS将自动恢复正常操作。
