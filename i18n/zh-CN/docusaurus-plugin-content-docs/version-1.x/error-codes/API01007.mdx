# API01007: 设备离线

## 这意味着什么

您的设备未连接到互联网。POS 检测到没有活动的网络连接。

## 常见原因

- **WiFi 断开** — 您的设备失去了 WiFi 连接
- **以太网未插入** — 网络电缆已断开
- **启用飞行模式** — 所有无线连接已禁用
- **网络适配器禁用** — 网络接口已关闭

## 如何修复

### 1. 检查 WiFi 连接

- 确保您的设备上已启用 WiFi
- 确保您连接到正确的网络
- 检查 WiFi 信号强度

### 2. 检查物理连接

如果使用以太网：
- 确认电缆已 securely 插入
- 尝试更换电缆或端口
- 检查网络交换机/路由器指示灯

### 3. 禁用飞行模式

检查飞行模式是否意外启用并将其禁用。

### 4. 重启网络适配器

**Windows:**
- 打开网络连接
- 右键单击您的适配器 → 禁用，然后启用

**macOS:**
- 从菜单栏关闭再开启 WiFi
- 或：系统偏好设置 → 网络 → 关闭/开启

### 5. 重启您的设备

有时，简单的重启可以解决网络检测问题。

## 离线模式

在离线状态下，WCPOS 仍然可以：
- 浏览和搜索本地缓存的产品
- 浏览和搜索本地缓存的客户
- 开始新订单并将商品添加到购物车
- 查看以前同步的数据

**需要连接：**
- 完成/结算订单
- 创建新客户

当连接恢复时，POS 将自动恢复正常操作。
