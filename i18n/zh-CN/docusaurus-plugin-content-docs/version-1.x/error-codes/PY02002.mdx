# PY02002: 支付超时

## 这意味着什么

支付处理花费的时间过长，出现了超时。支付网关未在预期时间内响应。

## 常见原因

- **网关过载** — 支付处理器正在经历高流量
- **网络延迟** — 与支付服务的连接缓慢
- **处理延迟** — 复杂的欺诈检查或验证
- **服务器问题** — 支付网关存在性能问题

## 如何修复

### 1. 检查支付是否成功

**重要：** 在重试之前，请验证支付状态：
- 检查您的支付网关仪表板
- 在WooCommerce中查找交易
- 如果实际上已成功，请勿再次处理

### 2. 小心重试

如果支付确实未成功：
- 等待30-60秒
- 再试一次
- 留意重复收费

### 3. 检查您的连接

确保连接稳定：
- 测试您的互联网连接
- 如果使用WiFi，尝试使用有线连接
- 检查网络问题

### 4. 在不太繁忙的时间尝试

如果网关过载：
- 零售高峰时段可能导致延迟
- 稍后再试
- 使用其他支付方式

### 5. 联系支付提供商

如果超时问题仍然存在：
- 检查他们的状态页面
- 联系他们的支持
- 报告持续的问题

## 避免重复收费

发生超时时：
1. **不要立即重试** — 先检查状态
2. **审核交易** — 检查网关和WooCommerce
3. **退款重复费用** — 对任何意外重复的费用处理退款
4. **通知客户** — 如果延迟，解释情况

## 替代措施

如果支付持续超时：
- 接受现金支付
- 如果允许，记录为“稍后支付”
- 使用不同的支付方式
