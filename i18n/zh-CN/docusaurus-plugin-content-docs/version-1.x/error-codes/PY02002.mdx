# PY02002：支付超时

## 含义

支付处理时间过长并超时。支付网关未在预期时间内响应。

## 常见原因

- **网关负载** — 支付处理器正在经历高流量
- **网络延迟** — 连接支付服务较慢
- **处理延迟** — 复杂的欺诈检查或验证
- **服务器问题** — 支付网关性能问题

## 解决方法

### 1. 确认支付状态

**重要：** 重试前，请验证支付状态：
- 检查支付网关仪表板
- 在 WooCommerce 中查找交易
- 如果已成功支付，请勿再次处理

### 2. 谨慎重试

如果支付确实未通过：
- 等待 30-60 秒
- 重新尝试
- 注意避免重复扣款

### 3. 检查连接

确保网络连接稳定：
- 测试您的互联网连接
- 如果使用 WiFi，尝试有线连接
- 检查网络问题

### 4. 选择较空闲时段

如果网关负载过高：
- 零售高峰期可能导致延迟
- 稍后再试
- 使用替代支付方式

### 5. 联系支付提供商

如果超时持续：
- 查看其状态页面
- 联系其支持团队
- 报告持续性问题

## 避免重复扣款

发生超时时：
1. **不要立即重试** — 先检查状态
2. **审查交易** — 检查网关和 WooCommerce
3. **退还重复款项** — 处理意外重复扣款的退款
4. **告知客户** — 如有延迟，解释情况

## 替代操作

如果支付持续超时：
- 接受现金支付
- 记录为"稍后支付"（如果允许）
- 使用其他支付方式
