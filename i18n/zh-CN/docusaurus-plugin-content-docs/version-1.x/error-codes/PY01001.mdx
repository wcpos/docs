# PY01001：支付被拒绝

## 这意味着什么

支付被支付处理器或发卡银行拒绝。交易无法完成。

## 常见原因

- **资金不足** — 账户中没有足够的资金
- **超过卡片限额** — 交易超过卡片限制
- **欺诈保护** — 银行标记为可疑
- **卡片详细信息不正确** — 卡号、CVV或有效期错误
- **卡片限制** — 卡片未启用此类购买

## 如何解决

### 1. 验证卡片详细信息

检查输入的信息是否正确：
- 卡号（所有数字）
- 到期日期
- CVV/安全码
- 账单地址（如果需要）

### 2. 重试

有时会出现临时问题：
- 稍等片刻
- 重新输入卡片详细信息
- 再次尝试交易

### 3. 使用其他支付方式

如果卡片继续被拒绝：
- 尝试另一张卡
- 使用现金支付
- 使用替代支付方式

### 4. 联系银行

持卡人可能需要：
- 与银行确认交易
- 解除欺诈拦截
- 确认卡片是否激活
- 检查消费限额

### 5. 检查限制

某些卡片有限制：
- 国际交易已禁用
- 在线/POS交易被阻止
- 某些商户类别被阻止

## 对于店主

如果频繁出现拒绝：
- 验证商户账户状态正常
- 检查支付网关配置
- 审查欺诈预防设置
- 考虑联系支付服务提供商
