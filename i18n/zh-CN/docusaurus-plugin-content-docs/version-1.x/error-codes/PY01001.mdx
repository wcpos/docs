# PY01001: 支付被拒绝

## 这意味着什么

支付被支付处理方或发卡行拒绝。交易无法完成。

## 常见原因

- **资金不足** — 账户余额不足
- **超出卡限额** — 交易超过卡限额
- **欺诈保护** — 银行标记为可疑
- **卡信息不正确** — 卡号、CVV或到期日期错误
- **卡限制** — 卡未启用此类购买

## 如何解决

### 1. 验证卡信息

检查输入的信息是否正确：
- 卡号（所有数字）
- 到期日期
- CVV/安全码
- 开票地址（如需）

### 2. 再试一次

有时会发生暂时性问题：
- 等待片刻
- 重新输入卡信息
- 再次尝试交易

### 3. 使用其他支付方式

如果卡仍然被拒绝：
- 尝试另一张卡
- 使用现金支付
- 使用其他支付方式

### 4. 联系银行

持卡人可能需要：
- 与银行核实交易
- 解除任何欺诈阻止
- 确认卡处于活动状态
- 检查消费限额

### 5. 检查限制

某些卡有限制：
- 禁止国际交易
- 在线/POS交易被阻止
- 一些商户类别被阻止

## 对于店主

如果您频繁看到支付被拒：
- 确认您的商户账户状态良好
- 检查支付网关配置
- 审核欺诈防范设置
- 考虑联系您的支付处理方
