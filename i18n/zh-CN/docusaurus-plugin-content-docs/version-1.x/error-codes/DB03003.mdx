# DB03003：缺少必填字段

## 这意味着什么

没有提供必填字段。数据库无法保存记录，因为缺少了基本信息。

## 常见原因

- **表单不完整** — 必填字段未填写
- **数据同步问题** — 必要数据未正确同步
- **绕过验证** — 未经验证提交表单
- **自定义字段必填** — 自定义必填字段为空

## 如何修复

### 1. 完成所有必填字段

检查表单：
- 带有星号(*)的字段
- 高亮或标记错误的字段
- 应有值的空字段

### 2. 刷新并重新输入

如果表单状态不正确：
1. 刷新页面/屏幕
2. 重新输入所有信息
3. 提交前验证所有字段

### 3. 检查WooCommerce设置

如果自定义字段为必填：
1. 查看WooCommerce结账设置
2. 检查必填的自定义字段
3. 确保POS提供这些字段

### 4. 同步必要数据

如果相关数据缺失：
- 刷新商品/客户等
- 等待同步完成
- 重试操作

### 5. 审查必填字段设置

在WooCommerce管理后台：
- 检查哪些字段标记为必填
- 考虑对POS是否真的都是必要的
- 必要时调整要求

## 常见必填字段

订单通常必填：
- 至少一个商品行
- 支付方式
- 客户（取决于设置）

商品通常必填：
- 商品名称
- 价格（对于简单商品）
