# SY01002：磁盘已满

## 含义

您的设备存储已满。POS 系统无法在没有可用磁盘空间的情况下保存数据或继续运行。

## 常见原因

- **存储已满** — 驱动器上没有可用空间
- **大型本地数据库** — POS 数据占用大量空间
- **临时文件** — 临时文件消耗空间
- **其他应用程序** — 其他应用填满磁盘

## 解决方法

### 1. 释放磁盘空间

删除不必要的文件：
- 清空回收站
- 删除旧的下载文件
- 卸载未使用的应用程序
- 清理临时文件

### 2. 检查可用空间

**Windows：**
- 打开文件资源管理器
- 右键单击驱动器 → 属性
- 查看已用和可用空间

**macOS：**
- 苹果菜单 → 关于本机
- 点击存储

### 3. 清理浏览器数据（网页版）

浏览器本地存储数据：
- 清除缓存和 Cookie
- 删除旧的站点数据
- 检查浏览器存储使用情况

### 4. 移动或删除大文件

查找并处理大文件：
- 旧视频或照片
- 大型下载文件
- 未使用的软件
- 可移动到外部存储的备份文件

### 5. 检查 POS 数据

如果 POS 数据非常大：
- 本地数据库可能需要清理
- 考虑清除并重新同步
- 检查是否可以清除旧数据

## 推荐可用空间

保持至少：
- **10GB** 用于正常操作
- **20GB+** 用于大型目录
- 更多空间始终更好

## 防止未来问题

- 定期清理不必要的文件
- 设置自动清理临时文件
- 监控磁盘空间使用情况
- 如果经常空间不足，考虑更大的存储
