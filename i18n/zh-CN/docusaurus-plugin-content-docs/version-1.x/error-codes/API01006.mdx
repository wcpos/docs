# API01006: 网络不可达

## 这意味着什么

POS 无法建立通往服务器的路由。这是一个网络级别的问题——数据包无法找到到达您服务器的路径。

## 常见原因

- **网络配置问题** — 网关或路由设置不正确
- **ISP 问题** — 您的互联网服务提供商出现问题
- **VPN 干扰** — VPN 可能错误地路由流量
- **服务器网络问题** — 服务器的网络可能已断开

## 如何修复

### 1. 检查您的互联网连接

- 尝试访问其他网站
- 重启您的路由器/调制解调器
- 验证您已连接到网络

### 2. 临时禁用 VPN

如果使用 VPN：
- 断开连接并重试
- 尝试不同的 VPN 服务器
- 检查 VPN 是否允许访问您服务器的位置

### 3. 检查您的 ISP

如果以下情况，请联系您的互联网服务提供商：
- 其他网站也无法访问
- 在重启网络设备后问题仍然存在

### 4. 验证服务器状态

- 检查您的主机提供商是否报告任何网络问题
- 从其他网络（例如，移动数据）尝试访问您的网站
- 使用在线工具检查该网站是否对所有人都无法访问

### 5. 检查防火墙设置

确保您设备的防火墙没有阻止与您服务器的出站连接。
