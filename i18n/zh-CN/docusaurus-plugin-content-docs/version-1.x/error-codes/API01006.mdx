# API01006：网络不可达

## 这意味着什么

POS无法建立到服务器的路由。这是一个网络层面的问题 — 数据包无法找到到您服务器的路径。

## 常见原因

- **网络配置问题** — 网关或路由设置不正确
- **ISP问题** — 您的互联网服务提供商遇到问题
- **VPN干扰** — VPN可能错误地路由流量
- **服务器网络问题** — 服务器的网络可能已关闭

## 如何解决

### 1. 检查您的互联网连接

- 尝试访问其他网站
- 重启路由器/调制解调器
- 验证您是否连接到网络

### 2. 暂时禁用VPN

如果正在使用VPN：
- 断开连接并重试
- 尝试不同的VPN服务器
- 检查VPN是否允许访问您服务器的位置

### 3. 与ISP联系

如果出现以下情况，请联系您的互联网服务提供商：
- 其他站点也无法访问
- 重启网络设备后问题仍然存在

### 4. 验证服务器状态

- 检查您的主机提供商是否报告任何网络问题
- 尝试从不同网络（如移动数据）访问您的站点
- 使用在线工具检查站点是否对所有人都不可用

### 5. 检查防火墙设置

确保您的设备防火墙未阻止到服务器的出站连接。
