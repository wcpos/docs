# API 错误 {#api-errors}

当与您的 WooCommerce 服务器通信时，会出现 API 错误。这些错误以 `API` 为前缀，并分为以下几类：

## 分类 {#categories}

| 类别 | 代码范围 | 描述 |
|------|----------|------|
| [连接](#connection-errors) | API01xxx | 网络和连通性问题 |
| [身份验证](#authentication-errors) | API02xxx | 登录、令牌和权限问题 |
| [请求](#request-errors) | API03xxx | 发送请求时的问题 |
| [响应](#response-errors) | API04xxx | 服务器响应的问题 |
| [插件/WordPress](#pluginwordpress-errors) | API05xxx | WordPress 或 WooCommerce 插件的问题 |
| [配置](#configuration-errors) | API06xxx | 设置和配置问题 |

---

## 连接错误 {#connection-errors}

POS 与您的服务器之间的网络和连通性问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [API01001](./API01001) | 连接超时 | 服务器响应时间过长 |
| [API01002](./API01002) | 连接被拒绝 | 服务器拒绝连接 |
| [API01003](./API01003) | 连接重置 | 连接意外关闭 |
| [API01004](./API01004) | DNS 解析失败 | 无法解析服务器地址 |
| [API01005](./API01005) | SSL 证书错误 | 网站的安全证书存在问题 |
| [API01006](./API01006) | 网络不可达 | 无法连接到网络 |
| [API01007](./API01007) | 设备离线 | 您的设备未连接到互联网 |
| [API01008](./API01008) | 网站不可用 | 网站未响应 |

## 身份验证错误 {#authentication-errors}

登录、会话和权限的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [API02001](./API02001) | 无效凭据 | 用户名或密码不正确 |
| [API02002](./API02002) | 令牌已过期 | 您的会话已过期 |
| [API02003](./API02003) | 令牌无效 | 身份验证令牌无效 |
| [API02004](./API02004) | 用户未授权 | 您没有权限执行此操作 |
| [API02005](./API02005) | 权限不足 | 您的用户角色缺少所需权限 |
| [API02006](./API02006) | API 密钥无效 | WooCommerce API 密钥无效 |
| [API02007](./API02007) | 令牌刷新失败 | 无法刷新您的会话 |
| [API02008](./API02008) | 刷新令牌无效 | 刷新令牌无效 |
| [API02009](./API02009) | 刷新令牌已过期 | 刷新令牌已过期 |
| [API02010](./API02010) | 需要认证 | 此操作需要认证 |

## 请求错误 {#request-errors}

发送到服务器的请求出现的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [API03001](./API03001) | 请求格式无效 | 请求格式不正确 |
| [API03002](./API03002) | 缺少必需参数 | 请求中缺少必需数据 |
| [API03003](./API03003) | 参数值无效 | 某参数的值无效 |
| [API03004](./API03004) | 请求过大 | 请求超出大小限制 |
| [API03005](./API03005) | 超过速率限制 | 短时间内请求过多 |
| [API03006](./API03006) | 不支持的方法 | 不支持该 HTTP 方法 |
| [API03007](./API03007) | 请求队列已满 | 待处理请求过多 |

## 响应错误 {#response-errors}

从服务器收到的响应出现的问题。

| 代码 | 名称 | 描述 |
|------|------|------|
| [API04001](./API04001) | 响应格式无效 | 服务器响应格式无效 |
| [API04002](./API04002) | 意外的响应代码 | 收到意外的 HTTP 状态代码 |
| [API04003](./API04003) | JSON 响应格式错误 | JSON 响应已损坏或无效 |
| [API04004](./API04004) | 缺少响应数据 | 响应中缺少预期数据 |
| [API04005](./API04005) | 尝试 JSON 恢复 | 尝试从格式错误的 JSON 中恢复 |
| [API04006](./API04006) | 资源未找到 | 请求的资源不存在（404） |

| [API05001](./API05001) | WooCommerce API 已禁用 | WooCommerce REST API 已禁用 |

| [API05003](./API05003) | WCPOS 插件过时 | WCPOS 插件需要更新 |
| [API05004](./API05004) | WordPress API 已禁用 | WordPress REST API 已禁用 |
| [API05005](./API05005) | 找不到插件 | 缺少所需的插件 |
