# 错误代码

当 WCPOS 遇到问题时，它会显示一个错误代码，帮助识别具体的问题。每个错误代码遵循 `[域][类别][具体代码]` 的格式，以帮助您快速了解错误的类型和性质。

## 错误代码格式

| 组件 | 描述 | 示例 |
|-----------|-------------|---------|
| **域** | 系统区域 | `API`, `DB`, `PY`, `SY` |
| **类别** | 错误类别 | `01`, `02`, `03` 等 |
| **具体代码** | 唯一标识符 | `001`, `002`, `003` 等 |

**示例：** `API04001` = API（域）+ 04（响应错误）+ 001（无效的响应格式）

## 错误域

### [API 错误](./api)

API 错误发生在与 WooCommerce 服务器通信时。

| 类别 | 代码范围 | 描述 |
|----------|------------|-------------|
| 连接 | API01xxx | 网络和连接问题 |
| 认证 | API02xxx | 登录、令牌和权限问题 |
| 请求 | API03xxx | 传出请求的问题 |
| 响应 | API04xxx | 服务器响应的问题 |
| 插件/WordPress | API05xxx | WordPress 或 WooCommerce 插件问题 |
| 配置 | API06xxx | 设置和配置问题 |

### [数据库错误](./db)

数据库错误发生在本地存储或检索数据时。

| 类别 | 代码范围 | 描述 |
|----------|------------|-------------|
| 连接 | DB01xxx | 数据库连接和事务问题 |
| 数据 | DB02xxx | 记录和约束问题 |
| 查询 | DB03xxx | 查询语法和数据类型问题 |

### [支付错误](./py)

支付错误发生在结账和支付处理期间。

| 类别 | 代码范围 | 描述 |
|----------|------------|-------------|
| 卡片 | PY01xxx | 支付卡片问题 |
| 网关 | PY02xxx | 支付网关通信问题 |

### [系统错误](./sy)

系统错误与设备资源和系统配置有关。

| 类别 | 代码范围 | 描述 |
|----------|------------|-------------|
| 资源 | SY01xxx | 内存、磁盘和权限问题 |
| 服务 | SY02xxx | 配置和服务可用性问题 |
