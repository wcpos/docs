# API05005: 插件未找到

## 这意味着什么

所需的 WordPress 插件未安装或未激活。可能是 WooCommerce 本身、所需的扩展或集成插件。

## 常见原因

- **WooCommerce 禁用** — WooCommerce 被禁用
- **缺少所需的扩展** — 所需的 WooCommerce 扩展未安装
- **插件冲突** — 另一个插件导致所需插件被禁用
- **更新失败** — 插件更新失败，并导致其处于禁用状态

## 如何修复

### 1. 验证 WooCommerce 是否激活

WCPOS 需要 WooCommerce：
1. 转到 插件 → 已安装插件
2. 找到 "WooCommerce"
3. 确保它已被激活

### 2. 检查所需的扩展

某些 WCPOS 功能可能需要扩展：
- 检查错误消息以获取特定插件名称
- 安装并激活所需的扩展

### 3. 审查插件冲突

如果插件由于冲突而被禁用：
1. 检查您的电子邮件以获取 WordPress 通知
2. 审查错误日志
3. 解决冲突并重新激活

### 4. 检查插件文件

如果插件的文件丢失：
1. 从 WordPress.org 重新安装该插件
2. 或从备份中恢复
3. 激活该插件

### 5. 更新所有插件

过时的插件可能导致问题：
1. 转到 仪表盘 → 更新
2. 更新所有插件
3. 验证所有所需插件都已激活

## 所需插件

至少，WCPOS 需要：
- **WooCommerce** — 电子商务平台
- **WCPOS** — POS 插件

其他附加功能可能需要其他插件。
