# API05005：插件未找到

## 这意味着什么

所需的WordPress插件未安装或未激活。这可能是WooCommerce本身、所需的扩展或集成插件。

## 常见原因

- **WooCommerce已停用** — WooCommerce被禁用
- **缺少必需的扩展** — 所需的WooCommerce扩展未安装
- **插件冲突** — 另一个插件导致所需插件被停用
- **更新失败** — 插件更新失败并导致其停用

## 如何修复

### 1. 验证WooCommerce已激活

WCPOS需要WooCommerce：
1. 转到插件 → 已安装插件
2. 找到"WooCommerce"
3. 确保已激活

### 2. 检查必需的扩展

某些WCPOS功能可能需要扩展：
- 查看错误消息以获取特定插件名称
- 安装并激活所需的扩展

### 3. 检查插件冲突

如果由于冲突导致插件被停用：
1. 检查WordPress通知邮件
2. 查看错误日志
3. 解决冲突并重新激活

### 4. 检查插件文件

如果插件文件丢失：
1. 从WordPress.org重新安装插件
2. 或从备份还原
3. 激活插件

### 5. 更新所有插件

过时的插件可能导致问题：
1. 转到仪表板 → 更新
2. 更新所有插件
3. 验证所有必需的插件已激活

## 必需的插件

最低要求，WCPOS需要：
- **WooCommerce** — 电子商务平台
- **WCPOS** — POS插件

其他功能可能需要其他插件。
