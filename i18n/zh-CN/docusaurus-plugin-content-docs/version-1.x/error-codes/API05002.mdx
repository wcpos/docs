# API05002: 未找到 WCPOS 插件

## 这意味着什么

WCPOS（WooCommerce POS）插件未在您的 WordPress 站点上安装或未激活。POS 应用程序需要此插件与 WooCommerce 进行通信。

## 常见原因

- **未安装插件** — 插件从未安装
- **插件已停用** — 插件已禁用
- **插件已删除** — 插件文件已被移除
- **错误的站点** — 连接到未安装 WCPOS 的站点

## 如何修复

### 1. 安装插件

如果未安装：
1. 转到 WordPress 管理 → 插件 → 添加新
2. 搜索 "WooCommerce POS"
3. 安装并激活插件

或从 [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) 下载

### 2. 激活插件

如果已安装但未激活：
1. 转到 插件 → 已安装插件
2. 找到 "WooCommerce POS"
3. 点击 "激活"

### 3. 检查错误

如果插件无法激活：
- 检查错误日志中的 PHP 错误
- 验证 WooCommerce 是否已安装并激活
- 检查 PHP 版本要求
- 查找插件冲突

### 4. 验证 URL

确保 POS 已配置为连接到正确的 WordPress 站点：
- 检查 POS 设置中的 URL
- 验证其与您的 WordPress 安装匹配

### 5. 如有必要，重新安装

如果插件已损坏：
1. 停用并删除插件
2. 从 WordPress.org 重新安装
3. 激活插件

## 要求

WCPOS 要求：
- WordPress（近期版本）
- WooCommerce（近期版本）
- PHP 7.4 或更高版本
