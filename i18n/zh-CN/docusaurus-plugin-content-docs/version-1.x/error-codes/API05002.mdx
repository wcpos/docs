# API05002：未找到WCPOS插件

## 含义

WooCommerce POS（WCPOS）插件未安装或未在您的WordPress站点上激活。POS应用程序需要此插件才能与WooCommerce通信。

## 常见原因

- **插件未安装** — 从未安装插件
- **插件已停用** — 插件已被禁用
- **插件已删除** — 插件文件已移除
- **错误的站点** — 连接到没有WCPOS的站点

## 解决方法

### 1. 安装插件

如果未安装：
1. 转到WordPress管理后台 → 插件 → 添加新插件
2. 搜索"WooCommerce POS"
3. 安装并激活插件

或从[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)下载

### 2. 激活插件

如果已安装但未激活：
1. 转到插件 → 已安装插件
2. 找到"WooCommerce POS"
3. 点击"激活"

### 3. 检查错误

如果插件无法激活：
- 检查错误日志中的PHP错误
- 验证WooCommerce已安装并激活
- 检查PHP版本要求
- 查找插件冲突

### 4. 验证URL

确保POS配置连接到正确的WordPress站点：
- 检查POS设置中的URL
- 验证是否与您的WordPress安装匹配

### 5. 必要时重新安装

如果插件已损坏：
1. 停用并删除插件
2. 从WordPress.org重新安装
3. 激活插件

## 系统要求

WCPOS需要：
- WordPress（最新版本）
- WooCommerce（最新版本）
- PHP 7.4 或更高版本
