# PY02001: 支付网关错误

## 这意味着什么

在与支付网关通信时发生错误。支付处理器无法完成请求。

## 常见原因

- **网关停机** — 支付服务遇到问题
- **配置错误** — 网关设置不正确
- **API凭证** — 无效或过期的API密钥
- **网络问题** — 与网关的连接问题

## 如何修复

### 1. 重试交易

有时网关会出现临时问题：
- 等待片刻
- 再次尝试处理支付
- 问题可能会自行解决

### 2. 检查网关状态

访问您的支付提供商状态页面：
- Stripe: status.stripe.com
- PayPal: 检查PayPal社区或状态
- Square: status.squareup.com
- 检查已知的停机情况

### 3. 验证网关配置

在WooCommerce管理中：
1. 转到WooCommerce → 设置 → 支付
2. 点击您的支付网关
3. 验证API密钥是否正确
4. 检查测试/实时模式是否与您的密钥匹配

### 4. 检查API凭证

确保凭证有效：
- API密钥未过期
- 使用实时密钥进行真实交易
- 使用测试密钥进行测试交易
- 凭证与正确的账户匹配

### 5. 使用不同的网关进行测试

如果可用：
- 尝试使用替代网关处理
- 使用现金支付作为后备
- 完成销售，稍后进行对账

## 对于商店所有者

常见配置问题：
- 混合使用测试和实时API密钥
- 不同账户的API密钥
- 禁用的支付方式
- 过期或被撤销的凭证

如果问题仍然存在，请联系您的支付提供商支持。
