# PY02001：支付网关错误

## 这意味着什么

在与支付网关通信时发生了错误。支付处理程序无法完成请求。

## 常见原因

- **网关宕机** — 支付服务正在遇到问题
- **配置错误** — 网关设置不正确
- **API凭证** — 无效或过期的API密钥
- **网络问题** — 与网关的连接出现问题

## 如何修复

### 1. 重试交易

有时网关会出现暂时性问题：
- 稍等片刻
- 尝试重新处理付款
- 问题可能会自行解决

### 2. 检查网关状态

访问支付提供商的状态页面：
- Stripe：status.stripe.com
- PayPal：查看PayPal社区或状态
- Square：status.squareup.com
- 检查已知的服务中断

### 3. 验证网关配置

在WooCommerce管理后台：
1. 转到WooCommerce → 设置 → 支付
2. 点击您的支付网关
3. 验证API密钥是否正确
4. 检查测试/生产模式是否与您的密钥匹配

### 4. 检查API凭证

确保凭证有效：
- API密钥未过期
- 生产交易使用生产密钥
- 测试交易使用测试密钥
- 凭证与正确的账户匹配

### 5. 尝试使用其他网关

如果可用：
- 尝试使用备用网关处理
- 使用现金支付作为后备
- 完成销售并稍后对账

## 对于商店所有者

常见配置问题：
- 混淆测试和生产API密钥
- 来自不同账户的API密钥
- 禁用的支付方式
- 过期或吊销的凭证

如果问题持续存在，请联系您的支付提供商支持。
