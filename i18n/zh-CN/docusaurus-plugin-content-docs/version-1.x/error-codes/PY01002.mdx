# PY01002：资金不足

## 含义

由于账户中没有足够的资金来支付交易，支付被拒绝。

## 常见原因

- **账户余额低** — 可用资金不足
- **待处理交易** — 其他待处理交易减少了可用余额
- **信用卡额度已达上限** — 信用卡已透支
- **资金被冻结** — 资金被其他交易暂时冻结

## 解决方法

### 1. 使用其他支付方式

最快的解决方案：
- 尝试另一张卡
- 使用现金
- 分多种方式支付（如果支持）

### 2. 减少订单金额

如果可以部分支付：
- 移除部分商品
- 应用折扣或优惠券
- 完成一个较小的交易

### 3. 客户操作

客户可能需要：
- 向账户转入资金
- 等待待处理交易清算
- 偿还信用卡余额
- 申请提高信用额度

### 4. 分割交易

如果POS系统支持分割支付：
- 用一张卡支付部分金额
- 用另一种方式支付剩余金额

## 隐私提示

与客户沟通时：
- 谨慎处理具体拒付原因
- 简单地说"卡片被拒"
- 允许客户私下尝试其他支付方式
