:::info 专业功能
产品屏幕需要 [WCPOS Pro](/getting-started/pro-license)。免费用户可以在 [POS产品面板](/pos/product-panel) 中查看和添加产品到购物车，但不能编辑库存或价格。
:::

产品屏幕是一个专用库存管理界面。与 [POS产品面板](/pos/product-panel)（用于销售）不同，此屏幕旨在管理您的产品目录。

{/* 截图: products-screen.png */}

## 产品屏幕与POS产品面板 {#products-screen-vs-pos-product-panel}

| 功能 | POS产品面板 | 产品屏幕 |
|---------|-------------------|-----------------|
| **目的** | 添加产品到购物车 | 管理库存 |
| **库存** | 仅查看 | 行内编辑 |
| **价格** | 仅查看 | 行内编辑 |
| **动作** | 添加到购物车 | 编辑、同步、删除 |
| **可用性** | 免费 + 专业 | 仅专业 |

## 界面概述 {#interface-overview}

### 搜索与过滤 {#search-filters}

在屏幕顶部：

- **搜索栏** - 按名称、SKU或条形码查找产品
- **过滤按钮** - 库存状态、推荐、特价、类别、标签、品牌
- **显示设置** (<Icon name="sliders" />) - 配置可见列

### 产品列表 {#product-list}

主区域以可滚动的表格显示您的产品：

- **图片** - 产品缩略图
- **产品名称** - 可选的SKU、条形码、属性
- **库存** - 可编辑的库存数量，带“管理”切换
- **库存状态** - 有库存（绿色）、缺货（红色）、缺货待补
- **类别** - 产品类别标签
- **价格** - 当前价格、原价、特价（均可编辑）
- **动作** - 三点菜单，包含编辑、同步、删除

### 变体产品 {#variable-products}

变体产品显示：
- 属性选项（颜色、尺寸等）
- “展开”链接以查看所有变体
- 单个变体行，分别显示库存/价格

### 页脚 {#footer}

- **税务状态** - 当前税收计算基础
- **产品数量** - “显示X中的Y”，带同步按钮 (<Icon name="arrow-rotate-right" />)。**长按**同步按钮可选清除和刷新选项

## 主要功能 {#key-features}

### 行内库存编辑 {#inline-stock-editing}

直接点击库存数量进行编辑：

1. 点击库存数字
2. 输入新数量
3. 按Enter键保存

更改会自动同步到WooCommerce。

### 管理库存切换 {#manage-stock-toggle}

“管理”切换控制WooCommerce是否跟踪此产品的库存：

- **开启** - 销售时跟踪并减少库存水平
- **关闭** - 产品始终可用（无限库存）

### 行内价格编辑 {#inline-price-editing}

点击任何价格字段进行编辑：

- **价格** - 当前销售价格
- **原价** - 非折扣价格
- **特价** - 折扣价格（当特价时）

### 产品动作 {#product-actions}

点击每个产品的三点菜单 (⋮)：

- **编辑** - 打开产品编辑模态窗口
- **同步** - 从WooCommerce刷新此产品
- **删除** - 从本地数据库中移除产品

:::note
从POS删除产品仅会在本地移除。该产品仍然保留在WooCommerce中，并将在下次同步时重新出现。
:::

### 可用列 {#available-columns}

| 列 | 描述 | 显示选项 |
|--------|-------------|-----------------|
| **图片** | 产品缩略图 | - |
| **ID** | WooCommerce产品ID | - |
| **产品** | 产品名称 | SKU、条形码、属性 |
| **类型** | 简单、变体、组合等 | - |
| **SKU** | 库存单位 | - |
| **条形码** | 产品条形码 | - |
| **库存** | 当前数量（可编辑） | - |
| **库存状态** | 有库存 / 缺货 / 缺货待补 | - |
| **类别** | 产品类别 | - |
| **标签** | 产品标签 | - |
| **品牌** | 品牌信息（如果插件已激活） | - |
| **销售成本** | 产品成本价格 | - |
| **价格** | 当前销售价格（可编辑） | 税务信息 |
| **原价** | 非折扣价格（可编辑） | 税务信息 |
| **特价** | 折扣价格（可编辑） | 税务信息 |
| **创建日期** | 产品创建时间 | - |
| **修改日期** | 最后修改日期 | - |
| **动作** | 编辑、同步、删除按钮 | - |

### 恢复默认设置 {#restore-default-settings}

点击以重置所有显示设置为原始默认值。

## 筛选产品 {#filtering-products}

使用过滤按钮缩小您的视图：

- **库存状态** - 有库存、缺货、缺货待补
- **推荐** - 标记为推荐的产品
- **促销** - 当前特价的产品
- **类别** - 按产品类别过滤
- **标签** - 按产品标签过滤
- **品牌** - 按品牌过滤（如果使用品牌插件）

## 排序 {#sorting}

点击任意列标题按该列排序。再次点击以反转排序顺序。
