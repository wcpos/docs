:::info 专业版功能
产品页面需要 [WCPOS Pro](/getting-started/pro-license)。免费用户可以在 [POS 产品面板](/pos/product-panel)中查看和添加产品到购物车，但无法编辑库存或价格。
:::

产品页面是一个专门的库存管理界面。与 [POS 产品面板](/pos/product-panel)（用于销售）不同，此页面专为管理您的产品目录而设计。

{/* Screenshot: products-screen.png */}

## 产品页面 vs POS 产品面板

| 功能 | POS 产品面板 | 产品页面 |
|---------|-------------------|-----------------|
| **目的** | 添加产品到购物车 | 管理库存 |
| **库存** | 仅查看 | 可内联编辑 |
| **价格** | 仅查看 | 可内联编辑 |
| **操作** | 添加到购物车 | 编辑、同步、删除 |
| **可用于** | 免费版 + 专业版 | 仅限专业版 |

## 界面概览

### 搜索和筛选

屏幕顶部：

- **搜索栏** - 按名称、SKU 或条形码查找产品
- **筛选按钮** - 库存状态、特色、促销、分类、标签、品牌
- **显示设置** (<Icon name="sliders" />) - 配置可见列

### 产品列表

主区域以可滚动表格显示您的产品：

- **图片** - 产品缩略图
- **产品名称** - 带可选 SKU、条形码、属性
- **库存** - 可编辑的库存数量，带"管理"开关
- **库存状态** - 有库存（绿色），无库存（红色），允许回货
- **分类** - 产品分类徽章
- **价格** - 当前价格、原价、促销价（均可编辑）
- **操作** - 三点菜单，包含编辑、同步、删除

### 多属性产品

多属性产品显示：
- 属性选项（颜色、尺码等）
- "展开"链接以查看所有变体
- 带单独库存/价格的单个变体行

### 底部

- **税收状态** - 当前税收计算基础
- **产品数量** - "显示 X 个，共 Y 个"，带同步按钮

## 关键功能

### 内联库存编辑

直接点击库存数量进行编辑：

1. 点击库存数字
2. 输入新数量
3. 按回车保存

更改会自动同步到 WooCommerce。

### 管理库存开关

"管理"开关控制 WooCommerce 是否跟踪产品库存：

- **开** - 跟踪库存水平并在销售时扣减
- **关** - 产品始终可用（无限库存）

### 内联价格编辑

点击任何价格字段进行编辑：

- **价格** - 当前销售价
- **原价** - 未打折价格
- **促销价** - 打折价格（促销期间）

### 产品操作

点击每个产品的三点菜单（⋮）：

- **编辑** - 打开产品编辑弹窗
- **同步** - 从 WooCommerce 刷新此产品
- **删除** - 从本地数据库移除产品

:::note
从 POS 删除产品仅在本地删除。产品仍保留在 WooCommerce 中，并将在下次同步时重新出现。
:::

### 可用列

| 列 | 描述 | 显示选项 |
|--------|-------------|-----------------|
| **图片** | 产品缩略图 | - |
| **ID** | WooCommerce 产品 ID | - |
| **产品** | 产品名称 | SKU、条形码、属性 |
| **类型** | 简单、多属性、分组等 | - |
| **SKU** | 库存单位 | - |
| **条形码** | 产品条形码 | - |
| **库存** | 当前数量（可编辑） | - |
| **库存状态** | 有库存 / 无库存 / 允许回货 | - |
| **分类** | 产品分类 | - |
| **标签** | 产品标签 | - |
| **品牌** | 品牌信息（如插件激活） | - |
| **销售成本** | 产品成本价 | - |
| **价格** | 当前销售价（可编辑） | 税收信息 |
| **原价** | 未打折价格（可编辑） | 税收信息 |
| **促销价** | 打折价格（可编辑） | 税收信息 |
| **创建日期** | 产品创建时间 | - |
| **修改日期** | 最后修改日期 | - |
| **操作** | 编辑、同步、删除按钮 | - |

### 恢复默认设置

点击重置所有显示设置为原始默认值。

## 筛选产品

使用筛选按钮缩小视图：

- **库存状态** - 有库存、无库存、允许回货
- **特色** - 标记为特色的产品
- **促销** - 当前正在促销的产品
- **分类** - 按产品分类筛选
- **标签** - 按产品标签筛选
- **品牌** - 按品牌筛选（如使用品牌插件）

## 排序

点击任何列标题按该列排序。再次点击反转排序顺序。
