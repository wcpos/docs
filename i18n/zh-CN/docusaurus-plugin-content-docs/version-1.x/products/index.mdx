:::info 专业功能
产品界面需要 [WCPOS Pro](/getting-started/pro-license)。免费用户可以从 [POS 产品面板](/pos/product-panel) 查看和添加产品到购物车，但无法编辑库存或价格。
:::

产品界面是一个专用的库存管理界面。与 [POS 产品面板](/pos/product-panel)（用于销售）不同，此界面旨在管理您的产品目录。

{/* 截图：products-screen.png */}

## 产品界面与 POS 产品面板

| 特性 | POS 产品面板 | 产品界面 |
|---------|-------------------|-----------------|
| **目的** | 将产品添加到购物车 | 管理库存 |
| **库存** | 仅查看 | 内联编辑 |
| **价格** | 仅查看 | 内联编辑 |
| **操作** | 添加到购物车 | 编辑、Sync、删除 |
| **提供于** | 免费 + 专业 | 仅专业 |

## 界面概述

### 搜索与筛选

在界面顶部：

- **搜索栏** - 按名称、SKU 或条形码查找产品
- **筛选按钮** - 库存状态、推荐、特价、类别、标签、品牌
- **显示设置** (<Icon name="sliders" />) - 配置可见列

### 产品列表

主要区域以可滚动表格显示您的产品：

- **图片** - 产品缩略图
- **产品名称** - 可选 SKU、条形码、属性
- **库存** - 可编辑的库存数量与“管理”切换
- **库存状态** - 有库存（绿色）、无库存（红色）、待补货
- **类别** - 产品类别标记
- **价格** - 当前价格、正常价格、特价（均可编辑）
- **操作** - 三点菜单，包含编辑、Sync、删除

### 变体产品

变体产品显示：
- 属性选项（颜色、尺寸等）
- “展开”链接以查看所有变体
- 单独变体行，具有单独的库存/价格

### 页脚

- **税务状态** - 当前税务计算基础
- **产品数量** - “显示 X 的 Y”与 Sync 按钮 (<Icon name="arrow-rotate-right" />)。**长按** Sync 按钮以获得清除和刷新选项

## 关键特性

### 内联库存编辑

直接点击库存数量以进行编辑：

1. 点击库存数字
2. 输入新数量
3. 按回车键保存

更改会自动同步到 WooCommerce。

### 管理库存切换

“管理”切换控制 WooCommerce 是否跟踪此产品的库存：

- **开启** - 销售时库存水平会被跟踪和减少
- **关闭** - 产品始终可用（无限库存）

### 内联价格编辑

点击任何价格字段进行编辑：

- **价格** - 当前销售价格
- **正常价格** - 未折扣的价格
- **特价** - 折扣价格（在特价时）

### 产品操作

点击每个产品的三点菜单（⋮）：

- **编辑** - 打开产品编辑模态框
- **Sync** - 从 WooCommerce 刷新该产品
- **删除** - 从本地数据库中移除产品

:::note
从 POS 删除产品仅会在本地移除。该产品仍保留在 WooCommerce 中，并会在下次同步时重新出现。
:::

### 可用列

| 列 | 描述 | 显示选项 |
|--------|-------------|-----------------|
| **图片** | 产品缩略图 | - |
| **ID** | WooCommerce 产品 ID | - |
| **产品** | 产品名称 | SKU、条形码、属性 |
| **类型** | 简单、变体、分组等 | - |
| **SKU** | 库存单位 | - |
| **条形码** | 产品条形码 | - |
| **库存** | 当前数量（可编辑） | - |
| **库存状态** | 有库存 / 无库存 / 待补货 | - |
| **类别** | 产品类别 | - |
| **标签** | 产品标签 | - |
| **品牌** | 品牌信息（如果插件激活） | - |
| **销售成本** | 产品成本 | - |
| **价格** | 当前销售价格（可编辑） | 税务信息 |
| **正常价格** | 未折扣价格（可编辑） | 税务信息 |
| **特价** | 折扣价格（可编辑） | 税务信息 |
| **创建日期** | 产品创建时间 | - |
| **修改日期** | 最后修改日期 | - |
| **操作** | 编辑、Sync、删除按钮 | - |

### 恢复默认设置

点击重置所有显示设置为原始默认值。

## 筛选产品

使用筛选按钮缩小您的视图：

- **库存状态** - 有库存、无库存、待补货
- **推荐** - 标记为推荐的产品
- **特价** - 当前特价的产品
- **类别** - 按产品类别筛选
- **标签** - 按产品标签筛选
- **品牌** - 按品牌筛选（如果使用品牌插件）

## 排序

点击任何列标题按该列排序。再次点击以反转排序顺序。
