仅限 POS 产品是一项功能，允许您控制产品是否出现在您的在线商店、POS，或两者皆有。当您拥有不同的在线和线下销售产品目录时，这非常有用。

## 概述 {#overview}

启用后，每个产品会有一个 **POS 可见性** 设置，包含三个选项：

| 可见性 | 在线商店 | POS |
|--------|----------|-----|
| **POS & 在线**（默认） | 可见 | 可见 |
| **仅 POS** | 隐藏 | 可见 |
| **仅在线** | 可见 | 隐藏 |

## 何时使用此功能 {#when-to-use-this-feature}

如果您需要，启用仅限 POS 产品：

- **销售专属店内产品** - 仅在客户访问您的实体店时可用的产品
- **从 POS 隐藏仅在线项目** - 不应在店内销售的数字产品、预购商品或其他项目
- **管理不同的目录** - 在线和零售渠道的不同产品系列
- **控制促销产品** - 店内专属特价或仅在线交易

:::tip
如果您的所有产品都应该出现在两个渠道中，则不需要此功能。将其保持禁用以提高性能。
:::

## 启用该功能 {#enabling-the-feature}

1. 转到 **WP Admin → WCPOS → 设置 → 常规**
2. 启用 **"启用仅限 POS 产品"**
3. 保存更改

有关常规设置的更多详情，请参见 [WP 管理常规设置](/settings/wp-admin/general)。

:::warning 性能影响
仅在必要时启用此设置。启用后，每个产品请求会增加额外的数据库查找，这可能会影响拥有大量目录的商店的性能。
:::

## 设置产品可见性 {#setting-product-visibility}

### 单个产品 {#individual-products}

要设置单个产品的可见性：

1. 转到 **WP Admin → 产品** 并编辑一个产品
2. 在 **产品数据** 面板中，查找 **POS 可见性**
3. 选择您想要的选项：
   - **POS & 在线** - 产品在所有地方可见（默认）
   - **仅 POS** - 产品在在线商店中隐藏
   - **仅在线** - 产品在 POS 中隐藏
4. 单击 **更新** 保存

### 批量编辑 {#bulk-editing}

要一次更改多个产品的可见性：

1. 转到 **WP Admin → 产品**
2. 使用复选框选择您想要修改的产品
3. 从 **批量操作** 下拉菜单中选择 **编辑**
4. 单击 **应用**
5. 在批量编辑面板中，查找 **POS 可见性** 下拉菜单
6. 选择您想要的可见性选项
7. 单击 **更新**

### 可变产品 {#variable-products}

对于可变产品，您可以在父级产品级别和每个单独变体上设置可见性。这使您能够进行精细控制 - 例如，您可以在网上显示所有变体，但在店内仅显示某些颜色或尺寸。

## 它对 POS 的影响 {#how-it-affects-the-pos}

当产品设置为 **仅在线** 时：

- 它不会出现在 [产品面板](/pos/product-panel) 中
- 不会通过搜索或条形码扫描找到
- 不能在 POS 中添加到购物车

当产品设置为 **仅 POS** 时：

- 它不会出现在您的 WooCommerce 商店前端
- 不会包含在在线搜索结果中
- 客户无法在线购买

:::note
如果您最近更改了产品的可见性，但它仍然在 POS 中出现（或不出现），请尝试同步。长按同步按钮并选择 **"清除并刷新"** 以重新加载所有产品数据。
:::

## 常见用例 {#common-use-cases}

<UseCases>
  <UseCaseCard icon="gift" title="仅限店内的礼品卡">

  需要在注册时激活的实体礼品卡：

  1. 创建礼品卡产品
  2. 将可见性设置为 **仅 POS**
  3. 卡片将不会在线显示，但工作人员可以在店内出售

  </UseCaseCard>

  <UseCaseCard icon="globe" title="在线预购">

  可以在线预购但尚未有存货的产品：

  1. 创建预购产品
  2. 将可见性设置为 **仅在线**
  3. 客户可以在线预购，但它不会占用 POS 的空间

  </UseCaseCard>

  <UseCaseCard icon="building" title="批发与零售">

  如果您在同一 WooCommerce 商店中同时处理批发和零售：

  1. 将仅限批发的产品设置为 **仅在线**（用于 B2B 门户访问）
  2. 将零售专属产品设置为 **仅 POS**
  3. 将常规产品保留为 **POS & 在线**

  </UseCaseCard>
</UseCases>

## 故障排除 {#troubleshooting}

<Accordion>
  <AccordionItem question="设置为仅在线的产品仍然出现">

  1. 打开 POS
  2. 在产品面板底部长按同步按钮
  3. 选择 **"清除并刷新"**
  4. 等待产品重新加载

  </AccordionItem>

  <AccordionItem question="产品上未显示设置">

  确保该功能已启用：

  1. 转到 **WP Admin → WCPOS → 设置 → 常规**
  2. 验证 **"启用仅限 POS 产品"** 已选中
  3. 如果您进行了更改，请保存设置

  </AccordionItem>
</Accordion>
