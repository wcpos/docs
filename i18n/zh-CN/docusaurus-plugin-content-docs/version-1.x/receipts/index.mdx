WCPOS 提供完整的模板系统，可用于小票、发票、礼品小票、装箱单、厨房订单和其他内容，所有模板均在 **WP 管理后台 → POS → 模板** 中管理。

:::tip 只想更改小票？
从 **[自定义小票](/receipts/customise)** 开始。这里介绍了三种最简单的方法：选择其他内置模板、让 AI 调整模板，或手动编辑。
:::

## 模板引擎 {#template-engines}

WCPOS 支持三种模板引擎。创建模板时，或在模板库卡片上点击 **使用模板** 时选择引擎；**之后无法更改**，如需切换引擎，请新建模板。

<FeatureGrid>
  <FeatureCard icon="code" title="无逻辑 HTML（推荐）">
    Mustache 风格 <code>{"{{variable}}"}</code> 普通 HTML 中的占位符。客户端渲染，因此收据**可离线使用**。用于浏览器打印和 PDF 收据。
  </FeatureCard>
  <FeatureCard icon="print" title="热敏 XML">
    可同时生成屏幕预览和热敏收据打印机所需**ESC/POS 命令**的 XML 模板。同一模板，两种输出。适用于 Epson、Star 和其他热敏打印机。
  </FeatureCard>
  <FeatureCard icon="terminal" title="旧版 PHP">
    使用 WooCommerce 函数的服务端 PHP 模板。为兼容现有的 <code>yourtheme/woocommerce-pos/receipt.php</code> 覆盖文件而保留。渲染时需要连接服务器。
  </FeatureCard>
</FeatureGrid>

## 模板库 {#template-gallery}

模板库提供可直接使用或自定义的现成模板。可按**类别**（收据、发票、礼品收据、厨房订单、报价单／采购订单）、**格式**（HTML／ESC/POS）和**方向**（LTR／RTL）筛选，并使用店铺的真实数据预览。每张卡片的页脚都有**预览**链接和主要的**使用模板**按钮——点击“使用模板”会在*你的模板*中创建一个可编辑副本。

### 第三方文档模板 {#third-party-document-templates}

如果已启用 WP Overnight 的 **PDF Invoices & Packing Slips for WooCommerce**，WCPOS 还会在收据模板列表中提供 **发票（WP Overnight）** 和 **装箱单（WP Overnight）**。这些由服务器渲染的 HTML 模板会调用 WP Overnight 的文档 API，并复用该插件中配置的发票和装箱单编号、布局、品牌、法律/税务字段以及模板自定义设置。

仅当 WP Overnight 插件处于启用状态时才能使用这些模板，并且渲染时需要连接服务器。如需离线打印，请使用 WCPOS 内置的 HTML 或热敏模板。

### 自适应税费显示 {#adaptive-tax-display}

大多数内置模板（标准、标准 RTL、极简/现代、窄版、发票、报价单、热敏简洁变体）会**自动适应 WooCommerce 的税费显示设置**：

- **含税店铺**（欧盟/英国/澳大利亚）显示含税价格，并附有“含税”行。
- **不含税店铺**（美国/加拿大）显示未含税价格，并将税额作为单独一行添加。
- 总计始终显示实际收取的含税金额。

**详细**系列为正式税务发票，无论店铺设置如何，始终按未含税价格逐项列出税额并提供明确明细。

## 工作原理 {#how-it-works}

1. **添加模板**：在图库卡片中点击**使用模板**，或从头创建自己的模板。
2. 使用应用内编辑器和实时预览进行**自定义**，或将其粘贴到 ChatGPT / Claude，并说明所需的修改。
3. **启用模板**：在*你的模板*中打开**已启用**开关。每个已启用的收据模板都会显示在收银台收据界面的下拉列表中。
4. **打印或显示**：结账后会打开收据，可选择打印、通过电子邮件发送或在屏幕上查看。模板也可用于收银台以外的场景：在线客户可[从“我的账户”下载收据](/receipts/storefront)，并使用相同的设计呈现。

模板根据标准化的[收据数据载荷](/receipts/receipt-data)呈现，其中包含店铺信息、商品行、总计、税费、付款、退款等部分。所有货币字段均包含预先格式化的 `_display` 变体（例如，`$29.99` 而非 `29.99`）。

## 专业版功能 {#pro-features}

使用 [WCPOS Pro](/getting-started/pro-license) 后，每个店铺都可拥有自己的模板分配、品牌设置（徽标、地址、联系方式）和模板排序。这适用于多地点运营的场景，其中每家店铺都会打印自己的信头。
