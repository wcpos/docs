:::caution
複数在庫は現在Proプラグインの機能ではありません。
以下の情報は、WCPOS Proが正式に複数在庫をサポートするまでのユーザー支援を目的としています。
:::

複数在庫店舗は、WCPOS Proにおける最も要望の多い機能の1つです。
この機能により、店舗オーナーは製品を異なる店舗に割り当て、各物理店舗とオンラインストア間の在庫を管理できるようになります。

複数在庫機能の作成は、単純なタスクではありません。
この問題に取り組んでいるプラグインがすでにいくつかあります。例えば、[WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646)や[WooCommerce MultiStore](http://woomultistore.com/)などです。
現時点では、他のコアPOS機能の開発に注力し、これらのプラグインが安定したソリューションに成熟した後に統合することが理にかなっています。

短期的には、[WordPress マルチサイト機能](https://codex.wordpress.org/Create_A_Network)を使用して、ほとんどの要件を満たすことができます。
各店舗のサイトを作成し、各サイトにWooCommerceとWCPOSをインストールすることで、各店舗の在庫を完全に分離できます。
上記のWooCommerce Multisite（前述）は、サイト間の在庫管理のための追加レイヤーを使用するこのアプローチを採用しています。
