---
title: "マルチ在庫ストア"
---

:::caution
マルチ在庫は現在、Proプラグインの機能ではありません。
以下の情報は、WCPOS Proが正式にマルチ在庫をサポートするまで、ユーザーを支援することを目的としています。
:::

マルチ在庫ストアは、WCPOS Proで最もリクエストされている機能の1つです。
この機能により、ストアのオーナーは製品を異なるストアに割り当て、それぞれの実店舗とオンラインストアの間で在庫を管理することができます。

マルチ在庫機能を作成することは、単純ではないタスクです。
現在、[WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646)や[WooCommerce MultiStore](http://woomultistore.com/)など、この問題に取り組んでいるプラグインがいくつかあります。
現在、他のコアPOS機能の開発に集中し、これらのプラグインが安定したソリューションとなった時に統合することが理にかなっています。

短期的には、[WordPress MultiSite機能](https://codex.wordpress.org/Create_A_Network)を使用してほとんどの要件を満たすことができるかもしれません。
各ストアのためにサイトを作成することで、各サイトにWooCommerceとWCPOSをインストールし、各ストアの在庫を完全に分離することができます。
上で言及したWooCommerce Multisiteは、このアプローチを使用し、サイト間での在庫管理のための追加のレイヤーを利用しています。
