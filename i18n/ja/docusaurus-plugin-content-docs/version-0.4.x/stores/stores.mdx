---
title: ストア
---

:::important
この機能は、[WCPOS Pro](http://wcpos.com/pro) へのアップグレードが必要です。
:::

![ストアリストの例](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "ストアリストの例")

WCPOS Pro は、物理的な販売のために別々の _ストア_ を作成する機能を追加します。
ストアには、それぞれの住所、ロゴ、その他の詳細を [領収書テンプレート](../receipts) 用に設定できます。
ストアは異なる税率を使用でき、[レポーティング](../reports#sales-by-store) のためにストアごとに販売を分けることができます。

![ストア編集ページの例](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "ストア編集ページの例")

### 今後の機能 {#coming-features}

ストア機能はバージョン 0.4 で追加されました。
[マルチインベントリストア](./multi-inventory) やストアにキャッシャーを割り当てる機能など、いくつかの _頻繁にリクエストされる機能_ があります。
これらの機能は、プラグインがバージョン 1.0 に近づくにつれて Pro プラグインに追加される予定です。
