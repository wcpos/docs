:::important
この機能は[WCPOS Pro](http://wcpos.com/pro)へのアップグレードが必要です。
:::

![店舗一覧の例](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "店舗一覧の例")

WCPOS Proは、物理店舗の_店舗_を個別に作成する機能を追加します。
店舗ごとに独自の住所、ロゴ、その他の詳細を[レシートテンプレート](../receipts)で使用できます。
店舗ごとに異なる税率を設定でき、売上を[店舗別に報告](../reports#sales-by-store)することができます。

![店舗編集ページの例](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "店舗編集ページの例")

### 今後の機能

店舗機能はバージョン0.4で追加されました。
[マルチ在庫店舗](./multi-inventory)やキャッシャーを店舗に割り当てる機能など、いくつかの_頻繁にリクエストされる機能_があります。
これらの機能は、プラグインがバージョン1.0に近づくにつれてProプラグインに追加される予定です。
