---
title: セキュリティ
description: 顧客情報のセキュリティを確保することはあなたの責任です
---

WCPOS は、インストールされている [WooCommerce ゲートウェイ](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/) を使用して支払いを行うことができます。支払いゲートウェイが個人情報を収集する場合、**顧客の情報のセキュリティを確保するのはあなたの責任です**。以下は、POSを安全に保つためのヒントです：

### SSL証明書を使用する

オンラインストアを運営している場合は、ウェブサイトのセキュリティ証明書が必要です。
これにより、ログインやクレジットカード番号などのフォームデータが暗号化されます。
セキュリティ証明書を使用しない場合、インターネット上で送信される情報は傍受され、読み取られる可能性があります。これは、ネットワークを共有している誰かや、あなたのコンピュータとサーバーを接続する数十のノードのいずれかによって行われる可能性があります。
基本的なセキュリティ証明書は[わずか16米ドル](https://www.servertastic.com/rapidssl/)で、設定にも1日もかかりません。セキュリティ証明書を持たない言い訳はありません。

### POSを放置しない

インターネット上での盗聴はセキュリティが侵害される一つの方法ですが、多くの場合、実際の被害は悪意のあるユーザーがデバイスに物理的にアクセスしたときに発生します。
これは、コンピュータやタブレットを人通りの多いエリアに置くことが多い店主に特に当てはまります。
最悪の事態、つまりコンピュータやデバイスが盗まれることを想定し、以下のようなセキュリティ対策を講じてリスクを最小限に抑えましょう：

- デバイスを5分または10分放置する場合はロック画面を使用する
- デバイスを離れる際はWordPressとWCPOSからログアウトする
- デバイスが盗まれた場合はすぐにパスワードをリセットする
- デバイスが盗まれた場合は遠隔でワイプする

### できるだけ少ないプラグインを使用する

WordPressの素晴らしい点の1つは、ボタンクリック1つで利用できる何千ものプラグインです...しかし、それは同時に最悪の点でもあります。
プラグインをインストールするたびに、そのプラグインにウェブサイトと通過するデータへのアクセスを事実上許可することになります。
すべてのプラグインで店をカスタマイズしたくなる誘惑に駆られますが、多くの場合、サイトの速度を低下させ、セキュリティを脅かす可能性があります。
どのプラグインをインストールする前に、自問自答してください：

1. 本当に必要か？
2. プラグイン作成者を信頼できるか？

WooCommerceストアのセキュリティについてさらに詳しく知りたい場合は、WooThemesウェブサイトの[この素晴らしい記事](http://www.woothemes.com/2013/09/improve-your-wordpress-security-with-these-10-tips/)をお読みください。
