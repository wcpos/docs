---
title: レガシーサーバー
description: Web ホストが最新のRESTful HTTPメソッドをブロックしている場合、レガシーサーバーサポートを有効にする必要があります。
---

HTTP（および HTTPS）は、すべてのブラウザがWeb アプリケーションと通信するために使用するプロトコルです。
プロトコルの重要な部分は [HTTPメソッド](https://tools.ietf.org/html/rfc2616#section-5.1.1) です。
従来、Webアプリケーションは、GET（Webページの取得）やPOST（Webフォームの送信）などの非常に限定されたHTTPメソッドのセットを使用していました。

WCPOS のような最新のWebアプリケーションは、以下のHTTPメソッドを使用するREST/HTTPアプローチを採用しています：

- GET
- POST
- PUT
- PATCH
- DELETE

まれなケースとして、WebホストがこれらのHTTPメソッドの一部をブロックすることがあります。
Webホストが最新のRESTful HTTPメソッドをブロックしている場合、レガシーサーバーサポートを有効にする必要があります。
