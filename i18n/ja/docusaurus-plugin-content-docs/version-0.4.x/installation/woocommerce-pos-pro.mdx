:::important

[WCPOS Pro](http://wcpos.com/pro) は、有料プラグインで、無料の [WCPOSプラグイン](http://wordpress.org/plugins/woocommerce-pos/) に機能を追加します。

:::

WCPOS Proのライセンスを購入された場合、以下の手順に従ってプラグインをインストールおよびアクティベートしてください：

1.  [https://wcpos.com/my-account/](https://wcpos.com/my-account/) にアクセスします。
2.  **ダウンロード** の下にあるダウンロードリンクをクリックし、プラグインをデスクトップに保存します。
3.  次に、サイトにログインし、`WP Admin > プラグイン > 新規追加 > プラグインのアップロード`に移動します。
4.  デスクトップからプラグインのzipファイルをアップロードし、アクティベートします。
5.  次に、`WP Admin > POS > 設定 > ライセンス` に移動し、ライセンスキーとライセンスメールを入力してアクティベーションを完了します。

![My Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "アカウントページからWCPOS Proをダウンロードできます")

新しいバージョンが準備できると、あなたの更新ダッシュボードに表示されます。
私が現在作業している内容については、[ロードマップ](http://wcpos.com/roadmap/)を確認してください。
また、各バージョンの新機能を一覧表示する変更ログにも目を光らせておき、[ドキュメントページ](http://docs.wcpos.com)も新機能の使用方法に関する情報で更新されます。
