:::important

[WCPOS Pro](http://wcpos.com/pro)は、無料の[WCPOSプラグイン](http://wordpress.org/plugins/woocommerce-pos/)に機能を追加する有料プラグインです。

:::

WCPOS Proのライセンスを購入された方は、以下の手順に従ってプラグインをインストールおよびアクティベートしてください：

1.  以下にアクセス：[https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  **ダウンロード**の下で、ダウンロードリンクをクリックし、プラグインをデスクトップに保存します。
3.  サイトにログインし、`WP管理画面 > プラグイン > 新規追加 > プラグインアップロード`に移動します
4.  デスクトップからプラグインのzipファイルをアップロードし、アクティベートします。
5.  次に、`WP管理画面 > POS > 設定 > ライセンス`に移動し、ライセンスキーとライセンスメールを入力して、アクティベーションを完了します。

![マイダウンロード](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "アカウントページでWCPOS Proをダウンロードできます")

新しいバージョンの準備ができると、更新ダッシュボードに表示されます。
現在取り組んでいる内容については、[ロードマップ](http://wcpos.com/roadmap/)をご確認ください。
また、変更ログで各バージョンの新機能を確認し、[ドキュメントページ](http://docs.wcpos.com)で新機能の使用方法に関する情報が更新されます。
