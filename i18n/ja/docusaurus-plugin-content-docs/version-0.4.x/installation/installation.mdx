### 自動インストール（推奨）

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "1分以内にWCPOSをインストールして有効化")

1. `WP管理画面 > プラグイン > 新規追加`に移動
2. WordPressプラグインディレクトリで`WCPOS`を検索
3. プラグインをインストール
4. プラグインを有効化

### 手動インストール

1. [WCPOSをダウンロード](https://wordpress.org/plugins/woocommerce-pos/)（WordPress.org）
2. `WP管理画面 > プラグイン > 新規追加 > プラグインをアップロード`に移動
3. プラグインを有効化

:::info
WCPOSの以前のバージョンは、WordPress.orgの[開発者](https://wordpress.org/plugins/woocommerce-pos/developers/)タブで見つけられます。
:::
