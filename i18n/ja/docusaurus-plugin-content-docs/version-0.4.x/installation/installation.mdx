### 自動インストール（推奨） {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "1分以内にWCPOSをインストールして有効化")

1. `WP Admin > プラグイン > 新規追加` に移動します
2. WordPressプラグインディレクトリで `WCPOS` を検索します
3. プラグインをインストールします
4. プラグインを有効化します

### 手動インストール {#manual-installation}

1. WordPress.orgから[WCPOSをダウンロード](https://wordpress.org/plugins/woocommerce-pos/)します
2. `WP Admin > プラグイン > 新規追加 > プラグインのアップロード` に移動します
3. プラグインを有効化します

:::info
以前のバージョンのWCPOSは、WordPress.orgの[開発者](https://wordpress.org/plugins/woocommerce-pos/developers/)タブから入手できます。
:::
