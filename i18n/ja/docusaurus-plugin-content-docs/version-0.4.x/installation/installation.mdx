### 自動インストール（推奨）

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "WCPOSを1分未満でインストールしてアクティブ化します")

1. `WP Admin > プラグイン > 新規追加` に移動
2. WordPressプラグインディレクトリで `WCPOS` を検索
3. プラグインをインストール
4. プラグインをアクティブ化

### 手動インストール

1. [WCPOSをダウンロード](https://wordpress.org/plugins/woocommerce-pos/)します
2. `WP Admin > プラグイン > 新規追加 > プラグインをアップロード` に移動
3. プラグインをアクティブ化

:::info
WCPOSの以前のバージョンは、Wordpress.orgの[開発者](https://wordpress.org/plugins/woocommerce-pos/developers/)タブで見つけることができます。
:::
