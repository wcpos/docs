WCPOSは、マウスまたはタッチスクリーンを使用して迅速な入力を可能にする、さまざまな数字パッドコンポーネントを使用しています。
以下は、異なるタイプの数字パッドの例です：

![+1と-1の迅速な入力が可能な数量数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "数量数字パッドに+1と-1の迅速な入力")

![割引率の迅速な入力が可能な商品価格数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "商品価格数字パッドに割引率の迅速な入力")

![よくある現金額の迅速な入力が可能な受取金額数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "受取金額数字パッドによくある現金額の迅速な入力")

![よくある釣銭額の迅速な入力が可能な釣銭数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "釣銭数字パッドによくある釣銭額の迅速な入力")
