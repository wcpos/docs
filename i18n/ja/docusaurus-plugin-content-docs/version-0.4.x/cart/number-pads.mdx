WCPOSは、マウスまたはタッチスクリーンを使用した迅速な入力を可能にするために、いくつかの数字パッドコンポーネントを使用しています。  
以下は、さまざまな種類の数字パッドの例です：

![数量入力用の+1および-1のクイックエントリーのある数量数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "数量入力用の+1および-1のクイックエントリーのある数量数字パッド")

![割引率のクイックエントリーのある商品価格数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "割引率のクイックエントリーのある商品価格数字パッド")

![可能な現金金額のクイックエントリーのある支払金額数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "可能な現金金額のクイックエントリーのある支払金額数字パッド")

![一般的な現金バック金額のクイックエントリーのあるキャッシュバック数字パッド。](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "一般的な現金バック金額のクイックエントリーのあるキャッシュバック数字パッド")
