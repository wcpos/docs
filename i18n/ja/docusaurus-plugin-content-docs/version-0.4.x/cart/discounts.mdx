---
title: 割引
description: WCPOSでは、販売時にすぐに割引を適用できます。
---

### 商品価格

商品価格をクリックすると、その場で価格を変更できます。
商品価格の数字パッドを使用して、新しい販売価格またはパーセンテージでの割引を入力できます。

![数字パッドを使用して商品価格をパーセンテージで削減する](https://wcpos.com/wp-content/uploads/2016/08/product-discount.png "数字パッドを使用して商品価格をパーセンテージで削減する")

### 注文割引

注文割引を適用するには、カートにマイナスの手数料を追加します。
タイトルをクリックして手数料の名前を変更できます。
矢印をクリックして、税のステータスと税率を変更できます。

![注文割引の数字パッド](https://wcpos.com/wp-content/uploads/2016/08/negative-fee.png "注文割引を適用するためのマイナス手数料の使用")
