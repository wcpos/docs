---
title: 割引
description: WCPOSでは、販売時点で割引を迅速に適用できます。
---

### アイテム価格

アイテム価格をクリックすると、リアルタイムで価格を変更できます。
アイテム価格の数字パッドを使用して、新しい販売価格またはパーセンテージとしての割引を入力できます。

![数字パッドを使用してアイテム価格をパーセンテージで減少させる](https://wcpos.com/wp-content/uploads/2016/08/product-discount.png "数字パッドを使用してアイテム価格をパーセンテージで減少させる")

### 注文割引

カートに負の手数料を追加することで、注文割引を適用できます。
手数料の名前を変更するには、タイトルをクリックします。
税のステータスと税率を変更するには、矢印をクリックします。

![注文割引の数字パッド](https://wcpos.com/wp-content/uploads/2016/08/negative-fee.png "負の手数料を使用して注文割引を適用する")
