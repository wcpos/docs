ホットキーは、WCPOS内のアクションをすばやく実行するためのキーボードショートカットです。

| キー                         | 説明      |
| --------------------------- | ---------------- |
| <div align="center">?</div> | ヘルプ画面      |
| <div align="center">B</div> | バーコードモード     |
| <div align="center">V</div> | 検索モード      |
| <div align="center">S</div> | サーバーと同期 |
