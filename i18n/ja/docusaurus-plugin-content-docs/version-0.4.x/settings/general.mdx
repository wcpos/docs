### 商品の表示 {#product-visibility}

POS専用商品は、オンラインストアまたは実店舗のいずれかから商品を非表示にする必要がある場合にのみ有効にする必要があります。
詳細については、[POS専用商品](../products/pos-only-products)ドキュメントをご覧ください。

### デフォルトPOS顧客 {#default-pos-customer}

新しい注文はデフォルトのPOS顧客で初期化されます。一般的なオプションは次のとおりです:

- **ゲスト:** POS注文はユーザーIDに関連付けられていません
- **キャッシャー:** POS注文はログインしているユーザーのアカウントに関連付けられています
- **来店顧客:** POS注文はカスタムユーザーアカウントに関連付けられています（_注: このアカウントは作成する必要があります_）

[WCPOS Pro](https://wcpos.com/pro)ユーザーは、チェックアウト中にユーザーアカウントを即座に作成できます。
新しいアカウントを作成するにはメールアドレスが必要で、これはWordPressの要件です。
**新しい顧客のためにアカウントを作成することをお勧めします**。これは、顧客の注文履歴を追跡し、将来的にプロモーションオファーで顧客に連絡する可能性を意味します。
