POSアクセス設定を使用することで、店舗管理者はポイントオブセール（POS）システムのユーザー権限を付与または削除できます。
デフォルトでは、**管理者**と**ショップマネージャー**の役割がPOSを使用するためのすべての機能で有効になっています。
影響を理解していない限り、デフォルト設定を維持することをお勧めします。
例えば、`購読者`の役割に`publish_shop_orders`を付与することは、_非常に_悪いアイデアです。

![POSアクセス設定の例](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "POSアクセス設定の例")

| 機能                                                                    | アクセス                                 |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | POS画面にアクセスするために必要           |
| manage_woocommerce_pos                                                  | POS管理設定にアクセスするために必要       |
| read_private_products                                                   | 商品を表示するために必要                 |
| edit_product <br /> edit_published_products <br /> edit_others_products | 商品とバリエーションを編集するために必要  |
| read_private_shop_orders                                                | 注文と領収書を表示するために必要          |
| publish_shop_orders                                                     | 新しい注文を作成するために必要            |
| list_users                                                              | 顧客を表示するために必要                 |
| create_users                                                            | 新しい顧客を作成するために必要            |
| edit_users                                                              | 既存の顧客を更新するために必要            |
| read_private_shop_coupons                                               | クーポンを表示するために必要              |

**参考資料:**

- [WooCommerce の役割と権限](http://docs.woothemes.com/document/roles-capabilities/)に関する詳細情報
