WCPOSは[WooCommerce](https://www.woothemes.com/woocommerce/)用のPOSプラグインです。
このドキュメントでは、オープンソースプラグインと、無料プラグインに機能を追加するProプラグインについて説明します。

- WordPress.orgから[WCPOS](https://wordpress.org/plugins/woocommerce-pos/)をダウンロード
- [WCPOS Pro](http://wcpos.com/pro)にアップグレード

:::danger

ドキュメントのバージョン番号が、インストール済みプラグインのバージョンと一致していることを確認してください。

:::
