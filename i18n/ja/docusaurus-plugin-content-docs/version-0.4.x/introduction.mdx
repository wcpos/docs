WCPOSは[WooCommerce](https://www.woothemes.com/woocommerce/)用のPoint of Saleプラグインです。
このドキュメントは、オープンソースプラグインと、無料プラグインに機能を追加するPro版プラグインの両方をカバーしています。

- [WCPOS](https://wordpress.org/plugins/woocommerce-pos/)をWordPress.orgからダウンロード
- [WCPOS Pro](http://wcpos.com/pro)にアップグレード

:::danger

インストールされているプラグインのバージョンと、ドキュメントのバージョン番号が一致していることを確認してください。

:::

### その他のドキュメント

**よくある質問（FAQ）**は[faq.wcpos.com](http://faq.wcpos.com)で利用可能です。
**開発者向けドキュメント**は[api.wcpos.com](http://api.wcpos.com)で利用可能です。

### ヘルプとサポート

このドキュメントやFAQでカバーされていない質問がある場合は、[support@wcpos.com](mailto:support@wcpos.com)までお問い合わせください。
