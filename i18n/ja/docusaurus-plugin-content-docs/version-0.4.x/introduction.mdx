WCPOSは、[WooCommerce](https://www.woothemes.com/woocommerce/)のための販売時点管理プラグインです。このドキュメントでは、オープンソースプラグインと、無料プラグインに機能を追加するProプラグインについて説明します。

- WordPress.orgから[WCPOS](https://wordpress.org/plugins/woocommerce-pos/)をダウンロード
- [WCPOS Pro](http://wcpos.com/pro)にアップグレード

:::danger

ドキュメントのバージョン番号がインストールされているプラグインのバージョンと一致することを確認してください。

:::
