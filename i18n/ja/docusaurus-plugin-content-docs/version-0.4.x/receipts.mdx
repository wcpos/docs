### Proレシートテンプレート

Proレシートテンプレートでは、[ストア](./stores)の管理画面からロゴ、店舗住所、営業時間、特別メッセージなどの追加情報を含めることができます。

### レシートの日付のカスタマイズ

WCPOSは[moment.js](http://momentjs.com/)を使用して日付文字列をローカライズします。デフォルトの日付形式は `"MMMM Do YYYY, h:mm:ss a"` です。例：5月31日 2015年、7:20:44 pm。日付形式のカスタマイズについては、[moment.jsのドキュメント](http://momentjs.com/docs/#/parsing/string-format/)を参照してください。

### 注文のプロパティ

WCPOSは、WC REST APIからのjsonアウトプットを使用して注文レシートテンプレートを作成します。[WC REST APIのドキュメント](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order)に標準的なjsonアウトプットの例が示されています。WCPOSは `woocommerce_api_order_response` フィルターを使用して、いくつかの追加のプロパティを追加しています。

| プロパティ                          | 説明                                                   |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | カートディスカウントの税金部分                          |
| cashier.id                        | キャッシャーのユーザーID                                        |
| cashier.first_name                | キャッシャーの名前                                     |
| cashier.last_name                 | キャッシャーの苗字                                     |
| payment_details.result            | 支払いゲートウェイの成功または失敗                            |
| payment_details.message           | 支払いゲートウェイからのメッセージ                         |
| payment_details.redirect          | オフサイト決済のリダイレクトURL（例：PayPalスタンダード）        |
| payment_details.method\_`gateway` | ゲートウェイ固有のメッセージ（例：入金額と釣銭）                 |
| shipping_lines[i].total_tax       | 各配送ラインの税金額を追加                         |
| subtotal_tax                      | 小計の税金部分                               |
