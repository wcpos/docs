---
title: POSのみの商品
description: POSの可視性設定を使用して、オンラインストアまたは販売時点で商品を非表示にできます。
---

オンラインストアまたはPOSのいずれかで商品を非表示にする必要がある状況があるかもしれません。
WCPOSでは、商品編集ページのPOS可視性設定を使用してこれを実現できます。

### POSのみの商品の有効化

POSのみの商品はデフォルトでは有効になっていません。
POSのみの商品を有効にするには、<code>WP管理画面 > POS > 設定 > 一般</code>に移動します

![POSのみの商品を有効化](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "POSのみの商品を有効化")

POSのみの商品を有効にすると、追加のデータベース検索が必要になり、わずかなパフォーマンスの低下が発生します。
POSのみの商品を使用しない場合は、このオプションを無効のままにしておくことをお勧めします。

### POS & オンラインの可視性

![POS可視性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "商品編集ページのPOS可視性設定")

| 設定                            | 説明                                                                       |
| ------------------------------- | --------------------------------------------------------------------------------- |
| **POS & オンライン**<br />（デフォルト） | 商品はオンラインのウェブストアと実店舗の両方で表示されます |
| **POSのみ**                    | 商品ページはオンラインでは表示されません                                           |
| **オンラインのみ**                 | 商品はPOSでは表示されません                                            |

### カタログの可視性

POSのみの商品ページに直接アクセスすると、<code>404ページが見つかりません</code>と表示されます。
ただし、POSのみの商品は、オンラインの検索やカテゴリーページに表示される場合があります。
商品を完全にウェブサイトから非表示にするには、カタログの可視性を「非表示」に設定する必要があります。

![カタログの可視性](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "商品編集ページのカタログ可視性設定")

### POSのみの商品をフィルタリング

商品管理ページのクイックリンクを使用して、どの商品が表示/非表示になっているかを簡単に確認できます：

![POS可視性フィルター](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "POS可視性フィルター")

### 一括編集

複数の商品のPOS可視性を変更するには、商品リストに移動し、<code>一括操作</code>から<code>編集</code>を選択します。

![一括編集POS可視性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "POS可視性の一括編集")
