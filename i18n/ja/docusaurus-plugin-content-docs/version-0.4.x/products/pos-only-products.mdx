---
title: "POS専用商品"
description: 商品は、オンラインストアまたはPOSでの可視性設定を使用して非表示にできます。
---

オンラインストアまたはPOSから商品を隠す必要がある状況があるかもしれません。
WCPOSでは、商品編集ページのPOS可視性設定を使用してこれを実現できます。

### POS専用商品の有効化

POS専用商品はデフォルトでは有効になっていません。
POS専用商品を有効にするには、<code>WP Admin > POS > 設定 > 一般</code>に移動します。

![POS専用商品の有効化](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "POS専用商品の有効化")

POS専用商品を有効にすると、データベースの追加検索が必要になり、わずかなパフォーマンスの影響が生じます。
POS専用商品を使用しない場合は、このオプションを無効のままにしておくことをお勧めします。

### POSとオンラインの可視性

![POS可視性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "商品編集ページのPOS可視性設定")

| 設定                             | 説明                                                                       |
| ------------------------------- | --------------------------------------------------------------------------------- |
| **POSとオンライン**<br />(デフォルト) | 商品はオンラインのウェブストアと物理ストアの両方で可視化されます              |
| **POS専用**                      | 商品ページはオンラインでは表示されません                                    |
| **オンライン専用**               | 商品はPOSでは表示されません                                                    |

### カタログの可視性

POS専用商品ページに直接移動すると、<code>404 ページが見つかりません</code>が表示されます。
ただし、POS専用商品はオンライン検索やカテゴリーページに表示される場合があります。
ウェブサイトから商品を完全に隠すには、カタログの可視性を非表示に設定する必要があります。

![カタログの可視性](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "商品編集ページのカタログ可視性設定")

### POS専用商品のフィルタリング

商品管理ページのクイックリンクを使用すると、どの商品の可視性があるか/非表示かをすぐに確認できます：

![POS可視性フィルター](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "POS可視性フィルター")

### バルク編集

複数の商品を一度にPOSの可視性を変更するには、商品リストに移動し、<code>編集</code>を<code>一括アクション</code>から選択します。

![バルク編集POS可視性](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "バルク編集POS可視性")
