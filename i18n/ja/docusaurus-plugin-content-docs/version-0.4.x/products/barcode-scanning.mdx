---

WCPOSはSKUフィールドで商品を絞り込むことができます。検索がバーコードフィールドと完全に一致すると、商品が即座にカートに追加され、検索フィールドがリセットされます。検索フィールドに入力できるバーコードスキャナーをお持ちであれば、WCPOSでバーコードスキャンをご利用いただけます。

SKU検索の動作を確認するには、http://demo.wcpos.com/pos/ をご覧ください。

`BARCODE` と入力（またはコピー＆ペースト）すると、商品がカートに追加されます。

`VARIATION1` と入力（またはコピー＆ペースト）すると、バリエーションがカートに追加されます。

![WCPOSでのバーコードスキャン](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "WCPOSでのSKUフィールドによる絞り込み")

以下の機能は今後のリリースで実装が予定されています：

- バーコードスキャナーを使用したWooCommerceへの商品追加。この機能は0.3.xリリースで予定されている棚卸しモードの一部として実装されます。
- モバイルデバイスのカメラを使用したバーコードスキャン。この機能にはデバイスのハードウェアにアクセスするためのネイティブWCPOSアプリが必要です。ネイティブアプリは要望リストに含まれていますが、現時点ではWeb版への機能追加が最優先事項です。
- ホットキーによる起動

### 更新情報 {#updates}

**バージョン 0.3.1 以降：** バーコードスキャン専用モードが追加されました。検索フィールドの左側にあるボタンをクリックすることで、「検索」モードと「バーコードスキャン」モードを切り替えることができます。
