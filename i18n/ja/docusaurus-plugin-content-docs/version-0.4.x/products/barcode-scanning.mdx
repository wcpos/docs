---

WCPOSはSKUフィールドで商品をフィルタリングできます。検索がバーコードフィールドと完全に一致する場合、商品は即座にカートに追加され、検索フィールドはリセットされます。バーコードスキャナーが検索フィールドに値を入力できる場合、WCPOSでバーコードスキャニングを使用できます。

バーコード検索の動作を確認するには、http://demo.wcpos.com/pos/ にアクセスしてください。

`BARCODE` を入力（またはコピー＆ペースト）すると、商品がカートに追加されます。

`VARIATION1` を入力（またはコピー＆ペースト）すると、バリエーション商品がカートに追加されます。

![WCPOSのバーコードスキャニング](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "WCPOSのSKUフィールドでのフィルタリング")

今後のリリースで計画されている機能は以下の通りです：

- バーコードスキャナーを使用してWooCommerceに商品を追加。これは0.3.x版でリリース予定の棚卸モードの一部となります。
- モバイルデバイスのカメラを使用したバーコードスキャン。これには、デバイスのハードウェアにアクセスするネイティブWCPOSアプリが必要になります。ネイティブアプリは希望リストにありますが、現時点ではウェブバージョンへの機能追加が最優先です。
- ホットキーアクティベーション

### アップデート

**バージョン0.3.1以降：** バーコードスキャニング用の専用モードが追加されました。検索フィールドの左側のボタンをクリックすることで、「検索」モードと「バーコードスキャン」モードを切り替えることができます。
