---
title: 検索とフィルタリング
description: 任意の属性（ID、カテゴリ、タイプなど）で商品を検索フレーズでフィルタリングできます。
---

WCPOSでは、商品をタイトルや属性で検索できます。例えば、[デモサイト](http://demo.wcpos.com/pos/)で`woo`を検索すると、タイトルでフィルタリングされ、`cat:posters`を検索するとポスターカテゴリの商品がフィルタリングされ、`cat:posters woo`を検索するとポスターカテゴリかつタイトルに'woo'を含む商品がフィルタリングされます。映画ポスターカテゴリを選択したい場合は、`cat:"movie posters"`を使用できます。
