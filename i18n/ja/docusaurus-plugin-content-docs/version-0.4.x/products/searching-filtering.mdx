---
title: "検索とフィルタリング"
description: "任意の属性（ID、カテゴリ、タイプなど）によって製品をフィルタリングするために検索フレーズを使用できます。"
---

WCPOSを使用すると、タイトルや属性によって製品を検索できます。たとえば、[デモサイト](http://demo.wcpos.com/pos/)では、`woo`の検索はタイトルでフィルタリングされ、`cat:posters`の検索はポスターカテゴリによるフィルタリングを行い、`cat:posters woo`の検索はタイトルに'woo'を含むポスターカテゴリの製品をフィルタリングします。ムービーポスターカテゴリを選択したい場合は、`cat:"movie posters"`を使用できます。
