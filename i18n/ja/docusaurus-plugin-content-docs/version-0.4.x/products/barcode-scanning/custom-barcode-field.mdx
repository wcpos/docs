デフォルトでWCPOSはSKUフィールドをバーコードとして使用します。

![デフォルトでWCPOSはSKUフィールドをバーコードとして使用します。](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "デフォルトでWCPOSはSKUフィールドをバーコードとして使用します")

[WCPOS Pro](http://wcpos.com/pro)ユーザーは、`WP管理画面 > POS > 設定 > 全般`でバーコードフィールドを選択することで、任意の商品メタフィールドを使用できます。

![WCPOSプロでカスタムバーコードフィールドを選択](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "WCPOSプロでカスタムバーコードフィールドを選択")

カスタムバーコードメタフィールドは、`woocommerce_pos_barcode_meta_key`フィルターを使用しても設定できます。

```php title="functions.php"
/**
 * バーコードとして使用するカスタム商品メタフィールド
 *
 * デフォルトでWCPOSはSKUフィールド（'_sku'）を使用します
 * 次のWordPressフィルターを使用してメタキーを変更できます
 *
 * 以下は人気のバーコードプラグインのメタフィールドです
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // カスタムメタキーに変更
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
バーコードは各商品で一意である必要があります。
:::
