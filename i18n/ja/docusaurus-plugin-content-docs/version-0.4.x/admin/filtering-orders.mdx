WooCommerceの管理画面の左上にある簡易リンクを使用して、POSとオンライン注文をすばやく分けることができます。
**注意:** これは0.3.1バージョン以降のPOS注文のみをフィルタリングします。0.3.1より前の注文は、デフォルトでオンラインとして表示されます。

![注文のフィルタリング](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "注文のフィルタリング")

さらに、管理エリアで完了した注文をすべてフィルタリングし、保留中/処理中の注文に集中したい場合は、[functions.php](http://codex.wordpress.org/Functions_File_Explained)ファイルに次のコードを追加できます。

```php title="functions.php"
/**
 * WP管理画面での注文のデフォルトフィルターを変更
 * functions.phpファイルに追加してください
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
