---

WooCommerce管理画面の左上にあるクイックリンクを使用して、POS注文とオンライン注文をすばやく分けることができます。
**ご注意:** この機能はバージョン0.3.1以降のPOS注文のみをフィルタリングします。0.3.1より前の注文はオンラインとして扱われます。

![注文のフィルタリング](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

さらに、保留中や処理中の注文に集中するために、完了済みの注文を管理画面からフィルタリングしたい場合は、以下のコードを[functions.php](http://codex.wordpress.org/Functions_File_Explained)ファイルに追加してください。

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
