---
title: "POSまたはオンライン販売による注文のフィルタリング"
description: "WooCommerceの管理画面でクイックリンクを使用してPOSとオンラインの注文を分けます。"
---

WooCommerceの管理画面の左上にあるクイックリンクを使用すると、POSとオンラインの注文をすばやく分けることができます。  
**ご注意:** これはバージョン0.3.1以降のPOS注文のみをフィルターします。0.3.1以前の注文はデフォルトでオンラインになります。

![注文のフィルター](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "注文のフィルター")

さらに、管理エリアからすべての完了した注文をフィルタリングして、保留中/処理中の注文に集中したい場合は、以下のコードを[functions.php](http://codex.wordpress.org/Functions_File_Explained)ファイルに追加できます。

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
