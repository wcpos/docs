title: 決済ゲートウェイ

---

WCPOSはデフォルトで3つの決済ゲートウェイをサポートしています：現金、カード（外部EFTPOS端末）、PayPal。
[WCPOS Pro](http://wcpos.com/pro)ユーザーは、インストールされたWooCommerceゲートウェイを有効化でき、クレジットカード、ビットコイン、[100以上の決済ゲートウェイ](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/)での支払い処理が可能です。

![WCPOSの決済ゲートウェイ設定](https://wcpos.com/wp-content/uploads/2014/09/payment-gateways.png "WCPOSの決済ゲートウェイ設定の例")

決済ゲートウェイは、WooCommerceとは独立して有効化できるため、店舗内では1つの支払い方法を受け入れても、オンラインストアでは必ずしも同じ方法を有効化する必要はありません。
決済の順序やデフォルトの支払い方法も独立して設定できるため、POSのチェックアウト画面をカスタマイズできます。

PayPal Standardなど一部のゲートウェイは、支払いプロセスを完了するために外部のウェブページを使用します。
リダイレクトが必要な場合、WCPOSは新しいウィンドウを開きます。
支払いが完了したら、レシートに戻り「更新」をクリックして支払いが完了したことを確認できます。

:::caution
**注意:** WCPOSの決済ゲートウェイはまだ実験的な機能です。
100以上のゲートウェイがありますが、すべてがすぐに使用できることを保証することはできません。
支払い処理に問題が発生した場合は、[サポート](mailto:support@wcpos.com)にご連絡ください。対応いたします。
:::
