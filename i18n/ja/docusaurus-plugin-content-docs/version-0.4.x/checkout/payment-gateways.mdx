---
title: 支払いゲートウェイ
description: WCPOS Proユーザーは、インストールされたWooCommerceゲートウェイからの支払いを有効化し、受け入れることができます。
---

WCPOSはデフォルトで3つの支払いゲートウェイをサポートしています：現金、カード（外部EFTPOS機）、およびPayPalです。
[WCPOS Pro](http://wcpos.com/pro)ユーザーは、ストアオーナーがクレジットカード、ビットコイン、または[100以上の他の支払いゲートウェイ](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/)を処理できるように、インストールされたWooCommerceゲートウェイを有効化できます。

![WCPOSの支払いゲートウェイ設定。](https://wcpos.com/wp-content/uploads/2014/09/payment-gateways.png "WCPOSの支払いゲートウェイ設定の例")

支払いゲートウェイはWooCommerceとは独立して有効化できるため、店舗での1つの支払いを受け入れることができますが、ウェブストアで必ずしも有効化する必要はありません。
支払いの順序とデフォルトの支払い方法も独立して設定できるため、POSのチェックアウト表示をカスタマイズできます。

PayPal Standardなどの一部のゲートウェイは、支払いプロセスを完了するために外部のWebページを使用します。
リダイレクトが必要な場合、WCPOSは新しいウィンドウを開きます。
支払いが完了すると、領収書に戻り、「更新」をクリックして支払いが完了したことを確認できます。

:::caution
**ご注意ください:** WCPOSの支払いゲートウェイはまだ実験的です。
100以上のゲートウェイが利用可能ですが、すべてがすぐに動作することは保証できません。
支払い処理に問題が発生した場合は、[サポート](mailto:support@wcpos.com)に連絡してください。ゲートウェイが正常に動作するようにお手伝いします。
:::
