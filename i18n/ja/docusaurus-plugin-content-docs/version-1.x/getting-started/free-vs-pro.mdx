WCPOS には2つのエディションがあります。**無料プラグイン**は、レジでの決済、レシート印刷、WooCommerce への注文同期に必要な機能をすべて備えています。**Pro版**では管理画面（商品、注文、顧客、レポート）、より多くの決済ゲートウェイ、クーポン、返金、マルチ店舗対応が追加されます。

主に売上を記録するレジが必要な場合は、無料プラグインで十分です。POS から店舗の日常業務（在庫管理、過去の注文検索、顧客への返金、カードリーダーによるカード決済など）も行いたい場合は、Pro版をご利用ください。

## 概要 {#at-a-glance}

| 項目 | 無料版 | Pro版 |
|---|---|---|
| 販売処理とレシートの印刷・メール送信 | ✓ | ✓ |
| 現金および外部カード決済 | ✓ | ✓ |
| カードリーダー連携（Stripe、SumUp、Vipps MobilePay） | — | ✓ |
| レジでのクーポン利用 | — | ✓ |
| POS からの返金 | — | ✓ |
| POSから在庫、価格、商品詳細を編集 | — | ✓ |
| 過去の注文を検索して再印刷 | — | ✓ |
| 顧客の追加・編集 | — | ✓ |
| 日次レポート | — | ✓ |
| 複数店舗の管理 | — | ✓ |
| 店舗ごとのレシートテンプレート、税ID、ブランディング | — | ✓ |
| 設定からPOS拡張機能をインストール・管理 | — | ✓ |
| Discordでの優先サポート | — | ✓ |

## 無料プラグインの内容 {#what-s-in-the-free-plugin}

[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)の無料プラグインは、基本的な「販売を処理する」ワークフローに対応しています。

<FeatureGrid>
  <FeatureCard icon="cash-register" title="現金・カード決済">
    2つの組み込み決済方法：おつり計算機能付きの**現金**と、外部端末で決済を行い売上のみを記録する場合に使用する**カード**。
  </FeatureCard>
  <FeatureCard icon="cart-shopping" title="カートと商品検索">
    商品パネルの検索、バーコードスキャン、明細の追加、明細ごとの割引適用、カタログにないオープン価格の「その他」商品の追加が可能です。
  </FeatureCard>
  <FeatureCard icon="receipt" title="レシートとサーマル印刷">
    内蔵の[テンプレートギャラリー](/receipts/customise)から選択できます — レシート、請求書、見積書、納品書、ギフトレシート、キッチンチケットなど — ネットワーク、Bluetooth、または USB 経由で 58 mm または 80 mm サーマルプリンターに印刷できます。
  </FeatureCard>
  <FeatureCard icon="sync" title="オフラインストレージ + 同期">
    商品と注文はローカルに保存されるため、レジは高速に動作し、接続が不安定な状況でも使用できます。ネットワークが復旧すると、注文は WooCommerce に同期されます。
  </FeatureCard>
  <FeatureCard icon="barcode" title="バーコード対応">
    USB または Bluetooth スキャナーで商品を直接カートにスキャンできます。デバイスのカメラも使用可能です。
  </FeatureCard>
  <FeatureCard icon="file-invoice" title="顧客の納税者番号">
    顧客レコードに納税者番号フィールドを内蔵 — VAT、ABN、GST、GSTIN、EIN、その他の地域形式に対応しています。
  </FeatureCard>
  <FeatureCard icon="globe" title="クロスプラットフォーム + 多言語対応">
    ブラウザ、デスクトップ（Windows/macOS）、モバイルアプリ（iOS/Android、ベータ版）に対応。ヨーロッパ、アジア、ラテンアメリカの主要言語に翻訳済みです。
  </FeatureCard>
  <FeatureCard icon="puzzle-piece" title="拡張機能カタログ（閲覧のみ）">
    設定から POS 拡張機能と連携機能の一覧を閲覧できます。インストールには Pro が必要です。
  </FeatureCard>
</FeatureGrid>

:::note 無料版での POS 限定の顧客・クーポンアクセス
無料版プラグインでも、既存の顧客を**選択**したり、レジでクーポンカタログを**閲覧**したりすることは可能です。ただし、顧客の追加・編集や会計時のクーポンコード適用はできません。これらの機能は Pro で利用可能になります。
:::

## Pro で追加される機能 {#what-pro-adds}

Pro では、残りのバックオフィス機能、追加の決済オプション、店舗ごとの機能が利用可能になります。以下の項目はすべて Pro 限定です。

### 管理画面 {#management-screens}

無料版プラグインでは、販売を行うための商品パネルが提供されます。Pro では完全な管理画面が追加されます：

| 画面 | できること |
|---|---|
| **[商品](/products)** | 在庫数、価格、売上原価をインラインで編集できます。カタログ全体での一括操作が可能です。 |
| **[注文](/orders)** | 過去の注文の検索、レシートの再印刷、注文の編集、返金フローへのアクセスが可能です。 |
| **[顧客](/customers)** | 新規顧客の追加や既存の顧客情報の編集が可能です（無料ユーザーは会計時にピッカーから選択のみ可能）。 |
| **[レポート](/reports)** | 日締め売上レポート — 決済方法別、レジ担当者別、店舗別の合計を表示し、現金精算に適しています。 |
| **[クーポン](/coupons)** | レジからWooCommerceのクーポンカタログを閲覧できます（無料ユーザーはぼかし表示のプレビューのみ）。 |

### 決済ゲートウェイ {#payment-gateways}

無料プラグインの「カード」ゲートウェイは、外部端末で行った決済を記録するだけで、リーダー自体との連携はありません。プロ版では**すべてのWooCommerce決済ゲートウェイ**をPOSで利用でき、さらに以下の統合済みゲートウェイが含まれています：

| ゲートウェイ | 用途 |
|---|---|
| **[Stripe Terminal](/payment/gateways/stripe-terminal)** | StripeのS700 / WisePOS Eリーダーとの直接連携。MOTO（電話注文）およびテスト用シミュレーターに対応。 |
| **[SumUp Terminal](/payment/gateways/sumup-terminal)** | SumUpカードリーダーをPOSにペアリングし、レジを離れずにカード決済を完了できます。 |
| **[Vipps MobilePay（ビップス モバイルペイ）](/payment/gateways/vipps-mobilepay)** | QRコードまたはプッシュ通知によるスマートフォン決済 — ノルウェーではVipps、デンマークとフィンランドではMobilePayとして利用可能。 |
| **[メール請求書](/payment/gateways/email-invoice)** | 顧客にメールで支払いリンクを送信し、オンラインで支払いが完了すると注文が確定します。 |
| **[Webチェックアウト](/payment/gateways/web-checkout)** | 顧客をホスト型チェックアウトページにリダイレクトし、オンラインで決済を完了します。 |
| **その他すべてのWooCommerceゲートウェイ** | [ゲートウェイテンプレート](/payment/gateways)を使用して、任意のWooCommerce決済プラグインをPOSチェックアウトに対応させることができます。 |

### クーポン、返金、レジ業務ワークフロー {#coupons-refunds-and-the-at-counter-workflow}

<FeatureGrid>
  <FeatureCard icon="gift" title="レジでのクーポン適用" variant="pro">
    会計時に WooCommerce のクーポンコードを検索・適用できます。クーポンはカート内に削除可能なピルとして表示され、順次割引に対応し、有効期限、使用回数制限、最小/最大利用金額、商品・カテゴリ制限など通常のルールがすべて適用されます。
  </FeatureCard>
  <FeatureCard icon="arrow-rotate-right" title="レジでの返金処理" variant="pro">
    POS から全額または一部の[返金](/orders/refunds)を処理できます。元の決済方法への返金または現金での返金が可能です。返金にはキャッシャーと店舗が記録され、完全な監査証跡が残ります。
  </FeatureCard>
</FeatureGrid>

### マルチ店舗、在庫、ブランディング {#multi-store-stock-and-branding}

<FeatureGrid>
  <FeatureCard icon="store" title="複数店舗拠点" variant="pro">
    複数の[店舗拠点](/stores)を作成でき、それぞれに独自の住所、ロゴ、税率、[税番号](/settings/wp-admin/store-tax-ids)、キャッシャーの割り当てを設定できます。注文やレポートは店舗ごとにフィルタリングできます。
  </FeatureCard>
  <FeatureCard icon="box" title="拠点別在庫管理（ATUM）" variant="pro">
    Pro の店舗を [ATUM Multi-Inventory](/extensions/atum) の拠点にリンクし、拠点別の在庫、価格、SKU を管理できます。
  </FeatureCard>
  <FeatureCard icon="receipt" title="店舗別レシートテンプレート" variant="pro">
    店舗ごとに異なるレシートや請求書の[テンプレート](/receipts)を割り当てられます。各店舗にはロゴ、住所、連絡先などの独自のブランディングとテンプレートの並び順を設定できます。
  </FeatureCard>
  <FeatureCard icon="puzzle-piece" title="POS 拡張機能のインストール" variant="pro">
    設定画面から POS の[拡張機能](/extensions)や連携機能を直接インストール、有効化、更新できます。無料ユーザーはカタログを閲覧できますが、インストール操作は無効になっています。
  </FeatureCard>
</FeatureGrid>

### サポート {#support}

## 料金 {#pricing}

Pro はサイト単位で [wcpos.com/pro](https://wcpos.com/pro) から購入できます：

- **年間ライセンス** — $129／年、ライセンス期間中のアップデートとサポートを含みます。
- **永久ライセンス** — $399 の一括払い、製品が存続する限りアップデートとサポートを含みます。

価格は 2026 年 5 月現在のものであり、変更される場合があります。

ライセンスの有効期限が切れても Pro 機能は引き続き使用できますが、アップデートとサポートの提供は停止します。有効化の手順やライセンスに関するトラブルシューティングについては、[WCPOS Pro のインストール](/getting-started/pro-license)を参照してください。

:::tip まだ検討中ですか？
公開デモを [demo.wcpos.com/pos](https://demo.wcpos.com/pos)（ログイン情報: `demo` / `demo`）で試すことができます。Pro の全機能が有効になっているため、管理画面、クーポン、返金フローを購入前に確認できます。
:::

## アップグレードの準備はできましたか？ {#ready-to-upgrade}

ダウンロードと有効化の手順については、[WCPOS Pro のインストール](/getting-started/pro-license)をご覧ください。

- **v1.8 以降の Pro** — まず無料版プラグインを無効化して削除してください。v1.8 以降、Pro はスタンドアロンプラグインであり、無料版と同時にインストールしないでください。
- **旧バージョンの Pro（v1.8 未満）** — お使いの Pro バージョンで必要な場合は、無料版プラグインをインストールしたままにしてください。
