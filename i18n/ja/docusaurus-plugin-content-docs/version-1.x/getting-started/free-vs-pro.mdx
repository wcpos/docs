WCPOSには2つのエディションがあります。**無料プラグイン**は、レジでの決済、レシート印刷、WooCommerceへの注文同期に必要なすべての機能を備えています。**Pro**では、管理画面（商品、注文、顧客、レポート）、追加の決済ゲートウェイ、クーポン、返金、マルチ店舗サポートが利用可能です。

主に売上を記録するレジが必要であれば、無料プラグインで十分です。POSから店舗の日常業務も行いたい場合 — 在庫管理、過去の注文検索、顧客への返金、統合リーダーによるカード決済 — それがPro版の役割です。

## 概要 {#at-a-glance}

| 項目 | 無料版 | Pro版 |
|---|---|---|
| 売上処理とレシートの印刷・メール送信 | ✓ | ✓ |
| 現金および外部カード決済 | ✓ | ✓ |
| 統合カードリーダー（Stripe、SumUp、Vipps/MobilePay） | — | ✓ |
| レジでのクーポン利用 | — | ✓ |
| POSからの返金 | — | ✓ |
| POSから在庫、価格、商品詳細を編集 | — | ✓ |
| 過去の注文の検索と再印刷 | — | ✓ |
| 顧客の追加と編集 | — | ✓ |
| 日次レポート | — | ✓ |
| 複数店舗 | — | ✓ |
| 店舗ごとのレシートテンプレート、税番号、ブランディング | — | ✓ |
| 設定からPOS拡張機能のインストールと管理 | — | ✓ |
| Discordでの優先サポート | — | ✓ |

## 無料プラグインの内容 {#what-s-in-the-free-plugin}

[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)の無料プラグインは、基本的な「販売会計」ワークフローをカバーしています。

<FeatureGrid>
  <FeatureCard icon="cash-register" title="現金＋カード決済">
    2つの組み込み決済方法：おつり計算機能付きの**現金**と、外部端末で決済を行い売上を記録するだけの**カード**。
  </FeatureCard>
  <FeatureCard icon="cart-shopping" title="カートと商品検索">
    商品パネルの検索、バーコードスキャン、明細の追加、明細ごとの割引適用、カタログにないオープン価格の「その他」商品の追加が可能です。
  </FeatureCard>
  <FeatureCard icon="receipt" title="レシートとサーマル印刷">
    組み込みの[テンプレートギャラリー](/templates/customise)から選択できます — レシート、請求書、見積書、梱包伝票、ギフトレシート、キッチンチケット — ネットワーク、Bluetooth、またはUSB経由で58 mmまたは80 mmサーマルプリンターに印刷できます。
  </FeatureCard>
  <FeatureCard icon="sync" title="オフラインストレージ + 同期">
    商品と注文はローカルに保存されるため、レジは高速に動作し、不安定な接続でも使用できます。ネットワークが復旧すると、注文はWooCommerceに同期されます。
  </FeatureCard>
  <FeatureCard icon="barcode" title="バーコード対応">
    USBまたはBluetoothスキャナーで商品をカートに直接スキャンするか、デバイスのカメラを使用できます。
  </FeatureCard>
  <FeatureCard icon="file-invoice" title="顧客税務ID">
    顧客レコードに組み込みの税務IDフィールドを搭載 — VAT、ABN、GST、GSTIN、EIN、その他の地域固有の形式に対応。
  </FeatureCard>
  <FeatureCard icon="globe" title="クロスプラットフォーム + 多言語対応">
    ブラウザ、デスクトップ（Windows/macOS）、モバイルアプリ（iOS/Android、ベータ版）に対応。主要なヨーロッパ、アジア、中南米の言語に翻訳済み。
  </FeatureCard>
  <FeatureCard icon="puzzle-piece" title="拡張機能カタログ（閲覧のみ）">
    設定から POS 拡張機能とインテグレーションのディレクトリを閲覧できます。インストールには Pro が必要です。
  </FeatureCard>
</FeatureGrid>

:::note 無料版での POS 限定の顧客・クーポンアクセス
無料プラグインでも、既存の顧客を**選択**したり、レジでクーポンカタログを**閲覧**したりすることは可能です。ただし、顧客の追加・編集やチェックアウト時のクーポンコード適用はできません。これらの機能は Pro で利用可能になります。
:::

## Pro の追加機能 {#what-pro-adds}

Pro ではバックオフィス機能の全体、追加の決済オプション、店舗ごとの機能が利用可能になります。以下の項目はすべて Pro 限定です。

### 管理画面 {#management-screens}

無料プラグインでは販売処理用の商品パネルが利用できます。Pro ではさらに完全な管理機能が追加されます：

| 画面 | できること |
|---|---|
| **[商品](/products)** | 在庫数、価格、売上原価をインラインで編集できます。カタログ全体の一括操作にも対応しています。 |
| **[注文](/orders)** | 過去の注文の検索、レシートの再印刷、注文の編集、返金フローへのアクセスが可能です。 |
| **[顧客](/customers)** | 新規顧客の追加や既存の顧客情報の編集が可能です（無料ユーザーは会計時にピッカーから選択することのみ可能です）。 |
| **[レポート](/reports)** | 日次売上レポート — 決済方法別、レジ担当者別、店舗別の合計を表示し、現金照合に適しています。 |
| **[クーポン](/coupons)** | レジからWooCommerceのクーポンカタログを参照できます（無料ユーザーにはぼかし表示のプレビューのみ表示されます）。 |

### 決済ゲートウェイ {#payment-gateways}

無料プラグインの「カード」ゲートウェイは、外部端末で行った決済を記録するだけで、リーダー自体との連携はありません。Proでは**すべてのWooCommerce決済ゲートウェイ**をPOSで利用でき、以下の統合ゲートウェイが同梱されています：

| ゲートウェイ | 用途 |
|---|---|
| **[Stripe Terminal](/payment/custom-gateways/stripe-terminal)** | StripeのS700 / WisePOS Eリーダーとの直接連携。MOTO（電話注文）およびテスト用シミュレーターに対応しています。 |
| **[SumUp Terminal](/payment/custom-gateways/sumup-terminal)** | SumUpカードリーダーをPOSとペアリングし、レジから離れることなくカード決済を完了できます。 |
| **[Vipps MobilePay](/payment/custom-gateways/vipps-mobilepay)** | QRコードまたはプッシュ通知によるスマートフォン決済 — ノルウェーではVipps、デンマークとフィンランドではMobilePayとして利用できます。 |
| **[メール請求書](/payment/custom-gateways/email-invoice)** | 顧客にメールで決済リンクを送信し、オンラインで支払いが完了すると注文が確定します。 |
| **[Webチェックアウト](/payment/custom-gateways/web-checkout)** | 顧客をホスト型チェックアウトページにリダイレクトし、オンラインで決済を完了します。 |
| **その他のWooCommerceゲートウェイ** | [ゲートウェイテンプレート](/payment/custom-gateways)を使用して、任意のWooCommerce決済プラグインをPOSチェックアウトに対応させることができます。 |

### クーポン、返金、レジ業務ワークフロー {#coupons-refunds-and-the-at-counter-workflow}

<FeatureGrid>
  <FeatureCard icon="gift" title="レジでのクーポン適用" variant="pro">
    チェックアウト時にWooCommerceのクーポンコードを検索・適用できます。クーポンはカート内に削除可能なピルとして表示され、複数の割引を順次適用でき、有効期限、使用回数制限、最小・最大金額、商品およびカテゴリの制限など、通常のルールがすべて適用されます。
  </FeatureCard>
  <FeatureCard icon="arrow-rotate-right" title="レジでの返金処理" variant="pro">
    POSから全額または一部の[返金](/orders/refunds)を実行できます。元の決済方法への返金または現金での返金が可能です。返金には担当キャッシャーと店舗が記録され、完全な監査証跡を確保します。
  </FeatureCard>
</FeatureGrid>

### マルチ店舗、在庫、ブランディング {#multi-store-stock-and-branding}

<FeatureGrid>
  <FeatureCard icon="store" title="マルチ店舗ロケーション" variant="pro">
    複数の[店舗ロケーション](/stores)を作成し、それぞれに独自の住所、ロゴ、税率、[税務ID](/settings/wp-admin/store-tax-ids)、キャッシャーの割り当てを設定できます。注文やレポートは店舗ごとにフィルタリングできます。
  </FeatureCard>
  <FeatureCard icon="box" title="ロケーション別在庫管理（ATUM）" variant="pro">
    Pro の店舗を [ATUM Multi-Inventory](/extensions/atum) のロケーションにリンクし、ロケーションごとの在庫、価格、SKU を管理できます。
  </FeatureCard>
  <FeatureCard icon="receipt" title="店舗ごとのレシートテンプレート" variant="pro">
    店舗ごとに異なるレシートや請求書の[テンプレート](/templates)を割り当てられます。各店舗にはロゴ、住所、連絡先などの独自のブランディングとテンプレートの順序設定があります。
  </FeatureCard>
  <FeatureCard icon="puzzle-piece" title="POS 拡張機能のインストール" variant="pro">
    設定画面から直接 POS の[拡張機能](/extensions)や連携機能をインストール、有効化、更新できます。無料ユーザーはカタログを閲覧できますが、インストール操作は無効になっています。
  </FeatureCard>
</FeatureGrid>

## 料金 {#pricing}

Pro はサイト単位で [wcpos.com/pro](https://wcpos.com/pro) から購入できます：

- **年間ライセンス** — $129／年、ライセンス期間中のアップデートとサポートを含みます。
- **永久ライセンス** — $399（一括払い）、製品の存続期間中のアップデートとサポートを含みます。

価格は2026年5月時点のものであり、変更される場合があります。

ライセンスの有効期限が切れても、Pro 機能は引き続き使用できますが、アップデートとサポートは受けられなくなります。有効化の手順やライセンスのトラブルシューティングについては、[WCPOS Pro のインストール](/getting-started/pro-license)を参照してください。

:::tip まだお悩みですか？
[demo.wcpos.com/pos](https://demo.wcpos.com/pos)（ログイン：`demo` / `demo`）で公開デモをお試しいただけます。Pro の全機能が動作しており、管理画面、クーポン、返金フローを購入前に確認できます。
:::

## アップグレードの準備はできましたか？ {#ready-to-upgrade}

ダウンロードと有効化の手順については、[WCPOS Pro のインストール](/getting-started/pro-license)を参照してください。

- **v1.8 以降の Pro** — まず無料版プラグインを無効化して削除してください。v1.8 以降、Pro はスタンドアロンプラグインとなっており、無料版と同時にインストールしないでください。
- **v1.8 より前の Pro** — お使いの Pro バージョンが無料版プラグインを必要とする場合は、無料版をインストールしたままにしてください。
