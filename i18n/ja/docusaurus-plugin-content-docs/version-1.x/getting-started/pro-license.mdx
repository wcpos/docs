:::info Pro機能
[WCPOS Pro](https://wcpos.com/pro)は、有料のプラグインで、無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)に機能を追加します。
:::

## Proに含まれているもの {#whats-included-in-pro}

WCPOS Proは追加機能をアンロックします：

- **製品画面** - POSで在庫レベルと価格を直接編集
- **注文画面** - 注文履歴の表示と管理
- **顧客画面** - 顧客情報の追加と編集
- **レポート画面** - 1日の終わりの売上レポートを生成
- **追加の決済ゲートウェイ** - Stripe Terminal、SumUp、およびカスタムゲートウェイ

## インストール {#installation}

WCPOS Proのライセンスを購入した場合は、以下の手順に従ってプラグインをインストールしてアクティブ化してください：

1. [https://wcpos.com/my-account/](https://wcpos.com/my-account/)にアクセス。
2. **ダウンロード**の下で、ダウンロードリンクをクリックしてプラグインをデスクトップに保存。
3. 次に、サイトに移動し、ログインして`WP Admin > プラグイン > 新規追加 > プラグインのアップロード`へ進む。
4. デスクトップからプラグインのzipファイルをアップロードし、アクティブ化。
5. 次に、`WP Admin > POS > 設定 > ライセンス`に移動し、ライセンスキーとライセンスメールを入力してアクティブ化を完了。

![マイダウンロード](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "アカウントページからWCPOS Proをダウンロードできます")

## 更新 {#updates}

新しいバージョンが準備できると、他のプラグインと同様に更新ダッシュボードに表示されます。最新のバージョンは[アカウントページ](https://wcpos.com/my-account/)からもダウンロードできます。

## ライセンスを購入 {#purchase-a-license}

[wcpos.com/pro](https://wcpos.com/pro)を訪れてProライセンスを購入してください。
