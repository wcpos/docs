:::info Pro 機能
[WCPOS Pro](https://wcpos.com/pro) は、無料の [WCPOS プラグイン](https://wordpress.org/plugins/woocommerce-pos/)に機能を追加する有料プラグインです。
:::

## Pro に含まれる機能

WCPOS Pro は、以下の追加機能を解放します：

- **商品画面** - POS で直接在庫レベルと価格を編集
- **注文画面** - 注文履歴の表示と管理
- **顧客画面** - 顧客情報の追加と編集
- **レポート画面** - 日次売上レポートの生成
- **追加の決済ゲートウェイ** - Stripe Terminal、SumUp、カスタムゲートウェイ

## インストール

WCPOS Pro のライセンスを購入された方は、以下の手順に従ってプラグインをインストールおよびアクティベートしてください：

1. 次のページに移動: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **ダウンロード**の下で、ダウンロードリンクをクリックし、プラグインをデスクトップに保存します。
3. サイトにログインし、`WP 管理画面 > プラグイン > 新規追加 > プラグインをアップロード`に移動
4. デスクトップからプラグインのzipファイルをアップロードし、アクティベートします。
5. 次に、`WP 管理画面 > POS > 設定 > ライセンス`に移動し、ライセンスキーとライセンスメールを入力してアクティベーションを完了します。

![マイダウンロード](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "WCPOS Proはアカウントページでダウンロードできます")

## 更新

新しいバージョンが準備できると、他のプラグインと同様に更新ダッシュボードに表示されます。また、[アカウントページ](https://wcpos.com/my-account/)から最新バージョンをダウンロードすることもできます。

## ライセンスの購入

Pro ライセンスを購入するには、[wcpos.com/pro](https://wcpos.com/pro) にアクセスしてください。
