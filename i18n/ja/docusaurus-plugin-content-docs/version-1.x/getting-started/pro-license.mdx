:::info Pro機能
[WCPOS Pro](https://wcpos.com/pro)は、有料のプラグインで、無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)に機能を追加します。
:::

## Proに含まれるものは？ {#whats-included-in-pro}

WCPOS Proは、追加の機能を解除します：

- **商品画面** - POSで直接在庫レベルと価格を編集
- **注文画面** - 注文履歴を表示・管理
- **顧客画面** - 顧客情報を追加・編集
- **レポート画面** - 日末の売上レポートを生成
- **追加の支払いゲートウェイ** - Stripe Terminal、SumUp、およびカスタムゲートウェイ

## インストール {#installation}

WCPOS Proのライセンスを購入した場合は、以下の手順に従ってプラグインをインストールおよびアクティブ化してください：

1. [https://wcpos.com/my-account/](https://wcpos.com/my-account/)に移動します。
2. **ダウンロード**の下で、ダウンロードリンクをクリックして、プラグインをデスクトップに保存します。
3. 次に、サイトに移動し、ログインして`WP Admin > プラグイン > プラグインを追加 > プラグインをアップロード`に進みます。
4. デスクトップからプラグインのzipファイルをアップロードし、アクティブ化します。
5. 次に、`WP Admin > POS > 設定 > ライセンス`に移動し、ライセンスキーとライセンスメールを入力して、アクティブ化を完了します。

![マイダウンロード](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "アカウントページでWCPOS Proをダウンロードできます")

## 更新 {#updates}

新しいバージョンが準備できると、他のプラグインと同様に更新ダッシュボードに表示されます。最新バージョンを[アカウントページ](https://wcpos.com/my-account/)からダウンロードすることもできます。

## ライセンスを購入する {#purchase-a-license}

[wcpos.com/pro](https://wcpos.com/pro)にアクセスして、Proライセンスを購入してください。
