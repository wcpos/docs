:::info Pro機能
[WCPOS Pro](https://wcpos.com/pro)は、WCPOSの管理画面、追加の決済ゲートウェイ、その他のPro限定機能を提供する有料プラグインです。

**v1.8以降、Proはスタンドアロンプラグインです** — 無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)を併せてインストールする必要はありません。
:::

## Proに含まれる機能 {#whats-included-in-pro}

WCPOS Proで利用できる追加機能：

- **商品画面** - POSで在庫数や価格を直接編集
- **注文画面** - 注文履歴の表示と管理
- **顧客画面** - 顧客情報の追加と編集
- **レポート画面** - 日次売上レポートを作成
- **追加決済ゲートウェイ** - Stripe Terminal、SumUp、およびカスタムゲートウェイ

## インストール {#installation}

WCPOS Pro のライセンスを購入済みの場合は、以下の手順に従ってプラグインをインストールし、有効化してください。

:::note 旧バージョンからのアップグレードですか？
以前に無料の WCPOS プラグインと旧バージョンの Pro アドオンの両方をインストールしていた場合は、新しいスタンドアロン版の Pro プラグインを有効化する前に、**無料プラグインを無効化して削除してください**。両方を同時に使用すると、*「Invalid response fetching remote state」* や致命的な *「Class not found」* などの有効化エラーが発生する可能性があります。
:::

1. [https://wcpos.com/my-account/](https://wcpos.com/my-account/) にアクセスします。
2. **ダウンロード**からダウンロードリンクをクリックし、プラグインをデスクトップに保存します。
3. 次にサイトにログインし、`WP Admin > プラグイン > Add Plugin > プラグインのアップロード` に移動します。
4. デスクトップからプラグインの zip ファイルをアップロードし、有効化します。
5. 次に、`WP Admin > POS > 設定 > License` に移動し、ライセンスキーとライセンスメールアドレスを入力して有効化を完了します。

![マイダウンロード](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "You can download WCPOS Pro on your account page")

## アップデート {#updates}

新しいバージョンがリリースされると、他のプラグインと同様にアップデートダッシュボードに表示されます。最新バージョンは[アカウントページ](https://wcpos.com/my-account/)からもダウンロードできます。

## トラブルシューティング {#troubleshooting}

### アクティベーションの失敗 {#activation-failures}

- **「無効なライセンスキー」** — 貼り付け時に末尾のスペースが含まれていないか確認し、購入時に使用したメールアドレスと一致しているか確認してください。
- **「ライセンス上限に達しました」** — 各ライセンスには対応サイト数の上限があります。[アカウントページ](https://wcpos.com/my-account/)から未使用のサイトを無効化するか、プランをアップグレードしてください。

### ライセンスの有効期限切れ {#expired-licence}

- ライセンスの有効期限が切れても Pro 機能は引き続き使用できますが、アップデートは受信できなくなります。
- [アカウントページ](https://wcpos.com/my-account/)から更新してください。既存のキーはそのまま使用できます。

### ドメインの移行 {#domain-transfers}

- サイトを新しいドメインに移行した場合は、まず旧ドメインでライセンスを無効化してから、新しいドメインで再度アクティベートしてください。
- 以前のドメインにアクセスできなくなった場合は、**support@wcpos.com** に連絡してアクティベーションをリセットしてください。

### ライセンスサーバーへの接続 {#licence-server-connectivity}

- アクティベーションには、WordPress サーバーから WCPOS ライセンスサーバーへの通信が必要です。
- アクティベーションが無応答で失敗する場合は、ホストのファイアウォールやセキュリティプラグインによって送信 HTTPS リクエストがブロックされていないか確認してください。
- 一部のマネージドホストでは外部 API 呼び出しがデフォルトでブロックされています。ホスティング会社に連絡して `wcpos.com` をホワイトリストに追加してください。

### ライセンス状態の確認 {#checking-licence-status}

- `WP Admin > POS > 設定 > License` で現在のアクティベーション状態を確認できます。
- [アカウントページ](https://wcpos.com/my-account/)で、すべての有効なサイトと有効期限を確認できます。

## ライセンスの購入 {#purchase-a-license}

Pro ライセンスは [wcpos.com/pro](https://wcpos.com/pro) から購入できます。
