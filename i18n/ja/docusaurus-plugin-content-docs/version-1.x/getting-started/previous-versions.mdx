## 以前のバージョンの再インストール

WCPOSプラグインの以前のバージョンに戻す必要がある場合は、次の手順に従ってください：

1. WordPress.orgの[詳細表示](https://wordpress.org/plugins/woocommerce-pos/advanced/)ページにアクセスします。

2. **詳細オプション**セクションまでスクロールし、**以前のバージョン**セレクトメニューを探します。

3. セレクトメニューを開くと、利用可能なプラグインバージョンのリストが表示されます。インストールするバージョンを見つけ、その横にある**ダウンロード**ボタンをクリックします。デスクトップや便利な場所にダウンロードしたファイルを保存します。

4. WordPress管理ダッシュボードにログインします。

5. 左側のメニューで**プラグイン**をクリックし、**プラグイン**ページに移動します。

6. 現在のWCPOSプラグインを非アクティブ化し、削除します。これによりプラグインがWordPressインストールから削除されます。心配しないでください、データは失われません。

7. ページ上部の**新規追加**ボタンをクリックします。

8. **プラグインの追加**ページで、**プラグインをアップロード**ボタンをクリックします。

9. 以前にダウンロードしたプラグインファイルを、デスクトップや保存した場所から選択します。

<div class="image-container">
  <Image
    alt="プラグインをアップロード"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">WordPress管理画面でプラグインをアップロード</p>
</div>

10. **今すぐインストール**ボタンをクリックし、インストールが完了するまで待ちます。

11. インストールが完了したら、**アクティブ化**ボタンをクリックして、WCPOSプラグインの以前のバージョンをアクティブ化します。

これで完了です！WCPOSプラグインの以前のバージョンを正常に再インストールしました。

再インストール中に問題が発生した場合や、さらに支援が必要な場合は、support@wcpos.comまでサポートチームにお問い合わせください。
