## 以前のバージョンを再インストールする {#re-installing-a-previous-version}

WCPOSプラグインの以前のバージョンに戻す必要がある場合は、以下の手順に従ってください。

1. WCPOSのWordPress.orgページ [Advanced View](https://wordpress.org/plugins/woocommerce-pos/advanced/) にアクセスします。

2. **Advanced Options** セクションまでスクロールダウンし、**Previous Versions** セレクトメニューを見つけます。

3. セレクトメニューを開くと、利用可能なプラグインバージョンのリストが表示されます。インストールしたいバージョンを見つけ、横にある **Download** ボタンをクリックしてください。ダウンロードしたファイルをデスクトップまたは便利な場所に保存します。

4. WordPressの管理ダッシュボードにログインします。

5. 左側のメニューから **Plugins** をクリックして **Plugins** ページに移動します。

6. 現在のWCPOSプラグインのバージョンを無効にして削除します。これにより、プラグインはWordPressインストールから削除されます。データは失われないので心配しないでください。

7. ページの上部にある **Add New** ボタンをクリックします。

8. **Add Plugins** ページで **Upload Plugin** ボタンをクリックします。

9. デスクトップまたは保存した場所から以前にダウンロードしたプラグインファイルを選択します。

  <Image

  />

</div>

10. **Install Now** ボタンをクリックし、インストールが完了するまで待ちます。

11. インストールが完了したら、**Activate** ボタンをクリックして以前のバージョンのWCPOSプラグインを有効化します。

以上です！以前のバージョンのWCPOSプラグインを正常に再インストールしました。

再インストールプロセス中に問題が発生したり、さらなる支援が必要な場合は、support@wcpos.comまでサポートチームにお問い合わせください。
