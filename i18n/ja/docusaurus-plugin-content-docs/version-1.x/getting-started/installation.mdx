WCPOSプラグインの適切な機能と最適なパフォーマンスを確保するために、次の最小要件を満たしていることを確認してください。

## サーバー要件 {#server-requirements}

- **WordPress:** バージョン5.6以上
- **WooCommerce:** バージョン5.3以上
- **PHP:** バージョン7.2以上

これらの最小要件を満たすことで、WCPOSプラグインとWordPress環境との互換性と安定性が確保されます。また、プラグインによって提供される最新の機能や拡張を利用できるようになります。

## ブラウザ要件 {#browser-requirements}

- **最新のウェブブラウザ:** WCPOSプラグインは、Google Chrome、Mozilla Firefox、Safari、Microsoft Edgeなどの最新のウェブブラウザでの動作を考慮して設計されています。ご自身の好みのブラウザの最新の安定版を使用していることを確認してください。
- **JavaScriptを有効にする:** WCPOSプラグインは、インタラクティブ機能にJavaScriptに依存しています。ブラウザ設定でJavaScriptが有効になっていることを確認してください。
- **プライベートブラウジングモードではないこと:** ブラウザがプライベートブラウジングモードになっていると、ブラウザのIndexedDB (IDB) データベースへのアクセスが制限される可能性があります。プライベートブラウジングモードは、WCPOSプラグインの特定の機能の正常な動作を妨げることがあります。

## プラグインのインストール {#installing-the-plugin}

1. `WP Admin > プラグイン > プラグインの追加`に移動
2. "wcpos"を検索
3. **今すぐインストール**をクリックし、次に**有効化**

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)からプラグインをダウンロードし、手動でアップロードすることもできます。

## POSへのアクセス {#accessing-the-pos}

有効化後、POSにアクセスできます：

- **ウェブ:** `yourdomain.com/pos`
- **デスクトップ:** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、 [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、 [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル:** [iOS](https://testflight.apple.com/join/JGBdVRrW)、 [Android](https://play.google.com/apps/testing/com.wcpos.main)

最小要件を満たし、互換性のあるウェブブラウザを使用することで、WCPOSプラグインを快適に使用し、その全機能を活用して販売業務を効率的に管理できます。

さらに質問がある場合やサポートが必要な場合は、[Discordチャット](https://wcpos.com/discord)に投稿したり、support@wcpos.comにメールを送信して、お気軽にサポートチームにお問い合わせください。
