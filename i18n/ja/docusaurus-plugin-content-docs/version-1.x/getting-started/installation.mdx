WCPOSプラグインを正常に機能させ、最適なパフォーマンスを確保するために、以下の最小要件を満たしていることを確認してください：

## サーバー要件

- **WordPress:** バージョン 5.6 以上
- **WooCommerce:** バージョン 5.3 以上
- **PHP:** バージョン 7.2 以上

これらの最小要件を満たすことで、WCPOSプラグインとWordPressの環境間の互換性と安定性を確保できます。また、プラグインが提供する最新の機能と拡張機能を活用できます。

## ブラウザ要件

- **最新のウェブブラウザ：** WCPOSプラグインは、Google Chrome、Mozilla Firefox、Safari、Microsoft Edgeなどの最新のウェブブラウザで動作するように設計されています。使用しているブラウザの最新の安定版を使用していることを確認してください。
- **JavaScriptを有効に：** WCPOSプラグインは、インタラクティブな機能のためにJavaScriptに依存しています。ブラウザ設定でJavaScriptが有効になっていることを確認してください。
- **プライベートブラウジングモードを使用しない：** ブラウザはプライベートブラウジングモードであってはいけません。このモードではブラウザのIndexedDB（IDB）データベースへのアクセスが制限される可能性があるためです。プライベートブラウジングモードは、WCPOSプラグインの特定の機能の正常な動作を妨げる可能性があります。

## プラグインのインストール

1. `WP管理画面 > プラグイン > 新規追加`に移動
2. 「WooCommerce POS」を検索
3. **今すぐインストール**をクリックし、続いて**有効化**をクリック

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)からプラグインをダウンロードし、手動でアップロードすることもできます。

## POSへのアクセス

有効化後、以下の方法でPOSにアクセスできます：

- **ウェブ：** `yourdomain.com/pos`
- **デスクトップ/モバイル：** [wcpos.com](https://wcpos.com)からアプリをダウンロードし、ストアに接続

最小要件を満たし、互換性のあるウェブブラウザを使用することで、WCPOSプラグインを使用してポイントオブセールの運営を効率的に管理できます。

さらに質問や支援が必要な場合は、[Discordチャット](https://wcpos.com/discord)に投稿するか、support@wcpos.comまでメールでお問い合わせください。
