WCPOSプラグインの適切な機能と最適なパフォーマンスを確保するために、環境が以下の最低要件を満たしていることを確認してください。

## サーバー要件

- **WordPress:** バージョン 5.6 以上
- **WooCommerce:** バージョン 5.3 以上
- **PHP:** バージョン 7.2 以上

これらの最低要件を満たすことで、WCPOSプラグインとWordPress環境との互換性と安定性が保証されます。また、プラグインが提供する最新の機能や改善点を利用できるようになります。

## ブラウザ要件

- **最新のウェブブラウザ:** WCPOSプラグインは、Google Chrome、Mozilla Firefox、Safari、Microsoft Edgeなどの最新のウェブブラウザで動作するように設計されています。お使いのブラウザの最新の安定版を使用していることを確認してください。
- **JavaScript 有効:** WCPOSプラグインは、そのインタラクティブな機能のためにJavaScriptに依存しています。ブラウザ設定でJavaScriptが有効になっていることを確認してください。
- **プライベートブラウジングモードではないこと:** ブラウザはプライベートブラウジングモードではない必要があります。このモードはブラウザのIndexedDB（IDB）データベースへのアクセスを制限する可能性があります。プライベートブラウジングモードでは、WCPOSプラグインの特定の機能が正しく動作しない場合があります。

## プラグインのインストール

1. `WP Admin > プラグイン > 新規追加` に移動
2. "WooCommerce POS" を検索
3. **今すぐインストール** をクリックし、次に **有効化** をクリック

あるいは、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)からプラグインをダウンロードし、手動でアップロードすることもできます。

## POSへのアクセス

有効化後、次の場所でPOSにアクセスできます：

- **ウェブ:** `yourdomain.com/pos`
- **デスクトップ:** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル:** [iOS](https://testflight.apple.com/join/JGBdVRrW)、[Android](https://play.google.com/apps/testing/com.wcpos.main)

最低要件を満たし、互換性のあるウェブブラウザを使用することで、WCPOSプラグインを利用してスムーズな体験を得て、販売時点管理業務を効率的に遂行するための機能を最大限に活用できます。
