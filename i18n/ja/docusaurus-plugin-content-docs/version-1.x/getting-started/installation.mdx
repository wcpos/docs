WCPOSプラグインの正しい機能と最適なパフォーマンスを確保するために、次の最低要件を満たしていることを確認してください。

## サーバー要件

- **WordPress:** バージョン 5.6 以上
- **WooCommerce:** バージョン 5.3 以上
- **PHP:** バージョン 7.2 以上

これらの最低要件を満たすことで、WCPOSプラグインとWordPress環境との互換性と安定性が確保されます。また、プラグインが提供する最新の機能と強化を享受できることも保証されます。

## ブラウザー要件

- **モダンウェブブラウザー:** WCPOSプラグインは、Google Chrome、Mozilla Firefox、Safari、Microsoft Edgeを含むモダンウェブブラウザーで動作するように設計されています。お好みのブラウザーの最新の安定版を使用していることを確認してください。
- **JavaScriptを有効にする:** WCPOSプラグインは、インタラクティブ機能のためにJavaScriptに依存しています。ブラウザー設定でJavaScriptが有効になっていることを確認してください。
- **プライベートブラウジングモードを使用しない:** インデックス付きDB (IDB) データベースへのアクセスが制限される可能性があるため、ブラウザーはプライベートブラウジングモードにしてはいけません。プライベートブラウジングモードは、WCPOSプラグインの特定の機能の正常な動作を妨げる可能性があります。

## プラグインのインストール

1. `WP Admin > プラグイン > 新規追加` に移動します
2. "WooCommerce POS" を検索します
3. **今すぐインストール** をクリックし、次に **有効化** をクリックします

あるいは、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からプラグインをダウンロードし、手動でアップロードすることもできます。

## POSへのアクセス

有効化後、以下の場所でPOSにアクセスできます。

- **ウェブ:** `yourdomain.com/pos`
- **デスクトップ:** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル:** [iOS](https://testflight.apple.com/join/JGBdVRrW)、[Android](https://play.google.com/apps/testing/com.wcpos.main)

最低要件を満たし、互換性のあるウェブブラウザーを使用することで、WCPOSプラグインをスムーズに体験し、販売業務を効率的に管理するためのフルポテンシャルを活用することができます。

ご不明な点がある場合やサポートが必要な場合は、[Discordチャット](https://wcpos.com/discord) に投稿するか、support@wcpos.com にメールを送信してサポートチームにお問い合わせください。
