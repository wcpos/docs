WCPOS プラグインが正しく動作し、最適なパフォーマンスを発揮するために、ご利用の環境が以下の最小要件を満たしていることをご確認ください。

## サーバー要件 {#server-requirements}

WCPOS プラグインは WordPress サイト上で動作し、サイトは以下の最小要件を満たす必要があります：

| コンポーネント | 最小バージョン | 備考 |
| ----------- | --------------- | --------------- |
| WordPress | 5.6 | — |
| WooCommerce | 5.3 | — |
| PHP | 7.4 | 8.x 推奨 |

これらの最小要件を満たすことで、WordPress 環境との互換性と安定性が確保され、最新の WCPOS 機能にアクセスできるようになります。

## ブラウザ要件 {#browser-requirements}

POS のウェブ版は、あらゆるモダンブラウザで動作します。最新の安定版リリースを使用してください：

| ブラウザ | 対応バージョン |
| --------------- | ----------------- |
| Google Chrome | 最新の安定版 |
| Mozilla Firefox | 最新の安定版 |
| Safari | 最新の安定版 |
| Microsoft Edge | 最新の安定版 |

:::warning プライベート／シークレットモードは避けてください
プライベートブラウジングは、WCPOS がデータをオフラインで保存するために使用するブラウザの IndexedDB データベースへのアクセスをブロックすることがあり、一部の機能が正しく動作しなくなります。また、JavaScript も有効になっている必要があります。
:::

## アプリ要件 {#app-requirements}

WCPOS はデスクトップアプリおよびモバイルアプリとしても利用できます。それぞれ、以下の最小オペレーティングシステム向けに構築されています：

| プラットフォーム | 最小 OS | アーキテクチャ |
| --------------- | ------------------------- | -------------------------- |
| Windows デスクトップ | Windows 10 | x64、ARM64 |
| macOS デスクトップ | macOS 12 (Monterey) | Intel (x64)、Apple Silicon |
| iOS / iPadOS | iOS 16.4 | — |
| Android | Android 10 (API level 29) | — |

:::note 印刷の権限
Bluetooth およびネットワーク印刷には、プロンプトが表示されたときに、アプリに関連する Bluetooth およびローカルネットワークの権限を付与する必要があります。
:::

## プラグインのインストール {#installing-the-plugin}

1. `WP Admin > Plugins > Add Plugin` に移動します
2. 「wcpos」を検索します
3. **Install Now** をクリックし、次に **Activate** をクリックします

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からプラグインをダウンロードして手動でアップロードすることもできます。

## POS へのアクセス {#accessing-the-pos}

有効化後、以下の場所から POS にアクセスできます：

- **Web：** `yourdomain.com/pos`
- **デスクトップ：** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル：** [iOS](https://testflight.apple.com/join/JGBdVRrW)、[Android](https://play.google.com/apps/testing/com.wcpos.main)

最小要件を満たし、対応する Web ブラウザを使用することで、WCPOS プラグインをスムーズに利用でき、その潜在能力を最大限に活用して POS 業務を効率的に管理できます。

ご不明な点やサポートが必要な場合は、[Discord チャット](https://wcpos.com/discord)に投稿するか、support@wcpos.com にメールでお気軽にお問い合わせください。
