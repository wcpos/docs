WCPOSプラグインが正しく動作し、最適なパフォーマンスを発揮するために、ご利用の環境が以下の最小要件を満たしていることをご確認ください。

## サーバー要件 {#server-requirements}

- **WordPress：** バージョン 5.6 以上
- **WooCommerce：** バージョン 5.3 以上
- **PHP：** バージョン 7.4 以上（8.x 推奨）

これらの最小要件を満たすことで、WCPOSプラグインとWordPress環境間の互換性と安定性が確保されます。また、プラグインが提供する最新の機能や改善点を活用できるようになります。

## ブラウザ要件 {#browser-requirements}

- **モダンウェブブラウザ：** WCPOSプラグインは、Google Chrome、Mozilla Firefox、Safari、Microsoft Edgeなどのモダンウェブブラウザで動作するよう設計されています。ご利用のブラウザが最新の安定版であることをご確認ください。
- **JavaScriptの有効化：** WCPOSプラグインは、インタラクティブな機能にJavaScriptを使用しています。ブラウザの設定でJavaScriptが有効になっていることをご確認ください。
- **プライベートブラウジングモードは使用しないでください：** ブラウザのIndexedDB（IDB）データベースへのアクセスが制限される場合があるため、プライベートブラウジングモードは使用しないでください。プライベートブラウジングモードでは、WCPOSプラグインの一部の機能が正常に動作しない可能性があります。

## プラグインのインストール {#installing-the-plugin}

1. `WP Admin > プラグイン > Add Plugin` に移動します
2. 「wcpos」を検索します
3. **今すぐインストール** をクリックし、次に **有効化** をクリックします

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からプラグインをダウンロードして手動でアップロードすることもできます。

## POSへのアクセス {#accessing-the-pos}

有効化後、以下のURLからPOSにアクセスできます：

- **Web：** `yourdomain.com/pos`
- **デスクトップ：** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac（Intel）](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac（Apple Silicon）](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル：** [iOS](https://testflight.apple.com/join/JGBdVRrW)、[Android](https://play.google.com/apps/testing/com.wcpos.main)

最小要件を満たし、対応するWebブラウザを使用することで、WCPOSプラグインをスムーズに利用でき、POS業務を効率的に管理するための機能を最大限に活用できます。

ご不明な点やサポートが必要な場合は、[Discordチャット](https://wcpos.com/discord)に投稿するか、support@wcpos.com にメールでお気軽にお問い合わせください。
