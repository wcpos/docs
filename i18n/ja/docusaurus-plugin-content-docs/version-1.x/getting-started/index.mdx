## WCPOSとは？

WCPOSはWooCommerce用のポイントオブセール（POS）システムです。これにより、ウェブブラウザまたは専用アプリを使用して、物理的な場所でWooCommerceストアの製品を対面で販売することができます。

**利用可能なプラットフォーム：**

- **Web：** `yourdomain.com/pos` で即座にアクセス
- **デスクトップ：** [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **モバイル：** [iOS (TestFlight)](https://testflight.apple.com/join/JGBdVRrW) および [Android (Google Play Beta)](https://play.google.com/apps/testing/com.wcpos.main)

## 5分で始める

### 1. プラグインをインストール

WordPressサイトに無料の [WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/) をインストールします：

1. `WP Admin > プラグイン > 新規追加` に移動
2. "WooCommerce POS" を検索
3. **今すぐインストール** をクリックし、その後 **有効化** 

詳細な要件については[インストール](/getting-started/installation)を参照してください。

### 2. POSにアクセス

**Webユーザー：**
ブラウザで `yourdomain.com/pos` に移動します。WordPressの資格情報でログインするよう求められます。

**デスクトップ/モバイルユーザー：**
1. アプリをダウンロード： [Windows](https://updates.wcpos.com/electron/download/win32-x64)、[Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64)、[Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)、[iOS](https://testflight.apple.com/join/JGBdVRrW)、または [Android](https://play.google.com/apps/testing/com.wcpos.main)
2. ストアのURLを入力し、**接続** をクリック
3. WordPressの資格情報でログイン

複数のストアとユーザーの管理については[接続](/getting-started/connect)を参照してください。

### 3. 初回同期を待つ

WCPOSは、サーバーの過負荷を避けるために、小さなバッチ（約50ずつ）で製品をダウンロードします。製品は検索、スクロール、フィルタリングに応じて順次ダウンロードされますので、販売を開始する前にすべてが同期するのを待つ必要はありません。詳細については[製品同期](/products/sync)を参照してください。

### 4. 販売を開始

- 検索またはスキャンして製品をカートに追加
- 顧客を選択するか、「ゲスト」を使用
- **チェックアウト** をクリックして支払いを処理
- レシートを印刷またはメール送信

## 主な機能

- **高速ローカル検索** - 製品はローカルに保存され、即座に結果が得られます
- **オフラインブラウジング** - インターネットなしで製品や顧客を検索（注文の完了には接続が必要です）
- **バーコードスキャン** - 任意のUSBまたはBluetoothバーコードスキャナを使用
- **複数の支払い方法** - 現金、カード、カスタムゲートウェイ
- **レシート印刷** - カスタマイズ可能なレシートを印刷またはメール送信
- **マルチストアサポート** - 複数のWooCommerceストアに接続

## プロ機能

[WCPOS Pro](https://wcpos.com/pro) は追加機能を提供します：

- **製品管理** - POS内で在庫と価格を直接編集
- **注文履歴** - 過去の注文を表示および管理
- **顧客管理** - 顧客の詳細を追加および編集
- **売上レポート** - 印刷オプション付きの終日レポート
- **追加の支払いゲートウェイ** - Stripe Terminal、SumUpなど
