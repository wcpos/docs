## WCPOSとは

WCPOSは、WooCommerceのポイントオブセール（POS）システムです。WooCommerceストアの商品を、Webブラウザや専用アプリを使用して、実店舗で販売することができます。

**利用可能な環境：**

- **Web:** `yourdomain.com/pos`ですぐにアクセス可能
- **デスクトップ：** [wcpos.com](https://wcpos.com)からWindows、macOS、Linuxにダウンロード可能
- **モバイル：** iOSおよびAndroidアプリ（近日公開）

## 5分で始められる

### 1. プラグインのインストール

WordPress サイトに無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)をインストールします：

1. `WP管理画面 > プラグイン > 新規追加`に移動
2. 「WooCommerce POS」を検索
3. **今すぐインストール**をクリック、その後**有効化**

詳細な要件は[インストール](/getting-started/installation)を参照してください。

### 2. POSへのアクセス

**Webユーザー：**
ブラウザで`yourdomain.com/pos`にアクセスします。WordPress資格情報でログインを求められます。

**デスクトップ/モバイルユーザー：**
1. [wcpos.com](https://wcpos.com)からアプリをダウンロード
2. ストアのURLを入力し、**接続**をクリック
3. WordPress資格情報でログイン

複数のストアとユーザーの管理については、[接続](/getting-started/connect)を参照してください。

### 3. 初期同期を待つ

WCPOSは、サーバーに負担をかけないよう、商品を小さなバッチ（約50個ずつ）でダウンロードします。商品は、検索、スクロール、またはフィルタリング時に徐々にダウンロードされるため、すべてが同期されるまで待つ必要はありません。詳細は[商品同期](/products/sync)を参照してください。

### 4. 販売を開始

- 商品を検索またはスキャンしてカートに追加
- 顧客を選択するか「ゲスト」を使用
- **チェックアウト**をクリックして支払いを処理
- レシートを印刷またはメール送信

## 主な機能

- **高速ローカル検索** - 商品はすぐに結果を表示するためローカルに保存
- **オフラインブラウジング** - インターネット接続なしで商品と顧客を検索可能（注文完了には接続が必要）
- **バーコードスキャン** - あらゆるUSBまたはBluetoothバーコードスキャナーを使用可能
- **複数の支払い方法** - 現金、カード、カスタムゲートウェイ
- **レシート印刷** - カスタマイズ可能なレシートの印刷またはメール送信
- **マルチストアサポート** - 複数のWooCommerceストアに接続

## プロ機能

[WCPOS Pro](https://wcpos.com/pro)で追加の機能が利用可能：

- **商品管理** - POSで在庫と価格を直接編集
- **注文履歴** - 過去の注文の表示と管理
- **顧客管理** - 顧客情報の追加と編集
- **売上レポート** - 印刷オプション付きの日次売上レポート
- **追加の支払いゲートウェイ** - Stripe Terminal、SumUpなど
