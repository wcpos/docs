## WCPOSとは何ですか？ {#what-is-wcpos}

WCPOSはWooCommerce用のポイントオブセール（POS）システムです。これにより、Webブラウザまたは専用アプリを使用して、物理的な場所でWooCommerceストアの商品を対面で販売できます。

## 5分で始める {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="プラグインをインストール">

WordPressサイトに無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)をインストールします：

1. **WP管理画面 → プラグイン → 新規追加**に移動します。
2. "WooCommerce POS"と検索します。
3. **今すぐインストール**をクリックし、次に**有効化**します。

詳細な要件については[インストール](/getting-started/installation)をご覧ください。

</StepCard>

<StepCard number={2} title="POSにアクセス">

**Webユーザー：** ブラウザで`yourdomain.com/pos`に移動します。WordPressの資格情報でログインするように求められます。

**デスクトップ/モバイルユーザー：** アプリをダウンロードし、ストアのURLを入力して**接続**をクリックし、WordPressの資格情報でログインします。

複数のストアやユーザーの管理に関する詳細については[接続](/getting-started/connect)をご覧ください。

</StepCard>

<StepCard number={3} title="初回同期を待つ">

WCPOSはサーバーの負荷を避けるために商品を小分けでダウンロードします。商品は検索、スクロール、フィルタリングする際に徐々にダウンロードされるため、販売を開始する前にすべての同期を待つ必要はありません。

詳細については[商品同期](/products/sync)をご覧ください。

</StepCard>

<StepCard number={4} title="販売を開始" isLast>

- 商品を検索またはスキャンしてカートに追加します。
- 顧客を選択するか、「ゲスト」を使用します。
- **チェックアウト**をクリックして支払いを処理します。
- レシートを印刷するか、メールで送信します。

</StepCard>

</Steps>

## 主な機能 {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="高速ローカル検索">
    商品がローカルに保存され、即座に検索結果が提供されます。
  </FeatureCard>
  <FeatureCard icon="wifi" title="オフラインブラウジング">
    インターネットなしで商品や顧客を検索できます（注文には接続が必要です）。
  </FeatureCard>
  <FeatureCard icon="barcode" title="バーコードスキャン">
    USBまたはBluetoothのバーコードスキャナーを使用できます。
  </FeatureCard>
  <FeatureCard icon="credit-card" title="現金およびカード支払い">
    現金と基本的なカード支払いに対するサポートが組み込まれています。
  </FeatureCard>
  <FeatureCard icon="print" title="レシート印刷">
    カスタマイズ可能なレシートを印刷またはメールで送信できます。
  </FeatureCard>
  <FeatureCard icon="calculator" title="自動税計算">
    WooCommerceの税設定に基づいた税金計算が行われます。
  </FeatureCard>
</FeatureGrid>

## プロ機能 {#pro-features}

[WCPOS Pro](https://wcpos.com/pro)は追加機能を提供します：

<FeatureGrid>
  <FeatureCard icon="box" title="Products Management" variant="pro">
    POS内で直接在庫と価格を編集できます。
  </FeatureCard>
  <FeatureCard icon="file-invoice" title="Orders History" variant="pro">
    過去の注文を表示および管理できます。
  </FeatureCard>
  <FeatureCard icon="users" title="Customers Management" variant="pro">
    顧客情報を追加および編集できます。
  </FeatureCard>
  <FeatureCard icon="chart-pie" title="Sales Reports" variant="pro">
    終日のレポートを印刷オプション付きで提供します。
  </FeatureCard>
  <FeatureCard icon="credit-card" title="Payment Gateways" variant="pro">
    Stripe Terminal、SumUp、およびカスタムゲートウェイ。
  </FeatureCard>
  <FeatureCard icon="store" title="Multi-Store Support" variant="pro">
    複数のWooCommerceストアに接続できます。
  </FeatureCard>
</FeatureGrid>
