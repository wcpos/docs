## WCPOSとは？ {#what-is-wcpos}

WCPOSは、WooCommerce用のポイントオブセール（POS）システムです。ウェブブラウザや専用アプリを使用して、実際の店舗でWooCommerceストアの商品を販売することができます。

## 5分で始める {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="プラグインをインストール">

無料の[WCPOSプラグイン](https://wordpress.org/plugins/woocommerce-pos/)をWordPressサイトにインストールします：

1. **WP管理 → プラグイン → 新規追加**に移動
2. "WooCommerce POS"を検索
3. **今すぐインストール**をクリックし、その後**有効化**します

詳細な要件については、[インストール](/getting-started/installation)を参照してください。

</StepCard>

<StepCard number={2} title="POSにアクセス">

**ウェブユーザー:** ブラウザで`yourdomain.com/pos`に移動します。WordPressの資格情報でログインするように求められます。

**デスクトップ/モバイルユーザー:** アプリをダウンロードし、店舗のURLを入力して**接続**をクリックし、WordPressの資格情報でログインします。

複数の店舗やユーザーを管理する詳細については、[接続](/getting-started/connect)を参照してください。

</StepCard>

<StepCard number={3} title="初期同期を待つ">

WCPOSはサーバーに負荷をかけないように、小分けにして商品をダウンロードします。商品は検索、スクロール、またはフィルタリングを行うと徐々にダウンロードされるため、販売を開始する前にすべてを同期する必要はありません。

詳細については、[商品同期](/products/sync)を参照してください。

</StepCard>

<StepCard number={4} title="販売を開始する" isLast>

- 検索またはスキャンして商品をカートに追加する
- 顧客を選択するか「ゲスト」を使用する
- **チェックアウト**をクリックして支払いを処理する
- 領収書を印刷またはメールする

</StepCard>

</Steps>

## 主な特徴 {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="迅速なローカル検索">
    商品はローカルに保存され、即座に検索結果が表示されます
  </FeatureCard>
  <FeatureCard icon="wifi" title="オフラインブラウジング">
    インターネットなしで商品や顧客を検索できます（注文には接続が必要です）
  </FeatureCard>
  <FeatureCard icon="barcode" title="バーコードスキャン">
    USBまたはBluetoothのバーコードスキャナーを使用できます
  </FeatureCard>
  <FeatureCard icon="credit-card" title="現金とカードの支払い">
    現金および基本的なカード支払いのための内蔵サポート
  </FeatureCard>
  <FeatureCard icon="print" title="領収書印刷">
    カスタマイズ可能な領収書を印刷またはメールできます
  </FeatureCard>
  <FeatureCard icon="calculator" title="自動税計算">
    WooCommerceの税設定に基づく税金計算
  </FeatureCard>
</FeatureGrid>
