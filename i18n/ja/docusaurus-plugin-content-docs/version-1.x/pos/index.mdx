POS画面は、製品を販売するための主なインターフェースです。WCPOSを使用する際に最も多くの時間を過ごす場所です。

{/* スクリーンショット: pos-desktop-layout.png */}

## 画面レイアウト

### ヘッダー

ヘッダーバーには以下が表示されます：

- **店舗名** - 接続されたWooCommerceストアの名前
- **接続状況インジケーター** - サーバーに接続されているときは緑の点、オフラインのときは黄色/赤
- **ユーザーメニュー** - 設定にアクセスしたり、ユーザーを切り替えたり、ログアウトしたりするためにクリック

### ナビゲーションドロワー

左のサイドバーはすべての主要画面へのクイックアクセスを提供します：

| アイコン | 画面 | 説明 |
|------|--------|-------------|
| <Icon name="cash-register" /> | **POS** | 主な販売インターフェース（ここにいます） |
| <Icon name="gift" /> | **製品** | 在庫管理（Pro） |
| <Icon name="receipt" /> | **注文** | 注文履歴（Pro） |

バージョン番号はドロワーの下部に表示されます。

## レスポンシブレイアウト

WCPOSは異なる画面サイズに適応します：

### デスクトップ / タブレット（大画面）

大きな画面では、POSは**二列レイアウト**を表示します：

- **左列：** 製品パネル - 製品を検索してブラウズする
- **右列：** カートパネル - 現在の注文を表示して管理する

カラムは**リサイズ可能**です - 両者の間の仕切りをドラッグして比率を調整します。

{/* スクリーンショット: pos-desktop-layout.png */}

### モバイル（小画面）

小さな画面では、POSは**タブベースのレイアウト**を使用します：

- **製品タブ：** 製品をブラウズしてカートに追加する
- **カートタブ：** カートを表示し、商品ラインアイテムを管理してチェックアウトする

画面の下部にあるナビゲーションバーを使用してタブ間を切り替えます。

{/* スクリーンショット: pos-mobile-layout.png */}

## 主なコンポーネント

### 製品パネル

POSの左側は[製品パネル](/pos/product-panel)で、ここで：

- 名前、SKU、バーコードで製品を検索
- 在庫状況、カテゴリー、タグなどで製品をフィルタリング
- 製品をカートに追加

[製品パネルについて詳しく学ぶ →](/pos/product-panel)

### カートパネル

POSの右側は[カートパネル](/pos/cart)で、ここで：

- 現在の注文のアイテムを表示
- 数量と価格を編集
- 顧客を選択または追加
- 注文アクション（ノート、無効、チェックアウト）にアクセス

[カートパネルについて詳しく学ぶ →](/pos/cart)

### チェックアウト

販売を完了する準備ができたら、**チェックアウト**をクリックして[チェックアウトモーダル](/pos/checkout)を開きます：

- クーポンコードを適用
- 支払い方法を選択
- 支払いを処理
- 領収書を印刷またはメールで送信

[チェックアウトについて詳しく学ぶ →](/pos/checkout)

## キーボードショートカット

WCPOSは高速操作のためにキーボードショートカットをサポートしています。設定モーダルから完全なリストにアクセスするか、[キーボードショートカット](/settings/store/hotkeys)を参照してください。

一般的なショートカット：

| ショートカット | アクション |
|----------|--------|
| `Ctrl + F` | 検索バーにフォーカス |
| `Ctrl + Shift + S` | 設定を開く |
| `Escape` | モーダル/ダイアログを閉じる |

## 接続性

ヘッダーの緑のインジケーターは接続状況を表示します：

- **緑** - WooCommerceサーバーに接続
- **黄色** - 接続に問題があり、再試行中
- **赤** - オフラインモード

オフラインの間も、キャッシュされた製品や顧客をブラウズし、新しい注文を開始することができます。注文を完了するには接続が必要です。
