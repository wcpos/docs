POSの画面は、製品を販売するためのメインインターフェースです。WCPOSを使用する際、最も多くの時間をここで過ごすことになります。

{/* スクリーンショット: pos-desktop-layout.png */}

## 画面レイアウト

### ヘッダー

ヘッダーバーには以下が表示されます：

- **ストア名** - 接続されたWooCommerceストアの名前
- **接続状態インジケーター** - サーバーに接続されている場合は緑のドット、オフライン時は黄色/赤
- **ユーザーメニュー** - クリックして設定、ユーザー切り替え、ログアウトにアクセス

### ナビゲーションドロワー

左サイドバーから、すべてのメイン画面に素早くアクセスできます：

| アイコン | 画面 | 説明 |
|------|------|-------------|
| <Icon name="cash-register" /> | **POS** | メイン販売インターフェース（現在地） |
| <Icon name="gift" /> | **製品** | 在庫管理（Pro） |
| <Icon name="receipt" /> | **注文** | 注文履歴（Pro） |
| <Icon name="users" /> | **顧客** | 顧客管理（Pro） |
| <Icon name="chart-line" /> | **レポート** | 売上レポート（Pro） |
| <Icon name="heart-pulse" /> | **ログ** | デバッグ用システムログ |
| <Icon name="circle-question" /> | **サポート** | Discordサポートチャンネル |

バージョン番号はドロワーの下部に表示されます。

## レスポンシブレイアウト

WCPOSは異なる画面サイズに適応します：

### デスクトップ / タブレット（大画面）

大画面では、POSは**2列レイアウト**を表示します：

- **左列：** 製品パネル - 製品の検索と閲覧
- **右列：** カートパネル - 現在の注文の表示と管理

列は**サイズ変更可能** - 両列の間の仕切りをドラッグして比率を調整できます。

{/* スクリーンショット: pos-desktop-layout.png */}

### モバイル（小画面）

小画面では、POSは**タブベースのレイアウト**を使用します：

- **製品タブ：** 製品の閲覧とカートへの追加
- **カートタブ：** カートの表示、ライン項目の管理、チェックアウト

画面下部のナビゲーションバーでタブを切り替えます。

{/* スクリーンショット: pos-mobile-layout.png */}

## メインコンポーネント

### 製品パネル

POSの左側は[製品パネル](/pos/product-panel)で、以下を行えます：

- 名前、SKU、またはバーコードで製品を検索
- 在庫状態、カテゴリー、タグなどで製品をフィルタリング
- カートに製品を追加

[製品パネルの詳細はこちら →](/pos/product-panel)

### カートパネル

POSの右側は[カートパネル](/pos/cart)で、以下を行えます：

- 現在の注文の商品を表示
- 数量と価格を編集
- 顧客の選択または追加
- 注文アクション（メモ、無効、チェックアウト）にアクセス

[カートパネルの詳細はこちら →](/pos/cart)

### チェックアウト

売上を完了する準備ができたら、**チェックアウト**をクリックして[チェックアウトモーダル](/pos/checkout)を開きます：

- クーポンコードの適用
- 支払い方法の選択
- 支払いの処理
- レシートの印刷またはメール送信

[チェックアウトの詳細はこちら →](/pos/checkout)

## キーボードショートカット

WCPOSは、より速い操作のためのキーボードショートカットをサポートしています。設定モーダルから完全なリストにアクセスするか、[キーボードショートカット](/settings/store/hotkeys)を参照してください。

よく使用されるショートカット：

| ショートカット | アクション |
|----------|----------|
| `Ctrl + F` | 検索バーにフォーカス |
| `Ctrl + Shift + S` | 設定を開く |
| `Escape` | モーダル/ダイアログを閉じる |

## 接続性

ヘッダーの緑のインジケーターは接続状態を示します：

- **緑** - WooCommerceサーバーに接続済み
- **黄色** - 接続に問題あり、再接続中
- **赤** - オフラインモード

オフライン中も、キャッシュされた製品と顧客を閲覧し、新しい注文を開始できます。注文の完了には接続が必要です。
