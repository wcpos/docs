POS画面は製品販売のための主要インターフェイスです。WCPOSを使用しているときに最も多くの時間を過ごす場所です。

{/* スクリーンショット: pos-desktop-layout.png */}

## 画面レイアウト {#screen-layout}

### ヘッダー {#header}

ヘッダーバーには以下が表示されます：

- **店舗名** - 接続されているWooCommerceストアの名前
- **接続インジケーター** - サーバーに接続されているときは緑のドット、オフラインのときは黄色/赤
- **ユーザーメニュー** - 設定にアクセスしたり、ユーザーを切り替えたり、ログアウトしたりするためにクリック

### ナビゲーションドロワー {#navigation-drawer}

左側のサイドバーからすべての主要画面に迅速にアクセスできます：

| アイコン | 画面 | 説明 |
|------|--------|-------------|
| <Icon name="cash-register" /> | **POS** | 主な販売インターフェイス（ここです） |
| <Icon name="gift" /> | **商品** | 在庫管理（Pro） |
| <Icon name="receipt" /> | **注文** | 注文履歴（Pro） |
| <Icon name="users" /> | **顧客** | 顧客管理（Pro） |
| <Icon name="chart-line" /> | **レポート** | 売上レポート（Pro） |
| <Icon name="heart-pulse" /> | **ログ** | デバッグ用のシステムログ |
| <Icon name="circle-question" /> | **サポート** | Discordサポートチャンネル |

バージョン番号はドロワーの下部に表示されます。

## レスポンシブレイアウト {#responsive-layout}

WCPOSはさまざまな画面サイズに適応します：

### デスクトップ / タブレット（大画面） {#desktop-tablet-large-screens}

大画面では、POSは**二列レイアウト**を表示します：

- **左列:** 商品パネル - 商品を検索・閲覧
- **右列:** カートパネル - 現在の注文を表示・管理

カラムは**サイズ変更可能**です - 列間の仕切りをドラッグして比率を調整します。

{/* スクリーンショット: pos-desktop-layout.png */}

### モバイル（小画面） {#mobile-small-screens}

小画面では、POSは**タブベースのレイアウト**を使用します：

- **商品タブ:** 商品を閲覧し、カートに追加
- **カートタブ:** カートを表示し、アイテムを管理し、チェックアウト

画面下部のナビゲーションバーを使ってタブ間を切り替えます。

{/* スクリーンショット: pos-mobile-layout.png */}

## 主なコンポーネント {#main-components}

### 商品パネル {#product-panel}

POSの左側は[商品パネル](/pos/product-panel)で、以下を行います：

- 名前、SKU、またはバーコードで商品を検索
- 在庫状況、カテゴリ、タグなどで商品をフィルター
- カートに商品を追加

[商品パネルについてさらに学ぶ →](/pos/product-panel)

### カートパネル {#cart-panel}

POSの右側は[カートパネル](/pos/cart)で、以下を行います：

- 現在の注文に含まれているアイテムを表示
- 数量や価格を編集
- 顧客を選択または追加
- 注文アクションにアクセス（メモ、無効、チェックアウト）

[カートパネルについてさらに学ぶ →](/pos/cart)

### チェックアウト {#checkout}

販売を完了する準備ができたら、**チェックアウト**をクリックして[チェックアウトモーダル](/pos/checkout)を開きます：

- クーポンコードを適用
- 支払い方法を選択
- 支払いを処理
- 領収書を印刷またはメール

[チェックアウトについてさらに学ぶ →](/pos/checkout)

## キーボードショートカット {#keyboard-shortcuts}

WCPOSは迅速な操作のためにキーボードショートカットをサポートしています。全ショートカットのリストは設定モーダルからアクセスできます。または[キーボードショートカット](/settings/store/hotkeys)を参照してください。

一般的なショートカット：

| ショートカット | アクション |
|----------|--------|
| `Ctrl + F` | 検索バーにフォーカス |
| `Ctrl + Shift + S` | 設定を開く |
| `Escape` | モーダル/ダイアログを閉じる |

## 接続性 {#connectivity}

ヘッダーの緑のインジケーターは接続状態を示します：

- **緑** - WooCommerceサーバーに接続されています
- **黄色** - 接続に問題、再接続中
- **赤** - オフラインモード

オフラインの間でもキャッシュされた商品や顧客を閲覧したり、新しい注文を開始したりできます。注文を完了するには接続が必要です。
