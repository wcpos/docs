カートパネルはPOS画面の右側で、現在の注文を管理する場所です。ここでは、アイテムを表示したり、数量や価格を編集したり、顧客を選択したり、チェックアウトに進んだりできます。

{/* スクリーンショット: pos-cart-panel.png */}

## インターフェース概要

### 顧客セレクター

カートパネルの上部：

- **顧客バッジ** - 現在の顧客を表示（例："ゲスト" または 顧客名）
- **顧客追加アイコン** (<Icon name="plus" />) - 新しい顧客を作成
- **表示設定** (<Icon name="sliders" />) - カートの列とオプションを構成

顧客バッジをクリックして、別の顧客を検索して選択します。

### 行項目

メインエリアには現在の注文のアイテムが表示されます：

- **数量** - 数量（編集可能）
- **名前** - バリエーション属性を含む製品名
- **価格** - 単価（編集可能）
- **合計** - 行の合計

編集の詳細については[行項目](/pos/cart/line-items)を参照してください。

### カートへの追加オプション

行項目の下に、特別なアイテムを追加できます：

- **雑多な製品を追加** - 手動価格入力でカスタムアイテムを追加
- **手数料を追加** - 手数料行を追加（例：ギフトラッピング、サービス料金）
- **配送を追加** - 手動コスト入力で配送行を追加

:::note 行項目のタイプ
WooCommerceは3種類の行項目を使用します：`line_item`（製品）、`fee_line`（手数料）、および`shipping_line`（配送）。現在、配送については手動入力のみがサポートされています—POSは自動的に配送コストを計算しません。
:::

### 注文の概要

- **小計** - 税金および手数料前の合計
- **税金** - 計算された税額（該当する場合）
- **合計** - 最終的な注文合計

### 注文アクション

カートパネルの下部：

- **注文ノート** - 顧客に見えるノートを追加
- **注文メタ** - カスタムメタデータを追加またはJSONを表示
- **サーバーに保存** - ステータスが`pos-open`のまま注文をWooCommerceに保存

詳細については[注文アクション](/pos/cart/order-actions)を参照してください。

### メインボタン

- **無効**（赤） - 現在の注文をキャンセル/削除
- **チェックアウト**（緑） - 支払いに進む

### オープン注文

最下部にはすべてのオープン注文が表示されるカルーセルがあります：

- 各カートはその合計を表示
- カートをクリックして切り替える
- 現在のカートはハイライト表示されます

詳細については[オープン注文](/pos/cart/open-orders)を参照してください。

## 表示設定

スライダーアイコン (<Icon name="sliders" />) をクリックしてカートパネルの表示をカスタマイズします。

### チェックアウト後にレシートを自動的に表示

有効にすると、販売が完了した後に自動的にレシート画面が表示されます。

### チェックアウト後にレシートを自動的に印刷

有効にすると、表示された時にレシートが自動的に印刷されます。これにより手動印刷ステップを減らし、時間を節約できます。

### クイック割引

カンマ区切りの割引率リストで、クイックショートカットボタンとして表示されます。

たとえば、`5,10,15,20`と入力すると、全体の注文に対して割合割引を適用する4つのボタンが追加されます。

### 列

カートに表示される列を構成します：

| 列      | 説明                       | 表示オプション                          |
|---------|--------------------------|------------------------------------|
| **数量**  | 各アイテムの数量              | 分割（アイテムの分割を許可）               |
| **名前**  | 製品名                     | SKU                                |
| **価格**  | 単価                       | セール表示インジケーター                      |
| **通常価格** | 割引されていない価格          | -                                  |
| **小計**  | 行の小計（数量 × 価格）        | 税額                                |
| **合計**  | 行の合計                   | 税額、セール表示インジケーター                |
| **アクション** | アイテム削除ボタン            | -                                  |

### デフォルト設定に戻す

クリックしてすべてのカート表示設定を元のデフォルトにリセットします。

## カートへのアイテム追加

### 製品パネルから

[製品パネル](/pos/product-panel)の製品をクリックして、カートに追加します。バリエーションのある製品の場合、最初にバリエーションを選択します。

### 雑多な製品

カタログにないアイテムの場合：

1. **雑多な製品を追加**をクリック
2. 名前と価格を入力
3. アイテムがカートに追加されます

### 手数料

手数料を追加するには（例：ギフトラッピング）：

1. **手数料を追加**をクリック
2. 名前と金額を入力
3. 手数料が別の行項目として表示されます

### 配送

配送を追加するには：

1. **配送を追加**をクリック
2. 配送方法の名前とコストを入力
3. 配送が注文に追加されます

## カートアイテムの編集

すべての行項目はカート内で直接編集可能です。詳細については[行項目](/pos/cart/line-items)を参照してください：

- 数量の編集
- 価格の変更
- 生のJSONデータの表示および編集
- アイテムの削除
