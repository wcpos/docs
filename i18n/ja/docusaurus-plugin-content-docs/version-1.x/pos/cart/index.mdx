カートパネルは、POSスクリーンの右側で、現在の注文を管理するための場所です。ここでは、アイテムを表示したり、数量や価格を編集したり、顧客を選択したり、チェックアウトに進んだりすることができます。

{/* スクリーンショット: pos-cart-panel.png */}

## インターフェースの概要

### 顧客セレクタ

カートパネルの上部には：

- **顧客バッジ** - 現在の顧客を表示（例："ゲスト" または 顧客名）
- **顧客を追加するアイコン** (<Icon name="plus" />) - 新しい顧客を作成
- **表示設定** (<Icon name="sliders" />) - カートの列とオプションを設定

顧客バッジをクリックして、別の顧客を検索し、選択できます。

### 行アイテム

メインエリアには現在の注文のアイテムが表示されます：

- **数量 (QTY)** - 数量（編集可能）
- **名称** - バリエーション属性付きの製品名
- **価格** - 単価（編集可能）
- **合計** - 行合計

編集の詳細については、[行アイテム](/pos/cart/line-items)を参照してください。

### カートに追加するオプション

行アイテムの下には、特別なアイテムを追加できます：

- **雑多商品を追加** - 手動価格入力でカスタムアイテムを追加
- **手数料を追加** - 手数料行を追加（例：ギフト包装、サービス料）
- **送料を追加** - 手動コスト入力で送料行を追加

:::note 行アイテムのタイプ
WooCommerceでは、3種類の行アイテムが使用されます： `line_item`（製品）、 `fee_line`（手数料）、および `shipping_line`（送料）。 現在、送料は手動入力のみがサポートされており、POSが送料を自動計算することはありません。
:::

### 注文の概要

- **小計** - 税金や手数料を除いた合計
- **税金** - 計算された税額（該当する場合）
- **合計** - 最終的な注文総額

### 注文アクション

カートパネルの下部には：

- **注文メモ** - 顧客に表示されるノートを追加
- **注文メタ** - カスタムメタデータを追加またはJSONを表示
- **サーバーに保存** - ステータス `pos-open` でWooCommerceに注文を保存

詳細については、[注文アクション](/pos/cart/order-actions)を参照してください。

### メインボタン

- **無効**（赤） - 現在の注文をキャンセル/削除
- **チェックアウト**（緑） - 支払いに進む

### オープン注文

一番下には、すべてのオープン注文を表示するカルーセルがあります：

- 各カートにはその合計が表示されます
- カートをクリックして切り替えます
- 現在のカートはハイライト表示されます

詳細については、[オープン注文](/pos/cart/open-orders)を参照してください。

## 表示設定

スライダーアイコン (<Icon name="sliders" />) をクリックすると、カートパネルの表示をカスタマイズできます。

### チェックアウト後にレシートを自動表示

有効にすると、販売が完了すると自動的にレシート画面が表示されます。

### チェックアウト後にレシートを自動印刷

有効にすると、表示されたときにレシートが自動的に印刷されます。これにより、手動印刷ステップを減らし、時間を節約できます。

### クイック割引

カンマで区切られた割引率のリストが、クイックショートカットボタンとして表示されます。

例えば、`5,10,15,20` と入力すると、全体の注文に対してワンクリックで割引を適用する4つのボタンが追加されます。

### 列

カートに表示する列を設定します：

| 列       | 説明                          | 表示オプション                         |
|----------|-------------------------------|-------------------------------------|
| **数量**  | 各アイテムの数量              | 分割（アイテムを分割可能にする）      |
| **名称**  | 製品名                        | SKU                                 |
| **価格**  | 単位あたりの価格              | セールインジケータ                     |
| **通常価格** | 割引がない価格                | -                                   |
| **小計**  | 行の小計（数量 × 価格）       | 税額                                 |
| **合計**  | 行合計                        | 税額、セールインジケータ               |
| **アクション** | アイテム削除ボタン           | -                                   |

### デフォルト設定を復元

すべてのカート表示設定を元のデフォルトにリセットするには、クリックします。

## カートにアイテムを追加

### 製品パネルから

[製品パネル](/pos/product-panel)内の製品をクリックして、カートに追加します。可変製品の場合は、最初にバリエーションを選択します。

### 雑多な製品

カタログにないアイテムの場合：

1. **雑多商品を追加** をクリック
2. 名前と価格を入力
3. アイテムがカートに追加されます

### 手数料

手数料を追加するには（例：ギフト包装）：

1. **手数料を追加** をクリック
2. 名前と金額を入力
3. 手数料が別の行アイテムとして表示されます

### 送料

送料を追加するには：

1. **送料を追加** をクリック
2. 送料の方法名とコストを入力
3. 送料が注文に追加されます

## カートアイテムの編集

すべての行アイテムは、カート内で直接編集可能です。編集の詳細については、[行アイテム](/pos/cart/line-items)を参照してください：

- 数量の編集
- 価格の変更
- 生のJSONデータの表示と編集
- アイテムの削除
