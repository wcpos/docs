カート画面は、POSの画面右側にあり、現在の注文を管理する場所です。ここで商品の表示、数量や価格の編集、顧客の選択、チェックアウトを行うことができます。

{/* Screenshot: pos-cart-panel.png */}

## インターフェース概要

### 顧客セレクター

カート画面の上部：

- **顧客バッジ** - 現在の顧客を表示（例：「ゲスト」または顧客名）
- **顧客追加アイコン**（<Icon name="plus" />） - 新規顧客作成
- **表示設定**（<Icon name="sliders" />） - カートの列とオプションを設定

顧客バッジをクリックして、別の顧客を検索・選択できます。

### 明細項目

現在の注文の商品を表示するメインエリア：

- **数量** - 数量（編集可能）
- **名前** - 商品名とバリエーション属性
- **価格** - 単価（編集可能）
- **合計** - 明細の合計

詳細は[明細項目](/pos/cart/line-items)を参照。

### カートに追加するオプション

明細項目の下に、特殊な項目を追加できます：

- **その他の商品を追加** - カスタム商品を手動価格で追加
- **手数料を追加** - 手数料の明細を追加（例：ギフトラッピング、サービス料）
- **配送を追加** - 手動で配送費を追加

:::note 明細項目のタイプ
WooCommerceは3種類の明細項目を使用します：`line_item`（商品）、`fee_line`（手数料）、`shipping_line`（配送）。現在、配送は手動入力のみ対応—POSは自動的に配送料を計算しません。
:::

### 注文サマリー

- **小計** - 税金と手数料を除いた合計
- **税金** - 計算された税金額（該当する場合）
- **合計** - 最終的な注文総額

### 注文アクション

カート画面の下部：

- **注文メモ** - 顧客に表示されるメモを追加
- **注文メタデータ** - カスタムメタデータを追加またはJSONを表示
- **サーバーに保存** - 注文をステータス`pos-open`でWooCommerceに保存

詳細は[注文アクション](/pos/cart/order-actions)を参照。

### メインボタン

- **無効化**（赤） - 現在の注文をキャンセル/削除
- **チェックアウト**（緑） - 支払いに進む

### 保留中の注文

最下部に、すべての保留中の注文をカルーセル表示：

- 各カートに合計額を表示
- カートをクリックして切り替え
- 現在のカートを強調表示

詳細は[保留中の注文](/pos/cart/open-orders)を参照。

スライダーアイコン（<Icon name="sliders" />）をクリックして、カート画面の表示をカスタマイズ。

### チェックアウト後に自動的にレシートを表示

有効にすると、販売完了後にレシート画面を自動的に表示します。

### チェックアウト後に自動的にレシートを印刷

有効にすると、レシート表示時に自動的に印刷します。手動印刷の手順を減らし、時間を節約できます。

### クイック割引

カンマ区切りの割引率のリストで、クイックショートカットボタンとして表示されます。

例えば、「5,10,15,20」と入力すると、1回のクリックで注文全体に割引率を適用する4つのボタンが追加されます。

### 列

カートに表示する列を設定：

| 列 | 説明 | 表示オプション |
|--------|-------------|-----------------|
| **数量** | 各商品の数量 | 分割（商品の分割を許可） |
| **名前** | 商品名 | SKU |
| **価格** | 単価 | セール中インジケーター |
| **通常価格** | 割引前の価格 | - |
| **小計** | 明細の小計（数量 × 価格） | 税金額 |
| **合計** | 明細の合計 | 税金額、セール中インジケーター |
| **アクション** | 商品削除ボタン | - |

### デフォルト設定に戻す

クリックして、すべてのカート表示設定を元のデフォルトに戻します。

## カートに商品を追加

### 商品パネルから

[商品パネル](/pos/product-panel)の商品をクリックしてカートに追加。可変商品の場合は、最初にバリエーションを選択。

### その他の商品

カタログにない商品：

1. **その他の商品を追加**をクリック
2. 名前と価格を入力
3. 商品がカートに追加されます

### 手数料

手数料の追加（例：ギフトラッピング）：

1. **手数料を追加**をクリック
2. 名前と金額を入力
3. 手数料が別の明細項目として追加されます

### 配送

配送の追加：

1. **配送を追加**をクリック
2. 配送方法名と費用を入力
3. 配送が注文に追加されます

## カート内商品の編集

すべての明細項目はカート内で直接編集可能。詳細は[明細項目](/pos/cart/line-items)を参照：

- 数量の編集
- 価格の変更
- 生のJSONデータの表示と編集
- 商品の削除
