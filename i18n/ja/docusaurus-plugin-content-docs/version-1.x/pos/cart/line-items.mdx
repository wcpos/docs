カート内の各アイテムは、直接編集可能な明細行です。WCPOSは数量、価格、その他の詳細について柔軟なインライン編集を提供します。

## 明細行の種類

WooCommerceは3つの種類の明細行を使用します：

| 種類 | 説明 | 例 |
|------|-------------|----------|
| **line_item** | カタログ内の商品 | Tシャツ、コーヒーマグ |
| **fee_line** | 追加料金 | ギフト包装、サービス料 |
| **shipping_line** | 配送費用 | 通常配送、速達配送 |

:::note
配送費用は手動で入力します。POSは配送料を自動的に計算しません。
:::

## 明細行の編集

### 数量

数量フィールドをクリックして直接編集します。新しい数字を入力するか、スピナーコントロールを使用します。

{/* Screenshot: cart-line-item-edit-qty.png */}

### 商品名

雑貨や追加料金の場合、名前をクリックして直接編集できます。

### 価格

価格フィールドをクリックして、単価を変更します。以下のような場合に便利です：

- 手動割引の適用
- 価格合わせ
- 価格エラーの修正

数量×価格に基づいて合計が自動的に更新されます。

### 3ドットメニュー

任意の明細行の **⋮**（3つのドット）をクリックして、追加オプションにアクセスします：

{/* Screenshot: cart-line-item-menu.png */}

#### 編集フォーム

明細行の詳細な編集フォームを開き、すべてのフィールドを変更できます。

#### 生のJSON表示

明細行の生のJSONデータを表示します。これは以下の場合に非常に役立ちます：

- **デバッグ** - WooCommerceに送信されるデータを正確に確認
- **トラブルシューティング** - カスタムフィールドやメタデータの問題を特定
- **開発** - 統合のためのデータ構造を理解

### アイテムの削除

赤い **×** ボタンをクリックして、カートからアイテムを削除します。

## バリエーション属性

可変商品の場合、選択されたバリエーション属性は商品名の下に表示されます：

```
パーカー
色: グリーン
サイズ: ラージ
```

これらの属性は明細行データの一部であり、注文に含まれます。

## メタデータ

[商品パネル](/pos/product-panel#meta-data-keys)の設定で[メタデータキー]を設定している場合、商品をカートに追加するときにそのデータが自動的にコピーされます。

生のJSON表示でこのメタデータを確認できます。

## 明細行の分割

[表示設定](/pos/cart#display-settings)で有効になっている場合、明細行を複数の行に分割できます。これは、顧客が同じ商品の一部に異なる割引を適用したい場合に便利です。

## ヒント

### クイック価格調整

簡単に割引するには：

1. 価格フィールドをクリック
2. 新しい価格を入力
3. Enterキーを押す

### キーボードナビゲーション

- **Tab** - 編集可能なフィールド間を移動
- **Enter** - 編集を確定
- **Escape** - 編集をキャンセル

### 一括編集

複雑な注文の場合、3ドットメニューから全フィールドを一度に表示できる完全な編集フォームの使用を検討してください。
