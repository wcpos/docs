カート内の各アイテムは直接編集できる行商品です。WCPOSは数量、価格、その他の詳細を柔軟にインライン編集できます。

## 行商品の種類

WooCommerceは3種類の行商品を使用します：

| タイプ | 説明 | 例 |
|------|-------------|----------|
| **line_item** | カタログからの製品 | Tシャツ、コーヒーマグ |
| **fee_line** | 追加料金 | ギフトラッピング、サービス料 |
| **shipping_line** | 配送料 | 標準配送、速達 |

:::note
配送料は手動で入力します。POSは配送料を自動で計算しません。
:::

## 行商品の編集

### 数量

数量フィールドをクリックして直接編集します。新しい数字を入力するか、スピナーコントロールを使用します。

{/* スクリーンショット: cart-line-item-edit-qty.png */}

### 製品名

雑貨や料金の場合、名前をクリックすることで直接編集できます。

### 価格

価格フィールドをクリックして単価を変更します。これは以下に便利です：

- 手動割引の適用
- 価格マッチング
- 価格エラーの訂正

合計は数量 × 価格に基づいて自動的に更新されます。

### 三点メニュー

任意の行商品にある**⋮**（三つの点）をクリックして追加オプションにアクセスします：

{/* スクリーンショット: cart-line-item-menu.png */}

#### 編集フォーム

行商品の詳細編集フォームが開き、すべてのフィールドを変更できます。

#### 生のJSONビュー

行商品の生のJSONデータを表示します。これは以下に非常に役立ちます：

- **デバッグ** - WooCommerceに送信されているデータを正確に確認
- **トラブルシューティング** - カスタムフィールドやメタデータの問題を特定
- **開発** - 統合用のデータ構造を理解

### アイテムの削除

赤い**×**ボタンをクリックしてカートからアイテムを削除します。

## バリエーション属性

可変製品の場合、選択されたバリエーション属性は製品名の下に表示されます：

```
Hoodie
Colour: Green
Size: Large
```

これらの属性は行商品データの一部であり、注文に含まれます。

## メタデータ

[メタデータキー](/pos/product-panel#meta-data-keys)を製品パネル設定で構成している場合、そのデータは製品がカートに追加されると自動的に行商品にコピーされます。

このメタデータは生のJSONビューで確認できます。

## 行商品の分割

[表示設定](/pos/cart#display-settings)で有効になっている場合、行商品を複数の行に分割できます。これは、顧客が同じ製品の一部に異なる割引を適用したいときに便利です。

## ヒント

### クイック価格調整

迅速に割引を提供するには：

1. 価格フィールドをクリック
2. 新しい価格を入力
3. Enterを押す

### キーボードナビゲーション

- **Tab** - 編集可能なフィールド間を移動
- **Enter** - 編集を確定
- **Escape** - 編集をキャンセル

### バッチ編集

複雑な注文の場合、三点メニューを使用して完全な編集フォームにアクセスすると、すべてのフィールドを一度に表示できます。
