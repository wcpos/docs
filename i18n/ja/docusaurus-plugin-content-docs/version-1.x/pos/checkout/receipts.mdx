決済が正常に処理されると、レシートモーダルが表示されます。完了した注文の概要が表示され、レシートの印刷またはメール送信のオプションが提供されます。

{/* Screenshot: receipt-modal.png */}

## レシートの内容 {#receipt-contents}

レシートには以下の情報が表示されます：

### 店舗情報 {#store-information}
- **店舗名**
- **住所**
- **電話番号**
- **メールアドレス**
- **ウェブサイト**
- **ロゴ**（設定されている場合）

### レシートヘッダー {#receipt-header}
- **レシートの種類**（例：「税務レシート」）
- **注文番号**
- **日時**
- **レジ担当者名**
- **支払い方法**

### 注文詳細 {#order-details}
- **明細項目**（以下を含む）:
  - 商品名
  - バリエーション属性（該当する場合）
  - 数量
  - 価格
  - 合計

### 合計 {#totals}
- **小計**
- **税金**（該当する場合）
- **合計**
- **預り金額**（現金の場合）
- **お釣り**（現金の場合）

### 顧客情報 {#customer-information}
- **請求先住所**
- **配送先住所**（異なる場合）

## レシートアクション {#receipt-actions}

### 閉じる {#close}
レシートモーダルを閉じてPOSに戻ります。

### レシートをメール送信 {#email-receipt}
顧客のメールアドレスにレシートを送信します：

1. **レシートをメール送信**をクリックします
2. メールアドレスを確認します
3. レシートはWooCommerceのメールシステムを通じて送信されます

顧客に注文に関連付けられたメールアドレスが必要です。

### レシートを印刷 {#print-receipt}
レシートを印刷します：

1. **レシートを印刷**をクリックします
2. ブラウザの印刷ダイアログが開きます
3. レシートプリンターを選択して印刷します

:::tip レシートプリンター
WCPOSはブラウザからアクセス可能なすべてのプリンターで動作します。最良の結果を得るには、以下をお勧めします：
- 専用のレシートプリンター（サーマルプリンターが一般的です）
- プリンター設定で用紙サイズを設定する
- 素早く印刷するためのプリントプリセットを設定する
:::

## 自動レシートオプション {#automatic-receipt-options}

[カート表示設定](/pos/cart#display-settings)で、以下を設定できます：

- **会計後にレシートを自動表示** - 支払い完了後すぐにレシートモーダルを開きます
- **会計後にレシートを自動印刷** - レシートが表示されると自動的に印刷を実行します

これらのオプションにより、大量の会計ワークフローを高速化できます。

## レシートのカスタマイズ {#customising-receipts}

レシートの外観は、WordPress のテンプレートで管理されます：

`WP Admin > POS > Templates`

カスタマイズの詳細については、[レシートテンプレート](/templates)を参照してください：

- 店舗情報の表示
- ロゴとブランディング
- レイアウトと書式設定
- 含める情報の選択

## レシートの再印刷 {#reprinting-receipts}

過去の注文のレシートを再印刷するには：

1. [注文画面](/orders)に移動します（Pro 機能）
2. 注文を検索する
3. クリックして注文を開く
4. レシートオプションを使用して再印刷する
