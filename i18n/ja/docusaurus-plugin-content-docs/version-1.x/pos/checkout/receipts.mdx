支払いが正常に処理された後、領収書モーダルが表示されます。これにより、完了した注文の概略が表示され、領収書を印刷またはメールで送信するオプションが提供されます。

{/* Screenshot: receipt-modal.png */}

## 領収書の内容

領収書には以下が表示されます：

### 店舗情報
- **店舗名**
- **住所**
- **電話番号**
- **メール**
- **ウェブサイト**
- **ロゴ**（設定されている場合）

### 領収書ヘッダー
- **領収書の種類**（例："税務領収書"）
- **注文番号**
- **日付と時刻**
- **店員名**
- **支払い方法**

### 注文詳細
- **品目**：
  - 商品名
  - バリエーション属性（該当する場合）
  - 数量
  - 価格
  - 合計

### 合計
- **小計**
- **税金**（該当する場合）
- **合計**
- **支払い金額**（現金の場合）
- **お釣り**（現金の場合）

### 顧客情報
- **請求先住所**
- **配送先住所**（異なる場合）

## 領収書の操作

### 閉じる
領収書モーダルを閉じて、POSに戻ります。

### 領収書をメール送信
領収書を顧客のメールアドレスに送信します：

1. **領収書をメール送信**をクリック
2. メールアドレスを確認
3. WooCommerceのメールシステムを通じて領収書が送信されます

顧客は、注文に関連付けられたメールアドレスを持っている必要があります。

### 領収書を印刷
領収書の物理コピーを印刷します：

1. **領収書を印刷**をクリック
2. ブラウザの印刷ダイアログが開きます
3. 領収書プリンターを選択し、印刷します

:::tip 領収書プリンター
WCPOSはブラウザからアクセスできる任意のプリンターで動作します。最良の結果を得るためには：
- 専用の領収書プリンター（サーマルプリンターが一般的）
- プリンター設定で用紙サイズを設定
- 迅速な印刷のための印刷プリセットを設定
:::

## 自動領収書オプション

[カート表示設定](/pos/cart#display-settings)では、以下を設定できます：

- **チェックアウト後に自動的に領収書を表示** - 支払い後にすぐに領収書モーダルが開きます
- **チェックアウト後に自動的に領収書を印刷** - 領収書が表示されたときに自動的に印刷がトリガーされます

これらのオプションは、ボリュームが多いチェックアウトワークフローを迅速化します。

## 領収書のカスタマイズ

領収書の外観はWordPressのテンプレートによって制御されます：

`WP Admin > POS > テンプレート`

カスタマイズに関する詳細は[領収書テンプレート](/templates/receipt-templates)を参照してください：

- 店舗情報の表示
- ロゴとブランディング
- レイアウトと書式設定
- 含める情報

## 領収書の再印刷

過去の注文の領収書を再印刷するには：

1. [注文画面](/orders)に移動します（Pro機能）
2. 注文を見つける
3. 注文を開くためにクリック
4. 領収書オプションを使用して再度印刷します
