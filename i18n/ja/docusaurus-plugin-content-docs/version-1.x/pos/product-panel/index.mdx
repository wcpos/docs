商品パネルは、カートに追加するために商品を検索、ブラウズ、選択するPOS画面の左側です。

{/* スクリーンショット: pos-product-panel.png */}

## インターフェースの概要 {#interface-overview}

### 検索バー {#search-bar}

パネルの上部にある検索バーでは、次の方法で商品を素早く見つけることができます:

- **商品名** - 商品名の任意の部分を入力
- **SKU** - ストックキーピングユニットで検索
- **バーコード** - バーコード番号をスキャンまたは入力

詳細については、[検索とフィルタリング](/pos/product-panel/search-filtering)を参照してください。

### 表示設定 {#display-settings}

検索バーの横にある**スライダーアイコン** (<Icon name="sliders" />) をクリックして表示設定を開きます。これにより、商品リストに表示される列と情報をカスタマイズできます。

### フィルターボタン {#filter-buttons}

検索バーの下にあるフィルターボタンを使用して、商品を絞り込むことができます:

- **在庫あり** - 利用可能な在庫を持つ商品のみを表示
- **注目商品** - WooCommerceで注目としてマークされた商品を表示
- **セール中** - 現在セール中の商品を表示
- **カテゴリ** - 商品カテゴリでフィルタリング
- **タグ** - 商品タグでフィルタリング
- **ブランド** - ブランドでフィルタリング（ブランドプラグインを使用している場合）

アクティブなフィルターはハイライト表示されます。フィルターを再度クリックすると削除されます。

### 商品リスト {#product-list}

メインエリアには、スクロール可能なリスト形式で商品の表示があります。各行には次の情報が表示されます:

- **商品画像** - 商品のサムネイル
- **商品名** - 任意の詳細情報（在庫、SKU、カテゴリなど）
- **価格** - 現在の販売価格（該当する場合はセール価格も表示）
- **アクションボタン** - カートに追加する (<Icon name="circle-plus" color="green" />) またはバリエーションを選択する (<Icon name="circle-arrow-right" color="green" />)

### 商品タイプ {#product-types}

- **シンプル商品**は、カートに直接追加するための緑の <Icon name="circle-plus" color="green" /> ボタンを表示します。
- **バリエーション商品**は、バリエーションセレクターを開くための緑の <Icon name="circle-arrow-right" color="green" /> 矢印を表示します。

詳細については、[バリエーション商品](/pos/product-panel/variations)を参照してください。

### フッター {#footer}

商品パネルの下部には:

- **税金ステータス** - 現在の税金計算の基準を表示します（例："税金の基準: 店舗の基本住所"）。アクティブな税率を表示するにはクリックします。
- **商品数** - 表示されている商品の数を示します（例："17中10件を表示"）
- **同期ボタン** (<Icon name="arrow-rotate-right" />) - サーバーから商品を更新します。**長押し**すると追加オプションが表示されます：
  - **同期** - サーバーからの標準の更新
  - **クリアして更新** - ローカルデータをクリアして全てを再読み込み

## 表示設定 {#display-settings}

スライダーアイコン (<Icon name="sliders" />) をクリックして商品パネルの表示をカスタマイズします。

### 在庫切れ商品の表示 {#show-out-of-stock-products}

現在在庫がない商品が表示されるかどうかを制御します。

- **有効**: 在庫切れの商品は表示されますが、カートに追加できません
- **無効**: 在庫切れの商品はリストから非表示になります

### 列 {#columns}

商品リストに表示される列を構成します:

| 列       | 説明               | 表示オプション                             |
|----------|------------------|-------------------------------------|
| **画像**   | 商品のサムネイル   | -                                   |
| **商品**   | 商品名と詳細情報   | 在庫、SKU、バーコード、カテゴリ、タグ、ブランド、属性、メタデータ |
| **SKU**    | ストックキーピングユニット（別列） | -                                   |
| **バーコード** | 商品バーコード（別列） | -                                   |
| **タイプ**  | 商品タイプ（simple, variable,など） | -                                   |
| **在庫**   | 現在の在庫数量     | -                                   |
| **売上原価**  | 販売コスト         | -                                   |
| **価格**   | 販売価格           | 税金情報、セール中インジケーター               |
| **アクション**| カートに追加するボタン | -                                   |

### メタデータキー {#meta-data-keys}

商品がカートに追加される際に、発注ラインアイテムに転送されるべき商品メタデータキーを指定します。

カンマ区切りのメタデータキーのリストを入力します:

```
_bottle_deposit,_custom_field,_tracking_code
```

**ユースケース：**
- ボトルデポジットプラグイン
- カスタム商品フィールド
- サードパーティ統合データ
- コンプライアンス/トラッキング情報

:::note
メタデータキーはケースセンシティブであり、商品のメタデータに保存されている通りに正確に一致させる必要があります。
:::

### デフォルト設定を復元 {#restore-default-settings}

すべての表示設定を元のデフォルトにリセットするにはクリックします。

## 商品をカートに追加する {#adding-products-to-cart}

### シンプル商品 {#simple-products}

緑の <Icon name="circle-plus" color="green" /> ボタンをクリックしてシンプル商品をカートに追加します。各クリックで1つの単位が追加されます。

### バリエーション商品 {#variable-products}

バリエーション商品（例：サイズや色のオプションがあるTシャツ）は、緑の <Icon name="circle-arrow-right" color="green" /> 矢印を表示します。クリックして:

1. **クイックポップオーバー** - ドロップダウンからバリエーションを選択
2. **インライン展開** - "展開"をクリックしてリスト内のすべてのバリエーションを表示

詳細については、[バリエーション商品](/pos/product-panel/variations)を参照してください。

### バーコードスキャン {#barcode-scanning}

USBまたはBluetoothのバーコードスキャナーを接続して、商品を素早く追加します。バーコードをスキャンすると、WCPOSが自動的に一致する商品を検索し追加します。

設定手順については、[バーコードスキャン](/pos/product-panel/barcode-scanning)を参照してください。

## なぜ一部の商品が表示されないのか？ {#why-cant-i-see-some-products}

商品パネルから商品が欠落している場合は、以下を確認してください:

- **POSの表示設定** - "オンラインのみ"に設定された商品はPOSに表示されません。[POS専用商品](/products/pos-only-products)を参照してください。
- **在庫設定** - 表示設定で"在庫切れ商品を表示"が無効になっている場合、在庫切れアイテムは非表示になります。
- **同期状況** - 新しいまたは最近更新された商品は同期が必要な場合があります。同期ボタンを長押しし、**"クリアして更新"**を選択します。
- **フィルター** - カテゴリやタグなどのアクティブなフィルターが商品を隠している可能性があります。
