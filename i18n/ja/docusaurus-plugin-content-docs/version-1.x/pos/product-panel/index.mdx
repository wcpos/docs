商品パネルは、POSの画面左側にあり、商品を検索、参照、選択してカートに追加するためのエリアです。

{/* Screenshot: pos-product-panel.png */}

## インターフェース概要

### 検索バー

パネル上部の検索バーで、以下の方法で商品をすぐに見つけられます：

- **商品名** - 商品名の一部を入力
- **SKU** - 在庫管理単位で検索
- **バーコード** - バーコード番号をスキャンまたは入力

詳細は[検索とフィルタリング](/pos/product-panel/search-filtering)を参照してください。

### 表示設定

検索バーの横の**スライダーアイコン**（<Icon name="sliders" />）をクリックして表示設定を開きます。商品リストに表示する列と情報をカスタマイズできます。

### フィルターボタン

検索バーの下にあるフィルターボタンで、商品を絞り込めます：

- **在庫あり** - 在庫のある商品のみ表示
- **おすすめ** - WooCommerceでおすすめとしてマークされた商品を表示
- **セール中** - 現在セール中の商品を表示
- **カテゴリー** - 商品カテゴリーでフィルター
- **タグ** - 商品タグでフィルター
- **ブランド** - ブランドプラグインを使用している場合、ブランドでフィルター

アクティブなフィルターは強調表示されます。もう一度クリックすると、フィルターを解除できます。

### 商品リスト

メインエリアには、スクロール可能な商品リストが表示されます。各行には以下が表示されます：

- **商品画像** - 商品のサムネイル
- **商品名** - オプションの詳細（在庫、SKU、カテゴリーなど）
- **価格** - 現在の販売価格（該当する場合、セール価格も表示）
- **アクションボタン** - カートに追加（<Icon name="circle-plus" color="green" />）または variation選択（<Icon name="circle-arrow-right" color="green" />）

### 商品タイプ

- **シンプル商品**は、緑の<Icon name="circle-plus" color="green" />ボタンでカートに直接追加できます
- **可変商品**は、緑の<Icon name="circle-arrow-right" color="green" />矢印でバリエーション選択画面を開きます

詳細は[可変商品](/pos/product-panel/variations)を参照してください。

### フッター

商品パネルの下部には：

- **税金ステータス** - 現在の税金計算の基準（例：「税金の基準：店舗のベースアドレス」）。クリックすると、アクティブな税率を表示
- **商品数** - 表示されている商品数（例：「10件中17件を表示」）
- **同期ボタン** - サーバーから商品を更新

スライダーアイコン（<Icon name="sliders" />）をクリックして、商品パネルの表示をカスタマイズします。

### 在庫切れ商品の表示

現在在庫切れの商品を表示するかどうかを制御します。

- **有効**: 在庫切れ商品は表示されますが、カートに追加できません
- **無効**: 在庫切れ商品はリストから非表示になります

### 列

商品リストに表示する列を設定します：

| 列 | 説明 | 表示オプション |
|--------|-------------|-----------------|
| **画像** | 商品サムネイル | - |
| **商品** | 商品名と詳細 | 在庫、SKU、バーコード、カテゴリー、タグ、ブランド、属性、メタデータ |
| **SKU** | 在庫管理単位（別の列） | - |
| **バーコード** | 商品バーコード（別の列） | - |
| **タイプ** | 商品タイプ（シンプル、可変など） | - |
| **在庫** | 現在の在庫数量 | - |
| **売上原価** | 商品の仕入れ価格 | - |
| **価格** | 販売価格 | 税金情報、セール中インジケーター |
| **アクション** | カートに追加ボタン | - |

### メタデータキー

カートに商品を追加する際に、注文明細に転送されるべき商品メタデータキーを指定します。

カンマ区切りでメタキーを入力します：

```
_bottle_deposit,_custom_field,_tracking_code
```

**ユースケース：**
- ボトルデポジットプラグイン
- カスタム商品フィールド
- サードパーティ連携データ
- コンプライアンス/追跡情報

:::note
メタキーは大文字と小文字を区別し、商品のメタデータに保存されているものと完全に一致する必要があります。
:::

### デフォルト設定の復元

クリックすると、すべての表示設定を元のデフォルトに戻します。

## カートへの商品追加

### シンプル商品

緑の<Icon name="circle-plus" color="green" />ボタンをクリックして、シンプル商品をカートに追加します。クリックするごとに1単位追加されます。

### 可変商品

可変商品（例：サイズと色のあるTシャツ）は、緑の<Icon name="circle-arrow-right" color="green" />矢印を表示します。クリックして：

1. **クイックポップオーバー** - ドロップダウンからバリエーションを選択
2. **インライン展開** - 「展開」をクリックしてリスト内のすべてのバリエーションを表示

詳細は[可変商品](/pos/product-panel/variations)を参照してください。

### バーコードスキャン

USBまたはBluetoothバーコードスキャナーを接続して、商品をすばやく追加できます。バーコードをスキャンすると、WCPOSは自動的に一致する商品を検索して追加します。

セットアップ手順は[バーコードスキャン](/pos/product-panel/barcode-scanning)を参照してください。
