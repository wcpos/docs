プロダクトパネルは、POS画面の左側にあり、カートに追加する商品を検索、閲覧、選択する場所です。

{/* スクリーンショット: pos-product-panel.png */}

## インターフェース概要

### 検索バー

パネルの上部にある検索バーを使用すると、以下の方法で素早く商品を見つけることができます：

- **商品名** - 商品名の一部を入力
- **SKU** - 在庫管理単位で検索
- **バーコード** - バーコード番号をスキャンまたは入力

詳細情報は[検索とフィルタリング](/pos/product-panel/search-filtering)を参照してください。

### 表示設定

検索バーの隣にある**スライダーのアイコン**(<Icon name="sliders" />)をクリックすると、表示設定を開くことができます。これにより、商品リストに表示される列と情報をカスタマイズできます。

### フィルターボタン

検索バーの下にあるフィルターボタンを使用して、商品を絞り込むことができます：

- **在庫あり** - 在庫のある商品のみ表示
- **特集** - WooCommerceで特集としてマークされた商品を表示
- **セール中** - 現在セール中の商品を表示
- **カテゴリー** - 商品カテゴリーでフィルタリング
- **タグ** - 商品タグでフィルタリング
- **ブランド** - ブランドでフィルタリング（ブランドプラグインを使用している場合）

アクティブなフィルターはハイライト表示されます。フィルターを再度クリックすると、フィルターを解除できます。

### 商品リスト

メインエリアには、スクロール可能なリストで商品が表示されます。各行には以下が表示されます：

- **商品画像** - 商品のサムネイル
- **商品名** - オプションの詳細（在庫、SKU、カテゴリーなど）付き
- **価格** - 現在の販売価格（適用される場合はセール価格付き）
- **アクションボタン** - カートに追加(<Icon name="circle-plus" color="green" />)またはバリエーションを選択(<Icon name="circle-arrow-right" color="green" />)

### 商品タイプ

- **シンプル商品**は、カートに直接追加するための緑色の<Icon name="circle-plus" color="green" />ボタンを表示します
- **バリエーション商品**は、バリエーションセレクターを開くための緑色の<Icon name="circle-arrow-right" color="green" />矢印を表示します

詳細については[バリエーション商品](/pos/product-panel/variations)を参照してください。

### フッター

プロダクトパネルの下部には：

- **税金ステータス** - 現在の税計算基準を表示（例："税金の基準: 店舗の基本住所"）。アクティブな税率を表示するにはクリックしてください。
- **商品数** - 表示されている商品の数を示します（例："17のうちの10を表示"）
- **同期ボタン** - サーバーから商品を更新します

## 表示設定

スライダーのアイコン(<Icon name="sliders" />)をクリックして、プロダクトパネルの表示をカスタマイズします。

### 在庫切れ商品を表示

現在在庫切れの商品が表示されるかどうかを制御します。

- **有効**: 在庫切れの商品は表示されますが、カートに追加することはできません
- **無効**: 在庫切れの商品はリストから非表示になります

### 列

商品リストに表示される列を設定します：

| 列      | 説明             | 表示オプション                       |
|---------|------------------|-------------------------------------|
| **画像**    | 商品のサムネイル    | -                                   |
| **商品**    | 商品名と詳細       | 在庫、SKU、バーコード、カテゴリー、タグ、ブランド、属性、メタデータ |
| **SKU**     | 在庫管理単位（別列） | -                                   |
| **バーコード** | 商品バーコード（別列） | -                                   |
| **タイプ**   | 商品タイプ（シンプル、バリエーションなど） | -                                   |
| **在庫**    | 現在の在庫数量      | -                                   |
| **売上原価** | 商品の原価価格      | -                                   |
| **価格**    | 販売価格           | 税情報、セールインジケーター             |
| **アクション** | カートに追加ボタン   | -                                   |

### メタデータキー

商品がカートに追加されたときに、注文の行アイテムに転送されるべき商品メタデータキーを指定します。

カンマ区切りのメタキーのリストを入力します：

```
_bottle_deposit,_custom_field,_tracking_code
```

**使用例:**
- ボトルデポジットプラグイン
- カスタム商品フィールド
- 第三者統合データ
- 準拠/追跡情報

:::note
メタキーは大文字と小文字を区別し、商品メタデータに保存されている通りに正確に一致する必要があります。
:::

### デフォルト設定の復元

すべての表示設定を元のデフォルトにリセットするにはクリックしてください。

## カートへの商品追加

### シンプル商品

緑色の<Icon name="circle-plus" color="green" />ボタンをクリックしてシンプル商品をカートに追加します。クリックするたびに1つの単位が追加されます。

### バリエーション商品

バリエーション商品（例：サイズと色の選択肢があるTシャツ）は、緑色の<Icon name="circle-arrow-right" color="green" />矢印を表示します。クリックして：

1. **クイックポップオーバー** - ドロップダウンからバリエーションを選択
2. **インライン展開** - "展開"をクリックしてリスト内のすべてのバリエーションを表示

詳細については[バリエーション商品](/pos/product-panel/variations)を参照してください。

### バーコードスキャン

USBまたはBluetoothバーコードスキャナーを接続して、商品を迅速に追加します。バーコードをスキャンすると、WCPOSは自動的に一致する商品を検索して追加します。

設定手順については[バーコードスキャン](/pos/product-panel/barcode-scanning)を参照してください。
