:::info プロ機能
顧客画面は[WCPOS Pro](/getting-started/pro-license)が必要です。無料ユーザーは既存の顧客を注文に選択できますが、すべての顧客リストを表示したり、顧客詳細を編集することはできません。
:::

顧客画面は、POS内で直接顧客管理を提供します。WooCommerceの管理画面に切り替えることなく、顧客を表示、編集、作成できます。

{/* スクリーンショット: customers-screen.png */}

## インターフェースの概要

### ヘッダーアクション

画面の上部には：

- **検索バー** - 名前、メールなどで顧客を検索
- **顧客の追加** (<Icon name="plus" />) - 新しい顧客を作成
- **表示設定** (<Icon name="sliders" />) - 表示する列を設定

### 顧客テーブル

メインエリアには顧客が表示され：

- **アバター** - 顧客のプロフィール写真（またはプレースホルダー）
- **名** - 顧客の名
- **姓** - 顧客の姓（ソート可能）
- **メール** - 連絡用メールアドレス
- **請求先住所** - 完全な請求先住所
- **作成日** - 顧客が追加された日
- **アクション** - 三点メニュー

### フッター

- 同期ボタン付きの顧客数

## 主な機能

### 顧客検索

顧客を素早く見つける方法：

- 名または姓
- メールアドレス
- 住所情報

### 新しい顧客の追加

POSから直接顧客を作成：

1. ヘッダーの<Icon name="plus" />アイコンをクリック
2. 顧客の詳細を入力
3. 新しい顧客を保存

顧客はWooCommerceに作成され、すぐに注文に利用可能です。

### 顧客の編集

顧客情報を更新：

1. 顧客の三点メニューをクリック
2. **編集**を選択
3. 詳細を修正（名前、メール、住所）
4. 変更を保存

変更はWooCommerceに自動的に同期されます。

### 利用可能な列

| 列       | 説明                |
|----------|---------------------|
| **画像**     | 顧客のアバター        |
| **ID**      | WooCommerce顧客ID   |
| **名**      | 顧客の名            |
| **姓**      | 顧客の姓            |
| **メール**    | メールアドレス        |
| **役割**     | ユーザーの役割（顧客、サブスクライバーなど） |
| **ユーザー名** | WordPressユーザー名  |
| **請求先住所** | 請求情報            |
| **配送先住所** | 配送情報            |

| **最終更新日** | 最後の更新         |
| **アクション** | 編集、同期、削除     |

## 顧客アクション

三点メニュー(⋮)をクリックしてオプションを表示：

- **編集** - 顧客の詳細を修正
- **同期** - サーバーから顧客を更新
- **削除** - ローカルデータベースから削除

:::note
POSから顧客を削除すると、ローカルでのみ削除されます。顧客はWooCommerceに残り、次回の同期時に再表示されます。
:::

## 注文での顧客の使用

顧客を注文に割り当てるには：

1. [カートパネル](/pos/cart)で顧客バッジをクリック
2. 名前またはメールで顧客を検索
3. 顧客を選択
4. 今、顧客がその注文に関連付けられます

顧客情報（請求先/配送先住所）は自動的に注文に使用されます。

## 同期

顧客データはPOSとWooCommerceの間で同期されます：

- **リアルタイム更新** - 変更がすべてのデバイスに同期
- **WooCommerce統合** - 顧客はWooCommerceに保存
- **オフライン機能** - オフラインでも顧客を表示可能
