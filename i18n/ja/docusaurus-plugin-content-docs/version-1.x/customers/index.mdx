:::info プロ機能
顧客画面には[WCPOS Pro](/getting-started/pro-license)が必要です。無料ユーザーは注文に既存の顧客を選択できますが、顧客リスト全体を表示したり、顧客詳細を編集したりすることはできません。
:::

顧客画面では、POSから直接包括的な顧客管理を提供します。WooCommerceの管理画面に切り替えることなく、顧客を表示、編集、作成できます。

{/* スクリーンショット: customers-screen.png */}

## インターフェース概要

### ヘッダーアクション

画面上部：

- **検索バー** - 名前、メールなどで顧客を検索
- **顧客追加** (<Icon name="plus" />) - 新規顧客作成
- **表示設定** (<Icon name="sliders" />) - 表示列の設定

### 顧客テーブル

メインエリアには以下の顧客情報が表示されます：

- **アバター** - 顧客のプロフィール画像（またはプレースホルダー）
- **名前** - 顧客の名前
- **苗字** - 顧客の苗字（ソート可能）
- **メール** - 連絡先メールアドレス
- **請求先住所** - 完全な請求先住所
- **作成日** - 顧客が追加された日
- **アクション** - 三点メニュー

### フッター

- 同期ボタン付きの顧客数

## 主な機能

### 顧客検索

以下で迅速に顧客を検索：

- 名前または苗字
- メールアドレス
- 住所情報

### 新規顧客追加

POSから直接顧客を作成：

1. ヘッダーの <Icon name="plus" /> アイコンをクリック
2. 顧客詳細を入力
3. 新規顧客を保存

顧客はWooCommerceに作成され、すぐに注文に使用できます。

### 顧客編集

顧客情報を更新：

1. 顧客の三点メニューをクリック
2. **編集**を選択
3. 詳細を変更（名前、メール、住所）
4. 変更を保存

変更は自動的にWooCommerceと同期されます。

### 利用可能な列

| 列 | 説明 |
|--------|-------------|
| **画像** | 顧客アバター |
| **ID** | WooCommerce顧客ID |
| **名前** | 顧客の名前 |
| **苗字** | 顧客の苗字 |
| **メール** | メールアドレス |
| **役割** | ユーザー役割（顧客、購読者など） |
| **ユーザー名** | WordPressユーザー名 |
| **請求先住所** | 請求情報 |
| **配送先住所** | 配送情報 |

| **更新日** | 最終更新日 |
| **アクション** | 編集、同期、削除 |

## 顧客アクション

三点メニュー（⋮）をクリックしてオプションを選択：

- **編集** - 顧客詳細を変更
- **同期** - サーバーから顧客情報を更新
- **削除** - ローカルデータベースから削除

:::note
POSから顧客を削除しても、ローカルからのみ削除されます。顧客はWooCommerceに残り、次の同期時に再表示されます。
:::

## 注文での顧客使用

注文に顧客を割り当てるには：

1. [カートパネル](/pos/cart)で顧客バッジをクリック
2. 名前またはメールで顧客を検索
3. 顧客を選択
4. 顧客が注文に関連付けられます

顧客情報（請求/配送先住所）が自動的に注文に使用されます。

## 同期

顧客データはPOSとWooCommerceの間で同期：

- **リアルタイム更新** - 変更は全デバイスに同期
- **WooCommerce連携** - 顧客はWooCommerceに保存
- **オフライン機能** - オフライン時も顧客を表示可能
