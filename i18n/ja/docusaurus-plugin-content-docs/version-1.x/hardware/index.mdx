WCPOS は、店舗のカウンターで既に使用している物理デバイスと連携します。このページは、レシートプリンター、バーコードスキャナー、カードリーダー、キャッシュドロワーなどのハードウェア接続を始めるための出発点です。各デバイス種別には、ここに専用の設定ページがあるか、実際の設定手順が掲載されているページへの案内があります。

:::note デバイスごとに設定場所は1つ
ハードウェアハブから各デバイスの設定へ移動できますが、実際の手順は1か所にまとめられています。カードリーダーのアカウント設定とペアリングの流れはその決済ゲートウェイページにあり、キャッシュドロワーを開く設定はレシートプリンターページにあります。
:::
