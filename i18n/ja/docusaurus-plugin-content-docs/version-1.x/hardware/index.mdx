:::note デバイスごとに1つの設定場所
ハードウェアハブでは各デバイスの設定先を案内していますが、実際の設定手順はそれぞれ1か所にまとめられています。カードリーダーのアカウントとペアリングの手順は対応する決済ゲートウェイページに、キャッシュドロワーのキック設定はレシートプリンターページにあります。
:::

## カードリーダーの互換性 {#card-reader-compatibility}

使用できるカードリーダーは、**WCPOSの利用環境**とリーダーの接続方式によって異なります。

- **Webアプリ**では、**Web（ブラウザ）SDK**を提供するリーダーのみ使用できます。通常はインターネット接続型の据え置きターミナルが該当します。**Bluetoothのみ**のリーダー（例：Stripe M2 / WisePad 3、SumUp Air）やスマートフォンの**タッチ決済**はWebアプリでは**対応していません**。
- 対応状況は**ゲートウェイごと**に異なります。購入前に、使用予定のゲートウェイ（[Stripe Terminal](/payment/gateways/stripe-terminal)、[SumUp](/payment/gateways/sumup-terminal)、[PayPal Reader](/payment/gateways/paypal-reader)）のページでリーダーの対応状況を確認してください。

各ゲートウェイページに対応リーダーの一覧が記載されています。
