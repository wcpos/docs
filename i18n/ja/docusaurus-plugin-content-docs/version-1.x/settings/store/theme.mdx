## 利用可能なテーマ

### システムテーマ

- **説明**: デバイス設定に従う
- **動作**: デバイスのシステム設定に基づいて自動的にライトモードとダークモードを切り替えます

### ライトテーマ

- **説明**: クリーンで明るい
- **最適な用途**: 照明が良好な環境や伝統的なライトインターフェースを好むユーザー

### ダークテーマ

- **説明**: 目に優しい
- **最適な用途**: 照明が少ない環境や長時間の使用時に目の疲れを軽減

### オーシャンテーマ

- **説明**: クールな青とティール
- **最適な用途**: リラックスしたプロフェッショナルな雰囲気を作り出す

### サンセットテーマ

- **説明**: 暖かいオレンジと紫
- **最適な用途**: 居心地の良い招待的な雰囲気を作り出す

### モノクロームテーマ

- **説明**: 高コントラストのグレースケール
- **最適な用途**: アクセシビリティのニーズや高コントラストが必要な環境

## テーマの選択

1. テーマ設定パネルには、利用可能なすべてのテーマがカードとして表示されます
2. 各テーマカードにはテーマ名と簡単な説明が表示されます
3. 選択したいテーマカードをクリックします
4. 現在のアクティブテーマは青い境界線で強調表示されます
5. 選択は自動的に保存され、すぐに適用されます

## テーマ選択のためのヒント

| テーマ | 最適な環境 |
|-------|------------------|
| **システム** | POSがデバイスに自動的に合わせてほしいとき |
| **ライト** | 照明が良好な明るい小売環境 |
| **ダーク** | 照明が薄い環境や夕方/夜のシフト |
| **オーシャン/サンセット** | 個性を加え、ブランド調整をするため |
| **モノクローム** | 最大コントラストが必要な場合 |

選択したテーマは、すべてのPOSセッションにわたって持続します。
