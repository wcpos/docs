## 利用可能なテーマ

### システムテーマ

- **説明**: デバイスの設定に従います
- **動作**: デバイスのシステム設定に基づいて、自動的にライトモードとダークモードを切り替えます

### ライトテーマ

- **説明**: クリーンで明るい
- **最適な環境**: 明るい環境および従来のライトインターフェースを好むユーザー

### ダークテーマ

- **説明**: 目に優しい
- **最適な環境**: 低照度環境および長時間使用時の目の疲れを軽減

### オーシャンテーマ

- **説明**: クールな青とティール
- **最適な環境**: リラックスしたプロフェッショナルな雰囲気を作る

### サンセットテーマ

- **説明**: 暖かいオレンジと紫
- **最適な環境**: 居心地の良い、魅力的な雰囲気を作る

### モノクロームテーマ

- **説明**: 高コントラストのグレースケール
- **最適な環境**: アクセシビリティのニーズや高コントラストが必要な環境

## テーマの選択

1. テーマ設定パネルでは、すべての利用可能なテーマがカードとして表示されます
2. 各テーマカードにはテーマ名と簡単な説明が表示されます
3. 任意のテーマカードをクリックして選択します
4. 現在のアクティブなテーマは青い枠で強調表示されます
5. あなたの選択は自動的に保存され、すぐに適用されます

## テーマ選択のためのヒント

| テーマ | 最適な環境 |
|-------|------------------|
| **システム** | POSをデバイスに自動的に合わせたいとき |
| **ライト** | 良好な照明の明るい小売環境 |
| **ダーク** | 薄暗い環境や夕方/夜間シフト |
| **オーシャン/サンセット** | 個性を加え、ブランドの整合性を持たせる |
| **モノクローム** | 最大のコントラストを必要とする場合 |

選択したテーマはすべてのPOSセッションを通じて持続します。
