## 利用可能なテーマ

### システムテーマ

- **説明**: デバイス設定に従う
- **動作**: デバイスのシステム設定に基づいて、自動的にライトモードとダークモードを切り替え

### ライトテーマ

- **説明**: クリーンで明るい
- **推奨環境**: 明るい環境と、従来のライトインターフェースを好むユーザー向け

### ダークテーマ

- **説明**: 目に優しい
- **推奨環境**: 暗い環境と、長時間使用時の目の疲れを軽減

### オーシャンテーマ

- **説明**: 涼しげな青とティール
- **推奨環境**: リラックスした、プロフェッショナルな雰囲気の演出

### サンセットテーマ

- **説明**: 温かいオレンジと紫
- **推奨環境**: 居心地の良い、温かい雰囲気の演出

### モノクロームテーマ

- **説明**: 高コントラストのグレースケール
- **推奨環境**: アクセシビリティのニーズと、高コントラストが必要な環境

## テーマの選択

1. テーマ設定パネルで、すべての利用可能なテーマがカードで表示されます
2. 各テーマカードにはテーマ名と簡単な説明が表示されます
3. テーマカードをクリックして選択します
4. 現在のアクティブなテーマは青い枠線で強調表示されます
5. 選択はすぐに自動保存され、即座に適用されます

## テーマ選択のヒント

| テーマ | 推奨環境 |
|-------|------------------|
| **システム** | デバイスに自動的に合わせたい場合 |
| **ライト** | 十分な照明がある明るい小売環境 |
| **ダーク** | 薄暗い環境または夕方/夜間のシフト |
| **オーシャン/サンセット** | パーソナリティとブランドの一致 |
| **モノクローム** | 最大のコントラストが必要な場合 |

選択したテーマは、すべてのPOSセッションで持続します。
