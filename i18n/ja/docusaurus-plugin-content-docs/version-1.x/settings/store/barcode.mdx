## バーコード検出のしくみ

ほとんどのバーコードスキャナーは、デバイスに接続されたキーボードのように動作します。バーコードをスキャンすると、POSは文字が通常のタイピングよりも速く入力されたことを検出します。これらの「高速キー入力」を使用して、入力がバーコードスキャンであることを識別します。

## 設定

| 設定 | 説明 |
|---------|-------------|
| **平均入力時間** | バーコードとして認識するための入力速度（ミリ秒）。値が低いほど、より速い入力が必要です |
| **最小長** | バーコードとして扱う連続した文字列の長さ |
| **プレフィックス削除** | スキャンされたバーコードの先頭から削除する文字 |
| **サフィックス削除** | スキャンされたバーコードの末尾から削除する文字 |

## プレフィックス/サフィックスの削除

スキャナーが追加文字（プレフィックスやサフィックス）を追加する場合、それらを自動的に削除し、メインのバーコードのみを残すことができます。

**例:** スキャナーがすべてのバーコードの前に「A」、末尾に「Z」を追加する場合、以下のように設定します：
- プレフィックス削除: `A`
- サフィックス削除: `Z`

## トラブルシューティング

### バーコードが検出されない

- **スキャナーが遅い**: 「平均入力時間」の値を増やしてみてください
- **バーコードが短すぎる**: 「最小長」の値を減らしてみてください
- **余分な文字がある**: スキャナーがプレフィックス/サフィックスを追加していないか確認してください

### 通常のタイピングがバーコードとして検出される

- **タイピングが速すぎる**: 「平均入力時間」の値を減らしてみてください
- **入力が長すぎる**: 「最小長」の値を増やしてみてください
