## バーコード検出の仕組み

ほとんどのバーコードスキャナーは、デバイスに接続されたキーボードのように動作します。バーコードをスキャンすると、POSは通常の入力よりも速く文字が入力されたことを検出します。これらの「高速キー入力」を利用して、入力をバーコードスキャンとして識別します。

## 設定

| 設定 | 説明 |
|---------|-------------|
| **平均入力時間** | バーコードとしてカウントされるための入力速度（ミリ秒単位）。低い値 = より速い入力が必要 |
| **最小長** | バーコードとして扱われるために必要な連続した文字列の長さ |
| **プレフィックス除去** | スキャンしたバーコードの先頭から除去する文字 |
| **サフィックス除去** | スキャンしたバーコードの末尾から除去する文字 |

## プレフィックス/サフィックス除去

スキャナーが余分な文字（プレフィックスやサフィックス）を追加する場合、自動でこれらを除去できるので、主要なバーコードのみが残ります。

**例:** スキャナーがすべてのバーコードの前に「A」を追加し、後に「Z」を追加する場合、次のように設定します：
- プレフィックス除去: `A`
- サフィックス除去: `Z`

## トラブルシューティング

### バーコードが検出されない

- **スキャナーが遅すぎる**: 「平均入力時間」値を増やしてみてください
- **バーコードが短すぎる**: 「最小長」値を減らしてみてください
- **余分な文字**: スキャナーがプレフィックスやサフィックスを追加しているか確認してください

### 通常の入力がバーコードとして検出される

- **入力が速すぎる**: 「平均入力時間」値を減らしてみてください
- **入力が長すぎる**: 「最小長」値を増やしてみてください
