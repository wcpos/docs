## 利用可能なショートカット

| キーの組み合わせ | アクション | 説明 |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 設定 | POS設定パネルを開く |
| `Ctrl + Shift + A` | POS | メインPOS画面に移動 |
| `Ctrl + Shift + P` | 製品 | 製品管理画面を開く |
| `Ctrl + Shift + O` | 注文 | 注文管理画面を開く |
| `Ctrl + Shift + C` | 顧客 | 顧客管理画面を開く |
| `Ctrl + Shift + ?` | サポート | サポート/ヘルプセクションを開く |
| `Ctrl + Shift + L` | ログアウト | 現在のセッションからログアウト |

## キーボードショートカットの使用

キーボードショートカットを使用するには：

1. **修飾キーを押し続ける** (`Ctrl + Shift`)
2. **アクションキーを押す**（希望するアクションに対応する文字）
3. **すべてのキーを放して** コマンドを実行

## 効率的な使用のためのヒント

- **よく使うショートカットを覚える**: 最も頻繁に行うアクションのショートカットを学ぶことに集中する
- **キーの組み合わせを練習する**: 定期的な使用が筋肉記憶を構築するのに役立つ
- **混雑時にショートカットを使用する**: キーボードショートカットは、ピーク時の操作を大幅に高速化できる

## カスタマイズ

:::info 機能リクエスト
現在、キーボードショートカットのカスタマイズは利用できません。上記にリストされたショートカットは固定されており、設定インターフェイスを通じて変更することはできません。

キーボードショートカットをカスタマイズしたい場合は、[Discord](https://wcpos.com/discord)でフィードバックを共有してください。
:::
