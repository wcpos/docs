## 利用可能なショートカット

| キーの組み合わせ | アクション | 説明 |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 設定 | POSの設定パネルを開きます |
| `Ctrl + Shift + A` | POS | メインPOS画面に移動します |
| `Ctrl + Shift + P` | 商品 | 商品管理画面を開きます |
| `Ctrl + Shift + O` | 注文 | 注文管理画面を開きます |
| `Ctrl + Shift + C` | 顧客 | 顧客管理画面を開きます |
| `Ctrl + Shift + ?` | サポート | サポート/ヘルプセクションを開きます |
| `Ctrl + Shift + L` | ログアウト | 現在のセッションからログアウトします |

## キーボードショートカットの使用方法

キーボードショートカットを使用するには：

1. **修飾キーを押したままにする** (`Ctrl + Shift`)
2. **アクションキーを押す** (目的のアクションに対応する文字)
3. **すべてのキーを離す** とコマンドが実行されます

## 効率的な使用のためのヒント

- **よく使うショートカットを覚える**: 最も頻繁に実行するアクションのショートカットに焦点を当てる
- **キーの組み合わせを練習する**: 定期的に使用することで、筋肉の記憶を形成できます
- **ピーク時にショートカットを使用する**: キーボードショートカットは、繁忙時に操作を大幅に高速化できます

## カスタマイズ

:::info 機能リクエスト
キーボードショートカットのカスタマイズは現在利用できません。上記のショートカットは固定されており、設定インターフェースから変更することはできません。

キーボードショートカットをカスタマイズしたい場合は、[Discord](https://wcpos.com/discord)でフィードバックをお寄せください。
:::
