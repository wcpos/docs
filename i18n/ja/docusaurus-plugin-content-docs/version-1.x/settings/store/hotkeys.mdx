## 利用可能なショートカット

| キーの組み合わせ | アクション | 説明 |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | 設定 | POS設定パネルを開く |
| `Ctrl + Shift + A` | POS | メインPOS画面に移動 |
| `Ctrl + Shift + P` | 商品 | 商品管理画面を開く |
| `Ctrl + Shift + O` | 注文 | 注文管理画面を開く |
| `Ctrl + Shift + C` | 顧客 | 顧客管理画面を開く |
| `Ctrl + Shift + ?` | サポート | サポート/ヘルプセクションを開く |
| `Ctrl + Shift + L` | ログアウト | 現在のセッションからログアウト |

## キーボードショートカットの使用方法

任意のキーボードショートカットを使用するには：

1. **修飾キーを押し続ける** (`Ctrl + Shift`)
2. **アクションキーを押す**（目的のアクションに対応する文字）
3. **すべてのキーを放す**ことでコマンドを実行

## 効率的な使用のためのヒント

- **頻繁に使用するショートカットを覚える**：最も頻繁に行うアクションのショートカットを学ぶことに集中する
- **キーの組み合わせを練習する**：定期的な使用が筋肉記憶を育てるのに役立つ
- **忙しい時間帯にショートカットを使用する**：キーボードショートカットはピーク時の操作を大幅にスピードアップできます

## カスタマイズ

:::info 機能リクエスト
キーボードショートカットのカスタマイズは現在利用できません。上記のショートカットは固定されており、設定インターフェースから修正することはできません。

キーボードショートカットをカスタマイズしたい場合は、[Discord](https://wcpos.com/discord)でフィードバックを共有してください。
:::
