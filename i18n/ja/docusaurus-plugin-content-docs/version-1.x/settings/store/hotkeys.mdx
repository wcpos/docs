## Available Shortcuts

| Key Combination | Action | Description |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | Settings | Opens the POS settings panel |
| `Ctrl + Shift + A` | POS | Navigates to the main POS screen |
| `Ctrl + Shift + P` | Products | Opens the products management screen |
| `Ctrl + Shift + O` | Orders | Opens the orders management screen |
| `Ctrl + Shift + C` | Customers | Opens the customers management screen |
| `Ctrl + Shift + ?` | Support | Opens the support/help section |
| `Ctrl + Shift + L` | Logout | Logs out of the current session |

## キーボードショートカットの使用 {#using-keyboard-shortcuts}

キーボードショートカットを使用するには：

1. **修飾キーを押し続ける** (`Ctrl + Shift`)
2. **アクションキーを押す**（希望するアクションに対応する文字）
3. **すべてのキーを放して** コマンドを実行

## 効率的な使用のためのヒント {#tips-for-efficient-use}

- **頻繁に使うショートカットを暗記する**: 最も頻繁に行うアクションのショートカットを学ぶことに集中する
- **キーの組み合わせを練習する**: 定期的な使用が筋肉記憶を構築するのに役立つ
- **混雑時にショートカットを使用する**: キーボードショートカットは、ピーク時の操作を大幅に高速化できる

## カスタマイズ {#customisation}

:::info Feature Request
現在、キーボードショートカットのカスタマイズは利用できません。上記にリストされたショートカットは固定されており、設定インターフェイスを通じて変更することはできません。

キーボードショートカットをカスタマイズしたい場合は、[Discord](https://wcpos.com/discord)でフィードバックを共有してください。
:::
