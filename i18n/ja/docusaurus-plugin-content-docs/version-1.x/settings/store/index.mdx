店舗設定は、POSの動作に影響するアプリ全体の設定です。これらの設定は、個々の画面ではなく、店舗全体に適用されます。

## 店舗設定へのアクセス

店舗設定を開くには：

- **ユーザーメニュー**（右上隅）をクリックし、**設定**を選択
- キーボードショートカット `Ctrl + Shift + S` を使用

## 設定タブ

店舗設定モーダルには、次のタブがあります：

| タブ | 説明 |
|-----|-------------|
| [一般](./general) | 店舗情報、言語、通貨、デフォルト顧客 |
| [税金](./tax) | 税金の計算と表示オプション |
| [バーコード](./barcode) | バーコードスキャナーの設定 |
| [キーボードショートカット](./hotkeys) | 利用可能なキーボードショートカット |
| [テーマ](./theme) | 外観とカラースキーム |
