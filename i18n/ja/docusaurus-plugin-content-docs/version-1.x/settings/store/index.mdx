店舗設定は、POS の動作に影響するアプリ全体の構成です。これらの設定は個別の画面ではなく、店舗全体に適用されます。

## 店舗設定へのアクセス {#accessing-store-settings}

店舗設定を開くには:

- **ユーザーメニュー**（右上隅）をクリックし、**設定**を選択します
- キーボードショートカット `Ctrl + Shift + S` を使用します

## 設定タブ {#settings-tabs}

店舗設定モーダルには次のタブがあります:

| タブ | 説明 |
|-----|-------------|
| [一般](./general) | 店舗情報、言語、通貨、デフォルト顧客 |
| [税](./tax) | 税額計算と表示オプション |
| [バーコード](./barcode) | バーコードスキャナーの設定 |
| [キーボードショートカット](./hotkeys) | 利用可能なキーボードショートカット |
| [テーマ](./theme) | 外観とカラースキーム |
| [プリンター](/hardware/printers) | レシートプリンターの接続と設定 |
