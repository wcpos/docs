ストア設定は、POSの動作に影響を与えるアプリ全体の構成です。これらの設定は、個々の画面ではなく、ストア全体に適用されます。

## ストア設定へのアクセス

ストア設定を開くには：

- **ユーザーメニュー**（右上隅）をクリックして**設定**を選択します
- キーボードショートカット`Ctrl + Shift + S`を使用します

## 設定タブ

ストア設定モーダルには、次のタブがあります：

| タブ | 説明 |
|-----|-------------|
| [一般](./general) | ストア情報、言語、通貨、デフォルト顧客 |
| [税金](./tax) | 税金計算および表示オプション |
| [バーコード](./barcode) | バーコードスキャナーの設定 |
| [キーボードショートカット](./hotkeys) | 利用可能なキーボードショートカット |
| [テーマ](./theme) | 外観とカラースキーム |
