:::warning 重要
初期ロード時、POSはWooCommerceからデフォルトの税設定を取得します。一般的に、ユーザーは特別な理由がない限り、WooCommerceと同じ設定を維持する必要があります。サーバーと異なる税設定は、POSとWooCommerceで異なる税金の計算を引き起こし、注文の同期に問題を生じる可能性があります。
:::

## すべての税率を表示

**すべての税率を表示**ボタンをクリックして、POSにダウンロードされたすべての税率を確認します。以下が表示されます：

- 税率名
- 税率の割合
- 適用される地域/場所

これは、正しい税率が読み込まれているか確認し、税金計算の問題をデバッグするのに役立ちます。

## 税金計算設定

### 税込み価格で入力

製品価格に税金が含まれるかどうかを決定します：

| オプション | 説明 |
|--------|-------------|
| **はい** | 製品価格に税金が含まれます。POSは総価格から逆算して税金を計算します |
| **いいえ** | 製品価格に税金は含まれません。POSは入力価格に税金を追加します |

### 税金計算の基準

税金計算に使用するアドレスを決定します：

| オプション | 説明 |
|--------|-------------|
| **顧客の配送先住所** | 注文が配送される場所に基づく税率 |
| **顧客の請求先住所** | 顧客の請求先住所に基づく税率 |
| **店舗の基本住所** | 店舗の物理的な所在地に基づく税率 |

### 配送の税区分

配送費用の課税方法を決定します：

| オプション | 説明 |
|--------|-------------|
| **カート内のアイテムの税区分に基づく配送税** | 配送税はカート内のアイテムの税区分に基づいて計算されます |
| **標準** | 配送は標準税率で課税されます |
| **軽減税率** | 配送は軽減税率で課税されます |
| **ゼロ税率** | 配送費用には税金が適用されません |

### 端数処理

| オプション | 説明 |
|--------|-------------|
| **小計レベルで税金を丸める** | 各明細行ではなく、小計全体に対して税金が計算され、丸められます |

## 表示設定

### ショップでの価格表示

| オプション | 説明 |
|--------|-------------|
| **税込み** | 表示される価格に税金が含まれます |
| **税抜き** | 表示される価格に税金が含まれません |

### カートとチェックアウト時の価格表示

| オプション | 説明 |
|--------|-------------|
| **税込み** | カートとチェックアウトの価格に税金が含まれます |
| **税抜き** | カートとチェックアウトの価格に税金が含まれません |

### 価格表示サフィックス

税の状態を示すために価格の後に表示されるテキスト（例：「税込」または「税抜」）。

### 税金合計の表示

| オプション | 説明 |
|--------|-------------|
| **1つの合計として** | すべての税金が1つの合計行に統合されます |
| **明細化** | 各税率が別の明細行として表示されます |

## サーバー設定の復元

クリックして、サーバーから現在のWooCommerce税設定をフェッチし、適用します。

:::tip ベストプラクティス
WordPress管理画面で税設定を変更した後は、POSの税金設定をWooCommerceストアと同期させるために、「サーバー設定の復元」を定期的に使用してください。
:::
