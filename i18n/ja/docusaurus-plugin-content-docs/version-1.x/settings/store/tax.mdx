:::warning 重要
初回読み込み時に、POSはWooCommerceからデフォルトの税設定を取得します。一般的に、特別な必要がない限り、ユーザーはWooCommerceと同じ設定を維持するべきです。サーバーと異なる税設定を持つと、POSとWooCommerceが異なる税金を計算することになり、差異や注文同期の問題が発生する可能性があります。
:::

## すべての税率を表示

**すべての税率を表示**ボタンをクリックして、POSにダウンロードされたすべての税率を確認します。これには以下が含まれます：

- 税率名
- 税率の百分率
- 適用される地域/場所

これは、正しい税率が読み込まれているかを確認し、税計算の問題をデバッグするのに役立ちます。

## 税計算設定

### 税込価格で入力された価格

商品価格に税金が含まれているかどうかを決定します：

| オプション | 説明 |
|------------|-------------|
| **はい** | 商品価格に税金が含まれています。POSは総価格から逆算して税金を計算します。 |
| **いいえ** | 商品価格に税金が含まれていません。POSは入力された価格に税金を追加します。 |

### 税計算の基準

税計算に使用する住所を決定します：

| オプション | 説明 |
|------------|-------------|
| **顧客の発送先住所** | 注文が発送される場所に基づく税率 |
| **顧客の請求先住所** | 顧客の請求先住所に基づく税率 |
| **店舗の基本住所** | 店舗の物理的な所在地に基づく税率 |

### 送料の税クラス

送料がどのように課税されるかを決定します：

| オプション | 説明 |
|------------|-------------|
| **カート内アイテムに基づく送料の税クラス** | 送料の税はカート内のアイテムの税クラスに基づいて計算されます。 |
| **標準** | 送料は標準税率で課税されます。 |
| **軽減税率** | 送料は軽減税率で課税されます。 |
| **ゼロ税率** | 送料には税金が適用されません。 |

### 四捨五入

| オプション | 説明 |
|------------|-------------|
| **小計レベルでの税金の四捨五入** | 税金は全体の小計に対して計算され、その後四捨五入されます。各項目ごとに税金を四捨五入するのではありません。 |

## 表示設定

### 店舗における価格の表示

| オプション | 説明 |
|------------|-------------|
| **税込み表示** | 表示される価格には税金が含まれます。 |
| **税抜き表示** | 表示される価格に税金は含まれません。 |

### カートとチェックアウト中の価格の表示

| オプション | 説明 |
|------------|-------------|
| **税込み表示** | カートとチェックアウトの価格には税金が含まれます。 |
| **税抜き表示** | カートとチェックアウトの価格には税金が含まれません。 |

### 価格表示の接尾辞

価格の後に表示されるテキスト、税金の状態を示します（例：「税込み」または「税抜き」）。

### 税金合計の表示

| オプション | 説明 |
|------------|-------------|
| **単一の合計として** | すべての税金が1つの合計行にまとめられます。 |
| **項目別** | 各税率が個別の項目として表示されます。 |

## サーバー設定の復元

クリックして、サーバーから現在のWooCommerce税設定を取得して適用します。

:::tip ベストプラクティス
WooCommerceストアの税設定に対してPOSの税構成が同期されるように、特にWordPress管理画面で税設定に変更を加えた後は「サーバー設定の復元」を定期的に使用してください。
:::
