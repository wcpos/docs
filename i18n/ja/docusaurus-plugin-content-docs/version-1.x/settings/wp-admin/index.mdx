これらの設定は、PHPおよびWooCommerce構成を含むPOSアプリケーションのバックエンド機能を構成します。
次のパスに移動することでアクセスできます: `WP Admin > POS > Settings`

主な構成項目は次のとおりです:

- すべてのレジ係のデフォルトカート顧客。
- 商品バーコードフィールド。
- 注文メール設定。
- ユーザーロールの権限。
