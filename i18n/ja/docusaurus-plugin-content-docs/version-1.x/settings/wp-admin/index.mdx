これらの設定は、POSアプリケーションのバックエンド機能を構成し、PHPおよびWooCommerceの設定を含みます。
これらには、次の手順でアクセスできます： `WP Admin > POS > Settings`

主要な設定項目は以下の通りです：

- すべてのキャッシャーのデフォルトのカート顧客。
- 商品バーコードフィールド。
- 注文メール設定。
- ユーザーロールの権限。
