これらの設定は、POSアプリケーションのバックエンド機能（PHPおよびWooCommerceの設定）を構成します。
`WP管理画面 > POS > 設定`から設定にアクセスできます。

主な設定項目は以下の通りです：

- すべてのレジ担当者のデフォルト顧客カート
- 商品バーコードフィールド
- 注文メール設定
- ユーザーロール権限
