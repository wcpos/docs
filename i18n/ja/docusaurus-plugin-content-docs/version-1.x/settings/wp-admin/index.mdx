これらの設定は、PHPおよびWooCommerce構成を含むPOSアプリケーションのバックエンド機能を構成します。
`WP Admin > POS > Settings`に移動することでアクセスできます。

主な設定項目には以下が含まれます：

- すべてのキャッシャーのデフォルトカート顧客。
- 商品バーコードフィールド。
- 注文メール設定。
- ユーザーロールの権限。
