WCPOSには、設定が構成される場所に基づいて整理された3種類の設定があります。

## 設定の見つけ方

### 1. WordPress管理設定

あなたのWordPressダッシュボードで構成されるバックエンド設定：

**場所:** `WP Admin > POS > 設定`

これらは次のことを制御します：
- すべてのレジ係のためのデフォルトのカート顧客
- 製品バーコードフィールドの設定
- 注文メール設定
- ユーザーロールの権限（POSにアクセスできる人）
- 決済ゲートウェイの可用性

詳細については、[WordPress管理設定](/settings/wp-admin)を参照してください。

### 2. POS店舗設定

POSアプリケーション内で構成されるアプリ全体の設定：

**場所:** ユーザーメニュー（右上）→ 設定、または `Ctrl + Shift + S`

これらは次のことを制御します：
- 税金計算オプション
- バーコードスキャナーの設定
- キーボードショートカット
- テーマ/外観

詳細については、[店舗設定](/settings/store)を参照してください。

### 3. ディスプレイ設定（画面ごと）

POSの各画面には、自分のデータテーブルをカスタマイズするためのディスプレイ設定があります：

**場所:** いずれかの画面でスライダーアイコン(<Icon name="sliders" />)をクリック

これらは次のことを制御します：
- 表示される列
- 列の表示オプション
- 画面特有の機能

ディスプレイ設定は各画面のドキュメント内で文書化されています：
- [POS製品パネル](/pos/product-panel#display-settings)
- [カートパネル](/pos/cart#display-settings)
- [製品画面](/products#display-settings)
- [注文画面](/orders#display-settings)
- [顧客画面](/customers#display-settings)
- [レポート画面](/reports#display-settings)
