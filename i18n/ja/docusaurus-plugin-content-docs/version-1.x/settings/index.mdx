WCPOSには、設定が構成されている場所によって整理された3種類の設定があります。

## 設定の見つけ方 {#where-to-find-settings}

### 1. WordPress管理設定 {#1-wordpress-admin-settings}

WordPressダッシュボードで構成されるバックエンド設定：

**場所:** `WP Admin > POS > Settings`

これらは以下を制御します：
- すべてのキャッシャーのデフォルトカート顧客
- 商品バーコードフィールドの構成
- 注文メール設定
- ユーザーロールの権限（誰がPOSにアクセスできるか）
- 支払いゲートウェイの利用可能性

詳細については[WordPress管理設定](/settings/wp-admin)をご覧ください。

### 2. POS店舗設定 {#2-pos-store-settings}

POSアプリケーション内で構成されたアプリ全体の設定：

**場所:** ユーザーメニュー（右上）→ 設定、または`Ctrl + Shift + S`

これらは以下を制御します：
- 税計算オプション
- バーコードスキャナーの構成
- キーボードショートカット
- テーマ/外観

詳細については[店舗設定](/settings/store)をご覧ください。

### 3. 表示設定（スクリーンごと） {#3-display-settings-per-screen}

POSの各スクリーンには、データテーブルをカスタマイズするための独自の表示設定があります：

**場所:** 任意のスクリーンでスライダーアイコン(<Icon name="sliders" />)をクリック

これらは以下を制御します：
- 表示される列
- 列の表示オプション
- スクリーン固有の機能

表示設定は各スクリーンのドキュメント内に記載されています：
- [POS商品パネル](/pos/product-panel#display-settings)
- [カートパネル](/pos/cart#display-settings)
- [商品スクリーン](/products#display-settings)
- [注文スクリーン](/orders#display-settings)
- [顧客スクリーン](/customers#display-settings)
- [レポートスクリーン](/reports#display-settings)
