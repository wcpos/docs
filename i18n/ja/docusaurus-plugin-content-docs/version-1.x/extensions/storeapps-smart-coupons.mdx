# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/) で作成されたストアクレジットに対する WCPOS 互換性を追加します。POS 注文で StoreApps ストアクレジットを利用し、StoreApps の残高を同期し、スタッフ向けに注文メモの監査証跡を残します。

この拡張機能は StoreApps / WooCommerce.com の Smart Coupons プラグイン専用です。類似の「Smart Coupons」という名称を持つ他のプラグインは別の製品であり、個別の連携が必要になる場合があります。

:::info Pro 機能
ディレクトリからの拡張機能のインストールと管理には [WCPOS Pro](/getting-started/pro-license) が必要です。
:::

## 機能概要 {#what-it-does}

- 通常のチェックアウトカートのコンテキストが利用できない場合に、POS で作成された注文に対して StoreApps の `smart_coupons_contribution` メタデータを記録します。
- POS チェックアウト後に、クーポン残高から正しいストアクレジットの一部金額を StoreApps が差し引けるようにします。
- POS注文がWooCommerceの通常の注文ライフサイクルでキャンセルまたは返金された場合に、StoreAppsのストアクレジット残高を復元します。
- クーポン適用後、クーポンコード、使用金額、現在の残高を記載した非公開の注文メモを追加します。
- 既存のレシートの割引行に、クーポンの説明/割引ラベルに付加する形でストアクレジットの残高を表示します。

この拡張機能はWooCommerceの標準的なクーポンフィールドとメタデータを使用します。WCPOSのカスタムクーポンレスポンスフィールドの追加やカスタムレシートテンプレートは不要です。

## インストール {#installation}

<Steps>
  <StepCard number="1" title="StoreApps Smart Couponsをインストール">

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/)をインストールして有効化します。WooCommerceで通常どおりストアクレジットクーポンやギフトカードを作成してください。

  </StepCard>
  <StepCard number="2" title="WCPOS StoreApps Smart Couponsをインストール">

`WP Admin > POS > 設定 > 拡張機能`からインストールするか、[GitHubリリースページ](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases)から最新版をダウンロードし、`プラグイン > 新規追加 > プラグインのアップロード`からアップロードしてください。

  </StepCard>
  <StepCard number="3" title="POSにクーポンを同期" isLast>

POSを開いてクーポンを同期します。レジ担当者は通常の**クーポンを追加**フローからStoreAppsのストアクレジットクーポンを適用できます。

  </StepCard>
</Steps>

## レシート出力 {#receipt-output}

WCPOS のレシートテンプレートは、割引行ラベルを使用してクーポン割引をすでに印刷しています。レシートのレンダリング時に、この拡張機能は既存のラベルに現在の StoreApps 残高を追加します。

たとえば、クーポンの説明が **ギフトカード** の場合、次のように印刷されます：

```text
Gift card — Store credit balance: £61.50
```

クーポンに説明がない場合、残高テキストが割引ラベルとして使用されます。既存のレシートテンプレートを変更する必要はありません。

## 注文監査証跡 {#order-audit-trail}

StoreApps が POS のストアクレジット利用を処理した後、この拡張機能は次のような非公開の注文メモを追加します：

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

このメモを使用して、どのストアクレジットクーポンが使用されたか、いくら利用されたか、販売後の残高がいくらだったかを確認できます。

## 互換性に関する注意事項 {#compatibility-notes}

- **StoreApps 専用：** この拡張機能は StoreApps / WooCommerce.com Smart Coupons を対象としています。WebToffee やその他のスマートクーポンプラグインには対応していません。
- **クーポンデータ：** WCPOS は引き続き通常の WooCommerce クーポンレスポンスを使用します。StoreApps 固有の状態は、`smart_coupons_contribution` やクーポン残高フィールドなどの WooCommerce メタデータに保持されます。
- **レシート：** 残高テキストは、レシートデータやテンプレートを変更するのではなく、既存のクーポン説明／割引ラベルのパスを通じて追加されます。
- **StoreApps プラグインが無効の場合：** StoreApps Smart Coupons が有効でない場合、この拡張機能は何も行いません。
