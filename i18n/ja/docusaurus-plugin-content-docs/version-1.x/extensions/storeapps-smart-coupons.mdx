# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/) で作成されたストアクレジットを WCPOS で利用できるようにします。POS 注文で StoreApps のストアクレジットを利用でき、StoreApps の残高を同期し、スタッフ向けに注文メモの監査記録を残します。

この拡張機能は、StoreApps / WooCommerce.com Smart Coupons プラグイン専用です。類似した「Smart Coupons」という名前の他のプラグインは別製品であり、個別の連携が必要になる場合があります。

:::info Pro 機能
ディレクトリから拡張機能をインストールして管理するには、[WCPOS Pro](/getting-started/pro-license) が必要です。
:::

## 機能 {#what-it-does}

- 通常のチェックアウトカートコンテキストを利用できない場合に、POS で作成された注文に StoreApps の `smart_coupons_contribution` メタデータを記録します。
- POS チェックアウト後に、StoreApps がクーポン残高から正しい一部ストアクレジット金額を差し引けるようにします。
- POS注文がWooCommerceの通常の注文ライフサイクルでキャンセルまたは返金されたときに、StoreAppsのストアクレジット残高を復元します。
- 利用後に、クーポンコード、使用額、現在の残高を示す非公開の注文メモを追加します。
- 残りのストアクレジット残高をクーポンの説明/割引ラベルに追加し、既存のレシート割引行に表示します。

この拡張機能は、通常のWooCommerceクーポンフィールドとメタデータを使用します。カスタムのWCPOSクーポン応答フィールドは追加せず、カスタムレシートテンプレートも必要ありません。

## インストール {#installation}

<Steps>
  <StepCard number="1" title="StoreApps Smart Couponsをインストール">

[Smart Coupons by StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/)をインストールして有効化します。WooCommerceで通常どおり、ストアクレジットクーポンまたはギフトカードを作成します。

  </StepCard>
  <StepCard number="2" title="WCPOS StoreApps Smart Couponsをインストール">

`WP Admin > POS > 設定 > 拡張機能`からインストールするか、[GitHubリリースページ](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases)から最新リリースをダウンロードし、`プラグイン > 新規追加 > プラグインのアップロード`からアップロードします。

  </StepCard>
  <StepCard number="3" title="クーポンをPOSに同期" isLast>

POSを開いてクーポンを同期します。その後、レジ担当者は通常の**クーポンを追加**フローからStoreAppsストアクレジットクーポンを適用できます。

  </StepCard>
</Steps>

## レシート出力 {#receipt-output}

WCPOS のレシートテンプレートは、すでに割引行ラベルを使用してクーポン割引を印刷します。レシートのレンダリング時に、この拡張機能は現在の StoreApps 残高を既存のラベルに追加します。

たとえば、クーポンの説明が **ギフトカード** の場合、次のように印刷されることがあります:

```text
Gift card — Store credit balance: £61.50
```

クーポンに説明がない場合、残高テキストが割引ラベルとして使用されます。既存のレシートテンプレートを変更する必要はありません。

## 注文監査履歴 {#order-audit-trail}

StoreApps が POS のストアクレジット利用を処理した後、この拡張機能は次のような非公開の注文メモを追加します:

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

このメモを使用して、どのストアクレジットクーポンが使用されたか、いくら利用されたか、販売後に残った残高を確認できます。

## 互換性に関する注意 {#compatibility-notes}

- **StoreApps のみ:** この拡張機能は StoreApps / WooCommerce.com Smart Coupons を対象としています。WebToffee やその他のスマートクーポンプラグインは対象外です。
- **クーポンデータ:** WCPOS は引き続き通常の WooCommerce クーポンレスポンスを使用します。StoreApps 固有の状態は、`smart_coupons_contribution` やクーポン残高フィールドなどの WooCommerce メタデータに保持されます。
- **レシート:** 残高テキストは、レシートデータやテンプレートを変更するのではなく、既存のクーポン説明/割引ラベルの経路を通じて追加されます。
- **非有効の StoreApps プラグイン:** StoreApps Smart Coupons が有効でない場合、この拡張機能は何もしません。
