# PY01003: カードの有効期限が切れました {#py01003-card-expired}

## これが意味すること {#what-this-means}

支払いカードの有効期限が切れており、取引に使用できなくなりました。

## 一般的な原因 {#common-causes}

- **期限切れのカード** — カードの有効期限が過ぎています
- **無効な有効期限が入力された** — 有効期限が間違って入力されました
- **新しいカードがまだ有効化されていない** — 代替カードが届いたが、まだ有効化されていない

## 修正方法 {#how-to-fix}

### 1. 有効期限を確認する {#1-check-the-expiration-date}

カードを見て：
- 有効期限を見つける（MM/YY形式）
- 現在の日付と比較する
- 入力された日付がカードと一致しているか確認する

### 2. 別のカードを使用する {#2-use-a-different-card}

カードが本当に期限切れの場合：
- 別の有効なカードを使用する
- 現金または代替の支払い方法を使用する
- 顧客に代替カードがあるか尋ねる

### 3. 更新されたカードを取得する {#3-get-the-updated-card}

カード所有者は：
- 代替カードが郵送されていないか確認する
- 受け取った場合は新しいカードを有効化する
- 代替カードが届かない場合は銀行に連絡する

### 4. 入力を確認する {#4-verify-entry}

有効期限の入力を再確認する：
- MM/YY形式
- 正しい月と年
- 誤字がないか

## カスタマーサービスのヒント {#customer-service-tips}

カードが期限切れの場合：
- 丁寧に顧客に知らせる
- 代替カードを確認するよう提案する
- 代替の支払いオプションを提供する
- 理解を示す — これは誰にでも起こり得ます
