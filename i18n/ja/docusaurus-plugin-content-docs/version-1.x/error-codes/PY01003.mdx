# PY01003: カード期限切れ

## この意味

支払いカードの有効期限が過ぎており、取引に使用できなくなっています。

## 一般的な原因

- **カード期限切れ** — カードの有効期限が過ぎている
- **有効期限の誤入力** — 有効期限が間違って入力された
- **新しいカードが未activation** — 交換カードを受け取ったが、まだ有効化されていない

## 解決方法

### 1. 有効期限を確認

カードを確認：
- 有効期限（MM/YY形式）を探す
- 現在の日付と比較する
- 入力された日付がカードと一致することを確認する

### 2. 別のカードを使用

カードが本当に期限切れの場合：
- 有効な別のカードを使用する
- 現金または代替の支払い方法を使用する
- 顧客に交換カードがあるか確認する

### 3. 更新されたカードを入手

カード所有者は以下を行うべきです：
- 郵便で交換カードを確認する
- 受け取った新しいカードを有効化する
- 交換カードが届いていない場合は銀行に連絡する

### 4. エントリーを確認

有効期限の入力を再確認：
- MM/YY形式
- 正しい月と年
- 入力ミスがないか

## カスタマーサービスのヒント

カードが期限切れの場合：
- 丁寧に顧客に伝える
- 交換カードの確認を提案する
- 代替の支払い方法を提案する
- 理解を示す — これは誰にでも起こりうること
