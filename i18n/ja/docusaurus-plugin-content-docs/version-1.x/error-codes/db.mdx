# データベースエラー {#database-errors}

データベースエラーは、ローカルデータベースにデータを保存したり、取得したりするときに発生します。WCPOSは、迅速なアクセスとオフライン機能のために、製品、顧客、およびその他のデータを保存するためにローカルデータベースを使用します。これらのエラーは`DB`で始まります。

## カテゴリー {#categories}

| カテゴリー | コード範囲 | 説明 |
|------------|------------|------|
| [接続](#connection-errors) | DB01xxx | データベース接続およびトランザクションの問題 |
| [データ](#data-errors) | DB02xxx | レコードおよび制約の問題 |
| [クエリ](#query-errors) | DB03xxx | クエリ構文およびデータ型の問題 |

---

## 接続エラー {#connection-errors}

ローカルデータベースへの接続または操作に関する問題。

| コード | 名前 | 説明 |
|--------|------|------|
| [DB01001](./DB01001) | 接続失敗 | ローカルデータベースに接続できませんでした |
| [DB01002](./DB01002) | クエリタイムアウト | データベースクエリに時間がかかりすぎました |
| [DB01003](./DB01003) | トランザクション失敗 | データベーストランザクションを完了できませんでした |

## データエラー {#data-errors}

レコード、重複、および制約に関する問題。

| コード | 名前 | 説明 |
|--------|------|------|
| [DB02001](./DB02001) | 重複レコード | このIDのレコードはすでに存在します |
| [DB02002](./DB02002) | レコードが見つかりません | 要求されたレコードは存在しません |
| [DB02003](./DB02003) | 制約違反 | データがデータベースの制約に違反しています |

## クエリエラー {#query-errors}

クエリ構文およびデータ型に関する問題。

| コード | 名前 | 説明 |
|--------|------|------|
| [DB03001](./DB03001) | クエリ構文エラー | データベースクエリに構文エラーがあります |
| [DB03002](./DB03002) | 無効なデータ型 | データ型が予期される型と一致しません |
| [DB03003](./DB03003) | 必須フィールドが欠落 | 必須フィールドが空です |
