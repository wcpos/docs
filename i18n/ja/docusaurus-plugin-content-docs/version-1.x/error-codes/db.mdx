# データベースエラー

データベースエラーは、ローカルデータベースにデータを保存または取得する際に発生します。WCPOSは、高速アクセスとオフライン機能のために、製品、顧客、その他のデータをローカルデータベースに保存します。これらのエラーは `DB` プレフィックスが付いています。

## カテゴリ

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| [接続](#接続エラー) | DB01xxx | データベース接続とトランザクションの問題 |
| [データ](#データエラー) | DB02xxx | レコードと制約の問題 |
| [クエリ](#クエリエラー) | DB03xxx | クエリ構文とデータ型の問題 |

---

## 接続エラー

ローカルデータベースへの接続または操作に関する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [DB01001](./DB01001) | 接続失敗 | ローカルデータベースに接続できませんでした |
| [DB01002](./DB01002) | クエリタイムアウト | データベースクエリの実行に時間がかかりすぎました |
| [DB01003](./DB01003) | トランザクション失敗 | データベーストランザクションを完了できませんでした |

## データエラー

レコード、重複、制約に関する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [DB02001](./DB02001) | 重複レコード | このIDのレコードが既に存在します |
| [DB02002](./DB02002) | レコード未検出 | 要求されたレコードは存在しません |
| [DB02003](./DB02003) | 制約違反 | データがデータベースの制約に違反しています |

## クエリエラー

クエリ構文とデータ型の問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [DB03001](./DB03001) | クエリ構文エラー | データベースクエリに構文エラーがあります |
| [DB03002](./DB03002) | 無効なデータ型 | データ型が期待される型と一致しません |
| [DB03003](./DB03003) | 必須フィールド欠落 | 必須フィールドが空です |
