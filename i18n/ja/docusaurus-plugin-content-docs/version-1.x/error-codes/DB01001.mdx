# DB01001: 接続失敗

## これの意味

POSはローカルデータベースに接続できませんでした。WCPOSは、製品、顧客、その他のデータを迅速にアクセスし、オフライン機能を提供するためにローカルデータベースを使用します。

## 一般的な原因

- **ストレージが満杯** — デバイスにストレージスペースが残っていない
- **データベースが破損** — ローカルデータベースファイルが損傷している
- **権限の問題** — アプリにストレージ権限がない
- **ブラウザストレージが無効** — IndexedDBが無効になっている（ウェブ版）

## 修正方法

### 1. ストレージスペースを確認

デバイスに利用可能なストレージがあることを確認してください：
- ストレージが満杯の場合はスペースを解放
- 使用していないアプリやファイルを削除
- 利用可能なディスクスペースを確認

### 2. アプリデータをクリア

ローカルデータベースをリセット：
- **デスクトップアプリ:** 設定内の「データをクリア」を探す
- **ウェブブラウザ:** POSドメインのサイトデータをクリア
- **注意:** これによりすべてのデータを再同期する必要があります

### 3. ブラウザ設定を確認（ウェブ）

ウェブ版を使用している場合：
1. IndexedDBが有効になっていることを確認
2. プライベート/シークレットモードを使用しない（ストレージが制限される）
3. サイトにデータを保存することを許可

### 4. 権限を確認

デスクトップ上で：
- アプリがデータディレクトリに書き込む権限があることを確認
- ウイルス対策ソフトがデータベースアクセスをブロックしていないか確認

### 5. アプリを再インストール

他の解決策が失敗した場合：
1. POSアプリをアンインストール
2. デバイスを再起動
3. アプリを再インストール
4. ログインし、データを同期

## データ復旧

未保存のデータがある場合：
- アプリに同期できなかった保留中の変更があるかもしれません
- データの復旧が必要な場合はサポートに連絡してください
- 大多数のケースでは、データはサーバーから再同期できます
