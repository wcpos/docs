# DB01001: 接続に失敗しました

## この意味

POSがローカルデータベースに接続できませんでした。WCPOSは、製品、顧客、その他のデータを高速アクセスとオフライン機能のためにローカルデータベースに保存します。

## 一般的な原因

- **ストレージ容量不足** — デバイスのストレージ容量が不足
- **データベース破損** — ローカルデータベースファイルが破損
- **権限の問題** — アプリがストレージ権限を持っていない
- **ブラウザストレージ無効** — IndexedDBが無効（Webバージョン）

## 修正方法

### 1. ストレージ容量を確認

デバイスの利用可能なストレージを確認します：
- ストレージ容量が満杯の場合は空き容量を確保
- 未使用のアプリやファイルを削除
- ディスクの空き容量を確認

### 2. アプリデータをクリア

ローカルデータベースをリセット：
- **デスクトップアプリ：** 設定で「データをクリア」を探す
- **Webブラウザ：** POSドメインのサイトデータをクリア
- **注意：** すべてのデータを再同期する必要があります

### 3. ブラウザ設定を確認（Web）

Webバージョンを使用している場合：
1. IndexedDBが有効になっていることを確認
2. プライベート/シークレットモードを使用しない（ストレージ制限あり）
3. サイトにデータ保存を許可

### 4. 権限を確認

デスクトップ：
- アプリがデータディレクトリに書き込む権限があることを確認
- ウイルス対策ソフトがデータベースアクセスをブロックしていないか確認

### 5. アプリを再インストール

他の解決策が失敗した場合：
1. POSアプリをアンインストール
2. デバイスを再起動
3. アプリを再インストール
4. ログインしてデータを同期

## データ復旧

未保存のデータがある場合：
- アプリに同期されていない保留中の変更がある可能性
- データ復旧が必要な場合はサポートに連絡
- ほとんどの場合、サーバーから再同期できます
