# API01008: サイト利用不可

## 意味

POSはインターネットに接続されていますが、特定のウェブサイトが応答していません。あなたのWooCommerceストアをホストしているサーバーがダウンしているか、到達不能な状態です。

## 一般的な原因

- **サーバーメンテナンス** — ホスティングプロバイダが更新作業を行っている可能性があります
- **サーバークラッシュ** — ウェブサーバーが予期せず停止しました
- **リソース制限の超過** — ホスティングアカウントが制限に達しました
- **DDoS攻撃** — サーバーが悪意のあるトラフィックによって圧倒されています
- **ドメイン/ホスティングの期限切れ** — ホスティングまたはドメインが失効しました

## 解決方法

### 1. サイトがダウンしているか確認します

ブラウザで直接サイトにアクセスしてみてください。読み込まれない場合、問題はサーバー側です。

### 2. ホスティングプロバイダのステータスを確認します

- ホスティングプロバイダのステータスページにアクセスします
- 定期メンテナンスの有無を確認します
- 報告された障害情報を探します

### 3. ホスティングアカウントを確認します

ホスティングコントロールパネルにログインします：
- アカウントがアクティブかを確認します
- リソース制限を超えていないか確認します
- 一時停止通知がないかチェックします

### 4. ホスティングサポートに連絡します

自分の側ですべてが正常に見える場合、ホスティングプロバイダに連絡してください。彼らは：
- サーバーステータスを確認します
- サーバーログをレビューします
- 必要に応じてサービスを再起動します

### 5. 待って再試行します

一時的な障害である場合：
- 15〜30分待ちます
- POSは自動的に接続を再試行します
- キャッシュされたデータはオフラインで利用可能です
