# API01008: ウェブサイト利用不可

## これが意味するもの

POSはインターネットに接続できますが、特定のウェブサイトが応答していません。WooCommerceストアをホストしているサーバーがダウンしているか、到達できない状態です。

## 一般的な原因

- **サーバーメンテナンス** — ホスティングプロバイダが更新作業を行っている
- **サーバークラッシュ** — ウェブサーバーが予期せず停止した
- **リソース制限超過** — ホスティングアカウントが制限に達した
- **DDoS攻撃** — サーバーが悪意のあるトラフィックに overwhelmed している
- **ドメイン/ホスティング期限切れ** — ホスティングまたはドメインの有効期限が切れている

## 解決方法

### 1. サイトがダウンしていることを確認

ブラウザで直接サイトにアクセスしてみてください。読み込めない場合、問題はサーバー側にあります。

### 2. ホスティングプロバイダのステータスを確認

- ホスティングプロバイダのステータスページを訪れる
- 計画されているメンテナンスを確認
- 報告されている障害を探す

### 3. ホスティングアカウントを確認

ホスティングコントロールパネルにログイン:
- アカウントがアクティブかどうかを確認
- リソース制限を超えていないか確認
- 停止通知がないか確認

### 4. ホスティングサポートに連絡

自分側で問題が見つからない場合、ホスティングプロバイダに連絡:
- サーバーステータスを確認
- サーバーログを確認
- 必要に応じてサービスを再起動

### 5. 待機して再試行

一時的な障害の場合:
- 15〜30分待つ
- POSは自動的に接続を再試行
- キャッシュデータはオフラインでも利用可能
