# PY02001: 決済ゲートウェイエラー

## 意味

決済ゲートウェイとの通信中にエラーが発生しました。決済プロセッサーがリクエストを完了できませんでした。

## よくある原因

- **ゲートウェイのダウンタイム** — 決済サービスに問題が発生しています
- **設定エラー** — ゲートウェイ設定が正しくありません
- **APIクレデンシャル** — 無効または期限切れのAPIキー
- **ネットワーク問題** — ゲートウェイへの接続に問題があります

## 解決方法

### 1. トランザクションを再試行

ゲートウェイに一時的な問題がある場合があります：
- 少し待ってみる
- 支払いの処理を再度試みる
- 問題が自動的に解決する可能性があります

### 2. ゲートウェイのステータスを確認

決済プロバイダーのステータスページを確認：
- Stripe: status.stripe.com
- PayPal: PayPalコミュニティまたはステータスを確認
- Square: status.squareup.com
- 既知の障害がないか確認

### 3. ゲートウェイ設定を確認

WooCommerceの管理画面で：
1. WooCommerce → 設定 → 支払い に移動
2. 使用している決済ゲートウェイをクリック
3. APIキーが正しいことを確認
4. テストモードとライブモードが一致していることを確認

### 4. APIクレデンシャルを確認

クレデンシャルが有効であることを確認：
- APIキーの期限が切れていないこと
- ライブ取引にはライブキーを使用
- テスト取引にはテストキーを使用
- クレデンシャルが正しいアカウントと一致していること

### 5. 別の決済ゲートウェイでテスト

可能な場合：
- 別の決済ゲートウェイで処理を試みる
- 現金決済を代替手段として使用
- 後で売上を調整

## 店舗オーナー向け

よくある設定の問題：
- テストキーとライブキーの混同
- 異なるアカウントのAPIキー
- 支払い方法が無効になっている
- 期限切れまたは無効化されたクレデンシャル

問題が続く場合は、決済プロバイダーのサポートに連絡してください。
