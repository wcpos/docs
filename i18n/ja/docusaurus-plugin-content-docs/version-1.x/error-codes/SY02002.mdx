# SY02002: サービス利用不可

## これは何を意味するか

必要なサービスまたはコンポーネントが利用できません。POSは機能するためにさまざまなサービスに依存しており、そのうちの1つが現在アクセスできません。

## 一般的な原因

- **バックグラウンドサービスが停止** — 必要なサービスが実行されていません
- **システムリソースの問題** — サービスを開始するためのリソースが不足しています
- **依存関係の失敗** — これに依存するサービスが失敗しました
- **起動の問題** — サービスが正しく起動しませんでした
- **サーバーエラー** — WooCommerceサーバーが5xxエラーを返しました

## サーバーエラーのマッピング

このエラーコードは、サーバーが以下を返すときにトリガーされます：

| サーバーコード | ソース |
|-------------|--------|
| HTTP 5xx | いかなるサーバーエラー応答（500、502、503、504など） |

## 修正方法

### 1. アプリケーションを再起動する

POSを閉じて再起動します：
1. アプリケーションを完全に閉じる
2. 数秒待つ
3. 再度起動する

### 2. デバイスを再起動する

完全な再起動は、サービスの問題を解決することがよくあります：
1. 重要な作業を保存する
2. コンピュータを再起動する
3. もう一度POSを試す

### 3. システムリソースを確認する

システムリソースが利用可能であることを確認します：
- 不要なアプリケーションを閉じる
- メモリ使用量を確認する
- ディスクスペースを確認する

### 4. アプリケーションを更新する

最新バージョンを使用していることを確認します：
- 利用可能な更新を確認する
- 保留中の更新をインストールする
- 更新後に再起動する

### 5. 必要に応じて再インストールする

サービスが引き続き利用できない場合：
1. アプリケーションをアンインストールする
2. デバイスを再起動する
3. 最新バージョンをダウンロードする
4. 新しいものをインストールする

## ブラウザ固有（Webバージョン）

Webバージョンの場合：
- JavaScriptが有効になっていることを確認する
- 必要なブラウザ機能がブロックされていないか確認する
- 別のブラウザを試す
- 干渉する可能性のあるブラウザ拡張機能を無効にする

## サポートに連絡するタイミング

このエラーが続く場合：
- 発生する時刻を記録する
- どの特定のアクションがそれを引き起こすか確認する
- 調査のために詳細を報告する
