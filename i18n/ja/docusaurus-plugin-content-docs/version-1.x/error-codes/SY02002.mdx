# SY02002: サービス利用不可

## この意味

必要なサービスまたはコンポーネントが利用できません。POSは様々なサービスに依存しており、それらの1つが現在アクセス不能です。

## 一般的な原因

- **バックグラウンドサービス停止** — 必要なサービスが実行されていない
- **システムリソースの問題** — サービスを開始するためのリソースが不足
- **依存関係の障害** — 依存しているサービスが失敗
- **起動の問題** — サービスが正常に起動しなかった
- **サーバーエラー** — WooCommerceサーバーが5xxエラーを返した

## サーバーエラーのマッピング

このエラーコードは、サーバーが以下を返す際にトリガーされます：

| サーバーコード | ソース |
|-------------|--------|
| HTTP 5xx | すべてのサーバーエラー応答（500、502、503、504など） |

## 修正方法

### 1. アプリケーションを再起動

POSを閉じて再度開きます：
1. アプリケーションを完全に閉じる
2. 数秒待つ
3. 再度起動する

### 2. デバイスを再起動

フルリスタートでサービスの問題が解決することが多いです：
1. 重要な作業を保存
2. コンピューターを再起動
3. POSを再試行

### 3. システムリソースを確認

システムリソースが利用可能であることを確認：
- 不要なアプリケーションを閉じる
- メモリ使用量を確認
- ディスクスペースを確認

### 4. アプリケーションを更新

最新バージョンであることを確認：
- 利用可能な更新を確認
- 保留中の更新をインストール
- 更新後に再起動

### 5. 必要に応じて再インストール

サービスが引き続き利用不可の場合：
1. アプリケーションをアンインストール
2. デバイスを再起動
3. 最新バージョンをダウンロード
4. 新規にインストール

## ブラウザ固有（ウェブ版）

ウェブ版の場合：
- JavaScriptが有効であることを確認
- 必要なブラウザ機能がブロックされていないことを確認
- 別のブラウザを試す
- 干渉する可能性のあるブラウザ拡張機能を無効化

## サポートに連絡するタイミング

このエラーが継続する場合：
- 発生したタイミングをメモ
- 特定のアクションでトリガーされるかを確認
- 調査のための詳細と共に報告
