# API05005: プラグインが見つかりません {#api05005-plugin-not-found}

## これが意味すること {#what-this-means}

必須のWordPressプラグインがインストールされていないか、アクティブではありません。これはWooCommerce自体、必要な拡張機能、または統合プラグインである可能性があります。

## 一般的な原因 {#common-causes}

- **WooCommerceが無効** — WooCommerceが無効になっています
- **必要な拡張が欠けている** — 必要なWooCommerceの拡張機能がインストールされていません
- **プラグインの競合** — 別のプラグインが必須のプラグインを無効にしました
- **更新失敗** — プラグインの更新が失敗し、無効のままになっています

## 修正方法 {#how-to-fix}

### 1. WooCommerceがアクティブであることを確認 {#1-verify-woocommerce-is-active}

WCPOSはWooCommerceを必要とします：
1. プラグイン → インストール済みプラグインに移動
2. "WooCommerce"を見つける
3. アクティブになっていることを確認

### 2. 必要な拡張を確認 {#2-check-for-required-extensions}

WCPOSの一部の機能は拡張を必要とする場合があります：
- エラーメッセージを確認して特定のプラグイン名を探す
- 必要な拡張機能をインストールしてアクティブにする

### 3. プラグインの競合を確認 {#3-review-plugin-conflicts}

プラグインが競合により無効になった場合：
1. WordPressの通知を確認するためにメールをチェック
2. エラーログを確認
3. 競合を解決し、再アクティブにする

### 4. プラグインファイルを確認 {#4-check-plugin-files}

プラグインのファイルが欠けている場合：
1. WordPress.orgからプラグインを再インストール
2. またはバックアップから復元
3. プラグインをアクティブにする

### 5. すべてのプラグインを更新 {#5-update-all-plugins}

古いプラグインが問題を引き起こす可能性があります：
1. ダッシュボード → 更新に移動
2. すべてのプラグインを更新
3. すべての必須プラグインがアクティブであることを確認

## 必要なプラグイン {#required-plugins}

最低限、WCPOSは次を必要とします：
- **WooCommerce** — Eコマースプラットフォーム
- **WCPOS** — POSプラグイン

追加機能には他のプラグインが必要な場合があります。
