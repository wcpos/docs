# API05005: プラグインが見つかりません

## この意味

必要なWordPressプラグインがインストールされていないか、アクティブになっていません。これはWooCommerceそのものや、必要な拡張機能、または統合プラグインの可能性があります。

## よくある原因

- **WooCommerceが無効化** — WooCommerceが無効になっている
- **必要な拡張機能の不足** — 必要なWooCommerce拡張機能がインストールされていない
- **プラグインの競合** — 別のプラグインにより、必要なプラグインが無効化された
- **更新の失敗** — プラグインの更新に失敗し、無効化された

## 修正方法

### 1. WooCommerceがアクティブであることを確認

WCPOSにはWooCommerceが必要です：
1. プラグイン → インストール済みプラグインに移動
2. 「WooCommerce」を探す
3. アクティブになっていることを確認

### 2. 必要な拡張機能を確認

一部のWCPOS機能には拡張機能が必要です：
- エラーメッセージで特定のプラグイン名を確認
- 必要な拡張機能をインストールしてアクティブ化

### 3. プラグインの競合を確認

プラグインが競合により無効化された場合：
1. WordPressの通知メールを確認
2. エラーログを確認
3. 競合を解決し、再アクティブ化

### 4. プラグインファイルを確認

プラグインファイルが見つからない場合：
1. WordPress.orgからプラグインを再インストール
2. バックアップから復元
3. プラグインをアクティブ化

### 5. すべてのプラグインを更新

古いプラグインは問題を引き起こす可能性があります：
1. ダッシュボード → 更新に移動
2. すべてのプラグインを更新
3. 必要なプラグインがすべてアクティブであることを確認

## 必要なプラグイン

最低限、WCPOSには以下が必要です：
- **WooCommerce** — Eコマースプラットフォーム
- **WCPOS** — POSプラグイン

追加機能には他のプラグインが必要な場合があります。
