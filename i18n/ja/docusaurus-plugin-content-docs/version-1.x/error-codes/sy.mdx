# システムエラー

システムエラーは、デバイスのリソースとシステム設定に関連しています。これらのエラーは `SY` で始まり、デバイスまたはPOSアプリケーション自体の問題を示します。

## カテゴリ

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| [リソース](#resource-errors) | SY01xxx | メモリ、ディスク、アクセス権の問題 |
| [サービス](#service-errors) | SY02xxx | 設定とサービスの可用性 |

---

## リソースエラー

メモリ、ストレージ、アクセス権といったデバイスリソースに関する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [SY01001](./SY01001) | メモリ不足 | デバイスのメモリが不足しています |
| [SY01002](./SY01002) | ディスクフル | デバイスのストレージがいっぱいです |
| [SY01003](./SY01003) | アクセス拒否 | システムのアクセス権が拒否されました |

## サービスエラー

システム設定とサービスの可用性に関する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [SY02001](./SY02001) | 無効な設定 | システム設定が無効です |
| [SY02002](./SY02002) | サービス利用不可 | 必要なサービスが利用できません |
