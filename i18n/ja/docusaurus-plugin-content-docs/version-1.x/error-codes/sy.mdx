# システムエラー {#system-errors}

システムエラーはデバイスリソースおよびシステム設定に関連するエラーです。これらのエラーには次のプレフィックスが付きます： `SY` これは、デバイスまたはPOSアプリケーション自体に関する問題を示しています。

## カテゴリ {#categories}

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| [リソース](#resource-errors) | SY01xxx | メモリ、ディスク、および権限に関する問題 |
| [サービス](#service-errors) | SY02xxx | 設定およびサービスの可用性 |

---

## リソースエラー {#resource-errors}

メモリ、ストレージ、権限などのデバイスリソースに関する問題。

| コード | 名称 | 説明 |
|------|------|-------------|
| [SY01001](./SY01001) | メモリ不足 | デバイスのメモリが不足しています |
| [SY01002](./SY01002) | ディスク容量不足 | デバイスのストレージが満杯です |
| [SY01003](./SY01003) | 権限拒否 | システムの権限が拒否されました |

## サービスエラー {#service-errors}

システム設定およびサービスの可用性に関する問題。

| コード | 名称 | 説明 |
|------|------|-------------|
| [SY02001](./SY02001) | 無効な構成 | システム構成が無効です |
| [SY02002](./SY02002) | サービス利用不可 | 必要なサービスが利用できません |
