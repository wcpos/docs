# システムエラー

システムエラーは、デバイスリソースとシステム構成に関連するエラーです。これらのエラーは `SY` の接頭辞が付けられ、デバイスまたはPOSアプリケーション自体の問題を示します。

## カテゴリ

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| [リソース](#リソースエラー) | SY01xxx | メモリ、ディスク、および権限の問題 |
| [サービス](#サービスエラー) | SY02xxx | 構成とサービスの可用性 |

---

## リソースエラー

メモリ、ストレージ、権限などのデバイスリソースに関する問題。

| コード | 名称 | 説明 |
|------|------|-------------|
| [SY01001](./SY01001) | メモリ不足 | デバイスのメモリが不足しています |
| [SY01002](./SY01002) | ディスク容量がいっぱい | デバイスストレージが満杯です |
| [SY01003](./SY01003) | 権限が拒否されました | システム権限が拒否されました |

## サービスエラー

システム構成とサービスの可用性に関する問題。

| コード | 名称 | 説明 |
|------|------|-------------|
| [SY02001](./SY02001) | 無効な構成 | システム構成が無効です |
| [SY02002](./SY02002) | サービス利用不可 | 必要なサービスが利用できません |
