# 支払いエラー

支払いエラーは、チェックアウトと支払い処理中に発生します。これらのエラーは `PY` プレフィックスが付けられ、支払いカード、ゲートウェイ、トランザクションに関する問題に関連しています。

## カテゴリ

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| [カード](#カードエラー) | PY01xxx | 支払いカードに関する問題 |
| [ゲートウェイ](#ゲートウェイエラー) | PY02xxx | 支払いゲートウェイ通信の問題 |

---

## カードエラー

顧客の支払いカードに関連する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [PY01001](./PY01001) | 支払い拒否 | 支払いが拒否されました |
| [PY01002](./PY01002) | 資金不足 | 支払いに十分な資金がありません |
| [PY01003](./PY01003) | カード期限切れ | 支払いカードの有効期限が切れています |
| [PY01004](./PY01004) | 無効なカード番号 | カード番号が無効です |

## ゲートウェイエラー

支払いゲートウェイとの通信に関する問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [PY02001](./PY02001) | 支払いゲートウェイエラー | 支払いゲートウェイとの通信中にエラーが発生しました |
| [PY02002](./PY02002) | 支払いタイムアウト | 支払い処理がタイムアウトしました |
