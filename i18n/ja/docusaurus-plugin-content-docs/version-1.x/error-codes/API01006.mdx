# API01006: ネットワーク到達不能

## この意味

POSがサーバーへのルートを確立できません。これはネットワークレベルの問題で、データパケットがサーバーへのパスを見つけられません。

## よくある原因

- **ネットワーク構成の問題** — ゲートウェイまたはルーティング設定が正しくない
- **ISPの問題** — インターネットサービスプロバイダに障害がある
- **VPNの干渉** — VPNがトラフィックを正しくルーティングできていない
- **サーバーネットワークの問題** — サーバーのネットワークがダウンしている

## 解決方法

### 1. インターネット接続を確認

- 他のウェブサイトにアクセスしてみる
- ルーターやモデムを再起動
- ネットワークに接続されていることを確認

### 2. VPNを一時的に無効化

VPNを使用している場合:
- 切断してもう一度試す
- 別のVPNサーバーを試す
- VPNがサーバーの場所へのアクセスを許可しているか確認

### 3. ISPに確認

以下の場合、インターネットサービスプロバイダに連絡:
- 他のサイトも到達できない
- ネットワーク機器を再起動しても問題が解決しない

### 4. サーバーステータスを確認

- ホスティングプロバイダがネットワーク障害を報告していないか確認
- 別のネットワーク（モバイルデータなど）からサイトにアクセスしてみる
- オンラインツールを使用して、サイトが全体的にダウンしているか確認

### 5. ファイアウォール設定を確認

デバイスのファイアウォールがサーバーへの発信接続をブロックしていないことを確認します。
