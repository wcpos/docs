# API01006: ネットワーク到達不可

## これが意味すること

POS はサーバーへのルートを確立できません。これはネットワークレベルの問題です — データパケットはあなたのサーバーへのパスを発見できません。

## 一般的な原因

- **ネットワーク構成の問題** — 誤ったゲートウェイまたはルーティング設定
- **ISPの問題** — インターネットサービスプロバイダーに問題があります
- **VPNの干渉** — VPNがトラフィックを誤ってルーティングしている可能性があります
- **サーバーのネットワーク問題** — サーバーのネットワークがダウンしている可能性があります

## 修正方法

### 1. インターネット接続をチェックする

- 他のウェブサイトにアクセスしてみてください
- ルーター/モデムを再起動します
- ネットワークに接続されていることを確認してください

### 2. VPNを一時的に無効にする

VPNを使用している場合：
- 切断して再試行してください
- 別のVPNサーバーを試してください
- VPNがサーバーの所在地へのアクセスを許可しているか確認してください

### 3. ISPに確認する

次の場合は、インターネットサービスプロバイダーに連絡してください：
- 他のサイトも到達不可である
- ネットワーク機器を再起動しても問題が続く

### 4. サーバーの状態を確認する

- ホスティングプロバイダーがネットワークの問題を報告しているか確認してください
- 異なるネットワーク（例：モバイルデータ）からサイトにアクセスを試みてください
- オンラインツールを使用して、サイトが全員に対してダウンしているか確認します

### 5. ファイアウォール設定を確認する

デバイスのファイアウォールがサーバーへの送信接続をブロックしていないことを確認してください。
