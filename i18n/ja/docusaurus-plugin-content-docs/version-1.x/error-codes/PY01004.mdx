# PY01004: 無効なカード番号

## 意味

入力されたカード番号が無効です。カード番号は特定のパターンに従い、検証用のチェックディジットを含んでいます。

## 一般的な原因

- **入力ミス** — 1つ以上の桁が正しく入力されていない
- **桁数不足** — すべての桁が入力されていない
- **桁数過多** — 入力された桁数が多すぎる
- **カードの誤り** — 物理的なカードと異なるカード番号
- **カードリーダーエラー** — チップ/スワイプが誤ったデータを読み取った

## 解決方法

### 1. カード番号を再入力

慎重にすべての桁を入力：
- 物理的なカードを確認
- 各桁をゆっくり入力
- 送信前に全番号を確認

### 2. カードの種類を確認

サポートされているカードの種類を確認：
- Visa（4で始まる）
- Mastercard（51-55または22-27で始まる）
- Amex（34または37で始まる）
- Discover（6011、622、644-649、65で始まる）

### 3. カードリーダーで再試行

チップ/スワイプリーダーを使用する場合：
- カードのチップを清掃
- 異なる読み取り方法を試す（チップ vs スワイプ）
- リーダーが失敗した場合は手動で入力

### 4. 別のカードを使用

カード番号が継続的に失敗する場合：
- カードが破損している可能性
- 別のカードを試す
- 代替の支払い方法を使用

### 5. カードの損傷を確認

物理的なカードの問題：
- 磁気ストライプの傷
- チップの損傷
- 摩耗または不鮮明な番号
- カードの交換が必要

## カード番号の形式

| カードの種類 | 長さ | 先頭番号 |
|-----------|--------|-------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55, 22-27 |
| Amex | 15 | 34, 37 |
| Discover | 16 | 6011, 622, 644-649, 65 |
