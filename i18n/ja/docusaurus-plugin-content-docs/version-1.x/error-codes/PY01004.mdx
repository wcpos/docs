# PY01004: 無効なカード番号

## 意味

入力されたカード番号は無効です。カード番号は特定のパターンに従い、検証用のチェックデジットを含みます。

## 一般的な原因

- **タイプミス** — 一つまたは複数の数字が誤って入力された
- **数字の欠落** — 全ての数字が入力されていない
- **余分な数字** — あまりにも多くの数字が入力されている
- **誤ったカード** — 物理カードとは異なるカード番号
- **カードリーダーエラー** — チップ/スワイプが誤ったデータを読み取った

## 修正方法

### 1. カード番号を再入力する

全ての数字を慎重に入力してください：
- 物理カードを確認する
- 各数字をゆっくり入力する
- 提出する前に全体の番号を確認する

### 2. カードタイプを確認する

カードタイプがサポートされているか確認してください：
- Visa（4で始まる）
- Mastercard（51-55または22-27で始まる）
- Amex（34または37で始まる）
- Discover（6011、622、644-649、65で始まる）

### 3. カードリーダーで再試行する

チップ/スワイプリーダーを使用している場合：
- カードのチップを清掃する
- 異なる読み取り方法を試す（チップ対スワイプ）
- リーダーが失敗した場合は手動で入力する

### 4. 別のカードを使用する

カード番号が引き続き失敗する場合：
- カードが損傷している可能性があります
- 別のカードを試してください
- 代替の支払い方法を使用する

### 5. カードの損傷を確認する

物理カードの問題：
- 磁気ストライプの傷
- チップの損傷
- 擦り切れたまたは色あせた数字
- カードは交換が必要かもしれません

## カード番号のフォーマット

| カードタイプ | 長さ | 始まりの数字 |
|-----------|--------|-------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55、22-27 |
| Amex | 15 | 34、37 |
| Discover | 16 | 6011、622、644-649、65 |
