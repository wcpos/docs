# SY02001: 無効な構成

## これは何を意味するか

システムの構成が無効または破損しています。POSアプリケーションは、意味がないか不完全な構成設定を見つけました。

## 一般的な原因

- **設定の破損** — 構成ファイルが破損した
- **設定の未完了** — 設定プロセスが完了していなかった
- **手動編集エラー** — 構成ファイルが不正に修正された
- **バージョンアップグレードの問題** — 古い構成が新しいバージョンと互換性がない

## 修正方法

### 1. デフォルト設定にリセット

アプリで利用可能な場合：
- 「設定をリセット」または「デフォルトに復元」を探します
- これによりカスタム設定がクリアされますが、破損は修正されます

### 2. セットアップを完了する

設定が完了していない場合：
1. 設定プロセスを再開始
2. 必要なすべてのステップを完了
3. 完了前に構成を確認

### 3. アプリケーションデータをクリア

破損した構成を削除：
- **デスクトップ:** アプリデータフォルダーを削除し、アプリを再起動
- **ブラウザ:** サイトデータをクリアし、リロード

### 4. アプリケーションを再インストール

問題が持続する場合：
1. 完全にアンインストール
2. 残ったデータフォルダーを削除
3. 新しく再インストール
4. 最初からセットアップを完了

### 5. アップデートを確認

構成形式が変更される可能性があります：
- POSアプリケーションを更新
- WCPOSプラグインを更新
- 更新後に再構成

## リセットされる内容

構成をリセットすると：
- 再度ログインする必要があります
- カスタム設定は失われます
- データの再同期が必要
- ただし、WooCommerceデータはサーバー上で安全です
