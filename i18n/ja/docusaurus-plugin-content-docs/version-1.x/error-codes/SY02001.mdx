# SY02001: 設定が無効です

## この意味

システム設定が無効または破損しています。POSアプリケーションが、意味をなさないか不完全な設定を見つけました。

## 一般的な原因

- **破損した設定** — 設定ファイルが破損しました
- **不完全なセットアップ** — セットアップ処理が完了していません
- **手動編集エラー** — 設定ファイルが不正に変更されました
- **バージョンアップグレードの問題** — 古い設定が新しいバージョンと互換性がありません

## 修正方法

### 1. デフォルト設定に戻す

アプリで利用可能な場合：
- 「設定をリセット」または「デフォルトに戻す」を探します
- これにより、カスタム設定がクリアされ、破損が修正されます

### 2. セットアップを完了する

セットアップが完了していない場合：
1. セットアップ処理を再度開始します
2. すべての必要な手順を完了します
3. 完了前に設定を確認します

### 3. アプリケーションデータを消去

破損した設定を削除：
- **デスクトップ：** アプリデータフォルダを削除し、アプリを再起動
- **ブラウザ：** サイトデータを消去し、リロード

### 4. アプリケーションを再インストール

永続的な問題がある場合：
1. 完全にアンインストール
2. 残りのデータフォルダを削除
3. 新規にインストール
4. 最初からセットアップを完了

### 5. アップデートを確認

設定フォーマットが変更される可能性があります：
- POSアプリケーションを更新
- WCPOSプラグインを更新
- 更新後に再設定

## リセットされる内容

設定をリセットする際：
- 再ログインが必要になります
- カスタム設定は失われます
- データは再同期が必要です
- ただし、WooCommerceのデータはサーバー上で安全です
