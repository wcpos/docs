# API01007: デバイスがオフライン {#api01007-device-offline}

## これが意味すること {#what-this-means}

あなたのデバイスはインターネットに接続されていません。POSは、アクティブなネットワーク接続がないことを検出しました。

## 一般的な原因 {#common-causes}

- **WiFiが切断された** — デバイスがWiFi接続を失った
- **イーサネットが抜かれた** — ネットワークケーブルが切断されている
- **機内モードが有効** — すべての無線接続が無効になっている
- **ネットワークアダプタが無効** — ネットワークインターフェイスがオフになっている

## 修正方法 {#how-to-fix}

### 1. WiFi接続を確認 {#1-check-wifi-connection}

- デバイスでWiFiが有効になっているか確認
- 正しいネットワークに接続されているか確認
- WiFi信号強度を確認

### 2. 物理接続を確認 {#2-check-physical-connections}

イーサネットを使用している場合：
- ケーブルがしっかりと接続されているか確認
- 別のケーブルやポートを試す
- ネットワークスイッチ/ルーターのランプを確認

### 3. 機内モードを無効にする {#3-disable-airplane-mode}

機内モードが誤って有効になっていないか確認し、無効にします。

### 4. ネットワークアダプタを再起動 {#4-restart-network-adapter}

**Windows:**
- ネットワーク接続を開く
- アダプタを右クリック → 無効にする、次に有効にする

**macOS:**
- メニューバーからWiFiをオフにして再度オンにする
- または: システム環境設定 → ネットワーク → オフにする/オンにする

### 5. デバイスを再起動 {#5-restart-your-device}

時にはシンプルな再起動がネットワーク検出の問題を解決します。

## オフラインモード {#offline-mode}

オフライン中でも、WCPOSは以下を実行できます：
- ローカルにキャッシュされた商品の閲覧と検索
- ローカルにキャッシュされた顧客の閲覧と検索
- 新しい注文を開始し、カートにアイテムを追加
- 以前に同期されたデータの表示

**接続が必要:**
- 注文の完了/チェックアウト
- 新しい顧客の作成

接続が復元されると、POSは自動的に通常の操作を再開します。
