# API01007: デバイスオフライン

## この意味

デバイスがインターネットに接続されていません。POSは、アクティブなネットワーク接続がないことを検出しました。

## 一般的な原因

- **WiFi切断** — デバイスのWiFi接続が失われました
- **イーサネット未接続** — ネットワークケーブルが外れています
- **機内モード有効** — すべての無線接続が無効になっています
- **ネットワークアダプターが無効** — ネットワークインターフェースがオフになっています

## 修正方法

### 1. WiFi接続を確認

- デバイスのWiFiが有効になっていることを確認
- 正しいネットワークに接続されていることを確認
- WiFi信号強度を確認

### 2. 物理的な接続を確認

イーサネット使用の場合:
- ケーブルが確実に接続されていることを確認
- 別のケーブルまたはポートを試す
- ネットワークスイッチ/ルーターのランプを確認

### 3. 機内モードを無効にする

機内モードが誤って有効になっていないか確認し、無効にします。

### 4. ネットワークアダプターを再起動

**Windows:**
- ネットワーク接続を開く
- アダプターを右クリック → 無効、次に有効にする

**macOS:**
- メニューバーからWiFiをオフ、次にオンにする
- または：システム環境設定 → ネットワーク → オフ/オンにする

### 5. デバイスを再起動

シンプルな再起動でネットワーク検出の問題が解決することがあります。

## オフラインモード

オフライン中、WCPOSは以下が可能です:
- ローカルにキャッシュされた商品を参照・検索
- ローカルにキャッシュされた顧客を参照・検索
- 新しい注文を開始し、カートに商品を追加
- 以前に同期されたデータを表示

**接続が必要:**
- 注文の完了/チェックアウト
- 新しい顧客の作成

接続が回復すると、POSは自動的に通常の操作を再開します。
