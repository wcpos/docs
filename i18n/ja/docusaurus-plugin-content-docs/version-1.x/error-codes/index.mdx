# エラーコード

WCPOSが問題に遭遇すると、問題を特定するのに役立つエラーコードが表示されます。各エラーコードは、エラーのタイプと性質を迅速に理解できるように`[DOMAIN][CATEGORY][SPECIFIC_CODE]`の形式に従っています。

## エラーコードの形式

| コンポーネント | 説明 | 例 |
|----------------|------|----|
| **ドメイン** | システムエリア | `API`, `DB`, `PY`, `SY` |
| **カテゴリ** | エラーのカテゴリ | `01`, `02`, `03` など |
| **特定コード** | 一意の識別子 | `001`, `002`, `003` など |

**例:** `API04001` = API (ドメイン) + 04 (応答エラー) + 001 (無効な応答形式)

## エラードメイン

### [APIエラー](./api)

APIエラーは、WooCommerceサーバーとの通信時に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|------|
| 接続 | API01xxx | ネットワークおよび接続の問題 |
| 認証 | API02xxx | ログイン、トークン、権限の問題 |
| リクエスト | API03xxx | 送信リクエストの問題 |
| 応答 | API04xxx | サーバーの応答の問題 |
| プラグイン/WordPress | API05xxx | WordPressまたはWooCommerceプラグインの問題 |
| 構成 | API06xxx | セットアップおよび構成の問題 |

### [データベースエラー](./db)

データベースエラーは、データをローカルに保存または取得する際に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|------|
| 接続 | DB01xxx | データベース接続およびトランザクションの問題 |
| データ | DB02xxx | レコードおよび制約の問題 |
| クエリ | DB03xxx | クエリ構文およびデータ型の問題 |

### [支払いエラー](./py)

支払いエラーは、チェックアウトおよび支払い処理中に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|------|
| カード | PY01xxx | 支払いカードの問題 |
| ゲートウェイ | PY02xxx | 支払いゲートウェイ通信の問題 |

### [システムエラー](./sy)

システムエラーは、デバイスリソースおよびシステム構成に関連しています。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|------|
| リソース | SY01xxx | メモリ、ディスク、および権限の問題 |
| サービス | SY02xxx | 構成およびサービスの可用性 |

## さらに支援が必要ですか？

トラブルシューティング手順に従った後もまだ問題が発生している場合：

1. より詳細なガイドについては[トラブルシューティング](/support/troubleshooting/response-error)セクションを確認してください
2. サポートのために[Discordコミュニティ](https://wcpos.com/discord)に参加してください
3. [GitHub](https://github.com/wcpos)で問題を報告してください
