# エラーコード

WCPOSで問題が発生した場合、エラーコードが表示され、問題の特定に役立ちます。各エラーコードは`[ドメイン][カテゴリ][具体的なコード]`の形式に従い、エラーの種類と性質をすぐに理解できるようになっています。

## エラーコードの形式

| コンポーネント | 説明 | 例 |
|-----------|-------------|---------|
| **ドメイン** | システムエリア | `API`、`DB`、`PY`、`SY` |
| **カテゴリ** | エラーカテゴリ | `01`、`02`、`03`など |
| **具体的なコード** | ユニークな識別子 | `001`、`002`、`003`など |

**例:** `API04001` = API（ドメイン）+ 04（レスポンスエラー）+ 001（無効なレスポンス形式）

## エラードメイン

### [APIエラー](./api)

APIエラーは、WooCommerceサーバーとの通信中に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| 接続 | API01xxx | ネットワークおよび接続の問題 |
| 認証 | API02xxx | ログイン、トークン、および権限の問題 |
| リクエスト | API03xxx | 送信リクエストの問題 |
| レスポンス | API04xxx | サーバーレスポンスの問題 |
| プラグイン/WordPress | API05xxx | WordPressまたはWooCommerceプラグインの問題 |
| 設定 | API06xxx | セットアップおよび設定の問題 |

### [データベースエラー](./db)

データベースエラーは、ローカルでのデータの保存または取得時に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| 接続 | DB01xxx | データベース接続およびトランザクションの問題 |
| データ | DB02xxx | レコードおよび制約の問題 |
| クエリ | DB03xxx | クエリ構文およびデータ型の問題 |

### [支払いエラー](./py)

支払いエラーは、チェックアウトおよび支払い処理中に発生します。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| カード | PY01xxx | 支払いカードの問題 |
| ゲートウェイ | PY02xxx | 決済ゲートウェイ通信の問題 |

### [システムエラー](./sy)

システムエラーは、デバイスリソースとシステム設定に関連しています。

| カテゴリ | コード範囲 | 説明 |
|----------|------------|-------------|
| リソース | SY01xxx | メモリ、ディスク、および権限の問題 |
| サービス | SY02xxx | 設定およびサービスの可用性 |

## さらにヘルプが必要ですか？

トラブルシューティングの手順に従っても問題が解決しない場合：

1. 詳細なガイドについては[トラブルシューティング](/support/troubleshooting/response-error)セクションを確認してください
2. [Discordコミュニティ](https://wcpos.com/discord)でサポートを受けてください
3. [GitHub](https://github.com/wcpos)で問題を報告してください
