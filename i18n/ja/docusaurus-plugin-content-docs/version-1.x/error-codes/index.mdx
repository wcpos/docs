# エラーコード

WCPOSが問題に遭遇すると、問題の特定に役立つエラーコードを表示します。各エラーコードは、エラーのタイプと性質をすぐに理解できるように、`[DOMAIN][CATEGORY][SPECIFIC_CODE]`の形式に従っています。

## エラーコードの形式

| コンポーネント | 説明 | 例 |
|-----------|-------------|---------|
| **ドメイン** | システム領域 | `API`, `DB`, `PY`, `SY` |
| **カテゴリー** | エラーのカテゴリー | `01`, `02`, `03`, 等 |
| **固有コード** | 一意の識別子 | `001`, `002`, `003`, 等 |

**例:** `API04001` = API (ドメイン) + 04 (レスポンスエラー) + 001 (無効なレスポンス形式)

## エラードメイン

### [APIエラー](./api)

APIエラーは、WooCommerceサーバーとの通信中に発生します。

| カテゴリー | コード範囲 | 説明 |
|----------|------------|-------------|
| 接続 | API01xxx | ネットワークと接続の問題 |
| 認証 | API02xxx | ログイン、トークン、権限の問題 |
| リクエスト | API03xxx | 外部リクエストに関する問題 |
| レスポンス | API04xxx | サーバーレスポンスに関する問題 |
| プラグイン/WordPress | API05xxx | WordPressまたはWooCommerceプラグインの問題 |
| 設定 | API06xxx | セットアップと設定の問題 |

### [データベースエラー](./db)

データベースエラーは、ローカルでデータを保存または取得する際に発生します。

| カテゴリー | コード範囲 | 説明 |
|----------|------------|-------------|
| 接続 | DB01xxx | データベース接続およびトランザクションの問題 |
| データ | DB02xxx | レコードと制約の問題 |
| クエリ | DB03xxx | クエリ構文およびデータ型の問題 |

### [支払いエラー](./py)

支払いエラーは、チェックアウトと支払い処理中に発生します。

| カテゴリー | コード範囲 | 説明 |
|----------|------------|-------------|
| カード | PY01xxx | 支払いカードに関する問題 |
| ゲートウェイ | PY02xxx | 支払いゲートウェイ通信の問題 |

### [システムエラー](./sy)

システムエラーは、デバイスリソースおよびシステム設定に関連しています。

| カテゴリー | コード範囲 | 説明 |
|----------|------------|-------------|
| リソース | SY01xxx | メモリ、ディスク、権限の問題 |
| サービス | SY02xxx | 設定およびサービスの可用性 |

## さらなる支援が必要ですか？

トラブルシューティングの手順に従った後も問題が発生する場合：

1. より詳細なガイドについては[トラブルシューティング](/support/troubleshooting/response-error)セクションを確認してください。
2. 支援のために[Discordコミュニティ](https://wcpos.com/discord)に参加してください。
3. [GitHub](https://github.com/wcpos)で問題を報告してください。
