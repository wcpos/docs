# APIエラー

APIエラーは、WooCommerceサーバーとの通信中に発生します。これらのエラーは`API`で始まり、以下のカテゴリに整理されています。

## カテゴリ

| カテゴリ | コードレンジ | 説明 |
|----------|--------------|------|
| [接続](#connection-errors) | API01xxx | ネットワークと接続の問題 |
| [認証](#authentication-errors) | API02xxx | ログイン、トークン、および権限の問題 |
| [リクエスト](#request-errors) | API03xxx | アウトゴーイングリクエストの問題 |
| [レスポンス](#response-errors) | API04xxx | サーバーのレスポンスに関する問題 |
| [プラグイン/WordPress](#pluginwordpress-errors) | API05xxx | WordPressまたはWooCommerceプラグインの問題 |
| [構成](#configuration-errors) | API06xxx | 設定と構成の問題 |

---

## 接続エラー

POSとサーバー間のネットワークおよび接続の問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API01001](./API01001) | 接続タイムアウト | サーバーが応答するまでに時間がかかりすぎました |
| [API01002](./API01002) | 接続拒否 | サーバーが接続を拒否しました |
| [API01003](./API01003) | 接続リセット | 接続が予期せず閉じられました |
| [API01004](./API01004) | DNS解決失敗 | サーバーアドレスを解決できませんでした |
| [API01005](./API01005) | SSL証明書エラー | サイトのセキュリティ証明書に問題があります |
| [API01006](./API01006) | ネットワーク到達不能 | ネットワークに到達できません |
| [API01007](./API01007) | デバイスオフライン | デバイスがインターネットに接続されていません |
| [API01008](./API01008) | ウェブサイト利用不可 | ウェブサイトが応答していません |

## 認証エラー

ログイン、セッション、および権限に関する問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API02001](./API02001) | 無効な資格情報 | ユーザー名またはパスワードが正しくありません |
| [API02002](./API02002) | トークンの有効期限切れ | セッションの有効期限が切れました |
| [API02003](./API02003) | 無効なトークン | 認証トークンは無効です |
| [API02004](./API02004) | ユーザー未承認 | このアクションを実行する権限がありません |
| [API02005](./API02005) | 権限不足 | ユーザーロールに必要な権限がありません |
| [API02006](./API02006) | APIキー無効 | WooCommerce APIキーは無効です |
| [API02007](./API02007) | トークンの更新に失敗しました | セッションの更新に失敗しました |
| [API02008](./API02008) | 更新トークン無効 | 更新トークンは無効です |
| [API02009](./API02009) | 更新トークンの有効期限切れ | 更新トークンの有効期限が切れました |
| [API02010](./API02010) | 認証が必要 | このアクションには認証が必要です |

## リクエストエラー

サーバーに送信されたリクエストに関する問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API03001](./API03001) | 無効なリクエスト形式 | リクエスト形式が正しくありません |
| [API03002](./API03002) | 必要なパラメータが欠けている | リクエストに必要なデータが欠けています |
| [API03003](./API03003) | 無効なパラメータ値 | パラメータに無効な値があります |
| [API03004](./API03004) | リクエストが大きすぎる | リクエストがサイズ制限を超えています |
| [API03005](./API03005) | レート制限を超えました | 短時間でのリクエストが多すぎます |
| [API03006](./API03006) | 非対応メソッド | HTTPメソッドがサポートされていません |
| [API03007](./API03007) | リクエストキューが満杯 | 保留中のリクエストが多すぎます |

## レスポンスエラー

サーバーから受け取ったレスポンスに関する問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API04001](./API04001) | 無効なレスポンス形式 | サーバーレスポンス形式が無効です |
| [API04002](./API04002) | 予期しないレスポンスコード | 予期しないHTTPステータスコードを受け取りました |
| [API04003](./API04003) | 不正なJSONレスポンス | JSONレスポンスが破損または無効です |
| [API04004](./API04004) | レスポンスデータが欠落しています | 期待されるデータがレスポンスから欠けています |
| [API04005](./API04005) | JSONからの復元を試みました | 不正なJSONからの復元を試みました |
| [API04006](./API04006) | リソースが見つかりません | 要求されたリソースが存在しません (404) |

## プラグイン/WordPressエラー

WordPress、WooCommerce、またはWCPOSプラグインに関する問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API05001](./API05001) | WooCommerce API無効 | WooCommerce REST APIは無効です |
| [API05002](./API05002) | WCPOSプラグインが見つかりません | WCPOSプラグインがインストールされていません |
| [API05003](./API05003) | WCPOSプラグインが古い | WCPOSプラグインは更新が必要です |
| [API05004](./API05004) | WordPress API無効 | WordPress REST APIは無効です |
| [API05005](./API05005) | プラグインが見つかりません | 必要なプラグインがインストールされていません |

設定および構成の問題。

| コード | 名称 | 説明 |
|------|------|------|
| [API06001](./API06001) | 無効なURL形式 | URL形式が無効です |
| [API06002](./API06002) | API URLが欠落しています | API URLが設定されていません |
| [API06003](./API06003) | 無効なサイト構成 | サイト構成が正しくありません |
