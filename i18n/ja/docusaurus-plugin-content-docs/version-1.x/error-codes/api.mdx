# APIエラー

APIエラーは、WooCommerceサーバーと通信する際に発生します。これらのエラーは`API`が接頭辞として付けられ、以下のカテゴリーに分類されます：

## カテゴリー

| カテゴリー | コード範囲 | 説明 |
|----------|------------|-------------|
| [接続](#接続エラー) | API01xxx | ネットワークと接続の問題 |
| [認証](#認証エラー) | API02xxx | ログイン、トークン、権限の問題 |
| [リクエスト](#リクエストエラー) | API03xxx | 送信リクエストの問題 |
| [レスポンス](#レスポンスエラー) | API04xxx | サーバーレスポンスの問題 |
| [プラグイン/WordPress](#プラグインwordpress-エラー) | API05xxx | WordPressまたはWooCommerceプラグインの問題 |
| [設定](#設定エラー) | API06xxx | セットアップと設定の問題 |

---

## 接続エラー

POSとサーバー間のネットワークと接続の問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [API01001](./API01001) | 接続タイムアウト | サーバーの応答に時間がかかりすぎました |
| [API01002](./API01002) | 接続拒否 | サーバーが接続を拒否しました |
| [API01003](./API01003) | 接続リセット | 接続が予期せず切断されました |
| [API01004](./API01004) | DNS解決失敗 | サーバーアドレスを解決できませんでした |
| [API01005](./API01005) | SSLサーティフィケートエラー | サイトのセキュリティ証明書に問題があります |
| [API01006](./API01006) | ネットワーク到達不能 | ネットワークに到達できません |
| [API01007](./API01007) | デバイスオフライン | デバイスがインターネットに接続されていません |
| [API01008](./API01008) | ウェブサイト利用不可 | ウェブサイトが応答しません |

## 認証エラー

ログイン、セッション、および権限の問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [API02001](./API02001) | 無効な認証情報 | ユーザー名またはパスワードが正しくありません |
| [API02002](./API02002) | トークン期限切れ | セッションの有効期限が切れています |
| [API02003](./API02003) | 無効なトークン | 認証トークンが無効です |
| [API02004](./API02004) | ユーザー未認証 | この操作を実行する権限がありません |
| [API02005](./API02005) | 権限不足 | ユーザーロールに必要な権限がありません |
| [API02006](./API02006) | 無効なAPIキー | WooCommerceのAPIキーが無効です |
| [API02007](./API02007) | トークンリフレッシュ失敗 | セッションをリフレッシュできませんでした |
| [API02008](./API02008) | 無効なリフレッシュトークン | リフレッシュトークンが無効です |
| [API02009](./API02009) | リフレッシュトークン期限切れ | リフレッシュトークンの有効期限が切れています |
| [API02010](./API02010) | 認証が必要 | この操作には認証が必要です |

## リクエストエラー

サーバーに送信されたリクエストの問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [API03001](./API03001) | 無効なリクエスト形式 | リクエスト形式が正しくありません |
| [API03002](./API03002) | 必須パラメータ不足 | リクエストに必要なデータが不足しています |
| [API03003](./API03003) | 無効なパラメータ値 | パラメータに無効な値が設定されています |
| [API03004](./API03004) | リクエストが大きすぎる | リクエストがサイズ制限を超えています |
| [API03005](./API03005) | レート制限超過 | 短時間に多すぎるリクエストがあります |
| [API03006](./API03006) | サポートされていないメソッド | HTTPメソッドがサポートされていません |
| [API03007](./API03007) | リクエストキュー満杯 | 保留中のリクエストが多すぎます |

## レスポンスエラー

サーバーから受信したレスポンスの問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [API04001](./API04001) | 無効なレスポンス形式 | サーバーレスポンス形式が無効です |
| [API04002](./API04002) | 予期しないレスポンスコード | 予期しないHTTPステータスコードを受信しました |
| [API04003](./API04003) | 不正なJSONレスポンス | JSONレスポンスが破損または無効です |
| [API04004](./API04004) | レスポンスデータ不足 | 期待されるデータがレスポンスに存在しません |
| [API04005](./API04005) | JSON復旧試行 | 不正なJSONからの復旧を試みました |
| [API04006](./API04006) | リソース未検出 | リクエストされたリソースが存在しません (404) |

## プラグイン/WordPress エラー

WordPress、WooCommerce、またはWCPOSプラグインの問題。

| コード | 名前 | 説明 |
|------|------|-------------|
| [API05001](./API05001) | WooCommerce API無効 | WooCommerceのREST APIが無効になっています |
| [API05002](./API05002) | WCPOSプラグイン未検出 | WCPOSプラグインがインストールされていません |
| [API05003](./API05003) | WCPOSプラグイン古い | WCPOSプラグインの更新が必要です |
| [API05004](./API05004) | WordPress API無効 | WordPress REST APIが無効になっています |
| [API05005](./API05005) | プラグイン未検出 | 必要なプラグインがインストールされていません |

セットアップと設定の問題。

|------|------|-------------|
| [API06001](./API06001) | 無効なURL形式 | URLの形式が無効です |

| [API06003](./API06003) | 無効なサイト設定 | サイト設定が正しくありません |
