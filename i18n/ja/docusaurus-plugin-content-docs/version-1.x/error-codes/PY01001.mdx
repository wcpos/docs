# PY01001: 支払い拒否

## これは何を意味するか

支払いが決済プロセッサーまたはカード発行会社によって拒否されました。トランザクションを完了できませんでした。

## 一般的な原因

- **資金不足** — アカウントに十分な資金がありません
- **カード限度額超過** — トランザクションがカードの限度額を超えています
- **不正防止** — 銀行が不審とフラグを立てました
- **カード詳細の誤り** — カード番号、CVV、または有効期限が間違っています
- **カード制限** — このタイプの購入に対してカードが有効になっていません

## 修正方法

### 1. カード情報の確認

入力された情報が正確であることを確認します：
- カード番号（すべての桁）
- 有効期限
- CVV/セキュリティコード
- 請求先住所（必要な場合）

### 2. 再試行

一時的な問題が発生することがあります：
- 少し待つ
- カード情報を再入力
- トランザクションを再度試行

### 3. 別の支払い方法を使用

カードが継続的に拒否される場合：
- 別のカードを試す
- 現金支払いを使用
- 代替の支払い方法を使用

### 4. 銀行に連絡

カード所有者は以下を行う必要がある場合があります：
- 銀行にトランザクションを確認
- 不正ブロックを解除
- カードがアクティブであることを確認
- 支出制限を確認

### 5. 制限の確認

一部のカードには制限があります：
- 国際取引が無効
- オンライン/POS取引がブロック
- 特定の商業カテゴリーがブロック

## 店舗所有者向け

頻繁に拒否が発生する場合：
- マーチャントアカウントが良好な状態であることを確認
- 決済ゲートウェイの設定を確認
- 不正防止設定を確認
- 決済プロセッサーに連絡することを検討
