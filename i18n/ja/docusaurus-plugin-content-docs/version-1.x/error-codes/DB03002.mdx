# DB03002: データ型が無効

## 意味

データ型がデータベースの期待する型と一致しません。例えば、数値が必要な箇所にテキストが入力されました。

## 一般的な原因

- **ユーザー入力エラー** — 数値フィールドにテキストを入力
- **インポートの問題** — インポートされたデータの形式が不正
- **データ破損** — 転送中に値が破損
- **プラグインの競合** — 別のプラグインがデータ型を変更

## 修正方法

### 1. 入力を確認

入力するデータを見直してください：
- **価格** — 数値である必要があります（例：`19.99`）
- **数量** — 整数である必要があります（例：`5`）
- **ID** — 数値である必要があります

### 2. クリアして再入力

データが破損している場合：
- フィールドをクリア
- 値を手動で再入力
- 外部ソースからのコピー＆ペーストを避ける

### 3. ソースデータを確認

WooCommerceから同期している場合：
- WordPress 管理画面でデータを確認
- 不正な形式のフィールドを探す
- ソース側でデータを修正

### 4. データを再同期

サーバーから新しいデータを取得：
- ローカルキャッシュをクリア
- データを再同期
- 問題が解決したか確認

### 5. プラグインの競合を確認

他のWooCommerceプラグインを使用している場合：
- 予期しない方法でデータを変更する可能性がある
- 一時的に無効化してテスト
- 非互換性を報告

## 一般的な例

| フィールド | 期待される値 | 無効な値 |
|-------|-------------|----------|
| 価格 | `19.99` | `$19.99` |
| 数量 | `5` | `five` |
| SKU | `ABC123` | — (任意の形式OK) |
| 商品ID | `42` | `product-42` |
