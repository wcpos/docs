# API05002: WCPOS プラグインが見つかりません {#api05002-wcpos-plugin-not-found}

## これは何を意味しますか {#what-this-means}

WCPOS (WooCommerce POS) プラグインが、あなたの WordPress サイトにインストールされていないか、アクティブでないということです。POS アプリケーションは、WooCommerce と通信するためにこのプラグインを必要とします。

## 一般的な原因 {#common-causes}

- **プラグインがインストールされていない** — プラグインがインストールされていない
- **プラグインが無効化されている** — プラグインが無効になっている
- **プラグインが削除されている** — プラグインのファイルが削除されている
- **間違ったサイト** — WCPOS がないサイトに接続している

## 修正方法 {#how-to-fix}

### 1. プラグインをインストールする {#1-install-the-plugin}

インストールされていない場合:
1. WordPress 管理画面 → プラグイン → 新規追加
2. "WooCommerce POS" を検索
3. プラグインをインストールして有効化

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からダウンロードします。

### 2. プラグインを有効化する {#2-activate-the-plugin}

インストールされているがアクティブでない場合:
1. プラグイン → インストール済みプラグインに移動
2. "WooCommerce POS" を見つける
3. "有効化" をクリック

### 3. エラーを確認する {#3-check-for-errors}

プラグインが有効化されない場合:
- エラーログで PHP エラーを確認
- WooCommerce がインストールされていてアクティブであることを確認
- PHP バージョン要件を確認
- プラグインの競合を探す

### 4. URL を確認する {#4-verify-the-url}

POS が正しい WordPress サイトに接続するように設定されていることを確認:
- POS 設定の URL を確認
- WordPress インストールと一致していることを確認

### 5. 必要に応じて再インストールする {#5-reinstall-if-necessary}

プラグインが破損している場合:
1. プラグインを無効化して削除
2. WordPress.org から再インストール
3. プラグインを有効化
