# API05002: WCPOSプラグインが見つかりません

## これは何を意味するのか

WCPOS（WooCommerce POS）プラグインがあなたのWordPressサイトにインストールされていないか、アクティブではありません。POSアプリケーションは、WooCommerceと通信するためにこのプラグインを必要とします。

## 一般的な原因

- **プラグインがインストールされていない** — プラグインはインストールされていません
- **プラグインが無効化されている** — プラグインが無効になっています
- **プラグインが削除された** — プラグインファイルが削除されました
- **間違ったサイト** — WCPOSがないサイトに接続されています

## 修正方法

### 1. プラグインをインストール

インストールされていない場合：
1. WordPress管理画面 → プラグイン → 新規追加
2. 「WooCommerce POS」を検索
3. プラグインをインストールしてアクティブ化する

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)からダウンロードします。

### 2. プラグインをアクティブ化

インストールされているがアクティブでない場合：
1. プラグイン → インストール済みプラグインに移動
2. 「WooCommerce POS」を見つける
3. 「アクティブ化」をクリック

### 3. エラーを確認

プラグインがアクティブ化できない場合：
- エラーログでPHPエラーを確認する
- WooCommerceがインストールされてアクティブであることを確認する
- PHPバージョン要件を確認する
- プラグインの競合を探す

### 4. URLを確認

POSが正しいWordPressサイトに接続するように設定されていることを確認します：
- POS設定内のURLを確認する
- WordPressインストールと一致することを確認する

### 5. 必要に応じて再インストール

プラグインが破損している場合：
1. プラグインを無効化して削除する
2. WordPress.orgから再インストールする
3. プラグインをアクティブ化する

## 要件

WCPOSは以下を必要とします：
- WordPress（最新バージョン）
- WooCommerce（最新バージョン）
- PHP 7.4以上
