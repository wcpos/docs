# API05002: WCPOSプラグインが見つかりません

## 意味

WCPOS（WooCommerce POS）プラグインがWordPressサイトにインストールされていないか、アクティブになっていません。POSアプリケーションはWooCommerceと通信するためにこのプラグインを必要とします。

## よくある原因

- **プラグインが未インストール** — プラグインがまだインストールされていない
- **プラグインが無効** — プラグインが無効化されている
- **プラグインが削除** — プラグインファイルが削除されている
- **間違ったサイト** — WCPOSのないサイトに接続している

## 解決方法

### 1. プラグインをインストール

インストールされていない場合:
1. WordPress管理画面 → プラグイン → 新規追加
2. 「WooCommerce POS」を検索
3. プラグインをインストールし、有効化

または[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)からダウンロード

### 2. プラグインを有効化

インストールされているが有効でない場合:
1. プラグイン → インストール済みプラグイン
2. 「WooCommerce POS」を探す
3. 「有効化」をクリック

### 3. エラーを確認

プラグインが有効化できない場合:
- エラーログでPHPエラーを確認
- WooCommerceがインストールされ、有効であることを確認
- PHPバージョンの要件を確認
- プラグインの競合を確認

### 4. URLを確認

POSが正しいWordPressサイトに接続していることを確認:
- POSの設定でURLを確認
- WordPress インストールと一致することを確認

### 5. 必要に応じて再インストール

プラグインが破損している場合:
1. プラグインを無効化し、削除
2. WordPress.orgから再インストール
3. プラグインを有効化

## 要件

WCPOSには以下が必要:
- WordPress (最新バージョン)
- WooCommerce (最新バージョン)
- PHP 7.4以上
