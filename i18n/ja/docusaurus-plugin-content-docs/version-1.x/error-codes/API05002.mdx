# API05002: WCPOS プラグインが見つかりません {#api05002-wcpos-plugin-not-found}

## これは何を意味するのか {#what-this-means}

WCPOS プラグインが WordPress サイトにインストールされていないか、アクティブでありません。この POS アプリケーションは、WooCommerce との通信にこのプラグインを必要とします。

## よくある原因 {#common-causes}

- **プラグイン未インストール** — プラグインが一度もインストールされていない
- **プラグイン無効化** — プラグインが無効になっている
- **プラグイン削除** — プラグインファイルが削除された
- **誤ったサイト** — WCPOS のないサイトに接続されている

## どうやって修正するか {#how-to-fix}

### 1. プラグインをインストール {#1-install-the-plugin}

インストールされていない場合:
1. WordPress 管理画面に移動 → プラグイン → 新規追加
2. 「WooCommerce POS」を検索
3. プラグインをインストールして有効化

または [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からダウンロードします。

### 2. プラグインを有効化 {#2-activate-the-plugin}

インストールされているがアクティブでない場合:
1. プラグイン → インストール済みプラグイン に移動
2. 「WooCommerce POS」を探す
3. 「有効化」をクリック

### 3. エラーを確認 {#3-check-for-errors}

プラグインが有効化できない場合:
- エラーログで PHP エラーをチェック
- WooCommerce がインストールされてアクティブであることを確認
- PHP バージョン要件を確認
- プラグインの競合を確認

### 4. URL を確認 {#4-verify-the-url}

POS が正しい WordPress サイトに接続されるように構成されていることを確認してください:
- POS 設定の URL をチェック
- あなたの WordPress インストールに一致することを確認

### 5. 必要に応じて再インストール {#5-reinstall-if-necessary}

プラグインが破損している場合:
1. プラグインを無効化して削除
2. WordPress.org から再インストール
3. プラグインを有効化
