# API05002: WCPOS プラグインが見つかりません

## これは何を意味しますか

WCPOS (WooCommerce POS) プラグインが、あなたの WordPress サイトにインストールされていないか、アクティブでないということです。POS アプリケーションは、このプラグインを使用して WooCommerce と通信します。

## 一般的な原因

- **プラグインがインストールされていない** — プラグインが一度もインストールされていない
- **プラグインが無効化されている** — プラグインが無効化されている
- **プラグインが削除されている** — プラグインファイルが削除されている
- **間違ったサイト** — WCPOS のないサイトに接続されています

## 修正方法

### 1. プラグインをインストールする

インストールされていない場合:
1. WordPress 管理 → プラグイン → 新規追加
2. "WooCommerce POS" を検索
3. プラグインをインストールして有効化

または、[WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) からダウンロードします。

### 2. プラグインを有効化する

インストールされているがアクティブでない場合:
1. プラグイン → インストール済みプラグイン
2. "WooCommerce POS" を見つける
3. "有効化" をクリック

### 3. エラーを確認する

プラグインが有効化されない場合:
- エラーログで PHP エラーを確認
- WooCommerce がインストールされていてアクティブであることを確認
- PHP バージョン要件を確認
- プラグインの競合を探す

### 4. URL を確認する

POS が正しい WordPress サイトに接続するように設定されていることを確認:
- POS 設定の URL を確認
- あなたの WordPress インストールと一致していることを確認

### 5. 必要に応じて再インストールする

プラグインが破損している場合:
1. プラグインを無効化して削除
2. WordPress.org から再インストール
3. プラグインを有効化
