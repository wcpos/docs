# SY01001: メモリ不足

## これは何を意味しますか？

デバイスのメモリ（RAM）が不足しており、要求された操作を完了できません。POSは適切に機能するために十分なメモリが必要です。

## 一般的な原因

- **アプリが多すぎる** — メモリを使用している他のアプリケーション
- **大規模なデータセット** — 多くの製品/顧客が読み込まれている
- **メモリリーク** — アプリケーションがメモリを適切に解放していない
- **RAM不足** — デバイスに十分なメモリがない

## 修正方法

### 1. 他のアプリケーションを閉じる

メモリを解放します：
- 使用していないブラウザタブを閉じる
- 他のアプリケーションを閉じる
- バックグラウンドプログラムを閉じる

### 2. POSを再起動する

アプリケーションのメモリをリフレッシュします：
1. POSを完全に閉じる
2. 数秒待つ
3. アプリケーションを再度開く

### 3. デバイスを再起動する

完全な再起動はすべてのメモリをクリアします：
1. 開いている作業を保存する
2. コンピュータ/デバイスを再起動する
3. 再起動時にPOSのみを開く

### 4. ブラウザデータをクリアする（Web版）

ブラウザで使用している場合：
- ブラウザのキャッシュをクリアする
- ブラウザを閉じて再度開く
- ブラウザタブを減らす

### 5. データ負荷を減らす

非常に大きなカタログがある場合：
- 一度に読み込むデータを減らすためにフィルタを使用する
- 結果のページネーションを考慮する
- 可能であれば同期範囲を制限する

## 最小要件

デバイスが最小スペックを満たしていることを確認してください：
- **RAM:** 最小4GB、推奨8GB
- **ブラウザ:** モダンブラウザ（Chrome、Firefox、Edge）
- **デスクトップアプリ:** 要件についてリリースノートを確認する

## 継続的な問題

頻繁にメモリ不足になる場合：
- デバイスのRAMのアップグレードを検討する
- より多くのリソースを持つデバイスを使用する
- 過剰な場合は問題を報告する
