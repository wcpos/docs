# SY01001: メモリ不足

## 意味

デバイスのメモリ（RAM）が不足しており、要求された操作を完了できません。POSは適切に機能するために十分なメモリを必要とします。

## 一般的な原因

- **多くのアプリが開いている** — 他のアプリケーションがメモリを使用
- **大量のデータセット** — 多くの商品/顧客が読み込まれている
- **メモリリーク** — アプリケーションがメモリを適切に解放していない
- **RAM不足** — デバイスのメモリが不十分

## 解決方法

### 1. 他のアプリケーションを閉じる

メモリを解放:
- 使用していないブラウザタブを閉じる
- 他のアプリケーションを閉じる
- バックグラウンドプログラムを閉じる

### 2. POSを再起動

アプリケーションのメモリをリフレッシュ:
1. POSを完全に閉じる
2. 数秒待つ
3. アプリケーションを再度開く

### 3. デバイスを再起動

メモリを完全にクリア:
1. 開いている作業を保存
2. コンピューター/デバイスを再起動
3. 起動時にPOSのみを開く

### 4. ブラウザデータをクリア（Web版）

ブラウザで使用している場合:
- ブラウザキャッシュをクリア
- ブラウザを閉じて再度開く
- ブラウザタブを少なくする

### 5. データ読み込みを削減

非常に大きなカタログがある場合:
- フィルターを使用してデータ読み込みを少なくする
- 結果をページ分割することを検討
- 可能であれば同期範囲を制限する

## 最小要件

デバイスが最小仕様を満たしていることを確認:
- **RAM:** 最小4GB、推奨8GB
- **ブラウザ:** 最新のブラウザ（Chrome、Firefox、Edge）
- **デスクトップアプリ:** リリースノートで要件を確認

## 継続的な問題

頻繁にメモリ不足になる場合:
- デバイスのRAMをアップグレードを検討
- より多くのリソースを持つデバイスを使用
- 過度に見える場合は問題を報告
