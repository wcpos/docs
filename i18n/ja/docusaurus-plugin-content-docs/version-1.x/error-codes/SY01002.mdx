# SY01002: ディスクがいっぱいです

## この意味

デバイスのストレージが一杯です。POSはデータを保存したり、利用可能なディスクスペースなしで動作を続けることができません。

## 一般的な原因

- **ストレージ満杯** — ドライブに空き容量がない
- **大きいローカルデータベース** — POSデータが大量のスペースを使用
- **一時ファイル** — 一時的なファイルがスペースを消費
- **他のアプリケーション** — 他のアプリがディスクを埋めている

## 修正方法

### 1. ディスクスペースを空ける

不要なファイルを削除：
- ゴミ箱/ごみ箱を空にする
- 古いダウンロードを削除
- 使用していないアプリケーションを削除
- 一時ファイルを消去

### 2. 利用可能な容量を確認

**Windows:**
- ファイルエクスプローラーを開く
- ドライブを右クリック → プロパティ
- 使用済みと空き容量を表示

**macOS:**
- Appleメニュー → このMacについて
- ストレージをクリック

### 3. ブラウザデータを消去（ウェブ版）

ブラウザはローカルにデータを保存：
- キャッシュとクッキーを消去
- 古いサイトデータを削除
- ブラウザのストレージ使用状況を確認

### 4. 大きなファイルを移動または削除

大きなファイルを見つけて処理：
- 古い動画や写真
- 大きなダウンロード
- 使用していないソフトウェア
- 外部ストレージに移動できるバックアップファイル

### 5. POSデータを確認

POSデータが非常に大きい場合：
- ローカルデータベースのクリーニングが必要かもしれません
- クリアして再同期を検討
- 古いデータを削除できるか確認

## 推奨される空き容量

以下を確保：
- **通常の操作に10GB** の空き容量
- **大きなカタログに20GB+**
- より多くの空き容量が常に望ましい

## 将来の問題を防ぐ

- 定期的に不要なファイルをクリーンアップ
- 一時ファイルの自動クリーンアップを設定
- ディスク容量の使用状況を監視
- 頻繁に一杯になる場合は大容量ストレージを検討
