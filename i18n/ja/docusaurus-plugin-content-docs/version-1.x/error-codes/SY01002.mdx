# SY01002: ディスクが満杯

## これは何を意味するか

デバイスのストレージが満杯です。ディスクスペースが空いていないと、POSはデータを保存したり、動作を続けたりできません。

## 一般的な原因

- **ストレージの満杯** — ドライブに空きスペースがない
- **大きなローカルデータベース** — POSデータがかなりのスペースを使用している
- **一時ファイル** — スペースを消費する一時ファイル
- **他のアプリケーション** — ディスクを埋める他のアプリ

## 修正方法

### 1. ディスクスペースを空ける

不要なファイルを削除します：
- ゴミ箱/リサイクルビンを空にする
- 古いダウンロードを削除する
- 使用していないアプリケーションを削除する
- 一時ファイルをクリアする

### 2. 使用可能なスペースを確認する

**Windows:**
- ファイルエクスプローラーを開く
- ドライブを右クリック → プロパティ
- 使用済みスペースと空きスペースを確認する

**macOS:**
- アップルメニュー → このMacについて
- ストレージをクリック

### 3. ブラウザデータをクリアする（Web版）

ブラウザはデータをローカルに保存します：
- キャッシュとクッキーをクリアする
- 古いサイトデータを削除する
- ブラウザのストレージ使用状況を確認する

### 4. 大きなファイルを移動または削除する

大きなファイルを見つけて処理します：
- 古い動画や写真
- 大きなダウンロード
- 使用していないソフトウェア
- 外部ストレージに移動できるバックアップファイル

### 5. POSデータを確認する

POSデータが非常に大きい場合：
- ローカルデータベースをクリーニングする必要があるかもしれません
- クリーンアップして再同期を検討する
- 古いデータが削除できるか確認する

## 推奨される空きスペース

少なくとも次の量を確保してください：
- **10GB** 通常の操作用
- **20GB以上** 大規模カタログ用
- できるだけ多くのスペースがある方が良い

## 将来の問題を防ぐために

- 定期的に不要なファイルを整理する
- 自動一時ファイルクリーナーを設定する
- ディスクスペース使用状況を監視する
- 頻繁に満杯になる場合は、大きなストレージを検討する
