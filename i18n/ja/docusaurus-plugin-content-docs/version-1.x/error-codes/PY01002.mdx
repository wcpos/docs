# PY01002: 資金不足

## 意味

トランザクションをカバーするのに十分な資金が口座にないため、支払いが拒否されました。

## 一般的な原因

- **口座残高が低い** — 利用可能な資金が不足
- **保留中の取引** — 他の保留中の取引が利用可能な残高を減少
- **クレジット限度額に達している** — クレジットカードの限度額が最大
- **資金の保留** — 他の取引のために資金が保留中

## 解決方法

### 1. 別の支払い方法を使用

最も迅速な解決策:
- 別のカードを試す
- 現金を使用
- 複数の方法で支払いを分割（サポートされている場合）

### 2. 注文金額を削減

部分的な支払いが可能な場合:
- いくつかのアイテムを削除
- 割引またはクーポンを適用
- 小さな取引を完了

### 3. 顧客のアクション

顧客は以下を行う必要があるかもしれません:
- 口座に資金を移動
- 保留中の取引がクリアされるまで待つ
- クレジットカードの残高を支払う
- クレジット限度額の引き上げを依頼

### 4. トランザクションを分割

POSが分割支払いをサポートする場合:
- 1枚のカードで一部を支払う
- 残りを別の方法で支払う

## プライバシーに関する注意

顧客とコミュニケーションを取る際:
- 具体的な拒否理由について控えめに
- 単に「カードが拒否されました」と述べる
- プライベートに別の支払い方法を試すことを許可
