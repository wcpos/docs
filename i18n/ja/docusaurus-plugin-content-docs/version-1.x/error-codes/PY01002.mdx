# PY01002: 不足の資金

## これは何を意味するか

支払いが拒否されたのは、トランザクションを賄うための十分な資金が口座にないからです。

## 一般的な原因

- **低い口座残高** — 利用可能な資金が不足している
- **保留中のトランザクション** — 他の保留中のトランザクションが利用可能な残高を減少させている
- **クレジット限度額に達した** — クレジットカードの場合、限度額に達している
- **資金が保留されている** — 他のトランザクションのために資金が保留されている

## 修正方法

### 1. 別の支払い方法を使用する

最も迅速な解決策：
- 別のカードを試す
- 現金を使用する
- 複数の方法で支払いを分割する（サポートされている場合）

### 2. 注文金額を減らす

部分的な支払いが可能な場合：
- 一部のアイテムを削除する
- 割引やクーポンを適用する
- より小さいトランザクションを完了する

### 3. 顧客の行動

顧客が行う必要があること：
- 口座に資金を振り込む
- 保留中のトランザクションがクリアされるのを待つ
- クレジットカードの残高を支払う
- クレジット限度額の引き上げを要求する

### 4. トランザクションを分割する

POSが分割払いに対応している場合：
- 1枚のカードで一部を支払う
- 残りを別の方法で支払う

## プライバシーノート

顧客への説明を行う際：
- 特定の拒否理由については控えめに伝える
- 単に「カードは拒否されました」と述べる
- 別の支払い方法をプライベートに試すことを許可する
