POS専用製品は、製品がオンラインストア、POS、またはその両方に表示されるかを制御できる機能です。これは、オンラインと店頭販売のために異なる製品カタログがある場合に便利です。

## 概要

有効になっている場合、各製品には3つのオプションを持つ**POS可視性**設定が付与されます：

| 可視性 | オンラインストア | POS |
|--------|------------------|-----|
| **POS & オンライン** (デフォルト) | 表示 | 表示 |
| **POS専用** | 非表示 | 表示 |
| **オンライン専用** | 表示 | 非表示 |

## この機能を使用するタイミング

POS専用製品を有効にする必要がある場合：

- **店頭限定商品を販売する** - 顧客が物理店舗を訪れたときにのみ利用可能な製品
- **オンライン専用アイテムをPOSから隠す** - デジタル製品、予約商品、または店舗で販売すべきでないアイテム
- **異なるカタログを管理する** - オンラインと小売チャネルのための別々の製品範囲
- **プロモーションアイテムを制御する** - 店頭専用の特別商品またはオンライン限定のディール

:::tip
すべての製品が両方のチャネルに表示される場合、この機能は必要ありません。パフォーマンス向上のために無効のままにしてください。
:::

## 機能の有効化

1. **WP Admin → WCPOS → 設定 → 一般**に移動
2. **「POS専用製品を有効にする」**を有効にします
3. 変更を保存します

一般設定の詳細については、[WP Admin一般設定](/settings/wp-admin/general)をご覧ください。

:::warning パフォーマンス影響
この設定は必要な場合のみ有効にしてください。有効にすると、すべての製品リクエストに対して追加のデータベースルックアップが追加され、大規模カタログの店舗のパフォーマンスに影響を与える可能性があります。
:::

## 製品可視性の設定

### 個別製品

単一の製品の可視性を設定するには：

1. **WP Admin → 製品**に移動し、製品を編集
2. **製品データ**パネルで**POS可視性**を探します
3. 希望のオプションを選択：
   - **POS & オンライン** - 製品はすべての場所に表示されます（デフォルト）
   - **POS専用** - 製品はオンラインストアから隠されます
   - **オンライン専用** - 製品はPOSから隠されます
4. **更新**をクリックして保存します

### 一括編集

複数の製品の可視性を一度に変更するには：

1. **WP Admin → 製品**に移動
2. チェックボックスを使用して変更したい製品を選択
3. **一括アクション**のドロップダウンから**編集**を選択
4. **適用**をクリック
5. 一括編集パネルで**POS可視性**のドロップダウンを見つけます
6. 希望の可視性オプションを選択
7. **更新**をクリックします

### 変数製品

変数製品の場合、親製品レベルと各個別のバリエーションの両方で可視性を設定できます。これにより詳細な制御が可能です - 例えば、すべてのバリエーションをオンラインで表示しつつ、特定の色やサイズのみを店頭で表示することができます。

## POSへの影響

製品が**オンライン専用**に設定されている場合：

- [製品パネル](/pos/product-panel)に表示されません
- 検索やバーコードスキャンでは見つかりません
- POSでカートに追加できません

製品が**POS専用**に設定されている場合：

- WooCommerceストアのフロントエンドには表示されません
- オンライン検索結果には含まれません
- 顧客はオンラインで購入できません

:::note
最近製品の可視性を変更した場合で、POSにまだ表示されている（または表示されていない）場合は、同期を試してください。同期ボタンを長押しし、**「クリアしてリフレッシュ」**を選択してすべての製品データを再読み込みします。
:::

## 一般的な使用例

<UseCases>
  <UseCaseCard icon="gift" title="店頭専用ギフトカード">

  レジでアクティベートが必要な物理的なギフトカード：

  1. ギフトカード製品を作成
  2. 可視性を**POS専用**に設定
  3. カードはオンラインに表示されませんが、スタッフは店頭で販売できます

  </UseCaseCard>

  <UseCaseCard icon="globe" title="オンライン予約販売">

  オンラインで予約販売可能ですが、まだ在庫がない製品：

  1. 予約商品を作成
  2. 可視性を**オンライン専用**に設定
  3. 顧客はオンラインで予約できますが、POSが混雑することはありません

  </UseCaseCard>

  <UseCaseCard icon="building" title="卸売と小売">

  同じWooCommerceストアを卸売と小売の両方で使用している場合：

  1. 卸売専用製品を**オンライン専用**に設定（B2Bポータルアクセス用）
  2. 小売専用を**POS専用**に設定
  3. 通常の製品は**POS & オンライン**として保持

  </UseCaseCard>
</UseCases>

## トラブルシューティング

<Accordion>
  <AccordionItem question="オンライン専用に設定した後も製品が表示される">

  1. POSを開く

  3. **「クリアしてリフレッシュ」**を選択
  4. 製品が再読み込みされるのを待つ

  </AccordionItem>

  <AccordionItem question="製品に設定が表示されない">

  1. **WP Admin → WCPOS → 設定 → 一般**に移動
  2. **「POS専用製品を有効にする」**がチェックされていることを確認
  3. 変更した場合は設定を保存

  </AccordionItem>
