:::info プロ機能
商品画面では、[WCPOS Pro](/getting-started/pro-license)が必要です。無料ユーザーは[POS商品パネル](/pos/product-panel)から商品を表示したりカートに追加したりできますが、在庫や価格を編集することはできません。
:::

商品画面は専用の在庫管理インターフェースです。[POS商品パネル](/pos/product-panel)（販売用）は異なり、この画面は商品カタログの管理用に設計されています。

{/* スクリーンショット: products-screen.png */}

## 商品画面とPOS商品パネルの違い

| 機能 | POS商品パネル | 商品画面 |
|---------|-------------------|-----------------|
| **目的** | 商品をカートに追加 | 在庫管理 |
| **在庫** | 表示のみ | インライン編集 |
| **価格** | 表示のみ | インライン編集 |
| **アクション** | カートに追加 | 編集、同期、削除 |
| **利用可能な場合** | 無料 + プロ | プロのみ |

## インターフェース概要

### 検索＆フィルター

画面の上部には：

- **検索バー** - 商品名、SKU、バーコードで商品を検索
- **フィルターボタン** - 在庫状況、特集、セール、カテゴリー、タグ、ブランド
- **表示設定** (<Icon name="sliders" />) - 表示する列を設定

### 商品リスト

メインエリアには、スクロール可能なテーブル形式で商品が表示されます：

- **画像** - 商品のサムネイル
- **商品名** - 任意のSKU、バーコード、属性付き
- **在庫** - "管理"トグル付きの編集可能な在庫数量
- **在庫状況** - 在庫あり（緑）、品切れ（赤）、バックオーダー
- **カテゴリー** - 商品カテゴリーのバッジ
- **価格** - 現在の価格、通常価格、販売価格（すべて編集可能）
- **アクション** - 編集、同期、削除の三点メニュー

### バリアブル商品

バリアブル商品には：
- 属性オプション（色、サイズなど）
- すべてのバリエーションを表示する「拡張」リンク
- 別々の在庫/価格を持つ各バリエーションの行

### フッター

- **税ステータス** - 現在の税計算の基準
- **商品数** - "XのYを表示中"と同期ボタン

## 主要機能

### インライン在庫編集

在庫数量を直接クリックして編集します：

1. 在庫数をクリック
2. 新しい数量を入力
3. Enterを押して保存

変更はWooCommerceに自動的に同期されます。

### 在庫管理トグル

"管理"トグルは、この商品の在庫をWooCommerceで追跡するかどうかを制御します：

- **オン** - 在庫レベルは追跡され、販売時に減算されます
- **オフ** - 商品は常に利用可能（在庫無限）

### インライン価格編集

任意の価格フィールドをクリックして編集：

- **価格** - 現在の販売価格
- **通常価格** - 割引されていない価格
- **販売価格** - 割引価格（セール中の場合）

### 商品アクション

各商品について三点メニュー（⋮）をクリック：

- **編集** - 商品編集モーダルを開く
- **同期** - WooCommerceからこの商品を更新
- **削除** - ローカルデータベースから商品を削除

:::note
POSから商品を削除すると、それはローカルでのみ削除されます。商品はWooCommerceには残り、次の同期で再表示されます。
:::

### 利用可能な列

| 列 | 説明 | 表示オプション |
|--------|-------------|-----------------|
| **画像** | 商品のサムネイル | - |
| **ID** | WooCommerce商品ID | - |
| **商品** | 商品名 | SKU、バーコード、属性 |
| **タイプ** | シンプル、バリアブル、グループ化など | - |
| **SKU** | 在庫管理単位 | - |
| **バーコード** | 商品バーコード | - |
| **在庫** | 現在の数量（編集可能） | - |
| **在庫状況** | 在庫あり / 品切れ / バックオーダー | - |
| **カテゴリー** | 商品カテゴリー | - |
| **タグ** | 商品タグ | - |
| **ブランド** | ブランド情報（プラグインがアクティブな場合） | - |
| **売上原価** | 商品原価 | - |
| **価格** | 現在の販売価格（編集可能） | 税情報 |
| **通常価格** | 割引のない価格（編集可能） | 税情報 |
| **販売価格** | 割引価格（編集可能） | 税情報 |
| **作成日** | 商品が作成された日 | - |
| **最終更新日** | 最後の更新日 | - |
| **アクション** | 編集、同期、削除ボタン | - |

### デフォルト設定の復元

すべての表示設定を元のデフォルトにリセットするにはクリックします。

## 商品フィルタリング

フィルターボタンを使用して表示を絞り込みます：

- **在庫状況** - 在庫あり、品切れ、バックオーダー
- **特集** - 特集としてマークされた商品
- **セール中** - 現在セール中の商品
- **カテゴリー** - 商品カテゴリーでフィルタリング
- **タグ** - 商品タグでフィルタリング
- **ブランド** - ブランドでフィルタリング（ブランドプラグインを使用している場合）

## ソート

任意の列ヘッダーをクリックして、その列でソートします。再度クリックしてソート順を反転します。
