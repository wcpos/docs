:::info プロ機能
商品画面には[WCPOS Pro](/getting-started/pro-license)が必要です。無料版ユーザーは[POSプロダクトパネル](/pos/product-panel)で商品の閲覧と買い物かごへの追加は可能ですが、在庫や価格の編集はできません。
:::

商品画面は専用の在庫管理インターフェースです。[POSプロダクトパネル](/pos/product-panel)（販売用）とは異なり、この画面は商品カタログを管理するために設計されています。

{/* Screenshot: products-screen.png */}

## 商品画面 vs POSプロダクトパネル

| 機能 | POSプロダクトパネル | 商品画面 |
|---------|-------------------|-----------------|
| **目的** | 買い物かごに商品を追加 | 在庫管理 |
| **在庫** | 表示のみ | インライン編集 |
| **価格** | 表示のみ | インライン編集 |
| **操作** | 買い物かごに追加 | 編集、同期、削除 |
| **利用可能な版** | 無料 + プロ | プロのみ |

## インターフェース概要

### 検索とフィルター

画面上部：

- **検索バー** - 商品名、SKU、バーコードで検索
- **フィルターボタン** - 在庫状況、おすすめ、セール中、カテゴリ、タグ、ブランド
- **表示設定** (<Icon name="sliders" />) - 表示列の設定

### 商品リスト

メインエリアに商品がスクロール可能な表で表示：

- **画像** - 商品サムネイル
- **商品名** - オプションでSKU、バーコード、属性
- **在庫** - 編集可能な在庫数と「管理」トグル
- **在庫状況** - 在庫あり（緑）、在庫なし（赤）、バックオーダー
- **カテゴリ** - 商品カテゴリバッジ
- **価格** - 現在の価格、通常価格、セール価格（すべて編集可能）
- **操作** - 3点メニュー（編集、同期、削除）

### バリエーション商品

バリエーション商品には：
- 属性オプション（色、サイズなど）
- すべてのバリエーションを表示する「展開」リンク
- 在庫・価格が個別に設定できるバリエーション行

### フッター

- **税金状況** - 現在の税計算基準
- **商品数** - 「X/Y を表示」と同期ボタン

## 主な機能

### インライン在庫編集

在庫数をクリックして直接編集：

1. 在庫数をクリック
2. 新しい数量を入力
3. Enterキーで保存

変更は自動的にWooCommerceと同期されます。

### 在庫管理トグル

「管理」トグルはWooCommerceの在庫追跡を制御：

- **オン** - 在庫レベルを追跡し、販売時に減算
- **オフ** - 常に在庫あり（無制限）

### インライン価格編集

任意の価格項目をクリックして編集：

- **価格** - 現在の販売価格
- **通常価格** - 割引前の価格
- **セール価格** - 割引価格（セール中）

### 商品操作

各商品の3点メニュー（⋮）をクリック：

- **編集** - 商品編集モーダルを開く
- **同期** - WooCommerceから商品を更新
- **削除** - ローカルデータベースから削除

:::note
POSから商品を削除しても、その商品はWooCommerceに残り、次の同期時に再表示されます。
:::

### 利用可能な列

| 列 | 説明 | 表示オプション |
|--------|-------------|-----------------|
| **画像** | 商品サムネイル | - |
| **ID** | WooCommerceの商品ID | - |
| **商品** | 商品名 | SKU、バーコード、属性 |
| **種類** | シンプル、バリエーション、グループなど | - |
| **SKU** | 在庫管理単位 | - |
| **バーコード** | 商品バーコード | - |
| **在庫** | 現在の数量（編集可） | - |
| **在庫状況** | 在庫あり / 在庫なし / バックオーダー | - |
| **カテゴリ** | 商品カテゴリ | - |
| **タグ** | 商品タグ | - |
| **ブランド** | ブランド情報（プラグイン有効時） | - |
| **売上原価** | 商品の仕入れ価格 | - |
| **価格** | 現在の販売価格（編集可） | 税情報 |
| **通常価格** | 割引前の価格（編集可） | 税情報 |
| **セール価格** | 割引価格（編集可） | 税情報 |
| **作成日** | 商品作成日 | - |
| **更新日** | 最終更新日 | - |
| **操作** | 編集、同期、削除ボタン | - |

### デフォルト設定に戻す

クリックすると、すべての表示設定を元のデフォルトに戻します。

## 商品のフィルタリング

フィルターボタンで表示を絞り込み：

- **在庫状況** - 在庫あり、在庫なし、バックオーダー
- **おすすめ** - おすすめ商品としてマークされた商品
- **セール中** - 現在セール中の商品
- **カテゴリ** - 商品カテゴリでフィルター
- **タグ** - 商品タグでフィルター
- **ブランド** - ブランドでフィルター（ブランドプラグイン使用時）

## 並べ替え

任意の列見出しをクリックしてその列で並べ替え。再度クリックで並べ替え順序を反転。
