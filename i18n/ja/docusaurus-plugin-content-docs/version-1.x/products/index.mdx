:::info プロ機能
商品画面は[WCPOS Pro](/getting-started/pro-license)が必要です。無料ユーザーは[POS商品パネル](/pos/product-panel)から製品を表示してカートに追加できますが、在庫や価格を編集することはできません。
:::

商品画面は専用の在庫管理インターフェースです。[POS商品パネル](/pos/product-panel)（販売用）とは異なり、この画面は製品カタログの管理用に設計されています。

{/* スクリーンショット: products-screen.png */}

## 商品画面 vs POS商品パネル

| 機能 | POS商品パネル | 商品画面 |
|------|----------------|-----------|
| **目的** | 商品をカートに追加 | 在庫管理 |
| **在庫** | 表示のみ | インライン編集 |
| **価格** | 表示のみ | インライン編集 |
| **アクション** | カートに追加 | 編集、同期、削除 |
| **利用可能** | 無料 + Pro | Proのみに対応 |

## インターフェース概要

### 検索 & フィルター

画面の上部にあります：

- **検索バー** - 名前、SKU、バーコードで製品を検索
- **フィルターボタン** - 在庫状況、特集、セール中、カテゴリー、タグ、ブランド
- **表示設定** (<Icon name="sliders" />) - 表示する列を設定

### 製品リスト

メインエリアには製品がスクロール可能なテーブルで表示されます：

- **画像** - 製品サムネイル
- **商品名** - オプションのSKU、バーコード、属性付き
- **在庫** - 「管理」トグル付きの編集可能な在庫数量
- **在庫状況** - 在庫あり（緑）、在庫切れ（赤）、バックオーダー
- **カテゴリー** - 製品カテゴリーのバッジ
- **価格** - 現在の価格、通常価格、セール価格（すべて編集可能）
- **アクション** - 編集、同期、削除の三点メニュー

### バリアブル製品

バリアブル製品は次のことを表示します：
- 属性オプション（色、サイズなど）
- すべてのバリエーションを表示するための「展開」リンク
- 別々の在庫/価格を持つ個別のバリエーション行

### フッター

- **税の状況** - 現在の税計算基準
- **商品数** - 「X の Y を表示中」と同期ボタン (<Icon name="arrow-rotate-right" />)。**長押し**で同期ボタンを押すとクリアとリフレッシュオプションが表示されます

## 主な機能

### インライン在庫編集

在庫数量を直接クリックして編集します：

1. 在庫番号をクリック
2. 新しい数量を入力
3. Enterを押して保存

変更はWooCommerceに自動的に同期されます。

### 在庫管理トグル

「管理」トグルは、この製品の在庫をWooCommerceで追跡するかどうかを制御します：

- **オン** - 売上に応じて在庫レベルが追跡され、減算されます
- **オフ** - 製品は常に利用可能（無限在庫）

### インライン価格編集

任意の価格フィールドをクリックして編集します：

- **価格** - 現在の販売価格
- **通常価格** - 割引されていない価格
- **セール価格** - 割引された価格（セール中のとき）

### 製品アクション

各製品の三点メニュー（⋮）をクリックします：

- **編集** - 製品編集モーダルを開く
- **同期** - WooCommerceからこの製品をリフレッシュ
- **削除** - ローカルデータベースから製品を削除

:::note
POSから製品を削除すると、ローカルからのみ削除されます。製品はWooCommerceには残り、次の同期で再表示されます。
:::

### 利用可能な列

| 列 | 説明 | 表示オプション |
|----|------|-----------------|
| **画像** | 製品サムネイル | - |
| **ID** | WooCommerce製品ID | - |
| **商品** | 商品名 | SKU、バーコード、属性 |
| **タイプ** | シンプル、バリアブル、グループなど | - |
| **SKU** | 在庫保持単位 | - |
| **バーコード** | 製品バーコード | - |
| **在庫** | 現在の数量（編集可能） | - |
| **在庫状況** | 在庫あり / 在庫切れ / バックオーダー | - |
| **カテゴリー** | 製品カテゴリー | - |
| **タグ** | 製品タグ | - |
| **ブランド** | ブランド情報（プラグインがアクティブな場合） | - |
| **売上原価** | 製品原価 | - |
| **価格** | 現在の販売価格（編集可能） | 税情報 |
| **通常価格** | 割引されていない価格（編集可能） | 税情報 |
| **セール価格** | 割引された価格（編集可能） | 税情報 |
| **作成日** | 製品が作成された日 | - |
| **最終更新日** | 最後の変更日 | - |
| **アクション** | 編集、同期、削除ボタン | - |

### デフォルト設定に戻す

すべての表示設定を元のデフォルトにリセットするにはクリックします。

## 商品のフィルタリング

フィルターボタンを使用して表示を絞り込みます：

- **在庫状況** - 在庫あり、在庫切れ、バックオーダー
- **特集** - 特集としてマークされた製品
- **セール中** - 現在セール中の製品
- **カテゴリー** - 製品カテゴリーでフィルタリング
- **タグ** - 製品タグでフィルタリング
- **ブランド** - ブランドでフィルタリング（ブランドプラグインを使用している場合）

## ソート

任意の列ヘッダーをクリックすると、その列でソートされます。再度クリックするとソート順序が逆になります。
