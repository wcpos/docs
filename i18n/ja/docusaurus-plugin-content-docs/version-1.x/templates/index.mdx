テンプレートを使用すると、POSシステムからのさまざまな出力の外観と内容をカスタマイズできます。テンプレートエディタには、次の場所に移動することでアクセスできます: `WP Admin > POS > テンプレート`

テンプレートシステムは、次のカスタマイズの柔軟性を提供します:

- **レシートテンプレート** - 印刷されたレシートとデジタルレシートをカスタマイズ
- **レポートテンプレート** - レポートのレイアウトをカスタマイズ _(近日公開)_
- **メールテンプレート** - メール通知をカスタマイズ _(近日公開)_

## テンプレートの種類

### デフォルトテンプレート

プラグインとテーマファイルから自動的に検出されます。これらは削除できませんが、カスタムバージョンを作成するためにコピーできます。

### カスタムテンプレート

ユーザーが作成したテンプレートで、WordPressの管理インターフェースを通じて完全にカスタマイズ、編集、管理することができます。
