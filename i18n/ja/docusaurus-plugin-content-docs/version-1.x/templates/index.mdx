テンプレートを使用すると、POSシステムからのさまざまな出力の外観や内容をカスタマイズできます。テンプレートエディタにアクセスするには、次のように移動します: `WP Admin > POS > テンプレート`

テンプレートシステムは、カスタマイズの柔軟性を提供します:

- **レシートテンプレート** - 印刷されたおよびデジタルレシートをカスタマイズ
- **レポートテンプレート** - レポートレイアウトをカスタマイズ _(近日公開)_
- **メールテンプレート** - メール通知をカスタマイズ _(近日公開)_

## テンプレートの種類 {#template-types}

### デフォルトテンプレート {#default-templates}

プラグインおよびテーマファイルから自動的に検出されます。これらは削除できませんが、カスタムバージョンを作成するためにコピーできます。

### カスタムテンプレート {#custom-templates}

ユーザーが作成したテンプレートで、WordPress管理インターフェイスを通じて完全にカスタマイズ、編集、および管理できます。
