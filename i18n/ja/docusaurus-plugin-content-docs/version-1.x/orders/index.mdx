:::info プロ機能
注文画面には [WCPOS Pro](/getting-started/pro-license) が必要です。
:::

注文画面は、POS 内で直接注文履歴へのアクセスを提供します。過去の注文を表示し、領収書を再印刷し、WooCommerce 管理画面に切り替えることなく注文詳細を管理します。

{/* スクリーンショット: orders-screen.png */}

## インターフェースの概要

### 検索とフィルター

画面の上部にあります:

- **検索バー** - 注文番号、顧客名などで注文を探します。
- **ステータスフィルター** - 注文のステータスでフィルタリングします（完了、処理中など）
- **顧客フィルター** - 顧客でフィルタリングします
- **キャッシャーフィルター** - 注文を処理した人でフィルタリングします
- **店舗フィルター** - 店舗でフィルタリングします（複数店舗設定の場合）
- **日付範囲** - 日付でフィルタリングします

アクティブなフィルターは、×をクリックすることで削除できる青いバッジとして表示されます。

### 注文テーブル

主な領域には次のように注文が表示されます:

- **ステータスアイコン** - ビジュアルインジケーター（✓ 完了、🕐 保留中、🛒 オープン）
- **注文番号** - 一意の注文 ID
- **顧客** - 顧客名（または「ゲスト」）
- **請求先住所** - 顧客の請求情報
- **ノートインジケーター** - 注文にノートがあるときに表示される吹き出し
- **Date Created** - 相対的なタイムスタンプ（「2 時間前」）
- **キャッシャー** - 注文を処理した人
- **支払い方法** - 現金、カードなど
- **合計** - 注文の合計金額
- **アクション** - 3 点メニュー

### フッター

- 注文数と同期ボタン (<Icon name="arrow-rotate-right" />)。**長押し**でクリアおよびリフレッシュオプション

## 主な機能

### 注文検索

次の方法で注文を迅速に見つけます:

- **注文番号検索**
- **顧客名検索**
- **日付範囲フィルタリング**
- **ステータスフィルタリング**
- **キャッシャーフィルタリング**

### 領収書の再印刷

任意の注文について、次の操作が可能です:

1. 3 点メニューをクリック
2. 領収書オプションを選択
3. 領収書を表示、印刷、またはメール送信

### 注文詳細

注文をクリックすると次の情報が表示されます:

- 完全な品目詳細
- 顧客情報
- 支払い詳細
- 注文ノート
- メタデータ

### 利用可能な列

| 列 | 説明 |
|--------|-------------|
| **ステータス** | 注文ステータスアイコン |
| **注文番号** | 一意の注文 ID |
| **顧客** | 顧客名と詳細 |
| **請求先住所** | 請求情報 |
| **配送先住所** | 配送情報 |
| **顧客ノート** | 顧客からのノート |
| **Date Created** | 注文が行われた日 |
| **Date Modified** | 最後の変更 |
| **Date Completed** | 注文が完了した日 |
| **Date Paid** | 支払いを受け取った日 |
| **Created Via** | 注文のソース（POS、オンラインなど） |
| **キャッシャー** | 注文を処理した人 |
| **支払い方法** | 使用された支払いタイプ |
| **合計** | 注文の合計 |
| **領収書** | 即時領収書アクセス |
| **アクション** | 編集、表示など |

## 注文アクション

3 点メニュー (⋮) をクリックしてオプションを表示します:

- **表示** - 注文の詳細を確認
- **編集** - 注文情報を変更
- **領収書** - 領収書を表示/印刷/メール送信
- **同期** - サーバーから注文を更新
