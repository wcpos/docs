:::info プロ機能
注文画面は[WCPOS Pro](/getting-started/pro-license)を必要とします。
:::

注文画面はPOS内で直接注文履歴にアクセスできます。過去の注文を表示し、レシートを再印刷し、WooCommerce管理に切り替えることなく注文の詳細を管理します。

{/* スクリーンショット: orders-screen.png */}

## インターフェースの概要

### 検索とフィルター

画面の上部に：

- **検索バー** - 注文番号、顧客名などで注文を検索します。
- **ステータスフィルター** - 注文ステータス（完了、処理中など）でフィルタリングします。
- **顧客フィルター** - 顧客でフィルタリングします。
- **キャッシャーフィルター** - 注文を処理した人でフィルタリングします。
- **店舗フィルター** - 店舗（マルチストア設定の場合）でフィルタリングします。
- **日付範囲** - 日付でフィルタリングします。

アクティブなフィルターは、×をクリックすることで削除できる青いバッジとして表示されます。

### 注文テーブル

主要エリアには以下の注文が表示されます：

- **ステータスアイコン** - ビジュアルインジケーター（✓ 完了、🕐 保留中、🛒 開いている）
- **注文番号** - ユニークな注文ID
- **顧客** - 顧客名（または「ゲスト」）
- **請求先住所** - 顧客の請求情報
- **ノートインジケーター** - 注文にノートがある時に表示される吹き出し
- **作成日** - 相対タイムスタンプ（「2時間前」）
- **キャッシャー** - 注文を処理した人
- **支払い方法** - 現金、カードなど
- **合計** - 注文の合計金額
- **アクション** - 三点リーダーメニュー

### フッター

- 同期ボタン付きの製品数

## 主な機能

### 注文検索

以下の方法で迅速に注文を見つけることができます：

- **注文番号検索**
- **顧客名検索**
- **日付範囲フィルタリング**
- **ステータスフィルタリング**
- **キャッシャーフィルタリング**

### レシート再印刷

任意の注文について、次のことができます：

1. 三点リーダーメニューをクリック
2. レシートオプションを選択
3. レシートを表示、印刷、またはメール送信

### 注文詳細

注文をクリックすると、以下の内容が表示されます：

- 完全な品目詳細
- 顧客情報
- 支払い詳細
- 注文ノート
- メタデータ

## 注文アクション

三点リーダーメニュー（⋮）をクリックすると、オプションが表示されます：

- **表示** - 完全な注文詳細を見る
- **編集** - 注文情報を修正する
- **レシート** - レシートを表示/印刷/メール送信
- **同期** - サーバーからの注文を更新

## 使用例

### 顧客サービス

- 過去の注文を調べて、返品や問い合わせを支援する
- 顧客が購入した商品を確認する
- 要求に応じてレシートを再印刷する

### 終日の作業

- その日のすべての注文をレビューする
- キャッシャーごとにフィルタリングして個別の合計を確認
- 照合のために[レポート](/reports)と照合する

### 注文追跡

- 注文のステータスを監視する
- 保留中と完了した注文を追跡する
- 注意が必要な注文を特定する
