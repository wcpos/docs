# Stripeターミナルゲートウェイ {#stripe-terminal-gateway}

Stripeターミナルゲートウェイを使用すると、WCPOS内でStripeターミナルハードウェアリーダーを使用して対面での支払いを受け入れることができます。このゲートウェイは、物理的なカードリーダーとテスト用のシミュレーターモードの両方をサポートしています。

  </FeatureCard>

## インストール {#installation}

<Steps>
  <StepCard number="1" title="ダウンロードとインストール">

1. [Stripeターミナル for WooCommerceのリリースページ](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)にアクセスします
2. 最新の**stripe-terminal-for-woocommerce.zip**ファイルをダウンロードします
3. WordPress管理画面で、`プラグイン > 新規追加 > プラグインのアップロード`に移動します
4. zipファイルをアップロードし、プラグインを有効化します

  </StepCard>
  <StepCard number="2" title="Stripe設定の構成">

1. `WP Admin > WooCommerce > 設定 > 支払い`に移動します
2. 支払い方法リストで**Stripeターミナル**を見つけます
3. **Stripeターミナル**をクリックして設定にアクセスします
4. **Stripeシークレットキー**を入力します（Stripeダッシュボードから取得できます）
5. 設定を保存します

:::note
WooCommerceの設定でStripeターミナルゲートウェイを有効にする必要はありません。次のステップでPOS専用に有効化されます。
:::

  </StepCard>
  <StepCard number="3" title="WCPOSでの有効化" isLast>

1. `WP Admin > POS > 設定 > チェックアウト`に移動します
2. リストで**Stripeターミナル**ゲートウェイを見つけます
3. POSで使用するためにゲートウェイを有効化します
4. 設定を保存します

  </StepCard>
</Steps>

## 使用方法 {#usage}

### リーダーの接続 {#connecting-a-reader}

POSでのチェックアウト中にStripeターミナルゲートウェイを選択すると：

1. **接続方法の選択**: 物理リーダーを接続するか、シミュレーターを使用できます
2. **物理リーダー**: 画面の指示に従ってStripeターミナルデバイスを接続します
3. **シミュレーター**: ハードウェアなしでさまざまな支払いシナリオをテストするためにシミュレーターモードを選択します

### 支払いの処理 {#processing-payments}

1. **アイテムの追加**: POSでカートに商品を追加します
2. **ゲートウェイの選択**: 支払い方法として「Stripeターミナル」を選択します
3. **リーダーの接続**: リーダーを接続するか、シミュレーターモードを選択します
4. **支払いの処理**: 取引を完了するためのプロンプトに従います
5. **確認**: 支払いが成功すると、注文が自動的に完了します

### シミュレーターでのテスト {#testing-with-simulator}

シミュレーターを使用すると、さまざまな支払い方法とシナリオをテストできます：

- **カード支払い**: 異なるカードタイプ（Visa、Mastercardなど）をテスト
- **非接触型支払い**: タップして支払う取引をシミュレート
- **エラーシナリオ**: 拒否された支払いおよびその他のエラー条件をテスト
- **異なる金額**: 様々な取引金額をテスト

## ハードウェアの互換性 {#hardware-compatibility}

:::important 接続要件
この実装はStripeのJavaScript SDKを使用しており、Webアプリケーションを通じて機能しますが、**インターネットに接続されたターミナルのみを必要とします**。Bluetoothターミナルは現在サポートされていません。
:::

### サポートされているターミナル（インターネット接続） {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - イーサネット/WiFi接続ターミナル
- **WisePOS E** ✅ - WiFi接続ターミナル

### サポートされていないターミナル（Bluetooth） {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Bluetooth専用
- **BBPOS WisePad 3** ❌ - Bluetooth専用  
- **Verifone P400** ❌ - Bluetooth専用

:::note 今後のサポート
Bluetoothターミナルのサポートは、**2026年第1四半期**にiOSおよびAndroidアプリケーションに追加されます。これにより、M2やWisePad 3を含むすべてのStripeターミナル認定リーダーのサポートが可能になります。
:::

### 一般的な問題 {#common-issues}

<Accordion>
  <AccordionItem question="リーダーが接続できない">

  - サポートされているインターネット接続ターミナル（S700またはWisePOS E）を使用していることを確認してください
  - ターミナルがWiFi/Ethernetに接続されていてオンラインであることを確認してください
  - Stripeアカウントにターミナルが有効化されていることを確認してください
  - ターミナルがStripeダッシュボードに登録されていることを確認してください

  </AccordionItem>

  <AccordionItem question="支払いが拒否された">

  - Stripeアカウントがアクティブで良好な状態であることを確認してください
  - 使用されているカードが有効であることを確認してください
  - 十分な資金があることを確認してください

  </AccordionItem>

  <AccordionItem question="SSL証明書エラー">

  - StripeターミナルはHTTPSを必要とします - サイトに有効なSSL証明書があることを確認してください
  - SSL証明書が正しく構成されていることを確認してください

  </AccordionItem>
</Accordion>
