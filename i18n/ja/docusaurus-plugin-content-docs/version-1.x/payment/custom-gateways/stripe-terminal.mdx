# Stripe Terminalゲートウェイ

Stripe Terminalゲートウェイを使用すると、Stripe Terminal物理端末を介して、WooCommerce POS内で対面での支払いを受け付けることができます。このゲートウェイは、物理的なカード読み取り端末とシミュレーターモードの両方をサポートしています。

## 機能

- **ハードウェア連携**: インターネット接続を介してStripe Terminal端末を接続
- **シミュレーターモード**: ハードウェアなしでStripeシミュレーターを使用して支払いをテスト
- **複数の支払い方法**: 各種カードタイプと支払い方法をサポート
- **リアルタイム処理**: 即時の支払い処理と確認
- **セキュアな取引**: Stripeを通じたPCI準拠の支払い処理
- **Web SDK**: Webアプリケーションとの互換性のためのStripe JavaScriptライブラリを使用

## インストール

### ステップ1: ダウンロードとインストール

1. [Stripe Terminal for WooCommerceリリースページ](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)を開く
2. 最新の **stripe-terminal-for-woocommerce.zip** ファイルをダウンロード
3. WordPress管理画面で、`プラグイン > 新規追加 > プラグインをアップロード`に移動
4. zipファイルをアップロードし、プラグインをアクティブ化

### ステップ2: Stripe設定の構成

1. `WP管理 > WooCommerce > 設定 > 支払い`に移動
2. 支払い方法のリストで**Stripe Terminal**を探す
3. **Stripe Terminal**をクリックして設定にアクセス
4. **Stripeシークレットキー**を入力（Stripeダッシュボードから取得可能）
5. 設定を保存

:::note
WooCommerce設定でStripe Terminalゲートウェイを有効にする必要はありません。次のステップで特にPOS用に有効化されます。
:::

### ステップ3: WooCommerce POSで有効化

1. `WP管理 > WooCommerce POS > 設定 > チェックアウト`に移動
2. リストで**Stripe Terminal**ゲートウェイを探す
3. POS内で使用するゲートウェイを有効化
4. 設定を保存

## 使用方法

### 端末の接続

POS内でStripe Terminalゲートウェイを選択する際：

1. **接続方法の選択**: 物理端末を接続するか、シミュレーターを使用
2. **物理端末**: 画面の指示に従ってStripe Terminal端末を接続
3. **シミュレーター**: ハードウェアなしでさまざまな支払いシナリオをテストするシミュレーターモードを選択

### 支払い処理

1. **商品の追加**: POSでカートに商品を追加
2. **ゲートウェイの選択**: 支払い方法として「Stripe Terminal」を選択
3. **端末の接続**: 端末を接続するか、シミュレーターモードを選択
4. **支払い処理**: プロンプトに従って取引を完了
5. **確認**: 支払い成功後、注文が自動的に完了

### シミュレーターでのテスト

シミュレーターでは、さまざまな支払い方法とシナリオをテストできます：

- **カード支払い**: 各種カードタイプ（Visa、Mastercardなど）をテスト
- **非接触型支払い**: タップ決済取引をシミュレート
- **エラーシナリオ**: 支払い拒否などのエラー状況をテスト
- **異なる金額**: さまざまな取引金額をテスト

## 要件

- **Stripeアカウント**: Terminalが有効化されたアクティブなStripeアカウント
- **APIキー**: ダッシュボードからのStripeシークレットキー
- **WooCommerce POS**: フリーまたはプロ版がインストールされアクティブ化
- **HTTPS**: セキュリティのためにサイトがSSL/HTTPSを使用

## ハードウェア互換性

:::important 接続要件
この実装ではStripeのJavaScriptライブラリを使用するため、Webアプリケーションを通じて動作しますが、**インターネットに接続された端末のみ**が必要です。Bluetooth端末は現在サポートされていません。
:::

### サポートされている端末（インターネット接続）

- **Stripe Reader S700** ✅ - イーサネット/WiFi接続端末
- **WisePOS E** ✅ - WiFi接続端末

### サポートされていない端末（Bluetooth）

- **BBPOS Chipper 2X BT** ❌ - Bluetoothのみ
- **BBPOS WisePad 3** ❌ - Bluetoothのみ
- **Verifone P400** ❌ - Bluetoothのみ

:::note 今後のサポート
Bluetooth端末のサポートは、**2026年第1四半期**にiOSおよびAndroidアプリケーションに追加されます。これにより、M2やWisePad 3を含むすべてのStripe Terminal認定端末のサポートが可能になります。
:::

## トラブルシューティング

### 一般的な問題

**端末が接続されない**
- サポートされているインターネット接続端末（S700またはWisePOS E）を使用していることを確認
- 端末がWiFi/イーサネットに接続されオンラインであることを確認
- Stripeアカウントでterminalが有効になっていることを確認
- 端末がStripeダッシュボードに登録されていることを確認

**支払いが拒否される**
- Stripeアカウントが有効で正常な状態であることを確認
- 使用しているカードが有効であることを確認
- 十分な資金があることを確認

**SSL証明書エラー**
- Stripe Terminalには HTTPS が必要 - サイトに有効なSSL証明書があることを確認
- SSL証明書が正しく設定されていることを確認

### サポートを得る

技術的なサポートについて：
- 問題を報告するには[Githubリポジトリ](https://github.com/wcpos/stripe-terminal-for-woocommerce)を訪問
- ハードウェア固有のガイダンスについては[Stripe Terminal ドキュメント](https://stripe.com/docs/terminal)を確認
- アカウント関連の問題については、Stripeサポートに連絡
