# ストライプターミナルゲートウェイ {#stripe-terminal-gateway}

ストライプターミナルゲートウェイを使用すると、WooCommerce POS内でStripe Terminalハードウェアリーダーを使って対面での支払いを受け付けることができます。このゲートウェイは、物理的なカードリーダーとテスト用のシミュレーターモードの両方をサポートします。

  <FeatureCard icon="credit-card" title="ハードウェア統合">

  </FeatureCard>
  <FeatureCard icon="desktop" title="シミュレーターモード">

  </FeatureCard>
  <FeatureCard icon="shield" title="安全なトランザクション">
    Stripeを通じてPCI準拠の支払い処理
  </FeatureCard>
</FeatureGrid>

## インストール {#installation}

<Steps>
  <StepCard number="1" title="ダウンロードとインストール">

1. [WooCommerce用ストライプターミナルのリリースページ](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)にアクセスします
2. 最新の**stripe-terminal-for-woocommerce.zip**ファイルをダウンロードします
3. WordPress管理画面で、`プラグイン > 新規追加 > プラグインのアップロード`に移動します
4. zipファイルをアップロードしてプラグインを有効化します

  </StepCard>
  <StepCard number="2" title="Stripe設定の構成">

1. `WP管理 > WooCommerce > 設定 > 支払い`に移動します
2. 支払い方法のリストで**Stripe Terminal**を見つけます
3. **Stripe Terminal**をクリックして設定を開きます
4. **Stripe Secret Key**を入力します（これをStripeのダッシュボードから取得できます）
5. 設定を保存します

:::note
WooCommerceの設定でStripe Terminalゲートウェイを有効にする必要はありません。次のステップでPOS専用に有効にされます。
:::

  </StepCard>
  <StepCard number="3" title="WooCommerce POSで有効にする" isLast>

1. `WP管理 > WooCommerce POS > 設定 > チェックアウト`に移動します
2. リストで**Stripe Terminal**ゲートウェイを見つけます
3. POSで使用するためにゲートウェイを有効にします
4. 設定を保存します

  </StepCard>
</Steps>

## 使用方法 {#usage}

### リーダーを接続する {#connecting-a-reader}

POSのチェックアウト中にStripe Terminalゲートウェイを選択すると：

1. **接続方法の選択**：物理リーダーを接続するか、シミュレーターを使用します
2. **物理リーダー**：Stripe Terminalデバイスを接続するための画面の指示に従います
3. **シミュレーター**：ハードウェアなしでさまざまな支払いシナリオをテストするためにシミュレーターモードを選択します

### 支払いを処理する {#processing-payments}

1. **アイテムを追加**：POSでカートに商品を追加します
2. **ゲートウェイを選択**：支払い方法として「Stripe Terminal」を選択します
3. **リーダーを接続**：リーダーを接続するか、シミュレーターモードを選択します
4. **支払い処理**：トランザクションを完了させるための指示に従います
5. **確認**：支払いが成功すると、注文が自動的に完了します

### シミュレーターでのテスト {#testing-with-simulator}

シミュレーターを使用してさまざまな支払い方法やシナリオをテストできます：

- **カード決済**：さまざまなカードタイプ（Visa、Mastercardなど）をテストします
- **非接触型決済**：タップして支払いを行う取引をシミュレートします
- **エラーシナリオ**：支払い拒否やその他のエラー条件をテストします
- **異なる金額**：さまざまなトランザクションの金額をテストします

## ハードウェア互換性 {#hardware-compatibility}

:::important 接続要件
この実装はStripeのJavaScript SDKを使用しているため、ウェブアプリケーションを介して機能しますが、**インターネット接続されたターミナルのみ**が必要です。Bluetoothターミナルは現在サポートされていません。
:::

### サポートされているターミナル（インターネット接続） {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - イーサネット/WiFi接続ターミナル
- **WisePOS E** ✅ - WiFi接続ターミナル

### サポートされていないターミナル（Bluetooth） {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Bluetoothのみ
- **BBPOS WisePad 3** ❌ - Bluetoothのみ  
- **Verifone P400** ❌ - Bluetoothのみ

:::note 今後のサポート
Bluetoothターミナルのサポートは、**2026年第1四半期**にiOSおよびAndroidアプリケーションに追加される予定です。これにより、M2やWisePad 3などのすべてのStripe Terminal認定リーダーがサポートされます。
:::

### よくある問題 {#common-issues}

<Accordion>
  <AccordionItem question="リーダーが接続されない">

  - サポートされているインターネット接続ターミナル（S700またはWisePOS E）を使用していることを確認してください
  - ターミナルがWiFi/Ethernetに接続されており、オンラインであることを確認してください
  - Stripeアカウントでターミナルが有効になっていることを確認してください
  - ターミナルがStripeのダッシュボードに登録されていることを確認してください

  </AccordionItem>

  <AccordionItem question="支払いが拒否される">

  - Stripeアカウントがアクティブで良好な状態であることを確認してください
  - 使用されているカードが有効であることを確認してください
  - 十分な資金が利用可能であることを確認してください

  </AccordionItem>

  <AccordionItem question="SSL証明書エラー">

  - Stripe TerminalはHTTPSを必要とします - サイトに有効なSSL証明書があることを確認してください
  - SSL証明書が正しく設定されていることを確認してください

  </AccordionItem>
</Accordion>
