# Stripeターミナルゲートウェイ {#stripe-terminal-gateway}

Stripeターミナルゲートウェイを使用すると、WooCommerce POS内でStripeターミナルハードウェアリーダーを使用して対面での支払いを受け付けることができます。このゲートウェイは、物理カードリーダーとシミュレーターモードの両方をサポートしています。

## 機能 {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="ハードウェア統合">

  </FeatureCard>
  <FeatureCard icon="desktop" title="シミュレーターモード">
    Stripeのシミュレーターを使用してハードウェアなしで支払いをテストします
  </FeatureCard>

  </FeatureCard>
  <FeatureCard icon="shield" title="安全なトランザクション">
    Stripeを通じたPCI準拠の支払い処理
  </FeatureCard>
</FeatureGrid>

## インストール {#installation}

<Steps>
  <StepCard number="1" title="ダウンロードとインストール">

1. [WooCommerce用Stripeターミナルのリリースページ](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)にアクセスします
2. 最新の**stripe-terminal-for-woocommerce.zip**ファイルをダウンロードします
3. WordPress管理画面で、`プラグイン > 新規追加 > プラグインのアップロード`に移動します
4. ZIPファイルをアップロードしてプラグインを有効化します

  </StepCard>
  <StepCard number="2" title="Stripe設定の構成">

1. `WP管理 > WooCommerce > 設定 > 支払い`に移動します
2. 支払い方法リストで**Stripeターミナル**を見つけます
3. **Stripeターミナル**をクリックして設定にアクセスします
4. **Stripe秘密鍵**を入力します（Stripeダッシュボードから取得できます）
5. 設定を保存します

:::note
WooCommerceの設定でStripeターミナルゲートウェイを有効にする必要はありません。次のステップでPOS専用に有効になります。
:::

  </StepCard>
  <StepCard number="3" title="WooCommerce POSでの有効化" isLast>

1. `WP管理 > WooCommerce POS > 設定 > チェックアウト`に移動します
2. リスト内で**Stripeターミナル**ゲートウェイを見つけます
3. POSで使用するためにゲートウェイを有効にします
4. 設定を保存します

  </StepCard>
</Steps>

## 使用方法 {#usage}

### リーダーの接続 {#connecting-a-reader}

POSでチェックアウト中にStripeターミナルゲートウェイを選択すると：

1. **接続方法の選択**：物理リーダーを接続するか、シミュレーターを使用します
2. **物理リーダー**：画面の指示に従ってStripeターミナルデバイスを接続します
3. **シミュレーター**：ハードウェアなしでさまざまな支払いシナリオをテストするためにシミュレーターモードを選択します

### 支払いの処理 {#processing-payments}

1. **アイテムの追加**：POSのカートに製品を追加します
2. **ゲートウェイの選択**：支払い方法として「Stripeターミナル」を選択します
3. **リーダーの接続**：リーダーを接続するかシミュレーターモードを選択します
4. **支払いの処理**：取引を完了するために指示に従います
5. **確認**：支払いに成功すると、注文は自動的に完了します

### シミュレーターでのテスト {#testing-with-simulator}

シミュレーターを使用すると、さまざまな支払い方法やシナリオをテストできます：

- **カード支払い**：異なるカードタイプ（Visa、Mastercardなど）をテストします
- **非接触型支払い**：タップで支払う取引をシミュレートします
- **エラーシナリオ**：支払い拒否や他のエラー条件をテストします
- **異なる金額**：さまざまな取引金額をテストします

## ハードウェア互換性 {#hardware-compatibility}

:::important 接続要件
この実装はStripeのJavaScript SDKを使用しているため、Webアプリケーションを介して機能しますが、**インターネット接続されたターミナルのみ**が必要です。Bluetoothターミナルは現在サポートされていません。
:::

### サポートされているターミナル（インターネット接続） {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Ethernet/WiFi接続ターミナル
- **WisePOS E** ✅ - WiFi接続ターミナル

### サポートされていないターミナル（Bluetooth） {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Bluetoothのみ
- **BBPOS WisePad 3** ❌ - Bluetoothのみ  
- **Verifone P400** ❌ - Bluetoothのみ

:::note 将来のサポート
Bluetoothターミナルのサポートは、**2026年第1四半期**にiOSおよびAndroidアプリケーションに追加される予定です。これにより、M2やWisePad 3を含むすべてのStripeターミナル認証リーダーのサポートが可能になります。
:::

### 一般的な問題 {#common-issues}

<Accordion>
  <AccordionItem question="リーダーが接続できない">

  - サポートされているインターネット接続ターミナル（S700またはWisePOS E）を使用していることを確認してください
  - ターミナルがWiFi/Ethernetに接続されておりオンラインであることを確認してください
  - あなたのStripeアカウントにターミナルが有効になっていることを確認してください
  - ターミナルがStripeダッシュボードに登録されていることを確認してください

  </AccordionItem>

  <AccordionItem question="支払いが拒否された">

  - あなたのStripeアカウントがアクティブで良好な状態であることを確認してください
  - 使用しているカードが有効であることを確認してください
  - 十分な資金が利用可能であることを確認してください

  </AccordionItem>

  <AccordionItem question="SSL証明書のエラー">

  - StripeターミナルはHTTPSが必要です - あなたのサイトに有効なSSL証明書があることを確認してください
  - SSL証明書が正しく構成されていることを確認してください

  </AccordionItem>
</Accordion>
