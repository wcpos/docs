# ストライプターミナルゲートウェイ

ストライプターミナルゲートウェイを使用すると、WooCommerce POS内でストライプターミナルハードウェアリーダーを使用して対面での支払いを受け付けることができます。このゲートウェイは、物理カードリーダーとテスト用のシミュレーターモードの両方をサポートしています。

## 機能

- **ハードウェア統合**: インターネット接続を介して物理ストライプターミナルリーダーを接続
- **シミュレーターモード**: ストライプのシミュレーターを使用してハードウェアなしで支払いをテスト
- **複数の支払い方法**: さまざまなカードタイプや支払い方法をサポート
- **リアルタイム処理**: 即時の支払い処理と確認
- **安全なトランザクション**: ストライプを通じたPCI準拠の支払い処理
- **ウェブベースのSDK**: ウェブアプリケーションとの互換性のためにストライプのJavaScript SDKを使用

## インストール

### ステップ 1: ダウンロードとインストール

1. [Stripe Terminal for WooCommerce のリリースページ](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)にアクセス
2. 最新の **stripe-terminal-for-woocommerce.zip** ファイルをダウンロード
3. WordPress管理画面で `プラグイン > 新規追加 > プラグインをアップロード` に移動
4. zipファイルをアップロードし、プラグインを有効化

### ステップ 2: ストライプ設定の構成

1. `WP管理 > WooCommerce > 設定 > 支払い` に移動
2. 支払い方法リストで **Stripe Terminal** を見つける
3. **Stripe Terminal** をクリックして設定にアクセス
4. **Stripe Secret Key** を入力（これはストライプダッシュボードから取得できます）
5. 設定を保存

:::note
WooCommerce 設定でストライプターミナルゲートウェイを有効にする必要はありません。次のステップで特にPOSのために有効になります。
:::

### ステップ 3: WooCommerce POSで有効化

1. `WP管理 > WooCommerce POS > 設定 > チェックアウト` に移動
2. リスト内で **Stripe Terminal** ゲートウェイを見つける
3. POSで使用するためにゲートウェイを有効化
4. 設定を保存

## 使用方法

### リーダーの接続

POSのチェックアウトでストライプターミナルゲートウェイを選択すると:

1. **接続方法を選択**: 物理リーダーを接続するかシミュレーターを使用するかを選択できます
2. **物理リーダー**: ストライプターミナルデバイスを接続するための画面の指示に従う
3. **シミュレーター**: ハードウェアなしでさまざまな支払いシナリオをテストするためにシミュレーターモードを選択

### 支払いの処理

1. **アイテムを追加**: POSに商品をカートに追加
2. **ゲートウェイを選択**: 支払い方法として「Stripe Terminal」を選択
3. **リーダーを接続**: リーダーを接続するかシミュレーターモードを選択
4. **支払いを処理**: トランザクションを完了するための指示に従う
5. **確認**: 支払いが成功すると、注文が自動的に完了します

### シミュレーターでのテスト

シミュレーターを使用すると、さまざまな支払い方法やシナリオをテストできます:

- **カード支払い**: 異なるカードタイプ（Visa、Mastercardなど）をテスト
- **非接触型支払い**: タップして支払うトランザクションをシミュレート
- **エラーシナリオ**: 支払い拒否やその他のエラー条件をテスト
- **さまざまな金額**: 様々なトランザクション金額をテスト

## 要件

- **ストライプアカウント**: 有効なストライプアカウントでターミナルが有効化されていること
- **APIキー**: ダッシュボードからのストライプシークレットキー
- **WooCommerce POS**: 無料またはプロバージョンがインストールされ、有効化されていること
- **HTTPS**: SSL/HTTPSを使用してセキュリティが確保されていること

## ハードウェア互換性

:::important 接続要件
この実装はストライプのJavaScript SDKを使用しており、これはウェブアプリケーションを介して機能しますが、**インターネット接続されたターミナルのみ**が必要です。Bluetoothターミナルは現在サポートされていません。
:::

### サポートされているターミナル（インターネット接続）

- **ストライプリーダー S700** ✅ - イーサネット/WiFi接続されたターミナル
- **WisePOS E** ✅ - WiFi接続されたターミナル

### サポートされていないターミナル（Bluetooth）

- **BBPOS Chipper 2X BT** ❌ - Bluetoothのみ
- **BBPOS WisePad 3** ❌ - Bluetoothのみ  
- **Verifone P400** ❌ - Bluetoothのみ

:::note 将来のサポート
Bluetoothターミナルのサポートは **2026年第1四半期** にiOSおよびAndroidアプリケーションに追加される予定です。これにより、M2およびWisePad 3を含むすべてのストライプターミナル認定リーダーがサポートされるようになります。
:::

## トラブルシューティング

### 一般的な問題

**リーダーが接続できない**
- サポートされているインターネット接続ターミナル（S700またはWisePOS E）を使用していることを確認
- ターミナルがWiFi/Ethernetに接続されており、オンラインであることを確認
- ストライプアカウントにターミナルが有効化されていることを確認
- ターミナルがストライプダッシュボードで登録されていることを確認

**支払いが拒否された**
- ストライプアカウントがアクティブであること、また良好な状態にあることを確認
- 使用されているカードが有効であることを確認
- 十分な資金が利用可能であることを確認

**SSL証明書エラー**
- ストライプターミナルはHTTPSを必要とします - サイトに有効なSSL証明書があることを確認
- SSL証明書が正しく設定されていることを確認

### ヘルプの取得

技術サポートについては:
- [GitHubリポジトリ](https://github.com/wcpos/stripe-terminal-for-woocommerce)にアクセスして問題を報告
- ハードウェア固有のガイダンスについては[ストライプターミナルのドキュメント](https://stripe.com/docs/terminal)を確認
- アカウント関連の問題についてはストライプサポートに連絡
