# ゲートウェイテンプレート

WooCommerceゲートウェイテンプレートは、WooCommerce POS用のカスタム支払いゲートウェイを作成するための出発点を提供します。このテンプレートには、完全に機能する支払いゲートウェイを構築するために必要なすべてのボイラープレートコードと構造が含まれています。

## 特徴

- **完全なテンプレート**: 必要なすべてのメソッドを備えた、すぐに使用可能なゲートウェイ構造
- **POS連携**: WooCommerce POS用に事前設定済み
- **自動セットアップ**: スクリプトベースのテンプレートカスタマイズ
- **ベストプラクティス**: WordPressとWooCommerceのコーディング標準に準拠
- **拡張性**: 特定の支払いプロバイダ向けに簡単に変更・拡張可能

## はじめに

### オプション1: 自動テンプレート生成

テンプレートには、特定のゲートウェイ用にテンプレートを自動的にカスタマイズするスクリプトが含まれています：

1. **リポジトリのクローン**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **セットアップスクリプトの実行**:
   ```bash
   ./create-gateway.sh
   ```

3. **プロンプトに従う**:
   - ゲートウェイ名を入力（例：「マイ決済ゲートウェイ」）
   - ゲートウェイスラッグを入力（例：「my-payment」）
   - 説明を入力
   - スクリプトがカスタマイズされたプラグインを生成

### オプション2: 手動テンプレート使用

手動カスタマイズを希望する場合：

1. **テンプレートのダウンロード**:
   - [ゲートウェイテンプレートリポジトリ](https://github.com/wcpos/woocommerce-gateway-template)にアクセス
   - 最新のリリースをダウンロードまたはリポジトリをクローン

2. **テンプレートのカスタマイズ**:
   - `{{GATEWAY_NAME}}` をゲートウェイの表示名に置き換え
   - `{{GATEWAY_SLUG}}` をゲートウェイの一意の識別子に置き換え
   - `{{GATEWAY_DESCRIPTION}}` をゲートウェイの説明に置き換え

3. **ファイル名の変更**:
   - `wcpos-{{GATEWAY_SLUG}}.php` をゲートウェイのスラッグに合わせて名前変更
   - ファイルヘッダーとプラグイン情報を更新

### メインプラグインファイル

メインプラグインファイル（`wcpos-{{GATEWAY_SLUG}}.php`）には以下が含まれます：

- **プラグインヘッダー**: WordPressプラグイン情報
- **ゲートウェイクラス**: メインの支払いゲートウェイクラス
- **初期化**: プラグインのセットアップとフック
- **連携**: WooCommerce POSとの互換性
