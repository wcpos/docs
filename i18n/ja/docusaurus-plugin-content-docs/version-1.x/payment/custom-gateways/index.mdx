# カスタム決済ゲートウェイ

WCPOSは、販売時点情報システムの機能を拡張するいくつかのカスタム決済ゲートウェイをサポートしています。これらのゲートウェイは、POS環境専用に設計されており、さまざまな支払い方法やサービスとシームレスに統合されます。

## 利用可能なカスタムゲートウェイ

### ハードウェアターミナルゲートウェイ

- **[Stripeターミナル](./stripe-terminal)** - Stripeターミナルハードウェアリーダーを使用して支払いを受け付けます
- **[SumUpターミナル](./sumup-terminal)** - SumUpカードリーダーを通じて支払いを処理します

### デジタル決済ゲートウェイ

- **[メールインボイス](./email-invoice)** - 顧客にメールで支払い請求書を送信します
- **[ウェブチェックアウト](./web-checkout)** - 顧客をリダイレクトしてオンラインで支払いを完了させます

## 自分のゲートウェイを作成する

特定の要件に必要なカスタム決済ゲートウェイが必要な場合は、私たちの**[ゲートウェイテンプレート](./gateway-template)**を使用して独自のWooCommerce POS決済ゲートウェイを作成できます。

## インストールの概要

すべてのカスタムゲートウェイは、同様のインストールプロセスに従います：

1. **最新リリース**をそれぞれのGitHubリポジトリからダウンロードします
2. **プラグインをインストール**します `WP Admin > プラグイン > 新規追加 > プラグインをアップロード`
3. **ゲートウェイ設定**を構成します `WP Admin > WooCommerce > 設定 > 支払い`
4. **ゲートウェイを有効にします** `WP Admin > WooCommerce POS > 設定 > チェックアウト`

## 要件

- WooCommerce POSプラグイン（無料版またはプロ版）
- WooCommerceがインストールされたWordPress
- 特定の決済サービス用の適切なAPIキーまたは資格情報

## サポート

カスタムゲートウェイに関する問題については、それぞれのGitHubリポジトリを訪問し、問題の詳細情報を記載した問題を作成してください。
