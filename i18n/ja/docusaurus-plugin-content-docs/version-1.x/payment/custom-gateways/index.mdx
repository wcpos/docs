# カスタム支払いゲートウェイ

WCPOSは、ポイント・オブ・セールシステムの機能を拡張するいくつかのカスタム支払いゲートウェイをサポートしています。これらのゲートウェイは、POS環境専用に設計されており、さまざまな支払い方法やサービスとのシームレスな統合を提供します。

## 利用可能なカスタムゲートウェイ

### ハードウェアターミナルゲートウェイ

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminalハードウェアリーダーを使用して支払いを受け取ります
- **[SumUp Terminal](./sumup-terminal)** - SumUpカードリーダーを通じて支払いを処理します

### デジタル支払いゲートウェイ

- **[Email Invoice](./email-invoice)** - 顧客に請求書を電子メールで送信します
- **[Web Checkout](./web-checkout)** - 顧客をオンラインで支払いを完了するためにリダイレクトします

## 独自のゲートウェイの作成

特定の要件に必要なカスタム支払いゲートウェイがある場合は、私たちの**[ゲートウェイテンプレート](./gateway-template)**を使用して独自のWooCommerce POS支払いゲートウェイを作成できます。

## インストールの概要

すべてのカスタムゲートウェイは、類似のインストールプロセスに従います。

1. **最新のリリースを**それぞれのGitHubリポジトリからダウンロードします
2. **プラグインを** `WP Admin > プラグイン > 新規追加 > プラグインのアップロード`からインストールします
3. **ゲートウェイの設定を** `WP Admin > WooCommerce > 設定 > 支払い`で構成します
4. **ゲートウェイを** `WP Admin > WooCommerce POS > 設定 > チェックアウト`で有効にします
