# カスタム決済ゲートウェイ

WCPOSは、ポイントオブセール（POS）システムの機能を拡張する複数のカスタム決済ゲートウェイをサポートしています。これらのゲートウェイは、POSの環境に特化して設計され、さまざまな決済方法とサービスとのシームレスな統合を提供します。

## 利用可能なカスタムゲートウェイ

### ハードウェアターミナルゲートウェイ

- **[Stripeターミナル](./stripe-terminal)** - Stripeターミナルハードウェアリーダーを使用した決済の受け入れ
- **[SumUpターミナル](./sumup-terminal)** - SumUpカードリーダーを通じた決済処理

### デジタル決済ゲートウェイ

- **[メール請求書](./email-invoice)** - メールで顧客に決済請求書を送信
- **[ウェブチェックアウト](./web-checkout)** - オンラインで決済を完了するために顧客をリダイレクト

## 独自のゲートウェイ作成

特定の要件に合わせたカスタム決済ゲートウェイが必要な場合は、**[ゲートウェイテンプレート](./gateway-template)**を使用して、独自のWooCommerce POS決済ゲートウェイを作成できます。

## インストール概要

すべてのカスタムゲートウェイは、同様のインストールプロセスに従います：

1. **ダウンロード** 各GitHubリポジトリから最新のリリースを取得
2. **インストール** `WP管理画面 > プラグイン > 新規追加 > プラグインをアップロード`からプラグインをインストール
3. **設定** `WP管理画面 > WooCommerce > 設定 > 決済方法`でゲートウェイの設定を構成
4. **有効化** `WP管理画面 > WooCommerce POS > 設定 > チェックアウト`でゲートウェイを有効にする

## 要件

- WooCommerce POSプラグイン（無料版または有料版）
- WooCommerceがインストールされたWordPress
- 特定の決済サービスに適したAPIキーまたは認証情報

## サポート

カスタムゲートウェイに関する問題については、各GitHubリポジトリにアクセスし、問題の詳細情報を記載してイシューを作成してください。
