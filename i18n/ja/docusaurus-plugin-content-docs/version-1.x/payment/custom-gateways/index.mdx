# カスタム支払いゲートウェイ {#custom-payment-gateways}

WCPOSは、販売時点情報管理（POS）システムの機能を拡張するいくつかのカスタム支払いゲートウェイをサポートしています。これらのゲートウェイは、POS環境専用に設計されており、さまざまな支払い方法やサービスとのシームレスな統合を提供します。

## 利用可能なカスタムゲートウェイ {#available-custom-gateways}

### ハードウェア端末ゲートウェイ {#hardware-terminal-gateways}

- **[Stripeターミナル](./stripe-terminal)** - Stripeターミナルのハードウェアリーダーを使用して支払いを受け付ける
- **[SumUpターミナル](./sumup-terminal)** - SumUpカードリーダーを通じて支払いを処理する

### モバイル支払いゲートウェイ {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QRコードまたはプッシュ通知を介して電話ベースの支払いを受け付ける

### デジタル支払いゲートウェイ

- **[メール請求書](./email-invoice)** - 顧客にメールで支払い請求書を送信する
- **[ウェブチェックアウト](./web-checkout)** - 顧客をリダイレクトしてオンラインで支払いを完了させる

## 独自のゲートウェイを作成する {#creating-your-own-gateway}

特定の要件に対するカスタム支払いゲートウェイが必要な場合は、**[ゲートウェイテンプレート](./gateway-template)**を使用して独自のWooCommerce POS支払いゲートウェイを作成できます。

## インストール概要 {#installation-overview}

すべてのカスタムゲートウェイは、同様のインストールプロセスに従います：

1. **最新リリースを**それぞれのGitHubリポジトリからダウンロードします
2. **プラグインをインストール**します `WP Admin > プラグイン > 新規追加 > プラグインをアップロード`
3. **ゲートウェイ設定を構成**します `WP Admin > WooCommerce > 設定 > 支払い`
4. **ゲートウェイを有効に**します `WP Admin > WooCommerce POS > 設定 > チェックアウト`
