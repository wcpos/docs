# カスタム決済ゲートウェイ {#custom-payment-gateways}

WCPOSは、POSシステムの機能を拡張する複数のカスタム決済ゲートウェイをサポートしています。これらのゲートウェイはPOS環境向けに設計されており、さまざまな決済方法やサービスとシームレスに統合できます。

## 利用可能なカスタムゲートウェイ {#available-custom-gateways}

### ハードウェア端末ゲートウェイ {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminalハードウェアリーダーを使用して決済を受け付けます
- **[SumUp Terminal](./sumup-terminal)** - SumUpカードリーダーを通じて決済を処理します

### モバイル決済ゲートウェイ {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QRコードまたはプッシュ通知を通じてスマートフォンでの決済を受け付けます

### デジタル決済ゲートウェイ {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - 顧客にメールで請求書を送信します
- **[Webチェックアウト](./web-checkout)** - 顧客をリダイレクトしてオンラインで支払いを完了します

## 独自のゲートウェイを作成する {#creating-your-own-gateway}

特定の要件に合わせたカスタム決済ゲートウェイが必要な場合は、**[ゲートウェイテンプレート](./gateway-template)** を使用して独自の WCPOS 決済ゲートウェイを作成できます。

## インストール概要 {#installation-overview}

すべてのカスタムゲートウェイは、同様のインストール手順に従います：

1. 該当する GitHub リポジトリから最新リリースを **ダウンロード** します
2. `WP Admin > Plugins > Add New > Upload Plugin` からプラグインを **インストール** します
3. `WP Admin > WooCommerce > Settings > Payments` でゲートウェイ設定を **構成** します
4. `WP Admin > POS > Settings > Checkout` でゲートウェイを **有効化** します
