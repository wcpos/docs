WCPOSは異なるビジネスニーズに対応するため、複数の支払い方法をサポートしています。

## デフォルトの支払い方法

無料版には2つの組み込み決済ゲートウェイが含まれています：

### 現金

- 受け取った金額を入力
- 自動的におつりを計算
- 追加の設定は不要

### カード

- 外部カード端末で使用
- 端末で処理後に支払いを完了としてマーク
- 直接の統合は不要

## その他の決済ゲートウェイ（Pro）

:::info Proの機能
追加の決済ゲートウェイには[WCPOS Pro](/getting-started/pro-license)が必要です。
:::

Proを使用すると、POSチェックアウトで追加のWooCommerce決済ゲートウェイを有効化できます：

- **Stripe Terminal** - Stripeカードリーダーとの直接統合
- **SumUp Terminal** - SumUpカードリーダーとの統合
- **任意のWooCommerceゲートウェイ** - ストアにインストールされているゲートウェイを有効化

## カスタムゲートウェイ

[カスタムゲートウェイ](/payment/custom-gateways)システムを使用して、独自の支払い連携を作成できます。
