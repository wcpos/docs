# カスタム支払いゲートウェイ {#custom-payment-gateways}

WCPOSは、販売時点管理システムの機能を拡張するいくつかのカスタム支払いゲートウェイをサポートしています。これらのゲートウェイは、POS環境専用に設計されており、さまざまな支払い方法やサービスとのシームレスな統合を提供します。

## 利用可能なカスタムゲートウェイ {#available-custom-gateways}

### ハードウェア端末ゲートウェイ {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminalハードウェアリーダーを使用して支払いを受け付けます
- **[SumUp Terminal](./sumup-terminal)** - SumUpカードリーダーを介して支払いを処理します

### モバイル支払いゲートウェイ {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - QRコードまたはプッシュ通知を通じて電話ベースの支払いを受け付けます

### デジタル支払いゲートウェイ {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - 顧客に支払い請求書をメールで送信します
- **[Web Checkout](./web-checkout)** - 顧客をオンラインで支払いを完了するためにリダイレクトします

## 自分のゲートウェイを作成する {#creating-your-own-gateway}

特定の要件に対してカスタム支払いゲートウェイが必要な場合は、当社の**[Gateway Template](./gateway-template)**を使用して独自のWCPOS支払いゲートウェイを作成できます。

## インストール概要 {#installation-overview}

すべてのカスタムゲートウェイは、似たようなインストールプロセスに従います：

1. **最新リリースを**それぞれのGitHubリポジトリからダウンロードします
2. **プラグインを** `WP Admin > プラグイン > 新規追加 > プラグインのアップロード`を通じてインストールします
3. **ゲートウェイの設定を** `WP Admin > WooCommerce > 設定 > 支払い`で構成します
4. **ゲートウェイを** `WP Admin > POS > 設定 > チェックアウト`で有効化します
