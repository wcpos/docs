WCPOSでスムーズなチェックアウト体験を提供するには、パフォーマンスの最適化が不可欠です。本セクションでは、よくあるパフォーマンスの問題とその解決方法について説明します。

## よくあるパフォーマンスの問題 {#common-performance-issues}

特によく報告されるパフォーマンスの問題は以下のとおりです。

- **チェックアウトが遅い** - 支払いモーダルの読み込みや支払い処理に時間がかかる
- **インターフェースの動作が重い** - POSインターフェースの反応が鈍い
- **同期に時間がかかる** - 商品や注文の同期に時間がかかりすぎる

## パフォーマンスのカテゴリ {#performance-categories}

### [チェックアウトのパフォーマンス](/support/performance/checkout) {#checkout-performancesupportperformancecheckout}
チェックアウトのプロセスを最適化し、支払い処理の高速化と顧客体験の向上を実現します。

### [サーバーパフォーマンス](/support/performance/server) {#server-performancesupportperformanceserver}
組み込みのメトリクスとトラブルシューティング手法を使用して、WordPress/WooCommerce サーバーを監視し、最適化します。

### ネットワークパフォーマンス（近日公開） {#network-performance-coming-soon}
ネットワーク関連のパフォーマンス問題をトラブルシューティングし、API 通信を最適化します。
