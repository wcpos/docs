パフォーマンスの最適化は、WCPOSでスムーズなチェックアウト体験を提供するために重要です。このセクションでは、一般的なパフォーマンスの問題とその解決方法を説明します。

## 一般的なパフォーマンスの問題 {#common-performance-issues}

最も一般的なパフォーマンスに関する苦情は次のとおりです：

- **チェックアウトプロセスの遅さ** - 支払いモーダルの読み込みや支払い処理に時間がかかります
- **鈍いインターフェース** - POS インターフェースの反応が遅いと感じます
- **長い同期時間** - 商品と注文の同期に時間がかかりすぎます

## パフォーマンスのカテゴリー {#performance-categories}

### [チェックアウトパフォーマンス](/support/performance/checkout) {#checkout-performancesupportperformancecheckout}
より早い支払い処理と優れた顧客体験のためにチェックアウトプロセスを最適化します。

### [サーバーパフォーマンス](/support/performance/server) {#server-performancesupportperformanceserver}
組み込みのメトリクスとトラブルシューティング技術を使用して、WordPress/WooCommerceサーバーを監視し最適化します。

### ネットワークパフォーマンス（近日公開） {#network-performance-coming-soon}
ネットワーク関連のパフォーマンスの問題をトラブルシューティングし、API通信を最適化します。
