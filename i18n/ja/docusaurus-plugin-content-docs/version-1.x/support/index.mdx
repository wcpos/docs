WCPOSに関してヘルプが必要ですか？サポートを受ける方法と質問への回答を見つける方法は以下の通りです。

## アプリ内サポート

POSには、左のナビゲーションドロワーからアクセスできるサポート画面が組み込まれています。この画面では、WCPOSのDiscordコミュニティを直接アプリ内で利用できるため、以下が可能です：

- 既存のディスカッションを閲覧する
- よくある質問の回答を検索する
- #generalチャンネルで質問をする
- アナウンスを常に把握する

{/* スクリーンショット: support-screen.png */}

## Discordコミュニティ

WCPOSの主要なサポートチャンネルは私たちの[Discordサーバー](https://wcpos.com/discord)です。

### チャンネル

| チャンネル | 目的 |
|---------|---------|
| **#welcome-and-rules** | コミュニティガイドライン |
| **#announcements** | 更新情報やニュース |
| **#general** | メインサポートチャンネル - ここで質問をする |
| **#forum** | 長文の議論や機能のリクエスト |
| **#github-activity** | 開発の更新情報 |

### 質問する前に

最良のヘルプを得るために：

1. **まずはテスト** - プラグインをダウンロードして数日間テストし、自分のニーズに合っているか確認してください
2. **ドキュメントを確認** - このドキュメントで多くの質問が回答されています
3. **Discordを検索** - あなたの質問はすでに回答されているかもしれません
4. **詳細を提供** - WooCommerceのバージョン、WCPOSのバージョン、および問題を再現する手順を含めてください

- [設定](/settings) - 設定オプション
- [トラブルシューティング](/support/troubleshooting/critical-error) - よくある問題と解決策

## メールサポート

ライセンス関連の問題やプライベートな問い合わせについては、次のメールアドレスに連絡できます：**support@wcpos.com**

:::note
一般的なサポートの質問はDiscordに投稿してください。コミュニティが助けてくれるほか、他の人もその回答から利益を得ることができます。
:::

## ログ画面

POSには、システムのアクティビティを記録する[ログ画面](/support/logs)が含まれています。これは以下にとって非常に重要です：

- シンクの問題をデバッグする
- エラーメッセージを理解する
- サポートをリクエストする際に詳細情報を提供する

助けを求める際は、ログ画面をチェックしてエラーメッセージを確認し、サポートリクエストに含めてください。

## プロサポート

[WCPOS Pro](https://wcpos.com/pro)のお客様は、Discordを通じて同じコミュニティサポートを受けています。Proライセンスは、無料およびPro機能の継続的な開発を資金提供します。
