WCPOSについての助けが必要ですか？サポートを受ける方法と質問に対する答えを見つける方法は以下の通りです。

## アプリ内サポート {#in-app-support}

POSには、左側のナビゲーションドロワーからアクセスできる組み込みのサポート画面があります。この画面はアプリ内にWCPOS Discordコミュニティを埋め込んでおり、以下のことができます：

- 既存のディスカッションを閲覧する
- よくある質問の答えを検索する
- #generalチャンネルで質問する
- お知らせを受け取る

{/* スクリーンショット: support-screen.png */}

## Discordコミュニティ {#discord-community}

WCPOSの主なサポートチャンネルは、私たちの[Discordサーバー](https://wcpos.com/discord)です。

### チャンネル {#channels}

| チャンネル | 目的 |
|---------|---------|
| **#welcome-and-rules** | コミュニティガイドライン |
| **#announcements** | 更新情報やニュース |
| **#general** | 主なサポートチャンネル - ここで質問してください |
| **#forum** | より長いディスカッションや機能リクエスト |
| **#github-activity** | 開発の最新情報 |

### 質問する前に {#before-asking}

最良の助けを得るために：

1. **まずテスト** - プラグインをダウンロードし、数日間テストしてニーズに合っているか確認してください
2. **ドキュメントを確認** - 多くの質問はこのドキュメントで回答されています
3. **Discordを検索** - あなたの質問はすでに回答されているかもしれません
4. **詳細を提供** - WooCommerceのバージョン、WCPOSのバージョン、問題を再現する手順を含めてください

## メールサポート {#email-support}

ライセンス関連の問題やプライベートな問い合わせについては、以下のメールに連絡してください：**support@wcpos.com**

:::note
一般的なサポートの質問は、Discordに投稿してください。コミュニティが助けを提供し、他の人がその回答から利益を得ることができます。
:::

## ログ画面 {#logs-screen}

POSには[ログ画面](/support/logs)が含まれており、システムの活動を記録します。これは以下に非常に価値があります：

- 同期問題のデバッグ
- エラーメッセージの理解
- サポートをリクエストする際に詳細情報を提供

助けを求める際には、ログ画面にエラーメッセージがあるか確認し、それをサポートリクエストに含めてください。

## プロサポート {#pro-support}

[WCPOS Pro](https://wcpos.com/pro)の顧客は、Discordを通じて同じコミュニティサポートを受けます。Proライセンスは無料機能とPro機能の継続的な開発を資金提供します。

- [GitHub](https://github.com/wcpos)でバグを報告する
- Discordで機能を提案する
- ドキュメントを翻訳する
