WCPOSでお困りですか？サポートを受けて質問に答えるための方法をご紹介します。

## アプリ内サポート

POSには、左側のナビゲーションドロワーからアクセスできる組み込みのサポート画面があります。この画面には、WCPOSのDiscordコミュニティが直接埋め込まれており、以下のことができます：

- 既存のディスカッションを閲覧
- よくある質問の答えを検索
- #generalチャンネルで質問する
- お知らせを確認する

{/* Screenshot: support-screen.png */}

## Discordコミュニティ

WCPOSの主要なサポートチャンネルは、[Discordサーバー](https://wcpos.com/discord)です。

### チャンネル

| チャンネル | 目的 |
|---------|---------|
| **#welcome-and-rules** | コミュニティガイドライン |
| **#announcements** | 更新とニュース |
| **#general** | メインサポートチャンネル - ここで質問してください |
| **#forum** | 長めの議論と機能リクエスト |
| **#github-activity** | 開発の最新情報 |

### 質問する前に

最適なヘルプを受けるために：

1. **まず試す** - プラグインをダウンロードし、数日間テストして、ニーズに合うかを理解する
2. **ドキュメントを確認** - 多くの質問はこのドキュメントで既に答えられています
3. **Discordで検索** - あなたの質問は既に回答されている可能性があります
4. **詳細を提供** - WooCommerceのバージョン、WCPOSのバージョン、問題を再現する手順を含めてください

## メールサポート

ライセンス関連の問題や個人的な問い合わせについては、以下のメールアドレスまで：**support@wcpos.com**

:::note
一般的なサポートの質問は、コミュニティが助けられ、他の人も回答から利益を得られるように、Discordに投稿してください。
:::

## ログ画面

POSには、システムアクティビティを記録する[ログ画面](/support/logs)が含まれています。これは以下に役立ちます：

- 同期の問題のデバッグ
- エラーメッセージの理解
- サポートリクエスト時の詳細な情報提供

サポートを求める際は、ログ画面で任意のエラーメッセージを確認し、サポートリクエストに含めてください。

## プロサポート

[WCPOS Pro](https://wcpos.com/pro)のお客様は、Discordを通じて同じコミュニティサポートを受けられます。Proライセンスは、無料版とPro版の両方の機能の継続的な開発に充てられます。
