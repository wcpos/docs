POS ヘッダーの**通知ベル**には、WCPOS からのメッセージが表示されます。製品のお知らせ、プラグイン更新の通知、Pro ライセンスのリマインダーなどです。ベル上の赤いバッジには未読通知の件数が表示されます（99 件を超える場合は **99+** と表示されます）。

## 通知センターを開く {#opening-the-notification-centre}

POS 上部の**ベルアイコン**をクリックすると、通知パネルが開きます。通知は新しい順に表示され、それぞれに次の情報が含まれます。

- **未読ドット** — 未読の間は塗りつぶされ、タイトルは太字で表示されます
- 短い**タイトル**と任意の**本文**
- **相対時刻**（例: _2 時間前_）

待機中の通知がない場合、パネルには **"通知はありません"** の空の状態が表示されます。

## 通知を読んで消去する {#reading-and-clearing-notifications}

- **1 件を既読にする** — リスト内の未読通知をタップします。
- **すべて既読にする** — パネルのヘッダーにある **すべて既読にする** ボタンを使用します（未読項目がある場合にのみ表示されます）。
- **バッジを消去する** — 各項目をまだ開いていなくても、パネルを開くだけでベルの未読件数バッジが消えます。

通知はデバイスに保存されるため、一覧はオフラインでも利用できます。接続中は新しい通知がリアルタイムで届きます。

## 表示される通知の種類 {#what-kinds-of-notifications-appear}

通知センターは、店舗の注文ではなく、システムメッセージとアカウントメッセージに使用されます。表示される内容は次のとおりです。

| 種類 | 例 |
| --- | --- |
| **ようこそ** | 初回接続時に表示される一度限りの歓迎メッセージ |
| **お知らせ** | WCPOS に関する一般的なニュースと更新情報 |
| **プラグイン更新** | WCPOS プラグインの更新が利用可能であることを知らせる通知 |
| **ライセンス期限間近** | Pro ライセンスの更新日が近づいていることを知らせるリマインダー。**更新** リンク付き |
| **ライセンス期限切れ** | Pro ライセンスの有効期限が切れていることを知らせる継続表示のアラート。**今すぐ更新** リンク付き |

重要なメッセージ（ライセンスアラートなど）は、ベルに加えて画面上のトーストとして短時間表示される場合もあります。

:::note
通知センターはWCPOSからのメッセージを配信します。**新しいオンライン注文**のリアルタイム通知は、別途予定されている機能です — [ロードマップ](https://github.com/orgs/wcpos/projects/4)を参照してください。
:::
