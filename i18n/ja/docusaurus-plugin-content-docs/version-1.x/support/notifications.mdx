POS ヘッダーの **通知ベル** は、WCPOS が製品のお知らせ、プラグインの更新通知、Pro ライセンスのリマインダーなどのメッセージを表示する場所です。ベルの赤いバッジは、未読の通知の数を示します (99 を超える数は **99+** と表示されます)。

## 通知センターを開く {#opening-the-notification-centre}

POS の上部にある **ベル アイコン** をクリックして、通知パネルを開きます。通知が新しいものから順にリストされ、それぞれ次の内容が含まれます。

- **未読ドット** — 未読の間は塗りつぶされ、タイトルは太字で表示されます
- 短い **タイトル** とオプションの **本文**
- **相対タイムスタンプ** (例: _2 時間前_)

何も待っていない場合、パネルには **「通知なし」** という空の状態が表示されます。

## 通知の読み取りとクリア {#reading-and-clearing-notifications}

- **1 つを既読としてマークします** — リスト内の未読の通知をタップします。
- **すべて既読としてマーク** — パネル ヘッダーの **すべて既読としてマーク** ボタンを使用します (未読項目がある場合にのみ表示されます)。
- **バッジのクリア** — 各項目を開いていない場合でも、パネルを開くだけでベルの未読数バッジがクリアされます。

通知はデバイスに保存されるため、リストはオフラインでも利用できます。接続中に新しい通知がリアルタイムで届きます。

## どのような通知が表示されるか {#what-kinds-of-notifications-appear}

通知センターは、店舗での注文ではなく、システムおよびアカウントのメッセージに使用されます。次のものが表示される場合があります。

| タイプ | 例 |
| --- | --- |
| **ようこそ** | 初めて接続したときの 1 回限りのウェルカム メッセージ |
| **お知らせ** | WCPOS に関する一般的なニュースと最新情報 |
| **プラグインのアップデート** | WCPOS プラグインのアップデートが利用可能になったというお知らせ |
| **ライセンスの有効期限が切れています** | Pro ライセンスの更新日が近づいていることを通知するリマインダー (**更新**) リンク |
| **ライセンスの有効期限が切れています** | Pro ライセンスの有効期限が切れたことを示す永続的なアラート (**今すぐ更新** リンク付き) |

重要なメッセージ (ライセンス アラートなど) は、ベルに加えて画面上のトーストとして短時間表示される場合もあります。

:::note
通知センターは WCPOS からのメッセージを配信します。 **新しいオンライン注文**に関するリアルタイム アラートは、別の計画された機能です。[ロードマップ](https://github.com/orgs/wcpos/projects/4)を参照してください。
:::
