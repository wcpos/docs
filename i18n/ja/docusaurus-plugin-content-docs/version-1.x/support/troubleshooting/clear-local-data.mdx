WCPOSは、製品、注文、顧客のローカルコピーを保存し、POSが迅速に動作し、オフラインで機能できるようにします。時には、このローカルデータがWooCommerceストアと同期しなくなることがあります。ローカルデータをすべてクリアすると、ローカルデータベースが削除され、サーバーからの新しい同期がトリガーされます。

:::warning
これは核オプションです。ローカルデータをクリアすると、デバイスに保存されているすべてが削除され、完全再同期が強制されます。店舗のサイズによっては、これに時間がかかることがあります。他のオプション（通常の [同期](/products/sync) など）で問題が解決されない場合にのみ使用してください。
:::

## 使用するタイミング {#when-to-use-it}

ローカルデータをクリアする必要がある場合：

- **同期の問題** — 製品、注文、または顧客が正しく更新されない
- **プラグインの更新後** — 新しいデータ構造との互換性を確保するため
- **古いデータ** — 価格、在庫レベル、または他の情報が古く見える

## 何がクリアされるか {#what-gets-cleared}

| クリアされる項目 | クリアされない項目 |
|----------------|-----------------|
| ローカル製品データベース | ログインセッション |
| ローカル注文データベース | サーバー側のデータ |
| ローカル顧客データベース | WooCommerce設定 |

ログアウトされることは **ありません**。あなたのWooCommerceストアデータには影響がなく、デバイス上のローカルコピーのみが削除されます。クリア後、すぐに新しい同期が自動的に開始します。

## ローカルデータをクリアする方法 {#how-to-clear-local-data}

### ウェブ {#web}

1. 右上隅の**ユーザーメニュー**をクリックしてメニューを開く
2. **すべてのローカルデータをクリア**をクリック
3. ローカルデータベースが削除され、新しい同期が自動的に開始します

### デスクトップ {#desktop}

1. 上部メニューバーの**WCPOS**アプリケーションメニューを開く
2. **アドバンスト**にマウスをホバーする
3. **アプリデータをクリア**をクリック
4. ローカルデータベースが削除され、新しい同期が自動的に開始します
