このエラーメッセージが表示されている場合、あなたのPOSは致命的なサーバーエラーに遭遇しており、続行できないことを意味します。
良いニュースは、このエラーメッセージがあなたのWooCommerceログに記録されており、私たちのサポートチームが問題のトラブルシューティングを手伝うことができることです。

## エラーを見つける手順

1. WordPress管理エリアにログインします。
2. WordPressサイドバーメニューで「WooCommerce」>「ステータス」>「ログ」に移動します。
3. 「fatal_error」ログセクションを見つけます。
4. エラーが発生した日のログエントリ（通常は今日の日付）をクリックして、エラーの詳細を表示します。
5. このセクションにログされたエラーの原因を特定します。
6. 関連するエラーメッセージをコピーするか、スクリーンショットを取得して、[Discordチャット](https://wcpos.com/discord)に投稿するか、support@wcpos.comにメールして私たちのサポートチームと共有してください。

## ヘルプを得る

エラーメッセージを私たちのサポートチームと共有してください：

- **Discord**: [wcpos.com/discord](https://wcpos.com/discord)
- **Email**: support@wcpos.com
