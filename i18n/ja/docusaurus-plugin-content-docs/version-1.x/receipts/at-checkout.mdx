支払いの処理が正常に完了すると、レシートモーダルが表示されます。完了した注文の概要が表示され、レシートを印刷またはメール送信するためのオプションが提供されます。

{/* Screenshot: receipt-modal.png */}

## レシートの内容 {#receipt-contents}

レシートには以下が表示されます。

### 店舗情報 {#store-information}
- **店舗名**
- **住所**
- **電話番号**
- **メールアドレス**
- **Webサイト**
- **ロゴ**（設定されている場合）

### レシートヘッダー {#receipt-header}
- **レシートの種類**（例:「税務領収書」）
- **注文番号**
- **日時**
- **レジ担当者名**
- **支払い方法**

### 注文の詳細 {#order-details}
- **明細項目**:
  - 商品名
  - バリエーション属性（該当する場合）
  - 数量
  - 価格
  - 合計

### 合計 {#totals}
- **小計**
- **税**（該当する場合）
- **合計**
- **お預かり金額**（現金の場合）
- **お釣り**（現金の場合）

### 顧客情報 {#customer-information}
- **請求先住所**
- **配送先住所**（異なる場合）

## レシート操作 {#receipt-actions}

### 閉じる {#close}
レシートのモーダルを閉じて POS に戻ります。

### レシートをメール送信 {#email-receipt}
顧客のメールアドレスにレシートを送信します:

1. **レシートをメール送信**をクリックします
2. メールアドレスを確認します
3. レシートは WooCommerce のメールシステム経由で送信されます

顧客には注文に関連付けられたメールアドレスが必要です。

### レシートを印刷 {#print-receipt}
レシートの紙のコピーを印刷します:

1. **レシートを印刷**をクリックします
2. ブラウザーの印刷ダイアログが開きます
3. レシートプリンターを選択して印刷します

:::tip レシートプリンター
WCPOS は、ブラウザーからアクセスできる任意のプリンターで動作します。最適な結果を得るには、次を使用してください。
- 専用のレシートプリンター（サーマルプリンターが一般的です）
- プリンター設定で用紙サイズを設定します
- すばやく印刷できるように印刷プリセットを設定します
:::

## 自動レシートオプション {#automatic-receipt-options}

[カート表示設定](/pos/cart#display-settings)では、次を設定できます。

- **チェックアウト後にレシートを自動表示** - 支払い直後にレシートモーダルを開きます
- **チェックアウト後にレシートを自動印刷** - レシートが表示されたときに自動で印刷を開始します

これらのオプションにより、処理件数の多い会計ワークフローを高速化できます。

## レシートのカスタマイズ {#customising-receipts}

レシートの外観は、WordPress のテンプレートで管理します。

`WP Admin > POS > Templates`

カスタマイズできる項目の詳細については、[レシートテンプレート](/receipts)を参照してください。

- 店舗情報の表示
- ロゴとブランディング
- レイアウトと書式
- 含める情報

## レシートの再印刷 {#reprinting-receipts}

過去の注文のレシートを再印刷するには、次の手順を実行します。

1. [注文画面](/orders)に移動します（Pro 機能）
2. 注文を見つける
3. クリックして注文を開く
4. 領収書オプションを使用して再度印刷する
