WCPOSには、レシート、請求書、ギフトレシート、納品書、キッチンチケットなどに対応する完全なテンプレートシステムがあり、すべて **WP Admin → POS → テンプレート** で管理できます。

:::tip レシートを変更するだけですか？
**[レシートをカスタマイズ](/receipts/customise)** から始めましょう。付属テンプレートを選ぶ、AIに調整を依頼する、手動で編集するという、最も簡単な3つの方法を説明しています。
:::

## テンプレートエンジン {#template-engines}

WCPOSは3つのテンプレートエンジンに対応しています。テンプレートの作成時、またはギャラリーカードの **テンプレートを使用** をクリックしたときにエンジンを選択します。選択後は**変更できません**。別のエンジンを使用する場合は、新しいテンプレートを作成してください。

<FeatureGrid>
  <FeatureCard icon="code" title="ロジックレスHTML（推奨）">
    Mustache形式 <code>{"{{variable}}"}</code> プレーン HTML 内のプレースホルダー。クライアント側でレンダリングされるため、領収書は**オフラインで動作**します。ブラウザー印刷および PDF 領収書に使用します。
  </FeatureCard>
  <FeatureCard icon="print" title="サーマル XML">
    画面プレビューとサーマルレシートプリンター向けの**ESC/POS コマンド**の両方を生成する XML テンプレート。同じテンプレートで、2 種類の出力を作成できます。Epson、Star、その他のサーマルプリンターに使用します。
  </FeatureCard>
  <FeatureCard icon="terminal" title="レガシー PHP">
    WooCommerce 関数を使用するサーバー側 PHP テンプレート。既存の <code>yourtheme/woocommerce-pos/receipt.php</code> オーバーライドとの後方互換性のために保持されています。レンダリングにはサーバー接続が必要です。
  </FeatureCard>
</FeatureGrid>

## テンプレートギャラリー {#template-gallery}

ギャラリーには、そのまま使用することもカスタマイズすることもできる既製テンプレートが用意されています。**カテゴリ**（領収書、請求書、ギフト領収書、キッチンチケット、見積書 / 発注書）、**形式**（HTML / ESC/POS）、**方向**（LTR / RTL）で絞り込み、店舗の実際のデータでプレビューできます。各カードのフッターには**プレビュー**リンクと主要な**テンプレートを使用**ボタンがあります。テンプレートを使用をクリックすると、*マイテンプレート*に編集可能なコピーが作成されます。

### サードパーティ製ドキュメントテンプレート {#third-party-document-templates}

WP Overnight の **PDF Invoices & Packing Slips for WooCommerce** が有効な場合、WCPOS はレシートテンプレートの一覧に **Invoice (WP Overnight)** と **Packing Slip (WP Overnight)** も表示します。これらのサーバーでレンダリングされるHTMLテンプレートは WP Overnight のドキュメントAPIを利用し、そのプラグインで設定した請求書と納品書の番号、レイアウト、ブランド設定、法的情報・税務項目、テンプレートのカスタマイズを再利用します。

これらは WP Overnight プラグインが有効な場合にのみ利用でき、レンダリングにはサーバー接続が必要です。オフライン印刷が必要な場合は、WCPOS に同梱されているHTMLまたはサーマルテンプレートを使用してください。

### 適応型税表示 {#adaptive-tax-display}

同梱テンプレートの大半（Standard、Standard RTL、Minimal/Modern、Narrow、Invoice、Quote、Thermal Simpleの各バリエーション）は、**WooCommerce の税表示設定に自動的に対応します**：

- **税込価格の店舗** (EU/UK/AU) では、総額価格が表示され、「税金込み」の行が表示されます。
- **税抜価格の店舗** (US/CA) では、税抜価格が表示され、税額は別の行に追加されます。
- 合計金額には、実際に請求された総額が常に表示されます。

**詳細** ファミリーは正式な税務請求書であり、店舗設定にかかわらず、税抜きの明細と明示的な内訳によって税額を常に明細化します。

## 仕組み {#how-it-works}

1. **テンプレートを追加** — ギャラリーカードで **テンプレートを使用** をクリックするか、一から作成します。
2. **カスタマイズ** — アプリ内エディターとライブプレビューを使用するか、ChatGPT / Claude に貼り付けて希望する変更を依頼します。
3. **有効化** — *マイテンプレート* で **有効** トグルをオンにします。有効なレシートテンプレートはすべて、レジのレシート画面にあるドロップダウンに表示されます。
4. **印刷または表示** — レシートはチェックアウト後に開き、印刷、メール送信、画面表示を選択できます。テンプレートはレジ以外でも使用できます。オンライン顧客は、同じデザインでレンダリングされた [マイアカウントからレシートをダウンロード](/receipts/storefront) できます。

テンプレートは、店舗情報、明細、合計、税金、支払い、返金などのセクションを含む標準化された [レシートデータペイロード](/receipts/receipt-data) に基づいてレンダリングされます。すべての通貨フィールドには、事前整形済みの `_display` バリアントが含まれます (例: `$29.99` ではなく `29.99`)。

## Pro 機能 {#pro-features}

[WCPOS Pro](/getting-started/pro-license) では、各店舗で独自のテンプレート割り当て、ブランディング（ロゴ、住所、連絡先）、テンプレートの並び順を設定できます。各店舗が専用のレターヘッドで印刷する複数拠点の運用に便利です。
