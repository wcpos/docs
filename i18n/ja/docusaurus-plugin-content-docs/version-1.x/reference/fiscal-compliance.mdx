一部の国では、POS ソフトウェアに脱税に対する**認定**、または国の財政当局との統合を法的に義務付けています。このページには WCPOS の立場がまとめられているため、それを信頼する前に**、それが管轄区域に適合するかどうかを確認できます。

:::warning WCPOS は認定会計 POS ではありません
WCPOS は現在、規制市場の会計認証を**保持していません**。お住まいの国で認定済みの POS ソフトウェアが必要な場合は、ライブ セールスに WCPOS を使用する前に、現地の要件を確認してください。 WCPOS は WooCommerce のミドルウェアとして機能します。請求書発行と VAT は最終的に WooCommerce レベルで処理されます。
:::

:::info 私たちはそれに取り組んでいます
財政コンプライアンスは、現在進行中の開発分野です。私たちの目標は、ユーザーが活動するすべての国で WCPOS に準拠することです。以下のステータスは **現在**を反映しており、私たちがどこに向かっているのかを示すものではありません。お住まいの地域でのコンプライアンスが重要な場合は、[コミュニティ](https://wcpos.com/discord) でお知らせください。次に取り組む市場の優先順位を付けるのに役立ちます。
:::

## WCPOS による税金の処理方法 {#how-wcpos-handles-tax}

WCPOS は、WooCommerce からダウンロードしたレートを使用してアプリ内でローカルに税金を計算し、WooCommerce で注文を作成します。これは高速でオフラインで動作しますが、認証済み/サーバー側の税務ロジック (一部のコンプライアンス プラグインなど) は、注文がサーバーに到着したときにのみ実行されることを意味します。 **「サーバーに保存」** アクションと **Web チェックアウト** / **電子メール請求書** ゲートウェイは、標準の WooCommerce チェックアウトを通じて注文を送信し、サーバー側の税金とコンプライアンスのルールが適用されます。

## 国のステータス {#country-status}

### フランス — NF525 {#france-nf525}

WCPOS は NF525 に対して **認定されていません**。フランスの法律では、POS システムが不正行為に対して認定されることを義務付けているため、これはフランスでの販売の可能性に影響します。 WCPOS は WooCommerce のミドルウェアであるため、請求書発行と VAT のコンプライアンスは WooCommerce レベルで処理されます。[フランス VAT 詐欺防止法](https://woocommerce.com/posts/french-vat-fraud-prevention-laws-how-to-certify-invoices-with-woocommerce/) に関する WooCommerce のガイダンスを参照してください。

### スペイン — VERIFACTU {#spain-verifactu}

WCPOS は VERIFACTU に対して**認定されていません**。スペインでは POS ソフトウェアの認定が必要です。 VERIFACTU の期限は **2027 年に延期されました**。チームは要件を認識していますが、認定のスケジュールはまだ決まっていません。

### ノルウェー — カサシステムロブン {#norway-kassasystemloven}

WCPOS はノルウェー現金登録法 (_Kassasystemloven_) の**認定を受けていません**。また、サプライヤー宣言は利用できません。ノルウェーの一部の販売者は POS を使用していますが、法的義務についてはご自身で確認する必要があります。

### チリ — SII {#chile-sii}

検証済みの販売領収書 (_boletas_) については、チリの Servicio de Impuestos Internos (SII) との**統合はありません**。領収書テンプレートは標準のカスタマイズ可能な WooCommerce テンプレートですが、SII 準拠は組み込まれていません。

### EU — ワンストップショップ (OSS) {#eu-oss}

POS は、ダウンロードされた税率からローカルで税を計算するため、国境を越えた販売のための EU ワン ストップ ショップ (OSS) VAT スキームをネイティブにはサポートしません**。

**回避策:**
- チェックアウト前に **サーバーに保存** を使用して、OSS プラグインを通じてサーバー側の税金の再計算をトリガーするか、
- **Web チェックアウト** または **電子メール請求書** ゲートウェイを使用して、OSS ルールが適用される標準の WooCommerce チェックアウトを通じて支払いを受け取ります。

## その他の規制市場 {#other-regulated-markets}

認定済み POS ソフトウェアを必要とする管轄区域については、まず地域の要件を確認してください。 WCPOS は現在、規制市場向けの認証を取得していません。認定ステータスが購入の決定に重要である場合は、コミットする前に [コミュニティ](https://wcpos.com/discord) で質問してください。
