---
title: Zahlungsgateways
description: WCPOS Pro-Benutzer können Zahlungen von jedem aktivierten WooCommerce-Gateway akzeptieren.
---

WCPOS unterstützt standardmäßig drei Zahlungsgateways: Barzahlung, Karte (externes EFTPOS-Gerät) und PayPal.  
[WCPOS Pro](http://wcpos.com/pro) Benutzer können jedes installierte WooCommerce-Gateway aktivieren, was es den Shop-Besitzern ermöglicht, Kreditkarten, Bitcoin oder [über 100 andere Zahlungsgateways](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/) zu verarbeiten.

![Einstellungen für Zahlungsgateways in WCPOS.](https://wcpos.com/wp-content/uploads/2014/09/payment-gateways.png "Ein Beispiel für die Einstellungen von Zahlungsgateways in WCPOS")

Zahlungsgateways können unabhängig von WooCommerce aktiviert werden, was bedeutet, dass Sie eine Zahlung im Geschäft akzeptieren können, aber sie nicht unbedingt in Ihrem Webshop aktivieren müssen.  
Zahlungsreihenfolge und die Standardzahlungsart werden ebenfalls unabhängig festgelegt, sodass Sie Ihr POS-Checkout-Display anpassen können.

Einige Gateways, wie PayPal Standard, verwenden eine externe Webseite, um den Zahlungsprozess abzuschließen.  
Wenn eine Weiterleitung erforderlich ist, öffnet WCPOS ein neues Fenster.  
Sobald die Zahlung abgeschlossen ist, können Sie zum Beleg zurückkehren und auf 'aktualisieren' klicken, um zu bestätigen, dass die Zahlung abgeschlossen wurde.

:::caution  
**Bitte beachten Sie:** Zahlungsgateways in WCPOS sind noch experimentell.  
Mit über 100 verfügbaren Gateways kann ich nicht garantieren, dass alle sofort funktionieren.  
Wenn Sie Probleme bei der Verarbeitung von Zahlungen haben, wenden Sie sich bitte an [Support](mailto:support@wcpos.com) und wir werden mit Ihnen zusammenarbeiten, um Ihr Gateway zum Laufen zu bringen.  
:::
