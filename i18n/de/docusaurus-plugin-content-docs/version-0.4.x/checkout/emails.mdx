---
title: "Emails"
description: "Mit den WCPOS-Einstellungen können Ladenbesitzer auswählen, ob sie die Email-Benachrichtigungen von WooCommerce aktivieren oder deaktivieren."
---

Standardmäßig sendet WooCommerce Emails, um den Ladenbesitzer und den Kunden über den Status ihrer Bestellungen zu informieren. Viele dieser Emails sind für Point of Sale-Bestellungen unnötig, da der Ladenbesitzer an einem Tag Hunderte von POS-Bestellungen aufgeben kann. Eine Benachrichtigung über jede neue Bestellung würde nur den Posteingang überladen und von anderen wichtigen Alerts über eine Online-Bestellung ablenken.

Mit den WCPOS-Einstellungen können Ladenbesitzer auswählen, ob sie die Email-Benachrichtigungen von WooCommerce aktivieren oder deaktivieren. Dies betrifft nur die über das POS-System bearbeiteten Bestellungen.

![Email-Einstellungen für POS-Bestellungen.](https://wcpos.com/wp-content/uploads/2015/07/email-settings.png "Email-Einstellungen für POS-Bestellungen")

Die Aktivierung der Admin Emails sendet Benachrichtigungen an die Email-Adresse des Administrators. Die Aktivierung der Kunden-Emails sendet Benachrichtigungen an die Rechnungs-Email-Adresse (es muss ein vorhandener Kunde beim Checkout ausgewählt werden). Benachrichtigungen bei niedrigem und keinem Lagerbestand werden immer an die Email-Adresse des Administrators gesendet.

| Typ                                               | Hooks                                                                                                                                                                                                                                                                                                                                                                                            |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Kunden-Emails** <br /> _standardmäßig deaktiviert_ | woocommerce_order_status_pending_to_processing_notification <br /> woocommerce_order_status_pending_to_on-hold_notification <br /> woocommerce_order_status_completed_notification                                                                                                                                                                                                               |
| **Admin Emails** <br /> _standardmäßig deaktiviert_ | woocommerce_order_status_pending_to_processing_notification <br /> woocommerce_order_status_pending_to_completed_notification <br /> woocommerce_order_status_pending_to_on-hold_notification <br /> woocommerce_order_status_failed_to_processing_notification <br /> woocommerce_order_status_failed_to_completed_notification <br /> woocommerce_order_status_failed_to_on-hold_notification  |
| **Admin Emails** <br /> _immer gesendet_           | woocommerce_low_stock_notification <br /> woocommerce_no_stock_notification <br /> woocommerce_product_on_backorder_notification                                                                                                                                                                                                                                                                 |
