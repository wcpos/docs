title: Sicherheit

---

WCPOS ermöglicht die Zahlung mit jedem installierten [WooCommerce Gateway](http://www.woothemes.com/product-category/woocommerce-extensions/payment-gateways/). Wenn das Zahlungsgateway persönliche Informationen erfasst, **liegt es an Ihnen**, die Sicherheit der Informationen Ihrer Kunden zu gewährleisten. Nachfolgend sind einige Tipps aufgeführt, um Ihr POS sicher zu halten:

### Verwenden Sie ein SSL-Zertifikat

Wenn Sie einen Online-Store betreiben, sollten Sie ein Sicherheitszertifikat für Ihre Website haben.
Dies wird sicherstellen, dass alle Formulardaten, die Sie senden, wie Anmeldedaten und Kreditkartennummern, verschlüsselt werden.
Wenn Sie kein Sicherheitszertifikat verwenden, können alle Informationen, die Sie über das Internet senden, abgefangen und gelesen werden, sei es von jemandem, der Ihr Netzwerk teilt, oder an einem der Dutzenden von Knoten, die Ihren Computer mit Ihrem Server verbinden.
Ein grundlegendes Sicherheitszertifikat [kostet nur 16 US-Dollar](https://www.servertastic.com/rapidssl/) und kann innerhalb eines Tages eingerichtet werden ... es gibt also keine Ausreden, kein Sicherheitszertifikat zu haben.

### Lassen Sie Ihr POS nicht unbeaufsichtigt

Das Ausspionieren über das Internet ist nur eine Möglichkeit, wie Ihre Sicherheit gefährdet werden kann, aber oft wird der echte Schaden angerichtet, wenn ein böswilliger Benutzer physischen Zugriff auf Ihr Gerät erhält.
Dies ist besonders relevant für Geschäftsinhaber, die ihren Computer oder ihr Tablet häufig in stark frequentierten Bereichen lassen.
Planen Sie für den schlimmsten Fall, dass jemand Ihren Computer oder Ihr Gerät stiehlt, und minimieren Sie das Risiko, indem Sie einige Sicherheitsmaßnahmen ergreifen, wie:

- Verwenden Sie einen Sperrbildschirm, wenn Ihr Gerät 5 oder 10 Minuten unbeaufsichtigt ist
- Melden Sie sich von WordPress und WCPOS ab, wenn Sie Ihr Gerät verlassen
- Setzen Sie Ihre Passwörter im Falle eines Diebstahls Ihres Geräts schnell zurück
- Löschen Sie Ihr Gerät aus der Ferne, falls es gestohlen wird

### Verwenden Sie so wenige Plugins wie möglich

Einer der großartigen Aspekte von WordPress sind die Tausenden von Plugins, die mit einem Klick verfügbar sind ... aber es ist auch einer der schlechtesten.
Jedes Mal, wenn Sie ein Plugin installieren, gewähren Sie diesem Plugin effektiv Zugriff auf Ihre Website und auf alle Daten, die darüber laufen.
Ich weiß, es ist verlockend, Ihren Shop mit jedem Plugin unter der Sonne "aufzupeppen", aber das macht Ihre Seite oft langsamer und kann sie unsicher machen.
Bevor Sie ein Plugin installieren, sollten Sie sich folgende Fragen stellen:

1.  Brauchen Sie es wirklich?
2.  Vertrauen Sie dem Autor des Plugins?
