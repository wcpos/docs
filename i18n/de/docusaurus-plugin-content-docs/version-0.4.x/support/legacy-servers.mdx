title: "Legacy-Server"

---

HTTP (und HTTPS) sind die Protokolle, die von jedem Browser verwendet werden, um mit Webanwendungen zu kommunizieren. Ein wichtiger Teil des Protokolls ist die [HTTP-Methode](https://tools.ietf.org/html/rfc2616#section-5.1.1). Traditionell verwendeten Webanwendungen eine sehr begrenzte Menge an HTTP-Methoden wie GET (eine Webseite abrufen) und POST (ein Webformular absenden).

Moderne Webanwendungen, wie WCPOS, verwenden einen REST/HTTP-Ansatz mit folgenden HTTP-Methoden:

- GET
- POST
- PUT
- PATCH
- DELETE

In einigen seltenen Fällen kann ein Webhoster entscheiden, einige dieser HTTP-Methoden zu blockieren. Wenn Ihr Webhost moderne RESTful-HTTP-Methoden blockiert, müssen Sie die Unterstützung für Legacy-Server aktivieren.
