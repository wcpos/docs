HTTP (und HTTPS) sind die Protokolle, die von jedem Browser verwendet werden, um mit Webanwendungen zu kommunizieren.
Ein Schlüsselelement des Protokolls ist die [HTTP-Methode](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Traditionell verwendeten Webanwendungen einen sehr begrenzten Satz von HTTP-Methoden wie GET (Webseite abrufen) und POST (Webformular absenden).

Moderne Webanwendungen wie WCPOS verwenden einen REST/HTTP-Ansatz mit folgenden HTTP-Methoden:

- GET
- POST
- PUT
- PATCH
- DELETE

In seltenen Fällen kann ein Webhost einige dieser HTTP-Methoden blockieren.
Wenn Ihr Webhost moderne RESTful-HTTP-Methoden blockiert, müssen Sie die Unterstützung für Legacy-Server aktivieren.
