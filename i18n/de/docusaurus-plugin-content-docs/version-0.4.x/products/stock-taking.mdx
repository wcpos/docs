title: Produktbestandsaufnahme
---

:::important
Diese Funktion erfordert ein Upgrade auf [WCPOS Pro](http://wcpos.com/pro).
:::

Die Auswahl der Seite `Produkte` aus dem `POS-Menü` öffnet eine Schnellbearbeitungsseite für Ihre WooCommerce-Produkte.
