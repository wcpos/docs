---
title: Benutzerdefiniertes Barcode-Feld
---

Standardmäßig verwendet WCPOS das SKU-Feld als Barcode.

![Standardmäßig verwendet WCPOS das SKU-Feld als Barcode. ](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Standardmäßig verwendet WCPOS das SKU-Feld als Barcode")

[WCPOS Pro](http://wcpos.com/pro) Benutzer können jedes Produktmetafeld verwenden, indem sie ein Barcode-Feld in `WP Admin > POS > Einstellungen > Allgemein` auswählen.

![Wählen Sie das benutzerdefinierte Barcode-Feld in WCPOS Pro aus](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Wählen Sie das benutzerdefinierte Barcode-Feld in WCPOS Pro aus")

Benutzerdefinierte Barcode-Metafelder können auch über den Filter `woocommerce_pos_barcode_meta_key` festgelegt werden.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Ein Barcode muss für jedes Produkt einzigartig sein.
:::
