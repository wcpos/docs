---
title: Benutzerdefiniertes Barcode-Feld
---

Standardmäßig verwendet WCPOS das SKU-Feld als Barcode.

![Standardmäßig verwendet WCPOS das SKU-Feld als Barcode. ](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Standardmäßig verwendet WCPOS das SKU-Feld als Barcode")

[WCPOS Pro](http://wcpos.com/pro) Benutzer können jedes Produkt-Metafeld verwenden, indem sie ein Barcode-Feld in `WP Admin > POS > Einstellungen > Allgemein` auswählen

![Benutzerdefiniertes Barcode-Feld in WCPOS Pro auswählen](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Benutzerdefiniertes Barcode-Feld in WCPOS Pro auswählen")

Benutzerdefinierte Barcode-Metafelder können auch mit dem `woocommerce_pos_barcode_meta_key` Filter gesetzt werden.

```php title="functions.php"
/**
 * Benutzerdefiniertes Produkt-Metafeld als Barcode
 *
 * Standardmäßig verwendet WCPOS das SKU-Feld - '_sku'
 * Sie können den Metakey mit dem folgenden WordPress-Filter ändern
 *
 * Unten sind Metafelder von beliebten Barcode-Plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // Ändern Sie dies zu Ihrem benutzerdefinierten Metakey
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Ein Barcode muss für jedes Produkt eindeutig sein.
:::
