Es kann Fälle geben, in denen ein Geschäft Artikel verkaufen muss, die nicht Teil des WooCommerce-Inventars sind. Unten finden Sie zwei Optionen für den Verkauf von „verschiedenen Produkten“ in WCPOS.

### Nur POS Verschiedenes Produkt (empfohlen)

![Nur POS Verschiedenes Produkt](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Erstellen Sie ein nur POS Verschiedenes Produkt")

Mit der Funktion [Nur POS](./pos-only-products) können Sie ein „verschiedenes Produkt“ für die Verwendung im POS erstellen. Der Titel, der Verkaufspreis, der reguläre Preis, der Steuersatz und die Metainformationen können während des Checkouts dynamisch geändert werden.

![Bearbeiten Sie das Verschiedene Produkt](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Bearbeiten Sie das Verschiedene Produkt im Warenkorb")

### Gebühr Artikel

![Gebühr Artikel](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Bearbeiten Sie den Gebühr Artikel")

Die Verwendung eines Gebühr Artikels ähnelt der Erstellung eines Nur POS Verschiedenen Produkts. Kassierer können Titel, Preis und Steuersatz der Gebühr schnell aktualisieren. Der Hauptnachteil ist, dass Gebühren in Ihren Berichten möglicherweise separat von den Produkten angezeigt werden.
