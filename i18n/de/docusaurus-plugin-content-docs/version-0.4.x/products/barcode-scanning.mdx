title: Barcode-Scanning
description: WCPOS kann Produkte nach dem SKU-Feld filtern. So können Produkte mithilfe eines Barcodescanners sofort zum Warenkorb hinzugefügt werden.
---

WCPOS kann Produkte nach dem SKU-Feld filtern. Wenn eine Suche exakt mit dem Barcode-Feld übereinstimmt, wird das Produkt sofort zum Warenkorb hinzugefügt und das Suchfeld zurückgesetzt. Wenn ein Barcodescanner vorhanden ist, der das Suchfeld befüllen kann, lässt sich WCPOS für das Barcode-Scanning verwenden.

Die SKU-Suche kann unter http://demo.wcpos.com/pos/ ausprobiert werden.

Geben Sie `BARCODE` ein (oder kopieren Sie den Wert), und ein Artikel wird zum Warenkorb hinzugefügt.

Geben Sie `VARIATION1` ein (oder kopieren Sie den Wert), und eine Variante wird zum Warenkorb hinzugefügt.

![Barcode-Scanning in WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtering by the SKU field in WCPOS")

Die folgenden Funktionen sind für zukünftige Versionen geplant:

- Produkte per Barcodescanner zu WooCommerce hinzufügen. Dies wird Teil eines Inventurmodus sein, der für eine 0.3.x-Version geplant ist.
- Barcode-Scanning über die Kamera des Mobilgeräts. Dafür wird eine native WCPOS-App benötigt, um auf die Gerätehardware zugreifen zu können. Native Apps stehen auf der Wunschliste, aber die Erweiterung der Webversion hat derzeit die höchste Priorität.
- Aktivierung per Tastenkürzel

### Aktualisierungen {#updates}

**Since Version 0.3.1:** There is now a dedicated mode for Barcode scanning. You can switch between 'Search' mode and 'Scan Barcode' by clicking on the button to the left of the search field.

![Search Scan Barcode modes](https://wcpos.com/wp-content/uploads/2014/07/search-scan-mode.png)
