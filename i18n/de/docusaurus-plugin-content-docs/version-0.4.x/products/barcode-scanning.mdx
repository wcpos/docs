---
title: Barcode-Scan
description: WCPOS kann Produkte nach dem SKU-Feld filtern, was es ermöglicht, Produkte sofort mit einem Barcode-Scanner zum Warenkorb hinzuzufügen.
---

WCPOS kann Produkte nach dem SKU-Feld filtern. Wenn eine Suche genau mit dem Barcode-Feld übereinstimmt, wird das Produkt sofort zum Warenkorb hinzugefügt und das Suchfeld zurückgesetzt. Wenn Sie einen Barcode-Scanner haben, der das Suchfeld füllen kann, sollten Sie WCPOS für den Barcode-Scan verwenden.

Um den SKU-Scan in Aktion zu sehen, besuchen Sie bitte http://demo.wcpos.com/pos/

Geben Sie (oder kopieren & fügen Sie ein) `BARCODE`, und ein Artikel wird zum Warenkorb hinzugefügt.

Geben Sie (oder kopieren & fügen Sie ein) `VARIATION1`, und eine Variante wird zum Warenkorb hinzugefügt.

![Barcode-Scan in WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtern nach dem SKU-Feld in WCPOS")

Die folgenden Funktionen sind für zukünftige Versionen geplant:

- Produkte mit Barcode-Scanner zu WooCommerce hinzufügen. Dies wird Teil eines Bestandsaufnahme-Modus sein, der für eine 0.3.x-Version geplant ist.
- Barcode-Scan mit der Kamera des mobilen Geräts. Dies erfordert eine native WCPOS-App, um Zugriff auf die Geräte-Hardware zu erhalten. Native Apps stehen auf der Wunschliste, aber Funktionen zur Webversion hinzuzufügen hat derzeit höchste Priorität.
- Hot-Key-Aktivierung

### Updates

**Seit Version 0.3.1:** Es gibt jetzt einen speziellen Modus für Barcode-Scan. Sie können zwischen 'Suche' und 'Barcode scannen' wechseln, indem Sie auf die Schaltfläche links neben dem Suchfeld klicken.
