title: Barcode-Scan
description: WCPOS kann Produkte nach dem SKU-Feld filtern, wodurch Sie Produkte sofort mit einem Barcode-Scanner in den Warenkorb legen können.
---

WCPOS kann Produkte nach dem SKU-Feld filtern. Wenn eine Suche genau mit dem Barcode-Feld übereinstimmt, wird das Produkt sofort in den Warenkorb gelegt und das Suchfeld wird zurückgesetzt. Wenn Sie einen Barcode-Scanner haben, der das Suchfeld befüllen kann, sollten Sie WCPOS zum Scannen von Barcodes verwenden.

Um die SKU-Suche in Aktion zu sehen, gehen Sie bitte zu http://demo.wcpos.com/pos/

Geben Sie `BARCODE` ein (oder kopieren und fügen Sie es ein), und ein Artikel wird in den Warenkorb gelegt.

Geben Sie `VARIATION1` ein (oder kopieren und fügen Sie es ein), und eine Variation wird in den Warenkorb gelegt.

![Barcode-Scan in WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtern nach dem SKU-Feld in WCPOS")

Die folgenden Funktionen sind für zukünftige Releases geplant:

- Produkte zu WooCommerce mithilfe eines Barcode-Scanners hinzufügen. Dies wird Teil eines Bestandsaufnahme-Modus sein, der für ein 0.3.x Release geplant ist.
- Barcode-Scanning mit der Kamera des mobilen Geräts. Dies wird eine native WCPOS-App erfordern, um auf die Hardware des Geräts zuzugreifen. Native Apps stehen auf der Wunschliste, aber die Hinzufügung von Funktionen zur Webversion hat derzeit die höchste Priorität.
- Hotkey-Aktivierung

### Updates

**Seit Version 0.3.1:** Es gibt nun einen speziellen Modus für das Scannen von Barcodes. Sie können zwischen dem 'Such'-Modus und dem 'Barcode scannen'-Modus wechseln, indem Sie auf die Schaltfläche links vom Suchfeld klicken.

![Such- und Scan-Modi](https://wcpos.com/wp-content/uploads/2014/07/search-scan-mode.png)
