### Artikelpreis

Durch Klicken auf den Artikelpreis können Sie den Preis _im Handumdrehen_ ändern.
Der Artikelpreisziffernblock ermöglicht es Ihnen, einen neuen Verkaufspreis oder einen Rabatt als Prozentsatz einzugeben.

![Reduzierung des Artikelpreises durch Prozentsatz mithilfe des Ziffernblocks](https://wcpos.com/wp-content/uploads/2016/08/product-discount.png "Reduzierung des Artikelpreises durch Prozentsatz mithilfe des Ziffernblocks")

### Bestellrabatt

Sie können einen Bestellrabatt anwenden, indem Sie eine negative Gebühr zum Warenkorb hinzufügen.
Klicken Sie auf den Titel, um die Gebühr umzubenennen.
Klicken Sie auf den Pfeil, um den Steuerstatus und -satz zu ändern.

![Bestellrabatt Ziffernblock](https://wcpos.com/wp-content/uploads/2016/08/negative-fee.png "Verwendung einer negativen Gebühr zur Anwendung eines Bestellrabatts")
