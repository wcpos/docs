WCPOS verwendet eine Anzahl von Ziffernblock-Komponenten, um eine schnelle Eingabe über die Maus oder den Touchscreen zu ermöglichen. Im Folgenden sind Beispiele für die verschiedenen Arten von Ziffernblöcken:

![Mengen-Ziffernblock mit schneller Eingabe für +1 und -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Mengen-Ziffernblock mit schneller Eingabe für +1 und -1")

![Artikelpreis-Ziffernblock mit schneller Eingabe für Rabattprozentsätze.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Artikelpreis-Ziffernblock mit schneller Eingabe für Rabattprozentsätze")

![Betrag gegeben-Ziffernblock mit schneller Eingabe für wahrscheinliche Bargeldbeträge.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Betrag gegeben-Ziffernblock mit schneller Eingabe für wahrscheinliche Bargeldbeträge")

![Cashback-Ziffernblock mit schneller Eingabe für gängige Cashback-Beträge.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Cashback-Ziffernblock mit schneller Eingabe für gängige Cashback-Beträge")
