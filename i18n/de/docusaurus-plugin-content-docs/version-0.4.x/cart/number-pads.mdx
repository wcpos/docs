![Cashback-Zahlenpad mit schneller Eingabe für gängige Cashback-Beträge.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Cashback-Zahlenpad mit schneller Eingabe für gängige Cashback-Beträge")
