### Berechtigungen {#capabilities}

Kassierer benötigen die Berechtigungen _list_users_, _create_users_ und _edit_users_, um alle Kundenfunktionen zu nutzen.

### WordPress Multisite {#wordpress-multisite}

Bei Verwendung von [WordPress Multisite](https://codex.wordpress.org/Create_A_Network) gibt es einige [zusätzliche Prüfungen](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php), um zu verhindern, dass Benutzer ohne [Super-Admin](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin)-Rechte Kunden erstellen und bearbeiten.

Damit Benutzer ohne Super-Admin-Rechte neue Kunden **erstellen** können, aktivieren Sie zunächst die folgende Option unter `Network Admin > Einstellungen`.

![Multisite-Option zum Erstellen neuer Benutzer](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Multisite-Option zum Erstellen neuer Benutzer")

Damit Benutzer ohne Super-Admin-Rechte Kundendaten **bearbeiten** können, müssen Sie dem Kassierer außerdem die Berechtigung `manage_network_users` hinzufügen.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
