### Funktionen

Kassierer benötigen die Fähigkeiten _list_users_, _create_users_ und _edit_users_, um alle Kundenfunktionen nutzen zu können.

### WordPress Multisite

Bei der Verwendung von [WordPress Multisite](https://codex.wordpress.org/Create_A_Network) gibt es einige [zusätzliche Überprüfungen](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php), um zu verhindern, dass Nicht-[Super-Administratoren](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) Kunden erstellen und bearbeiten.

Zuerst, damit Nicht-Super-Administratoren **neue** Kunden **erstellen** können, aktivieren Sie bitte die folgende Option in `Netzwerk-Admin > Einstellungen`.

![Multisite-Option zum Erstellen neuer Benutzer](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Multisite-Option zum Erstellen neuer Benutzer")

Darüber hinaus müssen Sie, damit Nicht-Super-Administratoren **Kundendetails** **bearbeiten** können, die Fähigkeit `manage_network_users` dem Kassierer hinzufügen.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
