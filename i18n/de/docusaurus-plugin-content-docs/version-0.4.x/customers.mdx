### Berechtigungen

Kassierer benötigen die Berechtigungen _list_users_, _create_users_ und _edit_users_, um alle Kundenfunktionen zu nutzen.

### WordPress Multisite

Bei Verwendung von [WordPress Multisite](https://codex.wordpress.org/Create_A_Network) gibt es zusätzliche [Überprüfungen](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php), um zu verhindern, dass Nicht-[Super-Administratoren](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) Kunden erstellen und bearbeiten.

Zunächst müssen Sie die folgende Option in `Netzwerk-Administration > Einstellungen` aktivieren, damit Nicht-Super-Administratoren neue Kunden **erstellen** können.

![Multisite-Option zum Erstellen neuer Benutzer](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Multisite-Option zum Erstellen neuer Benutzer")

Zusätzlich müssen Sie dem Kassierer die Berechtigung `manage_network_users` hinzufügen, damit er Kundendetails **bearbeiten** kann.

```php title="functions.php"
// Ändern Sie die CASHIER_ID und
// fügen Sie dies zu Ihrer functions.php hinzu

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
