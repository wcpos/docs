---
title: Bestellung nach POS- oder Online-Verkäufen filtern
description: Trennen Sie POS- und Online-Bestellungen mithilfe der Schnelllinks im WooCommerce-Admin.
---

Sie können Ihre POS- und Online-Bestellungen im WooCommerce-Admin schnell trennen, indem Sie die Schnelllinks oben links verwenden.
**Bitte beachten Sie:** Dies filtert nur POS-Bestellungen ab Version 0.3.1, Bestellungen vor 0.3.1 werden standardmäßig als Online angezeigt.

![Filter Orders](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

Wenn Sie zusätzlich alle abgeschlossenen Bestellungen aus dem Admin-Bereich herausfiltern möchten, um sich auf ausstehende/in Bearbeitung befindliche Bestellungen zu konzentrieren, können Sie den folgenden Code zu Ihrer [functions.php](http://codex.wordpress.org/Functions_File_Explained)-Datei hinzufügen.

```php title="functions.php"
/**
 * Standardfilter der Bestellungen im WP-Admin ändern
 * Fügen Sie dies zu Ihrer functions.php-Datei hinzu
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
