---
title: Bestellungen nach POS- oder Online-Verkäufen filtern
description: POS- und Online-Bestellungen mithilfe der Schnelllinks im WooCommerce-Adminbereich trennen.
---

POS- und Online-Bestellungen lassen sich im WooCommerce-Adminbereich schnell über die Schnelllinks oben links trennen.
**Bitte beachten:** Diese Filterung erfasst nur POS-Bestellungen ab Version 0.3.1. Bestellungen vor Version 0.3.1 werden standardmäßig als Online angezeigt.

![Bestellungen filtern](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

Wenn zusätzlich alle abgeschlossenen Bestellungen im Adminbereich ausgeblendet werden sollen, um sich auf ausstehende bzw. in Bearbeitung befindliche Bestellungen zu konzentrieren, kann der folgende Code in die Datei [functions.php](http://codex.wordpress.org/Functions_File_Explained) eingefügt werden.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
