---

Sie können Ihre POS- und Online-Bestellungen im WooCommerce-Admin schnell trennen, indem Sie die Schnelllinks oben links verwenden.  
**Bitte beachten:** Dies filtert nur POS-Bestellungen nach Version 0.3.1, Bestellungen vor 0.3.1 werden standardmäßig als Online betrachtet.

![Bestellungen filtern](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Bestellungen filtern")

Zusätzlich, wenn Sie alle abgeschlossenen Bestellungen aus Ihrem Admin-Bereich filtern möchten, damit Sie sich auf die ausstehenden/zu bearbeitenden Bestellungen konzentrieren können, können Sie den folgenden Code in Ihre [functions.php](http://codex.wordpress.org/Functions_File_Explained) Datei einfügen.

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
