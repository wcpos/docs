title: POS-Zugriff
description: Mit den POS-Zugriffseinstellungen können Ladenbesitzer die Benutzerrechte für das Point of Sale-System gewähren oder entfernen.
---

Mit den POS-Zugriffseinstellungen können Ladenbesitzer die Benutzerrechte für das Point of Sale-System gewähren oder entfernen.  
Standardmäßig sind die Rollen **Administrator** und **Shop-Manager** mit allen Fähigkeiten für die Nutzung des POS aktiviert.  
Es wird empfohlen, die Standardeinstellungen beizubehalten, es sei denn, Sie sind sich der Konsequenzen bewusst.  
Zum Beispiel wäre es eine _sehr_ schlechte Idee, die `publish_shop_orders`-Berechtigung der Rolle `Abonnenten` zu gewähren.

![Beispiel für POS-Zugriffseinstellungen](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Beispiel für POS-Zugriffseinstellungen")

| Fähigkeit                                                               | Zugriff                                     |
| ----------------------------------------------------------------------- | ------------------------------------------ |
| access_woocommerce_pos                                                   | Erforderlich, um auf die POS-Frontend zuzugreifen       |
| manage_woocommerce_pos                                                   | Erforderlich, um auf die POS-Admin-Einstellungen zuzugreifen |
| read_private_products                                                    | Erforderlich, um Produkte anzuzeigen                |
| edit_product <br /> edit_published_products <br /> edit_others_products | Erforderlich, um Produkte und Variationen zu bearbeiten  |
| read_private_shop_orders                                               | Erforderlich, um Bestellungen und Quittungen anzuzeigen      |
| publish_shop_orders                                                    | Erforderlich, um neue Bestellungen zu erstellen             |
| list_users                                                             | Erforderlich, um Kunden anzuzeigen                |
| create_users                                                           | Erforderlich, um neue Kunden zu erstellen          |
| edit_users                                                             | Erforderlich, um bestehende Kunden zu aktualisieren     |
| read_private_shop_coupons                                              | Erforderlich, um Gutscheine anzuzeigen                  |
