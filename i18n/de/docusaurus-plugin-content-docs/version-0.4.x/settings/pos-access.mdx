title: POS-Zugriff

---

Mit den POS-Zugriffseinstellungen können Ladenbesitzer Benutzerrechte für das Point-of-Sale-System gewähren oder entfernen.
Standardmäßig sind die Rollen **Administrator** und **Shop-Manager** mit allen Funktionen für die Nutzung des POS aktiviert.
Es wird empfohlen, die Standardeinstellungen beizubehalten, es sei denn, Sie sind sich der Konsequenzen bewusst.
Beispielsweise wäre es eine _sehr_ schlechte Idee, die Berechtigung `publish_shop_orders` der Rolle `Abonnenten` zu gewähren.

![Beispiel für POS-Zugriffseinstellungen](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Beispiel für POS-Zugriffseinstellungen")

| Berechtigung                                                            | Zugriff                                    |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | Erforderlich zum Zugriff auf die POS-Benutzeroberfläche |
| manage_woocommerce_pos                                                  | Erforderlich zum Zugriff auf POS-Admineinstellungen |
| read_private_products                                                   | Erforderlich zum Anzeigen von Produkten    |
| edit_product <br /> edit_published_products <br /> edit_others_products | Erforderlich zum Bearbeiten von Produkten und Varianten |
| read_private_shop_orders                                                | Erforderlich zum Anzeigen von Bestellungen und Belegen |
| publish_shop_orders                                                     | Erforderlich zum Erstellen neuer Bestellungen |
| list_users                                                              | Erforderlich zum Anzeigen von Kunden       |
| create_users                                                            | Erforderlich zum Erstellen neuer Kunden    |
| edit_users                                                              | Erforderlich zum Aktualisieren bestehender Kunden |
| read_private_shop_coupons                                               | Erforderlich zum Anzeigen von Gutscheinen  |
