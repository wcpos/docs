### Produktsichtbarkeit

POS-eigene Produkte sollten nur aktiviert werden, wenn Sie einige Produkte in entweder dem Online- oder physischen Laden verbergen möchten.
Weitere Informationen finden Sie in der Dokumentation zu [POS-eigenen Produkten](../products/pos-only-products).

### Standard-POS-Kunde

Eine neue Bestellung wird mit dem Standard-POS-Kunden initialisiert. Häufige Optionen sind:

- **Gast:** POS-Bestellungen sind keiner Benutzer-ID zugeordnet
- **Kassierer:** POS-Bestellungen sind dem Konto des angemeldeten Benutzers zugeordnet
- **Laufkundschaft:** POS-Bestellungen sind einem benutzerdefinierten Benutzerkonto zugeordnet (_Hinweis: Dieses Konto muss erstellt werden_)

[WCPOS Pro](https://wcpos.com/pro)-Benutzer können Benutzerkonten während des Checkouts _spontan_ erstellen.
Für die Erstellung eines neuen Kontos ist eine E-Mail-Adresse erforderlich, was eine Anforderung von WordPress ist.
**Die Erstellung eines Kontos für neue Kunden wird empfohlen**, da Sie so deren Bestellhistorie verfolgen und sie möglicherweise in Zukunft mit Werbeaktionen kontaktieren können.
