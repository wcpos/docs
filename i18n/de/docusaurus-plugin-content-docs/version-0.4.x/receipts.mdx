Die Druckvorlagen für die POS-Quittung befinden sich in der Datei `includes/views/print/tmpl-receipt.php` sowohl des kostenlosen als auch des Pro-Plugins. Quittungsvorlagen können angepasst werden, indem eine Datei `woocommerce-pos/print/tmpl-receipt.php` in Ihrem Themenverzeichnis erstellt wird. Der Code für beide Vorlagen ist unten aufgeführt.

### Anpassung des Quittungsdatums {#customising-the-receipt-date}

WCPOS verwendet [moment.js](http://momentjs.com/), um die Datumszeichenfolgen zu lokalisieren. Das standardmäßige Datumsformat ist `"MMMM Do YYYY, h:mm:ss a"`, z. B.: 31. Mai 2015, 7:20:44 pm. Bitte konsultieren Sie die [moment.js-Dokumentation](http://momentjs.com/docs/#/parsing/string-format/) für weitere Informationen zur Anpassung des Datumsformats.

### Bestell-Eigenschaften {#order-properties}

WCPOS verwendet die JSON-Ausgabe der WC REST API, um die Vorlage für die Bestellquittung zu füllen. [Die WC REST API-Dokumentation zeigt ein Beispiel](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) der Standard-JSON-Ausgabe. Einige zusätzliche Eigenschaften wurden von WCPOS mit dem Filter `woocommerce_api_order_response` hinzugefügt.

| Eigenschaft                         | Beschreibung                                                  |
| ----------------------------------- | -------------------------------------------------------------|
| cart_discount_tax                   | Der Steueranteil des Warenkorb-Rabatts                       |
| cashier.id                          | Benutzer-ID des Kassierers                                   |
| cashier.first_name                  | Vorname des Kassierers                                       |
| cashier.last_name                   | Nachname des Kassierers                                      |
| payment_details.result              | Erfolg oder Misserfolg des Zahlungsgateways                  |
| payment_details.message             | Alle Nachrichten vom Zahlungsgateway                         |
| payment_details.redirect            | Weiterleitungs-URL für externe Zahlungen, z. B.: PayPal Standard |
| payment_details.method\_`gateway`   | Gateway-spezifische Nachrichten, z. B.: Betrag und Wechselgeld |
| shipping_lines[i].total_tax         | Hinzufügen des Steuerbetrags für jede Versandlinie           |
| subtotal_tax                        | Der Steueranteil des Zwischensaldos                          |
