WCPOS ist das Point-of-Sale-Plugin für [WooCommerce](https://www.woothemes.com/woocommerce/).
Diese Dokumentation behandelt das Open-Source-Plugin sowie das Pro-Plugin, das dem kostenlosen Plugin zusätzliche Funktionen hinzufügt.

- [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org herunterladen
- Auf [WCPOS Pro](http://wcpos.com/pro) upgraden

:::danger

Bitte stellen Sie sicher, dass die Versionsnummer der Dokumentation mit der Version Ihrer installierten Plugins übereinstimmt.

:::
