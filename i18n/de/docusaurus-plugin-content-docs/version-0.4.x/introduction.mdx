WCPOS ist das Point of Sale Plugin für [WooCommerce](https://www.woothemes.com/woocommerce/).
Diese Dokumentation umfasst das Open-Source-Plugin sowie das Pro-Plugin, das zusätzliche Funktionen zum kostenlosen Plugin hinzufügt.

- Laden Sie [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org herunter
- Upgraden Sie auf [WCPOS Pro](http://wcpos.com/pro)

:::danger

Stellen Sie bitte sicher, dass die Versionsnummer der Dokumentation mit der Version Ihrer installierten Plugins übereinstimmt.

:::

### Weitere Dokumentationen
