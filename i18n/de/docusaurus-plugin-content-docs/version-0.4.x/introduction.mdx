WCPOS ist das Point of Sale-Plugin für [WooCommerce](https://www.woothemes.com/woocommerce/).
Diese Dokumentation behandelt das Open-Source-Plugin sowie das Pro-Plugin, das Funktionalität zum kostenlosen Plugin hinzufügt.

- Laden Sie [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org herunter
- Upgrade auf [WCPOS Pro](http://wcpos.com/pro)

:::danger

Bitte stellen Sie sicher, dass die Versionsnummer der Dokumentation mit der Version Ihrer installierten Plugins übereinstimmt.

:::

### Andere Dokumentation

Eine Liste der **Häufig gestellten Fragen** finden Sie unter [faq.wcpos.com](http://faq.wcpos.com).
**Entwicklerdokumentation** ist verfügbar unter [api.wcpos.com](http://api.wcpos.com).
