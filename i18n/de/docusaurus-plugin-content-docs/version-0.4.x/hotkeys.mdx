Hotkeys sind Tastenkürzel, die verwendet werden können, um schnell Aktionen in WCPOS auszulösen.

| Taste                        | Beschreibung     |
| ---------------------------- | ---------------- |
| <div align="center">?</div>  | Hilfebildschirm  |
| <div align="center">B</div>  | Barcode-Modus    |
| <div align="center">V</div>  | Suchmodus        |
| <div align="center">S</div>  | Mit Server sync  |
