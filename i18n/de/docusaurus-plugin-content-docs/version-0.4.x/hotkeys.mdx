Tastenkombinationen sind Tastaturkürzel, mit denen Aktionen in WCPOS schnell ausgelöst werden können.

| Taste                       | Beschreibung     |
| --------------------------- | ---------------- |
| <div align="center">?</div> | Hilfebildschirm  |
| <div align="center">B</div> | Barcode-Modus    |
| <div align="center">V</div> | Suchmodus        |
| <div align="center">S</div> | Mit Server synchronisieren |
