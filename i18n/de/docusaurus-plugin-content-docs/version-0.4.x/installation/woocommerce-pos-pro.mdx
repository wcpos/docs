:::important

[WCPOS Pro](http://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das der kostenlosen [WCPOS-Plugin](http://wordpress.org/plugins/woocommerce-pos/) Funktionen hinzufügt. 

:::

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, folgen Sie bitte den untenstehenden Schritten, um das Plugin zu installieren und zu aktivieren:

1.  Gehen Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Unter **Downloads** klicken Sie auf den Download-Link und speichern das Plugin auf Ihrem Desktop.
3.  Gehen Sie dann zu Ihrer Website, melden Sie sich an und gehen Sie zu `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
4.  Laden Sie die Plugin-ZIP-Datei von Ihrem Desktop hoch und aktivieren Sie sie.
5.  Gehen Sie als Nächstes zu `WP Admin > POS > Einstellungen > Lizenz` und geben Sie Ihren Lizenzschlüssel und Ihre Lizenz-E-Mail ein, um die Aktivierung abzuschließen.

![Meine Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Sie können WCPOS Pro auf Ihrer Kontoseite herunterladen")

Wenn neue Versionen bereit sind, erscheinen sie in Ihrem Updates-Dashboard. 
Für Informationen darüber, woran ich derzeit arbeite, behalten Sie bitte den [Fahrplan](http://wcpos.com/roadmap/) im Auge. 
Achten Sie auch auf das Änderungsprotokoll, das die neuen Funktionen jeder Version auflistet, und die [Dokumentationsseite](http://docs.wcpos.com), die mit Informationen zur Verwendung der neuen Funktionen aktualisiert wird.
