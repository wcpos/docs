:::important

[WCPOS Pro](http://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das Funktionalitäten zum kostenlosen [WCPOS-Plugin](http://wordpress.org/plugins/woocommerce-pos/) hinzufügt.

:::

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, folgen Sie bitte den folgenden Schritten zur Installation und Aktivierung des Plugins:

1.  Gehen Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Klicken Sie unter **Downloads** auf den Download-Link und speichern Sie das Plugin auf Ihrem Desktop.
3.  Gehen Sie dann zu Ihrer Website, melden Sie sich an und navigieren Sie zu `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
4.  Laden Sie die Plugin-ZIP-Datei von Ihrem Desktop hoch und aktivieren Sie es.
5.  Gehen Sie als Nächstes zu `WP Admin > POS > Einstellungen > Lizenz` und geben Sie Ihren Lizenzschlüssel und die Lizenz-E-Mail ein, um die Aktivierung abzuschließen.

![Meine Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Sie können WCPOS Pro auf Ihrer Kontoseite herunterladen")

Wenn neue Versionen bereit sind, erscheinen sie in Ihrem Update-Dashboard.
Informationen darüber, woran ich gerade arbeite, finden Sie auf der [Roadmap](http://wcpos.com/roadmap/).
Achten Sie auch auf das Änderungsprotokoll, das die neuen Funktionen jeder Version auflistet, und die [Dokumentationsseite](http://docs.wcpos.com) wird mit Informationen zur Nutzung der neuen Funktionen aktualisiert.
