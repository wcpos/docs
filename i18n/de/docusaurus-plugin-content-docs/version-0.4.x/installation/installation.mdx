### Automatische Installation (empfohlen)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installieren und aktivieren Sie WCPOS in weniger als 1 Minute")

1. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen`
2. Suchen Sie nach `WCPOS` im WordPress Plugin-Verzeichnis
3. Installieren Sie das Plugin
4. Aktivieren Sie das Plugin

### Manuelle Installation

1. [Laden Sie WCPOS](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org herunter
2. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. Aktivieren Sie das Plugin

:::info
Frühere Versionen von WCPOS finden Sie unter dem [Entwickler](https://wordpress.org/plugins/woocommerce-pos/developers/) Tab auf Wordpress.org.
:::
