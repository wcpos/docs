### Automatische Installation (empfohlen) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installieren und aktivieren Sie WCPOS in weniger als 1 Minute")

1. Gehe zu `WP Admin > Plugins > Add New`
2. Suche nach `WCPOS` im WordPress Plugin-Verzeichnis
3. Installiere das Plugin
4. Aktiviere das Plugin

### Manuelle Installation {#manual-installation}

1. [Laden Sie WCPOS herunter](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org
2. Gehe zu `WP Admin > Plugins > Add New > Upload Plugin`
3. Aktiviere das Plugin

:::info
Vorherige Versionen von WCPOS finden Sie im Tab [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) auf WordPress.org.
:::
