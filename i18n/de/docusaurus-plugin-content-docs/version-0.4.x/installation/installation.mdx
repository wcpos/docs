### Automatische Installation (empfohlen) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Install and activate WCPOS in less than 1 minute")

1. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen`
2. Suchen Sie nach `WCPOS` im WordPress-Plugin-Verzeichnis
3. Installieren Sie das Plugin
4. Aktivieren Sie das Plugin

### Manuelle Installation {#manual-installation}

1. [WCPOS herunterladen](https://wordpress.org/plugins/woocommerce-pos/) von WordPress.org
2. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. Aktivieren Sie das Plugin

:::info
Frühere Versionen von WCPOS finden Sie unter dem Tab [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) auf WordPress.org.
:::
