title: "Geschäfte"
---

:::important
Diese Funktion erfordert ein Upgrade auf [WCPOS Pro](http://wcpos.com/pro).
:::

![Beispiel für die Geschäfteliste](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Example of stores list")

WCPOS Pro fügt die Möglichkeit hinzu, separate _Geschäfte_ für Ihre physischen Verkäufe zu erstellen.
Geschäfte können ihre eigene Adresse, ihr eigenes Logo und andere Details für die Verwendung auf der [Belegvorlage](../receipts) haben.
Geschäfte können unterschiedliche Steuersätze verwenden und Verkäufe können nach Geschäft für [Berichte](../reports#sales-by-store) getrennt werden.

![Beispiel für die Bearbeitungsseite von Geschäften](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Example of store edit page")

### Kommende Funktionen {#coming-features}

Die Geschäfte-Funktion wurde in Version 0.4 hinzugefügt.
Es gibt mehrere _häufig angeforderte Funktionen_, wie [Multi-Inventory-Geschäfte](./multi-inventory) und die Möglichkeit, Kassierer Geschäften zuzuweisen.
Diese Funktionen werden zum Pro-Plugin hinzugefügt, während sich das Plugin der Version 1.0 nähert.
