---
title: Filialen
---

:::important
Diese Funktion erfordert ein Upgrade auf [WCPOS Pro](http://wcpos.com/pro).
:::

![Beispiel einer Filialliste](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Beispiel einer Filialliste")

WCPOS Pro bietet die Möglichkeit, separate _Filialen_ für Ihre physischen Verkäufe zu erstellen.
Filialen können eigene Adressen, Logos und andere Details für den [Kassenbon-Vorlagen](../receipts) haben.
Filialen können unterschiedliche Steuersätze verwenden, und Verkäufe können pro Filiale für die [Berichterstattung](../reports#sales-by-store) getrennt werden.

![Beispiel einer Filiale-Bearbeitungsseite](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Beispiel einer Filiale-Bearbeitungsseite")

### Geplante Funktionen

Die Filialen-Funktion wurde in Version 0.4 hinzugefügt.
Es gibt mehrere _häufig angeforderte Funktionen_, wie [Mehrfach-Bestandsfilialen](./multi-inventory) und die Möglichkeit, Kassierer Filialen zuzuweisen.
Diese Funktionen werden dem Pro-Plugin hinzugefügt, wenn sich das Plugin der Version 1.0 nähert.
