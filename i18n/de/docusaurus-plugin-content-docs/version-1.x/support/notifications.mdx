Die **Benachrichtigungsglocke** im POS-Header ist der Ort, an dem WCPOS Nachrichten anzeigt — Produktankündigungen, Hinweise zu Plugin-Updates und Erinnerungen zur Pro-Lizenz. Ein rotes Badge auf der Glocke zeigt an, wie viele ungelesene Benachrichtigungen vorhanden sind (Werte über 99 werden als **99+** angezeigt).

## Benachrichtigungszentrum öffnen {#opening-the-notification-centre}

Klicken Sie oben im POS auf das **Glockensymbol**, um das Benachrichtigungspanel zu öffnen. Es listet die Benachrichtigungen mit den neuesten zuerst auf, jeweils mit:

- Einem **Ungelesen-Punkt** — ausgefüllt, solange die Benachrichtigung ungelesen ist, und der Titel wird fett angezeigt
- Einem kurzen **Titel** und einem optionalen **Text**
- Einem **relativen Zeitstempel** (z. B. _vor 2 Stunden_)

Wenn keine Benachrichtigungen vorliegen, zeigt das Panel den leeren Zustand **"Keine Benachrichtigungen"** an.

## Benachrichtigungen lesen und entfernen {#reading-and-clearing-notifications}

- **Eine Benachrichtigung als gelesen markieren** — tippen Sie auf eine beliebige ungelesene Benachrichtigung in der Liste.
- **Alle als gelesen markieren** — verwenden Sie die Schaltfläche **Alle als gelesen markieren** in der Kopfzeile des Panels (wird nur angezeigt, wenn ungelesene Elemente vorhanden sind).
- **Badge löschen** — bereits das Öffnen des Panels löscht das Badge für die Anzahl ungelesener Benachrichtigungen an der Glocke, auch wenn Sie noch nicht jedes Element geöffnet haben.

Benachrichtigungen werden auf Ihrem Gerät gespeichert, sodass die Liste offline verfügbar bleibt. Neue Benachrichtigungen treffen in Echtzeit ein, während Sie verbunden sind.

## Welche Arten von Benachrichtigungen angezeigt werden {#what-kinds-of-notifications-appear}

Das Benachrichtigungscenter wird für System- und Kontomeldungen verwendet, nicht für Filialbestellungen. Möglicherweise sehen Sie:

| Typ | Beispiel |
| --- | --- |
| **Willkommen** | Eine einmalige Willkommensnachricht, wenn Sie sich zum ersten Mal verbinden |
| **Ankündigungen** | Allgemeine Neuigkeiten und Updates zu WCPOS |
| **Plugin-Update** | Ein Hinweis, dass ein WCPOS Plugin-Update verfügbar ist |
| **Lizenz läuft ab** | Eine Erinnerung, dass sich Ihre Pro-Lizenz ihrem Verlängerungsdatum nähert, mit einem Link **Verlängern** |
| **Lizenz abgelaufen** | Eine dauerhafte Warnung, dass Ihre Pro-Lizenz abgelaufen ist, mit einem Link **Jetzt verlängern** |

Wichtige Meldungen (z. B. Lizenzwarnungen) können zusätzlich zur Glocke auch kurz als Toast auf dem Bildschirm erscheinen.

:::note
Die Benachrichtigungszentrale liefert Nachrichten von WCPOS. Echtzeit-Benachrichtigungen für **neue Online-Bestellungen** sind eine separate, geplante Funktion — siehe die [Roadmap](https://github.com/orgs/wcpos/projects/4).
:::
