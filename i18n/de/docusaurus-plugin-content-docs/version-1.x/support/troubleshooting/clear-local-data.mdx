WCPOS speichert eine lokale Kopie Ihrer Produkte, Bestellungen und Kunden, damit das POS schnell arbeiten und offline funktionieren kann. Gelegentlich kann es vorkommen, dass diese lokalen Daten nicht mehr mit Ihrem WooCommerce-Shop synchron sind. Das Zurücksetzen aller lokalen Daten entfernt die lokale Datenbank und löst eine frische Synchronisierung von Ihrem Server aus.

:::warning
Dies ist die nukleare Option. Das Zurücksetzen lokaler Daten löscht alles, was auf dem Gerät gespeichert ist, und erzwingt eine vollständige Resynchronisierung. Je nach Größe Ihres Shops kann dies einige Zeit in Anspruch nehmen. Verwenden Sie es nur, wenn andere Optionen — wie eine normale [Synchronisierung](/products/sync) — das Problem nicht gelöst haben.
:::

## Wann Sie es verwenden sollten {#when-to-use-it}

Sie müssen möglicherweise die lokalen Daten zurücksetzen, wenn:

- **Sync-Probleme** — Produkte, Bestellungen oder Kunden werden nicht ordnungsgemäß aktualisiert
- **Nach einem Plugin-Update** — um die Kompatibilität mit neuen Datenstrukturen sicherzustellen
- **Veraltete Daten** — Preise, Bestände oder andere Informationen erscheinen veraltet

## Was zurückgesetzt wird {#what-gets-cleared}

| Zurückgesetzt | Nicht zurückgesetzt |
|---------|-------------|
| Lokale Produktdatenbank | Serverseitige Daten |
| Lokale Bestelldatenbank | WooCommerce-Einstellungen |
| Lokale Kundendatenbank | |
| Anmeldesitzung | |

Ihre WooCommerce-Shop-Daten sind nicht betroffen — nur die lokale Kopie auf Ihrem Gerät wird entfernt. Eine frische Synchronisierung beginnt automatisch nach dem Zurücksetzen.

:::info Plattformunterschied
- **Desktop-App:** Sie werden abgemeldet und müssen Ihren Shop erneut verbinden.
- **Web-App:** Die Anmeldedaten werden zurückgesetzt, aber da Sie weiterhin in WP Admin angemeldet sind, wird die Sitzung automatisch wiederhergestellt — Sie werden also nicht bemerken, dass Sie abgemeldet wurden.
:::

## So setzen Sie lokale Daten zurück {#how-to-clear-local-data}

### Web {#web}

1. Klicken Sie auf Ihr **Benutzermenü** in der oberen rechten Ecke, um das Menü zu öffnen
2. Klicken Sie auf **Alle lokalen Daten zurücksetzen**
3. Die lokale Datenbank wird gelöscht und eine frische Synchronisierung beginnt automatisch

### Desktop {#desktop}

1. Öffnen Sie das Anwendungsmenü **WCPOS** in der oberen Menüleiste
2. Fahren Sie mit der Maus über **Erweitert**
3. Klicken Sie auf **App-Daten zurücksetzen**
4. Die lokale Datenbank wird gelöscht und eine frische Synchronisierung beginnt automatisch

:::note
Die Desktop-App verwendet eine SQLite-Datenbank, während die Web-App IndexedDB verwendet. Beide Datenbanken werden vollständig zurückgesetzt, aber das Anmeldeverhalten unterscheidet sich — siehe den obigen Hinweis zum Plattformunterschied.
:::
