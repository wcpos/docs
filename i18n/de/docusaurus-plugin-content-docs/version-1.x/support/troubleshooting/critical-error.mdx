Wenn Sie diese Fehlermeldung sehen, bedeutet das, dass Ihr POS auf einen fatalen Serverfehler gestoßen ist und nicht fortfahren kann.
Die gute Nachricht ist, dass diese Fehlermeldung in Ihren WooCommerce-Protokollen gespeichert wird und unser Support-Team Ihnen bei der Fehlersuche helfen kann.

## Schritte zum Finden des Fehlers

1. Melden Sie sich in Ihrem WordPress-Admin-Bereich an.
2. Navigieren Sie zu "WooCommerce" > "Status" > "Protokolle" im WordPress-Seitenleistenmenü.
3. Suchen Sie den Abschnitt "fatal_error" Protokolle.
4. Klicken Sie auf den Protokolleintrag, der dem Tag entspricht, an dem der Fehler aufgetreten ist (normalerweise das heutige Datum), um die Details des Fehlers anzuzeigen.
5. Identifizieren Sie die Ursache des Fehlers, wie sie in diesem Abschnitt protokolliert ist.
6. Kopieren Sie alle relevanten Fehlermeldungen oder machen Sie einen Screenshot und teilen Sie diese mit unserem Support-Team, indem Sie sie im [Discord-Chat](https://wcpos.com/discord) posten oder an support@wcpos.com senden.

Teilen Sie die Fehlermeldung mit unserem Support-Team:
