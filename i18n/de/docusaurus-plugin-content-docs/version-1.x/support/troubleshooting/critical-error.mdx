Wenn Sie diese Fehlermeldung sehen, bedeutet dies, dass Ihr POS auf einen schwerwiegenden Serverfehler gestoßen ist und nicht fortfahren kann.
Die gute Nachricht ist, dass diese Fehlermeldung in Ihren WooCommerce-Protokollen protokolliert wird und unser Supportteam Ihnen bei der Fehlerbehebung helfen kann.

## Schritte zum Auffinden des Fehlers

1. Melden Sie sich im WordPress-Administrationsbereich an.
2. Navigieren Sie zu "WooCommerce" > "Status" > "Protokolle" im WordPress-Seitenmenü.
3. Suchen Sie den Abschnitt "fatal_error" Protokolle.
4. Klicken Sie auf den Protokolleintrag, der dem Tag entspricht, an dem der Fehler auftrat (normalerweise das heutige Datum), um die Details des Fehlers anzuzeigen.
5. Identifizieren Sie die Ursache des Fehlers, wie in diesem Abschnitt protokolliert.
6. Kopieren Sie alle relevanten Fehlermeldungen oder machen Sie einen Screenshot und teilen Sie diese mit unserem Supportteam, indem Sie sie im [Discord-Chat](https://wcpos.com/discord) posten oder an support@wcpos.com senden.

## Hilfe erhalten
