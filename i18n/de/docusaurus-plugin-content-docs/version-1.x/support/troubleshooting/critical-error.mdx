Wenn Sie diese Fehlermeldung sehen, bedeutet dies, dass Ihr POS auf einen schwerwiegenden Serverfehler gestoßen ist und nicht fortfahren kann. Die gute Nachricht ist, dass diese Fehlermeldung in Ihren WooCommerce-Protokollen erfasst wird und unser Support-Team Ihnen bei der Fehlersuche helfen kann.

## Schritte zur Fehlerfindung

1. Melden Sie sich in Ihrem WordPress-Adminbereich an.
2. Navigieren Sie zu "WooCommerce" > "Status" > "Protokolle" im WordPress-Seitenmenü.
3. Suchen Sie den Abschnitt "fatal_error"-Protokolle.
4. Klicken Sie auf den Protokolleintrag, der dem Tag entspricht, an dem der Fehler aufgetreten ist (normalerweise das heutige Datum), um die Einzelheiten des Fehlers anzuzeigen.
5. Ermitteln Sie die Ursache des Fehlers, wie sie in diesem Abschnitt protokolliert ist.
6. Kopieren Sie alle relevanten Fehlermeldungen oder machen Sie einen Screenshot und teilen Sie diese mit unserem Support-Team, indem Sie diese im [Discord-Chat](https://wcpos.com/discord) posten oder sie an support@wcpos.com senden.
