Brauchen Sie Hilfe mit WCPOS? Hier erfahren Sie, wie Sie Unterstützung bekommen und Antworten auf Ihre Fragen finden.

## Unterstützung in der App

Das POS enthält einen integrierten Support-Bildschirm, der über das linke Navigationsmenü zugänglich ist. Dieser Bildschirm bettet die WCPOS Discord-Community direkt in die App ein, sodass Sie:

- Bestehende Diskussionen durchsuchen
- Nach Antworten auf häufige Fragen suchen
- Fragen im Kanal #general stellen
- Über Ankündigungen auf dem Laufenden bleiben

## Discord-Community

Der primäre Supportkanal für WCPOS ist unser [Discord-Server](https://wcpos.com/discord).

### Kanäle

| Kanal | Zweck |
|-------|-------|
| **#welcome-and-rules** | Gemeinschaftsrichtlinien |
| **#announcements** | Updates und Neuigkeiten |
| **#general** | Hauptsupportkanal - stellen Sie hier Fragen |
| **#forum** | Längere Diskussionen und Feature-Anfragen |
| **#github-activity** | Entwicklungsupdates |

### Bevor Sie Fragen stellen

Um die beste Hilfe zu erhalten:

1. **Zuerst testen** - Laden Sie das Plugin herunter und testen Sie es einige Tage, um zu verstehen, ob es Ihren Bedürfnissen entspricht
2. **Dokumentation überprüfen** - Viele Fragen werden in dieser Dokumentation beantwortet
3. **Discord durchsuchen** - Ihre Frage könnte bereits beantwortet worden sein
4. **Details angeben** - Geben Sie Ihre WooCommerce-Version, WCPOS-Version und Schritte zur Reproduktion von Problemen an

## E-Mail-Support

Bei lizenzbezogenen Problemen oder privaten Anfragen können Sie eine E-Mail senden an: **support@wcpos.com**

:::note
Allgemeine Unterstützungsfragen sollten an Discord gerichtet werden, wo die Gemeinschaft helfen kann und andere von den Antworten profitieren können.
:::

## Logs Screen

Das POS enthält einen [Logs Screen](/support/logs), der die Systemaktivität aufzeichnet. Dies ist von unschätzbarem Wert für:

- Das Debuggen von Synchronisierungsproblemen
- Das Verständnis von Fehlermeldungen
- Das Bereitstellen detaillierter Informationen bei der Anforderung von Unterstützung

Wenn Sie um Hilfe bitten, überprüfen Sie den Logs Screen auf Fehlermeldungen und fügen Sie diese Ihrer Unterstützungsanfrage hinzu.

## Pro-Support

[Kunden von WCPOS Pro](https://wcpos.com/pro) erhalten den gleichen Community-Support über Discord. Die Pro-Lizenz finanziert die fortlaufende Entwicklung sowohl von kostenlosen als auch von Pro-Funktionen.

## Mitwirken

WCPOS ist Open Source! Sie können beitragen, indem Sie:

- Fehler auf [GitHub](https://github.com/wcpos) melden
- Funktionen im Discord vorschlagen
- Die Dokumentation übersetzen
- Ihre Erfahrungen mit anderen teilen
