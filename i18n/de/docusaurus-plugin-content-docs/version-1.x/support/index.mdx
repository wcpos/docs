Brauchen Sie Hilfe mit WCPOS? So erhalten Sie Unterstützung und finden Antworten auf Ihre Fragen.

## In-App Unterstützung {#in-app-support}

Die POS enthält einen integrierten Unterstützungsbildschirm, der über das linke Navigationsmenü erreichbar ist. Dieser Bildschirm bindet die WCPOS Discord-Community direkt in die App ein, sodass Sie:

- Vorhandene Diskussionen durchsuchen
- Nach Antworten auf häufige Fragen suchen
- Fragen im Kanal #general stellen
- Über Ankündigungen informiert bleiben

## Discord-Community

Der primäre Unterstützungs Kanal für WCPOS ist unser [Discord-Server](https://wcpos.com/discord).

### Kanäle {#channels}

| Kanal | Zweck |
|-------|-------|
| **#welcome-and-rules** | Gemeinschaftsrichtlinien |
| **#announcements** | Updates und Nachrichten |
| **#general** | Hauptunterstützungskanal - Fragen Sie hier |
| **#forum** | Längere Diskussionen und Feature-Anfragen |
| **#github-activity** | Entwicklungsupdates |

### Bevor Sie fragen {#before-asking}

Um die beste Hilfe zu erhalten:

1. **Zuerst testen** - Laden Sie das Plugin herunter und testen Sie es einige Tage, um zu verstehen, ob es Ihren Anforderungen entspricht
2. **Dokumentation überprüfen** - Viele Fragen werden in dieser Dokumentation beantwortet
3. **Discord durchsuchen** - Ihre Frage könnte bereits beantwortet worden sein
4. **Details angeben** - Geben Sie Ihre WooCommerce-Version, WCPOS-Version und Schritte zur Reproduktion der Probleme an

## E-Mail-Support {#email-support}

Bei lizenzbezogenen Problemen oder privaten Anfragen können Sie eine E-Mail senden an: **support@wcpos.com**

:::note
Allgemeine Supportfragen sollten in Discord gestellt werden, wo die Community helfen kann und andere von den Antworten profitieren können.
:::

## Protokollbildschirm {#logs-screen}

Die POS enthält einen [Protokollbildschirm](/support/logs), der die Systemaktivität aufzeichnet. Dies ist von unschätzbarem Wert für:

- Debuggen von Synchronisationsproblemen
- Verstehen von Fehlermeldungen
- Bereitstellen detaillierter Informationen bei Supportanfragen

Überprüfen Sie den Protokollbildschirm auf Fehlermeldungen und fügen Sie diese Ihrer Unterstützungsanfrage hinzu.

## Pro-Support

Kunden von [WCPOS Pro](https://wcpos.com/pro) erhalten dieselbe Community-Unterstützung über Discord. Die Pro-Lizenz finanziert die fortlaufende Entwicklung sowohl kostenloser als auch Pro-Funktionen.

## Mitwirken {#contributing}

WCPOS ist Open Source! Sie können beitragen, indem Sie:

- Fehler auf [GitHub](https://github.com/wcpos) melden
- Funktionen auf Discord vorschlagen
- Die Dokumentation übersetzen
- Ihre Erfahrungen mit anderen teilen
