Benötigen Sie Hilfe mit WCPOS? Hier erfahren Sie, wie Sie Support erhalten und Antworten auf Ihre Fragen finden.

## Integrierte Unterstützung

Die POS enthält einen integrierten Support-Bildschirm, der über die linke Navigationsleiste zugänglich ist. Dieser Bildschirm bindet die WCPOS Discord-Community direkt in die App ein, sodass Sie:

- Bestehende Diskussionen durchsuchen können
- Nach Antworten auf häufige Fragen suchen können
- Fragen im #general-Kanal stellen können
- Über Ankündigungen auf dem Laufenden bleiben

{/* Screenshot: support-screen.png */}

## Discord-Community

Der primäre Support-Kanal für WCPOS ist unser [Discord-Server](https://wcpos.com/discord).

### Kanäle

| Kanal | Zweck |
|---------|---------|
| **#welcome-and-rules** | Community-Richtlinien |
| **#announcements** | Updates und Neuigkeiten |
| **#general** | Hauptsupport-Kanal - hier können Sie Fragen stellen |
| **#forum** | Längere Diskussionen und Funktionsanfragen |
| **#github-activity** | Entwicklungsupdates |

### Vor dem Fragen

Um die beste Hilfe zu erhalten:

1. **Zuerst testen** - Laden Sie das Plugin herunter und testen Sie es ein paar Tage, um zu verstehen, ob es Ihren Anforderungen entspricht
2. **Dokumentation prüfen** - Viele Fragen werden in dieser Dokumentation beantwortet
3. **Discord durchsuchen** - Ihre Frage wurde möglicherweise bereits beantwortet
4. **Details bereitstellen** - Geben Sie Ihre WooCommerce-Version, WCPOS-Version und Schritte zur Reproduktion von Problemen an

## E-Mail-Support

Für lizenzbedingte Probleme oder private Anfragen können Sie eine E-Mail senden an: **support@wcpos.com**

:::note
Allgemeine Support-Fragen sollten an Discord gerichtet werden, wo die Community helfen und andere von den Antworten profitieren können.
:::

## Protokollbildschirm

Die POS enthält einen [Protokollbildschirm](/support/logs), der Systemaktivitäten aufzeichnet. Dieser ist unschätzbar wertvoll für:

- Debugging von Synchronisationsproblemen
- Verstehen von Fehlermeldungen
- Bereitstellen detaillierter Informationen bei Supportanfragen

Überprüfen Sie bei Hilfsanfragen den Protokollbildschirm auf Fehlermeldungen und fügen Sie diese Ihrer Supportanfrage hinzu.

## Pro-Support

[WCPOS Pro](https://wcpos.com/pro)-Kunden erhalten denselben Community-Support über Discord. Die Pro-Lizenz finanziert die weitere Entwicklung von sowohl kostenlosen als auch Pro-Funktionen.

## Mitwirken

WCPOS ist Open Source! Sie können beitragen durch:

- Melden von Fehlern auf [GitHub](https://github.com/wcpos)
- Vorschlagen von Funktionen auf Discord
- Übersetzen der Dokumentation
- Teilen Ihrer Erfahrungen mit anderen
