Variable Produkte in WooCommerce sind Produkte, die mehrere Optionen haben, wie verschiedene Größen, Farben oder Materialien. WCPOS bietet mehrere Möglichkeiten, um effizient mit variablen Produkten zu arbeiten.

## Identifizierung von Variablen Produkten {#identifying-variable-products}

Im Produktpanel werden variable Produkte unterschieden durch:

- **Pfeiltaste** (<Icon name="circle-arrow-right" color="green" />) statt der Plus-Taste (<Icon name="circle-plus" color="green" />), die für einfache Produkte verwendet wird
- **Attributoptionen**, die unter dem Produktnamen angezeigt werden (z. B. "Farbe: Blau, Grün, Rot")
- **"Erweitern" Link**, um alle Variationen inline anzuzeigen

## Hinzufügen von Variablen Produkten zum Warenkorb {#adding-variable-products-to-cart}

### Methode 1: Schnelles Popover {#method-1-quick-popover}

Klicken Sie auf die grüne <Icon name="circle-arrow-right" color="green" /> Pfeiltaste bei einem variablen Produkt, um ein Popover mit Dropdown-Menüs für jedes Attribut zu öffnen:

1. Klicken Sie auf die <Icon name="circle-arrow-right" color="green" /> Schaltfläche
2. Wählen Sie Optionen aus jedem Dropdown-Menü (z. B. Größe: Groß, Farbe: Blau)
3. Die passende Variation wird dem Warenkorb hinzugefügt

Dies ist die schnellste Methode, wenn Sie genau wissen, welche Variation Sie benötigen.

{/* Screenshot: variation-popover.png */}

### Methode 2: Inline-Erweiterung {#method-2-expand-inline}

Klicken Sie auf den **"Erweitern"** Link bei einem variablen Produkt, um alle seine Variationen direkt in der Produktliste anzuzeigen:

1. Klicken Sie auf **Erweitern** unter dem Produktnamen
2. Alle Variationen erscheinen als separate Zeilen unter dem übergeordneten Produkt
3. Klicken Sie auf die <Icon name="circle-plus" color="green" /> Schaltfläche bei einer beliebigen Variation, um sie dem Warenkorb hinzuzufügen
4. Klicken Sie auf **Einklappen**, um die Variationen wieder auszublenden

Diese Methode ist nützlich, wenn Sie alle verfügbaren Optionen, deren Lagerbestände und Preise sehen möchten.

{/* Screenshot: variations-expanded.png */}

### Methode 3: Barcode-Scannen {#method-3-barcode-scanning}

Jede Produktvariation kann ihren eigenen einzigartigen Barcode haben. Wenn Sie den Barcode einer Variation scannen:

- Wenn genau eine Variation übereinstimmt, wird sie direkt in den Warenkorb hinzugefügt
- Wenn mehrere Variationen den Barcode teilen, zeigt die Suchleiste übereinstimmende Ergebnisse an

Siehe [Barcode-Scannen](/pos/product-panel/barcode-scanning) für weitere Details.

## Variationsinformationen {#variation-information}

Jede Variation kann anzeigen:

- **Variationsattribute** - Die spezifischen Optionen für diese Variation (z. B. "Größe: Groß, Farbe: Blau")
- **SKU** - Eindeutige Kennung für die Variation (kann von dem übergeordneten Produkt abweichen)
- **Lagerbestand** - Individuelles Lagerlevel für die Variation
- **Preis** - Variationen können andere Preise haben als das übergeordnete Produkt oder untereinander

## Variationen im Warenkorb {#variations-in-the-cart}

Wenn eine Variation dem Warenkorb hinzugefügt wird, wird angezeigt:

- **Produktname** vom übergeordneten Produkt
- **Variationsattribute** unter dem Namen
- **Preis** für diese spezifische Variation

Sie können die Position genau wie jedes andere Produkt im Warenkorb bearbeiten.

## Tipps zur effizienten Auswahl von Variationen {#tips-for-efficient-variation-selection}

### Verwenden Sie Barcode-Scannen {#use-barcode-scanning}

Wenn Ihre Variationen individuelle Barcodes haben (empfohlen), ist das Scannen der schnellste Weg, um sie dem Warenkorb hinzuzufügen.

### Anzeigeoptionen konfigurieren {#configure-display-options}

Im Produktpanel [Anzeigeeinstellungen](/pos/product-panel#display-settings) aktivieren Sie die **Attribute** Option, um Variationsattribute direkt in der Produktliste ohne Erweitern anzuzeigen.

### Tastaturnavigation {#keyboard-navigation}

Wenn das Variationen-Popover geöffnet ist, können Sie die Pfeiltasten der Tastatur verwenden, um Optionen zu navigieren und die Eingabetaste zu wählen.

## F.A.Q.

<Accordion>
  <AccordionItem question="Warum kann ich eine bestimmte Variation nicht sehen?">

Überprüfen Sie, dass die Variation:
- Veröffentlicht und nicht im Entwurfsstatus ist
- Auf "Auf Lager" gesetzt ist oder eine Lagermenge > 0 hat (wenn das Lagerverwaltung aktiviert ist)
- Alle erforderlichen Attribute definiert hat

Stellen Sie auch sicher, dass Sie das Produkt mit dem "Auf Lager" Filter nicht herausgefiltert haben, wenn die Variation nicht auf Lager ist.

  </AccordionItem>

  <AccordionItem question="Wie füge ich individuelle Barcodes zu Variationen hinzu?">

Bearbeiten Sie in WooCommerce das variable Produkt und erweitern Sie jede Variation. Sie können jedem Variationsbarcode ein einzigartiges Barcode-Feld hinzufügen (typischerweise `_sku` oder `_global_unique_id`, abhängig von Ihrer Konfiguration).

  </AccordionItem>
