Variantenprodukte in WooCommerce sind Produkte mit mehreren Optionen, wie verschiedene Größen, Farben oder Materialien. WCPOS bietet mehrere Möglichkeiten, effizient mit Variantenprodukten zu arbeiten.

## Identifizierung von Variantenprodukten

In der Produktübersicht werden Variantenprodukte erkannt durch:

- **Pfeil-Schaltfläche** (<Icon name="circle-arrow-right" color="green" />) anstelle der Plus-Schaltfläche (<Icon name="circle-plus" color="green" />) für einfache Produkte
- **Attributoptionen** unterhalb des Produktnamens (z.B. „Farbe: Blau, Grün, Rot")
- **„Erweitern"-Link** zum Anzeigen aller Varianten in der Liste

## Hinzufügen von Variantenprodukten zum Warenkorb

### Methode 1: Schnelles Popover

Klicken Sie auf die grüne <Icon name="circle-arrow-right" color="green" /> Pfeil-Schaltfläche eines Variantenprodukts, um ein Popover mit Dropdown-Menüs für jedes Attribut zu öffnen:

1. Klicken Sie auf die <Icon name="circle-arrow-right" color="green" /> Schaltfläche
2. Wählen Sie Optionen aus den Dropdown-Menüs (z.B. Größe: Groß, Farbe: Blau)
3. Die passende Variante wird zum Warenkorb hinzugefügt

Dies ist die schnellste Methode, wenn Sie genau wissen, welche Variante Sie benötigen.

{/* Screenshot: variation-popover.png */}

### Methode 2: Inline Erweitern

Klicken Sie auf den **„Erweitern"**-Link eines Variantenprodukts, um alle Varianten direkt in der Produktliste anzuzeigen:

1. Klicken Sie unterhalb des Produktnamens auf **Erweitern**
2. Alle Varianten erscheinen als separate Zeilen unter dem Elternprodukt
3. Klicken Sie auf die <Icon name="circle-plus" color="green" /> Schaltfläche einer Variante, um sie zum Warenkorb hinzuzufügen
4. Klicken Sie auf **Zuklappen**, um die Varianten wieder auszublenden

Diese Methode ist nützlich, wenn Sie alle verfügbaren Optionen, deren Bestandsmengen und Preise sehen möchten.

{/* Screenshot: variations-expanded.png */}

### Methode 3: Barcode-Scan

Jede Produktvariante kann einen eigenen eindeutigen Barcode haben. Beim Scannen eines Varianten-Barcodes:

- Wenn genau eine Variante übereinstimmt, wird sie direkt zum Warenkorb hinzugefügt
- Wenn mehrere Varianten den gleichen Barcode haben, zeigt die Suchleiste übereinstimmende Ergebnisse

Weitere Details finden Sie unter [Barcode-Scanning](/pos/product-panel/barcode-scanning).

## Varianteninformationen

Jede Variante kann Folgendes anzeigen:

- **Varianten-Attribute** - Die spezifischen Optionen für diese Variante (z.B. „Größe: Groß, Farbe: Blau")
- **SKU** - Eindeutiger Bezeichner für die Variante (kann vom Elternprodukt abweichen)
- **Bestandsmenge** - Individueller Bestandsstand für die Variante
- **Preis** - Varianten können unterschiedliche Preise zum Elternprodukt oder untereinander haben

## Varianten im Warenkorb

Wenn eine Variante zum Warenkorb hinzugefügt wird, werden angezeigt:

- **Produktname** des Elternprodukts
- **Varianten-Attribute** unterhalb des Namens
- **Preis** für diese spezifische Variante

Sie können den Zeilenposten wie jedes andere Produkt im Warenkorb bearbeiten.

## Tipps für effiziente Variantenauswahl

### Barcode-Scanning verwenden

Wenn Ihre Varianten individuelle Barcodes haben (empfohlen), ist das Scannen der schnellste Weg, um sie zum Warenkorb hinzuzufügen.

### Anzeigeoptionen konfigurieren

Aktivieren Sie in den Produktübersichts-[Anzeigeeinstellungen](/pos/product-panel#display-settings) die Option **Attribute**, um Varianten-Attribute direkt in der Produktliste anzuzeigen, ohne zu erweitern.

### Tastaturnavigation

Wenn das Varianten-Popover geöffnet ist, können Sie Pfeiltasten zur Navigation und Enter zum Auswählen verwenden.

## Häufig gestellte Fragen

<details>
<summary>Warum kann ich eine bestimmte Variante nicht sehen?</summary>

Überprüfen Sie, ob die Variante:
- Veröffentlicht und nicht als Entwurf markiert ist
- Als „Auf Lager" eingestellt oder Bestandsmenge > 0 ist (wenn Bestandsverwaltung aktiviert ist)
- Alle erforderlichen Attribute definiert hat

Stellen Sie außerdem sicher, dass Sie die Produktliste nicht mit dem „Auf Lager"-Filter gefiltert haben, wenn die Variante ausverkauft ist.

</details>

<details>
<summary>Kann ich eine Variante hinzufügen, ohne alle Attribute auszuwählen?</summary>

Nein, Sie müssen für jedes Attribut eine Option auswählen, um eine spezifische Variante zu identifizieren. Wenn ein Produkt Größen- und Farbattribute hat, müssen Sie beide auswählen, bevor Sie zum Warenkorb hinzufügen.

</details>

<details>
<summary>Wie füge ich Barcodes zu einzelnen Varianten hinzu?</summary>

Bearbeiten Sie in WooCommerce das Variantenprodukt und erweitern Sie jede Variante. Sie können einen eindeutigen Barcode zum Barcode-Feld jeder Variante hinzufügen (typischerweise `_sku` oder `_global_unique_id` abhängig von Ihrer Konfiguration).

</details>
