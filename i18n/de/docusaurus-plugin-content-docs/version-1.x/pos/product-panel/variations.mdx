Variable Produkte in WooCommerce sind Produkte mit mehreren Optionen, wie unterschiedlichen Größen, Farben oder Materialien. WCPOS bietet mehrere Möglichkeiten, effizient mit variablen Produkten zu arbeiten.

## Variable Produkte erkennen {#identifying-variable-products}

Im Produktbereich werden variable Produkte gekennzeichnet durch:

- **Pfeil-Schaltfläche** (<Icon name="circle-arrow-right" color="green" />) anstelle der Plus-Schaltfläche (<Icon name="circle-plus" color="green" />) für einfache Produkte
- **Attributoptionen**, die unter dem Produktnamen angezeigt werden (z. B. "Farbe: Blau, Grün, Rot")
- **Link "Erweitern"**, um alle Varianten inline anzuzeigen

## Variable Produkte zum Warenkorb hinzufügen {#adding-variable-products-to-cart}

### Methode 1: Schnelles Popover {#method-1-quick-popover}

Klicken Sie auf die grüne <Icon name="circle-arrow-right" color="green" /> Pfeil-Schaltfläche bei einem variablen Produkt, um ein Popover mit Dropdown-Menüs für jedes Attribut zu öffnen:

1. Klicken Sie auf die <Icon name="circle-arrow-right" color="green" /> Schaltfläche
2. Wählen Sie Optionen aus jedem Dropdown-Menü aus (z. B. Größe: Groß, Farbe: Blau)
3. Die passende Variante wird dem Warenkorb hinzugefügt

Dies ist die schnellste Methode, wenn genau bekannt ist, welche Variante benötigt wird.

{/* Screenshot: variation-popover.png */}

### Methode 2: Inline erweitern {#method-2-expand-inline}

Klicken Sie auf den Link **"Erweitern"** bei einem variablen Produkt, um alle Varianten direkt in der Produktliste anzuzeigen:

1. Klicken Sie unter dem Produktnamen auf **Erweitern**
2. Alle Varianten werden als separate Zeilen unter dem übergeordneten Produkt angezeigt
3. Klicken Sie auf die <Icon name="circle-plus" color="green" /> Schaltfläche bei einer beliebigen Variante, um sie dem Warenkorb hinzuzufügen
4. Klicken Sie auf **Einklappen**, um die Varianten wieder auszublenden

Diese Methode ist nützlich, wenn alle verfügbaren Optionen, ihre jeweiligen Lagerbestände und Preise angezeigt werden sollen.

{/* Screenshot: variations-expanded.png */}

### Methode 3: Barcode-Scannen {#method-3-barcode-scanning}

Jede Produktvariante kann einen eigenen eindeutigen Barcode haben. Wenn der Barcode einer Variante gescannt wird:

- Wenn genau eine Variante übereinstimmt, wird sie direkt dem Warenkorb hinzugefügt
- Wenn mehrere Varianten denselben Barcode verwenden, zeigt die Suchleiste passende Ergebnisse an

Weitere Details finden Sie unter [Barcode-Scannen](/pos/product-panel/barcode-scanning).

## Informationen zu Varianten {#variation-information}

Jede Variante kann Folgendes anzeigen:

- **Variantenattribute** - Die spezifischen Optionen für diese Variante (z. B. "Größe: Groß, Farbe: Blau")
- **SKU** - Eindeutige Kennung für die Variante (kann sich vom übergeordneten Produkt unterscheiden)
- **Lagerbestand** - Individueller Lagerbestand für die Variante
- **Preis** - Varianten können andere Preise als das übergeordnete Produkt oder andere Varianten haben

## Varianten im Warenkorb {#variations-in-the-cart}

Wenn eine Variante zum Warenkorb hinzugefügt wird, zeigt sie Folgendes an:

- **Produktname** des übergeordneten Produkts
- **Variantenattribute** unter dem Namen
- **Preis** für diese bestimmte Variante

Sie können die Position wie jedes andere Produkt im Warenkorb bearbeiten.

## Tipps für eine effiziente Variantenauswahl {#tips-for-efficient-variation-selection}

### Barcode-Scannen verwenden {#use-barcode-scanning}

Wenn Ihre Varianten individuelle Barcodes haben (empfohlen), ist das Scannen der schnellste Weg, sie zum Warenkorb hinzuzufügen.

### Anzeigeoptionen konfigurieren {#configure-display-options}

Aktivieren Sie im Produktbereich unter [Anzeigeeinstellungen](/pos/product-panel#display-settings) die Option **Attribute**, um Variantenattribute direkt in der Produktliste anzuzeigen, ohne sie zu erweitern.

### Tastaturnavigation {#keyboard-navigation}

Wenn das Varianten-Popover geöffnet ist, können Sie mit den Pfeiltasten der Tastatur durch die Optionen navigieren und mit Enter auswählen.

## FAQ {#faq}

<Accordion>
  <AccordionItem question="Warum wird eine bestimmte Variante nicht angezeigt?">

Prüfen Sie, ob die Variante:
- Veröffentlicht ist und sich nicht im Entwurfsstatus befindet
- Auf „Vorrätig“ gesetzt ist oder eine Lagerbestandsmenge > 0 hat (wenn die Lagerverwaltung aktiviert ist)
- Alle erforderlichen Attribute sind definiert

Stellen Sie außerdem sicher, dass Sie das Produkt nicht mit dem Filter „Auf Lager“ herausgefiltert haben, wenn die Variante nicht vorrätig ist.

  </AccordionItem>

  <AccordionItem question="Wie füge ich einzelnen Varianten Barcodes hinzu?">

Bearbeiten Sie in WooCommerce das variable Produkt und erweitern Sie jede Variante. Sie können jeder Variante im Barcode-Feld einen eindeutigen Barcode hinzufügen (typischerweise `_sku` oder `_global_unique_id`, abhängig von Ihrer Konfiguration).

  </AccordionItem>
</Accordion>
