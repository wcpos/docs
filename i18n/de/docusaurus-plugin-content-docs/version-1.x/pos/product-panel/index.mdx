Der Produktbereich befindet sich auf der linken Seite des POS-Bildschirms und dient zum Suchen, Durchsuchen und Auswählen von Produkten für den Warenkorb.

{/* Screenshot: pos-product-panel.png */}

## Schnittstellenübersicht

### Suchleiste

Am oberen Rand des Bereichs ermöglicht die Suchleiste das schnelle Finden von Produkten durch:

- **Produktname** - Geben Sie einen Teil des Produktnamens ein
- **SKU** - Suche nach Lagerhaltungseinheit
- **Barcode** - Barcode scannen oder Nummer eingeben

Weitere Informationen finden Sie unter [Suche & Filterung](/pos/product-panel/search-filtering).

### Anzeigeeinstellungen

Klicken Sie auf das **Schieberegler-Symbol** (<Icon name="sliders" />) neben der Suchleiste, um die Anzeigeeinstellungen zu öffnen. Hier können Sie anpassen, welche Spalten und Informationen in der Produktliste angezeigt werden.

### Filterschaltflächen

Unterhalb der Suchleiste befinden sich Filterschaltflächen zur Eingrenzung der Produkte:

- **Vorrätig** - Nur Produkte mit verfügbarem Bestand anzeigen
- **Hervorgehoben** - Produkte anzeigen, die in WooCommerce als hervorgehoben markiert sind
- **Im Angebot** - Aktuell im Angebot befindliche Produkte anzeigen
- **Kategorie** - Nach Produktkategorie filtern
- **Tag** - Nach Produkttag filtern
- **Marke** - Nach Marke filtern (bei Verwendung eines Marken-Plugins)

Aktive Filter werden hervorgehoben. Klicken Sie erneut auf einen Filter, um ihn zu entfernen.

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Liste. Jede Zeile enthält:

- **Produktbild** - Vorschaubild des Produkts
- **Produktname** - Mit optionalen Details (Bestand, SKU, Kategorien usw.)
- **Preis** - Aktueller Verkaufspreis (mit Sonderpreisen, wenn zutreffend)
- **Aktionsschaltfläche** - In den Warenkorb legen (<Icon name="circle-plus" color="green" />) oder Variation auswählen (<Icon name="circle-arrow-right" color="green" />)

### Produkttypen

- **Einfache Produkte** zeigen eine grüne <Icon name="circle-plus" color="green" /> Schaltfläche zum direkten Hinzufügen in den Warenkorb
- **Variable Produkte** zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil zur Öffnung der Variantenauswahl

Weitere Informationen unter [Variable Produkte](/pos/product-panel/variations).

### Fußzeile

Am unteren Rand des Produktbereichs:

- **Steuerstatuts** - Zeigt die aktuelle Steuerbemessungsgrundlage (z.B. „Steuern basierend auf: Shop-Basisadresse"). Klicken Sie, um aktive Steuersätze anzuzeigen.
- **Produktanzahl** - Zeigt, wie viele Produkte angezeigt werden (z.B. „10 von 17 angezeigt")
- **Synchronisierungsschaltfläche** - Produkte vom Server aktualisieren

## Anzeigeeinstellungen

### Ausverkaufte Produkte anzeigen

Steuert, ob Produkte, die derzeit nicht vorrätig sind, angezeigt werden.

- **Aktiviert**: Ausverkaufte Produkte bleiben sichtbar, können aber nicht in den Warenkorb gelegt werden
- **Deaktiviert**: Ausverkaufte Produkte werden aus der Liste ausgeblendet

### Spalten

Konfigurieren Sie, welche Spalten in der Produktliste erscheinen:

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|-------------|-----------------|
| **Bild** | Produktvorschaubild | - |
| **Produkt** | Produktname und Details | Bestand, SKU, Barcode, Kategorien, Tags, Marken, Attribute, Metadaten |
| **SKU** | Lagerhaltungseinheit (separate Spalte) | - |
| **Barcode** | Produktbarcode (separate Spalte) | - |
| **Typ** | Produkttyp (einfach, variabel usw.) | - |
| **Bestand** | Aktuelle Bestandsmenge | - |
| **Wareneinsatz** | Produkteinkaufspreis | - |
| **Preis** | Verkaufspreis | Steuerinformationen, Angebots-Indikator |
| **Aktionen** | In den Warenkorb-Schaltfläche | - |

### Metadaten-Schlüssel

Geben Sie Produkt-Metadaten-Schlüssel an, die bei der Warenkorb-Hinzufügung auf Auftragszeilen übertragen werden sollen.

Geben Sie eine kommagetrennte Liste von Metaschlüsseln ein:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Anwendungsfälle:**
- Pfand-Plugins
- Benutzerdefinierte Produktfelder
- Daten von Drittanbieter-Integrationen
- Compliance-/Tracking-Informationen

:::note
Metaschlüssel sind Groß-/Kleinschreibung-sensitiv und müssen exakt mit den in den Produktmetadaten gespeicherten Schlüsseln übereinstimmen.
:::

### Standardeinstellungen wiederherstellen

Klicken Sie hier, um alle Anzeigeeinstellungen auf die ursprünglichen Standardwerte zurückzusetzen.

## Produkte in den Warenkorb legen

### Einfache Produkte

Klicken Sie auf die grüne <Icon name="circle-plus" color="green" /> Schaltfläche, um ein einfaches Produkt in den Warenkorb zu legen. Jeder Klick fügt eine weitere Einheit hinzu.

### Variable Produkte

Variable Produkte (z.B. ein T-Shirt mit Größen- und Farboptionen) zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil. Klicken Sie, um:

1. **Schnell-Popover** - Variante aus einem Dropdown auswählen
2. **Inline erweitern** - Auf „Erweitern" klicken, um alle Varianten in der Liste zu sehen

Weitere Informationen unter [Variable Produkte](/pos/product-panel/variations).

### Barcode-Scanning

Verbinden Sie einen USB- oder Bluetooth-Barcodescanner, um Produkte schnell hinzuzufügen. Beim Scannen eines Barcodes sucht WCPOS automatisch nach dem passenden Produkt und fügt es hinzu.

Anleitungen zur Einrichtung finden Sie unter [Barcode-Scanning](/pos/product-panel/barcode-scanning).
