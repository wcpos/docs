Das Produktpanel ist die linke Seite des POS-Bildschirms, auf der Sie Produkte suchen, durchsuchen und auswählen können, um sie zum Warenkorb hinzuzufügen.

{/* Screenshot: pos-product-panel.png */}

## Übersicht der Benutzeroberfläche

### Suchleiste

Oben im Panel ermöglicht die Suchleiste eine schnelle Produktsuche durch:

- **Produktname** - Geben Sie einen Teil des Produktnamens ein
- **SKU** - Suchen Sie nach der Artikelnummer
- **Barcode** - Scannen oder geben Sie eine Barcode-Nummer ein

Siehe [Suche & Filterung](/pos/product-panel/search-filtering) für detaillierte Informationen.

### Anzeigeeinstellungen

Klicken Sie auf das **Schieberegler-Symbol** (<Icon name="sliders" />) neben der Suchleiste, um die Anzeigeeinstellungen zu öffnen. Dadurch können Sie anpassen, welche Spalten und Informationen in der Produktliste angezeigt werden.

### Filterbuttons

Unter der Suchleiste ermöglichen die Filterbuttons eine Eingrenzung der Produkte:

- **Auf Lager** - Zeigt nur Produkte mit verfügbaren Beständen an
- **Empfohlen** - Zeigt Produkte an, die in WooCommerce als empfohlen markiert sind
- **Im Angebot** - Zeigt Produkte an, die aktuell im Angebot sind
- **Kategorie** - Filter nach Produktkategorie
- **Tag** - Filter nach Produkt-Tag
- **Marke** - Filter nach Marke (bei Verwendung eines Marken-Plugins)

Aktive Filter werden hervorgehoben angezeigt. Klicken Sie erneut auf einen Filter, um ihn zu entfernen.

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Liste. Jede Zeile zeigt:

- **Produktbild** - Vorschau des Produkts
- **Produktname** - Mit optionalen Details (Bestand, SKU, Kategorien usw.)
- **Preis** - Aktueller Verkaufspreis (bei Angeboten wird der Preis angezeigt, wenn verfügbar)
- **Aktionsbutton** - Zum Warenkorb hinzufügen (<Icon name="circle-plus" color="green" />) oder Variante auswählen (<Icon name="circle-arrow-right" color="green" />)

### Produktarten

- **Einfache Produkte** zeigen einen grünen <Icon name="circle-plus" color="green" /> Button, um direkt zum Warenkorb hinzuzufügen
- **Variable Produkte** zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil, um den Variationsauswähler zu öffnen

Siehe [Variable Produkte](/pos/product-panel/variations) für weitere Details.

### Fußzeile

Am unteren Ende des Produktpanels:

- **Umsatzsteuerstatus** - Zeigt die aktuelle Berechnungsgrundlage für die Steuer an (z. B. "Steuer basiert auf: Basisadresse des Shops"). Klicken Sie, um die aktiven Steuersätze anzuzeigen.
- **Produktanzahl** - Zeigt an, wie viele Produkte angezeigt werden (z. B. "Zeige 10 von 17")
- **Synchronisieren-Button** - Aktualisieren Sie die Produkte vom Server

### Ausverkaufte Produkte anzeigen

Steuert, ob Produkte, die momentan nicht vorrätig sind, angezeigt werden.

- **Aktiviert**: Ausverkaufte Produkte bleiben sichtbar, können jedoch nicht zum Warenkorb hinzugefügt werden
- **Deaktiviert**: Ausverkaufte Produkte werden aus der Liste ausgeblendet

### Spalten

Konfigurieren Sie, welche Spalten in der Produktliste angezeigt werden:

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|-------------|-----------------|
| **Bild** | Produktvorschau | - |
| **Produkt** | Produktname und Details | Bestand, SKU, Barcode, Kategorien, Tags, Marken, Attribute, Metadaten |
| **SKU** | Artikelnummer (separate Spalte) | - |
| **Barcode** | Produktbarcode (separate Spalte) | - |
| **Typ** | Produktart (einfach, variabel usw.) | - |
| **Bestand** | Aktuelle Bestandsmenge | - |
| **Herstellungskosten** | Produktkostenpreis | - |
| **Preis** | Verkaufspreis | Steuerinformationen, Angebotsindikator |
| **Aktionen** | Button zum Warenkorb hinzufügen | - |

### Metadaten-Schlüssel

Geben Sie Produkt-Metadaten-Schlüssel an, die zu den Bestellpositionen übertragen werden sollen, wenn Produkte zum Warenkorb hinzugefügt werden.

Geben Sie eine durch Kommas getrennte Liste von Metaschlüsseln ein:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Anwendungsfälle:**
- Flaschenpfand-Plugins
- Benutzerdefinierte Produktfelder
- Daten von Drittanbieter-Integrationen
- Compliance-/Verfolgungsinformationen

:::note
Metaschlüssel sind groß- und kleinschreibungsempfindlich und müssen genau so übereinstimmen, wie sie in den Metadaten des Produkts gespeichert sind.
:::

### Standardwerte wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen auf die ursprünglichen Standardwerte zurückzusetzen.

## Produkte in den Warenkorb hinzufügen

### Einfache Produkte

Klicken Sie auf den grünen <Icon name="circle-plus" color="green" /> Button, um ein einfaches Produkt zum Warenkorb hinzuzufügen. Jeder Klick fügt eine weitere Einheit hinzu.

### Variable Produkte

Variable Produkte (z. B. ein T-Shirt mit Größen- und Farboptionen) zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil. Klicken Sie, um:

1. **Schnelles Popover** - Wählen Sie eine Variation aus einem Dropdown-Menü
2. **Inline erweitern** - Klicken Sie auf "Erweitern", um alle Variationen in der Liste zu sehen

Siehe [Variable Produkte](/pos/product-panel/variations) für weitere Details.

### Barcode-Scanning

Schließen Sie einen USB- oder Bluetooth-Barcode-Scanner an, um Produkte schnell hinzuzufügen. Wenn Sie einen Barcode scannen, sucht WCPOS automatisch nach dem passenden Produkt und fügt es hinzu.

Siehe [Barcode-Scanning](/pos/product-panel/barcode-scanning) für Einrichtungshinweise.
