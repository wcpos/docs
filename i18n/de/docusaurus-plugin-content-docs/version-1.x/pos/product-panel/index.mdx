Das Produktpanel ist die linke Seite des POS-Bildschirms, wo Sie Produkte suchen, durchsuchen und auswählen können, um sie in den Warenkorb hinzuzufügen.

{/* Screenshot: pos-product-panel.png */}

## Benutzeroberfläche Übersicht {#interface-overview}

### Suchleiste {#search-bar}

Oben im Panel ermöglicht die Suchleiste das schnelle Finden von Produkten durch:

- **Produktname** - Geben Sie einen Teil des Produktnamens ein
- **SKU** - Suchen Sie mit der Stock Keeping Unit
- **Barcode** - Scannen oder geben Sie eine Barcodenummer ein

Siehe [Suchen & Filtern](/pos/product-panel/search-filtering) für detaillierte Informationen.

### Anzeigeeinstellungen {#display-settings}

Klicken Sie auf das **Symbol mit den Reglern** (<Icon name="sliders" />) neben der Suchleiste, um die Anzeigeeinstellungen zu öffnen. Dies ermöglicht es Ihnen, anzupassen, welche Spalten und Informationen in der Produktliste angezeigt werden.

### Filtertasten {#filter-buttons}

Unter der Suchleiste ermöglichen Filtertasten das Eingrenzen der Produkte:

- **Auf Lager** - Zeigt nur Produkte mit verfügbarem Inventar an
- **Featured** - Zeigt Produkte an, die in WooCommerce als Featured gekennzeichnet sind
- **Im Sonderangebot** - Zeigt Produkte, die derzeit im Sonderangebot sind
- **Kategorie** - Filtern nach Produktkategorie
- **Tag** - Filtern nach Produkt-Tag
- **Marke** - Filtern nach Marke (wenn ein Markenplugin verwendet wird)

Aktive Filter werden hervorgehoben angezeigt. Klicken Sie erneut auf einen Filter, um ihn zu entfernen.

### Produktliste {#product-list}

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Liste an. Jede Zeile zeigt:

- **Produktbild** - Miniaturbild des Produkts
- **Produktname** - Mit optionalen Details (Lager, SKU, Kategorien usw.)
- **Preis** - Aktueller Verkaufspreis (mit Sale-Preisen, wenn zutreffend)
- **Aktionsschaltfläche** - In den Warenkorb hinzufügen (<Icon name="circle-plus" color="green" />) oder Variation auswählen (<Icon name="circle-arrow-right" color="green" />)

### Produktarten {#product-types}

- **Einfache Produkte** zeigen eine grüne <Icon name="circle-plus" color="green" /> Schaltfläche, um sie direkt in den Warenkorb hinzuzufügen
- **Variable Produkte** zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil, um den Variationsauswähler zu öffnen

Siehe [Variable Produkte](/pos/product-panel/variations) für weitere Details.

### Fußzeile {#footer}

Unten im Produktpanel:

- **Steuerstatus** - Zeigt die aktuelle Steuerberechnungsgrundlage an (z.B. "Steuer basiert auf: Shop-Adresse"). Klicken Sie, um die aktiven Steuersätze anzuzeigen.
- **Produktanzahl** - Zeigt an, wie viele Produkte angezeigt werden (z.B. "Zeige 10 von 17")
- **Sync-Schaltfläche** (<Icon name="arrow-rotate-right" />) - Aktualisiert Produkte vom Server. **Lange drücken** für zusätzliche Optionen:
  - **Synchronisieren** - Standardaktualisierung vom Server
  - **Daten löschen und neu laden** - Löscht lokale Daten und lädt alles neu

### Ausverkaufte Produkte anzeigen {#show-out-of-stock-products}

Steuert, ob derzeit nicht vorrätige Produkte angezeigt werden.

- **Aktiviert**: Ausverkaufte Produkte bleiben sichtbar, können jedoch nicht in den Warenkorb gelegt werden
- **Deaktiviert**: Ausverkaufte Produkte werden aus der Liste ausgeblendet

### Spalten {#columns}

Konfigurieren Sie, welche Spalten in der Produktliste angezeigt werden:

| Spalte   | Beschreibung                           | Anzeigeoptionen                              |
|----------|---------------------------------------|---------------------------------------------|
| **Bild** | Produktminiatur                       | -                                           |
| **Produkt** | Produktname und Details              | Lager, SKU, Barcode, Kategorien, Tags, Marken, Attribute, Metadaten |
| **SKU**  | Stock Keeping Unit (separate Spalte)  | -                                           |
| **Barcode** | Produktbarcode (separate Spalte)     | -                                           |
| **Typ**  | Produkttyp (einfach, variabel usw.)   | -                                           |
| **Lager** | Aktuelle Lageranzahl                 | -                                           |
| **Kosten der verkauften Waren** | Produktkostenpreis                  | -                                           |
| **Preis** | Verkaufspreis                        | Steuerinfo, Angebotsindikator               |
| **Aktionen** | In den Warenkorb-Schaltfläche       | -                                           |

### Metadatenschlüssel {#meta-data-keys}

Geben Sie Produkt-Metadaten-Schlüssel an, die zu den Auftragspositionen übertragen werden sollen, wenn Produkte in den Warenkorb gelegt werden.

Geben Sie eine durch Kommas getrennte Liste von Metaschlüsseln ein:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Anwendungsbeispiele:**
- Flaschenpfand-Plugins
- Benutzerdefinierte Produktfelder
- Daten von Drittanbietersystemen
- Compliance-/Tracking-Informationen

:::note
Metaschlüssel sind case-sensitive und müssen genau mit den in den Produkt-Metadaten gespeicherten übereinstimmen.
:::

### Standardwerte wiederherstellen {#restore-default-settings}

Klicken Sie, um alle Anzeigeeinstellungen auf die ursprünglichen Standardeinstellungen zurückzusetzen.

## Produkte in den Warenkorb hinzufügen {#adding-products-to-cart}

### Einfache Produkte {#simple-products}

Klicken Sie auf die grüne <Icon name="circle-plus" color="green" /> Schaltfläche, um ein einfaches Produkt in den Warenkorb zu legen. Jeder Klick fügt eine weitere Einheit hinzu.

### Variable Produkte {#variable-products}

Variable Produkte (z.B. ein T-Shirt mit Größen- und Farbauswahl) zeigen einen grünen <Icon name="circle-arrow-right" color="green" /> Pfeil. Klicken Sie, um:

1. **Schnelles Popover** - Wählen Sie eine Variation aus einem Dropdown-Menü aus
2. **Inline erweitern** - Klicken Sie auf "Erweitern", um alle Variationen in der Liste zu sehen

Siehe [Variable Produkte](/pos/product-panel/variations) für weitere Details.

### Barcode-Scanning

Schließen Sie einen USB- oder Bluetooth-Barcode-Scanner an, um Produkte schnell hinzuzufügen. Wenn Sie einen Barcode scannen, sucht WCPOS automatisch nach dem passenden Produkt und fügt es hinzu.

Siehe [Barcode-Scanning](/pos/product-panel/barcode-scanning) für Installationsanleitungen.

## Warum kann ich einige Produkte nicht sehen? {#why-cant-i-see-some-products}

Wenn Produkte in Ihrem Produktpanel fehlen, überprüfen Sie:

- **POS-Sichtbarkeit** - Produkte, die auf "Nur Online" eingestellt sind, werden im POS nicht angezeigt. Siehe [Nur POS-Produkte](/products/pos-only-products).
- **Lager Einstellungen** - Wenn "Ausverkaufte Produkte anzeigen" in den Anzeigeeinstellungen deaktiviert ist, werden ausverkaufte Artikel ausgeblendet.
- **Synchronisierungsstatus** - Neue oder kürzlich aktualisierte Produkte müssen möglicherweise synchronisiert werden. Drücken Sie lange auf die Sync-Schaltfläche und wählen Sie **"Daten löschen und neu laden"**.
- **Filter** - Überprüfen Sie, ob Sie aktive Filter (Kategorie, Tag usw.) haben, die Produkte ausblenden könnten.
