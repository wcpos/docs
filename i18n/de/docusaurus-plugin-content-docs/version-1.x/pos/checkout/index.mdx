Wenn Sie bereit sind, einen Verkauf abzuschließen, klicken Sie auf die Schaltfläche **Kasse**, um das Kassenfenster zu öffnen. Hier verarbeiten Sie die Zahlung und schließen die Bestellung ab.

{/* Screenshot: checkout-modal.png */}

## Übersicht des Kassenfensters

Das Kassenfenster zeigt:

- **Bestellnummer** - Die WooCommerce-Bestell-ID
- **Zu zahlender Betrag** - Gesamtbetrag
- Schaltfläche **Kasseneinstellungen** - Problembehandlung bei Anzeigeissues
- **Kassierer** - Wer die Bestellung verarbeitet
- **Kunde** - Der Kunde für diese Bestellung (anklickbarer Link)
- Feld für **Gutscheincode** - Rabattcodes anwenden
- **Bestellübersicht** - Produkte, Mengen und Summen
- **Zahlungsmethoden** - Verfügbare Zahlungsoptionen
- Schaltflächen **Abbrechen / Zahlung verarbeiten**

## Zahlungsmethoden

### In der Gratis-Version verfügbar

Die kostenlose Version von WCPOS umfasst zwei Zahlungsgateways:

- **Bargeld** - Mit Barzahlungsbetrag und Wechselgeldrechner
- **Karte** - Für externe Kartenterminals

### Zusätzliche Gateways (Pro)

:::info Pro-Funktion
Zusätzliche Zahlungsgateways erfordern [WCPOS Pro](/getting-started/pro-license).
:::

Mit Pro können Sie aktivieren:

- **Stripe Terminal** - Direkte Integration mit Stripe-Kartenlesegeräten
- **SumUp Terminal** - Integration mit SumUp-Kartenlesegeräten
- **Benutzerdefinierte Gateways** - Eigene Zahlungsintegrationen erstellen

Weitere Informationen zur Konfiguration von Gateways finden Sie unter [Zahlung](/payment).

### Auswahl einer Zahlungsmethode

Klicken Sie auf eine Zahlungsmethode, um sie auszuwählen. Das Formular aktualisiert sich, um relevante Felder anzuzeigen:

**Bargeld:**
- **Erhaltener Betrag** - Geben Sie den Betrag ein, den der Kunde zahlt
- **Wechselgeld** - Automatisch berechnetes Wechselgeld

{/* Screenshot: checkout-cash-payment.png */}

**Karte:**
- Zahlung über das externe Kartenterminal verarbeiten
- Auf "Zahlung verarbeiten" klicken, um abzuschließen

## Gutscheincodes

So wenden Sie einen Rabatt an:

1. Gutscheincode ins Feld eingeben
2. Auf **Gutschein anwenden** klicken
3. Der Rabatt erscheint in der Bestellübersicht

Gutscheine müssen in WooCommerce konfiguriert werden (`WP Admin > Marketing > Gutscheine`).

## Zahlung verarbeiten

1. Zahlungsmethode auswählen
2. Bei Bargeld den erhaltenen Betrag eingeben
3. Auf **Zahlung verarbeiten** klicken
4. Die Bestellung wird abgeschlossen und der [Kassenbon](/pos/checkout/receipts) wird angezeigt

## Kasseneinstellungen (Problembehandlung)

Das Kassenfenster verwendet einen iframe/webview, um die WooCommerce-Zahlungsseite anzuzeigen. Dies nutzt die bestehende Zahlungsinfrastruktur von WooCommerce, was bedeutet, dass jedes Zahlungsgateway, das mit WooCommerce funktioniert, auch im POS funktionieren sollte.

Allerdings können Theme- und Plugin-Skripte gelegentlich stören. Klicken Sie auf **Kasseneinstellungen**, um Probleme zu behandeln:

{/* Screenshot: checkout-settings.png */}

### Alle Stile und Skripte deaktivieren

- **wp_head deaktivieren** - Entfernt alle Skripte/Stile aus der WordPress-Kopfzeile
- **wp_footer deaktivieren** - Entfernt alle Skripte/Stile aus der WordPress-Fußzeile

### Ausgewählte Stile deaktivieren

CSS selektiv deaktivieren, das Anzeigeprobleme verursachen kann:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- usw.

### Ausgewählte Skripte deaktivieren

JavaScript selektiv deaktivieren, das mit Zahlungsgateways interferieren kann:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- usw.

:::tip
Wenn ein Zahlungsgateway nicht korrekt angezeigt wird:
1. Versuchen Sie zunächst, Theme-Stile zu deaktivieren
2. Dann versuchen Sie, WooCommerce-Skripte zu deaktivieren, die nicht benötigt werden
3. Seien Sie vorsichtig, keine Skripte zu deaktivieren, die von Ihrem Zahlungsgateway benötigt werden
:::

## Abbrechen

Klicken Sie auf **Abbrechen**, um das Kassenfenster zu schließen, ohne die Bestellung abzuschließen. Die Bestellung bleibt als offener Warenkorb bestehen.
