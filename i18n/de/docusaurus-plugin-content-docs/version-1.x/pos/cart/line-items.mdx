Jedes Element im Warenkorb ist eine Position, die direkt bearbeitet werden kann. WCPOS bietet flexible Inline-Bearbeitung für Mengen, Preise und andere Details.

## Positionstypen

WooCommerce verwendet drei Typen von Positionen:

| Typ | Beschreibung | Beispiele |
|------|-------------|----------|
| **line_item** | Produkte aus Ihrem Katalog | T-Shirt, Kaffeetasse |
| **fee_line** | Zusätzliche Gebühren | Geschenkverpackung, Servicegebühr |
| **shipping_line** | Versandkosten | Standardversand, Expresslieferung |

:::note
Versandkosten werden manuell eingegeben. Das POS berechnet Versandpreise nicht automatisch.
:::

## Bearbeiten von Positionen

### Menge

Klicken Sie auf das Mengenfeld, um es direkt zu bearbeiten. Geben Sie eine neue Zahl ein oder verwenden Sie die Spinner-Steuerung.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Produktname

Für verschiedene Produkte und Gebühren können Sie den Namen direkt durch Klicken darauf bearbeiten.

### Preis

Klicken Sie auf das Preisfeld, um den Einzelpreis zu ändern. Dies ist nützlich für:

- Manuelle Rabatte anwenden
- Preisangleichung
- Korrektur von Preisfehlern

Der Gesamtbetrag aktualisiert sich automatisch basierend auf Menge × Preis.

### Drei-Punkte-Menü

Klicken Sie auf **⋮** (drei Punkte) bei einer Position, um auf zusätzliche Optionen zuzugreifen:

{/* Screenshot: cart-line-item-menu.png */}

#### Bearbeitungsformular

Öffnet ein detailliertes Bearbeitungsformular für die Position, mit dem Sie alle Felder ändern können.

#### Roh-JSON-Ansicht

Zeigt die Rohdaten im JSON-Format für die Position an. Dies ist äußerst hilfreich für:

- **Debugging** - Sehen Sie genau, welche Daten an WooCommerce gesendet werden
- **Fehlerbehebung** - Identifizieren Sie Probleme mit benutzerdefinierten Feldern oder Metadaten
- **Entwicklung** - Verstehen Sie die Datenstruktur für Integrationen

### Artikel Entfernen

Klicken Sie auf die rote **×**-Schaltfläche, um einen Artikel aus dem Warenkorb zu entfernen.

## Variationsattribute

Für variable Produkte werden die ausgewählten Variationsattribute unter dem Produktnamen angezeigt:

```
Hoodie
Colour: Green
Size: Large
```

Diese Attribute sind Teil der Positionsdaten und werden mit der Bestellung einbezogen.

## Metadaten

Wenn Sie [Metadaten-Schlüssel](/pos/product-panel#meta-data-keys) in den Einstellungen des Produkt-Panels konfiguriert haben, werden diese Daten automatisch in die Position kopiert, wenn das Produkt in den Warenkorb gelegt wird.

Sie können diese Metadaten in der Roh-JSON-Ansicht anzeigen.

## Aufteilen von Positionen

Wenn in [Anzeigeeinstellungen](/pos/cart#display-settings) aktiviert, können Sie eine Position in mehrere Zeilen aufteilen. Dies ist nützlich, wenn ein Kunde unterschiedliche Rabatte auf Teile des gleichen Produkts anwenden möchte.

## Tipps

### Schnelle Preisänderungen

Um schnell einen Rabatt zu gewähren:

1. Klicken Sie auf das Preisfeld
2. Geben Sie den neuen Preis ein
3. Drücken Sie Enter

### Tastaturnavigation

- **Tab** - Wechseln Sie zwischen bearbeitbaren Feldern
- **Enter** - Bearbeitung bestätigen
- **Escape** - Bearbeitung abbrechen

### Batch-Bearbeitungen

Für komplexe Bestellungen sollten Sie das Drei-Punkte-Menü verwenden, um auf das vollständige Bearbeitungsformular zuzugreifen, das alle Felder auf einmal anzeigt.
