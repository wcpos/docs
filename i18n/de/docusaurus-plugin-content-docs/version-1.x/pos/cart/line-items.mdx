Jedes Element im Warenkorb ist ein Posten, der direkt bearbeitet werden kann. WCPOS bietet eine flexible Inline-Bearbeitung für Mengen, Preise und andere Details.

## Postenarten

WooCommerce verwendet drei Arten von Posten:

| Typ | Beschreibung | Beispiele |
|-----|--------------|-----------|
| **line_item** | Produkte aus Ihrem Katalog | T-Shirt, Kaffeetasse |
| **fee_line** | Zusätzliche Gebühren | Geschenkverpackung, Servicegebühr |
| **shipping_line** | Versandkosten | Standardversand, Expresszustellung |

:::note
Die Versandkosten werden manuell eingegeben. Das POS berechnet die Versandkosten nicht automatisch.
:::

## Bearbeiten von Posten

### Menge

Klicken Sie auf das Mengenfeld, um es direkt zu bearbeiten. Geben Sie eine neue Zahl ein oder verwenden Sie die Spinnsteuerung.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Produktname

Bei verschiedenen Produkten und Gebühren können Sie den Namen direkt bearbeiten, indem Sie darauf klicken.

### Preis

Klicken Sie auf das Preisfeld, um den Stückpreis zu ändern. Dies ist nützlich für:

- Manuelle Rabatte anwenden
- Preisabgleich
- Preisfehler korrigieren

Der Gesamtbetrag wird automatisch basierend auf Menge × Preis aktualisiert.

### Drei-Punkte-Menü

Klicken Sie auf die **⋮** (drei Punkte) bei einem Posten, um auf zusätzliche Optionen zuzugreifen:

{/* Screenshot: cart-line-item-menu.png */}

#### Bearbeitungsformular

Öffnet ein detailliertes Bearbeitungsformular für den Posten, mit dem Sie alle Felder ändern können.

#### Roh-JSON-Ansicht

Zeigt die rohen JSON-Daten für den Posten an. Dies ist äußerst hilfreich für:

- **Debugging** - Sehen Sie genau, welche Daten an WooCommerce gesendet werden
- **Fehlerbehebung** - Probleme mit benutzerdefinierten Feldern oder Meta-Daten identifizieren
- **Entwicklung** - Die Datenstruktur für Integrationen verstehen

### Element entfernen

Klicken Sie auf die rote **×**-Schaltfläche, um ein Element aus dem Warenkorb zu entfernen.

## Variationsattribute

Für variable Produkte werden die ausgewählten Variationsattribute unter dem Produktnamen angezeigt:

```
Hoodie
Farbe: Grün
Größe: Groß
```

Diese Attribute sind Teil der Postendaten und werden in der Bestellung aufgeführt.

## Meta-Daten

Wenn Sie [Meta-Daten-Schlüssel](/pos/product-panel#meta-data-keys) in den Einstellungen des Produktpanels konfiguriert haben, werden diese Daten automatisch in den Posten kopiert, wenn das Produkt in den Warenkorb gelegt wird.

Sie können diese Meta-Daten in der Roh-JSON-Ansicht ansehen.

## Aufteilen von Posten

Wenn dies in den [Darstellungseinstellungen](/pos/cart#display-settings) aktiviert ist, können Sie einen Posten in mehrere Zeilen aufteilen. Dies ist nützlich, wenn ein Kunde unterschiedliche Rabatte auf Teile desselben Produkts anwenden möchte.

## Tipps

### Schnelle Preisanpassungen

Um schnell einen Rabatt zu gewähren:

1. Klicken Sie auf das Preisfeld
2. Geben Sie den neuen Preis ein
3. Drücken Sie Enter

### Tastaturnavigation

- **Tab** - Zwischen bearbeitbaren Feldern wechseln
- **Enter** - Bearbeitung bestätigen
- **Escape** - Bearbeitung abbrechen

### Batch-Bearbeitungen

Für komplexe Bestellungen ziehen Sie in Betracht, das Drei-Punkte-Menü zu verwenden, um auf das vollständige Bearbeitungsformular zuzugreifen, das alle Felder auf einmal anzeigt.
