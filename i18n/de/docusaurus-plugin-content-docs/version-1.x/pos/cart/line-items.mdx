Jeder Artikel im Warenkorb ist eine Position, die direkt bearbeitet werden kann. WCPOS bietet flexible Inline-Bearbeitung für Mengen, Preise und andere Details.

## Arten von Positionen

WooCommerce verwendet drei Arten von Positionen:

| Art | Beschreibung | Beispiele |
|------|-------------|----------|
| **line_item** | Produkte aus Ihrem Katalog | T-Shirt, Kaffeebecher |
| **fee_line** | Zusätzliche Gebühren | Geschenkverpackung, Servicegebühr |
| **shipping_line** | Versandkosten | Standardversand, Express-Lieferung |

:::note
Versandkosten werden manuell eingegeben. Der POS berechnet Versandtarife nicht automatisch.
:::

## Positionen bearbeiten

### Menge

Klicken Sie auf das Mengenfeld, um es direkt zu bearbeiten. Geben Sie eine neue Zahl ein oder verwenden Sie die Spin-Steuerelemente.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Produktname

Für verschiedene Produkte und Gebühren können Sie den Namen direkt bearbeiten, indem Sie darauf klicken.

### Preis

Klicken Sie auf das Preisfeld, um den Einzelpreis zu ändern. Dies ist nützlich für:

- Manuelle Rabatte
- Preisanpassung
- Korrektur von Preisfehlern

Der Gesamtbetrag wird automatisch basierend auf Menge × Preis aktualisiert.

### Drei-Punkte-Menü

Klicken Sie auf **⋮** (drei Punkte) bei einer Position, um zusätzliche Optionen zu öffnen:

{/* Screenshot: cart-line-item-menu.png */}

#### Bearbeitungsformular

Öffnet ein detailliertes Bearbeitungsformular für die Position, mit dem Sie alle Felder ändern können.

#### Rohe JSON-Ansicht

Zeigt die rohen JSON-Daten für die Position. Dies ist äußerst hilfreich für:

- **Fehlersuche** - Sehen Sie genau, welche Daten an WooCommerce gesendet werden
- **Fehlerbehebung** - Identifizieren Sie Probleme mit benutzerdefinierten Feldern oder Metadaten
- **Entwicklung** - Verstehen Sie die Datenstruktur für Integrationen

### Position entfernen

Klicken Sie auf die rote **×** Schaltfläche, um einen Artikel aus dem Warenkorb zu entfernen.

## Varianten-Attribute

Für variable Produkte werden die ausgewählten Variantenattribute unterhalb des Produktnamens angezeigt:

```
Hoodie
Farbe: Grün
Größe: Groß
```

Diese Attribute sind Teil der Positionsdaten und werden in die Bestellung aufgenommen.

## Metadaten

Wenn Sie [Metadaten-Schlüssel](/pos/product-panel#meta-data-keys) in den Produktpanel-Einstellungen konfiguriert haben, werden diese Daten automatisch in die Position kopiert, wenn das Produkt in den Warenkorb gelegt wird.

Sie können diese Metadaten in der Rohen JSON-Ansicht anzeigen.

## Positionen aufteilen

Wenn in den [Anzeigeeinstellungen](/pos/cart#display-settings) aktiviert, können Sie eine Position in mehrere Positionen aufteilen. Dies ist nützlich, wenn ein Kunde unterschiedliche Rabatte auf Teile desselben Produkts anwenden möchte.

## Tipps

### Schnelle Preisanpassungen

Um schnell einen Rabatt zu geben:

1. Klicken Sie auf das Preisfeld
2. Geben Sie den neuen Preis ein
3. Drücken Sie Enter

### Tastaturnavigation

- **Tab** - Zwischen bearbeitbaren Feldern wechseln
- **Enter** - Bearbeitung bestätigen
- **Escape** - Bearbeitung abbrechen

### Sammelbearbeitung

Für komplexe Bestellungen sollten Sie das Drei-Punkte-Menü verwenden, um das vollständige Bearbeitungsformular zu öffnen, das alle Felder gleichzeitig anzeigt.
