Jedes Element im Warenkorb ist eine Position, die direkt bearbeitet werden kann. WCPOS bietet eine flexible Inline-Bearbeitung für Mengen, Preise und andere Details.

## Positionstypen

WooCommerce verwendet drei Arten von Positionen:

| Typ | Beschreibung | Beispiele |
|-----|--------------|-----------|
| **line_item** | Produkte aus Ihrem Katalog | T-Shirt, Kaffeetasse |
| **fee_line** | Zusätzliche Gebühren | Geschenkverpackung, Servicegebühr |
| **shipping_line** | Versandkosten | Standardversand, Expresszustellung |

:::note
Die Versandkosten werden manuell eingegeben. Das POS berechnet die Versandkosten nicht automatisch.
:::

## Bearbeiten von Positionen

### Menge

Klicken Sie auf das Mengenfeld, um es direkt zu bearbeiten. Geben Sie eine neue Zahl ein oder verwenden Sie die Spinnsteuerung.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Produktname

Für verschiedene Produkte und Gebühren können Sie den Namen direkt bearbeiten, indem Sie darauf klicken.

### Preis

Klicken Sie auf das Preisfeld, um den Stückpreis zu ändern. Dies ist nützlich für:

- Manuelle Rabatte anwenden
- Preise anpassen
- Preisfehler korrigieren

Der Gesamtbetrag wird automatisch basierend auf Menge × Preis aktualisiert.

### Drei-Punkte-Menü

Klicken Sie auf die **⋮** (drei Punkte) in einer beliebigen Position, um auf zusätzliche Optionen zuzugreifen:

{/* Screenshot: cart-line-item-menu.png */}

#### Bearbeitungsformular

Öffnet ein detailliertes Bearbeitungsformular für die Position, mit dem Sie alle Felder ändern können.

#### Rohes JSON-Format

Zeigt die rohen JSON-Daten für die Position an. Dies ist äußerst hilfreich für:

- **Debugging** - Sehen Sie genau, welche Daten an WooCommerce gesendet werden
- **Fehlerbehebung** - Probleme mit benutzerdefinierten Feldern oder Metadaten identifizieren
- **Entwicklung** - Die Datenstruktur für Integrationen verstehen

### Element Entfernen

Klicken Sie auf die rote **×**-Schaltfläche, um ein Element aus dem Warenkorb zu entfernen.

## Variationsattribute

Für variable Produkte werden die ausgewählten Variationsattribute unter dem Produktnamen angezeigt:

```
Hoodie
Colour: Green
Size: Large
```

Diese Attribute sind Teil der Positiondaten und werden in der Bestellung aufgeführt.

## Metadaten

Wenn Sie [Metadaten-Schlüssel](/pos/product-panel#meta-data-keys) in den Einstellungen des Produktpanels konfiguriert haben, werden diese Daten automatisch in die Position kopiert, wenn das Produkt in den Warenkorb gelegt wird.

Sie können diese Metadaten im Roh-JSON-Format ansehen.

## Aufteilen von Positionen

Wenn dies in den [Darstellungseinstellungen](/pos/cart#display-settings) aktiviert ist, können Sie eine Position in mehrere Zeilen aufteilen. Dies ist nützlich, wenn ein Kunde unterschiedlich Rabatte auf Teile desselben Produkts anwenden möchte.

## Tipps

### Schnelle Preisänderungen

Um schnell einen Rabatt zu gewähren:

1. Klicken Sie auf das Preisfeld
2. Geben Sie den neuen Preis ein
3. Drücken Sie Enter

### Tastaturnavigation

- **Tab** - Zwischen bearbeitbaren Feldern wechseln
- **Enter** - Bearbeitung bestätigen
- **Escape** - Bearbeitung abbrechen

### Batch-Bearbeitungen

Für komplexe Bestellungen ziehen Sie in Betracht, das Drei-Punkte-Menü zu verwenden, um das vollständige Bearbeitungsformular zuzugreifen, das alle Felder auf einmal anzeigt.
