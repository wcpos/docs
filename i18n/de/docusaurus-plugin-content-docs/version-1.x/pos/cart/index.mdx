Das Warenkorb Panel befindet sich auf der rechten Seite des POS-Bildschirms, wo Sie die aktuelle Bestellung verwalten. Hier können Sie Artikel anzeigen, Mengen und Preise bearbeiten, Kunden auswählen und zur Kasse gehen.

{/* Screenshot: pos-cart-panel.png */}

## Übersicht der Benutzeroberfläche

### Kundenauswahl

Oben im Warenkorb Panel:

- **Kundenbadge** - Zeigt den aktuellen Kunden an (z. B. "Gast" oder Kundennamen)
- **Kunden hinzufügen Icon** (<Icon name="plus" />) - Erstellen Sie einen neuen Kunden
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Konfigurieren Sie die Warenkorbspalten und Optionen

Klicken Sie auf das Kundenbadge, um nach einem anderen Kunden zu suchen und ihn auszuwählen.

### Positionsartikel

Der Hauptbereich zeigt die Artikel in der aktuellen Bestellung an:

- **MENGE** - Menge (bearbeitbar)
- **Name** - Produktname mit Variationsattributen
- **Preis** - Stückpreis (bearbeitbar)
- **Gesamt** - Positionsgesamt

Siehe [Positionsartikel](/pos/cart/line-items) für Details zur Bearbeitung.

### Optionen zum Hinzufügen zum Warenkorb

Unter den Positionsartikeln können Sie spezielle Artikel hinzufügen:

- **Sonstiges Produkt hinzufügen** - Fügen Sie einen benutzerdefinierten Artikel mit manueller Preisangabe hinzu
- **Gebühr hinzufügen** - Fügen Sie eine Gebührenzeile hinzu (z. B. Geschenkverpackung, Servicegebühr)
- **Versand hinzufügen** - Fügen Sie eine Versandzeile mit manueller Kostenangabe hinzu

:::note Arten von Positionen
WooCommerce verwendet drei Arten von Positionen: `line_item` (Produkte), `fee_line` (Gebühren) und `shipping_line` (Versand). Derzeit wird nur die manuelle Eingabe für den Versand unterstützt - das POS berechnet die Versandkosten nicht automatisch.
:::

### Bestellübersicht

- **Zwischensumme** - Gesamtsumme vor Steuern und Gebühren
- **Steuer** - Berechneter Steuerbetrag (sofern zutreffend)
- **Gesamt** - Endgesamtbetrag der Bestellung

### Bestellaktionen

Unten im Warenkorb Panel:

- **Bestellnotiz** - Fügen Sie eine Notiz hinzu, die für den Kunden sichtbar ist
- **Bestellmetadaten** - Fügen Sie benutzerdefinierte Metadaten hinzu oder sehen Sie JSON an
- **Auf Server speichern** - Speichern Sie die Bestellung bei WooCommerce mit dem Status `pos-open`

Siehe [Bestellaktionen](/pos/cart/order-actions) für Details.

### Hauptschaltflächen

- **Stornieren** (rot) - Die aktuelle Bestellung stornieren/löschen
- **Kasse** (grün) - Zur Zahlung fortfahren

### Offene Bestellungen

Ganz unten wird ein Karussell mit allen offenen Bestellungen angezeigt:

- Jeder Warenkorb zeigt seine Gesamtsumme an
- Klicken Sie auf einen Warenkorb, um zu ihm zu wechseln
- Der aktuelle Warenkorb wird hervorgehoben

Siehe [Offene Bestellungen](/pos/cart/open-orders) für Details.

### Rezept automatisch nach der Kasse anzeigen

Wenn aktiviert, wird der Rezeptbildschirm automatisch nach Abschluss eines Verkaufs angezeigt.

### Rezept automatisch nach der Kasse drucken

Wenn aktiviert, wird das Rezept automatisch beim Anzeigen gedruckt. Dies spart Zeit, indem es manuelle Druckschritte reduziert.

### Schnelle Rabatte

Eine durch Kommas getrennte Liste von Rabattprozentsätzen, die als schnelle Schaltflächen erscheinen.

Wenn Sie beispielsweise `5,10,15,20` eingeben, werden vier Schaltflächen hinzugefügt, die prozentuale Rabatte auf die gesamte Bestellung mit einem einzigen Klick anwenden.

### Spalten

Konfigurieren Sie, welche Spalten im Warenkorb angezeigt werden:

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|-------------|-----------------|
| **Menge** | Menge jedes Artikels | Teilen (Erlauben Sie das Teilen von Artikeln) |
| **Name** | Produktname | SKU |
| **Preis** | Preis pro Einheit | Hinweis auf Sonderangebot |
| **Regulärer Preis** | Preis ohne Rabatt | - |
| **Zwischensumme** | Positionen-Zwischensumme ( Menge × Preis) | Steuerbetrag |
| **Gesamt** | Positionen-Gesamt | Steuerbetrag, Hinweis auf Sonderangebot |
| **Aktionen** | Schaltfläche zum Entfernen des Artikels | - |

### Standardeinstellungen wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen des Warenkorbs auf die ursprünglichen Standardwerte zurückzusetzen.

## Artikel zum Warenkorb hinzufügen

### Aus dem Produktpanel

Klicken Sie auf ein Produkt im [Produktpanel](/pos/product-panel), um es zum Warenkorb hinzuzufügen. Bei variablen Produkten wählen Sie zuerst die Variation aus.

### Sonstige Produkte

Für Artikel, die nicht in Ihrem Katalog sind:

1. Klicken Sie auf **Sonstiges Produkt hinzufügen**
2. Geben Sie einen Namen und Preis ein
3. Der Artikel wird dem Warenkorb hinzugefügt

### Gebühren

Um eine Gebühr hinzuzufügen (z. B. Geschenkverpackung):

1. Klicken Sie auf **Gebühr hinzufügen**
2. Geben Sie einen Namen und Betrag ein
3. Die Gebühr erscheint als separate Position

### Versand

Um Versand hinzuzufügen:

1. Klicken Sie auf **Versand hinzufügen**
2. Geben Sie den Namen und die Kosten der Versandmethode ein
3. Der Versand wird zur Bestellung hinzugefügt

## Bearbeiten von Warenkorbartikeln

Alle Positionen sind direkt im Warenkorb editierbar. Siehe [Positionsartikel](/pos/cart/line-items) für Details zu:

- Mengenänderung
- Preisänderung
- Rohdaten im JSON anzeigen und bearbeiten
- Artikel entfernen
