Der Warenkorb-Bereich ist die rechte Seite des POS-Bildschirms, in dem Sie die aktuelle Bestellung verwalten. Hier können Sie Artikel anzeigen, Mengen und Preise bearbeiten, Kunden auswählen und zum Checkout fortfahren.

{/* Screenshot: pos-cart-panel.png */}

## Überblick über die Oberfläche {#interface-overview}

### Kundenauswahl {#customer-selector}

Oben im Warenkorb-Bereich:

- **Kunden-Badge** - Zeigt den aktuellen Kunden an (z. B. "Gast" oder Kundenname)
- **Symbol "Kunde hinzufügen"** (<Icon name="plus" />) - Neuen Kunden erstellen
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Warenkorbspalten und Optionen konfigurieren

Klicken Sie auf das Kunden-Badge, um nach einem anderen Kunden zu suchen und ihn auszuwählen.

### Positionen {#line-items}

Der Hauptbereich zeigt Artikel in der aktuellen Bestellung an:

- **Menge** - Menge (bearbeitbar)
- **Bezeichnung** - Produktname mit Variantenattributen
- **Preis** - Stückpreis (bearbeitbar)
- **Gesamt** - Positionssumme

Weitere Informationen zur Bearbeitung finden Sie unter [Positionen](/pos/cart/line-items).

### Optionen zum Hinzufügen zum Warenkorb {#add-to-cart-options}

Unterhalb der Positionen können Sie besondere Artikel hinzufügen:

- **Sonstiges Produkt hinzufügen** - Einen benutzerdefinierten Artikel mit manueller Preiseingabe hinzufügen
- **Gebühr hinzufügen** - Fügt eine Gebührenzeile hinzu (z. B. Geschenkverpackung, Servicegebühr)
- **Versand hinzufügen** - Fügt eine Versandzeile mit manueller Kosteneingabe hinzu

:::note Typen von Zeilenpositionen
WooCommerce verwendet drei Typen von Zeilenpositionen: `line_item` (Produkte), `fee_line` (Gebühren) und `shipping_line` (Versand). Derzeit wird für den Versand nur die manuelle Eingabe unterstützt - der POS berechnet Versandkosten nicht automatisch.
:::

### Bestellübersicht {#order-summary}

- **Zwischensumme** - Gesamtbetrag vor Steuern und Gebühren
- **Steuer** - Berechneter Steuerbetrag (falls zutreffend)
- **Gesamtbetrag** - Endgültiger Bestellbetrag

### Bestellaktionen {#order-actions}

Unten im Warenkorbbereich:

- **Bestellnotiz** - Fügt eine für den Kunden sichtbare Notiz hinzu
- **Bestellmetadaten** - Benutzerdefinierte Metadaten hinzufügen oder JSON anzeigen
- **Auf Server speichern** - Die Bestellung mit dem Status `pos-open` in WooCommerce speichern

Weitere Informationen finden Sie unter [Bestellaktionen](/pos/cart/order-actions).

### Hauptschaltflächen {#main-buttons}

- **Stornieren** (rot) - Die aktuelle Bestellung stornieren/löschen
- **Kasse** (grün) - Mit der Zahlung fortfahren

### Offene Bestellungen {#open-orders}

Ganz unten zeigt ein Karussell alle offenen Bestellungen an:

- Jeder Warenkorb zeigt seine Gesamtsumme an
- Klicken Sie auf einen Warenkorb, um zu ihm zu wechseln
- Der aktuelle Warenkorb ist hervorgehoben

Weitere Informationen finden Sie unter [Offene Bestellungen](/pos/cart/open-orders).

### Beleg nach dem Checkout automatisch anzeigen {#automatically-show-receipt-after-checkout}

Wenn aktiviert, wird der Belegbildschirm nach Abschluss eines Verkaufs automatisch angezeigt.

### Beleg nach dem Checkout automatisch drucken {#automatically-print-receipt-after-checkout}

Wenn aktiviert, wird der Beleg automatisch gedruckt, sobald er angezeigt wird. Das spart Zeit, da weniger manuelle Druckschritte erforderlich sind.

### Schnellrabatte {#quick-discounts}

Eine durch Kommas getrennte Liste von Rabattprozentsätzen, die als Schnellzugriffsschaltflächen angezeigt werden.

Wenn Sie beispielsweise `5,10,15,20` eingeben, werden vier Schaltflächen hinzugefügt, mit denen prozentuale Rabatte mit einem einzigen Klick auf die gesamte Bestellung angewendet werden.

### Spalten {#columns}

Konfigurieren Sie, welche Spalten im Warenkorb angezeigt werden:

| Spalte | Beschreibung |
|--------|-------------|
| **Menge** | Menge jedes Artikels |
| **Bezeichnung** | Produktname |
| **Preis** | Preis pro Einheit |
| **Regulärer Preis** | Preis ohne Rabatt |
| **Zwischensumme** | Positionszwischensumme (Menge × Preis) |
| **Gesamt** | Positionsgesamtbetrag |
| **Aktionen** | Schaltfläche zum Entfernen des Artikels |

Einige Spalten bieten zusätzliche Anzeigeoptionen:

| Spalte | Einstellung |
|--------|--------|
| **Menge** | Aufteilen (Aufteilen von Artikeln erlauben) |
| **Bezeichnung** | SKU |
| **Preis** | Angebotskennzeichnung |
| **Zwischensumme** | Steuerbetrag |
| **Gesamt** | Steuerbetrag, Angebotskennzeichnung |

### Standardeinstellungen wiederherstellen {#restore-default-settings}

Klicken Sie hier, um alle Einstellungen für die Warenkorbanzeige auf die ursprünglichen Standardwerte zurückzusetzen.

## Artikel zum Warenkorb hinzufügen {#adding-items-to-cart}

### Über den Produktbereich {#from-product-panel}

Klicken Sie im [Produktbereich](/pos/product-panel) auf ein Produkt, um es zum Warenkorb hinzuzufügen. Wählen Sie bei variablen Produkten zuerst die Variante aus.

### Sonstige Produkte {#miscellaneous-products}

Für Artikel, die nicht in Ihrem Katalog enthalten sind:

1. Klicken Sie auf **Sonstiges Produkt hinzufügen**
2. Geben Sie einen Namen und einen Preis ein
3. Der Artikel wird dem Warenkorb hinzugefügt

### Gebühren {#fees}

So fügen Sie eine Gebühr hinzu (z. B. für Geschenkverpackung):

1. Klicken Sie auf **Gebühr hinzufügen**
2. Geben Sie einen Namen und einen Betrag ein
3. Die Gebühr wird als separate Position angezeigt

### Versand {#shipping}

So fügen Sie Versand hinzu:

1. Klicken Sie auf **Versand hinzufügen**
2. Geben Sie den Namen der Versandart und die Kosten ein
3. Der Versand wird der Bestellung hinzugefügt

## Warenkorbpositionen bearbeiten {#editing-cart-items}

Alle Positionen können direkt im Warenkorb bearbeitet werden. Details zu folgenden Themen finden Sie unter [Positionen](/pos/cart/line-items):

- Mengen bearbeiten
- Preise ändern
- JSON-Rohdaten anzeigen und bearbeiten
- Artikel entfernen
