- Öffnet einen Dialog, um eine Kundennotiz zur Bestellung hinzuzufügen.
- Kundennotizen sind für den Kunden sichtbar und erscheinen standardmäßig auf dem Beleg, im Bereich „Mein Konto“ des Kunden und in der `WP Admin > WooCommerce > Bestellungen > Bestellung` unter der Versandadresse.

:::note
Bestellnotizen sind für Kunden sichtbar.
:::

- Öffnet ein Modal, um zusätzliche Metadaten für die Bestellung zu verwalten.
- Features umfassen:
  - Ändern der Bestellwährung.
  - Hinzufügen einer Transaktions-ID (z. B. von einem externen Zahlungsterminal).
  - Hinzufügen von benutzerdefinierten Metadaten für die Bestellung.

:::note
Zusätzliche Features werden im Laufe der Zeit zu diesem Modal hinzugefügt, einschließlich der Integration mit anderen WooCommerce-Plugins.
:::

### Was sind Metadaten in WooCommerce?

Metadaten bieten zusätzliche Informationen zu Bestellungen und sind über die WooCommerce REST API zugänglich. Entwickler und Plugins verwenden diese Daten, um die Funktionalität zu erweitern, z. B. um Drittanbieter-Dienste zu integrieren.

### JSON-Ansicht

- Zeigt die rohe JSON-Darstellung der Bestellung an, nützlich für Debugging- oder Integrationszwecke.

## Auf Server speichern

- Ermöglicht das Speichern offener Warenkörbe auf dem Server, damit sie sicher gespeichert und für zukünftige Verwendungen zugänglich sind.
- Erstellt eine WooCommerce-Bestellung mit dem Status `pos-open`.

### Warum speichern?

Wenn eine neue Bestellung im POS erstellt wird, existiert sie nur lokal. 
Das Speichern der Bestellung stellt sicher, dass sie besteht, auch wenn die lokale Datenbank geleert wird.

Siehe [Offene Bestellungen](/pos/cart/open-orders) für weitere Details zur Verwaltung gespeicherter Bestellungen.

## Stornieren

- Zwei Szenarien:
  - Ungespeicherte Bestellungen:
    - Wenn die Bestellung nicht auf dem Server gespeichert wurde, wird sie dauerhaft aus der lokalen Datenbank gelöscht.
  - Gespeicherte Bestellungen:
    - Wenn die Bestellung auf dem Server gespeichert wurde, wird sie in den Papierkorb in WooCommerce verschoben und aus der lokalen Datenbank gelöscht.
- Um eine stornierte Bestellung wiederherzustellen:
  - Gehe zu `WP Admin > WooCommerce > Bestellungen > Papierkorb`.

## Kasse

- Speichert die Bestellung auf dem Server und leitet den Zahlungsprozess ein.
- Öffnet den [Kassenbildschirm](/pos/checkout), um die Zahlung abzuschließen.
