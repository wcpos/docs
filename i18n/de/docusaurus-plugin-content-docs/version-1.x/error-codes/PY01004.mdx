# PY01004: Ungültige Kartennummer {#py01004-invalid-card-number}

## Was das bedeutet {#what-this-means}

Die eingegebene Kartennummer ist nicht gültig. Kartennummern folgen bestimmten Mustern und enthalten Prüfziffern zur Validierung.

## Häufige Ursachen {#common-causes}

- **Tippfehler** — Eine oder mehrere Ziffern wurden falsch eingegeben
- **Fehlende Ziffern** — Nicht alle Ziffern wurden eingegeben
- **Zusätzliche Ziffern** — Zu viele Ziffern eingegeben
- **Falsche Karte** — Eine andere Kartennummer als die physische Karte
- **Kartenleserfehler** — Chip/Swipe hat falsche Daten gelesen

## So beheben Sie das Problem {#how-to-fix}

### 1. Geben Sie die Kartennummer erneut ein {#1-re-enter-the-card-number}

Geben Sie alle Ziffern sorgfältig ein:
- Überprüfen Sie die physische Karte
- Geben Sie jede Ziffer langsam ein
- Überprüfen Sie die vollständige Nummer vor der Übermittlung

### 2. Überprüfen Sie den Kartentyp {#2-check-card-type}

Stellen Sie sicher, dass der Kartentyp unterstützt wird:
- Visa (beginnt mit 4)
- Mastercard (beginnt mit 51-55 oder 22-27)
- Amex (beginnt mit 34 oder 37)
- Discover (beginnt mit 6011, 622, 644-649, 65)

### 3. Versuchen Sie es erneut mit dem Kartenleser {#3-try-again-with-card-reader}

Wenn Sie einen Chip/Swipe-Leser verwenden:
- Reinigen Sie den Kartenchip
- Versuchen Sie eine andere Lesemethode (Chip vs Swipe)
- Manuelle Eingabe, wenn der Leser versagt

### 4. Verwenden Sie eine andere Karte {#4-use-a-different-card}

Wenn die Kartennummer weiterhin fehlschlägt:
- Die Karte könnte beschädigt sein
- Versuchen Sie eine andere Karte
- Verwenden Sie eine alternative Zahlungsmethode

### 5. Überprüfen Sie auf Kartenschäden {#5-check-for-card-damage}

Physische Kartenprobleme:
- Zerkratzter Magnetstreifen
- Beschädigter Chip
- Abgenutzte oder verblasste Ziffern
- Karte muss möglicherweise ersetzt werden

## Formate der Kartennummern {#card-number-formats}

| Kartentyp | Länge | Beginnt mit |
|-----------|-------|--------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55, 22-27 |
| Amex | 15 | 34, 37 |
| Discover | 16 | 6011, 622, 644-649, 65 |
