# PY01001: Zahlung abgelehnt

## Was das bedeutet

Die Zahlung wurde vom Zahlungsanbieter oder Kartenemitter abgelehnt. Die Transaktion konnte nicht abgeschlossen werden.

## Häufige Ursachen

- **Unzureichende Mittel** — Nicht genug Geld auf dem Konto
- **Kartenlimit überschritten** — Transaktion überschreitet die Kartenlimits
- **Betrugsprävention** — Bank als verdächtig markiert
- **Falsche Kartendaten** — Kartennummer, CVV oder Ablaufdatum falsch
- **Kartenbeschränkungen** — Karte für diese Art von Kauf nicht aktiviert

## So beheben Sie das Problem

### 1. Kartendetails überprüfen

Überprüfen Sie, ob die eingegebenen Informationen korrekt sind:
- Kartennummer (alle Ziffern)
- Ablaufdatum
- CVV/Sicherheitscode
- Rechnungsadresse (sofern erforderlich)

### 2. Erneut versuchen

Manchmal treten vorübergehende Probleme auf:
- Warten Sie einen Moment
- Geben Sie die Kartendetails erneut ein
- Versuchen Sie die Transaktion erneut

### 3. Verwenden Sie eine andere Zahlungsmethode

Wenn die Karte weiterhin abgelehnt wird:
- Versuchen Sie eine andere Karte
- Nutzen Sie Bargeldzahlung
- Verwenden Sie eine alternative Zahlungsmethode

### 4. Kontaktieren Sie die Bank

Der Karteninhaber muss möglicherweise:
- Die Transaktion mit seiner Bank bestätigen
- Betrugsblockaden aufheben
- Bestätigen, dass die Karte aktiv ist
- Über Prüfung von Ausgabenlimits

### 5. Überprüfen Sie auf Beschränkungen

Einige Karten haben Beschränkungen:
- Internationale Transaktionen deaktiviert
- Online-/POS-Transaktionen blockiert
- Bestimmte Händlerkategorien blockiert

## Für Ladenbesitzer

Wenn Sie häufige Ablehnungen sehen:
- Überprüfen Sie, ob Ihr Händlerkonto in gutem Status ist
- Überprüfen Sie die Konfiguration des Zahlungsportals
- Überarbeiten Sie die Einstellungen zur Betrugsprävention
- Ziehen Sie in Betracht, Ihren Zahlungsanbieter zu kontaktieren
