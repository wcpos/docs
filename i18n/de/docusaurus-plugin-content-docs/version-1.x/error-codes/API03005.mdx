# API03005: Rate Limit Überschritten {#api03005-rate-limit-exceeded}

## Was das bedeutet {#what-this-means}

Sie haben in kurzer Zeit zu viele Anfragen an den Server gestellt. Die Ratenbegrenzung schützt den Server davor, überlastet zu werden.

## Häufige Ursachen {#common-causes}

- **Schnelle Sync-Vorgänge** — Mehrere Geräte synchronisieren gleichzeitig
- **Aggressives Refresh** — Daten zu häufig aktualisieren
- **Sicherheits-Plugin** — Ein Plugin erzwingt Anfragelimits
- **Hosting-Limits** — Ihr Hosting-Plan hat API-Ratenlimits

## Serverfehlerzuordnung {#server-error-mapping}

Dieser Fehlercode wird ausgelöst, wenn der Server zurückgibt:

| Server Code | Quelle |
|-------------|--------|
| HTTP 429 | Jede Serverantwort (Fallback) |

## So beheben Sie das Problem {#how-to-fix}

### 1. Warten und erneut versuchen {#1-wait-and-retry}

Ratenlimits sind vorübergehend:
- Warten Sie 1-5 Minuten
- Der POS wird automatisch erneut versuchen
- Vermeiden Sie wiederholtes manuelles Refreshen

### 2. Gleichzeitige Vorgänge reduzieren {#2-reduce-concurrent-operations}

Wenn Sie mehrere POS-Terminals verwenden:
- Staggern Sie die anfängliche Synchronisierung über die Geräte
- Vermeiden Sie das gleichzeitige Refreshen aller Geräte
- Lassen Sie eines abschließen, bevor Sie mit einem anderen beginnen

### 3. Sicherheits-Plugins prüfen {#3-check-security-plugins}

Plugins wie Wordfence können schnelle Anfragen blockieren:
- Setzen Sie die POS-Anwendung auf die Whitelist
- Setzen Sie die IPs Ihrer POS-Geräte auf die Whitelist
- Passen Sie die Schwellenwerte der Rate-Limitierung an

### 4. Hosting-Limits überprüfen {#4-review-hosting-limits}

Einige Hosts beschränken API-Anfragen:
- Überprüfen Sie die Einzelheiten Ihres Hosting-Plans
- Erwägen Sie ein Upgrade, wenn die Limits zu niedrig sind
- Kontaktieren Sie den Support, um die Limits zu erhöhen

### 5. Sync-Frequenz optimieren {#5-optimise-sync-frequency}

In den POS-Einstellungen:
- Passen Sie die Auto-Refresh-Intervalle an
- Verwenden Sie manuelle Synchronisierung, wenn sinnvoll
- Vermeiden Sie unnötige Daten-Refreshes
