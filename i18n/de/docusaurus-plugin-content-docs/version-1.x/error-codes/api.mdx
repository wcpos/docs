# API-Fehler

API-Fehler treten auf, wenn Sie mit Ihrem WooCommerce-Server kommunizieren. Diese Fehler sind mit `API` gekennzeichnet und in die folgenden Kategorien unterteilt:

## Kategorien

| Kategorie | Code-Bereich | Beschreibung |
|----------|--------------|-------------|
| [Verbindung](#connection-errors) | API01xxx | Netzwerk- und Konnektivitätsprobleme |
| [Authentifizierung](#authentication-errors) | API02xxx | Anmelden, Tokens und Berechtigungsprobleme |
| [Anfrage](#request-errors) | API03xxx | Probleme mit ausgehenden Anfragen |
| [Antwort](#response-errors) | API04xxx | Probleme mit Serverantworten |
| [Plugin/WordPress](#pluginwordpress-errors) | API05xxx | Probleme mit WordPress oder WooCommerce-Plugins |
| [Konfiguration](#configuration-errors) | API06xxx | Einrichtungs- und Konfigurationsprobleme |

---

## Verbindungsfehler

Netzwerk- und Konnektivitätsprobleme zwischen dem POS und Ihrem Server.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API01001](./API01001) | Verbindungszeitüberschreitung | Der Server hat zu lange für eine Antwort gebraucht |
| [API01002](./API01002) | Verbindung abgelehnt | Der Server hat die Verbindung abgelehnt |
| [API01003](./API01003) | Verbindung zurückgesetzt | Die Verbindung wurde unerwartet geschlossen |
| [API01004](./API01004) | DNS-Auflösung fehlgeschlagen | Die Serveradresse konnte nicht aufgelöst werden |
| [API01005](./API01005) | SSL-Zertifikatfehler | Problem mit dem Sicherheitszertifikat der Seite |
| [API01006](./API01006) | Netzwerk nicht erreichbar | Das Netzwerk kann nicht erreicht werden |
| [API01007](./API01007) | Gerät offline | Ihr Gerät ist nicht mit dem Internet verbunden |
| [API01008](./API01008) | Website nicht verfügbar | Die Website reagiert nicht |

## Authentifizierungsfehler

Probleme mit Anmeldungen, Sitzungen und Berechtigungen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API02001](./API02001) | Ungültige Anmeldedaten | Benutzername oder Passwort ist falsch |
| [API02002](./API02002) | Token abgelaufen | Ihre Sitzung ist abgelaufen |
| [API02003](./API02003) | Token ungültig | Das Authentifizierungs-Token ist nicht gültig |
| [API02004](./API02004) | Benutzer nicht autorisiert | Sie haben keine Berechtigung für diese Aktion |
| [API02005](./API02005) | Unzureichende Berechtigungen | Ihre Benutzerrolle hat nicht die erforderlichen Berechtigungen |
| [API02006](./API02006) | Ungültiger API-Schlüssel | Der WooCommerce API-Schlüssel ist nicht gültig |
| [API02007](./API02007) | Token Refresh fehlgeschlagen | Ihre Sitzung konnte nicht aktualisiert werden |
| [API02008](./API02008) | Refresh-Token ungültig | Das Refresh-Token ist nicht gültig |
| [API02009](./API02009) | Refresh-Token abgelaufen | Das Refresh-Token ist abgelaufen |
| [API02010](./API02010) | Authentifizierung erforderlich | Authentifizierung ist für diese Aktion erforderlich |

## Anfragefehler

Probleme mit den an den Server gesendeten Anfragen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API03001](./API03001) | Ungültiges Anforderungsformat | Das Anforderungsformat ist nicht korrekt |
| [API03002](./API03002) | Fehlende erforderliche Parameter | Erforderliche Daten fehlen in der Anfrage |
| [API03003](./API03003) | Ungültiger Parameterwert | Ein Parameter hat einen ungültigen Wert |
| [API03004](./API03004) | Anfrage zu groß | Die Anfrage überschreitet die Größenbeschränkungen |
| [API03005](./API03005) | Rate-Limit überschritten | Zu viele Anfragen innerhalb kurzer Zeit |
| [API03006](./API03006) | Nicht unterstützte Methode | Die HTTP-Methode wird nicht unterstützt |
| [API03007](./API03007) | Anfragewarteschlange voll | Zu viele ausstehende Anfragen |

## Antwortfehler

Probleme mit den Antworten, die vom Server empfangen werden.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API04001](./API04001) | Ungültiges Antwortformat | Das Antwortformat des Servers ist nicht gültig |
| [API04002](./API04002) | Unerwarteter Antwortcode | Unerwarteter HTTP-Statuscode empfangen |
| [API04003](./API04003) | Fehlerhaftes JSON-Antwort | Die JSON-Antwort ist beschädigt oder ungültig |
| [API04004](./API04004) | Fehlende Antwortdaten | Erwartete Daten fehlen in der Antwort |
| [API04005](./API04005) | JSON-Wiederherstellung versucht | Versuch, sich von fehlerhaftem JSON zu erholen |
| [API04006](./API04006) | Ressource nicht gefunden | Die angeforderte Ressource existiert nicht (404) |

## Plugin/WordPress-Fehler

Probleme mit WordPress, WooCommerce oder dem WCPOS-Plugin.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API05001](./API05001) | WooCommerce API deaktiviert | Die WooCommerce REST API ist deaktiviert |
| [API05002](./API05002) | WCPOS-Plugin nicht gefunden | Das WCPOS-Plugin ist nicht installiert |
| [API05003](./API05003) | WCPOS-Plugin veraltet | Das WCPOS-Plugin muss aktualisiert werden |
| [API05004](./API05004) | WordPress API deaktiviert | Die WordPress REST API ist deaktiviert |
| [API05005](./API05005) | Plugin nicht gefunden | Ein erforderliches Plugin ist nicht installiert |

Einrichtungs- und Konfigurationsprobleme.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API06001](./API06001) | Ungültiges URL-Format | Das URL-Format ist nicht gültig |
| [API06002](./API06002) | Fehlende API-URL | Es wurde keine API-URL konfiguriert |
| [API06003](./API06003) | Ungültige Site-Konfiguration | Die Site-Konfiguration ist inkorrekt |
