# API-Fehler

API-Fehler treten bei der Kommunikation mit Ihrem WooCommerce-Server auf. Diese Fehler sind mit `API` gekennzeichnet und in folgende Kategorien unterteilt:

## Kategorien

| Kategorie | Code-Bereich | Beschreibung |
|----------|------------|-------------|
| [Verbindung](#verbindungsfehler) | API01xxx | Netzwerk- und Konnektivitätsprobleme |
| [Authentifizierung](#authentifizierungsfehler) | API02xxx | Anmelde-, Token- und Berechtigungsprobleme |
| [Anfrage](#anfragefehler) | API03xxx | Probleme mit ausgehenden Anfragen |
| [Antwort](#antwortfehler) | API04xxx | Probleme mit Serverantworten |
| [Plugin/WordPress](#pluginwordpress-fehler) | API05xxx | Probleme mit WordPress oder WooCommerce-Plugins |
| [Konfiguration](#konfigurationsfehler) | API06xxx | Einrichtungs- und Konfigurationsprobleme |

---

## Verbindungsfehler

Netzwerk- und Konnektivitätsprobleme zwischen dem POS und Ihrem Server.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API01001](./API01001) | Verbindungs-Timeout | Der Server hat zu lange gebraucht, um zu antworten |
| [API01002](./API01002) | Verbindung abgelehnt | Der Server hat die Verbindung abgelehnt |
| [API01003](./API01003) | Verbindung zurückgesetzt | Die Verbindung wurde unerwartet geschlossen |
| [API01004](./API01004) | DNS-Auflösung fehlgeschlagen | Die Serveradresse konnte nicht aufgelöst werden |
| [API01005](./API01005) | SSL-Zertifikatsfehler | Problem mit dem Sicherheitszertifikat der Website |
| [API01006](./API01006) | Netzwerk nicht erreichbar | Netzwerk kann nicht erreicht werden |
| [API01007](./API01007) | Gerät offline | Ihr Gerät ist nicht mit dem Internet verbunden |
| [API01008](./API01008) | Website nicht verfügbar | Die Website antwortet nicht |

## Authentifizierungsfehler

Probleme mit Anmeldung, Sitzungen und Berechtigungen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API02001](./API02001) | Ungültige Zugangsdaten | Benutzername oder Passwort ist falsch |
| [API02002](./API02002) | Token abgelaufen | Ihre Sitzung ist abgelaufen |
| [API02003](./API02003) | Token ungültig | Der Authentifizierungs-Token ist nicht gültig |
| [API02004](./API02004) | Benutzer nicht autorisiert | Sie haben keine Berechtigung, diese Aktion durchzuführen |
| [API02005](./API02005) | Unzureichende Berechtigungen | Ihre Benutzerrolle verfügt nicht über die erforderlichen Berechtigungen |
| [API02006](./API02006) | API-Schlüssel ungültig | Der WooCommerce-API-Schlüssel ist nicht gültig |
| [API02007](./API02007) | Token-Aktualisierung fehlgeschlagen | Ihre Sitzung konnte nicht aktualisiert werden |
| [API02008](./API02008) | Refresh-Token ungültig | Der Refresh-Token ist nicht gültig |
| [API02009](./API02009) | Refresh-Token abgelaufen | Der Refresh-Token ist abgelaufen |
| [API02010](./API02010) | Authentifizierung erforderlich | Für diese Aktion ist eine Authentifizierung erforderlich |

## Anfragefehler

Probleme mit den an den Server gesendeten Anfragen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [API03001](./API03001) | Ungültiges Anforderungsformat | Das Anforderungsformat ist nicht korrekt |
| [API03002](./API03002) | Erforderliche Parameter fehlen | Erforderliche Daten fehlen in der Anfrage |
| [API03003](./API03003) | Ungültiger Parameterwert | Ein Parameter hat einen ungültigen Wert |
| [API03004](./API03004) | Anfrage zu groß | Die Anfrage überschreitet Größenlimits |
| [API03005](./API03005) | Ratenlimit überschritten | Zu viele Anfragen in kurzer Zeit |
| [API03006](./API03006) | Methode nicht unterstützt | Die HTTP-Methode wird nicht unterstützt |
| [API03007](./API03007) | Anforderungswarteschlange voll | Zu viele ausstehende Anfragen |

| Code | Name | Beschreibung |
|------|------|-------------|

| [API05003](./API05003) | WCPOS-Plugin veraltet | Das WCPOS-Plugin muss aktualisiert werden |
| [API05004](./API05004) | WordPress-API deaktiviert | Die WordPress-REST-API ist deaktiviert |
| [API05005](./API05005) | Plugin nicht gefunden | Ein erforderliches Plugin ist nicht installiert |
