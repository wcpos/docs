# API03007: Anfragenwarteschlange voll

## Was Das Bedeutet

Die POS hat zu viele ausstehende Anfragen, die gesendet werden sollen. Die interne Warteschlange ist voll und kann keine weiteren Anfragen akzeptieren, bis einige abgeschlossen sind.

## Häufige Ursachen

- **Langsames Netzwerk** — Anfragen stauen sich aufgrund einer langsamen Verbindung
- **Server nicht reaktionsfähig** — Server braucht zu lange zum Antworten
- **Schnelle Aktionen** — Viele Aktionen werden schneller ausgeführt, als sie verarbeitet werden können
- **Synchronisierungsüberlastung** — Große Synchronisierungsoperation blockiert die Warteschlange

## Wie Man Es Behebt

### 1. Warten bis Warteschlange sich leert

Geben Sie ausstehenden Anfragen Zeit zum Abschluss:
- 30-60 Sekunden warten
- Synchronisierungsindikatoren beobachten
- Keine zusätzlichen Aktionen durchführen, bis erledigt

### 2. Netzwerkverbindung überprüfen

Eine langsame Verbindung verursacht Rückstau:
- Internetgeschwindigkeit testen
- Schnellere Verbindung versuchen
- Näher an WLAN-Router gehen

### 3. Serverantwortzeit überprüfen

Wenn der Server langsam ist:
- Website in Browser testen
- Hosting kontaktieren, wenn Website langsam
- Warten bis Serverlast abnimmt

### 4. POS neu starten

Wenn die Warteschlange blockiert ist:
1. POS-Anwendung schließen
2. Einen Moment warten
3. POS wieder öffnen
4. Ausstehende Anfragen müssen möglicherweise neu initiiert werden

### 5. Schnelle Aktionen vermeiden

Während Synchronisierung oder langsamer Perioden:
- Warten, bis eine Aktion abgeschlossen ist, bevor eine neue beginnt
- Nicht wiederholt auf Schaltflächen klicken
- Geduldig bei Netzwerkoperationen sein

## Offline-Verhalten

Wenn die Warteschlange aufgrund von Offline-Bedingungen voll wird:
- Die POS speichert ausstehende Anfragen, bis Online-Status wiederhergestellt ist
- Sie können weiterhin zwischengespeicherte Produkte und Kunden durchsuchen
- Sie können neue Bestellungen starten und Artikel in den Warenkorb legen
- Bestellungen abschließen erfordert Wiederherstellung der Konnektivität
