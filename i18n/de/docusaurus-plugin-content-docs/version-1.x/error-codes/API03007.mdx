# API03007: Anfragewarteschlange Voll

## Was Das Bedeutet

Das POS hat zu viele ausstehende Anfragen, die auf den Versand warten. Die interne Warteschlange ist voll und kann keine weiteren Anfragen annehmen, bis einige abgeschlossen sind.

## Häufige Ursachen

- **Langsame Verbindung** — Anfragen stauen sich aufgrund einer langsamen Verbindung
- **Server nicht reagierend** — Server benötigt zu lange für die Antwort
- **Schnelle Aktionen** — Viele Aktionen schneller ausführen, als sie verarbeitet werden können
- **Synchronisationsüberlastung** — Große Synchronisationsoperation blockiert die Warteschlange

## So Beheben Sie Es

### 1. Warten Sie, Bis Die Warteschlange Leer Ist

Geben Sie ausstehenden Anfragen Zeit, um abzuschließen:
- Warten Sie 30-60 Sekunden
- Achten Sie darauf, dass die Synchronisationsanzeigen abgeschlossen sind
- Vermeiden Sie zusätzliche Aktionen, bis sie beendet sind

### 2. Überprüfen Sie Die Netzwerkverbindung

Eine langsame Verbindung führt zu Staus:
- Testen Sie Ihre Internetgeschwindigkeit
- Versuchen Sie eine schnellere Verbindung
- Nähern Sie sich dem WLAN-Router

### 3. Überprüfen Sie Die Serverantwortzeit

Wenn der Server langsam ist:
- Testen Sie den Zugriff auf Ihre Seite in einem Browser
- Kontaktieren Sie den Hosting-Anbieter, wenn die Seite langsam ist
- Warten Sie, bis die Serverauslastung abnimmt

### 4. Starten Sie Das POS Neu

Wenn die Warteschlange festhängt:
1. Schließen Sie die POS-Anwendung
2. Warten Sie einen Moment
3. Öffnen Sie das POS erneut
4. Ausstehende Anfragen müssen möglicherweise neu initiiert werden

### 5. Vermeiden Sie Schnelle Aktionen

Während Synchronisation oder langsamen Phasen:
- Warten Sie, bis eine Aktion abgeschlossen ist, bevor Sie eine andere starten
- Klicken Sie nicht wiederholt auf Schaltflächen
- Seien Sie geduldig mit Netzwerkoperationen

## Offline-Verhalten

Wenn die Warteschlange aufgrund von Offline-Bedingungen gefüllt ist:
- Das POS hält ausstehende Anfragen, bis die Verbindung wieder hergestellt ist
- Sie können weiterhin zwischengespeicherte Produkte und Kunden durchsuchen
- Sie können neue Bestellungen starten und Produkte zum Warenkorb hinzufügen
- Das Abschließen von Bestellungen erfordert eine wiederhergestellte Verbindung
