# Datenbankfehler

Datenbankfehler treten beim Speichern oder Abrufen von Daten in der lokalen Datenbank auf. WCPOS verwendet eine lokale Datenbank zum Speichern von Produkten, Kunden und anderen Daten für schnellen Zugriff und Offline-Funktionalität. Diese Fehler haben das Präfix `DB`.

## Kategorien

| Kategorie | Code-Bereich | Beschreibung |
|----------|------------|-------------|
| [Verbindung](#verbindungsfehler) | DB01xxx | Probleme mit Datenbankverbindung und -transaktionen |
| [Daten](#datenfehler) | DB02xxx | Datensatz- und Einschränkungsprobleme |
| [Abfrage](#abfragefehler) | DB03xxx | Probleme mit Abfragesyntax und Datentypen |

---

## Verbindungsfehler

Probleme bei der Verbindung zur oder beim Arbeiten mit der lokalen Datenbank.

| Code | Name | Beschreibung |
|------|------|-------------|
| [DB01001](./DB01001) | Verbindung fehlgeschlagen | Keine Verbindung zur lokalen Datenbank möglich |
| [DB01002](./DB01002) | Abfrage-Zeitüberschreitung | Datenbankabfrage dauerte zu lange |
| [DB01003](./DB01003) | Transaktion fehlgeschlagen | Datenbanktranskation konnte nicht abgeschlossen werden |

## Datenfehler

Probleme mit Datensätzen, Duplikaten und Einschränkungen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [DB02001](./DB02001) | Doppelter Datensatz | Ein Datensatz mit dieser ID existiert bereits |
| [DB02002](./DB02002) | Datensatz nicht gefunden | Der angeforderte Datensatz existiert nicht |
| [DB02003](./DB02003) | Einschränkungsverletzung | Daten verletzen Datenbankeinschränkungen |

## Abfragefehler

Probleme mit Abfragesyntax und Datentypen.

| Code | Name | Beschreibung |
|------|------|-------------|
| [DB03001](./DB03001) | Abfragesyntaxfehler | Die Datenbankabfrage hat Syntaxfehler |
| [DB03002](./DB03002) | Ungültiger Datentyp | Datentyp entspricht nicht dem erwarteten Typ |
| [DB03003](./DB03003) | Erforderliches Feld fehlt | Ein erforderliches Feld ist leer |
