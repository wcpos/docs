# Datenbankfehler

Datenbankfehler treten auf, wenn Daten in der lokalen Datenbank gespeichert oder abgerufen werden. WCPOS verwendet eine lokale Datenbank, um Produkte, Kunden und andere Daten für schnellen Zugriff und Offline-Funktionalität zu speichern. Diese Fehler beginnen mit `DB`.

## Kategorien

| Kategorie | Codebereich | Beschreibung |
|-----------|-------------|--------------|
| [Verbindung](#connection-errors) | DB01xxx | Probleme mit der Datenbankverbindung und Transaktionen |
| [Daten](#data-errors) | DB02xxx | Probleme mit Datensätzen und Einschränkungen |
| [Abfrage](#query-errors) | DB03xxx | Probleme mit der Abfragesyntax und Datentypen |

---

## Verbindungsfehler

Probleme beim Verbinden mit oder beim Arbeiten an der lokalen Datenbank.

| Code | Name | Beschreibung |
|------|------|--------------|
| [DB01001](./DB01001) | Verbindung Fehlgeschlagen | Konnte nicht mit der lokalen Datenbank verbinden |
| [DB01002](./DB01002) | Abfrage-Zeitüberschreitung | Datenbankabfrage dauerte zu lange |
| [DB01003](./DB01003) | Transaktion Fehlgeschlagen | Datenbanktransaktion konnte nicht abgeschlossen werden |

## Datenfehler

Probleme mit Datensätzen, Duplikaten und Einschränkungen.

| Code | Name | Beschreibung |
|------|------|--------------|
| [DB02001](./DB02001) | Duplizierter Datensatz | Ein Datensatz mit dieser ID existiert bereits |
| [DB02002](./DB02002) | Datensatz Nicht Gefunden | Der angeforderte Datensatz existiert nicht |
| [DB02003](./DB02003) | Einschränkungsverletzung | Daten verletzen Datenbankeinschränkungen |

## Abfragefehler

Probleme mit der Abfragesyntax und Datentypen.

| Code | Name | Beschreibung |
|------|------|--------------|
| [DB03001](./DB03001) | Abfragesyntaxfehler | Die Datenbankabfrage weist Syntaxfehler auf |
| [DB03002](./DB03002) | Ungültiger Datentyp | Datentyp stimmt nicht mit dem erwarteten Typ überein |
| [DB03003](./DB03003) | Fehlendes Erforderliches Feld | Ein erforderliches Feld ist leer |
