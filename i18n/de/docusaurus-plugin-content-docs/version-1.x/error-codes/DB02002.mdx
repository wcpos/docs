# DB02002: Datensatz nicht gefunden

## Was das bedeutet

Der angeforderte Datensatz existiert nicht in der lokalen Datenbank. Der POS hat versucht, auf ein Produkt, einen Kunden, eine Bestellung oder einen anderen Datensatz zuzugreifen, der nicht lokal gespeichert ist.

## Häufige Ursachen

- **Noch nicht synchronisiert** — Daten wurden noch nicht heruntergeladen
- **Auf dem Server gelöscht** — Datensatz wurde in WooCommerce entfernt
- **Synchronisierung nicht abgeschlossen** — Die erste Synchronisierung läuft noch
- **Filter angewendet** — Datensatz von aktiven Filtern ausgeschlossen

## So beheben Sie das Problem

### 1. Auf die Synchronisierung warten

Falls kürzlich gestartet:
- Warten Sie, bis die erste Synchronisierung abgeschlossen ist
- Überprüfen Sie den Fortschrittsindikator der Synchronisierung
- Der Datensatz sollte nach der Synchronisierung erscheinen

### 2. Aktualisieren/Neu synchronisieren

Ein Daten-Refresh auslösen:
- Verwenden Sie die Aktualisieren/Synchronisieren-Taste (<Icon name="arrow-rotate-right" />) - kurzer Druck für die Synchronisierung
- **Lange drücken** Sie die Synchronisierungstaste für die Optionen Löschen und Aktualisieren
- Ziehen Sie die neuesten Daten vom Server
- Warten Sie, bis die Verarbeitung abgeschlossen ist

### 3. WooCommerce überprüfen

Stellen Sie sicher, dass der Datensatz auf dem Server existiert:
- Melden Sie sich im WordPress-Admin-Dashboard an
- Überprüfen Sie Produkte, Bestellungen oder Kunden
- Bestätigen Sie, dass der Artikel noch existiert

### 4. Filter überprüfen

Der Datensatz könnte herausgefiltert sein:
- Überprüfen Sie die aktiven Filter im POS
- Überprüfen Sie die Kategoriefilter
- Überprüfen Sie die Sichtbarkeitseinstellungen

### 5. Löschen und neu synchronisieren

Falls die Synchronisierung anscheinend feststeckt:
1. Lokale Daten löschen
2. Erneut anmelden
3. Eine frische Synchronisierung durchführen

## Häufige Szenarien

- **Produkt wird nicht angezeigt:** Überprüfen, ob es veröffentlicht und sichtbar ist
- **Kunde nicht gefunden:** Könnte noch nicht synchronisiert worden sein
- **Bestellung fehlt:** Könnte außerhalb des Synchronisierungszeitraums liegen
