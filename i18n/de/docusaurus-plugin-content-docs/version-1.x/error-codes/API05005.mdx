# API05005: Plugin Nicht Gefunden {#api05005-plugin-not-found}

## Was Das Bedeutet {#what-this-means}

Ein erforderliches WordPress-Plugin ist nicht installiert oder aktiv. Dies könnte WooCommerce selbst, eine erforderliche Erweiterung oder ein Integrationsplugin sein.

## Häufige Ursachen {#common-causes}

- **WooCommerce deaktiviert** — WooCommerce wurde deaktiviert
- **Erforderliche Erweiterung fehlt** — Eine benötigte WooCommerce-Erweiterung ist nicht installiert
- **Plugin-Konflikt** — Ein anderes Plugin hat dazu geführt, dass ein erforderliches Plugin deaktiviert wurde
- **Fehlgeschlagene Aktualisierung** — Eine Plugin-Aktualisierung ist fehlgeschlagen und hat es deaktiviert zurückgelassen

## So Beheben Sie Es {#how-to-fix}

### 1. Überprüfen Sie, Ob WooCommerce Aktiv Ist {#1-verify-woocommerce-is-active}

WCPOS erfordert WooCommerce:
1. Gehen Sie zu Plugins → Installierte Plugins
2. Suchen Sie nach "WooCommerce"
3. Stellen Sie sicher, dass es aktiviert ist

### 2. Überprüfen Sie Auf Erforderliche Erweiterungen {#2-check-for-required-extensions}

Einige WCPOS-Funktionen benötigen möglicherweise Erweiterungen:
- Überprüfen Sie die Fehlermeldung auf den spezifischen Plugin-Namen
- Installieren und aktivieren Sie die erforderliche Erweiterung

### 3. Überprüfen Sie Plugin-Konflikte {#3-review-plugin-conflicts}

Wenn Plugins aufgrund von Konflikten deaktiviert wurden:
1. Überprüfen Sie Ihre E-Mails auf WordPress-Benachrichtigungen
2. Überprüfen Sie die Fehlerprotokolle
3. Lösen Sie Konflikte und reaktivieren Sie

### 4. Überprüfen Sie Die Plugin-Dateien {#4-check-plugin-files}

Wenn die Dateien eines Plugins fehlen:
1. Installieren Sie das Plugin erneut von WordPress.org
2. Oder stellen Sie aus einer Sicherungskopie wieder her
3. Aktivieren Sie das Plugin

### 5. Aktualisieren Sie Alle Plugins {#5-update-all-plugins}

Veraltete Plugins können Probleme verursachen:
1. Gehen Sie zum Dashboard → Updates
2. Aktualisieren Sie alle Plugins
3. Stellen Sie sicher, dass alle erforderlichen Plugins aktiv sind

## Erforderliche Plugins {#required-plugins}

Mindestens erfordert WCPOS:
- **WooCommerce** — Die E-Commerce-Plattform
- **WCPOS** — Das POS-Plugin

Zusätzliche Funktionen können andere Plugins benötigen.
