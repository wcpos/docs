# PY01002: Unzureichende Deckung

## Was Dies Bedeutet

Die Zahlung wurde abgelehnt, da nicht genügend Geld auf dem Konto vorhanden ist, um die Transaktion zu decken.

## Häufige Ursachen

- **Niedriger Kontostand** — Nicht genügend Mittel verfügbar
- **Ausstehende Transaktionen** — Andere ausstehende Transaktionen reduzieren den verfügbaren Saldo
- **Kreditlimit erreicht** — Bei Kreditkarten ist das Limit ausgeschöpft
- **Sperrung von Mitteln** — Gelder sind für andere Transaktionen blockiert

## Wie Sie Dies Beheben Können

### 1. Alternative Zahlungsmethode Verwenden

Die schnellste Lösung:
- Eine andere Karte versuchen
- Bargeld verwenden
- Zahlung auf mehrere Methoden aufteilen (falls unterstützt)

### 2. Bestellbetrag Reduzieren

Wenn Teilzahlung möglich ist:
- Artikel entfernen
- Rabatte oder Gutscheine anwenden
- Eine kleinere Transaktion durchführen

### 3. Kundenaktionen

Der Kunde muss möglicherweise:
- Geld auf das Konto überweisen
- Ausstehende Transaktionen abwarten
- Kreditkartensaldo begleichen
- Kreditlimiterhöhung beantragen

### 4. Transaktion Aufteilen

Wenn die Kasse Teilzahlungen unterstützt:
- Einen Teil mit einer Karte bezahlen
- Den Rest mit einer anderen Methode begleichen

## Hinweis zur Privatsphäre

Bei der Kommunikation mit Kunden:
- Diskretion bezüglich des spezifischen Ablehnungsgrunds wahren
- Einfach mitteilen, dass die Karte abgelehnt wurde
- Dem Kunden ermöglichen, privat eine andere Zahlungsmethode zu versuchen

## Verwandte Fehler

- [PY01001](./PY01001) — Zahlung Abgelehnt
- [PY01003](./PY01003) — Karte Abgelaufen
