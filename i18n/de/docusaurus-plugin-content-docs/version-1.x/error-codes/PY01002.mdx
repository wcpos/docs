# PY01002: Unzureichende Mittel {#py01002-insufficient-funds}

## Was das bedeutet {#what-this-means}

Die Zahlung wurde abgelehnt, da nicht genug Geld auf dem Konto vorhanden ist, um die Transaktion abzudecken.

## Häufige Ursachen {#common-causes}

- **Niedriger Kontostand** — Nicht genügend verfügbare Mittel
- **Ausstehende Transaktionen** — Andere ausstehende Transaktionen reduzieren das verfügbare Guthaben
- **Kreditlimit erreicht** — Bei Kreditkarten ist das Limit ausgeschöpft
- **Halten von Geldern** — Gelder sind für andere Transaktionen reserviert

## So beheben Sie es {#how-to-fix}

### 1. Verwenden Sie eine andere Zahlungsmethode {#1-use-a-different-payment-method}

Die schnellste Lösung:
- Versuchen Sie eine andere Karte
- Verwenden Sie Bargeld
- Teilen Sie die Zahlung auf mehrere Methoden auf (sofern unterstützt)

### 2. Reduzieren Sie den Bestellbetrag {#2-reduce-the-order-amount}

Wenn eine Teilzahlung möglich ist:
- Entfernen Sie einige Artikel
- Wenden Sie Rabatte oder Coupons an
- Schließen Sie eine kleinere Transaktion ab

### 3. Kundenaktionen {#3-customer-actions}

Der Kunde muss möglicherweise:
- Gelder auf das Konto überweisen
- Warten, bis ausstehende Transaktionen geklärt sind
- Den Saldo der Kreditkarte begleichen
- Eine Erhöhung des Kreditlimits beantragen

### 4. Teilen Sie die Transaktion {#4-split-the-transaction}

Wenn Ihr POS geteilte Zahlungen unterstützt:
- Bezahlen Sie einen Teil mit einer Karte
- Bezahlen Sie den Rest mit einer anderen Methode

## Datenschutzhinweis {#privacy-note}

Wenn Sie dies den Kunden mitteilen:
- Seien Sie diskret bezüglich des spezifischen Ablehnungsgrunds
- Sagen Sie einfach "Die Karte wurde abgelehnt"
- Ermöglichen Sie ihnen, eine andere Zahlungsmethode privat auszuprobieren
