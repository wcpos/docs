# API01007: Gerät Offline {#api01007-device-offline}

## Was das bedeutet {#what-this-means}

Ihr Gerät ist nicht mit dem Internet verbunden. Das POS hat erkannt, dass keine aktive Netzwerkverbindung besteht.

## Häufige Ursachen {#common-causes}

- **WiFi getrennt** — Ihr Gerät hat seine WiFi-Verbindung verloren
- **Ethernet abgezogen** — Das Netzwerkkabel ist abgezogen
- **Flugmodus aktiviert** — Alle drahtlosen Verbindungen sind deaktiviert
- **Netzwerkadapter deaktiviert** — Die Netzwerkschnittstelle ist ausgeschaltet

## So beheben Sie das Problem {#how-to-fix}

### 1. Überprüfen Sie die WiFi-Verbindung {#1-check-wifi-connection}

- Stellen Sie sicher, dass WiFi auf Ihrem Gerät aktiviert ist
- Vergewissern Sie sich, dass Sie mit dem richtigen Netzwerk verbunden sind
- Überprüfen Sie die WiFi-Signalstärke

### 2. Überprüfen Sie die physischen Verbindungen {#2-check-physical-connections}

Wenn Sie Ethernet verwenden:
- Stellen Sie sicher, dass das Kabel sicher eingesteckt ist
- Versuchen Sie ein anderes Kabel oder einen anderen Port
- Überprüfen Sie die Lichter am Netzwerkswitch/Router

### 3. Flugmodus deaktivieren {#3-disable-airplane-mode}

Überprüfen Sie, ob der Flugmodus versehentlich aktiviert ist, und deaktivieren Sie ihn.

### 4. Netzwerkadapter neu starten {#4-restart-network-adapter}

**Windows:**
- Öffnen Sie die Netzverbindungen
- Klicken Sie mit der rechten Maustaste auf Ihren Adapter → Deaktivieren, dann Aktivieren

**macOS:**
- Schalten Sie WiFi im Menü oben aus und wieder ein
- Oder: Systemeinstellungen → Netzwerk → Aus-/einschalten

### 5. Starten Sie Ihr Gerät neu {#5-restart-your-device}

Manchmal behebt ein einfacher Neustart Probleme mit der Netzwerkerkennung.

## Offline-Modus {#offline-mode}

Während des Offline-Betriebs kann WCPOS weiterhin:
- Lokal zwischengespeicherte Produkte durchsuchen und suchen
- Lokal zwischengespeicherte Kunden durchsuchen und suchen
- Neue Bestellungen starten und Artikel zum Warenkorb hinzufügen
- Früher synchronisierte Daten anzeigen

**Erfordert Konnektivität:**
- Abschließen/Checkout von Bestellungen
- Erstellung neuer Kunden

Wenn die Konnektivität wiederhergestellt ist, wird das POS automatisch die normalen Vorgänge fortsetzen.
