# PY02001: Zahlungs-Gateway-Fehler

## Was das bedeutet

Beim Kommunizieren mit dem Zahlungs-Gateway ist ein Fehler aufgetreten. Der Zahlungsabwickler konnte die Anfrage nicht abschließen.

## Häufige Ursachen

- **Gateway-Ausfall** — Der Zahlungsdienst hat Probleme
- **Konfigurationsfehler** — Gateway-Einstellungen sind falsch
- **API-Anmeldeinformationen** — Ungültige oder abgelaufene API-Schlüssel
- **Netzwerkprobleme** — Verbindungsprobleme zum Gateway

## So beheben Sie es

### 1. Versuchen Sie die Transaktion erneut

Manchmal haben Gateways vorübergehende Probleme:
- Warten Sie einen Moment
- Versuchen Sie, die Zahlung erneut zu verarbeiten
- Das Problem könnte sich von selbst lösen

### 2. Überprüfen Sie den Gateway-Status

Besuchen Sie die Statusseite Ihres Zahlungsanbieters:
- Stripe: status.stripe.com
- PayPal: Überprüfen Sie die PayPal-Community oder den Status
- Square: status.squareup.com
- Überprüfen Sie auf bekannte Ausfälle

### 3. Überprüfen Sie die Gateway-Konfiguration

Im WooCommerce-Admin:
1. Gehen Sie zu WooCommerce → Einstellungen → Zahlungen
2. Klicken Sie auf Ihr Zahlungs-Gateway
3. Überprüfen Sie, ob die API-Schlüssel korrekt sind
4. Stellen Sie sicher, dass der Test-/Live-Modus mit Ihren Schlüsseln übereinstimmt

### 4. Überprüfen Sie die API-Anmeldeinformationen

Stellen Sie sicher, dass die Anmeldeinformationen gültig sind:
- API-Schlüssel sind nicht abgelaufen
- Verwenden Sie Live-Schlüssel für Live-Transaktionen
- Verwenden Sie Testschlüssel für Testtransaktionen
- Anmeldeinformationen stimmen mit dem richtigen Konto überein

### 5. Testen Sie mit einem anderen Gateway

Wenn verfügbar:
- Versuchen Sie, mit einem alternativen Gateway zu verarbeiten
- Verwenden Sie Barzahlung als Notlösung
- Schließen Sie den Verkauf ab und versöhnen Sie später

## Für Shop-Besitzer

Häufige Konfigurationsprobleme:
- Mischen von Test- und Live-API-Schlüsseln
- API-Schlüssel von verschiedenen Konten
- Deaktivierte Zahlungsmethoden
- Abgelaufene oder widerrufene Anmeldeinformationen

Kontaktieren Sie den Support Ihres Zahlungsanbieters, wenn das Problem weiterhin besteht.
