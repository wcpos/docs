# PY01003: Karte Abgelaufen

## Was Das Bedeutet

Die Zahlungskarte hat ihr Ablaufdatum überschritten und kann nicht mehr für Transaktionen verwendet werden.

## Häufige Ursachen

- **Abgelaufene Karte** — Das Gültigkeitsdatum der Karte ist abgelaufen
- **Falsches Ablaufdatum eingegeben** — Das Ablaufdatum wurde falsch eingegeben
- **Neue Karte nicht aktiviert** — Ersatzkarte erhalten, aber noch nicht aktiviert

## So Beheben Sie Es

### 1. Überprüfen Sie Das Ablaufdatum

Schauen Sie sich die Karte an:
- Finden Sie das Ablaufdatum (MM/JJ-Format)
- Vergleichen Sie es mit dem aktuellen Datum
- Überprüfen Sie, ob das eingegebene Datum mit der Karte übereinstimmt

### 2. Verwenden Sie Eine Andere Karte

Wenn die Karte tatsächlich abgelaufen ist:
- Verwenden Sie eine andere, gültige Karte
- Verwenden Sie Bargeld oder alternative Zahlungsmethoden
- Fragen Sie, ob der Kunde eine Ersatzkarte hat

### 3. Holen Sie Die Aktualisierte Karte

Der Karteninhaber sollte:
- Die Post auf eine Ersatzkarte überprüfen
- Die neue Karte aktivieren, falls sie erhalten wurde
- Seine Bank kontaktieren, wenn keine Ersatzkarte angekommen ist

### 4. Überprüfen Sie Die Eingabe

Überprüfen Sie die Eingabe des Ablaufdatums:
- MM/JJ-Format
- Korrekter Monat und Jahr
- Keine Tippfehler

## Tipps Für Den Kundenservice

Wenn eine Karte abgelaufen ist:
- Informieren Sie den Kunden höflich
- Schlagen Sie vor, nach einer Ersatzkarte zu suchen
- Bieten Sie alternative Zahlungsmöglichkeiten an
- Seien Sie verständnisvoll — das passiert jedem

## Verwandte Fehler

- [PY01001](./PY01001) — Zahlung Abgelehnt
- [PY01004](./PY01004) — Ungültige Kartennummer
