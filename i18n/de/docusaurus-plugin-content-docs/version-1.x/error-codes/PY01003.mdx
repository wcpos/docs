# PY01003: Karte abgelaufen

## Was Dies Bedeutet

Die Zahlungskarte ist über ihr Ablaufdatum hinaus und kann nicht mehr für Transaktionen verwendet werden.

## Häufige Ursachen

- **Abgelaufene Karte** — Das Gültigkeitsdatum der Karte ist überschritten
- **Falsches Ablaufdatum eingegeben** — Ablaufdatum wurde falsch eingegeben
- **Neue Karte nicht aktiviert** — Ersatzkarte erhalten, aber noch nicht aktiviert

## Wie Man Es Behebt

### 1. Ablaufdatum überprüfen

Blick auf die Karte:
- Ablaufdatum finden (MM/JJ-Format)
- Mit aktuellem Datum vergleichen
- Eingabedatum mit Kartendatum abgleichen

### 2. Andere Zahlungsmethode verwenden

Wenn die Karte tatsächlich abgelaufen ist:
- Eine andere gültige Karte verwenden
- Bar bezahlen oder alternative Zahlungsmethode nutzen
- Fragen, ob der Kunde eine Ersatzkarte hat

### 3. Aktualisierte Karte beschaffen

Der Karteninhaber sollte:
- Post auf Ersatzkarte prüfen
- Neue Karte bei Erhalt aktivieren
- Bank kontaktieren, wenn keine Ersatzkarte angekommen ist

### 4. Eingabe überprüfen

Ablaufdatum nochmals genau prüfen:
- MM/JJ-Format
- Richtiger Monat und Jahr
- Keine Tippfehler

## Kundenservice-Tipps

Bei einer abgelaufenen Karte:
- Kunden freundlich informieren
- Vorschlagen, Ersatzkarte zu überprüfen
- Alternative Zahlungsmöglichkeiten anbieten
- Verständnisvoll sein — das passiert jedem

- [PY01004](./PY01004) — Ungültige Kartennummer
