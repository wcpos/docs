# SY02001: Ungültige Konfiguration

## Was Dies Bedeutet

Die Systemkonfiguration ist ungültig oder beschädigt. Die POS-Anwendung hat Konfigurationseinstellungen gefunden, die keinen Sinn ergeben oder unvollständig sind.

## Häufige Ursachen

- **Beschädigte Einstellungen** — Konfigurationsdatei wurde beschädigt
- **Unvollständige Einrichtung** — Einrichtungsprozess wurde nicht abgeschlossen
- **Fehler bei manueller Bearbeitung** — Konfigurationsdateien wurden falsch modifiziert
- **Versionsupgrade-Problem** — Alte Konfiguration nicht kompatibel mit neuer Version

## Problemlösung

### 1. Auf Standardeinstellungen zurücksetzen

Falls in der App verfügbar:
- Suchen Sie nach "Einstellungen zurücksetzen" oder "Standardwerte wiederherstellen"
- Dies löscht benutzerdefinierte Einstellungen, behebt aber Beschädigungen

### 2. Einrichtung abschließen

Wenn Einrichtung nicht beendet wurde:
1. Einrichtungsprozess erneut starten
2. Alle erforderlichen Schritte abschließen
3. Konfiguration vor Abschluss überprüfen

### 3. Anwendungsdaten löschen

Beschädigte Konfiguration entfernen:
- **Desktop:** App-Datenordner löschen, App neu starten
- **Browser:** Website-Daten löschen, neu laden

### 4. Anwendung neu installieren

Bei anhaltenden Problemen:
1. Komplett deinstallieren
2. Alle verbleibenden Datenordner entfernen
3. Neu installieren
4. Einrichtung von Grund auf neu durchführen

### 5. Nach Updates suchen

Konfigurationsformate können sich ändern:
- POS-Anwendung aktualisieren
- WCPOS-Plugin aktualisieren
- Nach Update neu konfigurieren

## Was zurückgesetzt wird

Beim Zurücksetzen der Konfiguration:
- Sie müssen sich erneut anmelden
- Benutzerdefinierte Einstellungen gehen verloren
- Daten müssen erneut synchronisiert werden
- Aber Ihre WooCommerce-Daten sind sicher auf dem Server
