# SY02001: Ungültige Konfiguration

## Was Das Bedeutet

Die Systemkonfiguration ist ungültig oder beschädigt. Die POS-Anwendung hat Konfigurationseinstellungen gefunden, die keinen Sinn ergeben oder unvollständig sind.

## Häufige Ursachen

- **Beschädigte Einstellungen** — Die Konfigurationsdatei wurde beschädigt
- **Unvollständige Einrichtung** — Der Einrichtungsprozess wurde nicht abgeschlossen
- **Fehler beim manuellen Bearbeiten** — Konfigurationsdateien wurden falsch bearbeitet
- **Version Upgrade Problem** — Alte Konfiguration inkompatibel mit neuer Version

## So Beheben Sie Es

### 1. Auf Standardeinstellungen Zurücksetzen

Falls in der App verfügbar:
- Suchen Sie nach "Einstellungen zurücksetzen" oder "Werkseinstellungen wiederherstellen"
- Dies löscht benutzerdefinierte Einstellungen, behebt jedoch die Beschädigung

### 2. Einrichtung Abschließen

Wenn die Einrichtung nicht abgeschlossen wurde:
1. Starten Sie den Einrichtungsprozess erneut
2. Schließen Sie alle erforderlichen Schritte ab
3. Überprüfen Sie die Konfiguration vor dem Beenden

### 3. Anwendungsdaten Löschen

Entfernen Sie die beschädigte Konfiguration:
- **Desktop:** Löschen Sie den Anwendungsdatenordner, starten Sie die App neu
- **Browser:** Löschen Sie die Website-Daten, laden Sie die Seite neu

### 4. Anwendung Neuinstallieren

Bei anhaltenden Problemen:
1. Vollständig deinstallieren
2. Alle verbleibenden Datenordner entfernen
3. Neu installieren
4. Einrichtung von Grund auf abschließen

### 5. Auf Updates Überprüfen

Konfigurationsformate können sich ändern:
- Aktualisieren Sie die POS-Anwendung
- Aktualisieren Sie das WCPOS-Plugin
- Nach dem Update neu konfigurieren

## Was Zurückgesetzt Wird

Beim Zurücksetzen der Konfiguration:
- Sie müssen sich erneut anmelden
- Benutzerdefinierte Einstellungen gehen verloren
- Daten müssen erneut synchronisiert werden
- Aber Ihre WooCommerce-Daten sind sicher auf dem Server
