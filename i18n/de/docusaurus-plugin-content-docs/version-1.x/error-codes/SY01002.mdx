# SY01002: Festplatte voll

## Was Das Bedeutet

Der Speicher Ihres Geräts ist voll. Das POS-System kann keine Daten speichern oder weiter betrieben werden, ohne verfügbaren Festplattenspeicher.

## Häufige Ursachen

- **Speicher voll** — Kein freier Platz auf dem Laufwerk
- **Große lokale Datenbank** — POS-Daten belegen erheblichen Speicherplatz
- **Temporäre Dateien** — Temporäre Dateien verbrauchen Speicherplatz
- **Andere Anwendungen** — Andere Apps füllen die Festplatte

## Behebung

### 1. Speicherplatz freigeben

Unnötige Dateien löschen:
- Papierkorb leeren
- Alte Downloads entfernen
- Ungenutzte Anwendungen löschen
- Temporäre Dateien löschen

### 2. Verfügbaren Speicherplatz überprüfen

**Windows:**
- Datei-Explorer öffnen
- Rechtsklick auf Laufwerk → Eigenschaften
- Belegten und freien Speicher anzeigen

**macOS:**
- Apple-Menü → Über diesen Mac
- Speicher anklicken

### 3. Browser-Daten löschen (Webversion)

Browser speichern Daten lokal:
- Cache und Cookies löschen
- Alte Website-Daten entfernen
- Browser-Speichernutzung überprüfen

### 4. Große Dateien verschieben oder löschen

Große Dateien finden und bearbeiten:
- Alte Videos oder Fotos
- Große Downloads
- Ungenutzte Software
- Sicherungsdateien, die auf externen Speicher verschoben werden können

### 5. POS-Daten überprüfen

Falls POS-Daten sehr groß sind:
- Die lokale Datenbank muss möglicherweise bereinigt werden
- Erwägen Sie Löschen und erneutes Synchronisieren
- Überprüfen Sie, ob alte Daten gelöscht werden können

## Empfohlener freier Speicher

Mindestens behalten:
- **10GB** für normalen Betrieb
- **20GB+** für große Kataloge
- Mehr Speicher ist immer besser

## Zukünftige Probleme vermeiden

- Regelmäßig unnötige Dateien aufräumen
- Automatische Bereinigung temporärer Dateien einrichten
- Festplattenspeichernutzung überwachen
- Bei häufiger Vollspeicherung größeren Speicher in Betracht ziehen

## Verwandte Fehler

- [SY01001](./SY01001) — Speicher voll
- [DB01001](./DB01001) — Verbindung fehlgeschlagen
