# SY01002: Festplatte voll {#sy01002-disk-full}

## Was das bedeutet {#what-this-means}

Der Speicherplatz Ihres Geräts ist voll. Der POS kann keine Daten speichern oder den Betrieb ohne verfügbaren Speicherplatz fortsetzen.

## Häufige Ursachen {#common-causes}

- **Festplatte voll** — Kein freier Speicherplatz auf der Festplatte
- **Große lokale Datenbank** — POS-Daten verwenden erheblichen Speicherplatz
- **Temporäre Dateien** — Temporäre Dateien belegen Speicherplatz
- **Andere Anwendungen** — Andere Apps füllen die Festplatte

## So beheben Sie das Problem {#how-to-fix}

### 1. Speicherplatz freigeben {#1-free-up-disk-space}

Löschen Sie unnötige Dateien:
- Leeren Sie den Papierkorb
- Entfernen Sie alte Downloads
- Löschen Sie nicht genutzte Anwendungen
- Temporäre Dateien löschen

### 2. Verfügbaren Speicherplatz überprüfen {#2-check-available-space}

**Windows:**
- Öffnen Sie den Datei-Explorer
- Klicken Sie mit der rechten Maustaste auf das Laufwerk → Eigenschaften
- Sehen Sie sich den belegten und freien Speicherplatz an

**macOS:**
- Apple-Menü → Über diesen Mac
- Klicken Sie auf Speicher

### 3. Browserdaten löschen (Web-Version) {#3-clear-browser-data-web-version}

Browser speichern Daten lokal:
- Cache und Cookies löschen
- Alte Site-Daten entfernen
- Überprüfen Sie die Speichernutzung des Browsers

### 4. Große Dateien verschieben oder löschen {#4-move-or-delete-large-files}

Finden und verwalten Sie große Dateien:
- Alte Videos oder Fotos
- Große Downloads
- Unbenutzte Software
- Backup-Dateien, die auf externen Speicher verschoben werden können

### 5. POS-Daten überprüfen {#5-check-pos-data}

Wenn die POS-Daten sehr groß sind:
- Die lokale Datenbank muss möglicherweise bereinigt werden
- Ziehen Sie in Betracht, die Daten zu löschen und erneut zu synchronisieren
- Überprüfen Sie, ob alte Daten gelöscht werden können

## Empfohlener freier Speicherplatz {#recommended-free-space}

Halten Sie mindestens:
- **10GB** frei für den normalen Betrieb
- **20GB+** für große Kataloge
- Mehr Speicherplatz ist immer besser

## Zukünftige Probleme vermeiden {#preventing-future-issues}

- Regelmäßig unnötige Dateien aufräumen
- Automatische Bereinigung temporärer Dateien einrichten
- Die Speichernutzung überwachen
- Größeren Speicher in Betracht ziehen, wenn regelmäßig voll
