# SY01003: Berechtigung verweigert {#sy01003-permission-denied}

## Was das bedeutet {#what-this-means}

Die POS-Anwendung hat keine Berechtigung, um auf eine erforderliche Ressource zuzugreifen. Dies könnte eine Datei, ein Ordner oder eine Systemfunktion sein.

## Häufige Ursachen {#common-causes}

- **Dateiberechtigungen** — Kann benötigte Dateien nicht lesen oder schreiben
- **Anwendungsberechtigungen** — Anwendung hat erforderliche Berechtigungen nicht
- **Geschützter Ordner** — Versucht, auf einen eingeschränkten Speicherort zuzugreifen
- **Sicherheitssoftware** — Antivirus blockiert den Zugriff

## So beheben Sie das {#how-to-fix}

### 1. Als Administrator ausführen (Windows) {#1-run-as-administrator-windows}

Klicken Sie mit der rechten Maustaste auf die App und wählen Sie "Als Administrator ausführen", falls für die erste Einrichtung erforderlich.

### 2. Anwendungsberechtigungen überprüfen {#2-check-application-permissions}

**macOS:**
- Systemeinstellungen → Sicherheit & Datenschutz → Datenschutz
- Überprüfen Sie, ob die App die erforderlichen Berechtigungen hat

**Windows:**
- Einstellungen → Datenschutz
- Überprüfen Sie die Berechtigungen der App

### 3. Ordnerberechtigungen überprüfen {#3-check-folder-permissions}

Stellen Sie sicher, dass die App auf ihren Datenordner zugreifen kann:
- Der Benutzer sollte das Verzeichnis für Anwendungsdaten besitzen
- Schreibberechtigungen sollten aktiviert sein
- Keine Nur-Lese-Flags auf dem Ordner

### 4. Antivirus-Einstellungen überprüfen {#4-review-antivirus-settings}

Sicherheitssoftware kann den Zugriff blockieren:
- Überprüfen Sie die Antivirus-Protokolle auf Blockierungen
- Fügen Sie die POS-App zur Whitelist hinzu
- Nehmen Sie den Ordner für Anwendungsdaten auf die Whitelist

### 5. Die Anwendung neu installieren {#5-reinstall-the-application}

Falls die Berechtigungen beschädigt sind:
1. Deinstallieren Sie die POS-Anwendung
2. Starten Sie Ihr Gerät neu
3. Installieren Sie an einem Standardort neu
4. Führen Sie mit normalen (nicht-admin) Berechtigungen aus

## Browser-Berechtigungen (Webversion) {#browser-permissions-web-version}

Wenn Sie die Webversion verwenden:
- Erlauben Sie lokalen Speicher für die Seite
- Aktivieren Sie Cookies für die Seite
- Verwenden Sie keine übermäßig restriktiven Browsereinstellungen
