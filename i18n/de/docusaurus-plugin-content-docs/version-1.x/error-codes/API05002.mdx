# API05002: WCPOS-Plugin nicht gefunden

## Was das bedeutet

Das WCPOS (WooCommerce POS) Plugin ist auf Ihrer WordPress-Seite nicht installiert oder nicht aktiv. Die POS-Anwendung benötigt dieses Plugin, um mit WooCommerce zu kommunizieren.

## Häufige Ursachen

- **Plugin nicht installiert** — Das Plugin wurde nie installiert
- **Plugin deaktiviert** — Das Plugin wurde deaktiviert
- **Plugin gelöscht** — Die Plugin-Dateien wurden entfernt
- **Falsche Seite** — Verbunden mit einer Seite ohne WCPOS

## So beheben Sie es

### 1. Installieren Sie das Plugin

Falls nicht installiert:
1. Gehen Sie zu WordPress Admin → Plugins → Neu hinzufügen
2. Suchen Sie nach "WooCommerce POS"
3. Installieren und aktivieren Sie das Plugin

Oder herunterladen von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Aktivieren Sie das Plugin

Falls installiert, aber nicht aktiv:
1. Gehen Sie zu Plugins → Installierte Plugins
2. Finden Sie "WooCommerce POS"
3. Klicken Sie auf "Aktivieren"

### 3. Überprüfen Sie auf Fehler

Falls das Plugin nicht aktiviert werden kann:
- Überprüfen Sie auf PHP-Fehler in Ihrem Fehlerprotokoll
- Stellen Sie sicher, dass WooCommerce installiert und aktiv ist
- Überprüfen Sie die Anforderungen an die PHP-Version
- Suchen Sie nach Plugin-Konflikten

### 4. Überprüfen Sie die URL

Stellen Sie sicher, dass die POS so konfiguriert ist, dass sie mit der richtigen WordPress-Seite verbunden ist:
- Überprüfen Sie die URL in den POS-Einstellungen
- Stellen Sie sicher, dass sie mit Ihrer WordPress-Installation übereinstimmt

### 5. Bei Bedarf neu installieren

Falls das Plugin beschädigt ist:
1. Deaktivieren und löschen Sie das Plugin
2. Neu installieren von WordPress.org
3. Aktivieren Sie das Plugin
