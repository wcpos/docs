# API05002: WCPOS-Plugin nicht gefunden

## Was Dies Bedeutet

Das WCPOS (WooCommerce POS) Plugin ist auf Ihrer WordPress-Seite nicht installiert oder nicht aktiv. Die POS-Anwendung benötigt dieses Plugin für die Kommunikation mit WooCommerce.

## Häufige Ursachen

- **Plugin nicht installiert** — Das Plugin wurde nie installiert
- **Plugin deaktiviert** — Das Plugin wurde deaktiviert
- **Plugin gelöscht** — Die Plugin-Dateien wurden entfernt
- **Falsche Seite** — Mit einer Seite ohne WCPOS verbunden

## Problemlösung

### 1. Plugin installieren

Wenn nicht installiert:
1. Gehen Sie zu WordPress Admin → Plugins → Neu hinzufügen
2. Suchen Sie nach "WooCommerce POS"
3. Plugin installieren und aktivieren

Oder herunterladen von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Plugin aktivieren

Wenn installiert, aber nicht aktiv:
1. Gehen Sie zu Plugins → Installierte Plugins
2. "WooCommerce POS" finden
3. Auf "Aktivieren" klicken

### 3. Nach Fehlern suchen

Wenn das Plugin nicht aktiviert werden kann:
- PHP-Fehler im Fehlerprotokoll prüfen
- Sicherstellen, dass WooCommerce installiert und aktiv ist
- PHP-Versionsanforderungen überprüfen
- Nach Plugin-Konflikten suchen

### 4. URL überprüfen

Sicherstellen, dass die POS mit der richtigen WordPress-Seite verbunden ist:
- URL in POS-Einstellungen prüfen
- Übereinstimmung mit WordPress-Installation verifizieren

### 5. Bei Bedarf neu installieren

Falls das Plugin beschädigt ist:
1. Plugin deaktivieren und löschen
2. Neu von WordPress.org installieren
3. Plugin aktivieren

## Anforderungen

WCPOS benötigt:
- WordPress (aktuelle Version)
- WooCommerce (aktuelle Version)
- PHP 7.4 oder höher
