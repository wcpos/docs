# DB01001: Verbindung fehlgeschlagen {#db01001-connection-failed}

## Was das bedeutet {#what-this-means}

Das POS konnte nicht mit seiner lokalen Datenbank verbinden. WCPOS verwendet eine lokale Datenbank, um Produkte, Kunden und andere Daten für schnellen Zugriff und Offline-Funktionalität zu speichern.

## Häufige Ursachen {#common-causes}

- **Speicher voll** — Gerät hat keinen Speicherplatz mehr
- **Datenbank beschädigt** — Lokale Datenbankdateien sind beschädigt
- **Berechtigungsproblem** — App hat keine Speicherberechtigungen
- **Browser-Speicher deaktiviert** — IndexedDB ist deaktiviert (Webversion)

## So beheben Sie das Problem {#how-to-fix}

### 1. Überprüfen Sie den Speicherplatz {#1-check-storage-space}

Stellen Sie sicher, dass Ihr Gerät über verfügbaren Speicherplatz verfügt:
- Schaffen Sie Platz, wenn der Speicher voll ist
- Löschen Sie ungenutzte Apps oder Dateien
- Überprüfen Sie den verfügbaren Speicherplatz auf der Festplatte

### 2. App-Daten löschen {#2-clear-app-data}

Setzen Sie die lokale Datenbank zurück:
- **Desktop-App:** Suchen Sie nach "Daten löschen" in den Einstellungen
- **Webbrowser:** Löschen Sie die Site-Daten für die POS-Domain
- **Hinweis:** Dies erfordert ein erneutes Synchronisieren aller Daten

### 3. Überprüfen Sie die Browsereinstellungen (Web) {#3-check-browser-settings-web}

Wenn Sie die Webversion verwenden:
1. Stellen Sie sicher, dass IndexedDB aktiviert ist
2. Verwenden Sie nicht den privaten/Inkognito-Modus (eingeschränkter Speicherplatz)
3. Lassen Sie der Seite das Speichern von Daten zu

### 4. Überprüfen Sie die Berechtigungen {#4-check-permissions}

Auf dem Desktop:
- Stellen Sie sicher, dass die App die Berechtigung hat, in ihr Datenverzeichnis zu schreiben
- Überprüfen Sie, ob Antivirus den Datenbankzugriff blockiert

### 5. Installieren Sie die App neu {#5-reinstall-the-app}

Wenn andere Lösungen fehlschlagen:
1. Deinstallieren Sie die POS-App
2. Starten Sie Ihr Gerät neu
3. Installieren Sie die App neu
4. Melden Sie sich an und synchronisieren Sie die Daten

## Datenwiederherstellung {#data-recovery}

Wenn Sie ungespeicherte Daten haben:
- Die App könnte ausstehende Änderungen haben, die nicht synchronisiert werden konnten
- Kontaktieren Sie den Support, wenn Sie Hilfe bei der Wiederherstellung von Daten benötigen
- In den meisten Fällen können die Daten vom Server erneut synchronisiert werden
