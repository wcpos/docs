# API01006: Netzwerk Nicht Erreichbar

## Was Das Bedeutet

Das POS kann keine Verbindung zum Server herstellen. Dies ist ein Problem auf Netzwerkebene – die Datenpakete können keinen Weg zu Ihrem Server finden.

## Häufige Ursachen

- **Netzwerkkonfigurationsprobleme** – Falsche Gateway- oder Routing-Einstellungen
- **ISP-Probleme** – Ihr Internetdienstanbieter hat Probleme
- **VPN-Interferenzen** – Ein VPN könnte den Verkehr falsch routen
- **Server-Netzwerkprobleme** – Das Netzwerk des Servers könnte ausgefallen sein

## So Beheben Sie Es

### 1. Überprüfen Sie Ihre Internetverbindung

- Versuchen Sie, auf andere Websites zuzugreifen
- Starten Sie Ihren Router/modem neu
- Überprüfen Sie, ob Sie mit dem Netzwerk verbunden sind

### 2. VPN Vorübergehend Deaktivieren

Wenn Sie ein VPN verwenden:
- Trennen Sie die Verbindung und versuchen Sie es erneut
- Versuchen Sie einen anderen VPN-Server
- Überprüfen Sie, ob das VPN den Zugriff auf den Standort Ihres Servers erlaubt

### 3. Kontaktieren Sie Ihren ISP

Kontaktieren Sie Ihren Internetdienstanbieter, wenn:
- Auch andere Seiten nicht erreichbar sind
- Das Problem nach dem Neustart der Netzwerkgeräte weiterhin besteht

### 4. Serverstatus Überprüfen

- Überprüfen Sie, ob Ihr Hosting-Anbieter Netzwerkprobleme meldet
- Versuchen Sie, auf Ihre Seite von einem anderen Netzwerk zuzugreifen (z.B. mobile Daten)
- Verwenden Sie Online-Tools, um zu überprüfen, ob die Seite für alle ausgefallen ist

### 5. Überprüfen Sie die Firewall-Einstellungen

Stellen Sie sicher, dass die Firewall Ihres Geräts keine ausgehenden Verbindungen zu Ihrem Server blockiert.
