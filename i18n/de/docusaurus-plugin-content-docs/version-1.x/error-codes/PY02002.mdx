# PY02002: Zahlungs-Timeout

## Was Das Bedeutet

Die Zahlungsabwicklung hat zu lange gedauert und ist im Timeout gescheitert. Das Zahlungsgateway hat nicht innerhalb der erwarteten Zeit geantwortet.

## Häufige Ursachen

- **Gateway-Überlastung** — Zahlungsanbieter erfährt hohe Auslastung
- **Netzwerkverzögerung** — Langsame Verbindung zum Zahlungsdienst
- **Verarbeitungsverzögerung** — Komplexe Betrugsprüfungen oder Verifizierung
- **Serverprobleme** — Zahlungsgateway mit Leistungsproblemen

## Wie Sie Es Beheben

### 1. Überprüfen Sie den Zahlungsstatus

**Wichtig:** Überprüfen Sie vor dem erneuten Versuch den Zahlungsstatus:
- Prüfen Sie Ihr Zahlungs-Gateway-Dashboard
- Suchen Sie die Transaktion in WooCommerce
- Verarbeiten Sie nicht erneut, wenn die Zahlung bereits erfolgt ist

### 2. Vorsichtig Erneut Versuchen

Wenn die Zahlung definitiv nicht durchgeführt wurde:
- 30-60 Sekunden warten
- Erneut versuchen
- Achten Sie auf doppelte Belastungen

### 3. Verbindung Überprüfen

Stellen Sie eine stabile Konnektivität sicher:
- Testen Sie Ihre Internetverbindung
- Versuchen Sie eine kabelgebundene Verbindung bei WLAN
- Überprüfen Sie Netzwerkprobleme

### 4. Zu Einer Weniger Beschäftigten Zeit Versuchen

Wenn das Gateway überlastet ist:
- Geschäftliche Spitzenzeiten können Verzögerungen verursachen
- Versuchen Sie es kurz darauf erneut
- Nutzen Sie eine alternative Zahlungsmethode

### 5. Zahlungsanbieter Kontaktieren

Bei anhaltenden Timeouts:
- Überprüfen Sie deren Statusseite
- Kontaktieren Sie deren Support
- Melden Sie anhaltende Probleme

## Doppelte Belastungen Vermeiden

Bei einem Timeout:
1. **Nicht sofort erneut versuchen** — Status zuerst prüfen
2. **Transaktionen überprüfen** — Gateway und WooCommerce prüfen
3. **Doppelte Belastungen zurückerstatten** — Rückerstattungen für versehentliche Duplikate durchführen
4. **Kunden informieren** — Situation bei Verzögerungen erklären

## Alternative Aktionen

Wenn Zahlung weiterhin im Timeout:
- Barzahlung akzeptieren
- Als "später bezahlen" erfassen, wenn erlaubt
- Andere Zahlungsmethode verwenden
