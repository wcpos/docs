# PY02002: Zahlungs-Timeout

## Was Das Bedeutet

Der Zahlungsprozess hat zu lange gedauert und ist abgelaufen. Das Zahlungs-Gateway hat innerhalb der erwarteten Zeit nicht reagiert.

## Häufige Ursachen

- **Gateway-Überlastung** — Der Zahlungsprozessor hat ein hohes Volumen
- **Netzwerklatenz** — Langsame Verbindung zum Zahlungsdienst
- **Verzögerung bei der Verarbeitung** — Komplexe Betrugsprüfungen oder Verifikation
- **Serverprobleme** — Zahlungs-Gateway hat Leistungsprobleme

## Wie Man Es Behebt

### 1. Überprüfen Sie, ob die Zahlung erfolgreich war

**Wichtig:** Überprüfen Sie den Zahlungsstatus, bevor Sie es erneut versuchen:
- Überprüfen Sie Ihr Zahlungs-Gateway-Dashboard
- Suchen Sie die Transaktion in WooCommerce
- Verarbeiten Sie es nicht erneut, wenn es tatsächlich erfolgreich war

### 2. Vorsichtig erneut versuchen

Wenn die Zahlung definitiv nicht durchgegangen ist:
- Warten Sie 30-60 Sekunden
- Versuchen Sie es erneut
- Achten Sie auf doppelte Gebühren

### 3. Überprüfen Sie Ihre Verbindung

Sichern Sie eine stabile Konnektivität:
- Testen Sie Ihre Internetverbindung
- Versuchen Sie eine kabelgebundene Verbindung, wenn Sie WLAN verwenden
- Überprüfen Sie auf Netzwerkprobleme

### 4. Versuchen Sie es zu einem weniger geschäftigen Zeitpunkt

Wenn das Gateway überlastet ist:
- Einzelhandels-Spitzenzeiten können Verlangsamungen verursachen
- Versuchen Sie es bald erneut
- Verwenden Sie eine alternative Zahlungsmethode

### 5. Kontaktieren Sie den Zahlungsanbieter

Wenn die Timeouts weiterhin bestehen:
- Überprüfen Sie deren Statusseite
- Kontaktieren Sie deren Support
- Melden Sie laufende Probleme

## Vermeidung doppelter Gebühren

Wenn ein Timeout auftritt:
1. **Nicht sofort erneut versuchen** — Überprüfen Sie zuerst den Status
2. **Transaktionen überprüfen** — Überprüfen Sie sowohl Gateway als auch WooCommerce
3. **Doppelte Rückerstattungen** — Bearbeiten Sie Rückerstattungen für versehentliche Duplikate
4. **Den Kunden informieren** — Erklären Sie die Situation, wenn es zu einer Verzögerung kommt

## Alternative Maßnahmen

Wenn die Zahlung weiterhin timet:
- Akzeptieren Sie Barzahlung
- Erfassen Sie es als "später bezahlen", wenn erlaubt
- Verwenden Sie eine andere Zahlungsmethode

- [PY02001](./PY02001) — Fehler beim Zahlungs-Gateway
- [API01001](./API01001) — Verbindungs-Timeout
