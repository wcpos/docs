# SY01001: Nicht genügend Speicher {#sy01001-out-of-memory}

## Was das bedeutet {#what-this-means}

Ihr Gerät hat wenig Speicher (RAM) und kann die angeforderte Operation nicht abschließen. Das POS benötigt ausreichend Speicher.

## Häufige Ursachen {#common-causes}

- **Zu viele geöffnete Apps** — Andere Anwendungen verwenden Speicher
- **Großer Datensatz** — Viele Produkte/Kunden geladen
- **Speicherleck** — Anwendung gibt Speicher nicht ordnungsgemäß frei
- **Unzureichender RAM** — Gerät hat nicht genug Speicher

## So beheben Sie das {#how-to-fix}

### 1. Andere Anwendungen schließen {#1-close-other-applications}

Speicher freigeben:
- Unbenutzte Browser-Tabs schließen
- Andere Anwendungen schließen
- Hintergrundprogramme schließen

### 2. Das POS neu starten {#2-restart-the-pos}

Den Anwendungs-Speicher aktualisieren:
1. Das POS vollständig schließen
2. Einige Sekunden warten
3. Die Anwendung erneut öffnen

### 3. Ihr Gerät neu starten {#3-restart-your-device}

Ein vollständiger Neustart löscht den gesamten Speicher:
1. Alle offenen Arbeiten speichern
2. Ihren Computer/Gerät neu starten
3. Nur das POS beim erneuten Starten öffnen

### 4. Browserdaten löschen (Webversion) {#4-clear-browser-data-web-version}

Wenn Sie es in einem Browser verwenden:
- Browser-Cache löschen
- Den Browser schließen und erneut öffnen
- Weniger Browser-Tabs verwenden

### 5. Datenmenge reduzieren {#5-reduce-data-load}

Wenn Sie einen sehr großen Katalog haben:
- Filter verwenden, um weniger Daten auf einmal zu laden
- In Betracht ziehen, Ergebnisse zu paginieren
- Den Synchronisationsumfang, wenn möglich, begrenzen

## Mindestanforderungen {#minimum-requirements}

Stellen Sie sicher, dass Ihr Gerät die Mindestanforderungen erfüllt:
- **RAM:** Mindestens 4GB, 8GB empfohlen
- **Browser:** Moderner Browser (Chrome, Firefox, Edge)
- **Desktop-App:** Überprüfen Sie die Versionshinweise für Anforderungen

## Anhaltende Probleme {#ongoing-issues}

Wenn Ihnen häufig der Speicher ausgeht:
- Erwägen Sie, den RAM Ihres Geräts aufzurüsten
- Verwenden Sie ein Gerät mit mehr Ressourcen
- Melden Sie das Problem, wenn es übermäßig erscheint
