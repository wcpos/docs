# SY02002: Dienst nicht verfügbar

## Was Das Bedeutet

Ein erforderlicher Dienst oder eine Komponente ist nicht verfügbar. Die POS hängt von verschiedenen Diensten ab, und einer von ihnen ist derzeit nicht zugänglich.

## Häufige Ursachen

- **Hintergrunddienst gestoppt** — Ein erforderlicher Dienst läuft nicht
- **Systemressourcenproblem** — Nicht genügend Ressourcen zum Starten des Dienstes
- **Abhängigkeitsfehler** — Ein abhängiger Dienst ist fehlgeschlagen
- **Startproblem** — Dienst konnte nicht ordnungsgemäß gestartet werden
- **Serverfehler** — Der WooCommerce-Server hat einen 5xx-Fehler zurückgegeben

## Server-Fehler-Zuordnung

Dieser Fehlercode wird ausgelöst, wenn der Server folgendes zurückgibt:

| Server-Code | Quelle |
|-------------|--------|
| HTTP 5xx | Jede Server-Fehlerantwort (500, 502, 503, 504, etc.) |

## Wie Sie Es Beheben Können

### 1. Anwendung Neu Starten

Schließen und Wiedereröffnen der POS:
1. Anwendung vollständig schließen
2. Einige Sekunden warten
3. Erneut starten

### 2. Gerät Neu Starten

Ein vollständiger Neustart löst oft Dienstprobleme:
1. Wichtige Arbeiten speichern
2. Computer neu starten
3. POS erneut versuchen

### 3. Systemressourcen Prüfen

Stellen Sie sicher, dass Systemressourcen verfügbar sind:
- Unnötige Anwendungen schließen
- Speichernutzung überprüfen
- Festplattenplatz prüfen

### 4. Anwendung Aktualisieren

Stellen Sie sicher, dass Sie die neueste Version haben:
- Nach verfügbaren Updates suchen
- Ausstehende Updates installieren
- Nach der Aktualisierung neu starten

### 5. Bei Bedarf Neu Installieren

Wenn der Dienst weiterhin nicht verfügbar ist:
1. Anwendung deinstallieren
2. Gerät neu starten
3. Neueste Version herunterladen
4. Frisch installieren

## Browserspezifisch (Web-Version)

Für die Web-Version:
- JavaScript aktiviert
- Prüfen, ob erforderliche Browser-Funktionen nicht blockiert sind
- Anderen Browser versuchen
- Browser-Erweiterungen deaktivieren, die möglicherweise stören

## Wann Sie Support Kontaktieren Sollten

Wenn dieser Fehler anhält:
- Notieren Sie, wann er auftritt
- Prüfen Sie, ob bestimmte Aktionen ihn auslösen
- Melden Sie ihn mit Details zur Untersuchung
