# SY02002: Dienst nicht verfügbar {#sy02002-service-unavailable}

## Was Das Bedeutet {#what-this-means}

Ein erforderlicher Dienst oder eine Komponente ist nicht verfügbar. Das POS hängt von verschiedenen Diensten ab, um zu funktionieren, und einer von ihnen ist derzeit nicht erreichbar.

## Häufige Ursachen {#common-causes}

- **Hintergrunddienst gestoppt** — Ein erforderlicher Dienst läuft nicht
- **Systemressourcenproblem** — Nicht genügend Ressourcen, um den Dienst zu starten
- **Abhängigkeitsausfall** — Ein Dienst, von dem dieser abhängt, ist ausgefallen
- **Startproblem** — Dienst konnte nicht richtig gestartet werden
- **Serverfehler** — Der WooCommerce-Server gab einen 5xx-Fehler zurück

## Serverfehler-Zuordnung {#server-error-mapping}

Dieser Fehlercode wird ausgelöst, wenn der Server zurückgibt:

| Servercode | Quelle |
|------------|--------|
| HTTP 5xx   | Jede Serverfehlerantwort (500, 502, 503, 504 usw.) |

## So Beheben Sie Es {#how-to-fix}

### 1. Die Anwendung Neu Starten {#1-restart-the-application}

Schließen Sie das POS und öffnen Sie es erneut:
1. Schließen Sie die Anwendung vollständig
2. Warten Sie ein paar Sekunden
3. Starten Sie sie erneut

### 2. Ihr Gerät Neu Starten {#2-restart-your-device}

Ein vollständiger Neustart löst oft Dienstprobleme:
1. Speichern Sie alle wichtigen Arbeiten
2. Starten Sie Ihren Computer neu
3. Versuchen Sie das POS erneut

### 3. Überprüfen Sie Die Systemressourcen {#3-check-system-resources}

Stellen Sie sicher, dass Systemressourcen verfügbar sind:
- Schließen Sie unnötige Anwendungen
- Überprüfen Sie die Speichernutzung
- Überprüfen Sie den Speicherplatz

### 4. Aktualisieren Sie Die Anwendung {#4-update-the-application}

Stellen Sie sicher, dass Sie die neueste Version verwenden:
- Überprüfen Sie verfügbare Updates
- Installieren Sie alle ausstehenden Updates
- Starten Sie nach dem Aktualisieren neu

### 5. Neuinstallieren, Falls Erforderlich {#5-reinstall-if-necessary}

Wenn der Dienst weiterhin nicht verfügbar ist:
1. Deinstallieren Sie die Anwendung
2. Starten Sie Ihr Gerät neu
3. Laden Sie die neueste Version herunter
4. Installieren Sie frisch

## Browser-Spezifisch (Web-Version) {#browser-specific-web-version}

Für die Web-Version:
- Stellen Sie sicher, dass JavaScript aktiviert ist
- Überprüfen Sie, ob erforderliche Browserfunktionen nicht blockiert sind
- Versuchen Sie einen anderen Browser
- Deaktivieren Sie Browsererweiterungen, die stören könnten

## Wann Man Den Support Kontaktieren Sollte {#when-to-contact-support}

Wenn dieser Fehler weiterhin besteht:
- Notieren Sie, wann er auftritt
- Überprüfen Sie, ob bestimmte Aktionen ihn auslösen
- Melden Sie es mit Details zur Untersuchung
