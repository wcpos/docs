:::info Pro-Funktion
[WCPOS Pro](https://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das die Verwaltungsbildschirme, zusätzliche Zahlungsgateways und weitere Pro-exklusive Funktionen in WCPOS bereitstellt.

**Ab v1.8+ ist Pro ein eigenständiges Plugin** — das kostenlose [WCPOS-Plugin](https://wordpress.org/plugins/woocommerce-pos/) muss nicht zusätzlich installiert sein.
:::

## Was ist in Pro enthalten? {#whats-included-in-pro}

WCPOS Pro bietet zusätzliche Funktionen:

- **Produktbildschirm** – Lagerbestände und Preise direkt im POS bearbeiten
- **Bestellbildschirm** – Bestellverlauf anzeigen und verwalten
- **Kundenbildschirm** – Kundeninformationen hinzufügen und bearbeiten
- **Berichtsansicht** – Tagesabschlussberichte erstellen
- **Zusätzliche Zahlungsgateways** – Stripe Terminal, SumUp und benutzerdefinierte Gateways

## Installation {#installation}

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, befolgen Sie bitte die nachstehenden Schritte zur Installation und Aktivierung des Plugins:

:::note Upgrade von älteren Versionen?
Falls zuvor sowohl das kostenlose WCPOS-Plugin als auch ein älteres Pro-Add-on installiert waren, **deaktivieren und löschen Sie das kostenlose Plugin**, bevor Sie das neue eigenständige Pro-Plugin aktivieren. Der gleichzeitige Betrieb beider Plugins kann zu Aktivierungsfehlern wie *"Invalid response fetching remote state"* oder fatalen *"Class not found"*-Fehlern führen.
:::

1. Navigieren Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Klicken Sie unter **Downloads** auf den Download-Link und speichern Sie das Plugin auf Ihrem Desktop.
3. Melden Sie sich anschließend auf Ihrer Website an und navigieren Sie zu `WP Admin > Plugins > Add Plugin > Plugin hochladen`.
4. Laden Sie die Plugin-ZIP-Datei von Ihrem Desktop hoch und aktivieren Sie das Plugin.
5. Navigieren Sie anschließend zu `WP Admin > POS > Einstellungen > License` und geben Sie Ihren Lizenzschlüssel sowie Ihre Lizenz-E-Mail-Adresse ein, um die Aktivierung abzuschließen.

![Meine Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Sie können WCPOS Pro auf Ihrer Kontoseite herunterladen")

## Aktualisierungen {#updates}

Wenn neue Versionen verfügbar sind, werden sie wie bei jedem anderen Plugin im Update-Dashboard angezeigt. Die neueste Version kann auch über die [Kontoseite](https://wcpos.com/my-account/) heruntergeladen werden.

## Fehlerbehebung {#troubleshooting}

### Aktivierungsfehler {#activation-failures}

- **„Ungültiger Lizenzschlüssel"** — Überprüfen Sie beim Einfügen auf nachgestellte Leerzeichen und bestätigen Sie, dass die E-Mail-Adresse mit der beim Kauf verwendeten übereinstimmt.
- **„Lizenzlimit erreicht"** — standardmäßig wird eine Lizenz auf **2 Domains** aktiviert (vorgesehen für eine Live-Site und eine Staging-Site). Beachten Sie, dass **Subdomains als separate Aktivierungen zählen** — `example.com`, `staging.example.com` und `dev.example.com` sind drei. Dies betrifft häufig Hoster wie Pantheon mit obligatorischen Dev-/Test-/Live-Umgebungen. Deaktivieren Sie nicht verwendete Sites über Ihre [Kontoseite](https://wcpos.com/my-account/) oder schreiben Sie an **support@wcpos.com** — zusätzliche Aktivierungen werden auf Anfrage routinemäßig gewährt.

### Abgelaufene Lizenz {#expired-licence}

- Wenn eine Lizenz abläuft, bleiben die Pro-Funktionen aktiv, es werden jedoch keine Updates mehr bereitgestellt.
- Die Verlängerung erfolgt über die [Kontoseite](https://wcpos.com/my-account/) — der bestehende Lizenzschlüssel bleibt unverändert.

### Domain-Übertragungen {#domain-transfers}

- Wenn die Website auf eine neue Domain umgezogen ist, deaktivieren Sie die Lizenz zuerst auf der alten Domain und aktivieren Sie sie dann auf der neuen.
- Wenn die alte Domain nicht mehr erreichbar ist, wenden Sie sich an **support@wcpos.com**, um die Aktivierung zurückzusetzen.

### Verbindung zum Lizenzserver {#licence-server-connectivity}

- Für die Aktivierung muss der WordPress-Server den WCPOS-Lizenzserver erreichen können.
- Wenn die Aktivierung ohne Fehlermeldung fehlschlägt, prüfen Sie, ob ausgehende HTTPS-Anfragen durch die Firewall des Hosts oder ein Sicherheits-Plugin blockiert werden.
- Einige Managed-Hosting-Anbieter blockieren standardmäßig externe API-Aufrufe — wenden Sie sich an Ihren Hoster, um `wcpos.com` auf die Whitelist zu setzen.

### Lizenzstatus prüfen {#checking-licence-status}

- Navigieren Sie zu `WP Admin > POS > Einstellungen > License`, um den aktuellen Aktivierungsstatus einzusehen.
- Auf der [Kontoseite](https://wcpos.com/my-account/) werden alle aktiven Websites und das Ablaufdatum angezeigt.

## Lizenz erwerben {#purchase-a-license}

Besuchen Sie [wcpos.com/pro](https://wcpos.com/pro), um eine Pro-Lizenz zu erwerben.

:::note Kauf und Upgrade erfolgen über die Website
Es gibt keine „Auf Pro upgraden"-Schaltfläche in der App — Käufe, Plan-Upgrades und Verlängerungen werden alle auf [wcpos.com](https://wcpos.com/pro) und der [Kontoseite](https://wcpos.com/my-account/) verwaltet. Nach dem Kauf kann Pro mit den [oben beschriebenen Schritten](#installation) installiert werden.
:::
