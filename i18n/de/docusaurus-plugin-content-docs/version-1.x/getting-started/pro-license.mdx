:::info Pro-Funktion
[WCPOS Pro](https://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das die Verwaltungsbildschirme, zusätzliche Zahlungsgateways und weitere Pro-exklusive Funktionen in WCPOS bereitstellt.

**Ab v1.8+ ist Pro ein eigenständiges Plugin** — das kostenlose [WCPOS-Plugin](https://wordpress.org/plugins/woocommerce-pos/) muss nicht zusätzlich installiert werden.
:::

## Was ist in Pro enthalten? {#whats-included-in-pro}

WCPOS Pro schaltet zusätzliche Funktionen frei:

- **Produktansicht** – Lagerbestände und Preise direkt im POS bearbeiten
- **Bestellansicht** – Bestellverlauf anzeigen und verwalten
- **Kundenansicht** – Kundeninformationen hinzufügen und bearbeiten
- **Berichtsansicht** - Tagesabschlussberichte erstellen
- **Zusätzliche Zahlungsgateways** - Stripe Terminal, SumUp und benutzerdefinierte Gateways

## Installation {#installation}

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, folgen Sie den nachstehenden Schritten, um das Plugin zu installieren und zu aktivieren:

1. Gehen Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Klicken Sie unter **Downloads** auf den Download-Link und speichern Sie das Plugin auf Ihrem Desktop.
3. Gehen Sie dann zu Ihrer Website, melden Sie sich an und navigieren Sie zu `WP Admin > Plugins > Add Plugin > Plugin hochladen`.
4. Laden Sie die Plugin-ZIP-Datei von Ihrem Desktop hoch und aktivieren Sie das Plugin.
5. Navigieren Sie anschließend zu `WP Admin > POS > Einstellungen > License` und geben Sie Ihren Lizenzschlüssel und Ihre Lizenz-E-Mail-Adresse ein, um die Aktivierung abzuschließen.

:::note Upgrade von einer älteren Version?
Wenn Sie zuvor sowohl das kostenlose WCPOS-Plugin als auch ein älteres Pro-Add-on installiert hatten, **deaktivieren und löschen Sie das kostenlose Plugin**, bevor Sie das neue eigenständige Pro-Plugin aktivieren. Der gleichzeitige Betrieb beider Plugins kann zu Aktivierungsfehlern wie *"Invalid response fetching remote state"* oder schwerwiegenden *"Class not found"*-Fehlern führen.
:::

## Aktualisierungen {#updates}

Sobald neue Versionen verfügbar sind, erscheinen sie wie bei jedem anderen Plugin im Aktualisierungs-Dashboard. Die neueste Version kann auch über die [Kontoseite](https://wcpos.com/my-account/) heruntergeladen werden.

## Fehlerbehebung {#purchase-a-license}

### Aktivierungsfehler {#activation-failures}

- **"Invalid licence key"** — Überprüfen Sie, ob beim Einfügen keine zusätzlichen Leerzeichen vorhanden sind, und stellen Sie sicher, dass die E-Mail-Adresse mit der beim Kauf verwendeten übereinstimmt.
- **"Licence limit reached"** — Jede Lizenz gilt für eine bestimmte Anzahl von Websites. Deaktivieren Sie nicht verwendete Websites über Ihre [Kontoseite](https://wcpos.com/my-account/) oder führen Sie ein Upgrade Ihres Plans durch.

### Abgelaufene Lizenz {#expired-licence}

- Nach Ablauf einer Lizenz bleiben die Pro-Funktionen aktiv, es werden jedoch keine Aktualisierungen mehr bereitgestellt.
- Die Verlängerung erfolgt über die [Kontoseite](https://wcpos.com/my-account/) — der bestehende Schlüssel bleibt unverändert.

### Domain-Übertragungen {#domain-transfers}

- Wenn die Website auf eine neue Domain umgezogen wurde, muss die Lizenz zunächst auf der alten Domain deaktiviert und anschließend auf der neuen Domain erneut aktiviert werden.
- Wenn die alte Domain nicht mehr erreichbar ist, wenden Sie sich an **support@wcpos.com**, um die Aktivierung zurückzusetzen.

### Lizenzserver-Konnektivität {#licence-server-connectivity}

- Für die Aktivierung muss Ihr WordPress-Server den WCPOS-Lizenzserver erreichen können.
- Wenn die Aktivierung ohne Fehlermeldung fehlschlägt, prüfen Sie, ob ausgehende HTTPS-Anfragen durch die Firewall Ihres Hosters oder ein Sicherheits-Plugin blockiert werden.
- Einige Managed-Hosting-Anbieter blockieren externe API-Aufrufe standardmäßig – wenden Sie sich an Ihren Hoster, um `wcpos.com` auf die Whitelist setzen zu lassen.

### Lizenzstatus prüfen {#checking-licence-status}

- Navigieren Sie zu `WP Admin > POS > Einstellungen > License`, um den aktuellen Aktivierungsstatus einzusehen.
- Auf Ihrer [Kontoseite](https://wcpos.com/my-account/) werden alle aktiven Websites und das Ablaufdatum angezeigt.

## Lizenz erwerben {#purchase-a-license}

Besuchen Sie [wcpos.com/pro](https://wcpos.com/pro), um eine Pro-Lizenz zu erwerben.
