:::info Pro-Funktion
[WCPOS Pro](https://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das zusätzliche Funktionen zum kostenlosen [WCPOS Plugin](https://wordpress.org/plugins/woocommerce-pos/) hinzufügt.
:::

## Was ist in Pro enthalten?

WCPOS Pro schaltet zusätzliche Funktionen frei:

- **Produktbildschirm** - Bearbeiten Sie Lagerbestände und Preise direkt in der POS
- **Bestellbildschirm** - Anzeigen und Verwalten der Bestellhistorie
- **Kundenbildschirm** - Hinzufügen und Bearbeiten von Kundeninformationen
- **Berichtsbildschirm** - Generieren von Tagesabschlussberichten
- **Zusätzliche Zahlungsgateways** - Stripe Terminal, SumUp und benutzerdefinierte Gateways

## Installation

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, folgen Sie bitte den nachstehenden Schritten zur Installation und Aktivierung des Plugins:

1. Gehen Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Klicken Sie unter **Downloads** auf den Download-Link und speichern Sie das Plugin auf Ihrem Desktop.
3. Gehen Sie dann zu Ihrer Website, melden Sie sich an und navigieren Sie zu `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
4. Laden Sie die Plugin-ZIP-Datei von Ihrem Desktop hoch und aktivieren Sie sie.
5. Gehen Sie als Nächstes zu `WP Admin > POS > Einstellungen > Lizenz` und geben Sie Ihren Lizenzschlüssel und Ihre Lizenz-E-Mail ein, um die Aktivierung abzuschließen.

![Meine Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Sie können WCPOS Pro auf Ihrer Kontoseite herunterladen")

## Updates

Wenn neue Versionen verfügbar sind, erscheinen sie in Ihrem Update-Dashboard wie bei jedem anderen Plugin. Sie können auch die neueste Version von Ihrer [Kontoseite](https://wcpos.com/my-account/) herunterladen.

## Lizenz erwerben

Besuchen Sie [wcpos.com/pro](https://wcpos.com/pro), um eine Pro-Lizenz zu erwerben.
