:::info Pro-Funktion
[WCPOS Pro](https://wcpos.com/pro) ist ein kostenpflichtiges Plugin, das der kostenlosen [WCPOS Plugin](https://wordpress.org/plugins/woocommerce-pos/) zusätzliche Funktionen hinzufügt.
:::

## Was ist in Pro enthalten? {#whats-included-in-pro}

WCPOS Pro schaltet zusätzliche Funktionen frei:

- **Produkte Bildschirm** - Bearbeiten Sie Bestandsmengen und Preise direkt im POS
- **Bestellungen Bildschirm** - Bestellhistorie anzeigen und verwalten
- **Kunden Bildschirm** - Kundeninformationen hinzufügen und bearbeiten
- **Berichte Bildschirm** - Verkaufsberichte zum Ende des Tages erstellen
- **Zusätzliche Zahlungs-Gateways** - Stripe Terminal, SumUp und benutzerdefinierte Gateways

## Installation

Wenn Sie eine Lizenz für WCPOS Pro erworben haben, befolgen Sie bitte die Schritte unten, um das Plugin zu installieren und zu aktivieren:

1. Gehen Sie zu: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Klicken Sie unter **Downloads** auf den Download-Link und speichern Sie das Plugin auf Ihrem Desktop.
3. Gehen Sie dann zu Ihrer Seite, melden Sie sich an und gehen Sie zu `WP Admin > Plugins > Plugin hinzufügen > Plugin hochladen`
4. Laden Sie die Plugin-Zip-Datei von Ihrem Desktop hoch und aktivieren Sie sie.
5. Gehen Sie nun zu `WP Admin > POS > Einstellungen > Lizenz` und geben Sie Ihren Lizenzschlüssel und Ihre Lizenz-E-Mail ein, um die Aktivierung abzuschließen.

![Meine Downloads](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Sie können WCPOS Pro auf Ihrer Kontoseite herunterladen")

## Updates

Wenn neue Versionen verfügbar sind, erscheinen sie in Ihrem Updates-Dashboard wie jedes andere Plugin. Sie können auch die neueste Version von Ihrer [Kontoseite](https://wcpos.com/my-account/) herunterladen.

## Lizenz erwerben {#purchase-a-license}

Besuchen Sie [wcpos.com/pro](https://wcpos.com/pro), um eine Pro-Lizenz zu erwerben.
