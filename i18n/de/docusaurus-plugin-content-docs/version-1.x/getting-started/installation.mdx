Um die ordnungsgemäße Funktionalität und optimale Leistung des WCPOS-Plugins sicherzustellen, stellen Sie bitte sicher, dass Ihre Umgebung die folgenden Mindestanforderungen erfüllt:

## Serveranforderungen {#server-requirements}

Das WCPOS-Plugin läuft auf Ihrer WordPress-Website, die diese Mindestanforderungen erfüllen muss:

| Komponente  | Mindestversion  | Hinweise        |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x empfohlen   |

Die Erfüllung dieser Mindestanforderungen gewährleistet die Kompatibilität und Stabilität mit Ihrer WordPress-Umgebung sowie den Zugriff auf die neuesten WCPOS-Funktionen.

## Browseranforderungen {#browser-requirements}

Die Web-Version des POS läuft in jedem modernen Browser. Verwenden Sie die neueste stabile Version:

| Browser         | Unterstützte Version |
| --------------- | -------------------- |
| Google Chrome   | Neueste stabile      |
| Mozilla Firefox | Neueste stabile      |
| Safari          | Neueste stabile      |
| Microsoft Edge  | Neueste stabile      |

:::warning Vermeiden Sie den privaten / Inkognito-Modus
Das private Surfen kann den Zugriff auf die IndexedDB-Datenbank des Browsers blockieren, die WCPOS zur Offline-Speicherung von Daten verwendet — einige Funktionen werden nicht korrekt arbeiten. JavaScript muss ebenfalls aktiviert sein.
:::

## App-Anforderungen {#app-requirements}

WCPOS ist auch als Desktop- und Mobil-App verfügbar. Jede wird für die folgenden Mindest-Betriebssysteme entwickelt:

| Plattform       | Mindest-Betriebssystem    | Architekturen              |
| --------------- | ------------------------- | -------------------------- |
| Windows-Desktop | Windows 10                | x64, ARM64                 |
| macOS-Desktop   | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Android         | Android 10 (API-Level 29) | —                          |

:::note Druckberechtigungen
Bluetooth- und Netzwerkdruck erfordern, dass der App die entsprechenden Bluetooth- und lokalen Netzwerkberechtigungen erteilt werden, wenn Sie dazu aufgefordert werden.
:::

## Installation des Plugins {#installing-the-plugin}

1. Gehen Sie zu `WP Admin > Plugins > Add Plugin`
2. Suchen Sie nach „wcpos"
3. Klicken Sie auf **Jetzt installieren** und dann auf **Aktivieren**

Alternativ können Sie das Plugin von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) herunterladen und manuell hochladen.

## Zugriff auf das POS {#accessing-the-pos}

Nach der Aktivierung können Sie auf das POS unter folgenden Adressen zugreifen:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Durch die Erfüllung der Mindestanforderungen und die Verwendung eines kompatiblen Webbrowsers können Sie das WCPOS-Plugin nahtlos nutzen und dessen volles Potenzial ausschöpfen, um Ihre Kassenvorgänge effizient zu verwalten.
