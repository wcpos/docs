Um die ordnungsgemäße Funktionalität und optimale Leistung des WCPOS-Plugins sicherzustellen, stellen Sie bitte sicher, dass Ihre Umgebung die folgenden Mindestanforderungen erfüllt:

## Serveranforderungen {#server-requirements}

- **WordPress:** Version 5.6 oder höher
- **WooCommerce:** Version 5.3 oder höher
- **PHP:** Version 7.2 oder höher

Die Erfüllung dieser Mindestanforderungen gewährleistet die Kompatibilität und Stabilität zwischen dem WCPOS-Plugin und Ihrer WordPress-Umgebung. Außerdem wird sichergestellt, dass Sie von den neuesten Funktionen und Erweiterungen profitieren können, die das Plugin bietet.

## Browseranforderungen {#browser-requirements}

- **Moderner Webbrowser:** Das WCPOS-Plugin ist so konzipiert, dass es mit modernen Webbrowsern funktioniert, darunter Google Chrome, Mozilla Firefox, Safari und Microsoft Edge. Bitte stellen Sie sicher, dass Sie die neueste stabile Version Ihres bevorzugten Browsers verwenden.
- **JavaScript aktiviert:** Das WCPOS-Plugin ist auf JavaScript für seine interaktiven Funktionen angewiesen. Bitte stellen Sie sicher, dass JavaScript in den Browsereinstellungen aktiviert ist.
- **Kein privater Browsing-Modus:** Der Browser sollte sich nicht im privaten Browsing-Modus befinden, da dies den Zugriff auf die IndexedDB (IDB)-Datenbank des Browsers einschränken kann. Der private Browsing-Modus kann die ordnungsgemäße Funktion bestimmter Funktionen im WCPOS-Plugin verhindern.

## Installation des Plugins {#installing-the-plugin}

1. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen`
2. Suchen Sie nach "WooCommerce POS"
3. Klicken Sie auf **Jetzt installieren**, dann **Aktivieren**

Alternativ können Sie das Plugin von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) herunterladen und manuell hochladen.

## Zugriff auf das POS {#accessing-the-pos}

Nach der Aktivierung können Sie auf das POS zugreifen unter:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Durch die Erfüllung der Mindestanforderungen und die Verwendung eines kompatiblen Webbrowsers können Sie eine nahtlose Erfahrung mit dem WCPOS-Plugin haben und dessen volles Potenzial nutzen, um Ihre Verkaufsstellenoperationen effizient zu verwalten.
