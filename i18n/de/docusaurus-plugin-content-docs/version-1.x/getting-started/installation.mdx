Um die ordnungsgemäße Funktionalität und optimale Leistung des WCPOS-Plugins sicherzustellen, stellen Sie bitte sicher, dass Ihre Umgebung die folgenden Mindestanforderungen erfüllt:

## Serveranforderungen

- **WordPress:** Version 5.6 oder höher
- **WooCommerce:** Version 5.3 oder höher
- **PHP:** Version 7.2 oder höher

Wenn diese Mindestanforderungen erfüllt sind, wird die Kompatibilität und Stabilität zwischen dem WCPOS-Plugin und Ihrer WordPress-Umgebung sichergestellt. Außerdem können Sie von den neuesten Funktionen und Verbesserungen des Plugins profitieren.

## Browseranforderungen

- **Moderner Webbrowser:** Das WCPOS-Plugin ist für die Verwendung mit modernen Webbrowsern wie Google Chrome, Mozilla Firefox, Safari und Microsoft Edge ausgelegt. Stellen Sie bitte sicher, dass Sie die neueste stabile Version Ihres bevorzugten Browsers verwenden.
- **JavaScript aktiviert:** Das WCPOS-Plugin ist auf JavaScript für seine interaktiven Funktionen angewiesen. Bitte stellen Sie sicher, dass JavaScript in Ihren Browsereinstellungen aktiviert ist.
- **Kein privater Browsing-Modus:** Der Browser sollte sich nicht im privaten Browsing-Modus befinden, da dies den Zugriff auf die IndexedDB (IDB)-Datenbank des Browsers einschränken kann. Im privaten Browsing-Modus kann die ordnungsgemäße Funktion bestimmter Funktionen im WCPOS-Plugin verhindert werden.

## Installation des Plugins

1. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen`
2. Suchen Sie nach "WooCommerce POS"
3. Klicken Sie auf **Jetzt installieren**, dann auf **Aktivieren**

Alternativ können Sie das Plugin von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) herunterladen und manuell hochladen.

## Zugriff auf das POS

Nach der Aktivierung können Sie auf das POS zugreifen unter:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Indem Sie die Mindestanforderungen erfüllen und einen kompatiblen Webbrowser verwenden, können Sie eine nahtlose Erfahrung mit dem WCPOS-Plugin haben und dessen volles Potenzial zur effizienten Verwaltung Ihrer Verkaufsstellenoperationen ausschöpfen.

Wenn Sie weitere Fragen haben oder Unterstützung benötigen, zögern Sie bitte nicht, sich an unser Support-Team zu wenden, indem Sie Ihre Fragen im [Discord-Chat](https://wcpos.com/discord) stellen oder eine E-Mail an support@wcpos.com senden.
