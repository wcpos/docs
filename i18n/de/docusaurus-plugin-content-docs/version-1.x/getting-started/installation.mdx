Um eine einwandfreie Funktionalität und optimale Leistung des WCPOS-Plugins zu gewährleisten, stellen Sie sicher, dass Ihre Umgebung die folgenden Mindestanforderungen erfüllt:

## Serveranforderungen

- **WordPress:** Version 5.6 oder höher
- **WooCommerce:** Version 5.3 oder höher
- **PHP:** Version 7.2 oder höher

Das Erfüllen dieser Mindestanforderungen gewährleistet die Kompatibilität und Stabilität zwischen dem WCPOS-Plugin und Ihrer WordPress-Umgebung. Außerdem stellen Sie sicher, dass Sie von den neuesten Funktionen und Verbesserungen des Plugins profitieren können.

## Browser-Anforderungen

- **Moderner Webbrowser:** Das WCPOS-Plugin ist für die Arbeit mit modernen Webbrowsern entwickelt, einschließlich Google Chrome, Mozilla Firefox, Safari und Microsoft Edge. Stellen Sie sicher, dass Sie die neueste stabile Version Ihres bevorzugten Browsers verwenden.
- **JavaScript aktiviert:** Das WCPOS-Plugin basiert auf JavaScript für seine interaktiven Funktionen. Stellen Sie bitte sicher, dass JavaScript in Ihren Browsereinstellungen aktiviert ist.
- **Kein privater Browsermodus:** Der Browser sollte sich nicht im privaten Browsermodus befinden, da dieser den Zugriff auf die IndexedDB-Datenbank des Browsers einschränken kann. Der private Browsermodus kann die ordnungsgemäße Funktion bestimmter Funktionen im WCPOS-Plugin verhindern.

## Plugin installieren

1. Gehen Sie zu `WP Admin > Plugins > Neue hinzufügen`
2. Suchen Sie nach "WooCommerce POS"
3. Klicken Sie auf **Jetzt installieren**, dann **Aktivieren**

Alternativ können Sie das Plugin von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) herunterladen und manuell hochladen.

## Zugriff auf das POS

Nach der Aktivierung können Sie auf das POS zugreifen unter:

- **Web:** `ihredomain.com/pos`
- **Desktop/Mobile:** Laden Sie die App von [wcpos.com](https://wcpos.com) herunter und verbinden Sie sich mit Ihrem Shop

Indem Sie die Mindestanforderungen erfüllen und einen kompatiblen Webbrowser verwenden, können Sie eine nahtlose Erfahrung mit dem WCPOS-Plugin machen und sein volles Potenzial zur effizienten Verwaltung Ihrer Point-of-Sale-Aktivitäten ausschöpfen.
