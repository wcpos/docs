Um die ordnungsgemäße Funktionalität und optimale Leistung des WCPOS-Plugins sicherzustellen, stellen Sie bitte sicher, dass Ihre Umgebung die folgenden Mindestanforderungen erfüllt:

## Serveranforderungen {#server-requirements}

- **WordPress:** Version 5.6 oder höher
- **WooCommerce:** Version 5.3 oder höher
- **PHP:** Version 7.4 oder höher (8.x empfohlen)

Die Erfüllung dieser Mindestanforderungen gewährleistet die Kompatibilität und Stabilität zwischen dem WCPOS-Plugin und Ihrer WordPress-Umgebung. Außerdem wird sichergestellt, dass Sie von den neuesten Funktionen und Verbesserungen des Plugins profitieren können.

## Browseranforderungen {#browser-requirements}

- **Moderner Webbrowser:** Das WCPOS-Plugin ist für die Verwendung mit modernen Webbrowsern konzipiert, darunter Google Chrome, Mozilla Firefox, Safari und Microsoft Edge. Bitte stellen Sie sicher, dass Sie die neueste stabile Version Ihres bevorzugten Browsers verwenden.
- **JavaScript aktiviert:** Das WCPOS-Plugin verwendet JavaScript für seine interaktiven Funktionen. Bitte stellen Sie sicher, dass JavaScript in Ihren Browsereinstellungen aktiviert ist.
- **Kein privates Surfen:** Der Browser sollte sich nicht im privaten Modus befinden, da dieser den Zugriff auf die IndexedDB-Datenbank (IDB) des Browsers einschränken kann. Der private Modus kann die ordnungsgemäße Funktion bestimmter Features des WCPOS-Plugins verhindern.

## Installation des Plugins {#installing-the-plugin}

1. Navigieren Sie zu `WP Admin > Plugins > Add Plugin`
2. Suchen Sie nach "wcpos"
3. Klicken Sie auf **Jetzt installieren** und dann auf **Aktivieren**

Alternativ können Sie das Plugin von [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) herunterladen und manuell hochladen.

## Zugriff auf das POS {#accessing-the-pos}

Nach der Aktivierung können Sie auf das POS unter folgenden Adressen zugreifen:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Durch die Erfüllung der Mindestanforderungen und die Verwendung eines kompatiblen Webbrowsers können Sie das WCPOS-Plugin nahtlos nutzen und dessen volles Potenzial für die effiziente Verwaltung Ihrer Kassenvorgänge ausschöpfen.
