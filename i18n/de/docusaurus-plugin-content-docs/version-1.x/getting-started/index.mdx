## Was ist WCPOS? {#what-is-wcpos}

WCPOS ist ein Point-of-Sale-System (POS) für WooCommerce. Es ermöglicht den Vor-Ort-Verkauf von Produkten aus dem WooCommerce-Shop an einem physischen Standort über einen Webbrowser oder eine eigene App.

## In 5 Minuten loslegen {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="Das Plugin installieren">

Installieren Sie das kostenlose [WCPOS-Plugin](https://wordpress.org/plugins/woocommerce-pos/) auf Ihrer WordPress-Website:

1. Gehen Sie zu **WP Admin → Plugins → Installieren**
2. Suchen Sie nach "WCPOS"
3. Klicken Sie auf **Jetzt installieren** und anschließend auf **Aktivieren**

Ausführliche Anforderungen finden Sie unter [Installation](/getting-started/installation).

</StepCard>

<StepCard number={2} title="Auf den POS zugreifen">

**Web-Nutzer:** Navigieren Sie in Ihrem Browser zu `yourdomain.com/pos`. Sie werden aufgefordert, sich mit Ihren WordPress-Anmeldedaten anzumelden.

**Desktop-/Mobil-Nutzer:** Laden Sie die App herunter, geben Sie die URL Ihrer Filiale ein, klicken Sie auf **Verbinden** und melden Sie sich mit Ihren WordPress-Anmeldedaten an.

Weitere Informationen zur Verwaltung mehrerer Filialen und Nutzer finden Sie unter [Verbinden](/getting-started/connect).

</StepCard>

<StepCard number={3} title="Auf die erste Synchronisierung warten">

WCPOS lädt Produkte in kleinen Stapeln herunter, um Ihren Server nicht zu überlasten. Produkte werden nach und nach heruntergeladen, während Sie suchen, scrollen oder filtern – Sie müssen nicht warten, bis alles synchronisiert ist, bevor Sie mit dem Verkauf beginnen.

Weitere Informationen finden Sie unter [Produktsynchronisierung](/products/sync).

</StepCard>

<StepCard number={4} title="Mit dem Verkauf beginnen" isLast>

- Suchen oder scannen Sie Produkte, um sie zum Warenkorb hinzuzufügen
- Wählen Sie einen Kunden aus oder verwenden Sie „Gast“
- Klicken Sie auf **Checkout**, um die Zahlung abzuwickeln
- Beleg drucken oder per E-Mail versenden

</StepCard>

</Steps>

## Wichtige Funktionen {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="Schnelle lokale Suche">
    Produkte werden lokal gespeichert, um sofortige Suchergebnisse zu liefern
  </FeatureCard>
  <FeatureCard icon="wifi" title="Offline-Durchsuchen">
    Produkte und Kunden ohne Internetverbindung durchsuchen (für Bestellungen ist eine Verbindung erforderlich)
  </FeatureCard>
  <FeatureCard icon="barcode" title="Barcode-Scannen">
    Verwenden Sie einen beliebigen USB- oder Bluetooth-Barcode-Scanner
  </FeatureCard>
  <FeatureCard icon="credit-card" title="Bar- & Kartenzahlungen">
    Integrierte Unterstützung für Bar- und einfache Kartenzahlungen
  </FeatureCard>
  <FeatureCard icon="print" title="Belegdruck">
    Anpassbare Belege drucken oder per E-Mail versenden
  </FeatureCard>
  <FeatureCard icon="calculator" title="Automatische Steuer">
    Steuerberechnungen basierend auf Ihren WooCommerce-Steuereinstellungen
  </FeatureCard>
</FeatureGrid>
