## Was ist WCPOS? {#what-is-wcpos}

WCPOS ist ein Point of Sale (POS) System für WooCommerce. Es ermöglicht Ihnen, Produkte aus Ihrem WooCommerce-Shop persönlich an einem physischen Standort über einen Webbrowser oder eine spezielle App zu verkaufen.

## Erste Schritte in 5 Minuten {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="Plugin installieren">

Installieren Sie das kostenlose [WCPOS-Plugin](https://wordpress.org/plugins/woocommerce-pos/) auf Ihrer WordPress-Website:

1. Gehen Sie zu **WP Admin → Plugins → Neu hinzufügen**
2. Suchen Sie nach "WooCommerce POS"
3. Klicken Sie auf **Jetzt installieren** und dann auf **Aktivieren**

Siehe [Installation](/getting-started/installation) für detaillierte Anforderungen.

</StepCard>

<StepCard number={2} title="Zugriff auf das POS">

**Webbenutzer:** Navigieren Sie zu `yourdomain.com/pos` in Ihrem Browser. Sie werden aufgefordert, sich mit Ihren WordPress-Anmeldeinformationen anzumelden.

**Desktop-/Mobilbenutzer:** Laden Sie die App herunter, geben Sie die URL Ihres Shops ein, klicken Sie auf **Verbinden** und melden Sie sich mit Ihren WordPress-Anmeldeinformationen an.

Siehe [Verbinden](/getting-started/connect) für Details zur Verwaltung mehrerer Shops und Benutzer.

</StepCard>

<StepCard number={3} title="Warten auf die erste Synchronisierung">

WCPOS lädt Produkte in kleinen Chargen herunter, um Ihre Server nicht zu überlasten. Produkte werden schrittweise heruntergeladen, während Sie suchen, scrollen oder filtern - Sie müssen nicht warten, bis alles synchronisiert ist, bevor Sie mit dem Verkauf beginnen.

Siehe [Produkt-Synchronisation](/products/sync) für Details.

</StepCard>

<StepCard number={4} title="Verkauf starten" isLast>

- Suchen oder scannen Sie Produkte, um sie zum Warenkorb hinzuzufügen
- Wählen Sie einen Kunden oder verwenden Sie "Gast"
- Klicken Sie auf **Kasse**, um die Zahlung abzuwickeln
- Drucken Sie den Beleg aus oder senden Sie ihn per E-Mail

</StepCard>

</Steps>

## Hauptmerkmale {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="Schnelle lokale Suche">
    Produkte werden lokal gespeichert, um sofortige Suchergebnisse zu erhalten
  </FeatureCard>
  <FeatureCard icon="wifi" title="Offline-Browsing">
    Suchen Sie Produkte und Kunden ohne Internet (Bestellungen erfordern Konnektivität)
  </FeatureCard>
  <FeatureCard icon="barcode" title="Barcode-Scanning">
    Verwenden Sie jeden USB- oder Bluetooth-Barcode-Scanner
  </FeatureCard>
  <FeatureCard icon="credit-card" title="Bargeld- und Kartenzahlungen">
    Eingebauter Support für Bargeld- und grundlegende Kartenzahlungen
  </FeatureCard>
  <FeatureCard icon="print" title="Belegdruck">
    Drucken oder senden Sie anpassbare Belege per E-Mail
  </FeatureCard>
  <FeatureCard icon="calculator" title="Automatische Steuer">
    Steuerberechnungen basierend auf Ihren WooCommerce-Steuereinstellungen
  </FeatureCard>
</FeatureGrid>
