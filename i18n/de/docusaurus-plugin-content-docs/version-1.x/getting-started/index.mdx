## Was ist WCPOS?

WCPOS ist ein Point of Sale (POS) System für WooCommerce. Es ermöglicht Ihnen, Produkte aus Ihrem WooCommerce-Shop direkt vor Ort an einem physischen Standort über einen Webbrowser oder eine dedizierte App zu verkaufen.

**Verfügbar auf:**

- **Web:** Sofortiger Zugriff unter `ihredomain.com/pos`
- **Desktop:** Download für Windows, macOS und Linux von [wcpos.com](https://wcpos.com)
- **Mobile:** iOS- und Android-Apps (demnächst)

## In 5 Minuten einrichten

### 1. Plugin installieren

Installieren Sie das kostenlose [WCPOS-Plugin](https://wordpress.org/plugins/woocommerce-pos/) auf Ihrer WordPress-Website:

1. Gehen Sie zu `WP Admin > Plugins > Neu hinzufügen`
2. Suchen Sie nach "WooCommerce POS"
3. Klicken Sie auf **Jetzt installieren**, dann **Aktivieren**

Weitere Informationen finden Sie unter [Installation](/getting-started/installation).

### 2. Zugriff auf die POS

**Web-Nutzer:**
Navigieren Sie in Ihrem Browser zu `ihredomain.com/pos`. Sie werden aufgefordert, sich mit Ihren WordPress-Zugangsdaten anzumelden.

**Desktop-/Mobile-Nutzer:**
1. Laden Sie die App von [wcpos.com](https://wcpos.com) herunter
2. Geben Sie Ihre Store-URL ein und klicken Sie auf **Verbinden**
3. Melden Sie sich mit Ihren WordPress-Zugangsdaten an

Weitere Details zur Verwaltung mehrerer Stores und Nutzer finden Sie unter [Verbinden](/getting-started/connect).

### 3. Auf erste Synchronisation warten

WCPOS lädt Produkte in kleinen Chargen (ca. 50 gleichzeitig) herunter, um den Server nicht zu überlasten. Produkte werden schrittweise heruntergeladen, während Sie suchen, scrollen oder filtern - Sie müssen nicht auf den vollständigen Sync warten, bevor Sie mit dem Verkauf beginnen. Details finden Sie unter [Produktsynchronisation](/products/sync).

### 4. Verkauf beginnen

- Produkte suchen oder scannen, um sie zum Warenkorb hinzuzufügen
- Kunden auswählen oder "Gast" verwenden
- Auf **Kasse** klicken, um die Zahlung zu verarbeiten
- Quittung drucken oder per E-Mail versenden

## Hauptfunktionen

- **Schnelle lokale Suche** - Produkte sind lokal gespeichert für sofortige Ergebnisse
- **Offline-Browsing** - Produkte und Kunden suchen ohne Internet (Auftragsabschluss erfordert Konnektivität)
- **Barcode-Scanning** - Jeder USB- oder Bluetooth-Barcodescanner nutzbar
- **Mehrere Zahlungsmethoden** - Bar, Karte und individuelle Zahlungswege
- **Quittungsdruck** - Anpassbare Quittungen drucken oder per E-Mail versenden
- **Multi-Store-Unterstützung** - Verbindung zu mehreren WooCommerce-Stores

## Pro-Funktionen

[WCPOS Pro](https://wcpos.com/pro) erweitert die Möglichkeiten:

- **Produktverwaltung** - Bestand und Preise direkt in der POS bearbeiten
- **Auftragshistorie** - Vergangene Aufträge anzeigen und verwalten
- **Kundenverwaltung** - Kundendetails hinzufügen und bearbeiten
- **Verkaufsberichte** - Tagesabschlussberichte mit Druckoption
- **Zusätzliche Zahlungswege** - Stripe Terminal, SumUp und mehr
