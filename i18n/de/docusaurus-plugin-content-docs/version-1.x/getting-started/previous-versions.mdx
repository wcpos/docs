## Eine frühere Version erneut installieren

Wenn Sie zu einer früheren Version des WCPOS-Plugins zurückkehren müssen, folgen Sie diesen Schritten:

1. Besuchen Sie die WordPress.org-Seite [Erweiterte Ansicht](https://wordpress.org/plugins/woocommerce-pos/advanced/) für WCPOS.

2. Scrollen Sie nach unten zum Abschnitt **Erweiterte Optionen** und suchen Sie das **Previous Versions**-Auswahlmenü.

3. Öffnen Sie das **Previous Versions**-Auswahlmenü und Sie sehen eine Liste der verfügbaren Plugin-Versionen. Finden Sie die Version, die Sie installieren möchten, und klicken Sie auf die Schaltfläche **Download** daneben. Speichern Sie die heruntergeladene Datei auf Ihrem Desktop oder an einem anderen praktischen Ort.

4. Melden Sie sich bei Ihrem WordPress-Admin-Dashboard an.

5. Navigieren Sie zur Seite **Plugins**, indem Sie im linken Menü auf **Plugins** klicken.

6. Deaktivieren und löschen Sie die aktuelle Version des WCPOS-Plugins. Dadurch wird das Plugin aus Ihrer WordPress-Installation entfernt. Keine Sorge, Ihre Daten gehen nicht verloren.

7. Klicken Sie auf die Schaltfläche **Add New** oben auf der Seite.

8. Klicken Sie auf der Seite **Add Plugins** auf die Schaltfläche **Upload Plugin**.

9. Wählen Sie die zuvor heruntergeladene Plugin-Datei von Ihrem Desktop oder dem Ort, an dem Sie sie gespeichert haben.

<div class="image-container">
  <Image

  />
  <p class="image-caption">Plugin im WordPress-Admin hochladen</p>
</div>

10. Klicken Sie auf die Schaltfläche **Install Now** und warten Sie, bis die Installation abgeschlossen ist.

11. Nachdem die Installation abgeschlossen ist, klicken Sie auf die Schaltfläche **Activate**, um die frühere Version des WCPOS-Plugins zu aktivieren.

Das war's! Sie haben erfolgreich eine frühere Version des WCPOS-Plugins erneut installiert.

Wenn Sie während des Reinstallationsprozesses auf Probleme stoßen oder weitere Unterstützung benötigen, wenden Sie sich bitte an unser Support-Team unter support@wcpos.com.
