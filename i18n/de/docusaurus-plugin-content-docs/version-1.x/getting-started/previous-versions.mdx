## Neuinstallation einer früheren Version

Wenn Sie zu einer früheren Version des WCPOS-Plugins zurückkehren müssen, folgen Sie diesen Schritten:

1. Besuchen Sie die WordPress.org-Seite [Erweiterte Ansicht](https://wordpress.org/plugins/woocommerce-pos/advanced/) für WCPOS.

2. Scrollen Sie nach unten zum Abschnitt **Erweiterte Optionen** und finden Sie das Auswahlmenü **Vorherige Versionen**.

3. Öffnen Sie das Auswahlmenü und Sie werden eine Liste der verfügbaren Plugin-Versionen sehen. Suchen Sie die Version, die Sie installieren möchten, und klicken Sie auf die **Download**-Schaltfläche daneben. Speichern Sie die heruntergeladene Datei auf Ihrem Desktop oder an einem bequemen Ort.

4. Melden Sie sich an Ihrem WordPress-Admin-Dashboard an.

5. Navigieren Sie zur Seite **Plugins**, indem Sie im linken Menü auf **Plugins** klicken.

6. Deaktivieren und löschen Sie die aktuelle Version des WCPOS-Plugins. Dies wird das Plugin von Ihrer WordPress-Installation entfernen. Keine Sorge, Ihre Daten gehen nicht verloren.

7. Klicken Sie auf die Schaltfläche **Neu hinzufügen** oben auf der Seite.

8. Klicken Sie auf der Seite **Plugins hinzufügen** auf die Schaltfläche **Plugin hochladen**.

9. Wählen Sie die zuvor heruntergeladene Plugin-Datei von Ihrem Desktop oder dem Speicherort, an dem Sie sie gespeichert haben.

<div class="image-container">

  <p class="image-caption">Plugin im WordPress-Admin hochladen</p>
</div>

10. Klicken Sie auf die Schaltfläche **Jetzt installieren** und warten Sie, bis die Installation abgeschlossen ist.

11. Klicken Sie nach Abschluss der Installation auf die Schaltfläche **Aktivieren**, um die vorherige Version des WCPOS-Plugins zu aktivieren.

Das war's! Sie haben erfolgreich eine frühere Version des WCPOS-Plugins neu installiert.

Wenn Sie während des Neuinstallationsprozesses auf Probleme stoßen oder weitere Unterstützung benötigen, kontaktieren Sie bitte unser Support-Team unter support@wcpos.com.
