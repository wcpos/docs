## Übersicht des Verbindungsbildschirms

Wenn Sie die WCPOS Desktop- oder Mobilanwendung öffnen, sehen Sie den Verbindungsbildschirm. Hier verwalten Sie Ihre Geschäftsanbindungen und Benutzeranmeldungen.

{/* Screenshot: connect-screen.png */}

## Hinzufügen eines neuen Geschäfts

1. Geben Sie die URL Ihres WooCommerce-Geschäfts in das Textfeld ein (z.B. `https://mystore.com`)
2. Klicken Sie auf **Verbinden**
3. Sie werden umgeleitet, um sich mit Ihren WordPress-Anmeldedaten anzumelden
4. Nach erfolgreichem Login werden Sie zurück zur Anwendung geleitet

## Mehrere Geschäfte

Sie können sich mit so vielen WooCommerce-Geschäften verbinden, wie Sie benötigen. Jedes Geschäft wird als separate Karte auf dem Verbindungsbildschirm angezeigt, die Folgendes zeigt:

- **Geschäftsname** und Favicon
- **Geschäfts-URL**
- **Angemeldete Benutzer** für dieses Geschäft

Dies ist nützlich, wenn Sie mehrere Standorte oder Unternehmen verwalten.

## Mehrere Benutzer pro Geschäft

Jedes Geschäft kann mehrere Benutzer gleichzeitig angemeldet haben. Dies ist hilfreich für:

- **Schichtwechsel** - Neuer Kassierer kann sich anmelden, bevor der vorherige sich abmeldet
- **Multi-Register-Setups** - Verschiedene Kassierer auf verschiedenen Geräten
- **Schnelles Wechseln** - Einfach zwischen Benutzerkonten wechseln

### Einen Benutzer hinzufügen

Klicken Sie auf die <Icon name="plus" /> Schaltfläche neben "Angemeldete Benutzer", um einen weiteren Benutzer zu diesem Geschäft hinzuzufügen.

### Benutzer wechseln

Klicken Sie auf ein Benutzerabzeichen (z.B. "Brenda"), um das POS als dieser Benutzer zu öffnen.

### Einen Benutzer entfernen

Klicken Sie auf das **×** auf einem Benutzerabzeichen, um diesen Benutzer aus dem Geschäft abzumelden.

## Ein Geschäft entfernen

Klicken Sie auf die rote **×**-Schaltfläche auf der Geschäftskarte, um es aus Ihrer Liste zu entfernen. Dadurch werden alle Benutzer abgemeldet und die Geschäftsanbindung aus der Anwendung entfernt.

## Demogeschäft

Unten auf dem Bildschirm sehen Sie möglicherweise einen Link "Demogeschäft betreten". Dies verbindet Sie mit einem DemowooCommerce-Geschäft, um die WCPOS-Funktionen auszuprobieren, ohne Ihre eigenen Geschäftsdaten zu beeinflussen.

## Fehlersuche

### Kann mich nicht mit meinem Geschäft verbinden?

- Stellen Sie sicher, dass das WCPOS-Plugin auf Ihrer WordPress-Seite installiert und aktiviert ist
- Überprüfen Sie, dass Sie die korrekte URL verwenden (schließen Sie `https://` ein, wenn Ihre Website SSL verwendet)
- Stellen Sie sicher, dass die WooCommerce REST API zugänglich ist
- Überprüfen Sie, dass Ihr Benutzerkonto POS-Zugriffsberechtigungen hat

### Verbindung schlägt ständig fehl?

- Versuchen Sie zuerst, `yourdomain.com/pos` in einem Webbrowser aufzurufen, um zu überprüfen, ob das Plugin funktioniert
- Überprüfen Sie die Fehlerprotokolle Ihrer Website auf Probleme
- Stellen Sie sicher, dass Ihr Server die [Mindestanforderungen](/getting-started/installation) erfüllt
