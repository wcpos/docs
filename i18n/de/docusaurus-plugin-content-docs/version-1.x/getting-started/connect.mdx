## Übersicht des Verbindungsbildschirms

Wenn Sie die WCPOS Desktop- oder Mobile-App öffnen, sehen Sie den Verbindungsbildschirm. Hier verwalten Sie Ihre Shopverbindungen und Benutzeranmeldungen.

{/* Screenshot: connect-screen.png */}

## Einen neuen Shop hinzufügen

1. Geben Sie die URL Ihres WooCommerce-Shops in das Textfeld ein (z.B. `https://meinshop.com`)
2. Klicken Sie auf **Verbinden**
3. Sie werden weitergeleitet, um sich mit Ihren WordPress-Anmeldedaten anzumelden
4. Nach erfolgreicher Anmeldung kehren Sie zur App zurück

## Mehrere Shops

Sie können sich mit so vielen WooCommerce-Shops verbinden, wie Sie benötigen. Jeder Shop erscheint als separate Karte auf dem Verbindungsbildschirm mit:

- **Shopname** und Favicon
- **Shop-URL**
- **Angemeldete Benutzer** für diesen Shop

Dies ist nützlich, wenn Sie mehrere Standorte oder Unternehmen verwalten.

## Mehrere Benutzer pro Shop

Jeder Shop kann mehrere Benutzer gleichzeitig angemeldet haben. Dies ist hilfreich für:

- **Schichtwechsel** - Neuer Kassierer kann sich anmelden, bevor der vorherige sich abmeldet
- **Mehrfach-Kassen-Setups** - Verschiedene Kassierer auf verschiedenen Geräten
- **Schnelles Wechseln** - Einfaches Wechseln zwischen Benutzerkonten

### Benutzer hinzufügen

Klicken Sie auf die <Icon name="plus" /> Schaltfläche neben "Angemeldete Benutzer", um einen weiteren Benutzer für diesen Shop hinzuzufügen.

### Benutzer wechseln

Klicken Sie auf einen Benutzerbadge (z.B. "Brenda"), um den POS als dieser Benutzer zu öffnen.

### Benutzer entfernen

Klicken Sie auf das **×** auf einem Benutzerbadge, um diesen Benutzer aus dem Shop abzumelden.

## Shop entfernen

Klicken Sie auf die rote **×** Schaltfläche auf der Shopkarte, um sie aus Ihrer Liste zu entfernen. Dies meldet alle Benutzer ab und entfernt die Shopverbindung aus der App.

## Demo-Shop

Am unteren Rand des Bildschirms finden Sie möglicherweise einen Link "Demo-Shop betreten". Dieser verbindet Sie mit einem Demo-WooCommerce-Shop, um WCPOS-Funktionen auszuprobieren, ohne Ihre eigenen Shopdaten zu beeinflussen.

## Fehlerbehebung

### Keine Verbindung zu meinem Shop?

- Stellen Sie sicher, dass das WCPOS-Plugin auf Ihrer WordPress-Seite installiert und aktiviert ist
- Überprüfen Sie, dass Sie die korrekte URL verwenden (fügen Sie `https://` hinzu, wenn Ihre Seite SSL verwendet)
- Stellen Sie sicher, dass die WooCommerce-REST-API zugänglich ist
- Überprüfen Sie, dass Ihr Benutzerkonto POS-Zugriffsberechtigungen hat

### Verbindung schlägt wiederholt fehl?

- Versuchen Sie zuerst, `ihredomain.com/pos` in einem Webbrowser zu öffnen, um zu überprüfen, ob das Plugin funktioniert
- Überprüfen Sie die Fehlerprotokolle Ihrer Website
- Stellen Sie sicher, dass Ihr Server die [Mindestanforderungen](/getting-started/installation) erfüllt
