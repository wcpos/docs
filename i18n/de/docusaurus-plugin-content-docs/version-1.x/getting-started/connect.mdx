## Übersicht des Verbindungsbildschirms

Wenn Sie die WCPOS Desktop- oder Mobil-App öffnen, sehen Sie den Verbindungsbildschirm. Hier verwalten Sie Ihre Store-Verbindungen und Benutzeranmeldungen.

{/* Screenshot: connect-screen.png */}

## Hinzufügen eines neuen Stores

1. Geben Sie die URL Ihres WooCommerce-Stores in das Textfeld ein (z.B. `https://mystore.com`)
2. Klicken Sie auf **Verbinden**
3. Sie werden weitergeleitet, um sich mit Ihren WordPress-Anmeldedaten anzumelden
4. Nach erfolgreichem Login werden Sie zur App zurückgeleitet

## Mehrere Stores

Sie können so viele WooCommerce-Stores hinzufügen, wie Sie benötigen. Jeder Store wird als separate Karte auf dem Verbindungsbildschirm angezeigt, die Folgendes zeigt:

- **Store-Name** und Favicon
- **Store-URL**
- **Angemeldete Benutzer** für diesen Store

Dies ist nützlich, wenn Sie mehrere Standorte oder Unternehmen verwalten.

## Mehrere Benutzer pro Store

Jeder Store kann mehrere Benutzer gleichzeitig angemeldet haben. Dies ist hilfreich für:

- **Schichtwechsel** - Neuer Kassierer kann sich anmelden, bevor der vorherige sich abmeldet
- **Multi-Register-Setups** - Verschiedene Kassierer auf verschiedenen Geräten
- **Schnelles Wechseln** - Einfach zwischen Benutzerkonten wechseln

### Einen Benutzer hinzufügen

Klicken Sie auf die <Icon name="plus" /> Schaltfläche neben "Angemeldete Benutzer", um einen weiteren Benutzer zu diesem Store hinzuzufügen.

### Benutzer wechseln

Klicken Sie auf ein Benutzerabzeichen (z.B. "Brenda"), um das POS als dieser Benutzer zu öffnen.

### Einen Benutzer entfernen

Klicken Sie auf das **×** auf einem Benutzerabzeichen, um diesen Benutzer aus dem Store abzumelden.

## Einen Store entfernen

Klicken Sie auf die rote **×**-Schaltfläche auf der Store-Karte, um ihn aus Ihrer Liste zu entfernen. Dadurch werden alle Benutzer abgemeldet und die Store-Verbindung aus der App entfernt.

## Demo Store

Unten auf dem Bildschirm sehen Sie möglicherweise einen Link "Demo Store betreten". Dies verbindet Sie mit einem Demo WooCommerce-Store, um die WCPOS-Funktionen auszuprobieren, ohne Ihre eigenen Store-Daten zu beeinflussen.

## Fehlersuche

### Kann nicht mit meinem Store verbinden?

- Stellen Sie sicher, dass das WCPOS-Plugin auf Ihrer WordPress-Seite installiert und aktiviert ist
- Überprüfen Sie, dass Sie die korrekte URL verwenden (schließen Sie `https://` ein, wenn Ihre Website SSL verwendet)
- Stellen Sie sicher, dass die WooCommerce REST API zugänglich ist
- Überprüfen Sie, dass Ihr Benutzerkonto POS-Zugriffsberechtigungen hat

### Verbindung schlägt ständig fehl?

- Versuchen Sie zuerst, `yourdomain.com/pos` in einem Webbrowser aufzurufen, um zu überprüfen, ob das Plugin funktioniert
- Überprüfen Sie die Fehlerprotokolle Ihrer Website auf Probleme
- Stellen Sie sicher, dass Ihr Server die [Mindestanforderungen](/getting-started/installation) erfüllt
