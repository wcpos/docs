## Übersicht über den Verbindungsscreen

Wenn Sie die WCPOS-Desktop- oder Mobile-App öffnen, sehen Sie den Verbindungsscreen. Hier verwalten Sie Ihre Geschäftsanbindungen und Benutzeranmeldungen.

{/* Screenshot: connect-screen.png */}

## Hinzufügen eines neuen Geschäfts

1. Geben Sie die URL Ihres WooCommerce-Geschäfts in das Textfeld ein (z. B. `https://mystore.com`)
2. Klicken Sie auf **Verbinden**
3. Sie werden weitergeleitet, um sich mit Ihren WordPress-Anmeldeinformationen anzumelden
4. Nach erfolgreicher Anmeldung werden Sie zur App zurückgeleitet

## Mehrere Geschäfte

Sie können sich mit so vielen WooCommerce-Geschäften verbinden, wie Sie benötigen. Jedes Geschäft erscheint als separate Karte im Verbindungsscreen, die anzeigt:

- **Geschäftsname** und Favicon
- **Geschäfts-URL**
- **Angemeldete Benutzer** für dieses Geschäft

Dies ist hilfreich, wenn Sie mehrere Standorte oder Unternehmen verwalten.

## Mehrere Benutzer pro Geschäft

Jedes Geschäft kann mehrere Benutzer gleichzeitig angemeldet haben. Dies ist nützlich für:

- **Schichtwechsel** - Neuer Kassierer kann sich anmelden, bevor der vorherige sich abmeldet
- **Multi-Register-Setups** - Verschiedene Kassierer auf verschiedenen Geräten
- **Schnelles Wechseln** - Einfach zwischen Benutzerkonten wechseln

### Hinzufügen eines Benutzers

Klicken Sie auf die <Icon name="plus" />-Schaltfläche neben "Angemeldete Benutzer", um einen weiteren Benutzer zu diesem Geschäft hinzuzufügen.

### Benutzer wechseln

Klicken Sie auf ein Benutzerabzeichen (z. B. "Brenda"), um das POS als dieser Benutzer zu öffnen.

### Benutzer entfernen

Klicken Sie auf das **×** auf einem Benutzerabzeichen, um diesen Benutzer aus dem Geschäft abzumelden.

## Geschäft entfernen

Klicken Sie auf die rote **×**-Schaltfläche auf der Geschäftskarte, um es aus Ihrer Liste zu entfernen. Dadurch werden alle Benutzer abgemeldet und die Geschäftsanbindung aus der App entfernt.

## Demogeschäft

Am unteren Bildschirmrand sehen Sie möglicherweise einen Link "Demogeschäft betreten". Dadurch werden Sie mit einem Demogeschäft von WooCommerce verbunden, um die Funktionen von WCPOS auszuprobieren, ohne Ihre eigenen Geschäftsdaten zu beeinträchtigen.

## Fehlersuche

<Accordion>
  <AccordionItem question="Kann mich nicht mit meinem Geschäft verbinden?">

  - Stellen Sie sicher, dass das WCPOS-Plugin auf Ihrer WordPress-Seite installiert und aktiviert ist
  - Überprüfen Sie, ob Sie die richtige URL verwenden (einschließlich `https://`, wenn Ihre Seite SSL verwendet)
  - Überprüfen Sie, ob die WooCommerce REST API zugänglich ist
  - Stellen Sie sicher, dass Ihr Benutzerkonto über POS-Zugriffsberechtigungen verfügt

  </AccordionItem>

  <AccordionItem question="Verbindung schlägt ständig fehl?">

  - Versuchen Sie zuerst, `yourdomain.com/pos` in einem Webbrowser aufzurufen, um zu überprüfen, ob das Plugin funktioniert
  - Überprüfen Sie die Fehlerprotokolle Ihrer Seite auf etwaige Probleme
  - Stellen Sie sicher, dass Ihr Server die [Mindestanforderungen](/getting-started/installation) erfüllt

  </AccordionItem>
</Accordion>
