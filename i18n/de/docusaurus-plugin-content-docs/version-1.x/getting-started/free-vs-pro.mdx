WCPOS ist in zwei Editionen erhältlich. Das **kostenlose Plugin** bietet alles, was Sie benötigen, um Zahlungen an der Kasse entgegenzunehmen, Belege zu drucken und Bestellungen mit WooCommerce zu synchronisieren. **Pro** fügt die Verwaltungsansichten (Produkte, Bestellungen, Kunden, Berichte), weitere Zahlungsgateways, Gutscheine, Rückerstattungen und Unterstützung mehrerer Filialen hinzu.

Wenn Sie hauptsächlich eine Kasse benötigen, die Verkäufe erfasst, reicht das kostenlose Plugin aus. Wenn Sie auch den täglichen Betrieb Ihrer Filiale im POS verwalten möchten — Lagerbestände pflegen, vergangene Bestellungen nachschlagen, Rückerstattungen vornehmen, Kartenzahlungen über ein integriertes Lesegerät akzeptieren — dafür ist Pro gedacht.

:::note Kunden-/Gutscheinzugriff nur im POS in der Free-Version
Mit dem kostenlosen Plugin können Sie weiterhin einen vorhandenen Kunden **auswählen** oder den Gutscheinkatalog an der Kasse **einsehen** — Sie können jedoch keine Kunden hinzufügen oder bearbeiten und keine Gutscheincodes beim Checkout einlösen. Diese Funktionen werden mit Pro verfügbar.
:::

## Was Pro bietet {#what-pro-adds}

Pro schaltet den Rest des Backoffice, weitere Zahlungsoptionen und filialspezifische Funktionen frei. Alle folgenden Punkte sind ausschließlich in Pro verfügbar.

### Verwaltungsbildschirme {#management-screens}

Das kostenlose Plugin bietet ein Produktpanel zum Abkassieren. Pro ergänzt die vollständige Verwaltungsoberfläche:

| Bildschirm | Funktionen |
|---|---|
| **[Produkte](/products)** | Lagerbestände, Preise und Wareneinsatzkosten direkt bearbeiten. Massenoperationen über den gesamten Katalog. |
| **[Bestellungen](/orders)** | Vergangene Bestellungen nachschlagen, Belege erneut drucken, Bestellungen bearbeiten und auf den Erstattungsablauf zugreifen. |
| **[Kunden](/customers)** | Neue Kunden hinzufügen und bestehende Kundendaten bearbeiten (kostenlose Nutzer können an der Kasse nur aus einer Auswahlliste wählen). |
| **[Berichte](/reports)** | Tagesabschlussberichte — Summen nach Zahlungsmethode, Kassierer und Filiale, geeignet für den Kassenabgleich. |
| **[Gutscheine](/coupons)** | Den WooCommerce-Gutscheinkatalog an der Kasse durchsuchen (kostenlose Nutzer sehen nur eine unscharfe Vorschau). |

### Zahlungsgateways {#payment-gateways}

Das "Karte"-Gateway des kostenlosen Plugins zeichnet lediglich einen Verkauf auf, der an einem externen Terminal durchgeführt wurde — es gibt keine Integration mit dem Lesegerät selbst. Pro ermöglicht die Nutzung **jedes WooCommerce-Zahlungsgateways** im POS und enthält mehrere integrierte Gateways:

| Gateway | Anwendungsfall |
|---|---|
| **[Stripe Terminal](/payment/gateways/stripe-terminal)** | Direkte Integration mit den S700-/WisePOS-E-Lesegeräten von Stripe. Unterstützt MOTO (telefonische Bestellungen) und einen Simulator zum Testen. |
| **[SumUp Terminal](/payment/gateways/sumup-terminal)** | Ein SumUp-Kartenlesegerät mit dem POS koppeln und Kartenzahlungen abschließen, ohne die Kasse zu verlassen. |
| **[Vipps MobilePay](/payment/gateways/vipps-mobilepay)** | Telefonbasierte Zahlungen per QR-Code oder Push-Benachrichtigung — Vipps in Norwegen, MobilePay in Dänemark und Finnland. |
| **[E-Mail-Rechnung](/payment/gateways/email-invoice)** | Dem Kunden wird per E-Mail ein Zahlungslink gesendet; die Bestellung wird abgeschlossen, sobald die Online-Zahlung erfolgt ist. |
| **[Web-Checkout](/payment/gateways/web-checkout)** | Der Kunde wird zum gehosteten Checkout weitergeleitet, um die Zahlung online abzuschließen. |
| **Jedes andere WooCommerce-Zahlungsgateway** | Mit der [Gateway-Vorlage](/payment/gateways) lässt sich jedes WooCommerce-Zahlungs-Plugin für den POS-Checkout einbinden. |

### Gutscheine, Erstattungen und der Workflow an der Kasse {#coupons-refunds-and-the-at-counter-workflow}

<FeatureGrid>
  <FeatureCard icon="gift" title="Gutscheine an der Kasse" variant="pro">

Gutscheine erscheinen als entfernbare Chips im Warenkorb, unterstützen aufeinanderfolgende Rabatte und berücksichtigen alle üblichen Regeln — Ablaufdatum, Nutzungslimits, Mindest-/Höchstbetrag sowie Produkt- und Kategoriebeschränkungen.

  </FeatureCard>
</FeatureGrid>

## Preise {#pricing}

Pro wird pro Website über [wcpos.com/pro](https://wcpos.com/pro) verkauft:

- **Jahreslizenz** — 129 $ / Jahr, einschließlich Updates und Support für die Laufzeit der Lizenz.
- **Lebenslange Lizenz** — einmalig 399 $, einschließlich Updates und Support für die gesamte Produktlebensdauer.

Preise gültig ab Mai 2026; Änderungen vorbehalten.

Wenn eine Lizenz abläuft, funktionieren die Pro-Funktionen weiterhin, aber es werden keine Updates und kein Support mehr bereitgestellt. Siehe [WCPOS Pro installieren](/getting-started/pro-license) für Aktivierungsschritte und Lizenz-Fehlerbehebung.

:::tip Noch unentschlossen?
Die öffentliche Demo unter [demo.wcpos.com/pos](https://demo.wcpos.com/pos) (Anmeldung `demo` / `demo`) kann jederzeit ausprobiert werden — sie enthält den vollständigen Pro-Funktionsumfang, sodass Verwaltungsbildschirme, Gutscheine und der Rückerstattungsablauf vor dem Kauf getestet werden können.
:::

## Bereit für das Upgrade? {#ready-to-upgrade}

Unter [WCPOS Pro installieren](/getting-started/pro-license) finden sich die Download- und Aktivierungsschritte.

- **v1.8+ Pro** — das kostenlose Plugin muss zuerst deaktiviert und gelöscht werden. Ab v1.8 ist Pro ein eigenständiges Plugin, das nicht zusammen mit der kostenlosen Version installiert werden sollte.
- **Älteres Pro (< v1.8)** — das kostenlose Plugin muss installiert bleiben, wenn die verwendete Pro-Version es erfordert.
