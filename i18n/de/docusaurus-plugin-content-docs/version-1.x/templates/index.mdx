Vorlagen ermöglichen es Ihnen, das Erscheinungsbild und den Inhalt verschiedener Ausgaben Ihres POS-Systems anzupassen. Sie können den Vorlageneditor erreichen, indem Sie zu: `WP Admin > POS > Vorlagen` navigieren.

Das Vorlagensystem bietet Flexibilität für Anpassungen:

- **Belegvorlagen** - Passen Sie gedruckte und digitale Belege an
- **Berichtsvorlagen** - Passen Sie das Layout von Berichten an _(kommt bald)_
- **E-Mail-Vorlagen** - Passen Sie E-Mail-Benachrichtigungen an _(kommt bald)_

## Vorlagentypen {#template-types}

### Standardvorlagen {#default-templates}

Automatisch aus Ihren Plugin- und Theme-Dateien erkannt. Diese können nicht gelöscht, aber kopiert werden, um benutzerdefinierte Versionen zu erstellen.

### Benutzerdefinierte Vorlagen {#custom-templates}

Von Benutzern erstellte Vorlagen, die Sie vollständig anpassen, bearbeiten und über die WordPress-Adminoberfläche verwalten können.
