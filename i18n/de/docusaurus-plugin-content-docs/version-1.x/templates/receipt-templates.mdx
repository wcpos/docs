Der Kassenbon-Vorlagen-Editor ermöglicht es Ihnen, das Aussehen und den Inhalt von Kassenbons zu personalisieren, die von Ihrem POS-System gedruckt werden. Sie können den Vorlagen-Editor aufrufen unter: `WP Admin > POS > Vorlagen`

## Übersicht

Das Vorlagensystem bietet zwei Arten von Vorlagen:

- **Standardvorlagen**: Automatisch erkannt aus Ihren Plugin- und Theme-Dateien. Diese können nicht gelöscht, aber kopiert werden, um benutzerdefinierte Versionen zu erstellen.
- **Benutzerdefinierte Vorlagen**: Vom Benutzer erstellte Vorlagen, die vollständig angepasst, bearbeitet und verwaltet werden können.

## Zugriff auf den Vorlagen-Editor

1. Navigieren Sie zum WordPress-Administrationsbereich
2. Gehen Sie zu **POS > Vorlagen**
3. Sie sehen eine Liste verfügbarer Vorlagen, organisiert in zwei Abschnitten:
   - **Standardvorlagen** (aus Plugin-/Theme-Dateien)
   - **Benutzerdefinierte Vorlagen** (Ihre eigenen Kreationen)

## Standardvorlagen

Das System erkennt Kassenbon-Vorlagen aus drei Quellen:

### Vorlagenquellen

| Quelle         | Beschreibung                                         | Status                               |
| -------------- | --------------------------------------------------- | ------------------------------------ |
| **Plugin**     | Zentrale Kassenbon-Vorlage mit WooCommerce POS       | Immer verfügbar                     |
| **Pro Plugin** | Erweiterte Vorlage von WooCommerce POS Pro          | Verfügbar mit aktivem Lizenzschlüssel|
| **Theme**      | Benutzerdefinierte Vorlage aus Ihrem aktiven Theme  | Verfügbar, wenn Theme Vorlage enthält|

### Vorlagen-Aktionen

Für jede Standardvorlage können Sie:

- **Vorschau**: Sehen, wie die Vorlage mit Beispieldaten aussieht
- **Kopieren**: Eine anpassbare Kopie erstellen
- **Aktivieren**: Als aktive Vorlage für Ihr POS festlegen

## Benutzerdefinierte Vorlagen erstellen

### Methode 1: Von Standardvorlage kopieren

1. Im Abschnitt **Standardvorlagen** die zu kopierende Vorlage finden
2. Auf **Kopieren** klicken
3. Namen für die benutzerdefinierte Vorlage eingeben
4. **Vorlage kopieren** anklicken
5. Die neu erstellte benutzerdefinierte Vorlage bearbeiten

### Methode 2: Von Grund auf neu erstellen

1. **Neue Vorlage hinzufügen** am Seitenobrand anklicken
2. Vorlagennamen eingeben
3. **Kassenbon** als Vorlagentyp auswählen
4. Im Editor mit dem Codieren beginnen
