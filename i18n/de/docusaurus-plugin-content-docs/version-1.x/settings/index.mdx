WCPOS hat drei Arten von Einstellungen, die nach ihrem Konfigurationsort organisiert sind.

## Wo man Einstellungen findet

### 1. WordPress Admin Einstellungen

Backend-Einstellungen, die im Dashboard von WordPress konfiguriert sind:

**Ort:** `WP Admin > POS > Einstellungen`

Diese steuern:
- Standardkarteneinstellungen für alle Kassierer
- Konfiguration des Produktbarcodefeldes
- Einstellungen für Bestell-E-Mails
- Berechtigungen für Benutzerrollen (wer auf POS zugreifen kann)
- Verfügbarkeit von Zahlungsmethoden

Siehe [WordPress Admin Einstellungen](/settings/wp-admin) für weitere Details.

### 2. POS Store Einstellungen

App-weite Einstellungen, die innerhalb der POS-Anwendung konfiguriert sind:

**Ort:** Benutzermenü (oben rechts) → Einstellungen oder `Ctrl + Shift + S`

Diese steuern:
- Optionen zur Steuerberechnung
- Konfiguration des Barcode-Scanners
- Tastenkombinationen
- Thema/Design

Siehe [Store Einstellungen](/settings/store) für weitere Details.

### 3. Anzeigeeinstellungen (Pro Bildschirm)

Jeder Bildschirm im POS hat seine eigenen Anzeigeeinstellungen zur Anpassung der Datentabelle:

**Ort:** Klicken Sie auf das Schieberegler-Symbol (<Icon name="sliders" />) auf einem beliebigen Bildschirm

Diese steuern:
- Welche Spalten sichtbar sind
- Optionen zur Spaltendarstellung
- Bildschirmspezifische Funktionen

Die Anzeigeeinstellungen sind in der Dokumentation jedes Bildschirms dokumentiert:
- [POS Produktpanel](/pos/product-panel#display-settings)
- [Warenkorb-Panel](/pos/cart#display-settings)
- [Produkte Bildschirm](/products#display-settings)
- [Bestellungen Bildschirm](/orders#display-settings)
- [Kunden Bildschirm](/customers#display-settings)
- [Berichte Bildschirm](/reports#display-settings)
