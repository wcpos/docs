WCPOS hat drei Arten von Einstellungen, die nach dem Ort ihrer Konfiguration organisiert sind.

## Wo man Einstellungen findet {#where-to-find-settings}

### 1. WordPress Admin Einstellungen {#1-wordpress-admin-settings}

Backend-Einstellungen, die in deinem WordPress-Dashboard konfiguriert sind:

**Ort:** `WP Admin > POS > Einstellungen`

Diese steuern:
- Standardwarenkorb-Kunden für alle Kassierer
- Konfiguration des Produktbarcode-Feldes
- Bestell-E-Mail-Einstellungen
- Berechtigungen für Benutzerrollen (wer auf POS zugreifen kann)
- Verfügbarkeit von Zahlungs-Gateways

Siehe [WordPress Admin Einstellungen](/settings/wp-admin) für Details.

### 2. POS Geschäftseinstellungen {#2-pos-store-settings}

App-weite Einstellungen, die innerhalb der POS-Anwendung konfiguriert sind:

**Ort:** Benutzermenü (oben rechts) → Einstellungen oder `Ctrl + Shift + S`

Diese steuern:
- Steuerberechnungsoptionen
- Konfiguration des Barcode-Scanners
- Tastenkombinationen
- Thema/ Erscheinungsbild

Siehe [Geschäftseinstellungen](/settings/store) für Details.

### 3. Anzeigeeinstellungen (Pro Bildschirm) {#3-display-settings-per-screen}

Jeder Bildschirm in der POS hat seine eigenen Anzeigeeinstellungen zur Anpassung der Datentabelle:

**Ort:** Klicke auf das Symbol für die Schieberegler (<Icon name="sliders" />) auf jedem Bildschirm

Diese steuern:
- Welche Spalten sichtbar sind
- Optionen zur Spaltenanzeige
- Bildschirm-spezifische Funktionen

Die Anzeigeeinstellungen sind in der Dokumentation jedes Bildschirms dokumentiert:
- [POS Produktpanel](/pos/product-panel#display-settings)
- [Warenkorb-Panel](/pos/cart#display-settings)
- [Produkten Bildschirm](/products#display-settings)
- [Bestellungen Bildschirm](/orders#display-settings)
- [Kunden Bildschirm](/customers#display-settings)
- [Berichte Bildschirm](/reports#display-settings)
