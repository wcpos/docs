WCPOS hat drei Arten von Einstellungen, die je nach Konfigurationsort organisiert sind.

## Wo Sie Einstellungen finden

### 1. WordPress Admin-Einstellungen

Backend-Einstellungen, die in Ihrem WordPress-Dashboard konfiguriert werden:

**Ort:** `WP Admin > POS > Einstellungen`

Diese steuern:
- Standard-Warenkorb-Kunde für alle Kassierer
- Produktstrichcode-Feldkonfiguration
- Bestell-E-Mail-Einstellungen
- Benutzerrolle-Berechtigungen (wer auf POS zugreifen kann)
- Verfügbarkeit von Zahlungsmethoden

Weitere Informationen finden Sie unter [WordPress Admin-Einstellungen](/settings/wp-admin).

### 2. POS Store-Einstellungen

App-weite Einstellungen, die innerhalb der POS-Anwendung konfiguriert werden:

**Ort:** Benutzermenü (oben rechts) → Einstellungen, oder `Strg + Umschalt + S`

Diese steuern:
- Optionen für Steuerberechnung
- Strichcodescanner-Konfiguration
- Tastenkombinationen
- Design/Erscheinungsbild

Weitere Informationen finden Sie unter [Store-Einstellungen](/settings/store).

### 3. Anzeigeeinstellungen (Pro Bildschirm)

Jeder Bildschirm in der POS-Anwendung hat seine eigenen Anzeigeeinstellungen zur Anpassung der Datentabelle:

**Ort:** Klicken Sie auf das Schieberegler-Symbol (<Icon name="sliders" />) auf einem beliebigen Bildschirm

Diese steuern:
- Sichtbare Spalten
- Spaltenanzeige-Optionen
- Bildschirmspezifische Funktionen

Die Anzeigeeinstellungen sind in der Dokumentation der einzelnen Bildschirme beschrieben:
- [POS-Produktbereich](/pos/product-panel#display-settings)
- [Warenkorb-Bereich](/pos/cart#display-settings)
- [Produkte-Bildschirm](/products#display-settings)
- [Bestellungen-Bildschirm](/orders#display-settings)
- [Kunden-Bildschirm](/customers#display-settings)
- [Berichte-Bildschirm](/reports#display-settings)
