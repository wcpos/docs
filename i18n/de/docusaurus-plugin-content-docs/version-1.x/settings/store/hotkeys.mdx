## Verfügbare Tastenkombinationen

| Tastenkombination | Aktion | Beschreibung |
|-----------------|--------|-------------|
| `Strg + Umschalt + S` | Einstellungen | Öffnet das POS-Einstellungspanel |
| `Strg + Umschalt + A` | POS | Navigiert zum Hauptbildschirm der Kasse |
| `Strg + Umschalt + P` | Produkte | Öffnet den Produktverwaltungsbildschirm |
| `Strg + Umschalt + O` | Aufträge | Öffnet den Auftragsverwaltungsbildschirm |
| `Strg + Umschalt + C` | Kunden | Öffnet den Kundenverwaltungsbildschirm |
| `Strg + Umschalt + ?` | Hilfe | Öffnet den Hilfe-/Supportbereich |
| `Strg + Umschalt + L` | Abmelden | Meldet die aktuelle Sitzung ab |

## Verwendung von Tastenkombinationen

So verwenden Sie eine Tastenkombination:

1. **Halten Sie die Modifikatortasten gedrückt** (`Strg + Umschalt`)
2. **Drücken Sie die Aktionstaste** (den Buchstaben, der der gewünschten Aktion entspricht)
3. **Lassen Sie alle Tasten los**, um den Befehl auszuführen

## Tipps für effiziente Nutzung

- **Merken Sie sich häufig verwendete Tastenkombinationen**: Konzentrieren Sie sich darauf, die Tastenkombinationen für die am häufigsten durchgeführten Aktionen zu lernen
- **Üben Sie die Tastenkombinationen**: Regelmäßige Nutzung hilft, die Bewegungsabläufe zu verinnerlichen
- **Nutzen Sie Tastenkombinationen während hektischer Phasen**: Diese können die Abläufe während Stoßzeiten erheblich beschleunigen

## Anpassung

:::info Funktionswunsch
Die Anpassung von Tastenkombinationen ist derzeit nicht verfügbar. Die oben aufgeführten Tastenkombinationen sind festgelegt und können nicht über die Einstellungsoberfläche geändert werden.

Wenn Sie Tastenkombinationen anpassen möchten, teilen Sie uns Ihr Feedback bitte auf [Discord](https://wcpos.com/discord) mit.
:::
