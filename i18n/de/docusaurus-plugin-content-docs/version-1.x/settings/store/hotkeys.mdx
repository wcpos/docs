## Verfügbare Tastenkombinationen

| Tastenkombination  | Aktion  | Beschreibung  |
|--------------------|---------|---------------|
| `Ctrl + Shift + S` | Einstellungen | Öffnet das POS-Einstellungsfeld |
| `Ctrl + Shift + A` | POS     | Navigiert zum Haupt-POS-Bildschirm |
| `Ctrl + Shift + P` | Produkte | Öffnet den Bildschirm zur Produktverwaltung |
| `Ctrl + Shift + O` | Bestellungen | Öffnet den Bildschirm zur Bestellverwaltung |
| `Ctrl + Shift + C` | Kunden | Öffnet den Bildschirm zur Kundenverwaltung |
| `Ctrl + Shift + ?` | Support | Öffnet den Support/Hilfebereich |
| `Ctrl + Shift + L` | Abmelden | Meldet sich von der aktuellen Sitzung ab |

## Verwendung von Tastenkombinationen

Um eine Tastenkombination zu verwenden:

1. **Halten Sie die Modifikatortasten gedrückt** (`Ctrl + Shift`)
2. **Drücken Sie die Aktionstaste** (den Buchstaben, der Ihrer gewünschten Aktion entspricht)
3. **Lassen Sie alle Tasten los**, um den Befehl auszuführen

## Tipps für eine effiziente Nutzung

- **Merken Sie sich häufig verwendete Tastenkombinationen**: Konzentrieren Sie sich darauf, die Tastenkombinationen für Aktionen zu lernen, die Sie am häufigsten ausführen
- **Üben Sie die Tastenkombinationen**: Regelmäßige Nutzung hilft, das Muskelgedächtnis aufzubauen
- **Verwenden Sie Tastenkombinationen in Stoßzeiten**: Tastenkombinationen können die Abläufe in Spitzenzeiten erheblich beschleunigen

## Anpassung

:::info Feature-Anfrage
Die Anpassung von Tastenkombinationen ist derzeit nicht verfügbar. Die oben aufgeführten Tastenkombinationen sind festgelegt und können nicht über die Einstellungen geändert werden.

Wenn Sie Tastenkombinationen anpassen möchten, teilen Sie uns bitte Ihr Feedback auf [Discord](https://wcpos.com/discord) mit.
:::
