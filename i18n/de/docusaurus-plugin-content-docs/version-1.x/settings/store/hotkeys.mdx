## Verfügbare Tastenkombinationen {#available-shortcuts}

| Tasten Kombination | Aktion | Beschreibung |
|-------------------|--------|--------------|
| `Strg + Shift + S` | Einstellungen | Öffnet das POS-Einstellungspanel |
| `Strg + Shift + A` | POS | Navigiert zum Haupt-POS-Bildschirm |
| `Strg + Shift + P` | Produkte | Öffnet den Bildschirm zur Produkteverwaltung |
| `Strg + Shift + O` | Bestellungen | Öffnet den Bildschirm zur Bestellverwaltung |
| `Strg + Shift + C` | Kunden | Öffnet den Bildschirm zur Kundenverwaltung |
| `Strg + Shift + ?` | Support | Öffnet den Support-/Hilfebereich |
| `Strg + Shift + L` | Abmelden | Meldet sich von der aktuellen Sitzung ab |

## Verwendung von Tastenkombinationen {#using-keyboard-shortcuts}

Um eine Tastenkombination zu verwenden:

1. **Halte die Modifikatortasten gedrückt** (`Strg + Shift`)
2. **Drücke die Aktionstaste** (den Buchstaben, der deiner gewünschten Aktion entspricht)
3. **Lasse alle Tasten los**, um den Befehl auszuführen

## Tipps für eine effiziente Nutzung {#tips-for-efficient-use}

- **Merke dir häufig verwendete Tastenkombinationen**: Konzentriere dich darauf, die Tastenkombinationen für Aktionen zu lernen, die du am häufigsten ausführst
- **Übe die Tastenkombinationen**: Regelmäßige Verwendung hilft, Muskelgedächtnis aufzubauen
- **Verwende Tastenkombinationen in Stoßzeiten**: Tastenkombinationen können die Abläufe während Stoßzeiten erheblich beschleunigen

## Anpassung {#customisation}

:::info Funktionsanfrage
Die Anpassung von Tastenkombinationen ist derzeit nicht verfügbar. Die oben aufgeführten Tastenkombinationen sind festgelegt und können nicht über die Einstellungsoberfläche geändert werden.

Wenn du Tastenkombinationen anpassen möchtest, teile uns bitte dein Feedback auf [Discord](https://wcpos.com/discord) mit.
:::
