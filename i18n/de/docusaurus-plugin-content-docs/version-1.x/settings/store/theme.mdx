## Available Themes

### System Theme

- **Beschreibung**: Folgt Ihren Geräteeinstellungen
- **Verhalten**: Wechselt automatisch zwischen hellen und dunklen Modi basierend auf den Systemeinstellungen Ihres Geräts

### Light Theme

- **Beschreibung**: Sauber und hell
- **Am besten geeignet für**: Gut beleuchtete Umgebungen und Nutzer, die traditionelle helle Schnittstellen bevorzugen

### Dark Theme

- **Beschreibung**: Augenschonend
- **Am besten geeignet für**: Beleuchtungen mit geringer Helligkeit und zur Reduzierung der Augenbelastung bei längerem Gebrauch

### Ocean Theme

- **Beschreibung**: Coole Blau- und Türkistöne
- **Am besten geeignet für**: Schaffung einer entspannten, professionellen Atmosphäre

### Sunset Theme

- **Beschreibung**: Warme Orangetöne und Violetttöne
- **Am besten geeignet für**: Schaffung einer gemütlichen, einladenden Atmosphäre

### Monochrome Theme

- **Beschreibung**: Hoher Kontrast in Graustufen
- **Am besten geeignet für**: Barrierefreiheitsbedürfnisse und Umgebungen, die hohen Kontrast benötigen

## Selecting a Theme

1. Im Theme-Einstellungsbereich sehen Sie alle verfügbaren Themes, die als Karten angezeigt werden
2. Jede Theme-Karte zeigt den Theme-Namen und eine kurze Beschreibung
3. Klicken Sie auf eine beliebige Theme-Karte, um sie auszuwählen
4. Das derzeit aktive Theme wird mit einem blauen Rand hervorgehoben
5. Ihre Auswahl wird automatisch gespeichert und sofort angewendet

## Tips for Theme Selection

| Theme | Beste Umgebung |
|-------|------------------|
| **System** | Wenn Sie möchten, dass das POS automatisch Ihrem Gerät entspricht |
| **Light** | Helle Einzelhandelsumgebungen mit guter Beleuchtung |
| **Dark** | Schlecht beleuchtete Umgebungen oder Abend/Nachtschichten |
| **Ocean/Sunset** | Persönlichkeit und Markenabgleich hinzufügen |
| **Monochrome** | Wenn Sie maximalen Kontrast benötigen |

Das von Ihnen ausgewählte Theme bleibt während aller Ihrer POS-Sitzungen erhalten.
