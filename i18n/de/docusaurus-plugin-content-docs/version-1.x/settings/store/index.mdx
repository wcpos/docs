Die Shop-Einstellungen sind appweite Konfigurationen, die beeinflussen, wie das POS funktioniert. Diese Einstellungen gelten für den gesamten Shop, nicht für einzelne Bildschirme.

## Zugriff auf die Shop-Einstellungen

Öffnen Sie die Shop-Einstellungen durch:

- Klicken Sie auf das **Benutzermenü** (oben rechts) und wählen Sie **Einstellungen**
- Verwenden Sie die Tastenkombination `Ctrl + Shift + S`

## Einstellungs-Tabs

Das Modal für die Shop-Einstellungen enthält diese Tabs:

| Tab | Beschreibung |
|-----|--------------|
| [Allgemein](./general) | Shop-Info, Sprache, Währung, Standardkunde |
| [Steuern](./tax) | Steuerberechnung und Anzeige-Optionen |
| [Barcode](./barcode) | Konfiguration des Barcode-Scanners |
| [Tastenkombinationen](./hotkeys) | Verfügbare Tastenkombinationen |
| [Thema](./theme) | Erscheinungsbild und Farbschema |
