Die Store-Einstellungen sind appweite Konfigurationen, die beeinflussen, wie das POS arbeitet. Diese Einstellungen gelten für den gesamten Store, nicht für einzelne Bildschirme.

## Zugriff auf die Store-Einstellungen

Öffnen Sie die Store-Einstellungen durch:

- Klicken Sie auf das **Benutzermenü** (oben rechts) und wählen Sie **Einstellungen**
- Verwenden Sie die Tastenkombination `Ctrl + Shift + S`

## Einstellungs-Tabs

Das Modal für die Store-Einstellungen enthält diese Tabs:

| Tab | Beschreibung |
|-----|--------------|
| [General](./general) | Store-Info, Sprache, Währung, Standardkunde |
| [Tax](./tax) | Steuerberechnung und Anzeigeoptionen |
| [Barcode](./barcode) | Barcode-Scanner-Konfiguration |
| [Keyboard Shortcuts](./hotkeys) | Verfügbare Tastenkombinationen |
| [Theme](./theme) | Erscheinungsbild und Farbschema |
