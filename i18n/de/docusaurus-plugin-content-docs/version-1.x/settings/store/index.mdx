Die Geschäftseinstellungen sind anwendungsweite Konfigurationen, die beeinflussen, wie das POS funktioniert. Diese Einstellungen gelten für das gesamte Geschäft, nicht für einzelne Bildschirme.

## Zugriff auf Geschäftseinstellungen

Öffnen Sie die Geschäftseinstellungen durch:

- Klicken auf das **Benutzermenü** (obere rechte Ecke) und Auswahl von **Einstellungen**
- Verwenden der Tastenkombination `Ctrl + Shift + S`

## Einstellungsregisterkarten

Das Modal für die Geschäftseinstellungen enthält diese Registerkarten:

| Registerkarte | Beschreibung |
|---------------|--------------|
| [Allgemein](./general) | Geschäftsinformationen, Sprache, Währung, Standardkunde |
| [Steuer](./tax) | Steuerberechnungs- und Anzeigeoptionen |
| [Barcode](./barcode) | Konfiguration des Barcode-Scanners |
| [Tastenkombinationen](./hotkeys) | Verfügbare Tastenkombinationen |
| [Thema](./theme) | Erscheinungsbild und Farbschema |
