Geschäftseinstellungen sind App-weite Konfigurationen, die beeinflussen, wie die POS funktioniert. Diese Einstellungen gelten für den gesamten Store, nicht für einzelne Bildschirme.

## Zugriff auf Geschäftseinstellungen

Öffnen Sie die Geschäftseinstellungen durch:

- Klicken Sie auf das **Benutzermenü** (oben rechts) und wählen Sie **Einstellungen**
- Verwenden Sie die Tastenkombination `Strg + Umschalt + S`

## Einstellungs-Tabs

Der Geschäftseinstellungen-Dialog enthält folgende Tabs:

| Tab | Beschreibung |
|-----|-------------|
| [Allgemein](./general) | Store-Informationen, Sprache, Währung, Standardkunde |
| [Steuern](./tax) | Steuerberechnungs- und Anzeigeoptionen |
| [Barcode](./barcode) | Barcode-Scanner-Konfiguration |
| [Tastaturkurzbefehle](./hotkeys) | Verfügbare Tastaturkurzbefehle |
| [Design](./theme) | Aussehen und Farbschema |
