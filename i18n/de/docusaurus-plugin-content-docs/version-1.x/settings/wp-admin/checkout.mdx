Die Seite Checkout-Einstellungen (`WP Admin > POS > Settings > Checkout`) steuert, welche Zahlungsgateways im POS angezeigt werden und wie Bestellungen verarbeitet werden.

## Zahlungsgateways {#payment-gateways}

In diesem Abschnitt werden alle auf Ihrer Website installierten WooCommerce-Zahlungsgateways aufgelistet. Jedes Gateway kann unabhängig für den POS aktiviert oder deaktiviert werden.

| Spalte | Beschreibung |
|--------|-------------|
| **Aktiviert** | Das Gateway für den POS-Checkout ein- oder ausschalten |
| **Gateway** | Der Name des Zahlungsgateways |
| **Gateway-ID** | Die WooCommerce-Gateway-Kennung (nützlich für Entwickler) |

### Ein Gateway aktivieren {#enabling-a-gateway}

1. Suchen Sie das Gateway in der Liste
2. Aktivieren Sie den Schalter **Enabled**
3. Das Gateway erscheint beim nächsten Laden der Seite im POS-Checkout

### Reihenfolge der Gateways ändern {#reordering-gateways}

Ziehen Sie die Gateways, um ihre Anzeigereihenfolge im POS-Checkout zu ändern. Das erste aktivierte Gateway ist standardmäßig ausgewählt, wenn ein Kassierer das Checkout-Modal öffnet.

### POS vs. Online-Shop {#pos-vs-online-store}

Die Gateway-Einstellungen hier sind **unabhängig** von Ihren WooCommerce-Zahlungseinstellungen. Ein Gateway kann sein:

- **Für den POS aktiviert, online deaktiviert** – nützlich für Barzahlungen, die für Online-Bestellungen nicht relevant sind
- **Online aktiviert, für den POS deaktiviert** – nützlich für Gateways wie PayPal, die für den Verkauf vor Ort nicht praktikabel sind
- **Für beide aktiviert** – das Gateway funktioniert überall

:::note
Ein Gateway muss zunächst installiert und aktiv sein in `WP Admin > WooCommerce > Settings > Payments` bevor es in der Liste der Checkout-Einstellungen im POS erscheint. Falls ein Gateway fehlt, prüfen Sie, ob das Plugin installiert und das Gateway in WooCommerce aktiviert ist.
:::

## Bestellstatus {#order-status}

Legen Sie den Standard-Bestellstatus für abgeschlossene POS-Bestellungen fest. Standardmäßig werden POS-Bestellungen auf **Abgeschlossen** gesetzt, da die Zahlung am Verkaufspunkt erfolgt.
