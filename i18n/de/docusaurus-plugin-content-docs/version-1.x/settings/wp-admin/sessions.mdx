Auf dem Bildschirm **Sitzungen** sehen Sie, wer aktuell im POS angemeldet ist, auf welchen Geräten, und können einzelne Geräte per Fernzugriff abmelden. Dies ist die richtige Stelle, wenn ein Gerät verloren gegangen ist oder gestohlen wurde, wenn ein Kassierer ausscheidet oder wenn Sie einfach aktive Anmeldungen prüfen möchten.

Sie finden den Bildschirm unter **`WP Admin > POS > Einstellungen > Sessions`**.

> _Aktive Benutzersitzungen verwalten. Sie können alle angemeldeten Benutzer anzeigen, deren Geräte einsehen und Sitzungen bei Bedarf beenden._

## Funktionsweise {#how-it-works}

Der Bildschirm hat zwei Bereiche:

- **Benutzer** (links) — jeder Benutzer mit mindestens einer aktiven Sitzung, mit einem grünen Punkt bei Personen, die in den letzten Minuten aktiv waren, einer **Sie**-Kennzeichnung für Ihr eigenes Konto und der Anzahl der Sitzungen pro Benutzer. Über ein Filterfeld können Sie nach Name oder Benutzername suchen.
- **Sitzungen** (rechts) — wählen Sie einen Benutzer aus, um pro aktiver Sitzung eine Karte anzuzeigen.

Jede Sitzungskarte zeigt:

| Feld | Bedeutung |
| --- | --- |
| **Anwendung** | Wo der Benutzer angemeldet ist — **iOS-App**, **Android-App**, **Desktop-App** oder **Web-App** — plus Browser/Version |
| **Aktuelles Gerät** | Eine Markierung für die Sitzung, die gerade verwendet wird |
| **Zuletzt aktiv** | Wie kürzlich die Sitzung eine Anfrage gestellt hat (z. B. _Gerade eben_, _vor 5 Minuten_) |
| **IP** | Die IP-Adresse, von der aus sich die Sitzung verbunden hat (oder _N/A_) |
| **Erstellt** | Wann die Sitzung erstmals angemeldet wurde |
| **Läuft ab** | Wann das Sitzungstoken abläuft |
| **User-Agent** | Aufklappen, um den unverarbeiteten User-Agent-String des Browsers/Geräts anzuzeigen |

Eine Sitzung wird als **Jetzt aktiv** angezeigt, wenn sie innerhalb der letzten 5 Minuten eine Anfrage gestellt hat.

## Geräte abmelden {#signing-devices-out}

Es sind drei Aktionen verfügbar, jeweils mit einem Bestätigungsdialog:

- **Beenden** — meldet eine einzelne Sitzung ab. Verwenden Sie die Schaltfläche auf der Karte dieser Sitzung.
- **Andere Geräte abmelden** — meldet den ausgewählten Benutzer von allen Geräten ab, **außer** von dem Gerät, das er derzeit verwendet. Wird angezeigt, wenn der Benutzer mehr als eine Sitzung hat.
- **Alle abmelden** — meldet den ausgewählten Benutzer von **jedem** Gerät ab, einschließlich des aktuellen Geräts.

:::tip Verlorenes oder gestohlenes Gerät
Wenn ein Telefon, Tablet oder Laptop mit installiertem POS verloren geht, öffnen Sie Sitzungen, wählen Sie den betroffenen Benutzer aus und verwenden Sie **Beenden** für die Sitzung dieses Geräts (oder **Alle abmelden**, um auf Nummer sicher zu gehen). Das Gerät verliert beim nächsten Request den Zugriff.
:::

:::note
Wenn Sie Ihre eigene aktuelle Sitzung beenden, werden Sie von dem Gerät abgemeldet, auf dem Sie sich gerade befinden — vorher werden Sie zur Bestätigung aufgefordert.
:::
