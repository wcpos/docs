Diese Einstellungen konfigurieren die Backend-Funktionalität der POS-Anwendung, einschließlich PHP- und WooCommerce-Konfigurationen. 
Sie können darauf zugreifen, indem Sie navigieren zu: `WP Admin > POS > Einstellungen`

Wichtige Konfigurationen umfassen:

- Standardkundeneinstellungen für alle Kassierer.
- Produktbarcode-Feld.
- E-Mail-Einstellungen für Bestellungen.
- Benutzerrollenberechtigungen.
