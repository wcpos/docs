Diese Einstellungen konfigurieren die Backend-Funktionalität der POS-Anwendung, einschließlich PHP- und WooCommerce-Konfigurationen. 
Sie können darauf zugreifen, indem Sie zu: `WP Admin > POS > Einstellungen` navigieren

Wichtige Konfigurationen sind:

- Standardwarenkorb-Kunde für alle Kassierer.
- Produktbarcode-Feld.
- Bestell-E-Mail-Einstellungen.
- Benutzerrollen-Berechtigungen.
