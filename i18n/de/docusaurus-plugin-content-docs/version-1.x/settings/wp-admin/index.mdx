Diese Einstellungen konfigurieren die Backend-Funktionalität der POS-Anwendung, einschließlich PHP- und WooCommerce-Konfigurationen. 
Sie können sie aufrufen über: `WP Admin > POS > Einstellungen`

Wichtige Konfigurationen umfassen:

- Standard-Warenkorb-Kunde für alle Kassierer.
- Produkt-Barcode-Feld.
- Einstellungen für Bestellbestätigungen.
- Benutzerrolle-Berechtigungen.
