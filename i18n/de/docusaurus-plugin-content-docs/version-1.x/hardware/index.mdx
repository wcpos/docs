WCPOS funktioniert mit den physischen Geräten, die bereits an der Kasse im Einsatz sind. Diese Seite ist der Ausgangspunkt für die Verbindung von Hardware — Belegdrucker, Barcodescanner, Kartenleser und Kassenschubladen. Jeder Gerätetyp hat entweder eine eigene Einrichtungsseite in diesem Abschnitt oder verweist auf die Seite, auf der die Einrichtung tatsächlich beschrieben wird.

:::note Ein Ort pro Gerät
Der Hardware-Hub verweist auf die Einrichtung jedes Geräts, die eigentlichen Schritte befinden sich jedoch an einem einzigen Ort. Das Konto und der Kopplungsvorgang eines Kartenlesers befinden sich auf der jeweiligen Zahlungsgateway-Seite; die Öffnungseinstellung einer Kassenschublade befindet sich auf der Seite Bondrucker.
:::
