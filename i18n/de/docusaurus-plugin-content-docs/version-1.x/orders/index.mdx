# Bestellungen <span class="badge badge--primary">Pro</span>

:::info Pro Feature
Der Bestellbildschirm erfordert [WCPOS Pro](/getting-started/pro-license).
:::

Der Bestellbildschirm bietet Zugriff auf Ihre Bestellhistorie direkt im POS. Sehen Sie sich vergangene Bestellungen an, drucken Sie Belege erneut und verwalten Sie Bestelldetails, ohne zum WooCommerce-Admin zu wechseln.

{/* Screenshot: orders-screen.png */}

## Schnittstellenübersicht {#interface-overview}

### Suche & Filter {#search-filters}

Oben auf dem Bildschirm:

- **Suchleiste** - Finden Sie Bestellungen nach Nummer, Kundenname usw.
- **Statusfilter** - Filtern nach Bestellstatus (Abgeschlossen, In Bearbeitung usw.)
- **Kundenfilter** - Filtern nach Kunde
- **Kassiererfilter** - Filtern nach dem, der die Bestellung bearbeitet hat
- **Geschäftsfilter** - Filtern nach Geschäft (für Multi-Store-Setups)
- **Datumsbereich** - Filtern nach Datum

Aktive Filter erscheinen als blaue Abzeichen, die durch Klicken auf das × entfernt werden können.

### Bestelltabelle {#orders-table}

Der Hauptbereich zeigt Bestellungen mit:

- **Statussymbol** - Visueller Indikator (✓ abgeschlossen, 🕐 ausstehend, 🛒 offen)
- **Bestellnummer** - Eindeutige Bestell-ID
- **Kunde** - Kundenname (oder "Gast")
- **Rechnungsadresse** - Rechnungsinformationen des Kunden
- **Hinweisindikator** - Sprechblase, wenn die Bestellung Notizen hat
- **Erstellungsdatum** - Relative Zeitstempel ("vor 2 Stunden")
- **Kassierer** - Wer die Bestellung bearbeitet hat
- **Zahlungsmethode** - Bargeld, Karte usw.
- **Gesamt** - Gesamtbetrag der Bestellung
- **Aktionen** - Drei-Punkte-Menü

### Fußzeile {#footer}

- Bestellanzahl mit Synchronisierungsbutton (<Icon name="arrow-rotate-right" />). **Langes Drücken** für Optionen "Clear and Refresh"

## Hauptmerkmale {#key-features}

### Bestellsuche {#order-lookup}

Finden Sie Bestellungen schnell mit:

- **Bestellnummernsuche**
- **Kundenname-Suche**
- **Datumsbereich-Filterung**
- **Status-Filterung**
- **Kassierer-Filterung**

### Belegneuauflage {#receipt-reprinting}

Für jede Bestellung können Sie:

1. Klicken Sie auf das Drei-Punkte-Menü
2. Wählen Sie die Belegoption
3. Beleg anzeigen, drucken oder per E-Mail senden

### Bestelldetails {#order-details}

Klicken Sie auf eine Bestellung, um anzuzeigen:

- Vollständige Artikelinformationen
- Kundeninformationen
- Zahlungsdetails
- Bestellnotizen
- Metadaten

### Verfügbare Spalten {#available-columns}

| Spalte | Beschreibung |
|--------|-------------|
| **Status** | Bestellstatussymbol |
| **Bestellnummer** | Eindeutige Bestell-ID |
| **Kunde** | Kundenname und Details |
| **Rechnungsadresse** | Rechnungsinformationen |
| **Lieferadresse** | Versandinformationen |
| **Kundenhinweis** | Notizen vom Kunden |
| **Erstellungsdatum** | Wann die Bestellung aufgegeben wurde |
| **Änderungsdatum** | Letzte Änderung |
| **Abschlussdatum** | Wann die Bestellung abgeschlossen wurde |
| **Zahlungsdatum** | Wann die Zahlung eingegangen ist |
| **Erstellt über** | Bestellquelle (POS, Online usw.) |
| **Kassierer** | Wer die Bestellung bearbeitet hat |
| **Zahlungsmethode** | Verwendeter Zahlungstyp |
| **Gesamt** | Bestellsumme |
| **Beleg** | Schnellzugriff auf Beleg |
| **Aktionen** | Bearbeiten, anzeigen usw. |

## Bestellaktionen {#order-actions}

Klicken Sie auf das Drei-Punkte-Menü (⋮) für Optionen:

- **Anzeigen** - Vollständige Bestelldetails anzeigen
- **Bearbeiten** - Bestellinformationen ändern
- **Beleg** - Beleg anzeigen/drucken/per E-Mail senden
- **Synchronisieren** - Bestellung vom Server aktualisieren
