WCPOS umfasst ein vollständiges Vorlagensystem für Belege, Rechnungen, Geschenkbelege, Lieferscheine, Küchenbons und mehr — alles verwaltet unter **WP Admin → POS → Vorlagen**.

:::tip Sie möchten nur Ihren Beleg ändern?
Unter **[Ihren Beleg anpassen](/receipts/customise)** finden Sie die drei einfachsten Möglichkeiten: eine andere mitgelieferte Vorlage auswählen, eine Vorlage per KI anpassen lassen oder sie manuell bearbeiten.
:::

## Vorlagen-Engines {#template-engines}

WCPOS unterstützt drei Vorlagen-Engines. Die Engine wird ausgewählt, wenn Sie eine Vorlage erstellen oder auf einer Galeriekarte auf **Vorlage verwenden** klicken, und **kann anschließend nicht geändert werden** — erstellen Sie eine neue Vorlage, wenn Sie die Engine wechseln möchten.

<FeatureGrid>
  <FeatureCard icon="code" title="Logikfreies HTML (empfohlen)">
    Im Mustache-Stil <code>{"{{variable}}"}</code> Platzhalter in einfachem HTML. Wird clientseitig gerendert, daher funktionieren Belege **offline**. Für Browserdruck und PDF-Belege verwenden.
  </FeatureCard>
  <FeatureCard icon="print" title="Thermisches XML">
    XML-Vorlagen, die sowohl eine Bildschirmvorschau als auch **ESC/POS-Befehle** für Thermobondrucker erzeugen. Dieselbe Vorlage, zwei Ausgaben. Für Epson, Star und andere Thermodrucker verwenden.
  </FeatureCard>
  <FeatureCard icon="terminal" title="Legacy-PHP">
    Serverseitige PHP-Vorlagen mit WooCommerce-Funktionen. Zur Abwärtskompatibilität mit bestehenden <code>yourtheme/woocommerce-pos/receipt.php</code> Überschreibungen. Zum Rendern ist eine Serververbindung erforderlich.
  </FeatureCard>
</FeatureGrid>

## Vorlagengalerie {#template-gallery}

Die Galerie enthält gebrauchsfertige Vorlagen, die unverändert verwendet oder angepasst werden können. Nach **Kategorie** (Beleg, Rechnung, Geschenkbeleg, Küchenbon, Angebot / Bestellung), **Format** (HTML / ESC/POS) und **Richtung** (LTR / RTL) filtern; Vorschau mit den echten Daten Ihrer Filiale anzeigen. Im Fußbereich jeder Karte befinden sich ein Link **Vorschau** und die primäre Schaltfläche **Vorlage verwenden** — durch Klicken auf Vorlage verwenden wird eine bearbeitbare Kopie in *Ihre Vorlagen* erstellt.

### Dokumentvorlagen von Drittanbietern {#third-party-document-templates}

Wenn **PDF Invoices & Packing Slips for WooCommerce** von WP Overnight aktiv ist, stellt WCPOS auch **Rechnung (WP Overnight)** und **Lieferschein (WP Overnight)** in der Liste der Belegvorlagen bereit. Diese serverseitig gerenderten HTML-Vorlagen verwenden die Dokument-API von WP Overnight und übernehmen die in diesem Plugin konfigurierten Nummerierungen, Layouts, das Branding, Rechts- und Steuerfelder sowie Vorlagenanpassungen für Rechnungen und Lieferscheine.

Sie sind nur verfügbar, solange das WP Overnight-Plugin aktiv ist, und erfordern zum Rendern eine Serververbindung. Verwenden Sie die integrierten HTML- oder Thermovorlagen von WCPOS, wenn Offline-Druck erforderlich ist.

### Adaptive Steueranzeige {#adaptive-tax-display}

Die meisten integrierten Vorlagen (Standard, Standard RTL, Minimal/Modern, Schmal, Rechnung, Angebot, Varianten von Thermisch Einfach) **passen sich automatisch an Ihre WooCommerce-Einstellung zur Steueranzeige an**:

- **Filialen mit Preisen inklusive Steuern** (EU/UK/AU) sehen Bruttopreise mit einer Zeile „Steuern enthalten“.
- **Filialen mit Preisen zuzüglich Steuern** (US/CA) sehen Nettopreise, bei denen die Steuer als separate Zeile hinzugefügt wird.
- Die Gesamtsumme zeigt stets den tatsächlich berechneten Bruttobetrag.

Die Vorlagenfamilie **Detailliert** ist eine formelle Steuerrechnung und weist die Steuer stets einzeln aus, mit Nettopreiszeilen und einer expliziten Aufschlüsselung, unabhängig von der Filialeinstellung.

## So funktioniert es {#how-it-works}

1. **Vorlage hinzufügen** — Klicken Sie auf einer Galeriekarte auf **Vorlage verwenden**, oder erstellen Sie eine eigene Vorlage von Grund auf.
2. **Anpassen** Sie die Vorlage im In-App-Editor mit Live-Vorschau, oder fügen Sie sie in ChatGPT / Claude ein und bitten Sie um die gewünschten Änderungen.
3. **Aktivieren** Sie die Vorlage — schalten Sie in *Ihre Vorlagen* den Umschalter **Aktiv** ein. Jede aktive Belegvorlage erscheint im Dropdown-Menü des Belegbildschirms an der Kasse.
4. **Drucken oder anzeigen** — Belege werden nach dem Checkout mit Optionen zum Drucken, Versenden per E-Mail oder Anzeigen auf dem Bildschirm geöffnet. Ihre Vorlagen funktionieren auch außerhalb der Kasse: Online-Kunden können [Belege über Mein Konto herunterladen](/receipts/storefront), die im selben Design erstellt werden.

Vorlagen werden anhand einer standardisierten [Belegdaten-Payload](/receipts/receipt-data) mit Abschnitten für Filialinformationen, Positionen, Summen, Steuern, Zahlungen, Rückerstattungen und mehr erstellt. Alle Währungsfelder enthalten vorformatierte `_display`-Varianten (z. B. `$29.99` statt `29.99`).

## Pro-Funktionen {#pro-features}

Mit [WCPOS Pro](/getting-started/pro-license) kann jede Filiale eigene Vorlagenzuweisungen, Branding (Logo, Adresse, Kontaktdaten) und eine eigene Vorlagenreihenfolge haben — nützlich für Betriebe mit mehreren Standorten, in denen jede Filiale ihr eigenes Briefpapier druckt.
