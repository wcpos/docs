Nach erfolgreicher Zahlungsabwicklung wird das Belegmodal angezeigt. Es zeigt eine Zusammenfassung der abgeschlossenen Bestellung und bietet Optionen zum Drucken oder Versenden des Belegs per E-Mail.

{/* Screenshot: receipt-modal.png */}

## Beleginhalt {#receipt-contents}

Der Beleg zeigt:

### Filialinformationen {#store-information}
- **Filialname**
- **Adresse**
- **Telefonnummer**
- **E-Mail**
- **Website**
- **Logo** (falls konfiguriert)

### Belegkopf {#receipt-header}
- **Belegart** (z. B. "Steuerbeleg")
- **Bestellnummer**
- **Datum und Uhrzeit**
- **Name des Kassierers**
- **Zahlungsmethode**

### Bestelldetails {#order-details}
- **Positionen** mit:
  - Produktname
  - Variationsattribute (falls zutreffend)
  - Menge
  - Preis
  - Gesamt

### Summen {#totals}
- **Zwischensumme**
- **Steuer** (falls zutreffend)
- **Gesamt**
- **Erhaltener Betrag** (bei Barzahlung)
- **Wechselgeld** (bei Barzahlung)

### Kundeninformationen {#customer-information}
- **Rechnungsadresse**
- **Lieferadresse** (falls abweichend)

## Belegaktionen {#receipt-actions}

### Schließen {#close}
Schließen Sie das Belegmodal und kehren Sie zum POS zurück.

### Beleg per E-Mail senden {#email-receipt}
Senden Sie den Beleg an die E-Mail-Adresse des Kunden:

1. Klicken Sie auf **Beleg per E-Mail senden**
2. Bestätigen Sie die E-Mail-Adresse
3. Der Beleg wird über das E-Mail-System von WooCommerce gesendet

Dem Kunden muss eine E-Mail-Adresse für die Bestellung zugeordnet sein.

### Beleg drucken {#print-receipt}
Drucken Sie eine physische Kopie des Belegs:

1. Klicken Sie auf **Beleg drucken**
2. Der Druckdialog Ihres Browsers wird geöffnet
3. Wählen Sie Ihren Bondrucker aus und drucken Sie

:::tip Bondrucker
WCPOS funktioniert mit jedem Drucker, der über Ihren Browser erreichbar ist. Für beste Ergebnisse verwenden Sie:
- Einen dedizierten Bondrucker (Thermodrucker sind üblich)
- Konfigurieren Sie das Papierformat in Ihren Druckereinstellungen
- Richten Sie eine Druckvoreinstellung für schnelles Drucken ein
:::

## Automatische Belegoptionen {#automatic-receipt-options}

In den [Warenkorb-Anzeigeeinstellungen](/pos/cart#display-settings) können Sie Folgendes konfigurieren:

- **Beleg nach dem Checkout automatisch anzeigen** - Öffnet das Beleg-Modal direkt nach der Zahlung
- **Beleg nach dem Checkout automatisch drucken** - Löst den Druck automatisch aus, wenn der Beleg angezeigt wird

Diese Optionen beschleunigen Checkout-Workflows mit hohem Bestellaufkommen.

## Belege anpassen {#customising-receipts}

Das Erscheinungsbild von Belegen wird über Vorlagen in WordPress gesteuert:

`WP Admin > POS > Templates`

Weitere Informationen zur Anpassung finden Sie unter [Belegvorlagen](/receipts):

- Anzeige von Filialinformationen
- Logo und Branding
- Layout und Formatierung
- Welche Informationen enthalten sein sollen

## Belege erneut drucken {#reprinting-receipts}

So drucken Sie einen Beleg für eine frühere Bestellung erneut:

1. Öffnen Sie die [Bestellungen-Ansicht](/orders) (Pro-Funktion)
2. Bestellung suchen
3. Anklicken, um die Bestellung zu öffnen
4. Die Belegoption verwenden, um erneut zu drucken
