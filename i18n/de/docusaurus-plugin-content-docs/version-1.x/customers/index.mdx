:::info Pro Feature
Der Kundenbildschirm erfordert [WCPOS Pro](/getting-started/pro-license). Kostenlose Benutzer können vorhandene Kunden für Bestellungen auswählen, können jedoch die vollständige Kundenliste nicht einsehen oder Kundendaten bearbeiten.
:::

Der Kundenbildschirm bietet ein umfassendes Kundenmanagement direkt im POS. Ansicht, Bearbeitung und Erstellung von Kunden, ohne zur WooCommerce-Verwaltung wechseln zu müssen.

{/* Screenshot: customers-screen.png */}

## Schnittstellenübersicht

### Header-Aktionen

Oben auf dem Bildschirm:

- **Suchleiste** - Finden Sie Kunden nach Name, E-Mail usw.
- **Kunden hinzufügen** (<Icon name="plus" />) - Erstellen Sie einen neuen Kunden
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Konfigurieren Sie die sichtbaren Spalten

### Kundentabelle

Der Hauptbereich zeigt Kunden mit:

- **Avatar** - Kundenprofilbild (oder Platzhalter)
- **Vorname** - Vorname des Kunden
- **Nachname** - Nachname des Kunden (sortierbar)
- **E-Mail** - Kontakt-E-Mail-Adresse
- **Rechnungsadresse** - Vollständige Rechnungsadresse
- **Erstellungsdatum** - Wann der Kunde hinzugefügt wurde
- **Aktionen** - Drei-Punkte-Menü

### Fußzeile

- Kundenanzahl mit Synchronisierungsbutton

## Hauptmerkmale

### Kundensuche

Finden Sie Kunden schnell durch:

- Vor- oder Nachnamen
- E-Mail-Adresse
- Adressinformationen

### Neuen Kunden hinzufügen

Kunden direkt im POS erstellen:

1. Klicken Sie auf das <Icon name="plus" /> Symbol im Header
2. Füllen Sie die Kundendaten aus
3. Speichern Sie den neuen Kunden

Der Kunde wird in WooCommerce erstellt und ist sofort für Bestellungen verfügbar.

### Kunden bearbeiten

Aktualisieren Sie die Kundendaten:

1. Klicken Sie auf das Drei-Punkte-Menü eines Kunden
2. Wählen Sie **Bearbeiten**
3. Ändern Sie die Details (Name, E-Mail, Adressen)
4. Änderungen speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

## Anzeigeeinstellungen

## Kundenaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für Optionen:

- **Bearbeiten** - Ändern Sie die Kundendetails
- **Synchronisieren** - Kunden vom Server aktualisieren
- **Löschen** - Aus der lokalen Datenbank entfernen

:::note
Das Löschen eines Kunden im POS entfernt ihn lediglich lokal. Der Kunde bleibt in WooCommerce und wird beim nächsten Synchronisieren wieder angezeigt.
:::

## Verwendung von Kunden in Bestellungen

Um einen Kunden einer Bestellung zuzuordnen:

1. Klicken Sie im [Warenkorb-Panel](/pos/cart) auf die Kundenabzeichen
2. Suchen Sie den Kunden nach Namen oder E-Mail
3. Wählen Sie den Kunden aus
4. Der Kunde ist jetzt mit der Bestellung verbunden

Kundeninformationen (Rechnungs-/Lieferadressen) werden automatisch für die Bestellung verwendet.

## Synchronisierung

Die Kundendaten werden zwischen dem POS und WooCommerce synchronisiert:

- **Echtzeit-Updates** - Änderungen synchronisieren sich über alle Geräte hinweg
- **WooCommerce-Integration** - Kunden werden in WooCommerce gespeichert
- **Offline-Fähigkeit** - Kunden auch ohne Internetverbindung anzeigen
