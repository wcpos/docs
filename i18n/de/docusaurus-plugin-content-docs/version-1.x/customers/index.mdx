:::info Pro Feature
Der Bildschirm „Kunden“ erfordert [WCPOS Pro](/getting-started/pro-license). Kostenlose Benutzer können vorhandene Kunden für Bestellungen auswählen, aber die vollständige Kundenliste nicht anzeigen oder Kundendetails bearbeiten.
:::

Der Bildschirm „Kunden“ bietet umfassende Kundenverwaltung direkt innerhalb des POS. Anzeigen, bearbeiten und erstellen Sie Kunden, ohne zum WooCommerce-Admin wechseln zu müssen.

{/* Screenshot: customers-screen.png */}

## Schnittstellenübersicht

### Header-Aktionen

Oben auf dem Bildschirm:

- **Suchleiste** - Finden Sie Kunden nach Name, E-Mail usw.
- **Kunden hinzufügen** (<Icon name="plus" />) - Erstellen Sie einen neuen Kunden
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Konfigurieren Sie sichtbare Spalten

### Kundentabelle

Der Hauptbereich zeigt Kunden mit:

- **Avatar** - Kundenprofilbild (oder Platzhalter)
- **Vorname** - Vorname des Kunden
- **Nachname** - Nachname des Kunden (sortierbar)
- **E-Mail** - Kontakt-E-Mail-Adresse
- **Rechnungsadresse** - Vollständige Rechnungsadresse
- **Date Created** - Wann der Kunde hinzugefügt wurde
- **Aktionen** - Drei-Punkte-Menü

### Fußzeile

- Kundenanzahl mit Synchronisierungsschaltfläche (<Icon name="arrow-rotate-right" />). **Lang drücken** für die Optionen Clear und Refresh

## Key Features

### Kundensuche

Finden Sie Kunden schnell nach:

- Vor- oder Nachname
- E-Mail-Adresse
- Adressinformationen

### Neuen Kunden hinzufügen

Erstellen Sie Kunden direkt vom POS:

1. Klicken Sie auf das <Icon name="plus" />-Symbol im Header
2. Füllen Sie die Kundendetails aus
3. Speichern Sie den neuen Kunden

Der Kunde wird in WooCommerce erstellt und steht sofort für Bestellungen zur Verfügung.

### Kunde bearbeiten

Aktualisieren Sie die Kundeninformationen:

1. Klicken Sie auf das Drei-Punkte-Menü eines Kunden
2. Wählen Sie **Bearbeiten**
3. Ändern Sie die Details (Name, E-Mail, Adressen)
4. Änderungen speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

## Anzeigeeinstellungen

## Kundenaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für Optionen:

- **Bearbeiten** - Kundendetails ändern
- **Synchronisieren** - Kunden vom Server aktualisieren
- **Löschen** - Aus der lokalen Datenbank entfernen

:::note
Das Löschen eines Kunden aus dem POS entfernt ihn nur lokal. Der Kunde bleibt in WooCommerce und wird beim nächsten Synchronisieren erneut angezeigt.
:::

## Verwendung von Kunden in Bestellungen

Um einen Kunden einer Bestellung zuzuweisen:

1. Klicken Sie im [Warenkorb-Panel](/pos/cart) auf das Kundenabzeichen
2. Suchen Sie den Kunden nach Name oder E-Mail
3. Wählen Sie den Kunden aus
4. Der Kunde ist nun mit der Bestellung verknüpft

Kundeninformationen (Rechnungs-/Versandadressen) werden automatisch für die Bestellung verwendet.

## Synchronisierung

Kundendaten synchronisieren sich zwischen dem POS und WooCommerce:

- **Echtzeit-Updates** - Änderungen synchronisieren sich über alle Geräte
- **WooCommerce-Integration** - Kunden werden in WooCommerce gespeichert
- **Offline-Funktionalität** - Kunden auch im Offline-Modus anzeigen
