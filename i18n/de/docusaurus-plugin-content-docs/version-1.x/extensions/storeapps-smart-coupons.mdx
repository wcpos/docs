# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

Fügt WCPOS-Kompatibilität für Guthaben hinzu, das mit [Smart Coupons von StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/) erstellt wurde. POS-Bestellungen können StoreApps-Guthaben einlösen, das StoreApps-Guthaben wird synchron gehalten und es wird ein Bestellnotiz-Protokoll für das Personal erstellt.

Diese Erweiterung ist speziell für das StoreApps / WooCommerce.com Smart Coupons Plugin vorgesehen. Andere Plugins mit ähnlichen „Smart Coupons"-Namen sind eigenständige Produkte und erfordern möglicherweise separate Integrationen.

:::info Pro-Funktion
Die Installation und Verwaltung von Erweiterungen aus dem Verzeichnis erfordert [WCPOS Pro](/getting-started/pro-license).
:::

## Funktionsübersicht {#what-it-does}

- Zeichnet die StoreApps-Metadaten `smart_coupons_contribution` für im POS erstellte Bestellungen auf, wenn der normale Checkout-Warenkorbkontext nicht verfügbar ist.
- Ermöglicht es StoreApps, nach dem POS-Checkout den korrekten anteiligen Gutschriftbetrag vom Gutscheinguthaben abzuziehen.
- Stellt das StoreApps-Guthaben wieder her, wenn die Bestellung im POS über den normalen WooCommerce-Bestellablauf storniert oder rückerstattet wird.
- Fügt nach der Einlösung eine private Bestellnotiz hinzu, die den Gutscheincode, den verwendeten Betrag und das aktuelle Guthaben anzeigt.
- Zeigt das verbleibende Guthabenguthaben in der vorhandenen Rabattzeile des Bons an, indem es an die Gutscheinbeschreibung/Rabattbezeichnung angehängt wird.

Die Erweiterung verwendet die normalen WooCommerce-Gutscheinfelder und -Metadaten. Sie fügt keine benutzerdefinierten WCPOS-Gutschein-Antwortfelder hinzu und erfordert keine benutzerdefinierten Bonvorlagen.

## Installation {#installation}

<Steps>
  <StepCard number="1" title="StoreApps Smart Coupons installieren">

Installieren und aktivieren Sie [Smart Coupons von StoreApps / WooCommerce.com](https://woocommerce.com/products/smart-coupons/). Erstellen Sie Ihre Guthabengutscheine oder Geschenkkarten wie gewohnt in WooCommerce.

  </StepCard>
  <StepCard number="2" title="WCPOS StoreApps Smart Coupons installieren">

Installieren Sie die Erweiterung über `WP Admin > POS > Einstellungen > Erweiterungen`, oder laden Sie die neueste Version von der [GitHub-Releases-Seite](https://github.com/wcpos/wcpos-storeapps-smart-coupons/releases) herunter und installieren Sie sie über `Plugins > Neu hinzufügen > Plugin hochladen`.

  </StepCard>
  <StepCard number="3" title="Gutscheine mit dem POS synchronisieren" isLast>

Öffnen Sie das POS und synchronisieren Sie die Gutscheine. Kassierer können dann StoreApps-Guthabengutscheine über den normalen **Gutschein hinzufügen**-Ablauf anwenden.

  </StepCard>
</Steps>

## Bonausgabe {#receipt-output}

WCPOS-Belegvorlagen drucken Gutscheinrabatte bereits über die Rabattzeile. Während der Belegerstellung hängt diese Erweiterung das aktuelle StoreApps-Guthaben an diese bestehende Bezeichnung an.

Beispielsweise kann eine Gutscheinbeschreibung **Geschenkkarte** wie folgt gedruckt werden:

```text
Gift card — Store credit balance: £61.50
```

Wenn der Gutschein keine Beschreibung hat, wird der Guthabentext als Rabattbezeichnung verwendet. Bestehende Belegvorlagen müssen nicht geändert werden.

## Bestellungs-Prüfprotokoll {#order-audit-trail}

Nachdem StoreApps eine POS-Guthabeneinlösung verarbeitet hat, fügt die Erweiterung eine private Bestellnotiz hinzu, ähnlich wie:

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

Anhand dieser Notiz lässt sich überprüfen, welcher Guthabengutschein verwendet wurde, wie viel eingelöst wurde und welches Restguthaben nach dem Verkauf verblieben ist.

## Kompatibilitätshinweise {#compatibility-notes}

- **Nur StoreApps:** Diese Erweiterung ist auf StoreApps / WooCommerce.com Smart Coupons ausgerichtet. Sie unterstützt nicht WebToffee oder andere Smart-Coupon-Plugins.
- **Gutscheindaten:** WCPOS verwendet weiterhin die regulären WooCommerce-Gutscheinantworten. StoreApps-spezifische Daten verbleiben in WooCommerce-Metadaten wie `smart_coupons_contribution` und Gutschein-Guthabenfeldern.
- **Belege:** Der Guthabentext wird über den bestehenden Pfad für Gutscheinbeschreibung/Rabattbezeichnung eingefügt, nicht durch Änderung der Belegdaten oder -vorlagen.
- **Inaktives StoreApps-Plugin:** Wenn StoreApps Smart Coupons nicht aktiv ist, hat die Erweiterung keine Funktion.
