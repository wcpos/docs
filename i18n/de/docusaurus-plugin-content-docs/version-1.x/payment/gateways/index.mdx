# Benutzerdefinierte Zahlungsmethoden {#custom-payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsmethoden, die die Funktionalität Ihres Point of Sale-Systems erweitern. Diese Zahlungsmethoden sind speziell für POS-Umgebungen entwickelt und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Zahlungsmethoden {#available-custom-gateways}

### Hardware-Terminal-Zahlungsmethoden {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Stripe Terminal Hardware-Lesegeräten akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenleser verarbeiten

### Mobile Zahlungsmethoden {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Telefonbasierte Zahlungen über QR-Code oder Push-Benachrichtigung akzeptieren

### Digitale Zahlungsmethoden {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** - Zahlungsrechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** - Kunden umleiten, um Zahlungen online abzuschließen

## Eigene Zahlungsmethode erstellen {#creating-your-own-gateway}

Wenn Sie eine benutzerdefinierte Zahlungsmethode für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Gateway-Vorlage](./gateway-template)** verwenden, um Ihre eigene WCPOS-Zahlungsmethode zu erstellen.

## Installationsübersicht {#installation-overview}

Alle benutzerdefinierten Zahlungsmethoden folgen einem ähnlichen Installationsprozess:

1. **Laden Sie** die neueste Version aus dem jeweiligen GitHub-Repository herunter
2. **Installieren** Sie das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren** Sie die Gateway-Einstellungen in `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren** Sie die Zahlungsmethode in `WP Admin > POS > Einstellungen > Checkout`
