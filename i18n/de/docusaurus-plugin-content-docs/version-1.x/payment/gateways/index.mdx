# Zahlungsgateways {#payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, die den Funktionsumfang Ihres Point-of-Sale-Systems erweitern. Diese Gateways wurden speziell für POS-Umgebungen entwickelt und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Gateways {#available-custom-gateways}

### Hardware-Terminal-Gateways {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** – Zahlungen über Stripe Terminal Hardware-Lesegeräte akzeptieren
- **[SumUp Terminal](./sumup-terminal)** – Zahlungen über SumUp-Kartenlesegeräte abwickeln
- **[Square Terminal](./square-terminal)** – Vor-Ort-Zahlungen auf Square Terminal-Geräten entgegennehmen
- **[PayPal Reader (Zettle)](./paypal-reader)** – Vor-Ort-Kartenzahlungen über ein PayPal Reader (Zettle)-Terminal akzeptieren

### Mobile Zahlungsgateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** – Telefonbasierte Zahlungen per QR-Code oder Push-Benachrichtigung akzeptieren

### Digitale Zahlungsgateways {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** – Zahlungsrechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** – Kunden zur Online-Zahlung weiterleiten

## Eigenes Gateway erstellen {#creating-your-own-gateway}

Für individuelle Anforderungen an ein Zahlungsgateway kann die **[Gateway-Vorlage](/reference/gateway-template)** verwendet werden, um ein eigenes WCPOS-Zahlungsgateway zu erstellen.

## Installationsübersicht {#installation-overview}

Alle benutzerdefinierten Gateways folgen einem ähnlichen Installationsprozess:

1. **Herunterladen** – die neueste Version aus dem jeweiligen GitHub-Repository herunterladen
2. **Installieren** – das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen` installieren
3. **Konfigurieren** – die Gateway-Einstellungen unter `WP Admin > WooCommerce > Einstellungen > Zahlungen` vornehmen
4. **Aktivieren** – das Gateway unter `WP Admin > POS > Einstellungen > Checkout` aktivieren
