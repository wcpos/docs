# Zahlungsgateways {#payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, die die Funktionalität Ihres Point-of-Sale-Systems erweitern. Diese Gateways wurden speziell für POS-Umgebungen entwickelt und ermöglichen eine nahtlose Integration verschiedener Zahlungsmethoden und Dienste.

## Verfügbare benutzerdefinierte Zahlungsgateways {#available-custom-gateways}

### Hardware-Terminal-Gateways {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Stripe Terminal-Hardware-Kartenlesern akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenleser verarbeiten
- **[Square Terminal](./square-terminal)** - Vor-Ort-Zahlungen auf Square Terminal-Geräten entgegennehmen
- **[PayPal Reader (Zettle)](./paypal-reader)** - Vor-Ort-Kartenzahlungen an einem PayPal Reader (Zettle)-Terminal entgegennehmen
- **[Mollie Terminal](./mollie-terminal)** - Vor-Ort-Zahlungen auf Mollie Terminal-Geräten entgegennehmen

### Mobile Zahlungsgateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** – Telefonbasierte Zahlungen per QR-Code oder Push-Benachrichtigung akzeptieren

### Digitale Zahlungsgateways {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** – Zahlungsrechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** – Kunden weiterleiten, damit sie Zahlungen online abschließen

## Eigenes Gateway erstellen {#creating-your-own-gateway}

Wenn ein individuelles Zahlungsgateway für bestimmte Anforderungen benötigt wird, kann unsere **[Gateway-Vorlage](/reference/gateway-template)** verwendet werden, um ein eigenes WCPOS-Zahlungsgateway zu erstellen.

## Installationsübersicht {#installation-overview}

Alle individuellen Gateways folgen einem ähnlichen Installationsprozess:

1. **Herunterladen** der neuesten Version aus dem jeweiligen GitHub-Repository
2. **Installieren** des Plugins über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren** der Gateway-Einstellungen unter `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren** Sie das Gateway unter `WP Admin > POS > Einstellungen > Kasse`
