# Zahlungsgateways {#payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, die die Funktionalität Ihres POS-Systems erweitern. Diese Gateways sind speziell für POS-Umgebungen konzipiert und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und Diensten.

## Verfügbare benutzerdefinierte Gateways {#available-custom-gateways}

### Gateways für Hardware-Terminals {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Stripe Terminal-Hardware-Lesegeräten akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenlesegeräte verarbeiten

### Mobile Zahlungsgateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Telefonbasierte Zahlungen per QR-Code oder Push-Benachrichtigung akzeptieren

### Digitale Zahlungsgateways {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** - Zahlungsrechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** - Kunden zur Online-Abwicklung von Zahlungen weiterleiten

## Eigenes Gateway erstellen {#creating-your-own-gateway}

Wenn ein individuelles Zahlungsgateway für spezifische Anforderungen benötigt wird, kann unsere **[Gateway-Vorlage](/reference/gateway-template)** verwendet werden, um ein eigenes WCPOS-Zahlungsgateway zu erstellen.

## Installationsübersicht {#installation-overview}

Alle individuellen Gateways folgen einem ähnlichen Installationsprozess:

1. **Herunterladen** der neuesten Version aus dem jeweiligen GitHub-Repository
2. **Installieren** des Plugins über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren** der Gateway-Einstellungen unter `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren** des Gateways unter `WP Admin > POS > Einstellungen > Checkout`
