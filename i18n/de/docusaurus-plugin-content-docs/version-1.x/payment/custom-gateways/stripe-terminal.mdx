# Stripe Terminal Gateway

Der Stripe Terminal-Gateway ermöglicht es Ihnen, Zahlungen vor Ort mit Stripe Terminal-Hardware-Lesegeräten direkt in WooCommerce POS zu akzeptieren. Dieser Gateway unterstützt sowohl physische Kartenlesegeräte als auch den Simulatormodus zum Testen.

## Funktionen

- **Hardware-Integration**: Verbinden physischer Stripe Terminal-Lesegeräte über Internetverbindung
- **Simulatormodus**: Testen von Zahlungen ohne Hardware mit Stripes Simulator
- **Mehrere Zahlungsmethoden**: Unterstützung verschiedener Kartentypen und Zahlungsmethoden
- **Echtzeitverarbeitung**: Sofortige Zahlungsverarbeitung und -bestätigung
- **Sichere Transaktionen**: PCI-konforme Zahlungsabwicklung durch Stripe
- **Web-basiertes SDK**: Verwendet Stripes JavaScript SDK für Webanwendungskompatibilität

## Installation

### Schritt 1: Herunterladen und Installieren

1. Besuchen Sie die [Stripe Terminal für WooCommerce Veröffentlichungsseite](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Laden Sie die neueste **stripe-terminal-for-woocommerce.zip**-Datei herunter
3. Gehen Sie in Ihrer WordPress-Administration zu `Plugins > Neu hinzufügen > Plugin hochladen`
4. Laden Sie die Zip-Datei hoch und aktivieren Sie das Plugin

### Schritt 2: Stripe-Einstellungen konfigurieren

1. Navigieren Sie zu `WP Admin > WooCommerce > Einstellungen > Zahlungen`
2. Finden Sie **Stripe Terminal** in der Liste der Zahlungsmethoden
3. Klicken Sie auf **Stripe Terminal**, um auf die Einstellungen zuzugreifen
4. Geben Sie Ihren **Stripe-Geheimschlüssel** ein (Sie können diesen aus Ihrem Stripe-Dashboard abrufen)
5. Speichern Sie die Einstellungen

:::note
Sie müssen den Stripe Terminal-Gateway in den WooCommerce-Einstellungen nicht aktivieren. Er wird speziell für den POS im nächsten Schritt aktiviert.
:::

### Schritt 3: In WooCommerce POS aktivieren

1. Gehen Sie zu `WP Admin > WooCommerce POS > Einstellungen > Checkout`
2. Finden Sie den **Stripe Terminal**-Gateway in der Liste
3. Aktivieren Sie den Gateway für die Nutzung im POS
4. Speichern Sie Ihre Einstellungen

## Nutzung

### Lesegerät verbinden

Wenn Sie während des Checkouts im POS den Stripe Terminal-Gateway auswählen:

1. **Verbindungsmethode wählen**: Sie können entweder ein physisches Lesegerät verbinden oder den Simulator verwenden
2. **Physisches Lesegerät**: Folgen Sie den Bildschirmanweisungen, um Ihr Stripe Terminal-Gerät zu verbinden
3. **Simulator**: Wählen Sie den Simulatormodus, um verschiedene Zahlungsszenarien ohne Hardware zu testen

### Zahlungen verarbeiten

1. **Artikel hinzufügen**: Fügen Sie Produkte zum Warenkorb im POS hinzu
2. **Gateway auswählen**: Wählen Sie "Stripe Terminal" als Zahlungsmethode
3. **Lesegerät verbinden**: Verbinden Sie Ihr Lesegerät oder wählen Sie den Simulatormodus
4. **Zahlung verarbeiten**: Folgen Sie den Aufforderungen, um die Transaktion abzuschließen
5. **Bestätigung**: Die Bestellung wird nach erfolgreicher Zahlung automatisch abgeschlossen

### Mit Simulator testen

Der Simulator ermöglicht Ihnen das Testen verschiedener Zahlungsmethoden und Szenarien:

- **Kartenzahlungen**: Testen verschiedener Kartentypen (Visa, Mastercard, etc.)
- **Kontaktlose Zahlungen**: Berührungsloses Bezahlen simulieren
- **Fehlerzenarien**: Abgelehnte Zahlungen und andere Fehler testen
- **Verschiedene Beträge**: Verschiedene Transaktionsbeträge testen

## Anforderungen

- **Stripe-Konto**: Aktives Stripe-Konto mit aktiviertem Terminal
- **API-Schlüssel**: Stripe-Geheimschlüssel aus Ihrem Dashboard
- **WooCommerce POS**: Freie oder Pro-Version installiert und aktiviert
- **HTTPS**: Ihre Website muss SSL/HTTPS für Sicherheit verwenden

## Hardware-Kompatibilität

:::important Konnektivitätsanforderungen
Diese Implementierung verwendet Stripes JavaScript SDK, was bedeutet, dass sie über Webanwendungen funktioniert, aber nur Internetverbundene Terminals erforderlich macht. Bluetooth-Terminals werden derzeit nicht unterstützt.
:::

### Unterstützte Terminals (Internetverbunden)

- **Stripe Reader S700** ✅ - Ethernet/WiFi-verbundenes Terminal
- **WisePOS E** ✅ - WiFi-verbundenes Terminal

### Nicht unterstützte Terminals (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Nur Bluetooth
- **BBPOS WisePad 3** ❌ - Nur Bluetooth
- **Verifone P400** ❌ - Nur Bluetooth

:::note Zukünftige Unterstützung
Bluetooth-Terminal-Unterstützung wird in den iOS- und Android-Anwendungen im **Q1 2026** hinzugefügt. Dies wird die Unterstützung für alle Stripe Terminal-zertifizierten Lesegeräte einschließlich M2 und WisePad 3 ermöglichen.
:::

## Fehlerbehebung

### Häufige Probleme

**Lesegerät verbindet sich nicht**
- Stellen Sie sicher, dass Sie ein unterstütztes internetverbundenes Terminal (S700 oder WisePOS E) verwenden
- Überprüfen Sie, ob das Terminal mit WiFi/Ethernet verbunden und online ist
- Prüfen Sie, ob Ihr Stripe-Konto Terminal aktiviert hat
- Bestätigen Sie, dass das Terminal in Ihrem Stripe-Dashboard registriert ist

**Zahlung abgelehnt**
- Überprüfen Sie, ob Ihr Stripe-Konto aktiv und in gutem Zustand ist
- Verifizieren Sie, dass die verwendete Karte gültig ist
- Stellen Sie sicher, dass ausreichend Mittel verfügbar sind

**SSL-Zertifikatsfehler**
- Stripe Terminal erfordert HTTPS - stellen Sie sicher, dass Ihre Website ein gültiges SSL-Zertifikat hat
- Überprüfen Sie, ob Ihr SSL-Zertifikat korrekt konfiguriert ist
