# Stripe Terminal Gateway

Das Stripe Terminal Gateway ermöglicht es Ihnen, persönlich Zahlungen direkt innerhalb von WooCommerce POS mit Stripe Terminal Hardwarelesern zu akzeptieren. Dieses Gateway unterstützt sowohl physische Kartenleser als auch den Simulator-Modus zum Testen.

## Funktionen {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Hardware-Integration">

  </FeatureCard>
  <FeatureCard icon="desktop" title="Simulator-Modus">

## Installation

<Steps>
  <StepCard number="1" title="Herunterladen und Installieren">

1. Besuchen Sie die [Stripe Terminal für WooCommerce Veröffentlichungsseite](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Laden Sie die neueste **stripe-terminal-for-woocommerce.zip** Datei herunter
3. Gehen Sie zu Ihrem WordPress-Admin-Bereich und wählen Sie `Plugins > Neu hinzufügen > Plugin hochladen`
4. Laden Sie die Zip-Datei hoch und aktivieren Sie das Plugin

  </StepCard>
  <StepCard number="2" title="Stripe-Einstellungen konfigurieren">

1. Navigieren Sie zu `WP Admin > WooCommerce > Einstellungen > Zahlungen`
2. Finden Sie **Stripe Terminal** in der Liste der Zahlungsmethoden
3. Klicken Sie auf **Stripe Terminal**, um die Einstellungen zuzugreifen
4. Geben Sie Ihren **Stripe Secret Key** ein (dieser ist in Ihrem Stripe-Dashboard zu finden)
5. Speichern Sie die Einstellungen

:::note
Es ist nicht erforderlich, das Stripe Terminal Gateway in den WooCommerce-Einstellungen zu aktivieren. Es wird speziell für die POS im nächsten Schritt aktiviert.
:::

  </StepCard>
  <StepCard number="3" title="Im WooCommerce POS aktivieren" isLast>

1. Gehen Sie zu `WP Admin > WooCommerce POS > Einstellungen > Checkout`
2. Finden Sie das **Stripe Terminal** Gateway in der Liste
3. Aktivieren Sie das Gateway für die Nutzung im POS
4. Speichern Sie Ihre Einstellungen

  </StepCard>
</Steps>

## Verwendung {#usage}

### Einen Leser anschließen {#connecting-a-reader}

Wenn Sie das Stripe Terminal Gateway während des Checkouts im POS auswählen:

1. **Verbindungsmethode wählen**: Sie können entweder einen physischen Leser anschließen oder den Simulator verwenden
2. **Physischer Leser**: Befolgen Sie die Anweisungen auf dem Bildschirm, um Ihr Stripe Terminal-Gerät anzuschließen
3. **Simulator**: Wählen Sie den Simulator-Modus, um verschiedene Zahlungsszenarien ohne Hardware zu testen

### Zahlungen verarbeiten {#processing-payments}

1. **Artikel hinzufügen**: Fügen Sie Produkte zu Ihrem Warenkorb im POS hinzu
2. **Gateway auswählen**: Wählen Sie "Stripe Terminal" als Zahlungsmethode
3. **Leser anschließen**: Schließen Sie Ihren Leser an oder wählen Sie den Simulator-Modus
4. **Zahlung verarbeiten**: Befolgen Sie die Anweisungen, um die Transaktion abzuschließen
5. **Bestätigung**: Die Bestellung wird automatisch nach erfolgreicher Zahlung abgeschlossen

### Testen mit dem Simulator {#testing-with-simulator}

Der Simulator ermöglicht es Ihnen, verschiedene Zahlungsmethoden und Szenarien zu testen:

- **Kartenzahlungen**: Testen Sie verschiedene Kartentypen (Visa, Mastercard usw.)
- **Kontaktlose Zahlungen**: Simulieren Sie Tap-to-Pay-Transaktionen
- **Fehlerfälle**: Testen Sie abgelehnte Zahlungen und andere Fehlerbedingungen
- **Verschiedene Beträge**: Testen Sie verschiedene Transaktionsbeträge

## Hardware-Kompatibilität {#hardware-compatibility}

:::important Anforderungen an die Konnektivität
Diese Implementierung verwendet Stripes JavaScript SDK, was bedeutet, dass sie über Webanwendungen funktioniert, aber nur **internetverbundene Terminals** benötigt. Bluetooth-Terminals werden derzeit nicht unterstützt.
:::

### Unterstützte Terminals (Internet-verbunden) {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Ethernet/WLAN-verbundenes Terminal
- **WisePOS E** ✅ - WLAN-verbundenes Terminal

### Nicht unterstützte Terminals (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Nur Bluetooth
- **BBPOS WisePad 3** ❌ - Nur Bluetooth  
- **Verifone P400** ❌ - Nur Bluetooth

:::note Zukünftige Unterstützung
Die Unterstützung für Bluetooth-Terminals wird in den iOS- und Android-Anwendungen im **Q1 2026** hinzugefügt. Dies ermöglicht die Unterstützung aller von Stripe Terminal zertifizierten Leser, einschließlich M2 und WisePad 3.
:::

### Häufige Probleme {#common-issues}

<Accordion>
  <AccordionItem question="Leser kann nicht verbinden">

  - Stellen Sie sicher, dass Sie ein unterstütztes internetverbundenes Terminal (S700 oder WisePOS E) verwenden
  - Überprüfen Sie, ob das Terminal mit WLAN/Ethernet verbunden und online ist
  - Überprüfen Sie, ob Ihr Stripe-Konto für das Terminal aktiviert ist
  - Bestätigen Sie, dass das Terminal in Ihrem Stripe-Dashboard registriert ist

  </AccordionItem>

  <AccordionItem question="Zahlung abgelehnt">

  - Überprüfen Sie, ob Ihr Stripe-Konto aktiv und in gutem Stand ist
  - Überprüfen Sie, ob die verwendete Karte gültig ist
  - Stellen Sie sicher, dass ausreichende Mittel verfügbar sind

  </AccordionItem>

  <AccordionItem question="SSL-Zertifikatfehler">

  - Stripe Terminal erfordert HTTPS - stellen Sie sicher, dass Ihre Seite ein gültiges SSL-Zertifikat hat
  - Überprüfen Sie, ob Ihr SSL-Zertifikat korrekt konfiguriert ist

  </AccordionItem>
</Accordion>
