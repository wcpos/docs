# Stripe Terminal Gateway

Das Stripe Terminal-Gateway ermöglicht es Ihnen, persönliche Zahlungen mit Stripe Terminal Hardware-Lesegeräten direkt in WooCommerce POS zu akzeptieren. Dieses Gateway unterstützt sowohl physische Kartenlesegeräte als auch den Simulator-Modus zum Testen.

## Funktionen {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Hardware-Integration">

  </FeatureCard>

## Installation

<Steps>
  <StepCard number="1" title="Herunterladen und Installieren">

1. Besuchen Sie die [Stripe Terminal für WooCommerce Veröffentlichungsseite](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Laden Sie die neueste **stripe-terminal-for-woocommerce.zip**-Datei herunter
3. Gehen Sie in Ihrem WordPress-Admin zu `Plugins > Neu hinzufügen > Plugin hochladen`
4. Laden Sie die ZIP-Datei hoch und aktivieren Sie das Plugin

  </StepCard>
  <StepCard number="2" title="Stripe-Einstellungen konfigurieren">

1. Navigieren Sie zu `WP Admin > WooCommerce > Einstellungen > Zahlungen`
2. Finden Sie **Stripe Terminal** in der Liste der Zahlungsmethoden
3. Klicken Sie auf **Stripe Terminal**, um auf die Einstellungen zuzugreifen
4. Geben Sie Ihren **Stripe Secret Key** ein (dieser kann von Ihrem Stripe Dashboard abgerufen werden)
5. Speichern Sie die Einstellungen

:::note
Sie müssen das Stripe Terminal-Gateway in den WooCommerce-Einstellungen nicht aktivieren. Es wird im nächsten Schritt speziell für das POS aktiviert.
:::

  </StepCard>
  <StepCard number="3" title="Im WooCommerce POS aktivieren" isLast>

1. Gehen Sie zu `WP Admin > WooCommerce POS > Einstellungen > Checkout`
2. Finden Sie das **Stripe Terminal**-Gateway in der Liste
3. Aktivieren Sie das Gateway zur Verwendung im POS
4. Speichern Sie Ihre Einstellungen

  </StepCard>
</Steps>

## Verwendung {#usage}

### Anschließen eines Lesegeräts {#connecting-a-reader}

Wenn Sie das Stripe Terminal-Gateway während des Checkouts im POS auswählen:

1. **Wählen Sie die Verbindungsmethode**: Sie können entweder ein physisches Lesegerät oder den Simulator verwenden
2. **Physisches Lesegerät**: Folgen Sie den Anweisungen auf dem Bildschirm, um Ihr Stripe Terminal-Gerät anzuschließen
3. **Simulator**: Wählen Sie den Simulator-Modus, um verschiedene Zahlungsszenarien ohne Hardware zu testen

### Zahlungen verarbeiten {#processing-payments}

1. **Artikel hinzufügen**: Fügen Sie Produkte zu Ihrem Warenkorb im POS hinzu
2. **Gateway auswählen**: Wählen Sie "Stripe Terminal" als Zahlungsmethode
3. **Lesegerät verbinden**: Schließen Sie Ihr Lesegerät an oder wählen Sie den Simulator-Modus
4. **Zahlung verarbeiten**: Folgen Sie den Anweisungen, um die Transaktion abzuschließen
5. **Bestätigung**: Die Bestellung wird automatisch bei erfolgreicher Zahlung abgeschlossen

### Testen mit dem Simulator {#testing-with-simulator}

Der Simulator ermöglicht es Ihnen, verschiedene Zahlungsmethoden und Szenarien zu testen:

- **Karten-Zahlungen**: Testen Sie verschiedene Kartentypen (Visa, Mastercard usw.)
- **Kontaktlose Zahlungen**: Simulieren Sie Tap-to-Pay-Transaktionen
- **Fehler-Szenarien**: Testen Sie abgelehnte Zahlungen und andere Fehlerbedingungen
- **Verschiedene Beträge**: Testen Sie verschiedene Transaktionsbeträge

## Hardware-Kompatibilität {#hardware-compatibility}

:::important Anforderungen an die Konnektivität
Diese Implementierung verwendet das JavaScript-SDK von Stripe, was bedeutet, dass es über Webanwendungen funktioniert, jedoch **nur internetverbundene Terminals** benötigt. Bluetooth-Terminals werden derzeit nicht unterstützt.
:::

### Unterstützte Terminals (Internet-Verbunden) {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Ethernet/WiFi-verbundenes Terminal
- **WisePOS E** ✅ - WiFi-verbundenes Terminal

### Nicht unterstützte Terminals (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Nur Bluetooth
- **BBPOS WisePad 3** ❌ - Nur Bluetooth  
- **Verifone P400** ❌ - Nur Bluetooth

:::note Zukünftige Unterstützung
Die Unterstützung für Bluetooth-Terminals wird in den iOS- und Android-Anwendungen im **Q1 2026** hinzugefügt. Dies wird die Unterstützung für alle stripe-terminalzertifizierten Lesegeräte einschließen, einschließlich des M2 und WisePad 3.
:::

### Häufige Probleme {#common-issues}

<Accordion>
  <AccordionItem question="Lesegerät verbindet sich nicht">

  - Stellen Sie sicher, dass Sie ein unterstütztes internetverbundes Terminal (S700 oder WisePOS E) verwenden
  - Überprüfen Sie, ob das Terminal mit WiFi/Ethernet verbunden und online ist
  - Überprüfen Sie, ob Ihr Stripe-Konto Terminal aktiviert hat
  - Bestätigen Sie, dass das Terminal in Ihrem Stripe Dashboard registriert ist

  </AccordionItem>

  <AccordionItem question="Zahlung abgelehnt">

  - Überprüfen Sie, ob Ihr Stripe-Konto aktiv und in gutem Stand ist
  - Vergewissern Sie sich, dass die verwendete Karte gültig ist
  - Stellen Sie sicher, dass ausreichende Mittel verfügbar sind

  </AccordionItem>

  <AccordionItem question="SSL-Zertifikatfehler">

  - Stripe Terminal benötigt HTTPS - stellen Sie sicher, dass Ihre Website über ein gültiges SSL-Zertifikat verfügt
  - Überprüfen Sie, ob Ihr SSL-Zertifikat richtig konfiguriert ist

  </AccordionItem>
</Accordion>
