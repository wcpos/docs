# Web-Checkout-Gateway

Das Web-Checkout-Gateway ermöglicht es Kunden, ihre POS-Transaktionen über das Checkout-System Ihres Webshops abzuschließen. Dieses Gateway schafft eine nahtlose Verbindung zwischen Ihrem POS im Laden und Online-Zahlungsmethoden, ideal für Situationen, in denen Kunden Zahlungen über ihre eigenen Geräte bevorzugen oder wenn Sie Zugriff auf webbasierte Zahlungsmethoden benötigen.

## Funktionen

- **Nahtlose Integration**: Direkte Verknüpfung von POS zum Webcheckout
- **Alle Zahlungsmethoden**: Zugriff auf alle aktivierten Zahlungsgateways Ihres Webshops
- **Kundenkontrolle**: Kunden können mit ihren bevorzugten Methoden bezahlen
- **Bestellsynchronisation**: Automatische Aktualisierungen zwischen POS und Webshop
- **Flexible Arbeitsweise**: Online-Transaktionen bei Beibehaltung des POS-Workflows

## Installation

### Schritt 1: Herunterladen und Installieren

1. Besuchen Sie die [Web-Checkout-Gateway-Versionsseite](https://github.com/wcpos/web-checkout-gateway/releases)
2. Laden Sie die neueste **woocommerce-pos-web-checkout-gateway.zip**-Datei herunter
3. Gehen Sie im WordPress-Admin zu `Plugins > Neu hinzufügen > Plugin hochladen`
4. Laden Sie die ZIP-Datei hoch und aktivieren Sie das Plugin

### Schritt 2: In WooCommerce POS aktivieren

1. Gehen Sie zu `WP Admin > WooCommerce POS > Einstellungen > Checkout`
2. Suchen Sie das **WooCommerce POS Web-Checkout-Gateway** in der Liste
3. Aktivieren Sie das Gateway für die Nutzung im POS
4. Speichern Sie Ihre Einstellungen

:::note
Dieses Gateway nutzt Ihre vorhandenen WooCommerce-Zahlungsmethoden, stellen Sie also sicher, dass mindestens ein Web-Zahlungsgateway ordnungsgemäß konfiguriert ist (Stripe, PayPal, etc.).
:::
