# Benutzerdefinierte Zahlungsmethoden

WCPOS unterstützt verschiedene benutzerdefinierte Zahlungsmethoden, die die Funktionalität Ihres Point of Sale-Systems erweitern. Diese Gateways sind speziell für POS-Umgebungen entwickelt und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Gateways

### Hardware-Terminal-Gateways

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Stripe Terminal-Kartenlesegeräten akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenleser abwickeln

### Digitale Zahlungsmethoden

- **[E-Mail-Rechnung](./email-invoice)** - Rechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** - Kunden zur Online-Zahlungsabwicklung weiterleiten

## Eigenes Gateway erstellen

Wenn Sie ein benutzerdefiniertes Zahlungsgateway für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Gateway-Vorlage](./gateway-template)** verwenden, um Ihr eigenes WooCommerce POS-Zahlungsgateway zu erstellen.

## Installationsübersicht

Alle benutzerdefinierten Gateways folgen einem ähnlichen Installationsprozess:

1. **Herunterladen** der neuesten Version aus dem jeweiligen GitHub-Repository
2. **Installieren** des Plugins über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren** der Gateway-Einstellungen unter `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren** des Gateways unter `WP Admin > WooCommerce POS > Einstellungen > Checkout`

## Anforderungen

- WooCommerce POS Plugin (Gratis- oder Pro-Version)
- WordPress mit installiertem WooCommerce
- Entsprechende API-Schlüssel oder Anmeldedaten für den spezifischen Zahlungsdienst

## Support

Bei Problemen mit benutzerdefinierten Gateways besuchen Sie bitte das jeweilige GitHub-Repository und erstellen Sie ein Issue mit detaillierten Informationen zu Ihrem Problem.
