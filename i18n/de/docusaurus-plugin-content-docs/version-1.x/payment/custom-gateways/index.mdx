# Benutzerdefinierte Zahlungsgateways

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, die die Funktionalität Ihres Point of Sale-Systems erweitern. Diese Gateways sind speziell für POS-Umgebungen konzipiert und bieten eine nahtlose Integration mit verschiedenen Zahlungsarten und Diensten.

## Verfügbare Benutzerdefinierte Gateways

### Hardware-Terminal-Gateways

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Stripe Terminal-Hardwarelesegeräten akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenleser abwickeln

### Digitale Zahlungsgateways

- **[E-Mail-Rechnung](./email-invoice)** - Zahlungsrechnungen an Kunden per E-Mail senden
- **[Web Checkout](./web-checkout)** - Kunden umleiten, um Zahlungen online abzuschließen

## Erstellen Ihres eigenen Gateways

Wenn Sie ein benutzerdefiniertes Zahlungsgateway für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Gateway-Vorlage](./gateway-template)** verwenden, um Ihr eigenes WooCommerce POS-Zahlungsgateway zu erstellen.

## Installationsübersicht

Alle benutzerdefinierten Gateways folgen einem ähnlichen Installationsprozess:

1. **Laden Sie** das neueste Release aus dem entsprechenden GitHub-Repository herunter
2. **Installieren Sie** das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren Sie** die Gateway-Einstellungen in `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren Sie** das Gateway in `WP Admin > WooCommerce POS > Einstellungen > Checkout`

## Anforderungen

- WooCommerce POS-Plugin (kostenlose oder Pro-Version)
- WordPress mit installiertem WooCommerce
- Geeignete API-Schlüssel oder Zugangsdaten für den spezifischen Zahlungsdienst

## Unterstützung

Bei Problemen mit benutzerdefinierten Gateways besuchen Sie bitte das entsprechende GitHub-Repository und erstellen Sie ein Issue mit detaillierten Informationen zu Ihrem Anliegen.
