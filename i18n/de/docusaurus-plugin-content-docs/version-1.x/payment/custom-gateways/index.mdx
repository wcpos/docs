# Benutzerdefinierte Zahlungsgateways {#custom-payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, die die Funktionalität Ihres Point of Sale-Systems erweitern. Diese Gateways sind speziell für POS-Umgebungen konzipiert und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Gateways {#available-custom-gateways}

### Hardware-Terminal-Gateways

- **[Stripe Terminal](./stripe-terminal)** - Akzeptieren Sie Zahlungen mit Stripe Terminal Hardwarelesegeräten
- **[SumUp Terminal](./sumup-terminal)** - Verarbeiten Sie Zahlungen über SumUp-Kartenlesegeräte

### Digitale Zahlungsgateways {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** - Senden Sie Zahlungsrechnungen an Kunden per E-Mail
- **[Web-Checkout](./web-checkout)** - Leiten Sie Kunden weiter, um Zahlungen online abzuschließen

## Erstellen Sie Ihr eigenes Gateway {#creating-your-own-gateway}

Wenn Sie ein benutzerdefiniertes Zahlungsgateway für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Gateway-Vorlage](./gateway-template)** verwenden, um Ihr eigenes WooCommerce POS-Zahlungsgateway zu erstellen.

## Installationsübersicht {#installation-overview}

Alle benutzerdefinierten Gateways folgen einem ähnlichen Installationsprozess:

1. **Laden Sie** die neueste Version aus dem jeweiligen GitHub-Repository herunter
2. **Installieren Sie** das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren Sie** die Gateway-Einstellungen in `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren Sie** das Gateway in `WP Admin > WooCommerce POS > Einstellungen > Checkout`
