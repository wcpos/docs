# Benutzerdefinierte Zahlungsgateways {#custom-payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsgateways, welche die Funktionalität Ihres Kassensystems erweitern. Diese Gateways sind speziell für POS-Umgebungen konzipiert und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Gateways {#available-custom-gateways}

### Hardware-Terminal-Gateways {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Zahlungen mit Hardware-Lesegeräten von Stripe Terminal akzeptieren
- **[SumUp Terminal](./sumup-terminal)** - Zahlungen über SumUp-Kartenlesegeräte verarbeiten

### Mobile Zahlungsgateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Telefonbasierte Zahlungen per QR-Code oder Push-Benachrichtigung akzeptieren

### Digitale Zahlungsgateways {#digital-payment-gateways}

- **[E-Mail-Rechnung](./email-invoice)** - Zahlungsrechnungen per E-Mail an Kunden senden
- **[Web-Checkout](./web-checkout)** – Kunden zur Online-Zahlungsabwicklung weiterleiten

## Eigenes Gateway erstellen {#creating-your-own-gateway}

Wenn Sie ein individuelles Zahlungsgateway für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Gateway-Vorlage](./gateway-template)** verwenden, um Ihr eigenes WCPOS-Zahlungsgateway zu erstellen.

## Installationsübersicht {#installation-overview}

Alle benutzerdefinierten Gateways folgen einem ähnlichen Installationsprozess:

1. **Laden Sie** die neueste Version aus dem jeweiligen GitHub-Repository herunter
2. **Installieren Sie** das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren Sie** die Gateway-Einstellungen unter `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren Sie** das Gateway unter `WP Admin > POS > Einstellungen > Checkout`
