# Benutzerdefinierte Zahlungsmethoden {#custom-payment-gateways}

WCPOS unterstützt mehrere benutzerdefinierte Zahlungsmethoden, die die Funktionalität Ihres Point of Sale-Systems erweitern. Diese Zahlungsmethoden sind speziell für POS-Umgebungen entwickelt und bieten eine nahtlose Integration mit verschiedenen Zahlungsmethoden und -diensten.

## Verfügbare benutzerdefinierte Zahlungsmethoden {#available-custom-gateways}

### Hardware-Terminal-Zahlungsmethoden {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Akzeptieren Sie Zahlungen mit Stripe Terminal-Hardwarelesern
- **[SumUp Terminal](./sumup-terminal)** - Verarbeiten Sie Zahlungen über SumUp-Kartenleser

### Mobile Zahlungsmethoden {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Akzeptieren Sie telefonbasierte Zahlungen über QR-Code oder Push-Benachrichtigung

### Digitale Zahlungsmethoden

- **[Email-Rechnung](./email-invoice)** - Senden Sie Zahlungsrechnungen an Kunden per E-Mail
- **[Web-Checkout](./web-checkout)** - Leiten Sie Kunden weiter, um Zahlungen online abzuschließen

## Erstellen Ihrer eigenen Zahlungsmethode {#creating-your-own-gateway}

Wenn Sie eine benutzerdefinierte Zahlungsmethode für Ihre spezifischen Anforderungen benötigen, können Sie unsere **[Zahlungsmethoden-Vorlage](./gateway-template)** verwenden, um Ihre eigene WooCommerce POS-Zahlungsmethode zu erstellen.

## Installationsübersicht {#installation-overview}

Alle benutzerdefinierten Zahlungsmethoden folgen einem ähnlichen Installationsprozess:

1. **Laden Sie** die neueste Version aus dem jeweiligen GitHub-Repository herunter
2. **Installieren Sie** das Plugin über `WP Admin > Plugins > Neu hinzufügen > Plugin hochladen`
3. **Konfigurieren Sie** die Zahlungsmethoden-Einstellungen in `WP Admin > WooCommerce > Einstellungen > Zahlungen`
4. **Aktivieren Sie** die Zahlungsmethode in `WP Admin > WooCommerce POS > Einstellungen > Checkout`
