# Gateway-Vorlage

Die WooCommerce Gateway-Vorlage bietet einen Ausgangspunkt für die Erstellung eigener benutzerdefinierter Zahlungsgateways für WooCommerce POS. Diese Vorlage enthält den gesamten notwendigen Boilerplate-Code und die Struktur für ein voll funktionsfähiges Zahlungsgateway.

## Funktionen

- **Vollständige Vorlage**: Einsatzbereite Gateway-Struktur mit allen erforderlichen Methoden
- **POS-Integration**: Vorkonfiguriert für WooCommerce POS-Kompatibilität
- **Automatisierte Einrichtung**: Skriptbasierte Vorlagen-Anpassung
- **Best Practices**: Folgt WordPress- und WooCommerce-Programmierstandards
- **Erweiterbar**: Einfach zu modifizieren und für spezifische Zahlungsanbieter anzupassen

## Erste Schritte

### Option 1: Automatisierte Vorlagenerstellung

Die Vorlage enthält ein Skript, das die Vorlage automatisch für Ihr spezifisches Gateway anpasst:

1. **Repository klonen**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **Setup-Skript ausführen**:
   ```bash
   ./create-gateway.sh
   ```

3. **Eingabeaufforderungen befolgen**:
   - Gateway-Namen eingeben (z.B. "Meine Zahlungsmethode")
   - Gateway-Slug eingeben (z.B. "meine-zahlung")
   - Beschreibung angeben
   - Das Skript generiert ein angepasstes Plugin

### Option 2: Manuelle Vorlagennutzung

Bei manueller Anpassung:

1. **Vorlage herunterladen**:
   - [Gateway-Vorlagen-Repository](https://github.com/wcpos/woocommerce-gateway-template) besuchen
   - Neueste Version herunterladen oder Repository klonen

2. **Vorlage anpassen**:
   - `{{GATEWAY_NAME}}` durch Ihren Gateway-Anzeigenamen ersetzen
   - `{{GATEWAY_SLUG}}` durch Ihren eindeutigen Gateway-Bezeichner ersetzen
   - `{{GATEWAY_DESCRIPTION}}` durch Gateway-Beschreibung ersetzen

3. **Dateien umbenennen**:
   - `wcpos-{{GATEWAY_SLUG}}.php` entsprechend dem Gateway-Slug umbenennen
   - Datei-Header und Plugin-Informationen aktualisieren
