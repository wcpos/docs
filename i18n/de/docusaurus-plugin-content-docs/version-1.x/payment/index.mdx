WCPOS unterstützt mehrere Zahlungsmethoden, um unterschiedlichen Geschäftsbedürfnissen gerecht zu werden.

## Standard Zahlungsmethoden

Die kostenlose Version enthält zwei integrierte Zahlungsgateways:

### Bargeld

- Geben Sie den eingezahlten Betrag ein
- Automatische Berechnung des Wechselgelds
- Keine zusätzliche Konfiguration erforderlich

### Karte

- Zur Verwendung mit externen Kartenterminals
- Markieren Sie die Zahlung einfach als abgeschlossen, nachdem Sie sie auf Ihrem Terminal bearbeitet haben
- Keine direkte Integration erforderlich

## Zusätzliche Zahlungsgateways (Pro)

:::info Pro-Funktion
Zusätzliche Zahlungsgateways erfordern [WCPOS Pro](/getting-started/pro-license).
:::

Mit Pro können Sie zusätzliche WooCommerce-Zahlungsgateways im POS-Checkout aktivieren:

- **Stripe Terminal** - Direkte Integration mit Stripe-Kartenlesern
- **SumUp Terminal** - Integration mit SumUp-Kartenlesern
- **Jedes WooCommerce-Gateway** - Aktivieren Sie jedes auf Ihrem Shop installierte Gateway

## Benutzerdefinierte Gateways

Erstellen Sie Ihre eigenen Zahlungsintegrationen mit dem [Benutzerdefinierte Gateways](/payment/custom-gateways) System.
