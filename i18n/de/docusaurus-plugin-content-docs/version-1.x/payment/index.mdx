WCPOS unterstützt mehrere Zahlungsmethoden, um unterschiedliche Geschäftsanforderungen zu erfüllen.

## Standardmäßige Zahlungsmethoden

Die kostenlose Version enthält zwei integrierte Zahlungsgateways:

### Bargeld

- Eingabe des gezahlten Betrags
- Automatische Wechselgeldberechnung
- Keine zusätzliche Konfiguration erforderlich

### Karte

- Zur Verwendung mit externen Kartenterminals
- Markieren Sie die Zahlung einfach als abgeschlossen, nachdem sie auf Ihrem Terminal verarbeitet wurde
- Keine direkte Integration erforderlich

## Zusätzliche Zahlungsgateways (Pro)

:::info Pro-Funktion
Zusätzliche Zahlungsgateways erfordern [WCPOS Pro](/getting-started/pro-license).
:::

Mit Pro können Sie zusätzliche WooCommerce-Zahlungsgateways an der POS-Kasse aktivieren:

- **Stripe Terminal** - Direkte Integration mit Stripe-Kartenlesegeräten
- **SumUp Terminal** - Integration mit SumUp-Kartenlesegeräten
- **Beliebiges WooCommerce-Gateway** - Aktivieren Sie ein beliebiges in Ihrem Shop installiertes Gateway

## Benutzerdefinierte Gateways

Erstellen Sie Ihre eigenen Zahlungsintegrationen mit dem [Benutzerdefinierte Gateways](/payment/custom-gateways) System.
