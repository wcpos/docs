:::info Pro Funktion
Der Bildschirm für Produkte erfordert [WCPOS Pro](/getting-started/pro-license). Kostenlose Benutzer können Produkte im [POS Produktpanel](/pos/product-panel) anzeigen und in den Warenkorb legen, können jedoch Bestände oder Preise nicht bearbeiten.
:::

Der Bildschirm für Produkte ist eine spezielle Benutzeroberfläche zur Verwaltung von Beständen. Im Gegensatz zum [POS Produktpanel](/pos/product-panel) (das zum Verkaufen gedacht ist) ist dieser Bildschirm zum Verwalten Ihres Produktkatalogs konzipiert.

{/* Screenshot: products-screen.png */}

## Produkte Bildschirm vs. POS Produktpanel

| Funktion | POS Produktpanel | Produkte Bildschirm |
|----------|------------------|---------------------|
| **Zweck** | Produkte in den Warenkorb legen | Bestände verwalten |
| **Bestand** | Nur anzeigen | Inline bearbeiten |
| **Preise** | Nur anzeigen | Inline bearbeiten |
| **Aktionen** | In den Warenkorb legen | Bearbeiten, Synchronisieren, Löschen |
| **Verfügbar in** | Frei + Pro | Nur Pro |

## Überblick über die Benutzeroberfläche

### Suche & Filter

Oben auf dem Bildschirm:

- **Suchleiste** - Produkte nach Name, SKU oder Barcode finden
- **Filtertasten** - Lagerstatus, Vorgestellt, Im Angebot, Kategorie, Tag, Marke
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Sichtbare Spalten konfigurieren

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Tabelle an:

- **Bild** - Produktminiatur
- **Produktname** - Mit optionaler SKU, Barcode, Attributen
- **Bestand** - Bearbeitbare Bestandsmenge mit "Verwalten" Umschalter
- **Lagerstatus** - Auf Lager (grün), Ausverkauft (rot), Nachbestellung
- **Kategorien** - Produktkategorie-Abzeichen
- **Preise** - Aktueller Preis, regulärer Preis, Sonderpreis (alle bearbeitbar)
- **Aktionen** - Drei-Punkte-Menü mit Bearbeiten, Synchronisieren, Löschen

### Variable Produkte

Variable Produkte zeigen:
- Attributoptionen (Farbe, Größe usw.)
- "Erweitern" Link, um alle Variationen anzuzeigen
- Einzelne Variationszeilen mit separatem Bestand/Preisen

### Fußzeile

- **Steuerstatus** - Aktuelle Steuerberechnungsgrundlage
- **Produktanzahl** - "Zeigt X von Y" mit Synchronisieren-Schaltfläche

## Hauptmerkmale

### Inline-Bestandsbearbeitung

Klicken Sie direkt auf die Bestandsmenge, um sie zu bearbeiten:

1. Klicken Sie auf die Bestandsnummer
2. Geben Sie die neue Menge ein
3. Drücken Sie die Eingabetaste, um zu speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

### Bestandsverwaltung Umschalter

Der "Verwalten" Umschalter steuert, ob WooCommerce den Bestand für dieses Produkt verfolgt:

- **Ein** - Bestände werden verfolgt und beim Verkauf vermindert
- **Aus** - Produkt ist immer verfügbar (unendlicher Bestand)

### Inline-Preisbearbeitung

Klicken Sie auf jedes Preisfeld, um es zu bearbeiten:

- **Preis** - Aktueller Verkaufspreis
- **Regulärer Preis** - Preis ohne Rabatt
- **Verkaufspreis** - Rabattierter Preis (wenn im Angebot)

### Produktaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für jedes Produkt:

- **Bearbeiten** - Öffnen Sie das Produktbearbeitungsmodul
- **Synchronisieren** - Dieses Produkt von WooCommerce aktualisieren
- **Löschen** - Produkt aus der lokalen Datenbank entfernen

:::note
Das Löschen eines Produkts aus dem POS entfernt es nur lokal. Das Produkt bleibt in WooCommerce und wird beim nächsten Synchronisieren wieder angezeigt.
:::

### Verfügbare Spalten

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|--------------|------------------|
| **Bild** | Produktminiatur | - |
| **ID** | WooCommerce Produkt-ID | - |
| **Produkt** | Produktname | SKU, Barcode, Attribute |
| **Typ** | Einfach, variable, gruppiert usw. | - |
| **SKU** | Lagerverwaltungseinheit | - |
| **Barcode** | Produktbarcode | - |
| **Bestand** | Aktuelle Menge (bearbeitbar) | - |
| **Lagerstatus** | Auf Lager / Ausverkauft / Nachbestellung | - |
| **Kategorien** | Produktkategorien | - |
| **Tags** | Produkt-Tags | - |
| **Marken** | Markeninfo (falls Plugin aktiv) | - |
| **Herstellungskosten** | Produktkostenpreis | - |
| **Preis** | Aktueller Verkaufspreis (bearbeitbar) | Steuerinfo |
| **Regulärer Preis** | Preis ohne Rabatt (bearbeitbar) | Steuerinfo |
| **Verkaufspreis** | Rabattierter Preis (bearbeitbar) | Steuerinfo |
| **Erstellungsdatum** | Wann das Produkt erstellt wurde | - |
| **Änderungsdatum** | Letztes Änderungsdatum | - |
| **Aktionen** | Buttons für Bearbeiten, Synchronisieren, Löschen | - |

### Standardwerte wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen auf die ursprünglichen Standardeinstellungen zurückzusetzen.

## Produkte filtern

Verwenden Sie die Filtertasten, um Ihre Ansicht einzuschränken:

- **Lagerstatus** - Auf Lager, Ausverkauft, Nachbestellung
- **Vorgestellt** - Produkte, die als hervorgehoben gekennzeichnet sind
- **Im Angebot** - Produkte, die derzeit im Angebot sind
- **Kategorie** - Nach Produktkategorie filtern
- **Tag** - Nach Produkt-Tag filtern
- **Marke** - Nach Marke filtern (falls Marken-Plugin verwendet wird)

## Sortierung

Klicken Sie auf die Überschrift einer beliebigen Spalte, um nach dieser Spalte zu sortieren. Klicken Sie erneut, um die Sortierreihenfolge umzukehren.
