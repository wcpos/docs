:::info Pro Feature
Der Produkte-Bildschirm erfordert [WCPOS Pro](/getting-started/pro-license). Kostenlose Benutzer können Produkte im [POS-Produkt-Panel](/pos/product-panel) anzeigen und in den Warenkorb legen, aber keine Bestände oder Preise bearbeiten.
:::

Der Produkte-Bildschirm ist eine dedizierte Schnittstelle für die Bestandsverwaltung. Im Gegensatz zum [POS-Produkt-Panel](/pos/product-panel) (das zum Verkaufen dient), ist dieser Bildschirm für die Verwaltung Ihres Produktkatalogs konzipiert.

{/* Screenshot: products-screen.png */}

## Produkte-Bildschirm vs POS-Produkt-Panel

| Funktion | POS-Produkt-Panel | Produkte-Bildschirm |
|----------|-------------------|---------------------|
| **Zweck** | Produkte in den Warenkorb hinzufügen | Bestände verwalten |
| **Bestand** | Nur anzeigen | Inline bearbeiten |
| **Preise** | Nur anzeigen | Inline bearbeiten |
| **Aktionen** | In den Warenkorb legen | Bearbeiten, Synchronisieren, Löschen |
| **Verfügbar in** | Kostenlos + Pro | Nur Pro |

## Übersicht der Schnittstelle

### Suche & Filter

Oben auf dem Bildschirm:

- **Suchleiste** - Produkte nach Name, SKU oder Barcode suchen
- **Filtertasten** - Bestandsstatus, Vorgestellt, Im Sale, Kategorie, Tag, Marke
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Sichtbare Spalten konfigurieren

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Tabelle an:

- **Bild** - Produktminiaturansicht
- **Produktname** - Mit optionaler SKU, Barcode, Attributen
- **Bestand** - Bearbeitbare Bestandsmenge mit „Verwalten“-Umschalter
- **Bestandsstatus** - Auf Lager (grün), Ausverkauft (rot), Nachbestellung
- **Kategorien** - Produktkategorie-Abzeichen
- **Preise** - Aktueller Preis, regulärer Preis, Verkaufspreis (alle bearbeitbar)
- **Aktionen** - Drei-Punkte-Menü mit Bearbeiten, Synchronisieren, Löschen

### Variable Produkte

Variable Produkte zeigen:
- Attributoptionen (Farbe, Größe usw.)
- "Erweitern"-Link, um alle Variationen anzuzeigen
- Einzelne Variationszeilen mit separatem Bestand/Preisen

### Fußzeile

- **Steuerstatus** - Aktuelle Steuerberechnungsgrundlage
- **Produktanzahl** - "Zeigt X von Y" mit Synchronisieren-Schaltfläche (<Icon name="arrow-rotate-right" />). **Lang drücken** der Synchronisieren-Schaltfläche für die Optionen Löschen und Aktualisieren

## Hauptmerkmale

### Inline-Bestandsbearbeitung

Klicken Sie direkt auf die Bestandsmenge, um sie zu bearbeiten:

1. Klicken Sie auf die Bestandsnummer
2. Geben Sie die neue Menge ein
3. Drücken Sie die Eingabetaste, um zu speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

### Bestand verwalten Umschalter

Der Umschalter „Verwalten“ steuert, ob WooCommerce den Bestand für dieses Produkt verfolgt:

- **Ein** - Bestandsmengen werden verfolgt und beim Verkauf verringert
- **Aus** - Produkt ist immer verfügbar (unendlicher Bestand)

### Inline-Preisbearbeitung

Klicken Sie auf ein beliebiges Preisfeld, um es zu bearbeiten:

- **Preis** - Aktueller Verkaufspreis
- **Regulärer Preis** - Preis ohne Rabatt
- **Verkaufspreis** - Preis mit Rabatt (bei Angeboten)

### Produktaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für jedes Produkt:

- **Bearbeiten** - Öffnen Sie das Produkt-Bearbeitungsmodal
- **Synchronisieren** - Aktualisieren Sie dieses Produkt von WooCommerce
- **Löschen** - Produkt aus der lokalen Datenbank entfernen

:::note
Das Löschen eines Produkts vom POS entfernt es nur lokal. Das Produkt bleibt in WooCommerce und wird beim nächsten Synchronisieren wieder angezeigt.
:::

Klicken Sie auf das Symbol „Regler“ (<Icon name="sliders" />), um den Produkte-Bildschirm anzupassen.

### Verfügbare Spalten

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|-------------|-----------------|
| **Bild** | Produktminiaturansicht | - |
| **ID** | WooCommerce-Produkt-ID | - |
| **Produkt** | Produktname | SKU, Barcode, Attribute |
| **Typ** | Einfach, variabel, gruppiert usw. | - |
| **SKU** | Lagerhaltungseinheit | - |
| **Barcode** | Produktbarcode | - |
| **Bestand** | Aktuelle Menge (bearbeitbar) | - |
| **Bestandsstatus** | Auf Lager / Ausverkauft / Nachbestellung | - |
| **Kategorien** | Produktkategorien | - |
| **Tags** | Produkt-Tags | - |
| **Marken** | Markeninformationen (wenn Plugin aktiv) | - |
| **Kosten der verkauften Waren** | Produktkosten | - |
| **Preis** | Aktueller Verkaufspreis (bearbeitbar) | Steuerinfo |
| **Regulärer Preis** | Preis ohne Rabatt (bearbeitbar) | Steuerinfo |
| **Verkaufspreis** | Preis mit Rabatt (bearbeitbar) | Steuerinfo |
| **Date Created** | Wann das Produkt erstellt wurde | - |
| **Date Modified** | Datum der letzten Änderung | - |
| **Aktionen** | Schaltflächen zum Bearbeiten, Synchronisieren, Löschen | - |

### Standardwerte wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen auf ihre ursprünglichen Standardeinstellungen zurückzusetzen.

## Produkte filtern

Verwenden Sie die Filtertasten, um Ihre Ansicht einzuschränken:

- **Bestandsstatus** - Auf Lager, Ausverkauft, Nachbestellung
- **Vorgestellt** - Produkte, die als vorgestellt markiert sind
- **Im Sale** - Produkte, die derzeit im Angebot sind
- **Kategorie** - Nach Produktkategorie filtern
- **Tag** - Nach Produkt-Tag filtern
- **Marke** - Nach Marke filtern (wenn Sie das Marken-Plugin verwenden)

## Sortieren

Klicken Sie auf eine beliebige Spaltenüberschrift, um nach dieser Spalte zu sortieren. Klicken Sie erneut, um die Sortierreihenfolge umzukehren.
