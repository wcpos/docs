:::info Pro Funktion
Der Produktbildschirm benötigt [WCPOS Pro](/getting-started/pro-license). Kostenlose Benutzer können Produkte im [POS-Produktpanel](/pos/product-panel) anzeigen und zum Warenkorb hinzufügen, aber keine Bestände oder Preise bearbeiten.
:::

Der Produktbildschirm ist eine dedizierte Benutzeroberfläche zur Verwaltung von Beständen. Im Gegensatz zum [POS-Produktpanel](/pos/product-panel) (das zum Verkaufen dient) ist dieser Bildschirm für die Verwaltung Ihres Produktkatalogs konzipiert.

{/* Screenshot: products-screen.png */}

## Produktbildschirm vs. POS-Produktpanel

| Funktion | POS-Produktpanel | Produktbildschirm |
|----------|------------------|-------------------|
| **Zweck** | Produkte in den Warenkorb hinzufügen | Bestände verwalten |
| **Bestand** | Nur anzeigen | Inline bearbeiten |
| **Preise** | Nur anzeigen | Inline bearbeiten |
| **Aktionen** | In den Warenkorb hinzufügen | Bearbeiten, Synchronisieren, Löschen |
| **Verfügbar in** | Kostenlos + Pro | Nur Pro |

## Übersicht der Benutzeroberfläche

### Suche & Filter

Oben auf dem Bildschirm:

- **Suchleiste** - Produkte nach Name, SKU oder Barcode suchen
- **Filtertasten** - Lagerbestand, Vorgestellt, Im Angebot, Kategorie, Tag, Marke
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Sichtbare Spalten konfigurieren

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Tabelle an:

- **Bild** - Produktvorschau
- **Produktname** - Mit optionaler SKU, Barcode, Attribute
- **Bestand** - Bearbeitbare Bestandsmenge mit "Verwalten"-Toggle
- **Bestandsstatus** - Auf Lager (grün), Nicht auf Lager (rot), Nachbestellung
- **Kategorien** - Produktkategorie-Abzeichen
- **Preise** - Aktueller Preis, regulärer Preis, Angebotspreis (alle bearbeitbar)
- **Aktionen** - Drei-Punkte-Menü mit Bearbeiten, Synchronisieren, Löschen

### Variable Produkte

Variable Produkte zeigen:
- Attributoptionen (Farbe, Größe usw.)
- "Erweitern"-Link, um alle Variationen anzuzeigen
- Einzelne Variationszeilen mit separatem Bestand/Preisen

### Fußzeile

- **Steuerstatus** - Aktuelle Grundlage für die Steuerberechnung
- **Produktanzahl** - "Zeige X von Y" mit Synchronisationsbutton (<Icon name="arrow-rotate-right" />). **Lange drücken** auf den Synchronisationsbutton für die Optionen Löschen und Aktualisieren

## Hauptfunktionen

### Inline-Bestandsbearbeitung

Klicken Sie direkt auf die Bestandsmenge, um diese zu bearbeiten:

1. Klicken Sie auf die Bestandsnummer
2. Geben Sie die neue Menge ein
3. Drücken Sie Enter, um zu speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

### Bestandsverwaltungs-Toggle

Der "Verwalten"-Toggle steuert, ob WooCommerce den Bestand für dieses Produkt verfolgt:

- **Ein** - Bestandslevel werden verfolgt und beim Verkauf verringert
- **Aus** - Produkt ist immer verfügbar (unendlicher Bestand)

### Inline-Preisbearbeitung

Klicken Sie auf ein beliebiges Preisfeld, um es zu bearbeiten:

- **Preis** - Aktueller Verkaufspreis
- **Regulärer Preis** - Nicht reduzierter Preis
- **Angebotspreis** - Reduzierter Preis (wenn im Angebot)

### Produktaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für jedes Produkt:

- **Bearbeiten** - Modal zur Produktbearbeitung öffnen
- **Synchronisieren** - Dieses Produkt aus WooCommerce aktualisieren
- **Löschen** - Produkt aus der lokalen Datenbank entfernen

:::note
Das Löschen eines Produkts aus dem POS entfernt es nur lokal. Das Produkt bleibt in WooCommerce und wird beim nächsten Synchronisieren wieder angezeigt.
:::

### Verfügbare Spalten

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|--------------|-----------------|
| **Bild** | Produktvorschau | - |
| **ID** | WooCommerce Produkt-ID | - |
| **Produkt** | Produktname | SKU, Barcode, Attribute |
| **Typ** | Einfach, variabel, gruppiert usw. | - |
| **SKU** | Lagerhaltungseinheit | - |
| **Barcode** | Produktbarcode | - |
| **Bestand** | Aktuelle Menge (bearbeitbar) | - |
| **Bestandsstatus** | Auf Lager / Nicht auf Lager / Nachbestellung | - |
| **Kategorien** | Produktkategorien | - |
| **Tags** | Produkt-Tags | - |
| **Marken** | Markeninfo (wenn Plug-in aktiv) | - |
| **Kosten der verkauften Waren** | Produktkosten | - |
| **Preis** | Aktueller Verkaufspreis (bearbeitbar) | Steuerinfo |
| **Regulärer Preis** | Nicht reduzierter Preis (bearbeitbar) | Steuerinfo |
| **Angebotspreis** | Reduzierter Preis (bearbeitbar) | Steuerinfo |
| **Erstellt am** | Wann das Produkt erstellt wurde | - |
| **Geändert am** | Datum der letzten Änderung | - |
| **Aktionen** | Schaltflächen für Bearbeiten, Synchronisieren, Löschen | - |

### Standardwerte wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen auf die ursprünglichen Standardwerte zurückzusetzen.

## Produkte filtern

Verwenden Sie die Filtertasten, um Ihre Ansicht einzuschränken:

- **Bestandsstatus** - Auf Lager, Nicht auf Lager, Nachbestellung
- **Vorgestellt** - Produkte, die als hervorgehoben markiert sind
- **Im Angebot** - Produkte, die derzeit im Angebot sind
- **Kategorie** - Nach Produktkategorie filtern
- **Tag** - Nach Produkt-Tag filtern
- **Marke** - Nach Marke filtern (wenn Marken-Plugin verwendet wird)

## Sortierung

Klicken Sie auf einen Spaltenkopf, um nach dieser Spalte zu sortieren. Klicken Sie erneut, um die Sortierreihenfolge umzukehren.
