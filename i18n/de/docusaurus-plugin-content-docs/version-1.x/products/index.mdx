:::info Pro-Funktion
Der Produktbildschirm erfordert [WCPOS Pro](/getting-started/pro-license). Benutzer der Gratis-Version können Produkte im [POS-Produktpanel](/pos/product-panel) anzeigen und zum Warenkorb hinzufügen, können aber Lagerbestand und Preise nicht bearbeiten.
:::

Der Produktbildschirm ist eine dedizierte Benutzeroberfläche zur Bestandsverwaltung. Im Gegensatz zum [POS-Produktpanel](/pos/product-panel) (das zum Verkaufen dient), ist dieser Bildschirm für die Verwaltung Ihres Produktkatalogs konzipiert.

{/* Screenshot: products-screen.png */}

## Produktbildschirm vs POS-Produktpanel

| Funktion | POS-Produktpanel | Produktbildschirm |
|---------|-------------------|-----------------|
| **Zweck** | Produkte zum Warenkorb hinzufügen | Bestand verwalten |
| **Lagerbestand** | Nur anzeigen | Inline bearbeiten |
| **Preise** | Nur anzeigen | Inline bearbeiten |
| **Aktionen** | Zum Warenkorb hinzufügen | Bearbeiten, Synchronisieren, Löschen |
| **Verfügbar in** | Gratis + Pro | Nur Pro |

## Schnittstellenübersicht

### Suche & Filter

Am oberen Bildschirmrand:

- **Suchleiste** - Produkte nach Name, SKU oder Barcode finden
- **Filterbuttons** - Lagerstatus, Empfohlen, Im Angebot, Kategorie, Tag, Marke
- **Anzeigeeinstellungen** (<Icon name="sliders" />) - Sichtbare Spalten konfigurieren

### Produktliste

Der Hauptbereich zeigt Ihre Produkte in einer scrollbaren Tabelle:

- **Bild** - Produktvorschaubild
- **Produktname** - Mit optionaler SKU, Barcode, Attributen
- **Lagerbestand** - Bearbeitbare Lagermenge mit "Verwalten"-Schalter
- **Lagerstatus** - Auf Lager (grün), Nicht auf Lager (rot), Vorbestellung
- **Kategorien** - Produkt-Kategorien-Badges
- **Preise** - Aktueller Preis, Grundpreis, Sonderpreis (alle bearbeitbar)
- **Aktionen** - Drei-Punkte-Menü mit Bearbeiten, Synchronisieren, Löschen

### Variantenprodukte

Variantenprodukte zeigen:
- Attributoptionen (Farbe, Größe, etc.)
- "Erweitern"-Link zur Ansicht aller Varianten
- Einzelne Variantenzeilen mit separatem Lagerbestand/Preisen

### Fußzeile

- **Steuerstatus** - Aktuelle Steuerberechnungsgrundlage
- **Produktanzahl** - "X von Y anzeigen" mit Synchronisierungsbutton

## Hauptfunktionen

### Inline-Lagerbestandsbearbeitung

Klicken Sie direkt auf die Lagermenge, um sie zu bearbeiten:

1. Klicken Sie auf die Lagerbestandszahl
2. Geben Sie die neue Menge ein
3. Drücken Sie Enter zum Speichern

Änderungen werden automatisch mit WooCommerce synchronisiert.

### Lagerbestand-Verwaltungsschalter

Der "Verwalten"-Schalter steuert, ob WooCommerce den Lagerbestand für dieses Produkt verfolgt:

- **An** - Lagerbestände werden verfolgt und bei Verkäufen reduziert
- **Aus** - Produkt ist immer verfügbar (unbegrenzter Lagerbestand)

### Inline-Preisbearbeitung

Klicken Sie auf beliebige Preisfelder zur Bearbeitung:

- **Preis** - Aktueller Verkaufspreis
- **Grundpreis** - Nicht rabattierter Preis
- **Sonderpreis** - Rabattierter Preis (bei Angeboten)

### Produktaktionen

Klicken Sie auf das Drei-Punkte-Menü (⋮) für jedes Produkt:

- **Bearbeiten** - Produktbearbeitungs-Modal öffnen
- **Synchronisieren** - Dieses Produkt von WooCommerce aktualisieren
- **Löschen** - Produkt aus lokaler Datenbank entfernen

:::note
Das Löschen eines Produkts aus dem POS entfernt es nur lokal. Das Produkt bleibt in WooCommerce und erscheint bei der nächsten Synchronisierung wieder.
:::

### Verfügbare Spalten

| Spalte | Beschreibung | Anzeigeoptionen |
|--------|-------------|-----------------|
| **Bild** | Produktvorschaubild | - |
| **ID** | WooCommerce-Produkt-ID | - |
| **Produkt** | Produktname | SKU, Barcode, Attribute |
| **Typ** | Einfach, variabel, Gruppe, etc. | - |
| **SKU** | Stock Keeping Unit | - |
| **Barcode** | Produktbarcode | - |
| **Lagerbestand** | Aktuelle Menge (bearbeitbar) | - |
| **Lagerstatus** | Auf Lager / Nicht auf Lager / Vorbestellung | - |
| **Kategorien** | Produktkategorien | - |
| **Tags** | Produkt-Tags | - |
| **Marken** | Markeninformationen (wenn Plugin aktiv) | - |
| **Wareneinsatz** | Produkteinkaufspreis | - |
| **Preis** | Aktueller Verkaufspreis (bearbeitbar) | Steuerinformationen |
| **Grundpreis** | Nicht rabattierter Preis (bearbeitbar) | Steuerinformationen |
| **Sonderpreis** | Rabattierter Preis (bearbeitbar) | Steuerinformationen |
| **Erstellungsdatum** | Wann das Produkt erstellt wurde | - |
| **Änderungsdatum** | Letztes Änderungsdatum | - |
| **Aktionen** | Bearbeiten, Synchronisieren, Löschen Buttons | - |

### Standardeinstellungen wiederherstellen

Klicken Sie, um alle Anzeigeeinstellungen auf ihre ursprünglichen Standardwerte zurückzusetzen.

## Produkte filtern

Verwenden Sie die Filterbuttons, um Ihre Ansicht einzuschränken:

- **Lagerstatus** - Auf Lager, Nicht auf Lager, Vorbestellung
- **Empfohlen** - Als empfohlen markierte Produkte
- **Im Angebot** - Produkte, die aktuell im Angebot sind
- **Kategorie** - Nach Produktkategorie filtern
- **Tag** - Nach Produkt-Tag filtern
- **Marke** - Nach Marke filtern (wenn Marken-Plugin verwendet wird)

## Sortieren

Klicken Sie auf eine Spaltenüberschrift, um nach dieser Spalte zu sortieren. Klicken Sie erneut, um die Sortierreihenfolge umzukehren.
