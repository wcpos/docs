POS-Exklusive Produkte ist ein Feature, mit dem Sie steuern können, ob Produkte in Ihrem Online-Shop, Ihrem POS oder in beiden angezeigt werden. Dies ist nützlich, wenn Sie unterschiedliche Produktkataloge für den Online- und den Einzelhandelsverkauf haben.

## Übersicht {#overview}

Wenn aktiviert, erhält jedes Produkt eine **POS-Sichtbarkeit**-Einstellung mit drei Optionen:

| Sichtbarkeit | Online-Shop | POS |
|--------------|-------------|-----|
| **POS & Online** (Standardeinstellung) | Sichtbar | Sichtbar |
| **POS-Exklusiv** | Ausgeblendet | Sichtbar |
| **Online-Exklusiv** | Sichtbar | Ausgeblendet |

## Wann man dieses Feature verwenden sollte {#when-to-use-this-feature}

Aktivieren Sie POS-Exklusive Produkte, wenn Sie:

- **Exklusive Produkte im Geschäft verkaufen** - Produkte, die nur verfügbar sind, wenn Kunden Ihr physisches Geschäft besuchen
- **Online-exklusive Artikel vom POS ausblenden** - Digitale Produkte, Vorbestellungen oder Artikel, die im Geschäft nicht verkauft werden sollten
- **Unterschiedliche Kataloge verwalten** - Getrennte Produktreihen für Online- und Einzelhandelskanäle
- **Werbeartikel steuern** - Nur im Geschäft verfügbare Sonderangebote oder exklusive Online-Angebote

:::tip
Wenn alle Ihre Produkte in beiden Kanälen erscheinen sollen, benötigen Sie dieses Feature nicht. Lassen Sie es deaktiviert für bessere Leistung.
:::

## Aktivierung des Features {#enabling-the-feature}

1. Gehen Sie zu **WP Admin → WCPOS → Einstellungen → Allgemein**
2. Aktivieren Sie **"POS-Exklusive Produkte aktivieren"**
3. Änderungen speichern

Für weitere Details zu den allgemeinen Einstellungen siehe [WP Admin Allgemeine Einstellungen](/settings/wp-admin/general).

:::warning Leistungsbeeinflussung
Aktivieren Sie diese Einstellung nur, wenn Sie sie benötigen. Wenn aktiviert, fügt sie für jede Produktanfrage einen zusätzlichen Datenbankabruf hinzu, was die Leistung bei Geschäften mit großen Katalogen beeinträchtigen kann.
:::

## Einstellung der Produktsichtbarkeit {#setting-product-visibility}

### Einzelne Produkte {#individual-products}

Um die Sichtbarkeit für ein einzelnes Produkt festzulegen:

1. Gehen Sie zu **WP Admin → Produkte** und bearbeiten Sie ein Produkt
2. Suchen Sie im **Produktdaten**-Panel nach **POS-Sichtbarkeit**
3. Wählen Sie Ihre gewünschte Option:
   - **POS & Online** - Produkt erscheint überall (Standardeinstellung)
   - **POS-Exklusiv** - Produkt ist im Online-Shop ausgeblendet
   - **Online-Exklusiv** - Produkt ist im POS ausgeblendet
4. Klicken Sie auf **Aktualisieren**, um zu speichern

### Massenbearbeitung {#bulk-editing}

Um die Sichtbarkeit für mehrere Produkte gleichzeitig zu ändern:

1. Gehen Sie zu **WP Admin → Produkte**
2. Wählen Sie die Produkte aus, die Sie mit den Kontrollkästchen ändern möchten
3. Wählen Sie im Dropdown-Menü **Massenaktionen** **Bearbeiten**
4. Klicken Sie auf **Anwenden**
5. Finden Sie im Massenbearbeitungsfeld das Dropdown-Menü **POS-Sichtbarkeit**
6. Wählen Sie Ihre gewünschte Sichtbarkeitsoption
7. Klicken Sie auf **Aktualisieren**

### Variable Produkte {#variable-products}

Für variable Produkte können Sie die Sichtbarkeit sowohl auf der Ebene des übergeordneten Produkts als auch für jede einzelne Variation festlegen. Dies gibt Ihnen eine feinkörnige Kontrolle - zum Beispiel können Sie alle Variationen online anzeigen, aber nur bestimmte Farben oder Größen im Geschäft.

## Wie es den POS beeinflusst {#how-it-affects-the-pos}

Wenn ein Produkt auf **Online-Exklusiv** eingestellt ist:

- Es wird nicht im [Produktpanel](/pos/product-panel) angezeigt
- Es kann nicht über die Suche oder das Scannen von Barcodes gefunden werden
- Es kann im POS nicht zum Warenkorb hinzugefügt werden

Wenn ein Produkt auf **POS-Exklusiv** eingestellt ist:

- Es wird nicht im Frontend Ihres WooCommerce-Shops angezeigt
- Es wird nicht in Online-Suchergebnissen einbezogen
- Kunden können es nicht online kaufen

:::note
Wenn Sie die Sichtbarkeit eines Produkts kürzlich geändert haben und es immer noch im POS erscheint (oder nicht erscheint), versuchen Sie eine Synchronisation. Halten Sie die Synchronisationstaste lange gedrückt und wählen Sie **"Löschen und Aktualisieren"**, um alle Produktdaten neu zu laden.
:::

## Häufige Anwendungsfälle {#common-use-cases}

<UseCases>
  <UseCaseCard icon="gift" title="Nur im Geschäft verfügbare Geschenkkarten">

  Physische Geschenkkarten, die an der Kasse aktiviert werden müssen:

  1. Erstellen Sie das Geschenkkartenprodukt
  2. Stellen Sie die Sichtbarkeit auf **POS-Exklusiv**
  3. Die Karte wird nicht online angezeigt, aber das Personal kann sie im Geschäft verkaufen

  </UseCaseCard>

  <UseCaseCard icon="globe" title="Online-Vorbestellungen">

  Produkte, die online zur Vorbestellung verfügbar sind, aber noch nicht auf Lager sind:

  1. Erstellen Sie das Vorbestellprodukt
  2. Stellen Sie die Sichtbarkeit auf **Online-Exklusiv**
  3. Kunden können online vorbestellen, aber es wird den POS nicht überladen

  </UseCaseCard>

  <UseCaseCard icon="building" title="Großhandel vs. Einzelhandel">

  Wenn Sie denselben WooCommerce-Shop für Großhandel und Einzelhandel nutzen:

  1. Stellen Sie Großhandelsprodukte auf **Online-Exklusiv** (für den B2B-Portalzugang)
  2. Stellen Sie Einzelhandelsprodukte auf **POS-Exklusiv**
  3. Halten Sie reguläre Produkte als **POS & Online**

  </UseCaseCard>
</UseCases>

## Fehlerbehebung {#troubleshooting}

<Accordion>
  <AccordionItem question="Produkte erscheinen weiterhin nach Einstellung auf Online-Exklusiv">

  1. Öffnen Sie den POS
  2. Halten Sie die Synchronisationstaste am unteren Ende des Produktpanels lange gedrückt
  3. Wählen Sie **"Löschen und Aktualisieren"**
  4. Warten Sie, bis die Produkte neu geladen werden

  </AccordionItem>

  <AccordionItem question="Einstellung wird nicht auf Produkten angezeigt">

  Stellen Sie sicher, dass das Feature aktiviert ist:

  1. Gehen Sie zu **WP Admin → WCPOS → Einstellungen → Allgemein**
  2. Überprüfen Sie, ob **"POS-Exklusive Produkte aktivieren"** angekreuzt ist
  3. Speichern Sie die Einstellungen, wenn Sie Änderungen vorgenommen haben

  </AccordionItem>
</Accordion>
