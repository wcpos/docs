I template di stampa delle ricevute POS risiedono nel file `includes/views/print/tmpl-receipt.php` sia nel plugin gratuito che in quello Pro. I template delle ricevute possono essere personalizzati creando un file `woocommerce-pos/print/tmpl-receipt.php` nella directory del tema. Il codice per entrambi i template è incluso di seguito.

### Personalizzazione della data della ricevuta

WCPOS utilizza [moment.js](http://momentjs.com/) per localizzare le stringhe delle date. Il formato data predefinito è `"MMMM Do YYYY, h:mm:ss a"`, es: 31 Maggio 2015, 19:20:44. Si prega di consultare la [documentazione di moment.js](http://momentjs.com/docs/#/parsing/string-format/) per ulteriori informazioni sulla personalizzazione del formato data.

### Proprietà dell'Ordine

WCPOS utilizza l'output json dall'API REST WC per popolare il template della ricevuta dell'ordine. [La documentazione dell'API REST WC mostra un esempio](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) dell'output json standard. Alcune proprietà aggiuntive sono state aggiunte da WCPOS utilizzando il filtro `woocommerce_api_order_response`.

| Proprietà                         | Descrizione                                                   |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | La parte fiscale dello sconto del carrello                    |
| cashier.id                        | ID utente del Cassiere                                        |
| cashier.first_name                | Nome del Cassiere                                             |
| cashier.last_name                 | Cognome del Cassiere                                          |
| payment_details.result            | Successo o fallimento del gateway di pagamento                |
| payment_details.message           | Eventuali messaggi del gateway di pagamento                   |
| payment_details.redirect          | URL di reindirizzamento per pagamenti fuori sito, es: PayPal Standard |
| payment_details.method\_`gateway` | Messaggi specifici del gateway, es: Importo versato e resto   |
| shipping_lines[i].total_tax       | Aggiungere l'importo delle tasse per ogni riga di spedizione  |
| subtotal_tax                      | La parte fiscale del subtotale                                |
