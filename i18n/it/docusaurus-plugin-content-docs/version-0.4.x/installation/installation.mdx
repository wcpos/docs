### Installazione automatica (consigliata)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installa e attiva WCPOS in meno di 1 minuto")

1. Vai su `WP Admin > Plugin > Aggiungi Nuovo`
2. Cerca `WCPOS` nel Directory dei Plugin di WordPress
3. Installa il plugin
4. Attiva il plugin

### Installazione manuale

1. [Scarica WCPOS](https://wordpress.org/plugins/woocommerce-pos/) da WordPress.org
2. Vai su `WP Admin > Plugin > Aggiungi Nuovo > Carica Plugin`
3. Attiva il plugin

:::info
Le versioni precedenti di WCPOS possono essere trovate nella scheda [Sviluppatori](https://wordpress.org/plugins/woocommerce-pos/developers/) su Wordpress.org.
:::
