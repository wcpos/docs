### Installazione automatica (consigliata) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Installa e attiva WCPOS in meno di 1 minuto")

1. Vai a `WP Admin > Plugin > Aggiungi Nuovo`
2. Cerca `WCPOS` nella directory dei plugin di WordPress
3. Installa il plugin
4. Attiva il plugin

### Installazione manuale {#manual-installation}

1. [Scarica WCPOS](https://wordpress.org/plugins/woocommerce-pos/) da WordPress.org
2. Vai a `WP Admin > Plugin > Aggiungi Nuovo > Carica plugin`
3. Attiva il plugin

:::info
Le versioni precedenti di WCPOS sono disponibili nella scheda [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) su WordPress.org.
:::
