:::important

[WCPOS Pro](http://wcpos.com/pro) è un plugin a pagamento che aggiunge funzionalità al [plugin WCPOS gratuito](http://wordpress.org/plugins/woocommerce-pos/).

:::

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi seguenti per installare e attivare il plugin:

1.  Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Sotto **Download**, fai clic sul link di download e salva il plugin sul tuo desktop.
3.  Poi vai sul tuo sito, accedi e vai su `WP Admin > Plugins > Aggiungi Nuovo > Carica Plugin`
4.  Carica il file zip del plugin dal tuo desktop e attivalo.
5.  Successivamente, vai su `WP Admin > POS > Impostazioni > Licenza` e inserisci la tua Chiave di Licenza e Email di Licenza per completare l'attivazione.

![I miei download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Puoi scaricare WCPOS Pro nella tua pagina account")

Quando saranno pronte nuove versioni, appariranno nel tuo dashboard Aggiornamenti. 
Per informazioni su cosa sto attualmente facendo, ti prego di tenere d'occhio il [Roadmap](http://wcpos.com/roadmap/). 
Inoltre, fai attenzione al changelog che elencherà le nuove funzionalità per ogni versione e la [pagina della documentazione](http://docs.wcpos.com) sarà aggiornata con informazioni su come utilizzare le nuove funzionalità.
