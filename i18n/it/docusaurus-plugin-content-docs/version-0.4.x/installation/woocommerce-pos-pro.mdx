:::important

[WCPOS Pro](http://wcpos.com/pro) è un plugin a pagamento che aggiunge funzionalità al plugin [WCPOS gratuito](http://wordpress.org/plugins/woocommerce-pos/). 

:::

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi seguenti per installare e attivare il plugin:

1.  Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2.  Sotto **Download**, fai clic sul link di download e salva il plugin sul tuo desktop.
3.  Poi vai sul tuo sito, effettua l'accesso e vai su `WP Admin > Plugin > Aggiungi nuovo > Carica plugin`
4.  Carica il file zip del plugin dal tuo desktop e attivalo.
5.  Successivamente, vai su `WP Admin > POS > Impostazioni > Licenza` e inserisci la tua Chiave Licenza e Email di Licenza per completare l'attivazione.

![I miei download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Puoi scaricare WCPOS Pro nella pagina del tuo account")

Quando sono pronti nuove versioni, appariranno nel tuo dashboard Aggiornamenti. 
Per informazioni su ciò su cui sto lavorando attualmente, tieni d'occhio la [Roadmap](http://wcpos.com/roadmap/). 
Tieni inoltre d'occhio il registro delle modifiche che elencherà le nuove funzionalità per ogni versione e la [pagina della documentazione](http://docs.wcpos.com) verrà aggiornata con informazioni su come utilizzare le nuove funzionalità.
