Utilizzando le impostazioni di Accesso POS, i proprietari dei negozi possono concedere o rimuovere i privilegi degli utenti al sistema Point of Sale.
Per impostazione predefinita, i ruoli **Amministratore** e **Manager del Negozio** sono abilitati con tutte le capacità per utilizzare il POS.
Si consiglia di mantenere le impostazioni predefinite a meno che non si sia a conoscenza delle conseguenze.
Ad esempio, concedere il `publish_shop_orders` al ruolo `Abbonati` sarebbe un _idea_ davvero _molto_ sbagliata.

![Esempio delle impostazioni di Accesso POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Esempio delle impostazioni di Accesso POS")

| Capacità                                                               | Accesso                                   |
| ---------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                 | Richiesta per accedere al frontend POS    |
| manage_woocommerce_pos                                                 | Richiesta per accedere alle impostazioni amministrative del POS |
| read_private_products                                                  | Richiesta per visualizzare i prodotti     |
| edit_product <br /> edit_published_products <br /> edit_others_products | Richiesta per modificare prodotti e variazioni |
| read_private_shop_orders                                               | Richiesta per visualizzare gli ordini e le ricevute |
| publish_shop_orders                                                    | Richiesta per creare nuovi ordini         |
| list_users                                                             | Richiesta per visualizzare i clienti      |
| create_users                                                           | Richiesta per creare nuovi clienti        |
| edit_users                                                             | Richiesta per aggiornare i clienti esistenti |
| read_private_shop_coupons                                              | Richiesta per visualizzare i coupon       |
