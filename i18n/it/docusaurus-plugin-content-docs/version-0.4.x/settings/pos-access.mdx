Utilizzando le impostazioni di Accesso POS, i proprietari del negozio possono concedere o rimuovere privilegi utente al sistema Point of Sale.
Per impostazione predefinita, i ruoli **Amministratore** e **Gestore Negozio** sono abilitati con tutte le capacità per l'utilizzo del POS.
Si consiglia di mantenere le impostazioni predefinite a meno che non si siano consapevoli delle conseguenze.
Ad esempio, concedere il permesso `publish_shop_orders` al ruolo `Sottoscrittori` sarebbe un'idea _molto_ cattiva.

![Esempio di impostazioni di Accesso POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Esempio di impostazioni di Accesso POS")

| Capacità                                                                | Accesso                                           |
| ----------------------------------------------------------------------- | ------------------------------------------------- |
| access_woocommerce_pos                                                  | Necessario per accedere al frontend POS           |
| manage_woocommerce_pos                                                  | Necessario per accedere alle impostazioni POS     |
| read_private_products                                                   | Necessario per visualizzare i prodotti            |
| edit_product <br /> edit_published_products <br /> edit_others_products | Necessario per modificare prodotti e variazioni   |
| read_private_shop_orders                                                | Necessario per visualizzare ordini e ricevute     |
| publish_shop_orders                                                     | Necessario per creare nuovi ordini                |
| list_users                                                              | Necessario per visualizzare i clienti             |
| create_users                                                            | Necessario per creare nuovi clienti               |
| edit_users                                                              | Necessario per aggiornare i clienti esistenti     |
| read_private_shop_coupons                                               | Necessario per visualizzare i coupon              |
