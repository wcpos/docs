WCPOS è il plugin di Punto Vendita per [WooCommerce](https://www.woothemes.com/woocommerce/).
Questa documentazione copre il plugin open source, così come il plugin Pro che aggiunge funzionalità al plugin gratuito.

- Scarica [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) da WordPress.org
- Aggiorna a [WCPOS Pro](http://wcpos.com/pro)

:::danger

Assicurati che il numero di versione della documentazione corrisponda alla versione dei plugin installati.

:::

### Altra documentazione
