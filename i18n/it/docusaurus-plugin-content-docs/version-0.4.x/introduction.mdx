WCPOS è il plugin Point of Sale per [WooCommerce](https://www.woothemes.com/woocommerce/).
Questa documentazione copre il plugin open source, nonché il plugin Pro che aggiunge funzionalità al plugin gratuito.

- Scarica [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) da WordPress.org
- Aggiorna a [WCPOS Pro](http://wcpos.com/pro)

:::danger

Assicurati che il numero di versione della documentazione corrisponda alla versione dei plugin installati.

:::

### Altra documentazione

Un elenco di **Domande frequenti** è disponibile su [faq.wcpos.com](http://faq.wcpos.com).
La **documentazione per sviluppatori** è disponibile su [api.wcpos.com](http://api.wcpos.com).
