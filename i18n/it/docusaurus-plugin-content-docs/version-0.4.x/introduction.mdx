WCPOS è il plugin Point of Sale per [WooCommerce](https://www.woothemes.com/woocommerce/).
Questa documentazione copre il plugin open source, così come il plugin Pro che aggiunge funzionalità al plugin gratuito.

- Scarica [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) da WordPress.org
- Passa a [WCPOS Pro](http://wcpos.com/pro)

:::danger

Assicurarsi che il numero di versione della documentazione corrisponda alla versione dei plugin installati.

:::

### Altra documentazione {#other-documentation}

Un elenco di **Domande frequenti** è disponibile su [faq.wcpos.com](http://faq.wcpos.com).
La **documentazione per sviluppatori** è disponibile su [api.wcpos.com](http://api.wcpos.com).

Per domande non trattate in questa documentazione o nelle FAQ, è possibile contattarci all'indirizzo [support@wcpos.com](mailto:support@wcpos.com).
