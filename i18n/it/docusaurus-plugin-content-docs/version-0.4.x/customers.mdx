### Capacità

I cassieri avranno bisogno delle capacità _list_users_, _create_users_ e _edit_users_ per utilizzare tutte le funzionalità Clienti.

### WordPress Multisite

Quando si utilizza [WordPress multisite](https://codex.wordpress.org/Create_A_Network), ci sono alcuni [controlli aggiuntivi](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) per impedire ai non [Super Admin](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) di creare e modificare i clienti.

Per prima cosa, affinché i non Super Admin possano **creare** nuovi clienti, abilitare la seguente opzione in `Network Admin > Settings`.

![Opzione multisite per creare nuovi utenti](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Opzione multisite per creare nuovi utenti")

Inoltre, affinché i non Super Admin possano **modificare** i dettagli dei clienti, è necessario aggiungere la capacità `manage_network_users` al cassiere.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
