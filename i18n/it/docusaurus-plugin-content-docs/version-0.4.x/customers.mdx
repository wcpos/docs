### Capacità

I cassieri avranno bisogno delle capacità _list_users_, _create_users_ e _edit_users_ per utilizzare tutte le funzionalità dei Clienti.

### WordPress Multisite

Quando si utilizza [WordPress multisite](https://codex.wordpress.org/Create_A_Network) ci sono alcuni [controlli aggiuntivi](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) per impedire ai non [Super Amministratori](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) di creare e modificare clienti.

Innanzitutto, per consentire ai non Super Amministratori di **creare** nuovi clienti, abilitare l'opzione seguente in `Amministrazione di Rete > Impostazioni`.

![Opzione Multisite per creare nuovi utenti](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Opzione Multisite per creare nuovi utenti")

Inoltre, per consentire ai non Super Amministratori di **modificare** i dettagli del cliente, sarà necessario aggiungere la capacità `manage_network_users` al cassiere.

```php title="functions.php"
// cambiare CASHIER_ID, e
// aggiungere al file functions.php

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
