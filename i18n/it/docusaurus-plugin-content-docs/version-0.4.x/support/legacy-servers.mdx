---

HTTP (e HTTPS) sono i protocolli utilizzati da ogni browser per comunicare con le applicazioni web.
Una parte chiave del protocollo è il [metodo HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Tradizionalmente, le applicazioni web utilizzavano un insieme molto limitato di metodi HTTP come GET (recupera una pagina web) e POST (invia un modulo web).

Le applicazioni web moderne, come WCPOS, utilizzano un approccio REST/HTTP con i seguenti metodi HTTP:

- GET
- POST
- PUT
- PATCH
- DELETE

In alcuni rari casi, un provider web può scegliere di bloccare alcuni di questi metodi HTTP.
Se il tuo provider web blocca i metodi HTTP RESTful moderni, dovrai abilitare il supporto per i server legacy.
