HTTP (e HTTPS) sono i protocolli utilizzati da ogni browser per comunicare con le applicazioni web.
Una parte chiave del protocollo è il [metodo HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Tradizionalmente, le applicazioni web utilizzavano un set molto limitato di metodi HTTP come GET (recuperare una pagina web) e POST (inviare un modulo web).

Le applicazioni web moderne, come WCPOS, utilizzano un approccio REST/HTTP con i seguenti metodi HTTP:

- GET
- POST
- PUT
- PATCH
- DELETE

In alcuni casi rari, un web host può scegliere di bloccare alcuni di questi metodi HTTP.
Se il tuo web host blocca i metodi HTTP RESTful moderni, dovrai abilitare il supporto per server legacy.
