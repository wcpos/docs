Per impostazione predefinita, WCPOS utilizzerà il campo SKU come codice a barre.

![Per impostazione predefinita WCPOS utilizzerà il campo SKU come codice a barre.](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Per impostazione predefinita WCPOS utilizzerà il campo SKU come codice a barre")

Gli utenti di [WCPOS Pro](http://wcpos.com/pro) possono utilizzare qualsiasi campo meta del prodotto selezionando un Campo Codice a Barre in `WP Admin > POS > Impostazioni > Generale`

![Seleziona un campo codice a barre personalizzato in WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Seleziona un campo codice a barre personalizzato in WCPOS Pro")

I campi meta dei codici a barre personalizzati possono essere impostati anche utilizzando il filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Campo meta del prodotto personalizzato da utilizzare come codice a barre
 *
 * Per impostazione predefinita WCPOS utilizzerà il campo SKU - '_sku'
 * È possibile modificare la chiave meta utilizzando il seguente filtro di WordPress
 *
 * Di seguito sono riportati i campi meta di plugin per codici a barre popolari
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // cambia con la tua chiave meta personalizzata
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un codice a barre deve essere univoco per ogni prodotto.
:::
