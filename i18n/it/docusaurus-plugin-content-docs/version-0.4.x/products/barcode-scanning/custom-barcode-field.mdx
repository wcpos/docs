---
title: Campo Barcode Personalizzato
---

Per impostazione predefinita, WCPOS utilizzerà il campo SKU come barcode.

![Per impostazione predefinita, WCPOS utilizzerà il campo SKU come barcode. ](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Per impostazione predefinita, WCPOS utilizzerà il campo SKU come barcode")

Gli utenti di [WCPOS Pro](http://wcpos.com/pro) possono utilizzare qualsiasi campo meta prodotto selezionando un Campo Barcode in `WP Admin > POS > Impostazioni > Generale`

![Seleziona campo barcode personalizzato in WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Seleziona campo barcode personalizzato in WCPOS Pro")

I campi meta barcode personalizzati possono anche essere impostati utilizzando il filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un barcode deve essere unico per ogni prodotto.
:::
