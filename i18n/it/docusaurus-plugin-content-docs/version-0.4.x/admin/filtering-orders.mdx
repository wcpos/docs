---
title: Filtrare gli ordini per vendite POS o Online
description: Separare gli ordini POS e Online utilizzando i link rapidi nell'area di amministrazione di WooCommerce.
---

È possibile separare rapidamente gli ordini POS e Online nell'area di amministrazione di WooCommerce utilizzando i link rapidi in alto a sinistra.
**Nota bene:** questo filtro funziona solo per gli ordini POS a partire dalla versione 0.3.1; gli ordini precedenti alla 0.3.1 verranno classificati come Online per impostazione predefinita.

![Filtra ordini](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

Inoltre, se si desidera nascondere tutti gli ordini completati dall'area di amministrazione per concentrarsi sugli ordini in attesa di pagamento o in lavorazione, è possibile aggiungere il seguente codice al file [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
