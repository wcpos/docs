Puoi separare rapidamente i tuoi ordini POS e Online nell'amministrazione di WooCommerce utilizzando i collegamenti rapidi in alto a sinistra.  
**Si prega di notare:** questo filtrer√† solo gli ordini POS dopo la versione 0.3.1, gli ordini precedenti alla 0.3.1 verranno impostati di default a Online.

![Filtra Ordini](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtra Ordini")

Inoltre, se desideri filtrare tutti gli ordini completati dalla tua area di amministrazione in modo da poterti concentrare sugli ordini in sospeso/processing, puoi aggiungere il seguente codice al tuo file [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
