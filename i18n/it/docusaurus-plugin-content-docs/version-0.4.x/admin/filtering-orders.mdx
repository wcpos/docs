Puoi separare rapidamente i tuoi ordini POS e Online nell'amministrazione di WooCommerce utilizzando i link rapidi in alto a sinistra.
**Nota:** questo filtrer√† solo gli ordini POS dalla versione 0.3.1, gli ordini precedenti a 0.3.1 verranno considerati come Online.

![Filtra Ordini](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtra Ordini")

Inoltre, se volessi filtrare tutti gli ordini completati dalla tua area di amministrazione per concentrarti sugli ordini in sospeso/in elaborazione, puoi aggiungere il seguente codice nel tuo file [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Cambia il filtro predefinito degli ordini in WP admin
 * aggiungi questo al tuo file functions.php
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
