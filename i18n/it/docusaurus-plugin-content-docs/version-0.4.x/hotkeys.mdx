Le scorciatoie da tastiera sono combinazioni che possono essere utilizzate per attivare rapidamente azioni in WCPOS.

| Tasto                       | Descrizione       |
| --------------------------- | ---------------- |
| <div align="center">?</div> | Schermata di aiuto |
| <div align="center">B</div> | Modalità codice a barre |
| <div align="center">V</div> | Modalità ricerca   |
| <div align="center">S</div> | Sincronizza con il server |
