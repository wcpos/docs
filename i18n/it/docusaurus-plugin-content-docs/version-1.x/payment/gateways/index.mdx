# Gateway di Pagamento Personalizzati {#custom-payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del tuo sistema di punto vendita. Questi gateway sono progettati specificamente per ambienti POS e offrono un'integrazione senza soluzione di continuità con vari metodi e servizi di pagamento.

## Gateway Personalizzati Disponibili {#available-custom-gateways}

### Gateway per Terminali Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti utilizzando lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora pagamenti tramite lettori di carte SumUp

### Gateway di Pagamento Mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accetta pagamenti tramite telefono tramite codice QR o notifiche push

### Gateway di Pagamento Digitale {#digital-payment-gateways}

- **[Fattura via Email](./email-invoice)** - Invia fatture di pagamento ai clienti via email
- **[Web Checkout](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare il Proprio Gateway {#creating-your-own-gateway}

Se hai bisogno di un gateway di pagamento personalizzato per le tue esigenze specifiche, puoi utilizzare il nostro **[Modello di Gateway](./gateway-template)** per creare il tuo gateway di pagamento WCPOS.

## Panoramica dell'Installazione {#installation-overview}

Tutti i gateway personalizzati seguono un processo di installazione simile:

1. **Scarica** l'ultima versione dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugin > Aggiungi Nuovo > Carica Plugin`
3. **Configura** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Abilita** il gateway in `WP Admin > POS > Impostazioni > Checkout`
