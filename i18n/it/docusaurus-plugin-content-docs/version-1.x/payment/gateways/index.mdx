# Gateway di pagamento {#payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del sistema punto vendita. Questi gateway sono progettati specificamente per gli ambienti POS e offrono un'integrazione fluida con vari metodi e servizi di pagamento.

## Gateway personalizzati disponibili {#available-custom-gateways}

### Gateway per terminali hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti tramite lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora i pagamenti tramite lettori di carte SumUp
- **[Square Terminal](./square-terminal)** - Accetta pagamenti di persona sui dispositivi Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Accetta pagamenti con carta di persona su un terminale PayPal Reader (Zettle)
- **[Mollie Terminal](./mollie-terminal)** - Accetta pagamenti di persona sui dispositivi Mollie Terminal

### Gateway di pagamento mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accetta pagamenti tramite telefono via codice QR o notifica push

### Gateway di pagamento digitali {#digital-payment-gateways}

- **[Fattura via email](./email-invoice)** - Invia fatture di pagamento ai clienti via email
- **[Checkout web](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare un gateway personalizzato {#creating-your-own-gateway}

Se è necessario un gateway di pagamento personalizzato per requisiti specifici, è possibile utilizzare il nostro **[Modello di gateway](/reference/gateway-template)** per creare un gateway di pagamento WCPOS.

## Panoramica dell'installazione {#installation-overview}

Tutti i gateway personalizzati seguono una procedura di installazione simile:

1. **Scaricare** l'ultima versione dal rispettivo repository GitHub
2. **Installare** il plugin tramite `WP Admin > Plugin > Aggiungi Nuovo > Carica plugin`
3. **Configurare** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Attivare** il gateway in `WP Admin > POS > Impostazioni > Pagamento`
