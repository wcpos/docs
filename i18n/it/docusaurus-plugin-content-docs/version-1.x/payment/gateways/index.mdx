# Gateway di pagamento {#payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del sistema Point of Sale. Questi gateway sono progettati specificamente per ambienti POS e offrono un'integrazione perfetta con vari metodi e servizi di pagamento.

## Gateway personalizzati disponibili {#available-custom-gateways}

### Gateway per terminali hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti tramite lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora pagamenti tramite lettori di carte SumUp
- **[Square Terminal](./square-terminal)** - Riscuoti pagamenti di persona sui dispositivi Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Accetta pagamenti con carta di persona tramite un terminale PayPal Reader (Zettle)

### Gateway per pagamenti mobili {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accetta pagamenti tramite telefono con codice QR o notifica push

### Gateway di pagamento digitali {#digital-payment-gateways}

- **[Fattura via email](./email-invoice)** - Invia fatture di pagamento ai clienti tramite email
- **[Checkout web](./web-checkout)** - Reindirizza i clienti per completare il pagamento online

## Creare un gateway personalizzato {#creating-your-own-gateway}

Per esigenze specifiche è possibile utilizzare il **[Modello di gateway](/reference/gateway-template)** per creare un gateway di pagamento WCPOS personalizzato.

## Panoramica dell'installazione {#installation-overview}

Tutti i gateway personalizzati seguono un processo di installazione simile:

1. **Scaricare** l'ultima versione dal rispettivo repository GitHub
2. **Installare** il plugin tramite `WP Admin > Plugin > Aggiungi Nuovo > Carica plugin`
3. **Configurare** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Attivare** il gateway in `WP Admin > POS > Impostazioni > Pagamento`
