# Gateway di pagamento {#payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del sistema POS. Questi gateway sono progettati specificamente per gli ambienti POS e offrono un'integrazione fluida con vari metodi e servizi di pagamento.

## Gateway personalizzati disponibili {#available-custom-gateways}

### Gateway per terminali hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti tramite i lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora i pagamenti tramite lettori di carte SumUp
- **[PayPal Reader (Zettle)](./paypal-reader)** - Accetta pagamenti con carta di persona su un terminale PayPal Reader (Zettle)

### Gateway di pagamento mobile {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accetta pagamenti da telefono tramite codice QR o notifica push

### Gateway di pagamento digitali {#digital-payment-gateways}

- **[Fattura via email](./email-invoice)** - Invia fatture di pagamento ai clienti via email
- **[Checkout web](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare il proprio gateway {#creating-your-own-gateway}

Se serve un gateway di pagamento personalizzato per requisiti specifici, è possibile usare il nostro **[Modello di gateway](/reference/gateway-template)** per creare il proprio gateway di pagamento WCPOS.

## Panoramica dell'installazione {#installation-overview}

Tutti i gateway personalizzati seguono una procedura di installazione simile:

1. **Scarica** l'ultima release dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugin > Aggiungi Nuovo > Carica plugin`
3. **Configura** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Abilita** il gateway in `WP Admin > POS > Impostazioni > Pagamento`
