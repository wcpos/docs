WCPOS supporta molteplici metodi di pagamento per soddisfare diverse esigenze aziendali.

## Metodi di Pagamento Predefiniti

La versione gratuita include due gateway di pagamento integrati:

### Contanti

- Inserire l'importo versato
- Calcolo automatico del resto
- Nessuna configurazione aggiuntiva richiesta

### Carta

- Per l'uso con terminali di pagamento esterni
- Semplicemente contrassegnare il pagamento come completato dopo l'elaborazione sul proprio terminale
- Nessuna integrazione diretta richiesta

## Gateway di Pagamento Aggiuntivi (Pro)

:::info Funzionalità Pro
I gateway di pagamento aggiuntivi richiedono [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, è possibile abilitare gateway di pagamento WooCommerce aggiuntivi durante il checkout POS:

- **Stripe Terminal** - Integrazione diretta con lettori di carte Stripe
- **SumUp Terminal** - Integrazione con lettori di carte SumUp
- **Qualsiasi Gateway WooCommerce** - Abilitare qualsiasi gateway installato nel proprio negozio

## Gateway Personalizzati

Crea integrazioni di pagamento personalizzate utilizzando il [sistema di Gateway Personalizzati](/payment/custom-gateways).
