## Metodi di Pagamento Predefiniti

La versione gratuita include due gateway di pagamento integrati:

### Contante

- Inserisci l'importo pagato
- Calcolo automatico del resto
- Nessuna configurazione aggiuntiva richiesta

### Carta

- Da utilizzare con terminali di pagamento esterni
- Basta contrassegnare il pagamento come completato dopo l'elaborazione sul tuo terminale
- Nessuna integrazione diretta necessaria

## Gateway di Pagamento Aggiuntivi (Pro)

:::info Funzione Pro
I gateway di pagamento aggiuntivi richiedono [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puoi abilitare ulteriori gateway di pagamento WooCommerce nel checkout POS:

- **Stripe Terminal** - Integrazione diretta con lettori di carte Stripe
- **SumUp Terminal** - Integrazione con lettori di carte SumUp
- **Qualsiasi Gateway WooCommerce** - Abilita qualsiasi gateway installato nel tuo negozio

## Gateway Personalizzati

Crea le tue integrazioni di pagamento utilizzando il sistema [Gateway Personalizzati](/payment/custom-gateways).
