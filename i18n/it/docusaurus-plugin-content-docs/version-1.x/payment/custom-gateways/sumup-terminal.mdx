# Gateway Terminal SumUp {#sumup-terminal-gateway}

Il gateway Terminal SumUp ti consente di accettare pagamenti con carta utilizzando i lettori di carte SumUp direttamente all'interno di WCPOS. Questo gateway offre un'integrazione fluida con il sistema di elaborazione dei pagamenti di SumUp per transazioni di persona.

## Installazione {#installation}

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina dei rilasci di SumUp Terminal per WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Scarica l'ultimo file **sumup-terminal-for-woocommerce.zip**
3. Nel tuo admin di WordPress, vai su `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Configura le Impostazioni di SumUp">

1. Naviga su `WP Admin > WooCommerce > Impostazioni > Pagamenti`
2. Trova **Terminal SumUp** nell'elenco dei metodi di pagamento
3. Clicca su **Terminal SumUp** per accedere alle impostazioni
4. Inserisci la tua **Chiave API di SumUp** (disponibile nel tuo cruscotto commerciante SumUp)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway Terminal SumUp nelle impostazioni di WooCommerce. Sarà abilitato specificamente per il POS in un passaggio successivo.
:::

  </StepCard>
  <StepCard number="3" title="Abbina il Tuo Terminal SumUp">

1. Nella stessa pagina delle impostazioni, trova la sezione **Abbina Lettore**
2. Sul tuo dispositivo SumUp, naviga alla schermata di abbinamento per visualizzare il codice di abbinamento
3. Inserisci il codice di abbinamento visualizzato sul tuo dispositivo SumUp
4. Clicca su **"Abbina Lettore"** per stabilire la connessione
5. Aspetta la conferma che il lettore è stato abbinato con successo

:::warning Importante
Il lettore deve essere abbinato con successo prima di poter elaborare i pagamenti. Assicurati che il processo di abbinamento sia completato prima di procedere.
:::

  </StepCard>
  <StepCard number="4" title="Abilita in WCPOS" isLast>

1. Vai su `WP Admin > POS > Impostazioni > Checkout`
2. Trova il gateway **Terminal SumUp** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

  </StepCard>
</Steps>

## Utilizzo {#usage}

### Elaborazione dei Pagamenti {#processing-payments}

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Terminal SumUp" come metodo di pagamento
3. **Inizia Pagamento**: Clicca per avviare un nuovo pagamento sul tuo dispositivo SumUp
4. **Pagamento del Cliente**: Il cliente completa il pagamento sul terminal SumUp
5. **Completamento Automatico**: L'ordine si completa automaticamente quando il pagamento è rilevato

### Controlli di Pagamento {#payment-controls}

Quando usi il gateway Terminal SumUp, hai le seguenti opzioni:

- **Inizia Nuovo Pagamento**: Inizia una richiesta di pagamento sul terminal collegato
- **Annulla Pagamento**: Annulla un pagamento attualmente in elaborazione
- **Stato del Pagamento**: Monitora lo stato attuale del processo di pagamento

### Gestione degli Ordini {#order-management}

- **Aggiornamenti Automatici**: Gli ordini vengono contrassegnati automaticamente come completati dopo un pagamento riuscito
- **Tracciamento dei Pagamenti**: Tutti i dettagli del pagamento sono registrati nelle note dell'ordine
- **Generazione di Ricevute**: Le ricevute standard del POS vengono generate dopo i pagamenti riusciti

## Compatibilità Hardware {#hardware-compatibility}

:::important Requisiti di Connettività
Questa implementazione utilizza il SDK JavaScript di SumUp, il che significa che funziona tramite applicazioni web ma richiede **terminali connessi a Internet soltanto**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - Terminale connesso a WiFi/Ethernet

### Terminali Non Supportati (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - Solo Bluetooth
- **SumUp 3G** ❌ - Solo Bluetooth

:::note Supporto Futura
Il supporto per i terminali Bluetooth sarà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo abiliterà il supporto per tutti i terminali certificati SumUp, inclusi i modelli Air e 3G.
:::

## Risoluzione dei Problemi {#troubleshooting}

### Problemi Comuni {#common-issues}

<Accordion>
  <AccordionItem question="Il lettore non si abbina">

  - Assicurati di utilizzare un terminale connesso a Internet supportato (Solo)
  - Verifica che il tuo dispositivo SumUp Solo sia connesso a WiFi e online
  - Controlla che il dispositivo sia in modalità di abbinamento e che il codice di abbinamento sia inserito correttamente
  - Conferma che il tuo account SumUp abbia abilitato l'accesso API
  - Assicurati di avere una connessione internet stabile durante l'abbinamento

  </AccordionItem>

  <AccordionItem question="Pagamento non elaborato">

  - Verifica che il lettore sia abbinato correttamente
  - Controlla che il tuo account SumUp sia attivo e in regola
  - Assicurati che il dispositivo abbia sufficiente carica e connettività
  - Conferma che la chiave API sia corretta e attiva

  </AccordionItem>

  <AccordionItem question="Ordine non completato">

  - Controlla la stabilità della connessione internet
  - Verifica le impostazioni del webhook nel tuo account SumUp
  - Assicurati che il pagamento sia stato completato con successo sul terminal SumUp
  - Controlla i log di errore di WordPress per problemi di comunicazione API

  </AccordionItem>
</Accordion>

### Limiti di Tasso API {#api-rate-limits}

Il gateway è ottimizzato per ridurre le chiamate API e evitare limitazioni di frequenza:
- Lo stato del pagamento viene controllato in modo efficiente
- Le richieste API non necessarie vengono minimize
- La logica di tentativo automatico gestisce i guasti temporanei
