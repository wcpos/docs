# Gateway Terminale SumUp

Il gateway SumUp Terminal consente di accettare pagamenti con carte utilizzando lettori di carte SumUp direttamente all'interno di WooCommerce POS. Questo gateway offre un'integrazione semplice con il sistema di elaborazione dei pagamenti di SumUp per transazioni in negozio.

## Funzionalità

- **Integrazione Hardware**: Connetti lettori di carte SumUp al tuo sistema POS tramite connessione internet
- **Elaborazione in Tempo Reale**: Elaborazione istantanea dei pagamenti e completamento degli ordini
- **Rilevamento Automatico**: Ordini completati automaticamente quando il pagamento viene rilevato
- **Transazioni Sicure**: Elaborazione dei pagamenti conforme PCI tramite SumUp
- **Abbinamento Semplice**: Processo di abbinamento del dispositivo semplice con codici di abbinamento
- **SDK Web**: Utilizza l'SDK JavaScript di SumUp per la compatibilità con applicazioni web

## Installazione

### Passaggio 1: Scarica e Installa

1. Visita la [pagina delle release di SumUp Terminal per WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Scarica l'ultimo file **sumup-terminal-for-woocommerce.zip**
3. Nell'amministrazione WordPress, vai su `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

### Passaggio 2: Configura Impostazioni SumUp

1. Vai su `Amministrazione WP > WooCommerce > Impostazioni > Pagamenti`
2. Trova **SumUp Terminal** nell'elenco dei metodi di pagamento
3. Fai clic su **SumUp Terminal** per accedere alle impostazioni
4. Inserisci la tua **Chiave API SumUp** (disponibile nel dashboard del tuo account SumUp)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway SumUp Terminal nelle impostazioni di WooCommerce. Verrà abilitato specificamente per il POS in un passaggio successivo.
:::

### Passaggio 3: Abbina il Tuo Terminale SumUp

1. Nella stessa pagina delle impostazioni, individua la sezione **Abbina Lettore**
2. Sul tuo dispositivo SumUp, vai alla schermata di abbinamento per visualizzare il codice di abbinamento
3. Inserisci il codice di abbinamento visualizzato sul tuo dispositivo SumUp
4. Fai clic su **"Abbina Lettore"** per stabilire la connessione
5. Attendi la conferma che il lettore è stato abbinato con successo

:::warning Importante
Il lettore deve essere abbinato correttamente prima di poter elaborare i pagamenti. Assicurati di completare il processo di abbinamento prima di procedere.
:::

### Passaggio 4: Abilita in WooCommerce POS

1. Vai su `Amministrazione WP > WooCommerce POS > Impostazioni > Checkout`
2. Trova il gateway **SumUp Terminal** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

## Utilizzo

### Elaborazione Pagamenti

1. **Aggiungi Articoli**: Aggiungi prodotti al carrello nel POS
2. **Seleziona Gateway**: Scegli "SumUp Terminal" come metodo di pagamento
3. **Avvia Pagamento**: Fai clic per avviare un nuovo pagamento sul tuo dispositivo SumUp
4. **Pagamento Cliente**: Il cliente completa il pagamento sul terminale SumUp
5. **Completamento Automatico**: L'ordine viene completato automaticamente quando il pagamento viene rilevato

### Controlli Pagamento

Utilizzando il gateway SumUp Terminal, hai le seguenti opzioni:

- **Avvia Nuovo Pagamento**: Avvia una richiesta di pagamento sul terminale connesso
- **Annulla Pagamento**: Annulla un pagamento in corso
- **Stato Pagamento**: Monitora lo stato corrente del processo di pagamento

### Gestione Ordini

- **Aggiornamenti Automatici**: Gli ordini vengono contrassegnati come completati al pagamento avvenuto con successo
- **Tracciamento Pagamenti**: Tutti i dettagli del pagamento vengono registrati nelle note dell'ordine
- **Generazione Ricevuta**: Vengono generate ricevute POS standard dopo pagamenti riusciti

## Requisiti

- **Account SumUp**: Account commerciante SumUp attivo
- **Accesso API**: Chiave API SumUp dal tuo dashboard commerciante
- **Hardware Compatibile**: Dispositivo lettore di carte SumUp
- **WooCommerce POS**: Versione Free o Pro installata e attivata
- **Connessione Stabile**: Connessione internet affidabile per comunicazione API

## Compatibilità Hardware

:::important Requisiti di Connettività
Questa implementazione utilizza l'SDK JavaScript di SumUp, il che significa che funziona tramite applicazioni web ma richiede **terminali connessi a internet**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet)

- **SumUp Solo** ✅ - Terminale connesso WiFi/Ethernet

### Terminali Non Supportati (Bluetooth)

- **SumUp Air** ❌ - Solo Bluetooth
- **SumUp 3G** ❌ - Solo Bluetooth

:::note Supporto Futuro
Il supporto per terminali Bluetooth verrà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo consentirà il supporto per tutti i terminali certificati SumUp, inclusi i modelli Air e 3G.
:::

## Risoluzione dei Problemi

### Problemi Comuni

**Il Lettore Non Si Abbina**
- Assicurati di utilizzare un terminale connesso a internet supportato (solo Solo)
- Verifica che il tuo dispositivo SumUp Solo sia connesso al WiFi e online
- Controlla che il dispositivo sia in modalità abbinamento e il codice di abbinamento sia inserito correttamente
- Conferma che il tuo account SumUp abbia l'accesso API abilitato
- Assicura una connessione internet stabile durante l'abbinamento

**Pagamento Non In Elaborazione**
- Verifica che il lettore sia correttamente abbinato
- Controlla che il tuo account SumUp sia attivo e in regola
- Assicurati che il dispositivo abbia batteria sufficiente e connettività
- Conferma che la chiave API sia corretta e attiva

**Ordine Non Completato**
- Controlla la stabilità della connessione internet
- Verifica le impostazioni webhook nel tuo account SumUp
- Assicurati che il pagamento sia avvenuto con successo sul terminale SumUp
- Controlla i registri degli errori di WordPress per problemi di comunicazione API

### Limiti di Utilizzo API

Il gateway è ottimizzato per ridurre le chiamate API ed evitare limiti di utilizzo:
- Lo stato del pagamento viene verificato in modo efficiente
- Le richieste API non necessarie vengono minimizzate
- La logica di riprova automatica gestisce guasti temporanei
