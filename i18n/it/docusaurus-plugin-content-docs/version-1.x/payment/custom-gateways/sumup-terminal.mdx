# Gateway Terminal SumUp

Il gateway Terminal SumUp ti consente di accettare pagamenti con carta utilizzando i lettori di carte SumUp direttamente all'interno di WooCommerce POS. Questo gateway offre un'integrazione fluida con il sistema di elaborazione dei pagamenti di SumUp per le transazioni in persona.

## Installazione

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina dei rilasci di SumUp Terminal per WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Scarica il file **sumup-terminal-for-woocommerce.zip** più recente
3. Nel tuo admin di WordPress, vai a `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Configura le Impostazioni di SumUp">

1. Naviga a `WP Admin > WooCommerce > Impostazioni > Pagamenti`
2. Trova **SumUp Terminal** nell'elenco dei metodi di pagamento
3. Clicca su **SumUp Terminal** per accedere alle impostazioni
4. Inserisci la tua **Chiave API di SumUp** (disponibile dal dashboard merchant di SumUp)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway Terminal SumUp nelle impostazioni di WooCommerce. Sarà abilitato specificamente per il POS in un passaggio successivo.
:::

  </StepCard>
  <StepCard number="3" title="Accoppia il Tuo Terminal SumUp">

1. Nella stessa pagina delle impostazioni, individua la sezione **Accoppia Lettore**
2. Sul tuo dispositivo SumUp, naviga nella schermata di accoppiamento per visualizzare il codice di accoppiamento
3. Inserisci il codice di accoppiamento visualizzato sul tuo dispositivo SumUp
4. Clicca **"Accoppia Lettore"** per stabilire la connessione
5. Attendi la conferma che il lettore è stato accoppiato con successo

:::warning Importante
Il lettore deve essere accoppiato con successo prima di poter elaborare i pagamenti. Assicurati che il processo di accoppiamento sia completato prima di procedere.
:::

  </StepCard>
  <StepCard number="4" title="Abilita in WooCommerce POS" isLast>

1. Vai a `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il gateway **SumUp Terminal** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

  </StepCard>
</Steps>

## Utilizzo

### Elaborazione dei Pagamenti

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "SumUp Terminal" come metodo di pagamento
3. **Inizia Pagamento**: Clicca per avviare un nuovo pagamento sul tuo dispositivo SumUp
4. **Pagamento del Cliente**: Il cliente completa il pagamento sul terminale SumUp
5. **Completamento Automatico**: L'ordine si completa automaticamente al rilevamento del pagamento

### Controlli di Pagamento

Quando utilizzi il gateway Terminal SumUp, hai le seguenti opzioni:

- **Inizia Nuovo Pagamento**: Avvia una richiesta di pagamento sul terminale connesso
- **Annulla Pagamento**: Annulla un pagamento attualmente in corso
- **Stato del Pagamento**: Monitora lo stato attuale del processo di pagamento

### Gestione degli Ordini

- **Aggiornamenti Automatici**: Gli ordini vengono automaticamente contrassegnati come completati al pagamento riuscito
- **Tracciamento dei Pagamenti**: Tutti i dettagli di pagamento vengono registrati nelle note dell'ordine
- **Generazione di Ricevute**: Ricevute standard del POS vengono generate dopo pagamenti riusciti

## Compatibilità Hardware

:::important Requisiti di Connettività
Questa implementazione utilizza l'SDK JavaScript di SumUp, il che significa che funziona tramite applicazioni web ma richiede **terminali solo con connessione internet**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet)

- **SumUp Solo** ✅ - Terminale connesso via WiFi/Ethernet

### Terminali Non Supportati (Bluetooth)

- **SumUp Air** ❌ - Solo Bluetooth
- **SumUp 3G** ❌ - Solo Bluetooth

:::note Supporto Futuro
Il supporto per terminali Bluetooth sarà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo abiliterà il supporto per tutti i terminali certificati SumUp, inclusi i modelli Air e 3G.
:::

## Risoluzione dei Problemi

### Problemi Comuni

<Accordion>
  <AccordionItem question="Il lettore non si accoppia">

  - Assicurati di utilizzare un terminale connesso a internet supportato (Solo)
  - Verifica che il tuo dispositivo SumUp Solo sia connesso a WiFi e online
  - Controlla che il dispositivo sia in modalità accoppiamento e che il codice di accoppiamento sia inserito correttamente
  - Conferma che il tuo account SumUp abbia l'accesso API attivato
  - Assicurati di avere una connessione internet stabile durante l'accoppiamento

  </AccordionItem>

  <AccordionItem question="Pagamento non elaborato">

  - Verifica che il lettore sia correttamente accoppiato
  - Controlla che il tuo account SumUp sia attivo e in buona posizione
  - Assicurati che il dispositivo abbia sufficiente batteria e connettività
  - Conferma che la chiave API sia corretta e attiva

  </AccordionItem>

  <AccordionItem question="Ordine non completato">

  - Controlla la stabilità della connessione internet
  - Verifica le impostazioni del webhook nel tuo account SumUp
  - Assicurati che il pagamento sia stato completato con successo sul terminale SumUp
  - Controlla i registri di errore di WordPress per problemi di comunicazione API

  </AccordionItem>
</Accordion>

### Limiti di Rate API

Il gateway è ottimizzato per ridurre le chiamate API e evitare il rate limiting:
- Lo stato del pagamento viene controllato in modo efficiente
- Le richieste API non necessarie vengono minimizzate
- La logica di ripetizione automatica gestisce i fallimenti temporanei
