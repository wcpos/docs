# Gateway Terminal SumUp {#sumup-terminal-gateway}

Il gateway Terminal SumUp ti consente di accettare pagamenti con carta utilizzando i lettori di carte SumUp direttamente all'interno di WooCommerce POS. Questo gateway offre un'integrazione senza soluzione di continuità con il sistema di elaborazione pagamenti di SumUp per le transazioni di persona.

## Installazione {#installation}

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina dei rilasci di SumUp Terminal per WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Scarica l'ultima versione del file **sumup-terminal-for-woocommerce.zip**
3. Nel tuo admin di WordPress, vai su `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Configura le Impostazioni di SumUp">

1. Naviga su `WP Admin > WooCommerce > Impostazioni > Pagamenti`
2. Trova **Terminal SumUp** nella lista dei metodi di pagamento
3. Clicca su **Terminal SumUp** per accedere alle impostazioni
4. Inserisci la tua **API Key di SumUp** (disponibile dal tuo dashboard commerciante SumUp)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway Terminal SumUp nelle impostazioni di WooCommerce. Sarà abilitato specificamente per il POS in un passaggio successivo.
:::

  </StepCard>
  <StepCard number="3" title="Associa il Tuo Terminal SumUp">

1. Nella stessa pagina delle impostazioni, trova la sezione **Associa Lettore**
2. Sul tuo dispositivo SumUp, naviga sulla schermata di associazione per visualizzare il codice di associazione
3. Inserisci il codice di associazione visualizzato sul tuo dispositivo SumUp
4. Clicca su **"Associa Lettore"** per stabilire la connessione
5. Attendi la conferma che il lettore è stato associato con successo

:::warning Importante
Il lettore deve essere associato con successo prima di poter elaborare i pagamenti. Assicurati che il processo di associazione sia completato prima di procedere.
:::

  </StepCard>
  <StepCard number="4" title="Abilita in WooCommerce POS" isLast>

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il gateway **Terminal SumUp** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

  </StepCard>
</Steps>

## Utilizzo {#usage}

### Elaborazione Pagamenti {#processing-payments}

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Terminal SumUp" come metodo di pagamento
3. **Inizia Pagamento**: Clicca per avviare un nuovo pagamento sul tuo dispositivo SumUp
4. **Pagamento Cliente**: Il cliente completa il pagamento sul terminal SumUp
5. **Completamento Automatico**: L'ordine si completa automaticamente quando il pagamento viene rilevato

### Controlli di Pagamento {#payment-controls}

Quando utilizzi il gateway Terminal SumUp, hai le seguenti opzioni:

- **Inizia Nuovo Pagamento**: Iniziare una richiesta di pagamento sul terminal collegato
- **Annulla Pagamento**: Annulla un pagamento attualmente in corso
- **Stato del Pagamento**: Monitora lo stato attuale del processo di pagamento

### Gestione Ordini {#order-management}

- **Aggiornamenti Automatici**: Gli ordini sono automaticamente contrassegnati come completati dopo il pagamento riuscito
- **Monitoraggio Pagamenti**: Tutti i dettagli di pagamento sono registrati nelle note dell'ordine
- **Generazione Ricevute**: Le ricevute standard POS vengono generate dopo i pagamenti riusciti

## Compatibilità Hardware {#hardware-compatibility}

:::important Requisiti di Connettività
Questa implementazione utilizza il SDK JavaScript di SumUp, il che significa che funziona tramite applicazioni web ma richiede **terminale connessi a internet solamente**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - terminale connesso via WiFi/Ethernet

### Terminali Non Supportati (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - solo Bluetooth
- **SumUp 3G** ❌ - solo Bluetooth

:::note Supporto Futuro
Il supporto dei terminali Bluetooth sarà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Ciò consentirà il supporto per tutti i terminali certificati SumUp, compresi i modelli Air e 3G.
:::

## Risoluzione dei Problemi {#troubleshooting}

### Problemi Comuni {#common-issues}

<Accordion>
  <AccordionItem question="Il lettore non si associa">

  - Assicurati di utilizzare un terminale connesso a internet supportato (Solo)
  - Verifica che il tuo dispositivo SumUp Solo sia connesso a WiFi e online
  - Controlla che il dispositivo sia in modalità di associazione e che il codice di associazione sia inserito correttamente
  - Conferma che il tuo account SumUp abbia accesso API abilitato
  - Assicurati di una connessione internet stabile durante l'associazione

  </AccordionItem>

  <AccordionItem question="Pagamento non elaborato">

  - Verifica che il lettore sia correttamente associato
  - Controlla che il tuo account SumUp sia attivo e in regola
  - Assicurati che il dispositivo abbia batteria e connettività sufficienti
  - Conferma che l'API key sia corretta e attiva

  </AccordionItem>

  <AccordionItem question="Ordine non completato">

  - Controlla la stabilità della connessione internet
  - Verifica le impostazioni webhook nel tuo account SumUp
  - Assicurati che il pagamento sia avvenuto con successo sul terminal SumUp
  - Controlla i log degli errori di WordPress per problemi di comunicazione API

  </AccordionItem>
</Accordion>

### Limiti di Frequenza API {#api-rate-limits}

Il gateway è ottimizzato per ridurre le chiamate API e evitare limiti di frequenza:
- Lo stato del pagamento viene controllato in modo efficiente
- Le richieste API non necessarie vengono ridotte al minimo
- La logica di ripetizione automatica gestisce i fallimenti temporanei
