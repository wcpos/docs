# Gateway Fattura via Email

Il gateway per fatture via email ti permette di inviare fatture di pagamento ai clienti direttamente dall'interfaccia WooCommerce POS. Ideale per situazioni in cui i clienti preferiscono pagare successivamente o devono completare il pagamento online utilizzando i metodi di pagamento del tuo negozio web.

## Funzionalità

- **Integrazione Email**: Invia fatture direttamente dall'interfaccia POS
- **Gestione Clienti**: Popola automaticamente l'email dai dati del cliente esistente o aggiungi nuovi indirizzi email
- **Ordini Guest**: Supporto per clienti guest con possibilità di aggiornare l'indirizzo di fatturazione
- **Pagamento Online**: I clienti ricevono un link per "Pagare questo ordine" per completare il pagamento online
- **Flusso di Lavoro Flessibile**: Completa gli ordini in negozio o consenti il completamento del pagamento da remoto

## Installazione

### Passaggio 1: Scarica e Installa

1. Visita la [pagina delle release del Gateway Fattura via Email](https://github.com/wcpos/email-invoice-gateway/releases)
2. Scarica l'ultimo file **woocommerce-pos-email-invoice-gateway.zip**
3. Nell'admin di WordPress, vai su `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

### Passaggio 2: Abilita in WooCommerce POS

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il **Gateway di Pagamento Invoice di WooCommerce POS** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

:::note
Questo gateway non richiede chiavi API aggiuntive o configurazioni di servizi esterni. Utilizza il sistema email WordPress esistente e i metodi di pagamento WooCommerce.
:::

## Utilizzo

### Invio Fatture

1. **Aggiungi Articoli**: Aggiungi prodotti al carrello nel POS
2. **Seleziona Gateway**: Scegli "Fattura via Email" come metodo di pagamento
3. **Indirizzo Email**: 
   - Per clienti esistenti: Email popolata automaticamente dai dati del cliente
   - Per ordini guest: Inserisci manualmente l'indirizzo email del cliente
4. **Opzionale**: Spunta "Salva nell'indirizzo di fatturazione" per aggiornare le informazioni di fatturazione del cliente
5. **Invia Fattura**: Completa il processo per inviare l'email della fattura

### Esperienza Cliente

Quando invii una fattura, il cliente riceve un'email contenente:

- **Dettagli Ordine**: Descrizione completa degli articoli, quantità e prezzi
- **Totale Ordine**: Importo finale dovuto inclusi tasse e commissioni
- **Link di Pagamento**: Link diretto per completare il pagamento online
- **Informazioni Ordine**: Numero ordine e dettagli di riferimento

### Completamento Pagamento

I clienti possono completare il pagamento in due modi:

1. **Pagamento Online**: Fare clic sul link "Paga questo ordine" nell'email
2. **Ritorno in Negozio**: Ritornare per completare il pagamento con altri metodi di pagamento POS

Il link "Paga questo ordine" porta i clienti a una pagina di pagamento dedicata sul tuo sito web dove possono utilizzare tutti i metodi di pagamento web abilitati (Stripe, PayPal, ecc.).
