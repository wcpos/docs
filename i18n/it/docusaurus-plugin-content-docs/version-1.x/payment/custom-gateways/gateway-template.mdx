# Modello di Gateway

Il Modello di Gateway WooCommerce fornisce un punto di partenza per creare gateway di pagamento personalizzati per WooCommerce POS. Questo modello include tutto il codice boilerplate e la struttura necessari per costruire un gateway di pagamento completamente funzionale.

## Funzionalità

- **Modello Completo**: Struttura del gateway pronta all'uso con tutti i metodi richiesti
- **Integrazione POS**: Pre-configurato per la compatibilità con WooCommerce POS
- **Configurazione Automatizzata**: Personalizzazione del modello basata su script
- **Migliori Pratiche**: Segue gli standard di codifica di WordPress e WooCommerce
- **Estensibile**: Facile da modificare ed estendere per specifici provider di pagamento

## Inizia

### Opzione 1: Generazione Automatica del Modello

Il modello include uno script che personalizza automaticamente il modello per il tuo gateway specifico:

1. **Clona il Repository**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **Esegui lo Script di Configurazione**:
   ```bash
   ./create-gateway.sh
   ```

3. **Segui le Istruzioni**:
   - Inserisci il nome del gateway (es. "Mio Gateway di Pagamento")
   - Inserisci uno slug del gateway (es. "mio-pagamento")
   - Fornisci una descrizione
   - Lo script genererà un plugin personalizzato

### Opzione 2: Utilizzo Manuale del Modello

Se preferisci la personalizzazione manuale:

1. **Scarica il Modello**:
   - Visita il [repository del Gateway Template](https://github.com/wcpos/woocommerce-gateway-template)
   - Scarica l'ultima release o clona il repository

2. **Personalizza il Modello**:
   - Sostituisci `{{GATEWAY_NAME}}` con il nome visualizzato del tuo gateway
   - Sostituisci `{{GATEWAY_SLUG}}` con l'identificatore unico del gateway
   - Sostituisci `{{GATEWAY_DESCRIPTION}}` con la descrizione del gateway

3. **Rinomina i File**:
   - Rinomina `wcpos-{{GATEWAY_SLUG}}.php` per adattarlo allo slug del gateway
   - Aggiorna le intestazioni dei file e le informazioni del plugin
