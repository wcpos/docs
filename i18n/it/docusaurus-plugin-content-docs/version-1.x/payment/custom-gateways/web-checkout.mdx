# Gateway di Web Checkout

Il gateway di Web Checkout consente ai clienti di completare le loro transazioni POS utilizzando il sistema di checkout del tuo negozio web. Questo gateway crea un ponte senza soluzione di continuità tra il tuo POS in negozio e i metodi di pagamento online, perfetto per situazioni in cui i clienti preferiscono pagare utilizzando i propri dispositivi o quando hai bisogno di accedere a metodi di pagamento disponibili solo online.

## Installazione

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina delle versioni del Gateway di Web Checkout](https://github.com/wcpos/web-checkout-gateway/releases)
2. Scarica l'ultimo file **woocommerce-pos-web-checkout-gateway.zip**
3. Nel tuo admin di WordPress, vai su `Plugins > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Abilita in WooCommerce POS" isLast>

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il **WooCommerce POS Web Checkout Gateway** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

:::note
Questo gateway sfrutta i tuoi metodi di pagamento WooCommerce esistenti, quindi assicurati di avere almeno un gateway di pagamento web configurato correttamente (Stripe, PayPal, ecc.).
:::

  </StepCard>
</Steps>

## Utilizzo

### Iniziare il Web Checkout

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Web Checkout" come metodo di pagamento
3. **Genera Link**: Il sistema crea un link unico di checkout per l'ordine
4. **Accesso Cliente**: Il cliente fa clic sul link per accedere al checkout web
5. **Pagamento Online**: Il cliente completa il pagamento utilizzando i metodi di pagamento del negozio web

### Processo di Pagamento

Il Gateway di Web Checkout segue questo flusso di lavoro:

1. **Creazione dell'Ordine**: Il POS crea un ordine in sospeso in WooCommerce
2. **Link di Checkout**: Il sistema genera un link di pagamento sicuro e unico
3. **Reindirizzamento del Cliente**: Il cliente viene indirizzato alla pagina di checkout web
4. **Selezione del Pagamento**: Il cliente sceglie tra i metodi di pagamento web disponibili
5. **Elaborazione del Pagamento**: Il processo di checkout standard di WooCommerce gestisce il pagamento
6. **Completamento dell'Ordine**: Lo stato dell'ordine si aggiorna automaticamente al termine del pagamento

### Flusso di Lavoro POS

Dopo aver avviato il Web Checkout:

1. **Monitora Stato**: Tieni aperta la schermata dell'ordine POS per monitorare lo stato del pagamento
2. **Pagamento del Cliente**: Il cliente completa il pagamento sul proprio dispositivo o sul tuo tablet
3. **Aggiornamento Automatico**: Lo stato dell'ordine si aggiorna in tempo reale quando il pagamento viene ricevuto
4. **Elaborazione del Pagamento**: Fai clic sul pulsante "Elabora Pagamento" nel POS per continuare alla ricevuta
5. **Generazione della Ricevuta**: Genera e stampa la ricevuta come di consueto

## Dettagli Tecnici

### Gestione degli Ordini

- **Ordini Pendenti**: Gli ordini vengono creati con stato "pagamento in sospeso"
- **Aggiornamenti di Stato**: Cambiamenti di stato automatici quando il pagamento viene ricevuto
- **Note sull'Ordine**: I dettagli di pagamento vengono registrati nelle note dell'ordine
- **Inventario**: Lo stock viene riservato durante il processo di checkout

### Sicurezza

- **Link Unici**: Ogni sessione di checkout ha un URL unico e limitato nel tempo
- **SSL Richiesto**: Tutta l'elaborazione dei pagamenti avviene su connessioni HTTPS sicure
- **Conformità PCI**: Sfrutta i tuoi gateway di pagamento web già conformi PCI
- **Gestione della Sessione**: La gestione sicura delle sessioni previene accessi non autorizzati

### Metodi di Pagamento

Il gateway fornisce accesso a tutti i tuoi metodi di pagamento WooCommerce configurati:

- **Carte di Credito/Debito**: Stripe, Square, Authorize.net, ecc.
- **Portafogli Digitali**: PayPal, Apple Pay, Google Pay
- **Bonifici Bancari**: Metodi di pagamento bancario diretti
- **Compra Ora, Paga Dopo**: Klarna, Afterpay, Sezzle
- **Criptovaluta**: Bitcoin e altri gateway di pagamento crypto
- **Metodi Locali**: Metodi di pagamento specifici per regione

## Risoluzione dei Problemi

### Problemi Comuni

<Accordion>
  <AccordionItem question="Il link di checkout non funziona">

  - Verifica che il certificato SSL sia installato correttamente
  - Controlla che i permalink di WooCommerce siano configurati correttamente
  - Assicurati che i gateway di pagamento web siano attivi e configurati
  - Prova con un browser o un dispositivo diverso

  </AccordionItem>

  <AccordionItem question="Il pagamento non viene elaborato">

  - Conferma che i gateway di pagamento web siano configurati correttamente
  - Controlla i log del gateway di pagamento per messaggi di errore
  - Verifica che il metodo di pagamento del cliente sia supportato
  - Verifica il processo di checkout web in modo indipendente

  </AccordionItem>

  <AccordionItem question="L'ordine non si aggiorna nel POS">

  - Controlla la stabilità della connessione Internet
  - Verifica che i cron job di WordPress siano in esecuzione correttamente
  - Aggiorna la schermata dell'ordine POS
  - Controlla lo stato dell'ordine nell'amministrazione di WooCommerce

  </AccordionItem>

  <AccordionItem question="Il cliente non riesce ad accedere al checkout">

  - Assicurati che il link di checkout non sia scaduto
  - Verifica che il dispositivo del cliente abbia connettività a Internet
  - Controlla che l'ordine non sia stato annullato o completato
  - Prova il link su un dispositivo diverso

  </AccordionItem>
</Accordion>

### Ottimizzazione delle Prestazioni

Per prestazioni ottimali:

- **Caching**: Configura i plugin di caching per escludere le pagine di checkout
- **CDN**: Assicurati che le impostazioni CDN non interferiscano con il processo di checkout
- **Database**: Ottimizza il database per una più rapida elaborazione degli ordini
- **Hosting**: Utilizza un hosting affidabile con buona disponibilità
