# Gateway Pagamento Web

Il gateway di Pagamento Web consente ai clienti di completare le loro transazioni POS utilizzando il sistema di checkout del proprio negozio online. Questo gateway crea un ponte trasparente tra il POS del punto vendita e i metodi di pagamento online, perfetto per situazioni in cui i clienti preferiscono pagare utilizzando i propri dispositivi o quando è necessario accedere a metodi di pagamento disponibili solo online.

## Funzionalità

- **Integrazione Trasparente**: Collegamento diretto dal POS al checkout web
- **Tutti i Metodi di Pagamento**: Accesso a tutti i gateway di pagamento abilitati nel negozio web
- **Controllo del Cliente**: I clienti possono pagare utilizzando i propri dispositivi e metodi preferiti
- **Sincronizzazione Ordini**: Aggiornamenti automatici dell'ordine tra POS e negozio web
- **Flusso di Lavoro Flessibile**: Completare transazioni online mantenendo il flusso di lavoro del POS

## Installazione

### Passaggio 1: Scarica e Installa

1. Visita la [pagina delle versioni del Gateway Web Checkout](https://github.com/wcpos/web-checkout-gateway/releases)
2. Scarica l'ultimo file **woocommerce-pos-web-checkout-gateway.zip**
3. Nell'amministrazione di WordPress, vai su `Plugins > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

### Passaggio 2: Attiva in WooCommerce POS

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il **Gateway Web Checkout di WooCommerce POS** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

:::note
Questo gateway sfrutta i metodi di pagamento WooCommerce esistenti, quindi assicurati di avere almeno un gateway di pagamento web configurato correttamente (Stripe, PayPal, ecc.).
:::

## Utilizzo

### Avvio Checkout Web

1. **Aggiungi Articoli**: Aggiungi prodotti al carrello nel POS
2. **Seleziona Gateway**: Scegli "Checkout Web" come metodo di pagamento
3. **Genera Link**: Il sistema crea un link di checkout unico per l'ordine
4. **Accesso Cliente**: Il cliente fa clic sul link per accedere al checkout web
5. **Pagamento Online**: Il cliente completa il pagamento utilizzando i metodi di pagamento del negozio web

### Processo di Pagamento

Il gateway di Checkout Web segue questo flusso di lavoro:

1. **Creazione Ordine**: Il POS crea un ordine in sospeso in WooCommerce
2. **Link Checkout**: Il sistema genera un link di pagamento sicuro e unico
3. **Reindirizzamento Cliente**: Il cliente viene indirizzato alla pagina di checkout web
4. **Selezione Pagamento**: Il cliente sceglie tra i metodi di pagamento web disponibili
5. **Elaborazione Pagamento**: Il processo di checkout standard di WooCommerce gestisce il pagamento
6. **Completamento Ordine**: Lo stato dell'ordine si aggiorna automaticamente al pagamento riuscito
