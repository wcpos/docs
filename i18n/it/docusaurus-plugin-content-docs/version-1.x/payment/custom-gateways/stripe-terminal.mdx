# Gateway Terminal Stripe {#stripe-terminal-gateway}

Il gateway Terminal Stripe ti consente di accettare pagamenti di persona utilizzando lettori hardware Stripe Terminal direttamente all'interno di WooCommerce POS. Questo gateway supporta sia lettori di carte fisici che la modalità simulatore per i test.

## Funzionalità {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Integrazione Hardware">
    Collega lettori fisici di Stripe Terminal tramite connessione internet
  </FeatureCard>
  <FeatureCard icon="desktop" title="Modalità Simulatore">
    Testa pagamenti senza hardware utilizzando il simulatore di Stripe
  </FeatureCard>
  <FeatureCard icon="bolt" title="Elaborazione in Tempo Reale">

## Installazione {#installation}

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina delle release di Stripe Terminal per WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Scarica il file **stripe-terminal-for-woocommerce.zip** più recente
3. Nel tuo admin di WordPress, vai su `Plugins > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Configura le Impostazioni Stripe">

1. Naviga su `WP Admin > WooCommerce > Impostazioni > Pagamenti`
2. Trova **Stripe Terminal** nell'elenco delle modalità di pagamento
3. Clicca su **Stripe Terminal** per accedere alle impostazioni
4. Inserisci la tua **Chiave Segreta Stripe** (puoi ottenerla dal tuo Dashboard Stripe)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway Terminal Stripe nelle impostazioni di WooCommerce. Sarà abilitato specificamente per il POS nel passaggio successivo.
:::

  </StepCard>
  <StepCard number="3" title="Abilita in WooCommerce POS" isLast>

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Checkout`
2. Trova il gateway **Stripe Terminal** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

  </StepCard>
</Steps>

## Utilizzo {#usage}

### Connessione di un Lettore {#connecting-a-reader}

Quando selezioni il gateway Terminal Stripe durante il checkout nel POS:

1. **Scegli Metodo di Connessione**: Puoi collegare un lettore fisico o utilizzare il simulatore
2. **Lettore Fisico**: Segui le istruzioni a schermo per collegare il tuo dispositivo Stripe Terminal
3. **Simulatore**: Seleziona la modalità simulatore per testare vari scenari di pagamento senza hardware

### Elaborazione dei Pagamenti {#processing-payments}

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Stripe Terminal" come metodo di pagamento
3. **Collega Lettore**: Collega il tuo lettore o scegli la modalità simulatore
4. **Elabora Pagamento**: Segui le istruzioni per completare la transazione
5. **Conferma**: L'ordine sarà automaticamente completato al pagamento riuscito

### Test con il Simulatore {#testing-with-simulator}

Il simulatore ti consente di testare vari metodi e scenari di pagamento:

- **Pagamenti con Carta**: Testa diversi tipi di carte (Visa, Mastercard, ecc.)
- **Pagamenti Contactless**: Simula transazioni tap-to-pay
- **Scenari di Errore**: Testa pagamenti rifiutati e altre condizioni di errore
- **Importi Diversi**: Testa vari importi di transazione

## Compatibilità Hardware {#hardware-compatibility}

:::important Requisiti di Connettività
Questa implementazione utilizza il SDK JavaScript di Stripe, il che significa che funziona tramite applicazioni web ma richiede esclusivamente **terminali connessi a internet**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet) {#supported-terminals-internet-connected}

- **Lettore Stripe S700** ✅ - terminale connesso tramite Ethernet/WiFi
- **WisePOS E** ✅ - terminale connesso tramite WiFi

### Terminali Non Supportati (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - solo Bluetooth
- **BBPOS WisePad 3** ❌ - solo Bluetooth  
- **Verifone P400** ❌ - solo Bluetooth

:::note Supporto Futuro
Il supporto per terminali Bluetooth sarà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo abiliterà il supporto per tutti i lettori certificati Stripe Terminal, inclusi M2 e WisePad 3.
:::

### Problemi Comuni {#common-issues}

<Accordion>
  <AccordionItem question="Il lettore non si connette">

  - Assicurati di utilizzare un terminale connesso a internet supportato (S700 o WisePOS E)
  - Verifica che il terminale sia connesso a WiFi/Ethernet e online
  - Controlla che il tuo account Stripe abbia Terminal abilitato
  - Conferma che il terminale sia registrato nel tuo Dashboard Stripe

  </AccordionItem>

  <AccordionItem question="Pagamento rifiutato">

  - Controlla che il tuo account Stripe sia attivo e in regola
  - Verifica che la carta utilizzata sia valida
  - Assicurati che siano disponibili fondi sufficienti

  </AccordionItem>

  <AccordionItem question="Errori di certificato SSL">

  - Stripe Terminal richiede HTTPS - assicurati che il tuo sito abbia un certificato SSL valido
  - Controlla che il tuo certificato SSL sia configurato correttamente

  </AccordionItem>
</Accordion>
