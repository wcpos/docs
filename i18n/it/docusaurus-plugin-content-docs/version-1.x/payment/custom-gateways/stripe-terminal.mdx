# Gateway Stripe Terminal

Il gateway Stripe Terminal ti consente di accettare pagamenti in negozio utilizzando i lettori hardware Stripe Terminal direttamente all'interno di WooCommerce POS. Questo gateway supporta sia lettori fisici che la modalità simulatore per i test.

## Funzionalità

- **Integrazione Hardware**: Connetti lettori fisici Stripe Terminal tramite connessione internet
- **Modalità Simulatore**: Testa i pagamenti senza hardware utilizzando il simulatore di Stripe
- **Metodi di Pagamento Multipli**: Supporto per diversi tipi di carte e metodi di pagamento
- **Elaborazione in Tempo Reale**: Elaborazione e conferma dei pagamenti istantanea
- **Transazioni Sicure**: Elaborazione dei pagamenti conforme a PCI tramite Stripe
- **SDK Web-based**: Utilizza l'SDK JavaScript di Stripe per la compatibilità con applicazioni web

## Installazione

### Passaggio 1: Scarica e Installa

1. Visita la [pagina dei rilasci di Stripe Terminal per WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Scarica l'ultimo file **stripe-terminal-for-woocommerce.zip**
3. Nell'admin di WordPress, vai su `Plugins > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

### Passaggio 2: Configura Impostazioni Stripe

1. Naviga su `Admin WP > WooCommerce > Impostazioni > Pagamenti`
2. Trova **Stripe Terminal** nell'elenco dei metodi di pagamento
3. Fai clic su **Stripe Terminal** per accedere alle impostazioni
4. Inserisci la tua **Chiave Segreta Stripe** (puoi ottenerla dal tuo Dashboard Stripe)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway Stripe Terminal nelle impostazioni di WooCommerce. Verrà abilitato specificamente per il POS nel passaggio successivo.
:::

### Passaggio 3: Abilita in WooCommerce POS

1. Vai su `Admin WP > WooCommerce POS > Impostazioni > Checkout`
2. Trova il gateway **Stripe Terminal** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

## Utilizzo

### Connessione di un Lettore

Quando selezioni il gateway Stripe Terminal durante il checkout nel POS:

1. **Scegli Metodo di Connessione**: Puoi connettere un lettore fisico o utilizzare il simulatore
2. **Lettore Fisico**: Segui le istruzioni a schermo per connettere il tuo dispositivo Stripe Terminal
3. **Simulatore**: Seleziona la modalità simulatore per testare vari scenari di pagamento senza hardware

### Elaborazione Pagamenti

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Stripe Terminal" come metodo di pagamento
3. **Connetti Lettore**: Connetti il tuo lettore o scegli la modalità simulatore
4. **Elabora Pagamento**: Segui le istruzioni per completare la transazione
5. **Conferma**: L'ordine verrà completato automaticamente al pagamento riuscito

### Test con Simulatore

Il simulatore ti permette di testare vari metodi di pagamento e scenari:

- **Pagamenti Carta**: Testa diversi tipi di carte (Visa, Mastercard, ecc.)
- **Pagamenti Contactless**: Simula transazioni tap-to-pay
- **Scenari di Errore**: Testa pagamenti rifiutati e altre condizioni di errore
- **Importi Diversi**: Testa vari importi di transazione

## Requisiti

- **Account Stripe**: Account Stripe attivo con Terminal abilitato
- **Chiavi API**: Chiave segreta Stripe dal tuo dashboard
- **WooCommerce POS**: Versione gratuita o Pro installata e attivata
- **HTTPS**: Il tuo sito deve utilizzare SSL/HTTPS per la sicurezza

## Compatibilità Hardware

:::important Requisiti di Connettività
Questa implementazione utilizza l'SDK JavaScript di Stripe, il che significa che funziona tramite applicazioni web ma richiede **terminali solo con connessione internet**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet)

- **Stripe Reader S700** ✅ - Terminale connesso tramite Ethernet/WiFi
- **WisePOS E** ✅ - Terminale connesso WiFi

### Terminali Non Supportati (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth
- **Verifone P400** ❌ - Solo Bluetooth

:::note Supporto Futuro
Il supporto per terminali Bluetooth verrà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo consentirà il supporto per tutti i lettori certificati Stripe Terminal, inclusi M2 e WisePad 3.
:::

## Risoluzione dei Problemi

### Problemi Comuni

**Lettore Non si Connette**
- Assicurati di utilizzare un terminale connesso a internet supportato (S700 o WisePOS E)
- Verifica che il terminale sia connesso a WiFi/Ethernet e online
- Controlla che il tuo account Stripe abbia Terminal abilitato
- Conferma che il terminale sia registrato nel tuo Dashboard Stripe

**Pagamento Rifiutato**
- Controlla che il tuo account Stripe sia attivo e in buono stato
- Verifica che la carta utilizzata sia valida
- Assicurati che siano disponibili fondi sufficienti

**Errori Certificato SSL**
- Stripe Terminal richiede HTTPS - assicurati che il tuo sito abbia un certificato SSL valido
- Verifica che il tuo certificato SSL sia configurato correttamente
