# Gateway di Stripe Terminal {#stripe-terminal-gateway}

Il gateway di Stripe Terminal ti consente di accettare pagamenti di persona utilizzando lettori hardware Stripe Terminal direttamente all'interno di WooCommerce POS. Questo gateway supporta sia lettori di carte fisiche che la modalità simulatore per i test.

## Installazione {#installation}

<Steps>
  <StepCard number="1" title="Scarica e Installa">

1. Visita la [pagina delle release di Stripe Terminal per WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Scarica il file più recente **stripe-terminal-for-woocommerce.zip**
3. Nel tuo admin di WordPress, vai su `Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip e attiva il plugin

  </StepCard>
  <StepCard number="2" title="Configura le Impostazioni di Stripe">

1. Naviga su `WP Admin > WooCommerce > Impostazioni > Pagamenti`
2. Trova **Stripe Terminal** nell'elenco dei metodi di pagamento
3. Clicca su **Stripe Terminal** per accedere alle impostazioni
4. Inserisci la tua **Chiave Segreta di Stripe** (puoi ottenerla dal tuo Dashboard Stripe)
5. Salva le impostazioni

:::note
Non è necessario abilitare il gateway di Stripe Terminal nelle impostazioni di WooCommerce. Sarà abilitato specificamente per il POS nel passaggio successivo.
:::

  </StepCard>
  <StepCard number="3" title="Abilita in WooCommerce POS" isLast>

1. Vai su `WP Admin > WooCommerce POS > Impostazioni > Pagamento`
2. Trova il gateway **Stripe Terminal** nell'elenco
3. Abilita il gateway per l'uso nel POS
4. Salva le tue impostazioni

  </StepCard>
</Steps>

## Utilizzo {#usage}

### Collegamento di un Lettore {#connecting-a-reader}

Quando selezioni il gateway di Stripe Terminal durante il pagamento nel POS:

1. **Scegli il Metodo di Connessione**: Puoi collegare un lettore fisico o utilizzare il simulatore
2. **Lettore Fisico**: Segui le istruzioni sullo schermo per collegare il tuo dispositivo Stripe Terminal
3. **Simulatore**: Seleziona la modalità simulatore per testare vari scenari di pagamento senza hardware

### Elaborazione dei Pagamenti {#processing-payments}

1. **Aggiungi Articoli**: Aggiungi prodotti al tuo carrello nel POS
2. **Seleziona Gateway**: Scegli "Stripe Terminal" come metodo di pagamento
3. **Collega Lettore**: Collega il tuo lettore o scegli la modalità simulatore
4. **Elabora Pagamento**: Segui le istruzioni per completare la transazione
5. **Conferma**: L'ordine sarà automaticamente completato dopo il pagamento riuscito

### Test con il Simulatore {#testing-with-simulator}

Il simulatore ti consente di testare vari metodi di pagamento e scenari:

- **Pagamenti con Carta**: Testa diversi tipi di carte (Visa, Mastercard, ecc.)
- **Pagamenti Contactless**: Simula transazioni tap-to-pay
- **Scenari di Errore**: Testa pagamenti rifiutati e altre condizioni di errore
- **Ammontari Diversi**: Testa vari importi di transazione

## Compatibilità Hardware {#hardware-compatibility}

:::important Requisiti di Connettività
Questa implementazione utilizza l'SDK JavaScript di Stripe, il che significa che funziona tramite applicazioni web ma richiede **terminali solo connessi a internet**. I terminali Bluetooth non sono attualmente supportati.
:::

### Terminali Supportati (Connessi a Internet) {#supported-terminals-internet-connected}

- **Lettore Stripe S700** ✅ - Terminale connesso Ethernet/WiFi
- **WisePOS E** ✅ - Terminale connesso WiFi

### Terminali Non Supportati (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth  
- **Verifone P400** ❌ - Solo Bluetooth

:::note Supporto Futuro
Il supporto per i terminali Bluetooth sarà aggiunto alle applicazioni iOS e Android nel **Q1 2026**. Questo abiliterà il supporto per tutti i lettori certificati Stripe Terminal, inclusi il M2 e il WisePad 3.
:::

### Problemi Comuni {#common-issues}

<Accordion>
  <AccordionItem question="Il lettore non si connette">

  - Assicurati di utilizzare un terminale connesso a internet supportato (S700 o WisePOS E)
  - Verifica che il terminale sia connesso a WiFi/Ethernet e sia online
  - Controlla che il tuo account Stripe abbia il Terminal abilitato
  - Conferma che il terminale sia registrato nel tuo Dashboard Stripe

  </AccordionItem>

  <AccordionItem question="Pagamento rifiutato">

  - Controlla che il tuo account Stripe sia attivo e in regola
  - Verifica che la carta utilizzata sia valida
  - Assicurati di avere fondi sufficienti disponibili

  </AccordionItem>

  <AccordionItem question="Errori di certificato SSL">

  - Stripe Terminal richiede HTTPS - assicurati che il tuo sito abbia un certificato SSL valido
  - Controlla che il tuo certificato SSL sia configurato correttamente

  </AccordionItem>
</Accordion>
