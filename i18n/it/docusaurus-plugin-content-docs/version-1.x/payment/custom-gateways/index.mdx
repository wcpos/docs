# Porte di pagamento personalizzate

WCPOS supporta diverse porte di pagamento personalizzate che estendono la funzionalità del tuo sistema Point of Sale. Queste porte sono progettate specificamente per ambienti POS e forniscono un'integrazione senza soluzione di continuità con vari metodi e servizi di pagamento.

## Porte personalizzate disponibili

### Porte di terminal hardware

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti utilizzando i lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora pagamenti tramite i lettori di carte SumUp

### Porte di pagamento digitale

- **[Fattura via email](./email-invoice)** - Invia fatture di pagamento ai clienti via email
- **[Web Checkout](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare la propria porta

Se hai bisogno di una porta di pagamento personalizzata per i tuoi requisiti specifici, puoi utilizzare il nostro **[Modello di Porta](./gateway-template)** per creare la tua porta di pagamento WooCommerce POS.

## Panoramica dell'installazione

Tutte le porte personalizzate seguono un simile processo di installazione:

1. **Scarica** l'ultima versione dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugins > Aggiungi nuovo > Carica plugin`
3. **Configura** le impostazioni della porta in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Abilita** la porta in `WP Admin > WooCommerce POS > Impostazioni > Checkout`

## Requisiti

- Plugin WooCommerce POS (versione gratuita o pro)
- WordPress con WooCommerce installato
- API keys o credenziali appropriate per il servizio di pagamento specifico

## Supporto

Per problemi con le porte personalizzate, visita il rispettivo repository GitHub e crea un problema con dettagliate informazioni sul tuo problema.
