# Gateway di Pagamento Personalizzati

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del sistema Point of Sale. Questi gateway sono progettati specificamente per ambienti POS e forniscono un'integrazione senza soluzione di continuità con vari metodi e servizi di pagamento.

## Gateway Disponibili

### Gateway per Terminali Hardware

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti utilizzando lettori hardware Stripe Terminal
- **[Terminale SumUp](./sumup-terminal)** - Elabora pagamenti tramite lettori di carte SumUp

### Gateway di Pagamento Digitali

- **[Fattura Email](./email-invoice)** - Invia fatture di pagamento ai clienti tramite email
- **[Checkout Web](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creazione di un Gateway Personalizzato

Se hai bisogno di un gateway di pagamento personalizzato per requisiti specifici, puoi utilizzare il nostro **[Modello di Gateway](./gateway-template)** per creare un gateway di pagamento WooCommerce POS personalizzato.

## Panoramica dell'Installazione

Tutti i gateway personalizzati seguono un processo di installazione simile:

1. **Scarica** l'ultima versione dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugins > Aggiungi Nuovo > Carica Plugin`
3. **Configura** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Attiva** il gateway in `WP Admin > WooCommerce POS > Impostazioni > Checkout`

## Requisiti

- Plugin WooCommerce POS (versione gratuita o pro)
- WordPress con WooCommerce installato
- Chiavi API o credenziali appropriate per il servizio di pagamento specifico

## Supporto

Per problemi relativi ai gateway personalizzati, visita il repository GitHub corrispondente e crea un issue con informazioni dettagliate sul tuo problema.
