# Gateway di Pagamento Personalizzati {#custom-payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono la funzionalitÃ  del tuo sistema di Punto Vendita. Questi gateway sono progettati specificamente per ambienti POS e forniscono un'integrazione fluida con vari metodi e servizi di pagamento.

## Gateway Personalizzati Disponibili {#available-custom-gateways}

### Gateway per Terminal Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti utilizzando lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora pagamenti tramite lettori di carte SumUp

### Gateway di Pagamento Digitale {#digital-payment-gateways}

- **[Fattura Email](./email-invoice)** - Invia fatture di pagamento ai clienti tramite email
- **[Checkout Web](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare il Proprio Gateway {#creating-your-own-gateway}

Se hai bisogno di un gateway di pagamento personalizzato per le tue esigenze specifiche, puoi utilizzare il nostro **[Template di Gateway](./gateway-template)** per creare il tuo gateway di pagamento WooCommerce POS.

## Panoramica dell'Installazione {#installation-overview}

Tutti i gateway personalizzati seguono un processo di installazione simile:

1. **Scarica** l'ultima release dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugin > Aggiungi Nuovo > Carica Plugin`
3. **Configura** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Abilita** il gateway in `WP Admin > WooCommerce POS > Impostazioni > Checkout`
