# Gateway di pagamento personalizzati {#custom-payment-gateways}

WCPOS supporta diversi gateway di pagamento personalizzati che estendono le funzionalità del sistema Point of Sale. Questi gateway sono progettati specificamente per ambienti POS e offrono un'integrazione fluida con vari metodi e servizi di pagamento.

## Gateway personalizzati disponibili {#available-custom-gateways}

### Gateway per terminali hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accetta pagamenti tramite i lettori hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Elabora i pagamenti tramite i lettori di carte SumUp

### Gateway per pagamenti mobili {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accetta pagamenti da telefono tramite codice QR o notifica push

### Gateway di pagamento digitali {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Invia fatture di pagamento ai clienti via email
- **[Web Checkout](./web-checkout)** - Reindirizza i clienti per completare i pagamenti online

## Creare il proprio gateway {#creating-your-own-gateway}

Se hai bisogno di un gateway di pagamento personalizzato per le tue esigenze specifiche, puoi utilizzare il nostro **[Modello di gateway](./gateway-template)** per creare il tuo gateway di pagamento WCPOS.

## Panoramica dell'installazione {#installation-overview}

Tutti i gateway personalizzati seguono una procedura di installazione simile:

1. **Scarica** l'ultima versione dal rispettivo repository GitHub
2. **Installa** il plugin tramite `WP Admin > Plugin > Aggiungi nuovo > Carica plugin`
3. **Configura** le impostazioni del gateway in `WP Admin > WooCommerce > Impostazioni > Pagamenti`
4. **Abilita** il gateway in `WP Admin > Cassa > Impostazioni > Checkout`
