:::info Funzione Pro
La schermata Prodotti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti gratuiti possono visualizzare e aggiungere prodotti al carrello dal [Pannello Prodotti POS](/pos/product-panel), ma non possono modificare l'inventario o i prezzi.
:::

La schermata Prodotti è un'interfaccia dedicata alla gestione dell'inventario. A differenza del [Pannello Prodotti POS](/pos/product-panel) (che è per la vendita), questa schermata è progettata per gestire il tuo catalogo prodotti.

{/* Screenshot: products-screen.png */}

## Schermata Prodotti vs Pannello Prodotti POS {#products-screen-vs-pos-product-panel}

| Caratteristica | Pannello Prodotti POS | Schermata Prodotti |
|----------------|-----------------------|---------------------|
| **Scopo** | Aggiungi prodotti al carrello | Gestisci inventario |
| **Stock** | Solo visualizzazione | Modifica in linea |
| **Prezzi** | Solo visualizzazione | Modifica in linea |
| **Azioni** | Aggiungi al carrello | Modifica, Sincronizza, Elimina |
| **Disponibile in** | Gratuito + Pro | Solo Pro |

## Panoramica dell'Interfaccia {#interface-overview}

### Ricerca e Filtri {#search-filters}

Nella parte superiore della schermata:

- **Barra di ricerca** - Trova prodotti per nome, SKU o codice a barre
- **Pulsanti di filtro** - Stato dello stock, In Evidenza, In Saldo, Categoria, Etichetta, Marchio
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Elenco Prodotti {#product-list}

L'area principale visualizza i tuoi prodotti in una tabella scorrevole:

- **Immagine** - Miniatura del prodotto
- **Nome prodotto** - Con SKU, codice a barre e attributi opzionali
- **Stock** - Quantità di stock modificabile con attivazione "Gestisci"
- **Stato dello Stock** - Disponibile (verde), Esaurito (rosso), Ordine arretrato
- **Categorie** - Distintivi della categoria del prodotto
- **Prezzi** - Prezzo attuale, prezzo regolare, prezzo in saldo (tutti modificabili)
- **Azioni** - Menu a tre punti con Modifica, Sincronizza, Elimina

### Prodotti Variabili {#variable-products}

I prodotti variabili mostrano:
- Opzioni di attributo (Colore, Taglia, ecc.)
- Link "Espandi" per visualizzare tutte le variazioni
- Righe di variazione individuali con stock/prezzi separati

### Piè di Pagina {#footer}

- **Stato fiscale** - Base attuale di calcolo delle tasse
- **Conteggio prodotti** - "Visualizzazione di X di Y" con pulsante di sincronizzazione (<Icon name="arrow-rotate-right" />). **Pressione prolungata** sul pulsante di sincronizzazione per opzione Cancella e Aggiorna

## Caratteristiche Principali {#key-features}

### Modifica dello Stock in Linea {#inline-stock-editing}

Clicca direttamente sulla quantità di stock per modificarla:

1. Clicca sul numero di stock
2. Inserisci la nuova quantità
3. Premi Invio per salvare

Le modifiche si sincronizzano automaticamente con WooCommerce.

### Attivazione Gestione Stock {#manage-stock-toggle}

L'attivazione "Gestisci" controlla se WooCommerce tiene traccia dell'inventario per questo prodotto:

- **Attivo** - I livelli di stock vengono monitorati e decrementati alle vendite
- **Disattivo** - Il prodotto è sempre disponibile (stock infinito)

### Modifica Prezzo in Linea {#inline-price-editing}

Clicca su qualsiasi campo prezzo per modificarlo:

- **Prezzo** - Prezzo di vendita attuale
- **Prezzo Regolare** - Prezzo non scontato
- **Prezzo in Saldo** - Prezzo scontato (quando in saldo)

### Azioni Prodotti {#product-actions}

Clicca sul menu a tre punti (⋮) per ciascun prodotto:

- **Modifica** - Apri il modulo di modifica prodotto
- **Sincronizza** - Aggiorna questo prodotto da WooCommerce
- **Elimina** - Rimuovi il prodotto dal database locale

:::note
Eliminare un prodotto dal POS lo rimuove solo localmente. Il prodotto rimane in WooCommerce e riapparirà alla prossima sincronizzazione.
:::

### Colonne Disponibili {#available-columns}

| Colonna | Descrizione | Opzioni di Visualizzazione |
|---------|-------------|----------------------------|
| **Immagine** | Miniatura del prodotto | - |
| **ID** | ID prodotto WooCommerce | - |
| **Prodotto** | Nome del prodotto | SKU, Codice a barre, Attributi |
| **Tipo** | Semplice, variabile, raggruppato, ecc. | - |
| **SKU** | Stock Keeping Unit | - |
| **Codice a barre** | Codice a barre del prodotto | - |
| **Stock** | Quantità attuale (modificabile) | - |
| **Stato dello Stock** | Disponibile / Esaurito / Ordine arretrato | - |
| **Categorie** | Categorie del prodotto | - |
| **Etichette** | Etichette del prodotto | - |
| **Marchi** | Informazioni sul marchio (se il plugin è attivo) | - |
| **Costo delle Merci Vendute** | Prezzo di costo del prodotto | - |
| **Prezzo** | Prezzo di vendita attuale (modificabile) | Informazioni fiscali |
| **Prezzo Regolare** | Prezzo non scontato (modificabile) | Informazioni fiscali |
| **Prezzo in Saldo** | Prezzo scontato (modificabile) | Informazioni fiscali |
| **Data di Creazione** | Quando è stato creato il prodotto | - |
| **Data di Modifica** | Data dell'ultima modifica | - |
| **Azioni** | Pulsanti Modifica, Sincronizza, Elimina | - |

### Ripristina Impostazioni Predefinite {#restore-default-settings}

Clicca per ripristinare tutte le impostazioni di visualizzazione ai loro valori predefiniti originali.

## Filtrare Prodotti {#filtering-products}

Usa i pulsanti di filtro per restringere la tua visualizzazione:

- **Stato dello Stock** - Disponibile, Esaurito, Ordine arretrato
- **In Evidenza** - Prodotti contrassegnati come in evidenza
- **In Saldo** - Prodotti attualmente in saldo
- **Categoria** - Filtra per categoria di prodotto
- **Etichetta** - Filtra per etichetta di prodotto
- **Marchio** - Filtra per marchio (se utilizzi il plugin marchio)

## Ordinamento {#sorting}

Clicca su qualsiasi intestazione di colonna per ordinare in base a quella colonna. Clicca di nuovo per invertire l'ordine di ordinamento.
