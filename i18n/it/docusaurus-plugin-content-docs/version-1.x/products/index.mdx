:::info Funzionalità Pro
La schermata Prodotti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti gratuiti possono visualizzare e aggiungere prodotti al carrello dal [Pannello Prodotti POS](/pos/product-panel), ma non possono modificare l'inventario o i prezzi.
:::

La schermata Prodotti è un'interfaccia dedicata alla gestione dell'inventario. A differenza del [Pannello Prodotti POS](/pos/product-panel) (che è per la vendita), questa schermata è progettata per gestire il tuo catalogo prodotti.

{/* Screenshot: products-screen.png */}

## Schermata Prodotti vs Pannello Prodotti POS

| Funzionalità | Pannello Prodotti POS | Schermata Prodotti |
|--------------|-----------------------|--------------------|
| **Scopo** | Aggiungere prodotti al carrello | Gestire l'inventario |
| **Inventario** | Solo visualizzazione | Modifica in linea |
| **Prezzi** | Solo visualizzazione | Modifica in linea |
| **Azioni** | Aggiungi al carrello | Modifica, Sincronizza, Elimina |
| **Disponibile in** | Gratuito + Pro | Solo Pro |

## Panoramica dell'Interfaccia

### Ricerca e Filtri

Nella parte superiore della schermata:

- **Barra di ricerca** - Trova prodotti per nome, SKU o codice a barre
- **Pulsanti di filtro** - Stato Magazzino, In Evidenza, In Vendita, Categoria, Tag, Marca
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Elenco Prodotti

L'area principale visualizza i tuoi prodotti in una tabella scrollabile:

- **Immagine** - Miniatura del prodotto
- **Nome prodotto** - Con SKU, codice a barre, attributi opzionali
- **Inventario** - Quantità di inventario modificabile con il toggle "Gestisci"
- **Stato Magazzino** - Disponibile (verde), Esaurito (rosso), In ordinazione
- **Categorie** - Distintivi delle categorie prodotto
- **Prezzi** - Prezzo attuale, prezzo normale, prezzo in promozione (tutti modificabili)
- **Azioni** - Menu a tre punti con Modifica, Sincronizza, Elimina

### Prodotti Variabili

I prodotti variabili mostrano:
- Opzioni di attributo (Colore, Taglia, ecc.)
- Link "Espandi" per visualizzare tutte le variazioni
- Righe di variazione individuali con stock/prezzi separati

### Piè di Pagina

- **Stato fiscale** - Base di calcolo fiscale attuale
- **Conteggio prodotti** - "Mostrando X di Y" con pulsante di sincronizzazione (<Icon name="arrow-rotate-right" />). **Premi a lungo** il pulsante di sincronizzazione per l'opzione Cancella e Aggiorna

## Caratteristiche Principali

### Modifica Stock in Linea

Clicca direttamente sulla quantità di inventario per modificarla:

1. Clicca sul numero di stock
2. Inserisci la nuova quantità
3. Premi Invio per salvare

Le modifiche si sincronizzano automaticamente con WooCommerce.

### Toggle Gestione Stock

Il toggle "Gestisci" controlla se WooCommerce tiene traccia dell'inventario per questo prodotto:

- **Attivo** - I livelli di stock vengono tracciati e decrementati con le vendite
- **Disattivo** - Il prodotto è sempre disponibile (stock infinito)

### Modifica Prezzo in Linea

Clicca su qualsiasi campo prezzo per modificare:

- **Prezzo** - Prezzo di vendita attuale
- **Prezzo Normale** - Prezzo non scontato
- **Prezzo in Vendita** - Prezzo scontato (quando in promozione)

### Azioni Prodotto

Clicca sul menu a tre punti (⋮) per ogni prodotto:

- **Modifica** - Apri la finestra di modifica del prodotto
- **Sincronizza** - Aggiorna questo prodotto da WooCommerce
- **Elimina** - Rimuovi il prodotto dal database locale

:::note
Eliminare un prodotto dal POS lo rimuove solo localmente. Il prodotto rimane in WooCommerce e riapparirà nella prossima sincronizzazione.
:::

### Colonne Disponibili

| Colonna | Descrizione | Opzioni di Visualizzazione |
|---------|-------------|----------------------------|
| **Immagine** | Miniatura del prodotto | - |
| **ID** | ID prodotto WooCommerce | - |
| **Prodotto** | Nome prodotto | SKU, Codice a barre, Attributi |
| **Tipo** | Semplice, variabile, raggruppato, ecc. | - |
| **SKU** | Stock Keeping Unit | - |
| **Codice a barre** | Codice a barre del prodotto | - |
| **Stock** | Quantità attuale (modificabile) | - |
| **Stato Stock** | Disponibile / Esaurito / In ordinazione | - |
| **Categorie** | Categorie prodotto | - |
| **Tag** | Tag prodotto | - |
| **Marche** | Info marca (se plugin attivo) | - |
| **Costo dei beni venduti** | Prezzo di costo del prodotto | - |
| **Prezzo** | Prezzo di vendita attuale (modificabile) | Info fiscale |
| **Prezzo Normale** | Prezzo non scontato (modificabile) | Info fiscale |
| **Prezzo in Vendita** | Prezzo scontato (modificabile) | Info fiscale |
| **Data di Creazione** | Quando è stato creato il prodotto | - |
| **Data di Modifica** | Ultima data di modifica | - |
| **Azioni** | Pulsanti Modifica, Sincronizza, Elimina | - |

### Ripristina Impostazioni Predefinite

Clicca per ripristinare tutte le impostazioni di visualizzazione ai loro valori predefiniti.

## Filtrare i Prodotti

Usa i pulsanti di filtro per ristrettare la tua visualizzazione:

- **Stato Stock** - Disponibile, Esaurito, In ordinazione
- **In Evidenza** - Prodotti contrassegnati come in evidenza
- **In Vendita** - Prodotti attualmente in vendita
- **Categoria** - Filtra per categoria prodotto
- **Tag** - Filtra per tag prodotto
- **Marca** - Filtra per marca (se usi il plugin marchi)

## Ordinamento

Clicca su qualsiasi intestazione di colonna per ordinare per quella colonna. Clicca di nuovo per invertire l'ordine di ordinamento.
