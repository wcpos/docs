:::info Funzionalità Pro
La schermata Prodotti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti del piano gratuito possono visualizzare e aggiungere prodotti al carrello dal [Pannello Prodotti POS](/pos/product-panel), ma non possono modificare l'inventario o i prezzi.
:::

La schermata Prodotti è un'interfaccia dedicata alla gestione dell'inventario. A differenza del [Pannello Prodotti POS](/pos/product-panel) (che è per la vendita), questa schermata è progettata per gestire il catalogo dei propri prodotti.

{/* Screenshot: products-screen.png */}

## Schermata Prodotti vs Pannello Prodotti POS

| Funzionalità | Pannello Prodotti POS | Schermata Prodotti |
|---------|-------------------|-----------------|
| **Scopo** | Aggiungere prodotti al carrello | Gestire l'inventario |
| **Scorte** | Solo visualizzazione | Modifica diretta |
| **Prezzi** | Solo visualizzazione | Modifica diretta |
| **Azioni** | Aggiungi al carrello | Modifica, Sincronizza, Elimina |
| **Disponibile in** | Gratuito + Pro | Solo Pro |

## Panoramica dell'Interfaccia

### Ricerca e Filtri

Nella parte superiore della schermata:

- **Barra di ricerca** - Trova prodotti per nome, SKU o codice a barre
- **Pulsanti di filtro** - Stato Scorte, In Evidenza, In Offerta, Categoria, Tag, Marca
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura colonne visibili

### Elenco Prodotti

L'area principale mostra i prodotti in una tabella scorrevole:

- **Immagine** - Miniatura del prodotto
- **Nome prodotto** - Con SKU, codice a barre, attributi opzionali
- **Scorte** - Quantità di scorte modificabile con interruttore "Gestisci"
- **Stato Scorte** - Disponibile (verde), Esaurito (rosso), Ordine Arretrato
- **Categorie** - Badge categoria prodotto
- **Prezzi** - Prezzo corrente, prezzo regolare, prezzo scontato (tutti modificabili)
- **Azioni** - Menu a tre punti con Modifica, Sincronizza, Elimina

### Prodotti Variabili

I prodotti variabili mostrano:
- Opzioni attributi (Colore, Taglia, ecc.)
- Link "Espandi" per visualizzare tutte le varianti
- Righe delle singole varianti con scorte/prezzi separati

### Footer

- **Stato fiscale** - Base di calcolo fiscale corrente
- **Conteggio prodotti** - "Visualizzazione X di Y" con pulsante di sincronizzazione

## Funzionalità Principali

### Modifica Scorte Inline

Fare clic direttamente sulla quantità di scorte per modificarla:

1. Fare clic sul numero delle scorte
2. Inserire la nuova quantità
3. Premere Invio per salvare

Le modifiche vengono sincronizzate automaticamente con WooCommerce.

### Interruttore Gestione Scorte

L'interruttore "Gestisci" controlla se WooCommerce tiene traccia dell'inventario per questo prodotto:

- **Attivo** - I livelli di scorte vengono tracciati e decrementati sulle vendite
- **Disattivo** - Prodotto sempre disponibile (scorte infinite)

### Modifica Prezzi Inline

Fare clic su qualsiasi campo prezzo per modificarlo:

- **Prezzo** - Prezzo di vendita corrente
- **Prezzo Regolare** - Prezzo non scontato
- **Prezzo Scontato** - Prezzo in promozione (quando in offerta)

### Azioni Prodotto

Fare clic sul menu a tre punti (⋮) per ogni prodotto:

- **Modifica** - Aprire la finestra modale di modifica del prodotto
- **Sincronizza** - Aggiornare questo prodotto da WooCommerce
- **Elimina** - Rimuovere il prodotto dal database locale

:::note
L'eliminazione di un prodotto dal POS lo rimuove solo localmente. Il prodotto rimane in WooCommerce e riapparirà alla successiva sincronizzazione.
:::

### Colonne Disponibili

| Colonna | Descrizione | Opzioni di Visualizzazione |
|--------|-------------|-----------------|
| **Immagine** | Miniatura prodotto | - |
| **ID** | ID prodotto WooCommerce | - |
| **Prodotto** | Nome prodotto | SKU, Codice a barre, Attributi |
| **Tipo** | Semplice, variabile, raggruppato, ecc. | - |
| **SKU** | Unità di Mantenimento Scorte | - |
| **Codice a Barre** | Codice a barre prodotto | - |
| **Scorte** | Quantità corrente (modificabile) | - |
| **Stato Scorte** | Disponibile / Esaurito / Ordine Arretrato | - |
| **Categorie** | Categorie prodotto | - |
| **Tag** | Tag prodotto | - |
| **Marche** | Informazioni marca (se plugin attivo) | - |
| **Costo del Venduto** | Prezzo di costo del prodotto | - |
| **Prezzo** | Prezzo di vendita corrente (modificabile) | Informazioni fiscali |
| **Prezzo Regolare** | Prezzo non scontato (modificabile) | Informazioni fiscali |
| **Prezzo Scontato** | Prezzo in promozione (modificabile) | Informazioni fiscali |
| **Data Creazione** | Quando il prodotto è stato creato | - |
| **Data Modifica** | Data ultima modifica | - |
| **Azioni** | Pulsanti Modifica, Sincronizza, Elimina | - |

### Ripristino Impostazioni Predefinite

Fare clic per reimpostare tutte le impostazioni di visualizzazione sui valori originali.

## Filtraggio Prodotti

Utilizzare i pulsanti di filtro per restringere la vista:

- **Stato Scorte** - Disponibile, Esaurito, Ordine Arretrato
- **In Evidenza** - Prodotti contrassegnati come in evidenza
- **In Offerta** - Prodotti attualmente in promozione
- **Categoria** - Filtra per categoria prodotto
- **Tag** - Filtra per tag prodotto
- **Marca** - Filtra per marca (se si usa il plugin delle marche)

## Ordinamento

Fare clic sull'intestazione di qualsiasi colonna per ordinare in base a quella colonna. Fare nuovamente clic per invertire l'ordine.
