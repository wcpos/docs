:::info Pro Feature
La schermata Prodotti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti gratuiti possono visualizzare e aggiungere prodotti al carrello dal [Pannello Prodotti POS](/pos/product-panel), ma non possono modificare l'inventario o i prezzi.
:::

La schermata Prodotti è un'interfaccia dedicata alla gestione dell'inventario. A differenza del [Pannello Prodotti POS](/pos/product-panel) (che è per le vendite), questa schermata è progettata per gestire il tuo catalogo prodotti.

{/* Screenshot: products-screen.png */}

## Schermata Prodotti vs Pannello Prodotti POS

| Caratteristica | Pannello Prodotti POS | Schermata Prodotti |
|----------------|-----------------------|--------------------|
| **Scopo** | Aggiungi prodotti al carrello | Gestisci l'inventario |
| **Stock** | Solo visualizzazione | Modifica in linea |
| **Prezzi** | Solo visualizzazione | Modifica in linea |
| **Azioni** | Aggiungi al carrello | Modifica, Sincronizza, Elimina |
| **Disponibile in** | Gratuito + Pro | Solo Pro |

## Panoramica dell'Interfaccia

### Ricerca e Filtri

Nella parte superiore della schermata:

- **Barra di ricerca** - Trova prodotti per nome, SKU o codice a barre
- **Pulsanti di filtro** - Stato dello Stock, In Evidenza, In Offerta, Categoria, Tag, Marca
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Elenco Prodotti

L'area principale visualizza i tuoi prodotti in una tabella scorrevole:

- **Immagine** - Miniatura del prodotto
- **Nome prodotto** - Con SKU opzionale, codice a barre, attributi
- **Stock** - Quantità di stock modificabile con attivazione "Gestisci"
- **Stato dello Stock** - In Stock (verde), Esaurito (rosso), In Preordine
- **Categorie** - Badge della categoria del prodotto
- **Prezzi** - Prezzo corrente, prezzo normale, prezzo in offerta (tutti modificabili)
- **Azioni** - Menu a tre punti con Modifica, Sincronizza, Elimina

### Prodotti Variabili

I prodotti variabili mostrano:
- Opzioni di attributo (Colore, Dimensione, ecc.)
- Link "Espandi" per visualizzare tutte le variazioni
- Righe di variazione individuali con stock/prezzi separati

### Piè di Pagina

- **Stato fiscale** - Base attuale per il calcolo delle tasse
- **Conteggio prodotti** - "Visualizzazione di X di Y" con pulsante di sincronizzazione

## Caratteristiche Principali

### Modifica Stock in Linea

Fai clic direttamente sulla quantità di stock per modificarla:

1. Fai clic sul numero di stock
2. Inserisci la nuova quantità
3. Premi Invio per salvare

Le modifiche si sincronizzano automaticamente con WooCommerce.

### Attivazione Gestione Stock

L'attivazione "Gestisci" controlla se WooCommerce tiene traccia dell'inventario per questo prodotto:

- **Attivo** - I livelli di stock vengono tenuti traccia e decrementati alle vendite
- **Disattivo** - Il prodotto è sempre disponibile (stock infinito)

### Modifica Prezzo in Linea

Fai clic su qualsiasi campo di prezzo per modificare:

- **Prezzo** - Prezzo di vendita corrente
- **Prezzo Normale** - Prezzo non scontato
- **Prezzo di Vendita** - Prezzo scontato (quando in offerta)

### Azioni sul Prodotto

Fai clic sul menu a tre punti (⋮) per ogni prodotto:

- **Modifica** - Apri il modulo di modifica del prodotto
- **Sincronizza** - Aggiorna questo prodotto da WooCommerce
- **Elimina** - Rimuovi il prodotto dal database locale

:::note
L'eliminazione di un prodotto dal POS rimuove solo localmente. Il prodotto rimane in WooCommerce e riapparirà alla successiva sincronizzazione.
:::

### Colonne Disponibili

| Colonna | Descrizione | Opzioni di Visualizzazione |
|---------|-------------|---------------------------|
| **Immagine** | Miniatura del prodotto | - |
| **ID** | ID prodotto WooCommerce | - |
| **Prodotto** | Nome del prodotto | SKU, Codice a Barre, Attributi |
| **Tipo** | Semplice, variabile, raggruppato, ecc. | - |
| **SKU** | Stock Keeping Unit | - |
| **Codice a Barre** | Codice a barre del prodotto | - |
| **Stock** | Quantità attuale (modificabile) | - |
| **Stato Stock** | In Stock / Esaurito / In Preordine | - |
| **Categorie** | Categorie di prodotto | - |
| **Tag** | Tag di prodotto | - |
| **Marche** | Informazioni sulla marca (se il plugin è attivo) | - |
| **Costo delle Merci Vendute** | Prezzo di costo del prodotto | - |
| **Prezzo** | Prezzo di vendita corrente (modificabile) | Informazioni fiscali |
| **Prezzo Normale** | Prezzo non scontato (modificabile) | Informazioni fiscali |
| **Prezzo di Vendita** | Prezzo scontato (modificabile) | Informazioni fiscali |
| **Data Creata** | Quando è stato creato il prodotto | - |
| **Data Modificata** | Ultima data di modifica | - |
| **Azioni** | Pulsanti Modifica, Sincronizza, Elimina | - |

### Ripristina Impostazioni Predefinite

Fai clic per ripristinare tutte le impostazioni di visualizzazione ai loro valori predefiniti originali.

## Filtrare Prodotti

Usa i pulsanti di filtro per restringere la tua vista:

- **Stato dello Stock** - In Stock, Esaurito, In Preordine
- **In Evidenza** - Prodotti contrassegnati come in evidenza
- **In Offerta** - Prodotti attualmente in offerta
- **Categoria** - Filtro per categoria di prodotto
- **Tag** - Filtro per tag di prodotto
- **Marca** - Filtro per marca (se usi il plugin per le marche)

## Ordinamento

Fai clic su qualsiasi intestazione di colonna per ordinare per quella colonna. Fai clic di nuovo per invertire l'ordine di ordinamento.
