WCPOS memorizza una copia locale dei tuoi prodotti, ordini e clienti in modo che il POS possa funzionare rapidamente e operare offline. Occasionalmente, questi dati locali possono diventare fuori sincrono con il tuo negozio WooCommerce. Cancellare tutti i dati locali rimuove il database locale e attiva una nuova sincronizzazione dal tuo server.

:::warning
Questa è l'opzione nucleare. Cancellare i dati locali elimina tutto ciò che è memorizzato sul dispositivo e forza una ri-sincronizzazione completa. A seconda della grandezza del tuo negozio, questo potrebbe richiedere del tempo. Usala solo quando altre opzioni — come una normale [sincronizzazione](/products/sync) — non hanno risolto il problema.
:::

## Quando Usarlo {#when-to-use-it}

Potresti dover cancellare i dati locali quando:

- **Problemi di sincronizzazione** — prodotti, ordini o clienti non si aggiornano correttamente
- **Dopo un aggiornamento del plugin** — per garantire la compatibilità con le nuove strutture di dati
- **Dati obsoleti** — prezzi, livelli di stock o altre informazioni sembrano obsolete

## Cosa Viene Cancellato {#what-gets-cleared}

| Cancellato | Non Cancellato |
|------------|----------------|
| Database locale dei prodotti | Dati lato server |
| Database locale degli ordini | Impostazioni di WooCommerce |
| Database locale dei clienti | |
| Sessione di accesso | |

I dati del tuo negozio WooCommerce non sono influenzati — solo la copia locale sul tuo dispositivo viene rimossa. Una nuova sincronizzazione inizia automaticamente dopo la cancellazione.

:::info Differenza tra piattaforme
- **App desktop:** Verrai disconnesso e dovrai ricollegare il tuo negozio.
- **App web:** I dati di accesso vengono cancellati, ma poiché sei ancora connesso a WP Admin la sessione si ri-idrata automaticamente — quindi non noterai la disconnessione.
:::

## Come Cancellare i Dati Locali {#how-to-clear-local-data}

### Web {#web}

1. Clicca sul tuo **menu utente** in alto a destra per aprire il menu
2. Clicca su **Cancella Tutti i Dati Locali**
3. Il database locale verrà eliminato e una nuova sincronizzazione inizierà automaticamente

### Desktop {#desktop}

1. Apri il menu dell'app **WCPOS** nella barra del menu in alto
2. Passa il mouse su **Advanced**
3. Clicca su **Clear App Data**
4. Il database locale verrà eliminato e una nuova sincronizzazione inizierà automaticamente

:::note
L'app desktop utilizza un database SQLite mentre l'app web utilizza IndexedDB. Entrambi i database vengono cancellati completamente, ma il comportamento di accesso differisce — vedi la nota sulla differenza tra piattaforme sopra.
:::
