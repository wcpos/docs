La pagina Impostazioni di pagamento (`WP Admin > POS > Settings > Checkout`) controlla quali gateway di pagamento vengono visualizzati nel POS e come vengono elaborati gli ordini.

## Gateway di pagamento {#payment-gateways}

Questa sezione elenca tutti i gateway di pagamento WooCommerce installati sul sito. Ogni gateway può essere abilitato o disabilitato in modo indipendente per il POS.

| Colonna | Descrizione |
|--------|-------------|
| **Abilitato** | Attiva o disattiva il gateway per il pagamento POS |
| **Gateway** | Il nome del gateway di pagamento |
| **ID Gateway** | L'identificatore del gateway WooCommerce (utile per gli sviluppatori) |

### Abilitare un gateway {#enabling-a-gateway}

1. Individuare il gateway nell'elenco
2. Attivare l'interruttore **Abilitato**
3. Il gateway comparirà al checkout del POS al successivo caricamento della pagina

### Riordinare i gateway {#reordering-gateways}

Trascinare i gateway per modificarne l'ordine di visualizzazione nel checkout del POS. Il primo gateway abilitato viene selezionato per impostazione predefinita quando un cassiere apre la finestra di checkout.

### POS e negozio online a confronto {#pos-vs-online-store}

Le impostazioni dei gateway in questa sezione sono **indipendenti** dalle impostazioni di pagamento di WooCommerce. Un gateway può essere:

- **Abilitato per il POS, disabilitato online** — utile per i pagamenti in contanti, che non si applicano agli ordini web
- **Abilitato online, disabilitato per il POS** — utile per gateway come PayPal che non risultano pratici per le vendite di persona
- **Abilitato per entrambi** — il gateway funziona ovunque

:::note
Un gateway deve prima essere installato e attivo in `WP Admin > WooCommerce > Settings > Payments` prima di apparire nell'elenco delle impostazioni di checkout del POS. Se un gateway è mancante, verificare che il plugin sia installato e che il gateway sia abilitato in WooCommerce.
:::

## Stato dell'ordine {#order-status}

Configura lo stato predefinito degli ordini completati dal POS. Per impostazione predefinita, gli ordini POS vengono impostati su **Completato** poiché il pagamento viene incassato al punto vendita.
