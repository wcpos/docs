## Scorciatoie Disponibili

| Combinazione di Tasti | Azione | Descrizione |
|----------------------|--------|-------------|
| `Ctrl + Shift + S`   | Impostazioni | Apre il pannello delle impostazioni POS |
| `Ctrl + Shift + A`   | POS    | Naviga alla schermata principale del POS |
| `Ctrl + Shift + P`   | Prodotti | Apre la schermata di gestione dei prodotti |
| `Ctrl + Shift + O`   | Ordini | Apre la schermata di gestione degli ordini |
| `Ctrl + Shift + C`   | Clienti | Apre la schermata di gestione dei clienti |
| `Ctrl + Shift + ?`   | Supporto | Apre la sezione di supporto/aiuto |
| `Ctrl + Shift + L`   | Disconnetti | Disconnette dalla sessione corrente |

## Utilizzo delle Scorciatoie da Tastiera

Per utilizzare qualsiasi scorciatoia da tastiera:

1. **Tieni premuti i tasti modificatori** (`Ctrl + Shift`)
2. **Premi il tasto d'azione** (la lettera corrispondente all'azione desiderata)
3. **Rilascia tutti i tasti** per eseguire il comando

## Suggerimenti per un Uso Efficiente

- **Memorizza le scorciatoie frequentemente utilizzate**: Concentrati sull'imparare le scorciatoie per le azioni che esegui più spesso
- **Pratica le combinazioni di tasti**: L'uso regolare aiuterà a costruire la memoria muscolare
- **Utilizza le scorciatoie durante i periodi di alta affluenza**: Le scorciatoie da tastiera possono accelerare significativamente le operazioni durante le ore di punta

## Personalizzazione
