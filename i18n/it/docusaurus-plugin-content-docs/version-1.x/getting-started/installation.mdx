Per garantire il corretto funzionamento e le prestazioni ottimali del plugin WCPOS, assicurarsi che il proprio ambiente soddisfi i seguenti requisiti minimi:

## Requisiti del server {#server-requirements}

Il plugin WCPOS viene eseguito sul tuo sito WordPress, che deve soddisfare questi requisiti minimi:

| Componente  | Versione minima | Note            |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x consigliata |

Il rispetto di questi requisiti minimi garantisce compatibilità e stabilità con il tuo ambiente WordPress, oltre all'accesso alle funzionalità più recenti di WCPOS.

## Requisiti del browser {#browser-requirements}

La versione web del POS funziona in qualsiasi browser moderno. Utilizza l'ultima versione stabile:

| Browser         | Versione supportata |
| --------------- | ------------------- |
| Google Chrome   | Ultima stabile      |
| Mozilla Firefox | Ultima stabile      |
| Safari          | Ultima stabile      |
| Microsoft Edge  | Ultima stabile      |

:::warning Evita la modalità di navigazione privata / in incognito
La navigazione privata può bloccare l'accesso al database IndexedDB del browser, che WCPOS utilizza per archiviare i dati offline — alcune funzionalità non funzioneranno correttamente. Anche JavaScript deve essere abilitato.
:::

## Requisiti dell'app {#app-requirements}

WCPOS è disponibile anche come app desktop e mobile. Ciascuna è sviluppata per i seguenti sistemi operativi minimi:

| Piattaforma     | Sistema operativo minimo     | Architetture               |
| --------------- | ---------------------------- | -------------------------- |
| Desktop Windows | Windows 10                   | x64, ARM64                 |
| Desktop macOS   | macOS 12 (Monterey)          | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                     | —                          |
| Android         | Android 10 (API livello 29)  | —                          |

:::note Permessi di stampa
La stampa Bluetooth e di rete richiede di concedere all'app i relativi permessi Bluetooth e di rete locale quando richiesto.
:::

## Installazione del plugin {#installing-the-plugin}

1. Andare su `WP Admin > Plugins > Add Plugin`
2. Cercare "wcpos"
3. Fare clic su **Installa ora**, poi su **Attiva**

In alternativa, è possibile scaricare il plugin da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e caricarlo manualmente.

## Accesso al POS {#accessing-the-pos}

Dopo l'attivazione, è possibile accedere al POS all'indirizzo:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Dispositivi mobili:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Soddisfacendo i requisiti minimi e utilizzando un browser web compatibile, è possibile ottenere un'esperienza fluida con il plugin WCPOS e sfruttarne appieno il potenziale per gestire le operazioni del punto vendita in modo efficiente.

Per ulteriori domande o necessità di assistenza, non esitare a contattare il nostro team di supporto pubblicando un messaggio nella [chat Discord](https://wcpos.com/discord) o inviando un'email a support@wcpos.com.
