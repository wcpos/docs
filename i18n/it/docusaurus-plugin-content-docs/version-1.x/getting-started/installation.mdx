Per garantire il corretto funzionamento e le prestazioni ottimali del plugin WCPOS, assicurarsi che il proprio ambiente soddisfi i seguenti requisiti minimi:

## Requisiti del server {#server-requirements}

- **WordPress:** versione 5.6 o superiore
- **WooCommerce:** versione 5.3 o superiore
- **PHP:** versione 7.4 o superiore (8.x consigliata)

Il rispetto di questi requisiti minimi garantisce compatibilità e stabilità tra il plugin WCPOS e il proprio ambiente WordPress. Consente inoltre di beneficiare delle funzionalità e dei miglioramenti più recenti offerti dal plugin.

## Requisiti del browser {#browser-requirements}

- **Browser web moderno:** il plugin WCPOS è progettato per funzionare con i browser web moderni, tra cui Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Assicurarsi di utilizzare l'ultima versione stabile del browser preferito.
- **JavaScript abilitato:** il plugin WCPOS si basa su JavaScript per le sue funzionalità interattive. Assicurarsi che JavaScript sia abilitato nelle impostazioni del browser.
- **Nessuna modalità di navigazione privata:** Il browser non deve essere in modalità di navigazione privata poiché potrebbe limitare l'accesso al database IndexedDB (IDB) del browser. La modalità di navigazione privata può impedire il corretto funzionamento di alcune funzionalità del plugin WCPOS.

## Installazione del plugin {#installing-the-plugin}

1. Andare su `WP Admin > Plugin > Add Plugin`
2. Cercare "wcpos"
3. Fare clic su **Installa ora**, poi su **Attiva**

In alternativa, è possibile scaricare il plugin da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e caricarlo manualmente.

## Accesso al POS {#accessing-the-pos}

Dopo l'attivazione, è possibile accedere al POS all'indirizzo:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Soddisfando i requisiti minimi e utilizzando un browser web compatibile, è possibile utilizzare il plugin WCPOS in modo ottimale e sfruttarne appieno il potenziale per gestire le operazioni del punto vendita in modo efficiente.

Per ulteriori domande o necessità di assistenza, non esitare a contattare il nostro team di supporto pubblicando un messaggio nella [chat Discord](https://wcpos.com/discord) o inviando un'email a support@wcpos.com.
