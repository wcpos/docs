Per garantire il corretto funzionamento e le prestazioni ottimali del plugin WCPOS, assicurati che il tuo ambiente soddisfi i seguenti requisiti minimi:

## Requisiti del Server

- **WordPress:** Versione 5.6 o superiore
- **WooCommerce:** Versione 5.3 o superiore
- **PHP:** Versione 7.2 o superiore

Soddisfare questi requisiti minimi garantisce compatibilità e stabilità tra il plugin WCPOS e il tuo ambiente WordPress. Inoltre, garantisce che tu possa beneficiare delle ultime funzionalità e miglioramenti forniti dal plugin.

## Requisiti del Browser

- **Browser web moderno:** Il plugin WCPOS è progettato per funzionare con browser web moderni, tra cui Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Assicurati di utilizzare l'ultima versione stabile del tuo browser preferito.
- **JavaScript attivato:** Il plugin WCPOS si basa su JavaScript per le sue funzionalità interattive. Assicurati che JavaScript sia attivato nelle impostazioni del tuo browser.
- **Nessuna modalità di navigazione privata:** Il browser non deve essere in modalità di navigazione privata poiché potrebbe limitare l'accesso al database IndexedDB (IDB) del browser. La modalità di navigazione privata può impedire il corretto funzionamento di alcune funzionalità nel plugin WCPOS.

## Installazione del Plugin

1. Vai a `WP Admin > Plugin > Aggiungi Nuovo`
2. Cerca "WooCommerce POS"
3. Clicca su **Installa Ora**, poi su **Attiva**

In alternativa, puoi scaricare il plugin da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e caricarlo manualmente.

## Accesso al POS

Dopo l'attivazione, puoi accedere al POS su:

- **Web:** `iltuodominio.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Soddisfacendo i requisiti minimi e utilizzando un browser web compatibile, puoi avere un'esperienza senza interruzioni utilizzando il plugin WCPOS e sfruttare appieno il suo potenziale per gestire efficientemente le tue operazioni di punto vendita.
