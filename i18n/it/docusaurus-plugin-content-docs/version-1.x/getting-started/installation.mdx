Per garantire il corretto funzionamento e le prestazioni ottimali del plugin WCPOS, ti preghiamo di assicurarti che il tuo ambiente soddisfi i seguenti requisiti minimi:

## Requisiti del Server {#server-requirements}

- **WordPress:** Versione 5.6 o superiore
- **WooCommerce:** Versione 5.3 o superiore
- **PHP:** Versione 7.2 o superiore

Soddisfare questi requisiti minimi assicura compatibilità e stabilità tra il plugin WCPOS e il tuo ambiente WordPress. Garantisce inoltre che tu possa beneficiare delle ultime funzionalità e dei miglioramenti forniti dal plugin.

## Requisiti del Browser {#browser-requirements}

- **Browser web moderno:** Il plugin WCPOS è progettato per funzionare con browser web moderni, tra cui Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Ti preghiamo di assicurarti che stai utilizzando l'ultima versione stabile del tuo browser preferito.
- **JavaScript attivato:** Il plugin WCPOS si basa su JavaScript per le sue funzionalità interattive. Assicurati che JavaScript sia attivato nelle impostazioni del tuo browser.
- **No modalità di navigazione privata:** Il browser non dovrebbe essere in modalità di navigazione privata poiché potrebbe limitare l'accesso al database IndexedDB (IDB) del browser. La modalità di navigazione privata può impedire il corretto funzionamento di alcune funzionalità nel plugin WCPOS.

## Installazione del Plugin {#installing-the-plugin}

1. Vai su `WP Admin > Plugin > Aggiungi Nuovo`
2. Cerca "WooCommerce POS"
3. Clicca **Installa Ora**, poi **Attiva**

In alternativa, puoi scaricare il plugin da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e caricarlo manualmente.

## Accesso al POS {#accessing-the-pos}

Dopo l'attivazione, puoi accedere al POS su:

- **Web:** `yourdomain.com/pos`
- **Desktop:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Mobile:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Soddisfacendo i requisiti minimi e utilizzando un browser web compatibile, puoi avere un'esperienza senza interruzioni nell'utilizzo del plugin WCPOS e sfruttare appieno il suo potenziale per gestire in modo efficiente le operazioni del tuo punto vendita.

Se hai ulteriori domande o hai bisogno di assistenza, ti preghiamo di non esitare a contattare il nostro team di supporto scrivendo nel [chat di Discord](https://wcpos.com/discord) o inviandoli via email a support@wcpos.com.
