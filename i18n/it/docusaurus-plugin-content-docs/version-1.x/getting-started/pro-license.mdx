:::info Caratteristica Pro
[WCPOS Pro](https://wcpos.com/pro) è un plugin a pagamento che aggiunge funzionalità al gratuito [plugin WCPOS](https://wordpress.org/plugins/woocommerce-pos/).
:::

## Cosa è incluso in Pro?

WCPOS Pro sblocca funzionalità aggiuntive:

- **Schermata Prodotti** - Modifica i livelli di stock e i prezzi direttamente nel POS
- **Schermata Ordini** - Visualizza e gestisci la cronologia degli ordini
- **Schermata Clienti** - Aggiungi e modifica le informazioni sui clienti
- **Schermata Report** - Genera report di vendita a fine giornata
- **Gateway di Pagamento Aggiuntivi** - Stripe Terminal, SumUp e gateway personalizzati

## Installazione

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi qui sotto per installare e attivare il plugin:

1. Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Sotto **Download**, clicca sul link di download e salva il plugin sul tuo desktop.
3. Poi vai al tuo sito, accedi e vai su `WP Admin > Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip del plugin dal tuo desktop e attivalo.
5. Successivamente, vai su `WP Admin > POS > Impostazioni > Licenza` e inserisci la tua Chiave di Licenza e l'Email di Licenza per completare l'attivazione.

![I miei Download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Puoi scaricare WCPOS Pro dalla tua pagina account")

## Aggiornamenti

Quando nuove versioni sono pronte, appariranno nel tuo dashboard degli Aggiornamenti come qualsiasi altro plugin. Puoi anche scaricare l'ultima versione dalla tua [pagina account](https://wcpos.com/my-account/).
