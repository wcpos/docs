:::info Funzionalità Pro
[WCPOS Pro](https://wcpos.com/pro) è un plugin a pagamento che aggiunge funzionalità al plugin gratuito [WCPOS](https://wordpress.org/plugins/woocommerce-pos/).
:::

## Cosa è Incluso in Pro?

WCPOS Pro sblocca funzionalità aggiuntive:

- **Schermata Prodotti** - Modifica i livelli di stock e i prezzi direttamente nel POS
- **Schermata Ordini** - Visualizza e gestisci la cronologia degli ordini
- **Schermata Clienti** - Aggiungi e modifica le informazioni dei clienti
- **Schermata Report** - Genera report delle vendite di fine giornata
- **Gateway di Pagamento Aggiuntivi** - Stripe Terminal, SumUp e gateway personalizzati

## Installazione

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi seguenti per installare e attivare il plugin:

1. Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Sotto **Downloads**, fai clic sul link di download e salva il plugin sul tuo desktop.
3. Poi vai sul tuo sito, effettua l'accesso e vai su `WP Admin > Plugin > Aggiungi Nuovo > Carica Plugin`
4. Carica il file zip del plugin dal tuo desktop e attivalo.
5. Successivamente, vai su `WP Admin > POS > Impostazioni > Licenza` e inserisci la tua Chiave Licenza e l'Email della Licenza per completare l'attivazione.

![I Miei Download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "Puoi scaricare WCPOS Pro nella pagina del tuo account")

## Aggiornamenti

Quando sono pronti nuove versioni, appariranno nel dashboard degli Aggiornamenti come qualsiasi altro plugin. Puoi anche scaricare l'ultima versione dalla tua [pagina dell'account](https://wcpos.com/my-account/).
