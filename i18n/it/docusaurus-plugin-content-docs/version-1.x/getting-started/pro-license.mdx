:::info Funzionalità Pro
[WCPOS Pro](https://wcpos.com/pro) è un plugin a pagamento che alimenta le schermate di gestione, i gateway di pagamento aggiuntivi e altre funzionalità disponibili solo nella versione Pro di WCPOS.

**A partire dalla versione v1.8+, Pro è un plugin autonomo** — non è necessario installare anche il [plugin WCPOS](https://wordpress.org/plugins/woocommerce-pos/) gratuito.
:::

## Cosa include Pro? {#whats-included-in-pro}

WCPOS Pro sblocca funzionalità aggiuntive:

- **Schermata Prodotti** - Modifica livelli di scorte e prezzi direttamente nel POS
- **Schermata Ordini** - Visualizza e gestisci la cronologia degli ordini
- **Schermata Clienti** - Aggiungi e modifica le informazioni dei clienti
- **Schermata dei report** - Genera report sulle vendite di fine giornata
- **Gateway di pagamento aggiuntivi** - Stripe Terminal, SumUp e gateway personalizzati

## Installazione {#installation}

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi riportati di seguito per installare e attivare il plugin:

:::note Aggiornamento da versioni precedenti?
Se in precedenza avevi installato sia il plugin gratuito WCPOS sia un componente aggiuntivo Pro precedente, **disattiva ed elimina il plugin gratuito** prima di attivare il nuovo plugin Pro autonomo. Eseguire entrambi affiancati può causare errori di attivazione come *"Invalid response fetching remote state"* o errori fatali *"Class not found"*.
:::

1. Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. In **Download**, fai clic sul link di download e salva il plugin sul desktop.
3. Quindi vai al tuo sito, accedi e vai a `WP Admin > Plugin > Add Plugin > Carica plugin`
4. Carica il file zip del plugin dal desktop e attivalo.
5. Successivamente, vai a `WP Admin > POS > Impostazioni > License` e inserisci la chiave di licenza e l'email della licenza per completare l'attivazione.

![I miei download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "È possibile scaricare WCPOS Pro dalla pagina del proprio account")

## Aggiornamenti {#updates}

Quando saranno disponibili nuove versioni, appariranno nella bacheca Aggiornamenti come qualsiasi altro plugin. È anche possibile scaricare la versione più recente dalla [pagina dell'account](https://wcpos.com/my-account/).

## Risoluzione dei problemi {#troubleshooting}

### Errori di attivazione {#activation-failures}

- **"Chiave di licenza non valida"** — verificare che non ci siano spazi finali quando si incolla la chiave e confermare che l'indirizzo email corrisponda a quello usato per l'acquisto.
- **"Limite di licenza raggiunto"** — per impostazione predefinita una licenza si attiva su **2 domini** (pensati per un sito di produzione e un sito di staging). Tenere presente che **i sottodomini contano come attivazioni separate** — `example.com`, `staging.example.com` e `dev.example.com` sono tre. Questo spesso crea problemi con host come Pantheon, che prevedono ambienti dev/test/live obbligatori. Disattivare i siti inutilizzati dalla [pagina dell'account](https://wcpos.com/my-account/) oppure inviare un'email a **support@wcpos.com** — attivazioni aggiuntive vengono normalmente concesse su richiesta.

### Licenza scaduta {#expired-licence}

- Quando una licenza scade, le funzionalità Pro restano attive, ma non si ricevono aggiornamenti.
- Rinnovare dalla [pagina dell'account](https://wcpos.com/my-account/) — la chiave esistente rimane la stessa.

### Trasferimenti di dominio {#domain-transfers}

- Se il sito è stato spostato su un nuovo dominio, disattiva prima la licenza sul vecchio dominio, quindi riattivala su quello nuovo.
- Se il vecchio dominio non è più accessibile, contatta **support@wcpos.com** per reimpostare l'attivazione.

### Connettività del server delle licenze {#licence-server-connectivity}

- L'attivazione richiede che il server WordPress possa raggiungere il server delle licenze WCPOS.
- Se l'attivazione non riesce senza mostrare messaggi, verifica che le richieste HTTPS in uscita non siano bloccate dal firewall dell'host o da un plugin di sicurezza.
- Alcuni host gestiti bloccano per impostazione predefinita le chiamate API esterne: contatta il tuo host per inserire `wcpos.com` nella lista consentita.

### Verifica dello stato della licenza {#checking-licence-status}

- Vai a `WP Admin > POS > Impostazioni > License` per vedere lo stato di attivazione corrente.
- La [pagina dell'account](https://wcpos.com/my-account/) mostra tutti i siti attivi e la data di scadenza.

## Acquistare una licenza {#purchase-a-license}

Visita [wcpos.com/pro](https://wcpos.com/pro) per acquistare una licenza Pro.

:::note Acquisto e aggiornamento avvengono sul sito web
Non c'è alcun pulsante "aggiorna a Pro" nell'app: acquisti, aggiornamenti del piano e rinnovi sono tutti gestiti su [wcpos.com](https://wcpos.com/pro) e nella tua [pagina dell'account](https://wcpos.com/my-account/). Dopo l'acquisto, installa Pro seguendo i [passaggi precedenti](#installation).
:::
