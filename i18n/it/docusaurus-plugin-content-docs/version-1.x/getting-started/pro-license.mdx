:::info Funzionalità Pro
[WCPOS Pro](https://wcpos.com/pro) è un plugin a pagamento che abilita le schermate di gestione, i gateway di pagamento aggiuntivi e altre funzionalità riservate alla versione Pro in WCPOS.

**A partire dalla v1.8+, Pro è un plugin autonomo** — non è necessario avere il [plugin WCPOS](https://wordpress.org/plugins/woocommerce-pos/) gratuito installato insieme.
:::

## Cosa include la versione Pro? {#whats-included-in-pro}

WCPOS Pro sblocca funzionalità aggiuntive:

- **Schermata Prodotti** - Modifica le giacenze e i prezzi direttamente nel POS
- **Schermata Ordini** - Visualizza e gestisci lo storico degli ordini
- **Schermata Clienti** - Aggiungi e modifica le informazioni dei clienti
- **Schermata Report** - Genera report di vendita di fine giornata
- **Gateway di pagamento aggiuntivi** - Stripe Terminal, SumUp e gateway personalizzati

## Installazione {#installation}

Se hai acquistato una licenza per WCPOS Pro, segui i passaggi seguenti per installare e attivare il plugin:

:::note Aggiornamento da versioni precedenti?
Se in precedenza avevi installato sia il plugin gratuito WCPOS sia un vecchio componente aggiuntivo Pro, **disattiva ed elimina il plugin gratuito** prima di attivare il nuovo plugin Pro standalone. L'esecuzione contemporanea di entrambi può causare errori di attivazione come *"Invalid response fetching remote state"* o errori fatali *"Class not found"*.
:::

1. Vai a: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. Nella sezione **Download**, fai clic sul link di download e salva il plugin sul desktop.
3. Quindi vai sul tuo sito, effettua l'accesso e vai su `WP Admin > Plugin > Add Plugin > Carica plugin`
4. Carica il file zip del plugin dal desktop e attivalo.
5. Quindi vai su `WP Admin > POS > Impostazioni > License` e inserisci la chiave di licenza e l'e-mail di licenza per completare l'attivazione.

![I miei download](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "You can download WCPOS Pro on your account page")

## Aggiornamenti {#updates}

Quando sono disponibili nuove versioni, queste appariranno nella dashboard degli aggiornamenti come qualsiasi altro plugin. È inoltre possibile scaricare l'ultima versione dalla [pagina del proprio account](https://wcpos.com/my-account/).

## Risoluzione dei problemi {#purchase-a-license}

### Errori di attivazione {#activation-failures}

- **"Invalid licence key"** — verificare che non siano presenti spazi aggiuntivi durante l'incollaggio e confermare che l'indirizzo e-mail corrisponda a quello utilizzato per l'acquisto.
- **"Licence limit reached"** — ogni licenza copre un numero definito di siti. Disattivare i siti non utilizzati dalla [pagina del proprio account](https://wcpos.com/my-account/) o aggiornare il piano.

### Licenza scaduta {#expired-licence}

- Quando una licenza scade, le funzionalità Pro rimangono attive ma non si riceveranno aggiornamenti.
- Rinnovare dalla [pagina del proprio account](https://wcpos.com/my-account/) — la chiave esistente rimane invariata.

### Trasferimenti di dominio {#domain-transfers}

- Se il sito è stato spostato su un nuovo dominio, disattivare prima la licenza sul vecchio dominio, quindi riattivarla su quello nuovo.
- Se il vecchio dominio non è più accessibile, contattare **support@wcpos.com** per reimpostare l'attivazione.

### Connettività al server licenze {#licence-server-connectivity}

- L'attivazione richiede che il server WordPress possa raggiungere il server licenze di WCPOS.
- Se l'attivazione non riesce senza errori visibili, verificare che le richieste HTTPS in uscita non siano bloccate dal firewall dell'hosting o da un plugin di sicurezza.
- Alcuni hosting gestiti bloccano le chiamate API esterne per impostazione predefinita — contattare il proprio provider di hosting per autorizzare `wcpos.com`.

### Verifica dello stato della licenza {#checking-licence-status}

- Accedere a `WP Admin > POS > Impostazioni > License` per visualizzare lo stato di attivazione corrente.
- La [pagina del proprio account](https://wcpos.com/my-account/) mostra tutti i siti attivi e la data di scadenza.

## Acquistare una licenza {#purchase-a-license}

Visitare [wcpos.com/pro](https://wcpos.com/pro) per acquistare una licenza Pro.
