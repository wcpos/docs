## Re-installazione di una Versione Precedente {#re-installing-a-previous-version}

Se hai bisogno di tornare a una versione precedente del plugin WCPOS, segui questi passaggi:

1. Visita la pagina di WordPress.org [Visualizzazione Avanzata](https://wordpress.org/plugins/woocommerce-pos/advanced/) per WCPOS.

2. Scorri verso il basso fino alla sezione **Opzioni Avanzate** e individua il menu a discesa **Versioni Precedenti**.

3. Apri il menu a discesa e vedrai un elenco delle versioni del plugin disponibili. Trova la versione che desideri installare e fai clic sul pulsante **Scarica** accanto ad essa. Salva il file scaricato sul tuo desktop o in una posizione comoda.

4. Accedi al tuo pannello di amministrazione di WordPress.

5. Naviga nella pagina **Plugins** facendo clic su **Plugins** nel menu a sinistra.

6. Disattiva ed elimina la versione attuale del plugin WCPOS. Questo rimuoverà il plugin dalla tua installazione di WordPress. Non preoccuparti, i tuoi dati non andranno persi.

7. Fai clic sul pulsante **Aggiungi Nuovo** nella parte superiore della pagina.

8. Nella pagina **Aggiungi Plugins**, fai clic sul pulsante **Carica Plugin**.

9. Scegli il file del plugin precedentemente scaricato dal tuo desktop o dalla posizione in cui lo hai salvato.

<div class="image-container">
  <Image
    alt="Carica Plugin"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"
  />
  <p class="image-caption">Carica plugin nel pannello di amministrazione di WordPress</p>
</div>

10. Fai clic sul pulsante **Installa Ora** e attendi che l'installazione sia completata.

11. Dopo che l'installazione è terminata, fai clic sul pulsante **Attiva** per attivare la versione precedente del plugin WCPOS.

Ecco fatto! Hai reinstallato con successo una versione precedente del plugin WCPOS.

Se riscontri problemi durante il processo di re-installazione o hai bisogno di ulteriore assistenza, contatta il nostro team di supporto all'indirizzo support@wcpos.com.
