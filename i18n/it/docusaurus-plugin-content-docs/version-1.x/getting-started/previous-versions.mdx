## Reinstallazione di una Versione Precedente

Se hai bisogno di tornare a una versione precedente del plugin WCPOS, segui questi passaggi:

1. Visita la pagina WordPress.org [Visualizzazione Avanzata](https://wordpress.org/plugins/woocommerce-pos/advanced/) per WCPOS.

2. Scorri fino alla sezione **Opzioni Avanzate** e individua il menu di selezione **Versioni Precedenti**.

3. Apri il menu di selezione e vedrai un elenco delle versioni del plugin disponibili. Trova la versione che desideri installare e fai clic sul pulsante **Download** accanto ad essa. Salva il file scaricato sul desktop o in un percorso comodo.

4. Accedi al pannello di amministrazione di WordPress.

5. Vai alla pagina **Plugin** facendo clic su **Plugin** nel menu a sinistra.

6. Disattiva ed elimina la versione corrente del plugin WCPOS. Questo rimuover√† il plugin dalla tua installazione WordPress. Non preoccuparti, i tuoi dati non andranno persi.

7. Fai clic sul pulsante **Aggiungi Nuovo** in alto nella pagina.

8. Nella pagina **Aggiungi Plugin**, fai clic sul pulsante **Carica Plugin**.

9. Scegli il file del plugin scaricato in precedenza dal tuo desktop o dal percorso in cui lo hai salvato.

10. Fai clic sul pulsante **Installa Ora** e attendi il completamento dell'installazione.

11. Al termine dell'installazione, fai clic sul pulsante **Attiva** per attivare la versione precedente del plugin WCPOS.

Ecco fatto! Hai reinstallato con successo una versione precedente del plugin WCPOS.

Se riscontri problemi durante il processo di reinstallazione o hai bisogno di ulteriore assistenza, contatta il nostro team di supporto all'indirizzo support@wcpos.com.
