## Re-installare una versione precedente {#re-installing-a-previous-version}

Se hai bisogno di tornare a una versione precedente del plugin WCPOS, segui questi passaggi:

1. Visita la pagina di WordPress.org [Visualizzazione Avanzata](https://wordpress.org/plugins/woocommerce-pos/advanced/) per WCPOS.

2. Scorri verso il basso fino alla sezione **Opzioni Avanzate** e individua il menu di selezione **Versioni Precedenti**.

3. Apri il menu di selezione e vedrai un elenco delle versioni disponibili del plugin. Trova la versione che desideri installare e fai clic sul pulsante **Scarica** accanto ad essa. Salva il file scaricato sul tuo desktop o in una posizione conveniente.

4. Accedi al tuo dashboard di WordPress Admin.

5. Naviga alla pagina **Plugin** facendo clic su **Plugin** nel menu a sinistra.

6. Disattiva ed elimina la versione attuale del plugin WCPOS. Questo rimuoverà il plugin dalla tua installazione di WordPress. Non preoccuparti, i tuoi dati non andranno persi.

7. Fai clic sul pulsante **Aggiungi Plugin** nella parte superiore della pagina.

8. Nella pagina **Aggiungi Plugin**, fai clic sul pulsante **Carica Plugin**.

9. Scegli il file del plugin precedentemente scaricato dal tuo desktop o dalla posizione in cui lo hai salvato.

<div class="image-container">
  <Image
    alt="Carica Plugin"
    img="https://wcpos.com/wp-content/uploads/2023/06/upload-plugin.png"

  <p class="image-caption">Carica plugin nell'Amministratore di WordPress</p>
</div>

10. Fai clic sul pulsante **Installa Ora** e attendi che l'installazione venga completata.

11. Dopo che l'installazione è stata completata, fai clic sul pulsante **Attiva** per attivare la versione precedente del plugin WCPOS.

Questo è tutto! Hai re-installato con successo una versione precedente del plugin WCPOS.

Se riscontri problemi durante il processo di re-installazione o hai bisogno di ulteriore assistenza, ti preghiamo di contattare il nostro team di supporto all'indirizzo support@wcpos.com.
