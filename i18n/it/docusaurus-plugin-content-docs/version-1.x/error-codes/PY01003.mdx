# PY01003: Carta Scaduta

## Cosa Significa

La carta di pagamento ha superato la sua data di scadenza e non può più essere utilizzata per transazioni.

## Cause Comuni

- **Carta scaduta** — La data di validità della carta è passata
- **Scadenza inserita in modo errato** — La data di scadenza è stata inserita in modo errato
- **Nuova carta non attivata** — Carta sostitutiva ricevuta ma non ancora attivata

## Come Risolvere

### 1. Verificare la Data di Scadenza

Guardare la carta:
- Trovare la data di scadenza (formato MM/AA)
- Confrontare con la data corrente
- Verificare che la data inserita corrisponda a quella sulla carta

### 2. Utilizzare una Carta Diversa

Se la carta è effettivamente scaduta:
- Utilizzare una carta diversa e valida
- Usare contanti o un metodo di pagamento alternativo
- Chiedere se il cliente ha una carta sostitutiva

### 3. Ottenere la Carta Aggiornata

Il titolare della carta dovrebbe:
- Controllare la posta per una carta sostitutiva
- Attivare la nuova carta se ricevuta
- Contattare la propria banca se non è arrivata alcuna sostituzione

### 4. Verificare l'Inserimento

Controllare nuovamente l'inserimento della scadenza:
- Formato MM/AA
- Mese e anno corretti
- Nessun errore di battitura

## Suggerimenti per l'Assistenza Clienti

Quando una carta è scaduta:
- Informare cortesemente il cliente
- Suggerire di verificare l'arrivo di una carta sostitutiva
- Offrire metodi di pagamento alternativi
- Essere comprensivi — capita a tutti
