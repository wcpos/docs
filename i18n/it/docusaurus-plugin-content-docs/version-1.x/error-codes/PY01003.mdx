# PY01003: Carta Scaduta {#py01003-card-expired}

## Cosa Significa {#what-this-means}

La carta di pagamento ha superato la sua data di scadenza e non può più essere utilizzata per transazioni.

## Cause Comuni {#common-causes}

- **Carta scaduta** — La data di validità della carta è scaduta
- **Scadenza inserita errata** — La data di scadenza è stata inserita in modo errato
- **Nuova carta non attivata** — Carta di sostituzione ricevuta ma non ancora attivata

## Come Risolvere {#how-to-fix}

### 1. Controllare la Data di Scadenza {#1-check-the-expiration-date}

Controlla la carta:
- Trova la data di scadenza (formato MM/AA)
- Confronta con la data attuale
- Verifica che la data inserita corrisponda a quella della carta

### 2. Utilizzare una Carta Diversa {#2-use-a-different-card}

Se la carta è effettivamente scaduta:
- Utilizza un'altra carta valida
- Utilizza contante o un metodo di pagamento alternativo
- Chiedi se il cliente ha una carta di sostituzione

### 3. Ottenere la Carta Aggiornata {#3-get-the-updated-card}

Il titolare della carta dovrebbe:
- Controllare la posta per una carta di sostituzione
- Attivare la nuova carta se ricevuta
- Contattare la propria banca se non è arrivata alcuna sostituzione

### 4. Verificare l'Inserimento {#4-verify-entry}

Controllare attentamente l'inserimento della scadenza:
- Formato MM/AA
- Mese e anno corretti
- Nessun errore di battitura

## Suggerimenti per il Servizio Clienti {#customer-service-tips}

Quando una carta è scaduta:
- Informare cortesemente il cliente
- Suggerire di controllare per una carta di sostituzione
- Offrire opzioni di pagamento alternative
- Essere comprensivi — questo succede a tutti
