# API05002: Plugin WCPOS Non Trovato {#api05002-wcpos-plugin-not-found}

## Cosa Significa {#what-this-means}

Il plugin WCPOS (WooCommerce POS) non è installato o non è attivo sul tuo sito WordPress. L'applicazione POS richiede questo plugin per comunicare con WooCommerce.

## Cause Comuni {#common-causes}

- **Plugin non installato** — Il plugin non è stato mai installato
- **Plugin disattivato** — Il plugin è stato disabilitato
- **Plugin cancellato** — I file del plugin sono stati rimossi
- **Sito sbagliato** — Connesso a un sito senza WCPOS

## Come Risolvere {#how-to-fix}

### 1. Installa il Plugin {#1-install-the-plugin}

Se non è installato:
1. Vai su WordPress Admin → Plugin → Aggiungi Nuovo
2. Cerca "WooCommerce POS"
3. Installa e attiva il plugin

Oppure scarica da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Attiva il Plugin {#2-activate-the-plugin}

Se installato ma non attivo:
1. Vai su Plugin → Plugin Installati
2. Trova "WooCommerce POS"
3. Clicca "Attiva"

### 3. Controlla gli Errori {#3-check-for-errors}

Se il plugin non si attiva:
- Controlla errori PHP nel tuo registro errori
- Verifica che WooCommerce sia installato e attivo
- Controlla i requisiti della versione PHP
- Cerca conflitti tra plugin

### 4. Verifica l'URL {#4-verify-the-url}

Assicurati che il POS sia configurato per connettersi al sito WordPress corretto:
- Controlla l'URL nelle impostazioni del POS
- Verifica che corrisponda alla tua installazione di WordPress

### 5. Reinstalla se Necessario {#5-reinstall-if-necessary}

Se il plugin è danneggiato:
1. Disattiva e cancella il plugin
2. Reinstalla da WordPress.org
3. Attiva il plugin
