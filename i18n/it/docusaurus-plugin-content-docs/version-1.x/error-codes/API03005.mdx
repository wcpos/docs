# API03005: Limite di Velocità Superato {#api03005-rate-limit-exceeded}

## Cosa Significa {#what-this-means}

Hai effettuato troppe richieste al server in un breve periodo. Il rate limiting protegge il server dall'essere sopraffatto.

## Cause Comuni {#common-causes}

- **Operazioni di sincronizzazione rapide** — Dispositivi multipli che sincronizzano simultaneamente
- **Aggressivo refresh** — Refresh dei dati troppo frequentemente
- **Plugin di sicurezza** — Un plugin sta applicando limiti alle richieste
- **Limiti di hosting** — Il tuo piano di hosting ha limiti di velocità API

## Mappatura degli Errori del Server {#server-error-mapping}

Questo codice di errore viene attivato quando il server restituisce:

| Codice Server | Origine |
|---------------|---------|
| HTTP 429 | Qualsiasi risposta del server (fallback) |

## Come Risolvere {#how-to-fix}

### 1. Aspetta e Riprova {#1-wait-and-retry}

I limiti di velocità sono temporanei:
- Aspetta 1-5 minuti
- Il POS riproverà automaticamente
- Evita di rinfrescare manualmente ripetutamente

### 2. Ridurre le Operazioni Concomitanti {#2-reduce-concurrent-operations}

Se utilizzi più terminali POS:
- Distribuisci la sincronizzazione iniziale tra i dispositivi
- Evita di rinfrescare tutti i dispositivi contemporaneamente
- Lascia completare uno prima di iniziarne un altro

### 3. Controlla i Plugin di Sicurezza {#3-check-security-plugins}

Plugin come Wordfence potrebbero bloccare richieste rapide:
- Inserisci l'applicazione POS nella whitelist
- Inserisci gli indirizzi IP dei tuoi dispositivi POS nella whitelist
- Regola le soglie di rate limiting

### 4. Rivedi i Limiti di Hosting {#4-review-hosting-limits}

Alcuni host limitano le richieste API:
- Controlla i dettagli del tuo piano di hosting
- Considera di aggiornarlo se i limiti sono troppo bassi
- Contatta il supporto per aumentare i limiti

### 5. Ottimizza la Frequenza di Sincronizzazione {#5-optimise-sync-frequency}

Nelle impostazioni del POS:
- Regola gli intervalli di refresh automatico
- Usa la sincronizzazione manuale quando appropriato
- Evita refresh dei dati non necessari
