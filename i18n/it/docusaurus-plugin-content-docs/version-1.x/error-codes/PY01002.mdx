# PY01002: Fondi Insufficienti

## Cosa Significa

Il pagamento è stato rifiutato perché non ci sono abbastanza soldi nel conto per coprire la transazione.

## Cause Comuni

- **Saldo del conto basso** — Fondi non sufficienti disponibili
- **Transazioni in attesa** — Altre transazioni in attesa che riducono il saldo disponibile
- **Limite di credito raggiunto** — Per le carte di credito, il limite è esaurito
- **Fondi in sospeso** — I fondi sono in sospeso per altre transazioni

## Come Risolvere

### 1. Usa un Metodo di Pagamento Differente

La soluzione più veloce:
- Prova una carta diversa
- Usa contante
- Dividi il pagamento tra più metodi (se supportato)

### 2. Riduci l'Importo dell'Ordine

Se il pagamento parziale è possibile:
- Rimuovi alcuni articoli
- Applica sconti o coupon
- Completa una transazione più piccola

### 3. Azioni del Cliente

Il cliente potrebbe dover:
- Trasferire fondi sul conto
- Aspettare che le transazioni in attesa vengano elaborate
- Estinguere il saldo della carta di credito
- Richiedere un aumento del limite di credito

### 4. Dividi la Transazione

Se il tuo POS supporta pagamenti divisi:
- Paga una parte con una carta
- Paga il resto con un altro metodo

## Nota sulla Privacy

Quando comunichi ciò ai clienti:
- Sii discreto riguardo al motivo specifico del rifiuto
- Dì semplicemente "la carta è stata rifiutata"
- Permetti loro di provare un altro metodo di pagamento in privato
