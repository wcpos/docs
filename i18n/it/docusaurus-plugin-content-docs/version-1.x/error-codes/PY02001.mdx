# PY02001: Errore Gateway di Pagamento

## Cosa Significa

Si è verificato un errore durante la comunicazione con il gateway di pagamento. Il processore di pagamento non è riuscito a completare la richiesta.

## Cause Comuni

- **Interruzione del gateway** — Il servizio di pagamento sta riscontrando problemi
- **Errore di configurazione** — Le impostazioni del gateway non sono corrette
- **Credenziali API** — Chiavi API non valide o scadute
- **Problemi di rete** — Problemi di connessione al gateway

## Come Risolvere

### 1. Ripetere la Transazione

A volte i gateway hanno problemi temporanei:
- Attendere un momento
- Provare a elaborare nuovamente il pagamento
- Il problema potrebbe risolversi da solo

### 2. Verificare lo Stato del Gateway

Visitare la pagina di stato del proprio fornitore di pagamento:
- Stripe: status.stripe.com
- PayPal: Controllare la community PayPal o lo stato
- Square: status.squareup.com
- Verificare eventuali interruzioni note

### 3. Verificare la Configurazione del Gateway

In WooCommerce Admin:
1. Andare su WooCommerce → Impostazioni → Pagamenti
2. Fare clic sul proprio gateway di pagamento
3. Verificare che le chiavi API siano corrette
4. Controllare che la modalità test/live corrisponda alle proprie chiavi

### 4. Controllare le Credenziali API

Assicurarsi che le credenziali siano valide:
- Le chiavi API non siano scadute
- Utilizzare chiavi live per transazioni live
- Utilizzare chiavi test per transazioni di test
- Le credenziali corrispondano al conto corretto

### 5. Testare con un Gateway Diverso

Se disponibile:
- Provare a elaborare con un gateway alternativo
- Utilizzare il pagamento in contanti come soluzione alternativa
- Completare la vendita e riconciliare successivamente

## Per i Proprietari di Negozi

Problemi di configurazione comuni:
- Mescolanza di chiavi API di test e live
- Chiavi API di account diversi
- Metodi di pagamento disabilitati
- Credenziali scadute o revocate

Contattare il supporto del proprio fornitore di pagamento se il problema persiste.
