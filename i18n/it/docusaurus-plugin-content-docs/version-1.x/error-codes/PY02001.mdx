# PY02001: Errore del Gateway di Pagamento {#py02001-payment-gateway-error}

## Cosa Significa {#what-this-means}

Si è verificato un errore durante la comunicazione con il gateway di pagamento. Il processore di pagamento non è riuscito a completare la richiesta.

## Cause Comuni {#common-causes}

- **Inattività del gateway** — Il servizio di pagamento sta riscontrando problemi
- **Errore di configurazione** — Le impostazioni del gateway sono errate
- **Credenziali API** — Chiavi API non valide o scadute
- **Problemi di rete** — Problemi di connessione al gateway

## Come Risolvere {#how-to-fix}

### 1. Riprovare la Transazione {#1-retry-the-transaction}

A volte i gateway hanno problemi temporanei:
- Aspetta un momento
- Prova a riprocessare il pagamento
- Il problema potrebbe risolversi da solo

### 2. Controlla lo Stato del Gateway {#2-check-gateway-status}

Visita la pagina di stato del tuo fornitore di pagamenti:
- Stripe: status.stripe.com
- PayPal: Controlla la comunità di PayPal o lo stato
- Square: status.squareup.com
- Controlla eventuali interruzioni note

### 3. Verifica la Configurazione del Gateway {#3-verify-gateway-configuration}

In WooCommerce Admin:
1. Vai su WooCommerce → Impostazioni → Pagamenti
2. Clicca sul tuo gateway di pagamento
3. Verifica che le chiavi API siano corrette
4. Controlla che la modalità test/live corrisponda alle tue chiavi

### 4. Controlla le Credenziali API {#4-check-api-credentials}

Assicurati che le credenziali siano valide:
- Le chiavi API non siano scadute
- Utilizza chiavi live per transazioni live
- Utilizza chiavi di test per transazioni di test
- Le credenziali corrispondono all'account corretto

### 5. Prova con un Gateway Diverso {#5-test-with-a-different-gateway}

Se disponibile:
- Prova a elaborare con un gateway alternativo
- Usa il pagamento in contanti come fallback
- Completa la vendita e riconcilia in seguito

## Per i Proprietari di Negozi {#for-store-owners}

Problemi di configurazione comuni:
- Miscelazione di chiavi API di test e live
- Chiavi API da account diversi
- Metodi di pagamento disabilitati
- Credenziali scadute o revocate

Contatta il supporto del tuo fornitore di pagamenti se il problema persiste.
