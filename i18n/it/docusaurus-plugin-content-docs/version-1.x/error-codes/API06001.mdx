# API06001: Formato URL non valido

## Cosa Significa

L'URL configurato per il tuo sito WooCommerce non è in un formato valido. Gli URL devono seguire le convenzioni standard degli indirizzi web.

## Cause Comuni

- **Protocollo mancante** — L'URL non inizia con `http://` o `https://`
- **Errore di battitura nell'URL** — Errori di ortografia nel dominio
- **Caratteri non validi** — Caratteri speciali non consentiti negli URL
- **URL incompleto** — L'URL manca di parti come l'estensione del dominio

## Come Risolvere

### 1. Controlla il Formato dell'URL

Un URL valido dovrebbe apparire così:
- ✅ `https://iltuosito.com`
- ✅ `https://www.iltuosito.com`
- ✅ `https://negozio.iltuodominio.com`
- ❌ `iltuosito.com` (protocollo mancante)
- ❌ `https://iltuosito` (estensione mancante)
- ❌ `https://il tuo sito.com` (contiene spazio)

### 2. Includi il Protocollo

Includi sempre `https://` o `http://`:
- Preferito: `https://` (sicuro)
- Di riserva: `http://` (se HTTPS non è disponibile)

### 3. Rimuovi Caratteri Extra

Assicurati che l'URL non contenga:
- Slash finali (opzionale ma sii coerente)
- Spazi extra
- Caratteri speciali
- Parametri URL (a meno che non siano necessari)

### 4. Verifica che l'URL Funzioni

Prova l'URL in un browser:
1. Copia l'URL che stai utilizzando
2. Incollalo in un browser
3. Dovrebbe caricare il tuo sito WordPress

### 5. Aggiorna la Configurazione del POS

Inserisci l'URL corretto nelle impostazioni del POS:
- Corrispondi esattamente a ciò che funziona nel tuo browser
- Includi o escludi `www.` come il tuo sito utilizza

## Esempi

| Non valido | Valido |
|------------|--------|
| `iltuodominio.com` | `https://iltuodominio.com` |
| `https://il tuo dominio.com` | `https://iltuodominio.com` |
| `https://iltuodominio` | `https://iltuodominio.com` |
| `htp://iltuodominio.com` | `https://iltuodominio.com` |
