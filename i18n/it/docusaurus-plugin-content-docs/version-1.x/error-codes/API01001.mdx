# API01001: Timeout di Connessione

## Cosa Significa

L'applicazione POS ha tentato di connettersi al tuo server WooCommerce, ma il server ha impiegato troppo tempo per rispondere. Questo di solito indica problemi di latenza di rete o un server sovraccarico.

## Cause Comuni

- **Connessione internet lenta** — La tua connessione di rete potrebbe stare subendo alta latenza
- **Sovraccarico del server** — Il tuo server WordPress potrebbe essere sotto pressione a causa di troppe richieste
- **Richieste di dati grandi** — Richiedere troppi dati in una volta può causare timeout
- **Ritardi di firewall o proxy** — I dispositivi di sicurezza di rete potrebbero aggiungere latenza

## Come Risolvere

### 1. Controlla la tua Connessione Internet

Testa la tua velocità e stabilità di internet. Prova a caricare il tuo pannello di amministrazione di WooCommerce in un browser per vedere se risponde lentamente.

### 2. Riduci il Carico del Server

Se il tuo server è sovraccarico:
- Considera di aggiornare il tuo piano di hosting
- Disabilita temporaneamente i plugin non necessari
- Controlla se ci sono cron job pesanti o processi in background

### 3. Ottimizza il Tempo di Risposta del Server

Contatta il tuo provider di hosting riguardo:
- Abilitare PHP opcache
- Aumentare i limiti di memoria PHP
- Utilizzare la cache lato server

### 4. Aumenta le Impostazioni di Timeout

Se il tuo server ha veramente bisogno di più tempo:
- Controlla se il tuo provider di hosting consente di aumentare il tempo di esecuzione di PHP
- Considera di utilizzare un CDN per ridurre la latenza
