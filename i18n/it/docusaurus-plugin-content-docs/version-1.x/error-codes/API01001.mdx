# API01001: Timeout di Connessione

## Cosa Significa

L'applicazione POS ha tentato di connettersi al tuo server WooCommerce, ma il server ha impiegato troppo tempo a rispondere. Questo solitamente indica problemi di latenza di rete o un server sovraccarico.

## Cause Comuni

- **Connessione internet lenta** — La tua connessione di rete potrebbe avere un'alta latenza
- **Sovraccarico del server** — Il tuo server WordPress potrebbe stare elaborando troppe richieste
- **Richieste di dati di grandi dimensioni** — Richiedere troppi dati contemporaneamente può causare timeout
- **Ritardi del firewall o del proxy** — I dispositivi di sicurezza di rete potrebbero aggiungere latenza

## Come Risolvere

### 1. Controlla la Connessione Internet

Testa la velocità e la stabilità della tua connessione internet. Prova a caricare l'admin panel di WooCommerce in un browser per vedere se risponde lentamente.

### 2. Riduci il Carico del Server

Se il tuo server è sovraccarico:
- Considera l'upgrade del tuo piano di hosting
- Disabilita temporaneamente i plugin non necessari
- Verifica la presenza di job cron o processi in background pesanti

### 3. Ottimizza il Tempo di Risposta del Server

Contatta il tuo provider di hosting per:
- Abilitare PHP opcache
- Aumentare i limiti di memoria PHP
- Utilizzare la cache lato server

### 4. Aumentare le Impostazioni di Timeout

Se il tuo server ha effettivamente bisogno di più tempo:
- Verifica se il tuo provider di hosting consente di aumentare il tempo di esecuzione PHP
- Considera l'uso di una CDN per ridurre la latenza
