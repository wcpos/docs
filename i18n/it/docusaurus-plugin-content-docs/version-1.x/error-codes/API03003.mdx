# API03003: Valore del parametro non valido {#api03003-invalid-parameter-value}

## Cosa significa {#what-this-means}

Uno dei valori nella tua richiesta non è valido. Il formato, il tipo o il valore dei dati non corrispondono a ciò che il server si aspetta.

## Cause comuni {#common-causes}

- **Tipo di dato errato** — Testo dove è atteso un numero (o viceversa)
- **Fuori intervallo** — Il valore supera i limiti consentiti
- **Formato non valido** — Email, telefono o altri campi formattati sono errati
- **Riferimento non valido** — Riferimento a un ID che non esiste

## Mappatura degli errori del server {#server-error-mapping}

Questo codice di errore viene attivato quando il server restituisce:

| Codice Server | Sorgente |
|---------------|----------|
| `rest_invalid_param` | WordPress REST API |
| `woocommerce_rest_invalid_id` | WooCommerce REST API |

## Come risolvere {#how-to-fix}

### 1. Controlla i valori di input {#1-check-input-values}

Esamina i dati che stai inviando:
- **Prezzi** — Devono essere numeri validi
- **Quantità** — Devono essere interi positivi
- **Email** — Deve essere in un formato email valido
- **ID** — Deve riferirsi a record esistenti

### 2. Cerca errori specifici nei campi {#2-look-for-specific-field-errors}

L'errore potrebbe indicare quale campo è non valido:
- Controlla i dettagli del messaggio di errore
- Correggi il campo specifico
- Ripeti l'azione

### 3. Verifica che i dati di riferimento esistano {#3-verify-referenced-data-exists}

Se fai riferimento a prodotti, clienti o altri record:
- Assicurati che l'elemento esista in WooCommerce
- Controlla se è stato recentemente eliminato
- Sincronizza i dati per aggiornare i record locali

### 4. Controlla i vincoli dei campi {#4-check-field-constraints}

WooCommerce potrebbe avere vincoli:
- Valori massimi/minimi
- Formati richiesti
- Opzioni consentite per i campi selezionati

### 5. Cancella e reinserisci {#5-clear-and-re-enter}

A volte i dati si corrompono:
- Cancella il campo problematico
- Reinserisci il valore da zero
- Evita di copiare e incollare da altre fonti
