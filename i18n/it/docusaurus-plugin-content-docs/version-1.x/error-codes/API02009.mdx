# API02009: Token di Aggiornamento Scaduto

## Cosa Significa

Il tuo token di aggiornamento ha superato la sua durata e non può più essere utilizzato per ottenere nuovi token di accesso. A differenza dei token di accesso (che scadono rapidamente), i token di aggiornamento di solito durano giorni o settimane, ma alla fine scadono.

## Cause Comuni

- **Inattività prolungata** — Non hai utilizzato il POS per molto tempo
- **Durata del token breve** — Server configurato con una scadenza del token di aggiornamento breve
- **Discrepanza dell'orologio** — L'ora del dispositivo o del server è significativamente errata

## Come Risolvere

### 1. Accedi Nuovamente

Quando il token di aggiornamento scade, devi autenticarti di nuovo:
1. Disconnettiti dal POS
2. Inserisci le tue credenziali
3. Un nuovo token di aggiornamento verrà emesso

### 2. Usa il POS Regolarmente

Per evitare ciò in futuro:
- Usa regolarmente il POS per mantenere i token aggiornati
- I token vengono aggiornati automaticamente quando sei attivo
- Periodi prolungati di inattività richiederanno un nuovo accesso

### 3. Controlla le Impostazioni di Tempo

Assicurati che l'ora sia accurata su tutti i sistemi:
- Abilita la sincronizzazione automatica dell'ora sul tuo dispositivo
- Verifica che l'ora del server sia corretta
- Differenze temporali possono causare scadenze premature

### 4. Regola la Durata del Token (Avanzato)

Se i token scadono troppo rapidamente per le tue esigenze, un amministratore del server può regolare la durata del token di aggiornamento nelle impostazioni di autenticazione.
