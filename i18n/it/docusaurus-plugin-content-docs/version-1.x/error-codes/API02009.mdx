# API02009: Token di Aggiornamento Scaduto

## Cosa Significa

Il tuo token di aggiornamento ha superato la sua durata di vita e non può più essere utilizzato per ottenere nuovi token di accesso. A differenza dei token di accesso (che scadono rapidamente), i token di aggiornamento di solito durano giorni o settimane ma alla fine scadono.

## Cause Comuni

- **Inattività prolungata** — Non hai utilizzato il POS per molto tempo
- **Breve durata del token** — Server configurato con breve scadenza del token di aggiornamento
- **Discrepanza oraria** — Ora del dispositivo o del server significativamente diversa

## Come Risolvere

### 1. Accedi Nuovamente

Quando il token di aggiornamento scade, devi autenticarti nuovamente:
1. Esci dal POS
2. Inserisci le tue credenziali
3. Verrà emesso un nuovo token di aggiornamento

### 2. Usa il POS Regolarmente

Per evitare questo in futuro:
- Usa il POS regolarmente per mantenere i token attivi
- I token vengono aggiornati automaticamente quando sei attivo
- Periodi prolungati di inattività richiederanno un nuovo accesso

### 3. Controlla le Impostazioni Orarie

Assicurati un'ora precisa su tutti i sistemi:
- Abilita la sincronizzazione automatica dell'ora sul tuo dispositivo
- Verifica che l'ora del server sia corretta
- Le differenze di tempo possono causare scadenze premature

### 4. Regola la Durata del Token (Avanzato)

Se i token scadono troppo rapidamente per le tue esigenze, un amministratore del server può modificare la durata del token di aggiornamento nelle impostazioni di autenticazione.
