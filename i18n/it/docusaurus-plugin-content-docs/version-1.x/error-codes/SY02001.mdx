# SY02001: Configurazione Non Valida

## Cosa Significa

La configurazione del sistema è non valida o danneggiata. L'applicazione POS ha trovato impostazioni di configurazione che non hanno senso o sono incomplete.

## Cause Comuni

- **Impostazioni danneggiate** — Il file di configurazione è diventato corrotto
- **Impostazione incompleta** — Il processo di configurazione non è stato completato
- **Errore di modifica manuale** — I file di configurazione sono stati modificati in modo errato
- **Problema di aggiornamento della versione** — La vecchia configurazione non è compatibile con la nuova versione

## Come Risolvere

### 1. Ripristina le Impostazioni Predefinite

Se disponibile nell'app:
- Cerca "Ripristina Impostazioni" o "Ripristina Predefiniti"
- Questo rimuove le impostazioni personalizzate ma risolve la corruzione

### 2. Completa la Configurazione

Se la configurazione non è stata terminata:
1. Avvia nuovamente il processo di configurazione
2. Completa tutti i passaggi richiesti
3. Verifica la configurazione prima di terminare

### 3. Cancella i Dati dell'Applicazione

Rimuovi la configurazione danneggiata:
- **Desktop:** Elimina la cartella dei dati dell'app, riavvia l'app
- **Browser:** Cancella i dati del sito, ricarica

### 4. Reinstalla l'Applicazione

Per problemi persistenti:
1. Disinstalla completamente
2. Rimuovi eventuali cartelle di dati residue
3. Reinstalla da zero
4. Completa la configurazione da zero

### 5. Controlla gli Aggiornamenti

I formati di configurazione potrebbero cambiare:
- Aggiorna l'applicazione POS
- Aggiorna il plugin WCPOS
- Riconfigura dopo l'aggiornamento

## Cosa Viene Ripristinato

Quando ripristini la configurazione:
- Dovrai accedere nuovamente
- Le impostazioni personalizzate andranno perse
- I dati dovranno risincronizzarsi
- Ma i tuoi dati WooCommerce sono al sicuro sul server
