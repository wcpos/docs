# SY02001: Configurazione non valida

## Cosa significa

La configurazione di sistema non è valida o è corrotta. L'applicazione POS ha trovato impostazioni di configurazione che non hanno senso o sono incomplete.

## Cause comuni

- **Impostazioni corrotte** — Il file di configurazione è diventato corrotto
- **Configurazione incompleta** — Il processo di configurazione non è stato completato
- **Errore di modifica manuale** — I file di configurazione sono stati modificati in modo errato
- **Problema di aggiornamento della versione** — La configurazione precedente non è compatibile con la nuova versione

## Come risolvere

### 1. Ripristina impostazioni predefinite

Se disponibile nell'app:
- Cerca "Ripristina impostazioni" o "Ripristina valori predefiniti"
- Questo cancella le impostazioni personalizzate ma risolve la corruzione

### 2. Completare la configurazione

Se la configurazione non è stata completata:
1. Riavvia il processo di configurazione
2. Completa tutti i passaggi richiesti
3. Verifica la configurazione prima di finire

### 3. Cancellare i dati dell'applicazione

Rimuovi la configurazione corrotta:
- **Desktop:** Elimina la cartella dei dati dell'app, riavvia l'app
- **Browser:** Cancella i dati del sito, ricarica

### 4. Reinstallare l'applicazione

Per problemi persistenti:
1. Disinstalla completamente
2. Rimuovi eventuali cartelle di dati residue
3. Reinstalla nuovamente
4. Completa la configurazione da zero

### 5. Verificare gli aggiornamenti

I formati di configurazione possono cambiare:
- Aggiorna l'applicazione POS
- Aggiorna il plugin WCPOS
- Riconfigura dopo l'aggiornamento

## Cosa viene ripristinato

Durante il ripristino della configurazione:
- Dovrai effettuare nuovamente l'accesso
- Le impostazioni personalizzate andranno perse
- I dati dovranno essere sincronizzati nuovamente
- Ma i tuoi dati WooCommerce sono al sicuro sul server
