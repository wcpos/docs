# SY01001: Memoria esaurita

## Cosa significa

Il dispositivo ha memoria (RAM) insufficiente e non può completare l'operazione richiesta. Il POS necessita di memoria sufficiente per funzionare correttamente.

## Cause comuni

- **Troppe app aperte** — Altre applicazioni che utilizzano memoria
- **Set di dati di grandi dimensioni** — Molti prodotti/clienti caricati
- **Memory leak** — Applicazione che non libera correttamente la memoria
- **RAM insufficiente** — Il dispositivo non ha abbastanza memoria

## Come risolvere

### 1. Chiudere altre applicazioni

Liberare memoria:
- Chiudere schede del browser non utilizzate
- Chiudere altre applicazioni
- Chiudere programmi in background

### 2. Riavviare il POS

Aggiornare la memoria dell'applicazione:
1. Chiudere completamente il POS
2. Attendere alcuni secondi
3. Riaprire l'applicazione

### 3. Riavviare il dispositivo

Un riavvio completo cancella tutta la memoria:
1. Salvare eventuali lavori aperti
2. Riavviare il computer/dispositivo
3. Aprire solo il POS all'avvio

### 4. Cancellare i dati del browser (versione web)

Se utilizzato in un browser:
- Cancellare la cache del browser
- Chiudere e riaprire il browser
- Utilizzare meno schede del browser

### 5. Ridurre il carico dei dati

Se si dispone di un catalogo molto grande:
- Utilizzare filtri per caricare meno dati
- Considerare la paginazione dei risultati
- Limitare l'ambito di sincronizzazione se possibile

## Requisiti minimi

Assicurarsi che il dispositivo soddisfi le specifiche minime:
- **RAM:** 4GB minimo, 8GB consigliato
- **Browser:** Browser moderno (Chrome, Firefox, Edge)
- **App Desktop:** Controllare le note di rilascio per i requisiti

## Problemi persistenti

Se si esaurisce frequentemente la memoria:
- Considerare l'aggiornamento della RAM del dispositivo
- Utilizzare un dispositivo con più risorse
- Segnalare il problema se sembra eccessivo
