# SY01001: Memoria Esaurita {#sy01001-out-of-memory}

## Cosa Significa {#what-this-means}

Il tuo dispositivo ha poca memoria (RAM) e non può completare l'operazione richiesta. Il POS ha bisogno di memoria sufficiente per funzionare correttamente.

## Cause Comuni {#common-causes}

- **Troppe app aperte** — Altre applicazioni che utilizzano memoria
- **Set di dati grande** — Molti prodotti/clienti caricati
- **Fuga di memoria** — Applicazione che non rilascia correttamente la memoria
- **RAM insufficiente** — Dispositivo non ha abbastanza memoria

## Come Risolvere {#how-to-fix}

### 1. Chiudi Altre Applicazioni {#1-close-other-applications}

Libera memoria:
- Chiudi le schede del browser non utilizzate
- Chiudi altre applicazioni
- Chiudi i programmi in background

### 2. Riavvia il POS {#2-restart-the-pos}

Aggiorna la memoria dell'applicazione:
1. Chiudi completamente il POS
2. Aspetta qualche secondo
3. Riapri l'applicazione

### 3. Riavvia il Tuo Dispositivo {#3-restart-your-device}

Un riavvio completo libera tutta la memoria:
1. Salva qualsiasi lavoro aperto
2. Riavvia il computer/dispositivo
3. Apri solo il POS al riavvio

### 4. Cancella i Dati del Browser (Versione Web) {#4-clear-browser-data-web-version}

Se stai usando un browser:
- Cancella la cache del browser
- Chiudi e riapri il browser
- Usa meno schede del browser

### 5. Riduci il Carico di Dati {#5-reduce-data-load}

Se hai un catalogo molto grande:
- Usa filtri per caricare meno dati alla volta
- Considera di paginare i risultati
- Limita l'ambito di sincronizzazione se possibile

## Requisiti Minimi {#minimum-requirements}

Assicurati che il tuo dispositivo soddisfi le specifiche minime:
- **RAM:** 4GB minimo, 8GB raccomandato
- **Browser:** Browser moderno (Chrome, Firefox, Edge)
- **App Desktop:** Controlla le note di rilascio per i requisiti

## Problemi Continuativi {#ongoing-issues}

Se esaurisci frequentemente la memoria:
- Considera di aggiornare la RAM del tuo dispositivo
- Usa un dispositivo con più risorse
- Riporta il problema se sembra eccessivo
