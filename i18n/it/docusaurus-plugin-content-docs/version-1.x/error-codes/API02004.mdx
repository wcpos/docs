# API02004: Utente Non Autorizzato {#api02004-user-not-authorized}

## Cosa Significa {#what-this-means}

Sei registrato, ma il tuo account utente non ha il permesso per eseguire l'azione richiesta. Questo è un problema di autorizzazione (cosa puoi fare) piuttosto che un problema di autenticazione (chi sei).

## Cause Comuni {#common-causes}

- **Limitazioni del ruolo utente** — Il tuo ruolo WordPress non include l'accesso al POS
- **Accesso al POS disabilitato** — Il tuo account non ha ricevuto i permessi per il POS
- **Restrizioni sulle funzionalità** — Alcune funzionalità sono limitate a ruoli specifici
- **Restrizioni sul negozio** — Potresti non avere accesso a questo negozio specifico

## Mappatura degli Errori del Server {#server-error-mapping}

Questo codice di errore viene attivato quando il server restituisce:

| Codice Server | Fonte |
|---------------|-------|
| `rest_cannot_view` | WordPress REST API |
| `woocommerce_rest_cannot_view` | WooCommerce REST API |

## Come Risolvere {#how-to-fix}

### 1. Controlla il Ruolo Utente {#1-check-user-role}

In WordPress Admin → Utenti:
1. Trova il tuo account utente
2. Verifica il ruolo (ad es. Responsabile Negozio, Amministratore)
3. Assicurati che il ruolo includa le capacità di WooCommerce

### 2. Abilita l'Accesso al POS {#2-enable-pos-access}

In WordPress Admin → WooCommerce → POS → Accesso:
1. Trova l'utente o il ruolo
2. Abilita i permessi di accesso al POS
3. Salva le modifiche

### 3. Richiedi Permessi Aggiuntivi {#3-request-additional-permissions}

Contatta l'amministratore del tuo negozio per:
- Concedere al tuo ruolo l'accesso al POS
- Assegnarti un ruolo con permessi adeguati
- Abilitare funzionalità specifiche di cui hai bisogno

### 4. Controlla l'Assegnazione del Negozio {#4-check-store-assignment}

Se utilizzi più negozi:
- Verifica che tu sia assegnato al negozio corretto
- Controlla i permessi specifici per il negozio

## Permessi Richiesti {#required-permissions}

Diverse azioni richiedono capacità diverse:
- **Visualizza prodotti**: Accesso in lettura ai prodotti
- **Crea ordini**: Capacità di creare/modificare ordini
- **Gestisci clienti**: Capacità di gestione dei clienti
- **Accedi ai rapporti**: Capacità di visualizzare rapporti
