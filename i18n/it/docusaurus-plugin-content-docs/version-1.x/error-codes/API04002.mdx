# API04002: Risposta Imprevista

## Cosa Significa

Il server ha restituito un codice di stato HTTP non previsto per questa richiesta. I codici comuni includono 500 (errore del server), 403 (proibito), 404 (non trovato), ecc.

## Cause Comuni

- **500 Errore Interno del Server** — PHP si è arrestato o ha riscontrato un errore
- **403 Proibito** — Accesso negato dalle impostazioni di sicurezza
- **404 Non Trovato** — L'endpoint non esiste
- **502/503/504** — Problemi di gateway o disponibilità del server

## Come Risolvere

### Per Errori 500 (Errore del Server)

1. Controllare i registri degli errori PHP
2. Aumentare il limite di memoria PHP
3. Cercare conflitti tra plugin
4. Controllare `wp-content/debug.log`

### Per Errori 403 (Proibito)

1. Controllare le impostazioni del plugin di sicurezza (Wordfence, Sucuri, ecc.)
2. Inserire in whitelist il POS o il proprio IP
3. Controllare le regole di blocco in `.htaccess`
4. Verificare che ModSecurity non stia bloccando le richieste

### Per Errori 404 (Non Trovato)

1. Verificare che il plugin WCPOS sia attivo
2. Svuotare i permalink di WordPress (Impostazioni → Permalink → Salva)
3. Controllare che l'API REST sia abilitata
4. Verificare la configurazione dell'URL

### Per Errori 502/503/504 (Problemi di Gateway)

1. Contattare il proprio fornitore di hosting
2. Attendere il ripristino del server
3. Verificare se il sito è sotto carico pesante
4. Verificare che il server sia in esecuzione

### Risoluzione Problemi Generale

1. Provare ad accedere al sito direttamente
2. Controllare il pannello di controllo dell'hosting per problemi
3. Rivedere i registri di accesso del server
4. Contattare il supporto hosting se necessario
