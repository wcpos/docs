# SY01003: Permesso Negato {#sy01003-permission-denied}

## Cosa Significa {#what-this-means}

L'app POS non ha permesso di accedere a una risorsa richiesta. Questo potrebbe essere un file, una cartella o una funzionalità di sistema.

## Cause Comuni {#common-causes}

- **Permessi dei file** — Impossibile leggere o scrivere nei file necessari
- **Permessi dell'app** — L'applicazione non ha i permessi richiesti
- **Cartella protetta** — Tentativo di accedere a una posizione ristretta
- **Software di sicurezza** — Antivirus che blocca l'accesso

## Come Risolvere {#how-to-fix}

### 1. Esegui come Amministratore (Windows) {#1-run-as-administrator-windows}

Fai clic destro sull'app e seleziona "Esegui come amministratore" se necessario per la configurazione iniziale.

### 2. Controlla i Permessi dell'Applicazione {#2-check-application-permissions}

**macOS:**
- Preferenze di Sistema → Sicurezza e Privacy → Privacy
- Controlla che l'app abbia i permessi necessari

**Windows:**
- Impostazioni → Privacy
- Esamina i permessi dell'app

### 3. Controlla i Permessi della Cartella {#3-check-folder-permissions}

Assicurati che l'app possa accedere alla propria cartella dati:
- L'utente dovrebbe possedere la directory dei dati dell'app
- I permessi in scrittura dovrebbero essere abilitati
- Nessun flag di sola lettura sulla cartella

### 4. Esamina le Impostazioni dell'Antivirus {#4-review-antivirus-settings}

Il software di sicurezza potrebbe bloccare l'accesso:
- Controlla i registri dell'antivirus per blocchi
- Aggiungi l'app POS all'elenco di autorizzazione
- Includi nella whitelist la cartella dei dati dell'app

### 5. Reinstalla l'Applicazione {#5-reinstall-the-application}

Se i permessi sono corrotti:
1. Disinstalla l'app POS
2. Riavvia il dispositivo
3. Reinstalla in una posizione standard
4. Esegui con permessi normali (non da amministratore)

## Permessi del Browser (Versione Web) {#browser-permissions-web-version}

Se utilizzi la versione web:
- Consenti l'archiviazione locale per il sito
- Abilita i cookie per il sito
- Non utilizzare impostazioni del browser eccessivamente restrittive
