# API02005: Autorizzazioni Insufficienti

## Cosa Significa

Il tuo account utente non dispone delle specifiche capacità WordPress necessarie per questa azione. Mentre hai un accesso di base, l'operazione che stai tentando richiede autorizzazioni aggiuntive.

## Cause Comuni

- **Ruolo utente limitato** — Il tuo ruolo non include tutte le capacità necessarie
- **Capacità non assegnata** — Una specifica capacità manca nel tuo ruolo
- **Restrizioni del plugin** — Un plugin di sicurezza sta limitando le capacità
- **Problemi con ruoli personalizzati** — I ruoli personalizzati potrebbero mancare di alcune capacità

## Mappatura degli Errori del Server

Questo codice di errore viene attivato quando il server restituisce:

| Codice Server | Origine |
|-------------|--------|
| `rest_forbidden` | WordPress REST API |
| `rest_cannot_create` | WordPress REST API |
| `rest_cannot_edit` | WordPress REST API |
| `rest_cannot_delete` | WordPress REST API |
| `woocommerce_rest_cannot_create` | WooCommerce REST API |
| `woocommerce_rest_cannot_edit` | WooCommerce REST API |
| `woocommerce_rest_cannot_delete` | WooCommerce REST API |
| HTTP 403 | Qualsiasi risposta del server (fallback) |

## Come Risolvere

### 1. Rivedere le Capacità Richieste

Capacità comuni necessarie per le operazioni POS:
- `manage_woocommerce` — Gestione generale di WooCommerce
- `edit_shop_orders` — Creare e modificare ordini
- `edit_products` — Modificare informazioni sui prodotti
- `edit_users` — Gestire account clienti

### 2. Aggiornare il Ruolo Utente

Chiedi a un amministratore di assegnare un ruolo con più capacità:
- **Shop Manager** — Accesso completo a WooCommerce
- **Amministratore** — Accesso completo al sito

### 3. Aggiungere Capacità Specifiche

Se hai bisogno di un ruolo personalizzato, aggiungi le capacità necessarie:

```php
// Esempio: Aggiungere capacità POS a un ruolo personalizzato
$role = get_role('tuo_ruolo_personalizzato');
$role->add_cap('manage_woocommerce');
$role->add_cap('edit_shop_orders');
```

### 4. Verificare Conflitti dei Plugin

Alcuni plugin di sicurezza o gestione ruoli possono limitare le capacità:
- Rivedere le impostazioni del plugin
- Verificare eventuali filtri delle capacità
- Disabilitare temporaneamente per eseguire test
