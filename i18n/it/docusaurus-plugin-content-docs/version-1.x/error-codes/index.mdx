# Codici di Errore {#error-codes}

Quando WCPOS incontra un problema, visualizza un codice di errore che aiuta a identificare il problema. Ogni codice di errore segue il formato `[DOMINIO][CATEGORIA][CODICE_SPECIFICO]` per aiutarti a comprendere rapidamente il tipo e la natura dell'errore.

## Formato del Codice di Errore {#error-code-format}

| Componente | Descrizione | Esempio |
|------------|-------------|---------|
| **Dominio** | L'area del sistema | `API`, `DB`, `PY`, `SY` |
| **Categoria** | La categoria di errore | `01`, `02`, `03`, ecc. |
| **Codice Specifico** | L'identificatore unico | `001`, `002`, `003`, ecc. |

**Esempio:** `API04001` = API (dominio) + 04 (errori di risposta) + 001 (formato di risposta non valido)

## Domini di Errore {#error-domains}

### [Errori API](./api) {#api-errorsapi}

Gli errori API si verificano durante la comunicazione con il server WooCommerce.

| Categoria | Intervallo Codici | Descrizione |
|-----------|-------------------|-------------|
| Connessione | API01xxx | Problemi di rete e connettività |
| Autenticazione | API02xxx | Problemi di accesso, token e permessi |
| Richiesta | API03xxx | Problemi con le richieste in uscita |
| Risposta | API04xxx | Problemi con le risposte del server |
| Plugin/WordPress | API05xxx | Problemi con i plugin di WordPress o WooCommerce |
| Configurazione | API06xxx | Problemi di installazione e configurazione |

### [Errori di Database](./db) {#database-errorsdb}

Gli errori di database si verificano durante la memorizzazione o il recupero di dati localmente.

| Categoria | Intervallo Codici | Descrizione |
|-----------|-------------------|-------------|
| Connessione | DB01xxx | Problemi di connessione e transazione del database |
| Dati | DB02xxx | Problemi con record e vincoli |
| Query | DB03xxx | Problemi di sintassi della query e tipo di dati |

### [Errori di Pagamento](./py) {#payment-errorspy}

Gli errori di pagamento si verificano durante il processo di checkout e pagamento.

| Categoria | Intervallo Codici | Descrizione |
|-----------|-------------------|-------------|
| Carta | PY01xxx | Problemi con la carta di pagamento |
| Gateway | PY02xxx | Problemi di comunicazione con il gateway di pagamento |

### [Errori di Sistema](./sy) {#system-errorssy}

Gli errori di sistema sono correlati alle risorse del dispositivo e alla configurazione del sistema.

| Categoria | Intervallo Codici | Descrizione |
|-----------|-------------------|-------------|
| Risorsa | SY01xxx | Problemi di memoria, disco e permessi |
| Servizio | SY02xxx | Configurazione e disponibilità del servizio |
