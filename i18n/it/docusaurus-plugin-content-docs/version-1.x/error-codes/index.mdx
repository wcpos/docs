# Codici di Errore

Quando WCPOS riscontra un problema, visualizza un codice di errore che aiuta a identificare il problema. Ogni codice di errore segue il formato `[DOMINIO][CATEGORIA][CODICE_SPECIFICO]` per aiutarti a comprendere rapidamente il tipo e la natura dell'errore.

## Formato del Codice di Errore

| Componente | Descrizione | Esempio |
|-----------|-------------|---------|
| **Dominio** | L'area di sistema | `API`, `DB`, `PY`, `SY` |
| **Categoria** | La categoria di errore | `01`, `02`, `03`, ecc. |
| **Codice Specifico** | L'identificatore univoco | `001`, `002`, `003`, ecc. |

**Esempio:** `API04001` = API (dominio) + 04 (errori di risposta) + 001 (formato di risposta non valido)

## Domini di Errore

### [Errori API](./api)

Gli errori API si verificano durante la comunicazione con il server WooCommerce.

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| Connessione | API01xxx | Problemi di rete e connettività |
| Autenticazione | API02xxx | Problemi di login, token e autorizzazioni |
| Richiesta | API03xxx | Problemi con le richieste in uscita |
| Risposta | API04xxx | Problemi con le risposte del server |
| Plugin/WordPress | API05xxx | Problemi con plugin WordPress o WooCommerce |
| Configurazione | API06xxx | Problemi di installazione e configurazione |

### [Errori Database](./db)

Gli errori database si verificano durante l'archiviazione o il recupero di dati locali.

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| Connessione | DB01xxx | Problemi di connessione e transazione del database |
| Dati | DB02xxx | Problemi di record e vincoli |
| Query | DB03xxx | Problemi di sintassi delle query e tipi di dati |

### [Errori di Pagamento](./py)

Gli errori di pagamento si verificano durante il checkout e l'elaborazione del pagamento.

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| Carta | PY01xxx | Problemi con la carta di pagamento |
| Gateway | PY02xxx | Problemi di comunicazione con il gateway di pagamento |

### [Errori di Sistema](./sy)

Gli errori di sistema sono relativi alle risorse del dispositivo e alla configurazione di sistema.

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| Risorsa | SY01xxx | Problemi di memoria, disco e autorizzazioni |
| Servizio | SY02xxx | Configurazione e disponibilità dei servizi |
