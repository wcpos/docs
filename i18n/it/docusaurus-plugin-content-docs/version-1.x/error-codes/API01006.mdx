# API01006: Rete Irraggiungibile {#api01006-network-unreachable}

## Cosa Significa {#what-this-means}

Il POS non può stabilire un percorso verso il server. Si tratta di un problema a livello di rete: i pacchetti di dati non riescono a trovare un percorso verso il tuo server.

## Cause Comuni {#common-causes}

- **Problemi di configurazione della rete** — Impostazioni errate del gateway o del routing
- **Problemi con il fornitore di servizi Internet** — Il tuo fornitore di servizi Internet ha problemi
- **Interferenza del VPN** — Un VPN potrebbe instradare il traffico in modo errato
- **Problemi di rete del server** — La rete del server potrebbe essere inattiva

## Come Risolvere {#how-to-fix}

### 1. Controlla la Tua Connessione Internet {#1-check-your-internet-connection}

- Prova ad accedere ad altri siti web
- Riavvia il tuo router/modem
- Verifica di essere connesso alla rete

### 2. Disabilita Temporaneamente il VPN {#2-disable-vpn-temporarily}

Se stai usando un VPN:
- Disconnetti e riprova
- Prova un server VPN diverso
- Controlla se il VPN consente l'accesso alla posizione del tuo server

### 3. Contatta il Tuo ISP {#3-check-with-your-isp}

Contatta il tuo fornitore di servizi Internet se:
- Altri siti sono anche irraggiungibili
- Il problema persiste dopo aver riavviato l'apparecchiatura di rete

### 4. Verifica lo Stato del Server {#4-verify-server-status}

- Controlla se il tuo fornitore di hosting riporta eventuali problemi di rete
- Prova ad accedere al tuo sito da una rete diversa (ad es., dati mobili)
- Usa strumenti online per verificare se il sito è inattivo per tutti

### 5. Controlla le Impostazioni del Firewall {#5-check-firewall-settings}

Assicurati che il firewall del tuo dispositivo non stia bloccando le connessioni in uscita verso il tuo server.
