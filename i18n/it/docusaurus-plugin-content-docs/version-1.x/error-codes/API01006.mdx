# API01006: Rete non raggiungibile

## Cosa significa

Il POS non riesce a stabilire una connessione con il server. Questo è un problema a livello di rete — i pacchetti dati non riescono a trovare un percorso verso il server.

## Cause comuni

- **Problemi di configurazione di rete** — Impostazioni di gateway o routing errate
- **Problemi del provider di servizi Internet** — Il proprio fornitore di servizi Internet sta riscontrando difficoltà
- **Interferenze VPN** — Una VPN potrebbe instradare il traffico in modo non corretto
- **Problemi di rete del server** — La rete del server potrebbe essere inattiva

## Come risolvere

### 1. Verificare la connessione Internet

- Provare ad accedere ad altri siti web
- Riavviare il router/modem
- Verificare la connessione alla rete

### 2. Disattivare temporaneamente la VPN

Se si utilizza una VPN:
- Disconnettersi e riprovare
- Provare un server VPN diverso
- Verificare se la VPN consente l'accesso alla posizione del server

### 3. Contattare il proprio provider di servizi Internet

Contattare il provider di servizi Internet se:
- Altri siti sono ugualmente irraggiungibili
- Il problema persiste dopo il riavvio delle apparecchiature di rete

### 4. Verificare lo stato del server

- Verificare se il proprio provider di hosting segnala problemi di rete
- Provare ad accedere al sito da una rete diversa (ad esempio, dati mobili)
- Utilizzare strumenti online per verificare se il sito è offline per tutti

### 5. Verificare le impostazioni del firewall

Assicurarsi che il firewall del dispositivo non stia bloccando le connessioni in uscita verso il server.
