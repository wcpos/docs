# Errori API

Gli errori API si verificano durante la comunicazione con il server WooCommerce. Questi errori hanno il prefisso `API` e sono organizzati nelle seguenti categorie:

## Categorie

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| [Connessione](#errori-di-connessione) | API01xxx | Problemi di rete e connettività |
| [Autenticazione](#errori-di-autenticazione) | API02xxx | Problemi di accesso, token e permessi |
| [Richiesta](#errori-di-richiesta) | API03xxx | Problemi con le richieste in uscita |
| [Risposta](#errori-di-risposta) | API04xxx | Problemi con le risposte del server |
| [Plugin/WordPress](#errori-di-pluginwordpress) | API05xxx | Problemi con WordPress o plugin WooCommerce |
| [Configurazione](#errori-di-configurazione) | API06xxx | Problemi di impostazione e configurazione |

---

## Errori di Connessione

Problemi di rete e connettività tra il POS e il server.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [API01001](./API01001) | Timeout Connessione | Il server ha impiegato troppo tempo a rispondere |
| [API01002](./API01002) | Connessione Rifiutata | Il server ha rifiutato la connessione |
| [API01003](./API01003) | Connessione Reimpostata | La connessione è stata chiusa inaspettatamente |
| [API01004](./API01004) | Risoluzione DNS Fallita | Impossibile risolvere l'indirizzo del server |
| [API01005](./API01005) | Errore Certificato SSL | Problema con il certificato di sicurezza del sito |
| [API01006](./API01006) | Rete Irraggiungibile | Impossibile raggiungere la rete |
| [API01007](./API01007) | Dispositivo Offline | Il dispositivo non è connesso a internet |
| [API01008](./API01008) | Sito Web Non Disponibile | Il sito web non risponde |

## Errori di Autenticazione

Problemi di accesso, sessioni e permessi.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [API02001](./API02001) | Credenziali Non Valide | Nome utente o password non corretti |
| [API02002](./API02002) | Token Scaduto | La sessione è scaduta |
| [API02003](./API02003) | Token Non Valido | Il token di autenticazione non è valido |
| [API02004](./API02004) | Utente Non Autorizzato | Non hai il permesso di eseguire questa azione |
| [API02005](./API02005) | Permessi Insufficienti | Il ruolo utente non ha i permessi richiesti |
| [API02006](./API02006) | Chiave API Non Valida | La chiave API di WooCommerce non è valida |
| [API02007](./API02007) | Aggiornamento Token Fallito | Impossibile aggiornare la sessione |
| [API02008](./API02008) | Token di Aggiornamento Non Valido | Il token di aggiornamento non è valido |
| [API02009](./API02009) | Token di Aggiornamento Scaduto | Il token di aggiornamento è scaduto |
| [API02010](./API02010) | Autenticazione Richiesta | L'autenticazione è necessaria per questa azione |

## Errori di Richiesta

Problemi con le richieste inviate al server.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [API03001](./API03001) | Formato Richiesta Non Valido | Il formato della richiesta non è corretto |
| [API03002](./API03002) | Parametri Obbligatori Mancanti | Dati obbligatori mancanti nella richiesta |
| [API03003](./API03003) | Valore Parametro Non Valido | Un parametro ha un valore non valido |
| [API03004](./API03004) | Richiesta Troppo Grande | La richiesta supera i limiti di dimensione |
| [API03005](./API03005) | Limite di Frequenza Superato | Troppe richieste in poco tempo |
| [API03006](./API03006) | Metodo Non Supportato | Il metodo HTTP non è supportato |
| [API03007](./API03007) | Coda Richieste Piena | Troppe richieste in attesa |

## Errori di Risposta

Problemi con le risposte ricevute dal server.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [API04001](./API04001) | Formato Risposta Non Valido | Il formato della risposta del server non è valido |
| [API04002](./API04002) | Codice Risposta Inatteso | Ricevuto un codice di stato HTTP inatteso |
| [API04003](./API04003) | Risposta JSON Malformata | La risposta JSON è corrotta o non valida |
| [API04004](./API04004) | Dati Risposta Mancanti | I dati attesi mancano nella risposta |
| [API04005](./API04005) | Recupero JSON Tentato | Tentativo di recupero da JSON malformato |
| [API04006](./API04006) | Risorsa Non Trovata | La risorsa richiesta non esiste (404) |

| Codice | Nome | Descrizione |

| [API05003](./API05003) | Plugin WCPOS Obsoleto | Il plugin WCPOS necessita di aggiornamento |
| [API05004](./API05004) | API WordPress Disabilitata | L'API REST di WordPress è disabilitata |
| [API05005](./API05005) | Plugin Non Trovato | Un plugin richiesto non è installato |
