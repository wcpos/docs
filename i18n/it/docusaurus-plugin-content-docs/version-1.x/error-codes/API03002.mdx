# API03002: Parametri Obbligatori Mancanti

## Cosa Significa

La richiesta non include i dati necessari al server per completare l'azione. I campi obbligatori non sono stati inclusi nella richiesta.

## Cause Comuni

- **Modulo incompleto** — Campi obbligatori non compilati
- **Dati non salvati** — Dati del modulo non acquisiti correttamente
- **Discrepanza di versione** — Il plugin si aspetta parametri diversi da quelli inviati dal POS
- **Campi personalizzati** — Campi personalizzati obbligatori mancanti

## Come Risolvere

### 1. Compilare i Campi Obbligatori

Verificare che tutte le informazioni necessarie siano inserite:
- Dettagli del cliente (se richiesti)
- Informazioni sul prodotto
- Dettagli dell'ordine

### 2. Aggiornare e Riprovare

Lo stato del modulo potrebbe essere incompleto:
1. Aggiornare il POS
2. Reinserire le informazioni richieste
3. Riprovare l'azione

### 3. Controllare le Versioni dei Plugin

Verificare la compatibilità:
- Aggiornare il plugin WCPOS
- Aggiornare l'applicazione POS
- Assicurarsi che siano su versioni compatibili

### 4. Verificare le Impostazioni dei Campi Obbligatori

In WooCommerce, alcuni campi possono essere contrassegnati come obbligatori:
- Rivedere i requisiti dei campi di checkout
- Controllare le configurazioni dei campi personalizzati
- Modificare i requisiti se necessario

### 5. Controllare Integrazioni Personalizzate

Se si utilizzano plugin o integrazioni personalizzate:
- Potrebbero richiedere campi aggiuntivi
- Rivedere la documentazione del plugin
- Verificare eventuali conflitti
