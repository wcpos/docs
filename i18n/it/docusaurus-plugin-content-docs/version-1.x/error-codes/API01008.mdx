# API01008: Sito Web Non Disponibile

## Cosa Significa

Il POS può raggiungere internet, ma il tuo sito web specifico non sta rispondendo. Il server che ospita il tuo negozio WooCommerce sembra essere inattivo o irraggiungibile.

## Cause Comuni

- **Manutenzione del server** — Il tuo fornitore di hosting potrebbe stare eseguendo degli aggiornamenti
- **Crash del server** — Il server web si è fermato inaspettatamente
- **Limiti di risorse superati** — Il tuo account di hosting ha raggiunto i suoi limiti
- **Attacco DDoS** — Il server è sovraccarico da traffico malevolo
- **Dominio/hosting scaduto** — Il tuo hosting o dominio è scaduto

## Come Risolvere

### 1. Verificare che il Sito sia Inattivo

Prova ad accedere al tuo sito direttamente in un browser. Se non si carica, il problema è lato server.

### 2. Controllare lo Stato del Fornitore di Hosting

- Visita la pagina di stato del tuo fornitore di hosting
- Controlla eventuali manutenzioni programmate
- Cerca segnalazioni di interruzioni del servizio

### 3. Esaminare l'Account di Hosting

Accedi al pannello di controllo del tuo hosting:
- Verifica che il tuo account sia attivo
- Controlla di non aver superato i limiti di risorse
- Cerca eventuali avvisi di sospensione

### 4. Contattare il Supporto Hosting

Se tutto sembra normale dalla tua parte, contatta il supporto del tuo fornitore di hosting. Possono:
- Verificare lo stato del server
- Esaminare i log del server
- Riavviare i servizi se necessario

### 5. Attendere e Riprovare

Se si tratta di un'interruzione temporanea:
- Attendi 15-30 minuti
- Il POS riproverà automaticamente le connessioni
- I dati in cache rimarranno disponibili offline
