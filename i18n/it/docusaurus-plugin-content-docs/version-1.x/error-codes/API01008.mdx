# API01008: Sito Web Non Disponibile

## Cosa Significa

Il POS può raggiungere internet, ma il tuo specifico sito web non risponde. Il server che ospita il tuo negozio WooCommerce sembra essere inattivo o irraggiungibile.

## Cause Comuni

- **Manutenzione del server** — Il tuo fornitore di hosting potrebbe essere in fase di aggiornamento
- **Crash del server** — Il server web si è bloccato in modo imprevisto
- **Limiti delle risorse superati** — Il tuo account di hosting ha raggiunto i suoi limiti
- **Attacco DDoS** — Il server è sopraffatto da traffico malevolo
- **Dominio/hosting scaduto** — Il tuo hosting o dominio è scaduto

## Come Risolvere

### 1. Verifica se il Sito è Giù

Prova ad accedere direttamente al tuo sito in un browser. Se non si carica, il problema è lato server.

### 2. Controlla lo Stato del Fornitore di Hosting

- Visita la pagina di stato del tuo fornitore di hosting
- Controlla per manutenzioni programmate
- Guarda per eventuali interruzioni segnalate

### 3. Rivedi l'Account di Hosting

Accedi al tuo pannello di controllo di hosting:
- Controlla se il tuo account è attivo
- Verifica di non aver superato i limiti delle risorse
- Cerca avvisi di sospensione

### 4. Contatta il Supporto Hosting

Se tutto sembra normale da parte tua, contatta il tuo fornitore di hosting. Possono:
- Controllare lo stato del server
- Rivedere i log del server
- Riavviare i servizi se necessario

### 5. Aspetta e Ripeti

Se si tratta di un'interruzione temporanea:
- Aspetta 15-30 minuti
- Il POS tenterà automaticamente di riconnettersi
- I dati memorizzati nella cache rimangono disponibili offline
