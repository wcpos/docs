# SY01002: Disco Pieno

## Cosa Significa

Lo spazio di archiviazione del dispositivo è esaurito. Il POS non può salvare dati o continuare a funzionare senza spazio su disco disponibile.

## Cause Comuni

- **Archiviazione piena** — Nessuno spazio libero sull'unità
- **Database locale di grandi dimensioni** — Dati POS che utilizzano molto spazio
- **File temporanei** — File temporanei che consumano spazio
- **Altre applicazioni** — Altre app che riempiono il disco

## Come Risolvere

### 1. Liberare Spazio su Disco

Eliminare file non necessari:
- Svuotare il cestino
- Rimuovere vecchi download
- Eliminare applicazioni non utilizzate
- Cancellare file temporanei

### 2. Verificare lo Spazio Disponibile

**Windows:**
- Aprire Esplora File
- Tasto destro sull'unità → Proprietà
- Visualizzare spazio utilizzato e libero

**macOS:**
- Menu Apple → Informazioni su Questo Mac
- Fare clic su Archiviazione

### 3. Cancellare Dati Browser (Versione Web)

I browser memorizzano dati localmente:
- Cancellare cache e cookie
- Rimuovere vecchi dati dei siti
- Verificare l'utilizzo dell'archiviazione del browser

### 4. Spostare o Eliminare File di Grandi Dimensioni

Trovare e gestire file di grandi dimensioni:
- Vecchi video o foto
- Download di grandi dimensioni
- Software non utilizzato
- File di backup che possono essere spostati su archiviazione esterna

### 5. Controllare i Dati POS

Se i dati POS sono molto grandi:
- Il database locale potrebbe aver bisogno di pulizia
- Considerare di cancellare e risincronizzare
- Verificare se è possibile eliminare vecchi dati

## Spazio Libero Consigliato

Mantenere almeno:
- **10GB** liberi per il normale funzionamento
- **20GB+** per cataloghi di grandi dimensioni
- Più spazio è sempre meglio

## Prevenire Problemi Futuri

- Pulire regolarmente file non necessari
- Configurare la pulizia automatica dei file temporanei
- Monitorare l'utilizzo dello spazio su disco
- Considerare un'archiviazione più grande se frequentemente piena
