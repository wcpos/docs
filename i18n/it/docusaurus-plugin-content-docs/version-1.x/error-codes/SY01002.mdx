# SY01002: Disco Pieno {#sy01002-disk-full}

## Cosa Significa {#what-this-means}

La memoria del tuo dispositivo è piena. Il POS non può salvare dati o continuare a funzionare senza spazio disponibile su disco.

## Cause Comuni {#common-causes}

- **Memoria piena** — Nessuno spazio libero sul disco
- **Grande database locale** — Dati POS che utilizzano molto spazio
- **File temporanei** — File temporanei che occupano spazio
- **Altre applicazioni** — Altre app che riempiono il disco

## Come Risolvere {#how-to-fix}

### 1. Liberare Spazio sul Disco {#1-free-up-disk-space}

Elimina file non necessari:
- Svuota il cestino
- Rimuovi download vecchi
- Elimina applicazioni inutilizzate
- Pulisci i file temporanei

### 2. Controlla Spazio Disponibile {#2-check-available-space}

**Windows:**
- Apri Esplora file
- Fai clic destro sul disco → Proprietà
- Visualizza spazio usato e libero

**macOS:**
- Menu Apple → Informazioni su questo Mac
- Clicca su Memoria

### 3. Pulisci Dati del Browser (Versione Web) {#3-clear-browser-data-web-version}

I browser memorizzano dati localmente:
- Pulisci cache e cookie
- Rimuovi dati di siti vecchi
- Controlla l'uso dello spazio di memoria del browser

### 4. Sposta o Elimina File Grandi {#4-move-or-delete-large-files}

Trova e gestisci file grandi:
- Video o foto vecchie
- Download grandi
- Software inutilizzati
- File di backup che possono essere spostati su memoria esterna

### 5. Controlla Dati del POS {#5-check-pos-data}

Se i dati del POS sono molto grandi:
- Il database locale potrebbe necessitare di pulizia
- Considera di svuotare e risincronizzare
- Controlla se i dati vecchi possono essere eliminati

## Spazio Libero Raccomandato {#recommended-free-space}

Tieni almeno:
- **10GB** liberi per un funzionamento normale
- **20GB+** per cataloghi grandi
- Maggiore spazio è sempre meglio

## Prevenire Problemi Futuri {#preventing-future-issues}

- Pulisci regolarmente i file non necessari
- Imposta la pulizia automatica dei file temporanei
- Monitora l'uso dello spazio su disco
- Considera uno storage più grande se frequentemente pieno
