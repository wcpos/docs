# Errori del Database

Gli errori del database si verificano durante il salvataggio o il recupero dei dati nel database locale. WCPOS utilizza un database locale per memorizzare prodotti, clienti e altri dati per un accesso veloce e funzionalità offline. Questi errori sono prefissi con `DB`.

## Categorie

| Categoria | Intervallo Codice | Descrizione |
|-----------|-------------------|-------------|
| [Connessione](#connection-errors) | DB01xxx | Problemi di connessione e transazione del database |
| [Dati](#data-errors) | DB02xxx | Problemi di registrazione e vincoli |
| [Query](#query-errors) | DB03xxx | Problemi di sintassi delle query e tipi di dati |

---

## Errori di Connessione

Problemi di connessione o di operazioni sul database locale.

| Codice | Nome | Descrizione |
|--------|------|-------------|
| [DB01001](./DB01001) | Connessione Fallita | Impossibile connettersi al database locale |
| [DB01002](./DB01002) | Timeout della Query | La query del database ha impiegato troppo tempo |
| [DB01003](./DB01003) | Transazione Fallita | La transazione del database non è riuscita a completarsi |

## Errori di Dati

Problemi con registrazioni, duplicati e vincoli.

| Codice | Nome | Descrizione |
|--------|------|-------------|
| [DB02001](./DB02001) | Registrazione Duplicata | Esiste già una registrazione con questo ID |
| [DB02002](./DB02002) | Registrazione Non Trovata | La registrazione richiesta non esiste |
| [DB02003](./DB02003) | Violazione del Vincolo | I dati violano i vincoli del database |

## Errori di Query

Problemi con la sintassi delle query e i tipi di dati.

| Codice | Nome | Descrizione |
|--------|------|-------------|
| [DB03001](./DB03001) | Errore di Sintassi della Query | La query del database presenta errori di sintassi |
| [DB03002](./DB03002) | Tipo di Dato Non Valido | Il tipo di dato non corrisponde al tipo previsto |
| [DB03003](./DB03003) | Campo Obbligatorio Mancante | Un campo obbligatorio è vuoto |
