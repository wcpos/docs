# Errori del Database

Gli errori del database si verificano durante l'archiviazione o il recupero di dati nel database locale. WCPOS utilizza un database locale per memorizzare prodotti, clienti e altri dati per un accesso rapido e funzionalità offline. Questi errori hanno un prefisso `DB`.

## Categorie

| Categoria | Intervallo Codici | Descrizione |
|----------|------------|-------------|
| [Connessione](#connection-errors) | DB01xxx | Problemi di connessione e transazione del database |
| [Dati](#data-errors) | DB02xxx | Problemi di record e vincoli |
| [Query](#query-errors) | DB03xxx | Problemi di sintassi delle query e tipi di dati |

---

## Errori di Connessione

Problemi di connessione o operazioni sul database locale.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [DB01001](./DB01001) | Connessione Fallita | Impossibile connettersi al database locale |
| [DB01002](./DB01002) | Timeout Query | La query del database ha richiesto troppo tempo |
| [DB01003](./DB01003) | Transazione Fallita | Impossibile completare la transazione del database |

## Errori di Dati

Problemi con record, duplicati e vincoli.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [DB02001](./DB02001) | Record Duplicato | Un record con questo ID esiste già |
| [DB02002](./DB02002) | Record Non Trovato | Il record richiesto non esiste |
| [DB02003](./DB02003) | Violazione Vincolo | I dati violano i vincoli del database |

## Errori di Query

Problemi di sintassi delle query e tipi di dati.

| Codice | Nome | Descrizione |
|------|------|-------------|
| [DB03001](./DB03001) | Errore di Sintassi Query | La query del database contiene errori di sintassi |
| [DB03002](./DB03002) | Tipo di Dati Non Valido | Il tipo di dati non corrisponde al tipo previsto |
| [DB03003](./DB03003) | Campo Obbligatorio Mancante | Un campo obbligatorio è vuoto |
