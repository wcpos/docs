# API02001: Credenziali non valide {#api02001-invalid-credentials}

## Cosa significa {#what-this-means}

Il nome utente o la password inseriti sono errati. Il server ha rifiutato il tentativo di accesso perché le credenziali non corrispondono a nessun account valido.

## Cause comuni {#common-causes}

- **Errore di battitura nel nome utente o nella password** — Controlla se il caps lock è attivo o se ci sono spazi extra
- **Account errato** — Utilizzo delle credenziali per un sito diverso
- **Password cambiata di recente** — La password è stata aggiornata ma il POS ha quella vecchia
- **Account eliminato** — L'account utente non esiste più

## Mappatura degli errori del server {#server-error-mapping}

Questo codice di errore viene attivato quando il server restituisce:

| Codice del server | Origine |
|-------------------|---------|
| `woocommerce_rest_authentication_error` | WooCommerce REST API |
| `jwt_auth_failed` | Plugin di autenticazione JWT |

## Come risolvere {#how-to-fix}

### 1. Verifica le tue credenziali {#1-verify-your-credentials}

- Controlla il tuo nome utente (di solito email o nome utente di WordPress)
- Verifica che la password sia corretta
- Prova a effettuare il login direttamente nell'amministrazione di WordPress per confermare

### 2. Reimposta la tua password {#2-reset-your-password}

Se hai dimenticato la tua password:
1. Vai alla tua pagina di login di WordPress
2. Clicca su "Hai dimenticato la password?"
3. Inserisci la tua email per ricevere un link di reimpostazione
4. Aggiorna la tua password
5. Accedi al POS con la nuova password

### 3. Controlla l'account utente {#3-check-the-user-account}

Nell'amministrazione di WordPress:
- Vai su Utenti → Tutti gli utenti
- Verifica che l'account esista
- Controlla che abbia le autorizzazioni di accesso appropriate al POS

### 4. Cancella le credenziali salvate {#4-clear-saved-credentials}

Nel POS:
- Disconnettiti completamente
- Cancella eventuali informazioni di accesso salvate/ memorizzate nella cache
- Inserisci di nuovo le credenziali fresche
