# API05003: Plugin WCPOS Obsoleto

## Cosa Significa

Il plugin WCPOS installato sul tuo server è obsoleto e non è compatibile con la tua versione dell'applicazione POS. Devi aggiornare il plugin.

## Cause Comuni

- **Plugin non aggiornato** — Gli aggiornamenti automatici potrebbero essere disattivati
- **Aggiornamento fallito** — Un precedente tentativo di aggiornamento non è riuscito
- **Discrepanza di versione** — App POS aggiornata ma plugin no
- **Sito di staging/sviluppo** — Test su una versione precedente

## Come Risolvere

### 1. Aggiornare il Plugin

In Amministrazione WordPress:
1. Vai a Plugin → Plugin Installati
2. Trova "WooCommerce POS"
3. Se è disponibile un aggiornamento, fai clic su "Aggiorna Ora"

### 2. Attivare gli Aggiornamenti Automatici

Per prevenire futuri disallineamenti:
1. Vai a Plugin → Plugin Installati
2. Trova "WooCommerce POS"
3. Fai clic su "Abilita aggiornamenti automatici"

### 3. Aggiornamento Manuale

Se l'aggiornamento automatico non funziona:
1. Scarica l'ultima versione da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Disattiva il plugin corrente
3. Elimina i vecchi file del plugin
4. Carica e attiva la nuova versione

### 4. Verificare Errori di Aggiornamento

Se gli aggiornamenti non riescono:
- Controlla i permessi dei file in wp-content/plugins
- Verifica di avere spazio su disco
- Controlla problemi di memoria PHP
- Controlla i registri degli errori per i dettagli

### 5. Downgrade dell'App POS (Temporaneo)

Se non puoi aggiornare immediatamente il plugin:
- Usa una versione precedente dell'app POS
- Questa è solo una soluzione temporanea
- Pianifica di aggiornare presto il plugin

## Compatibilità di Versione

Mantieni sempre entrambi i componenti aggiornati:
- L'applicazione POS
- Il plugin WordPress WCPOS

Controlla le note di rilascio per eventuali modifiche importanti durante l'aggiornamento.
