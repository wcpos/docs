# API05003: Plugin WCPOS Obsoleto

## Cosa Significa

Il plugin WCPOS installato sul tuo server è obsoleto e non è compatibile con la tua versione dell'applicazione POS. Devi aggiornare il plugin.

## Cause Comuni

- **Plugin non aggiornato** — Gli aggiornamenti automatici potrebbero essere disabilitati
- **Aggiornamento fallito** — Un precedente tentativo di aggiornamento non è riuscito
- **Incompatibilità di versione** — L'app POS è stata aggiornata ma il plugin no
- **Sito di staging/dev** — Test su una versione più vecchia

## Come Risolvere

### 1. Aggiorna il Plugin

In WordPress Admin:
1. Vai su Plugin → Plugin Installati
2. Trova "WooCommerce POS"
3. Se è disponibile un aggiornamento, fai clic su "Aggiorna Ora"

### 2. Abilita gli Aggiornamenti Automatici

Per prevenire future incompatibilità:
1. Vai su Plugin → Plugin Installati
2. Trova "WooCommerce POS"
3. Fai clic su "Abilita aggiornamenti automatici"

### 3. Aggiornamento Manuale

Se l'aggiornamento automatico fallisce:
1. Scarica l'ultima versione da [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Disattiva il plugin attuale
3. Elimina i vecchi file del plugin
4. Carica e attiva la nuova versione

### 4. Controlla gli Errori di Aggiornamento

Se gli aggiornamenti stanno fallendo:
- Controlla i permessi dei file su wp-content/plugins
- Verifica di avere spazio su disco
- Cerca problemi di memoria PHP
- Controlla i log degli errori per dettagli

### 5. Esegui il Downgrade dell'App POS (Temporaneo)

Se non puoi aggiornare immediatamente il plugin:
- Usa una versione più vecchia dell'app POS
- Questa è solo una soluzione temporanea
- Prevedi di aggiornare il plugin presto

## Compatibilità delle Versioni

Mantieni sempre entrambi i componenti aggiornati:
- L'applicazione POS
- Il plugin WordPress WCPOS

Controlla le note di rilascio per eventuali modifiche significative durante l'aggiornamento.
