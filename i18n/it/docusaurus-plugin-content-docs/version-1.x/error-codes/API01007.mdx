# API01007: Dispositivo Offline {#api01007-device-offline}

## Cosa Significa {#what-this-means}

Il tuo dispositivo non è connesso a Internet. Il POS ha rilevato che non c'è alcuna connessione di rete attiva.

## Cause Comuni {#common-causes}

- **WiFi disconnesso** — Il tuo dispositivo ha perso la connessione WiFi
- **Ethernet scollegato** — Il cavo di rete è scollegato
- **Modalità aereo attivata** — Tutte le connessioni wireless sono disattivate
- **Adattatore di rete disattivato** — L'interfaccia di rete è spenta

## Come Risolvere {#how-to-fix}

### 1. Controlla la Connessione WiFi {#1-check-wifi-connection}

- Verifica che il WiFi sia attivato sul tuo dispositivo
- Assicurati di essere connesso alla rete corretta
- Controlla la potenza del segnale WiFi

### 2. Controlla le Connessioni Fisiche {#2-check-physical-connections}

Se utilizzi ethernet:
- Verifica che il cavo sia collegato saldamente
- Prova un cavo o una porta diversi
- Controlla le luci dello switch/router di rete

### 3. Disabilita la Modalità Aereo {#3-disable-airplane-mode}

Controlla se la modalità aereo è attivata per errore e disattivala.

### 4. Riavvia l'Adattatore di Rete {#4-restart-network-adapter}

**Windows:**
- Apri le Connessioni di Rete
- Fai clic con il tasto destro sul tuo adattatore → Disabilita, quindi Attiva

**macOS:**
- Spegni e riaccendi il WiFi dalla barra dei menu
- Oppure: Preferenze di Sistema → Rete → Spegni/accendi

### 5. Riavvia il Tuo Dispositivo {#5-restart-your-device}

A volte un semplice riavvio risolve i problemi di rilevamento della rete.

## Modalità Offline {#offline-mode}

Mentre sei offline, WCPOS può comunque:
- Sfogliare e cercare prodotti memorizzati nella cache localmente
- Sfogliare e cercare clienti memorizzati nella cache localmente
- Iniziare nuovi ordini e aggiungere articoli al carrello
- Visualizzare i dati precedentemente sincronizzati

**Richiede connettività:**
- Completare/controllare gli ordini
- Creare nuovi clienti

Quando la connettività viene ripristinata, il POS riprenderà automaticamente le normali operazioni.
