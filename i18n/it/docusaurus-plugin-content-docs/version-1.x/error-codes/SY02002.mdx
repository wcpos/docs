# SY02002: Servizio non disponibile

## Cosa significa

Un servizio o componente necessario non è disponibile. Il POS dipende da vari servizi per funzionare e uno di essi è attualmente non raggiungibile.

## Cause comuni

- **Servizio in background interrotto** — Un servizio necessario non è in esecuzione
- **Problema di risorse di sistema** — Risorse insufficienti per avviare il servizio
- **Guasto di dipendenza** — Un servizio da cui dipende è fallito
- **Problema di avvio** — Il servizio non si è avviato correttamente
- **Errore del server** — Il server WooCommerce ha restituito un errore 5xx

## Mappatura degli errori del server

Questo codice di errore viene attivato quando il server restituisce:

| Codice Server | Origine |
|-------------|--------|
| HTTP 5xx | Qualsiasi risposta di errore del server (500, 502, 503, 504, ecc.) |

## Come risolvere

### 1. Riavviare l'applicazione

Chiudere e riaprire il POS:
1. Chiudere completamente l'applicazione
2. Attendere qualche secondo
3. Riavviare

### 2. Riavviare il dispositivo

Un riavvio completo spesso risolve i problemi di servizio:
1. Salvare eventuali lavori importanti
2. Riavviare il computer
3. Provare nuovamente il POS

### 3. Verificare le risorse di sistema

Assicurarsi che le risorse di sistema siano disponibili:
- Chiudere applicazioni non necessarie
- Controllare l'utilizzo della memoria
- Controllare lo spazio su disco

### 4. Aggiornare l'applicazione

Assicurarsi di essere sull'ultima versione:
- Verificare la disponibilità di aggiornamenti
- Installare eventuali aggiornamenti in sospeso
- Riavviare dopo l'aggiornamento

### 5. Reinstallare se necessario

Se il servizio rimane non disponibile:
1. Disinstallare l'applicazione
2. Riavviare il dispositivo
3. Scaricare l'ultima versione
4. Installare nuovamente

## Specifico per Browser (Versione Web)

Per la versione web:
- Assicurarsi che JavaScript sia abilitato
- Verificare che le funzionalità del browser richieste non siano bloccate
- Provare un browser diverso
- Disabilitare le estensioni del browser che potrebbero interferire

## Quando contattare l'assistenza

Se l'errore persiste:
- Annotare quando si verifica
- Verificare se azioni specifiche lo attivano
- Segnalare con dettagli per indagare
