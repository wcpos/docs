# SY02002: Servizio Non Disponibile {#sy02002-service-unavailable}

## Cosa Significa {#what-this-means}

Un servizio o componente richiesto non è disponibile. Il POS dipende da vari servizi per funzionare e uno di essi è attualmente inaccessibile.

## Cause Comuni {#common-causes}

- **Servizio in background fermo** — Un servizio richiesto non è in esecuzione
- **Problema di risorse di sistema** — Risorse insufficienti per avviare il servizio
- **Errore di dipendenza** — Un servizio da cui dipende ha fallito
- **Problema di avvio** — Il servizio non è riuscito ad avviarsi correttamente
- **Errore del server** — Il server WooCommerce ha restituito un errore 5xx

## Mappatura degli Errori del Server {#server-error-mapping}

Questo codice di errore viene attivato quando il server restituisce:

| Codice del Server | Origine |
|-------------------|---------|
| HTTP 5xx          | Qualsiasi risposta di errore del server (500, 502, 503, 504, ecc.) |

## Come Risolvere {#how-to-fix}

### 1. Riavvia l'Applicazione {#1-restart-the-application}

Chiudi e riapri il POS:
1. Chiudi completamente l'applicazione
2. Aspetta alcuni secondi
3. Riavviala

### 2. Riavvia il Tuo Dispositivo {#2-restart-your-device}

Un riavvio completo spesso risolve i problemi di servizio:
1. Salva eventuali lavori importanti
2. Riavvia il computer
3. Prova di nuovo il POS

### 3. Controlla le Risorse di Sistema {#3-check-system-resources}

Assicurati che le risorse di sistema siano disponibili:
- Chiudi applicazioni non necessarie
- Controlla l'uso della memoria
- Controlla lo spazio su disco

### 4. Aggiorna l'Applicazione {#4-update-the-application}

Assicurati di avere l'ultima versione:
- Controlla se ci sono aggiornamenti disponibili
- Installa eventuali aggiornamenti in sospeso
- Riavvia dopo l'aggiornamento

### 5. Reinstalla se Necessario {#5-reinstall-if-necessary}

Se il servizio rimane non disponibile:
1. Disinstalla l'applicazione
2. Riavvia il tuo dispositivo
3. Scarica l'ultima versione
4. Installa nuovamente

## Specifico per Browser (Versione Web) {#browser-specific-web-version}

Per la versione web:
- Assicurati che JavaScript sia abilitato
- Controlla che le funzionalità del browser richieste non siano bloccate
- Prova un browser diverso
- Disabilita le estensioni del browser che potrebbero interferire

## Quando Contattare il Supporto {#when-to-contact-support}

Se questo errore persiste:
- Nota quando si verifica
- Controlla se azioni specifiche lo attivano
- Riporta con dettagli per l'indagine
