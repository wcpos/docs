L'editor dei Template per Ricevute ti permette di personalizzare l'aspetto e il contenuto delle ricevute stampate dal tuo sistema POS. Puoi accedere all'editor dei template navigando in: `WP Admin > POS > Templates`

## Panoramica

Il sistema dei template fornisce due tipi di template:

- **Template Predefiniti**: Rilevati automaticamente dai file del plugin e del tema. Non possono essere eliminati ma possono essere copiati per creare versioni personalizzate.
- **Template Personalizzati**: Template creati dall'utente che puoi personalizzare, modificare e gestire completamente.

## Accesso all'Editor dei Template

1. Naviga nel pannello di amministrazione di WordPress
2. Vai su **POS > Templates**
3. Vedrai un elenco di template disponibili organizzati in due sezioni:
   - **Template Predefiniti** (dai file di plugin/tema)
   - **Template Personalizzati** (tue creazioni personalizzate)

## Template Predefiniti

Il sistema rileva automaticamente i template delle ricevute da tre fonti:

### Sorgenti Template

| Sorgente        | Descrizione                                          | Stato                                |
| -------------- | --------------------------------------------------- | ------------------------------------ |
| **Plugin**     | Template base incluso con WooCommerce POS           | Sempre disponibile                   |
| **Pro Plugin** | Template avanzato da WooCommerce POS Pro            | Disponibile con licenza attiva       |
| **Tema**       | Template personalizzato dal tuo tema attivo         | Disponibile se il tema include un template |

### Azioni sui Template

Per ogni template predefinito, puoi:

- **Anteprima**: Visualizzare come appare il template con dati di esempio
- **Copia**: Creare una copia personalizzata modificabile
- **Attiva**: Impostare come template attivo per il tuo POS

## Creazione di Template Personalizzati

### Metodo 1: Copia da Template Predefinito

1. Nella sezione **Template Predefiniti**, trova il template che vuoi personalizzare
2. Fai clic sul pulsante **Copia**
3. Inserisci un nome per il tuo template personalizzato
4. Fai clic su **Copia Template**
5. Modifica il template appena creato

### Metodo 2: Creazione da Zero

1. Fai clic su **Aggiungi Nuovo Template** in alto nella pagina
2. Inserisci un nome per il template
3. Seleziona **Ricevuta** come tipo di template
4. Inizia a codificare il tuo template nell'editor

## Funzionalità dell'Editor di Template

L'editor dei template offre:

### Editor di Codice

- **Evidenziazione Sintassi**: Evidenziazione sintassi PHP per una migliore leggibilità del codice
- **Numeri di Riga**: Navigazione e debug facilitati
- **Modifica a Schermo Intero**: Ambiente di editing senza distrazioni

### Pannello Impostazioni Template

- **Lingua**: Impostata su PHP per la corretta evidenziazione della sintassi
- **Tipo di Template**: Scegli tra Template Ricevute o Template Report
- **Stato**: Gestione stato Pubblicato/Bozza
- **Visibilità**: Controllo visibilità template

### Azioni sui Template

- **Anteprima**: Vedi come viene visualizzato il tuo template con dati di esempio
- **Imposta come Attivo**: Rendi questo template predefinito per il tuo POS
- **Aggiorna**: Salva le tue modifiche
- **Sposta nel Cestino**: Elimina template personalizzati

## Struttura del Template

I template delle ricevute sono file PHP che generano output HTML. Ecco la struttura base:

```php
<?php
/**
 * Template Ricevuta Personalizzato
 */

if ( ! defined( 'ABSPATH' ) ) {
    exit; // Esci se acceduto direttamente
}
?>
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
    <meta charset="<?php bloginfo( 'charset' ); ?>">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <style>
        /* Stili CSS personalizzati */
        body { font-family: sans-serif; font-size: 14px; }
        .receipt-header { text-align: center; margin-bottom: 20px; }
        /* Aggiungi altri stili secondo necessità */
    </style>
</head>
<body>
    <!-- Contenuto ricevuta -->
    <div class="receipt-header">
        <h1><?php echo get_bloginfo( 'name' ); ?></h1>
        <p><?php echo get_bloginfo( 'description' ); ?></p>
    </div>

    <!-- Dettagli ordine, articoli, totali, ecc. -->

</body>
</html>
```
