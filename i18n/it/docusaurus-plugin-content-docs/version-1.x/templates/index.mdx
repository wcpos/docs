I modelli ti permettono di personalizzare l'aspetto e il contenuto di vari output dal tuo sistema POS. Puoi accedere all'editor dei modelli navigando su: `WP Admin > POS > Modelli`

Il sistema di modelli offre flessibilit√† per la personalizzazione:

- **Modelli di Ricevuta** - Personalizza le ricevute stampate e digitali
- **Modelli di Report** - Personalizza i layout dei report _(in arrivo a breve)_
- **Modelli di Email** - Personalizza le notifiche email _(in arrivo a breve)_

## Tipi di Modelli

### Modelli Predefiniti

Rilevati automaticamente dai file del tuo plugin e tema. Questi non possono essere eliminati ma possono essere copiati per creare versioni personalizzate.

### Modelli Personalizzati

Modelli creati dagli utenti che puoi personalizzare completamente, modificare e gestire tramite l'interfaccia di amministrazione di WordPress.
