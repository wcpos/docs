I modelli ti consentono di personalizzare l'aspetto e il contenuto di vari output dal tuo sistema POS. Puoi accedere all'editor di modelli navigando a: `WP Admin > POS > Template`

Il sistema di modelli offre flessibilit√† per la personalizzazione:

- **Modelli di Ricevuta** - Personalizza le ricevute stampate e digitali
- **Modelli di Report** - Personalizza i layout dei report _(in arrivo)_
- **Modelli di Email** - Personalizza le notifiche email _(in arrivo)_

## Tipi di Modelli {#template-types}

### Modelli Predefiniti {#default-templates}

Rilevati automaticamente dai file del tuo plugin e tema. Questi non possono essere eliminati ma possono essere copiati per creare versioni personalizzate.

### Modelli Personalizzati {#custom-templates}

Modelli creati dagli utenti che puoi personalizzare, modificare e gestire completamente tramite l'interfaccia di amministrazione di WordPress.
