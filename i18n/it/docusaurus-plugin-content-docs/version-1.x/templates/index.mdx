I modelli consentono di personalizzare l'aspetto e il contenuto di vari output del proprio sistema POS. È possibile accedere all'editor dei modelli navigando in: `WP Admin > POS > Modelli`

Il sistema dei modelli offre flessibilità per personalizzare:

- **Modelli di Ricevute** - Personalizza ricevute stampate e digitali
- **Modelli di Report** - Personalizza layout dei report _(prossimamente)_
- **Modelli Email** - Personalizza notifiche email _(prossimamente)_

## Tipi di Modelli

### Modelli Predefiniti

Rilevati automaticamente dai file del plugin e del tema. Non possono essere eliminati ma possono essere copiati per creare versioni personalizzate.

### Modelli Personalizzati

Modelli creati dall'utente che è possibile personalizzare completamente, modificare e gestire tramite l'interfaccia di amministrazione di WordPress.
