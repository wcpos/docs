WCPOS funziona con i dispositivi fisici già in uso al bancone. Questa pagina è il punto di partenza per collegare l'hardware: stampanti per scontrini, lettori di codici a barre, lettori di carte e cassetti portadenaro. Ogni tipo di dispositivo ha una propria pagina di configurazione oppure rimanda alla pagina in cui viene effettivamente configurato.

:::note Una sola pagina per dispositivo
L'hub Hardware indirizza alla configurazione di ciascun dispositivo, ma le istruzioni effettive si trovano in un'unica pagina. L'account e la procedura di abbinamento di un lettore di carte si trovano nella relativa pagina Gateway di pagamento; l'impostazione di apertura di un cassetto portadenaro si trova nella pagina Stampanti per ricevute.
:::
