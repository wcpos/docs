:::info Funzionalità Pro
La schermata Clienti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti della versione gratuita possono selezionare clienti esistenti per gli ordini, ma non possono visualizzare l'elenco completo dei clienti né modificarne i dettagli.
:::

La schermata Clienti offre una gestione completa dei clienti direttamente dal POS. È possibile visualizzare, modificare e creare clienti senza passare al pannello di amministrazione di WooCommerce.

{/* Screenshot: customers-screen.png */}

## Panoramica dell'interfaccia {#interface-overview}

### Azioni dell'intestazione {#header-actions}

Nella parte superiore della schermata:

- **Barra di ricerca** — Ricerca clienti per nome, e-mail, ecc.
- **Aggiungi cliente** (<Icon name="plus" />) - Crea un nuovo cliente
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Tabella clienti {#customer-table}

L'area principale mostra i clienti con:

- **Avatar** - Immagine del profilo del cliente (o segnaposto)
- **Nome** - Nome del cliente
- **Cognome** - Cognome del cliente (ordinabile)
- **E-mail** - Indirizzo e-mail di contatto
- **Indirizzo di fatturazione** - Indirizzo di fatturazione completo
- **Data di creazione** - Data di aggiunta del cliente
- **Azioni** - Menu a tre punti

### Piè di pagina {#footer}

- Conteggio clienti con pulsante di sincronizzazione (<Icon name="arrow-rotate-right" />). **Pressione prolungata** per l'opzione Cancella e Aggiorna

## Funzionalità principali {#key-features}

### Ricerca clienti {#customer-search}

Trova rapidamente i clienti tramite:

- Nome o cognome
- Indirizzo e-mail
- Informazioni sull'indirizzo

### Aggiungi nuovo cliente {#add-new-customer}

Crea clienti direttamente dal POS:

1. Fare clic sull'icona <Icon name="plus" /> nell'intestazione
2. Compilare i dati del cliente
3. Salvare il nuovo cliente

Il cliente viene creato in WooCommerce ed è immediatamente disponibile per gli ordini.

### Modificare un cliente {#edit-customer}

Aggiornare le informazioni del cliente:

1. Fare clic sul menu a tre punti di un cliente
2. Selezionare **Modifica**
3. Modificare i dati (nome, e-mail, indirizzi)
4. Salvare le modifiche

Le modifiche vengono sincronizzate automaticamente con WooCommerce.

## Impostazioni di visualizzazione {#display-settings}

Fai clic sull'icona dei cursori (<Icon name="sliders" />) per personalizzare le colonne visibili.

## Azioni del cliente {#customer-actions}

Fare clic sul menu a tre punti (⋮) per le opzioni:

- **Modifica** - Modifica i dettagli del cliente
- **Sincronizza** - Aggiorna il cliente dal server
- **Elimina** - Rimuovi dal database locale

:::note
L'eliminazione di un cliente dal POS lo rimuove solo localmente. Il cliente rimane in WooCommerce e riapparirà alla sincronizzazione successiva.
:::

## Cliente predefinito (Ospite) {#default-customer-guest}

Quando non è selezionato alcun cliente, gli ordini vengono effettuati come ordini **Ospite**. Gli ordini Ospite:

- Non hanno associati nome del cliente, indirizzo email o indirizzo
- Non possono essere cercati per cliente nello storico ordini
- Sono comunque visibili nella schermata Ordini e nell'amministrazione di WooCommerce

Per evitare ordini Ospite, selezionare un cliente dal Pannello carrello prima del checkout. Se l'attività richiede un cliente per ogni ordine, è consigliabile istruire i cassieri ad assegnarne sempre uno — non esiste un'impostazione integrata per rendere obbligatoria questa operazione.

## Utilizzo dei clienti negli ordini {#using-customers-in-orders}

Per assegnare un cliente a un ordine:

1. Nel [Pannello carrello](/pos/cart), fare clic sul badge cliente
2. Cercare il cliente per nome, indirizzo e-mail o numero di telefono
3. Selezionare il cliente
4. Il cliente è ora associato all'ordine

Le informazioni del cliente (indirizzi di fatturazione/spedizione) vengono utilizzate automaticamente per l'ordine.

## Sincronizzazione {#synchronization}

I dati dei clienti si sincronizzano tra il POS e WooCommerce:

- **Aggiornamenti in tempo reale** - Le modifiche si sincronizzano su tutti i dispositivi
- **Integrazione con WooCommerce** - I clienti sono memorizzati in WooCommerce
- **Funzionalità offline** - Consultare i clienti anche senza connessione
