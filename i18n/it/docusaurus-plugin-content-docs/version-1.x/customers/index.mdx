:::info Pro Feature
Lo schermo Clienti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti gratuiti possono selezionare clienti esistenti per gli ordini ma non possono visualizzare l'elenco completo dei clienti o modificare i dettagli dei clienti.
:::

Lo schermo Clienti fornisce una gestione completa dei clienti direttamente all'interno del POS. Visualizza, modifica e crea clienti senza dover passare all'amministratore di WooCommerce.

{/* Screenshot: customers-screen.png */}

## Panoramica dell'interfaccia

### Azioni dell'intestazione

Nella parte superiore dello schermo:

- **Barra di ricerca** - Trova clienti per nome, email, ecc.
- **Aggiungi cliente** (<Icon name="plus" />) - Crea un nuovo cliente
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Tabella Clienti

L'area principale mostra i clienti con:

- **Avatar** - Immagine del profilo del cliente (o segnaposto)
- **Nome** - Nome di battesimo del cliente
- **Cognome** - Cognome del cliente (ordinabile)
- **Email** - Indirizzo email di contatto
- **Indirizzo di Fatturazione** - Indirizzo di fatturazione completo
- **Data di Creazione** - Quando il cliente è stato aggiunto
- **Azioni** - Menu a tre punti

### Piè di pagina

- Numero di clienti con pulsante di sincronizzazione (<Icon name="arrow-rotate-right" />). **Pressione lunga** per l'opzione Cancella e Aggiorna

## Caratteristiche principali

### Ricerca Clienti

Trova clienti rapidamente per:

- Nome o cognome
- Indirizzo email
- Informazioni sull'indirizzo

### Aggiungi Nuovo Cliente

Crea clienti direttamente dal POS:

1. Clicca sull'icona <Icon name="plus" /> nell'intestazione
2. Compila i dettagli del cliente
3. Salva il nuovo cliente

Il cliente viene creato in WooCommerce ed è immediatamente disponibile per gli ordini.

### Modifica Cliente

Aggiorna le informazioni del cliente:

1. Clicca sul menu a tre punti su un cliente
2. Seleziona **Modifica**
3. Modifica i dettagli (nome, email, indirizzi)
4. Salva le modifiche

Le modifiche vengono sincronizzate automaticamente con WooCommerce.

## Impostazioni di Visualizzazione

Clicca sull'icona dei cursori (<Icon name="sliders" />) per personalizzare le colonne visibili.

### Colonne Disponibili

| Colonna | Descrizione |
|---------|-------------|
| **Immagine** | Avatar del cliente |
| **ID** | ID cliente WooCommerce |
| **Nome** | Nome di battesimo del cliente |
| **Cognome** | Cognome del cliente |
| **Email** | Indirizzo email |
| **Ruolo** | Ruolo utente (Cliente, Abbonato, ecc.) |
| **Nome Utente** | Nome utente di WordPress |
| **Indirizzo di Fatturazione** | Informazioni di fatturazione |
| **Indirizzo di Spedizione** | Informazioni di spedizione |
| **Data di Creazione** | Quando è stato aggiunto il cliente |
| **Data Modificata** | Ultimo aggiornamento |
| **Azioni** | Modifica, Sincronizza, Elimina |

## Azioni Cliente

Clicca sul menu a tre punti (⋮) per opzioni:

- **Modifica** - Modifica i dettagli del cliente
- **Sincronizza** - Aggiorna il cliente dal server
- **Elimina** - Rimuovi dal database locale

:::note
Eliminare un cliente dal POS rimuove solo localmente. Il cliente rimane in WooCommerce e riapparirà alla prossima sincronizzazione.
:::

## Utilizzare i Clienti negli Ordini

Per assegnare un cliente a un ordine:

1. Nel [Pannello Carrello](/pos/cart), clicca sul distintivo del cliente
2. Cerca il cliente per nome o email
3. Seleziona il cliente
4. Il cliente è ora associato all'ordine

Le informazioni del cliente (indirizzi di fatturazione/spedizione) vengono utilizzate automaticamente per l'ordine.

## Sincronizzazione

I dati dei clienti vengono sincronizzati tra il POS e WooCommerce:

- **Aggiornamenti in tempo reale** - Le modifiche vengono sincronizzate su tutti i dispositivi
- **Integrazione WooCommerce** - I clienti sono memorizzati in WooCommerce
- **Capacità offline** - Visualizza i clienti anche quando sei offline
