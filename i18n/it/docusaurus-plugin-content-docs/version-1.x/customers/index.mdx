:::info Pro Feature
La schermata Clienti richiede [WCPOS Pro](/getting-started/pro-license). Gli utenti gratuiti possono selezionare clienti esistenti per gli ordini, ma non possono visualizzare l'elenco completo dei clienti o modificare i dettagli dei clienti.
:::

La schermata Clienti fornisce una gestione completa dei clienti direttamente all'interno del POS. Visualizza, modifica e crea clienti senza passare all'amministrazione di WooCommerce.

{/* Screenshot: customers-screen.png */}

## Panoramica dell'interfaccia

### Azioni dell'intestazione

Nella parte superiore dello schermo:

- **Barra di ricerca** - Trova clienti per nome, email, ecc.
- **Aggiungi Cliente** (<Icon name="plus" />) - Crea un nuovo cliente
- **Impostazioni di visualizzazione** (<Icon name="sliders" />) - Configura le colonne visibili

### Tabella Clienti

L'area principale visualizza i clienti con:

- **Avatar** - Immagine del profilo del cliente (o segnaposto)
- **Nome** - Nome di battesimo del cliente
- **Cognome** - Cognome del cliente (ordinabile)
- **Email** - Indirizzo email di contatto
- **Indirizzo di fatturazione** - Indirizzo di fatturazione completo
- **Data di creazione** - Quando il cliente è stato aggiunto
- **Azioni** - Menu a tre punti

### Piè di pagina

- Conteggio clienti con pulsante di sincronizzazione (<Icon name="arrow-rotate-right" />). **Pressione prolungata** per l'opzione Cancella e Aggiorna

## Caratteristiche principali

### Ricerca Clienti

Trova clienti rapidamente per:

- Nome o cognome
- Indirizzo email
- Informazioni sull'indirizzo

### Aggiungi Nuovo Cliente

Crea clienti direttamente dal POS:

1. Fai clic sull'icona <Icon name="plus" /> nell'intestazione
2. Compila i dettagli del cliente
3. Salva il nuovo cliente

Il cliente viene creato in WooCommerce e immediatamente disponibile per gli ordini.

### Modifica Cliente

Aggiorna le informazioni del cliente:

1. Fai clic sul menu a tre punti su un cliente
2. Seleziona **Modifica**
3. Modifica i dettagli (nome, email, indirizzi)
4. Salva le modifiche

Le modifiche si sincronizzano automaticamente con WooCommerce.

### Colonne Disponibili

| Colonna | Descrizione |
|--------|-------------|
| **Immagine** | Avatar del cliente |
| **ID** | ID cliente WooCommerce |
| **Nome** | Nome di battesimo del cliente |
| **Cognome** | Cognome del cliente |
| **Email** | Indirizzo email |
| **Ruolo** | Ruolo dell'utente (Cliente, Iscritto, ecc.) |
| **Nome utente** | Nome utente di WordPress |
| **Indirizzo di fatturazione** | Informazioni di fatturazione |
| **Indirizzo di spedizione** | Informazioni di spedizione |
| **Data di creazione** | Quando è stato aggiunto il cliente |
| **Data di modifica** | Ultimo aggiornamento |
| **Azioni** | Modifica, Sincronizza, Elimina |

## Azioni Cliente

Fai clic sul menu a tre punti (⋮) per le opzioni:

- **Modifica** - Modifica i dettagli del cliente
- **Sincronizza** - Aggiorna il cliente dal server
- **Elimina** - Rimuovi dal database locale

:::note
Eliminare un cliente dal POS rimuove solo localmente. Il cliente rimane in WooCommerce e riapparirà alla prossima sincronizzazione.
:::

## Utilizzo dei Clienti negli Ordini

Per assegnare un cliente a un ordine:

1. Nel [Pannello Carrello](/pos/cart), fai clic sul distintivo del cliente
2. Cerca il cliente per nome o email
3. Seleziona il cliente
4. Il cliente è ora associato all'ordine

Le informazioni del cliente (indirizzi di fatturazione/spedizione) sono automaticamente utilizzate per l'ordine.

## Sincronizzazione

I dati dei clienti si sincronizzano tra il POS e WooCommerce:

- **Aggiornamenti in tempo reale** - Le modifiche si sincronizzano su tutti i dispositivi
- **Integrazione con WooCommerce** - I clienti sono memorizzati in WooCommerce
- **Capacità offline** - Visualizza i clienti anche quando offline
