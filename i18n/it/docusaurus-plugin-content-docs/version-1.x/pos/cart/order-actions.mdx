## Nota Ordine

<div class="image-container">

- Apre una finestra di dialogo per aggiungere una nota cliente all'ordine.
- Le note dei clienti sono visibili al cliente e appaiono sulla ricevuta per impostazione predefinita, nell'area Il mio account del cliente e in `WP Admin > WooCommerce > Ordini > Ordine` sotto l'indirizzo di spedizione.

:::note
Le note degli ordini sono visibili ai clienti.
:::

## Meta Ordine

- Apre un modale per gestire metadati aggiuntivi per l'ordine.
- Le funzionalità includono:
  - Cambiare la valuta dell'ordine.
  - Aggiungere un ID transazione (ad es., da un terminale di pagamento esterno).
  - Aggiungere dati meta personalizzati per l'ordine.

:::note
Funzionalità aggiuntive verranno aggiunte a questo modale nel tempo, inclusa l'integrazione con altri plugin di WooCommerce.
:::

### Che cos'è il Meta Dati in WooCommerce?

Il meta dati fornisce informazioni extra sugli ordini ed è accessibile tramite l'API REST di WooCommerce. Sviluppatori e plugin utilizzano questi dati per estendere la funzionalità, come integrare servizi di terze parti.

### Visualizzazione JSON

- Visualizza la rappresentazione JSON grezza dell'ordine, utile per il debug o scopi di integrazione.

## Salva sul Server

- Ti permette di salvare i carrelli aperti sul server, in modo che siano memorizzati in modo sicuro e accessibili per utilizzi futuri.
- Crea un ordine WooCommerce con stato `pos-open`.

### Perché salvare?

Quando un nuovo ordine viene creato nel POS, esiste solo localmente. 
Salvare l'ordine assicura che persista, anche se il database locale viene svuotato.

Vedi [Ordini Aperti](/pos/cart/open-orders) per ulteriori dettagli sulla gestione degli ordini salvati.

## Annulla

- Due scenari:
  - Ordini non salvati:
    - Se l'ordine non è stato salvato sul server, verrà eliminato permanentemente dal database locale.
  - Ordini salvati:
    - Se l'ordine è salvato sul server, verrà spostato nella cartella Cestino in WooCommerce ed eliminato dal database locale.
- Per recuperare un ordine annullato:
  - Vai a `WP Admin > WooCommerce > Ordini > Cestino`.

## Pagamento

- Salva l'ordine sul server e avvia il processo di pagamento.
- Apre la [schermata di pagamento](/pos/checkout) per completare il pagamento.
