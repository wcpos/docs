## Nota Ordine {#order-note}

- Apre un dialogo per aggiungere una nota cliente all'ordine.
- Le note dei clienti sono visibili al cliente e appaiono nella ricevuta per impostazione predefinita, nell'area I miei ordini del cliente e in `WP Admin > WooCommerce > Ordini > Ordine` sotto l'indirizzo di spedizione.

:::note
Le note sugli ordini sono visibili ai clienti.
:::

- Apre una finestra modale per gestire metadati aggiuntivi per l'ordine.
- Le funzionalità includono:
  - Cambiare la valuta dell'ordine.
  - Aggiungere un ID transazione (ad esempio, da un terminale di pagamento esterno).
  - Aggiungere meta dati personalizzati per l'ordine.

:::note
Funzionalità aggiuntive saranno aggiunte a questa finestra modale nel tempo, inclusa l'integrazione con altri plugin WooCommerce.
:::

### Cos'è il Meta Dati in WooCommerce? {#what-is-meta-data-in-woocommerce}

I meta dati forniscono informazioni extra sugli ordini e sono accessibili tramite l'API REST di WooCommerce. Sviluppatori e plugin utilizzano questi dati per estendere la funzionalità, come l'integrazione con servizi di terze parti.

### Visualizzazione JSON {#json-view}

- Visualizza la rappresentazione JSON grezza dell'ordine, utile per il debug o per scopi di integrazione.

## Salva sul Server {#save-to-server}

- Ti consente di salvare i carrelli aperti sul server, in modo che siano memorizzati in modo sicuro e accessibili per un uso futuro.
- Crea un ordine WooCommerce con stato `pos-open`.

### Perché salvare? {#why-save}

Quando un nuovo ordine viene creato nel POS, esiste solo localmente. 
Salvare l'ordine garantisce che persista, anche se il database locale viene cancellato.

Consulta [Ordini Aperti](/pos/cart/open-orders) per ulteriori dettagli sulla gestione degli ordini salvati.

## Annulla {#void}

- Due scenari:
  - Ordini Non Salvati:
    - Se l'ordine non è stato salvato sul server, sarà eliminato permanentemente dal database locale.
  - Ordini Salvati:
    - Se l'ordine è salvato sul server, verrà spostato nella cartella Cestino in WooCommerce e cancellato dal database locale.
- Per recuperare un ordine annullato:
  - Vai a `WP Admin > WooCommerce > Ordini > Cestino`.

## Checkout

- Salva l'ordine sul server e avvia il processo di checkout.
- Apre la [schermata di checkout](/pos/checkout) per completare il pagamento.
