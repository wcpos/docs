Quando sei pronto a completare una vendita, fai clic sul pulsante **Checkout** per aprire il modulo di checkout. Qui è dove elabori il pagamento e completi l'ordine.

{/* Screenshot: checkout-modal.png */}

## Panoramica del Modulo di Checkout

Il modulo di checkout visualizza:

- **Numero d'ordine** - L'ID dell'ordine di WooCommerce
- **Importo da pagare** - Importo totale dovuto
- **Pulsante Impostazioni Checkout** - Risolvi i problemi di visualizzazione
- **Cassa** - Chi sta elaborando l'ordine
- **Cliente** - Il cliente per questo ordine (link cliccabile)
- **Campo codice coupon** - Applica i codici sconto
- **Riepilogo ordine** - Prodotti, quantità e totali
- **Metodi di pagamento** - Opzioni di pagamento disponibili
- **Annulla / Elabora pagamento** pulsanti

## Metodi di Pagamento

### Disponibili nella Versione Gratuita

La versione gratuita di WCPOS include due gateway di pagamento:

- **Contante** - Con importo dato e calcolatore di resto
- **Carta** - Per terminali di carta esterni

### Gateway Aggiuntivi (Pro)

:::info Caratteristica Pro
I gateway di pagamento aggiuntivi richiedono [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puoi abilitare:

- **Terminale Stripe** - Integrazione diretta con i lettori di carte Stripe
- **Terminale SumUp** - Integrazione con i lettori di carte SumUp
- **Gateway Personalizzati** - Crea le tue integrazioni di pagamento

Consulta [Pagamento](/payment) per i dettagli sulla configurazione dei gateway.

### Selezionare un Metodo di Pagamento

Fai clic su un metodo di pagamento per selezionarlo. Il modulo si aggiorna per mostrare i campi pertinenti:

**Contante:**

- **Importo Dato** - Inserisci l'importo che il cliente ti consegna
- **Restante** - Restante calcolato automaticamente da restituire

{/* Screenshot: checkout-cash-payment.png */}

**Carta:**

- Elabora il pagamento sul tuo terminale di carta esterno
- Fai clic su Elabora Pagamento per completare

## Codici Coupon

Per applicare uno sconto:

1. Inserisci il codice coupon nel campo
2. Fai clic su **Applica coupon**
3. Lo sconto appare nel riepilogo dell'ordine

I coupon devono essere configurati in WooCommerce (`WP Admin > Marketing > Coupons`).

## Elaborazione del Pagamento

1. Seleziona un metodo di pagamento
2. Per il contante, inserisci l'importo dato
3. Fai clic su **Elabora Pagamento**
4. L'ordine è completato e la [ricevuta](/pos/checkout/receipts) viene mostrata

## Impostazioni del Checkout (Risoluzione dei Problemi)

Il modulo di checkout utilizza un iframe/webview per visualizzare la pagina di pagamento dell'Ordine WooCommerce. Questo sfrutta l'infrastruttura di pagamento esistente di WooCommerce, il che significa che qualsiasi gateway di pagamento che funziona con WooCommerce dovrebbe funzionare nel POS.

Tuttavia, gli script del tema e dei plugin possono talvolta interferire. Fai clic su **Impostazioni Checkout** per risolvere i problemi:

### Disabilita Tutti gli Stili e gli Script

:::warning Opzione Nucleare
Questa è l'opzione nucleare e dovrebbe essere utilizzata solo per test o in rari casi in cui lo sviluppatore sa cosa sta facendo. Disabilitare tutti gli script wp_head rimuoverà anche gli script necessari di WooCommerce per espandere/contrarre i gateway di pagamento, interrompendo potenzialmente la funzionalità di pagamento.
:::

- **Disabilita wp_head** - Rimuove tutti gli script/stili dall'intestazione di WordPress
- **Disabilita wp_footer** - Rimuove tutti gli script/stili dal footer di WordPress

### Disabilita Stili Selezionati

Disabilita selettivamente il CSS che potrebbe causare problemi di visualizzazione:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- ecc.

### Disabilita Script Selezionati

Disabilita selettivamente il JavaScript che potrebbe interferire con i gateway di pagamento:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- ecc.

:::tip
Se un gateway di pagamento non viene visualizzato correttamente:

1. Prova a disabilitare prima gli stili del tema
2. Poi prova a disabilitare gli script di WooCommerce non necessari
3. Fai attenzione a non disabilitare gli script richiesti dal tuo gateway di pagamento
:::

## Annulla

Fai clic su **Annulla** per chiudere il modulo di checkout senza completare l'ordine. L'ordine rimane come un carrello aperto.
