Quando sei pronto per completare una vendita, fai clic sul pulsante **Pagamento** per aprire il modulo di pagamento. Qui è dove elabori il pagamento e completi l'ordine.

{/* Screenshot: checkout-modal.png */}

## Panoramica del Modulo di Pagamento

Il modulo di pagamento visualizza:

- **Numero d'ordine** - L'ID dell'ordine WooCommerce
- **Importo da Pagare** - Totale dovuto
- **Pulsante Impostazioni di Pagamento** - Risolvere problemi di visualizzazione
- **Cassiere** - Chi sta elaborando l'ordine
- **Cliente** - Il cliente per questo ordine (link cliccabile)
- **Campo codice coupon** - Applica i codici sconto
- **Riepilogo dell'ordine** - Prodotti, quantità e totali
- **Metodi di pagamento** - Opzioni di pagamento disponibili
- **Annulla / Elabora Pagamento** - Pulsanti

## Metodi di Pagamento

### Disponibile nella Versione Gratuita

La versione gratuita di WCPOS include due gateway di pagamento:

- **Contante** - Con importo ricevuto e calcolatore di resto
- **Carta** - Per terminali di carte esterni

### Gateway Aggiuntivi (Pro)

:::info Funzionalità Pro
I gateway di pagamento aggiuntivi richiedono [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puoi abilitare:

- **Terminale Stripe** - Integrazione diretta con lettori di carte Stripe
- **Terminale SumUp** - Integrazione con lettori di carte SumUp
- **Gateway Personalizzati** - Crea le tue integrazioni di pagamento

Vedi [Pagamento](/payment) per dettagli sulla configurazione dei gateway.

### Selezionare un Metodo di Pagamento

Fai clic su un metodo di pagamento per selezionarlo. Il modulo si aggiorna per mostrare i campi pertinenti:

**Contante:**
- **Importo Ricevuto** - Inserisci l'importo che il cliente ti dà
- **Resto** - Resto calcolato automaticamente da restituire

{/* Screenshot: checkout-cash-payment.png */}

**Carta:**
- Elabora il pagamento sul tuo terminale di carta esterno
- Fai clic su Elabora Pagamento per completare

## Codici Coupon

Per applicare uno sconto:

1. Inserisci il codice coupon nel campo
2. Fai clic su **Applica coupon**
3. Lo sconto appare nel riepilogo dell'ordine

I coupon devono essere configurati in WooCommerce (`WP Admin > Marketing > Coupon`).

## Elaborazione del Pagamento

1. Seleziona un metodo di pagamento
2. Per il contante, inserisci l'importo ricevuto
3. Fai clic su **Elabora Pagamento**
4. L'ordine è completato e la [ricevuta](/pos/checkout/receipts) viene mostrata

## Impostazioni di Pagamento (Risoluzione dei Problemi)

Il modulo di pagamento utilizza un iframe/webview per visualizzare la pagina di pagamento dell'ordine WooCommerce. Questo sfrutta l'infrastruttura di pagamento esistente di WooCommerce, il che significa che qualsiasi gateway di pagamento che funziona con WooCommerce dovrebbe funzionare nel POS.

Tuttavia, gli script di temi e plugin possono talvolta interferire. Fai clic su **Impostazioni di Pagamento** per risolvere i problemi:

### Disabilita Tutti gli Stili e gli Script

- **Disabilita wp_head** - Rimuove tutti gli script/stili dall'intestazione di WordPress
- **Disabilita wp_footer** - Rimuove tutti gli script/stili dal piè di pagina di WordPress

### Disabilita Stili Selezionati

Disabilita selettivamente CSS che possono causare problemi di visualizzazione:

- wp-emoji-styles
- wp-block-library
- classic-theme-styles
- woocommerce-layout
- woocommerce-smallscreen
- woocommerce-general
- ecc.

### Disabilita Script Selezionati

Disabilita selettivamente JavaScript che potrebbe interferire con i gateway di pagamento:

- wc-add-to-cart
- selectWoo
- wc-checkout
- woocommerce
- html5shiv
- ecc.

:::tip
Se un gateway di pagamento non viene visualizzato correttamente:
1. Prova a disabilitare prima gli stili del tema
2. Poi prova a disabilitare gli script di WooCommerce che non sono necessari
3. Fai attenzione a non disabilitare gli script richiesti dal tuo gateway di pagamento
:::

## Annulla

Fai clic su **Annulla** per chiudere il modulo di pagamento senza completare l'ordine. L'ordine rimane come un carrello aperto.
