# WCPOS StoreApps Smart Coupons {#wcpos-storeapps-smart-coupons}

[StoreApps / WooCommerce.com के Smart Coupons](https://woocommerce.com/products/smart-coupons/) से बनाए गए store credit के लिए WCPOS compatibility जोड़ता है। यह POS orders को StoreApps store credit redeem करने देता है, StoreApps balance को sync में रखता है, और staff के लिए order-note audit trail छोड़ता है।

यह extension खास तौर पर StoreApps / WooCommerce.com Smart Coupons plugin के लिए है। मिलते-जुलते "Smart Coupons" नाम वाले दूसरे plugins अलग products हैं और उन्हें अलग integrations की ज़रूरत हो सकती है।

:::info Pro सुविधा
Directory से extensions install और manage करने के लिए [WCPOS Pro](/getting-started/pro-license) आवश्यक है।
:::

## यह क्या करता है {#what-it-does}

- POS-created orders के लिए StoreApps `smart_coupons_contribution` metadata रिकॉर्ड करता है, जब सामान्य checkout cart context उपलब्ध नहीं होता।
- POS checkout के बाद StoreApps को coupon balance से सही partial store-credit amount deduct करने देता है।
- जब POS order WooCommerce के सामान्य order lifecycle से cancel या refund होता है, StoreApps store-credit balance restore करता है।
- Redemption के बाद coupon code, उपयोग की गई amount और current balance दिखाने वाली private order note जोड़ता है।
- Coupon description या discount label में जोड़कर receipt की existing discount row में बचा हुआ store-credit balance दिखाता है।

Extension सामान्य WooCommerce coupon fields और metadata का उपयोग करता है। यह custom WCPOS coupon response fields नहीं जोड़ता और custom receipt templates की आवश्यकता नहीं रखता।

## इंस्टॉलेशन {#installation}

<Steps>
  <StepCard number="1" title="StoreApps Smart Coupons इंस्टॉल करें">

[StoreApps / WooCommerce.com के Smart Coupons](https://woocommerce.com/products/smart-coupons/) को install और activate करें। WooCommerce में अपने store-credit coupons या gift cards सामान्य रूप से बनाएँ।

  </StepCard>
  <StepCard number="2" title="WCPOS StoreApps Smart Coupons इंस्टॉल करें">

  </StepCard>
  <StepCard number="3" title="Coupons को POS से sync करें" isLast>

POS खोलें और coupons sync करें। Cashiers फिर सामान्य **Add Coupon** flow से StoreApps store-credit coupons apply कर सकते हैं।

  </StepCard>
</Steps>

## Receipt output {#receipt-output}

WCPOS receipt templates coupon discounts को existing discount row label से ही print करते हैं। Receipt render होते समय यह extension current StoreApps balance को उसी existing label में जोड़ता है।

उदाहरण के लिए **Gift card** जैसी coupon description इस तरह print हो सकती है:

```text
Gift card — Store credit balance: £61.50
```

अगर coupon की कोई description नहीं है, तो balance text discount label के रूप में उपयोग होता है। Existing receipt templates को बदलने की ज़रूरत नहीं है।

## Order audit trail {#order-audit-trail}

StoreApps द्वारा POS store-credit redemption process करने के बाद, extension इस तरह की private order note जोड़ता है:

```text
StoreApps Smart Coupons store credit recorded for WCPOS: Coupon STORE100 used £38.50. Current balance: £61.50.
```

इस note से पुष्टि करें कि कौन सा store-credit coupon उपयोग हुआ, कितना redeem हुआ और sale के बाद कितना balance बचा।

## Compatibility notes {#compatibility-notes}

- **StoreApps only:** यह extension StoreApps / WooCommerce.com Smart Coupons को target करता है। यह WebToffee या दूसरे smart-coupon plugins को target नहीं करता।
- **Coupon data:** WCPOS सामान्य WooCommerce coupon responses का उपयोग करता रहता है। StoreApps-specific state WooCommerce metadata जैसे `smart_coupons_contribution` और coupon balance fields में रहती है।
- **Receipts:** Balance text existing coupon description/discount label path से जोड़ा जाता है, receipt data या templates बदलकर नहीं।
- **Inactive StoreApps plugin:** अगर StoreApps Smart Coupons active नहीं है, extension कुछ नहीं करता।
