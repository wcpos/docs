# SumUp टर्मिनल गेटवे

SumUp टर्मिनल गेटवे आपको WooCommerce POS के भीतर सीधे SumUp कार्ड रीडर्स का उपयोग करके कार्ड भुगतान स्वीकार करने में सक्षम बनाता है। यह गेटवे व्यक्तिगत लेन-देन के लिए SumUp के भुगतान प्रसंस्करण सिस्टम के साथ निर्बाध एकीकरण प्रदान करता है।

## विशेषताएँ

- **हार्डवेयर एकीकरण**: अपने POS सिस्टम को इंटरनेट कनेक्शन के माध्यम से SumUp कार्ड रीडर्स से कनेक्ट करें
- **वास्तविक समय प्रसंस्करण**: त्वरित भुगतान प्रक्रिया और आदेश पूर्णता
- **स्वचालित पहचान**: जब भुगतान का पता चला तो आदेश स्वचालित रूप से पूर्ण हो जाते हैं
- **सुरक्षित लेनदेन**: SumUp के माध्यम से PCI-अनुरूप भुगतान प्रसंस्करण
- **आसान पेयरिंग**: पेयरिंग कोड के साथ सरल डिवाइस पेयरिंग प्रक्रिया
- **वेब-आधारित SDK**: वेब एप्लिकेशन संगतता के लिए SumUp का JavaScript SDK का उपयोग करता है

## स्थापना

### चरण 1: डाउनलोड और स्थापित करें

1. [SumUp टर्मिनल के लिए WooCommerce रिलीज़ पृष्ठ](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases) पर जाएँ
2. नवीनतम **sumup-terminal-for-woocommerce.zip** फ़ाइल डाउनलोड करें
3. अपने WordPress प्रशासन में, `Plugins > Add New > Upload Plugin` पर जाएँ
4. ज़िप फ़ाइल अपलोड करें और प्लगइन सक्रिय करें

### चरण 2: SumUp सेटिंग्स की कॉन्फ़िगर करें

1. `WP Admin > WooCommerce > Settings > Payments` पर जाएँ
2. भुगतान विधियों की सूची में **SumUp टर्मिनल** खोजें
3. सेटिंग्स तक पहुँचने के लिए **SumUp टर्मिनल** पर क्लिक करें
4. अपना **SumUp API की** दर्ज करें (आपके SumUp व्यापारी डैशबोर्ड से उपलब्ध)
5. सेटिंग्स को सहेजें

:::note
आपको WooCommerce सेटिंग्स में SumUp टर्मिनल गेटवे को सक्षम करने की आवश्यकता नहीं है। यह बाद में एक चरण में POS के लिए विशेष रूप से सक्षम किया जाएगा।
:::

### चरण 3: अपने SumUp टर्मिनल को पेयर करें

1. उसी सेटिंग्स पृष्ठ पर, **पेयर रीडर** अनुभाग खोजें
2. अपने SumUp डिवाइस पर, पेयरिंग स्क्रीन पर जाएँ ताकि पेयरिंग कोड प्रदर्शित हो सके
3. अपने SumUp डिवाइस पर प्रदर्शित पेयरिंग कोड दर्ज करें
4. कनेक्शन स्थापित करने के लिए **"Pair Reader"** पर क्लिक करें
5. पुष्टि के लिए प्रतीक्षा करें कि रीडर सफलतापूर्वक पेयर हो चुका है

:::warning महत्वपूर्ण
आप भुगतान प्रक्रिया शुरू करने से पहले रीडर को सफलतापूर्वक पेयर करना होगा। आगे बढ़ने से पहले सुनिश्चित करें कि पेयरिंग प्रक्रिया पूरी हो गई है।
:::

### चरण 4: WooCommerce POS में सक्षम करें

1. `WP Admin > WooCommerce POS > Settings > Checkout` पर जाएँ
2. सूची में **SumUp टर्मिनल** गेटवे खोजें
3. POS में उपयोग के लिए गेटवे को सक्षम करें
4. अपनी सेटिंग्स को सहेजें

## उपयोग

### भुगतान प्रक्रिया

1. **आइटम जोड़ें**: POS में अपनी शॉपिंग कार्ट में उत्पाद जोड़ें
2. **गेटवे का चयन करें**: भुगतान विधि के रूप में "SumUp टर्मिनल" चुनें
3. **भुगतान शुरू करें**: अपने SumUp डिवाइस पर एक नया भुगतान आरंभ करने के लिए क्लिक करें
4. **ग्राहक भुगतान**: ग्राहक SumUp टर्मिनल पर भुगतान पूरा करता है
5. **स्वचालित पूर्णता**: भुगतान का पता चलने पर आदेश स्वचालित रूप से पूरा हो जाता है

### भुगतान नियंत्रण

SumUp टर्मिनल गेटवे का उपयोग करते समय, आपके पास निम्नलिखित विकल्प होते हैं:

- **नया भुगतान शुरू करें**: जुड़े टर्मिनल पर भुगतान अनुरोध प्रारंभ करें
- **भुगतान रद्द करें**: वर्तमान में प्रक्रिया में भुगतान रद्द करें
- **भुगतान स्थिति**: भुगतान प्रक्रिया की वर्तमान स्थिति की निगरानी करें

### आदेश प्रबंधन

- **स्वचालित अपडेट**: सफल भुगतान पर आदेश स्वचालित रूप से पूर्ण के रूप में चिह्नित होते हैं
- **भुगतान ट्रैकिंग**: सभी भुगतान विवरण आदेश नोट्स में रिकॉर्ड किए जाते हैं
- **रसीद उत्पन्न करना**: सफल भुगतान के बाद मानक POS रसीद उत्पन्न होती है

## आवश्यकताएँ

- **SumUp खाता**: सक्रिय SumUp व्यापारी खाता
- **API पहुँच**: अपने व्यापारी डैशबोर्ड से SumUp API की
- **संगत हार्डवेयर**: SumUp कार्ड रीडर डिवाइस
- **WooCommerce POS**: स्थापित और सक्रिय मुफ़्त या प्रो संस्करण
- **स्थिर कनेक्शन**: API संचार के लिए विश्वसनीय इंटरनेट कनेक्शन

## हार्डवेयर संगतता

:::important कनेक्टिविटी आवश्यकताएँ
यह कार्यान्वयन SumUp के JavaScript SDK का उपयोग करता है, जिसका अर्थ है कि यह वेब अनुप्रयोगों के माध्यम से काम करता है लेकिन केवल **इंटरनेट से जुड़े टर्मिनलों की आवश्यकता होती है**। वर्तमान में ब्लूटूथ टर्मिनलों का समर्थन नहीं किया गया है।
:::

### समर्थित टर्मिनल (इंटरनेट-से जुड़े)

- **SumUp Solo** ✅ - WiFi/Ethernet कनेक्टेड टर्मिनल

### असमर्थित टर्मिनल (ब्लूटूथ)

- **SumUp Air** ❌ - केवल ब्लूटूथ
- **SumUp 3G** ❌ - केवल ब्लूटूथ

:::note भविष्य का समर्थन
ब्लूटूथ टर्मिनल समर्थन को **Q1 2026** में iOS और Android अनुप्रयोगों में जोड़ा जाएगा। यह एयर और 3G मॉडल सहित सभी SumUp प्रमाणित टर्मिनलों के लिए समर्थन सक्षम करेगा।
:::

## ट्रबलशूटिंग

### सामान्य समस्याएँ

**रीडर पेयर नहीं हो रहा**
- सुनिश्चित करें कि आप एक समर्थित इंटरनेट-कनेक्टेड टर्मिनल (केवल Solo) का उपयोग कर रहे हैं
- सुनिश्चित करें कि आपका SumUp Solo डिवाइस WiFi से कनेक्टेड और ऑनलाइन है
- जांचें कि डिवाइस पेयरिंग मोड में है और पेयरिंग कोड सही ढंग से दर्ज किया गया है
- पुष्टि करें कि आपके SumUp खाते में API पहुँच सक्षम है
- पेयरिंग के दौरान स्थिर इंटरनेट कनेक्शन सुनिश्चित करें

**भुगतान प्रक्रिया नहीं हो रही**
- सुनिश्चित करें कि रीडर सही ढंग से पेयर हो चुका है
- सुनिश्चित करें कि आपका SumUp खाता सक्रिय और अच्छे स्थिति में है
- सुनिश्चित करें कि डिवाइस में पर्याप्त बैटरी और कनेक्टिविटी है
- पुष्टि करें कि API की सही और सक्रिय है

**आदेश पूर्ण नहीं हो रहा**
- इंटरनेट कनेक्शन की स्थिरता की जांच करें
- अपने SumUp खाते में वेबहुक सेटिंग्स को सत्यापित करें
- सुनिश्चित करें कि SumUp टर्मिनल पर भुगतान सफल रहा
- API संचार समस्याओं के लिए WordPress त्रुटि लॉग की जांच करें

### API दर सीमाएँ

गेटवे API कॉल को कम करने और दर सीमाओं से बचने के लिए अनुकूलित है:
- भुगतान स्थिति को कुशलता से जांचा जाता है
- अनावश्यक API अनुरोधों को न्यूनतम किया गया है
- स्वचालित पुन: प्रयास लॉजिक अस्थायी विफलताओं को संभालता है

### सहायता प्राप्त करना

तकनीकी समर्थन के लिए:
- मुद्दों की रिपोर्ट करने के लिए [GitHub रिपॉजिटरी](https://github.com/wcpos/sumup-terminal-for-woocommerce) पर जाएँ
- API-संबंधी प्रश्नों के लिए [SumUp डेवलपर दस्तावेज़](https://developer.sumup.com/) देखें
- खाता और हार्डवेयर मुद्दों के लिए SumUp समर्थन से संपर्क करें
