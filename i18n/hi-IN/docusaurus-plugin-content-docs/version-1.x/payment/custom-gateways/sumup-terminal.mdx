# SumUp टर्मिनल गेटवे

SumUp टर्मिनल गेटवे आपको WooCommerce POS के भीतर सीधे SumUp कार्ड रीडर का उपयोग करके कार्ड भुगतान स्वीकार करने में सक्षम बनाता है। यह गेटवे मौके पर लेनदेन के लिए SumUp के भुगतान प्रोसेसिंग सिस्टम के साथ सहज एकीकरण प्रदान करता है।

## विशेषताएं

- **हार्डवेयर एकीकरण**: इंटरनेट कनेक्शन के माध्यम से SumUp कार्ड रीडर को अपने POS सिस्टम से कनेक्ट करें
- **रीयल-टाइम प्रोसेसिंग**: तत्काल भुगतान प्रोसेसिंग और ऑर्डर पूरा करना
- **स्वचालित डिटेक्शन**: जब भुगतान का पता चलता है तो ऑर्डर स्वचालित रूप से पूरा होता है
- **सुरक्षित लेनदेन**: SumUp के माध्यम से PCI-अनुपालन भुगतान प्रोसेसिंग
- **आसान जोड़ना**: पेयरिंग कोड के साथ सरल उपकरण जोड़ने की प्रक्रिया
- **वेब-आधारित SDK**: वेब एप्लिकेशन संगतता के लिए SumUp के JavaScript SDK का उपयोग

## इंस्टॉलेशन

### चरण 1: डाउनलोड और इंस्टॉल करें

1. [SumUp टर्मिनल for WooCommerce रिलीज पृष्ठ](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases) पर जाएं
2. नवीनतम **sumup-terminal-for-woocommerce.zip** फ़ाइल डाउनलोड करें
3. अपने WordPress एडमिन में, `Plugins > Add New > Upload Plugin` पर जाएं
4. ज़िप फ़ाइल अपलोड करें और प्लगइन को सक्रिय करें

### चरण 2: SumUp सेटिंग्स कॉन्फ़िगर करें

1. `WP Admin > WooCommerce > Settings > Payments` पर नेविगेट करें
2. भुगतान विधियों की सूची में **SumUp टर्मिनल** ढूंढें
3. सेटिंग्स तक पहुंचने के लिए **SumUp टर्मिनल** पर क्लिक करें
4. अपने **SumUp API Key** दर्ज करें (आपके SumUp मर्चेंट डैशबोर्ड से उपलब्ध)
5. सेटिंग्स सेव करें

:::note
आपको WooCommerce सेटिंग्स में SumUp टर्मिनल गेटवे को सक्षम करने की आवश्यकता नहीं है। इसे बाद के चरण में विशेष रूप से POS के लिए सक्षम किया जाएगा।
:::

### चरण 3: अपना SumUp टर्मिनल जोड़ें

1. उसी सेटिंग्स पेज पर, **रीडर जोड़ें** अनुभाग ढूंढें
2. अपने SumUp डिवाइस पर, पेयरिंग कोड प्रदर्शित करने के लिए पेयरिंग स्क्रीन पर जाएं
3. अपने SumUp डिवाइस पर प्रदर्शित पेयरिंग कोड दर्ज करें
4. कनेक्शन स्थापित करने के लिए **"रीडर जोड़ें"** पर क्लिक करें
5. रीडर सफलतापूर्वक जोड़ा गया है, इसकी पुष्टि के लिए प्रतीक्षा करें

:::warning महत्वपूर्ण
भुगतान प्रोसेस करने से पहले रीडर को सफलतापूर्वक जोड़ा जाना चाहिए। आगे बढ़ने से पहले पेयरिंग प्रक्रिया पूरी होने सुनिश्चित करें।
:::

### चरण 4: WooCommerce POS में सक्षम करें

1. `WP Admin > WooCommerce POS > Settings > Checkout` पर जाएं
2. सूची में **SumUp टर्मिनल** गेटवे ढूंढें
3. POS में उपयोग के लिए गेटवे को सक्षम करें
4. अपनी सेटिंग्स सेव करें

## उपयोग

### भुगतान प्रोसेस करना

1. **आइटम जोड़ें**: POS में अपने कार्ट में उत्पाद जोड़ें
2. **गेटवे चुनें**: भुगतान विधि के रूप में "SumUp टर्मिनल" चुनें
3. **भुगतान शुरू करें**: अपने SumUp डिवाइस पर नया भुगतान शुरू करने के लिए क्लिक करें
4. **ग्राहक भुगतान**: ग्राहक SumUp टर्मिनल पर भुगतान पूरा करता है
5. **स्वचालित पूर्ति**: जब भुगतान का पता चलता है तो ऑर्डर स्वचालित रूप से पूरा होता है

### भुगतान नियंत्रण

SumUp टर्मिनल गेटवे का उपयोग करते समय, आपके पास निम्न विकल्प हैं:

- **नया भुगतान शुरू करें**: कनेक्टेड टर्मिनल पर भुगतान अनुरोध शुरू करें
- **भुगतान रद्द करें**: वर्तमान में प्रक्रियाधीन भुगतान को रद्द करें
- **भुगतान स्थिति**: भुगतान प्रक्रिया की वर्तमान स्थिति की निगरानी करें

### ऑर्डर प्रबंधन

- **स्वचालित अपडेट**: सफल भुगतान पर ऑर्डर स्वचालित रूप से पूरा माना जाता है
- **भुगतान ट्रैकिंग**: सभी भुगतान विवरण ऑर्डर नोट्स में दर्ज किए जाते हैं
- **रसीद जनरेशन**: सफल भुगतान के बाद मानक POS रसीदें उत्पन्न की जाती हैं

## आवश्यकताएं

- **SumUp अकाउंट**: सक्रिय SumUp मर्चेंट अकाउंट
- **API एक्सेस**: अपने मर्चेंट डैशबोर्ड से SumUp API कुंजी
- **संगत हार्डवेयर**: SumUp कार्ड रीडर डिवाइस
- **WooCommerce POS**: मुफ्त या प्रो संस्करण इंस्टॉल और सक्रिय
- **स्थिर कनेक्शन**: API संचार के लिए विश्वसनीय इंटरनेट कनेक्शन

## हार्डवेयर संगतता

:::important कनेक्टिविटी आवश्यकताएं
यह कार्यान्वयन SumUp के JavaScript SDK का उपयोग करता है, जिसका अर्थ है कि यह वेब एप्लिकेशन के माध्यम से काम करता है लेकिन **केवल इंटरनेट से जुड़े टर्मिनल** की आवश्यकता होती है। ब्लूटूथ टर्मिनल वर्तमान में समर्थित नहीं हैं।
:::

### समर्थित टर्मिनल (इंटरनेट से जुड़े)

- **SumUp Solo** ✅ - WiFi/Ethernet से जुड़ा टर्मिनल

### असमर्थित टर्मिनल (ब्लूटूथ)

- **SumUp Air** ❌ - केवल ब्लूटूथ
- **SumUp 3G** ❌ - केवल ब्लूटूथ

:::note भविष्य का समर्थन
iOS और Android एप्लिकेशन में **Q1 2026** में ब्लूटूथ टर्मिनल समर्थन जोड़ा जाएगा। यह Air और 3G मॉडल सहित सभी SumUp प्रमाणित टर्मिनल के लिए समर्थन सक्षम करेगा।
:::

## समस्या निवारण

### सामान्य समस्याएं

**रीडर जोड़ नहीं रहा**
- सुनिश्चित करें कि आप एक समर्थित इंटरनेट से जुड़े टर्मिनल (केवल Solo) का उपयोग कर रहे हैं
- जांचें कि आपका SumUp Solo डिवाइस WiFi से जुड़ा है और ऑनलाइन है
- जांचें कि डिवाइस पेयरिंग मोड में है और पेयरिंग कोड सही ढंग से दर्ज किया गया है
- पुष्टि करें कि आपके SumUp अकाउंट में API एक्सेस सक्षम है
- पेयरिंग के दौरान स्थिर इंटरनेट कनेक्शन सुनिश्चित करें

**भुगतान प्रोसेस नहीं हो रहा**
- सत्यापित करें कि रीडर ठीक से जोड़ा गया है
- जांचें कि आपका SumUp अकाउंट सक्रिय और अच्छी स्थिति में है
- सुनिश्चित करें कि डिवाइस के पास पर्याप्त बैटरी और कनेक्टिविटी है
- पुष्टि करें कि API कुंजी सही और सक्रिय है

**ऑर्डर पूरा नहीं हो रहा**
- इंटरनेट कनेक्शन की स्थिरता जांचें
- अपने SumUp अकाउंट में वेबहुक सेटिंग्स सत्यापित करें
- सुनिश्चित करें कि भुगतान SumUp टर्मिनल पर सफल हुआ है
- API संचार समस्याओं के लिए WordPress त्रुटि लॉग जांचें

### API दर सीमाएं

गेटवे को API कॉल को कम करने और दर सीमा से बचने के लिए अनुकूलित किया गया है:
- भुगतान स्थिति कुशलता से जांची जाती है
- अनावश्यक API अनुरोधों को कम किया जाता है
- स्वचालित पुनः प्रयास तर्क अस्थायी विफलताओं को संभालता है

### सहायता प्राप्त करना

तकनीकी सहायता के लिए:
- मुद्दे रिपोर्ट करने के लिए [GitHub रिपोजिटरी](https://github.com/wcpos/sumup-terminal-for-woocommerce) पर जाएं
- API-संबंधित प्रश्नों के लिए [SumUp डेवलपर दस्तावेज़](https://developer.sumup.com/) देखें
- अकाउंट और हार्डवेयर मुद्दों के लिए SumUp सहायता से संपर्क करें
