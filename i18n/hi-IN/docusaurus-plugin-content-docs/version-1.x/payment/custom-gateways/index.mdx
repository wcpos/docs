# कस्टम भुगतान गेटवे

WCPOS कई कस्टम भुगतान गेटवे का समर्थन करता है जो आपके पॉइंट ऑफ सेल प्रणाली की कार्यक्षमता को बढ़ाते हैं। ये गेटवे विशेष रूप से POS वातावरण के लिए डिज़ाइन किए गए हैं और विभिन्न भुगतान विधियों और सेवाओं के साथ निर्बाध एकीकरण प्रदान करते हैं।

## उपलब्ध कस्टम गेटवे

### हार्डवेयर टर्मिनल गेटवे

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminal हार्डवेयर रीडर का उपयोग करके भुगतान स्वीकार करें
- **[SumUp Terminal](./sumup-terminal)** - SumUp कार्ड रीडर के माध्यम से भुगतान संसाधित करें

### डिजिटल भुगतान गेटवे

- **[Email Invoice](./email-invoice)** - ग्राहकों को ईमेल के माध्यम से भुगतान चालान भेजें
- **[Web Checkout](./web-checkout)** - ग्राहकों को ऑनलाइन भुगतान पूरा करने के लिए पुनर्निर्देशित करें

## अपना स्वयं का गेटवे बनाना

यदि आपको अपनी विशिष्ट आवश्यकताओं के लिए एक कस्टम भुगतान गेटवे की आवश्यकता है, तो आप अपने स्वयं के WooCommerce POS भुगतान गेटवे बनाने के लिए **[Gateway Template](./gateway-template)** का उपयोग कर सकते हैं।

## स्थापना अवलोकन

सभी कस्टम गेटवे एक समान स्थापना प्रक्रिया का पालन करते हैं:

1. संबंधित GitHub रिपॉजिटरी से नवीनतम रिलीज़ **डाउनलोड** करें
2. `WP Admin > Plugins > Add New > Upload Plugin` के माध्यम से प्लगइन **स्थापित** करें
3. `WP Admin > WooCommerce > Settings > Payments` में गेटवे सेटिंग्स **कॉन्फ़िगर** करें
4. `WP Admin > WooCommerce POS > Settings > Checkout` में गेटवे **सक्षम** करें

## आवश्यकताएँ

- WooCommerce POS प्लगइन (फ्री या प्रो संस्करण)
- WooCommerce के साथ स्थापित WordPress
- विशेष भुगतान सेवा के लिए उचित API कुंजी या प्रमाण पत्र

## समर्थन

कस्टम गेटवे के साथ समस्याओं के लिए, कृपया संबंधित GitHub रिपॉजिटरी पर जाएं और आपकी समस्या के बारे में विस्तृत जानकारी के साथ एक समस्या बनाएं।
