# कस्टम भुगतान गेटवे {#custom-payment-gateways}

WCPOS कई कस्टम भुगतान गेटवे का समर्थन करता है जो आपके पॉइंट ऑफ सेल सिस्टम की कार्यशीलता को बढ़ाते हैं। ये गेटवे विशेष रूप से POS वातावरण के लिए डिज़ाइन किए गए हैं और विभिन्न भुगतान विधियों और सेवाओं के साथ सहज एकीकरण प्रदान करते हैं।

## उपलब्ध कस्टम गेटवे {#available-custom-gateways}

### हार्डवेयर टर्मिनल गेटवे {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Stripe Terminal हार्डवेयर रीडर्स का उपयोग करके भुगतान स्वीकार करें
- **[SumUp Terminal](./sumup-terminal)** - SumUp कार्ड रीडर्स के माध्यम से भुगतान प्रोसेस करें

### डिजिटल भुगतान गेटवे {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - ग्राहकों को ईमेल के माध्यम से भुगतान इनवॉइस भेजें
- **[Web Checkout](./web-checkout)** - ग्राहकों को ऑनलाइन भुगतान पूरा करने के लिए रीडायरेक्ट करें

## अपना स्वयं का गेटवे बनाना {#creating-your-own-gateway}

यदि आपको अपनी विशेष आवश्यकताओं के लिए कस्टम भुगतान गेटवे की आवश्यकता है, तो आप हमारे **[Gateway Template](./gateway-template)** का उपयोग करके अपना स्वयं का WooCommerce POS भुगतान गेटवे बना सकते हैं।

## स्थापना अवलोकन {#installation-overview}

सभी कस्टम गेटवे समान स्थापना प्रक्रिया का पालन करते हैं:

1. **डाउनलोड** करें नवीनतम रिलीज़ संबंधित GitHub रिपॉजिटरी से
2. **इंस्टॉल** करें प्लगइन को `WP Admin > Plugins > Add New > Upload Plugin` के माध्यम से
3. **कॉन्फ़िगर** करें गेटवे सेटिंग्स `WP Admin > WooCommerce > Settings > Payments` में
4. **सक्षम** करें गेटवे को `WP Admin > WooCommerce POS > Settings > Checkout` में

## समर्थन {#support}

कस्टम गेटवे के साथ समस्याओं के लिए, कृपया संबंधित GitHub रिपॉजिटरी पर जाएँ और अपनी समस्या के बारे में विस्तृत जानकारी के साथ एक समस्या बनाएँ।
