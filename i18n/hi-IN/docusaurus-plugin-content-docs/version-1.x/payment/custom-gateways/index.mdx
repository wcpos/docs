# कस्टम भुगतान गेटवे

WCPOS कई कस्टम भुगतान गेटवे का समर्थन करता है जो आपके पॉइंट ऑफ़ सेल सिस्टम की कार्यक्षमता को बढ़ाते हैं। ये गेटवे विशेष रूप से POS वातावरण के लिए डिज़ाइन किए गए हैं और विभिन्न भुगतान विधियों और सेवाओं के साथ सहज एकीकरण प्रदान करते हैं।

## उपलब्ध कस्टम गेटवे

### हार्डवेयर टर्मिनल गेटवे

- **[Stripe टर्मिनल](./stripe-terminal)** - Stripe टर्मिनल हार्डवेयर रीडर का उपयोग करके भुगतान स्वीकार करें
- **[SumUp टर्मिनल](./sumup-terminal)** - SumUp कार्ड रीडर के माध्यम से भुगतान प्रोसेस करें

### डिजिटल भुगतान गेटवे

- **[ईमेल इनवॉइस](./email-invoice)** - ग्राहकों को ईमेल द्वारा भुगतान इनवॉइस भेजें
- **[वेब चेकआउट](./web-checkout)** - ग्राहकों को ऑनलाइन भुगतान पूरा करने के लिए पुनर्निर्देशित करें

## अपना स्वयं का गेटवे बनाना

यदि आपको अपनी विशिष्ट आवश्यकताओं के लिए एक कस्टम भुगतान गेटवे की आवश्यकता है, तो आप अपना WooCommerce POS भुगतान गेटवे बनाने के लिए हमारे **[गेटवे टेम्प्लेट](./gateway-template)** का उपयोग कर सकते हैं।

## स्थापना अवलोकन

सभी कस्टम गेटवे एक समान स्थापना प्रक्रिया का पालन करते हैं:

1. **डाउनलोड** करें संबंधित GitHub रिपॉजिटरी से नवीनतम रिलीज
2. **इंस्टॉल** करें `WP व्यवस्थापक > प्लगइन > नया जोड़ें > प्लगइन अपलोड करें` के माध्यम से
3. **कॉन्फ़िगर** करें गेटवे सेटिंग्स `WP व्यवस्थापक > WooCommerce > सेटिंग्स > भुगतान` में
4. **सक्षम** करें गेटवे को `WP व्यवस्थापक > WooCommerce POS > सेटिंग्स > चेकआउट` में

## आवश्यकताएं

- WooCommerce POS प्लगइन (मुफ्त या प्रो संस्करण)
- WooCommerce के साथ WordPress
- विशिष्ट भुगतान सेवा के लिए उचित API कुंजियां या प्रमाणपत्र

## समर्थन

कस्टम गेटवे से संबंधित मुद्दों के लिए, कृपया संबंधित GitHub रिपॉजिटरी पर जाएं और अपनी समस्या के बारे में विस्तृत जानकारी के साथ एक मुद्दा बनाएं।
