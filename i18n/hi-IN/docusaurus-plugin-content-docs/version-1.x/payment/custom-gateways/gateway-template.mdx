# गेटवे टेम्पलेट

WooCommerce गेटवे टेम्पलेट WooCommerce POS के लिए अपना कस्टम भुगतान गेटवे बनाने के लिए एक शुरुआती बिंदु प्रदान करता है। यह टेम्पलेट एक पूरी तरह कार्यात्मक भुगतान गेटवे बनाने के लिए आवश्यक सभी बॉयलरप्लेट कोड और संरचना शामिल करता है।

## विशेषताएं

- **पूर्ण टेम्पलेट**: सभी आवश्यक विधियों के साथ तैयार-टू-यूज गेटवे संरचना
- **POS एकीकरण**: WooCommerce POS के लिए पूर्व-कॉन्फ़िगर किया गया
- **स्वचालित सेटअप**: स्क्रिप्ट-आधारित टेम्पलेट अनुकूलन
- **सर्वोत्तम प्रथाएं**: WordPress और WooCommerce कोडिंग मानकों का पालन करता है
- **विस्तार योग्य**: विशिष्ट भुगतान प्रदाताओं के लिए संशोधित करने में आसान

## शुरू करना

### विकल्प 1: स्वचालित टेम्पलेट जनरेशन

टेम्पलेट में एक स्क्रिप्ट शामिल है जो स्वचालित रूप से टेम्पलेट को आपके विशिष्ट गेटवे के लिए अनुकूलित करती है:

1. **रिपॉजिटरी क्लोन करें**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **सेटअप स्क्रिप्ट चलाएं**:
   ```bash
   ./create-gateway.sh
   ```

3. **प्रॉम्प्ट का पालन करें**:
   - अपना गेटवे नाम दर्ज करें (जैसे "मेरा भुगतान गेटवे")
   - गेटवे स्लग दर्ज करें (जैसे "my-payment")
   - विवरण प्रदान करें
   - स्क्रिप्ट एक अनुकूलित प्लगइन उत्पन्न करेगी

### विकल्प 2: मैनुअल टेम्पलेट उपयोग

यदि आप मैनुअल अनुकूलन पसंद करते हैं:

1. **टेम्पलेट डाउनलोड करें**:
   - [गेटवे टेम्पलेट रिपॉजिटरी](https://github.com/wcpos/woocommerce-gateway-template) पर जाएं
   - नवीनतम रिलीज़ डाउनलोड करें या रिपॉजिटरी क्लोन करें

2. **टेम्पलेट अनुकूलित करें**:
   - `{{GATEWAY_NAME}}` को अपने गेटवे का प्रदर्शन नाम से बदलें
   - `{{GATEWAY_SLUG}}` को अपने गेटवे के अद्वितीय पहचानकर्ता से बदलें
   - `{{GATEWAY_DESCRIPTION}}` को अपने गेटवे के विवरण से बदलें

3. **फ़ाइलें नाम बदलें**:
   - `wcpos-{{GATEWAY_SLUG}}.php` को अपने गेटवे स्लग के अनुरूप नाम दें
   - फ़ाइल हेडर और प्लगइन जानकारी अपडेट करें
