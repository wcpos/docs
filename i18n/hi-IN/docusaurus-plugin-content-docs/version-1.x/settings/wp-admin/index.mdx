ये सेटिंग्स POS अनुप्रयोग की बैकएंड कार्यक्षमता को कॉन्फ़िगर करती हैं, जिसमें PHP और WooCommerce कॉन्फ़िगरेशन शामिल हैं।  
आप इन्हें यहां नेविगेट करके एक्सेस कर सकते हैं: `WP Admin > POS > Settings`

मुख्य कॉन्फ़िगरेशन में शामिल हैं:

- सभी कैशियर्स के लिए डिफ़ॉल्ट कार्ट ग्राहक।
- उत्पाद बारकोड फ़ील्ड।
- ऑर्डर ईमेल सेटिंग्स।
- उपयोगकर्ता भूमिका अनुमतियाँ।
