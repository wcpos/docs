स्टोर सेटिंग्स ऐप-व्यापी कॉन्फ़िगरेशन हैं जो यह प्रभाव डालती हैं कि POS कैसे संचालित होता है। ये सेटिंग्स पूरे स्टोर पर लागू होती हैं, न कि व्यक्तिगत स्क्रीन पर।

## स्टोर सेटिंग्स तक पहुंचना {#accessing-store-settings}

स्टोर सेटिंग्स खोलने के लिए:

- **यूज़र मेन्यू** (ऊपरी दाएँ कोने) पर क्लिक करें और **सेटिंग्स** चुनें
- कीबोर्ड शॉर्टकट `Ctrl + Shift + S` का उपयोग करें

## सेटिंग्स टैब {#settings-tabs}

स्टोर सेटिंग्स मोडल में ये टैब शामिल हैं:

| टैब | विवरण |
|-----|-------------|
| [General](./general) | स्टोर जानकारी, भाषा, मुद्रा, डिफ़ॉल्ट ग्राहक |
| [Tax](./tax) | कर गणना और प्रदर्शन विकल्प |
| [Barcode](./barcode) | बारकोड स्कैनर कॉन्फ़िगरेशन |
| [Keyboard Shortcuts](./hotkeys) | उपलब्ध कीबोर्ड शॉर्टकट्स |
| [Theme](./theme) | रूप और रंग योजना |
