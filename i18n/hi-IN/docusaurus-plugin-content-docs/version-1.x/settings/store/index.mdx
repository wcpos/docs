स्टोर सेटिंग्स ऐप-व्यापी कॉन्फ़िगरेशन हैं जो POS के संचालन को प्रभावित करती हैं। ये सेटिंग्स संपूर्ण स्टोर पर लागू होती हैं, अलग-अलग स्क्रीन पर नहीं।

## स्टोर सेटिंग्स तक पहुँचना {#accessing-store-settings}

स्टोर सेटिंग्स खोलने के लिए:

- **उपयोगकर्ता मेनू** (ऊपर-दाएँ कोने में) पर क्लिक करें और **सेटिंग्स** चुनें
- कीबोर्ड शॉर्टकट `Ctrl + Shift + S` का उपयोग करें

## सेटिंग्स टैब {#settings-tabs}

स्टोर सेटिंग्स मोडल में ये टैब होते हैं:

| टैब | विवरण |
|-----|-------------|
| [सामान्य](./general) | स्टोर जानकारी, भाषा, मुद्रा, डिफ़ॉल्ट ग्राहक |
| [कर](./tax) | कर गणना और प्रदर्शन विकल्प |
| [बारकोड](./barcode) | बारकोड स्कैनर कॉन्फ़िगरेशन |
| [कीबोर्ड शॉर्टकट](./hotkeys) | उपलब्ध कीबोर्ड शॉर्टकट |
| [थीम](./theme) | दिखावट और रंग योजना |
| [प्रिंटर](/hardware/printers) | रसीद प्रिंटर कनेक्शन और कॉन्फ़िगरेशन |
