स्टोर सेटिंग्स ऐसी एप्लिकेशन-व्यापी कॉन्फ़िगरेशन हैं जो यह निर्धारित करती हैं कि POS कैसे कार्य करता है। ये सेटिंग्स पूरे स्टोर पर लागू होती हैं, न कि व्यक्तिगत स्क्रीन पर।

## स्टोर सेटिंग्स तक पहुंचना {#accessing-store-settings}

स्टोर सेटिंग्स खोलने के लिए:

- **यूज़र मेन्यू** (ऊपरी दाएँ कोने) पर क्लिक करें और **सेटिंग्स** चुनें
- कीबोर्ड शॉर्टकट `Ctrl + Shift + S` का उपयोग करें

## सेटिंग्स टैब {#settings-tabs}

स्टोर सेटिंग्स मोड में ये टैब शामिल हैं:

| टैब | विवरण |
|-----|-------------|
| [ सामान्य ](./general) | स्टोर जानकारी, भाषा, मुद्रा, डिफ़ॉल्ट ग्राहक |
| [ कर ](./tax) | कर गणना और प्रदर्शन विकल्प |
| [ बारकोड ](./barcode) | बारकोड स्कैनर कॉन्फ़िगरेशन |
| [ कीबोर्ड शॉर्टकट्स ](./hotkeys) | उपलब्ध कीबोर्ड शॉर्टकट्स |
| [ थीम ](./theme) | रूप और रंग योजना |
