WCPOS प्लगइन की उचित कार्यक्षमता और इष्टतम प्रदर्शन सुनिश्चित करने के लिए, कृपया सुनिश्चित करें कि आपका वातावरण निम्नलिखित न्यूनतम आवश्यकताओं को पूरा करता है:

## सर्वर आवश्यकताएँ {#server-requirements}

WCPOS प्लगइन आपकी WordPress साइट पर चलता है, जिसे इन न्यूनतम आवश्यकताओं को पूरा करना होगा:

| घटक        | न्यूनतम संस्करण | टिप्पणियाँ      |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x अनुशंसित   |

इन न्यूनतम आवश्यकताओं को पूरा करने से आपके WordPress वातावरण के साथ संगतता और स्थिरता सुनिश्चित होती है, और नवीनतम WCPOS सुविधाओं तक पहुँच मिलती है।

## ब्राउज़र आवश्यकताएँ {#browser-requirements}

POS का वेब संस्करण किसी भी आधुनिक ब्राउज़र में चलता है। नवीनतम स्थिर रिलीज़ का उपयोग करें:

| ब्राउज़र         | समर्थित संस्करण |
| --------------- | ----------------- |
| गूगल क्रोम   | नवीनतम स्थिर     |
| मोज़िला फ़ायरफ़ॉक्स | नवीनतम स्थिर     |
| सफ़ारी          | नवीनतम स्थिर     |
| माइक्रोसॉफ़्ट एज  | नवीनतम स्थिर     |

:::warning प्राइवेट / इनकॉग्निटो मोड से बचें
प्राइवेट ब्राउज़िंग ब्राउज़र के IndexedDB डेटाबेस तक पहुँच को अवरुद्ध कर सकती है, जिसका उपयोग WCPOS ऑफ़लाइन डेटा संग्रहीत करने के लिए करता है — कुछ सुविधाएँ सही ढंग से काम नहीं करेंगी। JavaScript भी सक्षम होना आवश्यक है।
:::

## ऐप आवश्यकताएँ {#app-requirements}

WCPOS डेस्कटॉप और मोबाइल ऐप के रूप में भी उपलब्ध है। प्रत्येक निम्नलिखित न्यूनतम ऑपरेटिंग सिस्टम के लिए बनाया गया है:

| प्लेटफ़ॉर्म        | न्यूनतम OS                | आर्किटेक्चर              |
| --------------- | ------------------------- | -------------------------- |
| Windows डेस्कटॉप | Windows 10                | x64, ARM64                 |
| macOS डेस्कटॉप   | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Android         | Android 10 (API level 29) | —                          |

:::note प्रिंटिंग अनुमतियाँ
Bluetooth और नेटवर्क प्रिंटिंग के लिए, संकेत मिलने पर ऐप को संबंधित Bluetooth और लोकल नेटवर्क अनुमतियाँ प्रदान करना आवश्यक है।
:::

## प्लगइन इंस्टॉल करना {#installing-the-plugin}

1. `WP Admin > प्लगइन्स > Add Plugin` पर जाएँ
2. "wcpos" खोजें
3. **अभी इंस्टॉल करें** पर क्लिक करें, फिर **सक्रिय करें**

वैकल्पिक रूप से, आप [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से प्लगइन डाउनलोड करके मैन्युअल रूप से अपलोड कर सकते हैं।

## POS तक पहुँच {#accessing-the-pos}

सक्रियण के बाद, आप POS को यहाँ एक्सेस कर सकते हैं:

- **वेब:** `yourdomain.com/pos`
- **डेस्कटॉप:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **मोबाइल:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

न्यूनतम आवश्यकताओं को पूरा करके और एक संगत वेब ब्राउज़र का उपयोग करके, आप WCPOS प्लगइन का उपयोग करते हुए एक सहज अनुभव प्राप्त कर सकते हैं और अपने पॉइंट ऑफ़ सेल संचालन को कुशलतापूर्वक प्रबंधित करने के लिए इसकी पूरी क्षमता का लाभ उठा सकते हैं।

यदि आपके कोई और प्रश्न हैं या सहायता चाहिए, तो कृपया [Discord चैट](https://wcpos.com/discord) पर पोस्ट करके या support@wcpos.com पर ईमेल करके हमारी सहायता टीम से संपर्क करने में संकोच न करें।
