:::info प्रो फीचर
[WCPOS Pro](https://wcpos.com/pro) एक पेड प्लगइन है जो मुफ्त [WCPOS प्लगइन](https://wordpress.org/plugins/woocommerce-pos/) में कार्यक्षमता जोड़ता है।
:::

## प्रो में क्या शामिल है? {#whats-included-in-pro}

WCPOS Pro अतिरिक्त सुविधाओं को अनलॉक करता है:

- **प्रोडक्ट्स स्क्रीन** - POS में सीधे स्टॉक स्तर और कीमतें संपादित करें
- **ऑर्डर्स स्क्रीन** - ऑर्डर इतिहास देखें और प्रबंधित करें
- **कस्टमर्स स्क्रीन** - ग्राहक जानकारी जोड़ें और संपादित करें
- **रिपोर्ट्स स्क्रीन** - दिन के अंत के बिक्री रिपोर्ट उत्पन्न करें
- **अतिरिक्त पेमेंट गेटवे** - Stripe Terminal, SumUp, और कस्टम गेटवे

## स्थापना {#installation}

यदि आपने WCPOS Pro के लिए एक लाइसेंस खरीदा है, तो कृपया प्लगइन को स्थापित और सक्रिय करने के लिए नीचे दिए गए चरणों का पालन करें:

1. जाएं: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **Downloads** के अंतर्गत, डाउनलोड लिंक पर क्लिक करें और प्लगइन को अपने डेस्कटॉप पर सहेजें।
3. फिर अपने साइट पर जाएं, लॉगिन करें और जाएं `WP Admin > Plugins > Add New > Upload Plugin`
4. अपने डेस्कटॉप से प्लगइन ज़िप फ़ाइल अपलोड करें और सक्रिय करें।
5. इसके बाद, जाएं `WP Admin > POS > Settings > License` और अपनी License Key और License Email दर्ज करें ताकि सक्रियण पूरा हो सके।

![मेरे डाउनलोड](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "आप अपने अकाउंट पेज पर WCPOS Pro डाउनलोड कर सकते हैं")

## अपडेट्स {#updates}

जब नए संस्करण तैयार होंगे, तो वे आपके Updates डैशबोर्ड में किसी अन्य प्लगइन की तरह दिखाई देंगे। आप अपने [अकाउंट पेज](https://wcpos.com/my-account/) से नवीनतम संस्करण भी डाउनलोड कर सकते हैं।
