:::info Pro विशेषता
[WCPOS Pro](https://wcpos.com/pro) एक सशुल्क प्लगइन है जो मुफ्त [WCPOS प्लगइन](https://wordpress.org/plugins/woocommerce-pos/) में कार्यक्षमता जोड़ता है।
:::

## Pro में क्या शामिल है? {#whats-included-in-pro}

WCPOS Pro अतिरिक्त सुविधाएँ अनलॉक करता है:

- **उत्पाद स्क्रीन** - सीधे POS में स्टॉक स्तर और कीमतें संपादित करें
- **ऑर्डर स्क्रीन** - ऑर्डर इतिहास को देखें और प्रबंधित करें
- **ग्राहक स्क्रीन** - ग्राहक जानकारी जोड़ें और संपादित करें
- **रिपोर्ट स्क्रीन** - दिन के अंत में बिक्री रिपोर्ट उत्पन्न करें
- **अतिरिक्त भुगतान गेटवे** - स्ट्राइप टर्मिनल, समअप, और कस्टम गेटवे

## स्थापना {#installation}

यदि आपने WCPOS Pro के लिए एक लाइसेंस खरीदा है, तो कृपया निम्नलिखित चरणों का पालन करें प्लगइन स्थापित और सक्रिय करने के लिए:

1. जाएँ: [https://wcpos.com/my-account/](https://wcpos.com/my-account/)
2. **डाउनलोड्स** के अंतर्गत, डाउनलोड लिंक पर क्लिक करें और प्लगइन को अपने डेस्कटॉप पर सहेजें।
3. फिर अपनी साइट पर जाएँ, लॉगिन करें और जाएँ `WP Admin > Plugins > Add Plugin > Upload Plugin`
4. अपने डेस्कटॉप से प्लगइन ज़िप फ़ाइल अपलोड करें और सक्रिय करें।
5. अगला, जाएँ `WP Admin > POS > Settings > License` और अपनी लाइसेंस कुंजी और लाइसेंस ईमेल दर्ज करें सक्रियण पूरा करने के लिए।

![मेरे डाउनलोड्स](https://wcpos.com/wp-content/uploads/2014/07/my-download.png "आप अपने खाता पृष्ठ पर WCPOS Pro डाउनलोड कर सकते हैं")

## अपडेट्स {#updates}

जब नए संस्करण तैयार होते हैं, तो वे आपके अपडेट डैशबोर्ड में अन्य प्लगइन की तरह दिखाई देंगे। आप अपने [खाता पृष्ठ](https://wcpos.com/my-account/) से नवीनतम संस्करण भी डाउनलोड कर सकते हैं।
