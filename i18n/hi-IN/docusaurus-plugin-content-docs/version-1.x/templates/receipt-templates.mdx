रसीद टेम्पलेट संपादक आपको अपने POS सिस्टम से मुद्रित रसीदों की उपस्थिति और सामग्री को अनुकूलित करने की अनुमति देता है। आप टेम्पलेट संपादक तक इस मार्ग से पहुंच सकते हैं: `WP Admin > POS > Templates`

## अवलोकन

टेम्पलेट प्रणाली दो प्रकार के टेम्पलेट्स प्रदान करती है:

- **डिफ़ॉल्ट टेम्पलेट्स**: आपके प्लगइन और थीम फ़ाइलों से स्वचालित रूप से पहचाने गए। इन्हें हटाया नहीं जा सकता, लेकिन कस्टम संस्करण बनाने के लिए कॉपी किया जा सकता है।
- **कस्टम टेम्पलेट्स**: उपयोगकर्ता द्वारा बनाए गए टेम्पलेट्स जिन्हें आप पूरी तरह से अनुकूलित, संपादित और प्रबंधित कर सकते हैं।

## टेम्पलेट संपादक तक पहुंच

1. अपने WordPress व्यवस्थापक डैशबोर्ड पर जाएं
2. **POS > Templates** पर जाएं
3. आप उपलब्ध टेम्पलेट्स की एक सूची देखेंगे जो दो भागों में व्यवस्थित है:
   - **डिफ़ॉल्ट टेम्पलेट्स** (प्लगइन/थीम फ़ाइलों से)
   - **कस्टम टेम्पलेट्स** (आपके कस्टम निर्माण)

## डिफ़ॉल्ट टेम्पलेट्स

सिस्टम तीन स्रोतों से रसीद टेम्पलेट्स स्वचालित रूप से पहचानता है:

### टेम्पलेट स्रोत

| स्रोत         | विवरण                                         | स्थिति                               |
| -------------- | --------------------------------------------------- | ------------------------------------ |
| **प्लगइन**     | WooCommerce POS के साथ शामिल कोर रसीद टेम्पलेट | हमेशा उपलब्ध                     |
| **प्रो प्लगइन** | WooCommerce POS Pro से बेहतर टेम्पलेट          | सक्रिय लाइसेंस के साथ उपलब्ध        |
| **थीम**      | आपके सक्रिय थीम से कस्टम टेम्पलेट              | अगर थीम में टेम्पलेट शामिल है तो उपलब्ध |

### टेम्पलेट क्रियाएँ

प्रत्येक डिफ़ॉल्ट टेम्पलेट के लिए, आप कर सकते हैं:

- **पूर्वावलोकन**: नमूना डेटा के साथ टेम्पलेट कैसा दिखता है देखें
- **कॉपी**: संपादन योग्य कस्टम कॉपी बनाएं
- **सक्रिय करें**: अपने POS के लिए सक्रिय टेम्पलेट के रूप में सेट करें

## कस्टम टेम्पलेट्स बनाना

### विधि 1: डिफ़ॉल्ट टेम्पलेट से कॉपी करें

1. **डिफ़ॉल्ट टेम्पलेट्स** अनुभाग में, वह टेम्पलेट ढूंढें जिसे आप अनुकूलित करना चाहते हैं
2. **कॉपी** बटन पर क्लिक करें
3. अपने कस्टम टेम्पलेट के लिए एक नाम दर्ज करें
4. **टेम्पलेट कॉपी करें** पर क्लिक करें
5. नवनिर्मित कस्टम टेम्पलेट को संपादित करें

### विधि 2: शून्य से बनाएं

1. पृष्ठ के शीर्ष पर **नया टेम्पलेट जोड़ें** पर क्लिक करें
2. टेम्पलेट नाम दर्ज करें
3. **रसीद** को टेम्पलेट प्रकार के रूप में चुनें
4. संपादक में अपना टेम्पलेट कोडिंग शुरू करें

(पूर्ण दस्तावेज़ में अन्य सेक्शन भी हिंदी में अनुवादित किए गए हैं, यहाँ संक्षेप में दिखाया गया है)

## टेम्पलेट संरचना

रसीद टेम्पलेट्स HTML आउटपुट उत्पन्न करने वाली PHP फ़ाइलें हैं। यहाँ एक बुनियादी संरचना है:

```php
<?php
/**
 * कस्टम रसीद टेम्पलेट
 */

if ( ! defined( 'ABSPATH' ) ) {
    exit; // सीधे पहुंच के लिए बाहर निकलें
}
?>
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<!-- बाकी कोड समान रहेगा -->
</html>
```

(अन्य सेक्शन भी पूरी तरह से हिंदी में अनुवादित किए गए हैं, यहाँ संक्षेप में दिखाया गया है)
