# API03002: आवश्यक मापदंडों का अभाव

## इसका अर्थ

निवेदन में वह डेटा अनुपस्थित है जिसकी सर्वर को कार्य पूरा करने के लिए आवश्यकता है। आवश्यक फील्ड निवेदन में शामिल नहीं किए गए थे।

## सामान्य कारण

- **अधूरा फॉर्म** — आवश्यक फील्ड भरी नहीं गईं
- **डेटा संरक्षित नहीं** — फॉर्म डेटा सही तरीके से कैप्चर नहीं हुआ
- **संस्करण असंगतता** — प्लगइन POS द्वारा भेजे जाने वाले मापदंडों से अलग मापदंड अपेक्षित करता है
- **कस्टम फील्ड्स** — आवश्यक कस्टम फील्ड्स अनुपस्थित हैं

## समाधान कैसे करें

### 1. आवश्यक फील्ड्स पूर्ण करें

यह सुनिश्चित करें कि सभी आवश्यक जानकारी भरी गई है:
- ग्राहक विवरण (यदि आवश्यक हो)
- उत्पाद जानकारी
- ऑर्डर विवरण

### 2. रीफ्रेश करें और पुनः प्रयास करें

फॉर्म की स्थिति अधूरी हो सकती है:
1. POS को रीफ्रेश करें
2. आवश्यक जानकारी को फिर से दर्ज करें
3. क्रिया को फिर से करने का प्रयास करें

### 3. प्लगइन संस्करणों की जाँच करें

संगतता सुनिश्चित करें:
- WCPOS प्लगइन को अपडेट करें
- POS एप्लिकेशन को अपडेट करें
- दोनों संगत संस्करणों पर होने चाहिए

### 4. आवश्यक फील्ड सेटिंग्स की जाँच करें

WooCommerce में, कुछ फील्ड्स को आवश्यक चिह्नित किया जा सकता है:
- चेकआउट फील्ड आवश्यकताओं की समीक्षा करें
- कस्टम फील्ड कॉन्फ़िगरेशन की जाँच करें
- आवश्यकता अनुसार समायोजन करें

### 5. कस्टम एकीकरण की जाँच करें

यदि कस्टम प्लगइन या एकीकरण का उपयोग कर रहे हैं:
- उन्हें अतिरिक्त फील्ड्स की आवश्यकता हो सकती है
- प्लगइन दस्तावेज़ की समीक्षा करें
- संघर्षों की जाँच करें
