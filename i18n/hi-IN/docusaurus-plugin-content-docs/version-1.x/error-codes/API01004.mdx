# API01004: DNS संकल्प विफल

## इसका अर्थ क्या है

पॉइंट ऑफ़ सेल आपकी वेबसाइट के डोमेन नाम (जैसे `yourstore.com`) को एक IP पते में परिवर्तित नहीं कर सका। DNS (डोमेन नाम प्रणाली) इंटरनेट का फोन बुक जैसा है — यदि इसे लिस्टिंग नहीं मिलती, तो यह कनेक्शन नहीं बना सकता।

## सामान्य कारण

- **डोमेन नाम में टाइपो** — URL में वर्तनी त्रुटि हो सकती है
- **DNS सर्वर समस्याएं** — आपका DNS सर्वर प्रतिक्रिया नहीं दे रहा
- **डोमेन समाप्त** — डोमेन पंजीकरण समाप्त हो गया हो सकता है
- **DNS प्रसार** — हाल ही में DNS परिवर्तन पूरी तरह से प्रसारित नहीं हुए
- **स्थानीय DNS कैश** — आपके डिवाइस पर पुरानी DNS जानकारी कैश में है

## समाधान कैसे करें

### 1. डोमेन नाम की पुष्टि करें

URL में टाइपो के लिए दोबारा जाँच करें। ब्राउज़र में साइट तक पहुँचने का प्रयास करें।

### 2. डोमेन स्थिति की जाँच करें

WHOIS खोज का उपयोग करके सत्यापित करें:
- डोमेन अभी भी पंजीकृत है
- समाप्त नहीं हुआ है
- DNS रिकॉर्ड कॉन्फ़िगर किए गए हैं

### 3. DNS कैश साफ़ करें

**अपने डिवाइस पर:**

macOS:
```bash
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```

Windows:
```bash
ipconfig /flushdns
```

### 4. वैकल्पिक DNS आज़माएं

सार्वजनिक DNS सर्वर पर स्विच करें जैसे:
- Google: `8.8.8.8` और `8.8.4.4`
- Cloudflare: `1.1.1.1`

### 5. DNS प्रसार के लिए प्रतीक्षा करें

यदि आपने हाल ही में DNS सेटिंग्स बदली हैं, तो पूर्ण प्रसार के लिए 24-48 घंटे प्रतीक्षा करें।
