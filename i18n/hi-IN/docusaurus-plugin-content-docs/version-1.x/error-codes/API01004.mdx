# API01004: DNS रिज़ॉल्यूशन विफल

## इसका क्या मतलब है

POS आपकी वेबसाइट के डोमेन नाम (जैसे, `yourstore.com`) को IP पते में परिवर्तित नहीं कर सका। DNS (डोमेन नाम प्रणाली) इंटरनेट की फोन बुक की तरह है — अगर यह लिस्टिंग नहीं खोज सकता, तो यह कनेक्शन नहीं बना सकता।

## सामान्य कारण

- **डोमेन नाम में टाइपो** — URL में स्पेलिंग की गलती हो सकती है
- **DNS सर्वर समस्याएँ** — आपका DNS सर्वर प्रतिक्रिया नहीं दे रहा है
- **डोमेन समाप्त** — डोमेन पंजीकरण समाप्त हो सकता है
- **DNS प्रचार** — हाल के DNS परिवर्तनों का प्रचार अभी नहीं हुआ है
- **स्थानीय DNS कैश** — आपके डिवाइस पर पुरानी DNS जानकारी कैश की गई है

## इसे कैसे ठीक करें

### 1. डोमेन नाम की पुष्टि करें

टाइपो के लिए URL को दोबारा जांचें। ब्राउज़र में साइट तक पहुँचने की कोशिश करें।

### 2. डोमेन स्थिति की जाँच करें

पुष्टि करने के लिए WHOIS खोज का उपयोग करें:
- डोमेन अभी भी पंजीकृत है
- यह समाप्त नहीं हुआ है
- DNS रिकॉर्ड कॉन्फ़िगर किए गए हैं

### 3. DNS कैश हटाएँ

**आपके डिवाइस पर:**

macOS:
```bash
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```

Windows:
```bash
ipconfig /flushdns
```

### 4. वैकल्पिक DNS आज़माएँ

एक सार्वजनिक DNS सर्वर पर स्विच करें जैसे:
- Google: `8.8.8.8` और `8.8.4.4`
- Cloudflare: `1.1.1.1`

### 5. DNS प्रचार का इंतजार करें

यदि आपने हाल ही में DNS सेटिंग्स में बदलाव किया है, तो पूर्ण प्रचार के लिए 24-48 घंटे का इंतजार करें।
