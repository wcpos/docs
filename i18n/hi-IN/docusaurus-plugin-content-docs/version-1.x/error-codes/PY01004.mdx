# PY01004: अमान्य कार्ड संख्या

## इसका अर्थ

दर्ज की गई कार्ड संख्या मान्य नहीं है। कार्ड संख्याएं विशिष्ट पैटर्न का पालन करती हैं और सत्यापन के लिए जांच अंक शामिल करती हैं।

## सामान्य कारण

- **टाइपो** — एक या अधिक अंक गलत दर्ज किए गए
- **अंक छूटे हुए** — सभी अंक दर्ज नहीं किए गए
- **अतिरिक्त अंक** — बहुत अधिक अंक दर्ज किए गए
- **गलत कार्ड** — भौतिक कार्ड से अलग कार्ड संख्या
- **कार्ड रीडर त्रुटि** — चिप/स्वाइप ने गलत डेटा पढ़ा

## समाधान कैसे करें

### 1. कार्ड संख्या फिर से दर्ज करें

सावधानीपूर्वक सभी अंक दर्ज करें:
- भौतिक कार्ड की जाँच करें
- प्रत्येक अंक धीरे-धीरे दर्ज करें
- सबमिट करने से पहले पूरी संख्या की पुष्टि करें

### 2. कार्ड प्रकार की जाँच करें

सुनिश्चित करें कि कार्ड प्रकार समर्थित है:
- वीजा (4 से शुरू होता है)
- मास्टरकार्ड (51-55 या 22-27 से शुरू होता है)
- एमेक्स (34 या 37 से शुरू होता है)
- डिस्कवर (6011, 622, 644-649, 65 से शुरू होता है)

### 3. कार्ड रीडर के साथ फिर से प्रयास करें

चिप/स्वाइप रीडर का उपयोग करते समय:
- कार्ड चिप को साफ करें
- अलग रीड विधि का प्रयास करें (चिप बनाम स्वाइप)
- रीडर विफल होने पर मैन्युअल रूप से दर्ज करें

### 4. अन्य कार्ड का उपयोग करें

यदि कार्ड संख्या लगातार विफल होती है:
- कार्ड क्षतिग्रस्त हो सकता है
- एक अलग कार्ड का प्रयास करें
- वैकल्पिक भुगतान विधि का उपयोग करें

### 5. कार्ड क्षति की जाँच करें

भौतिक कार्ड समस्याएं:
- खरोंच वाला चुंबकीय पट्टा
- क्षतिग्रस्त चिप
- घिसे या फीके हुए अंक
- कार्ड को बदलने की आवश्यकता हो सकती है
