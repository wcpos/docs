# API01002: कनेक्शन अस्वीकृत

## इसका अर्थ

सर्वर ने सक्रिय रूप से कनेक्शन प्रयास को अस्वीकार कर दिया। यह टाइमआउट से अलग है — सर्वर ने जवाब दिया, लेकिन "नहीं" कहा।

## सामान्य कारण

- **सर्वर बंद है** — वेब सर्वर (Apache/Nginx) शायद चल नहीं रहा है
- **गलत पोर्ट** — गलत पोर्ट पर कनेक्ट करने का प्रयास
- **फायरवॉल रोक** — फायरवॉल कनेक्शन को रोक रहा है
- **IP प्रतिबंध** — आपका IP पता अवरुद्ध हो सकता है

## समाधान कैसे करें

### 1. सर्वर चल रहा है यह सुनिश्चित करें

ब्राउज़र में सीधे अपनी WordPress साइट तक पहुंचने का प्रयास करें। यदि यह लोड नहीं हो रहा है, तो अपने होस्टिंग प्रदाता से संपर्क करें।

### 2. URL कॉन्फ़िगरेशन की जाँच करें

यह सुनिश्चित करें कि आप सही URL का उपयोग कर रहे हैं:
- `http://` बनाम `https://` की पुष्टि करें
- डोमेन नाम में टाइपो के लिए जाँच करें
- गैर-मानक पोर्ट का उपयोग करते समय सही पोर्ट की पुष्टि करें

### 3. फायरवॉल सेटिंग्स की जाँच करें

यदि आपके पास सर्वर तक पहुंच है:
- पोर्ट 80 (HTTP) या 443 (HTTPS) खुला है यह सत्यापित करें
- जाँच करें कि आपका IP व्हाइटलिस्ट में है
- किसी भी सुरक्षा प्लगइन की समीक्षा करें जो API पहुंच को रोक सकता है

### 4. अपने होस्टिंग प्रदाता से संपर्क करें

वे जाँच कर सकते हैं:
- सर्वर स्थिति और लॉग
- फायरवॉल कॉन्फ़िगरेशन
- क्या आपका IP अवरुद्ध किया गया है
