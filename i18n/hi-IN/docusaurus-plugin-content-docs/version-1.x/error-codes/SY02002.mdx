# SY02002: सेवा उपलब्ध नहीं

## इसका अर्थ

कोई आवश्यक सेवा या घटक उपलब्ध नहीं है। POS विभिन्न सेवाओं पर निर्भर करता है, और उनमें से एक वर्तमान में असुलभ है।

## सामान्य कारण

- **बैकग्राउंड सेवा रुकी हुई** — आवश्यक सेवा चल नहीं रही
- **सिस्टम संसाधन समस्या** — सेवा शुरू करने के लिए पर्याप्त संसाधन नहीं
- **निर्भरता असफलता** — जिस सेवा पर यह निर्भर है, वह विफल हो गई
- **स्टार्टअप समस्या** — सेवा ठीक से शुरू नहीं हुई
- **सर्वर त्रुटि** — WooCommerce सर्वर ने 5xx त्रुटि लौटाई

## सर्वर त्रुटि मैपिंग

यह त्रुटि कोड निम्न पर ट्रिगर होता है:

| सर्वर कोड | स्रोत |
|-------------|--------|
| HTTP 5xx | कोई भी सर्वर त्रुटि प्रतिक्रिया (500, 502, 503, 504, आदि) |

## समाधान कैसे करें

### 1. एप्लिकेशन पुनः शुरू करें

POS को बंद और फिर से खोलें:
1. पूरी तरह से एप्लिकेशन बंद करें
2. कुछ सेकंड प्रतीक्षा करें
3. फिर से शुरू करें

### 2. अपने डिवाइस को पुनः शुरू करें

पूर्ण पुनरारंभ अक्सर सेवा समस्याएं हल करता है:
1. महत्वपूर्ण काम सेव करें
2. अपना कंप्यूटर पुनः शुरू करें
3. POS को फिर से प्रयास करें

### 3. सिस्टम संसाधनों की जाँच करें

यह सुनिश्चित करें कि संसाधन उपलब्ध हैं:
- अनावश्यक एप्लिकेशन बंद करें
- स्मृति उपयोग की जाँच करें
- डिस्क स्थान की जाँच करें

### 4. एप्लिकेशन अपडेट करें

सुनिश्चित करें कि आप नवीनतम संस्करण पर हैं:
- उपलब्ध अपडेट की जाँच करें
- किसी भी लंबित अपडेट को इंस्टॉल करें
- अपडेट के बाद पुनः शुरू करें

### 5. आवश्यकता पड़ने पर पुनः इंस्टॉल करें

यदि सेवा अनुपलब्ध रहती है:
1. एप्लिकेशन अनइंस्टॉल करें
2. अपने डिवाइस को पुनः शुरू करें
3. नवीनतम संस्करण डाउनलोड करें
4. नया इंस्टॉल करें

## ब्राउज़र-विशिष्ट (वेब संस्करण)

वेब संस्करण के लिए:
- सुनिश्चित करें कि JavaScript सक्षम है
- जाँच करें कि आवश्यक ब्राउज़र सुविधाएं अवरुद्ध नहीं हैं
- एक अलग ब्राउज़र का प्रयास करें
- ब्राउज़र एक्सटेंशन को अक्षम करें जो बाधा डाल सकते हैं

## समर्थन से कब संपर्क करें

यदि यह त्रुटि बनी रहती है:
- ध्यान दें कि यह कब होता है
- जाँच करें कि क्या विशिष्ट क्रियाएं इसे ट्रिगर करती हैं
- जांच के लिए विवरण के साथ रिपोर्ट करें
