# SY02002: सेवा अनुपलब्ध {#sy02002-service-unavailable}

## इसका क्या मतलब है {#what-this-means}

एक आवश्यक सेवा या घटक उपलब्ध नहीं है। POS विभिन्न सेवाओं पर कार्य करने के लिए निर्भर करता है, और उनमें से एक वर्तमान में सुलभ नहीं है।

## सामान्य कारण {#common-causes}

- **पृष्ठभूमि सेवा बंद हो गई** — एक आवश्यक सेवा चल नहीं रही है
- **सिस्टम संसाधन समस्या** — सेवा शुरू करने के लिए पर्याप्त संसाधन नहीं हैं
- **निर्भरता विफलता** — जिस सेवा पर यह निर्भर है, वह विफल हो गई
- **स्टार्टअप समस्या** — सेवा ठीक से प्रारंभ करने में विफल
- **सर्वर त्रुटि** — WooCommerce सर्वर ने 5xx त्रुटि लौटाई

## सर्वर त्रुटि मानचित्रण {#server-error-mapping}

यह त्रुटि कोड तब सक्रिय होता है जब सर्वर लौटाता है:

| सर्वर कोड | स्रोत |
|-----------|-------|
| HTTP 5xx  | कोई भी सर्वर त्रुटि प्रतिक्रिया (500, 502, 503, 504, आदि) |

## कैसे ठीक करें {#how-to-fix}

### 1. एप्लिकेशन पुनरारंभ करें {#1-restart-the-application}

POS को बंद करें और फिर से खोलें:
1. एप्लिकेशन को पूरी तरह से बंद करें
2. कुछ सेकंड प्रतीक्षा करें
3. इसे फिर से प्रारंभ करें

### 2. अपने डिवाइस को पुनरारंभ करें {#2-restart-your-device}

पूरा पुनरारंभ अक्सर सेवा समस्याओं को हल करता है:
1. कोई भी महत्वपूर्ण काम सहेजें
2. अपने कंप्यूटर को पुनरारंभ करें
3. फिर से POS आजमाएँ

### 3. सिस्टम संसाधनों की जाँच करें {#3-check-system-resources}

सुनिश्चित करें कि सिस्टम संसाधन उपलब्ध हैं:
- अनावश्यक एप्लिकेशन बंद करें
- मेमोरी उपयोग की जाँच करें
- डिस्क स्पेस की जाँच करें

### 4. एप्लिकेशन को अपडेट करें {#4-update-the-application}

सुनिश्चित करें कि आप नवीनतम संस्करण पर हैं:
- उपलब्ध अपडेट की जाँच करें
- किसी भी लंबित अपडेट को इंस्टॉल करें
- अपडेट करने के बाद पुनरारंभ करें

### 5. यदि आवश्यक हो, तो पुनः स्थापित करें {#5-reinstall-if-necessary}

यदि सेवा अनुपलब्ध बनी रहती है:
1. एप्लिकेशन को अनइंस्टॉल करें
2. अपने डिवाइस को पुनरारंभ करें
3. नवीनतम संस्करण डाउनलोड करें
4. ताज़ा इंस्टॉल करें

## ब्राउज़र-विशिष्ट (वेब संस्करण) {#browser-specific-web-version}

वेब संस्करण के लिए:
- सुनिश्चित करें कि JavaScript सक्षम है
- जाँच करें कि आवश्यक ब्राउज़र सुविधाएँ अवरुद्ध नहीं हैं
- एक अलग ब्राउज़र आज़माएँ
- ब्राउज़र एक्सटेंशनों को निष्क्रिय करें जो विघटनकारी हो सकते हैं

## समर्थन से संपर्क कब करें {#when-to-contact-support}

यदि यह त्रुटि बनी रहती है:
- ध्यान दें कि यह कब होती है
- जाँच करें कि क्या विशेष क्रियाएँ इसे प्रेरित करती हैं
- जांच के लिए विवरण सहित रिपोर्ट करें
