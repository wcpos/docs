# API05003: WCPOS प्लगइन पुराना

## इसका अर्थ क्या है

आपके सर्वर पर स्थापित WCPOS प्लगइन पुराना है और POS एप्लिकेशन के आपके संस्करण के साथ संगत नहीं है। आपको प्लगइन को अपडेट करने की आवश्यकता है।

## सामान्य कारण

- **प्लगइन अपडेट नहीं किया गया** — स्वचालित अपडेट अक्षम हो सकते हैं
- **अपडेट विफल** — पिछला अपडेट प्रयास विफल हो गया
- **संस्करण असंगतता** — POS एप्लिकेशन अपडेट हो गया लेकिन प्लगइन नहीं
- **स्टेजिंग/डेव साइट** — पुराने संस्करण पर परीक्षण

## समाधान कैसे करें

### 1. प्लगइन अपडेट करें

WordPress व्यवस्थापक में:
1. प्लगइन → स्थापित प्लगइन पर जाएं
2. "WooCommerce POS" ढूंढें
3. यदि कोई अपडेट उपलब्ध है, तो "अभी अपडेट करें" पर क्लिक करें

### 2. स्वचालित अपडेट सक्षम करें

भविष्य में असंगतताओं को रोकने के लिए:
1. प्लगइन → स्थापित प्लगइन पर जाएं
2. "WooCommerce POS" ढूंढें
3. "स्वचालित अपडेट सक्षम करें" पर क्लिक करें

### 3. मैनुअल अपडेट

यदि स्वचालित अपडेट विफल होता है:
1. [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से नवीनतम संस्करण डाउनलोड करें
2. वर्तमान प्लगइन को निष्क्रिय करें
3. पुराने प्लगइन फ़ाइलों को हटाएं
4. नया संस्करण अपलोड और सक्रिय करें

### 4. अपडेट त्रुटियों की जाँच करें

यदि अपडेट विफल हो रहे हैं:
- wp-content/plugins पर फ़ाइल अनुमतियों की जाँच करें
- डिस्क स्पेस की पुष्टि करें
- PHP मेमोरी समस्याओं की जाँच करें
- विवरण के लिए त्रुटि लॉग देखें

### 5. POS एप्लिकेशन को डाउनग्रेड करें (अस्थायी)

यदि आप तुरंत प्लगइन अपडेट नहीं कर सकते:
- POS एप्लिकेशन का पुराना संस्करण उपयोग करें
- यह केवल एक अस्थायी समाधान है
- जल्द ही प्लगइन अपडेट करने की योजना बनाएं

## संस्करण संगतता

दोनों घटकों को अपडेट रखें:
- POS एप्लिकेशन
- WCPOS WordPress प्लगइन

अपडेट करते समय किसी भी ब्रेकिंग परिवर्तन के लिए रिलीज नोट्स देखें।
