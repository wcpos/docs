# API05003: WCPOS प्लगइन पुराना

## इसका मतलब

आपके सर्वर पर स्थापित WCPOS प्लगइन पुराना है और यह आपके POS एप्लिकेशन के संस्करण के साथ संगत नहीं है। आपको प्लगइन को अपडेट करने की आवश्यकता है।

## सामान्य कारण

- **प्लगइन अपडेट नहीं किया गया** — ऑटो-अपडेट्स बंद हो सकते हैं
- **अपडेट विफल** — पिछले अपडेट प्रयास में विफलता
- **संस्करण असंगति** — POS एप्लिकेशन अपडेट हुआ लेकिन प्लगइन नहीं हुआ
- **स्टेजिंग/डेव साइट** — पुराने संस्करण पर परीक्षण

## कैसे ठीक करें

### 1. प्लगइन अपडेट करें

WordPress एडमिन में:
1. Plugins → Installed Plugins पर जाएं
2. "WooCommerce POS" खोजें
3. यदि अपडेट उपलब्ध है, तो "Update Now" पर क्लिक करें

### 2. ऑटो-अपडेट सक्षम करें

भविष्य की असंगतियों को रोकने के लिए:
1. Plugins → Installed Plugins पर जाएं
2. "WooCommerce POS" खोजें
3. "Enable auto-updates" पर क्लिक करें

### 3. मैनुअल अपडेट

यदि स्वचालित अपडेट विफल हो जाता है:
1. [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से नवीनतम संस्करण डाउनलोड करें
2. वर्तमान प्लगइन को निष्क्रिय करें
3. पुराने प्लगइन फ़ाइलें हटाएं
4. नया संस्करण अपलोड और सक्रिय करें

### 4. अपडेट त्रुटियों की जांच करें

यदि अपडेट विफल हो रहे हैं:
- wp-content/plugins पर फ़ाइल अनुमतियों की जाँच करें
- सुनिश्चित करें कि आपके पास डिस्क स्थान है
- PHP मेमोरी मुद्दों की जांच करें
- विवरण के लिए त्रुटि लॉग की जांच करें

### 5. POS एप्लिकेशन को डाउनग्रेड करें (अस्थायी)

यदि आप तुरंत प्लगइन अपडेट नहीं कर सकते हैं:
- POS एप्लिकेशन के पुराने संस्करण का उपयोग करें
- यह केवल एक अस्थायी समाधान है
- जल्द ही प्लगइन को अपडेट करने की योजना बनाएं

## संस्करण संगतता

दोनों घटकों को हमेशा अपडेट रखें:
- POS एप्लिकेशन
- WCPOS WordPress प्लगइन

अपडेट करते समय किसी भी महत्वपूर्ण परिवर्तन के लिए रिलीज नोट्स की जांच करें।
