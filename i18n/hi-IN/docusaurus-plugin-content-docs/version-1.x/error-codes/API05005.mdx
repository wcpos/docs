# API05005: प्लगइन नहीं मिला

## इसका क्या अर्थ है

एक आवश्यक वर्डप्रेस प्लगइन इंस्टॉल या सक्रिय नहीं है। यह WooCommerce स्वयं, एक आवश्यक एक्सटेंशन, या एक इंटीग्रेशन प्लगइन हो सकता है।

## सामान्य कारण

- **WooCommerce निष्क्रिय** — WooCommerce को डिसेबल किया गया था
- **आवश्यक एक्सटेंशन गायब** — एक आवश्यक WooCommerce एक्सटेंशन इंस्टॉल नहीं है
- **प्लगइन संघर्ष** — एक अन्य प्लगइन ने एक आवश्यक प्लगइन को निष्क्रिय कर दिया
- **अपडेट असफल** — एक प्लगइन अपडेट विफल रहा और इसे निष्क्रिय छोड़ दिया

## कैसे ठीक करें

### 1. सुनिश्चित करें कि WooCommerce सक्रिय है

WCPOS को WooCommerce की आवश्यकता है:
1. Plugins → Installed Plugins पर जाएं
2. "WooCommerce" खोजें
3. सुनिश्चित करें कि यह सक्रिय है

### 2. आवश्यक एक्सटेंशन की जांच करें

कुछ WCPOS सुविधाओं के लिए एक्सटेंशन की आवश्यकता हो सकती है:
- विशेष प्लगइन नाम के लिए त्रुटि संदेश की जांच करें
- आवश्यक एक्सटेंशन इंस्टॉल और सक्रिय करें

### 3. प्लगइन संघर्षों की समीक्षा करें

यदि प्लगइन्स को संघर्षों के कारण निष्क्रिय कर दिया गया:
1. वर्डप्रेस सूचनाओं के लिए अपने ईमेल की जांच करें
2. त्रुटि लॉग की समीक्षा करें
3. संघर्षों को हल करें और पुनः सक्रिय करें

### 4. प्लगइन फ़ाइलों की जांच करें

यदि किसी प्लगइन की फाइलें गायब हैं:
1. WordPress.org से प्लगइन को पुनः स्थापित करें
2. या बैकअप से पुनर्स्थापित करें
3. प्लगइन को सक्रिय करें

### 5. सभी प्लगइन्स को अपडेट करें

पुराने अपडेट न हुए प्लगइन्स समस्याएँ पैदा कर सकते हैं:
1. Dashboard → Updates पर जाएं
2. सभी प्लगइन्स को अपडेट करें
3. सुनिश्चित करें कि सभी आवश्यक प्लगइन्स सक्रिय हैं

## आवश्यक प्लगइन्स

कम से कम, WCPOS को चाहिए:
- **WooCommerce** — ई-कॉमर्स प्लेटफॉर्म
- **WCPOS** — POS प्लगइन

अतिरिक्त सुविधाओं के लिए अन्य प्लगइन्स की आवश्यकता हो सकती है।
