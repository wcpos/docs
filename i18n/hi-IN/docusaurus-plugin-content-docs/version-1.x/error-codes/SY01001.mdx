# SY01001: मेमोरी समाप्त {#sy01001-out-of-memory}

## इसका क्या अर्थ है {#what-this-means}

आपका उपकरण मेमोरी (RAM) में कम चल रहा है और अनुरोधित ऑपरेशन पूरा नहीं कर सकता। POS को सही ढंग से कार्य करने के लिए पर्याप्त मेमोरी की आवश्यकता है।

## सामान्य कारण {#common-causes}

- **बहुत से ऐप्स खुले हैं** — अन्य अनुप्रयोग जो मेमोरी का उपयोग कर रहे हैं
- **बड़ा डेटा सेट** — कई उत्पाद / ग्राहक लोड किए गए हैं
- **मेमोरी लीक** — अनुप्रयोग सही तरीके से मेमोरी का विमोचन नहीं कर रहा है
- **पर्याप्त RAM नहीं है** — उपकरण में पर्याप्त मेमोरी नहीं है

## इसे कैसे ठीक करें {#how-to-fix}

### 1. अन्य अनुप्रयोग बंद करें {#1-close-other-applications}

मेमोरी खाली करें:
- अनुपयोगी ब्राउज़र टैब बंद करें
- अन्य अनुप्रयोग बंद करें
- बैकग्राउंड प्रोग्राम बंद करें

### 2. POS को पुनःस्टार्ट करें {#2-restart-the-pos}

अनुप्रयोग की मेमोरी को ताज़ा करें:
1. POS को पूरी तरह से बंद करें
2. कुछ सेकंड प्रतीक्षा करें
3. अनुप्रयोग को फिर से खोलें

### 3. अपने उपकरण को पुनःस्टार्ट करें {#3-restart-your-device}

पूर्ण पुनःस्टार्ट सभी मेमोरी को साफ करता है:
1. कोई भी खुला कार्य सहेजें
2. अपने कंप्यूटर / उपकरण को पुनःस्टार्ट करें
3. केवल POS को फिर से शुरू करते समय खोलें

### 4. ब्राउज़र डेटा साफ करें (वेब संस्करण) {#4-clear-browser-data-web-version}

यदि आप ब्राउज़र में उपयोग कर रहे हैं:
- ब्राउज़र कैश को साफ करें
- ब्राउज़र को बंद करें और पुनः खोलें
- कम ब्राउज़र टैब का उपयोग करें

### 5. डेटा लोड कम करें {#5-reduce-data-load}

यदि आपके पास एक बहुत बड़ा कैटलॉग है:
- एक बार में कम डेटा लोड करने के लिए फिल्टर का उपयोग करें
- परिणामों को पृष्ठबद्ध करने पर विचार करें
- यदि संभव हो तो सिंक दायरे को सीमित करें

## न्यूनतम आवश्यकताएँ {#minimum-requirements}

यह सुनिश्चित करें कि आपका उपकरण न्यूनतम विनिर्देशों को पूरा करता है:
- **RAM:** न्यूनतम 4GB, सिफारिशित 8GB
- **ब्राउज़र:** आधुनिक ब्राउज़र (Chrome, Firefox, Edge)
- **डेस्कटॉप ऐप:** आवश्यकताओं के लिए रिलीज़ नोट्स की जांच करें

## चल रही समस्याएँ {#ongoing-issues}

यदि आपको अक्सर मेमोरी समाप्त हो रही है:
- अपने उपकरण की RAM अपग्रेड करने पर विचार करें
- अधिक संसाधनों वाला उपकरण उपयोग करें
- यदि समस्या अधिक प्रतीत होती है तो रिपोर्ट करें
