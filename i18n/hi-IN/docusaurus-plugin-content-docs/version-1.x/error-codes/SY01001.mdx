# SY01001: मेमोरी समाप्त

## इसका अर्थ क्या है

आपका डिवाइस मेमोरी (RAM) में कम हो रहा है और अनुरोधित ऑपरेशन को पूरा नहीं कर सक रहा है। POS को ठीक से कार्य करने के लिए पर्याप्त मेमोरी की आवश्यकता होती है।

## सामान्य कारण

- **बहुत से ऐप्स खुले हैं** — अन्य एप्लिकेशन मेमोरी का उपयोग कर रहे हैं
- **बड़ा डेटा सेट** — कई उत्पाद/ग्राहक लोड किए गए
- **मेमोरी लीक** — एप्लिकेशन मेमोरी को ठीक से मुक्त नहीं कर रहा
- **अपर्याप्त RAM** — डिवाइस में पर्याप्त मेमोरी नहीं है

## समाधान कैसे करें

### 1. अन्य एप्लिकेशन बंद करें

मेमोरी मुक्त करें:
- अनुपयोगी ब्राउज़र टैब बंद करें
- अन्य एप्लिकेशन बंद करें
- बैकग्राउंड प्रोग्राम बंद करें

### 2. POS को पुनः शुरू करें

एप्लिकेशन मेमोरी को ताज़ा करें:
1. POS को पूरी तरह से बंद करें
2. कुछ सेकंड प्रतीक्षा करें
3. एप्लिकेशन को फिर से खोलें

### 3. अपने डिवाइस को पुनः शुरू करें

एक पूर्ण पुनरारंभ सभी मेमोरी को साफ करता है:
1. किसी भी खुले काम को सेव करें
2. अपने कंप्यूटर/डिवाइस को पुनः शुरू करें
3. शुरू करते समय केवल POS खोलें

### 4. ब्राउज़र डेटा साफ करें (वेब संस्करण)

यदि ब्राउज़र में उपयोग कर रहे हैं:
- ब्राउज़र कैश साफ करें
- ब्राउज़र को बंद और फिर से खोलें
- कम ब्राउज़र टैब का उपयोग करें

### 5. डेटा लोड कम करें

यदि आपके पास एक बहुत बड़ा कैटलॉग है:
- कम डेटा लोड करने के लिए फ़िल्टर का उपयोग करें
- परिणामों को पेजिनेट करने पर विचार करें
- सिंक दायरे को सीमित करें यदि संभव हो

## न्यूनतम आवश्यकताएं

सुनिश्चित करें कि आपका डिवाइस न्यूनतम विशिष्टताओं को पूरा करता है:
- **RAM:** 4GB न्यूनतम, 8GB अनुशंसित
- **ब्राउज़र:** आधुनिक ब्राउज़र (Chrome, Firefox, Edge)
- **डेस्कटॉप ऐप:** रिलीज़ नोट्स में आवश्यकताओं की जाँच करें

## चल रही समस्याएं

यदि आप बार-बार मेमोरी समाप्त होते हैं:
- अपने डिवाइस के RAM को अपग्रेड करने पर विचार करें
- अधिक संसाधनों वाले डिवाइस का उपयोग करें
- यदि यह अधिक लगता है तो मुद्दे की रिपोर्ट करें
