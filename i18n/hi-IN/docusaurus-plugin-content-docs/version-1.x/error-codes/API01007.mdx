# API01007: डिवाइस ऑफ़लाइन {#api01007-device-offline}

## इसका मतलब {#what-this-means}

आपका डिवाइस इंटरनेट से कनेक्ट नहीं है। POS ने यह पता लगाया कि कोई सक्रिय नेटवर्क कनेक्शन नहीं है।

## सामान्य कारण {#common-causes}

- **WiFi डिस्कनेक्टेड** — आपका डिवाइस अपना WiFi कनेक्शन खो बैठा
- **इथरनेट अनप्लग्ड** — नेटवर्क केबल डिस्कनेक्टेड है
- **एयरप्लेन मोड सक्षम** — सभी वायरलेस कनेक्शन निष्क्रिय हैं
- **नेटवर्क एडेप्टर निष्क्रिय** — नेटवर्क इंटरफ़ेस बंद है

## इसे ठीक कैसे करें {#how-to-fix}

### 1. WiFi कनेक्शन जांचें {#1-check-wifi-connection}

- सुनिश्चित करें कि आपके डिवाइस पर WiFi सक्षम है
- सुनिश्चित करें कि आप सही नेटवर्क से जुड़े हैं
- WiFi सिग्नल की ताकत चेक करें

### 2. भौतिक कनेक्शन जांचें {#2-check-physical-connections}

यदि आप इथरनेट का उपयोग कर रहे हैं:
- सुनिश्चित करें कि केबल securely प्लग इन है
- एक अलग केबल या पोर्ट आज़माएँ
- नेटवर्क स्विच/राउटर लाइट्स चेक करें

### 3. एयरप्लेन मोड को निष्क्रिय करें {#3-disable-airplane-mode}

जांचें कि एयरप्लेन मोड गलती से सक्षम है या नहीं और इसे निष्क्रिय करें।

### 4. नेटवर्क एडेप्टर को रीस्टार्ट करें {#4-restart-network-adapter}

**विंडोज़:**
- नेटवर्क कनेक्शनों को खोलें
- अपने एडेप्टर पर राइट-क्लिक करें → निष्क्रिय करें, फिर सक्षम करें

**मैक ओएस:**
- मेनू बार से WiFi बंद और चालू करें
- या: सिस्टम प्राथमिकताएँ → नेटवर्क → बंद/चालू करें

### 5. अपने डिवाइस को रीस्टार्ट करें {#5-restart-your-device}

कभी-कभी एक साधारण रीस्टार्ट नेटवर्क पहचान समस्याओं को हल करता है।

## ऑफ़लाइन मोड {#offline-mode}

ऑफ़लाइन रहते हुए, WCPOS अभी भी कर सकता है:
- स्थानीय कैश किए गए उत्पादों को ब्राउज़ और खोजें
- स्थानीय कैश किए गए ग्राहकों को ब्राउज़ और खोजें
- नए ऑर्डर शुरू करें और कार्ट में आइटम जोड़ें
- पहले से समन्वयित डेटा देखें

**कनेक्टिविटी की आवश्यकता है:**
- ऑर्डर पूरा करना/चेकआउट करना
- नए ग्राहकों का निर्माण

जब कनेक्टिविटी बहाल होती है, तो POS स्वचालित रूप से सामान्य कार्यों को फिर से शुरू करेगा।
