# API01007: डिवाइस ऑफ़लाइन

## इसका अर्थ

आपका डिवाइस इंटरनेट से कनेक्ट नहीं है। POS ने पाया कि कोई सक्रिय नेटवर्क कनेक्शन नहीं है।

## सामान्य कारण

- **WiFi डिसकनेक्ट** — आपके डिवाइस का WiFi कनेक्शन टूट गया है
- **ईथरनेट अनप्लग** — नेटवर्क केबल डिसकनेक्ट है
- **एयरप्लेन मोड सक्षम** — सभी वायरलेस कनेक्शन बंद हैं
- **नेटवर्क एडाप्टर अक्षम** — नेटवर्क इंटरफ़ेस बंद है

## समाधान कैसे करें

### 1. WiFi कनेक्शन की जाँच करें

- सुनिश्चित करें कि डिवाइस पर WiFi सक्षम है
- जाँचें कि आप सही नेटवर्क से कनेक्ट हैं
- WiFi सिग्नल की शक्ति की जाँच करें

### 2. भौतिक कनेक्शन की जाँच करें

ईथरनेट उपयोग करते समय:
- जाँचें कि केबल ठीक से प्लग है
- एक अलग केबल या पोर्ट का प्रयास करें
- नेटवर्क स्विच/राउटर के लाइट्स देखें

### 3. एयरप्लेन मोड बंद करें

जाँचें कि एयरप्लेन मोड अनजाने में सक्षम तो नहीं है और इसे बंद कर दें।

### 4. नेटवर्क एडाप्टर रीस्टार्ट करें

**Windows:**
- नेटवर्क कनेक्शन खोलें
- अपने एडाप्टर पर राइट-क्लिक करें → अक्षम, फिर सक्षम करें

**macOS:**
- मेनू बार से WiFi बंद और चालू करें
- या: सिस्टम प्राथमिकताएं → नेटवर्क → बंद/चालू करें

### 5. अपना डिवाइस रीस्टार्ट करें

कभी-कभी एक सरल रीस्टार्ट नेटवर्क पहचान की समस्याओं को हल कर देता है।

## ऑफ़लाइन मोड

ऑफ़लाइन होने पर, WCPOS अभी भी कर सकता है:
- स्थानीय रूप से कैश किए गए उत्पादों को ब्राउज़ और खोज
- स्थानीय रूप से कैश किए गए ग्राहकों को ब्राउज़ और खोज
- नए ऑर्डर शुरू करें और कार्ट में आइटम जोड़ें
- पहले सिंक किए गए डेटा देखें

**कनेक्टिविटी आवश्यक:**
- ऑर्डर पूरा करना/चेकआउट
- नए ग्राहक बनाना

जब कनेक्टिविटी पुनर्स्थापित होती है, POS स्वचालित रूप से सामान्य संचालन फिर से शुरू कर देगा।
