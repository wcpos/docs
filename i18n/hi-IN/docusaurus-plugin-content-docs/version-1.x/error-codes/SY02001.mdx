# SY02001: अमान्य कॉन्फ़िगरेशन

## इसका अर्थ क्या है

सिस्टम कॉन्फ़िगरेशन अमान्य या भ्रष्ट है। POS एप्लिकेशन ने कुछ ऐसी सेटिंग्स पाई हैं जो तार्किक नहीं हैं या अधूरी हैं।

## सामान्य कारण

- **भ्रष्ट सेटिंग्स** — कॉन्फ़िगरेशन फ़ाइल भ्रष्ट हो गई
- **अधूरा सेटअप** — सेटअप प्रक्रिया पूरी नहीं हुई
- **मैनुअल संपादन त्रुटि** — कॉन्फ़िग फ़ाइलों को गलत तरीके से संशोधित किया गया
- **संस्करण अपग्रेड समस्या** — पुराना कॉन्फ़िग नए संस्करण के साथ संगत नहीं है

## समाधान कैसे करें

### 1. डिफ़ॉल्ट सेटिंग्स पर रीसेट करें

यदि ऐप में उपलब्ध है:
- "सेटिंग्स रीसेट करें" या "डिफ़ॉल्ट पर पुनर्स्थापित करें" को खोजें
- यह कस्टम सेटिंग्स को साफ करता है लेकिन भ्रष्टता को ठीक करता है

### 2. सेटअप पूरा करें

यदि सेटअप पूरा नहीं हुआ है:
1. सेटअप प्रक्रिया फिर से शुरू करें
2. सभी आवश्यक चरण पूरे करें
3. समाप्त करने से पहले कॉन्फ़िगरेशन की पुष्टि करें

### 3. एप्लिकेशन डेटा साफ़ करें

भ्रष्ट कॉन्फ़िग हटाएं:
- **डेस्कटॉप:** ऐप डेटा फ़ोल्डर हटाएं, ऐप को पुनः शुरू करें
- **ब्राउज़र:** साइट डेटा साफ़ करें, पुनः लोड करें

### 4. एप्लिकेशन को फिर से इंस्टॉल करें

लगातार समस्याओं के लिए:
1. पूरी तरह से अनइंस्टॉल करें
2. किसी भी बचे हुए डेटा फ़ोल्डर को हटाएं
3. नया इंस्टॉल करें
4. शुरू से सेटअप पूरा करें

### 5. अपडेट के लिए जाँच करें

कॉन्फ़िगरेशन स्वरूप बदल सकते हैं:
- POS एप्लिकेशन को अपडेट करें
- WCPOS प्लगइन को अपडेट करें
- अपडेट के बाद पुनः कॉन्फ़िगर करें

## क्या रीसेट होगा

कॉन्फ़िगरेशन रीसेट करते समय:
- आपको फिर से लॉग इन करना होगा
- कस्टम सेटिंग्स खो जाएंगी
- डेटा को फिर से सिंक करना होगा
- लेकिन आका WooCommerce डेटा सर्वर पर सुरक्षित है
