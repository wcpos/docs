# SY02001: अमान्य कॉन्फ़िगरेशन

## इसका क्या मतलब है

सिस्टम कॉन्फ़िगरेशन अमान्य या भ्रष्ट है। POS एप्लिकेशन ने ऐसे कॉन्फ़िगरेशन सेटिंग्स पाई हैं जो समझ में नहीं आतीं या अधूरी हैं।

## सामान्य कारण

- **भ्रष्ट सेटिंग्स** — कॉन्फ़िगरेशन फ़ाइल भ्रष्ट हो गई
- **अधूरा सेटअप** — सेटअप प्रक्रिया पूरी नहीं हुई
- **मैनुअल संपादन त्रुटि** — कॉन्फ़िग फ़ाइलें गलत तरीके से संशोधित की गईं
- **संस्करण उन्नयन मुद्दा** — पुराना कॉन्फ़िग नया संस्करण के साथ असंगत

## इसे कैसे ठीक करें

### 1. डिफ़ॉल्ट सेटिंग्स पर रीसेट करें

यदि एप्लिकेशन में उपलब्ध हो:
- "रीसेट सेटिंग्स" या "डिफ़ॉल्ट पुनर्स्थापित करें" की तलाश करें
- यह कस्टम सेटिंग्स को साफ करता है लेकिन भ्रष्टता को ठीक करता है

### 2. सेटअप पूरा करें

यदि सेटअप पूरा नहीं हुआ:
1. फिर से सेटअप प्रक्रिया शुरू करें
2. सभी आवश्यक कदम पूरे करें
3. समाप्त करने से पहले कॉन्फ़िगरेशन की जांच करें

### 3. एप्लिकेशन डेटा साफ़ करें

भ्रष्ट कॉन्फ़िग को हटा दें:
- **डेस्कटॉप:** ऐप डेटा फ़ोल्डर हटाएं, ऐप को पुनरारंभ करें
- **ब्राउज़र:** साइट डेटा साफ़ करें, फिर से लोड करें

### 4. एप्लिकेशन को पुनर्स्थापित करें

जिद्दी समस्याओं के लिए:
1. पूरी तरह से अनइंस्टॉल करें
2. किसी भी शेष डेटा फ़ोल्डरों को हटा दें
3. नया इंस्टॉल करें
4. ज़ीरो से सेटअप पूरा करें

### 5. अपडेट के लिए जाँच करें

कॉन्फ़िगरेशन प्रारूप बदल सकते हैं:
- POS एप्लिकेशन को अपडेट करें
- WCPOS प्लगइन को अपडेट करें
- अपडेट के बाद पुनः कॉन्फ़िगर करें

## क्या रीसेट होता है

जब कॉन्फ़िगरेशन को रीसेट किया जाता है:
- आपको फिर से लॉग इन करना होगा
- कस्टम सेटिंग्स खो जाएँगी
- डेटा को फिर से सामंजस्य स्थापित करने की आवश्यकता होगी
- लेकिन आपका WooCommerce डेटा सर्वर पर सुरक्षित है
