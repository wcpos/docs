# API02005: अपर्याप्त अनुमतियाँ

## इसका अर्थ

आपके उपयोगकर्ता खाते में इस क्रिया के लिए आवश्यक विशिष्ट WordPress क्षमताएँ नहीं हैं। हालांकि आपको मूलभूत पहुंच है, लेकिन आप जिस विशेष संचालन का प्रयास कर रहे हैं, उसके लिए अतिरिक्त अनुमतियों की आवश्यकता है।

## सामान्य कारण

- **सीमित उपयोगकर्ता भूमिका** — आपकी भूमिका में सभी आवश्यक क्षमताएँ शामिल नहीं हैं
- **क्षमता असाइन नहीं की गई** — आपकी भूमिका में एक विशिष्ट क्षमता गायब है
- **प्लगइन प्रतिबंध** — एक सुरक्षा प्लगइन क्षमताओं को सीमित कर रहा है
- **कस्टम भूमिका समस्याएँ** — कस्टम भूमिकाओं में क्षमताएँ गायब हो सकती हैं

## सर्वर त्रुटि मैपिंग

यह त्रुटि कोड तब ट्रिगर होता है जब सर्वर लौटाता है:

| सर्वर कोड | स्रोत |
|-------------|--------|
| `rest_forbidden` | WordPress REST API |
| `rest_cannot_create` | WordPress REST API |
| `rest_cannot_edit` | WordPress REST API |
| `rest_cannot_delete` | WordPress REST API |
| `woocommerce_rest_cannot_create` | WooCommerce REST API |
| `woocommerce_rest_cannot_edit` | WooCommerce REST API |
| `woocommerce_rest_cannot_delete` | WooCommerce REST API |
| HTTP 403 | कोई भी सर्वर प्रतिक्रिया (फॉलबैक) |

## समाधान कैसे करें

### 1. आवश्यक क्षमताएँ की समीक्षा करें

POS संचालन के लिए आवश्यक सामान्य क्षमताएँ:
- `manage_woocommerce` — सामान्य WooCommerce प्रबंधन
- `edit_shop_orders` — ऑर्डर बनाएँ और संपादित करें
- `edit_products` — उत्पाद जानकारी संशोधित करें
- `edit_users` — ग्राहक खाते प्रबंधित करें

### 2. उपयोगकर्ता भूमिका अपग्रेड करें

किसी व्यवस्थापक से अधिक सक्षम भूमिका असाइन करने के लिए कहें:
- **शॉप प्रबंधक** — पूर्ण WooCommerce पहुंच
- **व्यवस्थापक** — पूर्ण साइट पहुंच

### 3. विशिष्ट क्षमताएँ जोड़ें

यदि आपको एक कस्टम भूमिका चाहिए, तो आवश्यक क्षमताएँ जोड़ें:

```php
// उदाहरण: एक कस्टम भूमिका में POS क्षमताएँ जोड़ें
$role = get_role('your_custom_role');
$role->add_cap('manage_woocommerce');
$role->add_cap('edit_shop_orders');
```

### 4. प्लगइन संघर्ष जाँचें

कुछ सुरक्षा या भूमिका प्रबंधन प्लगइन क्षमताओं को सीमित कर सकते हैं:
- प्लगइन सेटिंग्स की समीक्षा करें
- क्षमता फ़िल्टर की जाँच करें
- परीक्षण के लिए अस्थायी रूप से अक्षम करें
