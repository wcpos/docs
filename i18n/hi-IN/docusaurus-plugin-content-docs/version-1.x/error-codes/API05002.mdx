# API05002: WCPOS प्लगइन नहीं मिला

## इसका अर्थ क्या है

WCPOS (WooCommerce POS) प्लगइन आपकी WordPress साइट पर स्थापित नहीं है या सक्रिय नहीं है। POS एप्लिकेशन को WooCommerce से संवाद करने के लिए इस प्लगइन की आवश्यकता होती है।

## सामान्य कारण

- **प्लगइन स्थापित नहीं** — प्लगइन कभी स्थापित नहीं किया गया
- **प्लगइन निष्क्रिय** — प्लगइन को अक्षम कर दिया गया है
- **प्लगइन हटा दिया गया** — प्लगइन फ़ाइलें हटा दी गई हैं
- **गलत साइट** — WCPOS के बिना साइट से कनेक्ट किया गया

## समाधान कैसे करें

### 1. प्लगइन स्थापित करें

यदि स्थापित नहीं है:
1. WordPress एडमिन → प्लगइन → नया जोड़ें
2. "WooCommerce POS" खोजें
3. प्लगइन स्थापित और सक्रिय करें

या [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से डाउनलोड करें

### 2. प्लगइन सक्रिय करें

यदि स्थापित है लेकिन सक्रिय नहीं:
1. प्लगइन → स्थापित प्लगइन
2. "WooCommerce POS" ढूंढें
3. "सक्रिय करें" पर क्लिक करें

### 3. त्रुटियों की जाँच करें

यदि प्लगइन सक्रिय नहीं हो रहा है:
- अपने त्रुटि लॉग में PHP त्रुटियों की जाँच करें
- सुनिश्चित करें कि WooCommerce स्थापित और सक्रिय है
- PHP संस्करण आवश्यकताओं की जाँच करें
- प्लगइन संघर्षों की तलाश करें

### 4. URL सत्यापित करें

सुनिश्चित करें कि POS सही WordPress साइट से कनेक्ट होने के लिए कॉन्फ़िगर है:
- POS सेटिंग में URL की जाँच करें
- सत्यापित करें कि यह आपकी WordPress स्थापना से मेल खाता है

### 5. आवश्यकतानुसार पुनः स्थापित करें

यदि प्लगइन क्षतिग्रस्त है:
1. प्लगइन निष्क्रिय और हटा दें
2. WordPress.org से पुनः स्थापित करें
3. प्लगइन सक्रिय करें

## आवश्यकताएं

WCPOS को आवश्यक है:
- WordPress (नवीनतम संस्करण)
- WooCommerce (नवीनतम संस्करण)
- PHP 7.4 या उच्चतर
