# API05002: WCPOS प्लगइन नहीं पाया गया

## इसका अर्थ क्या है

WCPOS (WooCommerce POS) प्लगइन आपके वर्डप्रेस साइट पर स्थापित या सक्रिय नहीं है। POS एप्लिकेशन को WooCommerce के साथ संवाद करने के लिए इस प्लगइन की आवश्यकता होती है।

## सामान्य कारण

- **प्लगइन स्थापित नहीं है** — प्लगइन कभी स्थापित नहीं किया गया
- **प्लगइन निष्क्रिय है** — प्लगइन को निष्क्रिय कर दिया गया
- **प्लगइन हटाया गया** — प्लगइन की फाइलें हटा दी गईं
- **गलत साइट** — WCPOS के बिना किसी साइट से जुड़ा हुआ है

## इसे कैसे ठीक करें

### 1. प्लगइन स्थापित करें

यदि स्थापित नहीं है:
1. वर्डप्रेस एडमिन → प्लगइन्स → नया जोड़ें पर जाएं
2. "WooCommerce POS" के लिए खोजें
3. प्लगइन को स्थापित करें और सक्रिय करें

या [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से डाउनलोड करें

### 2. प्लगइन को सक्रिय करें

यदि स्थापित है लेकिन सक्रिय नहीं है:
1. प्लगइन्स → स्थापित प्लगइन्स पर जाएं
2. "WooCommerce POS" खोजें
3. "Activate" पर क्लिक करें

### 3. त्रुटियों के लिए जांचें

यदि प्लगइन सक्रिय नहीं होता है:
- अपनी त्रुटि लॉग में PHP त्रुटियों के लिए जांचें
- पुष्टि करें कि WooCommerce स्थापित और सक्रिय है
- PHP संस्करण आवश्यकताओं की जांच करें
- प्लगइन संघर्ष के लिए देखें

### 4. URL की पुष्टि करें

सुनिश्चित करें कि POS को सही वर्डप्रेस साइट से कनेक्ट करने के लिए कॉन्फ़िगर किया गया है:
- POS सेटिंग्स में URL की जांच करें
- पुष्टि करें कि यह आपके वर्डप्रेस इंस्टॉलेशन से मेल खाता है

### 5. आवश्यक होने पर पुनः स्थापित करें

यदि प्लगइन भ्रष्ट है:
1. प्लगइन को निष्क्रिय करें और हटा दें
2. WordPress.org से पुनः स्थापित करें
3. प्लगइन को सक्रिय करें

## आवश्यकताएँ

WCPOS को निम्नलिखित की आवश्यकता है:
- वर्डप्रेस (हाल का संस्करण)
- WooCommerce (हाल का संस्करण)
- PHP 7.4 या उससे अधिक
