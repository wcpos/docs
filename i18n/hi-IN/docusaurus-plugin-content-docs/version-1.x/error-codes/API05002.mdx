# API05002: WCPOS प्लगइन नहीं मिला

## इसका क्या मतलब है

WCPOS (WooCommerce POS) प्लगइन आपके WordPress साइट पर इंस्टॉल नहीं है या सक्रिय नहीं है। POS एप्लिकेशन को WooCommerce के साथ संवाद करने के लिए इस प्लगइन की आवश्यकता है।

## सामान्य कारण

- **प्लगइन स्थापित नहीं है** — प्लगइन कभी इंस्टॉल नहीं हुआ
- **प्लगइन निष्क्रिय** — प्लगइन अक्षम किया गया
- **प्लगइन हटाया गया** — प्लगइन फाइलें हटा दी गईं
- **गलत साइट** — WCPOS के बिना साइट से जुड़ा हुआ

## इसे कैसे ठीक करें

### 1. प्लगइन इंस्टॉल करें

यदि स्थापित नहीं है:
1. WordPress Admin → Plugins → नया जोड़ें पर जाएं
2. "WooCommerce POS" खोजें
3. प्लगइन इंस्टॉल और सक्रिय करें

या [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से डाउनलोड करें

### 2. प्लगइन सक्रिय करें

यदि इंस्टॉल हैं लेकिन सक्रिय नहीं है:
1. Plugins → Installed Plugins पर जाएं
2. "WooCommerce POS" खोजें
3. "सक्रिय करें" पर क्लिक करें

### 3. त्रुटियों की जांच करें

यदि प्लगइन सक्रिय नहीं हो रहा है:
- अपने त्रुटि लॉग में PHP त्रुटियों की जांच करें
- पुष्टि करें कि WooCommerce इंस्टॉल और सक्रिय है
- PHP संस्करण आवश्यकताओं की जांच करें
- प्लगइन संघर्ष देखें

### 4. URL की पुष्टि करें

सुनिश्चित करें कि POS को सही WordPress साइट से कनेक्ट करने के लिए कॉन्फ़िगर किया गया है:
- POS सेटिंग्स में URL की जांच करें
- पुष्टि करें कि यह आपके WordPress इंस्टॉलेशन से मेल खाता है

### 5. यदि आवश्यक हो तो पुन: स्थापित करें

यदि प्लगइन दूषित है:
1. प्लगइन को निष्क्रिय और हटा दें
2. WordPress.org से पुन: इंस्टॉल करें
3. प्लगइन सक्रिय करें
