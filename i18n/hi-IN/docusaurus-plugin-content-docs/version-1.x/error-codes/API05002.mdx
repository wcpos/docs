# API05002: WCPOS प्लगइन नहीं मिला

## इसका क्या मतलब है

WCPOS (WooCommerce POS) प्लगइन आपके वर्डप्रेस साइट पर स्थापित नहीं है या सक्रिय नहीं है। POS ऐप को WooCommerce के साथ संचार करने के लिए इस प्लगइन की आवश्यकता है।

## सामान्य कारण

- **प्लगइन स्थापित नहीं** — प्लगइन कभी स्थापित नहीं किया गया
- **प्लगइन निष्क्रिय** — प्लगइन को अक्षम कर दिया गया
- **प्लगइन हटाया गया** — प्लगइन फाइलें हटा दी गईं
- **गलत साइट** — एक ऐसी साइट से जुड़ा जिसे WCPOS नहीं है

## इसे कैसे ठीक करें

### 1. प्लगइन स्थापित करें

यदि स्थापित नहीं:
1. वर्डप्रेस प्रशासन → प्लगइन्स → नया जोड़ें पर जाएं
2. "WooCommerce POS" के लिए खोजें
3. प्लगइन स्थापित करें और इसे सक्रिय करें

या [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) से डाउनलोड करें

### 2. प्लगइन सक्रिय करें

यदि स्थापित है लेकिन सक्रिय नहीं:
1. प्लगइन्स → स्थापित प्लगइन्स पर जाएं
2. "WooCommerce POS" खोजें
3. "सक्रिय करें" पर क्लिक करें

### 3. त्रुटियों की जांच करें

यदि प्लगइन सक्रिय नहीं होता:
- अपनी त्रुटि लॉग में PHP त्रुटियों के लिए जांचें
- पुष्टि करें कि WooCommerce स्थापित और सक्रिय है
- PHP संस्करण आवश्यकताओं की जांच करें
- प्लगइन संघर्षों के लिए देखें

### 4. URL की पुष्टि करें

सुनिश्चित करें कि POS को सही वर्डप्रेस साइट से कनेक्ट करने के लिए कॉन्फ़िगर किया गया है:
- POS सेटिंग्स में URL की जांच करें
- पुष्टि करें कि यह आपके वर्डप्रेस स्थापना से मेल खाता है

### 5. आवश्यक होने पर पुनः स्थापित करें

यदि प्लगइन भ्रष्ट हो गया है:
1. प्लगइन को निष्क्रिय करें और हटा दें
2. WordPress.org से पुनः स्थापित करें
3. प्लगइन को सक्रिय करें

## आवश्यकताएँ

WCPOS की आवश्यकता है:
- वर्डप्रेस (हाल का संस्करण)
- WooCommerce (हाल का संस्करण)
- PHP 7.4 या उससे अधिक
