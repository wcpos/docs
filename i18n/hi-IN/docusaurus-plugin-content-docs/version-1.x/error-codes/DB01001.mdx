# DB01001: कनेक्शन विफल {#db01001-connection-failed}

## इसका अर्थ {#what-this-means}

POS अपने स्थानीय डेटाबेस से कनेक्ट नहीं हो सका। WCPOS एक स्थानीय डेटाबेस का उपयोग करता है ताकि उत्पादों, ग्राहकों और अन्य डेटा को तेज़ पहुँच और ऑफ़लाइन कार्यक्षमता के लिए संग्रहीत किया जा सके।

## सामान्य कारण {#common-causes}

- **स्टोरेज पूरा** — डिवाइस में कोई स्टोरेज स्थान नहीं बचा
- **डेटाबेस क्षतिग्रस्त** — स्थानीय डेटाबेस फ़ाइलें दूषित हैं
- **अनुमति समस्या** — ऐप को स्टोरेज अनुमतियां नहीं हैं
- **ब्राउज़र स्टोरेज निष्क्रिय** — IndexedDB निष्क्रिय है (वेब संस्करण)

## इसे कैसे ठीक करें {#how-to-fix}

### 1. स्टोरेज स्थान की जाँच करें {#1-check-storage-space}

सुनिश्चित करें कि आपके डिवाइस में उपलब्ध स्टोरेज है:
- यदि स्टोरेज पूरा है तो स्थान खाली करें
- अप्रयुक्त ऐप्स या फ़ाइलें हटाएं
- उपलब्ध डिस्क स्थान की जाँच करें

### 2. ऐप डेटा साफ़ करें {#2-clear-app-data}

स्थानीय डेटाबेस रीसेट करें:
- **डेस्कटॉप ऐप:** सेटिंग्स में "डेटा साफ़ करें" की खोज करें
- **वेब ब्राउज़र:** POS डोमेन के लिए साइट डेटा साफ़ करें
- **नोट:** इसके लिए सभी डेटा को फिर से समन्वयित करना आवश्यक होगा

### 3. ब्राउज़र सेटिंग्स की जाँच करें (वेब) {#3-check-browser-settings-web}

यदि आप वेब संस्करण का उपयोग कर रहे हैं:
1. सुनिश्चित करें कि IndexedDB सक्षम है
2. निजी/गोपनीय मोड का उपयोग न करें (सीमित स्टोरेज)
3. साइट को डेटा संग्रहीत करने की अनुमति दें

### 4. अनुमतियों की जाँच करें {#4-check-permissions}

डेस्कटॉप पर:
- सुनिश्चित करें कि ऐप को इसके डेटा निर्देशिका में लिखने की अनुमति है
- जाँच करें कि क्या एंटीवायरस डेटाबेस तक पहुँच को ब्लॉक कर रहा है

### 5. ऐप को फिर से स्थापित करें {#5-reinstall-the-app}

यदि अन्य समाधान विफल होते हैं:
1. POS ऐप को अनइंस्टॉल करें
2. अपने डिवाइस को रिस्टार्ट करें
3. ऐप को फिर से स्थापित करें
4. लॉगिन करें और डेटा को समन्वयित करें

## डेटा रिकवरी {#data-recovery}

यदि आपके पास असंचित डेटा है:
- हो सकता है ऐप में कुछ लंबित परिवर्तन हों जिन्हें समन्वयित नहीं किया जा सका
- यदि आपको डेटा पुनर्प्राप्त करने में सहायता की आवश्यकता हो तो समर्थन से संपर्क करें
- अधिकांश मामलों में, डेटा को सर्वर से फिर से समन्वयित किया जा सकता है
