# DB01001: कनेक्शन विफल

## इसका अर्थ

पीओएस अपने स्थानीय डेटाबेस से कनेक्ट नहीं हो सका। WCPOS तेज़ पहुंच और ऑफ़लाइन कार्यक्षमता के लिए एक स्थानीय डेटाबेस का उपयोग करता है।

## सामान्य कारण

- **भंडारण भरा हुआ** — डिवाइस में भंडारण स्थान शेष नहीं
- **डेटाबेस भ्रष्ट** — स्थानीय डेटाबेस फ़ाइलें क्षतिग्रस्त
- **अनुमति समस्या** — ऐप के पास भंडारण अनुमतियां नहीं
- **ब्राउज़र भंडारण निष्क्रिय** — इंडेक्स्डबी निष्क्रिय (वेब संस्करण)

## समाधान कैसे करें

### 1. भंडारण स्थान जांचें

सुनिश्चित करें कि आपके डिवाइस में उपलब्ध भंडारण है:
- यदि भंडारण भरा हुआ है तो जगह खाली करें
- अनुपयोगी ऐप्स या फ़ाइलें हटाएं
- उपलब्ध डिस्क स्थान की जांच करें

### 2. ऐप डेटा साफ़ करें

स्थानीय डेटाबेस रीसेट करें:
- **डेस्कटॉप ऐप:** सेटिंग्स में "डेटा साफ़ करें" देखें
- **वेब ब्राउज़र:** POS डोमेन के लिए साइट डेटा साफ़ करें
- **ध्यान दें:** इसके लिए सभी डेटा को फिर से सिंक करना होगा

### 3. ब्राउज़र सेटिंग्स जांचें (वेब)

वेब संस्करण का उपयोग करते समय:
1. सुनिश्चित करें कि इंडेक्स्डबी सक्षम है
2. निजी/इनकॉग्निटो मोड न चुनें (सीमित भंडारण)
3. साइट को डेटा संग्रहीत करने की अनुमति दें

### 4. अनुमतियां जांचें

डेस्कटॉप पर:
- सुनिश्चित करें कि ऐप को अपने डेटा निर्देशिका में लिखने की अनुमति है
- जांचें कि एंटीवायरस डेटाबेस तक पहुंच को अवरुद्ध तो नहीं कर रहा

### 5. ऐप पुनर्स्थापित करें

यदि अन्य समाधान विफल होते हैं:
1. POS ऐप अनइंस्टॉल करें
2. अपने डिवाइस को पुनरारंभ करें
3. ऐप को फिर से इंस्टॉल करें
4. लॉगिन करें और डेटा सिंक करें

## डेटा पुनर्प्राप्ति

यदि आपके पास असहेजा गया डेटा है:
- ऐप के पास लंबित परिवर्तन हो सकते हैं जो सिंक नहीं हो सके
- डेटा पुनर्प्राप्ति में सहायता के लिए समर्थन से संपर्क करें
- अधिकांश मामलों में, डेटा को सर्वर से फिर से सिंक किया जा सकता है
