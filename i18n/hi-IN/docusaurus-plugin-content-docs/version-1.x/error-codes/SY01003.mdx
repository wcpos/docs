# SY01003: अनुमति अस्वीकृत

## इसका अर्थ

POS एप्लिकेशन को एक आवश्यक संसाधन तक पहुंचने की अनुमति नहीं है। यह एक फ़ाइल, फ़ोल्डर या सिस्टम सुविधा हो सकती है।

## सामान्य कारण

- **फ़ाइल अनुमतियां** — आवश्यक फ़ाइलों को पढ़ने या लिखने में असमर्थ
- **ऐप अनुमतियां** — एप्लिकेशन को आवश्यक अनुमतियां नहीं मिलीं
- **संरक्षित फ़ोल्डर** — प्रतिबंधित स्थान तक पहुंचने का प्रयास
- **सुरक्षा सॉफ़्टवेयर** — एंटीवायरस द्वारा पहुंच अवरुद्ध

## समाधान कैसे करें

### 1. व्यवस्थापक के रूप में चलाएं (Windows)

यदि आरंभिक सेटअप के लिए आवश्यक हो तो ऐप पर राइट-क्लिक करें और "व्यवस्थापक के रूप में चलाएं" चुनें।

### 2. एप्लिकेशन अनुमतियां जांचें

**macOS:**
- सिस्टम प्राथमिकताएं → सुरक्षा और गोपनीयता → गोपनीयता
- जांचें कि ऐप को आवश्यक अनुमतियां मिली हैं

**Windows:**
- सेटिंग्स → गोपनीयता
- ऐप अनुमतियों की समीक्षा करें

### 3. फ़ोल्डर अनुमतियां जांचें

सुनिश्चित करें कि ऐप अपने डेटा फ़ोल्डर तक पहुंच सकता है:
- उपयोगकर्ता को ऐप डेटा निर्देशिका का स्वामित्व होना चाहिए
- लिखने की अनुमतियां सक्षम होनी चाहिए
- फ़ोल्डर पर कोई रीड-ओनली फ़्लैग न हो

### 4. एंटीवायरस सेटिंग्स की समीक्षा करें

सुरक्षा सॉफ़्टवेयर पहुंच अवरुद्ध कर सकता है:
- एंटीवायरस लॉग में अवरोधों की जांच करें
- POS ऐप को अनुमति सूची में जोड़ें
- ऐप डेटा फ़ोल्डर को व्हाइटलिस्ट करें

### 5. एप्लिकेशन पुनर्स्थापित करें

यदि अनुमतियां भ्रष्ट हो गई हैं:
1. POS एप्लिकेशन अनइंस्टॉल करें
2. अपने डिवाइस को पुनरारंभ करें
3. एक मानक स्थान पर फिर से इंस्टॉल करें
4. सामान्य (गैर-व्यवस्थापक) अनुमतियों के साथ चलाएं

## ब्राउज़र अनुमतियां (वेब संस्करण)

वेब संस्करण का उपयोग करते समय:
- साइट के लिए स्थानीय संग्रहण की अनुमति दें
- साइट के लिए कुकीज़ सक्षम करें
- अत्यधिक प्रतिबंधात्मक ब्राउज़र सेटिंग्स का उपयोग न करें
