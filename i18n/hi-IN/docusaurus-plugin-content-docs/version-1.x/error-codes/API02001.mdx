# API02001: अमान्य श्रेयांक

## इसका क्या अर्थ है

आपके द्वारा दर्ज किया गया उपयोगकर्ता नाम या पासवर्ड गलत है। सर्वर ने लॉगिन प्रयास को अस्वीकार कर दिया है क्योंकि श्रेयांक किसी भी वैध खाते से मेल नहीं खाते।

## सामान्य कारण

- **उपयोगकर्ता नाम या पासवर्ड में टाइपो** — कैप्स लॉक या अतिरिक्त स्पेस की जाँच करें
- **गलत खाता** — किसी अन्य साइट के श्रेयांक का उपयोग कर रहा है
- **हाल ही में पासवर्ड बदला गया** — पासवर्ड अपडेट किया गया था लेकिन POS में पुराना पासवर्ड है
- **खाता हटा दिया गया** — उपयोगकर्ता खाता अब मौजूद नहीं है

## सर्वर त्रुटि मानचित्रण

यह त्रुटि कोड तब ट्रिगर होता है जब सर्वर वापस लौटाता है:

| सर्वर कोड | स्रोत |
|-------------|--------|
| `woocommerce_rest_authentication_error` | WooCommerce REST API |
| `jwt_auth_failed` | JWT प्रमाणीकरण प्लगइन |

## कैसे ठीक करें

### 1. अपने श्रेयांक की पुष्टि करें

- अपने उपयोगकर्ता नाम (आमतौर पर ईमेल या WordPress उपयोगकर्ता नाम) की दोबारा जाँच करें
- सुनिश्चित करें कि पासवर्ड सही है
- सीधे WordPress एडमिन में लॉग इन करके पुष्टि करें

### 2. अपना पासवर्ड रीसेट करें

यदि आप अपना पासवर्ड भूल गए हैं:
1. अपने WordPress लॉगिन पेज पर जाएं
2. "पासवर्ड भूल गए?" पर क्लिक करें
3. रीसेट लिंक प्राप्त करने के लिए अपना ईमेल दर्ज करें
4. अपना पासवर्ड अपडेट करें
5. नए पासवर्ड के साथ POS में लॉग इन करें

### 3. उपयोगकर्ता खाते की जाँच करें

WordPress एडमिन में:
- उपयोगकर्ता → सभी उपयोगकर्ता पर जाएं
- खाते के मौजूद होने की पुष्टि करें
- जाँचें कि उसके पास उचित POS पहुंच अनुमतियां हैं

### 4. सहेजे गए श्रेयांक साफ करें

POS में:
- पूरी तरह से लॉग आउट करें
- किसी भी सहेजे गए/कैश किए गए लॉगिन जानकारी को साफ करें
- श्रेयांक को फिर से ताजा तरीके से दर्ज करें
