# API त्रुटियाँ

API त्रुटियाँ तब होती हैं जब आप अपने WooCommerce सर्वर के साथ संचार कर रहे होते हैं। इन त्रुटियों का प्रारंभ `API` से होता है और इन्हें निम्नलिखित श्रेणियों में वर्गीकृत किया गया है:

## श्रेणियाँ

| श्रेणी | कोड रेंज | विवरण |
|----------|------------|-------------|
| [कनेक्शन](#connection-errors) | API01xxx | नेटवर्क और कनेक्टिविटी समस्याएँ |
| [प्रमाणीकरण](#authentication-errors) | API02xxx | लॉगिन, टोकन और अनुमति संबंधित समस्याएँ |
| [अनुरोध](#request-errors) | API03xxx | बाहर जाने वाले अनुरोधों में समस्याएँ |
| [प्रतिक्रिया](#response-errors) | API04xxx | सर्वर प्रतिक्रियाओं में समस्याएँ |
| [प्लगइन/वर्डप्रेस](#pluginwordpress-errors) | API05xxx | वर्डप्रेस या WooCommerce प्लगइनों से संबंधित समस्याएँ |
| [कॉन्फ़िगरेशन](#configuration-errors) | API06xxx | सेटअप और कॉन्फ़िगरेशन की समस्याएँ |

---

## कनेक्शन त्रुटियाँ

POS और आपके सर्वर के बीच नेटवर्क और कनेक्टिविटी की समस्याएँ।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API01001](./API01001) | कनेक्शन टाइमआउट | सर्वर ने प्रतिक्रिया देने में अधिक समय लिया |
| [API01002](./API01002) | कनेक्शन अस्वीकृत | सर्वर ने कनेक्शन अस्वीकार कर दिया |
| [API01003](./API01003) | कनेक्शन रीसेट | कनेक्शन अप्रत्याशित रूप से बंद कर दिया गया |
| [API01004](./API01004) | DNS समाधान विफल | सर्वर पते को हल नहीं किया जा सका |
| [API01005](./API01005) | SSL प्रमाणपत्र त्रुटि | साइट के सुरक्षा प्रमाणपत्र में समस्या |
| [API01006](./API01006) | नेटवर्क अप्राप्य | नेटवर्क तक पहुँचना संभव नहीं है |
| [API01007](./API01007) | उपकरण ऑफ़लाइन | आपका उपकरण इंटरनेट से कनेक्ट नहीं है |
| [API01008](./API01008) | वेबसाइट अनुपलब्ध | वेबसाइट प्रतिक्रिया नहीं दे रही है |

## प्रमाणीकरण त्रुटियाँ

लॉगिन, सत्र और अनुमतियों से संबंधित समस्याएँ।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API02001](./API02001) | अवैध प्रमाण पत्र | उपयोगकर्ता नाम या पासवर्ड गलत है |
| [API02002](./API02002) | टोकन समाप्त | आपका सत्र समाप्त हो गया है |
| [API02003](./API02003) | टोकन अवैध | प्रमाणीकरण टोकन वैध नहीं है |
| [API02004](./API02004) | उपयोगकर्ता अधिकृत नहीं | आपको यह क्रिया करने की अनुमति नहीं है |
| [API02005](./API02005) | अपर्याप्त अनुमतियाँ | आपके उपयोगकर्ता भूमिका में आवश्यक अनुमतियों की कमी है |
| [API02006](./API02006) | API कुंजी अवैध | WooCommerce API कुंजी वैध नहीं है |
| [API02007](./API02007) | टोकन रीफ्रेश विफल | आपका सत्र रीफ्रेश नहीं किया जा सका |
| [API02008](./API02008) | रीफ्रेश टोकन अवैध | रीफ्रेश टोकन वैध नहीं है |
| [API02009](./API02009) | रीफ्रेश टोकन समाप्त | रीफ्रेश टोकन समाप्त हो गया है |
| [API02010](./API02010) | प्रमाणीकरण आवश्यक | इस क्रिया के लिए प्रमाणीकरण आवश्यक है |

## अनुरोध त्रुटियाँ

सर्वर को भेजे गए अनुरोधों में समस्याएँ।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API03001](./API03001) | अवैध अनुरोध प्रारूप | अनुरोध प्रारूप सही नहीं है |
| [API03002](./API03002) | आवश्यक पैरामीटर गायब | अनुरोध में आवश्यक डेटा गायब है |
| [API03003](./API03003) | अवैध पैरामीटर मान | किसी पैरामीटर का मान अवैध है |
| [API03004](./API03004) | अनुरोध बहुत बड़ा | अनुरोध आकार सीमा से अधिक है |
| [API03005](./API03005) | दर सीमा पार कर गई | एक छोटे समय में बहुत अधिक अनुरोध |
| [API03006](./API03006) | असमर्थित विधि | HTTP विधि समर्थित नहीं है |
| [API03007](./API03007) | अनुरोध कतार पूर्ण | बहुत अधिक लंबित अनुरोध हैं |

## प्रतिक्रिया त्रुटियाँ

सर्वर से प्राप्त प्रतिक्रियाओं में समस्याएँ।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API04001](./API04001) | अवैध प्रतिक्रिया प्रारूप | सर्वर प्रतिक्रिया प्रारूप मान्य नहीं है |
| [API04002](./API04002) | अप्रत्याशित प्रतिक्रिया कोड | अप्रत्याशित HTTP स्थिति कोड प्राप्त हुआ |
| [API04003](./API04003) | दूषित JSON प्रतिक्रिया | JSON प्रतिक्रिया भ्रष्ट या अवैध है |
| [API04004](./API04004) | प्रतिक्रिया डेटा गायब | अपेक्षित डेटा प्रतिक्रिया में गायब है |
| [API04005](./API04005) | JSON सुधार का प्रयास किया गया | दूषित JSON से उबरने का प्रयास किया गया |
| [API04006](./API04006) | संसाधन नहीं मिला | अनुरोधित संसाधन मौजूद नहीं है (404) |

| [API05004](./API05004) | वर्डप्रेस API अक्षम | वर्डप्रेस REST API अक्षम है |
