# API त्रुटियाँ

API त्रुटियाँ तब होती हैं जब आपके WooCommerce सर्वर से संचार करते समय। इन त्रुटियों के प्रीफ़िक्स `API` होते हैं और निम्न श्रेणियों में व्यवस्थित किए जाते हैं:

## श्रेणियाँ

| श्रेणी | कोड रेंज | विवरण |
|----------|------------|-------------|
| [कनेक्शन](#connection-errors) | API01xxx | नेटवर्क और कनेक्टिविटी समस्याएं |
| [प्रमाणीकरण](#authentication-errors) | API02xxx | लॉगिन, टोकन और अनुमति समस्याएं |
| [अनुरोध](#request-errors) | API03xxx | जाने वाले अनुरोधों में समस्याएं |
| [प्रतिक्रिया](#response-errors) | API04xxx | सर्वर की प्रतिक्रिया में समस्याएं |
| [प्लगइन/WordPress](#pluginwordpress-errors) | API05xxx | WordPress या WooCommerce प्लगइन में समस्याएं |
| [कॉन्फ़िगरेशन](#configuration-errors) | API06xxx | सेटअप और कॉन्फ़िगरेशन समस्याएं |

---

## कनेक्शन त्रुटियाँ

POS और आपके सर्वर के बीच नेटवर्क और कनेक्टिविटी समस्याएं।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API01001](./API01001) | कनेक्शन टाइमआउट | सर्वर ने बहुत अधिक समय तक प्रतिक्रिया नहीं दी |
| [API01002](./API01002) | कनेक्शन अस्वीकृत | सर्वर ने कनेक्शन अस्वीकार कर दिया |
| [API01003](./API01003) | कनेक्शन रीसेट | कनेक्शन अचानक बंद हो गया |
| [API01004](./API01004) | DNS रेज़ोल्यूशन विफल | सर्वर पते को हल नहीं कर सका |
| [API01005](./API01005) | SSL प्रमाणपत्र त्रुटि | साइट के सुरक्षा प्रमाणपत्र में समस्या |
| [API01006](./API01006) | नेटवर्क तक पहुंच नहीं | नेटवर्क तक पहुंच नहीं हो सकी |
| [API01007](./API01007) | डिवाइस ऑफ़लाइन | आपका डिवाइस इंटरनेट से कनेक्ट नहीं है |
| [API01008](./API01008) | वेबसाइट अनुपलब्ध | वेबसाइट प्रतिक्रिया नहीं दे रही |

## प्रमाणीकरण त्रुटियाँ

लॉगिन, सत्र और अनुमतियों की समस्याएं।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API02001](./API02001) | अमान्य क्रेडेंशियल | उपयोगकर्ता नाम या पासवर्ड गलत है |
| [API02002](./API02002) | टोकन समाप्त | आपका सत्र समाप्त हो गया है |
| [API02003](./API02003) | टोकन अमान्य | प्रमाणीकरण टोकन मान्य नहीं है |
| [API02004](./API02004) | उपयोगकर्ता अधिकृत नहीं | आपके पास यह क्रिया करने की अनुमति नहीं है |
| [API02005](./API02005) | अपर्याप्त अनुमतियाँ | आपकी उपयोगकर्ता भूमिका में आवश्यक अनुमतियाँ नहीं हैं |
| [API02006](./API02006) | API कुंजी अमान्य | WooCommerce API कुंजी मान्य नहीं है |
| [API02007](./API02007) | टोकन रीफ्रेश विफल | आपके सत्र को रीफ्रेश नहीं किया जा सका |
| [API02008](./API02008) | रीफ्रेश टोकन अमान्य | रीफ्रेश टोकन मान्य नहीं है |
| [API02009](./API02009) | रीफ्रेश टोकन समाप्त | रीफ्रेश टोकन समाप्त हो गया है |
| [API02010](./API02010) | प्रमाणीकरण आवश्यक | इस क्रिया के लिए प्रमाणीकरण आवश्यक है |

## अनुरोध त्रुटियाँ

सर्वर को भेजे गए अनुरोधों में समस्याएं।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API03001](./API03001) | अमान्य अनुरोध प्रारूप | अनुरोध प्रारूप सही नहीं है |
| [API03002](./API03002) | आवश्यक पैरामीटर अनुपस्थित | अनुरोध में आवश्यक डेटा गायब है |
| [API03003](./API03003) | अमान्य पैरामीटर मान | एक पैरामीटर का मान अमान्य है |
| [API03004](./API03004) | अनुरोध बहुत बड़ा | अनुरोध आकार सीमाओं से अधिक है |
| [API03005](./API03005) | दर सीमा पार | बहुत कम समय में बहुत अधिक अनुरोध |
| [API03006](./API03006) | असमर्थित विधि | HTTP विधि समर्थित नहीं है |
| [API03007](./API03007) | अनुरोध कतार भरी | बहुत से लंबित अनुरोध |

## प्रतिक्रिया त्रुटियाँ

सर्वर से प्राप्त प्रतिक्रियाओं में समस्याएं।

| कोड | नाम | विवरण |
|------|------|-------------|
| [API04001](./API04001) | अमान्य प्रतिक्रिया प्रारूप | सर्वर प्रतिक्रिया प्रारूप मान्य नहीं है |
| [API04002](./API04002) | अप्रत्याशित प्रतिक्रिया कोड | अप्रत्याशित HTTP स्थिति कोड प्राप्त हुआ |
| [API04003](./API04003) | विकृत JSON प्रतिक्रिया | JSON प्रतिक्रिया भ्रष्ट या अमान्य है |
| [API04004](./API04004) | प्रतिक्रिया डेटा अनुपस्थित | अपेक्षित डेटा प्रतिक्रिया में गायब है |
| [API04005](./API04005) | JSON पुनर्प्राप्ति का प्रयास | विकृत JSON से पुनर्प्राप्ति का प्रयास किया गया |
| [API04006](./API04006) | संसाधन नहीं मिला | अनुरोधित संसाधन मौजूद नहीं है (404) |
