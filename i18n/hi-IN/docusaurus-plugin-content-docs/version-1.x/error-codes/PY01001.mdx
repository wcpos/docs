# PY01001: भुगतान अस्वीकृत

## इसका क्या मतलब है

भुगतान को भुगतान प्रोसेसर या कार्ड जारीकर्ता द्वारा अस्वीकृत कर दिया गया। लेनदेन पूरा नहीं हो सका।

## सामान्य कारण

- **अपर्याप्त धन** — खाते में पर्याप्त पैसा नहीं है
- **कार्ड सीमा पार हो गई** — लेनदेन कार्ड सीमाओं से अधिक
- **धोखाधड़ी सुरक्षा** — बैंक द्वारा संदिग्ध के रूप में चिह्नित
- **गलत कार्ड विवरण** — कार्ड नंबर, CVV, या समाप्ति गलत
- **कार्ड प्रतिबंध** — इस प्रकार की खरीद के लिए कार्ड सक्षम नहीं है

## इसे कैसे ठीक करें

### 1. कार्ड विवरण की जांच करें

यह सुनिश्चित करें कि दर्ज की गई जानकारी सही है:
- कार्ड नंबर (सभी अंक)
- समाप्ति तिथि
- CVV/सुरक्षा कोड
- बिलिंग पता (अगर आवश्यक हो)

### 2. फिर से प्रयास करें

कभी-कभी अस्थायी समस्याएँ होती हैं:
- एक क्षण प्रतीक्षा करें
- कार्ड विवरण फिर से दर्ज करें
- लेन-देन फिर से करने का प्रयास करें

### 3. एक अलग भुगतान विधि का उपयोग करें

यदि कार्ड लगातार अस्वीकृत होता है:
- एक अलग कार्ड आजमाएँ
- नकद भुगतान का उपयोग करें
- एक वैकल्पिक भुगतान विधि का उपयोग करें

### 4. बैंक से संपर्क करें

कार्डधारक को आवश्यकता हो सकती है:
- अपने बैंक के साथ लेनदेन की पुष्टि करें
- किसी भी धोखाधड़ी ब्लॉकों को हटाएं
- पुष्टि करें कि कार्ड सक्रिय है
- व्यय सीमाओं की जांच करें

### 5. प्रतिबंधों की जांच करें

कुछ कार्डों पर प्रतिबंध होते हैं:
- अंतरराष्ट्रीय लेनदेन बंद
- ऑनलाइन/POS लेनदेन प्रतिबंधित
- कुछ व्यापारी श्रेणियों पर प्रतिबंध

## स्टोर मालिकों के लिए

यदि आप बार-बार अस्वीकृति देख रहे हैं:
- सुनिश्चित करें कि आपका व्यापारी खाता सही स्थिति में है
- भुगतान गेटवे कॉन्फ़िगरेशन की जांच करें
- धोखाधड़ी रोकथाम सेटिंग की समीक्षा करें
- अपने भुगतान प्रोसेसर से संपर्क करने पर विचार करें
