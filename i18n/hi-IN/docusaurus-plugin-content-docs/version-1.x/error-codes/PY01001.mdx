# PY01001: भुगतान नकारा गया

## इसका क्या मतलब है

भुगतान को भुगतान प्रोसेसर या कार्ड जारीकर्ता द्वारा नकारा गया। लेनदेन पूरा नहीं हो सका।

## सामान्य कारण

- **अपर्याप्त धनराशि** — खाते में पर्याप्त पैसा नहीं है
- **कार्ड सीमा से अधिक** — लेनदेन कार्ड की सीमाओं से अधिक है
- **धोखाधड़ी सुरक्षा** — बैंक द्वारा संदिग्ध के रूप में फ्लैग किया गया
- **गलत कार्ड विवरण** — कार्ड नंबर, CVV, या समाप्ति तिथि गलत
- **कार्ड प्रतिबंध** — इस प्रकार की खरीदारी के लिए कार्ड सक्षम नहीं है

## इसे कैसे ठीक करें

### 1. कार्ड विवरण सत्यापित करें

यह सुनिश्चित करें कि दर्ज की गई जानकारी सही है:
- कार्ड नंबर (सभी अंक)
- समाप्ति तिथि
- CVV/सुरक्षा कोड
- बिलिंग पता (अगर आवश्यक हो)

### 2. फिर से प्रयास करें

कभी-कभी अस्थायी समस्याएँ हो सकती हैं:
- एक क्षण प्रतीक्षा करें
- कार्ड विवरण फिर से दर्ज करें
- लेन-देन फिर से करने का प्रयास करें

### 3. एक अलग भुगतान विधि का उपयोग करें

यदि कार्ड लगातार नकारा जाता है:
- एक अलग कार्ड आजमाएँ
- नकद भुगतान का उपयोग करें
- एक वैकल्पिक भुगतान विधि का उपयोग करें

### 4. बैंक से संपर्क करें

कार्डधारक को आवश्यकता हो सकती है:
- अपने बैंक के साथ लेनदेन की पुष्टि करें
- किसी भी धोखाधड़ी ब्लॉकों को हटावें
- पुष्टि करें कि कार्ड सक्रिय है
- व्यय सीमाओं की जांच करें

### 5. प्रतिबंधों की जांच करें

कुछ कार्डों पर प्रतिबंध हो सकते हैं:
- अंतरराष्ट्रीय लेनदेन निष्क्रिय
- ऑनलाइन/POS लेनदेन प्रतिबंधित
- कुछ व्यापारी श्रेणियों पर प्रतिबंधित

## स्टोर मालिकों के लिए

यदि आप बार-बार नकारें देख रहे हैं:
- सुनिश्चित करें कि आपका व्यापारी खाता सही स्थिति में है
- भुगतान गेटवे कॉन्फ़िगरेशन की जांच करें
- धोखाधड़ी रोकथाम सेटिंग की समीक्षा करें
- अपने भुगतान प्रोसेसर से संपर्क करने पर विचार करें
