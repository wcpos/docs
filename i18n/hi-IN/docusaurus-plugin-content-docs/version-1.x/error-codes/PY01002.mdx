# PY01002: अपर्याप्त निधि

## इसका क्या अर्थ है

भुगतान को अस्वीकार कर दिया गया क्योंकि खाते में लेन-देन को कवर करने के लिए पर्याप्त धन नहीं है।

## सामान्य कारण

- **कम खाता शेष** — उपलब्ध धन पर्याप्त नहीं
- **लंबित लेनदेन** — अन्य लंबित लेनदेन उपलब्ध शेष को कम कर रहे हैं
- **क्रेडिट सीमा तक पहुंच गई** — क्रेडिट कार्ड के लिए, सीमा पूरी तरह से भर गई है
- **धन पर रोक** — अन्य लेनदेन के लिए धन रोके गए हैं

## कैसे ठीक करें

### 1. अन्य भुगतान विधि का उपयोग करें

सबसे तेज समाधान:
- एक अलग कार्ड का प्रयास करें
- नकद का उपयोग करें
- बहु भुगतान विधियों में विभाजित करें (यदि समर्थित है)

### 2. ऑर्डर राशि कम करें

यदि आंशिक भुगतान संभव है:
- कुछ वस्तुएं हटाएं
- छूट या कूपन लागू करें
- एक छोटा लेनदेन पूरा करें

### 3. ग्राहक क्रियाएं

ग्राहक को निम्न करने की आवश्यकता हो सकती है:
- खाते में धन स्थानांतरित करें
- लंबित लेनदेन के साफ होने की प्रतीक्षा करें
- क्रेडिट कार्ड शेष का भुगतान करें
- क्रेडिट सीमा में वृद्धि का अनुरोध करें

### 4. लेनदेन विभाजित करें

यदि आपका POS विभाजित भुगतान समर्थित करता है:
- एक कार्ड से आंशिक भुगतान करें
- शेष राशि अन्य विधि से चुकाएं

## गोपनीयता नोट

ग्राहकों के साथ संवाद करते समय:
- विशिष्ट अस्वीकृति कारण के बारे में सावधान रहें
- बस इतना कहें "कार्ड को अस्वीकार कर दिया गया"
- उन्हें निजी रूप से अन्य भुगतान विधि का प्रयास करने दें
