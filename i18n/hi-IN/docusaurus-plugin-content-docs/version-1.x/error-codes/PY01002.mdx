# PY01002: अपर्याप्त Funds

## इसका क्या मतलब है

भुगतान अस्वीकृत कर दिया गया क्योंकि खाते में लेनदेन को कवर करने के लिए पर्याप्त पैसा नहीं है।

## सामान्य कारण

- **कम खाता शेष** — उपलब्ध फंड की कमी
- **लंबित लेनदेन** — अन्य लंबित लेनदेन जो उपलब्ध शेष को कम कर रहे हैं
- **क्रेडिट सीमा पूरी हो गई** — क्रेडिट कार्ड के लिए, सीमा अधिकतम हो गई है
- **फंड पर रोक** — अन्य लेनदेन के लिए फंड रोक दिए गए हैं

## इसे कैसे ठीक करें

### 1. एक अलग भुगतान विधि का उपयोग करें

सबसे तेज़ समाधान:
- एक अलग कार्ड का प्रयास करें
- नकद का उपयोग करें
- कई विधियों के बीच भुगतान विभाजित करें (अगर समर्थित हो)

### 2. ऑर्डर राशि को घटाएं

अगर आंशिक भुगतान संभव है:
- कुछ वस्तुओं को हटा दें
- छूट या कूपन लागू करें
- एक छोटी लेनदेन पूरी करें

### 3. ग्राहक क्रियाएँ

ग्राहक को यह करने की आवश्यकता हो सकती है:
- खाते में फंड ट्रांसफर करें
- लंबित लेनदेन के साफ होने की प्रतीक्षा करें
- क्रेडिट कार्ड शेष चुकाएं
- क्रेडिट सीमा में वृद्धि का आग्रह करें

### 4. लेनदेन को विभाजित करें

यदि आपका POS विभाजित भुगतान का समर्थन करता है:
- एक कार्ड से हिस्सा चुकाएं
- बाकी राशि को अन्य विधि से चुकाएं

## गोपनीयता नोट

ग्राहकों को यह संवाद करते समय:
- विशेष अस्वीकृति कारण के बारे में विवेकपूर्ण रहें
- सरलता से कहें "कार्ड अस्वीकृत हो गया"
- उन्हें निजी तौर पर अन्य भुगतान विधि का प्रयास करने की अनुमति दें
