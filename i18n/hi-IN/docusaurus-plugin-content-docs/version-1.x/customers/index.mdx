# ग्राहक

## प्रो सुविधा
ग्राहक स्क्रीन के लिए [WCPOS Pro](/getting-started/pro-license) की आवश्यकता होती है। मुफ्त उपयोगकर्ता ऑर्डर के लिए मौजूदा ग्राहकों का चयन कर सकते हैं लेकिन पूर्ण ग्राहक सूची नहीं देख सकते या ग्राहक विवरण नहीं संपादित कर सकते।

POS के भीतर व्यापक ग्राहक प्रबंधन प्रदान करता है। WooCommerce व्यवस्थापक पर स्विच किए बिना ग्राहक देखें, संपादित करें और बनाएं।

## इंटरफ़ेस अवलोकन

### हेडर क्रियाएं

स्क्रीन के शीर्ष पर:

- **खोज पट्टी** - नाम, ईमेल आदि से ग्राहक खोजें
- **ग्राहक जोड़ें** (<Icon name="plus" />) - नया ग्राहक बनाएं
- **प्रदर्शन सेटिंग्स** (<Icon name="sliders" />) - दृश्यमान कॉलम कॉन्फ़िगर करें

### ग्राहक तालिका

मुख्य क्षेत्र में ग्राहक प्रदर्शित होते हैं:

- **अवतार** - ग्राहक प्रोफ़ाइल चित्र (या प्लेसहोल्डर)
- **पहला नाम** - ग्राहक का पहला नाम
- **अंतिम नाम** - ग्राहक का अंतिम नाम (क्रमबद्ध करने योग्य)
- **ईमेल** - संपर्क ईमेल पता
- **बिलिंग पता** - पूर्ण बिलिंग पता
- **निर्माण दिनांक** - ग्राहक कब जोड़ा गया
- **क्रियाएं** - तीन-बिंदु मेनू

### फ़ुटर

- सिंक बटन के साथ ग्राहक गणना

## मुख्य विशेषताएं

### ग्राहक खोज

तेजी से ग्राहक खोजें:

- पहला या अंतिम नाम
- ईमेल पता
- पता जानकारी

### नया ग्राहक जोड़ें

POS से सीधे ग्राहक बनाएं:

1. हेडर में <Icon name="plus" /> आइकन पर क्लिक करें
2. ग्राहक विवरण भरें
3. नए ग्राहक को सहेजें

ग्राहक WooCommerce में बनाया जाता है और तुरंत ऑर्डर के लिए उपलब्ध होता है।

### ग्राहक संपादित करें

ग्राहक जानकारी अपडेट करें:

1. ग्राहक पर तीन-बिंदु मेनू पर क्लिक करें
2. **संपादित करें** चुनें
3. विवरण संशोधित करें (नाम, ईमेल, पते)
4. परिवर्तन सहेजें

परिवर्तन स्वचालित रूप से WooCommerce के साथ सिंक होते हैं।

## प्रदर्शन सेटिंग्स

दृश्यमान कॉलम अनुकूलित करने के लिए स्लाइडर्स आइकन (<Icon name="sliders" />) पर क्लिक करें।

### उपलब्ध कॉलम

| कॉलम | विवरण |
|--------|-------------|
| **छवि** | ग्राहक अवतार |
| **आईडी** | WooCommerce ग्राहक आईडी |
| **पहला नाम** | ग्राहक पहला नाम |
| **अंतिम नाम** | ग्राहक अंतिम नाम |
| **ईमेल** | ईमेल पता |
| **भूमिका** | उपयोगकर्ता भूमिका (ग्राहक, सब्सक्राइबर, आदि) |
| **उपयोगकर्ता नाम** | WordPress उपयोगकर्ता नाम |
| **बिलिंग पता** | बिलिंग जानकारी |
| **शिपिंग पता** | शिपिंग जानकारी |
| **निर्माण दिनांक** | ग्राहक कब जोड़ा गया |
| **संशोधन दिनांक** | अंतिम अपडेट |
| **क्रियाएं** | संपादित करें, सिंक, हटाएं |

## ग्राहक क्रियाएं

तीन-बिंदु मेनू (⋮) पर क्लिक करें:

- **संपादित करें** - ग्राहक विवरण संशोधित करें
- **सिंक करें** - सर्वर से ग्राहक को ताज़ा करें
- **हटाएं** - स्थानीय डेटाबेस से हटाएं

ध्यान दें:
POS से ग्राहक हटाने से केवल स्थानीय रूप से हटाया जाता है। ग्राहक WooCommerce में बना रहता है और अगली सिंक पर फिर से दिखाई देगा।

## ऑर्डर में ग्राहक उपयोग

ऑर्डर में ग्राहक असाइन करने के लिए:

1. [कार्ट पैनल](/pos/cart) में ग्राहक बैज पर क्लिक करें
2. नाम या ईमेल से ग्राहक खोजें
3. ग्राहक का चयन करें
4. ग्राहक अब ऑर्डर से जुड़ा हुआ है

ग्राहक जानकारी (बिलिंग/शिपिंग पते) स्वचालित रूप से ऑर्डर के लिए उपयोग की जाती है।

## सिंक्रोनाइज़ेशन

ग्राहक डेटा POS और WooCommerce के बीच सिंक होता है:

- **रीयल-टाइम अपडेट** - सभी उपकरणों में परिवर्तन सिंक होते हैं
- **WooCommerce एकीकरण** - ग्राहक WooCommerce में संग्रहित होते हैं
- **ऑफ़लाइन क्षमता** - ऑफ़लाइन होने पर भी ग्राहक देख सकते हैं

## संबंधित दस्तावेज़ीकरण

- [कार्ट पैनल](/pos/cart) - ऑर्डर के लिए ग्राहक चयन
- [ऑर्डर](/orders) - ग्राहक ऑर्डर इतिहास देखें
