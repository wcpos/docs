POS रसीद प्रिंट टेम्पलेट्स मुफ्त और प्रो दोनों प्लगइन्स में `includes/views/print/tmpl-receipt.php` फ़ाइल में स्थित हैं। रसीद टेम्पलेट्स को अपने थीम डायरेक्टरी में `woocommerce-pos/print/tmpl-receipt.php` फ़ाइल बनाकर अनुकूलित किया जा सकता है। नीचे दोनों टेम्पलेट्स का कोड दिया गया है।

### बेसिक रसीद टेम्पलेट

### प्रो रसीद टेम्पलेट

प्रो रसीद टेम्पलेट [स्टोर्स](./stores) व्यवस्थापक से लोगो, स्टोर का पता, खुलने के घंटे और विशेष संदेश जैसी अतिरिक्त जानकारी की अनुमति देता है।

<Gist id="c9485366a73ceda12041" />

### रसीद की तिथि को अनुकूलित करना

WCPOS तिथि स्ट्रिंग को स्थानीयकृत करने के लिए [moment.js](http://momentjs.com/) का उपयोग करता है। डिफ़ॉल्ट तिथि प्रारूप `"MMMM Do YYYY, h:mm:ss a"` है, उदा.: 31 मई 2015, 7:20:44 शाम। तिथि प्रारूप को अनुकूलित करने के लिए कृपया [moment.js दस्तावेज़ीकरण](http://momentjs.com/docs/#/parsing/string-format/) देखें।

### ऑर्डर गुण

WCPOS WC REST API के json आउटपुट का उपयोग रसीद टेम्पलेट को भरने के लिए करता है। [WC REST API दस्तावेज़](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) मानक json आउटपुट का एक उदाहरण दिखाता है। WCPOS ने `woocommerce_api_order_response` फ़िल्टर का उपयोग करके कुछ अतिरिक्त गुण जोड़े हैं।

| गुण                               | विवरण                                                        |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | कार्ट छूट का कर भाग                                          |
| cashier.id                        | कैशियर का उपयोगकर्ता ID                                      |
| cashier.first_name                | कैशियर का पहला नाम                                           |
| cashier.last_name                 | कैशियर का अंतिम नाम                                          |
| payment_details.result            | भुगतान गेटवे की सफलता या असफलता                              |
| payment_details.message           | भुगतान गेटवे से कोई संदेश                                    |
| payment_details.redirect          | ऑफसाइट भुगतान के लिए रीडायरेक्ट URL, जैसे: PayPal Standard   |
| payment_details.method\_`gateway` | किसी भी गेटवे विशिष्ट संदेश, जैसे: भुगतान राशि और बाकी         |
| shipping_lines[i].total_tax       | प्रत्येक शिपिंग लाइन के लिए कर राशि जोड़ें                    |
| subtotal_tax                      | सबटोटल का कर भाग                                            |
