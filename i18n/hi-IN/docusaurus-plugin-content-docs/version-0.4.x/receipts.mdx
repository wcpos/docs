POS रसीद प्रिंट टेम्पलेट `includes/views/print/tmpl-receipt.php` फ़ाइल में स्थित हैं, जो निःशुल्क और प्रो प्लगइन दोनों में हैं। रसीद टेम्पलेट को आपके विषय निर्देशिका में `woocommerce-pos/print/tmpl-receipt.php` फ़ाइल बनाकर अनुकूलित किया जा सकता है। दोनों टेम्पलेट के लिए कोड नीचे शामिल है।

### बेसिक रसीद टेम्पलेट

### प्रो रसीद टेम्पलेट

प्रो रसीद टेम्पलेट अतिरिक्त जानकारी की अनुमति देता है, जैसे कि [स्टोर्स](./stores) प्रशासन से लोगो, स्टोर का पता, खुलने का समय और विशेष संदेश।

<Gist id="c9485366a73ceda12041" />

### रसीद तिथि को अनुकूलित करना

WCPOS तिथि स्ट्रिंग्स को स्थानीयकरण के लिए [moment.js](http://momentjs.com/) का उपयोग करता है। डिफ़ॉल्ट तिथि प्रारूप `"MMMM Do YYYY, h:mm:ss a"` है, जैसे: 31 मई 2015, 7:20:44 अपराह्न। तिथि प्रारूप को अनुकूलित करने के लिए कृपया [moment.js दस्तावेज़](http://momentjs.com/docs/#/parsing/string-format/) से परामर्श करें।

### ऑर्डर गुण

WCPOS ऑर्डर रसीद टेम्पलेट को भरने के लिए WC REST API से json आउटपुट का उपयोग करता है। [WC REST API दस्तावेज़ एक उदाहरण दिखाते हैं](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) मानक json आउटपुट का। `woocommerce_api_order_response` फ़िल्टर का उपयोग करके WCPOS द्वारा कुछ अतिरिक्त गुण जोड़े गए हैं।

| गुण                               | विवरण                                                        |
| ---------------------------------- | ------------------------------------------------------------ |
| cart_discount_tax                  | कार्ट छूट का कर हिस्सा                                       |
| cashier.id                         | कैशियर का उपयोगकर्ता आईडी                                   |
| cashier.first_name                 | कैशियर का पहला नाम                                         |
| cashier.last_name                  | कैशियर का अंतिम नाम                                        |
| payment_details.result             | भुगतान गेटवे की सफलता या विफलता                           |
| payment_details.message            | भुगतान गेटवे से कोई संदेश                                   |
| payment_details.redirect           | ऑफसाइट भुगतान के लिए पुनर्निर्देशन URL, जैसे: PayPal Standard |
| payment_details.method\_`gateway`  | कोई गेटवे विशिष्ट संदेश, जैसे: राशि दी गई और परिवर्तन     |
| shipping_lines[i].total_tax        | प्रत्येक शिपिंग लाइन के लिए कर राशि जोड़ें                  |
| subtotal_tax                       | उप-योग का कर हिस्सा                                        |
