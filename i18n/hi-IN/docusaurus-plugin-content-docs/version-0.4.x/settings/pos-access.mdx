पीओएस एक्सेस सेटिंग्स का उपयोग करके, स्टोर के मालिक पॉइंट ऑफ सेल सिस्टम के लिए उपयोगकर्ता अनुमतियाँ दे या हटा सकते हैं।  
डिफ़ॉल्ट रूप से, **व्यवस्थापक** और **दुकान प्रबंधक** की भूमिकाएँ पीओएस का उपयोग करने के लिए सभी क्षमताओं के साथ सक्षम हैं।  
यह अनुशंसा की जाती है कि आप डिफ़ॉल्ट सेटिंग्स को बनाए रखें जब तक कि आप इसके परिणामों के बारे में न जानते हों।  
उदाहरण के लिए, `publish_shop_orders` को `सदस्यों` की भूमिका में देना _बहुत_ बुरा विचार होगा।  

![पीओएस एक्सेस सेटिंग्स का उदाहरण](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "पीओएस एक्सेस सेटिंग्स का उदाहरण")

| क्षमता                                                                | एक्सेस                                      |
| --------------------------------------------------------------------- | ------------------------------------------- |
| access_woocommerce_pos                                                | पीओएस फ्रंटेंड को एक्सेस करने के लिए आवश्यक  |
| manage_woocommerce_pos                                                | पीओएस प्रशासन सेटिंग्स को एक्सेस करने के लिए आवश्यक |
| read_private_products                                                 | उत्पादों को देखने के लिए आवश्यक             |
| edit_product <br /> edit_published_products <br /> edit_others_products | उत्पादों और विविधताओं को संपादित करने के लिए आवश्यक |
| read_private_shop_orders                                              | ऑर्डर और रसीदें देखने के लिए आवश्यक        |
| publish_shop_orders                                                   | नए ऑर्डर बनाने के लिए आवश्यक               |
| list_users                                                            | ग्राहकों को देखने के लिए आवश्यक             |
| create_users                                                          | नए ग्राहकों को बनाने के लिए आवश्यक          |
| edit_users                                                            | मौजूदा ग्राहकों को अपडेट करने के लिए आवश्यक |
| read_private_shop_coupons                                             | कूपन देखने के लिए आवश्यक                     |

**संसाधन:**

- [WordPress भूमिका और क्षमताएँ](https://codex.wordpress.org/Roles_and_Capabilities) पर अधिक जानकारी
- [WooCommerce भूमिका और क्षमताएँ](http://docs.woothemes.com/document/roles-capabilities/) पर अधिक जानकारी
- [यूज़र रोल एडिटर](https://wordpress.org/plugins/user-role-editor/) - भूमिकाओं और क्षमताओं को प्रबंधित करने के लिए एक शानदार प्लगइन
- [मेंबर](https://wordpress.org/plugins/members/) - भूमिकाओं और क्षमताओं को प्रबंधित करने के लिए एक और प्लगइन
