पीओएस पहुंच सेटिंग्स का उपयोग करके, स्टोर के मालिक पॉइंट ऑफ़ सेल सिस्टम के लिए उपयोगकर्ता विशेषाधिकार दे या हटा सकते हैं।
डिफ़ॉल्ट रूप से, **व्यवस्थापक** और **शॉप प्रबंधक** भूमिकाएँ पीओएस का उपयोग करने के लिए सभी क्षमताओं के साथ सक्षम हैं।
यदि आप परिणामों से अवगत नहीं हैं तो अनुशंसा की जाती है कि आप डिफ़ॉल्ट सेटिंग्स को रखें।
उदाहरण के लिए, `सदस्य` भूमिका को `publish_shop_orders` देना एक _बहुत ही_ खराब विचार होगा।

![पीओएस पहुंच सेटिंग्स का उदाहरण](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "पीओएस पहुंच सेटिंग्स का उदाहरण")

| क्षमता                                                                 | पहुंच                                    |
| ----------------------------------------------------------------------- | ----------------------------------------- |
| access_woocommerce_pos                                                  | पीओएस फ्रंटएंड तक पहुंच के लिए आवश्यक    |
| manage_woocommerce_pos                                                  | पीओएस व्यवस्थापक सेटिंग्स तक पहुंच के लिए आवश्यक |
| read_private_products                                                   | उत्पादों को देखने के लिए आवश्यक           |
| edit_product <br /> edit_published_products <br /> edit_others_products | उत्पादों और प्रकारों को संपादित करने के लिए आवश्यक |
| read_private_shop_orders                                                | आदेशों और रसीदों को देखने के लिए आवश्यक   |
| publish_shop_orders                                                     | नए आदेश बनाने के लिए आवश्यक              |
| list_users                                                              | ग्राहकों को देखने के लिए आवश्यक            |
| create_users                                                            | नए ग्राहक बनाने के लिए आवश्यक             |
| edit_users                                                              | मौजूदा ग्राहकों को अपडेट करने के लिए आवश्यक |
| read_private_shop_coupons                                               | कूपन देखने के लिए आवश्यक                  |

**संसाधन:**

- [वूकॉमर्स भूमिकाएँ और क्षमताएँ](http://docs.woothemes.com/document/roles-capabilities/) के बारे में अधिक जानकारी
- [उपयोगकर्ता भूमिका संपादक](https://wordpress.org/plugins/user-role-editor/) - भूमिकाओं और क्षमताओं को प्रबंधित करने के लिए एक बढ़िया प्लगिन
- [सदस्य](https://wordpress.org/plugins/members/) - भूमिकाओं और क्षमताओं को प्रबंधित करने के लिए एक अन्य प्लगिन
