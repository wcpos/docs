---

आप WooCommerce प्रशासन में ऊपरी बाएँ कोने में त्वरित लिंक का उपयोग करके अपने पीओएस और ऑनलाइन ऑर्डर को जल्दी से अलग कर सकते हैं।
**कृपया ध्यान दें:** यह केवल संस्करण 0.3.1 के बाद के पीओएस ऑर्डरों को फ़िल्टर करेगा, 0.3.1 से पहले के ऑर्डर ऑनलाइन पर डिफ़ॉल्ट होंगे।

![Filter Orders](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

इसके अलावा, यदि आप अपने प्रशासन क्षेत्र से सभी पूर्ण ऑर्डरों को फ़िल्टर करना चाहते हैं ताकि आप लंबित/प्रसंस्करण ऑर्डरों पर ध्यान केंद्रित कर सकें, तो आप निम्नलिखित कोड को अपने [functions.php](http://codex.wordpress.org/Functions_File_Explained) फ़ाइल में जोड़ सकते हैं।

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
