description: WooCommerce व्यवस्थापक में क्विक-लिंक्स का उपयोग करके POS और ऑनलाइन ऑर्डर को अलग करें।
---

आप WooCommerce व्यवस्थापक में बाईं ओर क्विक-लिंक्स का उपयोग करके अपने POS और ऑनलाइन ऑर्डर को जल्दी से अलग कर सकते हैं।
**कृपया ध्यान दें:** यह केवल संस्करण 0.3.1 के बाद के POS ऑर्डर को फ़िल्टर करेगा, 0.3.1 से पहले के ऑर्डर ऑनलाइन माने जाएंगे।

![ऑर्डर फ़िल्टर करें](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "ऑर्डर फ़िल्टर करें")

इसके अलावा, यदि आप अपने व्यवस्थापक क्षेत्र में सभी पूरे किए गए ऑर्डर को फ़िल्टर करना चाहते हैं ताकि आप लंबित/प्रोसेसिंग ऑर्डर पर ध्यान केंद्रित कर सकें, तो आप अपनी [functions.php](http://codex.wordpress.org/Functions_File_Explained) फ़ाइल में निम्न कोड जोड़ सकते हैं।

```php title="functions.php"
/**
 * WP व्यवस्थापक में ऑर्डर के डिफ़ॉल्ट फ़िल्टर को बदलें
 * इसे अपनी functions.php फ़ाइल में जोड़ें
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
