title: POS या ऑनलाइन बिक्री के अनुसार ऑर्डर फ़िल्टर करना
description: WooCommerce एडमिन में क्विक-लिंक का उपयोग करके POS और ऑनलाइन ऑर्डर अलग करें।
---

आप WooCommerce एडमिन में ऊपर बाईं ओर दिए गए क्विक-लिंक का उपयोग करके अपने POS और ऑनलाइन ऑर्डर आसानी से अलग कर सकते हैं।
**कृपया ध्यान दें:** यह केवल संस्करण 0.3.1 के बाद के POS ऑर्डर फ़िल्टर करेगा, 0.3.1 से पहले के ऑर्डर डिफ़ॉल्ट रूप से ऑनलाइन के रूप में दिखेंगे।

![ऑर्डर फ़िल्टर करें](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "ऑर्डर फ़िल्टर करें")

इसके अतिरिक्त, यदि आप अपने एडमिन क्षेत्र से सभी पूर्ण किए गए ऑर्डर फ़िल्टर करना चाहते हैं ताकि आप लंबित/प्रोसेसिंग ऑर्डर पर ध्यान केंद्रित कर सकें, तो आप निम्नलिखित कोड अपनी [functions.php](http://codex.wordpress.org/Functions_File_Explained) फ़ाइल में जोड़ सकते हैं।

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
