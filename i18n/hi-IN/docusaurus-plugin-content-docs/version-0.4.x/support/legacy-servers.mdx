HTTP (और HTTPS) ब्राउज़र द्वारा वेब एप्लिकेशन से बातचीत करने के लिए उपयोग किए जाने वाले प्रोटोकॉल हैं।
प्रोटोकॉल का एक महत्वपूर्ण हिस्सा [HTTP मेथड](https://tools.ietf.org/html/rfc2616#section-5.1.1) है।
पारंपरिक रूप से, वेब एप्लिकेशन HTTP मेथड के एक बहुत ही सीमित सेट का उपयोग करते थे जैसे GET (वेब पेज प्राप्त करना) और POST (वेब फॉर्म जमा करना)।

आधुनिक वेब एप्लिकेशन, जैसे WCPOS, निम्नलिखित HTTP मेथड के साथ REST/HTTP दृष्टिकोण का उपयोग करते हैं:

- GET
- POST
- PUT
- PATCH
- DELETE

कुछ दुर्लभ मामलों में, एक वेब होस्ट इन HTTP मेथड में से कुछ को ब्लॉक करने का निर्णय ले सकता है।
यदि आपका वेब होस्ट आधुनिक RESTful HTTP मेथड को ब्लॉक करता है, तो आपको पुरानी सर्वर सहायता को सक्षम करना होगा।
