HTTP (और HTTPS) उन प्रोटोकॉल हैं जो हर ब्राउज़र वेब अनुप्रयोगों से बात करने के लिए उपयोग करता है। प्रोटोकॉल का एक मुख्य भाग [HTTP विधि](https://tools.ietf.org/html/rfc2616#section-5.1.1) है। पारंपरिक रूप से, वेब अनुप्रयोगों ने GET (एक वेब पृष्ठ प्राप्त करें) और POST (एक वेब फॉर्म सबमिट करें) जैसी बहुत सीमित HTTP विधियों का उपयोग किया।

आधुनिक वेब अनुप्रयोग, जैसे WCPOS, REST/HTTP दृष्टिकोण का उपयोग करते हैं जिसमें निम्नलिखित HTTP विधियाँ होती हैं:

- GET
- POST
- PUT
- PATCH
- DELETE

कुछ दुर्लभ मामलों में, एक वेब होस्ट इनमें से कुछ HTTP विधियों को ब्लॉक करने का विकल्प चुन सकता है। यदि आपकी वेब होस्ट आधुनिक RESTful HTTP विधियों को ब्लॉक करता है, तो आपको पुराने सर्वर समर्थन को सक्षम करने की आवश्यकता होगी।
