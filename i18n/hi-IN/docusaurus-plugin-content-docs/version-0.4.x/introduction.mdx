WCPOS [WooCommerce](https://www.woothemes.com/woocommerce/) के लिए पॉइंट ऑफ सेल प्लगइन है।
यह दस्तावेज़ ओपन सोर्स प्लगइन को कवर करता है, साथ ही प्रो प्लगइन जिसे मुफ्त प्लगइन में कार्यक्षमता जोड़ने के लिए डिज़ाइन किया गया है।

- [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) वर्डप्रेस.org से डाउनलोड करें
- [WCPOS Pro](http://wcpos.com/pro) में अपग्रेड करें

:::danger

कृपया सुनिश्चित करें कि दस्तावेज़ का संस्करण संख्या आपके इंस्टॉल किए गए प्लगइनों के संस्करण से मेल खाती है।

:::

### अन्य दस्तावेज़ 

**अक्सर पूछे जाने वाले प्रश्नों** की एक सूची [faq.wcpos.com](http://faq.wcpos.com) पर उपलब्ध है।
**डेवलपर दस्तावेज़** [api.wcpos.com](http://api.wcpos.com) पर उपलब्ध है।

### मदद और समर्थन

यदि आपके पास कोई प्रश्न है जो इस दस्तावेज़ या फ़ैक्यू के द्वारा कवर नहीं किया गया है, तो आप हमसे [support@wcpos.com](mailto:support@wcpos.com) पर संपर्क कर सकते हैं।

### इस दस्तावेज़ में योगदान करें

आप [GitHub](https://github.com/kilbot/WooCommerce-POS-Manual) पर मुद्दों को प्रस्तुत करके या परिवर्तनों का सुझाव देकर इस दस्तावेज़ को सुधार सकते हैं।
