---
title: कस्टम बारकोड फील्ड
---

डिफ़ॉल्ट रूप से WCPOS SKU फील्ड को बारकोड के रूप में उपयोग करेगा।

![डिफ़ॉल्ट रूप से WCPOS SKU फील्ड को बारकोड के रूप में उपयोग करेगा।](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "डिफ़ॉल्ट रूप से WCPOS SKU फील्ड को बारकोड के रूप में उपयोग करेगा")

[WCPOS प्रो](http://wcpos.com/pro) उपयोगकर्ता `WP एडमिन > POS > सेटिंग्स > सामान्य` में बारकोड फील्ड चुनकर किसी भी उत्पाद मेटा फील्ड का उपयोग कर सकते हैं

![WCPOS प्रो में कस्टम बारकोड फील्ड चुनें](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "WCPOS प्रो में कस्टम बारकोड फील्ड चुनें")

`woocommerce_pos_barcode_meta_key` फ़िल्टर का उपयोग करके कस्टम बारकोड मेटा फील्ड भी सेट किए जा सकते हैं।

```php title="functions.php"
/**
 * कस्टम उत्पाद मेटा फील्ड जिसे बारकोड के रूप में उपयोग किया जा सकता है
 *
 * डिफ़ॉल्ट रूप से WCPOS SKU फील्ड का उपयोग करेगा - '_sku'
 * आप निम्न WordPress फ़िल्टर का उपयोग करके मेटा कुंजी को बदल सकते हैं
 *
 * नीचे लोकप्रिय बारकोड प्लगइन से मेटा फील्ड हैं
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // अपनी कस्टम मेटा कुंजी के लिए बदलें
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
प्रत्येक उत्पाद के लिए बारकोड अद्वितीय होना चाहिए।
:::
