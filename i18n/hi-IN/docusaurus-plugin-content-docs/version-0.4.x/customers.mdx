### क्षमताएं

कैशियर को सभी ग्राहक सुविधाओं का उपयोग करने के लिए _list_users_, _create_users_ और _edit_users_ क्षमताओं की आवश्यकता होगी।

### वर्डप्रेस मल्टीसाइट

[वर्डप्रेस मल्टीसाइट](https://codex.wordpress.org/Create_A_Network) का उपयोग करते समय [अतिरिक्त जांच](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) होती हैं जो गैर [सुपर व्यवस्थापकों](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) को ग्राहक बनाने और संपादित करने से रोकती हैं।

सबसे पहले, गैर सुपर व्यवस्थापकों द्वारा नए ग्राहक **बनाने** के लिए, कृपया `नेटवर्क व्यवस्थापक > सेटिंग्स` में निम्न विकल्प को सक्षम करें।

![नए उपयोगकर्ता बनाने का मल्टीसाइट विकल्प](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "नए उपयोगकर्ता बनाने का मल्टीसाइट विकल्प")

इसके अलावा, गैर सुपर व्यवस्थापकों द्वारा ग्राहक विवरण **संपादित** करने के लिए, आपको कैशियर को `manage_network_users` क्षमता जोड़नी होगी।

```php title="functions.php"
// CASHIER_ID को बदलें, और
// अपनी functions.php फ़ाइल में जोड़ें

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
