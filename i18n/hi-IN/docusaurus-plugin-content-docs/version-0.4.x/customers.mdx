### क्षमताएँ {#capabilities}

कैशियर को सभी ग्राहक सुविधाओं का उपयोग करने के लिए _list_users_, _create_users_ और _edit_users_ क्षमताओं की आवश्यकता होगी।

### वर्डप्रेस मल्टीसाइट {#wordpress-multisite}

जब [वर्डप्रेस मल्टीसाइट](https://codex.wordpress.org/Create_A_Network) का उपयोग करते हैं, तो कुछ [अतिरिक्त चेक](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) होते हैं ताकि गैर [सुपर एडमिन्स](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) ग्राहक बनाने और संपादित करने से रोके जा सके।

पहले, गैर सुपर एडमिन्स द्वारा नए ग्राहक **बनाने** के लिए, कृपया `Network Admin > Settings` में निम्नलिखित विकल्प सक्षम करें।

![नए उपयोगकर्ताओं को बनाने के लिए मल्टीसाइट विकल्प](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "नए उपयोगकर्ताओं को बनाने के लिए मल्टीसाइट विकल्प")

इसके अतिरिक्त, गैर सुपर एडमिन्स द्वारा ग्राहक विवरणों को **संपादित** करने के लिए, आपको कैशियर को `manage_network_users` क्षमता जोड़ने की आवश्यकता होगी।

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
