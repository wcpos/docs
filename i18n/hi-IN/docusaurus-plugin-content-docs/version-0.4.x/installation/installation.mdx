### स्वचालित स्थापना (अनुशंसित)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "WCPOS को 1 मिनट से कम समय में स्थापित और सक्रिय करें")

1. `WP Admin > Plugins > Add New` पर जाएं
2. WordPress प्लगइन निर्देशिका में `WCPOS` के लिए खोजें
3. प्लगइन स्थापित करें
4. प्लगइन सक्रिय करें

### मैनुअल स्थापना

1. [WCPOS डाउनलोड करें](https://wordpress.org/plugins/woocommerce-pos/) WordPress.org से
2. `WP Admin > Plugins > Add New > Upload Plugin` पर जाएं
3. प्लगइन सक्रिय करें

:::info
WCPOS के पिछले संस्करण [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) टैब के अंतर्गत Wordpress.org पर पाए जा सकते हैं।
:::
