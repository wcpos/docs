### स्वचालित इंस्टॉलेशन (अनुशंसित)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "1 मिनट से भी कम समय में WCPOS इंस्टॉल और सक्रिय करें")

1. `WP Admin > Plugins > Add New` पर जाएं
2. WordPress प्लगइन डायरेक्टरी में `WCPOS` खोजें
3. प्लगइन इंस्टॉल करें
4. प्लगइन को सक्रिय करें

### मैनुअल इंस्टॉलेशन

1. WordPress.org से [WCPOS डाउनलोड करें](https://wordpress.org/plugins/woocommerce-pos/)
2. `WP Admin > Plugins > Add New > Upload Plugin` पर जाएं
3. प्लगइन को सक्रिय करें

:::info
WCPOS के पुराने संस्करण WordPress.org पर [डेवलपर्स](https://wordpress.org/plugins/woocommerce-pos/developers/) टैब के अंतर्गत पाए जा सकते हैं।
:::
