Las plantillas te permiten personalizar la apariencia y el contenido de varias salidas de tu sistema POS. Puedes acceder al editor de plantillas navegando a: `WP Admin > POS > Plantillas`

El sistema de plantillas proporciona flexibilidad para personalizar:

- **Plantillas de Recibos** - Personaliza recibos impresos y digitales
- **Plantillas de Informes** - Personaliza el diseño de informes _(próximamente)_
- **Plantillas de Correo Electrónico** - Personaliza las notificaciones por correo electrónico _(próximamente)_

## Tipos de Plantillas

### Plantillas Predeterminadas

Detectadas automáticamente a partir de los archivos de tu plugin y tema. Estas no se pueden eliminar, pero se pueden copiar para crear versiones personalizadas.

### Plantillas Personalizadas

Plantillas creadas por el usuario que puedes personalizar, editar y gestionar completamente a través de la interfaz de administración de WordPress.
