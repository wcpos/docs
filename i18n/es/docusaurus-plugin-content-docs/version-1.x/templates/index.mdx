Las plantillas te permiten personalizar la apariencia y el contenido de varias salidas de tu sistema POS. Puedes acceder al editor de plantillas navegando a: `WP Admin > POS > Plantillas`

El sistema de plantillas ofrece flexibilidad para personalizar:

- **Plantillas de Recibos** - Personalizar recibos impresos y digitales
- **Plantillas de Informes** - Personalizar diseños de informes _(próximamente)_
- **Plantillas de Correo Electrónico** - Personalizar notificaciones por correo _(próximamente)_

## Tipos de Plantillas

### Plantillas Predeterminadas

Detectadas automáticamente desde tus archivos de complementos y temas. No se pueden eliminar, pero se pueden copiar para crear versiones personalizadas.

### Plantillas Personalizadas

Plantillas creadas por el usuario que puedes personalizar completamente, editar y administrar a través de la interfaz de administración de WordPress.
