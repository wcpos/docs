WooCommerce POS v1.0 ha recibido importantes actualizaciones en comparación con las versiones anteriores 0.4.x:

- La aplicación ahora está construida usando [React Native](https://reactnative.dev/), haciendo uso de herramientas modernas de desarrollo.
- Esta nueva base de código nos permitirá agregar más características, comenzando con una [aplicación de escritorio](<(https://github.com/wcpos/electron/releases)>) dedicada. Las versiones de iOS y Android son las siguientes en nuestra lista.
- Algunas características han cambiado desde 0.4.x, lo que puede ser confuso para algunos usuarios.

Estamos trabajando para actualizar la documentación para ayudarlo a guiarlo a través de estos cambios lo más rápido posible. Agradecemos su paciencia durante esta transición.
