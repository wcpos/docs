# Pasarela de Pago de Checkout Web

La pasarela de Checkout Web permite a los clientes completar sus transacciones de POS utilizando el sistema de checkout de su tienda web. Esta pasarela crea un puente perfecto entre su POS en tienda y los métodos de pago en línea, ideal para situaciones donde los clientes prefieren pagar usando sus propios dispositivos o cuando necesita acceso a métodos de pago exclusivos de la web.

## Características

- **Integración Perfecta**: Enlace directo desde POS a checkout web
- **Todos los Métodos de Pago**: Acceso a todas las pasarelas de pago habilitadas en su tienda web
- **Control del Cliente**: Los clientes pueden pagar usando sus propios dispositivos y métodos preferidos
- **Sincronización de Pedidos**: Actualizaciones automáticas de pedidos entre POS y tienda web
- **Flujo de Trabajo Flexible**: Complete transacciones en línea manteniendo el flujo de trabajo de POS

## Instalación

### Paso 1: Descargar e Instalar

1. Visite la [página de lanzamientos de Pasarela de Checkout Web](https://github.com/wcpos/web-checkout-gateway/releases)
2. Descargue el archivo **woocommerce-pos-web-checkout-gateway.zip** más reciente
3. En su administración de WordPress, vaya a `Plugins > Añadir Nuevo > Subir Plugin`
4. Suba el archivo zip y active el plugin

### Paso 2: Habilitar en WooCommerce POS

1. Vaya a `WP Admin > WooCommerce POS > Configuración > Checkout`
2. Encuentre la **Pasarela de Checkout Web de WooCommerce POS** en la lista
3. Habilite la pasarela para su uso en el POS
4. Guarde su configuración

:::note
Esta pasarela aprovecha sus métodos de pago de WooCommerce existentes, así que asegúrese de tener al menos una pasarela de pago web configurada correctamente (Stripe, PayPal, etc.).
:::

## Uso

### Iniciando Checkout Web

1. **Añadir Artículos**: Añada productos a su carrito en el POS
2. **Seleccionar Pasarela**: Elija "Checkout Web" como método de pago
3. **Generar Enlace**: El sistema crea un enlace de checkout único para el pedido
4. **Acceso del Cliente**: El cliente hace clic en el enlace para acceder al checkout web
5. **Pago en Línea**: El cliente completa el pago usando métodos de pago de la tienda web

### Proceso de Pago

La pasarela de Checkout Web sigue este flujo de trabajo:

1. **Creación de Pedido**: El POS crea un pedido pendiente en WooCommerce
2. **Enlace de Checkout**: El sistema genera un enlace de pago seguro y único
3. **Redirección del Cliente**: El cliente es dirigido a la página de checkout web
4. **Selección de Pago**: El cliente elige de los métodos de pago web disponibles
5. **Procesamiento de Pago**: El proceso de checkout estándar de WooCommerce maneja el pago
6. **Finalización de Pedido**: El estado del pedido se actualiza automáticamente tras un pago exitoso

### Flujo de Trabajo de POS

Después de iniciar el checkout web:

1. **Monitorear Estado**: Mantenga abierta la pantalla de pedidos de POS para monitorear el estado de pago
2. **Pago del Cliente**: El cliente completa el pago en su dispositivo o su tableta
3. **Actualización Automática**: El estado del pedido se actualiza en tiempo real cuando se recibe el pago
4. **Procesar Pago**: Haga clic en el botón "Procesar Pago" en POS para continuar hasta el recibo
5. **Generación de Recibo**: Genere e imprima el recibo como de costumbre
