# Puerta de enlace Terminal SumUp {#sumup-terminal-gateway}

La puerta de enlace Terminal SumUp te permite aceptar pagos con tarjeta utilizando lectores de tarjetas SumUp directamente dentro de WooCommerce POS. Esta puerta de enlace proporciona una integración completa con el sistema de procesamiento de pagos de SumUp para transacciones en persona.

## Instalación {#installation}

<Steps>
  <StepCard number="1" title="Descargar e Instalar">

1. Visita la [página de lanzamientos de SumUp Terminal para WooCommerce](https://github.com/wcpos/sumup-terminal-for-woocommerce/releases)
2. Descarga el último archivo **sumup-terminal-for-woocommerce.zip**
3. En tu administrador de WordPress, ve a `Plugins > Añadir Nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

  </StepCard>
  <StepCard number="2" title="Configurar Ajustes de SumUp">

1. Navega a `WP Admin > WooCommerce > Ajustes > Pagos`
2. Busca **Terminal SumUp** en la lista de métodos de pago
3. Haz clic en **Terminal SumUp** para acceder a la configuración
4. Introduce tu **Clave API de SumUp** (disponible en tu panel de comerciante de SumUp)
5. Guarda los ajustes

:::note
No necesitas habilitar la puerta de enlace Terminal SumUp en los ajustes de WooCommerce. Se habilitará específicamente para el POS en un paso posterior.
:::

  </StepCard>
  <StepCard number="3" title="Emparejar Tu Terminal SumUp">

1. En la misma página de configuración, localiza la sección **Emparejar Lector**
2. En tu dispositivo SumUp, navega a la pantalla de emparejamiento para mostrar el código de emparejamiento
3. Introduce el código de emparejamiento mostrado en tu dispositivo SumUp
4. Haz clic en **"Emparejar Lector"** para establecer la conexión
5. Espera la confirmación de que el lector ha sido emparejado con éxito

:::warning Importante
El lector debe estar emparejado con éxito antes de que puedas procesar pagos. Asegúrate de que el proceso de emparejamiento se complete antes de continuar.
:::

  </StepCard>
  <StepCard number="4" title="Habilitar en WooCommerce POS" isLast>

1. Ve a `WP Admin > WooCommerce POS > Ajustes > Pago`
2. Busca la puerta de enlace **Terminal SumUp** en la lista
3. Habilita la puerta de enlace para su uso en el POS
4. Guarda tus ajustes

  </StepCard>
</Steps>

## Uso {#usage}

### Procesamiento de Pagos {#processing-payments}

1. **Agregar Artículos**: Agrega productos a tu carrito en el POS
2. **Seleccionar Puerta de Enlace**: Elige "Terminal SumUp" como método de pago
3. **Iniciar Pago**: Haz clic para iniciar un nuevo pago en tu dispositivo SumUp
4. **Pago del Cliente**: El cliente completa el pago en el terminal SumUp
5. **Finalización Automática**: El pedido se completa automáticamente cuando se detecta el pago

### Controles de Pago {#payment-controls}

Al usar la puerta de enlace Terminal SumUp, tienes las siguientes opciones:

- **Iniciar Nuevo Pago**: Inicia una solicitud de pago en el terminal conectado
- **Cancelar Pago**: Cancela un pago que se esté procesando actualmente
- **Estado del Pago**: Monitorea el estado actual del proceso de pago

### Gestión de Pedidos {#order-management}

- **Actualizaciones Automáticas**: Los pedidos se marcan automáticamente como completados tras un pago exitoso
- **Seguimiento de Pagos**: Todos los detalles del pago se registran en las notas del pedido
- **Generación de Recibos**: Se generan recibos estándar del POS tras pagos exitosos

## Compatibilidad de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividad
Esta implementación utiliza el SDK de JavaScript de SumUp, lo que significa que funciona a través de aplicaciones web pero requiere **terminales conectados a internet solamente**. Los terminales Bluetooth no son compatibles actualmente.
:::

### Terminales Soportados (Conectados a Internet) {#supported-terminals-internet-connected}

- **SumUp Solo** ✅ - Terminal conectado por WiFi/Ethernet

### Terminales No Soportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **SumUp Air** ❌ - Solo Bluetooth
- **SumUp 3G** ❌ - Solo Bluetooth

:::note Soporte Futuro
El soporte para terminales Bluetooth se añadirá a las aplicaciones de iOS y Android en **Q1 2026**. Esto habilitará el soporte para todos los terminales certificados por SumUp, incluidos los modelos Air y 3G.
:::

## Solución de Problemas {#troubleshooting}

### Problemas Comunes {#common-issues}

<Accordion>
  <AccordionItem question="El lector no se empareja">

  - Asegúrate de que estás usando un terminal conectado a internet compatible (Solo únicamente)
  - Verifica que tu dispositivo SumUp Solo esté conectado a WiFi y en línea
  - Comprueba que el dispositivo esté en modo de emparejamiento y que el código de emparejamiento se haya introducido correctamente
  - Confirma que tu cuenta de SumUp tenga habilitado el acceso a la API
  - Asegúrate de que la conexión a internet sea estable durante el emparejamiento

  </AccordionItem>

  <AccordionItem question="El pago no se procesa">

  - Verifica que el lector esté emparejado correctamente
  - Comprueba que tu cuenta de SumUp esté activa y en buen estado
  - Asegúrate de que el dispositivo tenga suficiente batería y conectividad
  - Confirma que la clave API sea correcta y esté activa

  </AccordionItem>

  <AccordionItem question="El pedido no se completa">

  - Comprueba la estabilidad de la conexión a internet
  - Verifica la configuración del webhook en tu cuenta de SumUp
  - Asegúrate de que el pago haya sido exitoso en el terminal SumUp
  - Revisa los registros de errores de WordPress en busca de problemas de comunicación de API

  </AccordionItem>
</Accordion>

### Límites de Tasa de API {#api-rate-limits}

La puerta de enlace está optimizada para reducir las llamadas a la API y evitar la limitación de tasa:
- El estado del pago se verifica de manera eficiente
- Se minimizan las solicitudes a la API innecesarias
- La lógica de reintento automática maneja fallos temporales
