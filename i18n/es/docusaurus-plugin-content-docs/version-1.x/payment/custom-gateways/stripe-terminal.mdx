# Puerta de enlace de Stripe Terminal

La puerta de enlace de Stripe Terminal te permite aceptar pagos en persona utilizando lectores de hardware de Stripe Terminal directamente dentro de WooCommerce POS. Esta puerta de enlace admite tanto lectores de tarjetas físicas como el modo simulador para pruebas.

## Características

- **Integración de Hardware**: Conecta lectores físicos de Stripe Terminal a través de conexión a internet
- **Modo Simulador**: Prueba pagos sin hardware utilizando el simulador de Stripe
- **Múltiples Métodos de Pago**: Soporte para varios tipos de tarjetas y métodos de pago
- **Procesamiento en Tiempo Real**: Procesamiento y confirmación de pagos instantáneos
- **Transacciones Seguras**: Procesamiento de pagos conforme a PCI a través de Stripe
- **SDK Basado en la Web**: Utiliza el SDK de JavaScript de Stripe para compatibilidad con aplicaciones web

## Instalación

### Paso 1: Descargar e Instalar

1. Visita la [página de lanzamientos de Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Descarga el archivo **stripe-terminal-for-woocommerce.zip** más reciente
3. En tu administración de WordPress, ve a `Plugins > Añadir Nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

### Paso 2: Configurar la Configuración de Stripe

1. Navega a `WP Admin > WooCommerce > Configuración > Pagos`
2. Busca **Stripe Terminal** en la lista de métodos de pago
3. Haz clic en **Stripe Terminal** para acceder a la configuración
4. Ingresa tu **Clave Secreta de Stripe** (puedes obtenerla desde tu Tablero de Stripe)
5. Guarda la configuración

:::note
No necesitas habilitar la puerta de enlace de Stripe Terminal en la configuración de WooCommerce. Se habilitará específicamente para el POS en el siguiente paso.
:::

### Paso 3: Habilitar en WooCommerce POS

1. Ve a `WP Admin > WooCommerce POS > Settings > Checkout`
2. Busca la puerta de enlace **Stripe Terminal** en la lista
3. Habilita la puerta de enlace para su uso en el POS
4. Guarda tu configuración

## Uso

### Conectando un Lector

Cuando seleccionas la puerta de enlace de Stripe Terminal durante el pago en el POS:

1. **Elegir Método de Conexión**: Puedes conectar un lector físico o usar el simulador
2. **Lector Físico**: Sigue las instrucciones en pantalla para conectar tu dispositivo de Stripe Terminal
3. **Simulador**: Selecciona el modo simulador para probar varios escenarios de pago sin hardware

### Procesando Pagos

1. **Agregar Ítems**: Agrega artículos a tu carrito en el POS
2. **Seleccionar Puerta de Enlace**: Elige "Stripe Terminal" como método de pago
3. **Conectar Lector**: Conecta tu lector o elige el modo simulador
4. **Procesar Pago**: Sigue las indicaciones para completar la transacción
5. **Confirmación**: El pedido se completará automáticamente al realizar un pago exitoso

### Pruebas con el Simulador

El simulador te permite probar varios métodos y escenarios de pago:

- **Pagos con Tarjeta**: Prueba diferentes tipos de tarjetas (Visa, Mastercard, etc.)
- **Pagos Sin Contacto**: Simula transacciones de pago por acercamiento
- **Escenarios de Error**: Prueba pagos rechazados y otras condiciones de error
- **Diferentes Cantidades**: Prueba varias cantidades de transacción

## Requisitos

- **Cuenta de Stripe**: Cuenta de Stripe activa con Terminal habilitado
- **Claves API**: Clave secreta de Stripe desde tu tablero
- **WooCommerce POS**: Versión gratuita o Pro instalada y activada
- **HTTPS**: Tu sitio debe usar SSL/HTTPS para seguridad

## Compatibilidad de Hardware

:::important Requisitos de Conectividad
Esta implementación utiliza el SDK de JavaScript de Stripe, lo que significa que funciona a través de aplicaciones web pero requiere **terminales conectados a internet únicamente**. Los terminales Bluetooth no son actualmente compatibles.
:::

### Terminales Soportados (Conectados a Internet)

- **Stripe Reader S700** ✅ - Terminal conectado por Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado por WiFi

### Terminales No Soportados (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth  
- **Verifone P400** ❌ - Solo Bluetooth

:::note Soporte Futuro
El soporte para terminales Bluetooth se añadirá a las aplicaciones de iOS y Android en **Q1 2026**. Esto habilitará el soporte para todos los lectores certificados de Stripe Terminal, incluyendo el M2 y el WisePad 3.
:::

## Solución de Problemas

### Problemas Comunes

**El Lector No Se Conecta**
- Asegúrate de estar utilizando un terminal conectado a internet soportado (S700 o WisePOS E)
- Verifica que el terminal esté conectado a WiFi/Ethernet y en línea
- Comprueba que tu cuenta de Stripe tenga habilitada la Terminal
- Confirma que el terminal esté registrado en tu Tablero de Stripe

**Pago Rechazado**
- Verifica que tu cuenta de Stripe esté activa y en buen estado
- Asegúrate de que la tarjeta utilizada sea válida
- Asegúrate de tener fondos suficientes disponibles

**Errores en el Certificado SSL**
- Stripe Terminal requiere HTTPS - asegúrate de que tu sitio tenga un certificado SSL válido
- Verifica que tu certificado SSL esté configurado correctamente
