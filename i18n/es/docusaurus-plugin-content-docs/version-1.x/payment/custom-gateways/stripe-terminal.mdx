# Pasarela Stripe Terminal {#stripe-terminal-gateway}

La pasarela Stripe Terminal permite que aceptes pagos en persona utilizando lectores de hardware de Stripe Terminal directamente en WooCommerce POS. Esta pasarela es compatible con lectores de tarjetas físicos y el modo simulador para pruebas.

## Características {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Integración de Hardware">
    Conecta lectores físicos de Stripe Terminal mediante conexión a internet
  </FeatureCard>
  <FeatureCard icon="desktop" title="Modo Simulador">

  </FeatureCard>

  </FeatureCard>
</FeatureGrid>

## Instalación {#installation}

<Steps>
  <StepCard number="1" title="Descargar e Instalar">

1. Visita la [página de lanzamientos de Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Descarga el último archivo **stripe-terminal-for-woocommerce.zip**
3. En tu administración de WordPress, ve a `Plugins > Añadir Nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

  </StepCard>
  <StepCard number="2" title="Configurar Ajustes de Stripe">

1. Navega a `WP Admin > WooCommerce > Settings > Payments`
2. Encuentra **Stripe Terminal** en la lista de métodos de pago
3. Haz clic en **Stripe Terminal** para acceder a los ajustes
4. Ingresa tu **Clave Secreta de Stripe** (puedes obtenerla desde tu Panel de Control de Stripe)
5. Guarda los ajustes

:::note
No necesitas habilitar la pasarela Stripe Terminal en los ajustes de WooCommerce. Se habilitará específicamente para el POS en el siguiente paso.
:::

  </StepCard>
  <StepCard number="3" title="Habilitar en WooCommerce POS" isLast>

1. Ve a `WP Admin > WooCommerce POS > Settings > Checkout`
2. Encuentra la pasarela **Stripe Terminal** en la lista
3. Habilita la pasarela para su uso en el POS
4. Guarda tus ajustes

  </StepCard>
</Steps>

## Uso {#usage}

### Conectando un Lector {#connecting-a-reader}

Cuando selecciones la pasarela Stripe Terminal durante el pago en el POS:

1. **Elegir Método de Conexión**: Puedes conectar un lector físico o usar el simulador
2. **Lector Físico**: Sigue las instrucciones en pantalla para conectar tu dispositivo Stripe Terminal
3. **Simulador**: Selecciona el modo simulador para probar diversos escenarios de pago sin hardware

### Procesando Pagos {#processing-payments}

1. **Agregar Artículos**: Agrega productos a tu carrito en el POS
2. **Seleccionar Pasarela**: Elige "Stripe Terminal" como método de pago
3. **Conectar Lector**: Conecta tu lector o elige el modo simulador
4. **Procesar Pago**: Sigue las indicaciones para completar la transacción
5. **Confirmación**: El pedido se completará automáticamente al realizar el pago exitosamente

### Pruebas con el Simulador {#testing-with-simulator}

El simulador te permite probar varios métodos de pago y escenarios:

- **Pagos con Tarjeta**: Prueba diferentes tipos de tarjetas (Visa, Mastercard, etc.)
- **Pagos Sin Contacto**: Simula transacciones de pago por acercamiento
- **Escenarios de Error**: Prueba pagos rechazados y otras condiciones de error
- **Diferentes Montos**: Prueba varios montos de transacción

## Compatibilidad de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividad
Esta implementación utiliza el SDK de JavaScript de Stripe, lo que significa que funciona a través de aplicaciones web pero requiere **terminales solo con conexión a internet**. Los terminales Bluetooth no son soportados actualmente.
:::

### Terminales Soportados (Con Conexión a Internet) {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Terminal conectado por Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado por WiFi

### Terminales No Soportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth  
- **Verifone P400** ❌ - Solo Bluetooth

:::note Soporte Futuro
El soporte para terminales Bluetooth se añadirá a las aplicaciones de iOS y Android en **Q1 2026**. Esto permitirá soporte para todos los lectores certificados de Stripe Terminal, incluyendo el M2 y WisePad 3.
:::

### Problemas Comunes {#common-issues}

<Accordion>
  <AccordionItem question="El lector no se conecta">

  - Asegúrate de estar utilizando un terminal conectado a internet soportado (S700 o WisePOS E)
  - Verifica que el terminal esté conectado a WiFi/Ethernet y en línea
  - Comprueba que tu cuenta de Stripe tenga activado Terminal
  - Confirma que el terminal esté registrado en tu Panel de Control de Stripe

  </AccordionItem>

  <AccordionItem question="Pago rechazado">

  - Verifica que tu cuenta de Stripe esté activa y en buen estado
  - Asegúrate de que la tarjeta utilizada sea válida
  - Confirma que hay fondos suficientes disponibles

  </AccordionItem>

  <AccordionItem question="Errores de certificado SSL">

  - Stripe Terminal requiere HTTPS - asegúrate de que tu sitio tenga un certificado SSL válido
  - Verifica que tu certificado SSL esté correctamente configurado

  </AccordionItem>
</Accordion>
