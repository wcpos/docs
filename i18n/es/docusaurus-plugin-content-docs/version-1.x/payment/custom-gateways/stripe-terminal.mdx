# Gateway de Terminal Stripe {#stripe-terminal-gateway}

El gateway de Terminal Stripe te permite aceptar pagos en persona utilizando lectores de hardware Stripe Terminal directamente dentro de WCPOS. Este gateway admite tanto lectores de tarjetas físicas como modo simulador para pruebas.

## Características {#features}

<FeatureGrid>
  <FeatureCard icon="credit-card" title="Integración de Hardware">
    Conecta lectores físicos de Stripe Terminal a través de la conexión a internet
  </FeatureCard>
  <FeatureCard icon="desktop" title="Modo Simulador">
    Prueba pagos sin hardware utilizando el simulador de Stripe
  </FeatureCard>
  <FeatureCard icon="bolt" title="Procesamiento en Tiempo Real">
    Procesamiento y confirmación de pagos instantáneos
  </FeatureCard>
  <FeatureCard icon="shield" title="Transacciones Seguras">

  </FeatureCard>
</FeatureGrid>

## Instalación {#installation}

<Steps>
  <StepCard number="1" title="Descargar e Instalar">

1. Visita la [página de lanzamientos de Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Descarga el archivo **stripe-terminal-for-woocommerce.zip** más reciente
3. En tu administrador de WordPress, ve a `Plugins > Añadir Nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

  </StepCard>
  <StepCard number="2" title="Configurar Ajustes de Stripe">

1. Navega a `WP Admin > WooCommerce > Ajustes > Pagos`
2. Encuentra **Stripe Terminal** en la lista de métodos de pago
3. Haz clic en **Stripe Terminal** para acceder a los ajustes
4. Ingresa tu **Clave Secreta de Stripe** (puedes obtenerla desde tu Panel de Control de Stripe)
5. Guarda los ajustes

:::note
No es necesario habilitar el gateway de Stripe Terminal en los ajustes de WooCommerce. Se habilitará específicamente para el POS en el siguiente paso.
:::

  </StepCard>
  <StepCard number="3" title="Habilitar en WCPOS" isLast>

1. Ve a `WP Admin > POS > Ajustes > Finalizar Compra`
2. Encuentra el gateway **Stripe Terminal** en la lista
3. Habilita el gateway para su uso en el POS
4. Guarda tus ajustes

  </StepCard>
</Steps>

## Uso {#usage}

### Conectar un Lector {#connecting-a-reader}

Cuando seleccionas el gateway de Terminal Stripe durante el pago en el POS:

1. **Elegir Método de Conexión**: Puedes conectar un lector físico o usar el simulador
2. **Lector Físico**: Sigue las instrucciones en pantalla para conectar tu dispositivo Terminal de Stripe
3. **Simulador**: Selecciona el modo simulador para probar varios escenarios de pago sin hardware

### Procesamiento de Pagos {#processing-payments}

1. **Agregar Artículos**: Agrega productos a tu carrito en el POS
2. **Seleccionar Gateway**: Elige "Stripe Terminal" como método de pago
3. **Conectar Lector**: Conecta tu lector o elige el modo simulador
4. **Procesar Pago**: Sigue las indicaciones para completar la transacción
5. **Confirmación**: El pedido se completará automáticamente una vez que el pago sea exitoso

### Pruebas con el Simulador {#testing-with-simulator}

El simulador te permite probar varios métodos y escenarios de pago:

- **Pagos con Tarjeta**: Prueba diferentes tipos de tarjetas (Visa, Mastercard, etc.)
- **Pagos Sin Contacto**: Simula transacciones de pago por acercamiento
- **Escenarios de Error**: Prueba pagos rechazados y otras condiciones de error
- **Diferentes Montos**: Prueba varios montos de transacción

## Compatibilidad de Hardware {#hardware-compatibility}

:::important Requisitos de Conectividad
Esta implementación utiliza el SDK de JavaScript de Stripe, lo que significa que funciona a través de aplicaciones web, pero requiere **terminales solo conectados a internet**. Los terminales Bluetooth no son compatibles actualmente.
:::

### Terminales Soportados (Conectados a Internet) {#supported-terminals-internet-connected}

- **Stripe Reader S700** ✅ - Terminal conectado por Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectado por WiFi

### Terminales No Soportados (Bluetooth) {#unsupported-terminals-bluetooth}

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth  
- **Verifone P400** ❌ - Solo Bluetooth

:::note Soporte Futuro
El soporte para terminales Bluetooth se añadirá a las aplicaciones de iOS y Android en **Q1 2026**. Esto permitirá el soporte para todos los lectores certificados de Terminal Stripe, incluyendo el M2 y WisePad 3.
:::

### Problemas Comunes {#common-issues}

<Accordion>
  <AccordionItem question="El lector no se conecta">

  - Asegúrate de estar usando un terminal conectado a internet admitido (S700 o WisePOS E)
  - Verifica que el terminal esté conectado a WiFi/Ethernet y en línea
  - Asegúrate de que tu cuenta de Stripe tenga habilitado el Terminal
  - Confirma que el terminal esté registrado en tu Panel de Control de Stripe

  </AccordionItem>

  <AccordionItem question="Pago rechazado">

  - Verifica que tu cuenta de Stripe esté activa y en buen estado
  - Asegúrate de que la tarjeta usada sea válida
  - Asegúrate de que haya fondos suficientes disponibles

  </AccordionItem>

  <AccordionItem question="Errores de certificado SSL">

  - Stripe Terminal requiere HTTPS - asegúrate de que tu sitio tenga un certificado SSL válido
  - Verifica que tu certificado SSL esté configurado correctamente

  </AccordionItem>
</Accordion>
