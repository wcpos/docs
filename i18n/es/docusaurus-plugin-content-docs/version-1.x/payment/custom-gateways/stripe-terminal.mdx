# Pasarela de Stripe Terminal

La pasarela de Stripe Terminal te permite aceptar pagos presenciales usando lectores de hardware de Stripe Terminal directamente dentro de WooCommerce POS. Esta pasarela admite lectores físicos de tarjetas y un modo de simulación para pruebas.

## Características

- **Integración de Hardware**: Conecta lectores físicos de Stripe Terminal a través de conexión a internet
- **Modo de Simulación**: Prueba pagos sin hardware usando el simulador de Stripe
- **Múltiples Métodos de Pago**: Compatibilidad con varios tipos de tarjetas y métodos de pago
- **Procesamiento en Tiempo Real**: Procesamiento y confirmación de pagos instantáneos
- **Transacciones Seguras**: Procesamiento de pagos compatible con PCI a través de Stripe
- **SDK Web**: Usa el SDK de JavaScript de Stripe para compatibilidad con aplicaciones web

## Instalación

### Paso 1: Descargar e Instalar

1. Visita la [página de lanzamientos de Stripe Terminal para WooCommerce](https://github.com/wcpos/stripe-terminal-for-woocommerce/releases)
2. Descarga el último archivo **stripe-terminal-for-woocommerce.zip**
3. En tu administrador de WordPress, ve a `Plugins > Añadir Nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

### Paso 2: Configurar Ajustes de Stripe

1. Navega a `Admin de WP > WooCommerce > Ajustes > Pagos`
2. Encuentra **Stripe Terminal** en la lista de métodos de pago
3. Haz clic en **Stripe Terminal** para acceder a los ajustes
4. Introduce tu **Clave Secreta de Stripe** (puedes obtenerla en tu Panel de Stripe)
5. Guarda los ajustes

:::note
No necesitas habilitar la pasarela de Stripe Terminal en los ajustes de WooCommerce. Se habilitará específicamente para el POS en el siguiente paso.
:::

### Paso 3: Habilitar en WooCommerce POS

1. Ve a `Admin de WP > WooCommerce POS > Ajustes > Checkout`
2. Encuentra la pasarela de **Stripe Terminal** en la lista
3. Habilita la pasarela para su uso en el POS
4. Guarda tus ajustes

## Uso

### Conectar un Lector

Al seleccionar la pasarela de Stripe Terminal durante el checkout en el POS:

1. **Elegir Método de Conexión**: Puedes conectar un lector físico o usar el simulador
2. **Lector Físico**: Sigue las instrucciones en pantalla para conectar tu dispositivo Stripe Terminal
3. **Simulador**: Selecciona el modo de simulación para probar varios escenarios de pago sin hardware

### Procesar Pagos

1. **Añadir Artículos**: Añade productos a tu carrito en el POS
2. **Seleccionar Pasarela**: Elige "Stripe Terminal" como método de pago
3. **Conectar Lector**: Conecta tu lector o elige el modo de simulación
4. **Procesar Pago**: Sigue las indicaciones para completar la transacción
5. **Confirmación**: El pedido se completará automáticamente tras un pago exitoso

### Probar con Simulador

El simulador te permite probar varios métodos de pago y escenarios:

- **Pagos con Tarjeta**: Prueba diferentes tipos de tarjetas (Visa, Mastercard, etc.)
- **Pagos sin Contacto**: Simula transacciones de pago con un toque
- **Escenarios de Error**: Prueba pagos rechazados y otras condiciones de error
- **Diferentes Importes**: Prueba varios importes de transacción

## Requisitos

- **Cuenta de Stripe**: Cuenta de Stripe activa con Terminal habilitado
- **Claves API**: Clave secreta de Stripe de tu panel
- **WooCommerce POS**: Versión gratuita o profesional instalada y activada
- **HTTPS**: Tu sitio debe usar SSL/HTTPS por seguridad

## Compatibilidad de Hardware

:::important Requisitos de Conectividad
Esta implementación usa el SDK de JavaScript de Stripe, lo que significa que funciona a través de aplicaciones web pero requiere **terminales conectados a internet únicamente**. Los terminales Bluetooth no están actualmente compatibles.
:::

### Terminales Compatibles (Conectadas a Internet)

- **Stripe Reader S700** ✅ - Terminal conectada por Ethernet/WiFi
- **WisePOS E** ✅ - Terminal conectada por WiFi

### Terminales No Compatibles (Bluetooth)

- **BBPOS Chipper 2X BT** ❌ - Solo Bluetooth
- **BBPOS WisePad 3** ❌ - Solo Bluetooth  
- **Verifone P400** ❌ - Solo Bluetooth

:::note Soporte Futuro
La compatibilidad con terminales Bluetooth se agregará en las aplicaciones de iOS y Android en el **Q1 2026**. Esto permitirá soporte para todos los lectores certificados por Stripe Terminal, incluidos el M2 y WisePad 3.
:::

## Resolución de Problemas

### Problemas Comunes

**El Lector No Se Conecta**
- Asegúrate de usar un terminal conectado a internet compatible (S700 o WisePOS E)
- Verifica que el terminal esté conectado a WiFi/Ethernet y en línea
- Comprueba que tu cuenta de Stripe tenga Terminal habilitado
- Confirma que el terminal está registrado en tu Panel de Stripe

**Pago Rechazado**
- Comprueba que tu cuenta de Stripe está activa y en buen estado
- Verifica que la tarjeta que se está usando sea válida
- Asegúrate de que haya fondos suficientes disponibles

**Errores de Certificado SSL**
- Stripe Terminal requiere HTTPS - asegúrate de que tu sitio tenga un certificado SSL válido
- Comprueba que tu certificado SSL esté configurado correctamente
