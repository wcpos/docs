# Pasarela de Factura por Correo Electrónico

La pasarela de factura por correo electrónico le permite enviar facturas de pago a clientes directamente por correo electrónico desde WooCommerce POS. Esto es perfecto para situaciones en las que los clientes prefieren pagar después o necesitan completar el pago en línea utilizando los métodos de pago de su tienda web.

## Características

- **Integración de Correo Electrónico**: Envía facturas directamente desde la interfaz del POS
- **Gestión de Clientes**: Autocompletar correo electrónico desde datos de clientes existentes o agregar nuevas direcciones de correo
- **Pedidos de Invitados**: Soporte para clientes invitados con actualizaciones opcionales de dirección de facturación
- **Pago en Línea**: Los clientes reciben un enlace para "Pagar este pedido" para completar el pago en línea
- **Flujo de Trabajo Flexible**: Completar pedidos en la tienda o permitir la finalización de pagos de forma remota

## Instalación

### Paso 1: Descargar e Instalar

1. Visita la [página de lanzamientos de Email Invoice Gateway](https://github.com/wcpos/email-invoice-gateway/releases)
2. Descarga el último archivo **woocommerce-pos-email-invoice-gateway.zip**
3. En tu administrador de WordPress, ve a `Plugins > Añadir nuevo > Subir Plugin`
4. Sube el archivo zip y activa el plugin

### Paso 2: Habilitar en WooCommerce POS

1. Ve a `Administrador de WP > WooCommerce POS > Configuración > Pago`
2. Encuentra la **Pasarela de Pago de Factura de WooCommerce POS** en la lista
3. Habilita la pasarela para su uso en el POS
4. Guarda tu configuración

:::note
Esta pasarela no requiere claves API o configuración de servicios externos. Utiliza su sistema de correo electrónico WordPress existente y métodos de pago de WooCommerce.
:::

## Uso

### Envío de Facturas

1. **Agregar Artículos**: Añade productos a tu carrito en el POS
2. **Seleccionar Pasarela**: Elige "Factura por Correo Electrónico" como método de pago
3. **Dirección de Correo Electrónico**: 
   - Para clientes existentes: El correo electrónico se completa automáticamente desde los datos del cliente
   - Para pedidos de invitados: Introduce manualmente la dirección de correo electrónico del cliente
4. **Opcional**: Marca "Guardar en dirección de facturación" para actualizar la información de facturación del cliente
5. **Enviar Factura**: Completa el proceso para enviar el correo electrónico de la factura

### Experiencia del Cliente

Cuando envías una factura, el cliente recibe un correo electrónico que contiene:

- **Detalles del Pedido**: Desglose completo de artículos, cantidades y precios
- **Total del Pedido**: Importe final debido, incluyendo impuestos y tarifas
- **Enlace de Pago**: Enlace directo para completar el pago en línea
- **Información del Pedido**: Número de pedido y detalles de referencia

### Completar Pago

Los clientes pueden completar el pago de dos maneras:

1. **Pago en Línea**: Hacer clic en el enlace "Pagar este pedido" en el correo electrónico
2. **Regreso a la Tienda**: Regresar para completar el pago con otros métodos de pago del POS

El enlace "Pagar este pedido" lleva a los clientes a una página de pago dedicada en su sitio web donde pueden utilizar cualquiera de sus pasarelas de pago web habilitadas (Stripe, PayPal, etc.).

## Gestión de Correos Electrónicos

### Clientes Existentes

- Las direcciones de correo electrónico se recuperan automáticamente de los perfiles de los clientes
- Se conserva la información de facturación del cliente
- El historial de pedidos se mantiene en la cuenta del cliente

### Clientes Invitados

- Introducir manualmente direcciones de correo electrónico para clientes ocasionales
- Opción de guardar el correo electrónico en la dirección de facturación para futuras referencias
- Los pedidos de invitados se rastrean correctamente en WooCommerce

### Plantillas de Correo Electrónico

La pasarela utiliza las plantillas de correo electrónico estándar de WooCommerce:

- **Personalizable**: Modificar la apariencia del correo electrónico a través de la configuración de correo electrónico de WooCommerce
- **Marca**: Incluir el logotipo y la marca de su tienda
- **Profesional**: Formato de factura limpio y profesional
- **Compatible con Dispositivos Móviles**: Diseño responsivo para todos los dispositivos

## Casos de Uso

### Perfecto Para

- **Ventas B2B**: Clientes comerciales que necesitan procesar pagos a través de sus sistemas contables
- **Pedidos Grandes**: Transacciones de alto valor que requieren aprobación o tiempo de procesamiento
- **Clientes Remotos**: Clientes que necesitan completar el pago después de salir de la tienda
- **Clientes con Cuenta**: Clientes regulares con términos de pago establecidos
- **Cotización a Pedido**: Convertir cotizaciones o presupuestos en facturas pagaderas

### Ejemplos de Flujo de Trabajo

**Tienda Minorista**
1. El cliente selecciona artículos pero necesita aprobación para la compra
2. Enviar factura por correo electrónico con todos los detalles
3. El cliente completa el pago en línea cuando esté listo
4. El pedido se actualiza automáticamente en su sistema

**Negocio de Servicios**
1. Completar trabajo de servicio y añadir artículos al POS
2. Enviar factura al departamento contable del cliente
3. El cliente paga en línea utilizando el método de pago preferido
4. Recibir confirmación de pago y completar pedido

## Requisitos

- **WooCommerce POS**: Versión gratuita o profesional instalada y activada
- **Sistema de Correo Electrónico**: Configuración de correo electrónico de WordPress en funcionamiento
- **Pasarelas Web de Pago**: Al menos un método de pago en línea habilitado en WooCommerce
- **Certificado SSL**: HTTPS requerido para procesamiento de pagos seguro

## Resolución de Problemas

### Problemas Comunes

**Correos Electrónicos No se Envían**
- Verificar la configuración de correo electrónico de WordPress
- Comprobar la configuración SMTP si se utiliza un servicio de correo electrónico personalizado
- Probar con plugins de prueba de correo electrónico de WordPress
- Revisar carpetas de spam para correos electrónicos de prueba

**Los Enlaces de Pago No Funcionan**
- Asegurarse de que WooCommerce esté configurado correctamente
- Verificar que al menos una pasarela de pago esté habilitada para pago web
- Comprobar que el certificado SSL esté instalado correctamente
- Confirmar que el estado del pedido permite el pago

**El Cliente No Puede Completar el Pago**
- Verificar que las pasarelas de pago web estén activas y configuradas
- Comprobar que el pedido no haya expirado o sido cancelado
- Asegurarse de que el cliente tenga límites suficientes en su método de pago
- Probar el proceso de pago usted mismo

### Entrega de Correo Electrónico

Para una entrega de correo electrónico confiable, considere:

- **Servicio SMTP**: Utilizar servicios como SendGrid, Mailgun o Amazon SES
- **Plugins de Correo Electrónico**: Instalar plugins SMTP de WordPress para mejor entrega
- **Autenticación de Dominio**: Configurar registros SPF, DKIM y DMARC
- **Monitoreo**: Rastrear la entrega y las tasas de apertura de correos electrónicos
