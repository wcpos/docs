# Pasarela de Factura por Email

La pasarela de factura por email permite enviar facturas de pago a los clientes por correo electrónico directamente desde WooCommerce POS. Esto es perfecto para situaciones en las que los clientes prefieren pagar más tarde o necesitan completar el pago en línea utilizando los métodos de pago de tu tienda en línea.

## Características

- **Integración de Email**: Envía facturas directamente desde la interfaz del POS
- **Gestión de Clientes**: Pre-poblar automáticamente el correo electrónico a partir de los datos del cliente existentes o agregar nuevas direcciones de correo electrónico
- **Órdenes de Invitados**: Soporte para clientes invitados con actualizaciones opcionales de la dirección de facturación
- **Pago en Línea**: Los clientes reciben un enlace "Pagar por este pedido" para completar el pago en línea
- **Flujo de Trabajo Flexible**: Completa pedidos en la tienda o permite la finalización del pago remoto

## Instalación

### Paso 1: Descargar e Instalar

1. Visita la [página de lanzamientos de la pasarela de factura por email](https://github.com/wcpos/email-invoice-gateway/releases)
2. Descarga el archivo **woocommerce-pos-email-invoice-gateway.zip** más reciente
3. En tu administración de WordPress, ve a `Plugins > Añadir Nuevo > Cargar Plugin`
4. Carga el archivo zip y activa el plugin

### Paso 2: Habilitar en WooCommerce POS

1. Ve a `WP Admin > WooCommerce POS > Configuración > Pago`
2. Busca la **Pasarela de Pago de Factura de WooCommerce POS** en la lista
3. Habilita la pasarela para su uso en el POS
4. Guarda tus configuraciones

:::note
Esta pasarela no requiere claves API adicionales ni configuración de servicios externos. Utiliza tu sistema de correo electrónico de WordPress existente y los métodos de pago de WooCommerce.
:::

## Uso

### Envío de Facturas

1. **Agregar Artículos**: Agrega productos a tu carrito en el POS
2. **Seleccionar Pasarela**: Elige "Factura por Email" como método de pago
3. **Dirección de Correo Electrónico**:
   - Para clientes existentes: El correo electrónico se llena automáticamente a partir de los datos del cliente
   - Para órdenes de invitados: Ingresa la dirección de correo electrónico del cliente manualmente
4. **Opcional**: Marca "Guardar en la dirección de facturación" para actualizar la información de facturación del cliente
5. **Enviar Factura**: Completa el proceso para enviar el correo electrónico de la factura

### Experiencia del Cliente

Cuando envías una factura, el cliente recibe un correo electrónico que contiene:

- **Detalles del Pedido**: Desglose completo de artículos, cantidades y precios
- **Total del Pedido**: Monto final a pagar incluyendo impuestos y tarifas
- **Enlace de Pago**: Enlace directo para completar el pago en línea
- **Información del Pedido**: Número de pedido y detalles de referencia

### Finalización del Pago

Los clientes pueden completar el pago de dos maneras:

1. **Pago en Línea**: Hacer clic en el enlace "Pagar por este pedido" en el correo electrónico
2. **Devolución en Tienda**: Regresar para completar el pago utilizando otros métodos de pago del POS

El enlace "Pagar por este pedido" lleva a los clientes a una página de pago dedicada en tu sitio web donde pueden utilizar cualquiera de tus pasarelas de pago web habilitadas (Stripe, PayPal, etc.).

## Gestión de Correo Electrónico

### Clientes Existentes

- Las direcciones de correo electrónico se recuperan automáticamente de los perfiles de cliente
- La información de facturación del cliente se preserva
- El historial de pedidos se mantiene bajo la cuenta del cliente

### Clientes Invitados

- Ingresa manualmente direcciones de correo electrónico para clientes únicos
- Opción de guardar el correo electrónico en la dirección de facturación para referencia futura
- Las órdenes de invitados se rastrean correctamente en WooCommerce

### Plantillas de Correo Electrónico

La pasarela utiliza las plantillas de correo electrónico estándar de WooCommerce:

- **Personalizable**: Modifica la apariencia del correo electrónico a través de la configuración de correo electrónico de WooCommerce
- **De Marca**: Incluye tu logotipo y marca de la tienda
- **Profesional**: Formato de factura limpio y profesional
- **Adaptable a Móviles**: Diseño responsivo para todos los dispositivos

## Casos de Uso

### Perfecto Para

- **Ventas B2B**: Clientes comerciales que necesitan procesar pagos a través de sus sistemas contables
- **Grandes Pedidos**: Transacciones de alto valor que requieren aprobación o tiempo de procesamiento
- **Clientes Remotos**: Clientes que necesitan completar el pago después de salir de la tienda
- **Clientes de Cuenta**: Clientes habituales con términos de pago establecidos
- **Cotización a Pedido**: Convertir cotizaciones o estimaciones en facturas pagaderas

### Ejemplos de Flujo de Trabajo

**Tienda Minorista**
1. El cliente selecciona artículos pero necesita obtener aprobación para la compra
2. Envía un correo electrónico de factura con todos los detalles
3. El cliente completa el pago en línea cuando esté listo
4. El pedido se actualiza automáticamente en tu sistema

**Negocio de Servicios**
1. Completa el trabajo de servicio y agrega artículos al POS
2. Envía la factura al departamento contable del cliente
3. El cliente paga en línea usando el método de pago preferido
4. Recibe la confirmación de pago y completa el pedido

## Requisitos

- **WooCommerce POS**: Versión gratuita o Pro instalada y activada
- **Sistema de Correo Electrónico**: Configuración de correo electrónico de WordPress funcionando
- **Pasarelas de Pago Web**: Al menos un método de pago en línea habilitado en WooCommerce
- **Certificado SSL**: Se requiere HTTPS para un procesamiento seguro de pagos

## Solución de Problemas

### Problemas Comunes

**Correos Electrónicos No Enviados**
- Verifica la configuración del correo electrónico de WordPress
- Confirma la configuración SMTP si usas un servicio de correo electrónico personalizado
- Prueba con plugins de prueba de correo electrónico de WordPress
- Verifica las carpetas de spam para los correos electrónicos de prueba

**Enlaces de Pago No Funcionan**
- Asegúrate de que WooCommerce esté correctamente configurado
- Verifica que al menos una pasarela de pago esté habilitada para el pago web
- Comprueba que el certificado SSL esté correctamente instalado
- Confirma que el estado del pedido permita el pago

**El Cliente No Puede Completar el Pago**
- Verifica que las pasarelas de pago web estén activas y configuradas
- Comprueba que el pedido no haya caducado o sido cancelado
- Asegúrate de que el cliente tenga límites de método de pago suficientes
- Prueba el proceso de pago tú mismo

### Entrega de Correo Electrónico

Para una entrega de correo electrónico confiable, considera:

- **Servicio SMTP**: Utiliza servicios como SendGrid, Mailgun o Amazon SES
- **Plugins de Correo Electrónico**: Instala plugins SMTP de WordPress para una mejor entrega
- **Autenticación de Dominio**: Configura registros SPF, DKIM y DMARC
- **Monitoreo**: Realiza un seguimiento de la entrega de correos y tasas de apertura
