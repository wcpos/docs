# Plantilla de Pasarela

La Plantilla de Pasarela de WooCommerce proporciona un punto de partida para crear tus propias pasarelas de pago personalizadas para WooCommerce POS. Esta plantilla incluye todo el código base necesario y la estructura para construir una pasarela de pago completamente funcional.

## Características

- **Plantilla Completa**: Estructura de pasarela lista para usar con todos los métodos requeridos
- **Integración POS**: Pre-configurada para compatibilidad con WooCommerce POS
- **Configuración Automatizada**: Personalización de plantilla basada en script
- **Mejores Prácticas**: Sigue los estándares de codificación de WordPress y WooCommerce
- **Extensible**: Fácil de modificar y extender para proveedores de pago específicos

## Primeros Pasos

### Opción 1: Generación Automatizada de Plantilla

La plantilla incluye un script que personaliza automáticamente la plantilla para tu pasarela específica:

1. **Clonar el Repositorio**:
   ```bash
   git clone https://github.com/wcpos/woocommerce-gateway-template.git
   cd woocommerce-gateway-template
   ```

2. **Ejecutar el Script de Configuración**:
   ```bash
   ./create-gateway.sh
   ```

3. **Seguir los Indicadores**:
   - Ingresa el nombre de tu pasarela (por ejemplo, "Mi Pasarela de Pago")
   - Ingresa un slug de pasarela (por ejemplo, "mi-pago")
   - Proporciona una descripción
   - El script generará un complemento personalizado

### Opción 2: Uso Manual de la Plantilla

Si prefieres personalización manual:

1. **Descargar la Plantilla**:
   - Visita el [repositorio de la Plantilla de Pasarela](https://github.com/wcpos/woocommerce-gateway-template)
   - Descarga la última versión o clona el repositorio

2. **Personalizar la Plantilla**:
   - Reemplaza `{{GATEWAY_NAME}}` con el nombre de visualización de tu pasarela
   - Reemplaza `{{GATEWAY_SLUG}}` con el identificador único de tu pasarela
   - Reemplaza `{{GATEWAY_DESCRIPTION}}` con la descripción de tu pasarela

3. **Renombrar Archivos**:
   - Renombra `wcpos-{{GATEWAY_SLUG}}.php` para que coincida con el slug de tu pasarela
   - Actualiza los encabezados de los archivos e información del complemento
