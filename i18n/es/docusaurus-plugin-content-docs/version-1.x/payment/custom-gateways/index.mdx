# Pasarelas de Pago Personalizadas

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos de POS y proporcionan una integración fluida con varios métodos y servicios de pago.

## Pasarelas Personalizadas Disponibles

### Pasarelas de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Aceptar pagos utilizando lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesar pagos a través de lectores de tarjetas SumUp

### Pasarelas de Pago Digital

- **[Factura por Correo Electrónico](./email-invoice)** - Enviar facturas de pago a los clientes por correo electrónico
- **[Pago Web](./web-checkout)** - Redirigir a los clientes para completar los pagos en línea

## Creando Su Propia Pasarela

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede usar nuestra **[Plantilla de Pasarela](./gateway-template)** para crear su propia pasarela de pago de WooCommerce POS.

## Resumen de Instalación

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión del repositorio respectivo de GitHub
2. **Instalar** el complemento a través de `WP Admin > Plugins > Agregar Nuevo > Cargar Complemento`
3. **Configurar** la configuración de la pasarela en `WP Admin > WooCommerce > Configuración > Pagos`
4. **Habilitar** la pasarela en `WP Admin > WooCommerce POS > Configuración > Pago`
