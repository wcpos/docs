# Pasarelas de pago personalizadas {#custom-payment-gateways}

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos POS y ofrecen una integración perfecta con diversos métodos y servicios de pago.

## Pasarelas personalizadas disponibles {#available-custom-gateways}

### Pasarelas con terminal de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acepte pagos mediante lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procese pagos a través de lectores de tarjetas SumUp

### Pasarelas de pago móvil {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acepte pagos desde el teléfono mediante código QR o notificación push

### Pasarelas de pago digital {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Envíe facturas de pago a los clientes por correo electrónico
- **[Pago web](./web-checkout)** - Redirige a los clientes para que completen los pagos en línea

## Crear tu propia pasarela {#creating-your-own-gateway}

Si necesitas una pasarela de pago personalizada para tus requisitos específicos, puedes utilizar nuestra **[plantilla de pasarela](./gateway-template)** para crear tu propia pasarela de pago de WCPOS.

## Resumen de la instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descarga** la última versión desde el repositorio de GitHub correspondiente
2. **Instala** el plugin desde `WP Admin > Plugins > Add New > Upload Plugin`
3. **Configura** los ajustes de la pasarela en `WP Admin > WooCommerce > Settings > Payments`
4. **Activa** la pasarela en `WP Admin > POS > Settings > Checkout`
