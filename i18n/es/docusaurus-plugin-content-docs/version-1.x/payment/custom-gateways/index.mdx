# Pasarelas de Pago Personalizadas {#custom-payment-gateways}

WCPOS soporta varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos POS y proporcionan una integración sin problemas con diversos métodos y servicios de pago.

## Pasarelas Personalizadas Disponibles {#available-custom-gateways}

### Pasarelas de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceptar pagos usando lectores de hardware de Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesar pagos a través de lectores de tarjetas SumUp

### Pasarelas de Pago Móvil {#digital-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceptar pagos basados en teléfonos a través de código QR o notificación push

### Pasarelas de Pago Digital

- **[Factura por Correo Electrónico](./email-invoice)** - Enviar facturas de pago a los clientes a través de correo electrónico
- **[Pago Web](./web-checkout)** - Redirigir a los clientes para completar los pagos en línea

## Creando Su Propia Pasarela {#creating-your-own-gateway}

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede usar nuestra **[Plantilla de Pasarela](./gateway-template)** para crear su propia pasarela de pago de WooCommerce POS.

## Resumen de Instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión del respectivo repositorio de GitHub
2. **Instalar** el plugin a través de `WP Admin > Plugins > Añadir Nuevo > Subir Plugin`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Configuración > Pagos`
4. **Habilitar** la pasarela en `WP Admin > WooCommerce POS > Configuración > Pago`
