# Pasarelas de Pago Personalizadas

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos de POS y proporcionan integración fluida con varios métodos y servicios de pago.

## Pasarelas Personalizadas Disponibles

### Pasarelas de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Aceptar pagos utilizando lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesar pagos a través de lectores de tarjetas SumUp

### Pasarelas de Pago Digitales

- **[Factura por Correo Electrónico](./email-invoice)** - Enviar facturas de pago a los clientes por correo electrónico
- **[Pago en la Web](./web-checkout)** - Redirigir a los clientes para completar los pagos en línea

## Creando Su Propia Pasarela

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede utilizar nuestra **[Plantilla de Pasarela](./gateway-template)** para crear su propia pasarela de pago para WooCommerce POS.

## Resumen de Instalación

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión desde el repositorio de GitHub correspondiente
2. **Instalar** el plugin a través de `WP Admin > Plugins > Agregar Nuevo > Subir Plugin`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Activar** la pasarela en `WP Admin > WooCommerce POS > Ajustes > Pago`

## Requisitos

- Plugin de WooCommerce POS (versión gratuita o pro)
- WordPress con WooCommerce instalado
- Claves API o credenciales apropiadas para el servicio de pago específico

## Soporte

Para problemas con pasarelas personalizadas, visite el repositorio de GitHub correspondiente y cree un problema con información detallada sobre su problema.
