# Pasarelas de Pago Personalizadas {#custom-payment-gateways}

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos de POS y proporcionan una integración fluida con varios métodos y servicios de pago.

## Pasarelas Personalizadas Disponibles {#available-custom-gateways}

### Pasarelas de Terminal de Hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acepta pagos utilizando lectores de hardware de Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesa pagos a través de lectores de tarjetas SumUp

### Pasarelas de Pago Móvil {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acepta pagos basados en teléfonos a través de código QR o notificaciones push

### Pasarelas de Pago Digital {#digital-payment-gateways}

- **[Factura por Email](./email-invoice)** - Envía facturas de pago a los clientes por correo electrónico
- **[Pago Web](./web-checkout)** - Redirige a los clientes para completar los pagos en línea

## Creando Su Propia Pasarela {#creating-your-own-gateway}

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede usar nuestra **[Plantilla de Pasarela](./gateway-template)** para crear su propia pasarela de pago WCPOS.

## Resumen de Instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión desde el repositorio de GitHub correspondiente
2. **Instalar** el complemento a través de `WP Admin > Plugins > Agregar Nuevo > Cargar Plugin`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Activar** la pasarela en `WP Admin > POS > Ajustes > Pago`
