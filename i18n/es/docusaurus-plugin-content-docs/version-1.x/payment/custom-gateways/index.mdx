# Pasarelas de pago personalizadas {#custom-payment-gateways}

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de punto de venta. Estas pasarelas están diseñadas específicamente para entornos de TPV y ofrecen una integración fluida con diversos métodos y servicios de pago.

## Pasarelas personalizadas disponibles {#available-custom-gateways}

### Pasarelas para terminales físicos {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acepte pagos utilizando lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procese pagos a través de lectores de tarjetas SumUp

### Pasarelas de pago móvil {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acepte pagos desde el teléfono mediante código QR o notificación push

### Pasarelas de pago digital {#digital-payment-gateways}

- **[Factura por correo electrónico](./email-invoice)** - Envíe facturas de pago a los clientes por correo electrónico
- **[Pago web](./web-checkout)** - Redirija a los clientes para completar los pagos en línea

## Crear su propia pasarela {#creating-your-own-gateway}

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede utilizar nuestra **[Plantilla de pasarela](./gateway-template)** para crear su propia pasarela de pago para WCPOS.

## Resumen de la instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargue** la última versión desde el repositorio de GitHub correspondiente
2. **Instale** el plugin mediante `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. **Configure** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Active** la pasarela en `WP Admin > POS > Ajustes > Finalizar compra`
