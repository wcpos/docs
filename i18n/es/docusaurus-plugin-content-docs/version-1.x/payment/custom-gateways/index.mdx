# Pasarelas de pago personalizadas {#custom-payment-gateways}

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de tu sistema de Terminal Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos de TPV y ofrecen una integración fluida con diversos métodos y servicios de pago.

## Pasarelas personalizadas disponibles {#available-custom-gateways}

### Pasarelas con terminal de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Acepta pagos utilizando los lectores de hardware de Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesa pagos mediante los lectores de tarjetas de SumUp

### Pasarelas de pago móvil {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acepta pagos desde el teléfono mediante código QR o notificación push

### Pasarelas de pago digital {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Envía facturas de pago a los clientes por correo electrónico
- **[Pago web](./web-checkout)** - Redirige a los clientes para completar los pagos en línea

## Crear tu propia pasarela {#creating-your-own-gateway}

Si necesitas una pasarela de pago personalizada para tus requisitos específicos, puedes usar nuestra **[Plantilla de pasarela](./gateway-template)** para crear tu propia pasarela de pago de WCPOS.

## Resumen de instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descarga** la última versión desde el repositorio de GitHub correspondiente
2. **Instala** el plugin desde `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. **Configura** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Activa** la pasarela en `WP Admin > TPV > Ajustes > Pago`
