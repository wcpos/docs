# Pasarelas de Pago Personalizadas

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad de su sistema de Punto de Venta. Estas pasarelas están diseñadas específicamente para entornos de POS y proporcionan una integración perfecta con diversos métodos de pago y servicios.

## Pasarelas Personalizadas Disponibles

### Pasarelas de Terminal de Hardware

- **[Stripe Terminal](./stripe-terminal)** - Acepte pagos utilizando lectores de hardware de Stripe Terminal
- **[Terminal SumUp](./sumup-terminal)** - Procese pagos a través de lectores de tarjetas SumUp

### Pasarelas de Pago Digital

- **[Factura por Correo Electrónico](./email-invoice)** - Envíe facturas de pago a clientes por correo electrónico
- **[Pago Web](./web-checkout)** - Redirija a los clientes para completar pagos en línea

## Creación de su Propia Pasarela

Si necesita una pasarela de pago personalizada para sus requisitos específicos, puede utilizar nuestra **[Plantilla de Pasarela](./gateway-template)** para crear su propia pasarela de pago para WooCommerce POS.

## Descripción General de la Instalación

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión del repositorio de GitHub correspondiente
2. **Instalar** el complemento a través de `WP Admin > Complementos > Añadir Nuevo > Subir Complemento`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Activar** la pasarela en `WP Admin > WooCommerce POS > Ajustes > Pago`

## Requisitos

- Complemento WooCommerce POS (versión gratuita o pro)
- WordPress con WooCommerce instalado
- Claves de API o credenciales apropiadas para el servicio de pago específico

## Soporte

Para problemas con pasarelas personalizadas, visite el repositorio de GitHub correspondiente y cree un informe con información detallada sobre su problema.
