WCPOS admite múltiples métodos de pago para satisfacer diferentes necesidades comerciales.

## Métodos de Pago Predeterminados {#default-payment-methods}

La versión gratuita incluye dos pasarelas de pago integradas:

### Efectivo {#cash}

- Ingresar cantidad recibida
- Cálculo automático de cambio
- No se requiere configuración adicional

### Tarjeta {#card}

- Para uso con terminales de tarjeta externos
- Simplemente marque el pago como completado después de procesarlo en su terminal
- No se requiere integración directa

## Pasarelas de Pago Adicionales (Pro) {#additional-payment-gateways-pro}

:::info Función Pro
Las pasarelas de pago adicionales requieren [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puede habilitar pasarelas de pago de WooCommerce adicionales en el proceso de pago del POS:

- **Stripe Terminal** - Integración directa con lectores de tarjetas Stripe
- **SumUp Terminal** - Integración con lectores de tarjetas SumUp
- **Cualquier Pasarela de WooCommerce** - Habilite cualquier pasarela instalada en su tienda

## Pasarelas Personalizadas {#custom-gateways}

Cree sus propias integraciones de pago utilizando el sistema de [Pasarelas Personalizadas](/payment/custom-gateways).
