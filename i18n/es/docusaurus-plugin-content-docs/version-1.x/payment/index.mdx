WCPOS admite múltiples métodos de pago para adaptarse a diferentes necesidades empresariales.

## Métodos de Pago Predeterminados

La versión gratuita incluye dos pasarelas de pago integradas:

### Efectivo

- Introducir monto entregado
- Cálculo automático de cambio
- No requiere configuración adicional

### Tarjeta

- Para usar con terminales de tarjeta externos
- Simplemente marcar el pago como completado después de procesarlo en su terminal
- No se requiere integración directa

## Pasarelas de Pago Adicionales (Pro)

:::info Función de Pro
Las pasarelas de pago adicionales requieren [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puede habilitar pasarelas de pago adicionales de WooCommerce en el checkout del POS:

- **Stripe Terminal** - Integración directa con lectores de tarjetas Stripe
- **SumUp Terminal** - Integración con lectores de tarjetas SumUp
- **Cualquier Pasarela de WooCommerce** - Habilitar cualquier pasarela instalada en su tienda

## Pasarelas Personalizadas

Cree sus propias integraciones de pago utilizando el sistema de [Pasarelas Personalizadas](/payment/custom-gateways).
