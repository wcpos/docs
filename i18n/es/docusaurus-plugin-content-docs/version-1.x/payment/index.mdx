WCPOS admite múltiples métodos de pago para adaptarse a diferentes necesidades comerciales.

## Métodos de Pago Predeterminados

La versión gratuita incluye dos pasarelas de pago integradas:

### Efectivo

- Introducir el monto entregado
- Cálculo automático del cambio
- No se requiere configuración adicional

### Tarjeta

- Para uso con terminales de tarjetas externos
- Simplemente marque el pago como completo después de procesarlo en su terminal
- No se requiere integración directa

## Pasarelas de Pago Adicionales (Pro)

:::info Característica Pro
Las pasarelas de pago adicionales requieren [WCPOS Pro](/getting-started/pro-license).
:::

Con Pro, puede habilitar pasarelas de pago adicionales de WooCommerce en el proceso de pago del POS:

- **Terminal de Stripe** - Integración directa con lectores de tarjetas Stripe
- **Terminal de SumUp** - Integración con lectores de tarjetas SumUp
- **Cualquier Pasarela de WooCommerce** - Habilitar cualquier pasarela instalada en su tienda

## Pasarelas Personalizadas

Cree sus propias integraciones de pago utilizando el sistema de [Pasarelas Personalizadas](/payment/custom-gateways).
