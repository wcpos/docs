# Pasarelas de pago {#payment-gateways}

WCPOS admite varias pasarelas de pago personalizadas que amplían la funcionalidad del sistema de punto de venta. Estas pasarelas se han diseñado específicamente para entornos de POS y ofrecen una integración fluida con diversos métodos y servicios de pago.

## Pasarelas personalizadas disponibles {#available-custom-gateways}

### Pasarelas para terminales de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Permite aceptar pagos mediante lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Permite procesar pagos mediante lectores de tarjetas SumUp
- **[Square Terminal](./square-terminal)** - Permite cobrar pagos presenciales en dispositivos Square Terminal
- **[PayPal Reader (Zettle)](./paypal-reader)** - Permite aceptar pagos con tarjeta presenciales en un terminal PayPal Reader (Zettle)
- **[Mollie Terminal](./mollie-terminal)** - Permite aceptar pagos presenciales en dispositivos Mollie Terminal

### Pasarelas de pago móviles {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Acepte pagos desde teléfonos mediante código QR o notificación push

### Pasarelas de pago digitales {#digital-payment-gateways}

- **[Factura por correo electrónico](./email-invoice)** - Envíe facturas de pago a los clientes por correo electrónico
- **[Pago web](./web-checkout)** - Redirija a los clientes para completar los pagos en línea

## Creación de una pasarela propia {#creating-your-own-gateway}

Si necesita una pasarela de pago personalizada para requisitos específicos, puede utilizar nuestra **[Plantilla de pasarela](/reference/gateway-template)** para crear su propia pasarela de pago WCPOS.

## Descripción general de la instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión del repositorio de GitHub correspondiente
2. **Instalar** el plugin mediante `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Habilite** la pasarela en `WP Admin > POS > Ajustes > Finalizar compra`
