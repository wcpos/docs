# Pasarelas de pago {#payment-gateways}

WCPOS es compatible con varias pasarelas de pago personalizadas que amplían la funcionalidad del sistema de punto de venta. Estas pasarelas están diseñadas específicamente para entornos POS y proporcionan una integración fluida con diversos métodos de pago y servicios.

## Pasarelas personalizadas disponibles {#available-custom-gateways}

### Pasarelas de terminal de hardware {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Aceptar pagos con lectores de hardware Stripe Terminal
- **[SumUp Terminal](./sumup-terminal)** - Procesar pagos a través de lectores de tarjetas SumUp
- **[PayPal Reader (Zettle)](./paypal-reader)** - Recibir pagos con tarjeta en persona mediante un terminal PayPal Reader (Zettle)

### Pasarelas de pago móvil {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Aceptar pagos mediante teléfono con código QR o notificación push

### Pasarelas de pago digitales {#digital-payment-gateways}

- **[Factura por correo electrónico](./email-invoice)** - Envío de facturas de pago a los clientes por correo electrónico
- **[Pago en línea](./web-checkout)** - Redirección de los clientes para completar los pagos en línea

## Creación de una pasarela propia {#creating-your-own-gateway}

Si se necesita una pasarela de pago personalizada para requisitos específicos, es posible utilizar nuestra **[Plantilla de pasarela](/reference/gateway-template)** para crear una pasarela de pago de WCPOS propia.

## Descripción general de la instalación {#installation-overview}

Todas las pasarelas personalizadas siguen un proceso de instalación similar:

1. **Descargar** la última versión desde el repositorio de GitHub correspondiente
2. **Instalar** el plugin a través de `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. **Configurar** los ajustes de la pasarela en `WP Admin > WooCommerce > Ajustes > Pagos`
4. **Activar** la pasarela en `WP Admin > POS > Ajustes > Finalizar compra`
