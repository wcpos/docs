Cada artículo en el carrito es un artículo de línea que se puede editar directamente. WCPOS ofrece edición en línea flexible para cantidades, precios y otros detalles.

## Tipos de Artículos de Línea {#line-item-types}

WooCommerce utiliza tres tipos de artículos de línea:

| Tipo | Descripción | Ejemplos |
|------|-------------|----------|
| **line_item** | Productos de su catálogo | Camiseta, Taza de café |
| **fee_line** | Tarifas adicionales | Envoltura de regalo, Cargo por servicio |
| **shipping_line** | Costos de envío | Envío estándar, Entrega exprés |

:::note
Los costos de envío se ingresan manualmente. El POS no calcula las tarifas de envío automáticamente.
:::

## Editando Artículos de Línea {#editing-line-items}

### Cantidad {#quantity}

Haga clic en el campo de cantidad para editarlo directamente. Introduzca un nuevo número o utilice los controles de ajuste.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nombre del Producto {#product-name}

Para productos diversos y tarifas, puede editar el nombre directamente haciendo clic en él.

### Precio {#price}

Haga clic en el campo de precio para cambiar el precio unitario. Esto es útil para:

- Aplicar descuentos manuales
- Igualar precios
- Corregir errores de precios

El total se actualiza automáticamente según la cantidad × precio.

### Menú de Tres Puntos {#three-dot-menu}

Haga clic en el **⋮** (tres puntos) en cualquier artículo de línea para acceder a opciones adicionales:

{/* Screenshot: cart-line-item-menu.png */}

#### Formulario de Edición {#edit-form}

Abre un formulario de edición detallado para el artículo de línea, permitiéndole modificar todos los campos.

#### Vista JSON Sin Procesar {#raw-json-view}

Muestra los datos JSON sin procesar para el artículo de línea. Esto es extremadamente útil para:

- **Depuración** - Ver exactamente qué datos se están enviando a WooCommerce
- **Solución de Problemas** - Identificar problemas con campos personalizados o meta data
- **Desarrollo** - Comprender la estructura de datos para integraciones

### Eliminar Artículo {#remove-item}

Haga clic en el botón rojo **×** para eliminar un artículo del carrito.

## Atributos de Variación {#variation-attributes}

Para productos variables, los atributos de variación seleccionados se muestran debajo del nombre del producto:

```
Hoodie
Colour: Green
Size: Large
```

Estos atributos son parte de los datos del artículo de línea y se incluyen en el pedido.

## Meta Data

Si ha configurado [Claves de Meta Data](/pos/product-panel#meta-data-keys) en la configuración del Panel de Productos, esos datos se copian automáticamente al artículo de línea cuando se agrega el producto al carrito.

Puede ver estos meta data en la vista JSON sin procesar.

## Dividiendo Artículos de Línea {#splitting-line-items}

Si está habilitado en [Configuraciones de Visualización](/pos/cart#display-settings), puede dividir un artículo de línea en múltiples líneas. Esto es útil cuando un cliente desea aplicar diferentes descuentos a porciones del mismo producto.

## Consejos {#tips}

### Ajustes Rápidos de Precio {#quick-price-adjustments}

Para dar rápidamente un descuento:

1. Haga clic en el campo de precio
2. Introduzca el nuevo precio
3. Presione Enter

### Navegación por Teclado {#keyboard-navigation}

- **Tab** - Moverse entre campos editables
- **Enter** - Confirmar edición
- **Escape** - Cancelar edición

### Ediciones por Lotes {#batch-edits}

Para pedidos complejos, considere usar el menú de tres puntos para acceder al formulario de edición completo, que muestra todos los campos a la vez.
