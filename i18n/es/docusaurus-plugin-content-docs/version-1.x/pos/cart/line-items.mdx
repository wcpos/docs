Cada artículo en el carrito es un artículo de línea que se puede editar directamente. WCPOS proporciona edición en línea flexible para cantidades, precios y otros detalles.

## Tipos de Artículos de Línea

WooCommerce utiliza tres tipos de artículos de línea:

| Tipo | Descripción | Ejemplos |
|------|-------------|----------|
| **line_item** | Productos de su catálogo | Camiseta, Taza de café |
| **fee_line** | Cargos adicionales | Envoltura de regalo, Cargo por servicio |
| **shipping_line** | Costos de envío | Envío estándar, Entrega exprés |

:::note
Los costos de envío se ingresan manualmente. El POS no calcula las tarifas de envío automáticamente.
:::

## Editar Artículos de Línea

### Cantidad

Haga clic en el campo de cantidad para editarlo directamente. Escriba un nuevo número o use los controles de ajuste.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nombre del Producto

Para productos y cargos diversos, puede editar el nombre directamente haciendo clic en él.

### Precio

Haga clic en el campo de precio para cambiar el precio unitario. Esto es útil para:

- Aplicar descuentos manuales
- Igualar precios
- Corregir errores de precios

El total se actualiza automáticamente en función de cantidad × precio.

### Menú de Tres Puntos

Haga clic en el **⋮** (tres puntos) en cualquier artículo de línea para acceder a opciones adicionales:

{/* Screenshot: cart-line-item-menu.png */}

#### Formulario de Edición

Abre un formulario de edición detallado para el artículo de línea, permitiéndole modificar todos los campos.

#### Vista JSON en Crudo

Muestra los datos JSON crudos para el artículo de línea. Esto es extremadamente útil para:

- **Depuración** - Ver exactamente qué datos se están enviando a WooCommerce
- **Solución de Problemas** - Identificar problemas con campos personalizados o metadatos
- **Desarrollo** - Comprender la estructura de datos para integraciones

### Eliminar Artículo

Haga clic en el botón rojo **×** para eliminar un artículo del carrito.

## Atributos de Variación

Para productos variables, los atributos de variación seleccionados se muestran debajo del nombre del producto:

```
Hoodie
Colour: Green
Size: Large
```

Estos atributos son parte de los datos del artículo de línea y se incluyen en el pedido.

## Metadatos

Si ha configurado [Claves de Metadatos](/pos/product-panel#meta-data-keys) en la configuración del Panel de Productos, esos datos se copian automáticamente al artículo de línea cuando el producto se agrega al carrito.

Puede ver estos metadatos en la vista JSON en crudo.

## División de Artículos de Línea

Si está habilitado en [Configuraciones de Pantalla](/pos/cart#display-settings), puede dividir un artículo de línea en múltiples líneas. Esto es útil cuando un cliente desea aplicar diferentes descuentos a porciones del mismo producto.

## Consejos

### Ajustes Rápidos de Precio

Para dar rápidamente un descuento:

1. Haga clic en el campo de precio
2. Ingrese el nuevo precio
3. Presione Enter

### Navegación por Teclado

- **Tab** - Moverse entre campos editables
- **Enter** - Confirmar edición
- **Escape** - Cancelar edición

### Ediciones por Lotes

Para pedidos complejos, considere usar el menú de tres puntos para acceder al formulario de edición completo, que muestra todos los campos a la vez.
