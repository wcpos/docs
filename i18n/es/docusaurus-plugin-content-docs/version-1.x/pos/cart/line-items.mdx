Cada elemento en el carrito es un elemento de línea que se puede editar directamente. WCPOS ofrece edición flexible en línea para cantidades, precios y otros detalles.

## Tipos de elementos de línea

WooCommerce utiliza tres tipos de elementos de línea:

| Tipo | Descripción | Ejemplos |
|------|-------------|----------|
| **line_item** | Productos de su catálogo | Camiseta, Taza de café |
| **fee_line** | Cargos adicionales | Envoltorio de regalo, Cargo por servicio |
| **shipping_line** | Costos de envío | Envío estándar, Entrega express |

:::note
Los costos de envío se ingresan manualmente. El POS no calcula las tarifas de envío automáticamente.
:::

## Editando elementos de línea

### Cantidad

Haga clic en el campo de cantidad para editarlo directamente. Escriba un nuevo número o use los controles de incremento.

{/* Screenshot: cart-line-item-edit-qty.png */}

### Nombre del producto

Para productos varios y cargos, puede editar el nombre directamente haciendo clic en él.

### Precio

Haga clic en el campo de precio para cambiar el precio unitario. Esto es útil para:

- Aplicar descuentos manuales
- Igualar precios
- Corregir errores de precios

El total se actualiza automáticamente según cantidad × precio.

### Menú de tres puntos

Haga clic en **⋮** (tres puntos) en cualquier elemento de línea para acceder a opciones adicionales:

{/* Screenshot: cart-line-item-menu.png */}

#### Formulario de edición

Abre un formulario de edición detallado para el elemento de línea, permitiéndole modificar todos los campos.

#### Vista JSON sin procesar

Muestra los datos JSON sin procesar del elemento de línea. Esto es extremadamente útil para:

- **Depuración** - Ver exactamente qué datos se envían a WooCommerce
- **Resolución de problemas** - Identificar problemas con campos personalizados o metadatos
- **Desarrollo** - Comprender la estructura de datos para integraciones

### Eliminar elemento

Haga clic en el botón rojo **×** para eliminar un elemento del carrito.

## Atributos de variación

Para productos variables, los atributos de variación seleccionados se muestran debajo del nombre del producto:

```
Sudadera
Color: Verde
Talla: Grande
```

Estos atributos son parte de los datos del elemento de línea y se incluyen en el pedido.

## Metadatos

Si ha configurado [Claves de metadatos](/pos/product-panel#meta-data-keys) en la configuración del Panel de Producto, esos datos se copian automáticamente al elemento de línea cuando se agrega el producto al carrito.

Puede ver estos metadatos en la vista JSON sin procesar.

## Dividir elementos de línea

Si está habilitado en [Configuración de visualización](/pos/cart#display-settings), puede dividir un elemento de línea en múltiples líneas. Esto es útil cuando un cliente quiere aplicar diferentes descuentos a porciones del mismo producto.

## Consejos

### Ajustes rápidos de precio

Para dar un descuento rápidamente:

1. Haga clic en el campo de precio
2. Ingrese el nuevo precio
3. Presione Intro

### Navegación con teclado

- **Tab** - Moverse entre campos editables
- **Enter** - Confirmar edición
- **Escape** - Cancelar edición

### Ediciones por lotes

Para pedidos complejos, considere usar el menú de tres puntos para acceder al formulario de edición completo, que muestra todos los campos a la vez.
