El Panel de Productos es el lado izquierdo de la pantalla del POS donde busca, navega y selecciona productos para agregar al carrito.

{/* Screenshot: pos-product-panel.png */}

## Descripción General de la Interfaz

### Barra de Búsqueda

En la parte superior del panel, la barra de búsqueda le permite encontrar productos rápidamente por:

- **Nombre del producto** - Escriba cualquier parte del nombre del producto
- **SKU** - Busque por Unidad de Mantenimiento de Stock
- **Código de barras** - Escanee o escriba un número de código de barras

Consulte [Búsqueda y Filtrado](/pos/product-panel/search-filtering) para información detallada.

### Configuración de Visualización

Haga clic en el **icono de deslizadores** (<Icon name="sliders" />) junto a la barra de búsqueda para abrir la Configuración de Visualización. Esto le permite personalizar qué columnas e información se muestran en la lista de productos.

### Botones de Filtro

Debajo de la barra de búsqueda, los botones de filtro le permiten reducir la selección de productos:

- **En Stock** - Mostrar solo productos con inventario disponible
- **Destacados** - Mostrar productos marcados como destacados en WooCommerce
- **En Venta** - Mostrar productos actualmente en venta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si utiliza un complemento de marca)

Los filtros activos aparecen resaltados. Haga clic en un filtro nuevamente para eliminarlo.

### Lista de Productos

El área principal muestra sus productos en una lista desplazable. Cada fila muestra:

- **Imagen del producto** - Miniatura del producto
- **Nombre del producto** - Con detalles opcionales (stock, SKU, categorías, etc.)
- **Precio** - Precio de venta actual (con precios de oferta mostrados cuando sea aplicable)
- **Botón de acción** - Agregar al carrito (<Icon name="circle-plus" color="green" />) o seleccionar variación (<Icon name="circle-arrow-right" color="green" />)

### Tipos de Producto

- **Productos simples** muestran un botón verde <Icon name="circle-plus" color="green" /> para agregar directamente al carrito
- **Productos variables** muestran una flecha verde <Icon name="circle-arrow-right" color="green" /> para abrir el selector de variaciones

Consulte [Productos Variables](/pos/product-panel/variations) para más detalles.

### Pie de Página

En la parte inferior del Panel de Productos:

- **Estado fiscal** - Muestra la base de cálculo de impuestos actual (p. ej., "Impuesto basado en: Dirección base de la tienda"). Haga clic para ver las tasas de impuestos activas.
- **Conteo de productos** - Muestra cuántos productos están exhibidos (p. ej., "Mostrando 10 de 17")
- **Botón de sincronización** (<Icon name="arrow-rotate-right" />) - Refrescar productos desde el servidor. **Presione prolongadamente** para opciones adicionales:
  - **Sincronización** - Actualización estándar desde el servidor
  - **Limpiar y Refrescar** - Limpiar datos locales y recargar todo

### Mostrar Productos Agotados

Controla si los productos que actualmente están agotados se muestran.

- **Habilitado**: Los productos agotados permanecen visibles pero no se pueden agregar al carrito
- **Deshabilitado**: Los productos agotados están ocultos de la lista

### Columnas

Configure qué columnas aparecen en la lista de productos:

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|--------------------------|
| **Imagen** | Miniatura del producto | - |
| **Producto** | Nombre del producto y detalles | Stock, SKU, Código de barras, Categorías, Etiquetas, Marcas, Atributos, Metadatos |
| **SKU** | Unidad de Mantenimiento de Stock (columna separada) | - |
| **Código de barras** | Código de barras del producto (columna separada) | - |
| **Tipo** | Tipo de producto (simple, variable, etc.) | - |
| **Stock** | Cantidad de stock actual | - |
| **Costo de los Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta | Información de impuestos, Indicador de En Venta |
| **Acciones** | Botón Agregar al carrito | - |

### Claves de Meta Datos

Especifique las claves de meta datos del producto que deben transferirse a los artículos de línea de pedido cuando los productos se agregan al carrito.

Ingrese una lista de claves de meta datos separadas por comas:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Casos de uso:**
- Complementos de depósito de botella
- Campos de producto personalizados
- Datos de integración de terceros
- Información de cumplimiento/seguimiento

:::note
Las claves de meta datos son sensibles a mayúsculas y minúsculas y deben coincidir exactamente con lo almacenado en los meta datos del producto.
:::

### Restaurar Configuraciones Predeterminadas

Haga clic para restablecer todas las configuraciones de visualización a sus valores predeterminados originales.

## Agregar Productos al Carrito

### Productos Simples

Haga clic en el botón verde <Icon name="circle-plus" color="green" /> para agregar un producto simple al carrito. Cada clic agrega una unidad más.

### Productos Variables

Los productos variables (p. ej., una camiseta con opciones de tamaño y color) muestran una flecha verde <Icon name="circle-arrow-right" color="green" />. Haga clic para:

1. **Popover rápido** - Seleccionar variación de un menú desplegable
2. **Expandir en línea** - Haga clic en "Expandir" para ver todas las variaciones en la lista

Consulte [Productos Variables](/pos/product-panel/variations) para más detalles.

### Escaneo de Códigos de Barras

Conecte un escáner de códigos de barras USB o Bluetooth para agregar productos rápidamente. Cuando escanea un código de barras, WCPOS busca y agrega automáticamente el producto correspondiente.

Consulte [Escaneo de Códigos de Barras](/pos/product-panel/barcode-scanning) para instrucciones de configuración.

## ¿Por Qué No Puedo Ver Algunos Productos?

Si faltan productos en su Panel de Productos, verifique:

- **Visibilidad del POS** - Los productos configurados como "Solo en línea" no aparecerán en el POS. Consulte [Productos Solo en el POS](/products/pos-only-products).
- **Configuraciones de inventario** - Si "Mostrar Productos Agotados" está deshabilitado en Configuración de Visualización, los artículos fuera de stock están ocultos.
- **Estado de sincronización** - Los productos nuevos o actualizados recientemente pueden necesitar una sincronización. Presione prolongadamente el botón de sincronización y seleccione **"Limpiar y Refrescar"**.
- **Filtros** - Verifique si tiene filtros activos (Categoría, Etiqueta, etc.) que podrían estar ocultando productos.
