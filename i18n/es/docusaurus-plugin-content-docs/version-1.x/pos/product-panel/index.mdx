El Panel de Productos es el lado izquierdo de la pantalla del TPV donde puedes buscar, navegar y seleccionar productos para añadir al carrito.

{/* Screenshot: pos-product-panel.png */}

## Vista General de la Interfaz

### Barra de Búsqueda

En la parte superior del panel, la barra de búsqueda te permite encontrar productos rápidamente por:

- **Nombre del producto** - Escribe cualquier parte del nombre del producto
- **SKU** - Busca por Unidad de Mantenimiento de Stock
- **Código de barras** - Escanea o escribe un número de código de barras

Consulta [Búsqueda y Filtrado](/pos/product-panel/search-filtering) para información detallada.

### Configuración de Visualización

Haz clic en el **ícono de deslizadores** (<Icon name="sliders" />) junto a la barra de búsqueda para abrir la Configuración de Visualización. Esto te permite personalizar qué columnas e información se muestran en la lista de productos.

### Botones de Filtro

Debajo de la barra de búsqueda, los botones de filtro te permiten reducir la cantidad de productos:

- **En stock** - Mostrar solo productos con inventario disponible
- **Destacados** - Mostrar productos marcados como destacados en WooCommerce
- **En oferta** - Mostrar productos actualmente en oferta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si se usa un plugin de marcas)

Los filtros activos aparecen resaltados. Haz clic en un filtro nuevamente para eliminarlo.

### Lista de Productos

El área principal muestra tus productos en una lista desplazable. Cada fila muestra:

- **Imagen del producto** - Miniatura del producto
- **Nombre del producto** - Con detalles opcionales (stock, SKU, categorías, etc.)
- **Precio** - Precio de venta actual (con precios en oferta mostrados cuando corresponde)
- **Botón de acción** - Añadir al carrito (<Icon name="circle-plus" color="green" />) o seleccionar variación (<Icon name="circle-arrow-right" color="green" />)

### Tipos de Productos

- **Productos simples** muestran un botón verde <Icon name="circle-plus" color="green" /> para agregar directamente al carrito
- **Productos variables** muestran una flecha verde <Icon name="circle-arrow-right" color="green" /> para abrir el selector de variaciones

Consulta [Productos Variables](/pos/product-panel/variations) para más detalles.

### Pie de Página

En la parte inferior del Panel de Productos:

- **Estado de impuestos** - Muestra la base de cálculo de impuestos actual (por ejemplo, "Impuesto basado en: Dirección base de la tienda"). Haz clic para ver las tasas de impuestos activas.
- **Conteo de productos** - Muestra cuántos productos se están mostrando (por ejemplo, "Mostrando 10 de 17")
- **Botón de sincronización** - Actualiza los productos desde el servidor

### Mostrar Productos sin Stock

Controla si los productos que actualmente están agotados se muestran.

- **Habilitado**: Los productos sin stock permanecen visibles pero no pueden añadirse al carrito
- **Deshabilitado**: Los productos sin stock se ocultan de la lista

### Columnas

Configura qué columnas aparecen en la lista de productos:

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|---------------------------|
| **Imagen** | Miniatura del producto | - |
| **Producto** | Nombre y detalles del producto | Stock, SKU, Código de barras, Categorías, Etiquetas, Marcas, Atributos, Metadatos |
| **SKU** | Unidad de Mantenimiento de Stock (columna separada) | - |
| **Código de Barras** | Código de barras del producto (columna separada) | - |
| **Tipo** | Tipo de producto (simple, variable, etc.) | - |
| **Stock** | Cantidad de stock actual | - |
| **Costo de Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta | Información de impuestos, Indicador de oferta |
| **Acciones** | Botón para añadir al carrito | - |

### Claves de Metadatos

Especifica las claves de metadatos del producto que deben trasladarse a los elementos de línea de pedido cuando los productos se añaden al carrito.

Ingresa una lista separada por comas de claves de metadatos:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Casos de uso:**
- Plugins de depósito por botellas
- Campos de productos personalizados
- Datos de integración de terceros
- Información de cumplimiento/seguimiento

:::note
Las claves de metadatos son sensibles a mayúsculas y minúsculas y deben coincidir exactamente con cómo se almacenan en los metadatos del producto.
:::

### Restaurar Configuraciones Predeterminadas

Haz clic para restablecer todas las configuraciones de visualización a sus valores predeterminados originales.

## Añadiendo Productos al Carrito

### Productos Simples

Haz clic en el botón verde <Icon name="circle-plus" color="green" /> para añadir un producto simple al carrito. Cada clic añade una unidad más.

### Productos Variables

Los productos variables (por ejemplo, una camiseta con opciones de tamaño y color) muestran una flecha verde <Icon name="circle-arrow-right" color="green" />. Haz clic para:

1. **Popover rápido** - Seleccionar variación de un menú desplegable
2. **Expandir en línea** - Haz clic en "Expandir" para ver todas las variaciones en la lista

Consulta [Productos Variables](/pos/product-panel/variations) para más detalles.

### Escaneo de Códigos de Barras

Conecta un escáner de códigos de barras USB o Bluetooth para añadir productos rápidamente. Cuando escaneas un código de barras, WCPOS busca automáticamente y añade el producto que coincide.

Consulta [Escaneo de Códigos de Barras](/pos/product-panel/barcode-scanning) para instrucciones de configuración.
