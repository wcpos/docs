El Panel de Productos es el lado izquierdo de la pantalla del POS donde puedes buscar, navegar y seleccionar productos para añadir al carrito.

{/* Screenshot: pos-product-panel.png */}

## Descripción general de la interfaz

### Barra de búsqueda

En la parte superior del panel, la barra de búsqueda te permite encontrar productos rápidamente por:

- **Nombre del producto** - Escribe cualquier parte del nombre del producto
- **SKU** - Buscar por Unidad de Mantenimiento de Existencias
- **Código de barras** - Escanear o escribir un número de código de barras

Consulta [Búsqueda y Filtrado](/pos/product-panel/search-filtering) para obtener información detallada.

### Configuración de visualización

Haz clic en el **icono de deslizadores** (<Icon name="sliders" />) junto a la barra de búsqueda para abrir la Configuración de visualización. Esto te permite personalizar qué columnas e información se muestran en la lista de productos.

### Botones de filtro

Debajo de la barra de búsqueda, los botones de filtro te permiten reducir los productos:

- **En stock** - Mostrar solo productos con inventario disponible
- **Destacados** - Mostrar productos marcados como destacados en WooCommerce
- **En oferta** - Mostrar productos actualmente en oferta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si se usa un plugin de marcas)

Los filtros activos aparecen resaltados. Haz clic en un filtro de nuevo para eliminarlo.

### Lista de productos

El área principal muestra tus productos en una lista desplazable. Cada fila muestra:

- **Imagen del producto** - Miniatura del producto
- **Nombre del producto** - Con detalles opcionales (stock, SKU, categorías, etc.)
- **Precio** - Precio de venta actual (con precios de oferta mostrados cuando sea aplicable)
- **Botón de acción** - Añadir al carrito (<Icon name="circle-plus" color="green" />) o seleccionar variación (<Icon name="circle-arrow-right" color="green" />)

### Tipos de productos

- Los **productos simples** muestran un botón verde <Icon name="circle-plus" color="green" /> para añadir directamente al carrito
- Los **productos variables** muestran una flecha verde <Icon name="circle-arrow-right" color="green" /> para abrir el selector de variaciones

Consulta [Productos Variables](/pos/product-panel/variations) para más detalles.

### Pie de página

En la parte inferior del Panel de Productos:

- **Estado de impuestos** - Muestra la base de cálculo de impuestos actual (p. ej., "Impuestos basados en: Dirección base de la tienda"). Haz clic para ver las tasas de impuestos activas.
- **Contador de productos** - Muestra cuántos productos se están mostrando (p. ej., "Mostrando 10 de 17")
- **Botón de sincronización** - Actualizar productos del servidor

### Mostrar productos sin stock

Controla si los productos que actualmente están sin stock se muestran.

- **Habilitado**: Los productos sin stock permanecen visibles pero no se pueden añadir al carrito
- **Deshabilitado**: Los productos sin stock se ocultan de la lista

### Columnas

Configura qué columnas aparecen en la lista de productos:

| Columna | Descripción | Opciones de visualización |
|--------|-------------|-----------------|
| **Imagen** | Miniatura del producto | - |
| **Producto** | Nombre del producto y detalles | Stock, SKU, Código de barras, Categorías, Etiquetas, Marcas, Atributos, Metadatos |
| **SKU** | Unidad de Mantenimiento de Existencias (columna separada) | - |
| **Código de barras** | Código de barras del producto (columna separada) | - |
| **Tipo** | Tipo de producto (simple, variable, etc.) | - |
| **Stock** | Cantidad de stock actual | - |
| **Costo de Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta | Información de impuestos, Indicador de oferta |
| **Acciones** | Botón de añadir al carrito | - |

### Claves de Metadatos

Especifica las claves de metadatos del producto que deben transferirse a los elementos de línea del pedido cuando se añaden productos al carrito.

Introduce una lista de claves de metadatos separadas por comas:

```
_bottle_deposit,_custom_field,_tracking_code
```

**Casos de uso:**
- Plugins de depósito de botellas
- Campos de producto personalizados
- Datos de integración de terceros
- Información de cumplimiento/seguimiento

:::note
Las claves de metadatos distinguen entre mayúsculas y minúsculas y deben coincidir exactamente con cómo se almacenan en los metadatos del producto.
:::

### Restaurar configuración predeterminada

Haz clic para restablecer toda la configuración de visualización a sus valores predeterminados originales.

## Añadir productos al carrito

### Productos simples

Haz clic en el botón verde <Icon name="circle-plus" color="green" /> para añadir un producto simple al carrito. Cada clic añade una unidad más.

### Productos variables

Los productos variables (p. ej., una camiseta con opciones de talla y color) muestran una flecha verde <Icon name="circle-arrow-right" color="green" />. Haz clic para:

1. **Ventana emergente rápida** - Seleccionar variación desde un menú desplegable
2. **Expandir en línea** - Haz clic en "Expandir" para ver todas las variaciones en la lista

Consulta [Productos Variables](/pos/product-panel/variations) para más detalles.

### Escaneo de código de barras

Conecta un escáner de código de barras USB o Bluetooth para añadir productos rápidamente. Cuando escaneas un código de barras, WCPOS busca automáticamente y añade el producto coincidente.

Consulta [Escaneo de Código de Barras](/pos/product-panel/barcode-scanning) para obtener instrucciones de configuración.
