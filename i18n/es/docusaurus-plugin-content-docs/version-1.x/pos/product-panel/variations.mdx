Los productos variables en WooCommerce son productos que tienen múltiples opciones, como diferentes tallas, colores o materiales. WCPOS ofrece varias formas de trabajar con productos variables de manera eficiente.

## Identificación de productos variables {#identifying-variable-products}

En el panel de productos, los productos variables se distinguen por:

- **Botón de flecha** (<Icon name="circle-arrow-right" color="green" />) en lugar del botón de suma (<Icon name="circle-plus" color="green" />) utilizado para productos simples
- **Opciones de atributos** mostradas debajo del nombre del producto (por ejemplo, "Color: Azul, Verde, Rojo")
- **Enlace "Expandir"** para ver todas las variaciones en línea

## Agregar productos variables al carrito {#adding-variable-products-to-cart}

### Método 1: Ventana emergente rápida {#method-1-quick-popover}

Hacer clic en el botón de <Icon name="circle-arrow-right" color="green" /> flecha verde en un producto variable para abrir una ventana emergente con menús desplegables para cada atributo:

1. Pulse el <Icon name="circle-arrow-right" color="green" /> botón
2. Seleccionar opciones de cada menú desplegable (por ejemplo, Talla: Grande, Color: Azul)
3. La variación correspondiente se agrega al carrito

Este es el método más rápido cuando se conoce exactamente la variación necesaria.

{/* Screenshot: variation-popover.png */}

### Método 2: Expandir en línea {#method-2-expand-inline}

Hacer clic en el enlace **"Expandir"** en un producto variable para mostrar todas sus variaciones directamente en la lista de productos:

1. Pulse **Expandir** debajo del nombre del producto
2. Todas las variaciones aparecen como filas separadas debajo del producto principal
3. Pulse el <Icon name="circle-plus" color="green" /> botón en cualquier variación para añadirla al carrito
4. Pulse **Contraer** para ocultar las variaciones de nuevo

Este método es útil cuando se desea ver todas las opciones disponibles, sus niveles de stock individuales y sus precios.

{/* Screenshot: variations-expanded.png */}

### Método 3: Escaneo de código de barras {#method-3-barcode-scanning}

Cada variación de producto puede tener su propio código de barras único. Al escanear el código de barras de una variación:

- Si coincide exactamente una variación, se añade directamente al carrito
- Si varias variaciones comparten el código de barras, la barra de búsqueda muestra los resultados coincidentes

Consulte [Escaneo de código de barras](/pos/product-panel/barcode-scanning) para obtener más información.

## Información de la variación {#variation-information}

Cada variación puede mostrar:

- **Atributos de variación** - Las opciones específicas de esta variación (p. ej., "Talla: Grande, Color: Azul")
- **SKU** - Identificador único de la variación (puede diferir del producto principal)
- **Cantidad en stock** - Nivel de inventario individual de la variación
- **Precio** - Las variaciones pueden tener precios diferentes al del producto principal o entre sí

## Variaciones en el carrito {#variations-in-the-cart}

Cuando se agrega una variación al carrito, se muestra:

- **Nombre del producto** del producto principal
- **Atributos de variación** debajo del nombre
- **Precio** de esa variación específica

Es posible editar el artículo de línea de la misma manera que cualquier otro producto en el carrito.

## Consejos para una selección eficiente de variaciones {#tips-for-efficient-variation-selection}

### Usar escaneo de código de barras {#use-barcode-scanning}

Si las variaciones tienen códigos de barras individuales (recomendado), el escaneo es la forma más rápida de añadirlas al carrito.

### Configurar opciones de visualización {#configure-display-options}

En la [Configuración de visualización](/pos/product-panel#display-settings) del panel de productos, se debe activar la opción **Atributos** para mostrar los atributos de variación directamente en la lista de productos sin necesidad de expandir.

### Navegación por teclado {#keyboard-navigation}

Cuando el cuadro emergente de variaciones está abierto, es posible utilizar las teclas de flecha del teclado para navegar entre las opciones y Enter para seleccionar.

## Preguntas frecuentes {#faq}

<Accordion>
  <AccordionItem question="¿Por qué no puedo ver una variación específica?">

Es necesario verificar que la variación:
- Esté publicada y no en estado de borrador
- Esté configurada como "En stock" o tenga una cantidad de stock > 0 (si la gestión de stock está habilitada)
- Tiene todos los atributos requeridos definidos

Asegúrese también de no haber filtrado el producto con el filtro "En stock" si la variación está agotada.

  </AccordionItem>

  <AccordionItem question="¿Cómo se agregan códigos de barras a variaciones individuales?">

En WooCommerce, edite el producto variable y expanda cada variación. Es posible agregar un código de barras único en el campo de código de barras de cada variación (normalmente `_sku` o `_global_unique_id` según la configuración).

  </AccordionItem>
</Accordion>
