Los productos variables en WooCommerce son productos que tienen múltiples opciones, como diferentes tamaños, colores o materiales. WCPOS proporciona varias formas de trabajar con productos variables de manera eficiente.

## Identificación de Productos Variables

En el Panel de Productos, los productos variables se distinguen por:

- **Botón de flecha** (<Icon name="circle-arrow-right" color="green" />) en lugar del botón de más (<Icon name="circle-plus" color="green" />) utilizado para productos simples
- **Opciones de atributos** mostradas debajo del nombre del producto (por ejemplo, "Color: Azul, Verde, Rojo")
- **Enlace de "Expandir"** para ver todas las variaciones en línea

## Agregar Productos Variables al Carrito

### Método 1: Ventana emergente rápida

Haz clic en el botón de flecha verde <Icon name="circle-arrow-right" color="green" /> en un producto variable para abrir una ventana emergente con menús desplegables para cada atributo:

1. Haz clic en el botón <Icon name="circle-arrow-right" color="green" />
2. Selecciona opciones de cada menú desplegable (por ejemplo, Tamaño: Grande, Color: Azul)
3. La variación correspondiente se agrega al carrito

Este es el método más rápido cuando sabes exactamente qué variación necesitas.

{/* Screenshot: variation-popover.png */}

### Método 2: Expandir en Línea

Haz clic en el enlace **"Expandir"** en un producto variable para mostrar todas sus variaciones directamente en la lista de productos:

1. Haz clic en **Expandir** debajo del nombre del producto
2. Todas las variaciones aparecen como filas separadas debajo del producto principal
3. Haz clic en el botón <Icon name="circle-plus" color="green" /> en cualquier variación para agregarla al carrito
4. Haz clic en **Colapsar** para ocultar las variaciones nuevamente

Este método es útil cuando deseas ver todas las opciones disponibles, sus niveles de inventario individuales y precios.

{/* Screenshot: variations-expanded.png */}

### Método 3: Escaneo de Código de Barras

Cada variación del producto puede tener su propio código de barras único. Cuando escaneas el código de barras de una variación:

- Si exactamente una variación coincide, se agrega directamente al carrito
- Si múltiples variaciones comparten el código de barras, la barra de búsqueda muestra los resultados coincidentes

Consulta [Escaneo de Códigos de Barras](/pos/product-panel/barcode-scanning) para más detalles.

## Información de Variación

Cada variación puede mostrar:

- **Atributos de variación** - Las opciones específicas para esta variación (por ejemplo, "Tamaño: Grande, Color: Azul")
- **SKU** - Identificador único para la variación (puede diferir del producto principal)
- **Cantidad en stock** - Nivel de inventario individual para la variación
- **Precio** - Las variaciones pueden tener precios diferentes al del producto principal o entre sí

## Variaciones en el Carrito

Cuando una variación se agrega al carrito, muestra:

- **Nombre del producto** del producto principal
- **Atributos de variación** debajo del nombre
- **Precio** para esa variación específica

Puedes editar el artículo en la línea como cualquier otro producto en el carrito.

## Consejos para una Selección Eficiente de Variaciones

### Usa Escaneo de Código de Barras

Si tus variaciones tienen códigos de barras individuales (recomendado), escanear es la forma más rápida de agregarlas al carrito.

### Configura Opciones de Visualización

En el Panel de Productos [Configuración de Visualización](/pos/product-panel#display-settings), habilita la opción **Atributos** para mostrar los atributos de variación directamente en la lista de productos sin necesidad de expandir.

### Navegación por Teclado

Cuando la ventana emergente de variación está abierta, puedes usar las teclas de dirección del teclado para navegar entre opciones y Enter para seleccionar.

## F.A.Q.

<Accordion>
  <AccordionItem question="¿Por qué no puedo ver una variación específica?">

Verifica que la variación:
- Esté publicada y no en estado de borrador
- Esté configurada como "En Stock" o tenga una cantidad en stock > 0 (si la gestión de inventario está habilitada)
- Tenga todos los atributos requeridos definidos

También asegúrate de no haber filtrado el producto con el filtro "En Stock" si la variación está fuera de stock.

  </AccordionItem>

  <AccordionItem question="¿Cómo agrego códigos de barras a variaciones individuales?">

En WooCommerce, edita el producto variable y expande cada variación. Puedes agregar un código de barras único al campo de código de barras de cada variación (típicamente `_sku` o `_global_unique_id` dependiendo de tu configuración).

  </AccordionItem>
</Accordion>
