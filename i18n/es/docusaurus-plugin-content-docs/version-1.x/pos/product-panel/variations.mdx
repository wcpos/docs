Los productos variables en WooCommerce son productos que tienen múltiples opciones, como diferentes tamaños, colores o materiales. WCPOS proporciona varias formas de trabajar con productos variables de manera eficiente.

## Identificación de Productos Variables

En el Panel de Productos, los productos variables se distinguen por:

- **Botón de flecha** (<Icon name="circle-arrow-right" color="green" />) en lugar del botón de más (<Icon name="circle-plus" color="green" />) utilizado para productos simples
- **Opciones de atributos** mostradas debajo del nombre del producto (por ejemplo, "Color: Azul, Verde, Rojo")
- **Enlace "Expandir"** para ver todas las variaciones en línea

## Agregar Productos Variables al Carrito

### Método 1: Ventana Emergente Rápida

Haga clic en el botón de flecha verde <Icon name="circle-arrow-right" color="green" /> de un producto variable para abrir una ventana emergente con menús desplegables para cada atributo:

1. Haga clic en el botón <Icon name="circle-arrow-right" color="green" />
2. Seleccione opciones de cada menú desplegable (por ejemplo, Talla: Grande, Color: Azul)
3. La variación coincidente se agrega al carrito

Este es el método más rápido cuando sabe exactamente qué variación necesita.

{/* Screenshot: variation-popover.png */}

### Método 2: Expandir en Línea

Haga clic en el enlace **"Expandir"** de un producto variable para mostrar todas sus variaciones directamente en la lista de productos:

1. Haga clic en **Expandir** debajo del nombre del producto
2. Todas las variaciones aparecen como filas separadas debajo del producto principal
3. Haga clic en el botón <Icon name="circle-plus" color="green" /> de cualquier variación para agregarla al carrito
4. Haga clic en **Contraer** para ocultar las variaciones nuevamente

Este método es útil cuando desea ver todas las opciones disponibles, sus niveles de stock individuales y precios.

{/* Screenshot: variations-expanded.png */}

### Método 3: Escaneo de Código de Barras

Cada variación de producto puede tener su propio código de barras único. Cuando escanea el código de barras de una variación:

- Si coincide exactamente una variación, se agrega directamente al carrito
- Si varias variaciones comparten el código de barras, la barra de búsqueda muestra los resultados coincidentes

Consulte [Escaneo de Código de Barras](/pos/product-panel/barcode-scanning) para más detalles.

## Información de Variaciones

Cada variación puede mostrar:

- **Atributos de variación** - Las opciones específicas para esta variación (por ejemplo, "Talla: Grande, Color: Azul")
- **SKU** - Identificador único para la variación (puede diferir del producto principal)
- **Cantidad en stock** - Nivel de stock individual para la variación
- **Precio** - Las variaciones pueden tener precios diferentes al producto principal o entre sí

## Variaciones en el Carrito

Cuando se agrega una variación al carrito, se muestra:

- **Nombre del producto** del producto principal
- **Atributos de variación** debajo del nombre
- **Precio** para esa variación específica

Puede editar la línea de artículo igual que cualquier otro producto en el carrito.

## Consejos para Selección Eficiente de Variaciones

### Usar Escaneo de Código de Barras

Si sus variaciones tienen códigos de barras individuales (recomendado), el escaneo es la forma más rápida de agregarlas al carrito.

### Configurar Opciones de Visualización

En la [Configuración de Visualización](/pos/product-panel#display-settings) del Panel de Productos, active la opción **Atributos** para mostrar los atributos de variación directamente en la lista de productos sin expandir.

### Navegación con Teclado

Cuando la ventana emergente de variación está abierta, puede usar las teclas de flecha para navegar por las opciones y Intro para seleccionar.

## Preguntas Frecuentes

<details>
<summary>¿Por qué no puedo ver una variación específica?</summary>

Verifique que la variación:
- Esté publicada y no en estado de borrador
- Esté configurada como "En Stock" o tenga cantidad de stock > 0 (si la gestión de stock está habilitada)
- Tenga todos los atributos requeridos definidos

Asegúrese también de no haber filtrado el producto con el filtro "En Stock" si la variación está agotada.

</details>

<details>
<summary>¿Puedo agregar una variación sin seleccionar todos los atributos?</summary>

No, debe seleccionar una opción para cada atributo para identificar una variación específica. Si un producto tiene atributos de Talla y Color, debe elegir ambos antes de agregar al carrito.

</details>

<details>
<summary>¿Cómo agrego códigos de barras a variaciones individuales?</summary>

En WooCommerce, edite el producto variable y expanda cada variación. Puede agregar un código de barras único a cada campo de código de barras de variación (típicamente `_sku` o `_global_unique_id` según su configuración).

</details>
