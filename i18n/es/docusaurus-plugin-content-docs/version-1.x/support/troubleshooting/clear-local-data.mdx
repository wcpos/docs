WCPOS almacena una copia local de tus productos, pedidos y clientes para que el POS pueda funcionar rápidamente y operar sin conexión. Ocasionalmente, estos datos locales pueden quedar fuera de sincronización con tu tienda WooCommerce. Limpiar todos los datos locales elimina la base de datos local y activa una nueva sincronización desde tu servidor.

:::warning
Esta es la opción nuclear. Limpiar datos locales elimina todo lo almacenado en el dispositivo y fuerza una re-sincronización completa. Dependiendo del tamaño de tu tienda, esto puede tardar un tiempo. Úsalo solo cuando otras opciones — como una [sincronización](/products/sync) normal — no hayan resuelto el problema.
:::

## Cuándo Usarlo {#when-to-use-it}

Puede que necesites limpiar los datos locales cuando:

- **Problemas de sincronización** — productos, pedidos o clientes no se actualizan correctamente
- **Después de una actualización de plugin** — para asegurar la compatibilidad con nuevas estructuras de datos
- **Datos obsoletos** — precios, niveles de stock u otra información parece desactualizada

## Qué se Limpia {#what-gets-cleared}

| Limpiado | No Limpiado |
|---------|-------------|
| Base de datos de productos local | Datos del lado del servidor |
| Base de datos de pedidos local | Configuraciones de WooCommerce |
| Base de datos de clientes local | |
| Sesión de inicio | |

Los datos de tu tienda WooCommerce no se ven afectados — solo se elimina la copia local en tu dispositivo. Una sincronización nueva comienza automáticamente después de limpiar.

:::info Diferencia entre plataformas
- **Aplicación de escritorio:** Se te cerrará la sesión y deberás volver a conectar tu tienda.
- **Aplicación web:** Los datos de inicio de sesión se limpian, pero como sigues con sesión iniciada en WP Admin, la sesión se rehidrata automáticamente — por lo que no notarás que se ha cerrado la sesión.
:::

## Cómo Limpiar Datos Locales {#how-to-clear-local-data}

### Web {#web}

1. Haz clic en tu **menú de usuario** en la esquina superior derecha para abrir el menú
2. Haz clic en **Limpiar Todos los Datos Locales**
3. La base de datos local será eliminada y comenzará automáticamente una nueva sincronización

### Escritorio {#desktop}

1. Abre el menú de la **aplicación WCPOS** en la barra de menú superior
2. Pasa el mouse sobre **Avanzado**
3. Haz clic en **Limpiar Datos de la Aplicación**
4. La base de datos local será eliminada y comenzará automáticamente una nueva sincronización

:::note
La aplicación de escritorio utiliza una base de datos SQLite mientras que la aplicación web usa IndexedDB. Ambas bases de datos se limpian por completo, pero el comportamiento de inicio de sesión difiere — consulta la nota sobre la diferencia entre plataformas más arriba.
:::
