# WCPOS Polylang {#wcpos-polylang}

Añade compatibilidad con [Polylang](https://polylang.pro/) a WCPOS para que el POS solo muestre productos de un único idioma, sin traducciones duplicadas en la búsqueda de productos, la cuadrícula del catálogo ni en los flujos de trabajo del cajero. Las tiendas con WCPOS Pro pueden fijar un idioma por tienda; las instalaciones gratuitas utilizan el idioma predeterminado de Polylang.

## Qué hace {#what-it-does}

- Filtra por idioma las consultas REST de productos y variaciones de WCPOS.
- Intercepta las rutas de **sincronización rápida** de WCPOS (las `posts_per_page=-1` + `fields` solicitudes ligeras que el POS utiliza para refrescar su índice local) para que los duplicados traducidos nunca lleguen al cliente.
- En instalaciones gratuitas, aplica el idioma predeterminado de Polylang.
- En instalaciones Pro, cada tienda puede elegir su propio idioma desde una nueva sección **Idioma** en el editor de tiendas.
- Respeta la visibilidad de productos **solo POS** de WCPOS al construir la carga útil de sincronización rápida.

La integración no realiza ninguna acción de forma limpia cuando Polylang no está activo: puede instalar el plugin antes de habilitar Polylang sin errores.

## Instalación {#installation}

<Steps>
  <StepCard number="1" title="Install Polylang">

Instale [Polylang](https://wordpress.org/plugins/polylang/) (o Polylang Pro) y configure los idiomas de su sitio con normalidad. Asegúrese de que al menos un idioma esté establecido como predeterminado.

  </StepCard>
  <StepCard number="2" title="Install WCPOS Polylang">

Instale desde el directorio de extensiones de WCPOS en `WP Admin > POS > Settings > Extensions`, o descargue la última versión desde la [página de versiones de GitHub](https://github.com/wcpos/wcpos-polylang/releases) y súbalo mediante `Plugins > Add New > Upload Plugin`.

  </StepCard>
  <StepCard number="3" title="(Pro) Set a per-store language" isLast>

Si gestiona [varias tiendas](/stores) en WCPOS Pro, vaya a `POS > Stores`, edita una tienda y elige su idioma en la sección **Idioma** de la barra lateral. Déjalo en *Predeterminado* para usar el idioma predeterminado de Polylang.

  </StepCard>
</Steps>

## Idioma por tienda (Pro) {#per-store-language-pro}

En WCPOS Pro, el plugin añade una sección **Idioma** a la barra lateral del editor de tiendas. Cada tienda puede fijarse a un único slug de idioma de Polylang: los productos servidos a esa tienda se filtran solo a ese idioma. Las tiendas que se dejan en *Predeterminado* usan el idioma predeterminado de Polylang.

El valor por tienda se guarda en el post de la tienda como `_wcpos_polylang_language` meta y se expone a través de la API REST de tiendas de WCPOS Pro (`/wcpos/v1/stores`), por lo que se transmite a través del POS como cualquier otro ajuste de tienda.

## Notas de compatibilidad {#compatibility-notes}

- **Productos solo POS:** cuando el modo solo POS está activado en los ajustes de WCPOS, los IDs de productos solo en línea se excluyen del payload de sincronización rápida para que no se filtren al POS.
- **Instalaciones gratuitas:** no hay interfaz para cambiar el idioma por tienda; el plugin utiliza el idioma predeterminado de Polylang. Si necesitas idiomas por tienda, actualiza a [WCPOS Pro](/getting-started/pro-license).
- **Plugin no disponible:** si Polylang está desactivado, el plugin no hace nada de forma silenciosa. No generará errores ni bloqueará el POS.

## Hooks para desarrolladores {#developer-hooks}

Para usos avanzados, el plugin expone algunos filtros:

| Filtro | Propósito |
| --- | --- |
| `wcpos_polylang_resolved_language` | Sobrescribe el idioma utilizado para una solicitud determinada. Recibe el slug resuelto y el `WP_REST_Request`. |
| `wcpos_polylang_default_language` | Sobrescribe el idioma de respaldo cuando no se ha definido un valor por tienda. |
| `wcpos_polylang_is_supported` | Fuerza la activación o desactivación del plugin independientemente de la detección de Polylang. |
| `wcpos_polylang_minimum_version` | Requiere una versión mínima de Polylang (por defecto: sin restricción de versión). |
