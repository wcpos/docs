---
title: Requisitos Mínimos
---

Para garantizar el correcto funcionamiento y un rendimiento óptimo del plugin WooCommerce POS, asegúrese de que su entorno cumpla con los siguientes requisitos mínimos:

### Requisitos del servidor:

- **WordPress:** Versión 5.6 o superior
- **WooCommerce:** Versión 5.3 o superior
- **PHP:** Versión 7.2 o superior

Cumplir con estos requisitos mínimos garantiza la compatibilidad y estabilidad entre el plugin WooCommerce POS y su entorno de WordPress. También asegura que pueda beneficiarse de las últimas características y mejoras proporcionadas por el plugin.

### Requisitos del navegador:

- **Navegador web moderno:** El plugin WooCommerce POS está diseñado para funcionar con navegadores web modernos, incluyendo Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. Asegúrese de estar utilizando la última versión estable de su navegador preferido.
- **JavaScript habilitado:** El plugin WooCommerce POS depende de JavaScript para sus características interactivas. Asegúrese de que JavaScript esté habilitado en la configuración de su navegador.
- **Sin modo de navegación privada:** El navegador no debe estar en modo de navegación privada, ya que puede restringir el acceso a la base de datos IndexedDB (IDB) del navegador. El modo de navegación privada puede evitar el correcto funcionamiento de ciertas características en el plugin WooCommerce POS.

Al cumplir con los requisitos mínimos y utilizar un navegador web compatible, puede tener una experiencia fluida utilizando el plugin WooCommerce POS y aprovechar todo su potencial para administrar eficientemente sus operaciones de punto de venta.
