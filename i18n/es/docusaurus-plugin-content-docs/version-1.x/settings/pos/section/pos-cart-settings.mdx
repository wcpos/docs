## Mostrar recibo automáticamente después de finalizar la compra

Si está habilitado, la aplicación mostrará automáticamente una pantalla de recibo inmediatamente después de completar una venta, lo que permite al personal revisar o compartir rápidamente el recibo final.

## Imprimir recibo automáticamente después de finalizar la compra

Si está habilitado, la aplicación activará inmediatamente una impresión cuando se muestre la pantalla de recibo. Esto ahorra tiempo al reducir los pasos de impresión manual.

## Descuentos rápidos

Una lista separada por comas de porcentajes de descuento que aparecen como botones de acceso rápido en el carrito.
Por ejemplo, si ingresa 5,10,15,20, se agregarán cuatro botones de descuento rápido que aplicarán un porcentaje de descuento a todo el pedido con un solo clic.

## Columnas

La sección Columnas controla qué puntos de datos se muestran para cada artículo en el carrito.
Activar o desactivar una columna determina si se muestra esa información.
En las "Opciones de visualización" de cada columna, puede elegir mostrar u ocultar detalles relacionados adicionales.

## Restaurar configuración predeterminada

Restablece todos los botones, descuentos rápidos y opciones de visualización a sus valores predeterminados originales.
Esto es útil si la configuración actual ya no cumple con los requisitos o necesita una línea base limpia.
