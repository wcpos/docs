Estas configuraciones definen la funcionalidad backend de la aplicación POS, incluyendo configuraciones de PHP y WooCommerce. 
Puedes acceder a ellas navegando a: `WP Admin > POS > Configuraciones`

Las configuraciones clave incluyen:

- Cliente predeterminado del carrito para todos los cajeros.
- Campo de código de barras del producto.
- Configuraciones de correo electrónico de pedidos.
- Permisos de roles de usuario.
