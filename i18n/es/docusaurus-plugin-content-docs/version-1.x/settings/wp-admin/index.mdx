Estos ajustes configuran la funcionalidad del backend de la aplicación POS, incluidas las configuraciones de PHP y WooCommerce. 
Puedes acceder a ellos navegando a: `WP Admin > POS > Settings`

Las configuraciones clave incluyen:

- Cliente de carrito predeterminado para todos los cajeros.
- Campo de código de barras del producto.
- Ajustes de correos electrónicos de pedidos.
- Permisos de roles de usuario.
