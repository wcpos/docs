Estas configuraciones configuran la funcionalidad del backend de la aplicación POS, incluyendo configuraciones de PHP y WooCommerce. 
Puedes acceder a ellas navegando a: `WP Admin > POS > Configuración`

Las configuraciones clave incluyen:

- Cliente de carrito predeterminado para todos los cajeros.
- Campo de código de barras del producto.
- Configuraciones de correo electrónico de pedido.
- Permisos de roles de usuario.
