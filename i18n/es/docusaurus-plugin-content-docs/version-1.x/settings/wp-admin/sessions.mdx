La pantalla de **Sesiones** permite ver quién tiene una sesión iniciada en el POS, en qué dispositivos, y cerrar sesión en dispositivos individuales de forma remota. Es el lugar indicado cuando se pierde o se roba un dispositivo, cuando un cajero deja de trabajar o simplemente cuando se desea revisar los inicios de sesión activos.

Se encuentra en **`WP Admin > POS > Ajustes > Sessions`**.

> _Administre las sesiones de usuario activas. Es posible ver todos los usuarios con sesión iniciada, consultar sus dispositivos y finalizar sesiones cuando sea necesario._

## Cómo funciona {#how-it-works}

La pantalla tiene dos paneles:

- **Usuarios** (izquierda) — cada usuario con al menos una sesión activa, con un punto verde en aquellos que han estado activos en los últimos minutos, una etiqueta **Tú** para la cuenta propia y un recuento de cuántas sesiones tiene cada usuario. Un cuadro de filtro permite buscar por nombre o nombre de usuario.
- **Sesiones** (derecha) — al seleccionar un usuario se muestra una tarjeta por cada sesión activa.

Cada tarjeta de sesión muestra:

| Campo | Significado |
| --- | --- |
| **Aplicación** | Dónde ha iniciado sesión el usuario — **Aplicación iOS**, **Aplicación Android**, **Aplicación de escritorio** o **Aplicación web** — además del navegador y la versión |
| **Dispositivo actual** | Una etiqueta que indica la sesión que se está utilizando en este momento |
| **Última actividad** | Cuánto tiempo ha pasado desde que la sesión realizó una solicitud (por ejemplo, _Ahora mismo_, _Hace 5 minutos_) |
| **IP** | La dirección IP desde la que se conectó la sesión (o _N/D_) |
| **Creada** | Cuándo se inició sesión por primera vez |
| **Caduca** | Cuándo caduca el token de sesión |
| **Agente de usuario** | Se puede expandir para ver la cadena de agente de usuario sin procesar del navegador o dispositivo |

Una sesión se muestra como **Activa ahora** cuando ha realizado una solicitud en los últimos 5 minutos.

## Cerrar sesión en dispositivos {#signing-devices-out}

Hay tres acciones disponibles, cada una con un cuadro de diálogo de confirmación:

- **Terminar** — cierra una sesión individual. Se utiliza el botón en la tarjeta de esa sesión.
- **Cerrar sesión en otros dispositivos** — cierra la sesión del usuario seleccionado en todos los dispositivos **excepto** el que está utilizando actualmente. Se muestra cuando el usuario tiene más de una sesión.
- **Cerrar todas las sesiones** — cierra la sesión del usuario seleccionado en **todos** los dispositivos, incluido el actual.

:::tip Dispositivo perdido o robado
Si se pierde un teléfono, una tableta o un portátil con el POS, abra Sesiones, seleccione el usuario afectado y utilice **Terminar** en la sesión de ese dispositivo (o **Cerrar todas las sesiones** para mayor seguridad). El dispositivo perderá el acceso en su siguiente solicitud.
:::

:::note
Terminar su propia sesión actual cierra la sesión en el dispositivo que está utilizando en este momento; se le pedirá confirmación antes de que esto ocurra.
:::
