## Atajos Disponibles

| Combinación de Teclas | Acción | Descripción |
|-----------------------|--------|-------------|
| `Ctrl + Shift + S`    | Settings | Abre el panel de configuración de POS |
| `Ctrl + Shift + A`    | POS    | Navega a la pantalla principal del POS |
| `Ctrl + Shift + P`    | Products | Abre la pantalla de gestión de productos |
| `Ctrl + Shift + O`    | Orders | Abre la pantalla de gestión de órdenes |
| `Ctrl + Shift + C`    | Customers | Abre la pantalla de gestión de clientes |
| `Ctrl + Shift + ?`    | Support | Abre la sección de soporte/ayuda |
| `Ctrl + Shift + L`    | Logout | Cierra la sesión actual |

## Uso de Atajos de Teclado

Para usar cualquier atajo de teclado:

1. **Mantén presionadas las teclas modificadoras** (`Ctrl + Shift`)
2. **Presiona la tecla de acción** (la letra correspondiente a la acción deseada)
3. **Suelta todas las teclas** para ejecutar el comando

## Consejos para un Uso Eficiente

- **Memoriza los atajos utilizados con frecuencia**: Concéntrate en aprender los atajos de las acciones que realizas con mayor frecuencia
- **Practica las combinaciones de teclas**: El uso regular ayudará a construir memoria muscular
- **Utiliza atajos durante períodos de máxima actividad**: Los atajos de teclado pueden acelerar significativamente las operaciones durante las horas pico

## Personalización

:::info Solicitud de Función
La personalización de atajos de teclado actualmente no está disponible. Los atajos enumerados arriba son fijos y no pueden ser modificados a través de la interfaz de configuración.

Si deseas personalizar atajos de teclado, por favor comparte tus comentarios en [Discord](https://wcpos.com/discord).
:::
