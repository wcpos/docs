## Atajos Disponibles

| Combinación de Teclas | Acción | Descripción |
|-----------------|--------|-------------|
| `Ctrl + Shift + S` | Configuración | Abre el panel de configuración del POS |
| `Ctrl + Shift + A` | POS | Navega a la pantalla principal del POS |
| `Ctrl + Shift + P` | Productos | Abre la pantalla de gestión de productos |
| `Ctrl + Shift + O` | Pedidos | Abre la pantalla de gestión de pedidos |
| `Ctrl + Shift + C` | Clientes | Abre la pantalla de gestión de clientes |
| `Ctrl + Shift + ?` | Soporte | Abre la sección de soporte/ayuda |
| `Ctrl + Shift + L` | Cerrar sesión | Cierra la sesión actual |

## Uso de Atajos de Teclado

Para usar cualquier atajo de teclado:

1. **Mantén presionadas las teclas modificadoras** (`Ctrl + Shift`)
2. **Presiona la tecla de acción** (la letra correspondiente a la acción deseada)
3. **Suelta todas las teclas** para ejecutar el comando

## Consejos para un Uso Eficiente

- **Memoriza los atajos más utilizados**: Concéntrate en aprender los atajos para las acciones que realizas con más frecuencia
- **Practica las combinaciones de teclas**: El uso regular ayudará a desarrollar memoria muscular
- **Usa atajos durante períodos ocupados**: Los atajos de teclado pueden acelerar significativamente las operaciones durante las horas pico

## Personalización

:::info Solicitud de Funcionalidad
Actualmente no está disponible la personalización de atajos de teclado. Los atajos listados están fijos y no se pueden modificar a través de la interfaz de configuración.

Si te gustaría personalizar los atajos de teclado, por favor comparte tu opinión en [Discord](https://wcpos.com/discord).
:::
