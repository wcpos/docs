La Configuración de la Tienda son configuraciones de toda la aplicación que afectan cómo opera el POS. Estos ajustes se aplican a toda la tienda, no a pantallas individuales.

## Acceder a la Configuración de la Tienda

Abra la Configuración de la Tienda:

- Haciendo clic en el **menú de usuario** (esquina superior derecha) y seleccionando **Configuración**
- Usando el atajo de teclado `Ctrl + Shift + S`

## Pestañas de Configuración

El modal de Configuración de la Tienda contiene estas pestañas:

| Pestaña | Descripción |
|---------|-------------|
| [General](./general) | Información de la tienda, idioma, moneda, cliente predeterminado |
| [Impuestos](./tax) | Opciones de cálculo y visualización de impuestos |
| [Código de Barras](./barcode) | Configuración del escáner de códigos de barras |
| [Atajos de Teclado](./hotkeys) | Atajos de teclado disponibles |
| [Tema](./theme) | Apariencia y esquema de color |
