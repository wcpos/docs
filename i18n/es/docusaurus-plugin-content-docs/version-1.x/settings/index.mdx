WCPOS tiene tres tipos de configuraciones organizadas por donde se configuran.

## Dónde Encontrar Configuraciones {#where-to-find-settings}

### 1. Configuraciones de WordPress {#1-wordpress-admin-settings}

Configuraciones del backend configuradas en tu panel de WordPress:

**Ubicación:** `WP Admin > POS > Configuración`

Estas controlan:
- Cliente de carrito predeterminado para todos los cajeros
- Configuración del campo de código de barras del producto
- Configuraciones del correo electrónico de pedidos
- Permisos de roles de usuario (quién puede acceder al POS)
- Disponibilidad de pasarelas de pago

Consulta [Configuraciones de WordPress](/settings/wp-admin) para más detalles.

### 2. Configuraciones de Tienda POS {#2-pos-store-settings}

Configuraciones a nivel de aplicación configuradas dentro de la aplicación POS:

**Ubicación:** Menú de usuario (esquina superior derecha) → Configuración, o `Ctrl + Shift + S`

Estas controlan:
- Opciones de cálculo de impuestos
- Configuración del escáner de códigos de barras
- Atajos de teclado
- Tema/apariencia

Consulta [Configuraciones de la Tienda](/settings/store) para más detalles.

### 3. Configuraciones de Pantalla (Por Pantalla) {#3-display-settings-per-screen}

Cada pantalla en el POS tiene sus propias configuraciones de pantalla para personalizar la tabla de datos:

**Ubicación:** Haz clic en el icono de deslizadores (<Icon name="sliders" />) en cualquier pantalla

Estas controlan:
- Qué columnas son visibles
- Opciones de visualización de columnas
- Características específicas de cada pantalla

Las configuraciones de visualización están documentadas dentro de la documentación de cada pantalla:
- [Panel de Productos POS](/pos/product-panel#display-settings)
- [Panel de Carrito](/pos/cart#display-settings)
- [Pantalla de Productos](/products#display-settings)
- [Pantalla de Pedidos](/orders#display-settings)
- [Pantalla de Clientes](/customers#display-settings)
- [Pantalla de Reportes](/reports#display-settings)
