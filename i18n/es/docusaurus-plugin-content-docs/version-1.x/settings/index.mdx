WCPOS tiene tres tipos de configuraciones organizadas según dónde se configuran.

## Dónde Encontrar Configuraciones

### 1. Configuración de WordPress Admin

Configuraciones del backend configuradas en tu panel de control de WordPress:

**Ubicación:** `WP Admin > POS > Configuración`

Estas controlan:
- Cliente de carrito predeterminado para todos los cajeros
- Configuración del campo de código de barras del producto
- Configuración de correo electrónico de pedidos
- Permisos de roles de usuario (quién puede acceder al POS)
- Disponibilidad de la pasarela de pago

Consulta [Configuración de WordPress Admin](/settings/wp-admin) para más detalles.

### 2. Configuración de la Tienda POS

Configuraciones a nivel de aplicación configuradas dentro de la aplicación POS:

**Ubicación:** Menú de usuario (parte superior derecha) → Configuración, o `Ctrl + Shift + S`

Estas controlan:
- Opciones de cálculo de impuestos
- Configuración del escáner de código de barras
- Atajos de teclado
- Tema/apariencia

Consulta [Configuración de la Tienda](/settings/store) para más detalles.

### 3. Configuración de Pantalla (Por Pantalla)

Cada pantalla en el POS tiene su propia configuración de visualización para personalizar la tabla de datos:

**Ubicación:** Haz clic en el ícono de deslizadores (<Icon name="sliders" />) en cualquier pantalla

Estas controlan:
- Qué columnas son visibles
- Opciones de visualización de columnas
- Características específicas de la pantalla

Las configuraciones de visualización están documentadas dentro de la documentación de cada pantalla:
- [Panel de Productos POS](/pos/product-panel#display-settings)
- [Panel de Carrito](/pos/cart#display-settings)
- [Pantalla de Productos](/products#display-settings)
- [Pantalla de Pedidos](/orders#display-settings)
- [Pantalla de Clientes](/customers#display-settings)
- [Pantalla de Informes](/reports#display-settings)
