Uno de los aspectos significativos del sistema WCPOS (Punto de Venta) es su enfoque respecto a la gestión de datos. Proporciona una forma eficiente de manejar un gran volumen de datos de productos. Esta guía explica cómo se descargan y sincronizan los productos dentro de WCPOS.

## Almacenamiento Local de Productos {#local-storage-of-products}

WCPOS descarga productos de tu tienda WooCommerce y los almacena localmente en tu dispositivo. Este enfoque significa que una vez que se descarga un producto, no es necesario descargarlo repetidamente. A medida que continúas utilizando el POS, descargará progresivamente todos los productos de tu tienda WooCommerce, creando una base de datos local de tus productos.

## Descarga por Lotes de Productos {#batch-downloading-of-products}

Para la mayoría de las tiendas, intentar descargar más de unos 50 productos a la vez podría llevar a una alta carga del servidor, tiempos de respuesta más lentos o incluso caídas del servidor. Para prevenir esto, WCPOS está diseñado para descargar productos en pequeños lotes, generalmente alrededor de 50 productos a la vez. Este proceso continúa hasta que todo el inventario ha sido almacenado localmente, asegurando que se pueda gestionar un gran volumen de datos sin poner una presión indebida en tu servidor.

## Uso de la Búsqueda de Productos {#using-the-product-search}

Una forma efectiva de iniciar la descarga por lotes de productos es utilizando la función de búsqueda de productos. Cada vez que realizas una búsqueda de productos, el sistema POS descarga y almacena otro lote de productos. Esta estrategia no solo ayuda a descargar gradualmente nuevos productos, sino que también mejora la velocidad y la eficiencia de la función de búsqueda a medida que se almacenan más productos localmente.

## Conclusión {#conclusion}

Es importante entender que no todos tus productos aparecerán de una vez en tu sistema POS, y esto es completamente normal. Con el tiempo, al utilizar la búsqueda de productos y otras operaciones del POS, descargarás todos tus productos. Para obtener más información detallada sobre cómo funciona WCPOS, consulta [Arquitectura](/reference/architecture).

WCPOS está diseñado para optimizar la carga del servidor y la eficiencia operativa. La función de descarga progresiva de productos asegura una experiencia de gestión minorista fluida, eficiente y fácil de usar.

## F.A.Q.

<Accordion>
  <AccordionItem question="¿Cómo puedo hacer que se carguen más productos?">

  La sincronización de productos en el POS se puede activar de varias maneras:

  - Cuando se carga por primera vez el POS.
  - Cada vez que realizas una búsqueda de productos.
  - Aplicando un filtro, por ejemplo, por categoría o etiqueta.
  - Desplazándote hasta la parte inferior de la lista de productos.
  - Presionando el botón de sincronización manualmente (presión corta para sincronizar, **presión larga** para la opción Limpiar y Refrescar).
  - Además, el POS sincronizará automáticamente cada 10 minutos.

  Una vez que todos los productos están descargados, las futuras sincronizaciones solo descargarán los productos que han sido actualizados desde la última sincronización.

  </AccordionItem>

  <AccordionItem question="He desplazado hasta el fondo y aún no se muestran todos los productos">

  Podría haber un par de razones por las cuales no se están mostrando todos los productos:

  - El POS ocultará los artículos fuera de stock por defecto. Esta configuración se puede modificar en la configuración de visualización de productos si deseas mostrar artículos fuera de stock.
  - La API REST de WooCommerce solo es compatible con los tipos de productos estándar de WooCommerce, que son simple, variable, agrupado y externo. Si estás utilizando un tipo de producto personalizado, es posible que no se muestren en el POS.

  </AccordionItem>
</Accordion>
