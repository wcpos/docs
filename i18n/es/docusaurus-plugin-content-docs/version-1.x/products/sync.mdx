Uno de los aspectos significativos del sistema WCPOS (Punto de Venta) es su enfoque en la gestión de datos. Proporciona una manera eficiente de manejar un gran volumen de datos de productos. Esta guía explica cómo se descargan y sincronizan los productos dentro de WCPOS.

## Almacenamiento Local de Productos {#local-storage-of-products}

WCPOS obtiene los productos de tu tienda WooCommerce y los almacena localmente en tu dispositivo. Este enfoque significa que una vez que un producto es descargado, no necesita ser obtenido repetidamente. A medida que continúas usando el POS, descargará progresivamente todos los productos de tu tienda WooCommerce, creando una base de datos local de tus productos.

## Descarga por Lotes de Productos {#batch-downloading-of-products}

Para la mayoría de las tiendas, intentar descargar más de unos 50 productos a la vez podría resultar en una alta carga del servidor, tiempos de respuesta más lentos o incluso caídas del servidor. Para prevenir esto, WCPOS está diseñado para descargar productos en pequeños lotes, generalmente alrededor de 50 productos a la vez. Este proceso continúa hasta que todo el inventario ha sido almacenado localmente, asegurando que un gran volumen de datos pueda ser gestionado sin poner un estrés indebido en tu servidor.

## Uso de la Búsqueda de Productos {#using-the-product-search}

Una forma efectiva de iniciar la descarga por lotes de productos es utilizando la función de búsqueda de productos. Cada vez que realizas una búsqueda de productos, el sistema POS obtiene y almacena otro lote de productos. Esta estrategia no solo te ayuda a descargar gradualmente nuevos productos, sino que también mejora la velocidad y eficiencia de la función de búsqueda a medida que más productos se almacenan localmente.

## Conclusión {#conclusion}

Es importante entender que todos tus productos no aparecerán de una vez en tu sistema POS, y esto es completamente normal. Con el tiempo, al usar la búsqueda de productos y otras operaciones del POS, descargarás todos tus productos. Para obtener información más detallada sobre cómo funciona WCPOS, consulta [Arquitectura](/reference/architecture).

WCPOS está diseñado para optimizar la carga del servidor y la eficiencia operativa. La función de descarga progresiva de productos garantiza una experiencia de gestión minorista fluida, eficiente y fácil de usar.

:::tip
Si estás experimentando problemas de sincronización persistentes que una sincronización normal no puede resolver, puedes [borrar todos los datos locales](/support/troubleshooting/clear-local-data) para forzar una nueva descarga de toda tu tienda.
:::

## F.A.Q.

<Accordion>
  <AccordionItem question="¿Cómo hago para que se carguen más productos?">

  La sincronización de productos en el POS puede ser activada de varias maneras:

  - Cuando se carga por primera vez el POS.
  - Cada vez que ejecutas una búsqueda de producto.
  - Aplicando un filtro, por ejemplo, por categoría o etiqueta.
  - Desplazándote hacia la parte inferior de la lista de productos.
  - Presionando el botón de sincronización manualmente (presión corta para sincronizar, **presión larga** para la opción de Borrar y Refrescar).
  - Además, el POS sincronizará automáticamente cada 10 minutos.

  Una vez que todos los productos están descargados, las sincronizaciones futuras solo obtendrán los productos que han sido actualizados desde la última sincronización.

  </AccordionItem>

  <AccordionItem question="He desplazado hasta abajo y aún no se muestran todos los productos">

  Podría haber un par de razones por las cuales no se muestran todos los productos:

  - El POS ocultará los artículos fuera de stock por defecto. Esta configuración se puede alterar en la configuración de visualización de productos si deseas mostrar artículos fuera de stock.
  - La API REST de WooCommerce solo es compatible con los tipos de productos estándar de WooCommerce, que son simple, variable, agrupado y externo. Si estás utilizando un tipo de producto personalizado, podrían no mostrarse en el POS.

  </AccordionItem>
</Accordion>
