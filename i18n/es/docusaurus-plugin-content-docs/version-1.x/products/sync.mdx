Uno de los aspectos significativos del sistema WCPOS (Punto de Venta) es su enfoque en la gestión de datos. Proporciona una manera eficiente de manejar un gran volumen de datos de productos. Esta guía explica cómo se descargan y sincronizan los productos dentro de WCPOS.

## Almacenamiento Local de Productos

WCPOS obtiene productos de tu tienda WooCommerce y los almacena localmente en tu dispositivo. Este enfoque significa que una vez que un producto es descargado, no necesita ser obtenido repetidamente. A medida que continúas utilizando el POS, irá descargando progresivamente todos los productos de tu tienda WooCommerce, creando una base de datos local de tus productos.

## Descarga por Lotes de Productos

Para la mayoría de las tiendas, intentar descargar más de unos 50 productos a la vez podría dar lugar a una alta carga en el servidor, tiempos de respuesta más lentos o incluso caídas del servidor. Para prevenir esto, WCPOS está diseñado para descargar productos en pequeños lotes, generalmente alrededor de 50 productos a la vez. Este proceso continúa hasta que todo el inventario ha sido almacenado localmente, asegurando que se pueda gestionar un gran volumen de datos sin poner una carga indebida en tu servidor.

## Uso de la Búsqueda de Productos

Una forma efectiva de iniciar la descarga por lotes de productos es mediante la función de búsqueda de productos. Cada vez que realizas una búsqueda de productos, el sistema POS obtiene y almacena otro lote de productos. Esta estrategia no solo te ayuda a descargar gradualmente nuevos productos, sino que también mejora la velocidad y eficiencia de la función de búsqueda a medida que más productos se almacenan localmente.

## Conclusión

Es importante entender que todos tus productos no aparecerán de una vez en tu sistema POS, y esto es completamente normal. Con el tiempo, al utilizar la búsqueda de productos y otras operaciones del POS, descargarás todos tus productos. Para obtener información más detallada sobre cómo opera WCPOS, consulta [Arquitectura](/reference/architecture).

WCPOS está diseñado para optimizar la carga del servidor y la eficiencia operativa. La función de descarga progresiva de productos asegura una experiencia de gestión minorista fluida, eficiente y fácil de usar.

## F.A.Q.

<details>

<summary>¿Cómo puedo hacer que se carguen más productos?</summary>

La sincronización de productos en el POS puede ser activada de varias maneras:

- Cuando el POS se carga por primera vez.
- Cada vez que realizas una búsqueda de productos.
- Aplicando un filtro, por ejemplo, por categoría o etiqueta.
- Desplazándote hacia la parte inferior de la lista de productos.
- Presionando el botón de sincronización manualmente (presión corta para sincronizar, **presión larga** para la opción de Clear and Refresh).
- Además, el POS sincronizará automáticamente cada 10 minutos.

Una vez que todos los productos están descargados, las sincronizaciones futuras solo obtendrán los productos que han sido actualizados desde la última sincronización.

</details>

<details>

<summary>
  Me he desplazado al fondo y aún no se muestran todos los productos
</summary>

Podría haber un par de razones por las que no se muestran todos los productos:

- El POS ocultará por defecto los artículos fuera de stock. Esta configuración puede ser alterada en la configuración de visualización de productos si deseas mostrar los artículos fuera de stock.

- La API REST de WooCommerce solo es compatible con los tipos de productos estándar de WooCommerce, que son simple, variable, agrupado y externo. Si estás utilizando un tipo de producto personalizado, es posible que no se muestren en el POS.

</details>
