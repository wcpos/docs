Uno de los aspectos más significativos del sistema WCPOS (Punto de Venta) es su enfoque para la gestión de datos. Proporciona una forma eficiente de manejar un gran volumen de datos de productos. Esta guía explica cómo se descargan y sincronizan los productos dentro de WCPOS.

## Almacenamiento Local de Productos

WCPOS obtiene productos de su tienda WooCommerce y los almacena localmente en su dispositivo. Este enfoque significa que una vez que se descarga un producto, no necesita ser buscado repetidamente. A medida que continúe usando el POS, irá descargando progresivamente todos los productos de su tienda WooCommerce, creando una base de datos local de sus productos.

## Descarga por Lotes de Productos

Para la mayoría de las tiendas, intentar descargar más de unos 50 productos a la vez podría provocar una alta carga del servidor, tiempos de respuesta más lentos o incluso bloqueos del servidor. Para prevenir esto, WCPOS está diseñado para descargar productos en pequeños lotes, generalmente alrededor de 50 productos a la vez. Este proceso continúa hasta que todo el inventario se haya almacenado localmente, asegurando que se pueda gestionar un gran volumen de datos sin poner un estrés indebido en su servidor.

## Usando la Búsqueda de Productos

Una forma efectiva de iniciar la descarga por lotes de productos es utilizando la función de búsqueda de productos. Cada vez que realiza una búsqueda de productos, el sistema POS obtiene y almacena otro lote de productos. Esta estrategia no solo le ayuda a descargar gradualmente nuevos productos, sino que también mejora la velocidad y eficiencia de la función de búsqueda a medida que se almacenan más productos localmente.

## Conclusión

Es importante entender que no todos sus productos aparecerán de inmediato en su sistema POS, y esto es completamente normal. Con el tiempo, mediante el uso de la búsqueda de productos y otras operaciones POS, descargará todos sus productos. Para obtener información más detallada sobre cómo opera WCPOS, consulte [Arquitectura](/reference/architecture).

WCPOS está diseñado para optimizar la carga del servidor y la eficiencia operativa. La característica de descarga progresiva de productos garantiza una experiencia de gestión minorista fluida, eficiente y fácil de usar.

## Preguntas Frecuentes

<details>

<summary>¿Cómo obtengo más productos para cargar?</summary>

La sincronización de productos en el POS se puede activar de varias maneras:

- Cuando se carga inicialmente el POS.
- Cada vez que ejecuta una búsqueda de productos.
- Aplicando un filtro, por ejemplo, por categoría o etiqueta.
- Desplazándose hasta el final de la lista de productos.
- Presionando manualmente el botón de sincronización.
- Además, el POS se sincronizará automáticamente cada 10 minutos.

Una vez que se hayan descargado todos los productos, las sincronizaciones futuras solo recuperarán los productos que se hayan actualizado desde la última sincronización.

</details>

<details>

<summary>
  He llegado al final de la lista y aún no muestra todos los productos
</summary>

Podría haber un par de razones por las que no se muestran todos los productos:

- El POS ocultará los elementos sin existencias de forma predeterminada. Esta configuración se puede modificar en la configuración de visualización de productos si desea mostrar elementos sin existencias.

- La API REST de WooCommerce solo es compatible con los tipos de productos estándar de WooCommerce, que son simples, variables, agrupados y externos. Si está utilizando un tipo de producto personalizado, es posible que no se muestren en el POS.

</details>
