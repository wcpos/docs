:::info Característica Pro
La pantalla de Productos requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden ver y añadir productos al carrito desde el [Panel de Productos del POS](/pos/product-panel), pero no pueden editar el inventario ni los precios.
:::

La pantalla de Productos es una interfaz dedicada a la gestión de inventario. A diferencia del [Panel de Productos del POS](/pos/product-panel) (que es para vender), esta pantalla está diseñada para gestionar tu catálogo de productos.

{/* Screenshot: products-screen.png */}

## Pantalla de Productos vs Panel de Productos del POS

| Característica | Panel de Productos del POS | Pantalla de Productos |
|----------------|----------------------------|-----------------------|
| **Propósito** | Añadir productos al carrito | Gestionar inventario |
| **Stock** | Solo visualizar | Editar en línea |
| **Precios** | Solo visualizar | Editar en línea |
| **Acciones** | Añadir al carrito | Editar, Sincronizar, Eliminar |
| **Disponible en** | Gratis + Pro | Solo Pro |

## Resumen de la Interfaz

### Búsqueda y Filtros

En la parte superior de la pantalla:

- **Barra de búsqueda** - Busca productos por nombre, SKU o código de barras
- **Botones de filtro** - Estado de Stock, Destacados, En Oferta, Categoría, Etiqueta, Marca
- **Configuración de visualización** (<Icon name="sliders" />) - Configurar columnas visibles

### Lista de Productos

El área principal muestra tus productos en una tabla desplazable:

- **Imagen** - Miniatura del producto
- **Nombre del producto** - Con SKU opcional, código de barras, atributos
- **Stock** - Cantidad de stock editable con conmutador "Gestionar"
- **Estado de Stock** - En Stock (verde), Fuera de Stock (rojo), Pedido en espera
- **Categorías** - Insignias de categoría del producto
- **Precios** - Precio actual, precio regular, precio en oferta (todos editables)
- **Acciones** - Menú de tres puntos con Editar, Sincronizar, Eliminar

### Productos Variables

Los productos variables muestran:
- Opciones de atributos (Color, Tamaño, etc.)
- Enlace "Expandir" para ver todas las variaciones
- Filas de variaciones individuales con stock/precios separados

### Pie de Página

- **Estado fiscal** - Base de cálculo fiscal actual
- **Conteo de productos** - "Mostrando X de Y" con botón de sincronización (<Icon name="arrow-rotate-right" />). **Mantén presionado** el botón de sincronización para las opciones de Limpiar y Actualizar

## Características Clave

### Edición de Stock en Línea

Haz clic directamente en la cantidad de stock para editarla:

1. Haz clic en el número de stock
2. Ingresa la nueva cantidad
3. Presiona Enter para guardar

Los cambios se sincronizan automáticamente con WooCommerce.

### Conmutador de Gestión de Stock

El conmutador "Gestionar" controla si WooCommerce rastrea el inventario para este producto:

- **Activado** - Los niveles de stock se rastrean y se decrecen en las ventas
- **Desactivado** - Producto siempre disponible (stock infinito)

### Edición de Precio en Línea

Haz clic en cualquier campo de precio para editar:

- **Precio** - Precio de venta actual
- **Precio Regular** - Precio sin descuento
- **Precio de Oferta** - Precio con descuento (cuando está en oferta)

### Acciones del Producto

Haz clic en el menú de tres puntos (⋮) para cada producto:

- **Editar** - Abrir el modal de edición del producto
- **Sincronizar** - Actualizar este producto desde WooCommerce
- **Eliminar** - Eliminar el producto de la base de datos local

:::note
Eliminar un producto del POS solo lo elimina localmente. El producto permanece en WooCommerce y reaparecerá en la próxima sincronización.
:::

### Columnas Disponibles

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|---------------------------|
| **Imagen** | Miniatura del producto | - |
| **ID** | ID del producto en WooCommerce | - |
| **Producto** | Nombre del producto | SKU, Código de barras, Atributos |
| **Tipo** | Simple, variable, agrupado, etc. | - |
| **SKU** | Unidad de Mantenimiento de Stock | - |
| **Código de barras** | Código de barras del producto | - |
| **Stock** | Cantidad actual (editable) | - |
| **Estado de Stock** | En Stock / Fuera de Stock / Pedido en espera | - |
| **Categorías** | Categorías de producto | - |
| **Etiquetas** | Etiquetas de producto | - |
| **Marcas** | Información de la marca (si el plugin está activo) | - |
| **Costo de Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta actual (editable) | Información fiscal |
| **Precio Regular** | Precio sin descuento (editable) | Información fiscal |
| **Precio de Oferta** | Precio con descuento (editable) | Información fiscal |
| **Fecha de Creación** | Cuándo se creó el producto | - |
| **Fecha de Modificación** | Fecha de última modificación | - |
| **Acciones** | Botones de Editar, Sincronizar, Eliminar | - |

### Restaurar Configuraciones por Defecto

Haz clic para restablecer todas las configuraciones de visualización a sus valores predeterminados originales.

## Filtrando Productos

Utiliza los botones de filtro para limitar tu vista:

- **Estado de Stock** - En Stock, Fuera de Stock, Pedido en espera
- **Destacados** - Productos marcados como destacados
- **En Oferta** - Productos actualmente en oferta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si se está utilizando el plugin de marca)

## Ordenar

Haz clic en cualquier encabezado de columna para ordenar por esa columna. Haz clic nuevamente para invertir el orden de clasificación.
