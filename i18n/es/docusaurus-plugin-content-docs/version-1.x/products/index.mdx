:::info Función Pro
La pantalla de Productos requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden ver y agregar productos al carrito desde el [Panel de Producto POS](/pos/product-panel), pero no pueden editar el inventario o los precios.
:::

La pantalla de Productos es una interfaz dedicada para la gestión de inventarios. A diferencia del [Panel de Producto POS](/pos/product-panel) (que es para vender), esta pantalla está diseñada para gestionar tu catálogo de productos.

{/* Screenshot: products-screen.png */}

## Pantalla de Productos vs Panel de Producto POS {#products-screen-vs-pos-product-panel}

| Característica | Panel de Producto POS | Pantalla de Productos |
|----------------|-------------------------|------------------------|
| **Propósito** | Agregar productos al carrito | Gestionar inventario |
| **Stock** | Solo vista | Editar en línea |
| **Precios** | Solo vista | Editar en línea |
| **Acciones** | Agregar al carrito | Editar, Sincronizar, Eliminar |
| **Disponible en** | Gratis + Pro | Solo Pro |

## Descripción General de la Interfaz {#interface-overview}

### Búsqueda y Filtros {#search-filters}

En la parte superior de la pantalla:

- **Barra de búsqueda** - Busca productos por nombre, SKU o código de barras
- **Botones de filtro** - Estado del Stock, Destacados, En Rebaja, Categoría, Etiqueta, Marca
- **Configuración de visualización** (<Icon name="sliders" />) - Configura las columnas visibles

### Lista de Productos {#product-list}

El área principal muestra tus productos en una tabla desplazable:

- **Imagen** - Miniatura del producto
- **Nombre del producto** - Con SKU opcional, código de barras, atributos
- **Stock** - Cantidad de stock editable con alternar "Gestionar"
- **Estado del Stock** - En Stock (verde), Sin Stock (rojo), Pedido Pendiente
- **Categorías** - Insignias de categoría de producto
- **Precios** - Precio actual, precio regular, precio en rebaja (todos editables)
- **Acciones** - Menú de tres puntos con Editar, Sincronizar, Eliminar

### Productos Variables {#variable-products}

Los productos variables muestran:
- Opciones de atributos (Color, Tamaño, etc.)
- Enlace "Expandir" para ver todas las variaciones
- Filas de variaciones individuales con stock/precios separados

### Pie de Página {#footer}

- **Estado fiscal** - Base de cálculo del impuesto actual
- **Conteo de productos** - "Mostrando X de Y" con botón de sincronización (<Icon name="arrow-rotate-right" />). **Presiona largo** el botón de sincronización para opción de Borrar y Actualizar

## Características Clave {#key-features}

### Edición de Stock en Línea {#inline-stock-editing}

Haz clic directamente en la cantidad de stock para editarla:

1. Haz clic en el número de stock
2. Ingresa la nueva cantidad
3. Presiona Enter para guardar

Los cambios se sincronizan automáticamente con WooCommerce.

### Alternar Gestión de Stock {#manage-stock-toggle}

El alternar "Gestionar" controla si WooCommerce rastrea el inventario para este producto:

- **Activado** - Los niveles de stock se rastrean y disminuyen en las ventas
- **Desactivado** - El producto siempre está disponible (stock infinito)

### Edición de Precios en Línea {#inline-price-editing}

Haz clic en cualquier campo de precio para editar:

- **Precio** - Precio de venta actual
- **Precio Regular** - Precio sin descuento
- **Precio de Rebaja** - Precio con descuento (cuando está en rebaja)

### Acciones del Producto {#product-actions}

Haz clic en el menú de tres puntos (⋮) para cada producto:

- **Editar** - Abrir el modal de edición del producto
- **Sincronizar** - Sincronizar este producto con WooCommerce
- **Eliminar** - Eliminar el producto de la base de datos local

:::note
Eliminar un producto del POS solo lo elimina localmente. El producto permanece en WooCommerce y volverá a aparecer en la próxima sincronización.
:::

### Columnas Disponibles {#available-columns}

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|---------------------------|
| **Imagen** | Miniatura del producto | - |
| **ID** | ID de producto de WooCommerce | - |
| **Producto** | Nombre del producto | SKU, Código de Barras, Atributos |
| **Tipo** | Simple, variable, agrupado, etc. | - |
| **SKU** | Unidad de Mantenimiento de Stock | - |
| **Código de Barras** | Código de barras del producto | - |
| **Stock** | Cantidad actual (editable) | - |
| **Estado del Stock** | En Stock / Sin Stock / Pedido Pendiente | - |
| **Categorías** | Categorías de producto | - |
| **Etiquetas** | Etiquetas de producto | - |
| **Marcas** | Información de marca (si el complemento está activo) | - |
| **Costo de los Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta actual (editable) | Información fiscal |
| **Precio Regular** | Precio sin descuento (editable) | Información fiscal |
| **Precio de Rebaja** | Precio con descuento (editable) | Información fiscal |
| **Fecha de Creación** | Fecha en que se creó el producto | - |
| **Fecha de Modificación** | Fecha de la última modificación | - |
| **Acciones** | Botones Editar, Sincronizar, Eliminar | - |

### Restaurar Configuraciones Predeterminadas {#restore-default-settings}

Haz clic para restablecer todas las configuraciones de visualización a sus valores predeterminados originales.

## Filtrando Productos {#filtering-products}

Utiliza los botones de filtro para reducir tu vista:

- **Estado del Stock** - En Stock, Sin Stock, Pedido Pendiente
- **Destacados** - Productos marcados como destacados
- **En Rebaja** - Productos actualmente en rebaja
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si se está utilizando el complemento de marca)

## Ordenar {#sorting}

Haz clic en cualquier encabezado de columna para ordenar por esa columna. Haz clic nuevamente para invertir el orden de clasificación.
