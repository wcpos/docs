:::info Función Pro
La pantalla de Productos requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden ver y añadir productos al carrito desde el [Panel de Productos POS](/pos/product-panel), pero no pueden editar inventario ni precios.
:::

La pantalla de Productos es una interfaz dedicada a la gestión de inventario. A diferencia del [Panel de Productos POS](/pos/product-panel) (que es para vender), esta pantalla está diseñada para administrar su catálogo de productos.

{/* Screenshot: products-screen.png */}

## Pantalla de Productos vs Panel de Productos POS

| Característica | Panel de Productos POS | Pantalla de Productos |
|---------------|------------------------|----------------------|
| **Propósito** | Añadir productos al carrito | Gestionar inventario |
| **Stock** | Solo ver | Editar en línea |
| **Precios** | Solo ver | Editar en línea |
| **Acciones** | Añadir al carrito | Editar, Sincronizar, Eliminar |
| **Disponible en** | Gratis + Pro | Solo Pro |

## Descripción general de la Interfaz

### Búsqueda y Filtros

En la parte superior de la pantalla:

- **Barra de búsqueda** - Buscar productos por nombre, SKU o código de barras
- **Botones de filtro** - Estado de Stock, Destacados, En Oferta, Categoría, Etiqueta, Marca
- **Configuración de visualización** (<Icon name="sliders" />) - Configurar columnas visibles

### Lista de Productos

El área principal muestra sus productos en una tabla desplazable:

- **Imagen** - Miniatura del producto
- **Nombre del producto** - Con SKU, código de barras y atributos opcionales
- **Stock** - Cantidad de stock editable con interruptor "Gestionar"
- **Estado de Stock** - En Stock (verde), Sin Stock (rojo), Pedido pendiente
- **Categorías** - Insignias de categoría de producto
- **Precios** - Precio actual, precio regular, precio de oferta (todos editables)
- **Acciones** - Menú de tres puntos con Editar, Sincronizar, Eliminar

### Productos Variables

Los productos variables muestran:
- Opciones de atributos (Color, Talla, etc.)
- Enlace "Expandir" para ver todas las variaciones
- Filas de variación individuales con stock y precios separados

### Pie de página

- **Estado fiscal** - Base de cálculo de impuestos actual
- **Recuento de productos** - "Mostrando X de Y" con botón de sincronización

## Características principales

### Edición de Stock en línea

Haga clic directamente en la cantidad de stock para editarla:

1. Haga clic en el número de stock
2. Introduzca la nueva cantidad
3. Presione Entrar para guardar

Los cambios se sincronizan automáticamente con WooCommerce.

### Interruptor de Gestión de Stock

El interruptor "Gestionar" controla si WooCommerce rastrea el inventario de este producto:

- **Activado** - Los niveles de stock se rastrean y disminuyen en las ventas
- **Desactivado** - El producto siempre está disponible (stock infinito)

### Edición de Precios en línea

Haga clic en cualquier campo de precio para editarlo:

- **Precio** - Precio de venta actual
- **Precio Regular** - Precio sin descuento
- **Precio de Oferta** - Precio con descuento (cuando está en oferta)

### Acciones de Producto

Haga clic en el menú de tres puntos (⋮) para cada producto:

- **Editar** - Abrir el modal de edición de producto
- **Sincronizar** - Actualizar este producto desde WooCommerce
- **Eliminar** - Eliminar producto de la base de datos local

:::note
Eliminar un producto del POS solo lo elimina localmente. El producto permanece en WooCommerce y volverá a aparecer en la próxima sincronización.
:::

### Columnas Disponibles

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|----------------------------|
| **Imagen** | Miniatura del producto | - |
| **ID** | ID de producto de WooCommerce | - |
| **Producto** | Nombre del producto | SKU, Código de barras, Atributos |
| **Tipo** | Simple, variable, agrupado, etc. | - |
| **SKU** | Unidad de Mantenimiento de Existencias | - |
| **Código de barras** | Código de barras del producto | - |
| **Stock** | Cantidad actual (editable) | - |
| **Estado de Stock** | En Stock / Sin Stock / Pedido pendiente | - |
| **Categorías** | Categorías de producto | - |
| **Etiquetas** | Etiquetas de producto | - |
| **Marcas** | Información de marca (si el plugin está activo) | - |
| **Costo de Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta actual (editable) | Información de impuestos |
| **Precio Regular** | Precio sin descuento (editable) | Información de impuestos |
| **Precio de Oferta** | Precio con descuento (editable) | Información de impuestos |
| **Fecha de Creación** | Cuándo se creó el producto | - |
| **Fecha de Modificación** | Fecha de última modificación | - |
| **Acciones** | Botones de Editar, Sincronizar, Eliminar | - |

### Restaurar Configuración Predeterminada

Haga clic para restablecer todas las configuraciones de visualización a sus valores originales.

## Filtrar Productos

Use los botones de filtro para reducir su vista:

- **Estado de Stock** - En Stock, Sin Stock, Pedido pendiente
- **Destacados** - Productos marcados como destacados
- **En Oferta** - Productos actualmente en oferta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si usa el plugin de marcas)

## Ordenar

Haga clic en cualquier encabezado de columna para ordenar por esa columna. Haga clic de nuevo para invertir el orden de clasificación.
