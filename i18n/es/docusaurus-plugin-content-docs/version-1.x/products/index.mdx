:::info Función Pro
La pantalla de Productos requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden ver y añadir productos al carrito desde el [Panel de Productos POS](/pos/product-panel), pero no pueden editar el inventario ni los precios.
:::

La pantalla de Productos es una interfaz dedicada a la gestión de inventario. A diferencia del [Panel de Productos POS](/pos/product-panel) (que es para vender), esta pantalla está diseñada para gestionar tu catálogo de productos.

{/* Screenshot: products-screen.png */}

## Pantalla de Productos vs Panel de Productos POS

| Función | Panel de Productos POS | Pantalla de Productos |
|---------|-----------------------|-----------------------|
| **Propósito** | Añadir productos al carrito | Gestionar inventario |
| **Inventario** | Solo ver | Editar en línea |
| **Precios** | Solo ver | Editar en línea |
| **Acciones** | Añadir al carrito | Editar, Sincronizar, Eliminar |
| **Disponible en** | Gratis + Pro | Solo Pro |

## Descripción General de la Interfaz

### Búsqueda y Filtros

En la parte superior de la pantalla:

- **Barra de búsqueda** - Encuentra productos por nombre, SKU o código de barras
- **Botones de filtro** - Estado del inventario, Destacados, En oferta, Categoría, Etiqueta, Marca
- **Configuraciones de visualización** (<Icon name="sliders" />) - Configura las columnas visibles

### Lista de Productos

El área principal muestra tus productos en una tabla desplazable:

- **Imagen** - Miniatura del producto
- **Nombre del producto** - Con SKU, código de barras y atributos opcionales
- **Inventario** - Cantidad de inventario editable con interruptor "Gestionar"
- **Estado del Inventario** - En Stock (verde), Agotado (rojo), Pedido pendiente
- **Categorías** - Insignias de categorías de productos
- **Precios** - Precio actual, precio regular, precio de venta (todos editables)
- **Acciones** - Menú de tres puntos con Editar, Sincronizar, Eliminar

### Productos Variables

Los productos variables muestran:
- Opciones de atributos (Color, Tamaño, etc.)
- Enlace "Expandir" para ver todas las variaciones
- Filas de variaciones individuales con inventario/precios separados

### Pie de Página

- **Estado fiscal** - Base de cálculo del impuesto actual
- **Conteo de productos** - "Mostrando X de Y" con botón de sincronización

## Funciones Clave

### Edición de Inventario en Línea

Haz clic directamente en la cantidad de inventario para editarla:

1. Haz clic en el número de inventario
2. Introduce la nueva cantidad
3. Presiona Enter para guardar

Los cambios se sincronizan automáticamente con WooCommerce.

### Interruptor de Gestión de Inventario

El interruptor "Gestionar" controla si WooCommerce rastrea el inventario para este producto:

- **Activado** - Los niveles de inventario se rastrean y se decrecen en ventas
- **Desactivado** - El producto siempre está disponible (inventario infinito)

### Edición de Precio en Línea

Haz clic en cualquier campo de precio para editar:

- **Precio** - Precio de venta actual
- **Precio Regular** - Precio sin descuento
- **Precio de Venta** - Precio descontado (cuando está en oferta)

### Acciones del Producto

Haz clic en el menú de tres puntos (⋮) para cada producto:

- **Editar** - Abre el modal de edición del producto
- **Sincronizar** - Actualiza este producto desde WooCommerce
- **Eliminar** - Elimina el producto de la base de datos local

:::note
Eliminar un producto del POS solo lo quita localmente. El producto permanece en WooCommerce y reaparecerá en la próxima sincronización.
:::

### Columnas Disponibles

| Columna | Descripción | Opciones de Visualización |
|---------|-------------|---------------------------|
| **Imagen** | Miniatura del producto | - |
| **ID** | ID del producto de WooCommerce | - |
| **Producto** | Nombre del producto | SKU, Código de barras, Atributos |
| **Tipo** | Simple, variable, agrupado, etc. | - |
| **SKU** | Unidad de Mantenimiento de Inventario | - |
| **Código de barras** | Código de barras del producto | - |
| **Inventario** | Cantidad actual (editable) | - |
| **Estado del Inventario** | En stock / Agotado / Pedido pendiente | - |
| **Categorías** | Categorías del producto | - |
| **Etiquetas** | Etiquetas del producto | - |
| **Marcas** | Información de la marca (si el plugin está activo) | - |
| **Costo de Bienes Vendidos** | Precio de costo del producto | - |
| **Precio** | Precio de venta actual (editable) | Información fiscal |
| **Precio Regular** | Precio sin descuento (editable) | Información fiscal |
| **Precio de Venta** | Precio descontado (editable) | Información fiscal |
| **Fecha de Creación** | Cuando se creó el producto | - |
| **Fecha de Modificación** | Última fecha de modificación | - |
| **Acciones** | Botones Editar, Sincronizar, Eliminar | - |

### Restaurar Configuraciones Predeterminadas

Haz clic para restablecer todas las configuraciones de visualización a sus valores predeterminados originales.

## Filtrando Productos

Usa los botones de filtro para reducir tu vista:

- **Estado del Inventario** - En stock, Agotado, Pedido pendiente
- **Destacados** - Productos marcados como destacados
- **En Oferta** - Productos actualmente en oferta
- **Categoría** - Filtrar por categoría de producto
- **Etiqueta** - Filtrar por etiqueta de producto
- **Marca** - Filtrar por marca (si se utiliza el plugin de marcas)

## Ordenación

Haz clic en cualquier encabezado de columna para ordenarlo por esa columna. Haz clic nuevamente para invertir el orden de la ordenación.
