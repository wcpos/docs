Algunos países exigen legalmente que el software de punto de venta esté **certificado** contra el fraude fiscal, o que se integre con una autoridad fiscal nacional. Esta página resume la situación actual de WCPOS para que pueda verificar si se ajusta a su jurisdicción **antes** de depender de él.

:::warning WCPOS no es un POS fiscal certificado
WCPOS **no** posee actualmente certificaciones fiscales para ningún mercado regulado. Si su país requiere software POS certificado, confirme los requisitos locales antes de utilizar WCPOS para ventas reales. WCPOS actúa como middleware para WooCommerce — la facturación y el IVA se gestionan en última instancia a nivel de WooCommerce.
:::

:::info Estamos trabajando en ello
El cumplimiento fiscal es un área de desarrollo activa y continua — nuestro objetivo es que WCPOS sea compatible con todos los países en los que operan nuestros usuarios. El estado que se muestra a continuación refleja la situación **actual**, no hacia dónde nos dirigimos. Si el cumplimiento normativo en su región es importante, puede comunicárnoslo en la [comunidad](https://wcpos.com/discord) — nos ayuda a priorizar qué mercados abordar a continuación.
:::

## Cómo gestiona WCPOS los impuestos {#how-wcpos-handles-tax}

WCPOS calcula los impuestos localmente en la aplicación utilizando las tasas descargadas de WooCommerce, y luego crea el pedido en WooCommerce. Esto es rápido y funciona sin conexión, pero implica que la lógica fiscal certificada o del lado del servidor (como la de algunos plugins de cumplimiento) solo se ejecuta cuando el pedido llega al servidor. La acción **"Guardar en el servidor"** y las pasarelas de **Pago web** / **Factura por correo electrónico** envían el pedido a través del proceso de pago estándar de WooCommerce, donde se aplican las reglas fiscales y de cumplimiento del lado del servidor.

## Estado por país {#country-status}

### Francia — NF525 {#france-nf525}

WCPOS **no está certificado** para NF525. La legislación francesa exige que los sistemas POS estén certificados contra el fraude, lo que afecta la viabilidad de ventas en Francia. Dado que WCPOS es un middleware para WooCommerce, la facturación y el cumplimiento del IVA se gestionan a nivel de WooCommerce — consulte la guía de WooCommerce sobre las [leyes francesas de prevención del fraude en el IVA](https://woocommerce.com/posts/french-vat-fraud-prevention-laws-how-to-certify-invoices-with-woocommerce/).

### España — VERIFACTU {#spain-verifactu}

WCPOS **no está certificado** para VERIFACTU. España exige que el software POS esté certificado; el plazo de VERIFACTU se ha **pospuesto a 2027**. El equipo conoce el requisito, pero no se ha establecido un calendario de certificación.

### Noruega — Kassasystemloven {#norway-kassasystemloven}

WCPOS **no está certificado** para la Ley Noruega de Cajas Registradoras (_Kassasystemloven_), y no se dispone de una declaración del proveedor. Algunos comerciantes en Noruega utilizan el POS, pero se recomienda confirmar las obligaciones legales propias.

### Chile — SII {#chile-sii}

**No existe integración** con el Servicio de Impuestos Internos (SII) de Chile para recibos de venta validados (_boletas_). Las plantillas de recibos son plantillas estándar y personalizables de WooCommerce, pero no incluyen cumplimiento con el SII.

### UE — Ventanilla Única (OSS) {#eu-oss}

El POS **no** es compatible de forma nativa con el régimen de IVA de Ventanilla Única (OSS) de la UE para ventas transfronterizas, ya que calcula los impuestos localmente a partir de las tasas descargadas.

**Soluciones alternativas:**
- Utilizar **Guardar en el servidor** antes del cobro para activar un recálculo de impuestos del lado del servidor a través del plugin de OSS, o
- Utilizar la pasarela **Cobro web** o **Enviar factura por correo electrónico** para realizar el pago a través del proceso de pago estándar de WooCommerce, donde se aplican las normas de OSS.

## Otros mercados regulados {#other-regulated-markets}

Para cualquier jurisdicción que requiera software de POS certificado, es necesario verificar primero los requisitos locales. WCPOS no posee actualmente certificaciones para mercados regulados. Si el estado de certificación es relevante para la decisión de compra, se recomienda consultar en la [comunidad](https://wcpos.com/discord) antes de comprometerse.
