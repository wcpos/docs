Para garantizar la funcionalidad adecuada y el rendimiento óptimo del plugin WCPOS, asegúrese de que su entorno cumpla con los siguientes requisitos mínimos:

## Requisitos del Servidor {#server-requirements}

- **WordPress:** Versión 5.6 o superior
- **WooCommerce:** Versión 5.3 o superior
- **PHP:** Versión 7.2 o superior

Cumplir con estos requisitos mínimos asegura la compatibilidad y estabilidad entre el plugin WCPOS y su entorno de WordPress. También garantiza que pueda beneficiarse de las últimas características y mejoras proporcionadas por el plugin.

## Requisitos del Navegador {#browser-requirements}

- **Navegador web moderno:** El plugin WCPOS está diseñado para funcionar con navegadores web modernos, incluyendo Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. Asegúrese de que está utilizando la última versión estable de su navegador preferido.
- **JavaScript habilitado:** El plugin WCPOS depende de JavaScript para sus características interactivas. Asegúrese de que JavaScript esté habilitado en la configuración de su navegador.
- **Sin modo de navegación privada:** El navegador no debe estar en modo de navegación privada, ya que esto puede restringir el acceso a la base de datos IndexedDB (IDB) del navegador. El modo de navegación privada puede impedir el funcionamiento correcto de ciertas características en el plugin WCPOS.

## Instalación del Plugin {#installing-the-plugin}

1. Vaya a `WP Admin > Plugins > Añadir Nuevo`
2. Busque "WooCommerce POS"
3. Haga clic en **Instalar Ahora**, luego en **Activar**

Alternativamente, puede descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) y subirlo manualmente.

## Accediendo al POS {#accessing-the-pos}

Después de la activación, puede acceder al POS en:

- **Web:** `su_dominio.com/pos`
- **Escritorio:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Al cumplir con los requisitos mínimos y utilizar un navegador web compatible, puede tener una experiencia sin problemas al usar el plugin WCPOS y aprovechar su completo potencial para gestionar sus operaciones de punto de venta de manera eficiente.
