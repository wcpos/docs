Para garantizar el funcionamiento adecuado y un rendimiento óptimo del plugin de WCPOS, asegúrese de que su entorno cumpla con los siguientes requisitos mínimos:

## Requisitos del Servidor

- **WordPress:** Versión 5.6 o superior
- **WooCommerce:** Versión 5.3 o superior
- **PHP:** Versión 7.2 o superior

Cumplir con estos requisitos mínimos garantiza la compatibilidad y estabilidad entre el plugin de WCPOS y su entorno de WordPress. También garantiza que pueda beneficiarse de las últimas características y mejoras proporcionadas por el plugin.

## Requisitos del Navegador

- **Navegador web moderno:** El plugin de WCPOS está diseñado para funcionar con navegadores web modernos, incluyendo Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. Asegúrese de estar utilizando la última versión estable de su navegador preferido.
- **JavaScript habilitado:** El plugin de WCPOS se basa en JavaScript para sus funciones interactivas. Asegúrese de que JavaScript esté habilitado en la configuración de su navegador.
- **Sin modo de navegación privada:** El navegador no debe estar en modo de navegación privada, ya que puede restringir el acceso a la base de datos IndexedDB (IDB) del navegador. El modo de navegación privada puede impedir el funcionamiento adecuado de ciertas características del plugin de WCPOS.

## Instalación del Plugin

1. Vaya a `WP Admin > Plugins > Add New`
2. Busque "WooCommerce POS"
3. Haga clic en **Install Now**, luego **Activate**

Alternativamente, puede descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) e instalarlo manualmente.

## Acceso al POS

Después de la activación, puede acceder al POS en:

- **Web:** `sudominio.com/pos`
- **Escritorio/Móvil:** Descargue la aplicación desde [wcpos.com](https://wcpos.com) y conéctese a su tienda

Cumpliendo con los requisitos mínimos y utilizando un navegador web compatible, podrá tener una experiencia fluida utilizando el plugin de WCPOS y aprovechar todo su potencial para gestionar eficientemente sus operaciones de punto de venta.
