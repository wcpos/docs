Para garantizar el funcionamiento correcto y el rendimiento óptimo del plugin WCPOS, asegúrate de que tu entorno cumpla con los siguientes requisitos mínimos:

## Requisitos del servidor {#server-requirements}

El plugin WCPOS se ejecuta en tu sitio de WordPress, que debe cumplir estos mínimos:

| Componente  | Versión mínima  | Notas               |
| ----------- | --------------- | ------------------- |
| WordPress   | 5.6             | —                   |
| WooCommerce | 5.3             | —                   |
| PHP         | 7.4             | se recomienda 8.x   |

Cumplir con estos mínimos garantiza la compatibilidad y estabilidad con tu entorno de WordPress, y el acceso a las últimas funciones de WCPOS.

## Requisitos del navegador {#browser-requirements}

La versión web del POS funciona en cualquier navegador moderno. Utiliza la última versión estable:

| Navegador       | Versión compatible |
| --------------- | ------------------ |
| Google Chrome   | Última estable     |
| Mozilla Firefox | Última estable     |
| Safari          | Última estable     |
| Microsoft Edge  | Última estable     |

:::warning Evita el modo privado / incógnito
La navegación privada puede bloquear el acceso a la base de datos IndexedDB del navegador, que WCPOS utiliza para almacenar datos sin conexión — algunas funciones no funcionarán correctamente. JavaScript también debe estar habilitado.
:::

## Requisitos de la aplicación {#app-requirements}

WCPOS también está disponible como aplicación de escritorio y móvil. Cada una está creada para los siguientes sistemas operativos mínimos:

| Plataforma          | SO mínimo                 | Arquitecturas              |
| ------------------- | ------------------------- | -------------------------- |
| Escritorio Windows  | Windows 10                | x64, ARM64                 |
| Escritorio macOS    | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS        | iOS 16.4                  | —                          |
| Android             | Android 10 (API level 29) | —                          |

:::note Permisos de impresión
La impresión por Bluetooth y por red requiere otorgar a la aplicación los permisos de Bluetooth y de red local correspondientes cuando se soliciten.
:::

## Instalación del plugin {#installing-the-plugin}

1. Ir a `WP Admin > Plugins > Add Plugin`
2. Buscar "wcpos"
3. Hacer clic en **Install Now** y luego en **Activate**

Alternativamente, se puede descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) y subirlo manualmente.

## Acceso al POS {#accessing-the-pos}

Después de la activación, se puede acceder al POS en:

- **Web:** `yourdomain.com/pos`
- **Escritorio:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Al cumplir con los requisitos mínimos y utilizar un navegador web compatible, se puede disfrutar de una experiencia fluida con el plugin WCPOS y aprovechar todo su potencial para gestionar las operaciones del punto de venta de manera eficiente.

Si tienes alguna pregunta adicional o necesitas asistencia, no dudes en comunicarte con nuestro equipo de soporte publicando tu consulta en el [chat de Discord](https://wcpos.com/discord) o enviando un correo electrónico a support@wcpos.com.
