Para asegurar el correcto funcionamiento y el rendimiento óptimo del plugin WCPOS, por favor asegúrate de que tu entorno cumpla con los siguientes requisitos mínimos:

## Requisitos del Servidor {#server-requirements}

- **WordPress:** Versión 5.6 o superior
- **WooCommerce:** Versión 5.3 o superior
- **PHP:** Versión 7.2 o superior

Cumplir con estos requisitos mínimos asegura la compatibilidad y estabilidad entre el plugin WCPOS y tu entorno de WordPress. También asegura que puedes beneficiarte de las últimas características y mejoras proporcionadas por el plugin.

## Requisitos del Navegador {#browser-requirements}

- **Navegador web moderno:** El plugin WCPOS está diseñado para funcionar con navegadores web modernos, incluyendo Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. Por favor, asegúrate de estar utilizando la última versión estable de tu navegador preferido.
- **JavaScript habilitado:** El plugin WCPOS depende de JavaScript para sus funciones interactivas. Asegúrate de que JavaScript esté habilitado en la configuración de tu navegador.
- **Sin modo de navegación privada:** El navegador no debe estar en modo de navegación privada ya que esto podría restringir el acceso a la base de datos IndexedDB (IDB) del navegador. El modo de navegación privada puede impedir el correcto funcionamiento de ciertas características del plugin WCPOS.

## Instalando el Plugin {#installing-the-plugin}

1. Ve a `WP Admin > Plugins > Añadir Plugin`
2. Busca "wcpos"
3. Haz clic en **Instalar Ahora**, luego en **Activar**

Alternativamente, puedes descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) y subirlo manualmente.

## Accediendo al POS {#accessing-the-pos}

Después de la activación, puedes acceder al POS en:

- **Web:** `tudominio.com/pos`
- **Escritorio:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Cumpliendo con los requisitos mínimos y utilizando un navegador web compatible, puedes tener una experiencia sin interrupciones al usar el plugin WCPOS y aprovechar su máximo potencial para gestionar tus operaciones de punto de venta de manera eficiente.

Si tienes alguna pregunta adicional o necesitas asistencia, no dudes en ponerte en contacto con nuestro equipo de soporte publicando en el [chat de Discord](https://wcpos.com/discord) o enviándolos por correo electrónico a support@wcpos.com.
