Para garantizar el funcionamiento correcto y el rendimiento óptimo del plugin WCPOS, asegúrese de que su entorno cumpla los siguientes requisitos mínimos:

## Requisitos del servidor {#server-requirements}

El plugin WCPOS se ejecuta en su sitio WordPress, que debe cumplir estos requisitos mínimos:

| Componente   | Versión mínima | Notas           |
| ----------- | --------------- | --------------- |
| WordPress   | 5.6             | —               |
| WooCommerce | 5.3             | —               |
| PHP         | 7.4             | 8.x recomendado |

Cumplir estos requisitos mínimos garantiza la compatibilidad y la estabilidad con su entorno WordPress, así como el acceso a las funciones más recientes de WCPOS.

## Requisitos del navegador {#browser-requirements}

La versión web del POS funciona en cualquier navegador moderno. Use la última versión estable:

| Navegador       | Versión compatible |
| --------------- | ----------------- |
| Chrome de Google | Última estable     |
| Firefox de Mozilla | Última estable     |
| Safari de Apple | Última estable     |
| Edge de Microsoft | Última estable     |

:::warning Evite el modo privado / de incógnito
La navegación privada puede bloquear el acceso a la base de datos IndexedDB del navegador, que WCPOS utiliza para almacenar datos sin conexión; algunas funciones no funcionarán correctamente. JavaScript también debe estar activado.
:::

## Requisitos de la aplicación {#app-requirements}

WCPOS también está disponible como aplicación de escritorio y móvil. Cada aplicación está diseñada para los siguientes sistemas operativos mínimos:

| Plataforma      | SO mínimo                 | Arquitecturas              |
| --------------- | ------------------------- | -------------------------- |
| Escritorio de Windows | Windows 10                | x64, ARM64                 |
| Escritorio de macOS   | macOS 12 (Monterey)       | Intel (x64), Apple Silicon |
| iOS / iPadOS    | iOS 16.4                  | —                          |
| Dispositivo Android         | Android 10 (nivel de API 29) | —                          |

:::note Permisos de impresión
La impresión por Bluetooth y red requiere conceder a la aplicación los permisos correspondientes de Bluetooth y red local cuando se soliciten.
:::

## Instalación del plugin {#installing-the-plugin}

1. Vaya a `WP Admin > Plugins > Add Plugin`
2. Busque "wcpos"
3. Haga clic en **Instalar ahora** y luego en **Activar**

También es posible descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) y subirlo manualmente.

## Acceso al POS {#accessing-the-pos}

Después de la activación, se puede acceder al POS en:

- **Web:** `yourdomain.com/pos`
- **Escritorio:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [dispositivos Android](https://play.google.com/apps/testing/com.wcpos.main)

Al cumplir los requisitos mínimos y utilizar un navegador web compatible, es posible disfrutar de una experiencia fluida con el plugin WCPOS y aprovechar todo su potencial para gestionar de forma eficiente las operaciones de punto de venta.
