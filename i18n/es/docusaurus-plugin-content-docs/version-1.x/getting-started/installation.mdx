Para garantizar el funcionamiento correcto y el rendimiento óptimo del plugin WCPOS, es necesario asegurarse de que el entorno cumple con los siguientes requisitos mínimos:

## Requisitos del servidor {#server-requirements}

- **WordPress:** Versión 5.6 o superior
- **WooCommerce:** Versión 5.3 o superior
- **PHP:** Versión 7.4 o superior (se recomienda 8.x)

Cumplir con estos requisitos mínimos garantiza la compatibilidad y estabilidad entre el plugin WCPOS y el entorno de WordPress. También permite aprovechar las últimas funcionalidades y mejoras proporcionadas por el plugin.

## Requisitos del navegador {#browser-requirements}

- **Navegador web moderno:** El plugin WCPOS está diseñado para funcionar con navegadores web modernos, incluyendo Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. Es necesario utilizar la última versión estable del navegador preferido.
- **JavaScript habilitado:** El plugin WCPOS depende de JavaScript para sus funciones interactivas. Es necesario asegurarse de que JavaScript esté habilitado en la configuración del navegador.
- **Sin modo de navegación privada:** El navegador no debe estar en modo de navegación privada, ya que esto puede restringir el acceso a la base de datos IndexedDB (IDB) del navegador. El modo de navegación privada puede impedir el funcionamiento correcto de ciertas funciones del plugin WCPOS.

## Instalación del plugin {#installing-the-plugin}

1. Ir a `WP Admin > Plugins > Add Plugin`
2. Buscar "wcpos"
3. Hacer clic en **Instalar ahora** y luego en **Activar**

Alternativamente, se puede descargar el plugin desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/) y subirlo manualmente.

## Acceso al POS {#accessing-the-pos}

Después de la activación, se puede acceder al POS en:

- **Web:** `yourdomain.com/pos`
- **Escritorio:** [Windows](https://updates.wcpos.com/electron/download/win32-x64), [Mac (Intel)](https://updates.wcpos.com/electron/download/darwin-x64), [Mac (Apple Silicon)](https://updates.wcpos.com/electron/download/darwin-arm64)
- **Móvil:** [iOS](https://testflight.apple.com/join/JGBdVRrW), [Android](https://play.google.com/apps/testing/com.wcpos.main)

Al cumplir con los requisitos mínimos y utilizar un navegador web compatible, se puede disfrutar de una experiencia fluida con el plugin WCPOS y aprovechar todo su potencial para gestionar las operaciones del punto de venta de manera eficiente.
