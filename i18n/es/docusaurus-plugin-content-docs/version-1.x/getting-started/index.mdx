## ¿Qué es WCPOS?

WCPOS es un sistema de Punto de Venta (POS) para WooCommerce. Te permite vender productos de tu tienda WooCommerce en persona, en una ubicación física, utilizando un navegador web o una aplicación dedicada.

**Disponible en:**

- **Web:** Accede instantáneamente en `tudominio.com/pos`
- **Escritorio:** Descarga para Windows, macOS y Linux desde [wcpos.com](https://wcpos.com)
- **Móvil:** Aplicaciones para iOS y Android (próximamente)

## Comienza en 5 Minutos

### 1. Instalar el Plugin

Instala el plugin gratuito de [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) en tu sitio WordPress:

1. Ve a `WP Admin > Plugins > Add New`
2. Busca "WooCommerce POS"
3. Haz clic en **Instalar Ahora**, luego **Activar**

Consulta [Instalación](/getting-started/installation) para conocer los requisitos detallados.

### 2. Acceder al POS

**Usuarios Web:**
Navega a `tudominio.com/pos` en tu navegador. Se te pedirá iniciar sesión con tus credenciales de WordPress.

**Usuarios de Escritorio/Móvil:**
1. Descarga la aplicación desde [wcpos.com](https://wcpos.com)
2. Introduce la URL de tu tienda y haz clic en **Conectar**
3. Inicia sesión con tus credenciales de WordPress

Consulta [Conectar](/getting-started/connect) para obtener detalles sobre la gestión de múltiples tiendas y usuarios.

### 3. Esperar la Sincronización Inicial

WCPOS descarga productos en pequeños lotes (alrededor de 50 a la vez) para evitar sobrecargar tu servidor. Los productos se descargan progresivamente mientras buscas, desplazas o filtras, no necesitas esperar a que todo se sincronice antes de comenzar a vender. Consulta [Sincronización de Productos](/products/sync) para más detalles.

### 4. Comenzar a Vender

- Busca o escanea productos para añadirlos al carrito
- Selecciona un cliente o usa "Invitado"
- Haz clic en **Pagar** para procesar el pago
- Imprime o envía el recibo por correo electrónico

## Características Principales

- **Búsqueda local rápida** - Los productos se almacenan localmente para resultados instantáneos
- **Navegación sin conexión** - Busca productos y clientes sin internet (completar pedidos requiere conectividad)
- **Escaneo de código de barras** - Usa cualquier escáner de código de barras USB o Bluetooth
- **Múltiples métodos de pago** - Efectivo, tarjeta y pasarelas personalizadas
- **Impresión de recibos** - Imprime o envía recibos personalizables
- **Soporte multi-tienda** - Conéctate a múltiples tiendas WooCommerce

## Características Pro

[WCPOS Pro](https://wcpos.com/pro) añade capacidades adicionales:

- **Gestión de productos** - Edita stock y precios directamente en el POS
- **Historial de pedidos** - Visualiza y gestiona pedidos anteriores
- **Gestión de clientes** - Añade y edita detalles de clientes
- **Informes de ventas** - Informes de fin de día con opción de impresión
- **Pasarelas de pago adicionales** - Stripe Terminal, SumUp y más
