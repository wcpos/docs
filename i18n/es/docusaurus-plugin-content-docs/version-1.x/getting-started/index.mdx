## ¿Qué es WCPOS? {#what-is-wcpos}

WCPOS es un sistema de Punto de Venta (POS) para WooCommerce. Te permite vender productos de tu tienda WooCommerce en persona, en una ubicación física, utilizando un navegador web o una aplicación dedicada.

## Comienza en 5 minutos {#get-started-in-5-minutes}

<Steps>

<StepCard number={1} title="Instala el Plugin">

Instala el [plugin WCPOS](https://wordpress.org/plugins/woocommerce-pos/) gratuito en tu sitio de WordPress:

1. Ve a **WP Admin → Plugins → Añadir Nuevo**
2. Busca "WooCommerce POS"
3. Haz clic en **Instalar Ahora**, luego en **Activar**

Consulta [Instalación](/getting-started/installation) para requisitos detallados.

</StepCard>

<StepCard number={2} title="Accede al POS">

**Usuarios de Web:** Navega a `yourdomain.com/pos` en tu navegador. Se te pedirá que inicies sesión con tus credenciales de WordPress.

**Usuarios de Escritorio/Móvil:** Descarga la aplicación, ingresa la URL de tu tienda, haz clic en **Conectar** e inicia sesión con tus credenciales de WordPress.

Consulta [Conectar](/getting-started/connect) para detalles sobre cómo gestionar múltiples tiendas y usuarios.

</StepCard>

<StepCard number={3} title="Espera la Sincronización Inicial">

WCPOS descarga productos en pequeños lotes para evitar sobrecargar tu servidor. Los productos se descargan progresivamente a medida que buscas, desplazas o filtras; no necesitas esperar a que todo se sincronice antes de comenzar a vender.

Consulta [Sincronización de Productos](/products/sync) para más detalles.

</StepCard>

<StepCard number={4} title="Comienza a Vender" isLast>

- Busca o escanea productos para añadirlos al carrito
- Selecciona un cliente o usa "Invitado"
- Haz clic en **Checkout** para procesar el pago
- Imprime o envía por correo el recibo

</StepCard>

</Steps>

## Características Clave {#key-features}

<FeatureGrid>
  <FeatureCard icon="search" title="Búsqueda Local Rápida">
    Los productos se almacenan localmente para resultados de búsqueda instantáneos
  </FeatureCard>
  <FeatureCard icon="wifi" title="Navegación Offline">
    Busca productos y clientes sin conexión a internet (los pedidos requieren conectividad)
  </FeatureCard>
  <FeatureCard icon="barcode" title="Escaneo de Códigos de Barras">
    Usa cualquier escáner de códigos de barras USB o Bluetooth
  </FeatureCard>
  <FeatureCard icon="credit-card" title="Pagos en Efectivo y Tarjeta">
    Soporte integrado para pagos en efectivo y pagos básicos con tarjeta
  </FeatureCard>
  <FeatureCard icon="print" title="Impresión de Recibos">
    Imprime o envía por correo recibos personalizables
  </FeatureCard>
  <FeatureCard icon="calculator" title="Impuesto Automático">
    Cálculos de impuestos basados en tus configuraciones fiscales de WooCommerce
  </FeatureCard>
</FeatureGrid>
