# DB02002: Registro No Encontrado {#db02002-record-not-found}

## Qué Significa Esto {#what-this-means}

El registro solicitado no existe en la base de datos local. El POS intentó acceder a un producto, customer, pedido u otro registro que no está almacenado localmente.

## Causas Comunes {#common-causes}

- **Aún no sincronizado** — Los datos aún no se han descargado
- **Eliminado en el servidor** — El registro fue eliminado en WooCommerce
- **Sincronización no completa** — La sincronización inicial aún se está ejecutando
- **Filtro aplicado** — Registro excluido por filtros activos

## Cómo Solucionarlo {#how-to-fix}

### 1. Esperar la Sincronización {#1-wait-for-sync}

Si comenzó recientemente:
- Esperar a que se complete la sincronización inicial
- Verificar el indicador de progreso de sincronización
- El registro debería aparecer una vez sincronizado

### 2. Refrescar/Re-sincronizar {#2-refreshre-sync}

Activar una actualización de datos:
- Usar el botón de refrescar/sync (<Icon name="arrow-rotate-right" />) - presionar brevemente para sincronizar
- **Presionar largo** el botón de sincronización para la opción de Limpiar y Refrescar
- Descargar los últimos datos del servidor
- Esperar a que se complete

### 3. Verificar WooCommerce {#3-check-woocommerce}

Verificar que el registro exista en el servidor:
- Iniciar sesión en WordPress Admin
- Revisar Productos, Pedidos o Customers
- Confirmar que el ítem aún exista

### 4. Comprobar Filtros {#4-check-filters}

El registro podría estar filtrado:
- Revisar los filtros activos en el POS
- Verificar filtros de categoría
- Confirmar la configuración de visibilidad

### 5. Limpiar y Re-sincronizar {#5-clear-and-re-sync}

Si la sincronización parece estar atascada:
1. Limpiar datos locales
2. Iniciar sesión nuevamente
3. Completar una nueva sincronización

## Escenarios Comunes {#common-scenarios}

- **Producto no visible:** Verificar si está publicado y visible
- **Customer no encontrado:** Puede que aún no se haya sincronizado
- **Pedido faltante:** Podría estar fuera del rango de fechas de sincronización
