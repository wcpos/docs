# Códigos de Error

Cuando WCPOS encuentra un problema, muestra un código de error que ayuda a identificar el problema. Cada código de error sigue el formato `[DOMINIO][CATEGORÍA][CÓDIGO_ESPECÍFICO]` para ayudarle a comprender rápidamente el tipo y la naturaleza del error.

## Formato del Código de Error

| Componente | Descripción | Ejemplo |
|-----------|-------------|---------|
| **Dominio** | El área del sistema | `API`, `DB`, `PY`, `SY` |
| **Categoría** | La categoría del error | `01`, `02`, `03`, etc. |
| **Código Específico** | El identificador único | `001`, `002`, `003`, etc. |

**Ejemplo:** `API04001` = API (dominio) + 04 (errores de respuesta) + 001 (formato de respuesta inválido)

## Dominios de Errores

### [Errores de API](./api)

Los errores de API ocurren al comunicarse con su servidor de WooCommerce.

| Categoría | Rango de Código | Descripción |
|----------|------------|-------------|
| Conexión | API01xxx | Problemas de red y conectividad |
| Autenticación | API02xxx | Problemas de inicio de sesión, tokens y permisos |
| Solicitud | API03xxx | Problemas con solicitudes salientes |
| Respuesta | API04xxx | Problemas con respuestas del servidor |
| Plugin/WordPress | API05xxx | Problemas con plugins de WordPress o WooCommerce |
| Configuración | API06xxx | Problemas de configuración |

### [Errores de Base de Datos](./db)

Los errores de base de datos ocurren al almacenar o recuperar datos localmente.

| Categoría | Rango de Código | Descripción |
|----------|------------|-------------|
| Conexión | DB01xxx | Problemas de conexión y transacción de base de datos |
| Datos | DB02xxx | Problemas de registros y restricciones |
| Consulta | DB03xxx | Problemas de sintaxis de consulta y tipos de datos |

### [Errores de Pago](./py)

Los errores de pago ocurren durante el pago y el procesamiento del pago.

| Categoría | Rango de Código | Descripción |
|----------|------------|-------------|
| Tarjeta | PY01xxx | Problemas con la tarjeta de pago |
| Pasarela | PY02xxx | Problemas de comunicación con la pasarela de pago |

### [Errores del Sistema](./sy)

Los errores del sistema están relacionados con recursos del dispositivo y configuración del sistema.

| Categoría | Rango de Código | Descripción |
|----------|------------|-------------|
| Recurso | SY01xxx | Problemas de memoria, disco y permisos |
| Servicio | SY02xxx | Configuración y disponibilidad de servicios |
