# Códigos de Error

Cuando WCPOS encuentra un problema, muestra un código de error que ayuda a identificar el problema. Cada código de error sigue el formato `[DOMINIO][CATEGORÍA][CÓDIGO_ESPECÍFICO]` para ayudarte a entender rápidamente el tipo y la naturaleza del error.

## Formato del Código de Error

| Componente | Descripción | Ejemplo |
|------------|-------------|---------|
| **Dominio** | El área del sistema | `API`, `DB`, `PY`, `SY` |
| **Categoría** | La categoría del error | `01`, `02`, `03`, etc. |
| **Código Específico** | El identificador único | `001`, `002`, `003`, etc. |

**Ejemplo:** `API04001` = API (dominio) + 04 (errores de respuesta) + 001 (formato de respuesta no válido)

## Dominios de Error

### [Errores de API](./api)

Los errores de API ocurren al comunicarse con tu servidor de WooCommerce.

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| Conexión | API01xxx | Problemas de red y conectividad |
| Autenticación | API02xxx | Problemas de inicio de sesión, tokens y permisos |
| Solicitud | API03xxx | Problemas con solicitudes salientes |
| Respuesta | API04xxx | Problemas con las respuestas del servidor |
| Plugin/WordPress | API05xxx | Problemas con plugins de WordPress o WooCommerce |
| Configuración | API06xxx | Problemas de configuración y ajuste |

### [Errores de Base de Datos](./db)

Los errores de base de datos ocurren al almacenar o recuperar datos localmente.

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| Conexión | DB01xxx | Problemas de conexión y transacción de base de datos |
| Datos | DB02xxx | Problemas con registros y restricciones |
| Consulta | DB03xxx | Problemas de sintaxis de consulta y tipo de datos |

### [Errores de Pago](./py)

Los errores de pago ocurren durante el proceso de pago y procesamiento de pagos.

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| Tarjeta | PY01xxx | Problemas con la tarjeta de pago |
| Pasarela | PY02xxx | Problemas de comunicación con la pasarela de pago |

### [Errores de Sistema](./sy)

Los errores de sistema están relacionados con los recursos del dispositivo y la configuración del sistema.

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| Recurso | SY01xxx | Problemas de memoria, disco y permisos |
| Servicio | SY02xxx | Configuración y disponibilidad del servicio |
