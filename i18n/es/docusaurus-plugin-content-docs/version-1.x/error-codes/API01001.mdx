# API01001: Tiempo de conexión agotado {#api01001-connection-timeout}

## Qué significa esto {#what-this-means}

La aplicación POS intentó conectarse al servidor de WooCommerce, pero el servidor tardó demasiado en responder. Esto generalmente indica problemas de latencia en la red o un servidor sobrecargado.

## Causas comunes {#common-causes}

- **Conexión a internet lenta** — La conexión de red puede estar experimentando alta latencia
- **Sobrecarga del servidor** — El servidor de WordPress puede estar procesando demasiadas solicitudes
- **Solicitudes de datos grandes** — Solicitar demasiados datos a la vez puede provocar tiempos de espera agotados
- **Retrasos por firewall o proxy** — Los dispositivos de seguridad de red pueden estar añadiendo latencia

## Cómo solucionarlo {#how-to-fix}

### 1. Verificar la conexión a internet {#1-check-your-internet-connection}

Pruebe la velocidad y estabilidad de su conexión a internet. Intente cargar el panel de administración de WooCommerce en un navegador para verificar si responde con lentitud.

### 2. Reducir la carga del servidor {#2-reduce-server-load}

Si el servidor está sobrecargado:
- Considere actualizar su plan de alojamiento
- Desactive temporalmente los plugins innecesarios
- Verifique si hay tareas cron pesadas o procesos en segundo plano

### 3. Optimizar el tiempo de respuesta del servidor {#3-optimise-server-response-time}

Contacte a su proveedor de alojamiento sobre:
- Habilitar PHP opcache
- Aumentar los límites de memoria de PHP
- Utilizar caché del lado del servidor

### 4. Aumentar la configuración de tiempo de espera {#4-increase-timeout-settings}

Si el servidor necesita legítimamente más tiempo:
- Verificar si el proveedor de hosting permite aumentar el tiempo de ejecución de PHP
- Considerar el uso de una CDN para reducir la latencia
