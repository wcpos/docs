# API03007: Cola de Solicitudes Llena

## Qué Significa

El POS tiene demasiadas solicitudes pendientes esperando para ser enviadas. La cola interna está llena y no puede aceptar más solicitudes hasta que algunas se completen.

## Causas Comunes

- **Red lenta** — Solicitudes acumulándose debido a una conexión lenta
- **Servidor sin respuesta** — El servidor tarda demasiado en responder
- **Acciones rápidas** — Realizar muchas acciones más rápido de lo que pueden procesar
- **Sobrecarga de sincronización** — Una operación de sincronización grande bloqueando la cola

## Cómo Solucionar

### 1. Esperar a que la Cola se Limpie

Dar tiempo para que las solicitudes pendientes se completen:
- Esperar 30-60 segundos
- Observar los indicadores de sincronización hasta que se completen
- Evitar acciones adicionales hasta que se limpie

### 2. Verificar la Conexión de Red

Una conexión lenta causa acumulación:
- Probar la velocidad de internet
- Intentar una conexión más rápida
- Moverse más cerca del router WiFi

### 3. Verificar el Tiempo de Respuesta del Servidor

Si el servidor está lento:
- Probar accediendo al sitio en un navegador
- Contactar al proveedor de alojamiento si el sitio está lento
- Esperar a que disminuya la carga del servidor

### 4. Reiniciar el POS

Si la cola está atascada:
1. Cerrar la aplicación POS
2. Esperar un momento
3. Reabrir el POS
4. Las solicitudes pendientes pueden necesitar reiniciarse

### 5. Evitar Acciones Rápidas

Durante la sincronización o periodos lentos:
- Esperar a que una acción se complete antes de iniciar otra
- No hacer clic repetidamente en botones
- Ser paciente con operaciones de red

## Comportamiento Sin Conexión

Cuando la cola se llena debido a condiciones sin conexión:
- El POS mantendrá las solicitudes pendientes hasta volver en línea
- Aún puede navegar productos y clientes en caché
- Puede iniciar nuevos pedidos y agregar artículos al carrito
- Completar pedidos requiere restablecer la conectividad
