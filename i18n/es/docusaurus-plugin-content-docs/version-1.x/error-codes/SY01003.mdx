# SY01003: Permiso Denegado {#sy01003-permission-denied}

## Qué Significa Esto {#what-this-means}

La aplicación POS no tiene permiso para acceder a un recurso requerido. Esto podría ser un archivo, carpeta o característica del sistema.

## Causas Comunes {#common-causes}

- **Permisos de archivo** — No se puede leer o escribir en los archivos necesarios
- **Permisos de la aplicación** — La aplicación no tiene los permisos requeridos
- **Carpeta protegida** — Intentando acceder a una ubicación restringida
- **Software de seguridad** — El antivirus está bloqueando el acceso

## Cómo Arreglarlo {#how-to-fix}

### 1. Ejecutar como Administrador (Windows) {#1-run-as-administrator-windows}

Haz clic derecho en la aplicación y selecciona "Ejecutar como administrador" si es necesario para la configuración inicial.

### 2. Verificar Permisos de la Aplicación {#2-check-application-permissions}

**macOS:**
- Preferencias del Sistema → Seguridad y Privacidad → Privacidad
- Verifica que la aplicación tenga los permisos necesarios

**Windows:**
- Configuración → Privacidad
- Revisa los permisos de la aplicación

### 3. Verificar Permisos de Carpeta {#3-check-folder-permissions}

Asegúrate de que la aplicación pueda acceder a su carpeta de datos:
- El usuario debe ser el propietario del directorio de datos de la aplicación
- Los permisos de escritura deben estar habilitados
- No debe haber flags de solo lectura en la carpeta

### 4. Revisar Configuraciones de Antivirus {#4-review-antivirus-settings}

El software de seguridad puede bloquear el acceso:
- Verifica los registros del antivirus por bloqueos
- Añade la aplicación POS a la lista de permitidos
- Pon en la lista blanca la carpeta de datos de la aplicación

### 5. Reinstalar la Aplicación {#5-reinstall-the-application}

Si los permisos están corruptos:
1. Desinstala la aplicación POS
2. Reinicia tu dispositivo
3. Reinstala en una ubicación estándar
4. Ejecuta con permisos normales (no de administrador)

## Permisos del Navegador (Versión Web) {#browser-permissions-web-version}

Si usas la versión web:
- Permite almacenamiento local para el sitio
- Habilita cookies para el sitio
- No uses configuraciones de navegador excesivamente restrictivas
