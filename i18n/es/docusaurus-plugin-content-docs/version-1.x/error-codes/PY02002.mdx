# PY02002: Tiempo de Espera de Pago

## Qué Significa

El procesamiento del pago tomó demasiado tiempo y agotó el tiempo de espera. La pasarela de pago no respondió dentro del tiempo esperado.

## Causas Comunes

- **Sobrecarga de la pasarela** — El procesador de pagos está experimentando alto volumen
- **Latencia de red** — Conexión lenta al servicio de pagos
- **Demora de procesamiento** — Verificaciones complejas de fraude o autenticación
- **Problemas de servidor** — La pasarela de pago tiene problemas de rendimiento

## Cómo Solucionar

### 1. Verificar si el Pago se Realizó

**Importante:** Antes de reintentar, verifique el estado del pago:
- Revise el panel de su pasarela de pagos
- Busque la transacción en WooCommerce
- No procese de nuevo si ya se completó

### 2. Reintentar con Precaución

Si el pago definitivamente no se realizó:
- Espere 30-60 segundos
- Intente de nuevo
- Esté atento a cargos duplicados

### 3. Verificar su Conexión

Asegure una conectividad estable:
- Pruebe su conexión a internet
- Intente una conexión por cable si está en WiFi
- Verifique problemas de red

### 4. Intentar en un Momento Menos Ocupado

Si la pasarela está sobrecargada:
- Los horarios pico de venta pueden causar ralentización
- Intente de nuevo pronto
- Use un método de pago alternativo

### 5. Contactar al Proveedor de Pagos

Si los tiempos de espera persisten:
- Revise su página de estado
- Contacte su soporte
- Reporte problemas continuos

## Evitar Cargos Duplicados

Cuando ocurre un tiempo de espera:
1. **No reintente inmediatamente** — Verifique el estado primero
2. **Revise las transacciones** — Compruebe la pasarela y WooCommerce
3. **Reembolse duplicados** — Procese reembolsos de duplicados accidentales
4. **Informe al cliente** — Explique la situación si hay un retraso

## Acciones Alternativas

Si el pago sigue agotando el tiempo de espera:
- Acepte pago en efectivo
- Registre como "pagar después" si está permitido
- Use un método de pago diferente
