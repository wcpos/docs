# PY02002: Tiempo de Espera de Pago

## Qué Significa Esto

El procesamiento del pago tomó demasiado tiempo y se agotó el tiempo de espera. La pasarela de pago no respondió dentro del tiempo esperado.

## Causas Comunes

- **Sobrecarga de la pasarela** — El procesador de pagos está experimentando un alto volumen
- **Latencia de red** — Conexión lenta al servicio de pago
- **Retraso en el procesamiento** — Comprobaciones de fraude o verificación complejas
- **Problemas con el servidor** — La pasarela de pago tiene problemas de rendimiento

## Cómo Solucionarlo

### 1. Verifica si el Pago Sucedió

**Importante:** Antes de volver a intentar, verifica el estado del pago:
- Revisa el panel de control de tu pasarela de pago
- Busca la transacción en WooCommerce
- No vuelvas a procesar si realmente tuvo éxito

### 2. Reintenta con Precaución

Si el pago definitivamente no se realizó:
- Espera de 30 a 60 segundos
- Intenta de nuevo
- Observa posibles cargos duplicados

### 3. Verifica Tu Conexión

Asegúrate de tener una conectividad estable:
- Prueba tu conexión a Internet
- Intenta una conexión por cable si estás en WiFi
- Verifica problemas de red

### 4. Intenta en un Momento Menos Ocupado

Si la pasarela está sobrecargada:
- Los picos de venta pueden causar ralentizaciones
- Inténtalo de nuevo en breve
- Utiliza un método de pago alternativo

### 5. Contacta al Proveedor de Pagos

Si los tiempos de espera persisten:
- Revisa su página de estado
- Contacta soporte
- Informa sobre problemas continuos

## Evitando Cargos Duplicados

Cuando ocurre un tiempo de espera:
1. **No intentes de inmediato** — Verifica el estado primero
2. **Revisa las transacciones** — Verifica tanto la pasarela como WooCommerce
3. **Reembolsa duplicados** — Procesa reembolsos por cualquier duplicado accidental
4. **Informa al cliente** — Explica la situación si hay un retraso

## Acciones Alternativas

Si el pago sigue agotando el tiempo de espera:
- Acepta pago en efectivo
- Registra como "pagar después" si se permite
- Usa un método de pago diferente
