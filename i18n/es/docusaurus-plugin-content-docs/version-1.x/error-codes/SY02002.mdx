# SY02002: Servicio No Disponible {#sy02002-service-unavailable}

## Lo Que Esto Significa {#what-this-means}

Un servicio o componente requerido no está disponible. El POS depende de varios servicios para funcionar, y uno de ellos actualmente no es accesible.

## Causas Comunes {#common-causes}

- **Servicio en segundo plano detenido** — Un servicio requerido no está en funcionamiento
- **Problema de recursos del sistema** — No hay suficientes recursos para iniciar el servicio
- **Fallo de dependencia** — Un servicio del que depende falló
- **Problema de inicio** — El servicio no pudo iniciarse correctamente
- **Error del servidor** — El servidor de WooCommerce devolvió un error 5xx

## Mapeo de Errores del Servidor {#server-error-mapping}

Este código de error se activa cuando el servidor devuelve:

| Código del Servidor | Origen |
|---------------------|--------|
| HTTP 5xx            | Cualquier respuesta de error del servidor (500, 502, 503, 504, etc.) |

## Cómo Solucionar {#how-to-fix}

### 1. Reiniciar la Aplicación {#1-restart-the-application}

Cierra y vuelve a abrir el POS:
1. Cierra completamente la aplicación
2. Espera unos segundos
3. Vuelve a iniciarlo

### 2. Reiniciar Tu Dispositivo {#2-restart-your-device}

Un reinicio completo a menudo resuelve problemas de servicio:
1. Guarda cualquier trabajo importante
2. Reinicia tu computadora
3. Prueba el POS nuevamente

### 3. Verificar Recursos del Sistema {#3-check-system-resources}

Asegúrate de que los recursos del sistema estén disponibles:
- Cierra aplicaciones innecesarias
- Verifica el uso de memoria
- Verifica el espacio en disco

### 4. Actualizar la Aplicación {#4-update-the-application}

Asegúrate de estar en la última versión:
- Verifica si hay actualizaciones disponibles
- Instala cualquier actualización pendiente
- Reinicia después de actualizar

### 5. Reinstalar si es Necesario {#5-reinstall-if-necessary}

Si el servicio sigue sin estar disponible:
1. Desinstala la aplicación
2. Reinicia tu dispositivo
3. Descarga la última versión
4. Instala de nuevo

## Específico del Navegador (Versión Web) {#browser-specific-web-version}

Para la versión web:
- Asegúrate de que JavaScript esté habilitado
- Verifica que las características requeridas del navegador no estén bloqueadas
- Prueba con un navegador diferente
- Desactiva las extensiones del navegador que puedan interferir

## Cuándo Contactar con Soporte {#when-to-contact-support}

Si este error persiste:
- Toma nota de cuándo ocurre
- Verifica si acciones específicas lo desencadenan
- Informa con detalles para la investigación
