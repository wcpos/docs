# SY02002: Servicio No Disponible

## Qué Significa

Un servicio o componente necesario no está disponible. El POS depende de varios servicios para funcionar, y uno de ellos está actualmente inaccesible.

## Causas Comunes

- **Servicio en segundo plano detenido** — Un servicio requerido no está ejecutándose
- **Problema de recursos del sistema** — No hay suficientes recursos para iniciar el servicio
- **Fallo de dependencia** — Un servicio del que depende ha fallado
- **Problema de inicio** — El servicio no se inició correctamente
- **Error del servidor** — El servidor de WooCommerce devolvió un error 5xx

## Mapeo de Errores del Servidor

Este código de error se activa cuando el servidor devuelve:

| Código del Servidor | Fuente |
|-------------------|--------|
| HTTP 5xx | Cualquier respuesta de error del servidor (500, 502, 503, 504, etc.) |

## Cómo Solucionar

### 1. Reiniciar la Aplicación

Cierre y vuelva a abrir el POS:
1. Cierre completamente la aplicación
2. Espere unos segundos
3. Iníciela de nuevo

### 2. Reiniciar su Dispositivo

Un reinicio completo a menudo resuelve problemas de servicio:
1. Guarde cualquier trabajo importante
2. Reinicie su computadora
3. Intente usar el POS de nuevo

### 3. Verificar Recursos del Sistema

Asegúrese de que los recursos del sistema estén disponibles:
- Cierre aplicaciones innecesarias
- Verifique el uso de memoria
- Revise el espacio en disco

### 4. Actualizar la Aplicación

Asegúrese de estar en la última versión:
- Busque actualizaciones disponibles
- Instale cualquier actualización pendiente
- Reinicie después de actualizar

### 5. Reinstalar si es Necesario

Si el servicio sigue sin estar disponible:
1. Desinstale la aplicación
2. Reinicie su dispositivo
3. Descargue la última versión
4. Instale de nuevo

## Específico para Navegadores (Versión Web)

Para la versión web:
- Asegúrese de que JavaScript esté habilitado
- Verifique que las características del navegador requeridas no estén bloqueadas
- Pruebe con un navegador diferente
- Desactive las extensiones del navegador que puedan interferir

## Cuándo Contactar a Soporte

Si este error persiste:
- Anote cuándo ocurre
- Verifique si acciones específicas lo provocan
- Informe con detalles para investigación
