# PY01004: Número de Tarjeta Inválido

## Qué Significa

El número de tarjeta ingresado no es válido. Los números de tarjetas siguen patrones específicos e incluyen dígitos de verificación para su validación.

## Causas Comunes

- **Error de escritura** — Se ingresaron uno o más dígitos incorrectamente
- **Dígitos faltantes** — No se ingresaron todos los dígitos
- **Dígitos adicionales** — Se ingresaron demasiados dígitos
- **Tarjeta equivocada** — Número de tarjeta diferente a la tarjeta física
- **Error del lector de tarjetas** — El chip/deslizamiento leyó datos incorrectos

## Cómo Solucionarlo

### 1. Volver a Ingresar el Número de Tarjeta

Ingrese los dígitos cuidadosamente:
- Revise la tarjeta física
- Ingrese cada dígito lentamente
- Verifique el número completo antes de enviarlo

### 2. Verificar el Tipo de Tarjeta

Asegúrese de que el tipo de tarjeta esté admitido:
- Visa (comienza con 4)
- Mastercard (comienza con 51-55 o 22-27)
- Amex (comienza con 34 o 37)
- Discover (comienza con 6011, 622, 644-649, 65)

### 3. Intentar Nuevamente con el Lector de Tarjetas

Si está usando un lector de chip/deslizamiento:
- Limpie el chip de la tarjeta
- Intente un método de lectura diferente (chip vs deslizamiento)
- Ingrese manualmente si el lector falla

### 4. Usar una Tarjeta Diferente

Si el número de tarjeta continúa fallando:
- La tarjeta puede estar dañada
- Pruebe con una tarjeta diferente
- Use un método de pago alternativo

### 5. Verificar Daños en la Tarjeta

Problemas físicos de la tarjeta:
- Banda magnética rayada
- Chip dañado
- Números desgastados o desvanecidos
- Es posible que la tarjeta necesite reemplazo

## Formatos de Números de Tarjeta

| Tipo de Tarjeta | Longitud | Comienza Con |
|----------------|----------|--------------|
| Visa | 16 | 4 |
| Mastercard | 16 | 51-55, 22-27 |
| Amex | 15 | 34, 37 |
| Discover | 16 | 6011, 622, 644-649, 65 |
