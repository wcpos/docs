# DB01001: Error de Conexión {#db01001-connection-failed}

## Qué Significa Esto {#what-this-means}

El POS no pudo conectarse a la base de datos local. WCPOS utiliza una base de datos local para almacenar productos, clientes y otros datos para un acceso rápido y funcionalidad sin conexión.

## Causas Comunes {#common-causes}

- **Almacenamiento lleno** — El dispositivo no tiene espacio de almacenamiento disponible
- **Base de datos corrupta** — Los archivos de la base de datos local están dañados
- **Problema de permisos** — La aplicación no tiene permisos de almacenamiento
- **Almacenamiento del navegador deshabilitado** — IndexedDB está deshabilitado (versión web)

## Cómo Solucionarlo {#how-to-fix}

### 1. Verificar Espacio de Almacenamiento {#1-check-storage-space}

Asegúrate de que tu dispositivo tenga almacenamiento disponible:
- Libera espacio si el almacenamiento está lleno
- Elimina aplicaciones o archivos no utilizados
- Verifica el espacio en disco disponible

### 2. Borrar Datos de la Aplicación {#2-clear-app-data}

Restablece la base de datos local:
- **Aplicación de escritorio:** Busca "Borrar Datos" en la configuración
- **Navegador web:** Borra los datos del sitio para el dominio del POS
- **Nota:** Esto requerirá volver a sincronizar todos los datos

### 3. Verificar la Configuración del Navegador (Web) {#3-check-browser-settings-web}

Si usas la versión web:
1. Asegúrate de que IndexedDB esté habilitado
2. No uses modo privado/incógnito (almacenamiento limitado)
3. Permite que el sitio almacene datos

### 4. Verificar Permisos {#4-check-permissions}

En el escritorio:
- Asegúrate de que la aplicación tenga permiso para escribir en su directorio de datos
- Verifica si el antivirus está bloqueando el acceso a la base de datos

### 5. Reinstalar la Aplicación {#5-reinstall-the-app}

Si otras soluciones fallan:
1. Desinstala la aplicación POS
2. Reinicia tu dispositivo
3. Reinstala la aplicación
4. Inicia sesión y sincroniza los datos

## Recuperación de Datos {#data-recovery}

Si tienes datos no guardados:
- La aplicación puede tener cambios pendientes que no se pudieron sincronizar
- Contacta con soporte si necesitas ayuda para recuperar datos
- En la mayoría de los casos, los datos pueden volver a sincronizarse desde el servidor
