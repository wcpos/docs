# API05003: Plugin WCPOS desactualizado {#api05003-wcpos-plugin-outdated}

## Qué significa {#what-this-means}

El plugin WCPOS instalado en el servidor está desactualizado y no es compatible con la versión de la aplicación POS. Es necesario actualizar el plugin.

## Causas comunes {#common-causes}

- **Plugin no actualizado** — Las actualizaciones automáticas pueden estar desactivadas
- **Actualización fallida** — Un intento de actualización anterior falló
- **Discrepancia de versión** — La aplicación POS se actualizó, pero el plugin no
- **Sitio de staging/desarrollo** — Pruebas en una versión anterior

## Cómo solucionarlo {#how-to-fix}

### 1. Actualizar el plugin {#1-update-the-plugin}

En el administrador de WordPress:
1. Vaya a Plugins → Plugins instalados
2. Busque «WCPOS»
3. Si hay una actualización disponible, haga clic en «Actualizar ahora»

### 2. Activar las actualizaciones automáticas {#2-enable-auto-updates}

Para evitar futuras discrepancias:
1. Vaya a Plugins → Plugins instalados
2. Busque «WCPOS»
3. Haga clic en «Activar las actualizaciones automáticas»

### 3. Actualización manual {#3-manual-update}

Si la actualización automática falla:
1. Descargue la última versión desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Desactive el plugin actual
3. Elimine los archivos del plugin antiguo
4. Suba y active la nueva versión

### 4. Compruebe si hay errores de actualización {#4-check-for-update-errors}

Si las actualizaciones fallan:
- Compruebe los permisos de archivo en wp-content/plugins
- Verifique que dispone de espacio en disco
- Busque problemas de memoria de PHP
- Consulte los registros de errores para obtener más detalles

### 5. Cambie a una versión anterior de la app POS (temporal) {#5-downgrade-pos-app-temporary}

Si no puede actualizar el plugin de inmediato:
- Utilice una versión anterior de la app POS
- Esta es solo una solución temporal
- Planifique actualizar el plugin pronto

## Compatibilidad de versiones {#version-compatibility}

Mantenga siempre ambos componentes actualizados:
- La aplicación POS
- El plugin de WordPress de WCPOS

Consulte las notas de la versión para ver posibles cambios incompatibles al actualizar.
