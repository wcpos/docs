# API05003: Plugin WCPOS Desactualizado

## Qué Significa Esto

El plugin WCPOS instalado en su servidor está desactualizado y no es compatible con su versión de la aplicación POS. Necesita actualizar el plugin.

## Causas Comunes

- **Plugin no actualizado** — Las actualizaciones automáticas pueden estar deshabilitadas
- **Actualización fallida** — Un intento de actualización anterior falló
- **Desajuste de versión** — La aplicación POS se actualizó, pero el plugin no
- **Sitio de pruebas/desarrollo** — Pruebas en una versión anterior

## Cómo Arreglarlo

### 1. Actualizar el Plugin

En WordPress Admin:
1. Vaya a Plugins → Plugins Instalados
2. Busque "WooCommerce POS"
3. Si hay una actualización disponible, haga clic en "Actualizar Ahora"

### 2. Habilitar Auto-Actualizaciones

Para prevenir discrepancias futuras:
1. Vaya a Plugins → Plugins Instalados
2. Busque "WooCommerce POS"
3. Haga clic en "Habilitar auto-actualizaciones"

### 3. Actualización Manual

Si la actualización automática falla:
1. Descargue la última versión desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)
2. Desactive el plugin actual
3. Elimine los archivos antiguos del plugin
4. Cargue y active la nueva versión

### 4. Verificar Errores de Actualización

Si las actualizaciones están fallando:
- Verifique los permisos de archivos en wp-content/plugins
- Asegúrese de tener espacio en disco
- Verifique problemas de memoria PHP
- Revise los registros de errores para obtener detalles

### 5. Degradar la Aplicación POS (Temporal)

Si no puede actualizar el plugin inmediatamente:
- Use una versión anterior de la aplicación POS
- Esta es una solución temporal únicamente
- Planifique actualizar el plugin pronto

## Compatibilidad de Versiones

Mantenga siempre ambos componentes actualizados:
- La aplicación POS
- El plugin WCPOS de WordPress

Verifique las notas de la versión para cualquier cambio importante al actualizar.
