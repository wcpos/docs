# SY02001: Configuración no válida

## Qué Significa

La configuración del sistema no es válida o está dañada. La aplicación POS encontró parámetros de configuración que no tienen sentido o están incompletos.

## Causas Comunes

- **Configuraciones dañadas** — El archivo de configuración se corrompió
- **Instalación incompleta** — El proceso de configuración no se completó
- **Error de edición manual** — Los archivos de configuración se modificaron incorrectamente
- **Problema de actualización de versión** — Configuración antigua incompatible con nueva versión

## Cómo Solucionar

### 1. Restablecer a Configuración Predeterminada

Si está disponible en la aplicación:
- Busque "Restablecer configuración" o "Restaurar valores predeterminados"
- Esto borra la configuración personalizada pero corrige la corrupción

### 2. Completar la Configuración

Si la configuración no se terminó:
1. Inicie de nuevo el proceso de configuración
2. Complete todos los pasos requeridos
3. Verifique la configuración antes de finalizar

### 3. Borrar Datos de la Aplicación

Eliminar configuración dañada:
- **Escritorio:** Borrar carpeta de datos de la aplicación, reiniciar aplicación
- **Navegador:** Borrar datos del sitio, recargar

### 4. Reinstalar la Aplicación

Para problemas persistentes:
1. Desinstalar completamente
2. Eliminar carpetas de datos restantes
3. Reinstalar desde cero
4. Completar configuración desde el principio

### 5. Verificar Actualizaciones

Los formatos de configuración pueden cambiar:
- Actualizar la aplicación POS
- Actualizar el plugin WCPOS
- Reconfigurar después de la actualización

## Qué Se Restablece

Al restablecer la configuración:
- Tendrá que iniciar sesión nuevamente
- Se perderán configuraciones personalizadas
- Los datos necesitarán re-sincronizarse
- Pero sus datos de WooCommerce están seguros en el servidor
