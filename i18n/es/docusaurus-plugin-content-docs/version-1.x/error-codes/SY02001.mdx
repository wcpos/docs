# SY02001: Configuración Inválida

## Qué significa esto

La configuración del sistema es inválida o está corrupta. La aplicación POS encontró ajustes de configuración que no tienen sentido o están incompletos.

## Causas Comunes

- **Ajustes corruptos** — El archivo de configuración se volvió corrupto
- **Configuración incompleta** — El proceso de configuración no se completó
- **Error de edición manual** — Los archivos de configuración se modificaron incorrectamente
- **Problema de actualización de versión** — Configuración antigua incompatible con la nueva versión

## Cómo Solucionar

### 1. Restablecer a Ajustes Predeterminados

Si está disponible en la app:
- Busca "Restablecer Ajustes" o "Restaurar Predeterminados"
- Esto borra los ajustes personalizados pero corrige la corrupción

### 2. Completar la Configuración

Si la configuración no se terminó:
1. Inicia el proceso de configuración nuevamente
2. Completa todos los pasos requeridos
3. Verifica la configuración antes de finalizar

### 3. Borrar Datos de la Aplicación

Elimina la configuración corrupta:
- **Escritorio:** Elimina la carpeta de datos de la app, reinicia la app
- **Navegador:** Borra los datos del sitio, recarga

### 4. Reinstalar la Aplicación

Para problemas persistentes:
1. Desinstala completamente
2. Elimina cualquier carpeta de datos sobrante
3. Reinstala desde cero
4. Completa la configuración desde el principio

### 5. Verificar si hay Actualizaciones

Los formatos de configuración pueden cambiar:
- Actualiza la aplicación POS
- Actualiza el plugin WCPOS
- Reconfigura después de la actualización

## Qué se Restablece

Al restablecer la configuración:
- Necesitarás iniciar sesión nuevamente
- Los ajustes personalizados se perderán
- Los datos deberán sincronizarse nuevamente
- Pero tus datos de WooCommerce están seguros en el servidor
