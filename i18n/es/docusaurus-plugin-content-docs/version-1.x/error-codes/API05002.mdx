# API05002: Plugin WCPOS no encontrado {#api05002-wcpos-plugin-not-found}

## Qué significa {#what-this-means}

El plugin de WCPOS no está instalado o no está activo en su sitio de WordPress. La aplicación POS necesita este plugin para comunicarse con WooCommerce.

## Causas comunes {#common-causes}

- **Plugin no instalado** — El plugin nunca se instaló
- **Plugin desactivado** — El plugin se deshabilitó
- **Plugin eliminado** — Los archivos del plugin se eliminaron
- **Sitio incorrecto** — Conectado a un sitio sin WCPOS

## Cómo solucionarlo {#how-to-fix}

### 1. Instalar el plugin {#1-install-the-plugin}

Si no está instalado:
1. Vaya a Administración de WordPress → Plugins → Añadir nuevo
2. Busque «WCPOS»
3. Instale y active el plugin

O descárguelo desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Active el plugin {#2-activate-the-plugin}

Si está instalado pero no está activo:
1. Vaya a Plugins → Plugins instalados
2. Localice «WCPOS»
3. Haga clic en «Activar»

### 3. Compruebe si hay errores {#3-check-for-errors}

Si el plugin no se activa:
- Revise si hay errores de PHP en su registro de errores
- Verifique que WooCommerce esté instalado y activo
- Compruebe los requisitos de versión de PHP
- Busque conflictos entre plugins

### 4. Verifique la URL {#4-verify-the-url}

Asegúrese de que el POS esté configurado para conectarse al sitio de WordPress correcto:
- Revise la URL en los ajustes del POS
- Verifique que coincida con su instalación de WordPress

### 5. Reinstale si es necesario {#5-reinstall-if-necessary}

Si el plugin está dañado:
1. Desactive y elimine el plugin
2. Reinstale desde WordPress.org
3. Active el plugin
