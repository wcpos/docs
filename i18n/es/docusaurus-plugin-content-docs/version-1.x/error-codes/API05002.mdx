# API05002: Plugin de WCPOS no encontrado

## Qué Significa

El plugin de WCPOS (WooCommerce POS) no está instalado o no está activo en su sitio WordPress. La aplicación POS requiere este plugin para comunicarse con WooCommerce.

## Causas Comunes

- **Plugin no instalado** — El plugin nunca se instaló
- **Plugin desactivado** — El plugin fue deshabilitado
- **Plugin eliminado** — Los archivos del plugin fueron removidos
- **Sitio incorrecto** — Conectado a un sitio sin WCPOS

## Cómo Solucionar

### 1. Instalar el Plugin

Si no está instalado:
1. Ir a Administración de WordPress → Plugins → Añadir nuevo
2. Buscar "WooCommerce POS"
3. Instalar y activar el plugin

O descargarlo desde [WordPress.org](https://wordpress.org/plugins/woocommerce-pos/)

### 2. Activar el Plugin

Si está instalado pero no activo:
1. Ir a Plugins → Plugins instalados
2. Buscar "WooCommerce POS"
3. Hacer clic en "Activar"

### 3. Verificar Errores

Si el plugin no se puede activar:
- Revisar errores de PHP en su registro de errores
- Verificar que WooCommerce esté instalado y activo
- Comprobar los requisitos de versión de PHP
- Buscar conflictos de plugins

### 4. Verificar la URL

Asegúrese de que el POS está configurado para conectarse al sitio WordPress correcto:
- Revisar la URL en la configuración de POS
- Verificar que coincida con su instalación de WordPress

### 5. Reinstalar si es Necesario

Si el plugin está dañado:
1. Desactivar y eliminar el plugin
2. Reinstalar desde WordPress.org
3. Activar el plugin

## Requisitos

WCPOS requiere:
- WordPress (versión reciente)
- WooCommerce (versión reciente)
- PHP 7.4 o superior
