# Errores de Pago

Los errores de pago ocurren durante el proceso de pago y procesamiento. Estos errores están precedidos por `PY` y se relacionan con problemas con tarjetas de pago, pasarelas y transacciones.

## Categorías

| Categoría | Rango de Código | Descripción |
|-----------|-----------------|-------------|
| [Tarjeta](#card-errors) | PY01xxx | Problemas con la tarjeta de pago |
| [Pasarela](#gateway-errors) | PY02xxx | Problemas de comunicación con la pasarela de pago |

---

## Errores de Tarjeta

Problemas relacionados con la tarjeta de pago del cliente.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [PY01001](./PY01001) | Pago Rechazado | El pago fue rechazado |
| [PY01002](./PY01002) | Fondos Insuficientes | No hay suficientes fondos para el pago |
| [PY01003](./PY01003) | Tarjeta Expirada | La tarjeta de pago ha expirado |
| [PY01004](./PY01004) | Número de Tarjeta Inválido | El número de tarjeta no es válido |

## Errores de Pasarela

Problemas de comunicación con la pasarela de pago.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [PY02001](./PY02001) | Error de Pasarela de Pago | Error en la comunicación con la pasarela de pago |
| [PY02002](./PY02002) | Tiempo de Espera en el Pago | El procesamiento del pago ha excedido el tiempo límite |
