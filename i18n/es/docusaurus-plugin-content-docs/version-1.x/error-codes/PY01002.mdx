# PY01002: Fondos Insuficientes {#py01002-insufficient-funds}

## Lo que Esto Significa {#what-this-means}

El pago fue declinado porque no hay suficiente dinero en la cuenta para cubrir la transacción.

## Causas Comunes {#common-causes}

- **Balance bajo en la cuenta** — No hay suficientes fondos disponibles
- **Transacciones pendientes** — Otras transacciones pendientes reduciendo el saldo disponible
- **Límite de crédito alcanzado** — Para tarjetas de crédito, el límite está al máximo
- **Fondos retenidos** — Los fondos están retenidos para otras transacciones

## Cómo Solucionarlo {#how-to-fix}

### 1. Usar un Método de Pago Diferente {#1-use-a-different-payment-method}

La solución más rápida:
- Prueba con una tarjeta diferente
- Usa efectivo
- Divide el pago en múltiples métodos (si es compatible)

### 2. Reducir el Monto del Pedido {#2-reduce-the-order-amount}

Si es posible el pago parcial:
- Elimina algunos artículos
- Aplica descuentos o cupones
- Completa una transacción más pequeña

### 3. Acciones del Cliente {#3-customer-actions}

El cliente puede necesitar:
- Transferir fondos a la cuenta
- Esperar a que se procesen las transacciones pendientes
- Pagar el saldo de la tarjeta de crédito
- Solicitar un aumento del límite de crédito

### 4. Dividir la Transacción {#4-split-the-transaction}

Si tu POS admite pagos divididos:
- Paga parte con una tarjeta
- Paga el resto con otro método

## Nota de Privacidad {#privacy-note}

Al comunicar esto a los clientes:
- Sé discreto sobre la razón específica del rechazo
- Simplemente indica "la tarjeta fue declinada"
- Permíteles intentar con otro método de pago de forma privada
