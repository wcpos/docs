# PY01002: Fondos Insuficientes

## Lo Que Significa

El pago fue rechazado porque no hay suficiente dinero en la cuenta para cubrir la transacción.

## Causas Comunes

- **Saldo de cuenta bajo** — No hay fondos disponibles suficientes
- **Transacciones pendientes** — Otras transacciones pendientes reducen el saldo disponible
- **Límite de crédito alcanzado** — Para tarjetas de crédito, el límite está al máximo
- **Retención de fondos** — Fondos retenidos para otras transacciones

## Cómo Solucionar

### 1. Usar un Método de Pago Diferente

La solución más rápida:
- Probar una tarjeta diferente
- Usar efectivo
- Dividir el pago en varios métodos (si es compatible)

### 2. Reducir el Monto del Pedido

Si es posible un pago parcial:
- Eliminar algunos artículos
- Aplicar descuentos o cupones
- Completar una transacción más pequeña

### 3. Acciones del Cliente

El cliente puede necesitar:
- Transferir fondos a la cuenta
- Esperar a que se liquiden las transacciones pendientes
- Pagar el saldo de la tarjeta de crédito
- Solicitar un aumento del límite de crédito

### 4. Dividir la Transacción

Si el POS permite pagos divididos:
- Pagar parte con una tarjeta
- Pagar el resto con otro método

## Nota de Privacidad

Al comunicarse con los clientes:
- Sea discreto sobre la razón específica del rechazo
- Simplemente indique que "la tarjeta fue rechazada"
- Permita que prueben otro método de pago de forma privada
