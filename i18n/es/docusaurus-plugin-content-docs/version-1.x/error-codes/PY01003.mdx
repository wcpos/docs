# PY01003: Tarjeta Caducada {#py01003-card-expired}

## Lo Que Esto Significa {#what-this-means}

La tarjeta de pago ha pasado su fecha de caducidad y ya no se puede utilizar para transacciones.

## Causas Comunes {#common-causes}

- **Tarjeta caducada** — La fecha de validez de la tarjeta ha pasado
- **Fecha de caducidad ingresada incorrectamente** — La fecha de caducidad se ingresó mal
- **Nueva tarjeta no activada** — Tarjeta de reemplazo recibida pero aún no activada

## Cómo Corregir {#how-to-fix}

### 1. Verificar la Fecha de Caducidad {#1-check-the-expiration-date}

Mire la tarjeta:
- Encuentre la fecha de caducidad (formato MM/AA)
- Compare con la fecha actual
- Verifique que la fecha ingresada coincida con la de la tarjeta

### 2. Usar una Tarjeta Diferente {#2-use-a-different-card}

Si la tarjeta está realmente caducada:
- Utilice una tarjeta diferente y válida
- Use efectivo u otro método de pago
- Pregunte si el cliente tiene una tarjeta de reemplazo

### 3. Obtener la Tarjeta Actualizada {#3-get-the-updated-card}

El titular de la tarjeta debe:
- Revisar el correo en busca de una tarjeta de reemplazo
- Activar la nueva tarjeta si la recibió
- Contactar a su banco si no llegó el reemplazo

### 4. Verificar la Entrada {#4-verify-entry}

Verifique la entrada de caducidad:
- Formato MM/AA
- Mes y año correctos
- Sin errores tipográficos

## Consejos de Servicio al Cliente {#customer-service-tips}

Cuando una tarjeta está caducada:
- Infórmele al cliente de manera cortés
- Sugiérale que revise si tiene una tarjeta de reemplazo
- Ofrezca opciones de pago alternativas
- Sea comprensivo: esto le sucede a todo el mundo
