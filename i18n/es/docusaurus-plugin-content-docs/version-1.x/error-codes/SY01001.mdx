# SY01001: Sin Memoria

## Qué Significa

Su dispositivo tiene poca memoria (RAM) y no puede completar la operación solicitada. El POS necesita suficiente memoria para funcionar correctamente.

## Causas Comunes

- **Demasiadas aplicaciones abiertas** — Otras aplicaciones utilizando memoria
- **Conjunto de datos grande** — Muchos productos/clientes cargados
- **Fuga de memoria** — Aplicación no liberando memoria correctamente
- **RAM insuficiente** — El dispositivo no tiene suficiente memoria

## Cómo Solucionar

### 1. Cerrar Otras Aplicaciones

Libere memoria:
- Cierre las pestañas del navegador no utilizadas
- Cierre otras aplicaciones
- Cierre programas en segundo plano

### 2. Reinicie el POS

Refresque la memoria de la aplicación:
1. Cierre completamente el POS
2. Espere unos segundos
3. Vuelva a abrir la aplicación

### 3. Reinicie Su Dispositivo

Un reinicio completo borra toda la memoria:
1. Guarde cualquier trabajo abierto
2. Reinicie su computadora/dispositivo
3. Abra solo el POS al iniciar nuevamente

### 4. Borrar Datos del Navegador (Versión Web)

Si está utilizando en un navegador:
- Borre la caché del navegador
- Cierre y vuelva a abrir el navegador
- Utilice menos pestañas del navegador

### 5. Reducir la Carga de Datos

Si tiene un catálogo muy grande:
- Use filtros para cargar menos datos a la vez
- Considere paginar los resultados
- Limite el alcance de la sincronización si es posible

## Requisitos Mínimos

Asegúrese de que su dispositivo cumpla con las especificaciones mínimas:
- **RAM:** Mínimo 4GB, recomendado 8GB
- **Navegador:** Navegador moderno (Chrome, Firefox, Edge)
- **Aplicación de Escritorio:** Consulte las notas de la versión para los requisitos

## Problemas Continuos

Si frecuentemente se queda sin memoria:
- Considere actualizar la RAM de su dispositivo
- Use un dispositivo con más recursos
- Informe el problema si parece excesivo
