# SY01001: Sin memoria disponible

## Qué Significa Esto

Su dispositivo tiene poca memoria (RAM) y no puede completar la operación solicitada. El POS necesita suficiente memoria para funcionar correctamente.

## Causas Comunes

- **Demasiadas aplicaciones abiertas** — Otras aplicaciones usando memoria
- **Conjunto de datos grande** — Muchos productos/clientes cargados
- **Pérdida de memoria** — Aplicación no liberando memoria correctamente
- **RAM insuficiente** — El dispositivo no tiene suficiente memoria

## Cómo Solucionarlo

### 1. Cerrar Otras Aplicaciones

Liberar memoria:
- Cerrar pestañas de navegador no utilizadas
- Cerrar otras aplicaciones
- Cerrar programas en segundo plano

### 2. Reiniciar el POS

Actualizar la memoria de la aplicación:
1. Cerrar completamente el POS
2. Esperar unos segundos
3. Volver a abrir la aplicación

### 3. Reiniciar Su Dispositivo

Un reinicio completo limpia toda la memoria:
1. Guardar cualquier trabajo abierto
2. Reiniciar su computadora/dispositivo
3. Abrir solo el POS al iniciar

### 4. Limpiar Datos del Navegador (Versión Web)

Si está usando en un navegador:
- Limpiar caché del navegador
- Cerrar y volver a abrir el navegador
- Usar menos pestañas de navegador

### 5. Reducir Carga de Datos

Si tiene un catálogo muy grande:
- Usar filtros para cargar menos datos a la vez
- Considerar paginar resultados
- Limitar el alcance de la sincronización si es posible

## Requisitos Mínimos

Asegurarse de que su dispositivo cumpla las especificaciones mínimas:
- **RAM:** 4GB mínimo, 8GB recomendado
- **Navegador:** Navegador moderno (Chrome, Firefox, Edge)
- **Aplicación de Escritorio:** Consultar notas de la versión para requisitos

## Problemas Continuos

Si queda sin memoria con frecuencia:
- Considerar actualizar la RAM de su dispositivo
- Usar un dispositivo con más recursos
- Informar el problema si parece excesivo
