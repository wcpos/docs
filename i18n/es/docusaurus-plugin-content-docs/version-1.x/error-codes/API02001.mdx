# API02001: Credenciales Inválidas {#api02001-invalid-credentials}

## Qué Significa Esto {#what-this-means}

El nombre de usuario o la contraseña que ingresaste son incorrectos. El servidor rechazó el intento de inicio de sesión porque las credenciales no coinciden con ninguna cuenta válida.

## Causas Comunes {#common-causes}

- **Error tipográfico en el nombre de usuario o la contraseña** — Verifica si el bloqueo de mayúsculas está activo o si hay espacios de más
- **Cuenta incorrecta** — Uso de credenciales de un sitio diferente
- **Contraseña cambiada recientemente** — La contraseña se actualizó pero el POS tiene la antigua
- **Cuenta eliminada** — La cuenta de usuario ya no existe

## Mapeo de Errores del Servidor {#server-error-mapping}

Este código de error se activa cuando el servidor devuelve:

| Código del Servidor | Fuente |
|--------------------|--------|
| `woocommerce_rest_authentication_error` | WooCommerce REST API |
| `jwt_auth_failed` | Complemento de Autenticación JWT |

## Cómo Solucionarlo {#how-to-fix}

### 1. Verifica tus Credenciales {#1-verify-your-credentials}

- Verifica tu nombre de usuario (normalmente un correo electrónico o nombre de usuario de WordPress)
- Verifica que la contraseña sea correcta
- Intenta iniciar sesión directamente en el admin de WordPress para confirmar

### 2. Restablece tu Contraseña {#2-reset-your-password}

Si has olvidado tu contraseña:
1. Ve a tu página de inicio de sesión de WordPress
2. Haz clic en "¿Perdiste tu contraseña?"
3. Ingresa tu correo electrónico para recibir un enlace de restablecimiento
4. Actualiza tu contraseña
5. Inicia sesión en el POS con la nueva contraseña

### 3. Verifica la Cuenta de Usuario {#3-check-the-user-account}

En el Admin de WordPress:
- Ve a Usuarios → Todos los Usuarios
- Verifica que la cuenta exista
- Verifica que tenga los permisos de acceso adecuados al POS

### 4. Borra las Credenciales Guardadas {#4-clear-saved-credentials}

En el POS:
- Cierra sesión por completo
- Borra cualquier información de inicio de sesión guardada/caché
- Ingresa nuevamente las credenciales frescas
