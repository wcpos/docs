# Errores de API {#api-errors}

Los errores de API ocurren al comunicarse con su servidor de WooCommerce. Estos errores tienen el prefijo `API` y se organizan en las siguientes categorías:

## Categorías {#categories}

| Categoría | Rango de Código | Descripción |
|-----------|-----------------|-------------|
| [Conexión](#errores-de-conexión) | API01xxx | Problemas de red y conectividad |
| [Autenticación](#errores-de-autenticación) | API02xxx | Problemas de inicio de sesión, tokens y permisos |
| [Solicitud](#errores-de-solicitud) | API03xxx | Problemas con las solicitudes salientes |
| [Respuesta](#errores-de-respuesta) | API04xxx | Problemas con las respuestas del servidor |
| [Plugin/WordPress](#errores-de-pluginwordpress) | API05xxx | Problemas con los plugins de WordPress o WooCommerce |
| [Configuración](#errores-de-configuración) | API06xxx | Problemas de implementación y configuración |

---

## Errores de Conexión {#connection-errors}

Problemas de red y conectividad entre el POS y su servidor.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [API01001](./API01001) | Tiempo de Espera de Conexión | El servidor tardó demasiado en responder |
| [API01002](./API01002) | Conexión Rechazada | El servidor rechazó la conexión |
| [API01003](./API01003) | Conexión Restablecida | La conexión se cerró inesperadamente |
| [API01004](./API01004) | Resolución DNS Fallida | No se pudo resolver la dirección del servidor |
| [API01005](./API01005) | Error de Certificado SSL | Problema con el certificado de seguridad del sitio |
| [API01006](./API01006) | Red Inalcanzable | No se puede alcanzar la red |
| [API01007](./API01007) | Dispositivo Fuera de Línea | Su dispositivo no está conectado a internet |
| [API01008](./API01008) | Sitio Web No Disponible | El sitio web no está respondiendo |

## Errores de Autenticación {#authentication-errors}

Problemas con el inicio de sesión, sesiones y permisos.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [API02001](./API02001) | Credenciales Inválidas | El nombre de usuario o la contraseña son incorrectos |
| [API02002](./API02002) | Token Expirado | Su sesión ha expirado |
| [API02003](./API02003) | Token Inválido | El token de autenticación no es válido |
| [API02004](./API02004) | Usuario No Autorizado | No tiene permiso para realizar esta acción |
| [API02005](./API02005) | Permisos Insuficientes | Su rol de usuario carece de permisos necesarios |
| [API02006](./API02006) | Clave API Inválida | La clave API de WooCommerce no es válida |
| [API02007](./API02007) | Fallo al Refrescar el Token | No se pudo refrescar su sesión |
| [API02008](./API02008) | Token de Actualización Inválido | El token de actualización no es válido |
| [API02009](./API02009) | Token de Actualización Expirado | El token de actualización ha expirado |
| [API02010](./API02010) | Autenticación Requerida | Se requiere autenticación para esta acción |

## Errores de Solicitud {#request-errors}

Problemas con las solicitudes enviadas al servidor.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [API03001](./API03001) | Formato de Solicitud Inválido | El formato de la solicitud no es correcto |
| [API03002](./API03002) | Parámetros Requeridos Faltantes | Faltan datos requeridos en la solicitud |
| [API03003](./API03003) | Valor de Parámetro Inválido | Un parámetro tiene un valor no válido |
| [API03004](./API03004) | Solicitud Demasiado Grande | La solicitud excede los límites de tamaño |
| [API03005](./API03005) | Límite de Tasa Excedido | Demasiadas solicitudes en poco tiempo |
| [API03006](./API03006) | Método No Soportado | El método HTTP no es compatible |
| [API03007](./API03007) | Cola de Solicitudes Llena | Demasiadas solicitudes pendientes |

## Errores de Respuesta {#response-errors}

Problemas con las respuestas recibidas del servidor.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [API04001](./API04001) | Formato de Respuesta Inválido | El formato de la respuesta del servidor no es válido |
| [API04002](./API04002) | Código de Respuesta Inesperado | Se recibió un código de estado HTTP inesperado |
| [API04003](./API04003) | Respuesta JSON Malformada | La respuesta JSON está dañada o es inválida |
| [API04004](./API04004) | Datos de Respuesta Faltantes | Faltan datos esperados en la respuesta |
| [API04005](./API04005) | Intento de Recuperación JSON Realizado | Se intentó recuperar de un JSON malformado |
| [API04006](./API04006) | Recurso No Encontrado | El recurso solicitado no existe (404) |

Problemas con WordPress, WooCommerce o el plugin WCPOS.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [API05001](./API05001) | API de WooCommerce Desactivada | La API REST de WooCommerce está desactivada |

| [API05003](./API05003) | Plugin WCPOS Desactualizado | El plugin WCPOS necesita actualización |
| [API05004](./API05004) | API de WordPress Desactivada | La API REST de WordPress está desactivada |
| [API05005](./API05005) | Plugin No Encontrado | Falta un plugin requerido |

Problemas de implementación y configuración.

| [API06003](./API06003) | Configuración del Sitio Inválida | La configuración del sitio es incorrecta |
