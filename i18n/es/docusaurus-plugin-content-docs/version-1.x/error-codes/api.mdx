# Errores de API

Los errores de API ocurren al comunicarse con su servidor WooCommerce. Estos errores tienen el prefijo `API` y están organizados en las siguientes categorías:

## Categorías

| Categoría | Rango de Código | Descripción |
|----------|------------|-------------|
| [Conexión](#errores-de-conexión) | API01xxx | Problemas de red y conectividad |
| [Autenticación](#errores-de-autenticación) | API02xxx | Problemas de inicio de sesión, tokens y permisos |
| [Solicitud](#errores-de-solicitud) | API03xxx | Problemas con solicitudes salientes |
| [Respuesta](#errores-de-respuesta) | API04xxx | Problemas con respuestas del servidor |
| [Plugin/WordPress](#errores-de-pluginwordpress) | API05xxx | Problemas con WordPress o plugins de WooCommerce |
| [Configuración](#errores-de-configuración) | API06xxx | Problemas de configuración y setup |

---

## Errores de Conexión

Problemas de red y conectividad entre el POS y su servidor.

| Código | Nombre | Descripción |
|------|------|-------------|
| [API01001](./API01001) | Tiempo de Conexión Agotado | El servidor tardó demasiado en responder |
| [API01002](./API01002) | Conexión Rechazada | El servidor rechazó la conexión |
| [API01003](./API01003) | Conexión Restablecida | La conexión se cerró inesperadamente |
| [API01004](./API01004) | Resolución DNS Fallida | No se pudo resolver la dirección del servidor |
| [API01005](./API01005) | Error de Certificado SSL | Problema con el certificado de seguridad del sitio |
| [API01006](./API01006) | Red Inaccesible | No se puede alcanzar la red |
| [API01007](./API01007) | Dispositivo Sin Conexión | Su dispositivo no está conectado a internet |
| [API01008](./API01008) | Sitio Web No Disponible | El sitio web no está respondiendo |
