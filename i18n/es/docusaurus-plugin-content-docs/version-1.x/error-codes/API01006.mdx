# API01006: Red Inaccesible

## Qué Significa

El POS no puede establecer una ruta al servidor. Este es un problema a nivel de red — los paquetes de datos no pueden encontrar un camino a su servidor.

## Causas Comunes

- **Problemas de configuración de red** — Configuración incorrecta de puerta de enlace o enrutamiento
- **Problemas del ISP** — Su proveedor de servicios de internet está teniendo problemas
- **Interferencia de VPN** — Una VPN puede estar enrutando el tráfico incorrectamente
- **Problemas de red del servidor** — La red del servidor puede estar caída

## Cómo Solucionar

### 1. Compruebe su Conexión a Internet

- Intente acceder a otros sitios web
- Reinicie su router/módem
- Verifique que está conectado a la red

### 2. Deshabilite la VPN Temporalmente

Si está usando una VPN:
- Desconecte y vuelva a intentarlo
- Pruebe con un servidor VPN diferente
- Verifique si la VPN permite el acceso a la ubicación de su servidor

### 3. Consulte con su ISP

Contacte a su proveedor de servicios de internet si:
- Otros sitios también son inaccesibles
- El problema persiste después de reiniciar el equipo de red

### 4. Verifique el Estado del Servidor

- Compruebe si su proveedor de alojamiento reporta problemas de red
- Intente acceder a su sitio desde una red diferente (p. ej., datos móviles)
- Use herramientas en línea para verificar si el sitio está caído para todos

### 5. Compruebe la Configuración del Firewall

Asegúrese de que el firewall de su dispositivo no esté bloqueando conexiones salientes al servidor.
