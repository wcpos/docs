# API01006: Red inalcanzable {#api01006-network-unreachable}

## Qué significa {#what-this-means}

El POS no puede establecer una ruta hacia el servidor. Se trata de un problema a nivel de red: los paquetes de datos no encuentran un camino hasta el servidor.

## Causas comunes {#common-causes}

- **Problemas de configuración de red** — Configuración incorrecta de la puerta de enlace o del enrutamiento
- **Problemas del ISP** — El proveedor de servicios de internet presenta inconvenientes
- **Interferencia de VPN** — Una VPN puede estar enrutando el tráfico de forma incorrecta
- **Problemas de red del servidor** — La red del servidor puede estar caída

## Cómo solucionarlo {#how-to-fix}

### 1. Verificar la conexión a internet {#1-check-your-internet-connection}

- Intentar acceder a otros sitios web
- Reiniciar el router/módem
- Verificar la conexión a la red

### 2. Desactivar la VPN temporalmente {#2-disable-vpn-temporarily}

Si se utiliza una VPN:
- Desconectar e intentar de nuevo
- Probar con un servidor VPN diferente
- Comprobar si la VPN permite el acceso a la ubicación del servidor

### 3. Consultar con el proveedor de servicios de Internet {#3-check-with-your-isp}

Contactar con el proveedor de servicios de Internet si:
- Otros sitios tampoco son accesibles
- El problema persiste después de reiniciar el equipo de red

### 4. Verificar el estado del servidor {#4-verify-server-status}

- Comprobar si el proveedor de hosting informa de algún problema de red
- Intentar acceder al sitio desde una red diferente (por ejemplo, datos móviles)
- Utilizar herramientas en línea para comprobar si el sitio está caído para todos

### 5. Verificar la configuración del firewall {#5-check-firewall-settings}

Asegurarse de que el firewall del dispositivo no esté bloqueando las conexiones salientes hacia el servidor.
