# API01006: Red Inalcanzable {#api01006-network-unreachable}

## Qué Significa Esto {#what-this-means}

El POS no puede establecer una ruta hacia el servidor. Este es un problema a nivel de red: los paquetes de datos no pueden encontrar un camino hacia su servidor.

## Causas Comunes {#common-causes}

- **Problemas de configuración de red** — Configuración incorrecta de la puerta de enlace o de enrutamiento
- **Problemas con el ISP** — Su proveedor de servicios de internet está teniendo problemas
- **Interferencia de VPN** — Una VPN puede estar enrutando el tráfico incorrectamente
- **Problemas de red del servidor** — La red del servidor puede estar caída

## Cómo Arreglarlo {#how-to-fix}

### 1. Verifique Su Conexión a Internet {#1-check-your-internet-connection}

- Intente acceder a otros sitios web
- Reinicie su router/módem
- Verifique que esté conectado a la red

### 2. Desactive la VPN Temporalmente {#2-disable-vpn-temporarily}

Si está usando una VPN:
- Desconéctese y pruebe de nuevo
- Pruebe un servidor VPN diferente
- Verifique si la VPN permite el acceso a la ubicación de su servidor

### 3. Verifique con Su ISP {#3-check-with-your-isp}

Contacte a su proveedor de servicios de internet si:
- Otros sitios también son inalcanzables
- El problema persiste después de reiniciar el equipo de red

### 4. Verifique el Estado del Servidor {#4-verify-server-status}

- Verifique si su proveedor de hosting reporta algún problema de red
- Intente acceder a su sitio desde una red diferente (por ejemplo, datos móviles)
- Utilice herramientas en línea para comprobar si el sitio está caído para todos

### 5. Asegúrese de la Configuración del Firewall {#5-check-firewall-settings}

Asegúrese de que el firewall de su dispositivo no esté bloqueando las conexiones salientes hacia su servidor.
