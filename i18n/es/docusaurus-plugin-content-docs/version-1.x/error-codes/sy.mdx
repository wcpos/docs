# Errores del Sistema

Los errores del sistema están relacionados con recursos del dispositivo y configuración del sistema. Estos errores tienen el prefijo `SY` e indican problemas con su dispositivo o con la aplicación POS.

## Categorías

| Categoría | Rango de Códigos | Descripción |
|----------|------------|-------------|
| [Recursos](#errores-de-recursos) | SY01xxx | Problemas de memoria, disco y permisos |
| [Servicio](#errores-de-servicio) | SY02xxx | Configuración y disponibilidad de servicios |

---

## Errores de Recursos

Problemas con recursos del dispositivo como memoria, almacenamiento y permisos.

| Código | Nombre | Descripción |
|------|------|-------------|
| [SY01001](./SY01001) | Sin Memoria | El dispositivo tiene memoria baja |
| [SY01002](./SY01002) | Disco Lleno | El almacenamiento del dispositivo está completo |
| [SY01003](./SY01003) | Permiso Denegado | Se denegó un permiso del sistema |

## Errores de Servicio

Problemas con la configuración del sistema y la disponibilidad de servicios.

| Código | Nombre | Descripción |
|------|------|-------------|
| [SY02001](./SY02001) | Configuración Inválida | La configuración del sistema no es válida |
| [SY02002](./SY02002) | Servicio No Disponible | Un servicio requerido no está disponible |
