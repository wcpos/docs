# Errores del Sistema

Los errores del sistema están relacionados con los recursos del dispositivo y la configuración del sistema. Estos errores están precedidos por `SY` e indican problemas con su dispositivo o la aplicación de POS.

## Categorías

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| [Recurso](#resource-errors) | SY01xxx | Problemas de memoria, disco y permisos |
| [Servicio](#service-errors) | SY02xxx | Configuración y disponibilidad del servicio |

---

## Errores de Recursos

Problemas con los recursos del dispositivo, como memoria, almacenamiento y permisos.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [SY01001](./SY01001) | Sin Memoria | El dispositivo está quedándose sin memoria |
| [SY01002](./SY01002) | Disco Lleno | El almacenamiento del dispositivo está lleno |
| [SY01003](./SY01003) | Permiso Denegado | Se negó el permiso del sistema |

## Errores de Servicio

Problemas con la configuración del sistema y la disponibilidad del servicio.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [SY02001](./SY02001) | Configuración Inválida | La configuración del sistema es inválida |
| [SY02002](./SY02002) | Servicio No Disponible | Un servicio requerido no está disponible |
