# Errores del Sistema {#system-errors}

Los errores del sistema están relacionados con los recursos del dispositivo y la configuración del sistema. Estos errores están precedidos por `SY` e indican problemas con su dispositivo o la propia aplicación de POS.

## Categorías {#categories}

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| [Recurso](#errores-de-recursos) | SY01xxx | Problemas de memoria, disco y permisos |
| [Servicio](#errores-de-servicio) | SY02xxx | Problemas de configuración y disponibilidad del servicio |

---

## Errores de Recursos {#resource-errors}

Problemas con los recursos del dispositivo, como memoria, almacenamiento y permisos.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [SY01001](./SY01001) | Sin Memoria | El dispositivo tiene poca memoria |
| [SY01002](./SY01002) | Disco Lleno | El disco del dispositivo está lleno |
| [SY01003](./SY01003) | Permiso Denegado | Se denegó el permiso del sistema |

## Errores de Servicio {#service-errors}

Problemas con la configuración del sistema y la disponibilidad del servicio.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [SY02001](./SY02001) | Configuración Inválida | La configuración del sistema es inválida |
| [SY02002](./SY02002) | Servicio No Disponible | Un servicio requerido no está disponible |
