# Errores del sistema {#system-errors}

Los errores del sistema están relacionados con los recursos del dispositivo y la configuración del sistema. Estos errores llevan el prefijo `SY` e indican problemas con su dispositivo o con la propia aplicación POS.

## Categorías {#categories}

| Categoría | Rango de códigos | Descripción |
|----------|------------|-------------|
| [Recursos](#resource-errors) | SY01xxx | Problemas de memoria, disco y permisos |
| [Servicio](#service-errors) | SY02xxx | Configuración y disponibilidad del servicio |

---

## Errores de recursos {#resource-errors}

Problemas con recursos del dispositivo como memoria, almacenamiento y permisos.

| Código | Nombre | Descripción |
|------|------|-------------|
| [SY01001](./SY01001) | Memoria insuficiente | El dispositivo se está quedando sin memoria |
| [SY01002](./SY01002) | Disco lleno | El almacenamiento del dispositivo está lleno |
| [SY01003](./SY01003) | Permiso denegado | Se denegó un permiso del sistema |

## Errores de servicio {#service-errors}

Problemas con la configuración del sistema y la disponibilidad de servicios.

| Código | Nombre | Descripción |
|------|------|-------------|
| [SY02001](./SY02001) | Configuración no válida | La configuración del sistema no es válida |
| [SY02002](./SY02002) | Servicio no disponible | Un servicio requerido no está disponible |
