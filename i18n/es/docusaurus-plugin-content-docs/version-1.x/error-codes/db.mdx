# Errores de Base de Datos

Los errores de base de datos ocurren al almacenar o recuperar datos en la base de datos local. WCPOS utiliza una base de datos local para almacenar productos, clientes y otros datos para un acceso rápido y funcionalidad fuera de línea. Estos errores están precedidos por `DB`.

## Categorías

| Categoría | Rango de Códigos | Descripción |
|-----------|------------------|-------------|
| [Conexión](#connection-errors) | DB01xxx | Problemas de conexión y transacción de base de datos |
| [Datos](#data-errors) | DB02xxx | Problemas de registros y restricciones |
| [Consulta](#query-errors) | DB03xxx | Problemas de sintaxis de consulta y tipos de datos |

---

## Errores de Conexión

Problemas al conectarse o al operar sobre la base de datos local.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [DB01001](./DB01001) | Conexión Fallida | No se pudo conectar a la base de datos local |
| [DB01002](./DB01002) | Tiempo de Espera de Consulta | La consulta de base de datos tardó demasiado |
| [DB01003](./DB01003) | Transacción Fallida | La transacción de base de datos no pudo completarse |

## Errores de Datos

Problemas con registros, duplicados y restricciones.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [DB02001](./DB02001) | Registro Duplicado | Ya existe un registro con este ID |
| [DB02002](./DB02002) | Registro No Encontrado | El registro solicitado no existe |
| [DB02003](./DB02003) | Violación de Restricción | Los datos violan las restricciones de la base de datos |

## Errores de Consulta

Problemas con la sintaxis de consulta y los tipos de datos.

| Código | Nombre | Descripción |
|--------|--------|-------------|
| [DB03001](./DB03001) | Error de Sintaxis de Consulta | La consulta de base de datos tiene errores de sintaxis |
| [DB03002](./DB03002) | Tipo de Dato Inválido | El tipo de dato no coincide con el tipo esperado |
| [DB03003](./DB03003) | Campo Requerido Faltante | Un campo requerido está vacío |
