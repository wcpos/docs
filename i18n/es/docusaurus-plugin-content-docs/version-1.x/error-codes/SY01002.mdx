# SY01002: Disco Lleno {#sy01002-disk-full}

## Qué Significa Esto {#what-this-means}

El almacenamiento de su dispositivo está lleno. El POS no puede guardar datos o continuar operando sin espacio disponible en el disco.

## Causas Comunes {#common-causes}

- **Almacenamiento lleno** — Sin espacio libre en la unidad
- **Base de datos local grande** — Los datos del POS ocupan un espacio significativo
- **Archivos temporales** — Archivos temporales consumiendo espacio
- **Otras aplicaciones** — Otras aplicaciones llenando el disco

## Cómo Arreglarlo {#how-to-fix}

### 1. Liberar Espacio en Disco {#1-free-up-disk-space}

Elimine archivos innecesarios:
- Vacíe la papelera de reciclaje
- Elimine descargas antiguas
- Elimine aplicaciones no utilizadas
- Elimine archivos temporales

### 2. Verifique el Espacio Disponible {#2-check-available-space}

**Windows:**
- Abra el Explorador de Archivos
- Haga clic derecho en la unidad → Propiedades
- Vea el espacio usado y libre

**macOS:**
- Menú de Apple → Acerca de Este Mac
- Haga clic en Almacenamiento

### 3. Borrar Datos del Navegador (Versión Web) {#3-clear-browser-data-web-version}

Los navegadores almacenan datos localmente:
- Limpie caché y cookies
- Elimine datos de sitios antiguos
- Verifique el uso de almacenamiento del navegador

### 4. Mover o Eliminar Archivos Grandes {#4-move-or-delete-large-files}

Encuentre y maneje archivos grandes:
- Videos o fotos antiguas
- Descargas grandes
- Software no utilizado
- Archivos de respaldo que pueden ser movidos a almacenamiento externo

### 5. Verificar Datos del POS {#5-check-pos-data}

Si los datos del POS son muy grandes:
- La base de datos local puede necesitar limpieza
- Considere limpiar y volver a sincronizar
- Verifique si los datos antiguos pueden ser eliminados

## Espacio Libre Recomendado {#recommended-free-space}

Mantenga al menos:
- **10GB** libres para operación normal
- **20GB+** para catálogos grandes
- Más espacio siempre es mejor

## Previniendo Problemas Futuros {#preventing-future-issues}

- Limpie regularmente archivos innecesarios
- Configure limpieza automática de archivos temporales
- Monitoree el uso del espacio en disco
- Considere almacenamiento más grande si frecuentemente está lleno
