# SY01002: Disco Lleno

## Qué Significa

El almacenamiento de tu dispositivo está lleno. El POS no puede guardar datos ni continuar operando sin espacio de disco disponible.

## Causas Comunes

- **Almacenamiento lleno** — No hay espacio libre en la unidad
- **Base de datos local grande** — Los datos del POS están utilizando un espacio significativo
- **Archivos temporales** — Archivos temporales consumiendo espacio
- **Otras aplicaciones** — Otras aplicaciones llenando el disco

## Cómo Solucionarlo

### 1. Liberar Espacio en Disco

Eliminar archivos innecesarios:
- Vaciar la papelera
- Eliminar descargas antiguas
- Borrar aplicaciones sin usar
- Limpiar archivos temporales

### 2. Verificar Espacio Disponible

**Windows:**
- Abrir Explorador de Archivos
- Clic derecho en la unidad → Propiedades
- Ver espacio usado y libre

**macOS:**
- Menú de Apple → Acerca de este Mac
- Hacer clic en Almacenamiento

### 3. Limpiar Datos del Navegador (Versión Web)

Los navegadores almacenan datos localmente:
- Borrar caché y cookies
- Eliminar datos de sitios antiguos
- Verificar uso de almacenamiento del navegador

### 4. Mover o Eliminar Archivos Grandes

Encontrar y manejar archivos grandes:
- Videos o fotos antiguos
- Descargas grandes
- Software sin usar
- Archivos de respaldo que se pueden mover a almacenamiento externo

### 5. Verificar Datos del POS

Si los datos del POS son muy grandes:
- La base de datos local puede necesitar limpieza
- Considerar limpiar y resincronizar
- Verificar si se pueden purgar datos antiguos

## Espacio Libre Recomendado

Mantener al menos:
- **10GB** libres para operación normal
- **20GB+** para catálogos grandes
- Más espacio es siempre mejor

## Prevenir Problemas Futuros

- Limpiar regularmente archivos innecesarios
- Configurar limpieza automática de archivos temporales
- Monitorear uso de espacio en disco
- Considerar almacenamiento más grande si se llena frecuentemente
