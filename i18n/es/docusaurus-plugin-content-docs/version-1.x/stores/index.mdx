:::info Función Pro
La funcionalidad multitienda requiere [WCPOS Pro](/getting-started/pro-license).
:::

WCPOS Pro te permite crear ubicaciones de tienda separadas dentro de una única instalación de WooCommerce. Cada tienda puede tener su propia dirección, logotipo y ajustes — útil si operas desde varias ubicaciones físicas.

## Qué ofrecen las tiendas {#what-stores-provide}

Cada tienda puede tener:

- **Nombre y dirección de la tienda** — se utilizan en recibos e informes
- **Logotipo de la tienda** — se muestra en los recibos impresos
- **Tipos de impuestos** — distintas ubicaciones pueden tener diferentes requisitos fiscales
- **Asignación de cajeros** — controla qué cajeros trabajan en cada tienda

## Configuración de tiendas {#setting-up-stores}

1. Ve a `WP Admin > POS > Settings`
2. Crea una nueva tienda con su nombre, dirección y logotipo
3. Configura los ajustes de impuestos específicos de la tienda
4. Asigna la tienda a los usuarios del POS

Cuando un cajero inicia sesión en el POS, selecciona desde qué tienda está operando. Esto determina los datos de la tienda que aparecen en los recibos y qué filtro de tienda se aplica a los informes.

## Filtrado por tienda {#filtering-by-store}

Una vez que tengas varias tiendas configuradas, el filtro de tienda aparece en todo el POS:

- **[Pedidos](/orders)** — filtra el historial de pedidos por ubicación de tienda
- **[Informes](/reports)** — genera informes de fin de día para una tienda específica
- **[Recibos](/pos/checkout/receipts)** — los recibos muestran automáticamente la dirección y el logotipo correctos de la tienda

## Precios por tienda {#store-specific-pricing}

WCPOS Pro admite **precios diferentes para distintas ubicaciones de tienda**. Puede establecer un precio normal y un precio rebajado independientes por tienda, tanto para productos como para variaciones.

Cuando un cajero opera desde una tienda específica, el POS utiliza automáticamente los precios de esa tienda. Si no se ha establecido un precio específico para la tienda, se utiliza el precio predeterminado de WooCommerce.

## Gestión de inventario {#inventory-management}

De forma predeterminada, todas las tiendas comparten los mismos niveles de stock de WooCommerce. Para el seguimiento de inventario por ubicación, WCPOS Pro se integra con el plugin [ATUM Inventory Management](https://www.atum.org/):

1. Instale y configure el plugin ATUM en su sitio de WooCommerce
2. Cree ubicaciones de inventario en ATUM
3. En WCPOS Pro, asocie cada tienda con una ubicación de inventario de ATUM
4. Los niveles de stock se actualizarán para la ubicación correcta cuando se realicen pedidos

### Alternativa: WordPress Multisite {#alternative-wordpress-multisite}

Para inventarios totalmente separados sin ATUM, puede utilizar [WordPress Multisite](https://developer.wordpress.org/advanced-administration/multisite/create-network/):

1. Cree un sitio de WordPress independiente para cada ubicación de tienda
2. Instala WooCommerce y WCPOS en cada sitio
3. Cada sitio mantiene su propio catálogo de productos y niveles de stock independientes

Esto proporciona una separación completa, pero implica gestionar múltiples instalaciones de WooCommerce.

## Limitaciones conocidas {#known-limitations}

- **Instancia única de WooCommerce** — las tiendas son ubicaciones dentro de una sola instalación de WooCommerce, no comercios independientes
- **Sin visibilidad de productos por tienda** — todas las tiendas ven el mismo catálogo de productos
