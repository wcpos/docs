WCPOS funciona con los dispositivos físicos que ya se utilizan en el mostrador. Esta página es el punto de partida para conectar hardware: impresoras de recibos, escáneres de códigos de barras, lectores de tarjetas y cajones de efectivo. Cada tipo de dispositivo tiene su propia página de configuración aquí, o apunta a la página donde se encuentra realmente su configuración.

:::note Un único lugar por dispositivo
El centro de Hardware dirige a la configuración de cada dispositivo, pero los pasos reales se encuentran en un único lugar. La cuenta y el proceso de vinculación de un lector de tarjetas se encuentran en su página de Pasarela de pago; la configuración de apertura de un cajón de dinero se encuentra en la página de Impresoras de recibos.
:::
