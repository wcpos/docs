WCPOS funciona con los dispositivos físicos que ya se utilizan en el mostrador. Esta página es el punto de partida para conectar hardware: impresoras de recibos, escáneres de códigos de barras, lectores de tarjetas y cajones de efectivo. Cada tipo de dispositivo tiene su propia página de configuración aquí o enlaza a la página donde se realiza realmente su configuración.

:::note Un único lugar por dispositivo
El centro de Hardware dirige a la configuración de cada dispositivo, pero los pasos reales se encuentran en un único lugar. La cuenta y el flujo de emparejamiento de un lector de tarjetas se encuentran en su página de Pasarela de pago; la configuración de apertura de un cajón de dinero se encuentra en la página de Impresoras de recibos.
:::

## Compatibilidad de lectores de tarjetas {#card-reader-compatibility}

Los lectores de tarjetas compatibles dependen de **cómo se ejecuta WCPOS** y del tipo de conexión del lector:

- **La aplicación web** solo puede controlar lectores que ofrezcan un **SDK web (navegador)** — generalmente terminales de mostrador con conexión a internet. Los lectores **solo Bluetooth** (p. ej., Stripe M2 / WisePad 3, SumUp Air) y el **Tap-to-Pay** del teléfono **no son compatibles** con la aplicación web.
- La compatibilidad es **por pasarela**, no universal — es necesario verificar el lector con la pasarela que se planea utilizar ([Stripe Terminal](/payment/gateways/stripe-terminal), [SumUp](/payment/gateways/sumup-terminal), [PayPal Reader](/payment/gateways/paypal-reader)) antes de comprarlo.

Cada página de pasarela enumera los lectores exactos que admite.
