:::info Característica Pro
La pantalla de Clientes requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden seleccionar clientes existentes para pedidos, pero no pueden ver la lista completa de clientes ni editar los detalles del cliente.
:::

La pantalla de Clientes proporciona una gestión integral de clientes directamente dentro del POS. Visualice, edite y cree clientes sin necesidad de cambiar a la administración de WooCommerce.

{/* Screenshot: customers-screen.png */}

## Descripción general de la interfaz

### Acciones del encabezado

En la parte superior de la pantalla:

- **Barra de búsqueda** - Encontrar clientes por nombre, correo electrónico, etc.
- **Añadir Cliente** (<Icon name="plus" />) - Crear un nuevo cliente
- **Configuración de visualización** (<Icon name="sliders" />) - Configurar columnas visibles

### Tabla de Clientes

El área principal muestra clientes con:

- **Avatar** - Imagen de perfil del cliente (o marcador de posición)
- **Nombre** - Nombre del cliente
- **Apellido** - Apellido del cliente (ordenable)
- **Correo electrónico** - Dirección de correo electrónico de contacto
- **Dirección de facturación** - Dirección de facturación completa
- **Fecha de creación** - Cuándo se agregó el cliente
- **Acciones** - Menú de tres puntos

### Pie de página

- Recuento de clientes con botón de sincronización

## Características principales

### Búsqueda de clientes

Encuentre clientes rápidamente por:

- Nombre o apellido
- Dirección de correo electrónico
- Información de dirección

### Añadir nuevo cliente

Cree clientes directamente desde el POS:

1. Haga clic en el icono <Icon name="plus" /> en el encabezado
2. Complete los detalles del cliente
3. Guarde el nuevo cliente

El cliente se crea en WooCommerce y está inmediatamente disponible para pedidos.

### Editar cliente

Actualizar información del cliente:

1. Haga clic en el menú de tres puntos de un cliente
2. Seleccione **Editar**
3. Modifique los detalles (nombre, correo electrónico, direcciones)
4. Guarde los cambios

Los cambios se sincronizan automáticamente con WooCommerce.

## Configuración de visualización

Haga clic en el icono de controles deslizantes (<Icon name="sliders" />) para personalizar las columnas visibles.

### Columnas disponibles

| Columna | Descripción |
|--------|-------------|
| **Imagen** | Avatar del cliente |
| **ID** | ID de cliente de WooCommerce |
| **Nombre** | Nombre del cliente |
| **Apellido** | Apellido del cliente |
| **Correo electrónico** | Dirección de correo electrónico |
| **Rol** | Rol de usuario (Cliente, Suscriptor, etc.) |
| **Nombre de usuario** | Nombre de usuario de WordPress |
| **Dirección de facturación** | Información de facturación |
| **Dirección de envío** | Información de envío |
| **Fecha de creación** | Cuándo se agregó el cliente |
| **Fecha de modificación** | Última actualización |
| **Acciones** | Editar, Sincronizar, Eliminar |

## Acciones de cliente

Haga clic en el menú de tres puntos (⋮) para ver las opciones:

- **Editar** - Modificar detalles del cliente
- **Sincronizar** - Actualizar cliente desde el servidor
- **Eliminar** - Eliminar de la base de datos local

:::note
Eliminar un cliente desde el POS solo lo elimina localmente. El cliente permanece en WooCommerce y volverá a aparecer en la próxima sincronización.
:::

## Usar clientes en pedidos

Para asignar un cliente a un pedido:

1. En el [Panel de Carrito](/pos/cart), haga clic en la insignia del cliente
2. Busque el cliente por nombre o correo electrónico
3. Seleccione el cliente
4. El cliente está ahora asociado al pedido

La información del cliente (direcciones de facturación/envío) se usa automáticamente para el pedido.

## Sincronización

Los datos de clientes se sincronizan entre el POS y WooCommerce:

- **Actualizaciones en tiempo real** - Los cambios se sincronizan en todos los dispositivos
- **Integración con WooCommerce** - Los clientes se almacenan en WooCommerce
- **Capacidad sin conexión** - Ver clientes incluso sin conexión
