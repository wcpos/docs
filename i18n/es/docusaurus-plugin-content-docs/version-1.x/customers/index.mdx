:::info Función Pro
La pantalla de Clientes requiere [WCPOS Pro](/getting-started/pro-license). Los usuarios gratuitos pueden seleccionar clientes existentes para pedidos, pero no pueden ver la lista completa de clientes ni editar los detalles del cliente.
:::

La pantalla de Clientes proporciona una gestión integral de clientes directamente en el POS. Visualiza, edita y crea clientes sin cambiar al administrador de WooCommerce.

{/* Screenshot: customers-screen.png */}

## Resumen de la Interfaz {#interface-overview}

### Acciones del Encabezado {#header-actions}

En la parte superior de la pantalla:

- **Barra de búsqueda** - Encuentra clientes por nombre, correo electrónico, etc.
- **Agregar Cliente** (<Icon name="plus" />) - Crea un nuevo cliente
- **Configuración de visualización** (<Icon name="sliders" />) - Configura las columnas visibles

### Tabla de Clientes {#customer-table}

El área principal muestra los clientes con:

- **Avatar** - Foto de perfil del cliente (o marcador de posición)
- **Nombre** - Primer nombre del cliente
- **Apellido** - Apellido del cliente (ordenable)
- **Correo Electrónico** - Dirección de correo electrónico de contacto
- **Dirección de Facturación** - Dirección de facturación completa
- **Fecha de Creación** - Cuando se añadió al cliente
- **Acciones** - Menú de tres puntos

### Pie de Página {#footer}

- Conteo de clientes con botón de sincronización (<Icon name="arrow-rotate-right" />). **Presionar prolongadamente** para la opción de Borrar y Actualizar

## Características Clave {#key-features}

### Búsqueda de Clientes {#customer-search}

Encuentra clientes rápidamente por:

- Nombre o apellido
- Dirección de correo electrónico
- Información de dirección

### Agregar Nuevo Cliente {#add-new-customer}

Crea clientes directamente desde el POS:

1. Haz clic en el icono <Icon name="plus" /> en el encabezado
2. Rellena los detalles del cliente
3. Guarda el nuevo cliente

El cliente se crea en WooCommerce y está inmediatamente disponible para pedidos.

### Editar Cliente {#edit-customer}

Actualiza la información del cliente:

1. Haz clic en el menú de tres puntos en un cliente
2. Selecciona **Editar**
3. Modifica los detalles (nombre, correo electrónico, direcciones)
4. Guarda los cambios

Los cambios se sincronizan automáticamente con WooCommerce.

## Configuración de Visualización {#display-settings}

Haz clic en el icono de deslizadores (<Icon name="sliders" />) para personalizar las columnas visibles.

### Columnas Disponibles {#available-columns}

| Columna       | Descripción                     |
|---------------|---------------------------------|
| **Imagen**    | Avatar del cliente              |
| **ID**        | ID del cliente en WooCommerce    |
| **Nombre**    | Primer nombre del cliente       |
| **Apellido**  | Apellido del cliente            |
| **Correo**    | Dirección de correo electrónico  |
| **Rol**       | Rol del usuario (Cliente, Suscriptor, etc.) |
| **Nombre de Usuario** | Nombre de usuario de WordPress |
| **Dirección de Facturación** | Información de facturación |
| **Dirección de Envío** | Información de envío      |

| **Fecha de Modificación** | Última actualización   |
| **Acciones**  | Editar, Sincronizar, Eliminar   |

## Acciones del Cliente {#customer-actions}

Haz clic en el menú de tres puntos (⋮) para opciones:

- **Editar** - Modificar los detalles del cliente
- **Sincronizar** - Actualizar cliente desde el servidor
- **Eliminar** - Eliminar de la base de datos local

:::note
Eliminar un cliente del POS solo lo elimina localmente. El cliente permanece en WooCommerce y reaparecerá en la próxima sincronización.
:::

## Uso de Clientes en Pedidos {#using-customers-in-orders}

Para asignar un cliente a un pedido:

1. En el [Panel del Carrito](/pos/cart), haz clic en la insignia del cliente
2. Busca al cliente por nombre o correo electrónico
3. Selecciona al cliente
4. El cliente ahora está asociado con el pedido

La información del cliente (direcciones de facturación/envío) se utiliza automáticamente para el pedido.

## Sincronización {#synchronization}

Los datos del cliente se sincronizan entre el POS y WooCommerce:

- **Actualizaciones en tiempo real** - Los cambios se sincronizan en todos los dispositivos
- **Integración con WooCommerce** - Los clientes se almacenan en WooCommerce
- **Capacidad sin conexión** - Ver clientes incluso cuando no estás en línea
