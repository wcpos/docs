WCPOS es el plugin de Punto de Venta para [WooCommerce](https://www.woothemes.com/woocommerce/).
Esta documentación cubre el plugin de código abierto, así como el plugin Pro que añade funcionalidad al plugin gratuito.

- Descarga [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) desde WordPress.org
- Mejora a [WCPOS Pro](http://wcpos.com/pro)

:::danger

Por favor, asegúrate de que el número de versión de la documentación coincida con la versión de tus plugins instalados.

:::

### Otra documentación

Una lista de **Preguntas Frecuentes** está disponible en [faq.wcpos.com](http://faq.wcpos.com).
**La documentación para desarrolladores** está disponible en [api.wcpos.com](http://api.wcpos.com).

### Ayuda y Soporte

Si tienes una pregunta que no está cubierta por esta documentación o las Preguntas Frecuentes, puedes contactarnos en [support@wcpos.com](mailto:support@wcpos.com).
