WCPOS es el complemento de punto de venta para [WooCommerce](https://www.woothemes.com/woocommerce/).
Esta documentación cubre el complemento de código abierto, así como el complemento Pro que añade funcionalidad al complemento gratuito.

- Descargar [WCPOS](https://wordpress.org/plugins/woocommerce-pos/) desde WordPress.org
- Actualizar a [WCPOS Pro](http://wcpos.com/pro)

:::danger

Asegúrese de que el número de versión de la documentación coincida con la versión de los complementos instalados.

:::

### Otra documentación

Una lista de **Preguntas frecuentes** está disponible en [faq.wcpos.com](http://faq.wcpos.com).
La **documentación para desarrolladores** está disponible en [api.wcpos.com](http://api.wcpos.com).
