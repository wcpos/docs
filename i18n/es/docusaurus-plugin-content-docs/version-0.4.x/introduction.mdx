WooCommerce POS es el plugin de Punto de Venta para [WooCommerce](https://www.woothemes.com/woocommerce/). 
Esta documentación cubre el plugin de código abierto, así como el plugin Pro que agrega funcionalidad al plugin gratuito.

* Descarga [WooCommerce POS](https://wordpress.org/plugins/woocommerce-pos/) desde WordPress.org
* Actualiza a [WooCommerce POS Pro](http://wcpos.com/pro)

:::danger

Por favor asegúrate de que el número de versión de la documentación coincida con la versión de tus plugins instalados.

:::
