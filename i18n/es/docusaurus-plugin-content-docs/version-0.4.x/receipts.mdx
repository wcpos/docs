Las plantillas de impresión de recibos del POS se encuentran en el archivo `includes/views/print/tmpl-receipt.php` tanto en los plugins gratuito como Pro. Las plantillas de recibos pueden personalizarse creando un archivo `woocommerce-pos/print/tmpl-receipt.php` en el directorio de tu tema. A continuación se incluye el código de ambas plantillas.

### Plantilla de Recibo Pro

La Plantilla de Recibo Pro permite información adicional desde la administración de [Tiendas](./stores) como logotipo, dirección de la tienda, horario de apertura y mensajes especiales.

### Personalizando la fecha del recibo

WCPOS usa [moment.js](http://momentjs.com/) para localizar las cadenas de fecha. El formato de fecha predeterminado es `"MMMM Do YYYY, h:mm:ss a"`, por ejemplo: 31 de mayo de 2015, 7:20:44 pm. Consulte la [documentación de moment.js](http://momentjs.com/docs/#/parsing/string-format/) para obtener más información sobre cómo personalizar el formato de fecha.

### Propiedades de la Orden

WCPOS utiliza la salida json de la API REST de WC para rellenar la plantilla de recibo de pedido. [La documentación de la API REST de WC muestra un ejemplo](http://woothemes.github.io/woocommerce-rest-api-docs/#view-an-order) de la salida json estándar. WCPOS ha agregado algunas propiedades adicionales utilizando el filtro `woocommerce_api_order_response`.

| Propiedad                         | Descripción                                                   |
| --------------------------------- | ------------------------------------------------------------- |
| cart_discount_tax                 | La parte de impuestos del descuento del carrito               |
| cashier.id                        | ID de usuario del Cajero                                      |
| cashier.first_name                | Nombre de pila del Cajero                                     |
| cashier.last_name                 | Apellido del Cajero                                           |
| payment_details.result            | Éxito o fallo de la pasarela de pago                          |
| payment_details.message           | Cualquier mensaje de la pasarela de pago                      |
| payment_details.redirect          | URL de redirección para pagos fuera de sitio, ej: PayPal Standard |
| payment_details.method\_`gateway` | Mensajes específicos de la pasarela, ej: Monto Ingresado y Cambio |
| shipping_lines[i].total_tax       | Añadir importe de impuestos para cada línea de envío          |
| subtotal_tax                      | La parte de impuestos del subtotal                            |
