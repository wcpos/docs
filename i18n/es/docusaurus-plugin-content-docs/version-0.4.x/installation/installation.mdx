### Instalación automática (recomendada)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Instala y activa WCPOS en menos de 1 minuto")

1. Ve a `WP Admin > Plugins > Añadir nuevo`
2. Busca `WCPOS` en el Directorio de Plugins de WordPress
3. Instala el plugin
4. Activa el plugin

### Instalación manual

1. [Descargar WCPOS](https://wordpress.org/plugins/woocommerce-pos/) de WordPress.org
2. Ve a `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. Activa el plugin

:::info
Las versiones anteriores de WCPOS se pueden encontrar en la pestaña de [Desarrolladores](https://wordpress.org/plugins/woocommerce-pos/developers/) en Wordpress.org.
:::
