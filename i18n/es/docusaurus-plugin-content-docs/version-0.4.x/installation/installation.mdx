### Instalación automática (recomendada) {#automatic-installation-recommended}

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Instalación y activación de WCPOS en menos de 1 minuto")

1. Ir a `WP Admin > Plugins > Añadir nuevo`
2. Buscar `WCPOS` en el directorio de plugins de WordPress
3. Instalar el plugin
4. Activar el plugin

### Instalación manual {#manual-installation}

1. [Descargar WCPOS](https://wordpress.org/plugins/woocommerce-pos/) desde WordPress.org
2. Ir a `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. Activar el plugin

:::info
Las versiones anteriores de WCPOS se pueden encontrar en la pestaña [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) en WordPress.org.
:::
