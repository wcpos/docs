### Instalación automática (recomendado)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Instala y activa WCPOS en menos de 1 minuto")

1. Ir a `WP Admin > Plugins > Añadir nuevo`
2. Buscar `WCPOS` en el Directorio de Plugins de WordPress
3. Instalar el plugin
4. Activar el plugin

### Instalación manual

1. [Descargar WCPOS](https://wordpress.org/plugins/woocommerce-pos/) desde WordPress.org
2. Ir a `WP Admin > Plugins > Añadir nuevo > Subir plugin`
3. Activar el plugin

:::info
Las versiones anteriores de WCPOS se pueden encontrar en la pestaña [Desarrolladores](https://wordpress.org/plugins/woocommerce-pos/developers/) en Wordpress.org.
:::
