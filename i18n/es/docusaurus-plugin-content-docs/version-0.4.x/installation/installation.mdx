### Automatic installation (recommended)

![](https://wcpos.com/wp-content/uploads/2016/06/wcpos-install.gif "Install and activate WCPOS in less than 1 minute")

1. Go to `WP Admin > Plugins > Add New`
2. Search for `WCPOS` in the WordPress Plugin Directory
3. Install the plugin
4. Activate the plugin

### Manual installation

1. [Download WCPOS](https://wordpress.org/plugins/woocommerce-pos/) from WordPress.org
2. Go to `WP Admin > Plugins > Add New > Upload Plugin`
3. Activate the plugin

:::important

Previous versions of WCPOS can be found under the [Developers](https://wordpress.org/plugins/woocommerce-pos/developers/) tab on Wordpress.org.

:::
