### Visibilidad de productos {#product-visibility}

Los productos exclusivos de POS solo deben habilitarse si se requiere que algunos productos estén ocultos en la tienda en línea o en la tienda física.
Consulte la documentación de [Productos exclusivos de POS](../products/pos-only-products) para obtener más información.

### Cliente POS predeterminado {#default-pos-customer}

Un nuevo pedido se inicializará con el cliente POS predeterminado. Las opciones comunes son:

- **Invitado:** Los pedidos POS no se asocian con ningún ID de usuario
- **Cajero:** Los pedidos POS se asocian con la cuenta del usuario que ha iniciado sesión
- **Cliente presencial:** Los pedidos POS se asocian con una cuenta de usuario personalizada (_nota: es necesario crear esta cuenta previamente_)

Los usuarios de [WCPOS Pro](https://wcpos.com/pro) pueden crear cuentas de usuario _sobre la marcha_ durante el proceso de pago.
Se requiere una dirección de correo electrónico para crear una nueva cuenta; este es un requisito de WordPress.
**Se recomienda crear una cuenta para los nuevos clientes**, ya que permite realizar un seguimiento de su historial de pedidos y contactarlos en el futuro con ofertas promocionales.
