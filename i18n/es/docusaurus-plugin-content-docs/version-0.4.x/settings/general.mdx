### Visibilidad de Productos

Los productos solo para POS deben habilitarse solo si necesita que algunos productos estén ocultos ya sea en la tienda en línea o en la tienda física.
Consulte el documento de [Productos solo para POS](../products/pos-only-products) para obtener más información.

### Cliente POS Predeterminado

Un nuevo pedido se iniciará con el cliente POS predeterminado. Las opciones comunes son:

- **Invitado:** Los pedidos POS no están asociados con ningún ID de usuario
- **Cajero:** Los pedidos POS están asociados con la cuenta del usuario que ha iniciado sesión
- **Cliente Ocasional:** Los pedidos POS están asociados con una cuenta de usuario personalizada (_nota: esta cuenta deberá crearse_)

Los usuarios de [WCPOS Pro](https://wcpos.com/pro) pueden crear cuentas de usuario _sobre la marcha_ durante el pago.
Se requiere una dirección de correo electrónico para crear una nueva cuenta, este es un requisito de WordPress.
**Se recomienda crear una cuenta para nuevos clientes** ya que esto le permitirá hacer un seguimiento del historial de pedidos y potencialmente contactar a sus clientes con ofertas promocionales en el futuro.
