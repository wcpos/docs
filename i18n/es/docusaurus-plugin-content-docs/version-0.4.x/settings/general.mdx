### Visibilidad del Producto

Los productos solo para POS deben habilitarse únicamente si necesita que algunos productos estén ocultos de la tienda en línea o física.
Consulte el documento [Productos Solo para POS](../products/pos-only-products) para obtener más información.

### Cliente Predeterminado de POS

Un nuevo pedido se inicializará con el cliente predeterminado de POS. Las opciones comunes son:

- **Invitado:** Los pedidos de POS no están asociados con ningún ID de usuario
- **Cajero:** Los pedidos de POS están asociados con la cuenta del usuario que ha iniciado sesión
- **Cliente de Entrada:** Los pedidos de POS están asociados con una cuenta de usuario personalizada (_nota: esta cuenta deberá ser creada_)

Los usuarios de [WCPOS Pro](https://wcpos.com/pro) pueden crear cuentas de usuario _al vuelo_ durante el proceso de pago.
Se requiere una dirección de correo electrónico para crear una nueva cuenta. Este es un requisito de WordPress.
**Se recomienda crear una cuenta para nuevos clientes** ya que significa que puede rastrear su historial de pedidos y potencialmente contactar a sus clientes con ofertas promocionales en el futuro.
