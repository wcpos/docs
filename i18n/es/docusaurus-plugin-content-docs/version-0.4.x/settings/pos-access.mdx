Mediante los ajustes de Acceso al POS, los propietarios de tiendas pueden otorgar o revocar privilegios de usuario en el sistema de Punto de Venta.
De forma predeterminada, los roles **Administrador** y **Gerente de tienda** están habilitados con todas las capacidades para utilizar el POS.
Se recomienda mantener los ajustes predeterminados a menos que se conozcan las consecuencias.
Por ejemplo, otorgar `publish_shop_orders` al rol `Subscribers` sería una _muy_ mala idea.

![Ejemplo de ajustes de Acceso al POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Example of POS Access settings")

| Capacidad                                                               | Acceso                                              |
| ----------------------------------------------------------------------- | --------------------------------------------------- |
| access_woocommerce_pos                                                  | Requerido para acceder a la interfaz del POS        |
| manage_woocommerce_pos                                                  | Requerido para acceder a los ajustes de admin del POS |
| read_private_products                                                   | Requerido para ver productos                        |
| edit_product <br /> edit_published_products <br /> edit_others_products | Necesario para editar productos y variaciones  |
| read_private_shop_orders                                                | Necesario para ver pedidos y recibos      |
| publish_shop_orders                                                     | Necesario para crear nuevos pedidos       |
| list_users                                                              | Necesario para ver clientes               |
| create_users                                                            | Necesario para crear nuevos clientes      |
| edit_users                                                              | Necesario para actualizar clientes existentes |
| read_private_shop_coupons                                               | Necesario para ver cupones                |
