Mediante la configuración de Acceso al POS, los propietarios de tiendas pueden conceder o eliminar privilegios de usuario al sistema de Punto de Venta.
De manera predeterminada, los roles de **Administrador** y **Administrador de Tienda** están habilitados con todas las capacidades para usar el POS.
Se recomienda mantener la configuración predeterminada a menos que conozca las consecuencias.
Por ejemplo, conceder el permiso `publish_shop_orders` al rol de `Suscriptores` sería una idea _muy_ mala.

![Ejemplo de configuración de Acceso al POS](https://wcpos.com/wp-content/uploads/2015/05/user-capabilities.png "Ejemplo de configuración de Acceso al POS")

| Capacidad                                                               | Acceso                                           |
| ----------------------------------------------------------------------- | ------------------------------------------------ |
| access_woocommerce_pos                                                  | Requerido para acceder al frontend del POS       |
| manage_woocommerce_pos                                                  | Requerido para acceder a la configuración del POS |
| read_private_products                                                   | Requerido para ver productos                     |
| edit_product <br /> edit_published_products <br /> edit_others_products | Requerido para editar productos y variaciones    |
| read_private_shop_orders                                                | Requerido para ver pedidos y recibos             |
| publish_shop_orders                                                     | Requerido para crear nuevos pedidos              |
| list_users                                                              | Requerido para ver clientes                      |
| create_users                                                            | Requerido para crear nuevos clientes             |
| edit_users                                                              | Requerido para actualizar clientes existentes    |
| read_private_shop_coupons                                               | Requerido para ver cupones                       |
