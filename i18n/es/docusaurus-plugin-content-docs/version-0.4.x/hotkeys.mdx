Los atajos de teclado son combinaciones de teclas que se pueden utilizar para activar rápidamente acciones en WCPOS.

| Tecla                      | Descripción       |
| --------------------------- | ---------------- |
| <div align="center">?</div> | Pantalla de ayuda |
| <div align="center">B</div> | Modo de código de barras |
| <div align="center">V</div> | Modo de búsqueda  |
| <div align="center">S</div> | Sincronizar con el servidor |
