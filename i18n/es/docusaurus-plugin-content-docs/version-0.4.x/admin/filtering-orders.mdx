---
title: Filtrar pedidos por ventas POS o en línea
description: Separar pedidos POS y en línea mediante los enlaces rápidos en el administrador de WooCommerce.
---

Es posible separar rápidamente los pedidos POS y en línea en el administrador de WooCommerce utilizando los enlaces rápidos en la parte superior izquierda.
**Nota:** esta función solo filtrará los pedidos POS a partir de la versión 0.3.1; los pedidos anteriores a la versión 0.3.1 se mostrarán como En línea de forma predeterminada.

![Filtrar pedidos](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filter Orders")

Además, si se desea filtrar todos los pedidos completados en el área de administración para centrarse en los pedidos pendientes de pago o en procesamiento, es posible agregar el siguiente código al archivo [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
