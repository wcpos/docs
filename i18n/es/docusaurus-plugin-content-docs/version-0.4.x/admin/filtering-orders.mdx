---
title: Filtrado de pedidos por ventas de POS o en línea
description: Separe los pedidos de POS y en línea utilizando los enlaces rápidos en el administrador de WooCommerce.
---

Puede separar rápidamente sus pedidos de POS y en línea en el administrador de WooCommerce utilizando los enlaces rápidos en la parte superior izquierda.  
**Por favor, tenga en cuenta:** esto solo filtrará los pedidos de POS después de la versión 0.3.1, los pedidos anteriores a 0.3.1 se establecerán automáticamente en en línea.

![Filtrar pedidos](https://wcpos.com/wp-content/uploads/2014/09/filter-orders-e1409558012346.png "Filtrar pedidos")

Además, si desea filtrar todos los pedidos completados desde su área de administrador para que pueda concentrarse en los pedidos pendientes/procesando, puede agregar el siguiente código a su archivo [functions.php](http://codex.wordpress.org/Functions_File_Explained).

```php title="functions.php"
/**
 * Change default filter of orders in WP admin
 * add this to your functions.php file
 */
add_action( 'admin_menu', 'change_default_order_filter', 99 );
function change_default_order_filter() {
	global $submenu;
	if (isset($submenu['woocommerce'])) {
		$submenu['woocommerce'][1][2] = 'edit.php?post_type=shop_order&shop_order_status=pending,processing';
	}
}
```
