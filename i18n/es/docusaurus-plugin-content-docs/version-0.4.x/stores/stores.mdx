---
title: Tiendas
---

:::important
Esta característica requiere una actualización a [WCPOS Pro](http://wcpos.com/pro).
:::

![Ejemplo de lista de tiendas](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Ejemplo de lista de tiendas")

WCPOS Pro agrega la capacidad de crear _tiendas_ separadas para sus ventas físicas.
Las tiendas pueden tener su propia dirección, logo y otros detalles para usar en la [plantilla de recibo](../receipts).
Las tiendas pueden usar diferentes tasas de impuestos y las ventas se pueden separar por tienda para [informes](../reports#sales-by-store).

![Ejemplo de página de edición de tienda](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Ejemplo de página de edición de tienda")

### Características próximas

La función de Tiendas se agregó en la versión 0.4.
Hay varios _características solicitadas con frecuencia_, como [tiendas de inventario múltiple](./multi-inventory) y la capacidad de asignar cajeros a tiendas.
Estas características se agregarán al complemento Pro a medida que el complemento se acerque a la versión 1.0.
