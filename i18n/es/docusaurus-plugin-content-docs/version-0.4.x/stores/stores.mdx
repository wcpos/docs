---
title: Tiendas
---

:::important
Esta función requiere una actualización a [WCPOS Pro](http://wcpos.com/pro).
:::

![Ejemplo de lista de tiendas](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Example of stores list")

WCPOS Pro añade la posibilidad de crear _tiendas_ independientes para las ventas físicas.
Las tiendas pueden tener su propia dirección, logotipo y otros datos para utilizar en la [plantilla de recibo](../receipts).
Las tiendas pueden utilizar diferentes tasas de impuestos, y las ventas se pueden separar por tienda para los [informes](../reports#sales-by-store).

![Ejemplo de página de edición de tienda](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Example of store edit page")

### Funciones próximas {#coming-features}

La función de Tiendas se añadió en la versión 0.4.
Existen varias _funciones solicitadas con frecuencia_, como las [tiendas con inventario múltiple](./multi-inventory) y la posibilidad de asignar cajeros a las tiendas.
Estas funciones se añadirán al complemento Pro a medida que el complemento se acerque a la versión 1.0.
