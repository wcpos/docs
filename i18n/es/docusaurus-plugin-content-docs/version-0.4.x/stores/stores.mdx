---
title: Tiendas
---

:::important
Esta función requiere una actualización a [WCPOS Pro](http://wcpos.com/pro).
:::

![Ejemplo de lista de tiendas](https://wcpos.com/wp-content/uploads/2016/08/stores-page.png "Ejemplo de lista de tiendas")

WCPOS Pro añade la capacidad de crear _tiendas_ separadas para tus ventas físicas.
Las tiendas pueden tener su propia dirección, logo y otros detalles para usar en la [plantilla de recibo](../receipts).
Las tiendas pueden utilizar diferentes tasas de impuestos y las ventas se pueden separar por tienda para [reportes](../reports#sales-by-store).

![Ejemplo de página de edición de tienda](https://wcpos.com/wp-content/uploads/2016/08/edit-store-page.png "Ejemplo de página de edición de tienda")

### Características venideras

La función de Tiendas se añadió en la versión 0.4.
Hay varias _funciones solicitadas frecuentemente_, como [tiendas con inventario múltiple](./multi-inventory) y la capacidad de asignar cajeros a tiendas.
Estas funciones se añadirán al plugin Pro a medida que el plugin se acerque a la versión 1.0.
