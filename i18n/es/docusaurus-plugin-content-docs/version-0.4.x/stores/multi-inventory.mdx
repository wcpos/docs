---
title: Tiendas de Inventario Múltiple
---

:::caution
El inventario múltiple no es actualmente una característica del complemento Pro.
La siguiente información está destinada a ayudar a los usuarios hasta que WCPOS Pro ofrezca oficialmente soporte para inventario múltiple.
:::

Las tiendas de inventario múltiple es una de las características más solicitadas para WCPOS Pro.
Esta función permitiría a los propietarios de tiendas asignar productos a diferentes tiendas y gestionar el stock entre cada tienda física y la tienda en línea.

Crear funcionalidad de inventario múltiple no es una tarea sencilla.
Ya hay un par de complementos trabajando en el problema, como [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646) y [WooCommerce MultiStore](http://woomultistore.com/).
Actualmente, tiene sentido concentrar el desarrollo en otras características principales del POS e integrarse con uno de estos complementos una vez que maduren como soluciones estables.

A corto plazo, es posible que pueda cumplir con la mayoría de sus requisitos utilizando la [función de Sitios Múltiples de WordPress](https://codex.wordpress.org/Create_A_Network).
Al crear un sitio para cada tienda, puede instalar WooCommerce y WCPOS en cada sitio, manteniendo completamente separado el inventario de cada tienda.
WooCommerce Multisite (mencionado anteriormente) utiliza este enfoque con una capa adicional para gestionar el inventario entre sitios.
