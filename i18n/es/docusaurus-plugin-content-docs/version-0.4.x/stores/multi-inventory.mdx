---
title: Tiendas con inventario múltiple
---

:::caution
El inventario múltiple no es actualmente una función del plugin Pro.
La siguiente información está destinada a ayudar a los usuarios hasta que WCPOS Pro admita oficialmente el inventario múltiple.
:::

Las tiendas con inventario múltiple son una de las funciones más solicitadas para WCPOS Pro.
Esta función permitiría a los propietarios de tiendas asignar productos a diferentes tiendas y gestionar el stock entre cada tienda física y la tienda en línea.

Desarrollar la funcionalidad de inventario múltiple es una tarea considerable.
Ya existen algunos plugins que abordan este problema, como [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646) y [WooCommerce MultiStore](http://woomultistore.com/).
Actualmente tiene más sentido centrar el desarrollo en otras funciones principales del POS e integrar con uno de estos plugins una vez que maduren como soluciones estables.

A corto plazo, es posible cumplir con la mayoría de los requisitos utilizando la [función MultiSite de WordPress](https://codex.wordpress.org/Create_A_Network).
Al crear un sitio para cada tienda, es posible instalar WooCommerce y WCPOS en cada uno, manteniendo el inventario de cada tienda completamente separado.
WooCommerce Multisite (mencionado anteriormente) utiliza este enfoque con una capa adicional para gestionar el inventario entre sitios.
