---
title: "Tiendas de Inventario Múltiple"
---

:::caution
El inventario múltiple actualmente no es una característica del plugin Pro.
La siguiente información está destinada a ayudar a los usuarios hasta que WCPOS Pro soporte oficialmente el inventario múltiple.
:::

Las tiendas de inventario múltiple son una de las características más solicitadas para WCPOS Pro.
Esta característica permitiría a los propietarios de tiendas asignar productos a diferentes tiendas y gestionar el stock entre cada tienda física y la tienda en línea.

Crear la funcionalidad de inventario múltiple es una tarea no trivial.
Hay un par de plugins que ya están trabajando en el problema, como [WooCommerce Warehouses](https://codecanyon.net/item/woocommerce-warehouses/13087646) y [WooCommerce MultiStore](http://woomultistore.com/).
Actualmente, tiene sentido enfocar el desarrollo en otras características centrales del POS e integrarse con uno de estos plugins una vez que maduren a soluciones estables.

A corto plazo, puede que pueda satisfacer la mayoría de sus requisitos utilizando la [funcionalidad MultiSitio de WordPress](https://codex.wordpress.org/Create_A_Network).
Al crear un sitio para cada tienda, puede instalar WooCommerce y WCPOS en cada sitio, manteniendo el inventario de cada tienda completamente separado.
WooCommerce Multisite (mencionado anteriormente) utiliza este enfoque con una capa adicional para gestionar el inventario entre los sitios.
