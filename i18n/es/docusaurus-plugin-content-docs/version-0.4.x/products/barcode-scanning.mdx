title: "Escaneo de Código de Barras"

---

WCPOS puede filtrar productos por el campo SKU. Si una búsqueda coincide exactamente con el campo de código de barras, el producto se agrega al carrito de inmediato y el campo de búsqueda se restablece. Si tienes un escáner de código de barras que puede llenar el campo de búsqueda, entonces deberías usar WCPOS para el escaneo de códigos de barras.

Para ver la búsqueda de SKU en acción, por favor visita http://demo.wcpos.com/pos/

Escribe (o copia y pega) `BARCODE` y un artículo será agregado al carrito.

Escribe (o copia y pega) `VARIATION1` y una variación será agregada al carrito.

![Escaneo de Código de Barras en WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtrando por el campo SKU en WCPOS")

Las siguientes características están siendo planeadas para futuras versiones:

- Agregar productos a WooCommerce usando un escáner de código de barras. Esto será parte de un modo de toma de inventario que se planea para una versión 0.3.x.
- Escaneo de códigos de barras usando la cámara de un dispositivo móvil. Esto requerirá una aplicación nativa de WCPOS para acceder al hardware del dispositivo. Las aplicaciones nativas están en la lista de deseos, pero agregar características a la versión web es la máxima prioridad en este momento.
- Activación por tecla de acceso rápido

### Actualizaciones

**Desde la versión 0.3.1:** Ahora hay un modo dedicado para el escaneo de códigos de barras. Puedes cambiar entre el modo 'Buscar' y 'Escanear Código de Barras' haciendo clic en el botón a la izquierda del campo de búsqueda.
