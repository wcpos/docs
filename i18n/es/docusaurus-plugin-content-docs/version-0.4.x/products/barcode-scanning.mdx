---
title: Escaneo de códigos de barras
description: WCPOS permite filtrar productos por el campo SKU, lo que permite agregar productos al carrito de forma instantánea utilizando un escáner de códigos de barras.
---

WCPOS permite filtrar productos por el campo SKU. Si una búsqueda coincide exactamente con el campo de código de barras, el producto se agrega al carrito de forma instantánea y el campo de búsqueda se restablece. Si se dispone de un escáner de códigos de barras capaz de completar el campo de búsqueda, es posible utilizar WCPOS para el escaneo de códigos de barras.

Para ver la búsqueda por SKU en acción, visite http://demo.wcpos.com/pos/

Escriba (o copie y pegue) `BARCODE` y se agregará un artículo al carrito.

Escriba (o copie y pegue) `VARIATION1` y se agregará una variación al carrito.

![Escaneo de códigos de barras en WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtering by the SKU field in WCPOS")

Las siguientes funcionalidades están planificadas para futuras versiones:

- Agregar productos a WooCommerce mediante un escáner de códigos de barras. Esta funcionalidad formará parte de un modo de inventario planificado para una versión 0.3.x.
- Escaneo de códigos de barras mediante la cámara del dispositivo móvil. Esto requerirá una aplicación nativa de WCPOS para acceder al hardware del dispositivo. Las aplicaciones nativas están en la lista de deseos, pero agregar funcionalidades a la versión web es la máxima prioridad en este momento.
- Activación mediante tecla de acceso rápido

### Actualizaciones {#updates}

**Desde la versión 0.3.1:** Ahora existe un modo dedicado para el escaneo de códigos de barras. Es posible alternar entre el modo 'Búsqueda' y 'Escanear código de barras' haciendo clic en el botón situado a la izquierda del campo de búsqueda.
