---
title: Escaneo de Código de Barras
description: WCPOS puede filtrar productos por el campo SKU, lo que permite agregar productos al carrito instantáneamente usando un escáner de códigos de barras.
---

WCPOS puede filtrar productos por el campo SKU. Si una búsqueda coincide exactamente con el campo de código de barras, el producto se agrega instantáneamente al carrito y el campo de búsqueda se restablece. Si tiene un escáner de códigos de barras capaz de completar el campo de búsqueda, debe usar WCPOS para escanear códigos de barras.

Para ver la búsqueda de SKU en acción, visite http://demo.wcpos.com/pos/

Escriba (o copie y pegue) `BARCODE` y se agregará un artículo al carrito.

Escriba (o copie y pegue) `VARIATION1` y se agregará una variación al carrito.

![Escaneo de Código de Barras en WCPOS](https://wcpos.com/wp-content/uploads/2014/07/barcode-search.png "Filtrado por el campo SKU en WCPOS")

Las siguientes características están planificadas para futuras versiones:

- Agregar productos a WooCommerce usando un escáner de códigos de barras. Esto será parte de un modo de inventario que se está planificando para una versión 0.3.x.
- Escaneo de códigos de barras utilizando la cámara de un dispositivo móvil. Esto requerirá una aplicación nativa de WCPOS para acceder al hardware del dispositivo. Las aplicaciones nativas están en la lista de deseos, pero agregar características a la versión web es la prioridad más alta en este momento.
- Activación con tecla de acceso rápido

### Actualizaciones

**Desde la Versión 0.3.1:** Ahora hay un modo dedicado para el escaneo de códigos de barras. Puede cambiar entre el modo 'Buscar' y 'Escanear Código de Barras' haciendo clic en el botón a la izquierda del campo de búsqueda.
