---
title: Productos solo para POS
description: Los productos pueden ocultarse en tu tienda en línea o en el punto de venta utilizando la configuración de visibilidad de POS.
---

Puede haber situaciones en las que necesites ocultar un producto de tu tienda en línea o de tu POS.
Con WCPOS esto se puede lograr usando la configuración de Visibilidad de POS en la página de edición del producto.

### Habilitando Productos Solo para POS

Los Productos Solo para POS no están habilitados de forma predeterminada.
Para habilitar Productos Solo para POS, ve a <code>WP Admin > POS > Configuración > General</code>

![Habilitar Productos Solo para POS](https://wcpos.com/wp-content/uploads/2014/09/enable-pos-only-products.png "Habilitar Productos Solo para POS")

Habilitar productos solo para POS requerirá una consulta adicional a la base de datos y, por lo tanto, incurrirá en una pequeña penalización de rendimiento.
Si no utilizas Productos Solo para POS, se recomienda dejar esta opción deshabilitada.

### Visibilidad de POS y en Línea

![Visibilidad de POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility.png "Configuración de Visibilidad de POS en la página de edición del producto")

| Configuración                   | Descripción                                                                                        |
| ------------------------------- | -------------------------------------------------------------------------------------------------- |
| **POS y En Línea**<br />(predeterminado) | Los productos serán visibles tanto en línea en tu tienda web como en tu tienda física               |
| **Solo POS**                    | La página del producto no será visible en línea                                                    |
| **Solo En Línea**               | El producto no será visible en el POS                                                              |

### Visibilidad del Catálogo

Si vas directamente a la página de un Producto Solo para POS, obtendrás <code>404 Página No Encontrada</code>.
Sin embargo, los Productos Solo para POS pueden seguir apareciendo en las páginas de búsqueda y categorías en línea.
Para ocultar completamente los productos de tu sitio web, debes configurar la Visibilidad del Catálogo como Oculto.

![Visibilidad del Catálogo](https://wcpos.com/wp-content/uploads/2016/08/catalog-visibility.png "Configuración de Visibilidad del Catálogo en la página de edición del producto")

### Filtrar Productos Solo para POS

Puedes ver rápidamente qué productos están visibles/ocultos usando los enlaces rápidos en la página de administración de Productos:

![Filtro de Visibilidad de POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-filter.png "Filtro de Visibilidad de POS")

### Edición Masiva

Para cambiar la Visibilidad de POS de varios productos, ve a tu lista de Productos y selecciona <code>Editar</code> desde las <code>Acciones Masivas</code>.

![Edición Masiva de Visibilidad de POS](https://wcpos.com/wp-content/uploads/2016/08/pos-visibility-bulk-edit.png "Edición masiva de Visibilidad de POS")
