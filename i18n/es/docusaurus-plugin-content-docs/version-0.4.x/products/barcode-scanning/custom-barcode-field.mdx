---
title: Campo de Código de Barras Personalizado
---

Por defecto, WCPOS utilizará el campo SKU como el código de barras.

![Por defecto, WCPOS utilizará el campo SKU como el código de barras. ](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "Por defecto, WCPOS utilizará el campo SKU como el código de barras.")

Los usuarios de [WCPOS Pro](http://wcpos.com/pro) pueden usar cualquier campo meta de producto seleccionando un Campo de Código de Barras en `WP Admin > POS > Configuración > General`

![Seleccionar campo de código de barras personalizado en WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Seleccionar campo de código de barras personalizado en WCPOS Pro")

Los campos meta de código de barras personalizados también se pueden establecer utilizando el filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Custom product meta field to use as barcode
 *
 * By default WCPOS will use the SKU field- '_sku'
 * You can change the meta key using the following WordPress filter
 *
 * Below are meta fields from popular barcode plugins
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // change for your custom meta key
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un código de barras debe ser único para cada producto.
:::
