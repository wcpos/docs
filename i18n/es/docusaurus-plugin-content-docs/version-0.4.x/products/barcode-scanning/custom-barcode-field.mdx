---
title: Campo de código de barras personalizado
---

De manera predeterminada, WCPOS utilizará el campo de SKU como código de barras.

![De manera predeterminada, WCPOS utilizará el campo de SKU como código de barras. ](https://wcpos.com/wp-content/uploads/2016/06/product-sku.png "De manera predeterminada, WCPOS utilizará el campo de SKU como código de barras")

Los usuarios de [WCPOS Pro](http://wcpos.com/pro) pueden utilizar cualquier campo de metadatos del producto seleccionando un Campo de código de barras en `WP Admin > POS > Configuración > General`

![Seleccionar campo de código de barras personalizado en WCPOS Pro](https://wcpos.com/wp-content/uploads/2016/06/select-custom-barcode-field.png "Seleccionar campo de código de barras personalizado en WCPOS Pro")

Los campos de metadatos de códigos de barras personalizados también se pueden configurar utilizando el filtro `woocommerce_pos_barcode_meta_key`.

```php title="functions.php"
/**
 * Campo de metadatos de producto personalizado para usar como código de barras
 *
 * De manera predeterminada, WCPOS utilizará el campo de SKU - '_sku'
 * Puede cambiar la clave de metadatos utilizando el siguiente filtro de WordPress
 *
 * A continuación se encuentran los campos de metadatos de plugins de códigos de barras populares
 * '_ywbc_barcode_value' - YITH WooCommerce Barcodes Premium
 */

function my_custom_pos_barcode_meta_key(){
  return '_barcode'; // cambiar por su clave de metadatos personalizada
}

add_filter('woocommerce_pos_barcode_meta_key', 'my_custom_pos_barcode_meta_key' );
```

:::danger
Un código de barras debe ser único para cada producto.
:::
