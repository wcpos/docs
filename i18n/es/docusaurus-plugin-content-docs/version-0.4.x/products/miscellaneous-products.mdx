---
title: Productos Varios
---

Puede haber casos en los que una tienda necesite vender artículos que no forman parte del inventario de WooCommerce.  
A continuación se presentan dos opciones para vender 'productos varios' en WCPOS.

### Producto Vario Solo POS (recomendado)

![Producto Vario Solo POS](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Crear un Producto Vario Solo POS")

Usando la función [Solo POS](./pos-only-products), puedes crear un 'producto vario' para usar en el POS.  
El título, el precio de venta, el precio regular, la tasa impositiva y la información meta se pueden cambiar sobre la marcha durante el proceso de pago.

![Editar el Producto Vario](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Editar el Producto Vario en el carrito")

### Cargo adicional

![Cargo adicional](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Editar el Cargo adicional")

Usar un Cargo adicional es similar a crear un Producto Vario Solo POS.  
Los cajeros pueden actualizar rápidamente el título, el precio y la tasa impositiva del Cargo.  
El principal inconveniente es que los Cargos pueden aparecer separados de los productos en tus informes.
