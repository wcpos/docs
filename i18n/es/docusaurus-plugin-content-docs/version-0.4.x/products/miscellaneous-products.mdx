---
title: Productos varios
---

Puede haber casos en los que una tienda necesite vender artículos que no forman parte del inventario de WooCommerce.
A continuación se presentan dos opciones para vender 'productos varios' en WCPOS.

### Producto varios solo para POS (recomendado) {#pos-only-miscellaneous-product-recommended}

![Producto varios solo para POS](https://wcpos.com/wp-content/uploads/2016/06/miscellaneous-product.png "Create a POS Only Miscellaneous Product")

Mediante la función [Solo para POS](./pos-only-products), es posible crear un 'producto varios' para su uso en el POS.
El título, el precio de oferta, el precio regular, la tasa de impuestos y la metainformación pueden modificarse sobre la marcha durante el proceso de cobro.

![Editar el producto varios](https://wcpos.com/wp-content/uploads/2016/06/custom-product-in-cart.png "Edit the Miscellaneous Product in the cart")

### Línea de cargo {#fee-line-item}

![Línea de cargo](https://wcpos.com/wp-content/uploads/2016/06/fee-line-item.png "Edit the Fee line item")

El uso de una línea de cargo es similar a la creación de un producto varios solo para POS.
Los cajeros pueden actualizar rápidamente el título, el precio y la tasa de impuestos del cargo.
La principal desventaja es que las tarifas pueden aparecer separadas de los productos en los informes.
