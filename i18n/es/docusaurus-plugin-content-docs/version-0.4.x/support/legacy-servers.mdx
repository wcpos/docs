---
title: Servidores heredados
description: Si el proveedor de alojamiento web bloquea los métodos HTTP RESTful modernos, será necesario habilitar la compatibilidad con servidores heredados.
---

HTTP (y HTTPS) son los protocolos utilizados por todos los navegadores para comunicarse con las aplicaciones web.
Una parte fundamental del protocolo es el [método HTTP](https://tools.ietf.org/html/rfc2616#section-5.1.1).
Tradicionalmente, las aplicaciones web utilizaban un conjunto muy limitado de métodos HTTP, como GET (recuperar una página web) y POST (enviar un formulario web).

Las aplicaciones web modernas, como WCPOS, utilizan un enfoque REST/HTTP con los siguientes métodos HTTP:

- GET
- POST
- PUT
- PATCH
- DELETE

En algunos casos excepcionales, un proveedor de alojamiento web puede optar por bloquear algunos de estos métodos HTTP.
Si el servidor web bloquea los métodos HTTP RESTful modernos, será necesario habilitar la compatibilidad con servidores legacy.
