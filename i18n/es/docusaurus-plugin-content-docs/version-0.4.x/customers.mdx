### Capacidades

Los cajeros necesitarán los permisos _list_users_, _create_users_ y _edit_users_ para utilizar todas las funciones de Clientes.

### Sitios Múltiples de WordPress

Al usar [sitios múltiples de WordPress](https://codex.wordpress.org/Create_A_Network) hay algunas [comprobaciones adicionales](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) para evitar que los usuarios que no son [Superadministradores](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) creen y editen clientes.

Primero, para permitir que los usuarios que no son Superadministradores **creen** nuevos clientes, habilite la siguiente opción en `Administración de Red > Configuración`.

![Opción de sitio múltiple para crear nuevos usuarios](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Opción de sitio múltiple para crear nuevos usuarios")

Adicionalmente, para permitir que los usuarios que no son Superadministradores **editen** detalles de clientes, necesitará agregar el permiso `manage_network_users` al cajero.

```php title="functions.php"
// cambiar CASHIER_ID, y
// agregar a su archivo functions.php

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
