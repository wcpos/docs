### Capacidades {#capabilities}

Los cajeros necesitarán las capacidades _list_users_, _create_users_ y _edit_users_ para usar todas las funciones de clientes.

### WordPress multisitio {#wordpress-multisite}

Al usar [WordPress multisitio](https://codex.wordpress.org/Create_A_Network), existen algunas [comprobaciones adicionales](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) para impedir que quienes no sean [superadministradores](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) creen y editen clientes.

En primer lugar, para que quienes no sean superadministradores puedan **crear** nuevos clientes, active la siguiente opción en `Network Admin > Ajustes`.

![Opción multisitio para crear nuevos usuarios](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Opción multisitio para crear nuevos usuarios")

Además, para que quienes no sean superadministradores puedan **editar** los detalles de clientes, deberá añadir la capacidad `manage_network_users` al cajero.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
