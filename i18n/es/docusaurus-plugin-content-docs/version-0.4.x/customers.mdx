### Capacidades {#capabilities}

Los cajeros necesitarán las capacidades _list_users_, _create_users_ y _edit_users_ para usar todas las funciones de Clientes.

### WordPress Multisite

Al usar [WordPress multisite](https://codex.wordpress.org/Create_A_Network), hay algunas [verificaciones adicionales](https://github.com/WordPress/WordPress/blob/master/wp-includes/capabilities.php) para evitar que los no [Super Admins](https://codex.wordpress.org/Roles_and_Capabilities#Super_Admin) creen y editen clientes.

Primero, para que los no Super Admins puedan **crear** nuevos clientes, habilite la siguiente opción en `Network Admin > Settings`.

![Opción multisite para crear nuevos usuarios](https://wcpos.com/wp-content/uploads/2016/09/multisite-create-new-users.png "Opción multisite para crear nuevos usuarios")

Además, para que los no Super Admins puedan **editar** los detalles del cliente, deberá agregar la capacidad `manage_network_users` al cajero.

```php title="functions.php"
// change the CASHIER_ID, and
// add to your functions.php file

add_filter('map_meta_cap', 'wc_pos_map_meta_cap', 10, 3);

function wc_pos_map_meta_cap($caps, $cap, $user_id){
  if(is_pos() && $cap == 'edit_users' && $user_id === CASHIER_ID){
    $caps = array('edit_users');
  }
  return $caps;
}
```
