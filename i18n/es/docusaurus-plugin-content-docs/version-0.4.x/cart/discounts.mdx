---

### Precio del artículo {#item-price}

Al hacer clic en el precio del artículo es posible modificar el precio _sobre la marcha_.
El teclado numérico de precio del artículo permite introducir un nuevo precio de venta o un descuento como porcentaje.

![Reducción del precio del artículo mediante un porcentaje usando el teclado numérico](https://wcpos.com/wp-content/uploads/2016/08/product-discount.png "Reducing the item price by a percentage using the number pad")

### Descuento de pedido {#order-discount}

Es posible aplicar un descuento de pedido añadiendo un cargo negativo al carrito.
Al hacer clic en el título se puede cambiar el nombre del cargo.
Al hacer clic en la flecha se puede cambiar el estado fiscal y la tasa.

![Teclado numérico de descuento de pedido](https://wcpos.com/wp-content/uploads/2016/08/negative-fee.png "Using a negative Fee to apply an Order Discount")
