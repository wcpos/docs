WCPOS utiliza varios componentes de teclado numérico para permitir la entrada rápida mediante el ratón o la pantalla táctil.
A continuación se muestran ejemplos de los diferentes tipos de teclados numéricos:

![Teclado numérico de cantidad con entrada rápida para +1 y -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Quantity number pad with quick entry for +1 and -1")

![Teclado numérico de precio del artículo con entrada rápida para porcentaje de descuento.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Item Price number pad with quick entry for discount percentage")

![Teclado numérico de monto entregado con entrada rápida para importes en efectivo habituales.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Amount Tendered number pad with quick entry for likely cash amounts")

![Teclado numérico de reembolso en efectivo con entrada rápida para importes de reembolso habituales.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Cashback number pad with quick entry for common cashback amounts")
