WCPOS utiliza una serie de componentes de teclado numérico para permitir una entrada rápida mediante el ratón o la pantalla táctil.
A continuación se presentan ejemplos de los diferentes tipos de teclados numéricos:

![Teclado numérico de cantidad con entrada rápida para +1 y -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Teclado numérico de cantidad con entrada rápida para +1 y -1")

![Teclado numérico de precio de artículo con entrada rápida para porcentaje de descuento.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Teclado numérico de precio de artículo con entrada rápida para porcentaje de descuento")

![Teclado numérico de cantidad entregada con entrada rápida para cantidades de efectivo probables.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Teclado numérico de cantidad entregada con entrada rápida para cantidades de efectivo probables")

![Teclado numérico de reembolso con entrada rápida para cantidades de reembolso comunes.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Teclado numérico de reembolso con entrada rápida para cantidades de reembolso comunes")
