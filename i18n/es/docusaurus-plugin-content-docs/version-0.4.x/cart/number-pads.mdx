WCPOS utiliza una serie de componentes de teclados numéricos para permitir la rápida introducción de datos mediante el ratón o la pantalla táctil.
A continuación se presentan ejemplos de los diferentes tipos de teclados numéricos:

![Teclado numérico de cantidad con entrada rápida para +1 y -1.](https://wcpos.com/wp-content/uploads/2014/09/quantity-numpad.png "Teclado numérico de cantidad con entrada rápida para +1 y -1")

![Teclado numérico de Item Price con entrada rápida para porcentaje de descuento.](https://wcpos.com/wp-content/uploads/2014/09/item-price-numpad-amount.png "Teclado numérico de Item Price con entrada rápida para porcentaje de descuento")

![Teclado numérico de Amount Tendered con entrada rápida para cantidades probables de efectivo.](https://wcpos.com/wp-content/uploads/2014/09/cash-sale-numpad.png "Teclado numérico de Amount Tendered con entrada rápida para cantidades probables de efectivo")

![Teclado numérico de Cashback con entrada rápida para cantidades comunes de cashback.](https://wcpos.com/wp-content/uploads/2014/09/card-sale-numpad.png "Teclado numérico de Cashback con entrada rápida para cantidades comunes de cashback")
