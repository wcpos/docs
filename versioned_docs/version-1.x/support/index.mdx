Need help with WCPOS? Here's how to get support and find answers to your questions.

## In-App Support

The POS includes a built-in Support screen accessible from the left navigation drawer. This screen embeds the WCPOS Discord community directly in the app, allowing you to:

- Browse existing discussions
- Search for answers to common questions
- Ask questions in the #general channel
- Stay updated with announcements

## Discord Community

The primary support channel for WCPOS is our [Discord server](https://wcpos.com/discord).

### Channels

| Channel | Purpose |
|---------|---------|
| **#welcome-and-rules** | Community guidelines |
| **#announcements** | Updates and news |
| **#general** | Main support channel - ask questions here |
| **#forum** | Longer discussions and feature requests |
| **#github-activity** | Development updates |

### Before Asking

To get the best help:

1. **Test first** - Download and test the plugin for a few days to understand if it meets your needs
2. **Check the docs** - Many questions are answered in this documentation
3. **Search Discord** - Your question may have already been answered
4. **Provide details** - Include your WooCommerce version, WCPOS version, and steps to reproduce issues

## Email Support

For license-related issues or private inquiries, you can email: **support@wcpos.com**

:::note
General support questions should go to Discord where the community can help and others can benefit from the answers.
:::

## Logs Screen

The POS includes a [Logs screen](/support/logs) that records system activity. This is invaluable for:

- Debugging sync issues
- Understanding error messages
- Providing detailed information when requesting support

When asking for help, check the Logs screen for any error messages and include them in your support request.

## Pro Support

[WCPOS Pro](https://wcpos.com/pro) customers receive the same community support through Discord. The Pro license funds continued development of both free and Pro features.
