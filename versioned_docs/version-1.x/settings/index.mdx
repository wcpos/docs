WCPOS has three types of settings organised by where they're configured.

## Where to Find Settings

### 1. WordPress Admin Settings

Backend settings configured in your WordPress dashboard:

**Location:** `WP Admin > POS > Settings`

These control:
- Default cart customer for all cashiers
- Product barcode field configuration
- Order email settings
- User role permissions (who can access POS)
- Payment gateway availability

See [WordPress Admin Settings](/settings/wp-admin) for details.

### 2. POS Store Settings

App-wide settings configured within the POS application:

**Location:** User menu (top-right) â†’ Settings, or `Ctrl + Shift + S`

These control:
- Tax calculation options
- Barcode scanner configuration
- Keyboard shortcuts
- Theme/appearance

See [Store Settings](/settings/store) for details.

### 3. Display Settings (Per Screen)

Each screen in the POS has its own display settings for customizing the data table:

**Location:** Click the sliders icon (<Icon name="sliders" />) on any screen

These control:
- Which columns are visible
- Column display options
- Screen-specific features

Display settings are documented within each screen's documentation:
- [POS Product Panel](/pos/product-panel#display-settings)
- [Cart Panel](/pos/cart#display-settings)
- [Products Screen](/products#display-settings)
- [Orders Screen](/orders#display-settings)
- [Customers Screen](/customers#display-settings)
- [Reports Screen](/reports#display-settings)
