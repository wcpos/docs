# Payment Gateways

WCPOS supports several custom payment gateways that extend the functionality of your Point of Sale system. These gateways are designed specifically for POS environments and provide seamless integration with various payment methods and services.

## Available Custom Gateways {#available-custom-gateways}

### Hardware Terminal Gateways {#hardware-terminal-gateways}

- **[Stripe Terminal](./stripe-terminal)** - Accept payments using Stripe Terminal hardware readers
- **[SumUp Terminal](./sumup-terminal)** - Process payments through SumUp card readers

### Mobile Payment Gateways {#mobile-payment-gateways}

- **[Vipps MobilePay](./vipps-mobilepay)** - Accept phone-based payments via QR code or push notification

### Digital Payment Gateways {#digital-payment-gateways}

- **[Email Invoice](./email-invoice)** - Send payment invoices to customers via email
- **[Web Checkout](./web-checkout)** - Redirect customers to complete payments online

## Creating Your Own Gateway {#creating-your-own-gateway}

If you need a custom payment gateway for your specific requirements, you can use our **[Gateway Template](/reference/gateway-template)** to create your own WCPOS payment gateway.

## Installation Overview {#installation-overview}

All custom gateways follow a similar installation process:

1. **Download** the latest release from the respective GitHub repository
2. **Install** the plugin via `WP Admin > Plugins > Add New > Upload Plugin`
3. **Configure** the gateway settings in `WP Admin > WooCommerce > Settings > Payments`
4. **Enable** the gateway in `WP Admin > POS > Settings > Checkout`
