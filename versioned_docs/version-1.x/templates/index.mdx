Templates allow you to customize the appearance and content of various outputs from your POS system. You can access the template editor by navigating to: `WP Admin > POS > Templates`

The template system provides flexibility for customizing:

- **Receipt Templates** - Customize printed and digital receipts
- **Report Templates** - Customize report layouts _(coming soon)_
- **Email Templates** - Customize email notifications _(coming soon)_

## Template Types

### Default Templates

Automatically detected from your plugin and theme files. These cannot be deleted but can be copied to create custom versions.

### Custom Templates

User-created templates that you can fully customize, edit, and manage through the WordPress admin interface.
